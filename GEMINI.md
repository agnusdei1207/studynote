## 기본 규칙

1. **스타일**: 서술형 + ASCII 다이어그램 하이브리드
2. **구조**: 핵심 인사이트 → Ⅰ~Ⅴ → 개념 맵 → 어린이 비유
3. **넘버링**: 각 과목 `_keyword_list.md` 순서 엄수
4. **커밋**: 3~5개마다 `git add -A && git commit -m "feat: [과목] #[범위] [주제]" && git push`
5. **업데이트**: 작업 후 아래 "현재 위치"만 수정
