# 수학 기호 용어집 (Mathematical Symbol Glossary)

정규분포 및 통계학에서 자주 사용되는 기호들의 의미, 발음, 사용 맥락을 상세히 설명합니다.

---

## 1. 그리스어 알파벳 (Greek Letters)

### 1.1 μ (Mu) - 뮤
```
소문자: μ (mu)
대문자: Μ (Mu)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 뮤 [뮤] |
| **영어 발음** | Mu [muː] 또는 [mjuː] |
| **의미** | 모평균(Population Mean), 집단 평균 |
| **사용 예시** | $X \sim N(\mu, \sigma^2)$: "X는 평균이 뮤이고 분산이 시그마 제곱인 정규분포를 따른다" |
| **관련 용어** | 모평균(Population Mean), 기대값(Expectation) |

**상세 설명**:
- **평균(Average)**을 나타내는 가장 기본적인 기호
- 전체 집단(모집단)의 평균을 의미할 때 $\mu$를 사용
- 표본(샘플)의 평균은 $\bar{x}$ (x-bar)로 표기하여 구분
- 정규분포에서 $\mu$는 종형 곡선의 가장 높은 지점, 즉 중심 위치를 나타냄

---

### 1.2 σ (Sigma) - 시그마
```
소문자: σ (sigma)
대문자: Σ (Sigma)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 시그마 [시-그-마] |
| **영어 발음** | Sigma [ˈsɪɡmə] |
| **소문자 σ의 의미** | 모표준편차(Population Standard Deviation) |
| **대문자 Σ의 의미** | 합(Summation), 시그마 연산자 |
| **사용 예시** | $\sigma^2$: "시그마 제곱은 분산" |
| **관련 용어** | 표준편차(Standard Deviation), 분산(Variance), 합(Sum) |

**상세 설명**:
**소문자 σ (시그마)**:
- 데이터가 평균에서 얼마나 퍼져 있는지 나타내는 척도
- **표준편차(Standard Deviation)**: 데이터의 퍼짐 정도를 원래 단위로 표현
  - 예: 키의 표준편차가 5cm라면 "대부분의 사람은 평균 키에서 ±5cm 정도 벗어남"
- **분산(Variance)**: 표준편차의 제곱인 $\sigma^2$로 표기
  - $\sigma^2 = V(X)$

**대문자 Σ (시그마)**:
- **합(Summation)**을 나타내는 연산자
- $\sum_{i=1}^{n} x_i$: "x₁부터 xₙ까지의 합"
- $\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n$

### 1.3 π (Pi) - 파이
```
소문자: π (pi)
대문자: Π (Pi)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 파이 [파-이] |
| **영어 발음** | Pi [paɪ] |
| **의미** | 원주율(Circumference Ratio) |
| **수값** | $\pi \approx 3.1415926535\dots$ (무리수) |
| **사용 예시** | $2\pi r$: "원의 둘레는 2곱하기 파이 곱하기 반지름" |
| **관련 용어** | 원주율, 무리수(Irrational Number) |

**상세 설명**:
- 원의 둘레(Circumference)를 지름(Diameter)으로 나눈 값
- 정규분포의 확률밀도함수에 등장하는 상수
- $\frac{1}{\sqrt{2\pi}}$: 정규분포의 정규화 상수(Normalizing Constant)
- 모든 원에 대해 동일한 값으로, 수학에서 가장 유명한 상수 중 하나

### 1.4 e - 자연 상수 (Euler's Number)
```
소문자: e
대문자: E (보통 다른 용도로 사용)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 이 [이] (자연 상수) |
| **영어 발음** | E [iː] (Euler's Number) |
| **의미** | 자연 상수(Natural Constant), 오일러 수 |
| **수값** | $e \approx 2.718281828\dots$ (무리수) |
| **사용 예시** | $e^x$: "이의 x제곱", 지수 함수(Exponential Function) |
| **관련 용어** | 지수함수, 자연로그(Natural Logarithm, ln), 오일러 상수 |

**상세 설명**:
- 자연의 성장, 붕괴, 확률 현상에 등장하는 중요한 상수
- $e = \lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n$
- 지수함수 $e^x$는 자신의 미분과 동일하다는 독특한 성질
- 정규분포에서 $e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ 형태로 등장하여 종형 곡선 형성

### 1.5 λ (Lambda) - 람다
```
소문자: λ (lambda)
대문자: Λ (Lambda)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 람다 [람-다] |
| **영어 발음** | Lambda [ˈlæmdə] |
| **의미** | 비율(Rate), 평균 발생 횟수 (주로 포이송분포, 지수분포) |
| **사용 예시** | $P(X=k) = \frac{e^{-\lambda}\lambda^k}{k!}$: "포이송분포에서 람다는 단위 시간당 평균 발생 횟수" |
| **관련 용어** | 포이송분포(Poisson Distribution), 비율 파라미터(Rate Parameter) |

**상세 설명**:
- **포이송분포(Poisson Distribution)**: 단위 시간/공간에서 사건이 평균적으로 몇 번 발생하는지 나타냄
- **지수분포(Exponential Distribution)**: 사건이 발생하는 비율(Rate)을 나타냄
- 람다가 클수록 사건이 더 자주 발생함
- 예: "1시간당 평균 5건의 오류가 발생한다" → $\lambda = 5$

---

### 1.6 ν (Nu) - 뉴
```
소문자: ν (nu)
대문자: N (Nu, 자주 안 쓰임)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 뉴 [뉴] |
| **영어 발음** | Nu [nuː] 또는 [njuː] |
| **의미** | 자유도(Degrees of Freedom), 주로 t-분포, 카이제곱분포에서 사용 |
| **사용 예시** | $t_{\nu}$: "자유도가 뉴인 t-분포" |
| **관련 용어** | 자유도(Degrees of Freedom, df) |

**상세 설명**:
- **자유도(Degrees of Freedom)**: 통계적 추정에서 독립적으로 변할 수 있는 데이터의 개수
- 예: 10개의 데이터가 있고 합이 100으로 고정되어 있다면, 9개는 자유롭게 정할 수 있고 마지막 1개는 자동으로 결정 → 자유도는 9
- t-분포에서 자유도가 낮을수록 꼬리가 두꺼워지고, 자유도가 커질수록 정규분포에 가까워짐
- 카이제곱분포에서 자유도는 포함된 독립 정규분포 변수의 개수

---

### 1.7 Φ, φ (Phi) - 파이
```
소문자: φ (phi)
대문자: Φ (Phi)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 파이 [파-이], [파-이] (영어: 파이 또는 피) |
| **영어 발음** | Phi [faɪ] 또는 [fiː] |
| **대문자 Φ의 의미** | 표준정규분포의 누적 분포 함수 (CDF) |
| **소문자 φ의 의미** | 표준정규분포의 확률밀도 함수 (PDF) |
| **사용 예시** | $\Phi(z)$: "표준정규분포에서 z보다 작거나 같은 확률" |
| **관련 용어** | 누적분포함수(CDF), 확률밀도함수(PDF) |

**상세 설명**:
**대문자 Φ (파이)**:
- 표준정규분포의 **누적 분포 함수(Cumulative Distribution Function, CDF)**
- $\Phi(z) = P(Z \le z)$: "표준정규분포에서 Z가 z보다 작거나 같을 확률"
- z=0일 때 $\Phi(0) = 0.5$ (정규분포의 절반)
- 표준정규분포표에서 $\Phi(z)$ 값들이 표로 제공됨

**소문자 φ (파이)**:
- 표준정규분포의 **확률밀도 함수(Probability Density Function, PDF)**
- $\phi(z) = \frac{1}{\sqrt{2\pi}} e^{-z^2/2}$: "z에서의 확률밀도"
- 종형 곡선의 높이를 나타냄

## 2. 수학 연산 기호

### 2.1 ∞ (Infinity) - 무한대
```
기호: ∞
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 무한대 [무-한-대] |
| **영어 발음** | Infinity [ɪnˈfɪnəti] |
| **의미** | 끝이 없음, 한계가 없음 |
| **사용 예시** | $(-\infty, \infty)$: "음의 무한대부터 양의 무한대까지" |
| **관련 용어** | 극한(Limit), 발산(Divergence) |

**상세 설명**:
- 정규분포의 정의역: $(-\infty, \infty)$ - 모든 실수 값
- 확률밀도함수가 끝없이 0에 가까워지지만 결코 0이 되지 않음
- 실무적으로 3$\sigma$ 범위를 "거의 전체"로 취급 (99.73%)
- 음의 무한대: $-\infty$ (마이너스 무한대), 양의 무한대: $\infty$

### 2.2 ∑ (Summation) - 시그마 (합)
```
기호: ∑
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 시그마 [시-그-마], 합 [합] |
| **영어 발음** | Sigma [ˈsɪɡmə] 또는 Summation [ˌsʌmˈmeɪʃn] |
| **의미** | 합(Sum), 시그마 연산자 |
| **사용 예시** | $\sum_{i=1}^{n} x_i$: "i는 1부터 n까지, 모든 x_i를 더하라" |
| **관련 용어** | 합(Sum), 연속 합(Consecutive Sum) |

**상세 설명**:
- $\sum_{i=1}^{n} x_i = x_1 + x_2 + x_3 + \cdots + x_n$
- **기대값(Expectation)** 계산에서 사용:
  - $E(X) = \sum_{i=1}^{n} x_i P(X=x_i)$: "각 값과 그 확률의 곱을 모두 더한 것"
- **분산(Variance)** 계산에서 사용:
  - $V(X) = E((X-\mu)^2) = \sum_{i=1}^{n} (x_i - \mu)^2 P(X=x_i)$

**계산 예시**:
- $\sum_{i=1}^{3} i = 1 + 2 + 3 = 6$
- $\sum_{i=1}^{4} 2i = 2 + 4 + 6 + 8 = 20$

---

### 2.3 ∫ (Integral) - 적분
```
기호: ∫
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 적분 [적-분], 인테그랄 |
| **영어 발음** | Integral [ˈɪntɪɡrəl] |
| **의미** | 면적(Area), 누적 합(Accumulated Sum) |
| **사용 예시** | $\int_{-\infty}^{z} \phi(t) \, dt$: "음의 무한대부터 z까지 φ(t)를 적분" |
| **관련 용어** | 누적 분포 함수(CDF), 면적(Area) |

**상세 설명**:
- **적분(Integration)**: 함수 아래의 면적을 계산
- **정적분(Definite Integral)**: 특정 구간 $[a, b]$에서의 면적
  - $\int_{a}^{b} f(x) \, dx$: "f(x)를 a부터 b까지 적분"
- **누적 분포 함수(CDF)** 정의:
  - $\Phi(z) = \int_{-\infty}^{z} \phi(t) \, dt$: "음의 무한대부터 z까지 확률밀도함수 아래의 면적"
- 확률 = 면적: 정규분포에서 특정 구간의 확률 = 해당 구간의 면적

### 2.4 √ (Square Root) - 제곱근, 루트
```
기호: √
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 제곱근 [제-곤-근], 루트 [루-트] |
| **영어 발음** | Square Root [skweər ruːt] 또는 Radical [ˈrædɪkəl] |
| **의미** | 제곱해서 원래 수가 되는 값 |
| **사용 예시** | $\sqrt{9} = 3$, $\sqrt{x}$ |
| **관련 용어** | 제곱(Square), 거듭제곱(Exponentiation) |

**상세 설명**:
- $\sqrt{x}$: "x의 제곱근", "x에 루트를 씌운다"
- 표준편차와 분산의 관계: $\sigma = \sqrt{\sigma^2}$
- 정규분포의 정규화 상수: $\frac{1}{\sigma\sqrt{2\pi}}$
- $\sqrt{2\pi} \approx \sqrt{6.283} \approx 2.507$

### 2.5 ! (Factorial) - 팩토리얼
```
기호: !
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 팩토리얼 [팩-토-리-얼] |
| **영어 발음** | Factorial [fækˈtɔːriəl] |
| **의미** | 1부터 그 수까지의 모든 정수의 곱 |
| **사용 예시** | $n! = n \times (n-1) \times \cdots \times 1$, $5! = 5 \times 4 \times 3 \times 2 \times 1 = 120$ |
| **관련 용어** | 순열(Permutation), 조합(Combination) |

**상세 설명**:
- $n! = n \times (n-1) \times (n-2) \times \cdots \times 2 \times 1$
- 특별한 경우: $0! = 1$, $1! = 1$
- 포이송분포 공식에서 등장: $\frac{e^{-\lambda}\lambda^k}{k!}$
- **계산 예시**:
  - $3! = 3 \times 2 \times 1 = 6$
  - $4! = 4 \times 3 \times 2 \times 1 = 24$
  - $5! = 5 \times 4 \times 3 \times 2 \times 1 = 120$

## 3. 확률/통계 기호

### 3.1 E(X) - 기대값
```
기호: E(X)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 기대값 [기-대-값], 이-오브-엑스 |
| **영어 발음** | Expectation of X [ɪkˌspɛkˈteɪʃən ʌv ɛks] |
| **의미** | 확률 변수 X의 기대값(평균) |
| **사용 예시** | $E(X) = \mu$: "X의 기대값은 뮤" |
| **관련 용어** | 평균(Mean), 기댓값(Expectation) |

**상세 설명**:
- **기대값(Expectation)**: 확률 변수의 장기적인 평균
- 이산 확률 변수: $E(X) = \sum_{i} x_i P(X=x_i)$
- 연속 확률 변수: $E(X) = \int_{-\infty}^{\infty} x f(x) \, dx$
- 정규분포에서 $E(X) = \mu$ (평균과 동일)
- 주사위 기대값: $E(X) = 1 \times \frac{1}{6} + 2 \times \frac{1}{6} + \cdots + 6 \times \frac{1}{6} = 3.5$

---

### 3.2 V(X) - 분산
```
기호: V(X)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 분산 [분-산], 브이-오브-엑스 |
| **영어 발음** | Variance of X [ˈveəriəns ʌv ɛks] |
| **의미** | 확률 변수 X의 분산(퍼짐 정도) |
| **사용 예시** | $V(X) = \sigma^2$: "X의 분산은 시그마 제곱" |
| **관련 용어** | 표준편차(Standard Deviation), 산포(Dispersion) |

**상세 설명**:
- **분산(Variance)**: 데이터가 평균에서 얼마나 퍼져 있는지
- $V(X) = E((X - \mu)^2)$: "평균에서 떨어진 거리의 제곱에 대한 기대값"
- 표준편차와의 관계: $\sigma = \sqrt{V(X)} = \sqrt{\sigma^2}$
- 정규분포에서 $V(X) = \sigma^2$
- 분산이 작을수록 데이터가 평균 근처에 집중됨

---

### 3.3 P(A) - 확률
```
기호: P(A)
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 에이일 확률 [확-률], 피-오브-에이 |
| **영어 발음** | Probability of A [ˌprɒbəˈbɪləti ʌv eɪ] |
| **의미** | 사건 A가 발생할 확률 |
| **사용 예시** | $P(X \le x)$: "X가 x보다 작거나 같을 확률" |
| **관련 용어** | 확률(Probability), 사건(Event) |

**상세 설명**:
- **확률(Probability)**: 사건이 발생할 가능성 (0 ~ 1 사이의 값)
- $0 \le P(A) \le 1$
- 누적 확률: $P(X \le x)$: "X가 x보다 작거나 같을 확률"
- 표준정규분포에서: $\Phi(z) = P(Z \le z)$

---

### 3.4 ~ (Distribution) - 분포를 따른다
```
기호: ~
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 분포를 따른다 [분-포를 따-른다] |
| **영어 발음** | Follows distribution [ˈfɒləʊz ˌdɪstrɪˈbjuːʃn] |
| **의미** | 확률 변수가 특정 분포를 따름 |
| **사용 예시** | $X \sim N(\mu, \sigma^2)$: "X는 평균 뮤, 분산 시그마 제곱의 정규분포를 따른다" |
| **관련 용어** | 분포(Distribution), 확률 분포(Probability Distribution) |

**상세 설명**:
- **분포를 따른다(Follows Distribution)**: 확률 변수의 확률적 성질을 정의
- $X \sim N(\mu, \sigma^2)$: "X는 정규분포(Normal)를 따른다"
- $X \sim B(n, p)$: "X는 이항분포(Binomial)를 따른다"
- $X \sim P(\lambda)$: "X는 포이송분포(Poisson)를 따른다"

---

### 3.5 σ² (Sigma Squared) - 분산
```
기호: σ²
```

| 속성 | 값 |
| :--- | :--- |
| **한국어 발음** | 시그마 제곱 [시-그-마 제-곱] |
| **영어 발음** | Sigma squared [ˈsɪɡmə skweəd] |
| **의미** | 분산(Variance), 표준편차의 제곱 |
| **사용 예시** | $N(\mu, \sigma^2)$: "평균 뮤, 분산 시그마 제곱인 정규분포" |
| **관련 용어** | 분산(Variance), 표준편차(Standard Deviation) |

**상세 설명**:
- $\sigma^2$: 시그마의 제곱, 즉 분산(Variance)
- 정규분포 표기법: $N(\mu, \sigma^2)$ - 평균과 분산으로 표기
- 표준편차와의 관계: $\sigma = \sqrt{\sigma^2}$
- 분산은 항상 음수가 아님: $\sigma^2 \ge 0$

**상세 설명**:
- $A \le B$: "A는 B보다 작거나 같다"
- 누적 확률: $P(X \le x)$: "X가 x보다 작거나 같을 확률"
- 68-95-99.7 규칙: $[\mu - \sigma, \mu + \sigma]$는 "뮤 마이너스 시그마 이상, 뮤 플러스 시그마 이하"

**상세 설명**:
- $A \ge B$: "A는 B보다 크거나 같다"
- 여집합 확률: $P(X \ge x) = 1 - P(X < x)$
- 오른쪽 꼬리 확률(Right-tail probability) 계산에 사용

- 무리수 표기: $\pi \approx 3.14$, $e \approx 2.72$
- 68-95-99.7 규칙: "약 68%, 약 95%, 약 99.7%"
- 실무적 계산에서 근사값을 자주 사용

## 10. 실전 응용 예시

### 10.1 정규분포 공식 읽기

**공식**: $f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}$

**한국어 발음**:
> "엑스의 함수(엠프)는, 시그마 곱하기 루트(2파이)의 역수에, 이의 마이너스 이분의 일, 괄호 엑스 마이너스 뮤를 시그마로 나눈 값의 제곱, 괄호를 곱한다."

**해석**:
1. $\frac{1}{\sigma\sqrt{2\pi}}$: 정규화 상수 (전체 확률이 1이 되게 하는 값)
2. $e^{-\frac{1}{2}(\frac{x-\mu}{\sigma})^2}$: 종형 곡선 형성하는 지수 부분
3. $\frac{x-\mu}{\sigma}$: 표준화 (x를 표준정규분포의 z로 변환)
4. 제곱: 항상 양수가 되게 함
5. 마이너스: 곡선이 가운데에서 높고 끝에서 낮게

### 10.2 표준화 공식 읽기

**공식**: $Z = \frac{X - \mu}{\sigma}$

**한국어 발음**:
> "제이는 엑스 마이너스 뮤를 시그마로 나눈 값이다."

**해석**:
1. $X - \mu$: 데이터에서 평균을 뺌 (평균에서 얼마나 떨어져 있는지)
2. $\frac{X - \mu}{\sigma}$: 표준편차 단위로 표현 (몇 시그마 떨어져 있는지)
3. $Z$: 표준정규분포의 값

### 10.3 기대값 공식 읽기

**공식**: $E(X) = \sum_{i=1}^{n} x_i P(X=x_i)$

**한국어 발음**:
> "엑스의 기대값은, 아이가 일부터 엔까지, 엑스 아이 곱하기 엑스가 엑스 아이일 확률의 합이다."

**해석**:
1. $x_i$: 각 가능한 값
2. $P(X=x_i)$: 해당 값이 나올 확률
3. $x_i P(X=x_i)$: 값과 확률의 곱
4. $\sum$: 모든 곱을 더함

## 12. 기억법 (Mnemonic)

### 12.1 μ (뮤) - 평균
- **M**ean (평균) → **M**u (뮤)
- "뮤"는 평균의 첫 글자 M과 발음이 비슷함

### 12.2 σ (시그마) - 표준편차
- **S**tandard **D**eviation → **S**igma (시그마)
- "시그마"의 "시"는 "Standard"의 "S"와 연상

### 12.3 π (파이) - 원주율
- **P**i → **P**ie (파이)
- 파이 모양이 원형임

### 12.4 e (이) - 자연 상수
- **E**xponential (지수함수) → **e**
- "이"는 자연의 상수, 자연스럽게 외워짐

### 12.5 λ (람다) - 비율
- **L**ambda → **L**ate (늦게), **L**ow (낮게)
- 람다가 작으면 사건이 늦게/적게 발생

### 12.6 Φ (파이) - CDF
- **C**umulative **D**istribution **F**unction → **C**apital **Φ** (대문자 파이)
- 대문자 Φ는 "누적(Cumulative)"을 의미
