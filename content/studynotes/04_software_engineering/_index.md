## 📋 핵심 키워드 리스트
- **개발 방법론**: 애자일 (Scrum, Kanban), 폭포수, UML 모델링, 도메인 주도 설계 (DDD)
- **아키텍처/패턴**: MSA (마이크로서비스), 디자인 패턴 (GoF 23종), 클린 아키텍처
- **품질 관리**: 소프트웨어 테스트 (V-Model), ISO 25010, TDD, 코드 리팩토링
- **프로젝트 관리**: WBS, 위험 관리, PMO, 형상 관리, 기능 점수 (FP)
