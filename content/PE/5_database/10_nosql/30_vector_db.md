# 벡터 데이터베이스 (Vector Database)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 벡터 데이터베이스는 **"고차원 **벡터(Embedding)를 **저장**하고 **유사 **벡터**(Approximate Nearest Neighbor) **검색**을 **지원**하는 **DB**"로, **ML**(Machine Learning), **RAG**(Retrieval Augmented Generation), **추천 **시스템**, **이미지 **검색**에 **사용**된다.
>  │ 2. **기술**: **Embedding**(실수 **벡터 변환), **ANN**(Approximate Nearest Neighbor), **HNSW**(Hierarchical Navigable Small World), **IVF**(Inverted File Index, **Product **Quantization)로 **구현**되며 **Pinecone**, **Weaviate**, **Milvus**, **Qdrant**가 **대표적**이다.
>  │ 3. **작동**: **Query**(쿼리 **벡터) → **Index Scan**(후보 **검색) → **Distance**(Cosine **유사도** 또는 **Euclidean** **계산**) → **Top-K** **결과**를 **반환**하며 **HNSW**가 **검색 **속도**를 **최적화**한다.

---

## Ⅰ. 개요 (Context & Background)

### 개념
벡터 데이터베이스는 **"임베딩 공간 유사성 검색 DB"**이다.

**벡터 vs 스칼라**:
| 구분 | 벡터 DB | 스칼라 DB |
|------|--------|-----------|
| **데이터** | 실수 벡터 | 텍스트/벡터 |
| **비교** | 유사도 | 일치 |
| **용도** | ML, 추천 | 검색 |
| **최적화** | ANN | Inverted Index |

### 💡 비유
벡터 DB는 ****책 ****도서관 ****과 같다.
- **책**: 문서
- **색인**: 임베딩
- **검색**: 유사한 책 찾기

## Ⅱ. 아키텍처 및 핵심 원리

### Embedding

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Word Embedding                                                     │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    Text → Token → Embedding (128-1536 dimensions)
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  "king" → [0.23, -0.51, 0.73, ..., 0.12]        (384d float array)                        │  │
    │  "queen" → [0.12, -0.23, 0.89, ..., -0.45]       (384d float array)                        │  │
    │                                                                                           │  │
    │  Similarity("king", "queen") = cosine(0.85) → High                                              │  │
    │  Similarity("king", "car") = cosine(0.12) → Low                                                │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    → ML Models: Word2Vec, GloVe, BERT, Sentence Transformers
```

### ANN (Approximate Nearest Neighbor)

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         ANN Search                                                           │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Query Vector: [0.1, -0.2, 0.5, ...] (128d)                                          │  │
    │                                                                                         │  │
    │  ANN Graph (V = 10, connections shown):                                                        │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Layer 0 (10,000 dims) → Projection → Layer 1 (128 dims)                              │  │  │
    │  │  ┌──────────────────────────────────────────────────────────────────────────────────┐  │  │  │  │
    │  │  │  Node 1    Node 5    Node 9        ...                                             │  │  │  │
    │  │  │    │  │         │  │                                                              │  │  │  │
    │  │  └──────────────────────────────────────────────────────────────────────────────────┘  │  │  │
    │  │           │                                                                              │  │  │
    │  │           ▼ Greedy search: Find top 2 closest nodes in layer 1                               │  │  │
    │  │                                                                                       │  │  │
    │  │  Layer 1 → Search candidates: [Node 2, Node 7] (closest)                             │  │  │
    │  │           │                                                                              │  │  │
    │  │           ▼ Continue until leaf layer (Layer log2(V))                                       │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │                                                                                         │  │
    │  → Returns: [Doc1: 0.89, Doc2: 0.85, Doc3: 0.78, ...]                                    │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

### HNSW (Hierarchical Navigable Small World)

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         HNSW Graph Structure                                                │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Layer 0: Entry Point (128 vectors, V=128)                                               │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Proximity Graph: Connects nearby points                                             │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │            │                                                                             │  │
    │            ▼                                                                             │  │
    │  Layer 1 (2,048 vectors, V=2×128):                                                           │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  │ ┌──────────────────────────────────────────────────────────────────────────────┐  │  │  │
    │  │  │  │ Node A (48 vectors)           Node B (48 vectors)           ...  │  │  │  │
    │  │  │  └──────────────────────────────────────────────────────────────────────────────┘  │  │  │
    │  │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │  │
    │  │          │                                                                             │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │            │                                                                             │  │  │
    │            ▼                                                                             │  │  │
    │  Layer 2 (16,384 vectors, V=2×2×128):                                                        │  │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  │  → Connects to 2 nodes each in Layer 1, now refined search                             │  │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │                                                                                         │  │  │
    │  → Hierarchical search: Fast exploration of vector space                                      │  │  │
    │  → Better accuracy than flat ANN at same speed                                               │  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용 및 기술사적 판단

### Pinecone 예시

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Vector Search with Pinecone                                           │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Upsert: Create index and add vectors                                                        │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  import pinecone                                                                           │  │  │
    │  │  │ pinecone.init(api_key="...")                                                            │  │  │
    │  │  │ index = pinecone.Index("wiki")                                                         │  │  │
    │  │  │                                                                                       │  │  │
    │  │  │ # Upsert documents (batch)                                                             │  │  │  │
    │  │  │  │ index.upsert([                                                                        │  │  │
    │  │  │  │     {"id": "doc1", "vector": [0.1, 0.2, ...], "metadata": {...}}                   │  │  │  │
    │  │  │  │     {"id": "doc2", "vector": [0.3, -0.1, ...], "metadata": {...}}                  │  │  │  │
    │  │  │  ])                                                                                   │  │  │  │
    │  │  │                                                                                       │  │  │  │
    │  │  │ # Query: Top 5 most similar                                                             │  │  │  │
    │  │  │  │ query = [0.1, 0.2, 0.5, ...]  # 128d                                     │  │  │  │
    │  │  │  │ results = index.query(                                                               │  │  │  │
    │  │  │  │     vector=query,                                                                   │  │  │  │
    │  │  │  │     top_k=5,                                                                       │  │  │  │
    │  │  │  │     include_metadata=True                                                          │  │  │  │
    │  │  │  │ )                                                                                   │  │  │  │
    │  │  │  │ for match in results:                                                                  │  │  │  │
    │  │  │  │     print(f"{match['id']}: {match['score']:.4f}")                                │  │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```
