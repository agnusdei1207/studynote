# 모드 비트 (Mode Bit)

## 정의

**모드 비트(Mode Bit)**는 CPU의 **프로세서 상태 레지스터(PSW)** 내에 존재하는 1비트 플래그로, 현재 CPU가 **사용자 모드(1)**인지 **커널 모드(0)**인지를 나타낸다. 이 비트를 통해 하드웨어는 특권 명령 실행을 제어한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           모드 비트 위치와 역할                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   프로세서 상태 워드 (PSW) 구조:                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ┌───┬───┬───┬───┬───┬───┬───┬───┬────────────────┐ │ │
│   │   │ M │ I │ O │ Z │ S │ P │ C │   │   기타 플래그    │ │ │
│   │   └───┴───┴───┴───┴───┴───┴───┴───┴────────────────┘ │ │
│   │     ↑                                                   │ │
│   │     └── 모드 비트 (Mode Bit)                            │ │
│   │                                                         │ │
│   │   M = 0: 커널 모드 (Kernel Mode)                        │ │
│   │   M = 1: 사용자 모드 (User Mode)                        │ │
│   │                                                         │ │
│   │   I = 인터럽트 플래그                                   │ │
│   │   O = 오버플로우 플래그                                 │ │
│   │   Z = 제로 플래그                                       │ │
│   │   S = 사인 플래그                                       │ │
│   │   P = 패리티 플래그                                     │ │
│   │   C = 캐리 플래그                                       │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 모드 비트 변경 시점

```
┌─────────────────────────────────────────────────────────────────┐
│           모드 비트 변경 시점                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   사용자 모드 → 커널 모드 (M: 1 → 0):                           │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ① 시스템 콜                                           │ │
│   │      SYSCALL/INT 0x80 실행 → 하드웨어가 자동으로 M=0    │ │
│   │                                                         │ │
│   │   ② 인터럽트                                            │ │
│   │      외부 신호 → 하드웨어가 자동으로 M=0                 │ │
│   │                                                         │ │
│   │   ③ 예외 (Exception)                                    │ │
│   │      잘못된 연산, 페이지 폴트 → 하드웨어가 자동으로 M=0  │ │
│   │                                                         │ │
│   │   💡 모드 전환은 하드웨어에 의해 자동 수행               │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   커널 모드 → 사용자 모드 (M: 0 → 1):                           │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ① IRET (Interrupt Return)                             │ │
│   │      인터럽트 처리 완료 후 복귀                          │ │
│   │      → 스택에서 PSW 복원, M=1로 설정                    │ │
│   │                                                         │ │
│   │   ② SYSRET (System Return)                              │ │
│   │      시스템 콜 처리 완료 후 복귀                         │ │
│   │      → M=1로 설정 후 사용자 코드로 복귀                 │ │
│   │                                                         │ │
│   │   ③ 컨텍스트 스위치                                     │ │
│   │      새 프로세스의 PSW 로드                              │ │
│   │      → 사용자 프로세스면 M=1                             │ │
│   │                                                         │ │
│   │   💡 복귀 명령은 특권 명령 (커널 모드에서만 실행)        │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. 아키텍처별 모드 비트 구현

```
┌─────────────────────────────────────────────────────────────────┐
│           아키텍처별 모드 비트                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  아키텍처  │  레지스터   │  모드 표현                    │ │
│   ├───────────┼────────────┼───────────────────────────────┤ │
│   │ x86       │ EFLAGS     │ CPL (Current Privilege Level) │ │
│   │ (32비트)  │ CS 하위    │ Ring 0~3 (2비트)              │ │
│   ├───────────┼────────────┼───────────────────────────────┤ │
│   │ x86-64    │ RFLAGS     │ CPL (Ring 0, 3 주로 사용)     │ │
│   │           │ CS 하위    │                               │ │
│   ├───────────┼────────────┼───────────────────────────────┤ │
│   │ ARM       │ CPSR       │ Mode 비트 (4비트)             │ │
│   │           │            │ User, FIQ, IRQ, SVC...       │ │
│   ├───────────┼────────────┼───────────────────────────────┤ │
│   │ MIPS      │ Status     │ Kernel/User 모드             │ │
│   │           │ Register   │ (1비트)                       │ │
│   ├───────────┼────────────┼───────────────────────────────┤ │
│   │ RISC-V    │ mstatus    │ MPP (이전 권한)               │ │
│   │           │            │ M/S/U 모드                    │ │
│   └───────────┴────────────┴───────────────────────────────┘ │
│                                                                 │
│   ARM 모드 상세:                                                │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   CPSR[4:0]  모드명     용도                            │ │
│   │   ─────────  ─────      ────                            │ │
│   │   10000     User        사용자 모드                     │ │
│   │   10001     FIQ         빠른 인터럽트                   │ │
│   │   10010     IRQ         일반 인터럽트                   │ │
│   │   10011     SVC         슈퍼바이저 (커널)               │ │
│   │   10111     Abort       메모리 예외                     │ │
│   │   11011     Undefined   정의되지 않은 명령              │ │
│   │   11111     System      시스템 모드                     │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **모드 비트**: PSW 내의 1비트 플래그로 현재 실행 모드 표시
> **0 = 커널 모드**: 모든 명령 실행 가능
> **1 = 사용자 모드**: 제한된 명령만 실행

### 2. 빈출 키워드

- **PSW (Processor Status Word)**: 모드 비트가 포함된 레지스터
- **CPL (Current Privilege Level)**: x86의 권한 레벨
- **특권 명령 보호**: 모드 비트 검사로 실행 제어
