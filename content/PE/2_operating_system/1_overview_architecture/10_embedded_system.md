# 임베디드 시스템 (Embedded System)

## 정의

**임베디드 시스템(Embedded System)**은 특정 기능을 수행하기 위해 **다른 시스템에 내장**되는 전용 컴퓨터 시스템이다. 제한된 자원(메모리, 전력, 처리능력) 하에서 특정 목적에 최적화되어 설계된다.

```
┌─────────────────────────────────────────────────────────────────┐
│           임베디드 시스템 개요                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   특징:                                                  │ │
│   │   • 특정 기능 전용 (Single-purpose)                     │ │
│   │   • 제한된 자원 (메모리, 전력, 성능)                     │ │
│   │   • 실시간성 요구 (많은 경우)                            │ │
│   │   • 높은 신뢰성/안정성 요구                              │ │
│   │   • 긴 제품 수명 주기                                    │ │
│   │                                                         │ │
│   │   구성 요소:                                             │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │  하드웨어  │  소프트웨어   │  통합 환경       │    │ │
│   │   ├───────────┼─────────────┼────────────────┤    │ │
│   │   │ MCU/SoC   │ 펌웨어       │ 센서/액추에이터 │    │ │
│   │   │ 메모리    │ RTOS        │ 통신 인터페이스 │    │ │
│   │   │ I/O 장치  │ 드라이버    │ 전원 관리      │    │ │
│   │   └───────────┴─────────────┴────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **임베디드 시스템**: 특정 기능 수행을 위해 다른 시스템에 내장된 전용 컴퓨터
> **RTOS**: 실시간성 보장을 위한 운영체제
> **제약사항**: 메모리, 전력, 성능, 비용 제약

### 2. 빈출 키워드

- **SoC (System on Chip)**: 단일 칩에 시스템 통합
- **RTOS**: 실시간 운영체제
- **BSP (Board Support Package)**: 하드웨어 추상화 계층
- **Cross-compilation**: 호스트에서 타겟용 코드 생성

### 3. 예상 문제

**Q1. 임베디드 시스템의 특징과 일반 컴퓨터 시스템과의 차이점을 설명하시오.**

**Q2. 임베디드 시스템에서 RTOS 선택 시 고려사항을 기술하시오.**

**Q3. 임베디드 소프트웨어의 계층 구조를 설명하시오.**
