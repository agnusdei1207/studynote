# 단방향/반이중/전이중 통신

## 정의

데이터 전송 방식은 **신호의 전달 방향**에 따라 **단방향(Simplex)**, **반이중(Half-Duplex)**, **전이중(Full-Duplex)**으로 구분된다. 각 방식은 전송 효율, 하드웨어 복잡도, 비용에서 차이를 보인다.

```
┌─────────────────────────────────────────────────────────────────┐
│           데이터 전송 방식 분류                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   단방향 (Simplex):                                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   송신기 ────────────────────────▶ 수신기               │ │
│   │   (한 방향만 전송 가능)                                  │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   반이중 (Half-Duplex):                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   장치A ◀──────────────────────▶ 장치B                  │ │
│   │   (양방향 전송 가능, 동시 전송 불가)                      │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   전이중 (Full-Duplex):                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   장치A ◀════════════════════════▶ 장치B                 │ │
│   │   (양방향 동시 전송 가능)                                │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 단방향 (Simplex)

```
┌─────────────────────────────────────────────────────────────────┐
│           단방향 통신 (Simplex)                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   특징:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 한 방향으로만 데이터 전송                             │ │
│   │  • 수신 측은 송신 불가                                   │ │
│   │  • 회선 구성 가장 단순                                   │ │
│   │  • 채널 효율 낮음 (수신 확인 불가)                        │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   응용 예시:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 라디오/TV 방송                                        │ │
│   │  • 키보드 → 컴퓨터                                       │ │
│   │  • 센서 → 모니터링 시스템                                │ │
│   │  • GPS 위성 → 수신기                                     │ │
│   │  • 프린터 (컴퓨터 → 프린터)                              │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   장단점:                                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  장점: 구조 단순, 비용 저렴                              │ │
│   │  단점: 오류 검출 어려움, 수신 확인 불가                  │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 반이중 (Half-Duplex)

```
┌─────────────────────────────────────────────────────────────────┐
│           반이중 통신 (Half-Duplex)                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   특징:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 양방향 전송 가능                                      │ │
│   │  • 동시 전송 불가 (교대로 전송)                          │ │
│   │  • 송/수신 전환 시간 필요 (Turnaround Time)              │ │
│   │  • 하나의 채널로 양방향 통신                             │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   동작 방식:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   시간 t₀~t₁:  A ────────▶ B (A 송신, B 수신)           │ │
│   │   시간 t₁~t₂:  [전환 시간]                              │ │
│   │   시간 t₂~t₃:  A ◀──────── B (B 송신, A 수신)           │ │
│   │                                                         │ │
│   │   A: ████░░░░░░░░░░░░░░░░░░████                        │ │
│   │   B: ░░░░░░░░░░░░████████░░░░░░░░░░                    │ │
│   │            ↑ 전환                                        │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   응용 예시:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 무전기 (Walkie-Talkie)                                │ │
│   │  • CB 라디오                                             │ │
│   │  • 이더넷 (Hub 기반, CSMA/CD)                            │ │
│   │  • 무선 LAN (802.11 원래 표준)                           │ │
│   │  • I2C, SPI 통신                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   장단점:                                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  장점: 전이중보다 회선 절약                              │ │
│   │  단점: 전환 지연, 동시 통신 불가                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 전이중 (Full-Duplex)

```
┌─────────────────────────────────────────────────────────────────┐
│           전이중 통신 (Full-Duplex)                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   특징:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 양방향 동시 전송 가능                                 │ │
│   │  • 두 개의 독립 채널 필요 (또는 주파수 분리)             │ │
│   │  • 전환 시간 불필요                                      │ │
│   │  • 최고 효율의 통신 방식                                 │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   구현 방식:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ① 공간 분리 (Space Division):                         │ │
│   │      • 별도의 송신/수신 회선 사용                       │ │
│   │      • 예: 4선식 회선 (Two-pair cable)                  │ │
│   │                                                         │ │
│   │   ② 주파수 분리 (Frequency Division):                   │ │
│   │      • 송신/수신에 다른 주파수 할당                     │ │
│   │      • 예: FDD (Frequency Division Duplexing)           │ │
│   │                                                         │ │
│   │   ③ 시간 분리 (Time Division):                          │ │
│   │      • 시간 슬롯 분할로 송수신 분리                     │ │
│   │      • 예: TDD (Time Division Duplexing)                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   응용 예시:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 전화망 (PSTN)                                         │ │
│   │  • 이더넷 스위치 (Full-Duplex Ethernet)                  │ │
│   │  • 휴대전화 (LTE, 5G FDD/TDD)                            │ │
│   │  • 광섬유 통신                                           │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   장단점:                                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  장점: 양방향 동시 통신, 최고 대역폭 효율                │ │
│   │  단점: 하드웨어 복잡, 비용 증가                          │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. 전송 방식 비교

```
┌─────────────────────────────────────────────────────────────────┐
│           전송 방식 종합 비교                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  구분       │ 단방향     │ 반이중     │ 전이중          │ │
│   ├─────────────┼───────────┼───────────┼────────────────┤ │
│   │ 전송 방향   │ 1방향     │ 양방향    │ 양방향 동시    │ │
│   │ 채널 수     │ 1개       │ 1개       │ 2개            │ │
│   │ 동시 통신   │ 불가      │ 불가      │ 가능           │ │
│   │ 회선 효율   │ 낮음      │ 중간      │ 높음           │ │
│   │ 하드웨어    │ 단순      │ 중간      │ 복잡           │ │
│   │ 비용        │ 저렴      │ 중간      │ 높음           │ │
│   │ 응답성      │ 없음      │ 전환지연  │ 즉시           │ │
│   └─────────────┴───────────┴───────────┴────────────────┘ │
│                                                                 │
│   대역폭 활용:                                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   단방향:  B ────────────────────▶ (전체 대역폭 송신)   │ │
│   │                                                         │ │
│   │   반이중:  B ───────▶ (50% 송신)                        │ │
│   │            B ◀─────── (50% 수신)                        │ │
│   │                                                         │ │
│   │   전이중:  B ════════▶ (100% 송신 + 100% 수신)          │ │
│   │            B ◀════════                                   │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **단방향(Simplex)**: 한 방향만 전송 (라디오)
> **반이중(Half-Duplex)**: 양방향 교대 전송 (무전기)
> **전이중(Full-Duplex)**: 양방향 동시 전송 (전화)

### 2. 빈출 키워드

- **Turnaround Time**: 반이중 전환 지연 시간
- **FDD/TDD**: 주파수/시간 분할 이중화
- **4선식/2선식**: 전이중/반이중 회선 구성

### 3. 예상 문제

**Q1. 단방향, 반이중, 전이중 통신 방식을 비교 설명하시오.**

**Q2. 반이중 통신의 문제점과 해결 방안을 기술하시오.**

**Q3. 이더넷에서 전이중 통신이 가능해진 이유를 설명하시오.**
