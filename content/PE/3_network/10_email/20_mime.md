# MIME (Multipurpose Internet Mail Extensions)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: MIME는 **"이메일로 **다양한 **데이터 형식**(텍스트, 이미지, 비디오, 바이너리)을 **전송**하기 위한 **확장 표준"**으로, **Content-Type**(미디어 타입), **Content-Transfer-Encoding**(인코딩), **Multipart**(다중 부분) **헤더**를 **사용**하여 **SMTP의 7비트 ASCII 제한**을 **극복**한다.
> 2. **가치**: **첨부 파일**(Attachment), **HTML 이메일**, **국제화**(UTF-8), **디지털 서명**(S/MIME), **암호화**를 **지원**하여 **단순 텍스트**만 **전송**하던 **전통 이메일**을 **멀티미디어 통신**으로 **확장**했다.
> 3. **융합**: **RFC 2045-2049**가 **표준**이며 **Base64**(64진수 인코딩)로 **바이너리를 **ASCII로 **변환**하고 **multipart/mixed**(여러 첨부), **multipart/alternative**(텍스트+HTML), **multipart/related**(인라인 이미지) **MIME 타입**이 **있다.

## Ⅰ. 개요 (Context & Background)

### 개념
MIME는 **"이메일에 다양한 데이터를 첨부하는 표준"**이다.

**MIME 특징**:
- **Content-Type**: 데이터 형식
- **Encoding**: 7비트 ASCII 변환
- **Multipart**: 여러 부분 결합
- **Extension**: SMTP 확장

### 💡 비유
MIME는 **"편지봉투**에 **물건**을 **넣는 것****과 같다.
- **편지**: 본문
- **봉투**: MIME 인코딩
- **물건**: 첨부 파일

### 등장 배경

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         이메일 표준 발전                                                  │
└─────────────────────────────────────────────────────────────────────────────────────────┘

SMTP (1982):
    • 7비트 ASCII만 전송
    • 영문 텍스트만 가능
         ↓
MIME (1992, RFC 2045):
    • 바이너리 전송 가능
    • 멀티미디어 지원
         ↓
현대:
    • UTF-8 국제화
    • S/MIME 보안
```

## Ⅱ. 아키텍처 및 핵심 원리

### MIME 헤더 구조

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         MIME Message 구조                                                │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  MIME: Message with Image Attachment                                                    │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │
    │  │  MIME-Version: 1.0                                          # MIME 버전                │  │
    │  │  From: sender@example.com                                                             │  │
    │  │  To: receiver@example.com                                                             │  │
    │  │  Subject: Photo attached                                                              │  │
    │  │  Content-Type: multipart/mixed; boundary="boundary123"       # 여러 부분 포함           │  │
    │  │  ───────────────────────────────────────────────────────────────────────────────────  │  │
    │  │  --boundary123                                                                         │  │
    │  │  Content-Type: text/plain; charset="UTF-8"                      # 텍스트 본문          │  │
    │  │  ───────────────────────────────────────────────────────────────────────────────────  │  │
    │  │  Please find attached photo.                                                          │  │
    │  │  --boundary123                                                                         │  │
    │  │  Content-Type: image/jpeg                                           # JPEG 이미지      │  │
    │  │  Content-Transfer-Encoding: base64                                    # Base64 인코딩  │  │
    │  │  Content-Disposition: attachment; filename="photo.jpg"                 # 첨부 파일       │  │
    │  │  ───────────────────────────────────────────────────────────────────────────────────  │  │
    │  │  /9j/4AAQSkZJRgABAQAAAQABAAD/2wBDAAgGBgcGBQgHBwcJCQgKDBQNDAsL                              │  │
    │  │  DBkSEw8UHRofHh0aHBwgJC4nICIsIxwcKDcpLDAxNDQ0Hyc5PTgyPC4zNDL/2wB...                          │  │
    │  │  --boundary123--                                                        # 끝             │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

### Content-Type (미디어 타입)

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Content-Type 형식                                                 │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  형식: type/subtype; parameter=value                                                   │  │
    │                                                                                         │  │
    │  ┌────────────────────────────────────────────────────────────────────────────────────┐  │
    │  │  Type (Top-level)         │  Subtype Examples              │  Description              │  │
    │  │  ───────────────────────────────────────────────────────────────────────────────────  │  │
    │  │  text                    │  plain, html, css             │  텍스트 데이터             │  │
    │  │  image                   │  jpeg, png, gif, webp         │  이미지                   │  │
    │  │  audio                   │  mp3, wav, ogg                │  오디오                   │  │
    │  │  video                   │  mp4, avi, mkv                │  비디오                   │  │
    │  │  application             │  pdf, json, zip, octet-stream │  바이너리 데이터           │  │
    │  │  multipart               │  mixed, alternative, related  │  다중 부분                 │  │
    │  │  message                 │  rfc822, partial              │  메시지 캡슐화             │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │
    │                                                                                         │  │
    │  Examples:                                                                              │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Content-Type: text/plain; charset="UTF-8"                        # UTF-8 텍스트    │  │  │
    │  │  Content-Type: text/html; charset="UTF-8"                         # HTML 이메일     │  │  │
    │  │  Content-Type: application/json; charset="UTF-8"                   # JSON 데이터     │  │  │
    │  │  Content-Type: application/octet-stream                            # 바이너리       │  │  │
    │  │  Content-Type: multipart/alternative; boundary="alt123"            # 대체 버전      │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

### Content-Transfer-Encoding

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Content-Transfer-Encoding                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Encoding              │  설명                      │  예시                            │  │
    │  ───────────────────────────────────────────────────────────────────────────────────  │
    │  7bit                  │  7비트 ASCII (기본)         │  Hello World                    │  │
    │  8bit                  │  8비트 데이터               │  UTF-8 (octets)                 │  │
    │  binary                │  바이너리 (제한 없음)       │  raw data                       │  │
    │  quoted-printable      │  주로 인쇄 가능 문자       │  =E3=84=9C (한글 인코딩)        │  │
    │  base64                │  64진수 인코딩             │  SGVsbG8gV29ybGQ=               │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Base64 Encoding Example                                                               │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Original:  "Hello"                                            │  │  │
    │  │  Binary:    01001000 01100101 01101100 01101100 01101111       │  │  │
    │  │  Base64:    "SGVsbG8="                                           │  │  │
    │  │  │  → 6비트씩 그룹화하여 64개의 ASCII 문자(A-Z, a-z, 0-9, +, /)로 매핑                 │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

### Multipart 타입

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Multipart 타입                                                    │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  multipart/mixed                                                                      │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  • 본문 + 첨부 파일                                                                   │  │  │
    │  │  • 순서대로 표시                                                                      │  │  │
    │  │                                                                                       │  │  │
    │  │  Content-Type: multipart/mixed; boundary="mixed123"                                 │  │  │
    │  │  --mixed123                                                                            │  │  │
    │  │  Content-Type: text/plain                                                            │  │  │
    │  │  본문입니다.                                                                           │  │  │
    │  │  --mixed123                                                                            │  │  │
    │  │  Content-Type: application/pdf                                                       │  │  │
    │  │  [PDF 첨부]                                                                            │  │  │
    │  │  --mixed123--                                                                          │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │
    │                                                                                         │  │
    │  multipart/alternative                                                                  │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  • 같은 내용의 여러 버전                                                              │  │  │
    │  │  • 클라이언트가 지원하는 버전 선택                                                     │  │  │
    │  │                                                                                       │  │  │
    │  │  Content-Type: multipart/alternative; boundary="alt123"                              │  │  │
    │  │  --alt123                                                                              │  │  │
    │  │  Content-Type: text/plain                                                            │  │  │
    │  │  순수 텍스트 본문                                                                      │  │  │
    │  │  --alt123                                                                              │  │  │
    │  │  Content-Type: text/html                                                             │  │  │
    │  │  <b>HTML</b> 본문                                                                      │  │  │
    │  │  --alt123--                                                                            │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │
    │                                                                                         │  │
    │  multipart/related                                                                      │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  • HTML + 인라인 이미지                                                               │  │  │
    │  │  • Content-ID로 참조                                                                   │  │  │
    │  │                                                                                       │  │  │
    │  │  Content-Type: multipart/related; boundary="rel123"                                  │  │  │
    │  │  --rel123                                                                              │  │  │
    │  │  Content-Type: text/html                                                             │  │  │
    │  │  <img src="cid:image1">                                                               │  │  │
    │  │  --rel123                                                                              │  │  │
    │  │  Content-Type: image/jpeg                                                             │  │  │
    │  │  Content-ID: <image1>                                                                 │  │  │
    │  │  [이미지 데이터]                                                                        │  │  │
    │  │  --rel123--                                                                            │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 융합 비교 및 다각도 분석

### 인코딩 방식 비교

| 인코딩 | 크기 증가 | 속도 | 사용처 |
|--------|----------|------|--------|
| **7bit** | 없음 | 빠름 | ASCII 텍스트 |
| **quoted-printable** | 약간 | 빠름 | 대부분 ASCII |
| **base64** | 33% | 중간 | 바이너리 |

### Multipart 타입 비교

| 타입 | 용도 | 순서 |
|------|------|------|
| **mixed** | 본문+첨부 | 순서대로 |
| **alternative** | 대체 버전 | 최선의 것 선택 |
| **related** | 관련 리소스 | 참조로 연결 |

### 기술사적 판단 (실무 시나리오)

#### 시나리오: HTML 이메일 전송
**상황**: HTML + 첨부 파일
**판단**:

```
From: sender@example.com
To: receiver@example.com
Subject: Newsletter with attachment
MIME-Version: 1.0
Content-Type: multipart/mixed; boundary="boundary123"

--boundary123
Content-Type: multipart/alternative; boundary="boundary456"

--boundary456
Content-Type: text/plain; charset="UTF-8"

Plain text version.

--boundary456
Content-Type: text/html; charset="UTF-8"

<html><body><h1>HTML Newsletter</h1></body></html>

--boundary456--

--boundary123
Content-Type: application/pdf
Content-Disposition: attachment; filename="report.pdf"
Content-Transfer-Encoding: base64

JVBERi0xLjUKJeLjz9MKMSAwIG9iago8PC9UeXBlIC9DYXRhbG9nCi9QYWdlcyAy...

--boundary123--
```

## Ⅴ. 기대효과 및 결론

### MIME 기대 효과

| 효과 | MIME 없음 | MIME |
|------|----------|------|
| **첨부 파일** | 불가 | 가능 |
| **HTML** | 불가 | 가능 |
| **국제화** | 제한됨 | UTF-8 |

### 미래 전망

1. **UTF-8**: 기본 인코딩
2. **S/MIME**: 보안 이메일
3. **Rich Media**: 비디오 임베드

### ※ 참고 표준/가이드
- **RFC 2045**: MIME Format
- **RFC 2046**: Media Types
- **RFC 2047**: Header Encoding
