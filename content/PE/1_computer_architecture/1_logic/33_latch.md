# 멀티플렉서 (Multiplexer, MUX)

## 핵심 인사이트 (3줄 요약)
1. MUX는 n개 입력 중 1개를 선택하여 출력으로 전달하는 데이터 선택 회로로, Select 입력에 의해 제어된다
2. 2^n-to-1 MUX는 2^n개 입력 중 1개를 선택하며, n비트 Select 신호가 필요하다
3. 기술사사 시험에서는 MUX 트리 구조, 전송 지연, 데이터 버스 구현, ALU의 데이터 경로 설계가 핵심이다

## Ⅰ. 개요 (500자 이상)

멀티플렉서(Multiplexer, MUX)는 **데이터 선택기(Data Selector)**라고도 하며, n개 입력 중 하나를 선택하여 출력으로 전달하는 조합 논리 회로이다. Select(선택) 신호에 따라 활성 입력이 출력으로 연결된다. 일반적으로 2^n-to-1 MUX가 사용되며, 2^n개 입력 중 1개를 선택하는 데 n비트 Select 신호가 필요하다.

```
4-to-1 MUX:
입력: D0 D1 D2 D3 (4개)
Select: S1 S0 (2비트, 00=0, 01=1, 10=2, 11=3)
출력: Y

진리표:
| S1 S0 | 선택 입력 | Y |
|-------|----------|---|
| 0  0 | D0 | D0 |
| 0  1 | D1 | D1 |
| 1  0 | D2 | D2 |
| 1  1 | D3 | D3 |
```

MUX는 **데이터 전달(Data Transfer)**, **시분 다중화(Time-Division Multiplexing, TDM)**, **ALU의 입력 선택**, **데이터 버스 스위칭** 등 다양한 응용을 가진다. 예를 들어 8비트 ALU는 4개 16비트 입력(레지스터 A, 레지스 B, 즉시 값, 상수) 중 하나를 MUX로 선택하여 ALU의 입력으로 공급한다.

## Ⅱ. 아키텍처 및 핵심 원리 (1000자 이상)

### MUX 회로 구조

```
4-to-1 MUX 회로:
D0 ──┐
     │
D1 ──┼───┬─── Y
     │   │
D2 ──┼─┤ │
     │ │ │
D3 ──┴─┤ │
       │ │
S0 S1 ─┴─┴─ Select 제어

동작:
S1 S0 = 00 → Y = D0
S1 S0 = 01 → Y = D1
S1 S0 = 10 → Y = D2
S1 S0 = 11 → Y = D3
```

### 불 대수식

```
Y = D0·S1'·S0' + D1·S1'·S0 + D2·S1·S0' + D3·S1·S0

S1' = NOT(S1)
S0' = NOT(S0)
```

### MUX 트리 구조

```
8-to-1 MUX 트리 구조 (4개 2-to-1 MUX 단계):
D0 D1 ── MUX0 ──┐
D2 D3 ── MUX1 ──┼─── MUX2 ─── Y
D4 D5 ── MUX2 ──┤
D6 D7 ── MUX3 ──┘

S2 S1 S0 (3비트 Select) → 3단 트리
```

## Ⅲ. 융합 비교 및 다각도 분석

### MUX vs 디코더

| 비교 항목 | MUX | 디코더 |
|----------|-----|--------|
| 입력 | 2^n개 데이터 + n비트 Select | n비트 코드 |
| 출력 | 1개 | 2^n개 |
| 기능 | 1개 선택 | 1개 활성화 |
| 응용 | 데이터 선택 | 주소 디코딩 |
| 복잡도 | O(2^n) 크기 | O(n) 작음 |

### MUX vs DEMUX

| 비교 항목 | MUX | DEMUX |
|----------|-----|-------|
| 기능 | 2^n→1 | 1→2^n |
| 역할 | 다중화 | 역다중화 |
| 전송 선 | 1개 공유 | n개 독립 |

## Ⅳ. 실무 적용 및 기술사적 판단

### ALU 입력 선택

```
ALU MUX:
Operand A: 레지스터 R0, R1, R2, R3
Operand B: 레지스터 R0, R1, R2, R3

S1 S0 → A 선택 (4개 중 1)
S3 S2 → B 선택 (4개 중 1)
```

## Ⅴ. 기대효과 및 결론

MUX는 데이터 선택의 핵심이다. ALU, 데이터 버스, 통신에 사용된다.

## 📌 관련 개념 맵

```
MUX
├── 정의: 2^n입력 → 1출력 (Select 제어)
├── 4-to-1 MUX
├── 8-to-1 MUX
└── 응용
    ├── ALU 입력 선택
    ├── 데이터 버스 스위치
    └── TDM 통신
```

## 👶 어린이를 위한 3줄비유 설명

1. MUX는 철도 선로에서 여러 입력 중 하나를 선택하는 스위치 역할이에요
2. 4개의 전선 중 1개를 선택하려면 2비트 제어 신호가 필요한데, "00이면 1번, 01이면 2번..."처럼 선택해요
3. CPU가 여러 레지스터 중 값을 가져올 때 MUX가 "어떤 레지스터에서 값을 가져올까?"를 선택해요
