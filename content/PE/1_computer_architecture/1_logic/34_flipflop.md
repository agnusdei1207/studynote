# 래치 (Latch)

## 핵심 인사이트 (3줄 요약)
1. 래치는 클럭(Enable) 신호가 1인 동안 입력을 유지하는 1비트 순차 기억 소자로, SRAM 셀과 함께 메모리의 기초이다
2. D 래치(D-Latch)와 JK 래치가 있으며, SET/RESET 제어로도 구현 가능하다
3. 기술사시험에서는 래치의 메타 안정성(Metastability), 설정/해지 시간, 플리핑 방지가 핵심이다

## Ⅰ. 개요 (500자 이상)

랫치(Latch)는 **1비트 순차 기억 소자**로, 클럭(Enable) 신호가 1인 동안 입력을 유지한다. 래치는 **레벨 트리거(Level Triggered)**로 동작하며, 클럭의 하강 에지(Positive Edge) 또는 상승 에지(Rising Edge)에서 입력을 샘플링한다. 래치는 투명성(Transparency)로 출력을 직접 제어할 수 있어, 플립플롭(Flip-Flop)과 함께 SRAM(Static RAM), 레지스터 파일, 설정 레지스터(Configuration Register)에 사용된다.

```
D 래치(D-Latch):
입력: D (Data), E (Enable)
출력: Q, Q' (Complement)

진리표:
| E | D | Q(t+1) | 설명 |
|---|---|---------|------|
| 0 | X | Q(t) | 유지 (Latch) |
| 1 | 0 | 0 | 리셋 |
| 1 | 1 | 1 | 셋 |

E=0: 입력 무시, 이전 상태 유지
E=1: 입력 D를 Q로 샘플링
```

SRAM 셀은 2개 D 래치를 교차 결합(Cross-Coupled)한 구조로, 1비트 정보를 저장한다. 6T SRAM은 2개 CMOS 인버터와 2개 액세스 트랜지스터로 구현된다.

```
6T SRAM 셀 구조:
     V_DD
      │
   ┌─┴─┐
   │   │
 PMOS PMOS
   │   │
  ─┴─┬─┴─┬─── Q
  │   │   │
  ─┬─┴───┴─── Q'
  │   │   │
 NMOS NMOS
  │   │   │
  ──┴───┴
     │
    GND

Wordline:
   BIT    BIT'   WORD LINE
    │  │   │
BIT ─┼──┴─── BIT'
```

## Ⅱ. 아키텍처 및 핵속 원리 (1000자 이상)

### D 래치(D-Latch) 회로

```
D 래치 회로:
D ──┬──┬─── Q
    │  │
E ──┤  AND  │
    │  NOR └─── Q'
```

### JK 래치

```
JK 래치 진리표:
| J | K | Q(t+1) | 설명 |
|---|---|---------|------|
| 0 | 0 | Q(t) | 유지 |
| 0 | 1 | 0 | 리셋 |
| 1 | 0 | 1 | 셋 |
| 1 | 1 | Q'(t) | 토글 |

JK 래치는 D 래치보다 강력한 기능을 가짐.
```

## Ⅲ. 융합 비교

### 래치 vs 플립플롭

| 비교 항목 | 래치 | 플립플롭 |
|----------|------|----------|
| 클럭 | 레벨 트리거 | 에지 트리거 |
| 투명성 | 출력 직접 제어 가능 | 클럭 시에만 변화 |
| 응용 | SRAM, 설정 레지스터 | 파이프라인, 카운터 |

## Ⅳ. 실무 적용

### SRAM 셀 래치 동작

```
쓰기(Write):
WL=1 → 셀 활성화
BL=0 → Q = D_L, Q' = D_H

읽기(Read):
WL=0 → 셀 유지 상태
BL 활성화 → Q, Q' 차단
```

## Ⅴ. 기대효과 및 결론

랫치는 1비트 기억의 기초이다.

## 📌 관련 개념 맵

```
랫치
├── 정의: Enable 동안 입력 유지
├── 종류
│   ├── D 래치
│   ├── JK 래치
│   └── SRAM 셀 (2개 래치 교차 결합)
└── 파라미터
    ├── Setup 시간
    ├── Hold 시간
    └── 메타 안정성
```

## 👶 어린이를 위한 3줄 비유 설명

1. 랫치는 스위치를 누르면 그 상태를 유지하는 전기 장치예요
2. "저장" 버튼을 누르면 입력 상태가 기억되는데, 래치도 입력을 기억하는 회로예요
3. 컴퓨터의 1비트 메모리 셀(SRAM)은 래치 두 개를 엇갈 연결해서 1비트 정보를 저장해요
