# 저항 (Resistance)

## ## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 저항은 전류의 흐름을 방해하는 전자기적 성질로, 회로의 전력 소모(P=I²R)와 전압 강하(V=IR)를 결정하는 기초 파라미터이다.
> 2. **가치**: 저항의 정밀한 제어는 신호 무결성, 임피던스 매칭, 풀업/풀다운 회로, 전력 제한 등 회로의 안정적 동작을 보장하는 핵심 설계 요소이다.
> 3. **융합**: 저항-커패시턴스(RC) 시정수는 신호의 상승/하강 시간을 결정하고, 온-칩 저항은 발열(P=I²R)의 주원인이며, 비저항 시트(R□)는 배선 폭 설계의 기준이 된다.

---

## Ⅰ. 개요 (Context & Background)

### 개념
저항(Resistance, 기호: R, 단위: 옴[Ω])은 도체 내에서 전자의 이동을 방해하는 재료의 고유한 성질로, 전류의 흐름에 반비례하는 관계를 가진다. 수학적으로 옴의 법칙(R = V/I)으로 정의되며, 1옴은 1볼트의 전압을 가했을 때 1암페어의 전류가 흐르는 저항값이다. 미시적으로는 도체 내의 **전자 산란(Electron Scattering)**에 기인하며, 거시적으로는 재료의 **비저항(Resistivity, ρ)**과 길이(L), 단면적(A)에 의해 R = ρ·L/A로 계산된다. 디지털 회로에서 저항은 신호 전달의 속도(RC Delay), 전력 소모(Power), 노이즈 내성을 결정하는 핵심 파라미터이다.

### 💡 비유
저항은 수도관의 **마찰 저항(Friction)**이나 파이프의 **좁은 구간**과 완벽하게 대응된다. 물이 파이프를 흐를 때 파이프 벽과의 마찰, 파이프가 좁아지는 구간에서 유속이 늦어지는 것처럼, 전자도 도체 내의 원자와 충돌하며 저항을 경험한다. 파이프가 길수록, 좁을수록 마찰이 커져 물이 잘 안 흐르듯, 도선이 길수록, 얇을수록 저항이 커져 전류가 잘 흐르지 않는다. 또한, 물을 빨리 보내려면 더 높은 수압(전압)이 필요하듯, 전류를 많이 흘리려면 더 높은 전압이 필요하다.

### 등장 배경 및 발전 과정

#### 1. 옴의 법칙의 발견 (1827)
독일의 물리학자 게오르크 시몬 옴(Georg Simon Ohm)은 전압과 전류의 비례 관계를 실험적으로 발견하고, 이를 수식화하여 **옴의 법칙(I = V/R)**을 확립했다. 이 발견은 전기 회로를 정량적으로 분석할 수 있는 토대를 마련했으나, 당시 학계에서는 이를 받아들이지 않았다. 옴의 업적은 사후에 인정받아, 1881년 국제 전기 회의에서 저항의 단위로 "옴(Ω)"이 채택되었다.

#### 2. 금속의 저항 기전 해명 (1900s)
20세기 초, 드루데(Drude)와 조머펠트(Sommerfeld)은 **자유 전자 모델(Free Electron Model)**을 통해 금속의 전기 전도도를 설명했다. 금속 내의 전자가 격자와 충돌하며 산란되어 저항이 발생한다는 이론이 확립되었다. 이후 **페르미 속도(Fermi Velocity)**와 **평균 자유 경로(Mean Free Path)** 개념이 도입되어, 저항의 미시적 기전이 밝혀졌다.

#### 3. 반도체와 저항의 제어 (1950s)
트랜지스터의 발명과 함께 **도핑(Doping)** 기술이 개발되었다. 반도체에 불순물을 주입하여 캐리어 농도를 조절하면, 저항을 10배에서 100만 배까지 변화시킬 수 있다. 이는 **MOSFET**의 채널 저항을 게이트 전압으로 제어할 수 있는 기반이 되었고, 디지털 스위칭 소자의 탄생을 이끌었다.

#### 4. 집적회로와 배선 저항의 문제 (1980s~현재)
VLSI(초대규모 집적회로)가 발전하며, **배선 저항(Interconnect Resistance)**이 병목이 되었다. 알루미늄 배선(저항률 2.65μΩ·cm)에서 **구리 배선**(저항률 1.68μΩ·cm)으로 전환하여 저항을 37% 감소시켰으나, 공정이 7nm 이하로 미세화됨에 따라 **표면 산란(Surface Scattering)**과 **그레인 경계 산란(Grain Boundary Scattering)**으로 인한 **저항률 증가(Resistivity Size Effect)**가 심각한 문제로 대두되었다. 이를 해결하기 위해 **코발트 합금(Cobalt Alloy)** 배선과 **공극(Void) 제어 기술**이 개발되고 있다.

### 심층 동작 원리

#### ① 저항의 미시적 기전: 전자 산란
금속 내의 자유 전자는 **페르미 속도(v_F ~ 10⁶ m/s)**로 열 운동을 하며 무질별하게 이동한다. 전압을 가하면 전자는 **드리프트 속도(v_d ~ 10⁻⁴ m/s)**를 얻어 전기장 방향으로 흐른다. 이때 전자는 격자의 이온과 **산란(Scattering)**하며, 에너지를 잃고 방향을 바꾼다. 산란이 빈번할수록 전자의 이동이 방해받아 저항이 커진다. 산란의 빈도는 온도에 비례하며(T), **저항률 ρ = ρ₀[1 + α(T - T₀)]**로 표현된다. (α: 온도 계수)

#### ② 배선 저항과 IR Drop
IC 내부의 배선은 **시트 저항(R_s = ρ/t)**으로 특성화된다. 폭 W, 길이 L인 배선의 저항은 **R = R_s × (L/W)**이다. 예를 들어, 구리 M9 층(R_s = 0.02Ω/□)에서 폭 100nm, 길이 10μm인 배선의 저항은 **R = 0.02 × (10/0.1) = 2Ω**이다. 수백 암페어의 전류가 흐를 때, 배선 저항으로 인한 전압 강하(IR Drop)는 **V_drop = I × R**로 계산된다. 예를 들어, 100A가 2Ω 배선을 통과하면 **200V**의 강하가 발생하므로, 배선 폭을 넓히거나 다층으로 병렬 연결하여 저항을 줄여야 한다.

#### ③ RC 지연과 신호 전달 속도
디지털 신호는 배선의 **기생 저항(R)**과 **기생 커패시턴스(C)**로 형성되는 **RC 회로**를 통해 전달된다. 입력이 계단 함수로 변할 때, 출력은 **V(t) = V₀(1 - e^(-t/RC))**로 지수적으로 증가하며, 63.2% 도달 시간을 **시정수(Time Constant) τ = RC**라고 한다. 전압이 90% 도달하는 데 **t_90% ≈ 2.3τ**, 99% 도달하는 데 **t_99% ≈ 4.6τ**이 소요된다. 예를 들어, R = 1kΩ, C = 1pF인 배선에서 **τ = 1ns**이며, 신호가 90% 도달하는 데 **2.3ns**가 걸린다. 이는 고속 디지털 시스템의 병목이 된다.

#### ④ 종단 저항과 반사 방지
전송로의 **특성 임피던스(Z₀)**는 **Z₀ = √(L/C)**로 계산된다. (L: 단위 길이당 인덕턴스, C: 단위 길이당 커패시턴스) 신호원 임피던스(Z_s)와 부하 임피드(Z_L)가 Z₀와 일치하지 않으면, **반사 계수(Reflection Coefficient) Γ = (Z_L - Z₀)/(Z_L + Z₀)**만큼 반사파가 발생한다. 반사파는 신호 왜곡(Overshoot, Ringback)을 일으킨다. 이를 방지하기 위해 부하에 **종단 저항(R_term = Z₀)**을 연결하여 **완전 흡수(Absorption)** 시킨다. DDR4의 **ODT(On-Die Termination)**는 칩 내부에 R_term = 240/120/80/60/40/30/20Ω을 선택적으로 연결하여 기판 공간을 절약한다.

### 핵심 알고리즘/공식 & 실무 코드 예시

#### 옴의 법칙과 전력 공식
```
V = I × R                    (옴의 법칙)
P = V × I = V²/R = I²R       (전력)
R = ρ × L / A                (저항 공식: 비저항 × 길이 / 단면적)
R_s = ρ / t                  (시트 저항: 비저항 / 두께)
R_total = R₁ + R₂ + ...      (직렬 연결)
1/R_total = 1/R₁ + 1/R₂ + ... (병렬 연결)
τ = R × C                    (RC 시정수)
f_c = 1/(2πRC)               (RC 필터 차단 주파수)
```

#### 전압 분배 법칙 (Voltage Divider)
```
V_out = V_in × R₂ / (R₁ + R₂)
```

#### Python: 배선 저항 및 IR Drop 계산기
```python
import numpy as np

def calculate_interconnect_resistance(
    sheet_resistivity: float,    # Ω/□ (시트 저항)
    length: float,               # μm (길이)
    width: float,                # μm (폭)
    num_metal_layers: int = 1,   # 금속 층 수 (병렬)
    temperature_coeff: float = 0.0039,  # Cu 온도 계수 (1/°C)
    temp_celsius: float = 85     # 동작 온도
) -> dict:
    """
    IC 배선 저항 계산기

    Args:
        sheet_resistivity: 시트 저항 (Ω/□)
        length: 배선 길이 (μm)
        width: 배선 폭 (μm)
        num_metal_layers: 병렬 금속 층 수
        temperature_coeff: 온도 계수 (1/°C)
        temp_celsius: 동작 온도 (°C)

    Returns:
        저항 분석 결과
    """
    # 기본 저항 (25°C 기준)
    resistance_base = sheet_resistivity * (length / width)

    # 온도 보정
    resistance_temp = resistance_base * (1 + temperature_coeff * (temp_celsius - 25))

    # 병렬 층으로 저항 감소
    resistance_total = resistance_temp / num_metal_layers

    return {
        "resistance_base_25C": resistance_base,
        "resistance_at_temp": resistance_temp,
        "resistance_total": resistance_total,
        "resistance_per_um": resistance_total / length,
        "temperature_factor": 1 + temperature_coeff * (temp_celsius - 25)
    }


def calculate_ir_drop(
    voltage_supply: float,
    interconnect_r: float,
    current_draw: float
) -> dict:
    """
    IR Drop 계산기

    Args:
        voltage_supply: 공급 전압 (V)
        interconnect_r: 배선 저항 (Ω)
        current_draw: 소비 전류 (A)

    Returns:
        전압 강하 분석 결과
    """
    voltage_drop = current_draw * interconnect_r
    voltage_delivered = voltage_supply - voltage_drop
    drop_percentage = (voltage_drop / voltage_supply) * 100
    margin_percentage = (voltage_delivered / voltage_supply) * 100

    return {
        "voltage_supply": voltage_supply,
        "voltage_drop": voltage_drop,
        "voltage_delivered": voltage_delivered,
        "drop_percentage": drop_percentage,
        "margin_percentage": margin_percentage,
        "is_acceptable": margin_percentage >= 95.0  # ±5% 허용
    }


# 실무 시나리오: 7nm 공정 배선 저항 분석
result = calculate_interconnect_resistance(
    sheet_resistivity=0.02,   # M9 구리 (Ω/□)
    length=100,               # 100μm 길이
    width=0.05,               # 50nm 폭 (최소)
    num_metal_layers=2,       # 2층 병렬
    temp_celsius=105          # 코어 온도
)

print(f"기본 저항 (25°C): {result['resistance_base_25C']:.2f} Ω")
print(f"동작 저항 (105°C): {result['resistance_at_temp']:.2f} Ω")
print(f"총 저항 (2층 병렬): {result['resistance_total']:.2f} Ω")
print(f"온도 보정 계수: {result['temperature_factor']:.3f}")

# IR Drop 계산
ir_result = calculate_ir_drop(
    voltage_supply=1.1,
    interconnect_r=result['resistance_total'],
    current_draw=50  # 50A 소비
)

print(f"\nIR Drop 분석:")
print(f"공급 전압: {ir_result['voltage_supply']} V")
print(f"전압 강하: {ir_result['voltage_drop']*1000:.1f} mV")
print(f"도달 전압: {ir_result['voltage_delivered']:.4f} V")
print(f"전압 마진: {ir_result['margin_percentage']:.2f}%")
print(f"판정: {'양호' if ir_result['is_acceptable'] else '개선 필요'}")

"""
출력 예시:
기본 저항 (25°C): 40.00 Ω
동작 저항 (105°C): 51.20 Ω
총 저항 (2층 병렬): 25.60 Ω
온도 보정 계수: 1.280

IR Drop 분석:
공급 전압: 1.1 V
전압 강하: 1280.0 mV
도달 전압: -0.1800 V
전압 마진: -16.36%
판정: 개선 필요
(※ 이 경우 배선 폭을 늘리거나 층 수를 늘려야 함)
"""
```

### 과목 융합 관점 분석: 저항 × [운영체제/네트워크/보안/데이터베이스]

#### 1. 운영체제와의 융합: GPIO 풀업/풀다운 설정
임베디드 리눅스에서 **Device Tree**나 **GPIO API**를 통해 핀의 풀업/풀다운 저항을 설정한다. 예를 들어, `pinctrl-single` 드라이버는 레지스터 비트를 조작하여 풀업(10kΩ~100kΩ), 풀다운, 또는 부동(Floating) 상태를 선택한다. I²C 버스에서는 **4.7kΩ 풀업 저항**이 표준으로 사용되며, **Standard Mode(100kHz)**, **Fast Mode(400kHz)**, **High Speed Mode(3.4MHz)**에 따라 저항값을 다르게 설계해야 한다.

#### 2. 네트워크와의 융합: 이더넷 종단 저항
**10/100BASE-TX**는 **RJ-45 커넥터 내부에 49.9Ω × 2 = 100Ω 종단 저항**이 내장된 magnetics 모듈을 사용한다. **1000BASE-T**는 4쌍 모두가 **100Ω differential termination**을 필요로 하며, **Echo Cancellation** 기술이 이를 대신할 수도 있다. **PoE(Power over Ethernet)**는 PSE의 **25kΩ 탐지 저항(Detection Resistor)**을 통해 PD의 존재를 확인한다.

#### 3. 보안과의 융합: 측면 채널 공격 완화
**전력 분석 공격(Power Analysis Attack)**은 회로의 저항에 따라 전력 소모(P=V²/R)가 달라지는 점을 악용한다. **상수 시간(Constant-Time)** 알고리즘과 **전력 평탄화(Power Flattening)** 기술은 연산에 따른 저항 변화를 최소화하여 공격자의 통계적 분석을 어렵게 만든다. **하드웨어 난수 생성기(TRNG)**는 **저항 열 잡음(Thermal Noise)**를 엔트로피 소스로 활용한다.

#### 4. 데이터베이스와의 융합: SSD 저항 기반 마모 평준화
**NAND 플래시**의 셀 저항(Threshold Voltage)을 측정하여 데이터를 판독한다. **Wear Leveling**은 쓰기 횟수에 따라 셀의 저항 변화(Programming Disturb)를 분산시켜 수명을 연장한다. **SLC/MLC/TLC/QLC**는 저항 상태를 2/4/8/16단계로 구별하여, 1/2/3/4비트를 저장한다.

## Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

### 기술사적 판단 (실무 시나리오)

#### 시나리오 1: DDR4 종단 저항 최적화
**상황**: DDR4-3200 메모리 인터페이스에서 신호 무결성(Signal Integrity) 테스트 중 **Eye Diagram**이 닫히는 현상이 발생했다.

**근본 원인 분석**:
1. 메모리 컨트롤러와 DIMM 간 거리: 50mm (단방향 25mm)
2. 특성 임피던스 Z₀ = 40Ω (미세 스트립 라인)
3. 종단 저항이 미조정: ODT = 240Ω (높음)
4. 반사 계수 Γ = (240 - 40)/(240 + 40) = 0.71 → 71% 반사

**의사결정**:
1. **ODT를 40Ω로 설정**: R_term = Z₀와 일치 → Γ = 0 (반사 제거)
2. **Fly-by Topology 확인**: 최종 DIMM에서만 ODT 활성화
3. **Write Leveling**: 각 DIMM의 지연을 보정

**결과**: Eye Opening 70% 증가, BER(Bit Error Rate) 10⁻⁶ → 10⁻¹² 개선

#### 시나리오 2: PoE 전류 감지 저항 선정
**상황**: PoE++ 802.3bt Type 4(90W) PSE의 전류 감지 회로 설계 필요

**분석**:
- 최대 전류: 1.9A @ 48V
- 감지 전압 범위: 0 ~ 2.5V (ADC)
- 손실 허용: 0.5W 이하

**의사결정**:
1. **저항값 선정**: R_sense = 2.5V / 1.9A = 1.3Ω
2. **전력 손실**: P_loss = (1.9A)² × 1.3Ω = 4.7W (과다)
3. **재설계**: 증폭기(G=20) 사용 → R_sense = 0.065Ω
4. **최종 손실**: P_loss = (1.9)² × 0.065 = 0.23W (허용)
5. **타입**: 4와이어 켈빈 구조의 **SMD Current Sense Resistor** (0.065Ω ±1%, 1W)

### 도입 시 고려사항 (체크리스트)

#### 기술적 고려사항
- [ ] **저항 허용 오차**: 임계 응용에서 ±1% 이하
- [ ] **온도 계수**: ±50ppm/°C 이하 권장 (정밀 회로)
- [ ] **전력 정격**: 실제 소비 전력의 2배 이상 여유
- [ ] **주파수 특성**: 고주파에서 기생 인덕턴스 고려
- [ ] **배열 방식**: 직렬/병렬 조합으로 원하는 저항 실현

#### 운영/보안적 고려사항
- [ ] **전력 제한**: 과전류 방지를 위한 저항 선택
- [ ] **발열 관리**: 밀폐 공간에서의 열 싱크 배치
- [ ] **안전 인증**: UL, CE, RoHS 준수
- [ ] **EMI/EMC**: 저항 배열에 의한 전자기 간섭 평가

### 주의사항 및 안티패턴 (Anti-patterns)

#### 안티패턴 1: 풀업 저항값 무작위 선택
> **실수**: "10kΩ이면 다 되겠지"라고 생각하여 회로마다 동일한 풀업 사용
> **결과**: 고속 I²C(400kHz)에서 상승 시간(Rise Time)이 느려져 통신 오류
> **올바른 접근**: 버스 커패시턴스(C_bus)를 측정하고, τ = R×C < 0.3×bit_period가 되도록 계산

#### 안티패턴 2: 전력 저항의 충분한 여유 미확보
> **실수**: 정격 10W 저항에 9W 지속 인가
> **결과**: 열화로 저항값 변화, 수명 단축, 화재 위험
> **올바른 접근**: 실제 소비 전력의 50% 수준에서 운용

#### 안티패턴 3: 온도 보정 없는 정밀 저항 사용
> **실수**: ±0.1% 저항을 사용했으나 온도 계수 무시
> **결과**: 50°C 상승 시 저항값 0.5% 변화로 정밀도 상실
> **올바른 접근**: ±10ppm/°C 이하의 저항 사용, 필요 시 온도 보상 회로 추가

### 미래 전망 및 진화 방향
1. **자기 저항 증가(Rise of Resistivity)**: 3nm 이하 공정에서 표면 산란과 그레인 경계 효과로 배선 저항률이 급격히 증가할 것이다. 이에 대한 대책으로 **탄소나노튜브(CNT) 배선**, **그래핀 배선**, **2D 재료(MoS₂) 배선** 등의 대체 기술이 연구되고 있다.
2. **레이저 어닐링(Laser Annealing)**: 배선의 결정립 크기(Grain Size)를 증가시켜 그레인 경계 산란을 줄이는 **레이저 어닐링** 기술이 7nm/5nm 공정에서 도입되고 있다.
3. **가변 저항 소자(ReRAM)**: **Memristor**는 저항 상태를 전기적으로 가변할 수 있어, **Neuromorphic Computing**의 시냅스 소자, **Storage Class Memory**, **인-메모리 컴퓨팅**에 활용될 것이다.

## 📌 관련 개념 맵 (Knowledge Graph)

- **전압(Voltage)**: V = IR로 저항과 관계
- **전류(Current)**: I = V/R로 저항에 반비례
- **전력(Power)**: P = I²R로 저항에 정비례
- **커패시터**: RC 필터 구성, τ = RC로 신호 속도 결정
- **인덕턴스**: RL 회로, R/(sL) 1차 저역 통과
- **임피던스(Impedance)**: 교류에서의 복소 저항 Z = R + jX
- **전송로(Transmission Line)**: 특성 임피던스 Z₀ = √(L/C)
- **시트 저항(Sheet Resistance)**: IC 배선 설계 기준

## 👶 어린이를 위한 3줄 비유 설명

1. **저항은 수도관의 막힘과 같아요**. 파이프가 좁거나 안이 거칠수록 물이 잘 안 흐르듯, 전선이 얇거나 재료가 나쁠수록 전기가 잘 안 흘러요.

2. **저항은 전기를 조절하는 밸브 같아요**. 물이 너무 세게 나오면 위험하듯, 전기도 너무 세게 흐르면 위험해요. 저항이 적당히 흐름을 제어해서 안전하게 만들어준답니다.

3. **저항은 전기를 열로 바꿔요**. 전기가 저항을 통과할 때 에너지를 잃고 열이 나는데, 이건 전기장판이나 히터가 따뜻해지는 원리예요. 하지만 컴퓨터에서는 이 열이 문제가 되어서, 저항을 최소화하는 설계를 한답니다.
