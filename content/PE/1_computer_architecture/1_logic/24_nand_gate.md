# NAND 게이트 (NAND Gate)

## 핵심 인사이트 (3줄 요약)
1. NAND 게이트는 AND 게이트의 출력을 반전시킨 논리 회로로, Y = (A·B)' = ¬(A∧B)로 표현되며 범용 게이트이다
2. NAND 게이트만으로 모든 논리 함수(AND, OR, NOT, XOR 등)를 구현할 수 있어 VLSI 설계의 표준 소자이다
3. 기술사 시험에서는 CMOS NAND 회로 구조, 트랜스컨덕턴스, 전송 지연, 범용성 증명이 핵심이다

## Ⅰ. 개요 (500자 이상)

NAND 게이트는 "NOT AND"의 줄임말로, AND 게이트의 출력을 반전시킨 논리 회로이다. 모든 입력이 1일 때만 0을 출력하며, 그 외의 경우에는 1을 출력한다. 불 대수식은 Y = (A·B)' = ¬(A∧B)로 표현한다.

NAND 게이트의 핵심 중요성은 **범용 게이트(Universal Gate)**라는 점이다. NAND 게이트만으로 모든 논리 함수(AND, OR, NOT, XOR, XNOR)를 구현할 수 있어, VLSI 설계에서 표준 소자로 사용된다. 이는 제조 공정을 단순화하고 칩 면적을 감소시키는 장점이 있다.

```
ANSI 기호          IEC 기호
    ┌─┐              ┌───┐
A ──┤ &│          A ─┤ & ├─── Y
    │   │─── Y         │   │
B ──┤   │          B ─┤   │
    └─┘○             └───┘
    (출력에 NOT 표시)   (내부에 &, 출력에 ○)
```

CMOS NAND 게이트는 PMOS 직렬(상단)과 NMOS 병렬(하단)로 구현된다.

## Ⅱ. 아키텍처 및 핵심 원리 (1000자 이상)

### CMOS NAND 게이트 회로

```
CMOS 2입력 NAND 게이트:
        V_DD
         │
      ┌──┴──┐
      │     │
     PMOS  PMOS
      │     │
   A ─┤     ├─┐
      │     │ │
   B ─┤     ├─┤─── Y
      │     │ │
      └──┬──┘
         │
     NMOS  NMOS
         │
        GND

동작:
A=0 또는 B=0: NMOS OFF, PMOS ON → Y=1
A=1, B=1: NMOS ON, PMOS OFF → Y=0
```

### NAND 범용성 증명

NAND 게이트로 기본 게이트 구현:

```
1) NOT 게이트:
   Y = NAND(A, A) = (A·A)' = A'

2) AND 게이트:
   Y = NOT(NAND(A, B)) = (A·B)'' = A·B

3) OR 게이트 (드 모르강 법칙):
   A + B = (A'·B')' = NAND(NOT(A), NOT(B))
        = NAND(NAND(A,A), NAND(B,B))

4) XOR 게이트:
   A⊕B = (A·B') + (A'·B)
       = NAND(NAND(A, NAND(B,B)), NAND(NAND(A,A), B))
```

### 전송 지연

```
t_p(NAND) ≈ 0.69 × R_eq × C_L

CMOS NAND는 2입력 단일 스테이지:
t_p ≈ 5ns (74HC, V_CC=5V, C_L=50pF)
```

## Ⅲ. 융합 비교 및 다각도 분석

### 범용 게이트 비교

| 게이트 | 범용성 | 구현 복잡도 | 응용 |
|--------|--------|-------------|------|
| NAND | 범용 (단독으로 모든 논리 구현 가능) | 낮음 | VLSI 표준 |
| NOR | 범용 (단독으로 모든 논리 구현 가능) | 중간 | 특수 응용 |

### NAND vs NOR 구현

| 비교 항목 | NAND (PMOS 직렬) | NOR (PMOS 병렬) | 설명 |
|----------|------------------|-----------------|------|
| PMOS 구조 | 직렬 | 병렬 | NAND가 전류 구동 우수 |
| NMOS 구조 | 병렬 | 직렬 | 동일 |
| 속도 | 빠름 | 느림 | PMOS 이동도 < NMOS |
| 면적 | 작음 | 큼음 | NAND가 고밀도에 유리 |
| 표준 여부 | 표준 | 비표준 | NAND가 VLSI 표준 |

## Ⅳ. 실무 적용 및 기술사적 판단

### SRAM 셀에서의 NAND

6T SRAM은 2개 CMOS 인버터를 교차 결합한 래치 구조이다. 인버터는 NAND가 아니나, 래치 안정성은 CMOS 인버터의 높은 노이즈 마진에 의존한다.

## Ⅴ. 기대효과 및 결론

NAND 게이트는 범용 게이트로서 모든 디지털 회로의 기초이다. CMOS NAND는 표준 VLSI 소자이다.

## 📌 관련 개념 맵

```
NAND 게이트
├── 정의: Y = (A·B)'
├── 범용성
│   ├── NOT: NAND(A,A)
│   ├── AND: NOT(NAND(A,B))
│   └── OR: 드 모르강 법칙
└── CMOS 구조
    ├── PMOS: 직렬
    └── NMOS: 병렬
```
