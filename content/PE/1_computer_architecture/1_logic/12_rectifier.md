# 정류기 (Rectifier)

## 핵심 인사이트 (3줄 요약
1. 정류기는 교류(AC)를 직류(DC)로 변환하는 전력 변환 회로로, 컴퓨터 전원 공급 장치(PSU)의 핵심 요소이다
2. 반파 정류, 전파 정류, 브리지 정류, 다상 정류 등 부하 요구에 따른 최적 위상 토폴로지가 존재한다
3. 기술사 시험에서는 정류 효율, 리플 전압, 역전압 내력(PIV), 다상 정류의 고조파 왜율 분석이 중요하다

## Ⅰ. 개요 (500자 이상)

정류기(Rectifier)는 교류 전압을 직류 전압으로 변환하는 비선형 전력 회로 장치이다. 교류 전원의 방향성을 제어하여 단방향 전류로 정류함으로써, 전자 장치에 필요한 DC 전원을 공급한다. 컴퓨터 시스템에서 정류기는 전원 공급 장치(PSU)의 입력단에 위치하며, 상용 교류 전원(110V/220V 50/60Hz)을 DC 버스 전압으로 변환하는 1차 전력 변환 단계를 담당한다.

정류기의 핵심 동작 원리는 다이오드의 순방향 바이어스 특성을 이용한 비대칭적 전류 흐름 제어이다. 이상적인 정류 다이오드는 순방향 시 단락 회로로 동작하여 전류를 자유롭게 통과시키고, 역방향 시 개방 회로로 동작하여 전류 흐름을 차단한다. 실제 다이오드는 순방향 시 약 0.7V(Silicon) 또는 0.3V(Schottky)의 순방향 강하 전압(VF)이 발생하며, 역방향 시 미세한 누설 전류(IR)가 흐른다. 정류기 설계 시 이러한 비이상적 특성을 고려한 전압 강하와 전력 손실 분석이 필수적이다.

정류기는 출력 품질과 효율 관점에서 여러 토폴로지로 분류된다. **반파 정류기(Half-Wave Rectifier)**는 입력 사이클의 절반만 출력으로 전달하여 간단하지만 리플이 크고 효율이 낮다(최대 40.6%). **전파 정류기(Full-Wave Rectifier)**는 중심 탭 변압기를 사용하여 입력 사이클의 양 반주를 모두 활용하며, 2배의 출력 주파수로 리플을 감소시킨다. **브리지 정류기(Bridge Rectifier)**는 4개 다이오드를 다리 형태로 배치하여 중심 탭 없이 전파 정류를 구현하며, 2개 다이오드의 순방향 강하(약 1.4V)가 발생하지만 변압기 활용도가 높다.

**다상 정류기(Polyphase Rectifier)**는 3상 전원을 사용하여 6펄스, 12펄스, 18펄스, 24펄스 등의 고주파수 리플을 생성하며, 대용량 데이터 센터 PSU와 서버 전원 시스템에 널리 사용된다. 3상 브리지 정류기는 6개 다이오드로 구성되어 입력 주파수의 6배(50Hz 입력 시 300Hz)의 리플 주파수를 생성하며, 필터링을 위한 커패시터 크기를 획기적으로 감소시킨다. 12펄스 정류기는 2개의 6펄스 정류기를 30° 위상차로 병렬 운영하여 고조파 왜율을 5차, 7차 고조파에서 12차, 24차 고조파로 이동시키며, THD(Total Harmonic Distortion)를 약 30%에서 10% 미만으로 감소시킨다.

컴퓨터 시스템에서 정류기의 효율은 전체 PSU 효율(Energy Star, 80 Plus 등급)에 직접적인 영향을 미친다. 실리콘 다이오드 대신 쇼트키 배리어 다이오드(SBD)를 사용하면 순방향 강하 전압을 0.3V 수준으로 감소시켜 정류 손실을 약 50% 감소시킬 수 있다. 최근 PSU에서는 다이오드 정류기 대신 MOSFET을 동기식 스위칭으로 제어하는 **동기 정류(Synchronous Rectification)**를 채택하여 전압 강하를 10mV 이내로 감소시키고 효율을 95% 이상으로 향상시킨다.

## Ⅱ. 아키텍처 및 핵심 원리 (1000자 이상)

### 정류기의 회로 토폴로지 및 동작 원리

정류기의 기본 동작은 PN 접합 다이오드의 전압-전류 비선형 특성에 기반한다. 쇼클리 다이오드 방정식에 따르면 다이오드 전류 I_D는 다음과 같다:

```
I_D = I_S × (exp(V_D / (n × V_T)) - 1)

여기서:
- I_S: 포화 전류 (Saturation Current), 약 10^-12 ~ 10^-6 A
- V_D: 다이오드 양단 전압
- n: 이상성 계수 (Ideality Factor), 1~2
- V_T: 열 전압 (Thermal Voltage) = k × T / q ≈ 25.9mV @ 25°C
```

순방향 바이어스(V_D > 0) 시 지수 함수적 전류 증가로 인해 낮은 순방향 강하에서도 큰 전류가 흐른다. 역방향 바이어스(V_D < 0) 시 -I_S에 수렴하는 미세한 누설 전류만 흐른다. 이러한 비대칭적 특성이 정류 동작의 기본 원리이다.

### 반파 정류기 (Half-Wave Rectifier)

```
입력 AC ──→|──┬─── 출력 DC (+)
              │
              └───→ (반환 경로)

구성 요소:
- 단일 다이오드
- 입력: 단상 AC
- 출력: 양의 반파만 통과 (음의 반파 차단)
```

반파 정류기는 입력 주파수 f_input에서 출력 리플 주파수가 f_input과 동일하다. 출력 전압 V_out(t)는 다음과 같이 표현된다:

```
V_out(t) = V_m × sin(ωt)  (0 < ωt < π)
         = 0              (π < ωt < 2π)

평균 출력 전압: V_avg = V_m / π ≈ 0.318 × V_m
RMS 출력 전압: V_rms = V_m / 2
리플 계수(Ripple Factor): γ = √((V_rms/V_avg)^2 - 1) ≈ 1.21 (121%)
정류 효율: η = 40.6%
```

높은 리플 계수로 인해 대용량 필터링이 필요하며, 낮은 효율로 인해 고용량 전원에는 적합하지 않다. 그러나 저비용으로 간단히 구현 가능하므로 저전력 센서나 충전기 등에 사용된다.

### 전파 정류기 - 중심 탭 (Full-Wave Center-Tapped)

```
          중심 탭
            ↓
    AC ──┬─┬───→|──┬─── 출력 (+)
         │ │      │
        [CT]      │
         │ │      │
    AC ──┴─┴───→|──┴─── 공통

구성 요소:
- 중심 탭 변압기
- 2개 다이오드
- 출력: 양/음 반파 모두 정류
```

중심 탭 전파 정류기는 입력 사이클의 양 반주 모두를 동일 극성의 출력으로 변환한다. 다이오드 D1은 0~π 구간에서 도통하여 V_m × sin(ωt)를 출력하고, D2는 π~2π 구간에서 도통하여 V_m × sin(ωt - π)의 절대값을 출력한다.

```
평균 출력 전압: V_avg = 2 × V_m / π ≈ 0.637 × V_m
RMS 출력 전압: V_rms = V_m / √2
리플 주파수: f_ripple = 2 × f_input
리플 계수: γ ≈ 0.48 (48%)
정류 효율: η = 81.2%
PIV (Peak Inverse Voltage): 2 × V_m
```

PIV는 다이오드가 차단 상태에서 견뎌야 하는 최대 역전압이다. 중심 탑 정류기의 PIV는 2×V_m으로 높아야 하며, 이는 다이오드 선정 시 중요한 설계 파라미터이다.

### 브리지 정류기 (Full-Wave Bridge)

```
     AC 입력 ──┬──→|──┬─── 출력 (+)
               │      │
               │      │
             ──┴──→|──┴───
             ──┬──→|──┬───
               │      │
               │      │
     AC 입력 ──┴──→|──┴─── 출력 (-)

구성 요소:
- 4개 다이오드 (브리지 구성)
- 중심 탭 불필요
```

브리지 정류기는 4개 다이오드를 휘튼스턴 브리지 형태로 배치하여 중심 탭 없이 전파 정류를 구현한다. 양의 반주에서 D1, D2가 도통하고, 음의 반주에서 D3, D4가 도통한다.

```
순방향 경로 전압 강하: 2 × V_F ≈ 1.4V (Si 다이오드)
평균 출력 전압: V_avg = 2 × (V_m - 2V_F) / π
RMS 출력 전압: V_rms = (V_m - 2V_F) / √2
PIV: V_m (중심 탑의 절반)
정류 효율: 약 81.2% (이상적), 실제는 2×V_F 강하로 감소
```

PIV가 V_m으로 낮아 다이오드 내압 요구 사항이 완화되나, 항상 2개 다이오드가 직렬 도통하므로 순방향 강하가 2배 발생한다. 이는 저전압 고전류 응용에서 전력 손실(P_loss = 2×V_F×I_avg)의 주요 원인이다.

### 3상 브리지 정류기 (3-Phase Full-Wave Bridge)

```
   A상 ──→|──┬───────→|──┬─── 출력 (+)
          │      │      │
   B상 ──→|──┼───────→|──┤
          │      │      │
   C상 ──→|──┴───────→|──┴───
          │             │
   A상 ──→|──┬───────→|──┬─── 출력 (-)
          │      │      │
   B상 ──→|──┼───────→|──┤
          │      │      │
   C상 ──→|──┴───────→|──┴───

구성 요소:
- 6개 다이오드 (3상 브리지)
- 평형 3상 입력
```

3상 정류기는 3상 교류 전원의 각 상이 120° 위상차를 가지므로, 다이오드 도통 각격이 60°씩 순차적으로 전환된다. 상간 전압(line-to-line voltage)이 최대인 상의 다이오드 쌍이 순차적으로 도통하여 연속적인 출력 전류를 생성한다.

```
상간 전압: V_L-L = √3 × V_phase (Y 연결)
평균 출력 전압: V_avg = (3 × √3 / π) × V_m ≈ 1.654 × V_m
리플 주파수: f_ripple = 6 × f_input (300Hz @ 50Hz 입력)
리플 계수: γ ≈ 0.042 (4.2%)
정류 효율: η > 95%
```

3상 정류기는 6펄스 출력을 생성하며, 단상 정류기의 2펄스에 비해 리플 주파수가 3배 높고 리플 계수가 1/30 이하로 감소한다. 이는 필터링 커패시터 크기를 획기적으로 감소시키며, 대용량 서버 PSU와 데이터 센터 전원 시스템의 표준 토폴로지이다.

### 12펄스 다상 정류기 (12-Pulse Polyphase Rectifier)

```
6펄스 정류기 1 ──→┬─── 병렬 결합 ──→ 출력
                  │
6펄스 정류기 2 ──→┴─── (30° 위상차)
                  │
              3상 변압기
             (Δ-Y, 30° 위상 이동)
```

12펄스 정류기는 2개의 6펄스 정류기를 30° 위상차로 병렬 운영한다. 일반적으로 3상 변압기의 1차 측을 Δ(델타) 결선, 2차 측을 Y(와이) 결선하여 30° 위상 이동을 생성한다. 두 정류기의 출력이 직렬 또는 병렬로 결합되어 12펄스 출력을 생성한다.

```
출력 리플 주파수: 12 × f_input (600Hz @ 50Hz)
고조파 주파수: 12k ± 1 차 고조파 (11차, 13차, 23차, 25차...)
저차 고조파(5차, 7차) 제거: 상쇄로 인한 소거
THD 감소: 30% → 10% 미만
전력 품질: IEC 61000-3-12 표준 준수 가능
```

높은 펄스 수는 입력 전류 고조파 왜율을 감소시켜 전력 품질을 향상시킨다. 데이터 센터와 대용량 UPS(Uninterruptible Power Supply) 시스템에서 표준으로 사용되며, 18펄스, 24펄스로 확장 가능하다.

### 다이오드 유형별 정류기 성능 비교

| 다이오드 유형 | 순방향 강하(V_F) | 역내전압(V_RRM) | 역복구 시간(t_rr) | 누설 전류(I_R) | 스위칭 속도 | 응용 분야 |
|---------------|------------------|----------------|-------------------|---------------|--------------|-----------|
| Si(실리콘) PN | 0.7-1.1V @ 1A | 50-5000V | 50-500ns | 1-10μA | 중간 (kHz) | 일반 정류, 저주파 |
| Schottky SBD | 0.2-0.4V @ 1A | 20-200V | 10-100ns | 0.1-1mA | 높음 (MHz) | 저전압 정류, 고속 |
| SiC(실리콘 카바이드) | 1.5-3V @ 10A | 600-10kV | 10-50ns | 1-10μA | 매우 높음 (MHz) | 고전력, 고온, EV |
| GaN(질화갈륨) | 1.2-2V @ 10A | 600-1200V | 5-20ns | 1-10μA | 초고속 (10MHz) | 고효률 정류, RF |
| 초고압 Si | 1.5-2V @ 10A | 5-20kV | 1-10μs | 10-100μA | 낮음 (Hz-kHz) | HVDC 전송, 산업용 |

**기술사적 판단 포인트**:
- **저전압 고전류 응용** (<24V): Schottky SBD 선택으로 V_F 감소, 효율 5-10% 향상
- **고전압 응용** (>400V): SiC 또는 Si PN 선택, PIV 여유도 1.5-2배 확보
- **고주파 스위칭** (>100kHz): SiC/GaN 선택으로 역복구 손실 감소
- **고온 환경** (>125°C): SiC 선택으로 열 안정성 확보

| 중심 탑 | 1.5x | 2x (중심탑) | 2x | 2.5x | 81.2% | 0.62 |
| 브리지 | 2x | 없음 | 4x | 2x | 81.2% | 0.41 (최상) |
| 3상 | 6x | 4x (3상) | 6x | 10x | 95% | 0.42 |
| 12펄스 | 12x | 8x (다상) | 12x | 20x | 97% | 0.62 |

**비용 효율성 분석**:
- 1kW 미만: 브리지 정류기가 최적의 비용/효율 비율 제공
- 1kW-10kW: 3상 브리지가 규형 있는 선택
- 10kW 이상: 12펄스 이상 다상 정류가 전력 품질 이득으로 비용 상쇄

## Ⅳ. 실무 적용 및 기술사적 판단 (800자 이상)

### 컴퓨터 PSU에서의 정류기 설계 고려사항

컴퓨터 전원 공급 장치(PSU)에서 정류기는 입력 필터와 PFC(Power Factor Correction) 회로 사이에 위치하며, 전체 시스템 효율과 신뢰성에 직접적인 영향을 미친다. 80 Plus 골드/플래티넘/티타늄 등급의 PSU는 90% 이상의 효율을 요구하며, 정류기 손실 최소화가 핵심 설계 목표이다.

**입력 전압 범위 고려**: 전 세계 상용 교류 전원은 100V-120V(북미), 220V-240V(유럽/아시아) 등 다양하다. PSU 설계 시 85V-265V의 광범위 입력 전압을 지원하는 Universal Input을 채택하면, 저전압 입력 시 입력 전류가 2-3배 증가하여 정류기 다이오드의 전류 용량과 열 발열이 설계 제약이 된다. 기술사는 다이오드 평균 전류 용량(I_FAV)이 최대 입력 전류의 2배 이상이 되도록 선정하여 열 마진을 확보해야 한다.

**역복구 손실(Reverse Recovery Loss) 관리**: PFC 회로의 스위칭 주파수가 50kHz-200kHz로 높아지면, 정류 다이오드의 역복구 시간(t_rr) 동안 발생하는 역복구 전류(I_rr)가 스위칭 손실의 주요 원인이 된다. Schottky SBD는 역복구 전류가 없으나 PIV 제한(최대 200V)으로 인해 230VAC 이상 입력에는 사용 불가하다. 230VAC 입력(맥값 325V)에서는 Si PN 다이오드를 사용하며, 초고속 다이오드(Ultra-Fast Recovery, t_rr < 35ns)를 선택하여 스위칭 손실을 최소화한다.

```
역복구 손실: E_rr = 0.5 × V_R × I_rr × t_rr
스위칭 주파수 100kHz, V_R = 325V, I_rr = 2A, t_rr = 50ns 인 경우:
E_rr = 0.5 × 325 × 2 × 50×10^-9 = 16.25μJ per transition
전력 손실: P_rr = E_rr × f_sw × 2 (상/하단) = 16.25μJ × 100kHz × 2 = 3.25W
```

SiC SBD(t_rr < 10ns)로 교체 시 손실을 0.65W로 80% 감소 가능하나, 다이오드 비용이 3-5배 증가한다. 기술사는 효율 등급 요구사항과 비용 트레이드오프를 분석하여 최적 다이오드를 선정해야 한다.

### 데이터 센터 PSU에서의 다상 정류기 적용

대용량 데이터 센터의 랙 마운트 PSU(1.5kW-3kW)는 3상 입력(380V/480V AC)을 사용하며, 6펄스 또는 12펄스 정류기를 채택한다. 3상 정류는 입력 전류를 3상으로 분산하여 위상별 전류가 평균 1/3로 감소하고, 케이블과 다이오드의 전류 용량 요구 사항을 완화한다.

**고조파 왜율 관리**: 6펄스 정류기의 입력 전류는 5차(250Hz), 7차(350Hz) 고조파가 주파수 대역에 포함되며, THD(Total Harmonic Distortion)가 30% 이상이다. 이는 입력 필터와 PFC 회로의 설계 부담을 증가시키며, IEC 61000-3-12 전력 품질 표준 준수를 위한 고조파 저감 필터가 필요하다. 12펄스 정류기는 5차, 7차 고조파를 상쇄하여 THD를 10% 미만으로 감소시키며, 입력 필터 크기를 50% 이상 감소시킨다.

```
6펄스 정류기 고조파 전류 비율:
- 기본파: 100%
- 5차: 20%
- 7차: 14%
- 11차: 9%
- 13차: 7%
- THD: 약 30%

12펄스 정류기 고조파 전류 비율 (2개 6펄스 정류기 30° 위상차):
- 기본파: 100%
- 5차: 0% (상쇄)
- 7차: 0% (상쇄)
- 11차: 9%
- 13차: 7%
- THD: 약 12%
```

### 동기 정류(Synchronous Rectification)의 적용

고전력 PSU(1kW 이상)와 서버용 VRM(Voltage Regulator Module)에서는 다이오드 대신 MOSFET을 사용한 동기 정류를 채택한다. MOSFET의 온-저항(R_DS(on))이 10mΩ 이하인 경우, 순방향 강하 전압 V_F = I × R_DS(on) = 20A × 10mΩ = 0.2V로 다이오드의 0.7V보다 70% 낮다.

```
다이오드 손실: P_diode = V_F × I_avg = 0.7V × 20A = 14W
MOSFET 손실: P_fet = I_RMS^2 × R_DS(on) = 20^2 × 0.01 = 4W
손실 감소: 10W (71% 감소)
```

동기 정류는 MOSFET의 게이트 드라이버 회로가 필요하여 비용이 증가하나, 500W 이상 PSU에서 효율 향상 2-3%로 전력 절감 효과가 비용 증가를 상쇄한다. 특히 12V/48V 저전압 고전류 출력에서는 필수적이다.

### 기술사 시험 대비 문제 분석

**문제 1**: 220VAC, 50Hz 단상 전원을 사용하는 500W PSU의 브리지 정류기 설계 시, 다이오드의 PIV(Peak Inverse Voltage)와 평균 전류 용량(I_FAV)을 산정하시오.

**해설**:
```
1) PIV 산정:
V_peak = √2 × V_RMS = √2 × 220V = 311V
PIV = V_peak = 311V
안전 계수 1.5-2배 적용: PIV_rating = 1.5 × 311V ≈ 466V
따라서 600V 등급 다이오드 선택

2) I_FAV 산정:
입력 유효 전력: P_in = P_out / η = 500W / 0.85 ≈ 588W
입력 평균 전류: I_in_avg = P_in / V_RMS = 588 / 220 ≈ 2.67A
브리지 정류기는 각 다이오드가 1/2 주기 도통하므로:
I_diode_avg = I_in_avg / 2 = 2.67 / 2 ≈ 1.34A
안전 계수 2배 적용: I_FAV_rating = 2 × 1.34 ≈ 2.68A
따라서 3A 등급 다이오드 선택
```

**문제 2**: 3상 380VAC, 50Hz 입력을 사용하는 10kW 데이터 센터 PSU의 6펄스 브리지 정류기 출력 리플 주파수와 평균 출력 전압을 계산하시오.

**해설**:
```
1) 리플 주파수:
f_ripple = 6 × f_input = 6 × 50Hz = 300Hz

2) 평균 출력 전압:
상간 전압(Line-to-Line): V_L-L = 380V
맥값: V_m = √2 × 380 ≈ 537V
V_avg = (3 × √3 / π) × V_m = (3√3 / π) × 537 ≈ 1.654 × 537 ≈ 888V

3) 리플 계수:
γ ≈ 4.2% (6펄스 이론값)
리플 전압(V_ripple) = γ × V_avg = 0.042 × 888 ≈ 37V
```

## Ⅴ. 기대효과 및 결론 (400자 이상)

정류기 기술 이해는 컴퓨터 전원 시스템 설계와 전력 품질 관리의 핵심이다. 적절한 정류 토폴로지 선정은 PSU 효율 5-10% 향상과 신뢰성 개선에 직결된다. 3상 12펄스 정류기와 SiC/GaN 다이오드의 채택은 데이터 센터 전력 비용 절감과 전력 품질 준수에 기여하며, 동기 정류 기술은 고밀도 전원 변환의 효율 한계를突破한다.

기술사는 부하 용량, 입력 전압 범위, 효율 요구사항, 비용 제약을 종합 분석하여 최적 정류 솔루션을 제시해야 한다. 단상 브리지는 1kW 미만 일반 응용에, 3상 6펄스는 1-10kW 산업용에, 12펄스 이상 다상은 10kW 이상 데이터 센터에 적합하다. 다이오드 선정 시 PIV 여유도 1.5-2배, 전류 용량 2배의 안전 계수 적용이 설계 원칙이다.

## 📌 관련 개념 맵

```
정류기 (Rectifier)
├── 회로 토폴로지
│   ├── 반파 정류 (Half-Wave)
│   ├── 전파 정류 (Full-Wave)
│   │   ├── 중심 탑 (Center-Tapped)
│   │   └── 브리지 (Bridge)
│   └── 다상 정류 (Polyphase)
│       ├── 3상 6펄스
│       ├── 12펄스
│       └── 18/24펄스
├── 다이오드 유형
│   ├── Si PN 다이오드
│   ├── Schottky SBD
│   ├── SiC 다이오드
│   └── GaN 다이오드
├── 성능 파라미터
│   ├── PIV (Peak Inverse Voltage)
│   ├── V_F (Forward Voltage Drop)
│   ├── I_FAV (Average Forward Current)
│   ├── t_rr (Reverse Recovery Time)
│   └── η (Rectification Efficiency)
├── 출력 특성
│   ├── V_avg (Average Output Voltage)
│   ├── V_ripple (Ripple Voltage)
│   ├── γ (Ripple Factor)
│   └── f_ripple (Ripple Frequency)
└── 응용 분야
    ├── 컴퓨터 PSU
    ├── 서버 VRM
    ├── 데이터 센터 UPS
    └── 산업용 전원 장치
```

## 👶 어린이를 위한 3줄 비유 설명

1. 정류기는 전기의 "일방 통행 다리" 같아서, 왔다 갔다하는 교류 전기를 한 방향으로만 흐르게 만들어준대요
2. 교류 전기를 정류기에 통과시키면, 물결치던 전기가 평평한 직류 전기가 되어 컴퓨터가 마시기 좋은 전기가 돼요
3. 브리지 정류기는 4개의 일방통행 문을 똑똑하게 배치해놓아서, 전기가 양쪽 방향에서 들어와도 항상 한쪽 방향으로 나가게 해줘요

## 💻 Python 코드: 3상 정류기 설계 및 리플 분석

```python
import numpy as np
import matplotlib.pyplot as plt

def three_phase_rectifier_analysis(V_phase_rms=220, f=50, R_load=10):
    """
    3상 브리지 정류기 해석
    """
    # 파라미터 설정
    omega = 2 * np.pi * f
    V_phase_peak = V_phase_rms * np.sqrt(2)
    V_ll_peak = V_phase_peak * np.sqrt(3)  # 상간 전압 맥값

    # 시간 배열 (2 주기)
    t = np.linspace(0, 2/f, 1000)

    # 3상 전압 생성 (120° 위상차)
    Va = V_phase_peak * np.sin(omega * t)
    Vb = V_phase_peak * np.sin(omega * t - 2*np.pi/3)
    Vc = V_phase_peak * np.sin(omega * t - 4*np.pi/3)

    # 상간 전압
    Vab = Va - Vb
    Vbc = Vb - Vc
    Vca = Vc - Va

    # 6펄스 정류 출력 (가장 높은 상간 전압 선택)
    V_out = np.maximum(np.maximum(Vab, Vbc), Vca)

    # 출력 전류 (부하 저항 통해)
    I_out = V_out / R_load

    # 출력 리플 주파수 (6배 입력 주파수)
    f_ripple = 6 * f

    # 평균 출력 전압 (이론적)
    V_avg_theoretical = (3 * np.sqrt(3) / np.pi) * V_ll_peak

    # 실제 평균 출력 전압 (수치 적분)
    V_avg_actual = np.mean(V_out)

    # 리플 전압 (최대-최소)
    V_ripple_pp = np.max(V_out) - np.min(V_out)

    # 리플 계수
    V_rms = np.sqrt(np.mean(V_out**2))
    ripple_factor = np.sqrt((V_rms / V_avg_actual)**2 - 1)

    # 정류 효율 (입력/출력 전력 비)
    P_in = 3 * (V_phase_rms**2) / (2 * R_load)  # 3상 입력 전력
    P_out = (V_avg_actual**2) / R_load
    efficiency = (P_out / P_in) * 100

    # 결과 출력
    print("=== 3상 브리지 정류기 해석 결과 ===")
    print(f"입력 상 전압: {V_phase_rms} Vrms")
    print(f"입력 주파수: {f} Hz")
    print(f"상간 전압: {V_ll_peak/np.sqrt(2):.1f} Vrms")
    print(f"리플 주파수: {f_ripple} Hz")
    print(f"평균 출력 전압: {V_avg_actual:.1f} V (이론: {V_avg_theoretical:.1f} V)")
    print(f"출력 리플 전압(p-p): {V_ripple_pp:.1f} V")
    print(f"리플 계수: {ripple_factor:.4f} ({ripple_factor*100:.2f}%)")
    print(f"입력 전력: {P_in:.1f} W")
    print(f"출력 전력: {P_out:.1f} W")
    print(f"정류 효율: {efficiency:.2f}%")

    # 다이오드 요구 사양
    I_diode_avg = P_out / (V_avg_actual * 3)  # 각 다이오드의 평균 전류
    PIV = V_ll_peak  # 최대 역전압

    print("\n=== 다이오드 요구 사양 ===")
    print(f"다이오드 평균 전류: {I_diode_avg:.2f} A")
    print(f"PIV (최대 역전압): {PIV:.1f} V")
    print(f"推奨 다이오드 전압 등급: {PIV*1.5:.0f} V (여유도 1.5배)")
    print(f"推奨 다이오드 전류 등급: {I_diode_avg*2:.1f} A (여유도 2배)")

    return {
        't': t,
        'Va': Va,
        'Vb': Vb,
        'Vc': Vc,
        'V_out': V_out,
        'I_out': I_out,
        'f_ripple': f_ripple,
        'V_avg': V_avg_actual,
        'ripple_factor': ripple_factor,
        'efficiency': efficiency
    }

# 실행
result = three_phase_rectifier_analysis(V_phase_rms=220, f=50, R_load=10)

# 그래프 생성 (선택적, 주피터 노트북 환경에서)
try:
    plt.figure(figsize=(12, 8))

    plt.subplot(2, 2, 1)
    plt.plot(result['t'] * 1000, result['Va'], label='A상')
    plt.plot(result['t'] * 1000, result['Vb'], label='B상')
    plt.plot(result['t'] * 1000, result['Vc'], label='C상')
    plt.xlabel('시간 (ms)')
    plt.ylabel('전압 (V)')
    plt.title('3상 입력 전압')
    plt.legend()
    plt.grid(True)

    plt.subplot(2, 2, 2)
    plt.plot(result['t'] * 1000, result['V_out'])
    plt.xlabel('시간 (ms)')
    plt.ylabel('전압 (V)')
    plt.title(f'정류 출력 (리플 주파수: {result["f_ripple"]} Hz)')
    plt.grid(True)

    plt.subplot(2, 2, 3)
    plt.plot(result['t'] * 1000, result['I_out'])
    plt.xlabel('시간 (ms)')
    plt.ylabel('전류 (A)')
    plt.title('출력 전류')
    plt.grid(True)

    plt.tight_layout()
    plt.savefig('/tmp/3phase_rectifier_analysis.png', dpi=150)
    print("\n그래프 저장 완료: /tmp/3phase_rectifier_analysis.png")
except Exception as e:
    print(f"\n그래프 생성 불가 (CLI 환경): {e}")
```

이 코드는 3상 정류기의 전기적 특성을 계산하고, 다이오드 선정을 위한 설계 파라미터를 제공한다. 기술사 시험에서는 정류 효율, 리플 계수, PIV, 다이오드 전류 용량 계산이 핵심 출제 포인트이다.
