# 절연체 (Insulator)

## 정의

**절연체(Insulator)**는 전기가 잘 흐르지 않는 물질로, 자유 전자가 거의 없어 **높은 저항으로 전류의 흐름을 차단**한다. 유전체(Dielectric)라고도 하며, 전기적 절연과 커패시터의 유전체로 사용된다.

```
┌─────────────────────────────────────────────────────────────────┐
│                   절연체의 전자 결합                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────┐   ┌─────┐   ┌─────┐   ┌─────┐   ┌─────┐             │
│   │ (+) │   │ (+) │   │ (+) │   │ (+) │   │ (+) │             │
│   │  ●  │───│  ●  │───│  ●  │───│  ●  │───│  ●  │             │
│   │     │   │     │   │     │   │     │   │     │             │
│   └─────┘   └─────┘   └─────┘   └─────┘   └─────┘             │
│                                                                 │
│   • 모든 전자가 원자핵에 강하게 결합                            │
│   • 자유 전자 거의 없음                                         │
│   • 전기장 인가 시 전자 이동 불가                                │
│   • 비저항: 10⁸ ~ 10¹⁸ Ω·m                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 절연 파괴 (Dielectric Breakdown)

```
┌─────────────────────────────────────────────────────────────────┐

│   │  │(+)|  │(+)|    │       │  │(+)|  │(+)|    │           │
│   │  │ ● │  │ ● │    │       │  │ ●→│  │←● │    │           │
```

### 2. 유전체 특성

```
┌─────────────────────────────────────────────────────────────────┐
│                   유전율과 분극                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   유전율 (Permittivity):                                        │
│                                                                 │
│   ε = ε₀ × εr                                                   │
│                                                                 │
│   ε: 유전율 (F/m)                                               │
│   ε₀: 진공 유전율 = 8.854 × 10⁻¹² F/m                          │
│   εr: 비유전율 (상대 유전율, Relative Permittivity)             │
│                                                                 │
│   분극 (Polarization):                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   외부 전계 없음        외부 전계 인가                    │ │
│   │   ┌─────────────┐      ┌─────────────┐                  │ │
│   │   │ (+)    (-) │      │ (+)    (-) │                  │ │
│   │   │  ●      ○  │      │  ● →  ← ○  │                  │ │
│   │   │  무작위    │      │  정렬(분극) │                  │ │
│   │   └─────────────┘      └─────────────┘                  │ │
│   │                                                         │ │
│   │   분극 → 내부 전계 감소 → 유효 전계 감소                 │ │
│   │   → 커패시터 정전용량 증가                               │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   주요 물질의 비유전율 (εr):                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  진공/공기: 1.0                                         │ │
│   │  테프론: 2.1                                             │ │
│   │  폴리에틸렌: 2.3                                          │ │
│   │  유리: 4-10                                              │ │
│   │  실리콘: 11.7                                            │ │
│   │  SiO₂: 3.9                                               │ │
│   │  물(H₂O): 80                                             │ │
│   │  BaTiO₃: 1000-10000                                      │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
│   액체 절연체   │  • 변압기유, 케이블유                        │

├─────────────────┼───────────────────────────────────────────────┤
│   고체 절연체   │  • 종이, 마이카, 세라믹, 에폭시              │

├─────────────────┼───────────────────────────────────────────────┤
│   무기 절연체   │  • 유리, 세라믹, 마이카                      │
│   (Inorganic)  │  • 고온 저항성, 내열성 우수                   │
│                │  • 내아크성 우수                               │
├─────────────────┼───────────────────────────────────────────────┤
│   유기 절연체   │  • 플라스틱, 고무, 에폭시                     │
│   (Organic)    │  • 가공성 우수, 저렴                          │
│                │  • 열적 한계 존재                              │
└─────────────────┴───────────────────────────────────────────────┘
```

## 상세 설명

### 절연 열화 (Insulation Degradation)

```
┌─────────────────────────────────────────────────────────────────┐
│                  절연 열화 요인                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 열적 열화 (Thermal Aging)                                   │
│      ┌─────────────────────────────────────────────────────┐   │
│      │ Arrhenius 법칙: 수명 ∝ exp(Ea/kT)                    │   │
│      │ • 온도 10°C 상승 → 수명 약 1/2 감소 (Montsinger 법칙) │   │
│      │ • 화학 결합 파괴, 기계적 특성 저하                    │   │
│      └─────────────────────────────────────────────────────┘   │
│                                                                 │
│   ② 전기적 열화 (Electrical Degradation)                        │
│      ┌─────────────────────────────────────────────────────┐   │
│      │ • 부분 방전 (Partial Discharge): 공극 내 미소 방전   │   │
│      │ • 트리잉 (Treeing): 나뭇가지 모양 열화 진행          │   │
│      │ • 전기적 스트레스에 의한 열화 누적                    │   │
│      └─────────────────────────────────────────────────────┘   │
│                                                                 │
│   ③ 기계적 열화 (Mechanical Degradation)                        │
│      ┌─────────────────────────────────────────────────────┐   │
│      │ • 진동, 충격, 반복 응력                               │   │
│      │ • 크리프 (Creep), 피로 (Fatigue)                     │   │
│      │ • 균열 발생 및 진전                                   │   │
│      └─────────────────────────────────────────────────────┘   │
│                                                                 │
│   ④ 환경적 열화 (Environmental Degradation)                     │
│      ┌─────────────────────────────────────────────────────┐   │
│      │ • 습도, 화학 물질, UV, 오존                           │   │
│      │ • 수분 흡수 → 유전 특성 저하                          │   │
│      └─────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 실무 적용

### 반도체 소자 절연

```
┌─────────────────────────────────────────────────────────────────┐
│              반도체에서의 절연막                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   SiO₂ (실리콘 산화막):                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ • 게이트 산화막: FET 채널과 게이트 간 절연              │ │
│   │ • 필드 산화막: 소자 간 격리 (LOCOS/STI)                 │ │
│   │ • 금속간 절연막: 배선 층 간 절연                        │ │
│   │ • Passivation: 수분, 이물질로부터 소자 보호             │ │
│   │                                                         │ │
│   │ 장점:                                                   │ │
│   │ • Si와 열적 안정성 우수 (Si-SiO₂ 계면)                  │ │
│   │ • 높은 절연 파괴 전계 (~10 MV/cm)                       │ │
│   │ • 안정한 계면 특성                                      │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   High-k 유전체 (45nm 이하):                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ • HfO₂ (하프늄 산화물): k ≈ 25                          │ │
│   │ • Al₂O₃ (알루미늄 산화물): k ≈ 9                        │ │
│   │ • ZrO₂ (지르코늄 산화물): k ≈ 25                        │ │
│   │                                                         │ │
│   │ • SiO₂ 대비 두께 증가로 누설 전류 감소                  │ │
│   │ • 게이트 누설 전류 억제                                  │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   Low-k 유전체 (배선 간):                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ • SiOₓCᵧ (Carbon-doped oxide): k ≈ 2.5-3.0             │ │
│   │ • Organic polymer: k ≈ 2.0-2.7                          │ │
│   │ • Air gap: k ≈ 1.0                                       │ │
│   │                                                         │ │
│   │ • 배선 간 커패시턴스 감소 → 신호 지연 감소              │ │
│   │ • RC 지연 감소로 동작 속도 향상                          │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **절연체**: 자유 전자가 거의 없어 높은 저항으로 전류 차단
> **절연 파괴**: 높은 전계에서 절연체가 도전성을 가지는 현상

### 2. 빈출 키워드

- **비유전율 εr**: 물질의 유전율/진공 유전율
- **절연 파괴 전계**: kV/mm 단위, 물질마다 다름
- **분극**: 외부 전계에 의한 전하 정렬
- **High-k/Low-k**: 게이트/배선 유전체
- **열화 요인**: 열, 전기, 기계, 환경

### 3. 예상 문제

**Q1. 도체, 반도체, 절연체의 차이점을 설명하시오.**

**Q2. 반도체 소자에서 High-k 유전체를 사용하는 이유를 기술하시오.**

**Q3. 절연 열화의 주요 원인 4가지를 설명하시오.**
