# 반도체 (Semiconductor)

## 정의

**반도체(Semiconductor)**는 도체와 부도체의 중간 특성을 가진 물질로, 온도, 불순물, 전기장 등의 외부 조건에 따라 **전도성을 제어**할 수 있는 물질이다. 대표적인 반도체로는 실리콘(Si), 게르마늄(Ge), 갈륨비소(GaAs) 등이 있다.

```
┌─────────────────────────────────────────────────────────────────┐
│                도체 vs 반도체 vs 부도체                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   비저항 (Ω·m)                                                  │
│                                                                 │
│   10⁻⁸ ─┬─ 도체 (Conductor)                                    │
│         │  • Cu, Al, Ag, Au                                    │
│         │  • 자유 전자 풍부                                     │
│         │                                                       │
│   10⁻³ ─┼─ 반도체 (Semiconductor)                              │
│         │  • Si, Ge, GaAs                                      │
│         │  • 전도성 제어 가능                                   │
│         │  • 불순물 도핑으로 특성 변화                          │
│         │                                                       │
│   10⁸  ─┼─ 부도체 (Insulator)                                  │
│         │  • 유리, 고무, 세라믹                                │
│         │  • 전기 절연                                          │
│         │                                                       │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 고유 반도체 (Intrinsic Semiconductor)

```
┌─────────────────────────────────────────────────────────────────┐
│              고유 반도체 (순수 Si 결정)                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   실리콘 결정 구조 (다이아몬드 구조):                           │
│                                                                 │
│         Si ───── Si                                             │
│         │╲      ╱│                                              │
│         │ ╲    ╱ │                                              │
│         │  Si  Si │                                             │
│         │ ╱    ╲ │                                              │
│         │╱      ╲│                                              │
│         Si ───── Si                                             │
│                                                                 │
│   • 각 Si 원자는 4개의 전자를 인접 원자와 공유                  │
│   • 공유 결합 (Covalent Bond) 형성                              │
│   • T = 0K에서: 모든 전자가 결합, 전도 불가능                   │
│   • 온도 상승 시: 일부 결합 파괴 → 자유 전자 생성               │
│                                                                 │
│   전자-정공 쌍 (Electron-Hole Pair):                            │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  온도 ↑ → 전자 결합 파괴 → 자유 전자 + 정공(Hole) 발생    │ │
│   │                                                         │ │
│   │  ┌───┐  ┌───┐  ┌───┐  ┌───┐                            │ │
│   │  │ ● │  │   │  │ ● │  │ ● │   ●: 전자                   │ │
│   │  └───┘  └─┬─┘  └───┘  └───┘   ○: 정공                   │ │
│   │          │                                              │ │
│   │       정공(Hole)                                        │ │
│   │       (전자가 빠져나간 자리)                             │ │
│   │                                                         │ │
│   │  전도 메커니즘:                                          │ │
│   │  • 전자 이동: ──▶ 전자의 직접 이동                       │ │
│   │  • 정공 이동: ◀── 인접 전자이 정공으로 이동               │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 불순물 반도체 (Extrinsic Semiconductor)

```
┌─────────────────────────────────────────────────────────────────┐
│                   N형 반도체 (N-Type)                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   5족 원소 도핑 (P, As, Sb): 전자 과잉                          │
│                                                                 │
│        Si       Si       Si                                     │
│         │╲      ╱│╲      ╱│                                     │
│         │ ╲    ╱ │ ╲    ╱ │                                     │
│         │  Si──P──Si  │  ← 5족 원소 (전자 5개)                  │
│         │ ╱    ╲ │ ╱    ╲ │                                     │
│         │╱      ╲│╱      ╲│                                     │
│        Si       Si       Si                                     │
│                                                                 │
│   • 4개의 전자는 결합, 1개의 전자 여분                          │
│   • 자유 전자 = 다수 캐리어 (Majority Carrier)                  │
│   • 정공 = 소수 캐리어 (Minority Carrier)                       │
│   • 도너(Donor) 불순물                                          │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                   P형 반도체 (P-Type)                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   3족 원소 도핑 (B, Al, Ga): 정공 발생                          │
│                                                                 │
│        Si       Si       Si                                     │
│         │╲      ╱│╲      ╱│                                     │
│         │ ╲    ╱ │ ╲    ╱ │                                     │
│         │  Si──B──Si  │  ← 3족 원소 (전자 3개)                  │
│         │ ╱  ○ ╲ │ ╱    ╲ │   ○: 정공                          │
│         │╱      ╲│╱      ╲│                                     │
│        Si       Si       Si                                     │
│                                                                 │
│   • 3개의 전자만 결합, 1개의 정공 발생                          │
│   • 정공 = 다수 캐리어 (Majority Carrier)                       │
│   • 전자 = 소수 캐리어 (Minority Carrier)                       │
│   • 억셉터(Acceptor) 불순물                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. PN 접합 (PN Junction)

```
┌─────────────────────────────────────────────────────────────────┐
│                    PN 접합 다이오드                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   공핍 영역 (Depletion Region) 형성:                            │
│                                                                 │
│   P형            │           N형                               │
│   ┌──────────────┼──────────────┐                              │
│   │ ○ ○ ○ ○ ○ ││ ● ● ● ● ● │   ○: 정공                       │
│   │ ○ ○ ○ ○ ○ ││ ● ● ● ● ● │   ●: 전자                       │
│   │ ○ ○ ○ ○ ○ ││ ● ● ● ● ● │                                  │
│   │ ○ ○ ○ ○ ○ ││ ● ● ● ● ● │                                  │
│   └──────────────┼──────────────┘                              │
│                  │                                              │
│          ───▶ 공핍 영역 ◀───                                    │
│            (전하가 없는 영역)                                    │
│                                                                 │
│   내부 전계 (Built-in Potential):                               │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  (-)  ←───────────────────────────────────▶  (+)        │ │
│   │   P측 공핍영역    │    N측 공핍영역                      │ │
│   │   (음이온)       │    (양이온)                          │ │
│   │                                                         │ │
│   │  확산 전류와 드리프트 전류가 평형을 이룸                  │ │
│   │  Si: V_bi ≈ 0.7V, Ge: V_bi ≈ 0.3V                       │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                   순방향 바이어스                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│       (+)                (-)                                    │
│        │                  │                                     │
│        ▼                  ▼                                     │
│   ┌─────────┐         ┌─────────┐                              │
│   │ P형     │ ──────▶ │ N형     │                              │
│   │         │  전류   │         │                              │
│   └─────────┘         └─────────┘                              │
│                                                                 │
│   • 외부 전계가 내부 전계를 상쇄                                 │
│   • 공핍 영역 축소                                              │
│   • 전류 흐름 (Turn-On)                                         │
│   • V > V_bi 에서 전도                                          │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                   역방향 바이어스                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│       (-)                (+)                                    │
│        │                  │                                     │
│        ▼                  ▼                                     │
│   ┌─────────┐         ┌─────────┐                              │
│   │ P형     │ ◀────── │ N형     │                              │
│   │         │ (전류 X)│         │                              │
│   └─────────┘         └─────────┘                              │
│                                                                 │
│   • 외부 전계가 내부 전계를 강화                                 │
│   • 공핍 영역 확장                                              │
│   • 전류 차단 (Turn-Off)                                        │
│   • 항복 전압 이상에서만 전류 (Avalanche)                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **반도체**: 도체와 부도체의 중간 전도성, 외부 조건으로 제어 가능
> **PN 접합**: P형과 N형 반도체의 접합, 다이오드의 기본 구조

### 2. 빈출 키워드

- **고유/불순물 반도체**: 순수 vs 도핑된 반도체
- **N형/P형**: 전자 vs 정공이 다수 캐리어
- **공핍 영역**: PN 접합에서 전하가 없는 영역
- **내부 전위**: Si ≈ 0.7V, Ge ≈ 0.3V
- **FinFET/GAA**: 3D 트랜지스터 구조

### 3. 예상 문제

**Q1. N형과 P형 반도체의 차이점을 설명하시오.**

**Q2. PN 접합 다이오드의 정류 원리를 기술하시오.**

**Q3. CMOS의 저전력 특성이 나타나는 이유를 설명하시오.**
