# 도체 (Conductor)

## 정의

**도체(Conductor)**는 전기가 잘 흐르는 물질로, 자유 전자가 풍부하여 **낮은 저항으로 전류를 전달**하는 물질이다. 대표적인 도체로는 구리, 알루미늄, 금, 은 등이 있다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    도체의 전자 이동                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   자유 전자 (Free Electrons)                                   │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 원자핵에 약하게 결합된 전자                            │ │
│   │  • 외부 전기장에 의해 쉽게 이동                           │ │
│   │  • 전류의 운반체 (Charge Carrier)                        │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ┌─────┐   ┌─────┐   ┌─────┐   ┌─────┐   ┌─────┐           │
│   │  ●  │ → │  ●  │ → │  ●  │ → │  ●  │ → │  ●  │           │
│   │     │   │     │   │     │   │     │   │     │           │
│   └─────┘   └─────┘   └─────┘   └─────┘   └─────┘           │
│    원자핵    자유전자 이동 방향 (전기장 방향)                   │
│                                                                 │
│   전기장 E ────────────────────────────────────▶               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 전도 메커니즘

```
┌─────────────────────────────────────────────────────────────────┐
│                   금속 도체의 전도 원리                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   금속 결정 구조:                                               │
│                                                                 │
│         (+)      (+)      (+)      (+)                         │
│          │        │        │        │                          │
│          ●  ─ ─  ●  ─ ─  ●  ─ ─  ●    ← 원자핵 (이온)          │
│          │        │        │        │                          │
│         (+)      (+)      (+)      (+)                         │
│                                                                 │
│   ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○ ○   ← 자유 전자 (전자 바다)   │
│                                                                 │
│   특징:                                                         │
│   • 원자가 전자이 약하게 결합                                    │
│   • 전자가 결정 전체를 자유롭게 이동                             │
│   • 전기장 인가 시 전자의 집단적 이동 = 전류                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 비저항과 전도도

```
┌─────────────────────────────────────────────────────────────────┐
│                   비저항 (Resistivity)                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   비저항 ρ (Ohm·m)                  전도도 σ (S/m)              │
│   ┌─────────────────────┐        ┌─────────────────────┐       │
│   │ R = ρ × (L/A)       │        │ σ = 1/ρ             │       │
│   │                     │        │                     │       │
│   │ ρ: 비저항           │        │ σ: 전도도           │       │
│   │ L: 길이             │        │ ρ: 비저항           │       │
│   │ A: 단면적           │        │                     │       │
│   └─────────────────────┘        └─────────────────────┘       │
│                                                                 │
│   주요 도체의 비저항 (20°C, ×10⁻⁸ Ω·m):                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  은 (Ag):    1.59  ← 최저 (최고 전도성)                │ │
│   │  구리 (Cu):  1.68  ← 전선 표준                         │ │
│   │  금 (Au):    2.44  ← 내식성 우수                       │ │
│   │  알루미늄(Al): 2.65 ← 가벼움, 송전선                   │ │
│   │  철 (Fe):    9.71  ← 비교적 높음                       │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 온도 의존성

```
┌─────────────────────────────────────────────────────────────────┐
│              도체 저항의 온도 의존성                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   R(T) = R₀ × [1 + α(T - T₀)]                                   │
│                                                                 │
│   R(T): 온도 T에서의 저항                                       │
│   R₀: 기준 온도 T₀에서의 저항                                   │
│   α: 온도 계수 (1/°C)                                           │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │    R │                                              ●   │ │
│   │      │                                           ●      │ │
│   │      │                                        ●         │ │
│   │      │                                     ●            │ │
│   │      │                                  ●               │ │
│   │      │                               ●                  │ │
│   │      └──────────────────────────────────────▶ T         │ │
│   │        T₀                                            │ │
│   │                                                         │ │
│   │   도체: 온도 ↑ → 저항 ↑ (양의 온도 계수)               │ │
│   │   반도체: 온도 ↑ → 저항 ↓ (음의 온도 계수)              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 상세 설명

### 도체 vs 부도체 vs 반도체

```
┌─────────────────────────────────────────────────────────────────┐
│              전도성에 따른 물질 분류                             │
├─────────────────┬─────────────────┬─────────────────────────────┤
│     구분        │     비저항      │         특성                │
├─────────────────┼─────────────────┼─────────────────────────────┤
│    도체         │   10⁻⁸ ~ 10⁻⁶  │  • 자유 전자 풍부           │
│  (Conductor)    │     Ω·m        │  • 금속 (Cu, Al, Ag, Au)   │
│                 │                 │  • 온도 ↑ → 저항 ↑          │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   반도체        │   10⁻³ ~ 10³   │  • 전도성 제어 가능          │
│  (Semiconductor)│     Ω·m        │  • Si, Ge, GaAs            │
│                 │                 │  • 온도 ↑ → 저항 ↓          │
│                 │                 │  • 불순물 도핑으로 특성 변화 │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   부도체        │   10⁸ ~ 10¹⁸   │  • 자유 전자 거의 없음       │
│  (Insulator)    │     Ω·m        │  • 유리, 고무, 세라믹       │
│                 │                 │  • 전기적 차폐              │
└─────────────────┴─────────────────┴─────────────────────────────┘
```

## 실무 적용

### PCB 트레이스 설계

```
┌─────────────────────────────────────────────────────────────────┐
│                PCB 트레이스 저항 계산                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   트레이스 저항: R = ρ × L / (W × T)                           │
│                                                                 │
│   ρ: 구리 비저항 = 1.68 × 10⁻⁸ Ω·m                            │
│   L: 트레이스 길이                                               │
│   W: 트레이스 폭                                                 │
│   T: 구리 두께 (1 oz ≈ 35 μm)                                   │
│                                                                 │
│   예시: 1oz PCB, 10mil 폭, 10inch 길이                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  R = 1.68×10⁻⁸ × 0.254m / (0.254mm × 35μm)             │ │
│   │    ≈ 0.05 Ω (약 50 mΩ)                                  │ │
│   │                                                         │ │
│   │  전류 1A 시 전압 강하: V = IR = 0.05V                   │ │
│   │  전력 손실: P = I²R = 0.05W                             │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   설계 고려사항:                                                 │
│   • 전원 트레이스: 충분한 폭 확보 (전압 강하 최소화)              │
│   • 고전류 경로: 병렬 트레이스 또는 전원 평면 사용               │
│   • 열 발생: I²R 손실로 인한 온도 상승 고려                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **도체**: 자유 전자가 풍부하여 낮은 저항으로 전류를 전달하는 물질
> **비저항 ρ**: 물질 고유의 저항 특성, 단위 Ω·m

### 2. 빈출 키워드

- **자유 전자**: 원자핵에 약하게 결합된 전자
- **온도 계수 α**: 도체는 양의 값, 반도체는 음의 값
- **전도 띠**: 전자가 자유롭게 이동할 수 있는 에너지 준위
- **스킨 효과**: 고주파에서 전류가 표면으로 집중

### 3. 예상 문제

**Q1. 도체, 반도체, 부도체의 차이점을 에너지 밴드 관점에서 설명하시오.**

**Q2. 구리가 전선의 표준 도체로 사용되는 이유를 기술하시오.**

**Q3. PCB 설계에서 트레이스 저항을 고려해야 하는 상황을 설명하시오.**
