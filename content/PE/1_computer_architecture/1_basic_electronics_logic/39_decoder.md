# 디코더 (Decoder)

## 정의

**디코더(Decoder)**는 **N개의 입력**을 받아 **최대 2ⁿ개의 출력** 중 하나를 활성화하는 조합 논리회로다. 인코더의 반대 동작을 수행하며, 주소 디코딩, 데이터 선택, 코드 변환에 사용된다.

```
┌─────────────────────────────────────────────────────────────────┐
│           디코더 기본 구조                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   N입력 → 2ⁿ출력 디코더 (N-to-2ⁿ Decoder)                       │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   입력: N비트                                            │ │
│   │   출력: 2ⁿ비트 (항상 1개만 활성)                         │ │
│   │                                                         │ │
│   │           ┌───────────┐                                │ │
│   │   I₁ I₀ ──│   2×4     │── Y₀                           │ │
│   │     │     │  Decoder  │── Y₁                           │ │
│   │     └─────│           │── Y₂                           │ │
│   │           │           │── Y₃                           │ │
│   │           └───────────┘                                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   💡 입력 조합에 해당하는 출력만 활성화 (보통 Active-Low)        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 2×4 디코더

```
┌─────────────────────────────────────────────────────────────────┐
│           2×4 디코더 진리표 및 회로                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   진리표 (Active-Low 출력):                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ I₁ │ I₀ │ Y₃ │ Y₂ │ Y₁ │ Y₀ │ 활성 출력               │ │
│   ├───┼───┼───┼───┼───┼───┼────────────────┤ │
│   │ 0 │ 0 │ 1 │ 1 │ 1 │ 0 │ Y₀ 활성 (0)      │ │
│   │ 0 │ 1 │ 1 │ 1 │ 0 │ 1 │ Y₁ 활성 (1)      │ │
│   │ 1 │ 0 │ 1 │ 0 │ 1 │ 1 │ Y₂ 활성 (2)      │ │
│   │ 1 │ 1 │ 0 │ 1 │ 1 │ 1 │ Y₃ 활성 (3)      │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   논리식 (Active-Low):                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   Y₀ = (I₁'I₀')' = I₁ + I₀                             │ │
│   │   Y₁ = (I₁'I₀ )' = I₁ + I₀'                            │ │
│   │   Y₂ = (I₁ I₀')' = I₁' + I₀                            │ │
│   │   Y₃ = (I₁ I₀ )' = I₁' + I₀'                           │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   회로도:                                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   I₁ ──┬──┤ NOT ├──┬─────────────────────────────────┐│ │
│   │        │           │                                 ││ │
│   │        │           ├──┤ NAND ├── Y₀                  ││ │
│   │        │           │   (I₁'I₀')                      ││ │
│   │   I₀ ──┼──┤ NOT ├──┘                                 ││ │
│   │        │                                             ││ │
│   │        └──────────────────┤ NAND ├── Y₁ (I₁'I₀)     ││ │
│   │                               │                      ││ │
│   │   (추가 NAND 게이트로 Y₂, Y₃ 생성)                    ││ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 3×8 디코더와 인에이블

```
┌─────────────────────────────────────────────────────────────────┐
│           3×8 디코더 (인에이블 포함)                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │           ┌─────────────────┐                          │ │
│   │   I₂ ─────│                 │── Y₀                     │ │
│   │   I₁ ─────│    3×8 Decoder  │── Y₁                     │ │
│   │   I₀ ─────│                 │── Y₂                     │ │
│   │           │   with Enable   │── Y₃                     │ │
│   │   E  ─────│                 │── Y₄                     │ │
│   │           │                 │── Y₅                     │ │
│   │           │                 │── Y₆                     │ │
│   │           │                 │── Y₇                     │ │
│   │           └─────────────────┘                          │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   인에이블(Enable) 동작:                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   E = 1: 정상 동작 (하나의 출력 활성)                    │ │
│   │   E = 0: 모든 출력 비활성                                │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   논리식 (E 포함):                                              │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   Yᵢ = E · mᵢ  (mᵢ는 해당 최소항)                      │ │
│   │   예: Y₀ = E · I₂'I₁'I₀'                               │ │
│   │       Y₇ = E · I₂I₁I₀                                  │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 디코더 확장 (계층 구조)

```
┌─────────────────────────────────────────────────────────────────┐
│           디코더 확장: 2×4 + 2×4 = 3×8                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   I₂ ─────┬─────────────────────────────────┐          │ │
│   │           │                                 │          │ │
│   │           ▼                                 ▼          │ │
│   │         ┌───┐                             ┌───┐       │ │
│   │         │NOT│                             │NOT│       │ │
│   │         └─┬─┘                             └─┬─┘       │ │
│   │           │                                 │          │ │
│   │           ▼ (E)                             ▼ (E)      │ │
│   │   ┌───────────────┐             ┌───────────────┐    │ │
│   │   │  2×4 Decoder  │             │  2×4 Decoder  │    │ │
│   │   │ (I₂'=E)       │             │ (I₂ =E)       │    │ │
│   │   └───────┬───────┘             └───────┬───────┘    │ │
│   │           │                             │             │ │
│   │    Y₀~Y₃ (I₂=0)                  Y₄~Y₇ (I₂=1)       │ │
│   │                                                         │ │
│   │   I₁I₀ = 00 01 10 11           I₁I₀ = 00 01 10 11    │ │
│   │   I₂=0:  Y₀ Y₁ Y₂ Y₃           I₂=1:  Y₄ Y₅ Y₆ Y₇    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   💡 상위 비트로 Enable을 제어하여 디코더 확장                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. 디코더 응용

```
┌─────────────────────────────────────────────────────────────────┐
│           디코더 응용 분야                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 메모리 주소 디코딩:                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   주소 버스 ──▶ [디코더] ──▶ 메모리 칩 선택             │ │
│   │                                                         │ │
│   │   예: 2비트 주소 → 4개 메모리 뱅크 선택                  │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② 조합 논리 함수 구현:                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   F(A,B,C) = Σm(1,3,5,7)                               │ │
│   │                                                         │ │
│   │   3×8 디코더 출력 Y₁, Y₃, Y₅, Y₇을 OR 결합             │ │
│   │   F = Y₁ + Y₃ + Y₅ + Y₇                                │ │
│   │                                                         │ │
│   │   💡 모든 SOP 함수를 디코더 + OR로 구현 가능             │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ③ 7세그먼트 디코더:                                           │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   BCD 입력 (4비트) ──▶ [7세그먼트 디코더] ──▶ a~g 출력  │ │
│   │                                                         │ │
│   │   ┌───┐                                                 │ │
│   │   │ a │                                                 │ │
│   │   ├───┤  BCD 0→abcdef (0 표시)                         │ │
│   │   │f│b│  BCD 1→bc (1 표시)                             │ │
│   │   ├───┤  ...                                            │ │
│   │   │ g │                                                 │ │
│   │   ├───┤                                                 │ │
│   │   │e│c│                                                 │ │
│   │   ├───┤                                                 │ │
│   │   │ d │                                                 │ │
│   │   └───┘                                                 │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ④ 디멀티플렉서 (DEMUX)로 사용:                                │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   데이터 ──▶ Enable 입력                                │ │
│   │   선택 ──▶ 디코더 입력                                  │ │
│   │   출력 ──▶ 선택된 출력으로 데이터 전송                  │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **디코더**: N입력 → 2ⁿ출력 (하나만 활성)
> **Active-Low**: 선택된 출력이 0 (나머지는 1)
> **Enable**: 디코더 동작 제어 신호

### 2. 빈출 키워드

- **N-to-2ⁿ**: 입력/출력 관계
- **최소항**: 각 출력이 하나의 최소항에 대응
- **메모리 디코딩**: 주소로 메모리 칩 선택
- **7세그먼트**: BCD→세그먼트 변환

### 3. 예상 문제

**Q1. 2×4 디코더의 진리표를 작성하고 논리식을 유도하시오.**

**Q2. 2×4 디코더 2개를 사용하여 3×8 디코더를 구성하시오.**

**Q3. 디코더를 사용하여 F(A,B,C) = Σm(0,2,4,6)을 구현하시오.**
