# 인덕터 (Inductor)

## 정의

**인덕터(Inductor)**는 도선을 코일 형태로 감아서 **자기장을 유도**하고 **전류의 변화를 방해**하는 수동 전자부품이다. 코일, 초크(Choke)라고도 한다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    인덕터의 기본 구조                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│         ┌─────────────────┐                                    │
│         │     Core      │ ←───── 감아 ───────→     Core       │
│         │    (자기장)    │    (자기장 집중)     │    (자기장)    │
│         └─────────────────┘                                    │
│              │                                    │             │
│              │     전류 흐름                                │
│              │     I ───▶ L ───▶ I                             │
│              │                                    │             │
│              ▼                                    ▼             │
│         (전류 증가 시)                          (전류 감소 시)  │
│         자기장 에너지 저장                       자기장 에너지 방출 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 인덕턴스 (Inductance)

```
┌─────────────────────────────────────────────────────────────────┐
│                    인덕턴스 공식                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│              V = L × (di/dt)                                     │
│                                                                 │
│              V: 유도 기전력 (Volt)                                │
│              L: 인덕턴스 (Henry, H)                               │
│              di/dt: 전류 변화율 (A/s)                             │
│              N: 코일 감은 수 (Turns)                              │
│                                                                 │
│              특성:                                               │
│              • 전류 변화 시 역기력 발생                          │
│              • 교류 회로에서 위상 차단 역할                       │
│              • DC에서 저항으로 작용                              │
│              • 주파수가 높을수록 임피던스 증가                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 인덕터의 주파수 응답

```
┌─────────────────────────────────────────────────────────────────┐
│                인덕터의 주파수 응답 특성                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   X_L = ωL                        X_L: 유도 리액턴스(임피던스)    │
│     ↑                              ω: 각주파수 (rad/s)             │
│     │                              L: 인덕턴스 (Henry)             │
│  저주파수                        고주파수                         │
│  (ω 작)                         (ω 큼)                           │
│     │                              │                              │
│     ▼                              ▼                              │
│   X_L 감소                      X_L 증가                          │
│   (단락에 가까움)               (개방에 가까움)                    │
│                                                                 │
│   DC(ω=0)에서: X_L = 0 → 단락                                   │
│   고주파에서: X_L → ∞ → 개방                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 에너지 저장

```
┌─────────────────────────────────────────────────────────────────┐
│                    인덕터 에너지 저장                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                    1                                           │
│           E = ─── × L × I²                                      │
│                    2                                            │
│                                                                 │
│           E: 저장된 에너지 (Joule)                               │
│           L: 인덕턴스 (Henry)                                    │
│           I: 전류 (Ampere)                                       │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  커패시터 vs 인덕터 에너지 저장                          │  │
│   │                                                         │  │
│   │  커패시터: E = ½CV²  (전기장 에너지)                    │  │
│   │  인덕터:  E = ½LI²  (자기장 에너지)                    │  │
│   │                                                         │  │
│   │  • C: 전압 기반 저장    L: 전류 기반 저장              │  │
│   │  • 전기장 형태          자기장 형태                    │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 상세 설명

### 인덕터의 종류

```
┌─────────────────────────────────────────────────────────────────┐
│                    인덕터 종류와 용도                          │
├─────────────────┬─────────────────┬─────────────────────────────┤
│      종류       │     구조        │           용도              │
├─────────────────┼─────────────────┼─────────────────────────────┤
│  공심 인덕터     │   공기 코어    │   고주파수 필터, RF 회로    │
│ (Air Core)      │                 │   저손실, 직선성 좋음        │
├─────────────────┼─────────────────┼─────────────────────────────┤
│  철심 인덕터     │   페라이트 코어 │   전원 필터, 저주파수       │
│ (Iron Core)     │   또는 철심     │   높은 인덕턴스值 가능       │
├─────────────────┼─────────────────┼─────────────────────────────┤
│  토로이드 인덕터 │   도넛 형태    │   전원 공급, 노이즈 필터   │
│ (Toroidal)      │   자기장 폐쇄   │   EMI 억제 효과 좋음        │
├─────────────────┼─────────────────┼─────────────────────────────┤
│  적층 인덕터     │   PCB 내장     │   DC-DC 컨버터, 모바일      │
│ (Multilayer)    │   소형 SMD      │   고밀도 설계에 적합         │
└─────────────────┴─────────────────┴─────────────────────────────┘
```

```
│                                                                 │

│           L                                                    │
```

```
│   Vout = Vin × D  (D: 듀티 사이클)                              │

│                      │            │                             │
│       L              │            │                             │
│       ┌──███████──┐  │            │                             │
│   ───┤            ├──┘            │                             │
│       └────────────┘              │                             │
│                                   │                             │

│                   │                                            │
│                   └── GND                                       │

│                                                                 │

```

```
┌─────────────────────────────────────────────────────────────────┐

```

## 기술사 시험 포인트

### 1. 핵심 정의 (반드시 암기)

> **인덕턴스(L)**: 전류 변화율에 대한 유도 기전력의 비율, 단위는 Henry(H)
> **유도 리액턴스(X_L)**: X_L = ωL, 주파수에 비례하여 증가

### 2. 빈출 키워드

- **V = L(di/dt)**: 인덕터 양단 전압 공식
- **X_L = 2πfL**: 주파수에 따른 임피던스
- **E = ½LI²**: 자기장 에너지 저장
- **LC 공진**: f = 1/(2π√LC)
- **LPF/HPF**: 저역/고역 통과 필터

### 3. 예상 문제

**Q1. 인덕터와 커패시터의 차이점을 설명하시오.**

**Q2. LC 필터의 원리와 종류에 대해 기술하시오.**

**Q3. DC-DC 컨버터에서 인덕터의 역할에 대해 설명하시오.**

### 4. 실무 연계

- VRM 다상 설계에서 인덕터 배치
- EMI/EMC 대책으로 토로이드 인덕터 사용
- 고주파수 회로에서 공심 인덕터 선택 기준
