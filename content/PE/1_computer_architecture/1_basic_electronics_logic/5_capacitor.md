# 커패시터 (Capacitor, 축전기)

## 정의

**커패시터(Capacitor)**는 두 개의 도체판 사이에 유전체(절연체)를 배치하여 **전기 에너지를 전기장 형태로 저장**하는 수동 전자부품이다. 축전기라고도 하며, 직류를 차단하고 교류를 통과시키는 특성을 가진다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    커패시터의 기본 구조                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│         도체판 A (+)          유전체           도체판 B (-)    │
│         ┌─────────┐      ┌───────────┐      ┌─────────┐       │
│         │  ████   │      │           │      │   ████  │       │
│         │  ████   │ ←─── │  절연체   │ ───▶ │  ████   │       │
│         │  ████   │      │  (유전체) │      │  ████   │       │
│         └────┬────┘      └───────────┘      └────┬────┘       │
│              │                                    │            │
│              ▼                                    ▼            │
│            (+ 전하)                           (- 전하)         │
│                                                                 │
│         전기장(E-field)이 유전체 내부에 형성됨                  │
│         에너지가 전기장 형태로 저장됨                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 정전용량 (Capacitance)

```
┌─────────────────────────────────────────────────────────────────┐
│                     정전용량 공식                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                     Q                                          │
│              C = ───  (단위: Farad, F)                         │
│                     V                                          │
│                                                                 │
│  C: 정전용량 (Farad)                                           │
│  Q: 저장된 전하량 (Coulomb)                                    │
│  V: 전압 (Volt)                                                │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  평행판 커패시터의 정전용량                              │   │
│  │                                                          │   │
│  │            ε₀ × εr × A                                   │   │
│  │     C = ─────────────                                    │   │
│  │                 d                                        │   │
│  │                                                          │   │
│  │  ε₀: 진공 유전율 (8.854 × 10⁻¹² F/m)                    │   │
│  │  εr: 유전체의 비유전율 (상대 유전율)                     │   │
│  │  A : 도체판 면적 (m²)                                    │   │
│  │  d : 도체판 간 거리 (m)                                  │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**정전용량 결정 요소**
- 면적(A) 증가 → 정전용량 증가
- 거리(d) 감소 → 정전용량 증가
- 유전율(εr) 증가 → 정전용량 증가

### 2. 커패시터의 특성

```
┌─────────────────────────────────────────────────────────────────┐
│                  커패시터의 주요 특성                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ① 주파수 의존적 임피던스                                      │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                          │   │
│  │              1              1                            │   │
│  │     Xc = ────── = ───────────────                        │   │
│  │            ωC       2πf × C                              │   │
│  │                                                          │   │
│  │  Xc: 커패시턴스 리액턴스 (Ω)                             │   │
│  │  f : 주파수 (Hz)                                         │   │
│  │  C : 정전용량 (F)                                        │   │
│  │                                                          │   │
│  │  → 주파수 ↑ → 임피던스 ↓ (고주파 통과)                   │   │
│  │  → 주파수 ↓ → 임피던스 ↑ (저주파 차단)                   │   │
│  │  → DC (f=0) → 임피던스 = ∞ (직류 차단)                   │   │
│  │                                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ② 위상차 (전류가 전압보다 90° 앞섬)                           │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                          │   │
│  │      V(t) = V₀ sin(ωt)                                   │   │
│  │      I(t) = I₀ sin(ωt + 90°) = I₀ cos(ωt)               │   │
│  │                                                          │   │
│  │      전류 ─────────────────▶  전압                       │   │
│  │                     (90° 위상 앞섬)                       │   │
│  │                                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ③ 에너지 저장                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                          │   │
│  │              1       1                                   │   │
│  │     E = ─── CV² = ─── QV                                │   │
│  │              2       2                                   │   │
│  │                                                          │   │
│  │  E: 저장된 에너지 (Joule)                                │   │
│  │                                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 커패시터의 종류

```
┌─────────────────────────────────────────────────────────────────┐
│                    커패시터 종류별 특성                         │
├─────────────┬──────────────┬─────────────┬─────────────────────┤
│    종류     │   유전체     │  정전용량   │       용도          │
├─────────────┼──────────────┼─────────────┼─────────────────────┤
│ 세라믹      │  세라믹      │ pF ~ μF    │ 고주파, 바이패스    │
│ 전해(알루미늄)│ 산화알루미늄 │ μF ~ mF    │ 전원 필터, 커플링   │
│ 전해(탄탈륨) │ 산화탄탈륨   │ μF ~ 수백μF│ 소형, 고신뢰성      │
│ 필름        │  폴리에스터  │ nF ~ μF    │ 오디오, 타이밍      │
│ 마일러      │  폴리에스터  │ nF ~ μF    │ 일반 목적           │
│ 가변        │   공기/세라믹│ 조절 가능  │ 튜닝 회로           │
└─────────────┴──────────────┴─────────────┴─────────────────────┘
```

## 상세 설명

### 커패시터 회로 해석

```
┌─────────────────────────────────────────────────────────────────┐
│                  직렬/병렬 연결                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ① 직렬 연결 (Series)                                          │
│     ┌────[C1]────[C2]────[C3]────┐                             │
│     │                              │                             │
│     └──────────────────────────────┘                             │
│                                                                 │
│              1          1     1     1                           │
│     ──────── = ───  +  ─── +  ───  + ...                        │
│             Ceq        C1    C2    C3                           │
│                                                                 │
│     → 직렬 연결 시 정전용량 감소                                │
│     → 내압(전압 내성) 증가                                      │
│                                                                 │
│  ② 병렬 연결 (Parallel)                                        │
│     ┌────[C1]────┐                                             │
│     ├────[C2]────┤                                             │
│     └────[C3]────┘                                             │
│                                                                 │
│     Ceq = C1 + C2 + C3 + ...                                   │
│                                                                 │
│     → 병렬 연결 시 정전용량 증가                                │
│     → 전체 전류 용량 증가                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### RC 회로와 시정수

```
┌─────────────────────────────────────────────────────────────────┐
│                   RC 회로의 충방전                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  [RC 저통과 필터]                                               │
│                                                                 │
│       R                    C                                    │
│    ─/\/\/\─┬──────────┬──||──┐                                │
│    Vin     │          │      │                                │
│             └──────────┴──────┴─ Vout                          │
│                         (GND)                                   │
│                                                                 │
│  시정수 (Time Constant): τ = R × C                             │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 충전 곡선                                                │   │
│  │                                                          │   │
│  │  Vc(t) = V₀(1 - e^(-t/τ))                               │   │
│  │                                                          │   │
│  │  t = τ  → Vc = 0.632V₀ (63.2% 충전)                     │   │
│  │  t = 3τ  → Vc = 0.950V₀ (95% 충전)                      │   │
│  │  t = 5τ  → Vc = 0.993V₀ (99.3% 충전)                    │   │
│  │                                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 방전 곡선                                                │   │
│  │                                                          │   │
│  │  Vc(t) = V₀ × e^(-t/τ)                                  │   │
│  │                                                          │   │
│  │  t = τ  → Vc = 0.368V₀ (36.8% 잔존)                     │   │
│  │  t = 3τ  → Vc = 0.050V₀ (5% 잔존)                       │   │
│  │                                                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ① 전원 디커플링 (Decoupling / Bypass)                         │

│  │  [VCC]──┬───────────────▶ IC VCC Pin                     │   │

│  │         │  (세라믹)                                      │   │

│                                                                 │

│                                                                 │

│  ┌─────────────────────────────────────────────────────────┐   │

│  │  - 리프레시 필요 (누설 전류로 방전)                      │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ④ 스위칭 전원 공급장치 (SMPS)                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  - 입력 필터 커패시터 (EMI 제거)                         │   │
│  │  - 출력 필터 커패시터 (리플 제거)                        │   │
│  │  - resonant 커패시터 (공진 회로)                         │   │
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **커패시터**: 전기 에너지를 전기장 형태로 저장하는 수동 소자, 정전용량 C = εA/d

### 2. 빈출 키워드

- **정전용량**: C = Q/V, 단위 Farad(F)
- **리액턴스**: Xc = 1/(2πfC), 주파수에 반비례
- **위상차**: 전류가 전압보다 90° 앞섬
- **시정수**: τ = RC, 충방전 시간 결정
- **직류 차단**: DC에서 무한대 임피던스

### 3. 예상 문제

**Q1. 커패시터의 임피던스가 주파수에 따라 어떻게 변하는지 설명하시오.**

**Q2. RC 회로의 시정수 τ의 의미와 충전 곡선을 설명하시오.**

**Q3. 디지털 회로에서 디커플링 커패시터의 역할과 배치 원칙을 기술하시오.**
