# 캐리 예측 가산기 (Carry Look-ahead Adder)

## 정의

**캐리 예측 가산기(Carry Look-ahead Adder, CLA)**는 모든 자리올림(Carry)을 **동시에 계산**하여 리플 캐리 가산기의 직렬 전파 지연을 극복한 고속 가산기다. O(N) → O(log N)으로 지연 시간을 단축한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           캐리 예측 가산기 기본 개념                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   리플 캐리 가산기 (RCA):                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   C₀ → C₁ → C₂ → C₃ → ... → Cₙ                        │ │
│   │         ↑ 자리올림이 순차적으로 전파 (O(N))              │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   캐리 예측 가산기 (CLA):                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   모든 Cᵢ를 입력(A, B)만으로 동시에 계산                 │ │
│   │   C₀, C₁, C₂, C₃, ... Cₙ ← 병렬 생성 (O(log N))        │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 생성(G)과 전파(P) 함수

```
┌─────────────────────────────────────────────────────────────────┐
│           생성(G)과 전파(P) 함수 정의                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   각 비트 i에서:                                                │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   생성 함수 (Generate):                                 │ │
│   │   Gᵢ = Aᵢ · Bᵢ                                        │ │
│   │   → A와 B가 모두 1이면 무조건 자리올림 발생              │ │
│   │                                                         │ │
│   │   전파 함수 (Propagate):                                │ │
│   │   Pᵢ = Aᵢ ⊕ Bᵢ  또는  Aᵢ + Bᵢ                         │ │
│   │   → 이전 자리올림이 있으면 다음으로 전파                  │ │
│   │                                                         │ │
│   │   자리올림 생성:                                         │ │
│   │   Cᵢ₊₁ = Gᵢ + Pᵢ · Cᵢ                                 │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   진리표:                                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │ Aᵢ │ Bᵢ │ Gᵢ=Aᵢ·Bᵢ │ Pᵢ=Aᵢ⊕Bᵢ │ 의미               │ │
│   ├───┼───┼─────────┼─────────┼────────────────┤ │
│   │ 0 │ 0 │    0    │    0    │ 생성X, 전파X    │ │
│   │ 0 │ 1 │    0    │    1    │ 생성X, 전파가능 │ │
│   │ 1 │ 0 │    0    │    1    │ 생성X, 전파가능 │ │
│   │ 1 │ 1 │    1    │    0    │ 생성O           │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 캐리 예측 논리식 전개

```
┌─────────────────────────────────────────────────────────────────┐
│           캐리 예측 논리식 유도                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   기본 식: Cᵢ₊₁ = Gᵢ + Pᵢ · Cᵢ                                │
│                                                                 │
│   4비트 CLA 캐리 전개:                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   C₁ = G₀ + P₀C₀                                       │ │
│   │                                                         │ │
│   │   C₂ = G₁ + P₁C₁                                       │ │
│   │      = G₁ + P₁(G₀ + P₀C₀)                              │ │
│   │      = G₁ + P₁G₀ + P₁P₀C₀                              │ │
│   │                                                         │ │
│   │   C₃ = G₂ + P₂C₂                                       │ │
│   │      = G₂ + P₂(G₁ + P₁G₀ + P₁P₀C₀)                     │ │
│   │      = G₂ + P₂G₁ + P₂P₁G₀ + P₂P₁P₀C₀                   │ │
│   │                                                         │ │
│   │   C₄ = G₃ + P₃G₂ + P₃P₂G₁ + P₃P₂P₁G₀ + P₃P₂P₁P₀C₀    │ │
│   │                                                         │ │
│   │   💡 모든 캐리가 A, B, C₀만으로 표현됨                   │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   일반화:                                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Cᵢ₊₁ = Gᵢ + Σ(k=0 to i-1)[Gₖ · Π(j=k+1 to i)Pⱼ]    │ │
│   │          + C₀ · Π(j=0 to i)Pⱼ                          │ │
│   │                                                         │ │
│   │   → 모든 입력에서 바로 계산 가능                         │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

│   │         └─┬─┘ └─┬─┘ └─┬─┘ └─┬─┘                     │ │

│   │                                                         │ │

│                                                                 │

### 4. 지연 시간 분석

```
┌─────────────────────────────────────────────────────────────────┐
│           CLA 지연 시간 분석                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   게이트 지연 가정:                                              │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   • AND, OR: 1단위                                      │ │
│   │   • XOR: 2단위                                          │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   CLA 지연 시간:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   4비트 CLA:                                             │ │
│   │   ① P, G 생성: 2단위 (XOR)                              │ │
│   │   ② C₁~C₄ 생성: 2단위 (AND-OR 2단)                      │ │
│   │   ③ Sum 계산: 2단위 (XOR)                               │ │
│   │   총 지연: 2 + 2 + 2 = 6단위                            │ │
│   │                                                         │ │
│   │   리플 캐리 4비트: 7단위                                 │ │
│   │   → CLA가 더 빠름                                        │ │
│   │                                                         │ │
│   │   N비트 일반화:                                          │ │
│   │   • RCA: O(N) = N × 게이트지연                          │ │
│   │   • CLA: O(log N) = 로그 단계                           │ │
│   │                                                         │ │
│   │   💡 비트 수가 많을수록 CLA 우위 확대                     │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **캐리 예측 가산기**: 자리올림을 병렬로 계산하는 고속 가산기
> **G(생성) = A·B, P(전파) = A⊕B**: 캐리 예측의 핵심 함수
> **지연 시간**: O(log N), 리플 캐리의 O(N)보다 빠름

### 2. 빈출 키워드

- **G(Generate), P(Propagate)**: 캐리 생성 및 전파 함수
- **Cᵢ₊₁ = Gᵢ + Pᵢ·Cᵢ**: 캐리 예측 기본 식
- **병렬 계산**: 모든 캐리를 동시에 생성
- **O(log N)**: 지연 시간 복잡도

### 3. 예상 문제

**Q1. 4비트 캐리 예측 가산기를 설계하고 지연 시간을 분석하시오.**

**Q2. 생성(G)과 전파(P) 함수를 정의하고 캐리 예측 논리식을 유도하시오.**

**Q3. 리플 캐리 가산기와 캐리 예측 가산기의 장단점을 비교하시오.**
