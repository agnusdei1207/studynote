# 저항 (Resistance)

## 정의

**저항(Resistance)**은 전기 회로에서 전류의 흐름을 방해하는 물리적 성질로, 도체 양단의 전압과 그 도체를 흐르는 전류의 비율로 정량화된다. 국제단위계(SI)에서는 옴(Ohm, 기호: Ω)을 사용하며, 1Ω은 1V의 전압을 가했을 때 1A의 전류가 흐르는 도체의 저항값으로 정의된다.

```
수식: R = V/I (옴의 법칙)
- R: 저항 (Ohm, Ω)
- V: 전압 (Volt, V)
- I: 전류 (Ampere, A)
```

컴퓨터 아키텍처 관점에서 저항은 전력 손실(Power Loss)과 열 발생(Heat Dissipation)의 주원인이며, 전원 배선 설계, 신호 무결성(Signal Integrity), 그리고 전력 효율 최적화의 핵심 설계 인자로 작용한다. 또한 반도체 소자의 온-저항(RDS(on)), 인터커넥트 배선 저항, 그리고 메모리 셀 저항 등은 칩 성능과 전력 소모에 직접적인 영향을 미친다.

## 핵심 개념

### 1. 저항의 물리적 본질

```
    [ 저항의 미시적 메커니즘 ]

    금속 도체 내부:
    +------------------------------------------------------+
    |                                                      |
    |   전자 이동 방향 --->                                 |
    |                                                      |
    |   e- ->  O  O  O  O  O  O  O  O  O  O               |
    |          |  |  |  |  |  |  |  |  |  |               |
    |          v  v  v  v  v  v  v  v  v  v               |
    |         (+)(+)(+)(+)(+)(+)(+)(+)(+)(+)  <-- 원자핵    |
    |          ^  ^  ^  ^  ^  ^  ^  ^  ^  ^               |
    |          |  |  |  |  |  |  |  |  |  |               |
    |         충돌 산란 (Scattering)                       |
    |                                                      |
    +------------------------------------------------------+

    * 전자가 원자핵과 충돌하며 운동 에너지 손실
    * 이 손실이 저항으로 나타나며 열로 방출
    * 온도 상승 -> 원자 진동 증가 -> 충돌 빈도 증가 -> 저항 증가
```

- **전자 산란(Electron Scattering)**: 전자가 격자 결함, 불순물, 열 진동(포논)과 충돌하며 운동량을 잃는 현상
- **비저항(Resistivity, ρ)**: 재료 고유의 저항 특성. 단위: Ω·m
- **전도도(Conductivity, σ)**: 비저항의 역수. 전류가 흐르기 쉬운 정도. σ = 1/ρ

### 2. 저항 공식과 기하학적 의존성

```
    [ 도체 저항 공식 ]

    R = ρ × L / A

    - R: 저항 (Ω)
    - ρ: 비저항 (Ω·m) - 재료 고유값
    - L: 길이 (m)
    - A: 단면적 (m²)

    [ 단면적 A와 길이 L의 영향 ]

    길이 L이 길어지면          단면적 A가 넓어지면
    +----------------+         +----------------+
    |  e- -> -> -> ->|         |     /|         |
    |  충돌 기회 증가 |         |    / |  통로 넓|
    |  저항 증가      |         |   /  |  저항 감|
    +----------------+         |  /   |  소     |
                               | /    |         |
                               |/     |         |
                               +----------------+
```

**주요 도체 재료의 비저항 (20°C)**:

| 재료 | 비저항 ρ (Ω·m) | 비전도도 σ (S/m) | 용도 |
|:---:|:---:|:---:|:---|
| **은(Silver)** | 1.59 × 10⁻⁸ | 6.30 × 10⁷ | 고주파 접점, 특수 배선 |
| **구리(Copper)** | 1.68 × 10⁻⁸ | 5.96 × 10⁷ | PCB 배선, 전력 케이블 |
| **금(Gold)** | 2.44 × 10⁻⁸ | 4.10 × 10⁷ | 본딩 와이어, 접점 도금 |
| **알루미늄** | 2.82 × 10⁻⁸ | 3.50 × 10⁷ | IC 배선, 송전선 |
| **텅스텐** | 5.60 × 10⁻⁸ | 1.79 × 10⁷ | 히터, 전극 |

### 3. 옴의 법칙과 선형/비선형 저항

```
    [ 선형 저항 (옴의 법칙 준수) ]

    전압(V)
      ^
      |                    /
      |                  /
      |                /   기울기 = 1/R (컨덕턴스)
      |              /
      |            /
      |          /
      |        /
      +----------------------------> 전류(I)

    특징: V/I = 일정 (R = 상수)

    [ 비선형 저항 소자들 ]

    전압(V)                  전압(V)                  전압(V)
      ^                        ^                        ^
      |        /               |    ___                 |
      |      /                 |   /                    |   /
      |    /  (다이오드)       |  /  (Varistor)         |  /  (NTC 써미스터)
      |  /                     | /                      | /
      +--------> 전류(I)       +--------> 전류(I)       +--------> 전류(I)

    * 비선형 소자는 옴의 법칙을 따르지 않음
    * 동작점에 따라 미분 저항(dV/dI)이 변화
```

- **선형 저항(Linear Resistor)**: 전압-전류 특성이 직선인 저항. 금속 선, 탄소막 저항 등
- **비선형 저항(Non-linear Resistor)**: 전압-전류 특성이 곡선인 소자. 다이오드, 써미스터, 바리스터 등
- **미분 저항(Differential Resistance)**: rd = dV/dI. 비선형 소자의 특정 동작점에서의 저항

### 4. 직렬 및 병렬 저항 회로

```
    [ 직렬 연결 (Series Connection) ]

    +--[R1]--[R2]--[R3]--+

    R_total = R1 + R2 + R3

    특징:
    - 전류가 모든 저항에 동일하게 흐름
    - 전압이 저항 비율로 분배 (전압 분배 법칙)
    - 총 저항이 증가

    [ 병렬 연결 (Parallel Connection) ]

    +---[R1]---+
    |          |
    +---[R2]---+
    |          |
    +---[R3]---+

    1/R_total = 1/R1 + 1/R2 + 1/R3

    특징:
    - 전압이 모든 저항에 동일하게 인가
    - 전류가 저항 비율로 분배 (전류 분배 법칙)
    - 총 저항이 감소 (가장 작은 저항값보다 작아짐)

    [ 두 저항 병렬 간소화 공식 ]

    R_parallel = (R1 × R2) / (R1 + R2)
```

| 연결 방식 | 총 저항 공식 | 전압 분배 | 전류 분배 |
|:---:|:---|:---|:---|
| **직렬** | R_total = ΣR | V_n = V × R_n/R_total | I 동일 |
| **병렬** | 1/R_total = Σ1/R | V 동일 | I_n = I × R_total/R_n |

## 상세 설명

### 1. 온도에 따른 저항 변화

```
    [ 금속 도체의 온도 계수 ]

    R(T) = R₀ × [1 + α × (T - T₀)]

    - R(T): 온도 T에서의 저항
    - R₀: 기준 온도 T₀에서의 저항
    - α: 온도 계수 (1/°C)
    - T: 현재 온도

    [ 온도 계수의 종류 ]

    금속 (양의 온도 계수, PTC)     반도체 (음의 온도 계수, NTC)
    저항                            저항
      ^                              ^
      |                            /
      |                          /
      |                        /
      |                      /
      +------------> 온도    +------------> 온도

    구리: α ≈ +0.00393 /°C
    실리콘: α ≈ -0.075 /°C (순수)

    [ 반도체 저항의 온도 의존성 ]

    R(T) = R₀ × e^(Eg / 2kT)

    - Eg: 밴드갭 에너지
    - k: 볼츠만 상수
    - T: 절대 온도 (K)
```

**주요 재료의 온도 계수**:

| 재료 | 온도 계수 α (1/°C) | 특성 |
|:---:|:---:|:---|
| 구리 | +0.00393 | PTC (온도 상승 시 저항 증가) |
| 알루미늄 | +0.00429 | PTC |
| 백금 | +0.00392 | 온도 센서 (RTD)용 |
| 탄소 | -0.0005 | NTC (온도 상승 시 저항 감소) |
| 실리콘 | -0.075 ~ -0.08 | NTC, 온도 민감 |

### 2. CMOS 회로에서의 저항 성분

```
    [ MOSFET의 온-저항 (RDS(on)) ]

    VDD
      |
    +-+---+
    | PMOS |    Rds(on)_p
    +-----+
      |-----------> Vout
      |
    +-+---+
    | NMOS |    Rds(on)_n
    +-----+
      |
    GND

    [ RDS(on)의 구성 요소 ]

    RDS(on) = R_channel + R_accumulation + R_spreading + R_contact

    +--------------------------------------------------------+
    |                 MOSFET 단면도                           |
    +--------------------------------------------------------+
    |                                                        |
    |    소스(Source)         채널(Channel)        드레인(Drain)|
    |    +--------+                              +--------+   |
    |    | n+     |                              | n+     |   |
    |    +---+----+                              +----+---+   |
    |        |      R_channel                        |        |
    |        +---------------------------------------+        |
    |                                                        |
    |        p-type substrate (바디)                         |
    +--------------------------------------------------------+

    [ 공정 축소에 따른 RDS(on) 변화 ]

    공정 노드(nm)   RDS(on) (mΩ·mm²)
    ────────────────────────────────
    180             ~50
    90              ~30
    45              ~20
    28              ~15
    14              ~12
    7               ~10
```

**CMOS 회로의 주요 저항 성분**:

| 저항 성분 | 위치 | 영향 | 최적화 방안 |
|:---:|:---|:---|:---|
| **RDS(on)** | MOSFET 채널 | 전력 손실, 열 발생 | 와이드 트랜지스터, FinFET |
| **인터커넥트** | 금속 배선 | RC 지연, IR Drop | 구리 배선, 저-k 유전체 |
| **콘택트 저항** | 소스/드레인 접촉 | 직렬 저항 증가 | 실리사이드 형성 |
| **웰 저항** | 기판/웰 | 래치업 위험 | 가드 링, 저항 분리 |

### 3. 인터커넥트 저항과 RC 지연

```
    [ 인터커넥트 RC 모델 ]

    드라이버                    부하
    +------+                    +------+
    |      |---[R_wire]---+-----|      |
    |      |              |     |      |
    +------+             ===    +------+
                         C_wire

    RC 지연: τ = R × C

    [ 공정 축소에 따른 배선 저항 증가 ]

    R = ρ × L / (W × H)

    공정 축소 -> W, H 감소 -> R 급격히 증가

    [ 계층적 배선 구조로 해결 ]

    +----------------------------------------------------------+
    |                    상위 배선층 (Global)                    |
    |  ═════════════════════════════════════════════════════    |
    |    두껍고 넓은 배선 (저항 작음, 긴 거리 연결)              |
    +----------------------------------------------------------+
    |                    중위 배선층 (Semi-global)              |
    |    ───────────────────────────────────────────           |
    |    중간 두께 배선                                         |
    +----------------------------------------------------------+
    |                    하위 배선층 (Local)                    |
    |    ─────────────────────────────────                     |
    |    얇고 좁은 배선 (저항 큼, 짧은 거리만 연결)              |
    +----------------------------------------------------------+
```

**인터커넥트 저항의 영향**:

| 영향 | 설명 | 완화 기술 |
|:---|:---|:---|
| **RC 지연** | 배선 저항×정전용량 | 구리 배선, 저-k 유전체 |
| **IR Drop** | 전류×저항 = 전압 강하 | 전원 평면 확장, 비아 다수화 |
| **전력 손실** | P = I²R | 저저항 재료, 굵은 배선 |
| **신호 왜곡** | 분포 RC 효과 | 리피터 삽입 |

### 4. 전력 전자에서의 저항과 효율

```
    [ 전력 변환 효율과 저항 ]

    P_loss = I² × R

    효율 η = P_out / P_in = P_out / (P_out + P_loss)

    [ 저항에 의한 전력 손실 분석 ]

    +----------------------------------------------------------+
    |                    DC-DC 컨버터 손실 분포                  |
    +----------------------------------------------------------+
    |                                                          |
    |    MOSFET 전도 손실 (RDS(on)):       35~45%              |
    |    MOSFET 스위칭 손실:               20~30%              |
    |    인덕터 DCR 손실:                  15~25%              |
    |    PCB 배선 손실:                    5~10%               |
    |    기타 (커패시터 ESR 등):           5~10%               |
    +----------------------------------------------------------+

    [ 저항 감소를 통한 효율 향상 예시 ]

    기존 RDS(on) = 10mΩ, I = 20A
    P_loss = (20)² × 0.01 = 4W

    개선 RDS(on) = 5mΩ
    P_loss = (20)² × 0.005 = 2W

    -> 2W 절감 (50% 손실 감소)
```

### 5. 접지 저항과 전원 평면 설계

```
    [ PCB 전원 분배 네트워크 (PDN) ]

    VRM ----[PCB 트레이스]----[비아]----[전원 평면]---- IC

    각 구간의 저항:
    - 트레이스 저항: R = ρ × L / (W × t)
    - 비아 저항: R_via = ρ × h / (π × r²) × (1/병렬 개수)
    - 전원 평면 저항: 매우 낮음 (평면 저항)

    [ 목표 임피던스와 저항 ]

    Z_target = ΔV / ΔI

    예: ΔV = 100mV, ΔI = 10A
    Z_target = 10mΩ

    -> 전체 PDN 임피던스를 10mΩ 이하로 설계해야 함

    [ 전원 평면 설계 예시 ]

    +----------------------------------------------------------+
    |                      PCB 적층 구조                         |
    +----------------------------------------------------------+
    |  Layer 1  |  신호 (Signal)                                |
    |  Layer 2  |  접지 평면 (GND Plane) <- 저항 최소화         |
    |  Layer 3  |  전원 평면 (PWR Plane) <- 저항 최소화         |
    |  Layer 4  |  신호 (Signal)                                |
    +----------------------------------------------------------+
```

## 실무 적용

### 1. 전력 케이블 및 배선 용량 산정

```
    [ 전류 용량과 저항 계산 ]

    구리 케이블 단면적 vs 전류 용량:

    단면적(mm²)  | DC 저항(mΩ/m) | 허용 전류(A) @ 30°C
    -------------|---------------|--------------------
    1.5          | 12.1          | 15~20
    2.5          | 7.27          | 20~25
    4.0          | 4.55          | 30~40
    6.0          | 3.03          | 40~50
    10           | 1.82          | 55~70
    16           | 1.14          | 80~100

    [ 전압 강하 계산 ]

    V_drop = I × R_total = I × (ρ × 2L / A)

    예: 10A, 10m 거리, 2.5mm² 케이블
    R = 7.27mΩ/m × 10m × 2(왕복) = 145.4mΩ
    V_drop = 10A × 0.1454Ω = 1.45V

    -> 12V 공급 시 10.55V 도달 (11.5% 강하 - 부적절)

    [ 케이블 선정 기준 ]

    1. 전류 용량: 허용 전류 > 부하 전류 × 안전율(1.25~1.5)
    2. 전압 강하: V_drop < 허용 강하 (일반적으로 3~5%)
    3. 발열: 케이블 온도 상승 < 한계값
```

### 2. 정밀 저항과 측정 회로

```
    [ 션트 저항을 이용한 전류 측정 ]

    부하 회로:
    +--------[R_shunt]--------+
    |            |            |
    |            v            |
    |      전압 측정          |
    |            |            |
    +------------|------------+
                 |
    I = V_measured / R_shunt

    [ 션트 저항 선정 기준 ]

    파라미터          | 권장 범위           | 고려사항
    ------------------|--------------------|-----------------
    저항값            | 0.1mΩ ~ 100mΩ      | 측정 감도 vs 전력 손실
    정밀도            | ±0.1% ~ ±1%        | 측정 정확도 요구
    온도 계수 (TCR)   | < 50ppm/°C         | 온도 드리프트
    전력 정격         | P = I²R 여유 있게   | 자체 발열 고려

    [ 4단자 측정 (Kelvin Connection) ]

    전류 소스 ---+---[R_x]---+--- 전류 경로
                 |           |
                 +---[---]---+    <-- 전압 측정점

    * 전류 경로의 접촉 저항이 전압 측정에 영향 없음
    * mΩ급 저항 정밀 측정에 필수
```

### 3. 서버 전원 설계에서의 저항 관리

```
    [ 서버급 VRM 설계 - 저항 최소화 ]

    +----------------------------------------------------------+
    |                    다상 VRM 구조                          |
    +----------------------------------------------------------+
    |                                                          |
    |    12V 입력                                               |
    |      |                                                    |
    |      +----+----+----+----+                                |
    |      |    |    |    |    |                                |
    |    [M1] [M2] [M3] [M4] [M5] [M6]  <- 6상 병렬            |
    |      |    |    |    |    |                                |
    |      +----+----+----+----+----+                           |
    |           |                                               |
    |      V_out (0.8~1.4V)                                    |
    |           |                                               |
    |    +------+------+                                        |
    |    | CPU/GPU Core |                                      |
    |    +-------------+                                        |
    |                                                          |
    +----------------------------------------------------------+

    [ 저항 관리 체크리스트 ]

    항목              | 목표 저항        | 설계 방안
    ------------------|------------------|--------------------
    MOSFET RDS(on)    | < 5mΩ/상        | 병렬 MOSFET, 고품질 소자
    인덕터 DCR        | < 1mΩ           | 저저항 인덕터
    PCB 트레이스      | < 0.5mΩ         | 넓은 트레이스, 다층 평면
    비아 저항         | < 0.1mΩ/개      | 다수 비아 병렬
    커넥터 접촉       | < 0.5mΩ         | 고품질 커넥터

    [ 전압 강하 예산 ]

    구간              | 허용 강하
    ------------------|-----------
    VRM 출력          | ±2% (기준)
    PCB 배선          | < 50mV
    패키지 내부       | < 30mV
    총 허용 강하      | < 100mV @ 100A
```

## 기술사 시험 포인트

### 1. 핵심 정의 및 공식
- **저항 정의**: R = V/I (전압/전류), 단위: 옴(Ω)
- **비저항 공식**: R = ρL/A (비저항×길이/단면적)
- **전력 손실**: P = I²R = V²/R
- **온도 계수**: R(T) = R₀[1 + α(T - T₀)]

### 2. 시험 빈출 키워드
1. **비저항(Resistivity, ρ)**: 재료 고유의 저항 특성. 도체는 10⁻⁸Ω·m 수준
2. **컨덕턴스(Conductance, G)**: 저항의 역수, 전류 흐름 용이성. 단위: 지멘스(S)
3. **온도 계수(TCR)**: 온도 변화에 따른 저항 변화율. 금속은 양(PTC), 반도체는 음(NTC)
4. **RDS(on)**: MOSFET의 온-저항. 전력 변환 효율의 핵심 인자
5. **IR Drop**: 전류×저항에 의한 전압 강하. 전원 설계의 핵심 고려사항

### 3. 단답형 예상 문제
- **Q**: 구리 도선의 저항을 줄이기 위한 방법을 설명하시오.
- **A**: ① 단면적을 넓힌다(R ∝ 1/A), ② 길이를 줄인다(R ∝ L), ③ 저비저항 재료(은 등)로 교체한다, ④ 온도를 낮춘다(금속은 PTC 특성).

### 4. 논술형 예상 문제
- **Q**: 고성능 CPU의 전원 설계에서 저항으로 인한 문제점과 해결 방안을 기술하시오.
- **A**:
  1. **문제점**:
     - VRM에서 CPU까지의 전압 강하(IR Drop)
     - 인터커넥트 RC 지연으로 인한 성능 저하
     - 저항에 의한 전력 손실(I²R)과 열 발생
     - 전원 노이즈 증가

  2. **해결 방안**:
     - 다상 VRM으로 전류 분산 (상당 전류 감소 → I²R 손실 감소)
     - 전원/접지 평면 다층 설계로 배선 저항 최소화
     - 고품질 MOSFET (저 RDS(on)) 사용
     - 적극적 전력 관리로 평균 전류 감소
