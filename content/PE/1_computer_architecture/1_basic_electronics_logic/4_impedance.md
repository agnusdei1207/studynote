# 임피던스 (Impedance)

## 정의

**임피던스(Impedance)**는 교류(AC) 회로에서 전압과 전류의 복소비(Complex Ratio)로 정의되는 물리량으로, 저항(Resistance), 유도성 리액턴스(Inductive Reactance), 그리고 용량성 리액턴스(Capacitive Reactance)를 통합한 개념이다. 국제단위계(SI)에서는 저항과 동일하게 옴(Ohm, 기호: Ω)을 사용하며, 복소수로 표현된다.

```
수식: Z = V/I = R + jX
- Z: 임피던스 (Ohm, Ω) - 복소수
- V: 전압 (Phasor, 복소수 표현)
- I: 전류 (Phasor, 복소수 표현)
- R: 저항 (Real Part, 실수부)
- X: 리액턴스 (Imaginary Part, 허수부)
- j: 허수 단위 (j² = -1)
```

컴퓨터 아키텍처 관점에서 임피던스는 고속 디지털 신호의 신호 무결성(Signal Integrity), 전원 분배 네트워크(PDN) 설계, 전송선로(Transmission Line) 특성, 그리고 EMI/EMC(Electromagnetic Interference/Compatibility) 규제 준수의 핵심 설계 인자로 작용한다. 현대 고속 인터페이스(PCIe, DDR, USB, HDMI)에서는 정밀한 임피던스 매칭이 필수적이다.

## 핵심 개념

### 1. 임피던스의 복소수 표현

```
    [ 임피던스의 복소 평면 표현 ]

                      Im (허수축)
                        ^
                        |
                  X_L   |  (유도성, Inductive)
                  (양의 X)  Z = R + jX_L
                        |
                        |     * Z (임피던스 벡터)
                        |    /
                        |   /
                        |  /  |Z| = √(R² + X²)
                        | /   (크기)
                        |/θ
    ─────────────────────+──────────────────> Re (실수축)
                        /|      R (저항)
                       / |
                      /  |
                     /   |
                X_C /    |  (용량성, Capacitive)
              (음의 X)   |  Z = R - jX_C
                        |

    * 위상각 θ = arctan(X/R)
    * |Z| = √(R² + X²) (임피던스 크기)
```

- **저항(Resistance, R)**: 에너지를 소모하는 성분. 위상 변화 없음. 실수부
- **리액턴스(Reactance, X)**: 에너지를 저장하는 성분. 위상 변화 발생. 허수부
- **임피던스 크기(Magnitude)**: |Z| = √(R² + X²)
- **위상각(Phase Angle)**: θ = arctan(X/R)

### 2. 리액턴스의 종류

```
    [ 유도성 리액턴스 (Inductive Reactance) ]

    X_L = ωL = 2πfL

    - ω: 각주파수 (rad/s)
    - f: 주파수 (Hz)
    - L: 인덕턴스 (Henry)

    특성:
    - 주파수가 높을수록 리액턴스 증가
    - 전류가 전압보다 90° 늦음 (전류 지상)
    - 인덕터, 변압기, 전선의 자체 인덕턴스

    [ 용량성 리액턴스 (Capacitive Reactance) ]

    X_C = 1/(ωC) = 1/(2πfC)

    - C: 정전용량 (Farad)

    특성:
    - 주파수가 높을수록 리액턴스 감소
    - 전압이 전류보다 90° 늦음 (전압 지상)
    - 커패시터, 기생 커패시턴스

    [ 주파수에 따른 리액턴스 변화 ]

    |X|
      ^
      |        /
      |       /  X_L = 2πfL (직선 증가)
      |      /
      |     /
      |____/_______________> f (주파수)
      |     \
      |      \  X_C = 1/(2πfC) (쌍곡선 감소)
      |       \
      |        \
```

| 구분 | 유도성 리액턴스 (X_L) | 용량성 리액턴스 (X_C) |
|:---:|:---:|:---:|
| **공식** | X_L = 2πfL | X_C = 1/(2πfC) |
| **주파수 의존성** | 비례 (∝ f) | 반비례 (∝ 1/f) |
| **위상 관계** | 전류가 전압보다 90° 지상 | 전압이 전류보다 90° 지상 |
| **에너지 저장** | 자기장 (Magnetic Field) | 전기장 (Electric Field) |
| **고주파 특성** | 차단 (Open) | 통과 (Short) |

### 3. 어드미턴스 (Admittance)

```
    [ 어드미턴스 정의 ]

    Y = 1/Z = G + jB

    - Y: 어드미턴스 (Siemens, S)
    - G: 컨덕턴스 (Conductance, 실수부)
    - B: 서셉턴스 (Susceptance, 허수부)

    [ 어드미턴스 성분 ]

    컨덕턴스:  G = R / (R² + X²)
    서셉턴스:  B = -X / (R² + X²)

    [ 병렬 회로 분석에 유리 ]

    직렬: Z_total = Z1 + Z2 + Z3 ... (임피던스 합)
    병렬: Y_total = Y1 + Y2 + Y3 ... (어드미턴스 합)
          Z_total = 1/Y_total
```

### 4. 임피던스와 위상

```
    [ 전압-전류 위상 관계 ]

    순수 저항 (R만):
    V(t) = V_m sin(ωt)
    I(t) = I_m sin(ωt)
    -> 위상차 = 0° (동상)

    인덕터 (L만):
    V(t) = V_m sin(ωt)
    I(t) = I_m sin(ωt - 90°)
    -> 전류가 90° 지상

    커패시터 (C만):
    V(t) = V_m sin(ωt)
    I(t) = I_m sin(ωt + 90°)
    -> 전류가 90° 진상

    [ RLC 직렬 회로의 위상 ]

    Z_total = R + j(ωL - 1/ωC)
            = R + j(X_L - X_C)

    X_L > X_C: 유도성 (전류 지상)
    X_L < X_C: 용량성 (전류 진상)
    X_L = X_C: 공진 (순수 저항)

    [ 파형도 ]

    전압 V(t)        전압
      ^               ^
      |   /\          |   /\
      |  /  \         |  /  \
    --+-/----\--    --+-/----\--  (저항: 동상)
      |/      \       |/      \
      +----------> t  +----------> t

    인덕터 전류:       커패시터 전류:
      ^               ^
      |     /\        | /\
      |    /  \       |/  \
    --+--/----\--   --+/----\--  (90° 위상차)
      | /      \      |      \
      +----------> t  +----------> t
```

## 상세 설명

### 1. 전송선로 임피던스

```
    [ 전송선로 모델 ]

    무손실 전송선로의 특성 임피던스:

    Z_0 = √(L/C)

    - L: 단위 길이당 인덕턴스 (H/m)
    - C: 단위 길이당 커패시턴스 (F/m)

    [ 분포 소자 모델 ]

    +----------------------------------------------------------+
    |                                                          |
    |   ----[L]----+----[L]----+----[L]----+----[L]----       |
    |              |           |           |                   |
    |             ===         ===         ===                  |
    |              C           C           C                   |
    |              |           |           |                   |
    |   -----------+-----------+-----------+-----------       |
    |                                                          |
    +----------------------------------------------------------+
           -> 신호 전파 방향 ->

    [ PCB 트레이스 특성 임피던스 ]

    마이크로스트립 (Microstrip):

    Z_0 ≈ (87/√(ε_r + 1.41)) × ln(5.98h / (0.8w + t))

    - ε_r: 기판 유전율
    - h: 트레이스 높이 (기판 두께)
    - w: 트레이스 폭
    - t: 트레이스 두께

    스트립라인 (Stripline):

    Z_0 ≈ (60/√ε_r) × ln(4h / (0.67π × (0.8w + t)))

    [ 일반적 임피던스 값 ]

    용도              | Z_0 (Ω)
    ------------------|--------
    PCB 일반 신호     | 50
    PCIe, SATA        | 85~100 (차동)
    USB               | 90 (차동)
    DDR4/5 DQ         | 40~50 (단일)
    DDR4/5 Clock      | 100~120 (차동)
    HDMI              | 100 (차동)
    Ethernet          | 100 (차동)
    RF/무선           | 50
    TV/영상           | 75
```

### 2. 임피던스 매칭과 반사

```
    [ 임피던스 불연속에 의한 반사 ]

    신호가 Z1에서 Z2로 진행할 때:

    반사 계수 (Reflection Coefficient):
    Γ = (Z_2 - Z_1) / (Z_2 + Z_1)

    투과 계수 (Transmission Coefficient):
    T = 2Z_2 / (Z_2 + Z_1) = 1 + Γ

    [ 반사 파형 ]

    완전 매칭 (Z_2 = Z_1):
    Γ = 0, 반사 없음

    개방 (Z_2 = ∞):
    Γ = +1, 100% 정반사 (동상)

    단락 (Z_2 = 0):
    Γ = -1, 100% 정반사 (역상)

    [ 반사로 인한 신호 왜곡 ]

    송신단           불연속점            수신단
    +----[Z_0]----+---[Z_1]---+----[Z_2]----+
                  |           |
              입사파         반사파
                  |           |
                  v           v
                  / \         / \
                 /   \       /   \
    ────────────/─────\─────/─────\─────────

    * 반사로 인한 신호 왜곡, 오버슈트/언더슈트 발생
    * 고속 디지털 회로에서 비트 오류 원인
```

**임피던스 매칭 기법**:

| 기법 | 설명 | 적용 분야 |
|:---|:---|:---|
| **직렬 종단** | 송신단에 R = Z_0 연결 | 단방향 버스, 저속 |
| **병렬 종단** | 수신단에 R = Z_0 연결 (GND 또는 VTT) | 고속 양방향 |
| **더비 종단** | 수신단에 2×Z_0 두 개를 VCC/GND 연결 | TTL/CMOS |
| **AC 종단** | 수신단에 R-C 직렬 연결 | 저전력, 가변 주파수 |
| **액티브 종단** | VTT 전압원으로 종단 | DDR 메모리 |

### 3. 전원 분배 네트워크(PDN) 임피던스

```
    [ 목표 임피던스 개념 ]

    Z_target = ΔV / ΔI

    예: 코어 전압 1.0V, 허용 리플 ±5% (50mV)
        최대 전류 변화 20A

    Z_target = 50mV / 20A = 2.5mΩ

    -> 전체 주파수 범위에서 PDN 임피던스를 2.5mΩ 이하로 유지

    [ PDN 임피던스 곡선 ]

    |Z|
      ^
      |                    +++++  <-- 인덕턴스 영역 (고주파)
      |                  ++
      |                ++
      |              ++        목표 임피던스
      |            ++          - - - - - - - - - - - - -
      |          ++            |               ++
      |        ++              |             ++
      |______++________________|___________++______
      |      |                 |         ++  커패시턴스
      |      |                 |       ++    영역 (저주파)
      |      |                 |     ++
      +------+-----------------+---+----------------> f
           VRM              decap  package
                                  resonance

    [ PDN 구성 요소별 임피던스 기여 ]

    주파수 범위        | 주요 소자        | 임피던스 특성
    -------------------|------------------|----------------
    DC ~ 1kHz          | VRM              | 매우 낮음
    1kHz ~ 1MHz        | 벌크 커패시터    | 감소
    1MHz ~ 100MHz      | 세라믹 커패시터  | 평탄 (목표값)
    100MHz ~ 1GHz      | 패키지/온다이    | 인덕턴스 증가
```

### 4. 차동 신호와 임피던스

```
    [ 차동 신호의 임피던스 ]

    +Z_single ────────[===]──────── +V_diff
                       |    |
    -Z_single ────────[===]──────── -V_diff

    차동 임피던스: Z_diff = 2 × (Z_single + Z_coupling)

    - Z_single: 단일 선 임피던스
    - Z_coupling: 결합 임피던스 (선간 커패시턴스/인덕턴스)

    [ 차동 쌍 기하구조 ]

    단면도:
              w           s           w
        +---------+   +---------+   +---------+
        |  D+     |   |  Gap    |   |  D-     |
        |  Trace  |   |  (s)    |   |  Trace  |
        +---------+   +---------+   +---------+
        ─────────────────────────────────────
                    기판 (유전체)
        ─────────────────────────────────────
                   접지 평면 (GND)

    [ 차동 임피던스 설계 파라미터 ]

    파라미터        | Z_diff 증가 | Z_diff 감소
    ----------------|-------------|-------------
    트레이스 폭(w)  | 폭 감소     | 폭 증가
    간격(s)         | 간격 증가   | 간격 감소
    유전체 두께(h)  | 두께 증가   | 두께 감소
    유전율(ε_r)     | 유전율 감소 | 유전율 증가

    [ 차동 신호의 장점 ]

    1. 노이즈 면역성: 공통 모드 노이즈 제거
    2. EMI 감소: 반대 방향 전류가 자기장 상쇄
    3. 신호 무결성: 지터 감소, 눈 다이어그램 개선
```

### 5. 고주파에서의 기생 임피던스

```
    [ 기생 성분의 등가 회로 ]

    이상적 커패시터        실제 커패시터
         |                    |
        ===                  ===
         |                    |
                              |
                             [L] ESL (등가 직렬 인덕턴스)
                              |
                             [R] ESR (등가 직렬 저항)

    실제 커패시터 임피던스:
    Z_C = ESR + j(ω×ESL - 1/(ωC))

    [ 공진 주파수 ]

    자기 공진 주파수 (SRF):
    f_res = 1 / (2π√(L×C))

    f < f_res: 용량성 동작 (C가 지배적)
    f = f_res: 순수 저항 (ESR만 남음) - 최소 임피던스
    f > f_res: 유도성 동작 (ESL이 지배적)

    [ 커패시터 SRF 예시 ]

    커패시터 타입    | 용량    | SRF (자기공진주파수)
    -----------------|---------|--------------------
    10μF 전해        | 10μF    | ~100kHz
    1μF 세라믹       | 1μF     | ~1MHz
    100nF 세라믹     | 100nF   | ~10MHz
    10nF 세라믹      | 10nF    | ~30MHz
    1nF 세라믹       | 1nF     | ~100MHz
    100pF 세라믹     | 100pF   | ~300MHz

    [ 다중 커패시터 병렬 사용 ]

    |Z|
      ^
      |    ++      ++
      |   +  +    +  +     개별 커패시터
      |  +    +  +    +    임피던스 곡선
      | +      ++      ++
      |
      |          ___
      |         /   \           병렬 조합:
      |________/     \______    넓은 주파수 범위에서
      |                           낮은 임피던스 유지
      +-------------------------------> f
         10μF  1μF  100nF  10nF  1nF
```

## 실무 적용

### 1. PCB 임피던스 제어 설계

```
    [ PCB 스택업 설계 예시 - 4층 기판 ]

    Layer 1  |  신호 (50Ω 마이크로스트립)    | 0.5oz Cu
    ─────────────────────────────────────────────────────
    Layer 2  |  접지 평면 (GND)              | 1oz Cu
             |  유전체 FR-4, h = 0.2mm       |
    ─────────────────────────────────────────────────────
    Layer 3  |  전원 평면 (PWR)              | 1oz Cu
             |  유전체 FR-4, h = 0.2mm       |
    ─────────────────────────────────────────────────────
    Layer 4  |  신호 (50Ω 마이크로스트립)    | 0.5oz Cu

    [ 50Ω 마이크로스트립 설계 ]

    조건: ε_r = 4.5 (FR-4), h = 0.2mm, t = 0.035mm

    목표 Z_0 = 50Ω

    계산된 트레이스 폭: w ≈ 0.38mm (15mil)

    [ 차동 쌍 설계 (100Ω) ]

    조건: 동일 스택업, Z_diff = 100Ω

    계산된 값:
    - 트레이스 폭 w ≈ 0.2mm (8mil)
    - 간격 s ≈ 0.15mm (6mil)

    [ 제작 공차 ]

    파라미터        | 일반 공차    | 정밀 공차
    ----------------|--------------|--------------
    임피던스        | ±10%         | ±5%
    트레이스 폭     | ±20%         | ±10%
    유전체 두께     | ±10%         | ±5%
```

### 2. 고속 인터페이스 임피던스 요구사항

```
    [ 주요 고속 인터페이스 임피던스 규격 ]

    인터페이스      | 데이터율    | 단일/차동 | Z_target
    ----------------|-------------|-----------|----------
    PCIe Gen 3      | 8 GT/s      | 차동      | 85Ω ±10%
    PCIe Gen 4      | 16 GT/s     | 차동      | 85Ω ±5%
    PCIe Gen 5      | 32 GT/s     | 차동      | 85Ω ±5%
    DDR4-3200       | 3.2 GT/s    | 단일(DQ)  | 40Ω ±10%
    DDR4-3200       | 3.2 GT/s    | 차동(CLK) | 120Ω ±10%
    DDR5-5600       | 5.6 GT/s    | 단일(DQ)  | 40Ω ±5%
    USB 3.2 Gen 2   | 10 Gbps     | 차동      | 90Ω ±5%
    USB4            | 40 Gbps     | 차동      | 90Ω ±3%
    HDMI 2.1        | 48 Gbps     | 차동      | 100Ω ±10%
    SATA 3.0        | 6 Gbps      | 차동      | 100Ω ±10%
    Ethernet 10G    | 10 Gbps     | 차동      | 100Ω ±5%

    [ TDR (Time Domain Reflectometry) 측정 ]

    측정 원리:
    - 펄스를 전송선로에 주입
    - 반사 파형을 시간 영역에서 분석
    - 임피던스 불연속 위치와 크기 파악

    TDR 파형 예시:

    임피던스(Ω)
      ^
    60 |          +-------+
      |         /         \
    50 |--------+           \--------+
      |                              \
    40 |                               \
      +----------------------------------> 시간
          커넥터   케이블    PCB 트레이스  IC 패키지
          (저임피던스) (정상)  (약간 높음)

    [ VNA (Vector Network Analyzer) 측정 ]

    주파수 영역 측정:
    - S-parameter (Scattering Parameter) 측정
    - S11: 반사 계수 (Return Loss)
    - S21: 투과 계수 (Insertion Loss)

    Return Loss (RL) = -20 log|Γ| (dB)

    목표 RL > 10~20dB (반사 10~1% 이하)
```

### 3. EMI/EMC와 임피던스

```
    [ 전도성 EMI와 임피던스 ]

    전도성 노이즈 경로:

    노이즈원 --[Z_source]---[Z_path]---[Z_victim]--> 피해회로

    노이즈 전류: I_noise = V_noise / (Z_source + Z_path + Z_victim)

    [ 공통 모드와 차동 모드 ]

    공통 모드 (Common Mode):         차동 모드 (Differential Mode):
    +--------->                      +--------->
               |                                |
    +--------->                      <---------+

    공통 모드 초크 (CMC):
    - 공통 모드 노이즈에 높은 임피던스
    - 차동 모드 신호에는 낮은 임피던스
    - USB, Ethernet 등에 필수

    [ 전원 필터 임피던스 설계 ]

    +--------+    +---[L]---+----+----[L]---+
    | 노이즈 |    |         |    |          |
    | 소스   |----+        ===   +---------+---- DC 출력
    +--------+    |         C    |          |
                  +----+----+    +---[L]----+
                       |
                      === GND

    필터 임피던스:
    - 저주파: 커패시터 높은 임피던스 (감쇠 작음)
    - 고주파: 인덕터 높은 임피던스 (감쇠 큼)

    감쇠량: Attenuation = 20 log(Z_filter / Z_load) (dB)
```

### 4. 메모리 인터페이스 임피던스 설계

```
    [ DDR 메모리 버스 임피던스 ]

    +----------------------------------------------------------+
    |                    DDR 메모리 채널                        |
    +----------------------------------------------------------+
    |                                                          |
    |   Controller                                              |
    |   +--------+                                             |
    |   |        |---[ODT]---+---[Z_0]---+---[ODT]---|        |
    |   |        |            |           |           |        |
    |   +--------+           ===          +----------+ Memory |
    |                        |            |           |        |
    |                       GND          GND          +--------+
    |                                                          |
    +----------------------------------------------------------+

    [ ODT (On-Die Termination) ]

    - 메모리 칩 내장 종단 저항
    - Z_0 매칭을 위해 가변 저항 사용
    - DDR3: 120Ω, 60Ω, 40Ω, 30Ω 선택 가능
    - DDR4: 240Ω ~ 34Ω 범위
    - DDR5: 자동 교정 (ZQ Calibration)

    [ DDR4 임피던스 요구사항 ]

    신호군        | Z_target    | 공차    | 비고
    --------------|-------------|---------|------------------
    DQ/DQS (단일) | 40Ω         | ±10%    | 데이터 라인
    CLK (차동)    | 120Ω        | ±10%    | 클럭
    ADDR/CMD      | 40Ω         | ±10%    | 어드레스/커맨드
    CTRL          | 40Ω         | ±10%    | 컨트롤 신호

    [ VTT 종단 전압 ]

    DDR3/4: VTT = VDD/2 (예: 1.35V/2 = 0.675V)
    DDR5: VTT 없음 (ODT만 사용)
```

## 기술사 시험 포인트

### 1. 핵심 정의 및 공식
- **임피던스 정의**: Z = V/I = R + jX (복소수), 단위: 옴(Ω)
- **유도성 리액턴스**: X_L = 2πfL (주파수에 비례)
- **용량성 리액턴스**: X_C = 1/(2πfC) (주파수에 반비례)
- **특성 임피던스**: Z_0 = √(L/C) (전송선로)
- **반사 계수**: Γ = (Z_2 - Z_1)/(Z_2 + Z_1)

### 2. 시험 빈출 키워드
1. **특성 임피던스(Characteristic Impedance, Z_0)**: 전송선로의 고유 임피던스. PCB에서는 50Ω, 100Ω(차동)이 표준
2. **임피던스 매칭(Impedance Matching)**: 반사를 방지하기 위해 소스/부하/선로 임피던스를 일치시키는 기술
3. **반사 계수(Reflection Coefficient, Γ)**: 임피던스 불연속에 의한 신호 반사 비율. |Γ| ≤ 1
4. **VSWR (Voltage Standing Wave Ratio)**: 정재파비. 반사에 의한 전압 최대/최소 비율
5. **S-parameter**: 산란 파라미터. 고주파 네트워크 특성 표현 (S11 반사, S21 투과)

### 3. 단답형 예상 문제
- **Q**: 고속 디지털 PCB에서 50Ω 특성 임피던스를 갖는 트레이스 폭을 결정하는 요소를 설명하시오.
- **A**: 기판 유전체의 유전율(ε_r), 기판 두께(h), 트레이스 두께(t)에 의해 결정되며, 일반적으로 Z_0 ≈ (87/√(ε_r+1.41)) × ln(5.98h/(0.8w+t)) 공식을 사용하여 트레이스 폭(w)을 계산한다.

### 4. 논술형 예상 문제
- **Q**: 고속 디지털 시스템에서 임피던스 매칭이 필요한 이유와 매칭하지 않을 때 발생하는 문제점을 기술하시오.
- **A**:
  1. **필요성**:
     - 신호 반사 방지: 임피던스 불연속 시 신호 일부가 반사되어 원신호와 간섭
     - 전력 전달 최적화: 최대 전력 전달을 위해 소스와 부하 임피던스 매칭 필요
     - 신호 무결성 보장: 반사로 인한 오버슈트, 언더슈트, 링잉 방지

  2. **매칭하지 않을 때 문제점**:
     - 신호 왜곡: 반사파와 입사파의 중첩으로 신호 형태 변형
     - 타이밍 오류: 반사로 인한 지터(Jitter) 증가, 셋업/홀드 타임 위반
     - 비트 오류: 심볼 간 간섭(ISI)으로 인한 데이터 오류
     - EMI 증가: 반사로 인한 고주파 복사 증가
     - 전력 손실: 반사된 신호 에너지 손실
