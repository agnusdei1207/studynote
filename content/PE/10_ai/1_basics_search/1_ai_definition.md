# 인공지능 (Artificial Intelligence)의 정의

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 인간의 지적 능력(학습, 추론, 문제 해결, 인식)을 기계 시스템으로 구현하여, 데이터로부터 패턴을 학습하고 새로운 상황에서 합리적인 의사결정을 수행하는 컴퓨터 과학의 핵심 분야.
> 2. **진화**: 기호주의(Symbolic AI)에서 연결주의(Connectionism/Neural Network)로 패러다임이 전환되며, 규칙 기반 시스템이 데이터 기반 학습 시스템으로 대체됨.
> 3. **범위**: 약인공지능(Narrow AI)에서 강인공지능(AGI)으로의 진화를 목표로 하며, 지각(Perception), 인지(Cognition), 행동(Action)의 3단계 지능 처리 체계를 포괄.

## Ⅰ. 개요 및 역사적 맥락 (Context & Background)

### 1. 정의의 다층적 해석

인공지능(Artificial Intelligence, AI)은 **"지능적 기계 및 지능적 에이전트를 설계하는 학문"**으로 정의된다. 이 정의는 1956년 다트머스 회의(Dartmouth Conference)에서 존 매카시(John McCarthy)가 처음 제안하였으며, 이후 다양한 관점에서 확장되어 왔다.

```
+------------------------------------------------------------------+
|                    인공지능 정의의 다층 구조                       |
+------------------------------------------------------------------+
|                                                                  |
|  [ Layer 1: 기계 지능 (Machine Intelligence) ]                   |
|    - 기계가 인간의 지적 작업을 수행할 수 있는 능력                  |
|    - 계산, 기억, 검색 등 기본적 인지 기능 모방                     |
|                                                                  |
|  [ Layer 2: 학습 능력 (Learning Capability) ]                    |
|    - 경험(데이터)으로부터 자동으로 개선되는 능력                    |
|    - 명시적 프로그래밍 없이 패턴 습득                             |
|                                                                  |
|  [ Layer 3: 추론 능력 (Reasoning Capability) ]                   |
|    - 불확실한 상황에서 합리적 결론 도출                           |
|    - 논리적 사고, 계획 수립, 문제 해결                            |
|                                                                  |
|  [ Layer 4: 적응 능력 (Adaptability) ]                           |
|    - 새로운 환경과 과업에 유연하게 대응                           |
|    - 전이 학습(Transfer Learning) 및 일반화                       |
|                                                                  |
+------------------------------------------------------------------+
```

```
|  - 보상 신호 기반 행동 최적화                                    |
|                                                                  |

```

## Ⅶ. 결론 및 전망 (Conclusion & Future Outlook)

### 핵심 요약

1. **정의의 본질**: 인공지능은 인간의 지적 능력을 기계로 구현하는 학문으로, 학습, 추론, 인식, 적응 능력을 포괄한다.

2. **진화 경로**: 기호주의(규칙 기반)에서 연결주의(신경망)로, 약인공지능에서 강인공지능으로의 진화가 진행 중이다.

3. **실무 적용**: 엔터프라이즈 AI는 데이터 계층, 학습 계층, 서빙 계층, 응용 계층의 통합 아키텍처로 구현된다.

4. **윤리적 고려**: 투명성, 공정성, 책임성, 프라이버시, 안전성을 균형있게 확보해야 한다.

### 미래 전망

- **AGI(범용 인공지능)**: 인간 수준의 범용 지능을 갖춘 AI로의 진화
- **멀티모달 AI**: 텍스트, 이미지, 오디오, 비디오를 통합 이해 및 생성
- **온디바이스 AI**: 프라이버시 보호와 실시간 처리를 위한 엣지 컴퓨팅
- **AI 거버넌스**: 국제적 규제 프레임워크 및 윤리 가이드라인 정착
