# RS 플립플롭 (RS Flip-Flop / SR 래치)

## 핵심 인사이트 (3줄 요약)
> 가장 기본적인 플립플롭으로, Set과 Reset 입력으로 1비트 정보를 저장한다. **금지 상태(S=R=1)**가 존재하며, 이를 해결한 것이 JK 플립플롭이다. 레지스터와 메모리의 기본 구조다.

## 1. 개념
RS 플립플롭(Set-Reset Flip-Flop)은 **Set(S)과 Reset(R) 입력으로 상태를 저장**하는 1비트 메모리 소자다. 래치(Latch) 형태로 구현되며, 레벨 트리거 방식이다.

> 비유: "스위치 두 개로 불 켜고 끄기" - Set 불 켜기, Reset 불 끄기

## 2. 진리표

| S | R | Q(n+1) | 상태 |
|---|---|--------|------|
| 0 | 0 | Q(n) | 유지 (Hold) |
| 0 | 1 | 0 | 리셋 (Reset) |
| 1 | 0 | 1 | 설정 (Set) |
| 1 | 1 | ? | **금지** (Invalid) |

## 3. 회로 구조

```
NAND 게이트 RS 래치:

    R ──┐       ┌── Q̄
        ├─NAND─┤
    ┌───┘       └──┐
    │              │
    └──┐       ┌───┘
       ├─NAND─┤
    S ─┘       └── Q

특징: NAND 버전은 입력이 Active Low (S̄, R̄)
```

## 4. 금지 상태 문제

```
S=1, R=1일 때:
- Q = NOT(Q̄)
- Q̄ = NOT(Q)
- 두 출력이 모두 1이 됨 (모순)
- 이후 S, R이 0이 되면 불확정 상태
```

## 5. 장단점

| 장점 | 단점 |
|-----|------|
| 구조가 간단함 | 금지 상태 존재 |
| 기본 메모리 소자 | 클럭 동기화 없음 |
| 쉽게 이해 가능 | 레이스 컨디션 가능 |

## 6. 활용 사례

1. **스위치 디바운싱**: 기계적 스위치의 채터링 제거
2. **기본 메모리 셀**: SRAM의 기본 구조
3. **상태 저장**: FSM의 상태 유지
4. **데이터 래치**: 버스 데이터 임시 저장

## 7. 코드 예시

```verilog
// 기본 SR 래치 (비동기)
module sr_latch(
    input s,      // Set
    input r,      // Reset
    output q,     // 출력
    output q_bar  // 반전 출력
);
    // NAND 게이트로 구현 (Active Low)
    nand n1(q_bar, s, q);
    nand n2(q, r, q_bar);
endmodule
```

## 8. 클럭 동기 RS 플립플롭

```
클럭(Clock) 추가:

    S ──┐
        ├─AND─┐
    CLK ┘     │
              ├─RS 래치─ Q
    R ──┐     │
        ├─AND─┘
    CLK ┘

특징: CLK=1일 때만 동작
```

## 9. 다른 플립플롭과 비교

| 플립플롭 | 금지 상태 | 토글 | 주요 용도 |
|---------|----------|------|---------|
| RS | 있음 (S=R=1) | 없음 | 기본 교육 |
| D | 없음 | 없음 | 데이터 저장 |
| JK | 없음 | 있음 | 범용 카운터 |
| T | 없음 | 있음 | 주파수 분주 |

## 10. 실무에선? (기술사적 판단)
- **현업 사용**: RS 래치는 거의 사용 안 함
- **대안**: D 플립플롭이 표준, JK는 카운터용
- **디바운싱**: 하드웨어 스위치 입력에 여전히 사용

## 11. 관련 개념
- 래치 vs 플립플롭
- 레벨 트리거 vs 엣지 트리거
- SRAM 셀

## 어린이를 위한 종합 설명

**RS 플립플롭은 "기억하는 전등 스위치"야!**

방에 전등 스위치가 두 개 있어요:
- **S(Set) 스위치**: 불을 켜요! 💡
- **R(Reset) 스위치**: 불을 꺼요! ⬛

```
S 누름 → 불 켜짐! (기억해요)
R 누름 → 불 꺼짐! (기억해요)
아무것도 안 누름 → 그대로 유지! (기억하고 있어요)
S랑 R 동시에 누름 → 💥 고장! (하면 안 돼요!)
```

**기억의 마법**: 스위치를 손에서 떼어도 불이 켜진 채로 유지돼요! 이게 바로 컴퓨터가 정보를 기억하는 방법이에요!

⚠️ **주의**: S와 R을 동시에 누르면 안 돼요! 컴퓨터가 헷갈려하거든요!
