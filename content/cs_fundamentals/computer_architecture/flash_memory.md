# 플래시 메모리 (Flash Memory)

## 핵심 인사이트 (3줄 요약)
> **전기적으로 지우고 쓸 수 있는** 비휘발성 메모리. EEPROM의 일종으로 NAND와 NOR 타입이 있다. SSD, USB, 스마트폰 저장장치의 핵심 기술이다.

## 1. 개념
플래시 메모리(Flash Memory)는 **전원이 꺼져도 데이터가 유지되는 비휘발성 메모리**다. 블록 단위로 지우기가 가능하여 빠른 대량 저장이 가능하다.

> 비유: "전자 화이트보드" - 지우고 다시 쓸 수 있음

## 2. 동작 원리

```
플래시 셀 구조:

   비트 라인
       │
    ┌──┴──┐
    │ FG  │ ← 플로팅 게이트 (전하 저장)
    └──┬──┘
       │
    컨트롤 게이트
       │
   소스 ─── 드레인

쓰기: 높은 전압 → 전자가 FG로 주입
지우기: 높은 전압 → 전자가 FG에서 제거
읽기: 게이트 전압 → 전류 흐름 감지
```

## 3. NAND vs NOR 플래시

| 특징 | NAND | NOR |
|------|------|-----|
| 구조 | 직렬 | 병렬 |
| 읽기 속도 | 느림 | 빠름 |
| 쓰기 속도 | 빠름 | 느림 |
| 밀도 | 높음 | 낮음 |
| XIP | 불가 | 가능 |
| 용도 | SSD, USB | 펌웨어 |

## 4. SLC/MLC/TLC/QLC

```
셀당 비트 수:

SLC (Single Level Cell): 1 bit/cell
  - 속도: 빠름
  - 수명: 100,000 P/E
  - 가격: 비쌈

MLC (Multi Level Cell): 2 bits/cell
  - 속도: 보통
  - 수명: 3,000 P/E
  - 가격: 중간

TLC (Triple Level Cell): 3 bits/cell
  - 속도: 느림
  - 수명: 1,000 P/E
  - 가격: 저렴

QLC (Quad Level Cell): 4 bits/cell
  - 속도: 더 느림
  - 수명: 100 P/E
  - 가격: 가장 저렴
```

## 5. 읽기/쓰기/지우기

```
동작 특성:

읽기: 페이지 단위 (4KB~16KB)
쓰기: 페이지 단위
지우기: 블록 단위 (수 MB)

중요: 지워진 곳에만 쓰기 가능!
→ 쓰기 전에 반드시 지우기 필요
```

## 6. 웨어 레벨링 (Wear Leveling)

```
문제: 같은 블록만 계속 쓰면 빨리 닳음

해결: 모든 블록을 골고루 사용

정적 웨어 레벨링: 자주 안 쓰는 데이터 이동
동적 웨어 레벨링: 쓰기 시 가장 적게 쓴 블록 선택
```

## 7. SSD 구조

```
SSD 내부:

┌─────────────────────────────────┐
│           컨트롤러              │
│  (FTL, 웨어레벨링, ECC)         │
└────────────┬────────────────────┘
             │
    ┌────────┼────────┐
    ↓        ↓        ↓
┌──────┐ ┌──────┐ ┌──────┐
│ NAND │ │ NAND │ │ NAND │
│ 패키지│ │ 패키지│ │ 패키지│
└──────┘ └──────┘ └──────┘

FTL (Flash Translation Layer):
논리 주소 → 물리 주소 변환
```

## 8. 장단점

| 장점 | 단점 |
|-----|------|
| 비휘발성 | 쓰기 횟수 제한 |
| 기계적 부품 없음 | 지우기 후 쓰기 |
| 빠른 속도 | 비용 |
| 저전력 | 데이터 보존 기간 |

## 9. 3D NAND

```
2D NAND: 평면 구조
┌───┬───┬───┬───┐
│   │   │   │   │
└───┴───┴───┴───┘

3D NAND: 수직 적층 (96~232층)
┌───┐
│   │
├───┤
│   │
├───┤
│   │
└───┘

→ 밀도 대폭 증가, 비용 절감
```

## 10. 실무에선? (기술사적 판단)
- **기업용 SSD**: SLC/MLC, 높은 내구성
- **소비자 SSD**: TLC/QLC, 가격 중심
- **모바일**: UFS (Universal Flash Storage)
- **데이터센터**: NVMe SSD

## 11. 관련 개념
- EEPROM
- SSD
- NVMe

## 어린이를 위한 종합 설명

**플래시 메모리는 "전자 칠판"이야!**

### 비유
```
일반 칠판: 지우면 사라져요
전자 칠판: 지워도 저장돼요! (비휘발성)
```

### 셀 = 작은 방
```
방 하나에 정보를 저장해요:

SLC: 0 또는 1 (1비트) - 2가지
MLC: 00, 01, 10, 11 (2비트) - 4가지
TLC: 3비트 - 8가지
QLC: 4비트 - 16가지

더 많이 넣을수록 싸지만, 빨리 닳아요!
```

### 수명 문제
```
칠판은 몇 번 지우면 닳아요

SLC: 100,000 번 지울 수 있어요 💪
MLC: 3,000 번
TLC: 1,000 번
QLC: 100 번

→ 골고루 써야 해요! (Wear Leveling)
```

**비밀**: 스마트폰, USB, SSD... 모든 게 플래시 메모리예요! 📱💾
