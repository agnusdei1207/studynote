# 하이브리드 컴퓨터 (Hybrid Computer)

## 핵심 인사이트 (3줄 요약)
> **아날로그와 디지털 컴퓨터의 장점을 결합**한 컴퓨터. 아날로그의 연속적 처리와 디지털의 정확성을 모두 활용한다. 의료, 군사, 공장 자동화에서 사용된다.

## 1. 개념
하이브리드 컴퓨터(Hybrid Computer)는 **아날로그 컴퓨터와 디지털 컴퓨터의 기능을 결합**한 시스템이다.

> 비유: "디지털 카메라 + 필름 카메라" - 두 방식의 장점 활용

## 2. 아날로그 vs 디지털 vs 하이브리드

```
아날로그:
- 연속적 신호
- 빠른 실시간 처리
- 정확도 낮음

디지털:
- 이산적 신호 (0, 1)
- 정확한 계산
- 처리 속도 느림

하이브리드:
- 아날로그 입력 → 디지털 처리 → 아날로그 출력
- 실시간 + 정확성
```

## 3. 구조

```
┌─────────────────────────────────────┐
│           하이브리드 컴퓨터          │
│  ┌─────────┐     ┌─────────┐       │
│  │ 아날로그 │ ←→  │  디지털  │       │
│  │  프로세서│     │  프로세서│       │
│  └─────────┘     └─────────┘       │
│       ↑               │            │
│    센서              ↓             │
│  (온도,압력)      [ADC]            │
│       │              ↓             │
│       └────────── [DAC] ──→ 제어기  │
└─────────────────────────────────────┘

ADC: Analog-to-Digital Converter
DAC: Digital-to-Analog Converter
```

## 4. 동작 과정

```
1. 아날로그 입력 (온도, 압력, 속도)
2. ADC로 디지털 변환
3. 디지털 연산 (정확한 계산)
4. DAC로 아날로그 변환
5. 아날로그 출력 (모터 제어 등)
```

## 5. 응용 분야

| 분야 | 용도 |
|------|------|
| 의료 | ECG, MRI 신호 처리 |
| 군사 | 레이더, 미사일 유도 |
| 공장 | PLC, 공정 제어 |
| 항공 | 비행 시뮬레이터 |
| 통신 | 신호 변환 |

## 6. 장단점

| 장점 | 단점 |
|-----|------|
| 실시간 처리 | 복잡한 구조 |
| 높은 정확도 | 비용 높음 |
| 유연성 | 유지보수 어려움 |

## 7. 예시: 의료 장비

```
ECG (심전도) 시스템:

심장 신호 (아날로그)
    ↓
증폭기
    ↓
ADC (디지털 변환)
    ↓
디지털 신호 처리
    ↓
화면 표시 + 알람

→ 실시간 모니터링 + 정확한 진단
```

## 8. PLC (Programmable Logic Controller)

```
공장 자동화의 하이브리드:

┌─────────┐
│   PLC   │
├─────────┤
│ 센서 입력│ → 디지털 처리 → 모터 제어
└─────────┘

온도, 압력 등 아날로그 입력
→ 디지털 연산
→ 아날로그 제어 출력
```

## 9. 실무에선? (기술사적 판단)
- **공장 자동화**: PLC가 대세
- **의료 기기**: 디지털화 추세
- **IoT**: 센서-디지털-클라우드
- **AI**: 하이브리드 AI (신경망+규칙)

## 10. 관련 개념
- 아날로그 컴퓨터
- ADC/DAC
- PLC

## 어린이를 위한 종합 설명

**하이브리드 컴퓨터는 "듀얼 파워"야!**

### 아날로그 (자연)
```
물의 온도, 바람의 세기...
연속적으로 변해요 🌊
→ 빠르지만 정확하지 않아요
```

### 디지털 (컴퓨터)
```
0 또는 1, 정확해요! 💻
→ 정확하지만 느려요
```

### 하이브리드 (둘 다!)
```
센서 → [변환] → 컴퓨터 → [변환] → 모터
아날로그         디지털        아날로그

→ 빠르고 정확해요! ✨
```

### 예시
```
🎮 게임 컨트롤러:
아날로그 스틱 (연속) → 디지털 변환 → 게임

🏥 병원 기계:
심장 박동 → 컴퓨터 분석 → 화면 표시
```
