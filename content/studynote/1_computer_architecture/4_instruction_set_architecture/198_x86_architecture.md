# 198. x86 아키텍처 (x86 Architecture)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 1978년 8086부터 현재 x86-64까지 이어진 CISC ISA
> 2. **가치**: 데스크톱/서버 시장 지배, 45년+ 이진 호환성
> 3. **융합**: Intel/AMD, Windows/Linux, 게임/워크스테이션

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

x86 아키텍처는 **Intel 8086(1978)에서 시작하여 현재 x86-64까지 이어지는 CISC ISA**다. 45년 이상의 이진 호환성을 유지하며 데스크톱, 서버, 워크스테이션 시장을 지배한다.

### 💡 비유: 45년 된 식당의 레시피

x86은 **45년 된 식당의 레시피**와 같다. 1978년 메뉴(8086 명령어)가 지금도 주문 가능하다. 메뉴가 계속 늘어났지만, 옛날 메뉴도 여전히 만들 수 있다. 주방(하드웨어)은 복잡하지만, 단골손님(소프트웨어)은 편하다.

## Ⅱ. x86 명령어 집합

### 명령어 상세

```
┌─────────────────────────────────────────────────────────────────────┐
│                x86 명령어 집합                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【주요 명령어】                                                     │
│  ──────────────────                                                  │
│                                                                     │
│  ; 데이터 이동                                                        │
│  MOV RAX, RBX            ; RAX = RBX                                │
│  MOV RAX, [RDI]          ; RAX = Mem[RDI]                           │
│  MOV [RDI], RAX          ; Mem[RDI] = RAX                           │
│  LEA RAX, [RBX+RCX*4]    ; 주소 계산                                │
│                                                                     │
│  ; 산술                                                               │
│  ADD RAX, RBX            ; RAX += RBX                               │
│  SUB RAX, RBX            ; RAX -= RBX                               │
│  MUL RBX                 ; RDX:RAX = RAX * RBX                      │
│  DIV RBX                 ; RAX = RDX:RAX / RBX                      │
│  INC RAX                 ; RAX++                                    │
│  DEC RAX                 ; RAX--                                    │
│                                                                     │
│  ; 논리                                                               │
│  AND RAX, RBX            ; RAX &= RBX                               │
│  OR RAX, RBX             ; RAX |= RBX                               │
│  XOR RAX, RBX            ; RAX ^= RBX                               │
│  NOT RAX                 ; RAX = ~RAX                               │
│  TEST RAX, RAX           ; RAX & RAX (플래그만)                      │
│                                                                     │
│  ; 분기                                                               │
│  CMP RAX, RBX            ; RAX - RBX (플래그만)                      │
│  JE label                ; if (ZF) goto label                        │
│  JNE label               ; if (!ZF) goto label                      │
│  JG label                ; if (RAX > RBX) goto label                │
│  JL label                ; if (RAX < RBX) goto label                │
│  JMP label               ; goto label                               │
│  CALL label              ; PUSH ret_addr; JMP label                  │
│  RET                     ; POP PC                                   │
│                                                                     │
│  ; 스택                                                               │
│  PUSH RAX                ; RSP -= 8; [RSP] = RAX                    │
│  POP RBX                 ; RBX = [RSP]; RSP += 8                    │
│                                                                     │
│  ; SIMD (SSE/AVX)                                                    │
│  ADDPS XMM0, XMM1        ; 4개 float 병렬 덧셈                      │
│  ADDPD XMM0, XMM1        ; 2개 double 병렬 덧셈                     │
│  VADDPS YMM0, YMM1, YMM2 ; 8개 float (AVX)                         │
│  VADDPS ZMM0, ZMM1, ZMM2 ; 16개 float (AVX-512)                    │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  x86 가변 길이 인코딩 예시:                                         │ │
│  │  ────────────────────────────────                                  │ │
│  │  RET               ; C3           (1바이트)                        │ │
│  │  INC EAX           ; 40           (1바이트)                        │ │
│  │  ADD EAX, 1        ; 83 C0 01     (3바이트)                        │ │
│  │  MOV EAX, [EBX]    ; 8B 03        (2바이트)                        │ │
│  │  MOV RAX, [RBX+RCX*4+100]  ; 48 8B 44 8B 64  (5바이트)           │ │
│  │  MOV RAX, 0x123456789ABCDEF0 ; 48 B8 F0 DE BC 9A 78 56 34 12     │ │
│  │                          ; (10바이트)                              │ │
│  │                                                               │ │
│  │  → 1바이트부터 15바이트까지                                         │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. x86 × 타 과목 융합

### 과목 융합 분석

```

│  【보안】                                                            │
│  • NX bit: 실행 방지                                                 │
│  • ASLR: 주소 무작위화                                                │
│  • SMEP/SMAP: 커널 보호                                              │

│  • 클럭: 3~6 GHz                                                     │

│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  x86 현황:                                                           │ │
│  │  ────────                                                             │ │
│  │  • 데스크톱: ~90% 점유                                              │ │
│  │  • 서버: ~90% 점유                                                  │ │
│  │  • 노트북: ~70% 점유 (ARM 증가)                                      │ │
│  │  • 모바일: ~0% (ARM 지배)                                            │ │
│  │  • 슈퍼컴퓨터: ~50% (ARM 증가)                                       │ │
│  │                                                               │ │
│  │  ARM 위협: Apple M1/M2, AWS Graviton, Ampere                      │ │
│  └───────────────────────────────────────────────────────────────┘ │
```

### 핵심 요약

```

```
