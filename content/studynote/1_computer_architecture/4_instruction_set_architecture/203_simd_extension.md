# 203. SIMD 명령어 확장 (SIMD Extensions)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단일 명령어로 다중 데이터 동시 처리
> 2. **가치**: 이미지, 오디오, 영상, AI 처리 4~16배 가속
> 3. **융합**: SSE/AVX(x86), NEON(ARM), V(RISC-V), GPU

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

SIMD(Single Instruction Multiple Data) 확장은 **하나의 명령어로 여러 데이터를 동시에 처리**하는 병렬 처리 기술이다. 이미지, 오디오, 비디오, AI 등 데이터 병렬성이 높은 작업에서 큰 성능 향상을 제공한다.

### 💡 비유: 8인분 한 번에 요리

SIMD는 **8인분을 한 번에 요리하는 큰 팬**과 같다. 일반 팬(스칼라)은 한 번에 1인분만 요리한다. SIMD 팬은 8인분을 동시에 요리한다. 같은 시간에 8배 많은 결과를 얻는다.

### SIMD 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                SIMD 구조                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【SIMD 원리】                                                       │
│  ──────────────────                                                  │
│                                                                     │
│  스칼라 (일반):                                                      │
│  ┌───┐     ┌───┐     ┌───┐     ┌───┐                              │
│  │ A │     │ B │     │ C │     │ D │                              │
│  └─┬─┘     └─┬─┘     └─┬─┘     └─┬─┘                              │
│    │ ADD     │ ADD     │ ADD     │ ADD                              │
│  ┌─▼─┐     ┌─▼─┐     ┌─▼─┐     ┌─▼─┐                              │
│  │ E │     │ F │     │ G │     │ H │                              │
│  └───┘     └───┘     └───┘     └───┘                              │
│  → 4번의 ADD 명령어 필요                                             │
│                                                                     │
│  SIMD:                                                               │
│  ┌───┬───┬───┬───┐                                                 │
│  │ A │ B │ C │ D │                                                 │
│  └─┬─┴─┬─┴─┬─┴─┬─┘                                                 │
│    │   │   │   │                                                    │
│    └─ADD──┴───┴──┘ (1번의 SIMD ADD)                                 │
│  ┌─▼─┬─▼─┬─▼─┬─▼─┐                                                 │
│  │ E │ F │ G │ H │                                                 │
│  └───┴───┴───┴───┘                                                 │
│  → 1번의 SIMD ADD로 4개 동시 처리                                    │
│                                                                     │
│  【SIMD 레지스터 크기】                                              │
│  ──────────────────────────                                         │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  레지스터    비트    float32    int8                            │ │
│  │  ─────────  ─────  ─────────  ────────                          │ │
│  │  MMX        64     2          8                                 │ │
│  │  XMM (SSE)  128    4          16                                │ │
│  │  YMM (AVX)  256    8          32                                │ │
│  │  ZMM (AVX-512) 512   16         64                               │ │
│  │  SVE         128~2048  가변    가변                              │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【SIMD 확장 비교】                                                  │
│  ──────────────────────────                                         │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  아키텍처   확장        레지스터    크기                          │ │
│  │  ─────────  ──────────  ────────  ───────                       │ │
│  │  x86       MMX         MM0-7     64비트                         │ │
│  │  x86       SSE/SSE2-4  XMM0-15   128비트                        │ │
│  │  x86       AVX/AVX2    YMM0-15   256비트                        │ │
│  │  x86       AVX-512     ZMM0-31   512비트                        │ │
│  │  ARM       NEON        Q0-31     128비트                        │ │
│  │  ARM       SVE/SVE2    Z0-31     128~2048비트                   │ │
│  │  RISC-V    V           v0-31     가변 (VLEN)                    │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  SIMD 장점:                                                        │ │
│  │  ──────────                                                        │ │
│  │  1. 데이터 병렬성 활용                                              │ │
│  │  2. 전력 효율: 동일 작업에 적은 명령어                              │ │
│  │  3. 메모리 대역폭 활용                                              │ │
│  │  4. AI/ML 처리 가속                                                │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. SIMD 명령어 상세

### 명령어 상세

```
┌─────────────────────────────────────────────────────────────────────┐
│                SIMD 명령어 상세                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【x86 SSE/AVX】                                                     │
│  ──────────────────                                                  │
│  ; 4개 float 덧셈 (SSE)                                             │
│  MOVAPS XMM0, [a]         ; 4개 float 로드                         │
│  MOVAPS XMM1, [b]                                                  │
│  ADDPS XMM0, XMM1         ; 4개 동시 덧셈                           │
│  MOVAPS [c], XMM0         ; 결과 저장                               │
│                                                                     │
│  ; 8개 float 덧셈 (AVX)                                             │
│  VMOVAPS YMM0, [a]        ; 8개 float 로드                         │
│  VMOVAPS YMM1, [b]                                                  │
│  VADDPS YMM0, YMM0, YMM1  ; 8개 동시 덧셈                           │
│  VMOVAPS [c], YMM0         ; 결과 저장                               │
│                                                                     │
│  ; 16개 float 덧셈 (AVX-512)                                        │
│  VMOVAPS ZMM0, [a]                                                  │
│  VMOVAPS ZMM1, [b]                                                  │
│  VADDPS ZMM0, ZMM0, ZMM1                                            │
│  VMOVAPS [c], ZMM0                                                  │
│                                                                     │
│  【ARM NEON】                                                        │
│  ──────────────────                                                  │
│  ; 4개 float 덧셈                                                   │
│  VLD1.32 {Q0}, [R0]        ; 4개 float 로드                         │
│  VLD1.32 {Q1}, [R1]                                                 │
│  VADD.F32 Q0, Q0, Q1       ; 4개 동시 덧셈                           │
│  VST1.32 {Q0}, [R2]        ; 결과 저장                               │
│                                                                     │
│  ; 8개 int8 덧셈                                                    │
│  VLD1.8 {D0}, [R0]         ; 8개 byte 로드                          │
│  VLD1.8 {D1}, [R1]                                                  │
│  VADD.I8 D0, D0, D1        ; 8개 동시 덧셈                           │
│  VST1.8 {D0}, [R2]                                                  │
│                                                                     │
│  【RISC-V V 확장】                                                   │
│  ──────────────────────                                              │
│  ; 가변 길이 벡터 덧셈                                               │
│  vle32.v v0, (a0)          ; 벡터 로드                               │
│  vle32.v v1, (a1)                                                   │
│  vadd.vv v2, v0, v1        ; 벡터 덧셈                               │
│  vse32.v v2, (a2)          ; 벡터 저장                               │
│  ; VLEN에 따라 자동으로 처리 개수 결정                               │
│                                                                     │
│  【주요 SIMD 연산】                                                  │
│  ──────────────────                                                  │
│  • ADD/ADDPS: 덧셈                                                  │
│  • SUB/SUBPS: 뺄셈                                                  │
│  • MUL/MULPS: 곱셈                                                  │
│  • DIV/DIVPS: 나눗셈                                                │
│  • FMADDPS: 융합 곱셈-덧셈 (a*b+c)                                  │
│  • SQRT/SQRTPS: 제곱근                                              │
│  • MIN/MAX: 최소/최대                                               │
│  • BLENDVPS: 조건부 혼합                                             │
│  • PERMUTE: 재배열                                                   │
│  • GATHER/SCATTER: 불연속 메모리                                    │
│  • REDUCE: 리덕션 (합, 최대 등)                                      │
│                                                                     │
│  【SIMD 인트린직 (C/C++)】                                           │
│  ─────────────────────────────                                      │
│  #include <immintrin.h>                                             │
│                                                                     │
│  // AVX 8개 float 덧셈                                              │
│  __m256 a = _mm256_load_ps(ptr_a);                                  │
│  __m256 b = _mm256_load_ps(ptr_b);                                  │
│  __m256 c = _mm256_add_ps(a, b);                                    │
│  _mm256_store_ps(ptr_c, c);                                         │
│                                                                     │
│  // AVX-512 16개 float 덧셈                                         │
│  __m512 a = _mm512_load_ps(ptr_a);                                  │
│  __m512 b = _mm512_load_ps(ptr_b);                                  │
│  __m512 c = _mm512_add_ps(a, b);                                    │
│  _mm512_store_ps(ptr_c, c);                                         │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  SIMD 활용 분야:                                                   │ │
│  │  ──────────────────                                                │ │
│  │  • 이미지 처리: 픽셀 연산 (밝기, 필터)                             │ │
│  │  • 오디오: DSP, 믹싱, 효과                                        │ │
│  │  • 비디오: 인코딩/디코딩, 스케일링                                 │ │
│  │  • 3D 그래픽: 정점 변환, 라이트                                    │ │
│  │  • 암호화: AES, 해시                                              │ │
│  │  • AI/ML: 추론, 학습 (INT8/FP16/BF16)                            │ │
│  │  • 과학 연산: 행렬, FFT                                           │ │
│  │  • 문자열: 검색, 비교                                              │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### SIMD 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                SIMD 분석                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【벡터 덧셈 성능 비교】                                             │
│  ──────────────────────────                                         │
│  // 스칼라 버전                                                      │
│  void add_scalar(float* a, float* b, float* c, int n) {             │
│      for (int i = 0; i < n; i++)                                    │
│          c[i] = a[i] + b[i];                                        │
│  }                                                                  │
│  // 1억 개: 250ms                                                    │
│                                                                     │
│  // AVX2 버전                                                        │
│  void add_avx2(float* a, float* b, float* c, int n) {               │
│      for (int i = 0; i < n; i += 8) {                               │
│          __m256 va = _mm256_load_ps(&a[i]);                         │
│          __m256 vb = _mm256_load_ps(&b[i]);                         │
│          __m256 vc = _mm256_add_ps(va, vb);                         │
│          _mm256_store_ps(&c[i], vc);                                │
│      }                                                              │
│  }                                                                  │
│  // 1억 개: 35ms (7배 향상)                                          │
│                                                                     │
│  【자동 벡터화 확인】                                                │
│  ──────────────────────────                                         │
│  $ gcc -O3 -march=native -fopt-info-vec prog.c                      │
│  prog.c:5:3: note: loop vectorized                                  │
│  prog.c:5:3: note: loop versioned for vectorization                 │
│                                                                     │
│  【OpenMP SIMD】                                                     │
│  ──────────────────                                                  │
│  #pragma omp simd                                                   │
│  for (int i = 0; i < n; i++)                                        │
│      c[i] = a[i] + b[i];                                            │
│  // 컴파일러가 자동으로 SIMD 활용                                    │
│                                                                     │
│  【SIMD 성능 측정】                                                  │
│  ──────────────────────────                                         │
│  $ perf stat -e cycles,instructions ./simd_prog                     │
│  cycles: 1,000,000,000                                              │
│  instructions: 500,000,000                                          │
│  IPC: 0.50                                                          │
│                                                                     │
│  $ perf stat -e simd_fp_256 ./simd_prog                             │
│  SIMD FP 256-bit: 100,000,000                                       │
│  // 256비트 SIMD 연산 횟수                                           │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  SIMD 최적화 팁:                                                    │ │
│  │  ──────────────────                                                │ │
│  │  1. 데이터 정렬: 16/32/64바이트 경계                               │ │
│  │  2. 연속 메모리: 캐시 친화적 접근                                  │ │
│  │  3. 루프 언롤링: 분기 오버헤드 감소                                │ │
│  │  4. FMA 활용: 곱셈+덧셈 1클럭                                     │ │
│  │  5. 프리페치: 메모리 지연 숨김                                     │ │
│  │  6. 마스크 활용: 조건부 연산                                       │ │
│  │  7. GATHER/SCATTER 최소화                                          │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• SIMD: 단일 명령어로 다중 데이터 처리
• 레지스터: XMM(128), YMM(256), ZMM(512)
• x86: MMX→SSE→AVX→AVX2→AVX-512
• ARM: NEON(128), SVE(가변 128~2048)
• RISC-V: V 확장 (가변 VLEN)
• 활용: 이미지, 오디오, 비디오, AI, 그래픽
• 성능: 4~32배 향상 (이론적)
• 최적화: 정렬, 연속 메모리, FMA, 프리페치
```
