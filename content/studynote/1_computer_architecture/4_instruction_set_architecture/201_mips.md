# 201. MIPS (Microprocessor without Interlocked Pipeline Stages)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 1985년 Stanford 개발, 교육용 표준 RISC ISA
> 2. **가치**: 파이프라인 단순화, 교과서 예제, SGI/임베디드
> 3. **융합**: 라우터, 게임기, 교육, MIPS Open

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

MIPS(Microprocessor without Interlocked Pipeline Stages)는 **1985년 Stanford의 John Hennessy가 개발한 RISC ISA**로, 파이프라인 효율성을 위해 하드웨어 인터락을 최소화하고 컴파일러가 해저드를 처리하도록 설계되었다. 컴퓨터 구조 교육의 표준이다.

### 💡 비유: 수학 교과서의 예제

MIPS는 **수학 교과서의 예제**와 같다. 실제 복잡한 문제보다 깔끔하고 이해하기 쉽다. ARM은 실제 시장에서 쓰이는 복잡한 버전이다. MIPS로 기본을 배우면 다른 ISA도 쉽게 이해할 수 있다.

## Ⅱ. MIPS 명령어 집합

### 명령어 상세

```
┌─────────────────────────────────────────────────────────────────────┐
│                MIPS 명령어 집합                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【R-type (레지스터)】                                               │
│  ──────────────────────                                              │
│  add $t0, $t1, $t2    ; $t0 = $t1 + $t2 (오버플로우 트랩)            │
│  addu $t0, $t1, $t2   ; $t0 = $t1 + $t2 (오버플로우 무시)           │
│  sub $t0, $t1, $t2    ; $t0 = $t1 - $t2                             │
│  subu $t0, $t1, $t2   ; $t0 = $t1 - $t2 (무부호)                    │
│  and $t0, $t1, $t2    ; $t0 = $t1 & $t2                             │
│  or $t0, $t1, $t2     ; $t0 = $t1 | $t2                             │
│  xor $t0, $t1, $t2    ; $t0 = $t1 ^ $t2                             │
│  nor $t0, $t1, $t2    ; $t0 = ~($t1 | $t2)                          │
│  slt $t0, $t1, $t2    ; $t0 = ($t1 < $t2) ? 1 : 0                   │
│  sltu $t0, $t1, $t2   ; $t0 = ($t1 < $t2) ? 1 : 0 (무부호)          │
│  sll $t0, $t1, shamt  ; $t0 = $t1 << shamt                          │
│  srl $t0, $t1, shamt  ; $t0 = $t1 >> shamt (논리)                   │
│  sra $t0, $t1, shamt  ; $t0 = $t1 >> shamt (산술)                   │
│  sllv $t0, $t1, $t2   ; $t0 = $t1 << $t2                            │
│  srlv $t0, $t1, $t2   ; $t0 = $t1 >> $t2 (논리)                     │
│  srav $t0, $t1, $t2   ; $t0 = $t1 >> $t2 (산술)                     │
│  jr $ra               ; PC = $ra                                    │
│  jalr $t0             ; $ra = PC+8; PC = $t0                        │
│                                                                     │
│  【I-type (즉시값)】                                                 │
│  ──────────────────                                                  │
│  addi $t0, $t1, imm   ; $t0 = $t1 + imm                             │
│  addiu $t0, $t1, imm  ; $t0 = $t1 + imm (무부호)                    │
│  andi $t0, $t1, imm   ; $t0 = $t1 & imm                             │
│  ori $t0, $t1, imm    ; $t0 = $t1 | imm                             │
│  xori $t0, $t1, imm   ; $t0 = $t1 ^ imm                             │
│  slti $t0, $t1, imm   ; $t0 = ($t1 < imm) ? 1 : 0                   │
│  sltiu $t0, $t1, imm  ; $t0 = ($t1 < imm) ? 1 : 0 (무부호)          │
│  lui $t0, imm         ; $t0 = imm << 16                             │
│                                                                     │
│  【Load/Store】                                                      │
│  ──────────────────                                                  │
│  lw $t0, offset($t1)  ; $t0 = Mem[$t1 + offset]                     │
│  lh $t0, offset($t1)  ; halfword (부호 확장)                        │
│  lhu $t0, offset($t1) ; halfword (0 확장)                           │
│  lb $t0, offset($t1)  ; byte (부호 확장)                            │
│  lbu $t0, offset($t1) ; byte (0 확장)                               │
│  sw $t0, offset($t1)  ; Mem[$t1 + offset] = $t0                     │
│  sh $t0, offset($t1)  ; halfword                                    │
│  sb $t0, offset($t1)  ; byte                                        │
│                                                                     │
│  【분기】                                                            │
│  ──────────────────                                                  │
│  beq $t0, $t1, label  ; if ($t0 == $t1) goto label                  │
│  bne $t0, $t1, label  ; if ($t0 != $t1) goto label                  │
│  bgtz $t0, label      ; if ($t0 > 0) goto label                     │
│  blez $t0, label      ; if ($t0 <= 0) goto label                    │
│  bgez $t0, label      ; if ($t0 >= 0) goto label                    │
│  bltz $t0, label      ; if ($t0 < 0) goto label                     │
│  j label              ; PC = label                                  │
│  jal label            ; $ra = PC+8; PC = label                      │
│                                                                     │
│  【지연 슬롯 (Delayed Branch)】                                      │
│  ──────────────────────────────────                                 │
│  beq $t0, $t1, target ; 분기                                        │
│  addi $t2, $t2, 1     ; ← 항상 실행! (지연 슬롯)                     │
│  ; 분기 여부와 관계없이 addi 실행됨                                  │
│  ; 컴파일러가 유용한 명령어로 채움                                   │
│                                                                     │
│  【곱셈/나눗셈 (HI/LO 사용)】                                        │
│  ────────────────────────────────                                   │
│  mult $t0, $t1        ; HI:LO = $t0 * $t1                           │
│  multu $t0, $t1       ; HI:LO = $t0 * $t1 (무부호)                  │
│  div $t0, $t1         ; LO = $t0 / $t1, HI = $t0 % $t1              │
│  divu $t0, $t1        ; 무부호                                       │
│  mfhi $t0             ; $t0 = HI                                    │
│  mflo $t0             ; $t0 = LO                                    │
│                                                                     │
│  【MIPS32 곱셈 (결과가 레지스터로)】                                  │
│  ──────────────────────────────────                                 │
│  mul $t0, $t1, $t2    ; $t0 = $t1 * $t2 (하위 32비트)               │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  MIPS 명령어 형식 (32비트):                                         │ │
│  │  ────────────────────────────                                      │ │
│  │  R-type: [opcode][rs][rt][rd][shamt][funct]                     │ │
│  │          6      5   5   5    5      6                            │ │
│  │  I-type: [opcode][rs][rt][imm]                                   │ │
│  │          6      5   5   16                                        │ │
│  │  J-type: [opcode][target]                                        │ │
│  │          6      26                                                │ │
│  │                                                               │ │
│  │  → 규칙적인 인코딩                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### MIPS 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                MIPS 분석                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【팩토리얼 함수 (MIPS)】                                            │
│  ──────────────────────────                                         │
│  # int factorial(int n)                                             │
│  factorial:                                                         │
│      addi $sp, $sp, -8      # 스택 할당                              │
│      sw $ra, 4($sp)         # 반환 주소 저장                         │
│      sw $s0, 0($sp)         # $s0 저장                               │
│      move $s0, $a0          # n = $a0                                │
│      slti $t0, $s0, 2       # n < 2?                                 │
│      beq $t0, $zero, recur  # no → 재귀                             │
│      nop                   # 지연 슬롯                               │
│      addi $v0, $zero, 1     # return 1                              │
│      j done                                                         │
│      nop                   # 지연 슬롯                               │
│  recur:                                                             │
│      addi $a0, $s0, -1      # n - 1                                  │
│      jal factorial         # 재귀 호출                              │
│      nop                   # 지연 슬롯                               │
│      mul $v0, $s0, $v0     # n * factorial(n-1)                     │
│  done:                                                              │
│      lw $ra, 4($sp)         # 반환 주소 복원                         │
│      lw $s0, 0($sp)         # $s0 복원                               │
│      addi $sp, $sp, 8       # 스택 복원                              │
│      jr $ra                                                         │
│      nop                   # 지연 슬롯                               │
│                                                                     │
│  【MARS 시뮬레이터】                                                 │
│  ──────────────────────                                              │
│  # MARS (MIPS Assembler and Runtime Simulator)                      │
│  # 교육용 MIPS 시뮬레이터                                            │
│  # http://courses.missouristate.edu/kenvollmar/mars/                │
│                                                                     │
│  【SPIM 시뮬레이터】                                                 │
│  ──────────────────────                                              │
│  $ spim -file program.asm                                           │
│  (spim) run                                                         │
│  (spim) print $t0                                                   │
│  (spim) step                                                        │
│                                                                     │
│  【지연 슬롯 최적화 예시】                                           │
│  ──────────────────────────────                                     │
│  # 비최적                                                            │
│  beq $t0, $zero, skip                                               │
│  nop                       # 낭비                                   │
│  add $t1, $t2, $t3                                                  │
│                                                                     │
│  # 최적화 (지연 슬롯 활용)                                           │
│  beq $t0, $zero, skip                                               │
│  add $t1, $t2, $t3         # 유용한 명령어로 채움                    │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  MIPS vs RISC-V:                                                   │ │
│  │  ──────────────────                                                │ │
│  │  특성           MIPS          RISC-V                              │ │
│  │  ───────────   ───────────   ────────────                         │ │
│  │  지연 슬롯      있음          없음                                 │ │
│  │  라이선스       제한적         오픈소스                             │ │
│  │  HI/LO         있음          없음 (mul rd 사용)                   │ │
│  │  조건부 분기    제한적         다양                                │ │
│  │  현재          교육용         산업/교육                            │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• MIPS: 1985년 Stanford 개발 RISC ISA
• 특징: 고정 길이, Load/Store, 지연 슬롯
• 레지스터: $0-$31 (32개) + HI/LO
• 교육: Patterson & Hennessy 교재 표준
• 역사: SGI, PlayStation, Nintendo 64
• 현재: 교육용, 임베디드 (감소 추세)
• 미래: RISC-V로 전환
```
