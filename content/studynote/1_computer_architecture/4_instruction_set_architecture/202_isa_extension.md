# 202. 명령어 집합 확장 (ISA Extensions)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 기본 ISA에 추가 기능을 제공하는 명령어 확장
> 2. **가치**: 성능 향상, 특수 목적 최적화, 선택적 구현
> 3. **융합**: SSE/AVX(x86), NEON(ARM), V(RISC-V), 가속기

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

명령어 집합 확장(ISA Extension)은 **기본 ISA에 추가 기능을 제공하는 명령어 세트**로, 특정 작업(SIMD, 암호화, 부동소수점 등)을 가속화한다. 선택적 구현으로 비용-성능 균형을 맞출 수 있다.

### 💡 비유: 스마트폰 앱 추가 설치

ISA 확장은 **스마트폰 앱 추가 설치**와 같다. 기본 ISA는 기본 앱(전화, 문자)이다. 확장은 필요한 앱(카메라, 게임)을 추가로 설치하는 것이다. 기본 기능 + 원하는 추가 기능으로 구성한다.

## Ⅱ. 주요 확장 상세

### 확장 상세

```
┌─────────────────────────────────────────────────────────────────────┐
│                주요 확장 상세                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【암호화 확장】                                                     │
│  ──────────────────                                                  │
│  • AES-NI (x86): AES 암호화 10배+ 가속                              │
│  • SHA (x86): SHA-1/256/512 하드웨어                                │
│  • ARMv8 Crypto: AES, SHA-1, SHA-256                               │
│  • RISC-V K: 암호화 확장                                            │
│  • PCLMULQDQ: 캐리리스 곱셈 (CRC, GCM)                              │
│                                                                     │
│  【비트 조작 확장】                                                  │
│  ──────────────────────────                                         │
│  • POPCNT: 1비트 개수 세기                                          │
│  • LZCNT/TZCNT: 선행/후행 0 개수                                    │
│  • BMI1/BMI2: 비트 조작 (ANDN, BLSI, BLSR, etc.)                    │
│  • RISC-V B: CLZ, CTZ, POPCNT, ROT                                 │
│                                                                     │
│  【부동소수점 확장】                                                 │
│  ──────────────────────────                                         │
│  • FMA: 융합 곱셈-덧셈 (a*b+c를 한 번에)                            │
│  • FP16: 반정밀도 (16비트)                                          │
│  • BF16: BFloat16 (AI용)                                           │
│  • RISC-V F/D/Q: FP32/FP64/FP128                                   │
│                                                                     │
│  【가상화 확장】                                                     │
│  ──────────────────                                                  │
│  • Intel VT-x: 하드웨어 가상화                                      │
│  • AMD-V: AMD 가상화                                                │
│  • ARM H Extension: 하이퍼바이저                                    │
│  • RISC-V H: 하이퍼바이저 확장                                      │
│                                                                     │
│  【AI/ML 확장】                                                      │
│  ──────────────────                                                  │
│  • Intel VNNI: INT8 딥러닝 가속                                     │
│  • Intel AMX: 행렬 연산 (AI)                                        │
│  • ARM i8mm: INT8 행렬 곱셈                                         │
│  • ARM BF16: BFloat16 딥러닝                                        │
│  • ARM DotProd: 도트 프로덕트                                       │
│  • ARM SVE/SVE2: 스케일러블 벡터                                    │
│                                                                     │
│  【보안 확장】                                                       │
│  ──────────────────                                                  │
│  • Intel SGX: Enclave (보호 메모리)                                 │
│  • Intel TDX: 신뢰 도메인                                           │
│  • ARM TrustZone: 보안 세계                                         │
│  • ARM PAC: 포인터 인증                                             │
│  • ARM MTE: 메모리 태깅                                             │
│  • ARM BTI: 분기 타겟 식별                                          │
│  • RISC-V PMP: 물리 메모리 보호                                     │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  CPUID로 확장 확인 (x86):                                          │ │
│  │  ────────────────────────────                                      │ │
│  │  $ cat /proc/cpuinfo | grep flags                                 │ │
│  │  flags: fpu vme de pse tsc msr pae mce cx8 apic sep              │ │
│  │        mtrr pge mca cmov pat pse36 clflush mmx fxsr sse sse2     │ │
│  │        ss ht syscall nx pdpe1gb rdtscp lm constant_tsc           │ │
│  │        arch_perfmon nopl xtopology tsc_reliable nonstop_tsc      │ │
│  │        cpuid pni pclmulqdq ssse3 fma cx16 pcid sse4_1 sse4_2     │ │
│  │        x2apic movbe popcnt tsc_deadline_timer aes xsave avx      │ │
│  │        f16c rdrand hypervisor lahf_lm abm 3dnowprefetch          │ │
│  │        cpuid_fault invpcid_single pti ssbd ibrs ibpb stibp       │ │
│  │        fsgsbase tsc_adjust bmi1 avx2 smep bmi2 invpcid avx512f   │ │
│  │                                                               │ │
│  │  → sse, sse2, avx, avx2, avx512f, aes 등 확인                    │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

```
│  • VNNI: INT8 추론 가속                                             │
│  • BF16: 딥러닝 훈련                                                │
│  • AMX: 행렬 곱셈 가속                                              │
│  • SVE: 가변 길이 벡터                                              │
│                                                                     │

│  │  확장 활용 예시:                                                   │ │

│  │                                                               │ │
│  │  // 8개 float 동시 덧셈                                         │ │
│  │  __m256 a = _mm256_load_ps(ptr_a);                              │ │
│  │  __m256 b = _mm256_load_ps(ptr_b);                              │ │
│  │  __m256 c = _mm256_add_ps(a, b);                                │ │
│  │  _mm256_store_ps(ptr_c, c);                                     │ │

│  │  // 8배 빠른 연산!                                              │ │
```

## Ⅳ. 실무 적용

### ISA 확장 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                ISA 확장 분석                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【AES-NI 성능 비교】                                                │
│  ──────────────────────────                                         │
│  # AES-256-CBC 암호화 (1GB)                                         │
│  # 소프트웨어: 12초                                                  │
│  # AES-NI: 0.8초                                                    │
│  # → 15배 향상                                                       │
│                                                                     │
│  $ openssl speed aes-256-cbc                                        │
│  # AES-NI 활성화 시 처리량 확인                                      │
│                                                                     │
│  【SIMD 벡터화 비교】                                                │
│  ──────────────────────────                                         │
│  // 스칼라 (1개씩)                                                   │
│  for (int i = 0; i < N; i++)                                        │
│      c[i] = a[i] + b[i];                                            │
│                                                                     │
│  // AVX2 (8개씩)                                                     │
│  for (int i = 0; i < N; i += 8) {                                   │
│      __m256 va = _mm256_load_ps(&a[i]);                             │
│      __m256 vb = _mm256_load_ps(&b[i]);                             │
│      __m256 vc = _mm256_add_ps(va, vb);                             │
│      _mm256_store_ps(&c[i], vc);                                    │
│  }                                                                  │
│  // → 8배 이론적 향상                                                │
│                                                                     │
│  【컴파일러 플래그】                                                  │
│  ──────────────────                                                  │
│  $ gcc -march=native -O3 prog.c                                     │
│  # 현재 CPU의 모든 확장 활용                                         │
│                                                                     │
│  $ gcc -mavx2 -mfma prog.c                                          │
│  # 특정 확장만 활성화                                                │
│                                                                     │
│  【런타임 CPUID 체크】                                                │
│  ──────────────────────────                                         │
│  #include <cpuid.h>                                                 │
│                                                                     │
│  void check_avx2() {                                                │
│      unsigned int eax, ebx, ecx, edx;                               │
│      __cpuid(7, eax, ebx, ecx, edx);                                │
│      if (ebx & (1 << 5)) {                                          │
│          // AVX2 지원                                                │
│          use_avx2_version();                                        │
│      } else {                                                       │
│          use_scalar_version();                                      │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  확장 확인 명령어:                                                 │ │
│  │  ──────────────────                                                │ │
│  │  $ lscpu | grep -i flags                                          │ │
│  │  $ cat /proc/cpuinfo | grep flags                                 │ │
│  │  $ cpuid -1 | grep -i feature                                     │ │
│  │                                                               │ │
│  │  주요 플래그:                                                       │ │
│  │  • sse sse2 sse3 ssse3 sse4_1 sse4_2                             │ │
│  │  • avx avx2 avx512f avx512bw                                     │ │
│  │  • aes fma bmi1 bmi2 popcnt                                     │ │
│  │  • vmx (VT-x) svm (AMD-V)                                        │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```
