# 볼티지 글리칭 (Voltage Glitching)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU 전원 공급에 순간적인 전압 변화(글리치)를 가해 연산 오류를 유발하고 보안 기능을 우회하는 물리적 공격
> 2. **가치**: 하드웨어 보안 이해, 전압 기반 공격, 방어 설계, 보안 평가
> 3. **융합**: 클럭 글리칭, EMFI, FA(Fault Attack)와 통합된 물리적 공격

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

볼티지 글리칭(Voltage Glitching)은 CPU 전원 공급에 순간적인 전압 변화(글리치)를 가해 연산 오류를 유발하고 보안 기능을 우회하는 물리적 공격입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    볼티지 글리칭 (Voltage Glitching)                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전압 글리치 파형                                   │ │
│   │                                                              │ │
│   │   전압 (V)                                                      │ │
│   │      ▲                                                        │ │
│   │   1.2V ────┬────────────────────────────────────             │ │
│   │            │                                                 │ │
│   │   1.0V ────┼────────────────────────────────────             │ │
│   │            │        ┌──┐                                     │ │
│   │   0.8V ────┼────────┤  ├────────────────────                │ │
│   │            │        │  │  글리치                             │ │
│   │   0.5V ────┼────────┤  ├────────────────────                │ │
│   │            │        └──┘                                     │ │
│   │            │             ↑                                   │ │
│   │            │        연산 오류 발생 시점                       │ │
│   │      └─────┼────────────────────────────────────────────→ 시간│ │
│   │            │                                                 │ │
│   │   효과:                                                      │ │
│   │   - 명령어 스킵                                               │ │
│   │   - 조건 분기 우회                                            │ │
│   │   - 암호화 연산 오류                                          │ │
│   │   - 보안 체크 우회                                            │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              공격 시나리오                                     │ │
│   │                                                              │ │
│   │   if (password_correct) {                                    │ │
│   │       grant_access();      ← 글리치로 이 분기 우회            │ │
│   │   } else {                                                   │ │
│   │       deny_access();       ← 글리치로 여기 실행 유도          │ │
│   │   }                                                          │ │
│   │                                                              │ │
│   │   글리치 타이밍에 맞춰 분기 조건 오염                         │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 볼티지 글리칭은 전압 순간 강하로 연산 오류를 유발합니다.

**💡 비유**: 볼티지 글리칭은 정전 순간을 이용하는 것과 같습니다. 잠깐 불이 나가면 문이 풀릴 수 있습니다.

**📢 섹션 요약 비유**: 볼티지 글리칭은 정전 같아요. 순간 전압 강하예요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **글리치 장비** | 전압 제어 | FET 스위치 | 스위치 |
| **타이밍 제어** | 정밀 동기 | ns 단위 | 시계 |
| **대상 회로** | 오류 발생 | CPU/VCC | 표적 |
| **모니터링** | 결과 확인 | 통신 인터페이스 | 관찰 |

**글리치 파라미터**

| 파라미터 | 범위 | 효과 |
|:---|:---:|:---|
| **강하 깊이** | 10-90% | 오류 유발 |
| **지속 시간** | 1ns-1μs | 오류 유형 |
| **타이밍** | 명령어 사이클 동기 | 공격 위치 |

**핵심 알고리즘**: 타이밍 분석 → 글리치 인가 → 오류 유발 → 결과 확인

**📢 섹션 요약 비유**: 타이밍이 핵심이에요. ns 단위로 맞춰요!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 볼티지 글리칭 vs 클럭 글리칭 vs EMFI**

| 비교 항목 | 볼티지 글리칭 | 클럭 글리칭 | EMFI |
|:---|:---:|:---:|:---:|
| **방법** | 전압 변화 | 클럭 변조 | 전자기 펄스 |
| **접근** | 전원선 | 클럭선 | 비접촉 |
| **정밀도** | 높음 | 높음 | 중간 |
| **장비** | 간단 | 간단 | 복잡 |

**📢 섹션 요약 비유**: 전압, 클럭, EMFI 다 방법이 달라요!
