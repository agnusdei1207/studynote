# 물리적 분해 분석 (Reverse Engineering)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 하드웨어를 물리적으로 분해하고 회로를 분석하여 설계 정보를 추출하는 공격 기법
> 2. **가치**: 지적 재산 보호, 보안 분석, 호환성 개발, 하드웨어 보안 이해
> 3. **융합**: 칩 사진, FIB, SEM, 물리적 보안과 통합된 분석 기법

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

물리적 분해 분석(Reverse Engineering)은 하드웨어를 물리적으로 분해하고 회로를 분석하여 설계 정보를 추출하는 공격 기법입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    물리적 분해 분석 (Reverse Engineering)              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              분해 분석 단계                                    │ │
│   │                                                              │ │
│   │   단계 1: 패키지 제거 (Decapsulation)                          │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                     │   │ │
│   │   │   ┌───────────────────────────┐                     │   │ │
│   │   │   │     패키지 (Plastic/Ceramic)  │                     │   │ │
│   │   │   │   ┌───────────────────┐   │                     │   │ │
│   │   │   │   │      Die (칩)       │   │                     │   │ │
│   │   │   │   └───────────────────┘   │                     │   │ │
│   │   │   └───────────────────────────┘                     │   │ │
│   │   │         ▼ 산 또는 레이저로 제거                        │   │ │
│   │   │   ┌───────────────────┐                              │   │ │
│   │   │   │      Die 노출       │                              │   │ │
│   │   │   └───────────────────┘                              │   │ │
│   │   │                                                     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   │   단계 2: 레이어 제거 (Delayering)                            │ │
│   │   - 연마 또는 에칭으로 금속층 제거                            │ │
│   │   - 각 층별 사진 촬영                                        │ │
│   │   - 금속1 → 유전체 → 금속2 → ...                            │ │
│   │                                                              │ │
│   │   단계 3: 이미징 (Imaging)                                    │ │
│   │   - 광학 현미경 (저배율)                                      │ │
│   │   - SEM (주사 전자 현미경)                                    │ │
│   │   - FIB (집속 이온 빔)                                        │ │
│   │                                                              │ │
│   │   단계 4: 회로 추출 (Circuit Extraction)                      │ │
│   │   - 이미지에서 회로도 복원                                    │ │
│   │   - 넷리스트 추출                                             │ │
│   │   - 기능 분석                                                 │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 분해 분석은 패키지 제거 → 레이어 제거 → 이미징 → 회로 추출 순서입니다.

**💡 비유**: 분해 분석은 시계를 분해하는 것과 같습니다. 부품을 보고 어떻게 작동하는지 알아냅니다.

**📢 섹션 요약 비유**: 분해 분석은 시계 분해 같아요. 부품을 봐요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **Decap** | 패키지 제거 | 산/레이저 | 케이스 열기 |
| **Delayer** | 층 제거 | 연마/에칭 | 껍질 벗기기 |
| **SEM** | 고배율 이미징 | 전자빔 | 돋보기 |
| **FIB** | 단면/수정 | 이온빔 | 칼 |

**장비별 용도**

| 장비 | 해상도 | 용도 |
|:---|:---:|:---|
| **광학 현미경** | ~1μm | 전체 구조 |
| **SEM** | ~10nm | 회로 패턴 |
| **FIB** | ~5nm | 단면, 회로 수정 |
| **TEM** | ~0.1nm | 트랜지스터 구조 |

**핵심 알고리즘**: 패키지 제거 → 층별 이미징 → 스티칭 → 회로 추출

**📢 섹션 요약 비유**: 층층이 벗겨서 사진 찍어요!
