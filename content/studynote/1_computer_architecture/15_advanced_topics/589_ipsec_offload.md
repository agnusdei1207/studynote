> **핵심 인사이트**
> 1. **IPsec (Internet Protocol Security)**은 패킷의 내용(Payload)뿐만 아니라 헤더까지 통째로 암호화하여(Tunnel Mode), 인터넷 위에서 우리 회사만의 안전한 비밀 통로(VPN)를 뚫어주는 강력한 네트워크 보안 프로토콜이다.
> 2. AES-256이나 SHA 암호화를 10Gbps 이상의 속도로 실시간으로 풀고 묶는 작업은 범용 CPU에게는 끔찍한 고문(Overhead)이며 통신 지연의 주범이 된다.
> 3. 이를 해결하기 위해 방화벽 장비나 서버 랜카드(SmartNIC) 내부에는 패킷이 통과하는 즉시 파이프라인에서 암호를 걸어버리는 **IPsec 전용 하드웨어 가속기**가 장착된다.



## Ⅰ. IPsec VPN의 강력함과 연산의 저주

코로나19 팬데믹 이후 수많은 직장인들이 재택근무를 하며 회사 망에 접속할 때 **VPN (가상 사설망)**을 썼습니다. 
이 VPN 터널을 뚫어주는 가장 표준적이고 강력한 기법이 **IPsec**입니다.

인터넷이라는 열린 고속도로를 달릴 때, IPsec은 내 데이터를 꽁꽁 싸매서 남들이 볼 수 없는 까만 장갑차(Tunnel Mode) 안에 넣고 달립니다.
* **ESP (Encapsulating Security Payload)**: 데이터를 AES 같은 강력한 암호로 암호화합니다.
* **AH (Authentication Header)**: 데이터가 가는 도중 위조되지 않았음을 증명하기 위해 거대한 해시(SHA-256) 도장을 찍습니다.

**문제**: 1초에 천만 개의 패킷이 장갑차에 타고 내리려면, 문을 열고 닫는(암호화/복호화) 속도가 엄청나게 빨라야 합니다. 범용 CPU 코어를 쓰면 트래픽이 1Gbps만 넘어가도 버티지 못하고 네트워크 지연(Ping)이 미친 듯이 솟구칩니다.

> 📢 **섹션 요약 비유**: 물건을 보낼 때마다 3중 쇠사슬로 묶고 티타늄 자물쇠를 채워(IPsec) 택배를 보냅니다. 안전하긴 한데, 사장님(CPU)이 맨손으로 이 자물쇠를 일일이 채우고 풀다 보니 손가락이 다 까져서 하루에 택배 10개밖에 처리를 못 하는 상황입니다.



## Ⅱ. IPsec 하드웨어 오프로드의 두 가지 형태

하드웨어 가속은 위치에 따라 두 가지로 나뉩니다.

### 1. 룩어사이드 가속기 (Look-aside Accelerator)
* 메인보드의 칩셋이나 CPU 바로 옆에 붙어 있는 전용 암호화 보조 칩(예: Intel QAT)입니다.
* CPU가 패킷을 받으면, "어, 이거 IPsec 암호 걸린 거네? 야 가속기야, 너 이거 암호 좀 풀어서 다시 나한테 줘!"라고 옆으로 던집니다(Look-aside).
* 가속기가 빛의 속도로 풀어서 CPU에 돌려주면, CPU가 그제야 데이터를 읽고 처리합니다. PCIe 버스를 두 번 타야 하는 단점이 있습니다.

### 2. 인라인 (In-line) IPsec 오프로드 (최신 랜카드 기술)
* 아예 랜카드(NIC) 칩 내부에 암호화 엔진을 융합해버린 궁극의 형태입니다.
* 랜카드 포트로 암호화된 빛(신호)이 들어오자마자, 랜카드 칩 내부의 실리콘 파이프라인을 지나가면서 **물 흐르듯 실시간으로 암호가 풀려버립니다 (Bump-in-the-wire).**
* CPU 메모리에 도착했을 때는 이미 평문(Plain text) 상태이므로, CPU는 암호화의 존재조차 모르고 쾌적하게 100Gbps 속도로 데이터를 씹어 먹습니다.

### 인라인 오프로드 다이어그램 (ASCII)

```text
 [ 인터넷망 ] ──(AES 암호화된 패킷)──▶ ┌──────────── 랜카드 (NIC) ────────────┐
                                   │  ┌─────────┐   ┌────────────────┐        │
                                   │  │ 수신 부 │─▶│ 인라인 IPsec   │   │ (하드웨어
                                   │  │ (MAC)   │   │ 복호화 가속기  │   │  파이프라인)
                                   │  └─────────┘   └───────┬────────┘        │
                                   └────────────────────────│─────────────────┘
                                                            ▼ (평문 상태로 쏙 들어옴)
                                                       [ 메인 메모리 (RAM) ]
```

> 📢 **섹션 요약 비유**: 룩어사이드는 사장님이 자물쇠 풀기 힘드니 비서를 고용해서 자물쇠 달린 박스를 비서 책상으로 넘겼다가 돌려받는 것이고, 인라인 오프로드는 아예 택배 하차 레일 끝에 '자동 자물쇠 절단기'를 설치해서 물건이 레일을 타고 내려오자마자 상자가 확 벗겨지게 만든 혁신입니다.



## Ⅲ. 통신사와 5G/6G 환경의 핵심

이 기술은 특히 통신사(SKT, KT 등)의 5G 기지국과 코어망(EPC)에서 절대적으로 중요합니다.
스마트폰에서 기지국으로 날아가는 모든 5G 전파는 100% IPsec으로 암호화되어야 한다는 표준 규격이 있기 때문입니다. 수만 명의 넷플릭스 4K 영상을 실시간으로 IPsec 암호화하여 쏴주기 위해 통신사 장비에는 반드시 이 거대한 IPsec 하드웨어 가속 칩이 내장되어 있습니다.
