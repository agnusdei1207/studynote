# 단락 보호 (SCP, Short Circuit Protection)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 출력 단자 간 단락(0Ω) 발생 시 즉시 전원을 차단하는 초고속 보호 회로
> 2. **가치**: 단락으로 인한 화재 방지, 장비 보호, 케이블 손상 방지, 인명 안전
> 3. **융합**: OCP, OVP, OTP, PSU 보호 회로와 통합된 초고속 전원 보호

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

단락 보호(SCP, Short Circuit Protection)는 출력 단자 간 단락(0Ω) 발생 시 즉시 전원을 차단하는 초고속 보호 회로입니다. 나노초(ns) 단위로 차단하여 화재를 방지합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    단락 보호 (SCP) 작동 원리                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   정상 상태                        단락 발생                         │
│   ┌─────────────┐                ┌─────────────┐                    │
│   │    Vout     │                │    Vout     │                    │
│   │     │       │                │     │ 0V   │ 단락             │
│   │     ▼       │                │     ▼       │                    │
│   │    Load     │                │  ┌──┴──┐   │                    │
│   │   (정상)    │                │  │단락 │   │                    │
│   │             │                │  └─────┘   │                    │
│   │  I_normal   │                │  I_short    │ (수백 앰페어)       │
│   │   ~50A      │                │  ~수천 A    │                    │
│   └─────────────┘                └─────────────┘                    │
│                                                                     │
│   SCP 작동:                                                         │
│   ┌────────────────────────────────────────────────────────────┐  │
│   │ 1. 전류 급증 감지 (di/dt > 임계값)                              │  │
│   │ 2. ~나노초 내 차단                                              │  │
│   │ 3. 출력 완전 차단                                                │  │
│   │ 4. 랫치 (수동 리셋 필요)                                         │  │
│   └────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 단락은 전류가 수천 암페어로 급증합니다. SCP는 나노초 단위로 즉시 차단합니다.

**💡 비유**: SCP는 자동차의 에어백과 같습니다. 충돌(단락) 시 즉시 작동합니다.

**📢 섹션 요약 비유**: SCP는 에어백 같아요. 사고(단락) 나면 즉시 터져요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **전류 변화율 센서** | di/dt 감지 | 빠른 응답 | 가속도계 |
| **비교기** | 급증 감지 | 고속 Op-Amp | 판단 |
| **차단 스위치** | 즉시 차단 | MOSFET | 밸브 |
| **랫치 회로** | 상태 유지 | 플립플롭 | 잠금 |

**SCP vs OCP 차이**

| 항목 | SCP | OCP |
|:---|:---:|:---:|
| **감지 방식** | di/dt (변화율) | I (절대값) |
| **응답 시간** | ~ns | ~μs |
| **용도** | 단락 | 과부하 |

**핵심 알고리즘**: di/dt > 임계값 → 즉시 차단 (~나노초)

**📢 섹션 요약 비유**: SCP는 OCP보다 1000배 빠릅니다. 단락은 매우 위험해요!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: SCP vs OCP vs OVP**

| 비교 항목 | SCP | OCP | OVP |
|:---|:---:|:---:|:---:|
| **감지 대상** | 전류 급증 | 전류 초과 | 전압 초과 |
| **응답 시간** | ~ns | ~μs | ~μs |
| **위험도** | 최고 | 높음 | 높음 |

**📢 섹션 요약 비유**: SCP > OCP > OVP 순서로 빠릅니다. 단락이 가장 위험해요!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데이터센터** → SCP 필수, 정기 테스트
**시나리오 2: 산업용** → SCP + 퓨즈 이중화
**시나리오 3: 가전** → 내장형 SCP

**도입 체크리스트**: 응답 시간 (<100ns), 랫치 방식, 정기 테스트

**📢 섹션 요약 비유**: SCP는 가장 빠른 보호입니다. 정기 테스트가 필요해요!
