### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **RISC-V 코어** | 보안 실행 | 오픈 소스 ISA | 직원 |
| **암호화 엔진** | 암호화 | AES, SHA, RSA | 금고 |
| **TRNG** | 난수 생성 | 물리적 엔트로피 | 주사위 |
| **키 저장소** | 키 보관 | 하드웨어 fuses | 열쇠함 |

**Titan 칩 세대**

| 세대 | 용도 | 특징 |
|:---|:---|:---|
| **Titan M** | Pixel 3+ | 디바이스 보안 |
| **Titan M2** | Pixel 6+ | 향상된 성능 |
| **Titan C** | Chromebook | 크롬OS 보안 |
| **Titan Server** | Cloud | 서버 보안 |

**핵심 알고리즘**: 부트 검증 → 키 보호 → 런타임 보안 → 인증

**📢 섹션 요약 비유**: M은 모바일, C는 크롬OS, Server는 클라우드!
