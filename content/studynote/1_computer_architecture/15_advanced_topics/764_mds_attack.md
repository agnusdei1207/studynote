# 마이크로아키텍처 데이터 샘플링 (MDS) 공격

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU 마이크로아키텍처의 버퍼(L1D, Store Buffer, Load Port)에서 잔류 데이터를 유출하는 사이드 채널 공격
> 2. **가치**: 보안 취약점 이해, 스펙터/멜트다운 계열, 완화 조치, CPU 설계 보안
> 3. **융합**: 사이드 채널 공격, 스펙터, 멜트다운, CPU 아키텍처와 통합된 보안 분석

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

마이크로아키텍처 데이터 샘플링(MDS, Microarchitectural Data Sampling)은 CPU 마이크로아키텍처의 버퍼에서 잔류 데이터를 유출하는 사이드 채널 공격입니다. 2019년 공개되었습니다.

> **해설**: MDS는 CPU 내부 버퍼에 잔류하는 데이터를 샘플링하여 유출합니다.

**💡 비유**: MDS는 휴지통 뒤지기와 같습니다. 버려진 문서(잔류 데이터)에서 정보를 얻습니다.

**📢 섹션 요약 비유**: MDS는 휴지통 뒤지기 같아요. 버려진 데이터를 찾아요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **Store Buffer** | 쓰기 버퍼 | 메모리 쓰기 지연 | 우편함 |
| **Fill Buffer** | L1D 채움 | 캐시 미스 처리 | 배달 |
| **Load Port** | 읽기 포트 | 메모리 읽기 | 수령 |
| **L1D Cache** | 1차 캐시 | 데이터 저장 | 보관함 |

**MDS 공격 원리**

| 변종 | 버퍼 | 유출 데이터 | CVE |
|:---|:---|:---|:---|
| **MSBDS** | Store Buffer | 쓰기 데이터 | CVE-2018-12126 |
| **MFBDS** | Fill Buffer | 캐시 라인 | CVE-2018-12127 |
| **MLPDS** | Load Port | 읽기 데이터 | CVE-2018-12130 |
| **MDSUM** | Uncacheable | 비캐시 메모리 | CVE-2019-11091 |

**핵심 알고리즘**: 버퍼 상태 조작 → 타이밍 측정 → 데이터 추론

**📢 섹션 요약 비유**: 버퍼에 남은 흔적으로 데이터를 유추해요!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: MDS vs Spectre vs Meltdown**

| 비교 항목 | MDS | Spectre | Meltdown |
|:---|:---:|:---:|:---:|
| **대상** | 마이크로 버퍼 | 분기 예측 | 권한 체크 |
| **유형** | 데이터 샘플링 | 추론 실행 | 순차 실행 위반 |
| **공개** | 2019 | 2018 | 2018 |
| **영향** | Intel 주요 | 전 CPU | Intel 주요 |

**📢 섹션 요약 비유**: MDS는 Spectre/Meltdown의 사촌이에요!
