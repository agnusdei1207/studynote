# 배스터브 곡선 (Bathtub Curve) 고장률

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 하드웨어의 수명 주기에 따른 고장률 변화를 나타낸 U자형 곡선 (초기고장-우발고장-마모고장)
> 2. **가치**: 수명 예측, 유지보수 전략, 보증 기간 설정, 교체 시기 결정
> 3. **융합**: MTBF, Weibull 분포, 신뢰성 공학, 예방 유지보수와 통합된 수명 분석

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

배스터브 곡선(Bathtub Curve)은 하드웨어의 수명 주기에 따른 고장률 변화를 나타낸 U자형 곡선입니다. 초기 고장기, 우발 고장기, 마모 고장기로 구성됩니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    배스터브 곡선 (Bathtub Curve)                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   고장률 (λ)                                                         │
│      ▲                                                              │
│   높음│  ┌───────────────────────────────────────────────┐          │
│       │  │ 초기고장기   우발 고장기 (정상 수명)  마모고장기 │          │
│       │  │ (Infant)     (Useful Life)        (Wear-out)  │          │
│       │  │                                               │          │
│       │  │    ╲                                         ╱          │
│       │  │     ╲                                       ╱           │
│       │  │      ╲                                     ╱            │
│       │  │       ╲___________________________________╱             │
│       │  │              ─────────────────                             │
│       │  │               일정한 고장률 (λ = constant)                │
│       │  └───────────────────────────────────────────────┘          │
│       └──┼───────────────┼───────────────────────┼─────────→ 시간   │
│          0          TBurn-in                   EOL                   │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              각 구간 특징                                       │ │
│   │                                                              │ │
│   │   초기 고장기: 제조 결함, 조립 불량, 초기 스트레스              │ │
│   │   우발 고장기: 랜덤 고장, 낮은 일정 고장률, 정상 운영           │ │
│   │   마모 고장기: 기계적 마모, 열화, 수명 종료                     │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 배스터브 곡선은 욕조 모양의 U자형 곡선입니다. 처음과 끝에 고장률이 높습니다.

**💡 비유**: 배스터브 곡선은 사람의 건강과 같습니다. 유아기(취약) → 청장년(건강) → 노년기(쇠약).

**📢 섹션 요약 비유**: 배스터브는 욕조 모양이에요. 처음과 끝이 높아요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 구간명 | 기간 | 고장률 특성 | 주요 원인 | 대응 |
|:---|:---|:---|:---|:---|
| **초기 고장기** | 0 ~ 수개월 | 높음 → 감소 | 제조 결함 | 번인 테스트 |
| **우발 고장기** | 수개월 ~ 수년 | 일정 (λ) | 랜덤 | 모니터링 |
| **마모 고장기** | EOL 전후 | 급격히 증가 | 마모/열화 | 교체 |

**구간별 특성**

| 구간 | λ(t) 형태 | 분포 | 대응 전략 |
|:---|:---:|:---|:---|
| **초기** | 감소 함수 | Weibull (β<1) | 번인, 스트레스 테스트 |
| **우발** | 일정 | 지수 분포 | 예비 부품, 모니터링 |
| **마모** | 증가 함수 | Weibull (β>1) | 예방 교체, 수명 관리 |

**핵심 알고리즘**: 시간 → 구간 판별 → λ(t) 계산 → 신뢰도 예측

**📢 섹션 요약 비유**: 초기엔 번인, 중간엔 모니터링, 끝엔 교체해요!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 하드웨어 vs 소프트웨어 고장 패턴**

| 비교 항목 | 하드웨어 | 소프트웨어 |
|:---|:---:|:---:|
| **초기 고장** | 제조 결함 | 버그 |
| **우발 고장** | 랜덤 | 드묾 |
| **마모 고장** | 물리적 열화 | 없음 (업데이트) |
| **고장률** | U자형 | 감소 함수 |

**📢 섹션 요약 비유**: 하드웨어는 늙어요, 소프트웨어는 고쳐져요!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 2: 산업용** → 예방 유지보수로 마모기 전 교체

**도입 체크리스트**: 번인 기간 설정, EOL 예측, 교체 정책, 비용 분석
