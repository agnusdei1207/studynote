# 안티 탬퍼 (Anti-Tamper) 메시/쉴드

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 칩 내부에 메시/쉴드를 배치하여 물리적 공격 시 감지하고 보안 데이터를 소거하는 방어 기술
> 2. **가치**: 물리적 공격 방지, IP 보호, 보안 키 보호, 인증 요건 충족
> 3. **융합**: 제로화, 보안 키 소거, TPM, HSM과 통합된 물리적 보안

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **메시 레이어** | 물리적 감지 | 와이어 그리드 | 철망 |
| **감지 회로** | 끊김 감지 | 연속성 체크 | 센서 |
| **제어 로직** | 반응 결정 | 상태 머신 | 두뇌 |
| **키 소거** | 데이터 파괴 | 메모리 삭제 | 자폭 |

**안티 탬퍼 유형**

| 유형 | 구조 | 감지 방식 |
|:---|:---|:---|
| **Passive Mesh** | 금속 와이어 | 연속성 끊김 |
| **Active Shield** | 활성 회로 | 신호 위상 변화 |
| **Glue Logic** | 난수화 회로 | 예상 패턴 위반 |
| **Memory Shield** | 메모리 커버 | 접근 시도 감지 |

**핵심 알고리즘**: 연속성 모니터링 → 이상 감지 → 키 소거 → 잠금

**📢 섹션 요약 비유**: 메시가 끊기면 바로 키를 지워요!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 스마트카드** → Active Shield 필수
**시나리오 2: HSM** → 다층 Mesh + Glue Logic

**📢 섹션 요약 비유**: 레이어가 많을수록 안전해요!
