# 마르코프 모델 신뢰성 분석 (Markov Model Reliability Analysis)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시스템 상태 전이를 확률적으로 모델링하여 시간에 따른 신뢰성을 분석하는 수학적 기법
> 2. **가치**: 복잡한 복구/고장 패턴 분석, 시간 의존 신뢰도, 상태 전이 확률 계산
> 3. **융합**: RBD, FTA, 큐잉 이론, 가용성 공학과 통합된 확률적 신뢰성 분석

### Ⅰ. 개요 (Context & Background)

**개념 정의**

마르코프 모델 신뢰성 분석(Markov Model Reliability Analysis)은 시스템 상태 전이를 확률적으로 모델링하여 시간에 따른 신뢰성을 분석하는 수학적 기법입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    마르코프 상태 전이 모델                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│                      ┌─────────────┐                                │
│                      │   State 0   │                                │
│                      │  (정상)     │                                │
│                      │  R(t)=P0    │                                │
│                      └──────┬──────┘                                │
│                             │ λ (고장률)                             │
│                             ▼                                       │
│                      ┌─────────────┐                                │
│           μ (복구율) │   State 1   │ λ (고장률)                      │
│        ┌────────────│  (고장 1개)  │────────────┐                    │
│        │            │  R(t)=P1    │            │                    │
│        │            └──────┬──────┘            │                    │
│        │                   │                   ▼                    │
│        │                   │            ┌─────────────┐             │
│        └───────────────────┤            │   State 2   │             │
│                            │            │  (고장 2개)  │             │
│                            │            │  시스템 다운 │             │
│                            │            └─────────────┘             │
│                            │                                         │
│                    상태 전이 확률                                     │
│                    P'(t) = P(t) × Q (전이 행렬)                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 마르코프 모델은 상태(정상, 고장) 간 전이를 확률로 모델링합니다.

**💡 비유**: 마르코프 모델은 날씨 예보와 같습니다. 현재 상태에서 내일 상태 확률을 예측합니다.

**📢 섹션 요약 비유**: 마르코프는 상태 예측 같아요. 다음 상태를 확률로 알아요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **상태** | 시스템 상태 | 정상/고장/복구 | 방 |
| **전이** | 상태 변화 | λ(고장), μ(복구) | 문 |
| **전이 행렬** | 확률 행렬 | Q = [qij] | 지도 |
| **체류 시간** | 상태 지속 | 지수 분포 | 머무는 시간 |

**주요 파라미터**

| 파라미터 | 의미 | 단위 | 관계식 |
|:---|:---|:---|:---|
| **λ** | 고장률 | 1/시간 | λ = 1/MTBF |
| **μ** | 복구율 | 1/시간 | μ = 1/MTTR |
| **A** | 가용성 | 무차원 | A = μ/(λ+μ) |
| **R(t)** | 신뢰도 | 무차원 | R(t) = e^(-λt) |

**핵심 알고리즘**: 상태 정의 → 전이 행렬 구성 → 미분방정식 풀이 → R(t), A 계산

**📢 섹션 요약 비유**: 고장률 λ는 1/MTBF, 복구율 μ는 1/MTTR이에요!
