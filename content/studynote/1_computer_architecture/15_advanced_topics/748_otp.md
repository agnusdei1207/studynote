# 과열 보호 (OTP, Over Temperature Protection)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 온도가 설정 임계값을 초과하면 전원을 차단하거나 성능을 제한하는 열 보호 회로
> 2. **가치**: 과열로 인한 하드웨어 손상 방지, 화재 예방, CPU/GPU 보호, 수명 연장
> 3. **융합**: TjMax, PROCHOT#, Thermal Throttling, 쿨링 시스템과 통합된 열 보호

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

과열 보호(OTP, Over Temperature Protection)는 온도가 설정 임계값을 초과하면 전원을 차단하거나 성능을 제한하는 열 보호 회로입니다. CPU, GPU, PSU 등 모든 전자기기에 필수적입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    과열 보호 (OTP) 작동 원리                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OTP 작동 단계                                     │ │
│   │                                                              │ │
│   │   온도 (°C)                                                   │ │
│   │      ▲                                                       │ │
│   │   105 ────┬── OTP Shutdown (전원 차단)                       │ │
│   │      │     │                                                  │ │
│   │   100 ────┼── TjMax (Thermal Throttling 시작)                │ │
│   │      │     │                                                  │ │
│   │    90 ────┼── Thermal Warning (경고)                         │ │
│   │      │     │                                                  │ │
│   │    80 ────┼── 정상 운전 범위                                  │ │
│   │      │     │                                                  │ │
│   │   ───┴─────┴────────────────────────────────────────────    │ │
│   │                                                              │ │
│   │   단계별 대응:                                                │ │
│   │   1. Warning: 팬 속도 증가, 알림                              │ │
│   │   2. Throttling: 클럭/전압 감소                               │ │
│   │   3. Shutdown: 전원 완전 차단                                 │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: OTP는 온도에 따라 단계별로 대응합니다. 경고 → Throttling → Shutdown 순서입니다.

**💡 비유**: OTP는 자동차 냉각수 경고등과 같습니다. 엔진이 너무 뜨거우면 경고하고, 더 뜨거우면 멈춥니다.

**📢 섹션 요약 비유**: OTP는 엔진 과열 경고등 같아요. 뜨거우면 알려주고, 더 뜨거우면 멈춰요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **온도 센서** | 온도 측정 | 다이오드/NTC | 온도계 |
| **비교기** | 임계값 비교 | Op-Amp | 판단 |
| **Throttling** | 성능 제한 | 클럭 감소 | 슬로우다운 |
| **Shutdown** | 전원 차단 | MOSFET | 비상 정지 |

**OTP 임계값 예시**

| 장비 | 정상 온도 | Warning | Throttle | Shutdown |
|:---|:---:|:---:|:---:|:---:|
| CPU | 50-80°C | 85°C | 95-100°C | 105°C |
| GPU | 60-85°C | 90°C | 95-100°C | 105°C |
| PSU | 30-50°C | 60°C | 70°C | 80°C |

**핵심 알고리즘**: 온도 센서 → 비교기 → Throttling/Shutdown

**📢 섹션 요약 비유**: OTP는 온도에 따라 점점 강하게 대응해요!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: Thermal Throttling vs OTP Shutdown**

| 비교 항목 | Throttling | Shutdown |
|:---|:---:|:---:|
| **목적** | 온도 하락 | 완전 보호 |
| **성능** | 저하 | 정지 |
| **복구** | 자동 | 수동/자동 |

**📢 섹션 요약 비유**: Throttling은 천천히, Shutdown은 완전 멈춤입니다.

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데이터센터** → OTP + BMC 모니터링
**시나리오 2: 노트북** → Thermal Throttling 중심
**시나리오 3: 산업용** → OTP Shutdown + 알림

**도입 체크리스트**: 온도 센서, 임계값 설정, 모니터링

**📢 섹션 요약 비유**: OTP는 건강 관리 같아요. 열이 나면 쉬어요!
