# 플런더버그 (Plundervolt)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU 전압 조작(언더볼팅)을 통해 SGX 인클레이브 내부 연산 오류를 유발하여 비밀을 유출하는 공격
> 2. **가치**: SGX 보안 이해, 전압 기반 공격, 완화 조치, 하드웨어 보안
> 3. **융합**: SGX, Rowhammer, 전압 공격, 사이드 채널과 통합된 물리적 공격

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

> **해설**: Plundervolt는 전압 강하로 SGX 내부 연산 오류를 유발합니다.

**💡 비유**: Plundervolt는 금고에 진동을 줘서 잠금을 푸는 것과 같습니다. 정상 동작을 방해합니다.

**📢 섹션 요약 비유**: Plundervolt는 금고 흔들기 같아요. 전압을 낮춰요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **전압 제어 MSR** | 전압 설정 | MSR 주소 | 다이얼 |
| **언더볼팅** | 전압 강하 | Vcore 감소 | 흔들기 |
| **연산 오류** | 계산 오류 | 비트 플립 | 잠금 풀림 |
| **SGX Enclave** | 보안 영역 | 격리 실행 | 금고 |

**Plundervolt 특징**

| 항목 | 내용 |
|:---|:---|
| **CVE** | CVE-2019-11157 |
| **대상** | Intel SGX |
| **방법** | 전압 조작 |
| **전제** | 전압 제어 권한 |

**핵심 알고리즘**: 권한 획득 → 전압 강하 → 연산 오류 → 오류 분석 → 키 유출

**📢 섹션 요약 비유**: 전압을 낮추면 계산이 틀려요. 그걸로 키를 찾아요!
