# 534. 무어의 법칙 이후 (More than Moore)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 'More than Moore (무어의 법칙 이후)'는 트랜지스터 선폭을 줄이는 미세 공정 한계(More Moore)를 넘어, 이기종 칩 통합, 3D 패키징, 특화 기능 추가 등을 통해 시스템 전체의 성능과 가치를 높이는 새로운 반도체 패러다임이다.
> 2. **가치**: 단일 칩의 밀도 한계를 패키징 수준에서 극복하여 개발 비용을 절감하고, AI, 5G, 자율주행 등 특정 도메인에 최적화된 고성능 시스템을 신속하게 구현한다.
> 3. **융합**: 칩렛 (Chiplet), TSV (Through Silicon Via), 시스템 온 패키지 (SiP), 이기종 집적 (Heterogeneous Integration) 등 설계, 공정, 패키징 전 영역의 기술 혁신이 결집된 형태이다.

---

## Ⅰ. 개요 (Context & Background)

- **개념**: 무어의 법칙 (Moore's Law)이 2년마다 트랜지스터 수가 2배로 늘어나는 '공정 미세화 (Scaling)'에 집중했다면, **More than Moore**는 칩의 크기를 줄이는 대신 서로 다른 기능을 가진 칩들을 하나의 패키지로 묶어 시스템 성능을 극대화하는 '기능적 확장 (Functional Diversity)'을 지향한다.

- **💡 비유**: 한 장의 종이에 글씨를 점점 더 작게 쓰는 것(More Moore)이 한계에 다다르자, 여러 장의 종이를 입체적으로 겹치거나 포스트잇을 붙여서 책 한 권(More than Moore)을 만드는 것과 같습니다.

- **등장 배경**:
  1. **경제적 한계 (Moore's Law is Dead?)**: 공정 미세화 속도가 늦춰지고 있으며, 3nm 이하 공정으로 갈수록 설계 및 제조 비용이 기하급수적으로 상승하여 가성비가 떨어지고 있다.
  2. **물리적 한계**: 누설 전류, 열 방산 문제, 양자 터널링 현상 등으로 인해 트랜지스터를 더 이상 작게 만들기 어려워졌다.
  3. **다양한 서비스 요구**: 단순히 빠른 연산뿐 아니라, 센서, 통신(RF), 아날로그, 전력 관리 등 다양한 기능이 한 시스템 내에 긴밀하게 통합되어야 하는 시대가 되었다.

- **📢 섹션 요약 비유**: 한 명의 천재(단일 칩)를 키우는 대신, 각 분야의 전문가들(칩렛)을 모아 최강의 팀(시스템 패키지)을 구성하는 협력의 시대로 전환된 것과 같습니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 구성 요소 (More than Moore의 핵심 기술들)

| 요소명 | 역할 | 내부 동작 | 관련 기술 | 비유 |
|:---|:---|:---|:---|:---|
| **Chiplet (칩렛)** | 기능을 분할한 작은 칩 단위 | 연산, 메모리, I/O 등을 각각 최적 공정으로 제작 | D2D (Die-to-Die) | 레고 블록 |
| **TSV (Through Silicon Via)** | 수직 데이터 통로 | 실리콘 웨이퍼를 관통하여 상하 칩 연결 | 3D Stacking, HBM | 건물 내 엘리베이터 |
| **Interposer (인터포저)** | 칩 간 연결 가교 | 미세 회로를 가진 중간 기판으로 칩렛 연결 | 2.5D Packaging | 아파트 복도 |
| **RDL (Redistribution Layer)** | 회로 재배선 | 패키지 내 단자 위치를 자유롭게 재배치 | Fan-out Packaging | 복잡한 배선 정리 |
| **Hybrid Bonding** | 직접 구리 접합 | 범프 없이 칩과 칩을 직접 연결하여 전송 속도 향상 | Cu-to-Cu Bonding | 본드 없이 자석으로 합체 |

### More Moore vs More than Moore 구조 비교

두 패러다임이 추구하는 방향의 차이를 시각화한다.

```text
  ┌─────────────────────────────────────────────────────────────┐
  │           More Moore vs More than Moore 아키텍처 비교          │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │ [1] More Moore (미세화 중심)      [2] More than Moore (융합 중심) │
  │                                                             │
  │    ┌───────────────┐               ┌───────────────┐        │
  │    │   Monolithic  │               │   Packaging   │        │
  │    │      Die      │               │ ┌───┐ ┌───┐   │        │
  │    │ (Single Chip) │               │ │CPU│ │GPU│   │        │
  │    │               │               │ └───┘ └───┘   │        │
  │    │   2nm / 1nm   │               │ ┌───┐ ┌───┐   │        │
  │    │   Scaling     │               │ │Mem│ │RF │   │        │
  │    └───────────────┘               │ └───┘ └───┘   │        │
  │                                    └──────┬────────┘        │
  │                                           │                 │
  │  * 핵심: 더 작게, 더 촘촘히          * 핵심: 더 다양하게, 입체적으로 │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** **More Moore**는 하나의 커다란 실리콘 다이(Die) 위에 모든 회로를 다 집어넣고 선폭을 줄이는 방식이다. 하지만 선폭이 좁아질수록 수율이 낮아지고 비용이 폭증한다. **More than Moore**는 이를 해결하기 위해 기능을 쪼갠다. 연산이 중요한 CPU는 비싼 5nm 공정으로, 성능보다 비용이 중요한 I/O나 아날로그는 저렴한 28nm 공정으로 각각 만든 뒤(칩렛), 이를 고속 인터커넥트(인터포저)로 연결하여 하나의 패키지로 묶는다. 이를 통해 전체 칩 생산 비용을 낮추면서도 물리적 크기 한계를 넘어선 거대 시스템 구현이 가능해진다.

---

### 심층 동작 원리: 이기종 집적 (Heterogeneous Integration)
1. **최적 공정 선택**: 각 기능 블록(IP)별로 가장 가성비가 좋은 공정 노드를 선택하여 제작한다.
2. **고속 인터페이스**: 칩렛 간 통신을 위해 UCIe (Universal Chiplet Interconnect Express)와 같은 표준화된 저지연 인터페이스를 사용한다.
3. **열 관리**: 3D로 쌓을 경우 발생하는 발열 문제를 해결하기 위해 고방열 소재 및 구조 설계를 병행한다.

- **📢 섹션 요약 비유**: 엔진, 타이어, 오디오를 모두 한 공장에서 만드는 대신, 각 전문 공장에서 최고의 부품을 가져와 조립 센터(패키징)에서 명차를 완성하는 것과 같습니다.

## Ⅲ. 융합 비교 및 다각도 분석

### More Moore vs More than Moore 상세 비교

| 비교 항목 | More Moore (무어 지속) | More than Moore (무어 확장) |
|:---|:---|:---|
| **주요 동력** | 리소그래피 (EUV 등) 공정 개선 | **고급 패키징 (Advanced Packaging)** |
| **확장 방향** | 트랜지스터 밀도 (수평적) | **기능 다양성 (수직/수평 융합)** |
| **비용 효율** | 공정 미세화에 따른 비용 폭증 | 이기종 집적을 통한 비용 최적화 |
| **주요 기술** | FinFET, GAA, High-NA EUV | **HBM, 칩렛, SiP, 2.5D/3D** |
| **한계점** | 물리적/경제적 한계 봉착 | 패키징 복잡도 및 열 관리 어려움 |

---

### 과목 융합 관점
- **컴퓨터 구조 (메모리 벽)**: 프로세서와 메모리를 수직으로 쌓는 HBM (High Bandwidth Memory) 기술은 More than Moore의 대표 사례로, 데이터 전송 지연을 물리적으로 단축한다.
- **네트워크 (5G/6G)**: 디지털 연산 칩과 고주파(RF) 통신 칩을 하나의 패키지에 묶어 신호 손실을 최소화하고 기기 크기를 줄인다.

- **📢 섹션 요약 비유**: 도서관 건물을 계속 넓히는 것(More Moore)보다, 층수를 높이고(3D) 근처에 카페와 식당을 합치는(기능 융합) 것이 사용자에게 더 큰 가치를 주는 것과 같습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오
1. **AI 가속기 (GPU) 설계**: 수만 개의 코어를 가진 거대 GPU를 단일 칩으로 만들면 수율이 0%에 가깝다. 대신 4~8개의 연산 칩렛을 인터포저로 연결하여 수율을 높이고 성능을 8배 확장한다 (예: NVIDIA Hopper/Blackwell).
2. **저전력 모바일 SoC**: 고성능 CPU 코어와 저전력 통신 칩을 최적의 패키징으로 결합하여 스마트폰 내부 공간을 확보하고 배터리 효율을 극대화한다.
3. **차량용 반도체 (ADAS)**: 센서 인터페이스와 연산 유닛을 통합하여 데이터 처리 속도를 높이고 혹독한 차량 환경에서의 신뢰성을 확보한다.

### 도입 체크리스트
- 단일 칩(Monolithic) 대비 칩렛 구조의 수율 개선 효과가 비용 상승(패키징 비용)을 상쇄하는가?
- 칩렛 간 데이터 전송 지연이 전체 시스템 성능 목표를 만족하는가?
- 3D 적층 시 발생하는 열 밀도(Heat Density)를 제어할 냉각 솔루션이 있는가?

- **📢 섹션 요약 비유**: 하나의 거대한 통유리를 만드는 것보다, 작은 유리 조각들을 정교하게 이어 붙인(칩렛) 스테인드글라스가 더 화려하고 만들기 쉬운 것과 같습니다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과
- **개발 기간 단축**: 이미 검증된 칩렛(IP)을 재사용하여 신규 칩 설계 시간(Time-to-Market) 획기적 단축.
- **성능의 한계 돌파**: 단일 칩 면적 제한(Reticle Limit)을 넘어선 초대형 연산 시스템 구축 가능.
- **가치 창출**: 단순 연산을 넘어 센싱, 통신, 보안이 결합된 '지능형 시스템'으로의 진화.

### 미래 전망
- 향후 **'시스템의 칩화 (System on Chip)'**를 넘어 **'데이터 센터의 칩화 (Data Center on Package)'** 수준으로 발전할 것이며, 광통신(Optical Interconnect)을 패키지 내부에 도입하여 전송 속도 혁명을 일으킬 전망이다. 반도체 패권은 이제 '누가 더 작게 만드는가'에서 '누가 더 잘 쌓고 연결하는가'로 이동하고 있다.

- **📢 섹션 요약 비유**: 이제 반도체는 단일 건물이 아니라, 모든 기능이 완벽하게 연결된 '수직 도시'로 진화하며 인류의 지능을 무한히 확장할 것입니다.
