# MIL-HDBK-217 고장률 예측

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 군사 및 산업용 전자 장비의 부품별 고장률을 예측하는 미 국방부 신뢰성 예측 표준
> 2. **가치**: 설계 단계 신뢰성 예측, 부품 선정 기준, MTBF 계산, 유지보수 계획

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MIL-HDBK-217은 군사 및 산업용 전자 장비의 부품별 고장률을 예측하는 미 국방부 신뢰성 예측 표준입니다. 부품 수, 환경, 품질 등을 고려하여 시스템 MTBF를 계산합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MIL-HDBK-217 고장률 예측 프로세스                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              고장률 계산 공식                                   │ │
│   │                                                              │ │
│   │   λp = λb × πT × πS × πQ × πE × πC × ...                    │ │
│   │                                                              │ │
│   │   λp: 부품 고장률 (Predicted Failure Rate)                    │ │
│   │   λb: 기본 고장률 (Base Failure Rate)                         │ │
│   │   πT: 온도 계수 (Temperature Factor)                          │ │
│   │   πS: 전기적 스트레스 계수 (Stress Factor)                     │ │
│   │   πQ: 품질 계수 (Quality Factor)                              │ │
│   │   πE: 환경 계수 (Environment Factor)                          │ │
│   │   πC: 복잡도 계수 (Complexity Factor)                         │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              환경 계수 (πE) 예시                                │ │
│   │                                                              │ │
│   │   ┌──────────────────────┬──────────┬───────────┐           │ │
│   │   │ 환경                  │ 코드      │ πE 값     │           │ │
│   │   ├──────────────────────┼──────────┼───────────┤           │ │
│   │   │ 지상 고정 (Ground)    │ GF       │ 1.0       │           │ │
│   │   │ 지상 이동 (Mobile)    │ GM       │ 4.0       │           │ │
│   │   │ 해군 보호 (Naval)     │ NS       │ 4.0       │           │ │
│   │   │ 항공 (Airborne)       │ AIC      │ 4.0-6.0   │           │ │
│   │   │ 우주 (Space)          │ SF       │ 0.5       │           │ │
│   │   │ 미사일 (Missile)      │ ML       │ 8.0-12.0  │           │ │
│   │   └──────────────────────┴──────────┴───────────┘           │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: MIL-HDBK-217은 기본 고장률에 다양한 계수를 곱하여 예측 고장률을 계산합니다.

**💡 비유**: MIL-HDBK-217은 보험료 계산과 같습니다. 기본 요율에 환경, 품질 등을 반영합니다.

**📢 섹션 요약 비유**: MIL-HDBK-217은 보험료 계산 같아요. 환경이 나쁘면 올라요!
