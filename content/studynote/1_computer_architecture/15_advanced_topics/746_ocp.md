# 과전류 보호 (OCP, Over Current Protection)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전류가 설정 임계값을 초과하면 전원을 차단하여 장비를 보호하는 안전 회로
> 2. **가치**: 과부하 방지, 화재 예방, VRM/PSU 보호, 케이블 손상 방지
> 3. **융합**: OVP, SCP, OTP, PSU 보호 회로, 전류 센싱과 통합된 전원 보호

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

과전류 보호(OCP, Over Current Protection)는 전류가 설정 임계값을 초과하면 전원을 차단하여 장비를 보호하는 안전 회로입니다. PSU, VRM, 케이블 등에 필수적으로 탑재됩니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    과전류 보호 (OCP) 구조                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   I_load (부하 전류)                                                 │
│       │                                                             │
│       ▼                                                             │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │                    OCP 회로                                   │   │
│   │                                                             │   │
│   │   ┌─────────┐    ┌─────────┐    ┌─────────┐                │   │
│   │   │ 전류    │    │ 비교기  │    │ 차단    │                │   │
│   │   │ 센서    │───►│ (Comp)  │───►│ 스위치  │                │   │
│   │   │ (Shunt) │    │         │    │         │                │   │
│   │   └─────────┘    └────┬────┘    └────┬────┘                │   │
│   │                       │              │                        │   │
│   │                       │ I_threshold  │                        │   │
│   │                       ▼              │                        │   │
│   │               ┌──────────────┐       │                        │   │
│   │               │ 기준 전류    │       │                        │   │
│   │               │ (Threshold)  │       │                        │   │
│   │               └──────────────┘       │                        │   │
│   │                                                             │   │
│   │   I_sense > I_threshold → SHUTDOWN                           │   │
│   │                                                             │   │
│   └─────────────────────────────────────────────────────────────┘   │
│       │                                                             │
│       ▼ (정상 시)                                                    │
│   Load                                                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: OCP는 전류 센서로 전류를 측정하고, 임계값과 비교하여 초과 시 차단합니다.

**💡 비유**: OCP는 회로 차단기와 같습니다. 전류가 너무 많이 흐르면 자동으로 끊어집니다.

**📢 섹션 요약 비유**: OCP는 전기 차단기 같아요. 전기가 너무 많이 흐르면 끊어요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **전류 센서** | 전류 측정 | Shunt 저항 | 미터기 |
| **비교기** | 임계값 비교 | Op-Amp | 판단 |
| **기준 전류** | 임계값 | 설정 가능 | 기준점 |
| **차단 스위치** | 전원 차단 | MOSFET/릴레이 | 스위치 |

**OCP 임계값 예시**

| 레일 | 정상 전류 | OCP 임계값 | 여유율 |
|:---|:---:|:---:|:---:|
| 12V 레일 | 50A | 65-75A | +30-50% |
| 5V 레일 | 30A | 40-45A | +30-50% |
| Vcore | 150A | 180-200A | +20-35% |

**핵심 알고리즘**: 전류 센서(Shunt) → 비교기 → 차단 (I > I_threshold)

**📢 섹션 요약 비유**: OCP는 전류가 임계값을 넘으면 즉시 차단합니다.

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: OCP vs OVP vs SCP**

| 비교 항목 | OCP | OVP | SCP |
|:---|:---:|:---:|:---:|
| **감지 대상** | 전류 | 전압 | 전류 급증 |
| **트리거** | I > 임계값 | V > 임계값 | I 급증 |
| **차단 속도** | ~μs | ~μs | ~ns |
| **주요 원인** | 과부하 | VRM 고장 | 단락 |

**📢 섹션 요약 비유**: OCP는 과부하, OVP는 과전압, SCP는 단락을 보호합니다.

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데이터센터** → 고품질 PSU + OCP/OVP/SCP 이중화
**시나리오 2: 오버클러킹** → OCP 설정 조정 주의
**시나리오 3: 임베디드** → OCP + 퓨즈 이중화

**도입 체크리스트**: OCP 임계값 (+30-50%), 응답 시간 (<수 μs), 모니터링

**📢 섹션 요약 비유**: OCP는 보험과 같습니다. 적절한 임계값 설정이 중요합니다.

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 하드웨어 손상 방지, 화재 예방, 다운타임 단축

**미래 전망**: 디지털 OCP, AI 예지, 다단계 보호, 자가 진단

**📢 섹션 요약 비유**: OCP의 미래는 AI가 관리합니다.
