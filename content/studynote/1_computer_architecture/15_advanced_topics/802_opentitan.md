# 오픈소스 하드웨어 RoT (OpenTitan)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: Google이 주도하는 오픈소스 하드웨어 RoT(Root of Trust)로 투명하고 검증 가능한 보안 칩
> 2. **가치**: 오픈 소스, 커뮤니티 검증, 투명성, 맞춤형 구현, 공급망 보안
> 3. **융합**: TPM, HSM, RISC-V, PUF, 안티 탬퍼와 통합된 오픈 보안

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **Ibex 코어** | 메인 CPU | RV32IMC | 두뇌 |
| **암호화 엔진** | AES, KMAC | 하드웨어 가속 | 암호기 |
| **OTBN** | Big Number | RSA/ECC | 계산기 |
| **PUF** | 디바이스 ID | SRAM PUF | 지문 |

**OpenTitan IP 블록**

| IP | 기능 |
|:---|:---|
| **AES** | AES-128/256 암호화 |
| **HMAC** | SHA-256 해시 |
| **KMAC** | SHA-3 기반 MAC |
| **OTBN** | Big Number 연산 (RSA/ECC) |
| **CSRNG** | 암호화 난수 생성 |
| **FLASH** | 보안 저장소 |

**핵심 알고리즘**: 부팅 → OTBN 검증 → 키 로드 → 보안 서비스

**📢 섹션 요약 비유**: 모든 보안 IP가 들어 있어요!
