# 350. PCI (Peripheral Component Interconnect)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 병렬 버스 기반 확장 카드 인터페이스 표준
> 2. **가치**: Plug and Play, IRQ 공유,广泛 채택
> 3. **융합**: 버스 중재, 버스트, 설정 공간

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

PCI(Peripheral Component Interconnect)는 **인텔이 개발한 32/64비트 병렬 버스 표준으로, 컴퓨터 내부의 확장 카드와 주변장치를 연결하는 인터페이스**다.

### 💡 비유: 확장 슬롯

PCI는 **확장 슬롯**과 같다. 책상에 서랍을 추가한다. 필요하면 하나 더 단다. 표준 규격이다. 어떤 서랍이든 맞는다. 꽂으면 바로 쓴다. PCI도 똑같다. 카드를 꽂으면 바로 쓴다.

## Ⅱ. PCI 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                PCI 상세                                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【PCI 버스 신호】                                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  주소/데이터:                                                            │ │
│  │  • AD[31:0]: 주소/데이터 멀티플렉스 (32비트)                             │ │
│  │  • AD[63:32]: 확장 (64비트)                                           │ │
│  │                                                               │ │
│  │  제어 신호:                                                              │ │
│  │  • FRAME#: 프레임 시작/끝                                              │ │
│  │  • IRDY#: 이니시에이터 준비                                            │ │
│  │  • TRDY#: 타겟 준비                                                    │ │
│  │  • STOP#: 타겟 중단 요청                                               │ │
│  │  • DEVSEL#: 장치 선택                                                  │ │
│  │                                                               │ │
│  │  명령/바이트 인에이블:                                                    │ │
│  │  • C/BE[3:0]#: 명령/바이트 인에이블                                     │ │
│  │  • PAR: 패리티                                                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【PCI 설정 공간】                                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  256바이트 설정 공간:                                                    │ │
│  │  ┌────────────────────────────────────────┐                        │ │
│  │  │ 오프셋   필드             설명             │                        │ │
│  │  │ ──────   ────             ────             │                        │ │
│  │  │ 00h      Vendor ID        제조사 ID        │                        │ │
│  │  │ 02h      Device ID        장치 ID          │                        │ │
│  │  │ 04h      Command          명령 레지스터    │                        │ │
│  │  │ 06h      Status           상태 레지스터    │                        │ │
│  │  │ 08h      Revision ID      리비전           │                        │ │
│  │  │ 09h      Class Code       클래스 코드      │                        │ │
│  │  │ 10h-24h  BAR 0-5          기본 주소 레지스터│                        │ │
│  │  │ 3Ch      Interrupt Line   인터럽트 라인    │                        │ │
│  │  │ 3Dh      Interrupt Pin    인터럽트 핀      │                        │ │
│  │  └────────────────────────────────────────┘                        │ │
│  │                                                               │ │
│  │  → Plug and Play를 위한 장치 정보                                      │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

### PCI 분석

```

│  00:00.0 Host bridge: Intel Corporation                             │
│  00:01.0 PCI bridge: Intel Corporation                              │

│  01:00.0 VGA compatible controller: NVIDIA                          │

│  ──────────────────                                                  │
│  $ lspci -vv -s 01:00.0                                             │
│  # Vendor ID, Device ID, BAR, IRQ 등                                 │
│                                                                     │

│  $ sudo setpci -s 01:00.0 VENDOR_ID                                 │

│                                                                     │

│  4. IRQ: 인터럽트 공유 확인                                           │
│  5. 리셋: 시스템 재부팅                                               │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• PCI: 병렬 버스 확장 인터페이스
• 폭: 32/64비트
• 속도: 33/66/133 MHz
• 대역폭: 133-1066 MB/s
• Plug and Play: 자동 설정
• IRQ 공유: 인터럽트 공유
• 설정 공간: 256바이트
• BAR: 기본 주소 레지스터
• 버스트: 연속 전송 지원
• 현재: PCIe로 대체됨
```
