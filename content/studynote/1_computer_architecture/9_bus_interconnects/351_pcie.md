# 351. PCIe (PCI Express)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고속 직렬 인터커넥트, 레인 기반 확장
> 2. **가치**: 높은 대역폭, 낮은 지연, 확장성
> 3. **융합**: 직렬 전송, 트랜잭션 계층, P2P

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

PCIe(PCI Express)는 **PCI의 후속으로 개발된 고속 직렬 인터커넥트 표준으로, 레인을 통해 확장 가능한 대역폭과 낮은 지연 시간을 제공**한다.

### 💡 비유: 고속도로 차선

PCIe는 **고속도로 차선**과 같다. 1차선, 4차선, 8차선, 16차선. 차선이 많을수록 많은 차가 지나간다. 각 차선은 독립적이다. 빠르다. 직렬로 데이터가 지나간다. PCIe도 똑같다. 레인이 차선이다.

### PCIe 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                PCIe 개요                                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【PCIe 레인 구조】                                                              │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  레인 (Lane):                                                           │ │
│  │  ┌───────────────────────────────────────────────────────┐    │ │
│  │  │  송신 (TX)  ────────────────────────────→  수신 (RX)  │    │ │
│  │  │  수신 (RX)  ←────────────────────────────  송신 (TX)  │    │ │
│  │  └───────────────────────────────────────────────────────┘    │ │
│  │  → 1쌍의 차등 신호 쌍 (TX+, TX-, RX+, RX-) = 1 레인                │ │
│  │  → 전이중 (동시 송수신)                                             │ │
│  │                                                               │ │
│  │  레인 구성:                                                              │ │
│  │  ┌─────────────────────────────────────────────────────┐      │ │
│  │  │ x1:  1레인  → 최저 대역폭, 저비용                    │      │ │
│  │  │ x4:  4레인  → 중간 성능                              │      │ │
│  │  │ x8:  8레인  → 고성능                                 │      │ │
│  │  │ x16: 16레인 → 최고 성능 (그래픽)                     │      │ │
│  │  └─────────────────────────────────────────────────────┘      │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【PCIe 세대별 속도】                                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  세대     데이터 전송률    x1        x4        x8        x16        │ │
│  │  ────     ──────────────   ────      ────      ────      ────        │ │
│  │  Gen1     2.5 GT/s         250 MB/s  1 GB/s    2 GB/s    4 GB/s     │ │
│  │  Gen2     5.0 GT/s         500 MB/s  2 GB/s    4 GB/s    8 GB/s     │ │
│  │  Gen3     8.0 GT/s         ~1 GB/s   ~4 GB/s   ~8 GB/s   ~16 GB/s   │ │
│  │  Gen4     16.0 GT/s        ~2 GB/s   ~8 GB/s   ~16 GB/s  ~32 GB/s   │ │
│  │  Gen5     32.0 GT/s        ~4 GB/s   ~16 GB/s  ~32 GB/s  ~64 GB/s   │ │
│  │  Gen6     64.0 GT/s        ~8 GB/s   ~32 GB/s  ~64 GB/s  ~128 GB/s  │ │
│  │                                                               │ │
│  │  → 레인 수와 세대가 대역폭 결정                                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. PCIe 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                PCIe 상세                                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【PCIe 계층 구조】                                                              │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  ┌─────────────────────────────────────────┐                  │ │
│  │  │  소프트웨어 계층 (Software Layer)         │                  │ │
│  │  │  • 장치 드라이버                         │                  │ │
│  │  └─────────────────────────────────────────┘                  │ │
│  │                         │                                      │ │
│  │  ┌─────────────────────────────────────────┐                  │ │
│  │  │  트랜잭션 계층 (Transaction Layer)       │                  │ │
│  │  │  • TLP 패킷 생성/파싱                    │                  │ │
│  │  │  • 주소 변환, VC                        │                  │ │
│  │  └─────────────────────────────────────────┘                  │ │
│  │                         │                                      │ │
│  │  ┌─────────────────────────────────────────┐                  │ │
│  │  │  데이터 링크 계층 (Data Link Layer)       │                  │ │
│  │  │  • 오류 검출/복구                        │                  │ │
│  │  │  • ACK/NAK, 순서 번호                    │                  │ │
│  │  └─────────────────────────────────────────┘                  │ │
│  │                         │                                      │ │
│  │  ┌─────────────────────────────────────────┐                  │ │
│  │  │  물리 계층 (Physical Layer)              │                  │ │
│  │  │  • 직렬화/역직렬화                       │                  │ │
│  │  │  • 인코딩, 클럭 복구                     │                  │ │
│  │  └─────────────────────────────────────────┘                  │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【PCIe 트랜잭션 타입】                                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  Memory Read/Write: 메모리 읽기/쓰기                                   │ │
│  │  I/O Read/Write: I/O 포트 읽기/쓰기                                    │ │
│  │  Configuration Read/Write: 설정 공간 접근                              │ │
│  │  Message: 인터럽트, 전원 관리 등                                        │ │
│  │  Completion: 요청에 대한 응답                                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【PCIe vs PCI】                                                                │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  항목        PCI             PCIe                             │ │
│  │  ────        ────             ────                             │ │
│  │  전송        병렬             직렬                              │ │
│  │  클럭        공유             독립 (각 레인)                      │ │
│  │  대역폭      133 MB/s        64 GB/s+ (Gen5 x16)               │ │
│  │  핫플러그    제한적           완전 지원                           │ │
│  │  전력        높음             낮음                              │ │
│  │  핀 수       많음             적음                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. PCIe × 타 과목 융합

### 과목 융합 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                PCIe × 타 과목 융합                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【컴퓨터 구조】                                                      │
│  • 직렬 전송: 차등 신호                                              │
│  • 레인: 대역폭 확장                                                 │
│  • 스위치: PCIe 스위치                                               │
│  • 루트 컴플렉스: 호스트 인터페이스                                    │
│                                                                     │
│  【운영체제】                                                          │
│  • 드라이버: PCIe 드라이버                                           │
│  • MSI: 메시지 시그널드 인터럽트                                       │
│  • ACS: 액세스 제어                                                  │

│                                                                     │
│  【데이터베이스】                                                        │
│  • NVMe: PCIe SSD                                                   │
│  • 네트워크: 고속 NIC                                                │

│  • 메모리: CXL                                                      │
│                                                                     │
│  【성능】                                                              │
│  • 대역폭: 최대 128 GB/s                                             │
│  • 지연: <1us                                                       │
│  • IOPS: 100만+                                                     │
│  • P2P: 장치 간 직접                                                 │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  PCIe 용도:                                                                    │ │
│  │  ──────────────────────                                              │ │
│  │  장치        레인     세대      대역폭                               │ │
│  │  ────        ────     ────      ──────                               │ │
│  │  GPU         x16     Gen4      32 GB/s                              │ │
│  │  NVMe SSD    x4      Gen4      8 GB/s                               │ │
│  │  10GbE NIC   x4      Gen3      4 GB/s                               │ │
│  │  WiFi        x1      Gen3      1 GB/s                               │ │
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• PCIe: 고속 직렬 인터커넥트
• 레인: x1, x4, x8, x16
• 세대: Gen1~Gen6
• 대역폭: 최대 128 GB/s
• 전이중: 동시 송수신
• 계층: 트랜잭션, 링크, 물리
• TLP: 트랜잭션 계층 패킷
• 핫플러그: 완전 지원
• P2P: 장치 간 직접
• 용도: GPU, SSD, NIC, 가속기
```
