# 339. DAS (Direct Attached Storage)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서버에 직접 연결된 스토리지
> 2. **가치**: 낮은 지연, 높은 성능, 간단한 구성
> 3. **융합**: SATA/SAS, 로컬 디스크, 비용 효율

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

DAS(Direct Attached Storage)는 **서버나 워크스테이션에 네트워크를 거치지 않고 직접 연결된 스토리지 장치**다. 내장 디스크, 외장 HDD/SSD, 직접 연결된 RAID 어레이 등이 모두 DAS다.

### 💡 비유: 책상 서랍

DAS는 **책상 서랍**과 같다. 내 책상에 서랍이 있다. 바로 옆에 있다. 손만 뻗으면 된다. 빠르다. 남이 쓸 수 없다. 내 전용이다. 네트워크가 필요 없다. DAS도 똑같다. 서버에 직접 연결된다. 빠르다. 공유는 어렵다.

## Ⅱ. DAS 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                DAS 상세                                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【연결 인터페이스】                                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. SATA (Serial ATA):                                               │ │
│  │     • 소비자용                                                       │ │
│  │     • 6 Gbps                                                        │ │
│  │     • 저렴함                                                        │ │
│  │                                                               │ │
│  │  2. SAS (Serial Attached SCSI):                                      │ │
│  │     • 엔터프라이즈용                                                  │ │
│  │     • 12/24 Gbps                                                    │ │
│  │     • 높은 신뢰성                                                   │ │
│  │     • 다중 포인트                                                   │ │
│  │                                                               │ │
│  │  3. NVMe (PCIe):                                                     │ │
│  │     • 최고 성능                                                      │ │
│  │     • PCIe 직접 연결                                                 │ │
│  │     • 낮은 지연                                                     │ │
│  │                                                               │ │
│  │  ┌─────────────────────────────────────────────────────┐        │ │
│  │  │  인터페이스  속도       용도         비용             │        │ │
│  │  │  ────────   ────        ────         ────             │        │ │
│  │  │  SATA       6 Gbps      소비자       낮음             │        │ │
│  │  │  SAS        24 Gbps     엔터프라이즈  높음             │        │ │
│  │  │  NVMe       32 Gbps     고성능       높음             │        │ │
│  │  │                                                       │        │ │
│  │  └─────────────────────────────────────────────────────┘        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【DAS vs NAS vs SAN】                                                          │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  항목       DAS          NAS           SAN                    │ │
│  │  ────       ────          ────          ────                   │ │
│  │  연결       직접          네트워크      전용 네트워크            │ │
│  │  접근       블록          파일          블록                    │ │
│  │  공유       불가          가능          가능                    │ │
│  │  성능       최고          보통          높음                    │ │
│  │  비용       낮음          중간          높음                    │ │
│  │  확장       제한          쉬움          쉬움                    │ │
│  │  관리       개별          중앙          중앙                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### DAS 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                DAS 분석                                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【DAS 디스크 확인】                                                            │)
│  ──────────────────                                                  │
│  $ lsblk                                                            │
│  NAME   SIZE TYPE MOUNTPOINT                                        │
│  sda    500G disk                                                   │
│  ├─sda1 500M part /boot                                             │
│  └─sda2 499G part /                                                 │
│  sdb    1T disk   ← DAS 추가 디스크                                  │
│                                                                     │
│  【RAID 구성】                                                                  │)
│  ──────────────────                                                  │
│  # 로컬 RAID 1 구성                                                  │
│  $ mdadm --create /dev/md0 --level=1 --raid-devices=2 \             │
│        /dev/sdb1 /dev/sdc1                                          │
│                                                                     │
│  【파일 시스템 생성】                                                            │)
│  ──────────────────                                                  │
│  $ mkfs.xfs /dev/sdb1                                               │
│  $ mount /dev/sdb1 /data                                            │
│                                                                     │
│  【성능 측정】                                                                   │)
│  ──────────────────                                                  │
│  $ fio --name=test --ioengine=libaio --rw=randread --bs=4k \        │
│        --direct=1 --size=1G /dev/sdb1                               │
│  read: IOPS=100,000, BW=400MB/s                                     │
│                                                                     │
│  【DAS 적합 용도】                                                              │)
│  ──────────────────                                                  │
│  1. OS: 부팅, 시스템 파일                                            │
│  2. DB: OLTP, 높은 IOPS                                             │
│  3. 캐시: Redis, 로컬 캐시                                           │
│  4. 개발: 워크스테이션                                               │
│  5. 로그: 앱 로그                                                    │
│  6. 임시: /tmp, 스왑                                                 │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• DAS: 서버 직접 연결 스토리지
• 연결: SATA, SAS, NVMe
• 장점: 낮은 지연, 높은 성능
• 단점: 공유 불가, 확장 제한
• 내부: 내장 디스크
• 외부: 외장 RAID 박스
• JBOD: 디스크 묶음
• 비용: 가장 낮음
• DB: DAS가 최적
• 관리: 개별 서버 단위
```
