# 341. SATA (Serial ATA)

#### 핵심 인사이트 (3줄 요약)

## Ⅰ. 개요 (Context & Background)

### 개념 정의

SATA(Serial ATA)는 **하드디스크와 SSD를 컴퓨터에 연결하는 직렬 방식의 인터페이스 표준**이다. 기존 병렬 ATA(PATA/IDE)를 대체하여 더 빠르고 얇은 케이블로 데이터를 전송한다.

### 💡 비유: USB 케이블

SATA는 **USB 케이블**과 같다. 옛날에는 큰 케이블이 많았다. 복잡했다. USB는 작고 단순하다. 꽂으면 된다. SATA도 똑같다. 얇은 케이블 하나다. 꽂으면 된다. 핫스왑도 된다. 켰다 켰다.

## Ⅱ. SATA 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                SATA 상세                                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【NCQ (Native Command Queuing)】                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  개념: 디스크가 명령어를 재정렬하여 성능 향상                           │ │
│  │                                                               │ │
│  │  NCQ 없이:                                                             │ │
│  │  요청: 섹터 100 → 섹터 50 → 섹터 150                                   │ │
│  │  헤드: 100 → 50 → 150 (많은 이동)                                      │ │
│  │                                                               │ │
│  │  NCQ 있이:                                                             │ │
│  │  요청: 섹터 100 → 섹터 50 → 섹터 150                                   │ │
│  │  실행: 섹터 50 → 섹터 100 → 섹터 150 (최적 경로)                        │ │
│  │                                                               │ │
│  │  • 최대 32개 명령어 큐잉                                              │ │
│  │  • 회전 최적화                                                        │ │
│  │  • HDD에 효과적, SSD에는 영향 적음                                     │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【AHCI (Advanced Host Controller Interface)】                                  │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  역할: SATA 컨트롤러와 OS 간 표준 인터페이스                             │ │
│  │                                                               │ │
│  │  기능:                                                                │ │
│  │  • NCQ 지원                                                          │ │
│  │  • 핫스왑 지원                                                        │ │
│  │  • 전원 관리                                                         │ │
│  │  • 포트 멀티플라이어                                                  │ │
│  │                                                               │ │
│  │  모드:                                                                │ │
│  │  • IDE: 구형 호환 모드                                                │ │
│  │  • AHCI: 표준 모드 (권장)                                             │ │
│  │  • RAID: RAID 모드                                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【핫스왑】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  정의: 전원을 끄지 않고 장치 교체                                        │ │
│  │                                                               │ │
│  │  조건:                                                                │ │
│  │  • AHCI 모드                                                         │ │
│  │  • OS 지원                                                           │ │
│  │  • 핫스왑 지원 컨트롤러                                                │ │
│  │                                                               │ │
│  │  용도:                                                                │ │
│  │  • 외장 HDD/SSD                                                      │ │
│  │  • 서버 디스크 교체                                                   │ │
│  │  • 백업 장치                                                         │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• SATA: 직렬 ATA 인터페이스
• 세대: 1.5→3→6 Gbps
• 케이블: 7선 데이터, 15선 전원
• NCQ: 명령어 큐잉 (32개)
• AHCI: 컨트롤러 인터페이스
• 핫스왑: 전원 켠 채 교체
• 속도: 최대 600 MB/s
• 용도: 소비자용 HDD/SSD
• 비용: 저렴, 널리 사용
• 비교: NVMe보다 느림, SAS보다 느림
```
