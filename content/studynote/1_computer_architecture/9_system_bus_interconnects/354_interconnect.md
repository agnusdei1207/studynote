# 354. 인터커넥트 (Interconnect)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 칩/프로세서/노드 간 고속 연결 네트워크
> 2. **가치**: 확장성, 낮은 지연, 높은 대역폭
> 3. **융합**: NoC, 메시, 트리, 하이퍼큐브

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

인터커넥트(Interconnect)는 **프로세서 코어, 캐시, 메모리, I/O 등 시스템 구성요소 간 데이터 전송을 위한 통신 네트워크**다. 단순 버스에서 복잡한 네트워크온칩(NoC)까지 다양하다.

### 💡 비유: 도시 도로망

인터커넥트는 **도시 도로망**과 같다. 집이 있다. 회사가 있다. 학교가 있다. 도로로 연결된다. 직선 도로, 교차로, 고속도로. 어디서 어디로든 갈 수 있다. 네트워크다. 인터커넥트도 똑같다. 모든 것을 연결한다.

### 인터커넥트 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                인터커넥트 개요                                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【인터커넥트 종류】                                                              │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. 온칩 인터커넥트 (On-chip Interconnect):                             │ │
│  │     • 코어, 캐시, 메모리 컨트롤러 연결                                  │ │
│  │     • NoC (Network-on-Chip)                                           │ │
│  │     • Mesh, Ring, Crossbar                                            │ │
│  │                                                               │ │
│  │  2. 칩 간 인터커넥트 (Chip-to-Chip Interconnect):                        │ │
│  │     • CPU-CPU, CPU-메모리, CPU-가속기                                   │ │
│  │     • QPI/UPI, HyperTransport, Infinity Fabric                        │ │
│  │                                                               │ │
│  │  3. 시스템 간 인터커넥트 (System-to-System Interconnect):                │ │
│  │     • 서버-서버, 노드-노드                                              │ │
│  │     • Ethernet, InfiniBand, NVLink                                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【토폴로지】                                                                     │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. 버스 (Bus):                                                        │ │
│  │     • 단순, 공유 매체                                                  │ │
│  │     • 확장성 낮음                                                      │ │
│  │                                                               │ │
│  │  2. 링 (Ring):                                                          │ │
│  │     • 순환 구조                                                        │ │
│  │     • Intel 등 사용                                                   │ │
│  │                                                               │ │
│  │  3. 메시 (Mesh):                                                        │ │
│  │     • 2D 그리드                                                        │ │
│  │     • 확장성 높음                                                      │ │
│  │                                                               │ │
│  │  4. 크로스바 (Crossbar):                                                │ │
│  │     • 전면 연결                                                        │ │
│  │     • 고비용, 최고 성능                                                │ │
│  │                                                               │ │
│  │  5. 하이퍼큐브 (Hypercube):                                              │ │
│  │     • 고차원 연결                                                      │ │
│  │     • HPC에서 사용                                                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 인터커넥트: 구성요소 간 통신 네트워크

• 칩 간: QPI/UPI, HyperTransport
• 시스템 간: Ethernet, InfiniBand
• 토폴로지: 버스, 링, 메시, 크로스바

• NUMA: 인터커넥트 기반 메모리
• 확장: 코어/노드 확장성
• NoC: 네트워크온칩
• 성능: 토폴로지에 따라 결정
```
