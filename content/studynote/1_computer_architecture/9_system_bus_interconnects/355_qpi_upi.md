# 355. QPI/UPI (QuickPath/UltraPath Interconnect)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: Intel의 고속 프로세서 간 인터커넥트
> 2. **가치**: 멀티소켓, NUMA, 캐시 일관성
> 3. **융합**: 직렬 전송, 코히런시, 메모리 분산

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

QPI/UPI(QuickPath/UltraPath Interconnect)는 **Intel이 개발한 고속 직렬 인터커넥트로, 멀티소켓 시스템에서 CPU 간 및 CPU와 메모리 간 저지연 고대역폭 통신을 제공**한다.

### 💡 비유: 도시 간 고속도로

QPI/UPI는 **도시 간 고속도로**와 같다. 서울과 부산이 있다. 고속도로로 연결된다. 직접 간다. 빠르다. 많은 차가 지나간다. 두 CPU가 두 도시다. QPI가 고속도로다. 데이터가 오간다.

## Ⅱ. QPI/UPI 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                QPI/UPI 상세                                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【QPI/UPI 특징】                                                                │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. 점대점 (Point-to-Point):                                           │ │
│  │     • 전용 링크                                                        │ │
│  │     • 높은 대역폭                                                      │ │
│  │     • 낮은 지연                                                        │ │
│  │                                                               │ │
│  │  2. 직렬 전송 (Serial):                                                 │ │
│  │     • 20레인 (각 방향 10레인)                                           │ │
│  │     • 차등 신호                                                        │ │
│  │                                                               │ │
│  │  3. 전이중 (Full-Duplex):                                               │ │
│  │     • 동시 송수신                                                      │ │
│  │     • 효율적 대역폭 활용                                                │ │
│  │                                                               │ │
│  │  4. 코히런시 (Coherency):                                               │ │
│  │     • MESIF 프로토콜                                                   │ │
│  │     • 하드웨어 캐시 일관성                                              │ │
│  │                                                               │ │
│  │  ┌─────────────────────────────────────────────────────┐        │ │
│  │  │  세대       속도        대역폠        도입년          │        │ │
│  │  │  ────       ────        ──────        ────            │        │ │
│  │  │  QPI 1.0    6.4 GT/s    25.6 GB/s    2008           │        │ │
│  │  │  QPI 1.1    8.0 GT/s    32.0 GB/s    2011           │        │ │
│  │  │  UPI 1.0    9.6 GT/s    38.4 GB/s    2017           │        │ │
│  │  │  UPI 2.0    10.4 GT/s   41.6 GB/s    2019           │        │ │
│  │  │                                                       │        │ │
│  │  └─────────────────────────────────────────────────────┘        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【MESIF 코히런시 프로토콜】                                                       │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  M (Modified): 수정됨, 이 CPU만 소유                                    │ │
│  │  E (Exclusive): 배타적, 이 CPU만 소유                                   │ │
│  │  S (Shared): 공유됨, 여러 CPU가 소유                                    │ │
│  │  I (Invalid): 무효, 데이터 없음                                         │ │
│  │  F (Forward): 전달 가능한 공유 상태                                      │ │
│  │                                                               │ │
│  │  → QPI/UPI가 캐시 라인 상태를 자동으로 관리                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

```

│  【운영체제】                                                          │

│  • 페이지 마이그레이션: 원격→로컬                                      │
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• QPI/UPI: Intel 고속 인터커넥트
• FSB 대체: 공유 버스 → 점대점
• 속도: 6.4 → 10.4 GT/s
• 대역폭: ~40 GB/s
• 특징: 전이중, 직렬, 코히런시
• NUMA: 비균일 메모리 지원
• MESIF: 캐시 일관성 프로토콜
• 멀티소켓: CPU 간 연결
• 확장: 소켓 추가 용이
• 최적화: NUMA 인식 중요
```
