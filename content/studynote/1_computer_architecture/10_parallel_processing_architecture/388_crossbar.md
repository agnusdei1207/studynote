# 388. 크로스바 스위치 (Crossbar Switch)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 모든 입력을 모든 출력과 직접 연결하는 비차단 스위치
> 2. **가치**: 최고 대역폭, 최저 지연, 비차단
> 3. **융합**: N×N, 비차단, 하드웨어 복잡도 O(N²)

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

크로스바 스위치(Crossbar Switch)는 **N개의 입력 포트와 M개의 출력 포트를 교차점(crosspoint)으로 연결하여, 모든 입력이 동시에 서로 다른 출력으로 연결될 수 있는 비차단(non-blocking) 스위치**다.

### 💡 비유: 전화 교환대

크로스바 스위치는 **전화 교환대**와 같다. 전화기가 여러 대다. 교환원이 있다. 누가 누구와도 연결된다. 동시에 여러 통화. 서로 방해하지 않는다. 크로스바가 똑같다. 모든 입력, 모든 출력, 동시 연결, 비차단.

### 크로스바 스위치 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                크로스바 스위치 개요                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【크로스바 구조】                                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │           출력 (Output)                                        │ │
│  │            O0    O1    O2    O3                                │ │
│  │            │     │     │     │                                 │ │
│  │   I0 ─────┼─────X─────┼─────┼─────  입력                       │ │
│  │            │     │     │     │                                 │ │
│  │   I1 ─────┼─────┼─────X─────┼─────                            │ │
│  │            │     │     │     │                                 │ │
│  │   I2 ─────X─────┼─────┼─────┼─────                            │ │
│  │            │     │     │     │                                 │ │
│  │   I3 ─────┼─────┼─────┼─────X─────                            │ │
│  │            │     │     │     │                                 │ │
│  │                                                               │ │
│  │   X = 교차점 (Crosspoint) = 스위치                              │ │
│  │   • I0 → O2, I1 → O3, I2 → O0, I3 → O1 동시 연결 가능           │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【크로스바 특징】                                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  • 비차단 (Non-blocking): 모든 입력이 서로 다른 출력으로 동시 연결  │ │
│  │  • 최고 대역폭: N개 동시 연결 = N × 링크 대역폭                    │ │
│  │  • 최저 지연: 단일 홉 (1 hop)                                     │ │
│  │  • 하드웨어 복잡도: O(N × M) 교차점                                │ │
│  │  • 비용: N²에 비례                                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 크로스바 스위치 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                크로스바 스위치 상세                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【동작 원리】                                                                   │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. 연결 설정:                                                            │ │
│  │     • 입력 포트에서 원하는 출력 포트의 교차점 활성화                │ │
│  │     • 교차점 = 트랜지스터 스위치                                   │ │
│  │                                                               │ │
│  │  2. 데이터 전송:                                                          │ │
│  │     • 활성화된 경로로 데이터 전송                                   │ │
│  │     • 동시에 여러 경로 전송 가능                                    │ │
│  │                                                               │ │
│  │  3. 연결 해제:                                                            │ │
│  │     • 교차점 비활성화                                              │ │
│  │     • 다른 연결에 사용 가능                                        │ │
│  │                                                               │ │
│  │  제약: 하나의 출력은 하나의 입력만 연결 가능                          │ │
│  │       (여러 입력 → 하나의 출력 = 충돌)                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  장점:                                                                  │ │
│  │  • 비차단: 항상 연결 가능                                          │ │
│  │  • 최고 성능: 최대 대역폭, 최소 지연                               │ │
│  │  • 단순 제어: 교차점 on/off                                       │ │
│  │  • 예측 가능: 일정한 성능                                          │ │
│  │                                                               │ │
│  │  단점:                                                                  │ │
│  │  • 비용: N² 복잡도 (교차점 수)                                    │ │
│  │  • 확장성: 포트 증가 시 비용 급증                                  │ │
│  │  • 전력: 많은 교차점                                               │ │
│  │  • 면적: 큰 실리콘 면적                                            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【활용 분야】                                                                  │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  • 소규모 스위치: 4-32 포트                                        │ │
│  │  • CPU 내부: 코어-캐시 연결                                        │ │
│  │  • 메모리 컨트롤러: 채널 간 스위칭                                  │ │
│  │  • 네트워크 스위치: 고성능 스위치                                   │ │
│  │  • FPGA: 프로그래밍 가능 연결                                       │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 크로스바 스위치 × 타 과목 융합

### 과목 융합 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                크로스바 스위치 × 타 과목 융합                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【컴퓨터 구조】                                                      │

│  • 메모리: 채널 스위칭                                               │
│  • NoC: 소규모 온칩 스위치                                           │
│  • 프로세서: 내부 데이터패스                                         │
│                                                                     │
│  【네트워크】                                                          │
│  • 스위칭: 비차단 스위치                                             │
│  • 패브릭: 고속 네트워크 패브릭                                      │
│  • 데이터센터: TOR 스위치                                            │
│  • 전송: 동시 다중 연결                                              │
│                                                                     │
│  【디지털 회로】                                                        │
│  • 교차점: 트랜지스터 스위치                                         │
│  • 제어: 디코더, 어드레싱                                            │
│  • 아비트레이션: 충돌 해결                                           │
│  • 복잡도: O(N²)                                                    │
│                                                                     │
│  【성능】                                                              │
│  • 대역폭: N × 링크 대역폭                                          │
│  • 지연: 1 홉                                                       │
│  • 비용: N² 교차점                                                  │
│  • 확장: 비용 급증                                                  │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  크로스바 비용 계산:                                                               │ │
│  │  ──────────────────────                                              │ │
│  │  포트 수    교차점 수    상대 비용                                   │ │
│  │  ──────     ────────     ────────                                   │ │
│  │  4          16           1x                                          │ │
│  │  8          64           4x                                          │ │
│  │  16         256          16x                                         │ │
│  │  32         1024         64x                                         │ │
│  │  64         4096         256x                                        │ │
│  │                                                               │ │
│  │  → 포트 2배 = 비용 4배                                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 크로스바: 모든 입력-출력 직접 연결
• 특징: 비차단, 최고 대역폭, 최저 지연
• 구조: N×M 교차점 그리드
• 복잡도: O(N×M) 교차점
• 장점: 최고 성능, 단순 제어
• 단점: 비용 N², 확장성 제한
• 활용: 소규모 스위치, CPU 내부
• 대안: 다단 연결망 (대규모)
• 비교: 크로스바 vs 버스 vs MIN
• 현대: 고성능 스위치, NoC
```
