# 257. NAND 플래시 (NAND Flash)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 셀이 직렬 연결된 고밀도 비휘발성 메모리
> 2. **가치**: SSD, USB, 스마트폰 저장의 핵심
> 3. **융합**: 3D NAND, SLC/MLC/TLC/QLC

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

NAND 플래시는 **메모리 셀이 NAND 게이트처럼 직렬로 연결된 플래시 메모리**이다. 높은 밀도와 낮은 비용으로 대용량 저장에 최적화되어 있어 SSD, USB, SD카드, 스마트폰 저장에 주로 사용된다.

### 💡 비유: 비닐봉지 사슬

NAND 플래시는 **비닐봉지 사슬**과 같다. 비닐봉지들을 줄줄이 연결해 놓았다. 하나를 꺼내려면 전체를 확인해야 한다. 하지만 많은 양을 작은 공간에 담을 수 있다.

## Ⅱ. NAND 플래시 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                NAND 플래시 상세                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【NAND 동작 특성】                                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 읽기: 페이지 단위 (4~16KB)                                       │ │
│  │     • 랜덤 읽기: 25~100μs                                          │ │
│  │     • 순차 읽기: 50~100MB/s                                        │ │
│  │                                                               │ │
│  │  2. 쓰기: 페이지 단위 (빈 페이지만)                                    │ │
│  │     • 프로그램 시간: 200~700μs                                       │ │
│  │     • 순차 쓰기: 50~200MB/s                                        │ │
│  │                                                               │ │
│  │  3. 소거: 블록 단위 (필수)                                           │ │
│  │     • 소거 시간: 2~5ms                                              │ │
│  │     • 쓰기 전소거 (Erase-Before-Write)                              │ │
│  │                                                               │ │
│  │  4. 제한된 수명:                                                    │ │
│  │     • P/E 사이클 (프로그램/소거): 1,000~100,000회                    │ │
│  │     • 마모 평준화로 수명 연장                                         │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【3D NAND】                                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  수직 적층으로 밀도 향상                                              │ │
│  │                                                               │ │
│  │  2D (Planar) NAND:                                                │ │
│  │  ┌───────────────────────┐                                        │ │
│  │  │ 셀 셀 셀 셀 셀 셀 셀 셀 │ ← 1층                                  │ │
│  │  └───────────────────────┘                                        │ │
│  │  → 수평 확장 한계                                                   │ │
│  │                                                               │ │
│  │  3D (Vertical) NAND:                                               │ │
│  │  ┌───────────────────────┐                                        │ │
│  │  │ 셀 셀 셀 셀 셀 셀 셀 셀 │ ← 96층                                 │ │
│  │  │ 셀 셀 셀 셀 셀 셀 셀 셀 │ ← ...                                  │ │
│  │  │ 셀 셀 셀 셀 셀 셀 셀 셀 │ ← 1층                                  │ │
│  │  └───────────────────────┘                                        │ │
│  │  → 수직 적층으로 밀도 10배+                                          │ │
│  │                                                               │ │
│  │  적층 수: 32 → 64 → 96 → 128 → 176 → 232층                         │ │
│  │  → 높을수록 같은 면적에 더 많은 용량                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【NAND 용량 추이】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  연도    적층 수    다이 용량    SSD 용량                            │ │
│  │  ────    ──────    ────────    ────────                            │ │
│  │  2016    48층      256Gb      2TB                                 │ │
│  │  2018    96층      512Gb      4TB                                 │ │
│  │  2020    128층     1Tb        8TB                                 │ │
│  │  2022    176층     1Tb        16TB                                │ │
│  │  2024    232층     2Tb        30TB+                               │ │
│  │  → 지속적 밀도 증가                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• NAND 플래시: 셀이 직렬 연결된 고밀도 비휘발성 메모리
• 구조: 셀 → 페이지(4~16KB) → 블록(256KB~8MB) → 플레인 → 다이
• 동작: 읽기/쓰기(페이지), 소거(블록)
• 장점: 높은 밀도, 낮은 비용, 빠른 순차 접근
• 단점: 느린 랜덤 접근, 블록 단위 소거, 제한된 수명
• 3D NAND: 수직 적층으로 밀도 향상 (32→232층)
• 셀 타입: SLC/MLC/TLC/QLC (밀도↑, 속도/내구성↓)
• 용도: SSD, USB, SD카드, 스마트폰
```
