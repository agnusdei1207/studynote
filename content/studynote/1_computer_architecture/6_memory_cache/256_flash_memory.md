# 256. 플래시 메모리 (Flash Memory)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전기적으로 소거/재프로그래밍 가능한 비휘발성 메모리
> 2. **가치**: SSD, USB, 스마트폰 저장의 핵심 기술
> 3. **융합**: NAND vs NOR, SLC/MLC/TLC/QLC

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

플래시 메모리(Flash Memory)는 **전기적으로 블록 단위로 소거하고 재프로그래밍할 수 있는 비휘발성 메모리**이다. EEPROM의 일종으로, 현대의 SSD, USB 드라이브, 스마트폰 저장에 핵심적으로 사용된다.

### 💡 비유: 화이트보드

플래시 메모리는 **화이트보드**와 같다. 글을 쓸 수 있고, 지울 수도 있다. 지울 때는 한 글자씩이 아니라 영역 전체를 지운다. 전원이 꺼져도 글은 그대로 남아 있다.

## Ⅱ. 플래시 메모리 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                플래시 메모리 상세                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【셀 타입 (SLC/MLC/TLC/QLC)】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  SLC (Single Level Cell): 1비트/셀                                 │ │
│  │  ┌───┐                                                         │ │
│  │  │ 0 │ 또는 │ 1 │                                               │ │
│  │  └───┘     └───┘                                                │ │
│  │  → 빠름, 높은 내구성, 비쌈                                           │ │
│  │                                                               │ │
│  │  MLC (Multi Level Cell): 2비트/셀                                  │ │
│  │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐                                  │ │
│  │  │ 00  │ │ 01  │ │ 10  │ │ 11  │                                  │ │
│  │  └─────┘ └─────┘ └─────┘ └─────┘                                  │ │
│  │  → 중간 속도, 중간 내구성, 중간 가격                                   │ │
│  │                                                               │ │
│  │  TLC (Triple Level Cell): 3비트/셀                                 │ │
│  │  → 8 상태 (000~111)                                               │ │
│  │  → 느림, 낮은 내구성, 저렴                                          │ │
│  │                                                               │ │
│  │  QLC (Quad Level Cell): 4비트/셀                                   │ │
│  │  → 16 상태 (0000~1111)                                            │ │
│  │  → 매우 느림, 매우 낮은 내구성, 매우 저렴                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【셀 타입 비교】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  타입    비트/셀  쓰기 속도  내구성(PE)     용도                      │ │
│  │  ────    ────────  ────────  ───────────    ──────                    │ │
│  │  SLC     1         매우 빠름   ~100,000     기업용 SSD                │ │
│  │  MLC     2         빠름      ~10,000       고급 SSD                  │ │
│  │  TLC     3         보통      ~3,000        일반 SSD                  │ │
│  │  QLC     4         느림      ~1,000        보급형 SSD                 │ │
│  │  → 비트/셀 증가 = 밀도 증가, 속도/내구성 감소                           │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【플래시 메모리 구조】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  페이지 (Page): 읽기/쓰기 단위 (4KB~16KB)                            │ │
│  │  블록 (Block): 소거 단위 (256KB~4MB = 여러 페이지)                    │ │
│  │  플레인 (Plane): 병렬 처리 단위                                       │ │
│  │  다이 (Die): 독립 동작 단위                                          │ │
│  │                                                               │ │
│  │  특성:                                                            │ │
│  │  • 읽기: 페이지 단위                                                │ │
│  │  • 쓰기: 페이지 단위 (빈 페이지만)                                    │ │
│  │  • 소거: 블록 단위 (필수)                                            │ │
│  │  • 쓰기 증폭: 데이터 갱신 시 전체 블록 복사                            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 플래시 메모리: 전기적 소거/재프로그래밍 가능한 비휘발성 메모리
• 종류: NOR (펌웨어용), NAND (대용량 저장용)
• 셀 타입: SLC(1bit) → MLC(2bit) → TLC(3bit) → QLC(4bit)
• 구조: 페이지(읽기/쓰기) → 블록(소거) → 플레인 → 다이
• 특성: 비휘발성, 빠른 읽기, 블록 단위 소거, 제한된 쓰기 횟수
• 최적화: FTL, 마모 평준화, 가비지 컬렉션, TRIM
• 용도: SSD, USB, SD카드, 스마트폰, 임베디드
• 발전: 3D NAND, PLC (5bit/cell)
```
