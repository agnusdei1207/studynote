# 253. DDR SDRAM (Double Data Rate)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 클럭 상승/하강 에지 모두에서 데이터 전송
> 2. **가치**: SDR 대비 2배 대역폭, 같은 클럭 주파수
> 3. **융합**: DDR2/3/4/5 발전, 프리패치, 채널

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

DDR SDRAM(Double Data Rate SDRAM)은 **클럭의 상승 에지와 하강 에지 모두에서 데이터를 전송하여 대역폭을 2배로 늘린 SDRAM**이다. 같은 클럭 주파수로 SDR 대비 2배의 데이터 전송이 가능하다.

### 💡 비유: 양방향 도로

DDR은 **양방향 도로**와 같다. SDR은 왕복 1차선(한 번에 한 방향만)이다. DDR은 왕복 2차선(양쪽 동시)이다. 같은 시간에 2배의 차량이 지나갈 수 있다.

## Ⅱ. DDR 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                DDR 상세                                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【DDR 프리패치】                                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  내부 클럭보다 낮은 속도로 동작하면서 높은 대역폭 달성                   │ │
│  │                                                               │ │
│  │  DDR:  2n 프리패치 (2개 데이터를 한 번에)                            │ │
│  │  DDR2: 4n 프리패치 (4개 데이터를 한 번에)                            │ │
│  │  DDR3: 8n 프리패치 (8개 데이터를 한 번에)                            │ │
│  │  DDR4: 8n 프리패치 (뱅크 그룹 추가)                                  │ │
│  │  DDR5: 16n 프리패치 (2 채널/모듈)                                   │ │
│  │                                                               │ │
│  │  DDR4-3200 예시:                                                  │ │
│  │  내부 클럭: 400 MHz                                                │ │
│  │  I/O 클럭: 1600 MHz                                               │ │
│  │  프리패치: 8n → 8비트 × 400 MHz × 2 = 6400 MT/s                     │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【DDR 세대별 특징】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  세대    전압    프리패치   뱅크   최대용량/모듈   특징                  │ │
│  │  ────    ────    ──────    ────    ───────────    ──────              │ │
│  │  DDR    2.5V     2n       4      2GB          최초 DDR               │ │
│  │  DDR2   1.8V     4n       4/8    4GB          저전압, 고속            │ │
│  │  DDR3   1.5V     8n       8      16GB         더 낮은 전압            │ │
│  │  DDR4   1.2V     8n       16     128GB        뱅크 그룹               │ │
│  │  DDR5   1.1V     16n      32     256GB        2 채널/모듈             │ │
│  │  → 전압 감소, 프리패치 증가, 용량 증가                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【DDR4 vs DDR5】                                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  특성           DDR4              DDR5                           │ │
│  │  ──────────     ───────────────   ────────────────                │ │
│  │  속도           1600~3200 MT/s    4800~8400 MT/s                   │ │
│  │  전압           1.2V              1.1V                            │ │
│  │  채널/모듈       1                2 (독립)                          │ │
│  │  버스트 길이      8                16                              │ │
│  │  뱅크           16 (4 그룹)        32 (8 그룹)                       │ │
│  │  리프레시        전체              Same Bank Refresh               │ │
│  │  ECC           선택               온디에이                          │ │
│  │  → DDR5: 2배 대역폭, 더 효율적                                       │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• DDR: 클럭 상승/하강 에지 모두 데이터 전송
• 장점: SDR 대비 2배 대역폭
• 프리패치: 2n(DDR) → 4n(DDR2) → 8n(DDR3/4) → 16n(DDR5)
• 속도 표기: DDR4-3200 = 3200 MT/s = 25.6 GB/s/채널
• 전압: 2.5V(DDR) → 1.2V(DDR4) → 1.1V(DDR5)
• 특수 변형: LPDDR(모바일), GDDR(GPU), HBM(AI)
• 최적화: 듀얼 채널, XMP/EXPO, 낮은 타이밍
• 발전: DDR5로 6400 MT/s+ 달성
```
