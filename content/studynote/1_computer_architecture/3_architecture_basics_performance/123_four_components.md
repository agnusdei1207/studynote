# 123. 컴퓨터의 4대 구성요소

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 폰 노이만 구조의 핵심으로, 모든 컴퓨터는 연산 장치(CU+ALU), 기억 장치(Memory), 입출력 장치(I/O)로 구성
> 2. **가치**: 하드웨어 설계의 기본 단위로, CPU-GPU-NPU-메모리-스토리지의 역할 분담 체계
> 3. **융합**: 시스템 프로그래밍, 운영체제 설계, 임베디드 개발의 기초 개념

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

컴퓨터의 4대 구성요소는 입력(Input), 처리(Process), 출력(Output), 저장(Storage) 기능을 수행하는 하드웨어 단위다. 전통적으로 연산 장치(ALU), 제어 장치(CU), 기억 장치(Memory), 입출력 장치(I/O)로 분류한다.

### 💡 비유: 요리 식당

컴퓨터는 **요리 식당**과 같다. 주방장(CU)이 주문을 받아 지시하고, 요리사(ALU)가 실제로 요리하며, 창고(Memory)에 재료를 보관하고, 홀 서빙(I/O)이 주문을 받고 음식을 내어놓는다.

### 등장 배경

| 시기 | 발전 |
|:---:|:---|
| 1945 | 폰 노이만 구조 제안 |
| 1971 | Intel 4004 (최초의 마이크로프로세서) |
| 현재 | SoC로 4대 요소 통합 |

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 구성요소별 상세

| 구성요소 | 역할 | 주요 부품 | 데이터 흐름 |
|:---|:---|:---|:---|
| **연산 장치 (ALU)** | 산술/논리 연산 | 가산기, 시프터, 비교기 | 피연산자 → 결과 |
| **제어 장치 (CU)** | 명령어 해석, 제어 신호 | 명령어 디코더, 제어기 | 명령어 → 제어신호 |
| **기억 장치** | 데이터/명령어 저장 | 레지스터, 캐시, RAM | 주소 → 데이터 |
| **입출력 장치** | 외부와 데이터 교환 | 키보드, 모니터, NIC | 외부 ↔ 내부 |

### 👶 어린이를 위한 3줄 비유 설명

**개념**: 컴퓨터는 4가지 일을 하는 기계예요! 생각하는 뇌(CPU), 기억하는 창고(메모리), 정보를 받고 주는 손발(입출력)이 있어요.

**원리**: 요리 식당처럼, 주방장이 지시하고(CU), 요리사가 만들고(ALU), 창고에 재료를 두고(Memory), 서빈홀이 음식을 내어놓아요(I/O).

**효과**: 이 4가지만 있으면 어떤 계산이든 할 수 있어요! 스마트폰부터 슈퍼컴퓨터까지 모두 이 원리예요!
