# 131. SoC (System on Chip)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU, GPU, 메모리 컨트롤러, I/O, 가속기 등 시스템 전체를 단일 칩에 집적한 초고밀도 반도체
> 2. **가치**: 부품 간 통신 지연 최소화, 전력 효율 극대화, 모바일/IoT 기기의 핵심 엔진
> 3. **융합**: AP(Application Processor)로 진화, 스마트폰·자동차·AI 엣지 디바이스에 필수

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

SoC(System on Chip)는 컴퓨터나 전자시스템의 모든 핵심 구성요소를 단일 집적회로(IC) 칩에 통합한 반도체다. CPU뿐만 아니라 GPU, 메모리 컨트롤러, 통신 모듈, 각종 가속기(NPU, DSP) 등을 하나의 칩에 집적한다.

### 💡 비유: 원룸 아파트 vs 단독 주택

SoC는 **원룸 아파트**와 같다. 침실, 주방, 욕실, 거실이 하나의 공간에 효율적으로 배치된다. 이동 시간이 짧고 공간을 절약한다. 반면 MPU+별도 칩들은 **단독 주택**처럼 각 방이 따로 있어 이동(통신)에 시간이 걸린다.

### SoC 구성요소 상세

| 구성요소 | 역할 | 예시 |
|:---|:---|:---|
| **CPU** | 범용 연산 | ARM Cortex-A, Apple Firestorm |
| **GPU** | 그래픽/병렬 연산 | Mali, Adreno, Apple GPU |
| **NPU** | AI/ML 추론 가속 | Neural Engine, Hexagon |
| **DSP** | 디지털 신호 처리 | Hexagon DSP, Cadence |
| **ISP** | 이미지 신호 처리 | 카메라 입력 처리 |
| **Video** | 비디오 인코딩/디코딩 | H.264, HEVC, AV1 |
| **Memory Ctrl** | 외부 메모리 제어 | DDR4/LPDDR5 컨트롤러 |
| **NoC** | 칩 내부 통신망 | ARM CCI, Mesh |
| **PMU** | 전력 관리 | DVFS, 클럭 게이팅 |

| 특성 | MPU | MCU | SoC |

| **유연성** | 높음 | 낮음 | 낮음 |
| **개발 복잡도** | 높음 | 낮음 | 높음 |
| **용도** | PC/서버 | 임베디드/IoT | 모바일/자동차 |

## Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

### 실무 시나리오

**시나리오 1: 스마트폰 개발**
- 요구: 고성능, 저전력, 5G, AI
- 선택: Qualcomm Snapdragon 8 Gen 3
- 이유: 통합 5G 모뎀, 우수한 GPU, NPU

**시나리오 2: 자율주행 ECU**
- 요구: 실시간 AI 추론, 다중 센서
- 선택: NVIDIA Orin X
- 이유: 254 TOPS, 다중 카메라 입력

**시나리오 3: AI 엣지 디바이스**
- 요구: 소형, 저전력, AI 추론
- 선택: Google Coral Edge TPU
- 이유: 4 TOPS, 2W 전력

**시나리오 4: 스마트홈 허브**
- 요구: WiFi, 저비용, 음성 처리
- 선택: ESP32-S3
- 이유: $3, WiFi/BT, AI 가속

|:---|:---:|:---:|
| **전력 효율** | 기준 | 30~50% 향상 |
| **통신 지연** | 높음 | 낮음 (~10× 감소) |
| **PCB 크기** | 큼 | 작음 |
| **BOM 비용** | 높음 | 낮음 |

### SoC 트렌드

- **Chiplet**: 기능별 칩을 패키지에 통합 (Intel, AMD)
- **2.5D/3D 적층**: HBM, CoWoS (TSMC)
- **RISC-V**: 오픈소스 CPU 코어 채택 증가
- **AI 가속**: NPU 성능 10× 향상 목표
