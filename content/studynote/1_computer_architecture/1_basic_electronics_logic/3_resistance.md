# 저항 (Resistance)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 저항 (Resistance)은 도체 내에서 전하(전자)의 이동을 물리적으로 방해하는 성질 ($R = \rho L/A$)로, 전기 에너지를 열 에너지로 변환시켜 손실을 유발하고 회로 내 전압 강하 (IR Drop)를 일으키는 주범이다.
> 2. **가치**: 반도체 초미세 공정(3nm 이하)에서 트랜지스터 크기는 한계까지 줄어들지만, 이를 연결하는 구리 배선 폭이 원자 단위로 좁아짐에 따라 배선 저항과 기생 저항이 폭증하여, 신호 지연 (RC Delay)이 전체 프로세서의 클럭 주파수를 가로막는 최대 병목(Bottleneck)이 되었다.
> 3. **융합**: 저항은 단순히 없애야 할 악재만은 아니다. 정적 노이즈 마진 (SNM)을 결정하고, 풀업/풀다운 저항 (Pull-up/Pull-down)을 통해 플로팅 (Floating) 상태를 방지하며, 종단 저항 (Termination)으로 고속 신호의 무결성 (Signal Integrity)을 지켜내는 필수 설계 요소로도 작용한다.

---

## Ⅰ. 개요 (Context & Background)

- **개념**: 저항 (Resistance)은 전압(전위차)이 가해져 전자가 이동할 때, 매질 내부의 원자핵이나 다른 전자들과 충돌하며 겪는 "흐름의 방해 정도"를 수치화한 것이다. 단위는 옴 ($\Omega$, Ohm)을 사용하며, 전기 회로에서 가장 기본이 되는 옴의 법칙 ($V = I \cdot R$)으로 정의된다. 시스템 아키텍처 관점에서 저항은 전류를 제어하는 '유용한 보호/제어 소자'인 동시에, 미세 배선에서 발생하는 '치명적인 기생 성분 (Parasitic Element)'이라는 이중적 성격을 띤다.

- **💡 비유**: 저항은 자동차가 달리는 **"도로의 마찰력과 병목 구간"**과 같다. 도로가 좁고 거칠수록(저항이 클수록) 자동차(전자)는 속도를 내기 힘들고 엔진은 과열되며 땀(발열)을 흘린다. 하지만 톨게이트(의도적 저항)를 세우면 차들이 엉뚱한 길로 폭주하는 것을 막아주는 안전장치가 되기도 한다.

- **등장 배경 및 기술적 딜레마 (Size Effect)**:
  1. **고전적 회로 시대**: 과거 매크로 단위의 회로에서는 저항 소자를 직접 기판에 납땜하여 전압을 나누고 전류를 통제하는 아날로그적 튜닝이 주를 이루었다.
  2. **미세 공정의 저주 (Interconnect Bottleneck)**: 무어의 법칙에 따라 트랜지스터는 작고 빨라졌다. 그러나 칩 내부에 거미줄처럼 깔린 배선(Interconnect)은 두께와 폭이 수십 나노미터 이하로 좁아지면서 저항이 기하급수적으로 폭증했다. 트랜지스터 연산은 순식간에 끝나는데 데이터가 배선을 타고 이동하는 데 시간이 다 걸리는 배선 지연 (Wire Delay) 현상이 발생한 것이다.

배선 폭 (Node)이 줄어듦에 따라 배선 저항이 어떻게 비선형적으로 급증하는지 '크기 효과 (Size Effect)'의 양상을 시각화하면 다음과 같다.

```text
  ┌───────────────────────────────────────────────────────────────┐
  │           공정 미세화(Node Scaling)에 따른 배선 저항 폭증 현상        │
  ├───────────────────────────────────────────────────────────────┤
  │                                                               │
  │  단위 길이당 배선 저항 (R)                                      │
  │     ▲                                       /                 │
  │     │                                      /                  │
  │     │                                     /                   │
  │     │                                    /  ◀─ Size Effect     │
  │     │                                   /   (표면 산란 + 결정립 산란) │
  │     │                                  /                      │
  │     │                               __/                       │
  │     │                       _______/                          │
  │     │  ____________________/                                  │
  │     └─┴─────────┴─────────┴─────────┴─────────┴─────▶ 공정 노드 │
  │       130nm     65nm      28nm      10nm      3nm             │
  │                                                               │
  │  해석: 배선 폭이 전자의 평균 자유 행로(Mean Free Path, 약 39nm)보다 │
  │        좁아지는 10nm 이하부터는, 전자가 금속 내부를 직선으로 통과하지│
  │        못하고 벽면에 계속 부딪혀 비저항(ρ) 자체가 폭발적으로 증가함.  │
  └───────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 그래프는 10nm 이하 초미세 공정에서 TSMC와 삼성전자 등 파운드리(Foundry)가 직면한 가장 큰 물리적 장벽을 보여준다. 배선 폭이 구리(Cu) 전자의 평균 자유 행로보다 좁아지면, 전자가 벽면에 부딪히는 '표면 산란 (Surface Scattering)'과 결정 경계면에서 부딪히는 '결정립 산란 (Grain Boundary Scattering)' 빈도가 폭증한다. 결국 단면적이 줄어드는 기하학적 저항 증가분을 넘어, 물질 자체의 비저항($\rho$)이 나빠지는 최악의 상황 (크기 효과)이 발생하며, 이는 칩 속도 향상을 가로막는 절대적 병목이 된다.

- **📢 섹션 요약 비유**: 도로가 너무 좁아지다 못해 자동차(전자)가 도로 양옆 벽에 계속 긁히며(산란 현상) 속도를 전혀 낼 수 없는 극심한 정체 상태에 도달한 것이 현대 반도체의 가장 큰 골칫거리입니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 구성 요소 및 저항의 종류

반도체 칩 내부에는 다양한 형태의 저항이 존재하며 각각 다른 역할을 수행하거나 다른 문제를 일으킨다.

| 요소명 | 역할 및 발생 위치 | 내부 원리 | 관련 문제 / 최신 기술 | 비유 |
|:---|:---|:---|:---|:---|
| **배선 저항 (Line Resistance)** | 칩 내부 신호와 전력을 전달하는 금속망 | 물질 고유의 비저항에 비례 ($R = \rho L/A$) | RC Delay, 구리 대체재 (코발트, 루테늄) 연구 | 기나긴 고속도로의 아스팔트 마찰력 |
| **접촉 저항 (Contact Resistance)** | 서로 다른 층이나 물질 간의 연결부 (Via) | 이종 물질 간 계면의 전자 이동 장벽 (Barrier) | 3D 패키징 및 수직 적층의 최대 발열/속도 병목 | 요금소(톨게이트) 통과 시의 지체 현상 |
| **채널 저항 (Channel Resistance)** | 켜진(ON) 트랜지스터 내부 (소스~드레인) | 게이트 전압($V_{gs}$)에 의해 유도된 전자의 이동도 | 스위칭 전류($I_{on}$) 크기 제한, 트랜지스터 발열 | 댐의 수문을 열었을 때 물이 쏟아지는 통로의 폭 |
| **풀업/풀다운 저항 (Pull-up/down)** | 디지털 입력 신호의 부동(Floating) 방지 | 전원($V_{dd}$)이나 접지($V_{ss}$)로 미세 전류를 흘림 | 신호 무결성 및 노이즈 마진 확보 | 문이 바람에 덜컹거리지 않게 잡아주는 스프링 |
| **종단 저항 (Termination Resistor)** | 고속 신호의 반사(Reflection) 에너지 흡수 | 전송 선로의 특성 임피던스($Z_0$)와 값을 물리적으로 일치시킴 | ODT (On-Die Termination), 고속 메모리 (DDR5) 필수 기술 | 파도가 절벽에 부딪혀 튕기지 않게 에너지를 흡수하는 방파제 |

### 저항의 물리적 결정 요인 및 면저항 (Sheet Resistance)

저항값은 물질의 고유한 성질인 비저항 ($\rho$), 도체의 길이 ($L$), 그리고 단면적 ($A = W \times t$)에 의해 결정된다. 실무 레이아웃 설계에서는 이를 단순화한 '면저항 (Sheet Resistance, $R_s$)' 개념을 주로 사용한다.

```text
 ┌───────────────────────────────────────────────────────────────┐
 │       도체의 저항 결정 요소와 면저항 (Sheet Resistance, Rs)       │
 ├───────────────────────────────────────────────────────────────┤
 │                                                               │
 │             ◀─────────────── L (길이) ───────────────▶        │
 │           ┌───────────────────────────────────────────┐       │
 │           │                                           │ ▲    │
 │   전류 ──▶│         물질의 고유 비저항 (ρ)             │ │ t   │
 │   (I)     │                                           │ ▼    │
 │           └───────────────────────────────────────────┘       │
 │             ▲                                                 │
 │             │ W (폭, Width)                                   │
 │             ▼                                                 │
 │                                                               │
 │   * 기본 공식: R = ρ * (L / A) = ρ * (L / W*t)               │
 │   * 면저항 (Rs) = ρ / t  (두께가 일정한 박막의 고유 저항값, Ω/sq)   │
 │   * 실무 공식: R = Rs * (L / W) = Rs * (Square 갯수)           │
 └───────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 도체 내의 전자는 이동하면서 금속 격자 원자들과 충돌하여 전기 에너지를 잃는다. 길이가 길어질수록($L$) 충돌 횟수가 비례하여 늘고, 폭($W$)이나 두께($t$)가 좁아질수록 전자가 지나갈 통로가 협소해져 저항이 커진다. 반도체 칩은 수많은 층(Layer)의 얇은 막으로 만들어지며 각 층의 두께($t$)는 이미 공정에서 고정되어 있다. 따라서 설계자는 비저항을 두께로 나눈 '면저항($R_s$)'이라는 상수를 받아, 오직 배선을 얼마나 좁고($W$) 길게($L$) 그릴 것인지만 결정하여 전체 저항을 계산한다. 폭이 1um이고 길이가 10um인 선은 정사각형(Square) 10개가 직렬로 연결된 것과 같아 $10 \times R_s$의 저항을 갖게 된다.

### 심층 동작 원리: 기생 저항과 신호 지연 (RC Delay)

컴퓨터 내부의 수십 억 개 트랜지스터를 잇는 신호선은 완벽한 전도체가 아니다. 선 자체의 저항 ($R$)과 배선-기판 사이의 커패시턴스 ($C$)가 결합된 로우패스 필터(Low-pass Filter)로 작용하며, 이를 RC 지연 (RC Delay)이라 부른다.

```text
  ┌───────────────────────────────────────────────────────────────┐
  │             RC Delay에 의한 디지털 신호의 파형 왜곡 (왜곡도)        │
  ├───────────────────────────────────────────────────────────────┤
  │                                                               │
  │  전압(V)                                                      │
  │    ▲                                                          │
  │  1 ├─ ─ ─ ─ ─ ─┌───────────────┐  ◀─ 송신단의 이상적 입력 파형 │
  │    │           │               │                              │
  │    │          /│               │                              │
  │ 0.5├─ ─ ─ ─ ─/─┼─ ─ ─ ─ ─ ─ ─ ─┼─ ─ ─ 수신단의 논리 임계치 (Vth) │
  │    │        /  │               │╲                             │
  │    │      /    │               │  ╲  ◀─ R과 C를 거치며 뭉개진 출력│
  │  0 └───/──────┴───────────────┴────╲─────────▶ 시간 (t)    │
  │        |← τ →|                                               │
  │                                                               │
  │   - 시정수 타우(τ) = R * C : 목표 전압의 약 63%에 도달하는 시간      │
  │   - 저항(R)이 크면 파형이 완만해져 '1'을 인식하는 시점이 심각하게 지연됨│
  └───────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 송신 코어에서 완벽한 사각형 펄스(Square Wave)를 쏘아도, 저항($R$)이라는 마찰과 커패시터($C$)라는 충전 웅덩이를 거치면서 수신단에는 둥글게 뭉개진 지수함수 파형이 도착한다. 클럭 주파수를 4GHz, 5GHz로 끌어올리려면 이 뭉개진 파형이 0.5V(임계치)를 통과하는 시간이 극도로 짧아야 한다. 그러나 미세 공정으로 선폭이 좁아져 저항($R$)이 커지면 신호가 임계치를 넘는 데 걸리는 시간($\tau$)이 길어지고, 다음 클럭이 치기 전에 데이터가 도착하지 못하는 타이밍 위반 (Setup Time Violation)이 발생해 시스템이 다운된다.

- **📢 섹션 요약 비유**: 직선으로 뻗은 고속도로에서는 짐(신호)을 금방 나를 수 있지만, 바닥이 울퉁불퉁하고(저항 R) 중간중간 화물을 부려놓아야 하는 창고(커패시터 C)가 많으면 목적지에 도착하는 시간이 하염없이 지연되는 이치입니다.

## Ⅲ. 융합 비교 및 다각도 분석

### 1. 배선 저항 (Line Resistance) vs 접촉 저항 (Contact Resistance) 심층 비교

3D 패키징 및 다층 배선 구조가 표준이 된 현대 아키텍처에서는 이 두 저항의 관리가 칩 수율을 결정한다.

| 비교 항목 | 배선 저항 (Line Resistance) | 접촉 저항 (Contact Resistance) |
|:---|:---|:---|
| **발생 위치** | M1, M2 등 수평으로 뻗은 금속 배선 층 내부 | 층과 층을 잇는 수직 비아(Via) 또는 실리콘과 금속의 경계 (Silicide) |
| **지배적 요인** | 배선의 길이($L$)와 폭($W$) 비례 (기하학적 요소) | 접촉 면적의 크기와 계면 물질의 장벽 에너지 (물리·화학적 요소) |
| **스케일링 부작용** | 선폭 감소에 따라 점진적(Size Effect 후 급진적) 증가 | 미세화로 접촉 면적이 줄어들면 기하급수적으로 폭증 (최대 발열원) |
| **최신 개선 기술** | 배선 높이를 키우는 High-Aspect-Ratio 구조, 신소재 | 배리어층(Barrier Layer) 두께 최소화, 셀프 얼라인(Self-aligned) 비아 |

### 2. 풀업 (Pull-up) 저항 vs 풀다운 (Pull-down) 저항 비교

MCU나 프로세서의 입력 핀이 어디에도 연결되지 않은 부동 상태 (Floating)를 방지하기 위해 논리적으로 의도된 저항이다.

| 항목 | 풀업 (Pull-up) 저항 | 풀다운 (Pull-down) 저항 |
|:---|:---|:---|
| **연결 위치** | 신호 입력단과 Vdd(전원) 사이에 배치 | 신호 입력단과 Vss(접지) 사이에 배치 |
| **스위치 개방 시 상태**| 입력 핀을 안정적인 '1' (High) 상태로 고정 | 입력 핀을 안정적인 '0' (Low) 상태로 고정 |
| **저항값 크기** | 보통 10k$\Omega$ 이상 (강한 전류에 의한 전력 낭비 방지) | 보통 10k$\Omega$ 이상 |
| **통신 프로토콜 적용** | I2C 등 Open-drain (오픈 드레인) 버스 구조에 필수 | 일반적인 제어 핀의 초기값 설정에 사용 |

- **과목 융합 관점 (네트워크 & 보안)**:
  - **네트워크**: 물리 계층(L1)에서 동축 케이블이나 UTP 케이블의 도체 저항은 거리에 비례해 신호의 진폭을 깎아먹는 감쇠 (Attenuation) 현상을 일으켜, 최대 전송 거리(예: 100m)를 물리적으로 제한한다.
  - **하드웨어 보안**: 공격자가 I2C 통신선 등을 인위적으로 단락(Short)시켜 칩을 파괴하거나 오류를 유발하는 고장 주입 (Fault Injection) 공격을 막기 위해, 직렬 핀에 적절한 전류 제한 저항 (Current Limiting Resistor)을 배치하는 것이 필수적이다.

- **📢 섹션 요약 비유**: 도로 포장 상태를 좋게 하는 것(고유 저항 개선)도 중요하지만, 층간을 잇는 엘리베이터(접촉 저항)가 좁고 막히면 빌딩 전체의 물류가 마비되는 것처럼, 반도체는 수평과 수직 모든 방향의 저항을 동시에 낮춰야 합니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오: 고성능 AI 가속기의 전압 강하 (IR Drop)로 인한 셧다운 극복

1. **문제 상황**: 400W 이상의 극단적인 전력을 소모하는 데이터센터용 딥러닝 가속 칩(NPU)이 100% 부하(Full-load)로 연산을 수행할 때, 칩의 중앙 부근에 위치한 코어들에서 지속적으로 논리 오류가 발생하며 서버가 블루스크린을 띄우고 강제 재부팅됨.
2. **원인 분석 (IR Drop)**:
   - 칩 전체에 수백 암페어($I$)의 막대한 전류가 공급되며 가장자리에서 칩 중앙부까지 이동함.
   - 이동 경로에 있는 얇은 금속 전원망의 누적 기생 저항($R$)과 만나 옴의 법칙($V = I \cdot R$)에 따라 전압이 소실됨.
   - 외부에서 1.0V를 공급했으나, 저항이 누적된 칩 한가운데 코어에는 0.8V밖에 도달하지 못함. 이 전압은 트랜지스터 임계치($V_{th}$) 근처의 불확실 구간에 걸쳐 있어, 플립플롭이 '1'을 제대로 캡처하지 못하는 메타스테이블(Metastable) 상태에 빠짐.
3. **아키텍처적 해결책**:
   - **Power Mesh (전원망) 계층화**: 저항이 가장 낮은 최상단 두꺼운 금속 층 (Top Metal)을 전원 공급 전용망으로 빼곡하게 할당하여 물리적인 병렬 저항을 극단적으로 낮춤.
   - **FIVR (Fully Integrated Voltage Regulator)**: 마더보드가 아닌 프로세서 패키지 내부, 코어 바로 옆에 전압 조절기를 배치하여 막대한 전류가 이동해야 하는 거리($L$) 자체를 근본적으로 최소화.

이러한 RC 지연 및 저항 문제를 해결하기 위한 실무(Layout) 엔지니어의 의사결정 플로우는 다음과 같다.

```text
  ┌───────────────────────────────────────────────────────────────┐
  │         신호 지연(RC Delay) 병목 타개를 위한 의사결정 플로우          │
  ├───────────────────────────────────────────────────────────────┤
  │                                                               │
  │   [STA (정적 타이밍 분석) 결과, Setup Time 위반 경로 식별]         │
  │                │                                              │
  │                ▼                                              │
  │      해당 신호선(Wire)의 물리적 길이가 너무 길어 R이 누적되는가?    │
  │          ├─ 예 ─────▶ [중간에 버퍼(Repeater) 여러 개 분할 삽입]  │
  │          │                     │                              │
  │          │                     └─▶ [선로를 쪼개어 R과 C의 누적 곱 방지]│
  │          │                                                    │
  │          └─ 아니오                                            │
  │                │                                              │
  │                ▼                                              │
  │      해당 배선 층의 얇은 두께로 인해 면저항(Rs)이 큰가?            │
  │          ├─ 예 ─────▶ [배선 폭(Width)을 임의로 두 배 넓힘 (NDR)]│
  │          │                     │                              │
  │          │                     └─▶ [또는 저항이 낮은 상위 메탈층으로 승격]│
  │          │                                                    │
  │          └─ 아니오 ──▶ [신호를 쏘는 드라이버(Cell) 크기를 키워 전류 구동력 향상]│
  │                                                               │
  │   최종 전략: 길이를 자르거나, 차선을 넓히거나, 밀어내는 엔진을 키운다  │
  └───────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 단순히 칩을 작게 만들면 끝나는 것이 아니다. 거리가 길어 RC 지연이 생길 때는 긴 도로(선)를 한 번에 쏘지 않고 중간에 릴레이 중계소(버퍼)를 여러 개 달아 신호를 재생산한다. 폭이 좁아서 저항이 크면 예외적으로 해당 선로만 두껍게 설정(NDR, Non-Default Rule)하거나 고속도로 상층부(Top Metal)로 끌어올린다. 그래도 안 되면 신호 출발점의 트랜지스터 크기를 키워 밀어내는 전류의 힘 자체를 폭발적으로 늘린다. 이 3가지 튜닝 조합이 반도체 성능 최적화의 알파와 오메가다.

### 안티패턴 및 도입 체크리스트
- **안티패턴 (스네이크 라우팅 남용)**: 데이터 버스(Data Bus)를 구성하는 수십 가닥의 선 길이가 다르면 도착 시간이 어긋나는 스큐(Skew)가 발생한다. 이를 맞추기 위해 짧은 선을 구불구불하게 뱀처럼 꼬아(Snake Routing) 길이를 억지로 늘리는 설계 기법이 있다. 하지만 이를 맹목적으로 남용하면 불필요하게 길이($L$)가 증가해 전체 저항이 폭증하고, 결과적으로 지연 시간과 발열이 기하급수적으로 악화된다.
- **도입 체크리스트**:
  - 온도 계수 (TCR, Temperature Coefficient of Resistance)를 고려했는가? 구리는 온도가 오를수록 저항이 증가하므로, 칩이 최고 구동 온도(예: 105℃)에 도달했을 때 늘어난 저항을 감안하고도 타이밍 위반이 발생하지 않는지 마진(Margin)을 전수 검사해야 한다.

- **📢 섹션 요약 비유**: 수압이 낮아지지 않도록 메인 파이프를 아주 굵게 만들고 촘촘한 그물망으로 짜듯이, 전력 공급망의 저항을 최소화하여 칩 한가운데까지 강력한 에너지를 배달하는 것이 고성능 설계의 기본입니다.

### 정량/정성 기대효과

| 구분 | 최적화 전 (단순 미세화 및 단일 메탈) | 최적화 후 (계층적 메탈 및 Repeater 최적화) | 개선 효과 |
|:---|:---|:---|:---|
| **정량 (성능)** | 배선 저항 폭증으로 RC 지연 심각 | 리피터 삽입 및 상위 메탈(Top Metal) 승격 | RC Delay 대폭 감소로 최대 동작 클럭 **15% 이상 상향** |
| **정량 (전력)** | 트리(Tree) 구조의 취약한 전원망 | Power Mesh 촘촘화 및 IVR 적용 | IR Drop을 Vdd의 **5% 이내로 방어**, 전력 효율 극대화 |
| **정성 (수율)** | 미세 Via의 높은 접촉 저항으로 핫스팟 | 배리어 메탈 최소화 및 신소재 공정 | 접촉 발열 억제로 3nm 이하 최첨단 공정 양산 수율 안정화 |

### 미래 전망
- **BSPDN (Backside Power Delivery Network)**: 기존에는 논리 신호선과 전원 공급선이 웨이퍼 앞면(Front-side)에서 복잡하게 얽혀 좁은 면적을 나눠 써야 했으므로 저항이 높았다. 2nm 이하 차세대 공정(Intel 20A 등)에서는 전원 공급선만 웨이퍼 뒷면(Back-side)으로 완전히 빼내어 극도로 굵고 저항이 낮은 전용 초고속 전력망을 구축하는 혁명적 아키텍처가 표준이 될 것이다.
- **2D 신소재 배선 패러다임**: 구리(Cu)의 크기 효과(Size Effect)로 인한 근본적 저항 폭증을 타개하기 위해, 비저항이 극도로 낮고 전류 밀도 한계가 압도적인 탄소 나노튜브 (CNT)나 그래핀 (Graphene)을 활용한 신소재 배선 연구가 활발히 진행 중이며, 이는 1nm 이하 옹스트롬 시대를 여는 열쇠가 될 것이다.

### 참고 표준
- **IEEE 1528**: 전자 기기 내에서의 전파 및 전력 전송 시 저항 및 매질 특성, 인체 흡수율 분석 기준 규격.

- **📢 섹션 요약 비유**: 저항과의 전쟁은 좁은 골목길에 자동차를 구겨 넣는 한계를 넘어, 칩의 뒷면에 아예 거대한 지하 고속도로(BSPDN)를 뚫거나, 구리라는 재질을 벗어나 하늘을 나는 자동차(신소재)를 개발하는 차원으로 웅장하게 진화하고 있습니다.

## 📌 관련 개념 맵 (Knowledge Graph)

| 개념 명칭 | 관계 및 시너지 설명 |
|:---|:---|
| **전압 강하 (IR Drop)** | 저항에 대규모 전류가 흐를 때 옴의 법칙에 의해 필연적으로 발생하는 전압 소실로, 칩 안정성을 심각하게 위협한다. |
| **커패시터 (Capacitor)** | 저항(R)과 함께 묶여 신호를 뭉개뜨리는 RC 지연 (RC Delay) 회로를 형성하며 시스템 속도의 상한선을 결정한다. |
| **임피던스 (Impedance)** | 저항(직류 방해)의 개념을 교류 및 고주파 영역으로 확장한 것으로, 고속 신호 전송(SI) 무결성의 핵심 분석 대상이다. |
| **열 설계 전력 (TDP)** | 저항으로 인해 발생하는 마찰 열 에너지 ($P=I^2R$)가 칩 전체에 누적되어 시스템 쿨링 솔루션의 설계 기준이 된다. |
| **TSV (Through-Silicon Via)** | 여러 칩을 적층할 때 접촉 저항과 배선 저항을 혁신적으로 줄이기 위해 칩에 미세 구멍을 뚫어 수직으로 최단거리 연결을 구성하는 3D 패키징 기술이다. |

## 👶 어린이를 위한 3줄 비유 설명
1. 저항은 전기 요정들이 달려가는 길에 쏟아져 있는 **"까끌까끌한 모래밭"**과 같아요. 모래밭이 길고 좁을수록 요정들이 뛰어가기 힘들고 땀(열)을 뻘뻘 흘리게 돼요.
2. 컴퓨터가 손톱보다 작아지면서 이 길들이 눈에 보이지 않을 만큼 좁아졌고, 전기가 통과하는 속도가 너무 느려져서 스마트폰이 느려지는 것이 가장 큰 고민이에요.
3. 하지만 때로는 전기가 엉뚱한 곳으로 너무 많이 흐르지 못하게 일부러 튼튼한 문(저항)을 세워두기도 하는데, 이 문을 잘 만들어야 컴퓨터가 터지지 않고 안전하답니다!
