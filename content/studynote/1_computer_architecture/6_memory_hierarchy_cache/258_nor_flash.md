# NOR 플래시 (NOR Flash)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: NOR 플래시는 NOR 게이트 구조를 기반으로 한 **비휘발성 메모리**로, 셀이 병렬로 연결되어 **바이트 단위 랜덤 접근과 XIP(Execute In Place)**가 가능하다.
> 2. **가치**: NAND 대비 **빠른 랜덤 읽기 속도와 높은 신뢰성**으로 인해 BIOS/UEFI 펌웨어, 임베디드 부트로더, 소규모 코드 저장에 최적화되어 있다.
> 3. **융합**: 셀당 면적이 커서 대용량에는 부적합하지만, **전원 인가 후 즉시 코드 실행**이 필요한 시스템 초기화 과정에서 필수적인 역할을 수행한다.

+++

## Ⅰ. NOR 플래시의 개요

### 1. 정의
- NOR 플래시는 플로팅 게이트 구조를 사용하며, 셀들이 NOR 게이트처럼 **병렬로 연결**된 구조를 가진 비휘발성 메모리다. 각 셀에 독립적으로 접근할 수 있어 랜덤 접근이 가능하다.

### 2. 등장 배경: 코드 실행과 데이터 저장의 분리
- NAND는 대용량 데이터 저장에 유리하지만, 랜덤 접근이 느려 코드 실행에 부적합하다.
- 전원이 켜진 직후 CPU가 **바로 실행할 수 있는 코드 저장소**가 필요했고, 이를 위해 NOR 플래시가 개발되었다.

### 3. 💡 비유: '각 방마다 독립적인 문이 있는 건물'
- NOR 플래시은 **'각 방(셀)마다 별도의 출입문이 있어 원하는 방에 바로 들어갈 수 있는 건물'**과 같다.
- NAND는 복도를 따라 순서대로 지나가야 원하는 방에 도달하지만, NOR는 엘리베이터로 원하는 층에 바로 내려갈 수 있다.

- **📢 섹션 요약 비유**: 도서관에서 책을 찾을 때, 바로 원하는 책 앞으로 갈 수 있는 개방형 서가입니다.

### 2. XIP (Execute In Place)
- **XIP**: CPU가 메모리에서 직접 명령어를 인출하여 실행하는 기능.
- NOR 플래시은 랜덤 읽기가 빨라 **RAM으로 복사하지 않고도 코드 실행이 가능**하다.
- 임베디드 시스템에서 부팅 시간 단축과 RAM 절약에 활용.

### 3. 쓰기/삭제 특성
- **읽기**: 바이트/워드 단위, 매우 빠름 (~100ns)
- **쓰기(Program)**: 워드/바이트 단위, 느림 (마이크로초~밀리초)
- **삭제(Erase)**: 섹터/블록 단위, 매우 느림 (수백 밀리초~초)
- NAND보다 쓰기/삭제가 느려 대용량 데이터 저장에는 부적합.

- **📢 섹션 요약 비유**: 책을 읽을 때는 바로 펼칠 수 있지만(NOR), 책을 다시 쓰려면 한 챕터 전체를 지우고 다시 써야 합니다.

## Ⅲ. NOR vs NAND 플래시 비교

| 비교 항목 | NOR 플래시 | NAND 플래시 |
|:---|:---|:---|
| **구조** | 셀 병렬 연결 | 셀 직렬 연결 |
| **랜덤 읽기** | 빠름 (~100ns) | 느림 (~25μs) |
| **순차 읽기** | 보통 | 빠름 |
| **쓰기/삭제** | 느림 | 빠름 |
| **밀도** | 낮음 | 높음 |
| **비용/비트** | 높음 | 낮음 |
| **XIP** | 가능 | 불가능 |
| **주 용도** | BIOS, 펌웨어 | SSD, USB |

## Ⅳ. 실무 적용 및 아키텍처적 판단 (Strategy)

### 1. 기술사적 인사이트: BIOS/UEFI 저장 매체
- PC 메인보드의 BIOS/UEFI는 NOR 플래시에 저장된다.
- 전원 인가 → CPU는 NOR 플래시의 부트 코드(XIP) 실행 → 하드웨어 초기화 → OS 로드.
- SPI(Serial Peripheral Interface) NOR 플래시가 일반적.

### 2. 임베디드 시스템에서의 활용
- 마이크로컨트롤러(MCU) 내부 플래시 메모리는 대부분 NOR 구조.
- OTA(Over-The-Air) 펌웨어 업데이트 시 두 개의 NOR 뱅크를 교체하며 업데이트(A/B 업데이트).

### 3. 안티패턴: 대용량 데이터 저장
- NOR는 비싸고 쓰기가 느려 대용량 데이터 저장에 부적합.
- 임베디드에서도 사용자 데이터는 외부 NAND/eMMC에 저장.

+++

## Ⅴ. 기대효과 및 결론

### 1. 정량/정성 기대효과
- **빠른 부팅**: XIP로 펌웨어 즉시 실행.
- **높은 신뢰성**: 셀당 면적이 커 비트 오류율이 낮음.

### 2. 미래 전망
- NOR 플래시는 BIOS, 임베디드 펌웨어 등 **소규모 고신뢰 코드 저장**에 계속 사용될 것.
- 고속 SPI NOR 플래시(Quad SPI, Octal SPI)로 대역폭 확장 중.
