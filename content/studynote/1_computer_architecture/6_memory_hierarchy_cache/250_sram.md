# 250. SRAM (Static RAM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 플립플롭 기반, 리프레시 불필요한 고속 메모리
> 2. **가치**: 캐시 메모리, 레지스터 파일의 기술
> 3. **융합**: 속도 vs 밀도 vs 전력 트레이드오프

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

SRAM(Static RAM)은 **플립플롭을 사용하여 데이터를 저장하는 휘발성 메모리**이다. 리프레시가 필요 없고 접근 속도가 매우 빠르지만, 셀당 트랜지스터 수가 많아 밀도가 낮고 비용이 비싸다.

### 💡 비유: 스위치

SRAM은 **스위치**와 같다. 스위치는 켜거나 끈 상태가 그대로 유지된다. 전원이 켜져 있는 한 상태가 바뀌지 않는다. 계속 누르고 있을 필요가 없다.

## Ⅱ. SRAM 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                SRAM 상세                                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【SRAM 읽기 동작】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 프리차지: BL, BLB를 VDD/2로 프리차지                             │ │
│  │  2. 워드라인 활성화: WL = HIGH                                      │ │
│  │  3. 셀 데이터 읽기: BL 또는 BLB가 변화                               │ │
│  │  4. 센스 앰프: BL-BLB 차이 증폭                                      │ │
│  │  5. 출력: 데이터 출력                                               │ │
│  │                                                               │ │
│  │  타이밍:                                                           │ │
│  │  ──────────────────────────────────────────────────────────      │ │
│  │  클럭   WL   BL    BLB    출력                                     │ │
│  │  ────   ───  ───   ───    ────                                     │ │
│  │   0     0    V/2   V/2    -                                        │ │
│  │   1     1    V/2   V/2    -   (프리차지)                             │ │
│  │   2     1    V/2↓  V/2    -   (BL 방전 시작)                         │ │
│  │   3     1    V/4   V/2    -                                        │ │
│  │   4     1    0     V/2    0    (센스 앰프)                           │ │
│  │  ──────────────────────────────────────────────────────────      │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【SRAM 쓰기 동작】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 데이터 준비: BL=0, BLB=1 (0 쓰기) 또는 BL=1, BLB=0 (1 쓰기)       │ │
│  │  2. 워드라인 활성화: WL = HIGH                                      │ │
│  │  3. 셀 덮어쓰기: BL/BLB가 셀 상태 강제 변경                           │ │
│  │  4. 워드라인 비활성화: WL = LOW                                      │ │
│  │  5. 데이터 유지: 새로운 상태가 저장됨                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【SRAM 용량과 성능】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  SRAM 타입       용량        접근 시간    용도                        │ │
│  │  ──────────      ──────       ────────     ──────                    │ │
│  │  레지스터 파일    ~1KB        0.3~1ns     CPU 레지스터                │ │
│  │  L1 캐시         32~64KB      1~2ns       CPU 캐시                   │ │
│  │  L2 캐시         256KB~2MB    3~5ns       CPU 캐시                   │ │
│  │  L3 캐시         8~64MB       10~20ns     CPU 캐시                   │ │
│  │  외부 SRAM       1~16MB       10~20ns     임베디드                   │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• SRAM: 플립플롭 기반 정적 메모리
• 구조: 6 트랜지스터 (표준)
• 특징: 리프레시 불필요, 고속, 비휘발성
• 장점: 빠른 접근 (1~10ns), 단순한 인터페이스
• 단점: 낮은 밀도, 높은 비용, 높은 전력
• 용도: 캐시, 레지스터, TLB, BTB
• vs DRAM: 10배 빠름, 10배 비쌈, 10배 전력
• 트렌드: eSRAM (임베디드), STT-MRAM (차세대)
```
