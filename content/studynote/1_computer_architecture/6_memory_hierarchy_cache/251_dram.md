# 251. DRAM (Dynamic RAM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 캐패시터 기반, 리프레시 필요한 고밀도 메모리
> 2. **가치**: 주 메모리의 핵심 기술, 대용량 저비용
> 3. **융합**: 메모리 컨트롤러, 버스트 모드, 인터리빙

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

DRAM(Dynamic RAM)은 **캐패시터의 충전 상태로 데이터를 저장하는 휘발성 메모리**이다. 전하가 시간이 지남에 따라 누설되므로 주기적인 리프레시가 필요하다. 셀당 1개의 트랜지스터와 1개의 캐패시터만 필요하여 고밀도 구현이 가능하다.

### 💡 비유: 양동이

DRAM은 **물이 새는 양동이**와 같다. 물(전하)을 담아두면 시간이 지나면서 조금씩 새어 나간다. 가득 차 있으면 1, 비어 있으면 0이다. 주기적으로 물을 채워줘야(리프레시) 데이터를 유지할 수 있다.

## Ⅱ. DRAM 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                DRAM 상세                                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【DRAM 접근 과정】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 행 주소 스트로브 (RAS): 행 주소 입력                              │ │
│  │  2. 행 디코딩: 해당 워드라인 활성화                                   │ │
│  │  3. 센스 앰프 활성화: 비트라인 데이터 증폭                             │ │
│  │  4. 열 주소 스트로브 (CAS): 열 주소 입력                              │ │
│  │  5. 데이터 출력: 선택된 비트라인 데이터 출력                           │ │
│  │  6. 프리차지: 다음 접근을 위해 워드라인 비활성화                        │ │
│  │                                                               │ │
│  │  타이밍:                                                           │ │
│  │  ┌──────────────────────────────────────────────────────────┐   │ │
│  │  │  tRCD  │  tCL   │  tRP   │                                │   │ │
│  │  │ (RAS→CAS)│ (CAS Latency)│(Precharge)│                       │   │ │
│  │  │  10~15ns │  10~15ns   │  10~15ns  │                        │   │ │
│  │  │   ┌──┐   │    ┌──┐    │   ┌──┐   │                        │   │ │
│  │  │───┤  ├───┼────┤  ├────┼───┤  ├───                        │   │ │
│  │  │   RAS    │    CAS     │  PRECHARGE│                        │   │ │
│  │  └──────────────────────────────────────────────────────────┘   │ │
│  │  → 총 접근 시간 = tRCD + tCL + tRP ≈ 50ns                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【리프레시】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 주기: 64ms 내 모든 행 리프레시                                    │ │
│  │  • 방식: 자동 리프레시 (Auto Refresh)                               │ │
│  │  • 시간: 8192행 × tREFI (7.8μs 간격)                               │ │
│  │  • 영향: 리프레시 중 접근 불가 → 대역폭 감소                           │ │
│  │                                                               │ │
│  │  리프레시 최적화:                                                    │ │
│  │  • FGR (Fine Granularity Refresh): 더 자주, 더 적게                │ │
│  │  • RRA (Refresh Rate Adjustment): 온도 기반 조정                    │ │
│  │  • SBRA (Segmented Burst Refresh): 분산 리프레시                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【DRAM 성능 파라미터】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  파라미터       설명               일반값                         │ │
│  │  ──────────     ───────────────    ────────                       │ │
│  │  tCL           CAS 지연 시간        14~18 클럭                     │ │
│  │  tRCD          RAS→CAS 지연         14~18 클럭                     │ │
│  │  tRP           프리차지 시간         14~18 클럭                     │ │
│  │  tRAS          행 활성 시간          30~40 클럭                     │ │
│  │  tRC           행 사이클 시간         40~50 클럭                     │ │
│  │  → 낮을수록 좋음 (고성능 메모리)                                      │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• DRAM: 캐패시터 기반 동적 메모리
• 구조: 1T1C (1 트랜지스터 + 1 캐패시터)
• 특징: 리프레시 필요, 고밀도, 저비용
• 장점: 대용량, 저비용, 높은 밀도
• 단점: 리프레시 오버헤드, 느린 속도 (vs SRAM)
• 접근: RAS → CAS → 데이터 → 프리차지
• 타이밍: tCL, tRCD, tRP (각 10~15ns)
• 용도: 주 메모리, 그래픽 메모리
• 발전: DDR4 → DDR5 → HBM3
```
