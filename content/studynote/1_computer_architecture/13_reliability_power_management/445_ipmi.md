# 445. IPMI (Intelligent Platform Management Interface)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서버 원격 관리를 위한 하드웨어 표준 인터페이스
> 2. **가치**: OOB 관리, 상태 모니터링, 원격 제어
> 3. **융합**: BMC, 센서, SEL, SOL, Redfish

---

## Ⅰ. 개요

### 개념 정의

IPMI(Intelligent Platform Management Interface)는 **서버의 전원, 온도, 팬, 전압 등을 OS와 독립적으로 원격 모니터링하고 제어하는 하드웨어 기반 표준 인터페이스**다.

### 💡 비유: 원격 리모컨

IPMI는 **원격 리모컨**과 같다. TV가 있다. (서버) 리모컨이 있다. (IPMI) 전원 켜고 끈다. 온도 본다. 채널 바꾼다. (재부팅) OS가 꺼져도 된다. IPMI가 똑같다. 원격 관리, OS 독립, 하드웨어 제어.

### IPMI 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                IPMI 개요                                                 │

│                                                                     │

│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. BMC (Baseboard Management Controller):                                 │ │
│  │     • IPMI의 핵심 컨트롤러                                                │ │
│  │     • 센서, 전원, 팬 관리                                                  │ │
│  │     • 독립적인 마이크로컨트롤러                                              │ │
│  │                                                               │ │
│  │  2. 센서 (Sensors):                                                         │ │
│  │     • 온도, 전압, 팬 속도                                                   │ │
│  │     • 전력 소모, 전원 상태                                                  │ │
│  │     • 침입 감지                                                            │ │
│  │                                                               │ │
│  │  3. SEL (System Event Log):                                                 │ │
│  │     • 하드웨어 이벤트 로그                                                  │ │
│  │     • 온도 초과, 팬 고장 등                                                 │ │
│  │     • NVRAM에 저장                                                         │ │
│  │                                                               │ │
│  │  4. SOL (Serial Over LAN):                                                  │ │
│  │     • 원격 직렬 콘솔                                                       │ │
│  │     • 네트워크를 통한 콘솔 접근                                             │ │
│  │     • OS 부팅 과정 모니터링                                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【IPMI 기능】                                                                  │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  기능              설명                    OOB 가능             │ │
│  │  ────              ────                    ────────             │ │
│  │  전원 제어         On/Off/Reset/Cycle        ✓                   │ │
│  │  상태 모니터링      센서 값 읽기              ✓                   │ │
│  │  이벤트 로그       SEL 조회                 ✓                   │ │
│  │  직렬 콘솔         원격 터미널               ✓                   │ │
│  │  가상 미디어       ISO 마운트               ✓                   │ │
│  │  사용자 관리       계정 생성/삭제            ✓                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. IPMI 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                IPMI 상세                                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【IPMI 메시지 구조】                                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  IPMI 메시지 = RMCP + IPMI Session + IPMI Payload                          │ │
│  │                                                               │ │
│  │  RMCP (Remote Management Control Protocol):                                 │ │
│  │  • UDP 포트 623                                                            │ │
│  │  • ASF (Alert Standard Framework)                                          │ │
│  │                                                               │ │
│  │  인증:                                                                    │ │
│  │  • None (인증 없음)                                                        │ │
│  │  • MD5, MD2                                                                │ │
│  │  • 비밀번호                                                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【주요 IPMI 명령어】                                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  명령어              기능                    NetFn                  │ │
│  │  ────                ────                    ─────                  │ │
│  │  Get Device ID      장치 정보               06h (App)              │ │
│  │  Get System Info    시스템 정보              06h (App)              │ │
│  │  Chassis Power      전원 제어               00h (Chassis)          │ │
│  │  Set Power Restore  전원 복구 정책           00h (Chassis)          │ │
│  │  Read Sensor        센서 값 읽기             04h (Sensor)           │ │
│  │  Get SEL Info       이벤트 로그 정보         0Ah (Storage)          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【IPMI vs Redfish】                                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  특성           IPMI                Redfish                  │ │
│  │  ────           ────                ───────                  │ │
│  │  프로토콜       RMCP/UDP            REST/HTTPS               │ │
│  │  데이터         Binary              JSON                      │ │
│  │  보안           MD5                 TLS/OAuth                 │ │
│  │  확장성         제한적               높음                      │ │
│  │  표준           Intel 등            DMTF                     │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### IPMI 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                IPMI 분석                                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【ipmitool 기본 명령어】                                                          │)
│  ──────────────────                                                  │
│  # BMC 정보                                                            │
│  $ ipmitool mc info                                                   │
│                                                                     │
│  # 센서 목록                                                            │
│  $ ipmitool sensor list                                               │
│                                                                     │
│  # 전원 상태                                                            │
│  $ ipmitool power status                                              │
│                                                                     │
│  # 전원 켜기/끄기                                                       │
│  $ ipmitool power on                                                  │
│  $ ipmitool power off                                                 │
│  $ ipmitool power cycle                                               │
│                                                                     │
│  【SEL 이벤트 로그】                                                              │)
│  ──────────────────                                                  │
│  $ ipmitool sel list                                                  │
│  # 이벤트 목록                                                          │
│                                                                     │
│  $ ipmitool sel elist                                                 │
│  # 확장 이벤트 목록                                                     │
│                                                                     │
│  【SOL (직렬 콘솔)】                                                              │)
│  ──────────────────                                                  │
│  $ ipmitool -I lanplus -H <host> -U <user> -P <pass> sol activate    │
│  # 원격 콘솔 활성화                                                      │
│                                                                     │
│  【IPMI 모니터링 팁】                                                              │)
│  ──────────────────                                                  │
│  1. 온도: CPU/메모리/디스크 온도 모니터링                                 │
│  2. 팬: 팬 속도 및 상태                                                  │
│  3. 전원: 전력 소모 및 전압                                               │
│  4. SEL: 이벤트 로그 정기 확인                                           │
│  5. 펌웨어: BMC 펌웨어 정기 업데이트                                       │
│  6. 보안: 강력한 비밀번호, VLAN 분리                                       │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• IPMI: 서버 원격 관리 표준
• BMC: 베이스보드 관리 컨트롤러
• 센서: 온도, 전압, 팬, 전력
• SEL: 시스템 이벤트 로그
• SOL: 직렬 오버 LAN
• OOB: 대역외 관리
• RMCP: 원격 관리 제어 프로토콜
• 장점: 원격 관리, OS 독립
• 단점: 보안 취약점, 복잡성
• Redfish: 차세대 표준
```
