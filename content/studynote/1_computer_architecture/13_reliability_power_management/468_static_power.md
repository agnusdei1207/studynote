# 정적 전력 (Static Power)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 정적 전력 (Static Power) 또는 누설 전력 (Leakage Power)은 트랜지스터가 완전한 '오프' 상태에서도 게이트 산화막 터널링 (Quantum Tunneling)과 서브스레숄드 전도로 인해 소량의 전류가 지속적으로 흐르면서 발생하는 전력 소모다.
> 2. **가치**: 공정이 7nm, 5nm, 3nm로 미세화될수록 산화막이 얇아져 게이트 누설이 기하급수로 증가하며, 7nm 이하 공정에서는 정적 전력이 전체 전력의 30~50%를 차지하는 설계의 핵심 문제가 된다.
> 3. **융합**: 고-k 유전체 (High-k Dielectric, HfO₂ 등), 핀펫 (FinFET), GAAFET 트랜지스터 구조, 파워 게이팅, 멀티 임계 전압 (Multi-Vt) 기법이 정적 전력 억제의 최전선 기술이다.

---

## Ⅰ. 개요 및 필요성

이상적인 트랜지스터는 '오프' 상태에서 전혀 전류가 흐르지 않아야 한다. 하지만 현실의 반도체 트랜지스터는 산화막이 극도로 얇은 나노 공정(~1nm 두께 SiO₂)에서 두 가지 원인으로 전류가 새어 나온다:

1. **서브스레숄드 누설**: 게이트 전압이 임계 전압보다 0일 때도 약간の 드레인 전류가 흐름
2. **게이트 산화막 터널링**: 너무 얇은 산화막에서 전자가 양자역학적으로 통과함

**💡 비유**: 수도꼭지를 완전히 잠가도 노후한 밸브에서 물이 조금씩 새는 것 — 밸브(트랜지스터)가 작아질수록(공정 미세화) 더 많이 셉니다.

```text
┌──────────────────────────────────────────────────────────────┐
│         정적 전력 성분 분석                                 │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  P_sta = V_DD × I_leak                                      │
│                                                              │
│  I_leak 구성 요소:                                          │
│  ① 서브스레숄드 (Subthreshold) 누설:                       │
│     I ∝ e^(Vgs/nVT), 공정 미세화 → Vt 감소 → 누설 지수 증가│
│                                                              │
│  ② 게이트 산화막 (Gate Oxide) 누설:                        │
│     SiO₂ 두께 1.5nm 이하 → 양자 터널링 발생               │
│     Intel → 고-k 유전체(HfO₂, 3nm 두께로 동일 효과) 도입  │
│                                                              │
│  ③ PN 접합 누설 (역방향 바이어스):                         │
│     소수 반송자 확산, 고온에서 급증                         │
│                                                              │
│  공정별 정적 전력 비율:                                     │
│  130nm → 전체의 5~10%                                       │
│  28nm  → 전체의 15~25%                                      │
│  7nm   → 전체의 30~50%                                      │
│  3nm   → 50% 이상 예상                                      │
└──────────────────────────────────────────────────────────────┘
```

**📢 섹션 요약 비유**: 정적 전력은 불을 끄고 잠을 자도 새는 전기 — 트랜지스터를 아무리 껐어도 나노 공정에선 전자가 얇은 벽(산화막)을 그냥 통과해요.

## Ⅱ. 아키텍처 및 핵심 원리

### 정적 전력 억제 기술

```text
┌──────────────────────────────────────────────────────────────┐
│         정적 전력 억제 기술 진화                            │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  시대 → 핵심 기술                                           │
│                                                              │
│  2003년: 고-k 유전체 도입 (Intel 45nm HIgh-K Metal Gate)   │
│  HfO₂ (k=25 vs SiO₂ k=3.9):                               │
│  → 물리적으로 두껍지만 전기적 효과는 SiO₂의 1/6.4 두께    │
│  → 게이트 터널링 누설 10배 이상 감소                       │
│                                                              │
│  2011년: FinFET 도입 (Intel 22nm Tri-Gate)                 │
│  → 3D 핀 구조로 채널을 3면에서 감쌈                        │
│  → 단채널 효과(Short Channel Effect) 억제                   │
│  → 동일 성능에서 전력 ~50% 감소                            │
│                                                              │
│  2022년~: GAAFET / NSFET (삼성 3nm, TSMC N2)              │
│  Gate-All-Around: 채널을 4면 전체에서 감쌈                 │
│  → FinFET 대비 누설 추가 20~30% 감소                       │
│                                                              │
│  설계 기법: Multi-Vt (다중 임계 전압)                       │
│  고속 경로: 저-Vt 셀 (빠르지만 누설 많음)                  │
│  비핵심 경로: 고-Vt 셀 (누설 매우 적음)                    │
│  → 성능과 누설의 맞춤 최적화                               │
└──────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 핀펫(FinFET)의 핵심은 평면 트랜지스터와 달리 실리콘 핀(Fin)을 세워서 게이트가 채널을 3방향에서 감싸는 것이다. 더 강한 전기장 제어로 서브스레숄드 누설을 억제하면서도 더 낮은 임계 전압으로 더 빠른 스위칭이 가능하다.

**📢 섹션 요약 비유**: FinFET은 수도관을 납작한 파이프에서 좁고 둥근 파이프로 바꾸는 것 — 더 꽉 막히니까 새는 물(누설)이 훨씬 줄어들어요.

## Ⅲ. 융합 비교 및 다각도 분석

### 정적 vs 동적 전력 트레이드오프

임계 전압(Vt)을 낮추면:
- 스위칭 속도↑ (동적 성능 향상)
- 서브스레숄드 누설↑ (정적 전력 증가)

반대로 Vt를 높이면:
- 서브스레숄드 누설↓
- 스위칭 속도↓ (타이밍 마진 감소)

**이것이 바로 Multi-Vt 기법이 필요한 이유**: 핵심 타이밍 경로에는 저-Vt, 나머지에는 고-Vt를 사용해 전체 성능을 유지하면서 누설을 최소화.

**📢 섹션 요약 비유**: Vt는 수도꼭지 잠금 강도 — 느슨하게 잠그면(저-Vt) 빨리 열리지만 물이 새고, 꽉 잠그면(고-Vt) 안 새지만 열기 어렵습니다. 핵심 수도관만 느슨하게, 나머지는 꽉 잠그는 게 최적이에요.

## Ⅳ. 실무 적용 및 기술사적 판단

**실무 시나리오**:
1. **서버 Idle 전력**: 최신 7nm 서버 CPU는 Idle 시에도 TDP의 20~30%를 누설로 소모. C6/C10 파워 게이팅으로 코어를 완전 차단하면 80% 이상 절감.
2. **스마트폰 대기 전력**: SoC 완전 Sleep (파워 게이팅)에서도 누설 때문에 배터리가 서서히 방전. GAAFET 도입으로 누설 감소 → 대기 시간 연장.

**안티패턴**:
- **저-Vt 셀 과용**: 타이밍에 여유 있는 경로에도 저-Vt 사용 → 불필요한 누설 증가. 합성 시 타이밍 슬랙 기반 자동 Vt 최적화 필수.

**📢 섹션 요약 비유**: 저-Vt 과용은 사무실 모든 수도꼭지를 느슨하게 설치하는 것 — 화장실 세면대까지 빠른 수압이 필요하지는 않잖아요.

## Ⅴ. 기대효과 및 결론

| 공정 | 대표 트랜지스터 | 정적 전력 |
|:---|:---|:---|
| 130nm | 평면 MOSFET + SiO₂ | ~5% |
| 45nm | 고-k HKMG | ~15% |
| 22nm | FinFET (3-gate) | ~20% |
| 3nm | GAAFET | 여전히 30~50% |

정적 전력은 나노 공정의 피할 수 없는 대가다. 고-k 유전체, FinFET, GAAFET으로 억제하고, 파워 게이팅으로 유휴 시 차단하며, Multi-Vt 설계로 성능-누설 균형을 맞추는 것이 현대 반도체 설계의 핵심 전략이다.

## 👶 어린이를 위한 3줄 비유 설명
1. 정적 전력은 수도꼭지를 잠가도 새는 물 — 트랜지스터를 꺼도 나노 공정에선 전자가 얇은 벽을 통과해요.
2. 핀펫(FinFET)은 수도꼭지를 둥그런 모양으로 바꿔서 물이 덜 새게 만든 거예요.
3. 파워 게이팅은 아예 수도 메인 밸브를 닫는 것 — 그래야 새는 물(정적 전력)을 완전히 막을 수 있어요!
