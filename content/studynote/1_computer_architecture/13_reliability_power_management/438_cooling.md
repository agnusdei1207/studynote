# 438. 발열 관리 및 냉각 (Thermal Management & Cooling)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU/GPU 발열을 제어하여 온도 유지
> 2. **가치**: 성능 유지, 수명 연장, 안정성 확보
> 3. **융합**: 히트싱크, 팬, 액체 냉각, P-state, 서멀 스로틀링

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

발열 관리 및 냉각(Thermal Management & Cooling)은 **CPU/GPU 등 반도체에서 발생하는 열을 효과적으로 방출하여 적정 온도를 유지하는 기술**이다.

### 💡 비유: 자동차 라디에이터

냉각 시스템은 **자동차 라디에이터**와 같다. 엔진이 뜨거워진다. (발열) 냉각수가 순환한다. (히트싱크) 팬이 바람을 보낸다. (냉각) 온도가 유지된다. 컴퓨터가 똑같다. 열 발생, 전달, 방출, 온도 유지.

### 발열 관리 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                발열 관리 개요                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【발열 원리】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  전력 소모 → 열 발생:                                                      │ │
│  │  Q = P × t (열 = 전력 × 시간)                                             │ │
│  │                                                               │ │
│  │  발열량:                                                                    │ │
│  │  • 100W CPU: ~100W 열                                                   │ │
│  │  • 300W GPU: ~300W 열                                                   │ │
│  │  • 1000W 서버: ~1000W 열                                                │ │
│  │                                                               │ │
│  │  온도 한계:                                                                    │ │
│  │  • TjMax: 접합 온도 최대 (100-105°C)                                      │ │
│  │  • Tcase: 케이스 온도                                                    │ │
│  │  • Tambient: 주변 온도                                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【냉각 방식】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. 공랭 (Air Cooling):                                                    │ │
│  │     • 히트싱크 + 팬                                                        │ │
│  │     • 가장 일반적                                                         │ │
│  │                                                               │ │
│  │  2. 수랭 (Liquid Cooling):                                                 │ │
│  │     • 워터블록 + 라디에이터 + 펌프                                         │ │
│  │     • 높은 효율                                                            │ │
│  │                                                               │ │
│  │  3. 기화 냉각 (Vapor Chamber):                                             │ │
│  │     • 상변화 열전달                                                       │ │
│  │     • 고효율, 소형                                                        │ │
│  │                                                               │ │
│  │  4. 히트파이프 (Heat Pipe):                                                │ │
│  │     • 작동 유체 순환                                                      │ │
│  │     • 노트북, 서버                                                       │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

```
│                                                                     │
│  $ cat /proc/acpi/ibm/fan                                            │
│  # ThinkPad 팬                                                        │
│                                                                     │
│  【서멀 스로틀링 확인】                                                            │)
│  ──────────────────                                                  │
│  $ cat /sys/devices/system/cpu/cpu*/thermal_throttle/*               │

│  $ nvidia-smi -q -d PERFORMANCE                                      │
│  # GPU 스로틀링                                                        │
│                                                                     │
│  【냉각 최적화 팁】                                                                  │)
│  ──────────────────                                                  │
│  1. TIM: 고품질 서멀 페이스트                                          │
│  2. 공기 흐름: 전면 흡기 후면 배기                                       │
│  3. 팬 곡선: 온도 기반 RPM 조절                                         │
│  4. 먼지 제거: 정기 청소                                               │
│  5. 케이스: 통풍良好 케이스                                              │
│  6. 실리콘: 적절량 도포                                                 │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 발열 관리: 열 발생 → 방출 제어
• 냉각: 공랭, 수랭, 기화, 히트파이프
• TjMax: 접합 온도 한계 (100-105°C)
• TIM: 열전달 매체
• IHS: 열분산 캡
• 서멀 스로틀링: 과열 시 주파수 감소
• TDP: 열 설계 전력
• 장점: 성능, 수명, 안정성
• 단점: 비용, 소음, 복잡성
• 아레니우스: 온도 10°C↑ = 수명 1/2
```
