# 601-604. 사물인터넷(IoT)의 기초와 WPAN

> **핵심 인사이트**: 사물인터넷은 단순히 물건에 센서를 다는 것이 아니라, 세상 모든 정보가 데이터가 되어 흐르게 하는 거대한 신경망이다. 정보 수집(Device), 전달(Network), 가공(Platform)이라는 3대 요소가 맞물려 돌아가며, 그 중 우리 몸 주변의 연결을 책임지는 것이 WPAN이다.

---

## Ⅰ. 사물인터넷 (IoT, Internet of Things) 개요
세상에 존재하는 사물들을 유무선 네트워크로 연결하여 정보를 공유하는 기술입니다.

### 1. IoT의 3대 구성 요소
1. **Device (디바이스)**: 센서와 액추에이터를 통해 현실의 데이터를 수집하거나 명령을 수행함. (예: 스마트 조명, 온도 센서)
2. **Network (네트워크)**: 데이터를 전달하는 통로. (예: Wi-Fi, LTE, ZigBee, LPWAN)
3. **Service/Platform (클라우드/플랫폼)**: 수집된 거대한 데이터를 저장, 분석하고 가치 있는 서비스로 가공함. (예: AWS IoT, 구글 홈)

### 2. M2M (Machine to Machine) vs IoT
* **M2M**: 기기 간의 직접적인 통신(1:1)에 초점. 폐쇄적인 네트워크가 많음.
* **IoT**: 인터넷이라는 공용망을 통해 수많은 기기와 클라우드가 연결되는 광범위한 개념.

---

## Ⅱ. WPAN (Wireless Personal Area Network)
사람을 중심으로 반경 10m 이내의 좁은 공간에서 장치들을 연결하는 무선 네트워크입니다.

* **특징**: 저전력, 소형화, 저가격을 지향합니다.
* **기술 종류**: 블루투스(Bluetooth), 지그비(ZigBee), UWB 등.

---

## Ⅲ. WSN (Wireless Sensor Network)
특정 지역의 정보를 수집하기 위해 수많은 센서 노드들을 뿌려놓은 네트워크입니다.

* **구조**:
  - **Sensor Node**: 온도, 습도 등을 측정하는 개별 단말.
  - **Sink Node (Gateway)**: 각 노드에서 모인 데이터를 취합하여 상위 네트워크(인터넷)로 전달하는 중계기.

---

## Ⅳ. 개념 맵 및 요약

```ascii
[IoT 데이터 흐름도]

  (수집) [ Device ] ───(수치 수집)───┐
                                    │
  (전달) [ Network ] ──(WPAN/LPWAN)──┼───> [ Sink Node / Gateway ]
                                    │           │
  (가공) [ Platform ] <──(인터넷)─────┘           │
                                                ▼
                                         [ 최종 서비스 (App) ]
```

📢 **섹션 요약 비유**: **IoT**는 우리 집 모든 가구에 '뇌'와 '무전기'를 달아주는 혁명입니다. **M2M**은 리모컨으로 TV를 켜는 단순한 동작이라면, **IoT**는 스마트폰으로 밖에서 에어컨을 켜고 전력 소모량을 분석하는 거대한 시스템입니다. **WPAN**은 내 몸 주변 1~2미터 안에서 이어폰과 폰이 대화하는 '사적인 대화방'이며, **WSN**은 산속 곳곳에 파수꾼(센서)을 배치해 산불이 나는지 감시하는 '감시망'과 같습니다.
