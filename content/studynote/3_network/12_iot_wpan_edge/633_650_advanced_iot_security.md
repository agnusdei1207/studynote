# 633-650. 고도화된 IoT 기술과 보안 (AIoT, CPS, 보안)

> **핵심 인사이트**: IoT는 이제 단순한 연결을 넘어 '스스로 판단하는 지능(AIoT)'과 '현실과 가상의 일치(CPS)'를 추구하고 있다. 하지만 장치의 자원이 워낙 부족해 기존 보안 기술을 그대로 쓸 수 없기에, 아주 가볍고 단단한 '경량 암호'와 '제로 트러스트' 모델이 생존의 열쇠가 되고 있다.

---

## Ⅰ. AIoT와 CPS: 지능형 사물 세상
1. **AIoT (AI + IoT)**: 사물이 수집한 데이터를 클라우드로 보내지 않고, 사물 자체가 내장된 AI로 즉시 판단하고 실행하는 기술입니다. (On-device AI)
2. **CPS (Cyber-Physical System)**: 현실 세계(Physical)와 가상 세계(Cyber)를 긴밀하게 통합한 시스템입니다. 현실의 데이터를 실시간으로 가상의 모델(Digital Twin)에 반영하고, 가상에서의 시뮬레이션 결과를 다시 현실의 기계 제어에 반영합니다.

---

## Ⅱ. IoT 보안 위협과 대응
IoT 기기는 PC보다 보안이 매우 취약하여 해커들의 주요 타겟이 됩니다.

### 1. 주요 공격 사례
* **Mirai Botnet**: 보안이 취약한 가정용 CCTV, 공유기를 감염시켜 거대한 좀비 네트워크를 만들고 대규모 DDoS 공격을 수행한 사례.

### 2. 보안 정책
* **망분리 (Network Separation)**: 중요한 업무망과 보안이 취약한 IoT 망을 물리적/논리적으로 분리.
* **제로 트러스트 (Zero Trust)**: "아무도 믿지 마라." 모든 기기와 접속에 대해 매번 검증하고 최소한의 권한만 부여하는 방식.

---

## Ⅲ. 경량 암호 (Lightweight Cryptography)
IoT 기기의 낮은 연산 능력과 적은 메모리 환경에서도 동작하도록 설계된 암호 알고리즘입니다.

* **LEA (Lightweight Encryption Algorithm)**: 우리나라에서 개발한 국산 경량 암호 표준. (AES보다 훨씬 가벼움)
* **특징**: 복잡한 수학 연산 대신 단순한 비트 이동(Shift)과 배타적 논리합(XOR)을 활용하여 속도를 높였습니다.

---

## Ⅳ. 미래형 IoT 기술
* **백스캐터 통신 (Ambient Backscatter)**: 배터리 없이 주변에 떠다니는 TV나 Wi-Fi 전파를 반사하여 데이터를 전송하는 초절전 기술입니다. (Energy Harvesting 연계)
* **WoT (Web of Things)**: 모든 사물에 URL을 부여하여, 마치 웹 페이지를 탐색하듯 브라우저로 전 세계 사물을 제어하는 개념입니다.

---

## Ⅴ. 개념 맵 및 요약

```ascii
[IoT의 진화 단계]

 (연결) Connected IoT : 사물이 인터넷에 연결됨
      │
 (지능) Intelligent IoT : 사물이 스스로 판단함 (AIoT)
      │
 (통합) CPS / Digital Twin : 가상과 현실이 동기화됨
```

📢 **섹션 요약 비유**: **AIoT**는 시키는 일만 하던 로봇이 스스로 생각해서 행동하는 '똑똑한 비서'가 된 것입니다. **CPS**는 현실의 공장을 거울처럼 비추는 가상 세계의 공장을 만들어, 거울 속에서 미리 기계를 돌려보고 안전한지 확인하는 '마법의 거울'입니다. **경량 암호**는 무거운 갑옷을 입을 체력이 없는 꼬마 기사(IoT 기기)에게 아주 가볍지만 튼튼한 '티타늄 방패'를 쥐여주는 것과 같습니다.
