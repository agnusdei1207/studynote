# 690-700. 네트워크 보안 장비의 진화 (Firewall, IPS, NAC)

> **핵심 인사이트**: 보안 장비는 점점 '똑똑해지고' 하나로 '합쳐지고' 있다. 단순한 문지기(방화벽)에서 범죄의 패턴을 읽는 형사(IPS)로, 이제는 애플리케이션의 내용까지 낱낱이 파악하는 정보원(NGFW)으로 진화하며 입체적인 방어망을 구축하고 있다.

---

## Ⅰ. 방화벽 (Firewall)
네트워크 경계에서 허용되지 않은 트래픽을 차단하는 기본 장비입니다.

* **1세대 (Packet Filter)**: IP와 포트 번호만 보고 단순 차단. (상태 기억 못 함)
* **2세대 (Stateful Inspection)**: **세션 상태**를 기억하여, 나간 요청에 대한 응답은 자동으로 허용함. (현대 방화벽의 기본)
* **3세대 (Application Proxy)**: 데이터 본문까지 열어서 검사하고 대행함.

---

## Ⅱ. 침입 탐지 및 차단 (IDS / IPS)
* **IDS (Intrusion Detection System)**: 네트워크 트래픽을 복사해서 보고, 공격 징후를 **탐지(Alert)**함. (방관자 모델)
* **IPS (Intrusion Prevention System)**: 트래픽이 흐르는 길목(In-line)에 서서, 공격이 발견되면 즉시 **차단(Drop)**함. (적극적 방어)
* **엔진 방식**:
  - **Misuse (오용)**: 해커의 공격 패턴(Signature)과 비교.
  - **Anomaly (이상)**: 평소와 다른 비정상적 행동 감지.

---

## Ⅲ. 차세대 및 통합 보안 장비
* **WAF (Web Application Firewall)**: 웹 서버 특화 방화벽. SQL Injection, XSS 등 L7 공격 방어.
* **UTM (Unified Threat Management)**: 방화벽, IPS, 백신 등을 장비 하나에 다 때려 넣은 가성비 모델.
* **NGFW (Next-Generation Firewall)**: UTM에서 한 단계 진화하여, 사용자 ID 기반 제어와 딥 패킷 분석(DPI)을 수행하는 고성능 장비.

---

## Ⅳ. 내부 보안: NAC (Network Access Control)
외부 공격이 아닌, **내부 단말기(PC, 폰)**를 통제하는 시스템입니다.

* **동작**: 백신이 안 깔려 있거나 OS 업데이트가 안 된 단말은 네트워크 접속을 격리(Quarantine)시킵니다.
* **장점**: 사내망의 무결성을 유지하고 비인가 장치의 접속을 원천 차단합니다.

---

## Ⅴ. 개념 맵 및 요약

```ascii
[네트워크 방어 레이어]

  (외부) ──> [ Firewall ] ──> [ IPS ] ──> [ WAF ] ──> (서버)
               (문지기)        (경찰)      (특수 경호)
                                 │
                          [ NAC (내부 통제) ]
```

📢 **섹션 요약 비유**: **방화벽**은 건물 입구에서 출입증(IP/Port)을 확인하는 경비원입니다. **IPS**는 건물 내부를 돌아다니며 수상한 행동(공격 패턴)을 하는 사람을 즉시 잡아내는 형사입니다. **WAF**는 귀빈실(웹 서버) 앞에서 위장한 자객(웹 공격)을 걸러내는 특수 요원입니다. **NAC**은 사원들이 출근할 때 복장 불량(백신 미설치)이나 전염병(바이러스) 여부를 체크하여 격리시키는 검역소와 같습니다.
