# 328-331. IPv6 주소 유형과 자동 설정(SLAAC)

> **핵심 인사이트**: IPv6는 불필요한 전체 방송(Broadcast)을 없애고, 가장 가까운 노드와 대화하는 애니캐스트(Anycast)를 도입했다. 또한, 관리자가 없어도 기기가 자신의 MAC 주소를 이용해 스스로 IP를 만드는 '무상태 자동 설정'이 가능하다.

---

## Ⅰ. IPv6 주소의 3대 유형
IPv6에는 **브로드캐스트(Broadcast)가 존재하지 않습니다.** 대신 멀티캐스트가 그 역할을 대신합니다.

1. **유니캐스트 (Unicast)**: 1:1 통신. 특정 단말 하나를 지정합니다.
2. **멀티캐스트 (Multicast)**: 1:N 통신. 특정 그룹에 가입한 여러 단말에게 보냅니다. (`FF00::/8` 대역 사용)
3. **애니캐스트 (Anycast)**: **1:가장 가까운 1명** 통신. 똑같은 주소를 가진 여러 단말 중, 네트워크 거리(Hop)상 가장 가까운 1명에게만 패킷이 전달됩니다. (주로 DNS 서버나 CDN 노드 분산에 사용)

---

## Ⅱ. IPv6의 범위별 주소 체계

* **링크 로컬 주소 (Link-Local Address)**:
  - **`FE80::/10`** 으로 시작합니다.
  - 라우터를 넘지 못하는, 로컬 망 안에서만 통신할 때 씁니다. IPv4의 APIPA와 비슷하지만, IPv6에서는 모든 인터페이스가 기본적으로 반드시 가져야 하는 필수 주소입니다.
* **글로벌 유니캐스트 주소 (Global Unicast)**:
  - **`2000::/3`** 대역입니다.
  - 전 세계 어디서든 통신 가능한 공인 IP 주소입니다.

---

## Ⅲ. 주소 자동 설정 (SLAAC과 EUI-64)
DHCP 서버 없이도 단말이 스스로 IP 주소를 설정하는 방식입니다.

### 1. EUI-64 (Extended Unique Identifier)
단말의 48비트 MAC 주소를 이용하여 64비트의 호스트 식별자(Interface ID)를 만드는 기술입니다.
* **방법**: 48비트 MAC 주소를 반으로 쪼개고, 그 사이에 **`FF FE`** (16비트)를 끼워 넣습니다. 
  - 예: `00:11:22:33:44:55` ➔ `0211:22FF:FE33:4455` (첫 번째 바이트의 7번째 비트를 반전시키는 규칙 포함)

### 2. SLAAC (Stateless Address Autoconfiguration)
* **단계**:
  1. 단말이 라우터에게 "이 동네 네트워크 번호(Prefix)가 뭐야?"라고 묻습니다. (**RS**: Router Solicitation)
  2. 라우터가 "여기는 `2001:DB8:1::/64` 야"라고 답해줍니다. (**RA**: Router Advertisement)
  3. 단말은 받은 Prefix 뒤에 자신의 **EUI-64** 식별자를 붙여서 128비트 IP를 완성합니다.
* **장점**: '무상태(Stateless)'라는 이름답게 서버가 누가 어떤 IP를 쓰는지 일일이 기록하고 관리할 필요가 없어 관리가 매우 편합니다.

---

## Ⅳ. 개념 맵 및 요약

```ascii
[IPv6 주소 자동 생성 과정]

1. [ PC ]  --- (RS: 동네 번호 알려줘!) --->  [ Router ]
2. [ PC ]  <--- (RA: 여기는 2001:A::/64 야) ---  [ Router ]

3. [ PC 의 판단 ]
   Prefix (라우터가 준 것) : 2001:000A:0000:0000
   Interface ID (내 MAC 기반): 0211:22FF:FE33:4455
   -------------------------------------------------
   최종 IP 완성 : 2001:A::211:22FF:FE33:4455
```

📢 **섹션 요약 비유**: **애니캐스트**는 "근처에 있는 편의점 아무 데나 한 곳만 가라"는 명령어와 같습니다. **SLAAC**은 전학 온 학생(새 기기)이 선생님(라우터)에게 "우리 반 이름이 뭐예요?"라고 묻고, 선생님이 "3학년 1반(Prefix)"이라고 답해주면, 학생이 자기 이름표(EUI-64)를 그 뒤에 붙여서 스스로 출석 번호를 만드는 아주 독립적인 시스템입니다.
