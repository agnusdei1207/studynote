# 312-316. 주소 분석 프로토콜(ARP)과 변종들

> **핵심 인사이트**: IP 주소만으로는 옆집에 데이터를 보낼 수 없다. 최종적으로는 랜카드 주소(MAC)가 필요하며, 이 둘 사이의 다리 역할을 하는 것이 ARP이다. 또한 자신의 주소를 광고하거나 남의 대답을 대신해 주는 특수 ARP들도 존재한다.

---

## Ⅰ. ARP (Address Resolution Protocol)
논리적 주소인 **IP 주소를 물리적 주소인 MAC 주소로 변환**해 주는 2.5계층 프로토콜입니다.

### 1. 작동 원리 (Request & Reply)
1. **ARP Request**: 송신자는 "IP가 192.168.1.5인 사람 누구니? MAC 주소 좀 알려줘!"라고 **브로드캐스트(FF:FF...)**를 날립니다.
2. **ARP Reply**: 해당 IP를 가진 단말이 "그거 난데, 내 MAC은 00:1A... 야"라고 **유니캐스트**로 대답합니다.
3. **ARP Cache**: 매번 물어보기 귀찮으므로, 한 번 알아낸 정보는 자신의 메모리(Table)에 잠시 저장해 둡니다.

---

## Ⅱ. ARP의 다양한 변종 기술

### 1. RARP (Reverse ARP)
* **개념**: MAC 주소는 아는데 자신의 IP 주소를 모를 때 사용하는 과거의 방식입니다. 
* **용도**: 하드디스크가 없는 터미널(Diskless Workstation)이 부팅될 때 서버로부터 IP를 할당받기 위해 썼으나, 현재는 **DHCP**에 밀려 쓰이지 않습니다.

### 2. Proxy ARP (프록시 ARP)
* **개념**: 라우터가 자신과 연결된 다른 네트워크에 있는 단말의 IP 요청을 받았을 때, **대신 자신의 MAC 주소를 알려주는** 속임수(?) 기술입니다.
* **용도**: 호스트가 서브넷 마스크 설정이 잘못되어 있거나, 디폴트 게이트웨이가 없을 때도 외부와 통신할 수 있게 도와줍니다.

### 3. Gratuitous ARP (G-ARP)
* **개념**: 아무도 물어보지 않았는데 스스로 "내 IP는 이거고 내 MAC은 이거야!"라고 브로드캐스트를 날리는 것입니다.
* **용도**:
  - **IP 중복 감지**: 만약 누가 똑같은 IP를 쓰고 있다면 응답이 올 것이고, 이를 통해 충돌을 확인합니다.
  - **캐시 갱신**: 서버 이중화(VRRP/HSRP)에서 메인 서버가 바뀌었을 때, 주변 스위치들의 MAC 테이블을 즉시 업데이트하기 위해 씁니다.

---

## Ⅲ. ARP의 보안 취약점 (ARP Spoofing)
ARP는 "먼저 대답하는 놈이 임자"라는 아주 순진한 프로토콜입니다.

* **공격 원리**: 해커가 가짜 ARP Reply를 지속적으로 보내어, "내가 게이트웨이다!" 혹은 "내가 저 컴퓨터다!"라고 희생자를 속입니다.
* **결과**: 희생자의 패킷이 해커를 거쳐 가게 되어 데이터를 훔쳐보는 **Man-in-the-Middle (MITM)** 공격이 가능해집니다.

---

## Ⅳ. 개념 맵 및 요약

```ascii
[ARP 통신 흐름]

[ PC A ] ------------------- (LAN) ------------------- [ PC B ]
  IP: .10                                                IP: .20
  MAC: AA                                                MAC: BB

1. (Request) "IP .20 누구야? MAC 알려줘!" (Broadcast FF:FF...)
2. [ LAN 상의 모든 PC가 들음 ]
3. [ PC B 만 응답 ] (Reply) "나 .20인데 내 MAC은 BB야!" (Unicast)
4. [ PC A 의 ARP Table ] 에 " .20 = BB " 기록 완료
```

📢 **섹션 요약 비유**: **ARP**는 거실에서 "누구 휴대폰 뒷번호가 1234니? 내 이름은 철수야"라고 소리치는 것입니다. 주인인 영희가 "그거 내 폰이야! 내 이름은 영희야"라고 말해주는 것이 **Reply**입니다. **Proxy ARP**는 영희가 화장실 갔을 때 엄마가 대신 "그거 우리 딸 거야"라고 말해주는 것이고, **G-ARP**는 학교에 전학 온 친구가 교실 문 열자마자 "안녕 난 짱구야!"라고 자기소개를 하는 것과 같습니다.
