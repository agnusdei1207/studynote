# 626-632. 엣지 컴퓨팅과 산업용 통신 (MEC, TSN)

> **핵심 인사이트**: 모든 데이터를 멀리 있는 클라우드로 보내기에는 세상이 너무 빨라졌다. 0.001초의 오차도 허용하지 않는 자율주행과 스마트 팩토리를 위해, 연산 능력은 현장(Edge)으로 내려오고 있으며, 통신 규격 또한 나노초 단위의 정밀한 시간 약속(TSN)을 지키는 방향으로 진화하고 있다.

---

## Ⅰ. 엣지 컴퓨팅 (Edge Computing)
데이터가 생성되는 곳(장치 근처)에서 즉시 데이터를 처리하는 분산 컴퓨팅 기술입니다.

* **필요성**: 클라우드까지 왕복하는 지연 시간(Latency) 감소, 대역폭 절약, 데이터 프라이버시 강화.
* **MEC (Multi-access Edge Computing)**: 5G 기지국 등에 서버를 두어 모바일 네트워크의 끝단에서 초저지연 연산을 수행하는 기술입니다.

---

## Ⅱ. 산업용 이더넷 및 제어 통신
공장의 정밀 기계들을 제어하기 위해 일반 이더넷의 불확실성(충돌, 지연)을 제거한 기술들입니다.

### 1. 주요 표준
* **EtherCAT**: 초고속 산업용 이더넷. 패킷이 지나가면서 각 장치가 필요한 데이터를 실시간으로 읽고 쓰는 'On-the-fly' 방식.
* **Modbus TCP**: 전통적인 시리얼 통신을 이더넷 위로 올린 방식.
* **OPC UA**: 제조사 상관없이 장비 간 데이터를 안전하게 교환하기 위한 통합 통신 표준.

### 2. TSN (Time-Sensitive Networking)
* **개념**: 표준 이더넷(IEEE 802.1)을 확장하여 **확정적 지연(Deterministic)**을 보장하는 기술입니다.
* **핵심 기능**: 시간 동기화(AS), 자원 예약, 트래픽 스케줄링.
* **가치**: 일반 데이터와 정밀 제어 데이터를 하나의 망에서 섞어 써도 제어 신호의 골든 타임을 완벽히 보장합니다.

---

## Ⅲ. 에너지 및 그리드 통신
* **스마트 그리드 (Smart Grid)**: IT 기술을 접목하여 전력 공급과 소비를 실시간으로 조절하는 지능형 전력망입니다.
* **AMI (Advanced Metering Infrastructure)**: 전력 사용량을 실시간으로 검침하고 양방향 통신을 수행하는 핵심 인프라입니다.

---

## Ⅳ. 개념 맵 및 요약

```ascii
[컴퓨팅 계층 구조의 변화]

  [ Cloud ] (중앙 집중, 대량 분석)
      │
  [ Edge / Fog ] (지역 거점, 실시간 제어) <─── ( MEC 가 위치하는 곳 )
      │
  [ Device ] (데이터 발생, 센서/액추에이터)
```

📢 **섹션 요약 비유**: **엣지 컴퓨팅**은 모든 문제를 서울 본사(클라우드)에 물어보는 대신, 현장 소장님(엣지)이 즉석에서 결정하게 하는 시스템입니다. **TSN**은 군대 의장대 사열과 같아서, 모든 병사가 나노초 단위의 정확한 박자에 맞춰 발을 맞추는 초정밀 시간 동기화 기술입니다. **스마트 그리드**는 전기가 남을 때 저축하고 모자랄 때 꺼내 쓰는 '똑똑한 전력 가계부'와 같습니다.
