# 609-612. 스마트홈 무선 표준 (ZigBee, Z-Wave, Thread, Matter)

> **핵심 인사이트**: 집안의 전등과 센서들이 대화하려면 전기를 아주 적게 쓰면서도 벽을 뚫고 멀리 전달되어야 한다. 이를 위해 지그비와 스레드는 서로 신호를 릴레이하는 '메시(Mesh)' 구조를 채택했으며, 최근에는 '매터(Matter)'라는 통일 표준이 등장하여 제조사 상관없이 모든 기기가 하나로 묶이고 있다.

---

## Ⅰ. 지그비 (ZigBee)
IEEE 802.15.4 표준을 기반으로 하는 저전력, 저가격, 저속도 무선 네트워크 기술입니다.

* **특징**:
  - **Mesh Topology**: 기기들이 서로 중계기 역할을 하여 네트워크 범위를 무한히 확장할 수 있습니다.
  - **초저전력**: 전구 하나로 수년을 버팁니다.
* **한계**: Wi-Fi와 똑같은 2.4GHz 대역을 써서 간섭이 발생할 수 있습니다.

---

## Ⅱ. Z-Wave (Z웨이브)
지그비의 강력한 라이벌로, 주로 북미 스마트홈 시장에서 인기 있는 기술입니다.

* **특징**: 900MHz 대역(Sub-GHz)을 사용하여 장애물 통과 능력이 뛰어나고 Wi-Fi 간섭이 전혀 없습니다.
* **단점**: 특정 칩셋 제조사(실리콘랩스)에 의존적인 폐쇄적 생태계입니다.

---

## Ⅲ. 스레드 (Thread)
구글(네스트) 주도로 만들어진 IPv6 기반의 저전력 메시 네트워크 프로토콜입니다.

* **차별점**: 지그비와 하드웨어(802.15.4)는 같지만, **IP 주소를 직접 가집니다.** 즉, 게이트웨이 없이도 인터넷과 직접 연결될 수 있는 지능형 스마트홈 표준입니다.

---

## Ⅳ. 매터 (Matter)
애플, 구글, 아마존, 삼성 등 거대 IT 기업들이 손잡고 만든 **상호 운용성 통합 표준**입니다.

* **가치**: "애플 홈킷 전용"이나 "삼성 스마트싱스 전용" 제품을 구분할 필요가 없어집니다.
* **구조**: Wi-Fi, Ethernet, Thread 위에서 동작하는 상위 애플리케이션 계층 표준입니다. 
* **의미**: 제조사 장벽을 허물어 스마트홈 대중화를 이끄는 최종 끝판왕 기술입니다.

---

## Ⅴ. 개념 맵 및 요약

```ascii
[스마트홈 기술 계층도]

  [ App Layer ] ──────>  [  Matter (공통 언어)  ]
                             /        \
  [ Transport ] ───> [ Thread / Wi-Fi / Bluetooth ]
                             \        /
  [ Physical  ] ───> [ IEEE 802.15.4 / 802.11 ]
```

📢 **섹션 요약 비유**: **지그비**는 동네 사람들이 한 줄로 서서 옆 사람에게 귓속말로 소식을 전하는 '릴레이 메시'입니다. **Z-Wave**는 벽이 많은 집에서도 소리가 잘 들리는 '낮은 저음의 무전기'입니다. **스레드**는 각 사람에게 직접 전화번호(IP)를 부여한 세련된 연락망이며, **매터**는 한국인, 미국인, 중국인이 모두 모여도 대화가 통하게 하는 '공용어(에스페란토)'와 같은 표준 규격입니다.
