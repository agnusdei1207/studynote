# 567-573. 이동통신 세대별 기술 (1G to 4G LTE)

> **핵심 인사이트**: 이동통신의 역사는 '어떻게 더 많은 데이터를 더 좁은 길(주파수)에 쑤셔 넣을 것인가'의 투쟁사다. 소리를 전파로 쏘던 1G를 지나, 코드로 구별하던 3G를 거쳐, 이제는 모든 것이 인터넷 데이터(All-IP)가 된 4G LTE 시대에 도달했다.

---

## Ⅰ. 1G & 2G: 음성 통화의 시대
1. **1G (아날로그)**: 음성을 그대로 전파에 실어 보냄. 보안이 없고 품질이 낮음. (**FDMA**)
2. **2G (디지털 전환)**: 음성을 디지털로 변환. 문자 메시지(SMS) 시작.
   - **CDMA (Code)**: 북미/한국 주도. 신호에 코드를 입혀 구별.
   - **GSM (Time)**: 유럽 주도. 시간을 쪼개어 구별. (SIM 카드의 시초)

---

## Ⅱ. 3G: 멀티미디어의 시작
음성뿐만 아니라 영상 통화와 인터넷이 가능해진 세대입니다.
* **W-CDMA**: GSM에서 발전한 세계 표준. 광대역 코드를 사용.
* **HSPA / HSPA+**: 3.5세대 기술로, 데이터 속도를 획기적으로 높여 스마트폰 대중화를 이끔.

---

## Ⅲ. 4G LTE: All-IP의 완성
"Long Term Evolution"의 약자로, 음성망을 아예 없애고 **모든 데이터를 IP 패킷**으로만 처리하는 시대입니다.

### 1. 핵심 기술 (OFDM & MIMO)
* **OFDM (Orthogonal Frequency Division Multiplexing)**: 수많은 부반송파를 겹치지 않게(직교) 정렬하여 데이터를 쏘는 기술. LTE 속도의 비결입니다.
* **MIMO (Multiple-Input Multiple-Output)**: 안테나를 여러 개 써서 데이터를 동시에 주고받는 기술.

### 2. 주요 표준
* **LTE-Advanced (LTE-A)**: 4G의 최종 완성형.
  - **Carrier Aggregation (CA)**: 서로 떨어진 주파수 대역을 하나처럼 묶어 속도를 높임.
  - **VoLTE**: 음성 통화도 데이터망(IP) 위에서 고품질로 처리.

---

## Ⅳ. 이동통신 세대별 요약표

| 세대 | 핵심 기술 | 주요 서비스 | 특징 |
| :--- | :--- | :--- | :--- |
| **1G** | FDMA (주파수 분할) | 아날로그 음성 | 최초의 휴대폰 |
| **2G** | TDMA / CDMA | 디지털 음성, SMS | 보안 강화, 로밍 시작 |
| **3G** | W-CDMA | 영상 통화, 무선 인터넷 | 세계 단일 표준화 노력 |
| **4G** | **OFDMA / MIMO** | 고화질 영상, 초고속 데이터 | **All-IP 망 (VoLTE)** |

---

## Ⅴ. 개념 맵 및 요약

```ascii
[LTE-A의 핵심: Carrier Aggregation]

  [ 주파수 대역 A (10MHz) ] + [ 주파수 대역 B (20MHz) ]
               │                       │
               └───────────┬───────────┘
                           ▼
              [ 가상 대역 (30MHz) ] ──> 초고속 데이터 전송
```

📢 **섹션 요약 비유**: **1G**는 실전화기로 대화하는 것이고, **2G**는 무전기로 암호화된 대화를 하는 것입니다. **3G**는 화상 전화기가 보급된 시기이며, **4G LTE**는 전화선 자체가 사라지고 모든 대화와 영상이 인터넷 광랜(IP)으로 통합된 '움직이는 광통신' 시대입니다. **CA**는 좁은 차선 여러 개를 합쳐서 널찍한 8차선 고속도로를 만드는 마법과 같습니다.
