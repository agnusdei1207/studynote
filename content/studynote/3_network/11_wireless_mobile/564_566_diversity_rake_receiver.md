# 564-566. 무선 수신 최적화: 다이버시티와 레이크 수신기

> **핵심 인사이트**: 무선 신호는 산에 부딪히고 건물에 튀며 수많은 '잔상'을 남긴다. 이 잔상들은 서로를 방해하는 노이즈가 되기 십상이지만, 다이버시티와 레이크 수신기 기술은 오히려 이 조각들을 긁어모아 더 크고 선명한 신호로 합쳐버리는 역발상을 보여준다.

---

## Ⅰ. 다이버시티 (Diversity) 시스템
여러 경로를 통해 들어온 똑같은 신호 중 상태가 좋은 것만 고르거나, 조각난 신호들을 합쳐서 수신 품질을 높이는 기술입니다. (페이딩 방지 핵심)

1. **공간 다이버시티 (Space)**: 안테나를 2개 이상 설치하여, 서로 다른 위치에서 오는 신호를 받습니다.
2. **시간 다이버시티 (Time)**: 똑같은 데이터를 시간 간격을 두고 여러 번 보냅니다.
3. **주파수 다이버시티 (Frequency)**: 서로 다른 주파수 대역으로 동시에 쏩니다.
4. **편파 다이버시티 (Polarization)**: 전파의 진동 방향(수직/수평)을 다르게 하여 받습니다.

---

## Ⅱ. 레이크 수신기 (Rake Receiver)
건물 등에 반사되어 미세하게 **시간 차(지연)를 두고 도착하는 여러 개의 다중 경로 신호**를 각각의 갈퀴(Finger)로 잡아낸 뒤, 하나로 합치는 장치입니다. (주로 CDMA 방식에서 사용)

* **원리**: 늦게 온 신호들을 잠시 기다렸다가 위상을 맞춰서 합치면, 신호의 세기가 증폭되는 효과(에너지 결합)가 발생합니다.
* **비유**: 흩어진 퍼즐 조각들을 모아 하나의 그림을 완성하는 것과 같습니다.

---

## Ⅲ. 등화기 (Equalizer)
* **개념**: 전파가 전달되는 과정에서 발생하는 왜곡(ISI, 인접 심볼 간섭)을 보정하기 위해 수신단에 설치하는 필터입니다.
* **역할**: "이 채널은 저주파가 잘 안 들리네?"라고 판단되면 해당 부분을 강제로 키워주는 식으로 신호를 평평하게(Equal) 만듭니다.

---

## Ⅳ. 개념 맵 및 요약

```ascii
[레이크 수신기의 구조]

 신호 유입 ───┬───> [ Finger 1 ] (직접파) ──┐
              │                             │
              ├───> [ Finger 2 ] (반사파) ──┼───> [ 신호 결합기 ] ──> 깨끗한 데이터
              │                             │      (Combiner)
              └───> [ Finger 3 ] (굴절파) ──┘
```

📢 **섹션 요약 비유**: **다이버시티**는 로또 당첨 확률을 높이기 위해 복권을 여러 장(여러 경로) 사는 것입니다. 그중 하나라도 맞으면 성공인 셈이죠. **레이크 수신기**는 마당에 떨어진 낙엽(조각난 신호)들을 갈퀴(Finger)로 싹싹 긁어모아 커다란 더미를 만드는 청소 도구입니다. **등화기**는 울퉁불퉁하게 녹음된 목소리를 보정하여 다시 원래의 맑은 목소리로 돌려주는 오디오 이퀄라이저와 같습니다.
