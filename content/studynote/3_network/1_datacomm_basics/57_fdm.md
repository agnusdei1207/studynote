# FDM (주파수 분할 다중화) 기술 분석

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전체 가용 주파수 대역폭을 여러 개의 작은 대역폭(채널)으로 나누어, **서로 다른 주파수 영역**에 각 사용자 데이터를 실어 보내는 다중화 방식.
> 2. **가치**: 인접한 채널 간의 상호 간섭을 방지하기 위해 사용하지 않는 빈 공간인 **가드 밴드(Guard Band)**를 두어 신호의 독립성을 보장한다.
> 3. **융합**: 고전적인 라디오/TV 방송의 근간이며, 현대의 ADSL 및 유선 케이블 통신에서 다수의 서비스를 하나의 선로로 통합 전송하는 데 핵심적인 역할을 한다.

---

### Ⅰ. FDM (Frequency Division Multiplexing)의 매커니즘

- **작동 원리**: 
  1. 각 송신기의 신호를 서로 다른 반송파(Carrier) 주파수로 변조(Modulation).
  2. 변조된 신호들을 하나로 합쳐서 전송.
  3. 수신측에서 필터(Filter)를 사용하여 원하는 주파수 대역만 추출.
- **특징**: 주로 아날로그 신호 전송에 사용되지만, 디지털 데이터도 반송파에 실어 보낼 수 있음.

---

### Ⅱ. 주파수 영역 아키텍처 및 가드 밴드 (ASCII)

주파수 축 상에서 채널들이 어떻게 배치되는지 보여준다.

```ascii
    [ Frequency Spectrum ]
    
    Power
      ^      CH 1        CH 2        CH 3
      |     +----+      +----+      +----+
      |     |    |      |    |      |    |
      |     |    |      |    |      |    |
    --+-----+----+------+----+------+----+------> Frequency
            |    |      |    |      |    |
            |____|      |____|      |____|
               ^          ^
               |          |
          [ Guard Band ] [ Guard Band ]
          (Interference Prevention)
```

- **Guard Band**: 채널 사이의 여백. 필터의 성능 한계로 인해 신호가 겹치는 것을 막기 위한 필수적인 자원 낭비 구간.

---

### Ⅲ. FDM 방식의 장단점

| 구분 | 장점 (Pros) | 단점 (Cons) |
|:---|:---|:---|
| **동시성** | 모든 사용자가 **상시 전송 가능**. 시분할 지연 없음. | - |
| **자원 효율** | - | **가드 밴드 오버헤드** 발생. 주파수 자원 낭비. |
| **복잡도** | 아날로그 회로로 단순 구현 가능. | 장비마다 다른 필터 설계 필요. |
| **간섭** | - | **상호변조 잡음**에 취약 (비선형 왜곡 시). |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

#### 1. 광대역 유선 가입자망 (ADSL/VDSL) 설계
- **현상**: 하나의 전화선으로 인터넷과 전화를 동시에 써야 함.
- **기술사적 결단**: 
  - 저주파 대역(0~4kHz)은 음성 전화용으로, 고주파 대역은 데이터용(Up/Down)으로 나누는 **FDM**을 적용한다. 
  - 이때 전화기의 잡음이 데이터 대역을 침범하지 않도록 **스플리터(Splitter)**를 설치하여 가드 밴드 역할을 물리적으로 수행하게 한다.

#### 2. 기술사적 인사이트: FDM에서 OFDM으로의 진화
- FDM의 최대 약점인 가드 밴드 낭비를 해결하기 위해, 신호들을 서로 직교(Orthogonal)하게 겹쳐서 가드 밴드 없이도 전송하는 **OFDM** 기술이 현대 무선 통신의 대세가 되었음을 연계하여 설명해야 한다.

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

#### 1. 정량/정성 기대효과
- **실시간 방송 서비스 구현**: 수백 개의 채널을 지연 없이 동시 송출.
- **이기종 서비스 융합**: 전화, 데이터, TV를 하나의 물리 매체에 통합.

#### 2. 미래 전망
전통적인 FDM은 주파수 효율성 문제로 점차 OFDM이나 가상화된 다중화로 대체되고 있다. 하지만 위성 통신의 채널 할당이나 고출력 레이저 전송 분야에서는 여전히 주파수 물리 분리 방식인 FDM이 가장 안정적인 기술로 평가받으며, 극한 환경 통신의 최후의 보루로 남을 것이다.
