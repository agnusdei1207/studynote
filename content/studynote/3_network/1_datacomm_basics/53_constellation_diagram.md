# 성상도 (Constellation Diagram) - 진폭과 위상 표시

> 1. **본질**: 디지털 기저대역 데이터가 아날로그 반송파에 실릴 때, 각 심볼이 갖는 **진폭(Amplitude)**과 **위상(Phase)** 상태를 2차원 복소 평면(I-Q Plane) 상의 점(Point)으로 맵핑하여 시각화한 분석 도구이다.
> 2. **가치**: 통신 시스템의 변조 방식(PSK, QAM 등)을 직관적으로 이해하게 해주며, 점들 간의 거리와 퍼짐 정도를 통해 **신호 대 잡음비(SNR)**, **에러 벡터 크기(EVM)** 등 물리 계층의 품질을 즉각적으로 진단하는 기준이 된다.
> 3. **융합**: 통신 이론의 수학적 모델(Vector Space)을 실제 하드웨어 측정 장비(오실로스코프, 스펙트럼 분석기)의 화면으로 옮겨놓은 것으로, 물리 계층 설계 및 디버깅의 절대적인 나침반 역할을 한다.

+++

## Ⅰ. 개요 (Context & Background)

- **개념**: **성상도(Constellation Diagram)**는 마치 밤하늘의 별자리처럼 신호가 위치해야 할 이상적인 목표점(Ideal Symbol)과 실제 수신된 점(Received Symbol)들을 보여주는 2차원 그래프이다. X축은 동위상(In-phase, $I$), Y축은 직교위상(Quadrature, $Q$)을 나타낸다.

- **💡 비유**: 양궁 과녁판과 같습니다. 중앙에 명중시켜야 할 점수판(이상적 심볼 위치)이 있고, 바람이나 수전증(노이즈) 때문에 실제로 화살이 꽂힌 위치(수신된 심볼)들이 과녁판 위에 흩어져 있는 모습을 보여주는 것입니다.

- **등장 배경**:
  1. **고차 변조의 복잡성**: 16-QAM, 64-QAM처럼 한 심볼에 여러 비트가 담기면서 단순히 시간축(Time Domain)의 파형만으로는 데이터가 제대로 전송되는지 눈으로 확인하기 불가능해졌다.
  2. **직교성(Orthogonality)의 시각화**: $\cos$(I)와 $\sin$(Q)이라는 두 개의 독립된 축을 사용하여 신호를 2차원 벡터로 완벽히 분해하고 표현할 수학적 도구가 필요했다.
  3. **장애 원인 분석**: 신호가 찌그러지거나 에러가 날 때, 그 원인이 진폭의 문제인지, 위상이 틀어진 것인지, 단순 백색 잡음인지를 성상도의 모양만 보고 단번에 파악하기 위해 도입되었다.

- **📢 섹션 요약 비유**: 복잡하게 엉킨 실타래(시간축 파형)를 엑스레이로 찍어, 뼈대(진폭과 위상)가 어떻게 구성되어 있는지 한눈에 보여주는 통신 공학의 투시경입니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### I-Q 복소 평면 (Complex Plane) 구조

성상도는 직교하는 두 반송파($\cos, \sin$)를 축으로 삼는다.

```text
 [ Constellation Diagram의 기본 구조 ]
 
            Q (Quadrature / 직교위상 / sin 성분 / 허수부)
            ▲
            │       ● (A, θ)  <-- 심볼 (Symbol)
            │      /|
            │   r / │
            │    /  │
            │   /   │ y (진폭 * sin(θ))
            │  / θ  │
 ───────────┼───────┴─────▶ I (In-phase / 동위상 / cos 성분 / 실수부)
            │      x (진폭 * cos(θ))
            │
            │
            
 * 원점으로부터의 거리 (r) = 신호의 진폭 (Amplitude)
 * I축으로부터의 각도 (θ) = 신호의 위상 (Phase)
```

**[다이어그램 해설]** 하나의 점(심볼)은 원점으로부터의 거리와 각도를 갖는다. 예를 들어 16-QAM은 이런 점이 사분면 전체에 걸쳐 16개가 찍혀 있다. 점이 원점에서 멀수록 송신 전력이 큰 것이고, 각도가 다르면 파동의 시작 위치가 다른 것이다. 이 2차원 평면은 오일러 공식($e^{j\theta} = \cos\theta + j\sin\theta$)에 기반한 복소수 벡터 공간을 시각화한 것이다.

+++

### 성상도를 통한 주요 에러 패턴(Impairment) 분석

이상적인 점(Ideal Point) 주변에 실제 수신된 점들이 어떻게 흩어지는지(Scatter)를 보면 장애 원인을 알 수 있다.

| 산포도 (Scatter) 형태 | 발생 원인 (Impairment) | 물리적 의미 | 비유 |
|:---|:---|:---|:---|
| **원형으로 뭉침 (Cloud)** | **열 잡음 (Thermal Noise/AWGN)** | 무작위적인 백색 잡음 유입 | 과녁 전체에 흩뿌려진 화살 |
| **원호를 따라 퍼짐 (Arc)** | **위상 잡음 (Phase Noise) / 지터** | 클럭이 흔들려 각도가 틀어짐 | 과녁판 자체가 좌우로 회전함 |
| **원점 방향으로 찌그러짐** | **증폭기 비선형성 (Compression)** | 최대 출력 구간에서 전압이 깎임 | 강하게 쏜 화살이 바람막이에 걸림 |
| **I 또는 Q 한쪽만 퍼짐** | **I-Q 불균형 (I/Q Imbalance)** | 송/수신기의 아날로그 변조기 오차 | 활시위가 한쪽으로만 팽팽함 |

+++

### 에러 벡터 크기 (EVM, Error Vector Magnitude)

성상도에서 통신 품질을 정량화하는 가장 중요한 지표다.

- **EVM = (에러 벡터의 RMS 크기) / (가장 큰 이상적 심볼의 크기)**
- 이상적인 점(Reference)과 실제 찍힌 점(Measured) 사이의 거리(벡터)를 계산하여 백분율(%)이나 데시벨(dB)로 나타낸다. 고차 QAM일수록(예: 1024-QAM) 점 사이의 간격이 좁으므로 EVM 규격이 극도로 엄격해진다.

- **📢 섹션 요약 비유**: 성상도는 단순히 예쁜 그림이 아니라, "과녁의 정중앙에서 화살이 평균적으로 몇 밀리미터나 빗나갔는지(EVM)"를 측정하는 초정밀 판독기입니다.

## Ⅲ. 융합 비교 및 다각도 분석

### 시간 영역(Time Domain) vs 성상도(Vector Domain)

| 비교 항목 | 오실로스코프 (Time Domain) | 성상도 (Constellation) |
|:---|:---|:---|
| **X축 / Y축** | 시간 (Time) / 전압 (Voltage) | 동위상 (I) / 직교위상 (Q) |
| **관찰 대상** | 파형의 연속적인 변화 모양 | 특정 샘플링 시점에서의 이산적 심볼 |
| **강점** | 타이밍 오류, 글리치(Glitch) 확인 용이 | 변조 품질, 에러 원인, 노이즈 마진 직관적 확인 |
| **복잡도 해석** | 64-QAM 이상이면 눈으로 해독 불가 | 1024-QAM도 점의 분포로 즉시 파악 가능 |

### 과목 융합 관점

- **소프트웨어 공학 (SE)**: 머신러닝이나 패턴 인식 AI를 통신 모뎀에 적용할 때, 이 2차원 성상도의 I-Q 좌표값들은 CNN(합성곱 신경망) 등 딥러닝 모델의 훌륭한 입력 피처(Input Feature)로 활용된다.
- **수학 (선형대수)**: 성상도의 각 점은 직교 기저(Orthogonal Basis) 함수인 $\cos$과 $\sin$을 축으로 하는 벡터 공간의 선형 결합(Linear Combination)을 의미한다.

- **📢 섹션 요약 비유**: 음악을 들을 때 스피커의 펄럭이는 진동판(시간 영역)을 보는 대신, 악보에 찍힌 음표(성상도)를 보면 곡의 구조를 훨씬 명확하게 이해할 수 있습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 5G 기지국(RU) 송신 불량 트러블슈팅**: 기지국 설치 후 스펙트럼 분석기로 신호를 찍어보니, 성상도의 바깥쪽 점들(가장 진폭이 큰 심볼들)이 원점 쪽으로 둥글게 찌그러져(Compression) 있었다. 기술사는 즉시 **전력 증폭기(PA)**의 출력이 포화 상태(Saturation)에 빠져 비선형 구간에서 동작하고 있음을 파악하고, 증폭기의 입력 전력(Back-off)을 낮추거나 DPD(Digital Pre-Distortion) 알고리즘을 재보정하여 문제를 해결한다.

2. **시나리오 — 광통신 QAM 수신기 최적화**: 100Gbps 이상을 지원하는 Coherent 광통신 수신기에서 성상도를 모니터링하니 점들이 도넛 모양(위상 잡음)으로 빙글빙글 돌고 있었다. 이는 송수신 레이저 간의 위상 동기화가 흔들리는 것이므로, 광학적 위상 고정 루프(OPLL)를 점검하거나 DSP의 위상 추적(Phase Tracking) 알고리즘 계수를 조정해야 한다.

### 도입 체크리스트
- **측정 장비 준비**: 기저대역 I/Q 신호를 분리해낼 수 있는 벡터 신호 분석기(VSA)가 준비되어 있는가?
- **EVM 허용치**: 사용하려는 변조 방식(예: 256-QAM)이 요구하는 최소 EVM 규격(예: -32dB)을 타겟 시스템이 통과하는가?
- **결정 경계(Decision Boundary)**: 점과 점 사이의 거리가 노이즈 분산보다 충분히 넓게 설정되어 있는가?

### 안티패턴
- **단순 스펙트럼(주파수) 관찰만 맹신**: 주파수 대역폭과 출력 전력이 정상이라고 해서 통신 품질이 좋다고 단정하는 것. 실제 데이터의 품질(모양)이 찌그러졌는지는 오직 성상도(EVM)를 통해서만 확인할 수 있다.
- **EVM 최적화만 고집하다 출력 희생**: EVM을 아주 깨끗하게 만들려고 증폭기 출력을 너무 낮추면(Deep Back-off), 신호 모양은 예쁘지만 정작 신호 세기(SNR)가 낮아져 수신단에서 에러가 폭증하는 딜레마에 빠진다.

- **📢 섹션 요약 비유**: 건강검진에서 겉모습(주파수 스펙트럼)만 멀쩡하다고 안심할 것이 아니라, 피를 뽑아 현미경(성상도)으로 적혈구 모양이 일그러지지 않았는지 확인해야 진짜 건강을 알 수 있습니다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과

| 구분 | 도입 전 (Time Domain 분석) | 도입 후 (Constellation 분석) | 개선 효과 |
|:---|:---|:---|:---|
| **장애 진단** | 원인 파악 불분명 | 형태별 직관적 원인 식별 | 트러블슈팅 시간 **90% 이상 단축** |
| **품질 관리** | 막연한 노이즈 측정 | EVM 수치 기반의 정밀 관리 | 제품 출하 불량률 제로화 |
| **설계 한계** | 복잡한 변조 설계 어려움 | 기하학적 매핑 최적화 | 초고차 QAM(4096 등) 구현 가능 |

### 미래 전망
현대 통신의 성상도는 고정된 사각형 격자를 넘어 진화하고 있다. 섀넌 한계에 더욱 근접하기 위해, 점들의 간격과 확률 분포를 비균일하게 찌그러뜨리는 **확률적 성상 형성(Probabilistic Constellation Shaping, PCS)** 기술이 해저 광케이블과 차세대 무선망에 도입되고 있다. 이제 성상도는 단순한 '분석 도구'를 넘어 '채널 최적화 알고리즘' 그 자체가 되었다.

### 참고 표준
- **3GPP TS 38.104**: 5G NR 단말기 및 기지국의 성상도 기반 EVM 테스트 규격
- **IEEE 802.11**: Wi-Fi 세대별 변조 정확도(EVM) 제한값 표준

- **📢 섹션 요약 비유**: 우주를 이해하기 위해 천문학자들이 별자리의 지도를 그렸듯이, 통신 공학자들은 정보의 바다를 지배하기 위해 이 2차원 평면 위에 자신들만의 정밀한 별자리(성상도)를 그려왔습니다.

## 📌 관련 개념 맵 (Knowledge Graph)
- **QAM (Quadrature Amplitude Modulation)**: 성상도를 가장 적극적이고 복잡하게 활용하는 변조 방식.
- **I-Q 변조 (I-Q Modulation)**: 성상도의 X축과 Y축을 물리적으로 만들어내는 송수신기 아키텍처.
- **EVM (Error Vector Magnitude)**: 성상도 위에서 신호의 품질을 평가하는 가장 강력한 수학적 잣대.
- **그레이 부호 (Gray Code)**: 성상도의 점들에 0과 1의 비트를 할당할 때 에러를 최소화하는 매핑 규칙.
