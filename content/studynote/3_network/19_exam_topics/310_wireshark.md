### Ⅰ. 개요 (Context & Background)

**개념 정의**

Wireshark는 오픈소스 GUI 기반 패킷 분석 도구입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Wireshark                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Wireshark 화면                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  [필터 바] tcp.port == 80                            │   │ │
│   │   │  ───────────────────────────────────────────────────│   │ │
│   │   │  [패킷 리스트]                                       │   │ │
│   │   │  No. Time   Source      Dest        Protocol Info    │   │ │
│   │   │  1   0.000  10.0.0.1    10.0.0.2    TCP      SYN     │   │ │
│   │   │  2   0.001  10.0.0.2    10.0.0.1    TCP      SYN,ACK │   │ │
│   │   │  3   0.002  10.0.0.1    10.0.0.2    TCP      ACK     │   │ │
│   │   │  ───────────────────────────────────────────────────│   │ │
│   │   │  [패킷 상세]                                          │   │ │

│   │   │  ▶ Transmission Control Protocol      │   │ │
│   │   │  ───────────────────────────────────────────────────│   │ │
│   │   │  [패킷 바이트]                                        │   │ │
│   │   │  00 0c 29 1a 2b 3c ...                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │

│   │   │     • TCP Stream                                     │   │ │

│   │   │                                                      │   │ │
```

> **해설**: Wireshark는 GUI로 패킷을 시각화하고 분석합니다.

**💡 비유**: Wireshark는 네트워크 현미경입니다. 모든 것을 확대해서 봅니다.

**📢 섹션 요약 비유:** Wireshark = 패킷 분석!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Capture Filter** | 캡처 필터 |
| **Display Filter** | 표시 필터 |
| **Follow Stream** | 스트림 추적 |
| **Expert Info** | 전문가 정보 |

**비교표: 필터 문법**

```
+------------+----------+----------+
| 필터       │ 설명     │ 예시     │
+------------+----------+----------+
| Capture    │ BPF      │ host x.x.x.x│
| Display    │ Wireshark│ ip.addr==x  │
+------------+----------+----------+
```

**핵심 알고리즸:** 프로토콜 디코딩

**📢 섹션 요약 비유:** Follow TCP Stream = 대화 보기!
