# IPv6 - 인터넷 프로토콜 버전 6

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 128비트 주소 체계
> 2. **가치**: 주소 고갈 해결, 단순화
> 3. **융합:** IPv4, Dual Stack, Tunneling

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IPv6는 128비트 주소를 사용하는 차세대 IP입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IPv6                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IPv6 주소 형식                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  IPv4: 192.168.1.1 (32비트, 43억 개)                 │   │ │
│   │   │                                                      │   │ │
│   │   │  IPv6: 2001:0db8:85a3:0000:0000:8a2e:0370:7334       │   │ │
│   │   │        (128비트, 3.4×10^38 개)                       │   │ │
│   │   │                                                      │   │ │
│   │   │  축약 표기:                                          │   │ │
│   │   │  2001:db8:85a3::8a2e:370:7334                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IPv6는 사실상 무한한 주소 공간을 제공합니다.

**📢 섹션 요약 비유:** IPv6 = 미래의 주소!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | IPv4 | IPv6 |
|:---|:---|:---|
| **주소 길이** | 32비트 | 128비트 |
| **헤더** | 복잡 | 단순 |
| **단편화** | 라우터 | 송신측 |
| **체크섬** | 있음 | 없음 |
| **IPsec** | 옵션 | 필수 |

**핵심 알고리즸:** 단순화된 헤더 처리
