# Delayed ACK - 지연 승인

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: ACK 전송 지연
> 2. **가치**: 패킷 수 감소, 효율
> 3. **융합:** TCP, Nagle, Piggybacking

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Delayed ACK는 ACK 전송을 잠시 지연합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Delayed ACK                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Delayed ACK 동작                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  송신측                    수신측                    │   │ │
│   │   │     │                         │                      │   │ │
│   │   │     │ ─── Data 1 ───────────→ │                      │   │ │
│   │   │     │                         │ (지연, ACK 보류)     │   │ │
│   │   │     │ ─── Data 2 ───────────→ │                      │   │ │
│   │   │     │                         │                      │   │ │
│   │   │     │ ←── ACK (2개 승인) ──── │ 200ms 내             │   │ │
│   │   │     │                         │                      │   │ │
│   │   │  → ACK 패킷 수 감소                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Delayed ACK는 ACK를 모아서 보내 효율을 높입니다.

**📢 섹션 요약 비유:** Delayed ACK = 모아서 승인!
