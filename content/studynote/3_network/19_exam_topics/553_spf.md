# SPF (Sender Policy Framework) - 발신자 정책 프레임워크

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이메일 발신자 검증
> 2. **가치**: 스팸/피싱 방지
> 3. **융합:** DNS TXT, DKIM, DMARC

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SPF는 DNS TXT 레코드로 발신 서버를 지정합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SPF                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SPF 동작                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  DNS TXT 레코드:                                     │   │ │
│   │   │  example.com. IN TXT "v=spf1 ip4:192.0.2.0/24       │   │ │
│   │   │                          include:_spf.google.com     │   │ │
│   │   │                          -all"                       │   │ │
│   │   │                                                      │   │ │
│   │   │  메일 수신 시:                                       │   │ │
│   │   │  1. 발신자 도메인 TXT 레코드 조회                    │   │ │
│   │   │  2. 발신 IP가 허용 목록에 있는지 확인                │   │ │
│   │   │  3. Pass/Fail/SoftFail 결과                          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SPF는 이메일 스푸핑을 방지합니다.

**📢 섹션 요약 비유:** SPF = 발신자 신원 확인!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **v=spf1** | SPF 버전 |
| **ip4** | IPv4 허용 |
| **ip6** | IPv6 허용 |
| **include** | 다른 도메인 포함 |
| **a** | A 레코드 허용 |
| **mx** | MX 서버 허용 |
| **all** | 기본 정책 |

**핵심 알고리즸:** TXT 조회 → IP 매칭 → 결과

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**SPF 결과**

| 결과 | 설명 |
|:---|:---|
| **Pass** | 허용 |
| **Fail** | 거부 |
| **SoftFail** | 의심 |
| **Neutral** | 판단 불가 |
