# Protocol Buffers - 프로토콜 버퍼

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 구조화된 데이터 직렬화
> 2. **가치**: 작은 크기, 빠른 처리
> 3. **융합:** gRPC, Cross-platform, Schema Evolution

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Protocol Buffers는 구조화된 데이터를 직렬화합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Protocol Buffers                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Protocol Buffers 구조                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  .proto 정의:                                        │   │ │
│   │   │  syntax = "proto3";                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  message Person {                                   │   │ │
│   │   │    string name = 1;                                 │   │ │
│   │   │    int32 age = 2;                                   │   │ │
│   │   │    repeated string emails = 3;                      │   │ │
│   │   │  }                                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  직렬화 비교 (Person {name: "Kim", age: 30}):        │   │ │
│   │   │  JSON:   {"name":"Kim","age":30}  → 27 bytes        │   │ │
│   │   │  Protobuf: [0A 03 4B 69 6D 10 1E]  → 8 bytes        │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                │   │ │
│   │   │  - 작은 크기 (3-10배 작음)                            │   │ │
│   │   │  - 빠른 직렬화/역직렬화                                │   │ │
│   │   │  - 스키마 진화                                        │   │ │
│   │   │  - 다국어 지원                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Protocol Buffers는 이진 형식으로 효율적입니다.

**📢 섹션 요약 비유:** Protocol Buffers = 압축 파일!
