# 전파 전파 (Propagation) - 지상파/천파/공간파

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전파가 공간을 이동하는 방식
> 2. **가치**: 통신 거리와 품질 결정
> 3. **융합**: 안테나, 주파수와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

전파 전파(Propagation)는 전파가 송신 안테나에서 수신 안테나로 이동하는 과정입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    전파 전파 (Propagation)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전파 경로 종류                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │          ＿＿＿＿＿＿＿＿ 전리층 ＿＿＿＿＿＿＿＿       │   │ │
│   │   │         /                                        \     │   │ │
│   │   │        /      천파 (Sky Wave)                     \    │   │ │
│   │   │       /       ↗                        ↖           \   │   │ │
│   │   │      /      ↗                              ↖         \  │   │ │
│   │   │     /     ↗                                    ↖       │   │ │
│   │   │    ──────── 지상파 (Ground Wave) ────────────────     │   │ │
│   │   │   송신   ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~   수신        │   │ │
│   │   │   ○     ────────────────────────────────►   ○          │   │ │
│   │   │          직진파 (Direct Wave)                          │   │ │
│   │   │                                                      │   │ │
│   │   │  ───────────────── 지표면 ──────────────────           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전파 경로별 특성                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 지상파 (Ground Wave / Surface Wave):               │   │ │
│   │   │     • 지표면을 따라 전파                               │   │ │
│   │   │     • 저주파 (LF, MF)에서 효과적                        │   │ │
│   │   │     • 장거리 통신 가능                                  │   │ │
│   │   │     • 야간보다 주간에 감쇠 큼                           │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 천파 (Sky Wave / Ionospheric Wave):                │   │ │
│   │   │     • 전리층 반사                                       │   │ │
│   │   │     • 단파 (HF)에서 효과적                              │   │ │
│   │   │     • 전 세계 통신 가능                                 │   │ │
│   │   │     • 계절/시간/태양 활동 영향                          │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 직진파/공간파 (Direct/Space Wave):                  │   │ │
│   │   │     • LOS (Line of Sight) 전파                         │   │ │
│   │   │     • VHF 이상에서 효과적                               │   │ │
│   │   │     • 거리 제한 (지평선)                                │   │ │
│   │   │     • 반사파/회절파 포함                                │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 주파수에 따라 최적의 전파 경로가 다릅니다.

**💡 비유**: 지상파는 도로, 천파는 비행기, 직진파는 터널입니다.

**📢 섹션 요약 비유:** 저주파 = 지상파, 단파 = 천파!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 전파 경로 | 주파수 | 특징 | 용도 |
|:---|:---|:---|:---|
| **지상파** | LF/MF | 지표면 따라 | AM 라디오 |
| **천파** | HF | 전리층 반사 | 국제방송 |
| **직진파** | VHF+ | LOS | TV/이동통신 |

**비교표**

```
+------------+----------+----------+
| 항목       | 지상파   | 천파     |
+------------+----------+----------+
| 주파수     │ LF/MF    │ HF       │
| 거리       │ 수백km   │ 전 세계  │
| 시간 영향   │ 주간감쇠 │ 야간증가 │
| 안정성     │ 높음     │ 낮음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 스넬의 법칙, 전리층 반사

**📢 섹션 요약 비유:** 전리층 = 거울!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 장거리 방송** → HF 천파
**시나리오 2: 지역 FM** → VHF 직진파
**시나리오 3: 이동통신** → UHF 직진파+반사파

**도입 체크리스트:** 주파수, 거리, 지형

**📢 섹션 요약 비유:** 주파수 선택 = 경로 결정!
