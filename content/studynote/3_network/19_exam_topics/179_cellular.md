# 셀룰러 (Cellular) - 셀/클러스터/재사용

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 영역 분할로 주파수 재사용
> 2. **가치**: 용량 증대, 커버리지 확장
> 3. **융합**: 이동통신의 기본 구조

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

셀룰러(Cellular)는 서비스 영역을 셀(Cell)로 분할하고 주파수를 재사용하여 용량을 높이는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    셀룰러 (Cellular)                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              셀 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │         ◆───◆───◆                                    │   │ │
│   │   │        / \ / \ / \        클러스터 (Cluster)         │   │ │
│   │   │       ◆───◆───◆───◆       (N=7)                      │   │ │
│   │   │      / \ / \ / \ / \                                 │   │ │
│   │   │     ◆───◆───◆───◆───◆                                │   │ │
│   │   │      \ / \ / \ / \ /                                 │   │ │
│   │   │       ◆───◆───◆───◆                                  │   │ │
│   │   │        \ / \ / \ /                                   │   │ │
│   │   │         ◆───◆───◆                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  ◆: 기지국 (BS)                                       │   │ │
│   │   │  셀 반경: R                                          │   │ │
│   │   │  클러스터 크기: N                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              주파수 재사용                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  주파수 재사용 거리: D = R·√(3N)                      │   │ │
│   │   │                                                      │   │ │
│   │   │  SIR (Signal-to-Interference Ratio):                 │   │ │
│   │   │  SIR = (D/R)^γ / i  (γ: 전파 감쇠 지수)               │   │ │
│   │   │                                                      │   │ │
│   │   │  N (클러스터 크기): 3, 4, 7, 12, 19, 21...           │   │ │
│   │   │  • N 작음: 용량 높음, 간섭 높음                        │   │ │
│   │   │  • N 큼: 용량 낮음, 간섭 낮음                          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │

```

> **해설**: 주파수 재사용은 셀룰러 용량의 핵심 원리입니다.

**💡 비유**: 셀룰러는 여러 사람이 같은 주파수를 다른 지역에서 쓰는 것입니다.

**📢 섹션 요약 비유:** 재사용 = 용량의 핵심!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소 | 설명 | 역할 |
|:---|:---|:---|
| **셀** | 기지국 커버리지 | 기본 단위 |
| **클러스터** | 재사용 그룹 | 간섭 관리 |
| **재사용 거리** | D = R√(3N) | 간섭 방지 |

**비교표**

```
+------------+----------+----------+
| 클러스터 N │ 재사용 거리│ 간섭    │
+------------+----------+----------+
| 3          │ 3R       │ 높음     │
| 7          │ 4.6R     │ 중간     │
| 12         │ 6R       │ 낮음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 핸드오프, 채널 할당

**📢 섹션 요약 비유:** N=7 = 표준!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 셀 타입**

| 셀 타입 | 반경 | 용도 |
|:---|:---:|:---|
| **펨토셀** | 10m | 실내 |
| **피코셀** | 100m | 실내/실외 |
| **마이크로** | 1km | 도심 |
| **매크로** | 10km | 광역 |

**📢 섹션 요약 비유:** HetNet = 이기종 네트워크!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 도심** → N=3, 소형 셀
**시나리오 2: 교외** → N=7, 매크로 셀
**시나리오 3: 실내** → 피코/펨토 셀

**도입 체크리스트:** 트래픽, 간섭, 비용

**📢 섹션 요약 비유:** 트래픽 높으면 소형 셀!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)
