# DiffServ (차등화 서비스) - Differentiated Services

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 클래스 기반 QoS
> 2. **가치**: 확장성, 단순성
> 3. **융합**: DSCP, PHB, EF/AF

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DiffServ는 트래픽을 클래스별로 차등 처리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DiffServ 구조                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PHB (Per-Hop Behavior)                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 클래스     │ DSCP     │ 용도     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ EF         │ 46       │ 음성     │               │   │ │
│   │   │  │ AF41-43    │ 34-36    │ 비디오   │               │   │ │
│   │   │  | AF31-33    │ 26-28    │ 중요데이터│              │   │ │
│   │   │  │ BE         │ 0        │ 일반     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: DiffServ는 에지에서 마킹, 코어에서 스케줄링합니다.

**📢 섹션 요약 비유:** DiffServ = 클래스 기반 QoS!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **DSCP** | DS Field 마킹 |
| **PHB** | Per-Hop Behavior |
| **EF** | Expedited Forwarding |
| **AF** | Assured Forwarding |

**핵심 알고리즸:** 분류, 마킹, 스케줄링

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**EF vs AF**

| 항목 | EF | AF |
|:---|:---|:---|
| **지연** | 최소 | 보통 |
| **손실** | 없음 | 가능 |
| **용도** | 음성 | 데이터 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 음성 → EF
**시나리오 2:** 영상 → AF41
**시나리오 3:** 일반 → BE
