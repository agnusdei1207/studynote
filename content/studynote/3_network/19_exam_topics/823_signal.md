# Signal - 신호

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 정보 운반 매체
> 2. **가치**: 통신 기반
> 3. **융합:** Analog, Digital, Bandwidth

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Signal은 정보를 운반하는 물리적 현상입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Signal                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              신호 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  아날로그 신호 (Analog):                               │   │ │
│   │   │  연속적, 무한 값                                       │   │ │
│   │   │        ∿∿∿∿∿∿∿∿∿∿∿∿                                 │   │ │
│   │   │  소리, 온도, 조도                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  디지털 신호 (Digital):                                │   │ │
│   │   │  이산적, 유한 값                                       │   │ │
│   │   │        ▔▔▔▔▁▁▁▁▔▔▔▔▁▁▁▁                              │   │ │
│   │   │  0과 1                                                │   │ │
│   │   │                                                      │   │ │
│   │   │  신호 특성:                                            │   │ │
│   │   │  - 진폭 (Amplitude): 세기                              │   │ │
│   │   │  - 주파수 (Frequency): 초당 진동수                      │   │ │
│   │   │  - 위상 (Phase): 시점                                  │   │ │
│   │   │  - 대역폭 (Bandwidth): 주파수 범위                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 신호는 통신 시스템의 기본 단위입니다.

**📢 섹션 요약 비유:** Signal = 정보의 파도!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Amplitude** | 진폭 |
| **Frequency** | 주파수 |
| **Phase** | 위상 |
| **Wavelength** | 파장 |
| **Period** | 주기 |

**핵심 알고리즸:** 소스 → 신호 생성 → 전송 → 수신 → 복원
