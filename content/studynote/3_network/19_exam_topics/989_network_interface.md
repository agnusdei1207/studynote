# Network Interface - 네트워크 인터페이스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 연결 지점
> 2. **가치**: 하드웨어/소프트웨어 연결
> 3. **융합:** NIC, MAC, IP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Network Interface는 네트워크 연결의 지점입니다.

**📢 섹션 요약 비유:** Network Interface = 출입구!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NIC** | 네트워크 카드 |
| **Driver** | 드라이버 |
| **MAC** | MAC 주소 |
| **IP** | IP 주소 |
| **MTU** | MTU |

**핵심 알고리듬:** 프레임 수신 → 드라이버 처리 → OS 전달

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**인터페이스 명명 규칙**

| 시스템 | 명명 규칙 |
|:---|:---|
| **Linux** | eth0, enp0s3 |
| **macOS** | en0, en1 |
| **Windows** | Ethernet, Wi-Fi |
