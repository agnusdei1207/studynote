# Gratuitous ARP - 자기 자신 ARP 광고

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 자신의 IP-MAC 매핑을 브로드캐스트
> 2. **가치**: IP 충돌 감지, ARP 캐시 갱신
> 3. **융합**: HA, VRRP, 클러스터링과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Gratuitous ARP는 호스트가 자신의 IP-MAC 매핑을 네트워크에 브로드캐스트하는 ARP 패킷입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Gratuitous ARP                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Gratuitous ARP 동작                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Host A (192.168.1.10)                              │   │ │
│   │   │                                                      │   │ │
│   │   │  ARP Request (Broadcast):                           │   │ │
│   │   │  "Who has 192.168.1.10? Tell 192.168.1.10"         │   │ │
│   │   │  (Sender IP = Target IP)                            │   │ │
│   │   │                                                      │   │ │
│   │   │  목적:                                                │   │ │
│   │   │  1. IP 충돌 감지                                     │   │ │
│   │   │  2. ARP 캐시 갱신                                    │   │ │
│   │   │  3. MAC 변경 알림                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              사용 사례                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 부팅 시 IP 충돌 확인                             │   │ │
│   │   │  2. VRRP/HSRP 장애 조치                              │   │ │
│   │   │  3. MAC 주소 변경 알림                               │   │ │
│   │   │  4. 가상 IP 인계 (Failover)                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Gratuitous ARP는 Sender IP = Target IP인 ARP 요청입니다.

**💡 비유**: Gratuitous ARP는 "나 여기 있어!"라고 방송하는 것입니다.

**📢 섹션 요약 비유:** Gratuitous = 자기 자신 광고!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Sender IP** | 자신의 IP |
| **Target IP** | 자신의 IP |
| **Sender MAC** | 자신의 MAC |
| **Target MAC** | 00:00:00:00:00:00 |

**비교표: 일반 ARP vs Gratuitous ARP**

```
+------------+----------+----------+
| 항목       │ 일반 ARP │ GARP     │
+------------+----------+----------+
| Sender IP  │ 자신     │ 자신     │
| Target IP  │ 타인     │ 자신     │
| 목적       │ 타인 MAC │ 알림     │
+------------+----------+----------+
```

**핵심 알고리즸:** 브로드캐스트, 캐시 갱신

**📢 섹션 요약 비유:** Sender IP = Target IP!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: VRRP와 GARP**

| 상황 | GARP 용도 |
|:---|:---|
| **Master 전환** | 가상 MAC 갱신 |
| **Failover** | ARP 캐시 갱신 |

**📢 섹션 요약 비유:** HA = GARP 필수!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 부팅** → IP 충돌 감지
**시나리오 2: HA** → Failover 시 GARP 전송
**시나리오 3: 보안** → GARP 차단 검토

**도입 체크리스트:** GARP 전송 설정, 보안 정책

**📢 섹션 요약 비유:** GARP 차단 = HA 문제!
