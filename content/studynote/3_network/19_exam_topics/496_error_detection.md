# 에러 검출 - Error Detection

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전송 오류 탐지
> 2. **가치**: 데이터 무결성 보장
> 3. **융합**: CRC, 체크섬, 패리티

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

에러 검출은 전송 중 오류를 탐지합니다.

```

│                    에러 검출 기법                                     │

│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              에러 검출 방식                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │

│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 패리티     │ 1bit     │ 문자     │               │   │ │
```

**📢 섹션 요약 비유:** Error Detection = 무결성 검사!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Parity** | 단일 비트 검출 |
| **Checksum** | 합계 기반 |
| **CRC** | 다항식 기반 |
| **FCS** | Frame Check Sequence |
