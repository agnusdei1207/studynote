# Port - 포트

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스 식별자
> 2. **가치**: 멀티플렉싱 지원
> 3. **융합:** Well-Known, Registered, Dynamic

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Port는 호스트 내 프로세스를 식별하는 번호입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Port                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              포트 범위                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  포트 범위 (16-bit: 0-65535):                            │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Well-Known (0-1023):                           ││   │ │
│   │   │  │    20/21: FTP                                   ││   │ │
│   │   │  │    22: SSH                                      ││   │ │
│   │   │  │    23: Telnet                                   ││   │ │
│   │   │  │    25: SMTP                                     ││   │ │
│   │   │  │    53: DNS                                      ││   │ │
│   │   │  │    80: HTTP                                     ││   │ │
│   │   │  │    443: HTTPS                                   ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Registered (1024-49151):                       ││   │ │
│   │   │  │    3306: MySQL                                  ││   │ │
│   │   │  │    5432: PostgreSQL                             ││   │ │
│   │   │  │    6379: Redis                                  ││   │ │
│   │   │  │    8080: HTTP Proxy                             ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Dynamic (49152-65535):                         ││   │ │
│   │   │  │    임시 포트 (Ephemeral)                          ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 포트는 하나의 IP에서 여러 서비스를 구분합니다.

**📢 섹션 요약 비유:** Port = 건물 호수!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)
