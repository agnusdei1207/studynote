# DDoS - 분산 서비스 거부 공격

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 다수 시스템으로 자원 고갈
> 2. **가치**: 서비스 마비, 비용 손실
> 3. **융합**: 방화벽, CDN, 클라우드 방어

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DDoS(Distributed Denial of Service)는 여러 시스템에서 서비스를 마비시킵니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DDoS                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DDoS 공격 구조                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [공격자]                                           │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ C&C 서버 제어                                   │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │          봇넷 (Botnet)                      │    │   │ │
│   │   │  │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐          │    │   │ │
│   │   │  │  │봇1  │ │봇2  │ │봇3  │ │...  │          │    │   │ │
│   │   │  │  └─────┘ └─────┘ └─────┘ └─────┘          │    │   │ │
│   │   │  │           │                               │    │   │ │
│   │   │  │           └──────────┬────────────────────┘    │   │ │
│   │   │  │                      ▼                         │   │ │
│   │   │  │              [타겟 서버]                       │    │   │ │
│   │   │  │              마비 (X)                          │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DDoS 유형                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 유형       │ 대상     │ 예시     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ Volumetric │ 대역폭   │ UDP Flood│               │   │ │
│   │   │  │ Protocol   │ 프로토콜 │ SYN Flood│               │   │ │
│   │   │  | Application│ L7      │ HTTP Flood│              │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: DDoS는 봇넷을 이용해 대규모 트래픽을 전송합니다.

**💡 비유**: DDoS는 많은 사람이 동시에 출입문을 막는 것입니다.

**📢 섹션 요약 비유:** DDoS = 서비스 마비!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Botnet** | 봇 네트워크 |
| **C&C Server** | 명령 제어 서버 |
| **Amplification** | 증폭 공격 |
| **Reflection** | 반사 공격 |

**비교표: DDoS 유형**

```
+------------+----------+----------+
| 항목       │ Volumetric│ App     │
+------------+----------+----------+
| 계층       │ L3/4     │ L7       │
| 탐지       │ 쉬움     │ 어려움   │
| 방어       │ 트래픽필터│ WAF      │
+------------+----------+----------+
```

**핵심 알고리즸:** 트래픽 분석, 속도 제한

**📢 섹션 요약 비유:** CDN = DDoS 방어!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 방어 방안**

| 항목 | 온프레미스 | CDN | 클라우드 |
|:---|:---|:---|:---|
| **용량** │ 제한 | 무제한 | 대규모 |
| **비용** │ 고정 | 사용량 | 사용량 |
| **효과** │ 제한 | 높음 | 높음 |

**📢 섹션 요약 비유:** 대역폭 공격 = CDN!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 대역폭 공격 → CDN/클라우드 scrubbing
**시나리오 2:** L7 공격 → WAF + 속도 제한
**시나리오 3:** SYN Flood → SYN Cookie

**도입 체크리스트:** 모니터링, 대응 계획, CDN 계약

**📢 섹션 요약 비유:** 대비 = DDoS 방어!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 서비스 가용성 보호

**미래 전망:** AI 기반 탐지, 자동 대응

**참고 표준:** NIST SP 800-83

**📢 섹션 요약 비유:** DDoS는 가용성 위협!
