# Memory Pool - 메모리 풀

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 메모리 블록 사전 할당
> 2. **가치**: 단편화 감소, 할당 속도
> 3. **융합:** Arena, Slab Allocator, jemalloc

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Memory Pool은 메모리를 사전 할당합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Memory Pool                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              메모리 풀 구조                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Without Pool:                                        │   │ │
│   │   │  malloc(100) → [  ] malloc(50) → [  ][  ] ...        │   │ │
│   │   │  (단편화 발생)                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  With Pool (Slab Allocator):                          │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │  Slab Class 32B                              │    │   │ │
│   │   │  │  [32][32][32][32][32][32][32][32]           │    │   │ │
│   │   │  ├─────────────────────────────────────────────┤    │   │ │
│   │   │  │  Slab Class 64B                              │    │   │ │
│   │   │  │  [64][64][64][64][64][64]                   │    │   │ │
│   │   │  ├─────────────────────────────────────────────┤    │   │ │
│   │   │  │  Slab Class 128B                             │    │   │ │
│   │   │  │  [128][128][128][128]                        │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                │   │ │
│   │   │  - 빠른 할당/해제                                      │   │ │
│   │   │  - 단편화 감소                                        │   │ │
│   │   │  - 캐시 친화적                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 메모리 풀은 할당 성능과 단편화를 개선합니다.
