# 경사 과부하 / 과립러 잡음 (Slope Overload / Granular Noise)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: DM의 두 가지 상충 문제
> 2. **가치**: 스텝 크기 설계의 딜레마 이해
> 3. **융합**: ADM, ADPCM과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

경사 과부하 잡음(Slope Overload Noise)은 스텝이 너무 작아 급변 신호를 따라가지 못하는 문제이고, 과립러 잡음(Granular Noise)은 스텝이 너무 커 천천히 변하는 신호에서 발생하는 진동입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    경사 과부하 / 과립러 잡음                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              경사 과부하 잡음 (Slope Overload)                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  발생 조건:                                               │   │ │
│   │   │  • 신호 변화율 > 스텝/샘플링 주기                         │   │ │
│   │   │  • Δ/Ts < |dx/dt|max                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  파형:                                                   │   │ │
│   │   │  원본:     ╱╱╱╱╱ (급경사)                                │   │ │
│   │   │                                                      │   │ │
│   │   │  DM:      ┌┌┌┌┌  (따라가기 실패)                         │   │ │
│   │   │           ↑                                            │   │ │
│   │   │          Δ 너무 작음                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  결과:                                                   │   │ │
│   │   │  • 신호 왜곡                                            │   │ │
│   │   │  • 급변 구간 에러                                       │   │ │
│   │   │  • 고주파 성분 손실                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  해결: Δ 증가 (또는 ADM)                                 │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              과립러 잡음 (Granular Noise)                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  발생 조건:                                               │   │ │
│   │   │  • 신호 변화율 << 스텝/샘플링 주기                        │   │ │
│   │   │  • Δ가 신호 변화보다 너무 큼                              │   │ │
│   │   │                                                      │   │ │
│   │   │  파형:                                                   │   │ │
│   │   │  원본:     ───────────── (평탄)                          │   │ │
│   │   │                                                      │   │ │
│   │   │  DM:      ┌┐┌┐┌┐┌┐  (진동)                              │   │ │
│   │   │           ↑                                            │   │ │
│   │   │          Δ 너무 큼                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  결과:                                                   │   │ │
│   │   │  • 계단형 잡음                                          │   │ │
│   │   │  • 저주파 대역 에러                                     │   │ │
│   │   │  • 배경 잡음                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  해결: Δ 감소 (또는 ADM)                                 │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              딜레마와 해결                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  딜레마:                                                 │   │ │
│   │   │  • Δ 크게 → 과부하 ↓, 과립러 ↑                          │   │ │
│   │   │  • Δ 작게 → 과부하 ↑, 과립러 ↓                          │   │ │
│   │   │  → 상충 관계!                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  해결:                                                   │   │ │
│   │   │  • ADM (적응형 Δ)                                       │   │ │
│   │   │  • 오버샘플링 (fs 증가)                                  │   │ │
│   │   │  • 다비트 DPCM                                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 이 두 잡음은 트레이드오프 관계에 있어 ADM이 최적 해법입니다.

**💡 비유**: 경사 과부하는 터무니없이 낮은 계단, 과립러는 터무니없이 높은 계단입니다.

**📢 섹션 요약 비유**: ADM이 두 잡음 동시 해결!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 잡음 유형 | 원인 | 증상 | 해결 |
|:---|:---|:---|:---|
| **과부하** | Δ 작음 | 급변 추적 실패 | Δ 증가 |
| **과립러** | Δ 큼 | 평탄 구간 진동 | Δ 감소 |
| **딜레마** | 상충 | 최적값 없음 | ADM |

**비교표**

```
+------------+----------+----------+
| 스텝 Δ     | 과부하   | 과립러   |
+------------+----------+----------+
| 작음       | 심함     | 적음     |
| 중간       | 중간     | 중간     |
| 큼         | 적음     | 심음     |
+------------+----------+----------+
```

**핵심 알고리즘**: 신호 분석 → 최적 Δ 탐색 → 적응 조절

**📢 섹션 요약 비유**: Δ 선택은 트레이드오프!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 음성 (급변 많음)** → 과부하 주의
