# 패치 안테나 (Patch/Microstrip Antenna)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 기판 위 금속 패치
> 2. **가치**: 소형, 경량, 저비용
> 3. **융합**: 모바일, 위성, 5G와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

패치 안테나(Patch Antenna)는 유전체 기판 위에 금속 패치를 형성한 평면 안테나입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    패치 안테나 (Patch Antenna)                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              패치 안테나 구조                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │   ┌─────────────────────────────┐                   │   │ │
│   │   │   │                             │  ← 패치 (Radiating)│   │ │
│   │   │   │      W (폭)                 │                    │   │ │
│   │   │   │    ←───────────→            │                   │   │ │
│   │   │   │                             │                   │   │ │
│   │   │   │         L (길이)            │                   │   │ │
│   │   │   │         ↓                   │                   │   │ │
│   │   │   └─────────────────────────────┘                   │   │ │
│   │   │   ═══════════════════════════════════                │   │ │
│   │   │        유전체 기판 (Substrate)                        │   │ │
│   │   │   ═══════════════════════════════════                │   │ │
│   │   │        접지면 (Ground Plane)                         │   │ │
│   │   │                                                      │   │ │
│   │   │  L ≈ λ/2·√(εr)  (공진 길이)                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              패치 안테나 특성                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 소형, 경량, 박형                                      │   │ │
│   │   │  • 제작 용이 (PCB 공정)                                  │   │ │
│   │   │  • 저비용                                                │   │ │
│   │   │  • 회로와 일체화 가능                                    │   │ │
│   │   │  • 배열 안테나 구현 용이                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                   │   │ │
│   │   │  • 대역폭 좁음 (1-5%)                                   │   │ │
│   │   │  • 이득 낮음 (5-9 dBi)                                  │   │ │
│   │   │  • 효율 낮음                                             │   │ │
│   │   │                                                      │   │ │
│   │   │  응용:                                                   │   │ │
│   │   │  • 스마트폰, 태블릿                                      │   │ │
│   │   │  • WiFi, 블루투스                                        │   │ │
│   │   │  • GPS                                                   │   │ │
│   │   │  • 5G mmWave 배열                                        │   │ │
│   │   │  • 위성 통신 단말                                        │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 패치 안테나는 모바일 기기와 5G mmWave의 핵심 안테나입니다.

**💡 비유**: 패치 안테나는 스티커처럼 얇아서 기기에 쉽게 붙일 수 있습니다.

**📢 섹션 요약 비유:** 패치 = 얇고 작고 저렴!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 파라미터 | 전형값 | 영향 |
|:---|:---|:---|
| **L (길이)** | λ/2·√εr | 공진 주파수 |
| **W (폭)** | 1-2×L | 임피던스 |
| **h (높이)** | 0.01-0.05λ | 대역폭 |
| **εr (유전율)** | 2-10 | 크기 |

**비교표**

```
+------------+----------+----------+
| 기판       │ εr       │ 용도     │
+------------+----------+----------+
| FR4        │ 4.4      │ 저가     │
│ Rogers     │ 2.2-10   │ 고주파   │
│ LTCC       │ 5-7      │ 모듈     │
│ PI (FPCB)  │ 3.5      │ 휨       │
+------------+----------+----------+
```

**핵심 알고리즸:** 공진 조건, 임피던스 정합

**📢 섹션 요약 비유:** 기판 높이↑ = 대역폭↑!
