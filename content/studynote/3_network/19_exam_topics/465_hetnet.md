# HetNet (이종 네트워크) - Heterogeneous Network

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 다양한 셀 유형의 혼합 네트워크
> 2. **가치**: 용량 증대, 커버리지 확장
> 3. **융합**: Macro + Small Cell, Wi-Fi, 5G

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

**💡 비유**: HetNet은 도로 계층(고속도로-일반도로-골목길)입니다.

**📢 섹션 요약 비유:** HetNet = 계층적 네트워크!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Macro** | 기본 커버리지 |
| **Small Cell** | 용량 증대 |
| **Wi-Fi** | 트래픽 오프로드 |
| **SON** | Self-Organizing Network |

**비교표: 셀 간 간섭 관리**

```
+------------+----------+----------+----------+
| 기술       │ 방식     │ 효과     │ 세대     │
+------------+----------+----------+----------+
| Frequency  │ 주파수   │ 격리     │ 기본     │
| eICIC      │ 시간     │ ABS      │ LTE      │
| CoMP       │ 협력     │ 다이버시티│ LTE-A   │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 간섭 조정, 핸드오버

**📢 섹션 요약 비유:** SON = 자동 최적화!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 오프로드 기술**

| 기술 | 설명 | 이점 |
|:---|:---|:---|
| **Wi-Fi Offload** | Wi-Fi로 분산 | 비용 |
| **LAA** | LTE + 비인가 | 대역폭 |
| **CBRS** | 공유 스펙트럼 | 용량 |

**📢 섹션 요약 비유:** Wi-Fi 오프로드 = 비용 절감!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 도심 → Macro + Small Cell
**시나리오 2:** 실내 → DAS + Wi-Fi
**시나리오 3:** 교외 → Macro 중심

**도입 체크리스트:** 트래픽, 간섭, 백홀

**📢 섹션 요약 비유:** 밀집 지역 = HetNet!
