# 주파수 분할 다중화 (FDM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서로 다른 주파수 대역으로 동시 전송
> 2. **가치**: 아날로그/디지털 모두 가능, 단순 구현
> 3. **융합**: 라디오, TV, 케이블 TV와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

FDM(Frequency Division Multiplexing)은 사용 가능한 대역폭을 여러 주파수 대역으로 나누어 각 신호가 다른 주파수로 전송되는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FDM (주파수 분할 다중화)                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FDM 원리                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  주파수 분할:                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  주파수                                               │   │ │
│   │   │    ▲                                                  │   │ │
│   │   │    │ ┌───┐GB┌───┐GB┌───┐GB┌───┐                    │   │ │
│   │   │    │ │CH1│ │CH2│ │CH3│ │CH4│                      │   │ │
│   │   │    │ └───┘ └───┘ └───┘ └───┘                      │   │ │
│   │   │    ├──────────────────────────────→                │   │ │
│   │   │    │    f1   f2   f3   f4   f5                     │   │ │
│   │   │    │         ↑                                      │   │ │
│   │   │    │    보호대역(Guard Band)                        │   │ │
│   │   │                                                      │   │ │
│   │   │  채널 할당:                                           │   │ │
│   │   │  • CH1: f1 ~ f2 (대역폭 B)                           │   │ │
│   │   │  • CH2: f2+GB ~ f3+GB                               │   │ │
│   │   │  • CH3: f3+2GB ~ f4+2GB                             │   │ │
│   │   │  • CH4: f4+3GB ~ f5+3GB                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FDM 송수신 구조                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  송신:                                                 │   │ │
│   │   │  CH1 ─→ [변조 f1] ─┐                                  │   │ │
│   │   │  CH2 ─→ [변조 f2] ─┼─→ [결합] ─→ 전송               │   │ │
│   │   │  CH3 ─→ [변조 f3] ─┤                                  │   │ │
│   │   │  CH4 ─→ [변조 f4] ─┘                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  수신:                                                 │   │ │
│   │   │  수신 ─→ [BPF f1] ─→ [복조] ─→ CH1                   │   │ │
│   │   │       ─→ [BPF f2] ─→ [복조] ─→ CH2                   │   │ │
│   │   │       ─→ [BPF f3] ─→ [복조] ─→ CH3                   │   │ │
│   │   │       ─→ [BPF f4] ─→ [복조] ─→ CH4                   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: FDM은 각 채널이 고유 주파수를 갖습니다.

**💡 비유**: FDM은 라디오 방송국이 각기 다른 주파수를 사용하는 것입니다.

**📢 섹션 요약 비유**: FDM = 주파수로 나누기!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **변조기** | 주파수 이동 | 혼합기 | 채널 |
| **BPF** | 대역 통과 | 필터 | 문지기 |
| **보호대역** | 간섭 방지 | 여유 공간 | 완충지 |
| **결합기** | 신호 합성 | 가산 | 병합 |

**비교표**

```
+------------+----------+----------+
| 항목       | FDM      | TDM      |
+------------+----------+----------+
| 분할 자원  | 주파수   | 시간     |
| 아날로그   | 가능     | 불가     |
| 보호대역   | 필요     | 불필요   |
| 동기       | 불필요   | 필요     |
+------------+----------+----------+
```

**핵심 알고리즸**: N신호 → 각각 변조(f1~fN) → 결합 → BPF로 분리 → 복조

**📢 섹션 요약 비유**: 보호대역은 FDM의 필수!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: FDM 응용**

| 응용 | 채널 수 | 대역폭 |
|:---|:---:|:---:|
| **AM 라디오** | 107 | 10kHz |
| **FM 라디오** | 100 | 200kHz |
| **TV 방송** | 68 | 6MHz |
| **CATV** | 100+ | 6MHz |

**📢 섹션 요약 비유**: 라디오/TV는 FDM!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 라디오 방송** → FDM
**시나리오 2: 케이블 TV** → FDM
**시나리오 3: 위성 통신** → FDM

**도입 체크리스트**: 대역폭, 채널 수, 보호대역

**📢 섹션 요약 비유**: FDM은 아날로그 친화적!
