# TDD (시분할 이중화)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 업링크와 다운링크를 시간으로 분리
> 2. **가치**: 주파수 효율, 비대칭 트래픽 지원
> 3. **융합**: 5G, WiFi, Massive MIMO와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

TDD(Time Division Duplexing)는 같은 주파수로 업링크(UL)와 다운링크(DL)를 시간으로 분리하여 전송하는 이중화 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TDD (시분할 이중화)                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TDD 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  주파수 사용:                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  시간                                                 │   │ │
│   │   │  ──────────────────────────────────────────→         │   │ │
│   │   │  ├─────┼─────┼─────┼─────┼─────┤                     │   │ │
│   │   │  │ DL  │ UL  │ DL  │ UL  │ DL  │                     │   │ │
│   │   │  └─────┴─────┴─────┴─────┴─────┘                     │   │ │
│   │   │   ↑                    ↑                              │   │ │
│   │   │  같은 주파수        같은 주파수                        │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 하나의 주파수 대역 사용                              │   │ │
│   │   │  • UL/DL 비율 유연 조절                                 │   │ │
│   │   │  • 채널 상호성 활용 (Massive MIMO)                      │   │ │
│   │   │  • Guard Period 필요                                   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TDD vs FDD                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  TDD:                                                   │   │ │
│   │   │  주파수                                                 │   │ │
│   │   │    ▲ ┌──────────────────────────────────┐            │   │ │
│   │   │    │ │ DL │ UL │ DL │ UL │ DL │ UL │   │            │   │ │
│   │   │    │ └──────────────────────────────────┘            │   │ │
│   │   │    └──────────────────────────────→ 시간             │   │ │
│   │   │      ←── 하나의 주파수 대역 ──→                       │   │ │
│   │   │                                                      │   │ │
│   │   │  FDD:                                                   │   │ │
│   │   │  주파수                                                 │   │ │
│   │   │    ▲ ┌────────────────────────────┐                  │   │ │
│   │   │    │ │ DL DL DL DL DL DL DL DL   │ ← DL 주파수       │   │ │
│   │   │    │ ├────────────────────────────┤                  │   │ │
│   │   │    │ │ UL UL UL UL UL UL UL UL   │ ← UL 주파수       │   │ │
│   │   │    │ └────────────────────────────┘                  │   │ │
│   │   │    └──────────────────────────────→ 시간             │   │ │
│   │   │      ←── 두 개의 주파수 대역 ──→                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: TDD는 5G Massive MIMO에 필수적인 이중화 방식입니다.

**💡 비유**: TDD는 왕복 도로에서 시간대별로 방향을 바꾸는 것입니다.

**📢 섹션 요약 비유**: TDD = 같은 주파수, 다른 시간!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **DL/UL 비율** | 트래픽 조절 | 가변 | 차선 |
| **GP** | 전환 시간 | 보호 구간 | 여유 |
| **상호성** | 채널 대칭 | UL→DL 활용 | 거울 |

**비교표**

```
+------------+----------+----------+
| 항목       | TDD      | FDD      |
+------------+----------+----------+
| 주파수     | 1개      | 2개      |
| UL/DL 비율 | 유연     | 고정     |
| 상호성     | O        | X        |
| 지연       | GP       | 낮음     |
+------------+----------+----------+
```

**핵심 알고리즸**: DL 전송 → GP → UL 전송 → GP → DL 전송

**📢 섹션 요약 비유**: Massive MIMO에는 TDD가 유리!
