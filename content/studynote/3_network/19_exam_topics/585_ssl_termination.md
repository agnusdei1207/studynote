# SSL Termination - SSL 종료

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프록시에서 HTTPS 복호화
> 2. **가치**: 서버 부하 감소, 인증서 관리
> 3. **융합:** Reverse Proxy, Load Balancer, TLS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SSL Termination은 프록시에서 암호화를 해제합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SSL Termination                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SSL Termination 흐름                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                LB/Proxy       Server        │   │ │
│   │   │     │                    │               │           │   │ │
│   │   │     │ ── HTTPS ───────→ │               │           │   │ │
│   │   │     │   (Encrypted)      │               │           │   │ │
│   │   │     │                    │               │           │   │ │
│   │   │     │               [Decrypt]           │           │   │ │
│   │   │     │                    │               │           │   │ │
│   │   │     │                    │ ── HTTP ────→ │           │   │ │
│   │   │     │                    │  (Plain)      │           │   │ │
│   │   │     │                    │               │           │   │ │
│   │   │  → 서버는 SSL 처리 부담 없음                           │   │ │
│   │   │  → 인증서는 LB/Proxy에서 관리                          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SSL Termination은 프록시에서 암호화를 처리합니다.

**📢 섹션 요약 비유:** SSL Termination = 문지기!
