# 혼잡 제어 - Congestion Control

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 혼잡 방지/완화
> 2. **가치**: 네트워크 안정성 유지
> 3. **융합**: TCP 혼잡 제어, AQM, ECN

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

혼잡 제어는 네트워크 과부하를 방지합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    혼잡 제어 기법                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              혼잡 제어 방식                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 방식       │ 설명     │ 위치     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ End-to-End │ TCP      │ 단말     │               │   │ │
│   │   │  │ Network    │ AQM      │ 라우터   │               │   │ │
│   │   │  | Link       │ Flow     │ 스위치   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 혼잡 제어는 단말과 네트워크가 협력합니다.

**📢 섹션 요약 비유:** Congestion Control = 교통 제어!
