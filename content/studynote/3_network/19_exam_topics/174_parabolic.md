# 파라볼라 안테나 (Parabolic Antenna) - 식탁/카세그레인

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 포물면 반사경 + 급전기
> 2. **가치**: 초고이득, 좁은 빔
> 3. **융합**: 위성, 마이크로파, 레이더와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

파라볼라 안테나(Parabolic Antenna)는 포물면 반사경을 사용하여 전파를 집중시키는 고이득 안테나입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    파라볼라 안테나 (Parabolic Antenna)               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              파라볼라 안테나 종류                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 식���형 (Prime Focus):                               │   │ │
│   │   │     • 급전기가 초점에 위치                              │   │ │
│   │   │     • 구조 단순                                        │   │ │
│   │   │     • 위성 TV 수신                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 카세그레인형 (Cassegrain):                          │   │ │
│   │   │     • 주반사경 + 부반사경                               │   │ │
│   │   │     • 급전기가 반사경 뒤                                │   │ │
│   │   │     • 위성 통신 지구국                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 그레고리형 (Gregorian):                             │   │ │
│   │   │     • 오목 부반사경                                     │   │ │
│   │   │     • 카세그레인 변형                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 오프셋형 (Offset):                                  │   │ │
│   │   │     • 급전기 그림자 없음                                │   │ │
│   │   │     • 위성 방송 수신                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              파라볼라 파라미터                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  이득 (Gain):                                         │   │ │
│   │   │  G(dBi) = 10·log₁₀(η·π·D/λ)² ≈ 10·log₁₀(k·D²·f)     │   │ │
│   │   │                                                      │   │ │
│   │   │  빔폭 (Beamwidth):                                     │   │ │
│   │   │  θ ≈ 70·λ/D (도)                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  D: 직경    λ: 파장    f: 주파수    η: 효율           │   │ │
│   │   │                                                      │   │ │
│   │   │  예: D=3m, f=12GHz → G≈40dBi, θ≈0.6°                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 파라볼라 안테나는 직경과 주파수에 비례하여 높은 이득을 얻습니다.

**💡 비유**: 파라볼라 안테나는 손전등의 반사경과 같습니다. 빛을 한 곳으로 모읍니다.

**📢 섹션 요약 비유:** D↑ = G↑ = 빔폭↓!

|:---|:---:|:---:|:---:|
| **급전기 위치** | 초점 | 주반사경 뒤 | 측면 |

```

| 0.5m       │ 25       │ 3.5      │

+------------+----------+----------+
```
