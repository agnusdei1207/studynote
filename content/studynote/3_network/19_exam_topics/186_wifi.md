# WiFi (802.11a/b/g/n/ac/ax/be)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 무선 LAN 표준 (IEEE 802.11)
> 2. **가치**: 무선 인터넷 접속
> 3. **융합**: 가정, 기업, 공공장소와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

WiFi는 IEEE 802.11 표준 기반의 무선 LAN 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    WiFi 표준별                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  표준    │ 연도  │ 주파수     │ 속도      │ 특징          │   │
│   │  ───────┼──────┼───────────┼─────────┼──────────────│   │
│   │  802.11a │ 1999  │ 5GHz       │ 54Mbps   │ OFDM          │   │
│   │  802.11b │ 1999  │ 2.4GHz     │ 11Mbps   │ DSSS          │   │
│   │  802.11g │ 2003  │ 2.4GHz     │ 54Mbps   │ OFDM          │   │
│   │  802.11n │ 2009  │ 2.4/5GHz   │ 600Mbps  │ MIMO          │   │
│   │  802.11ac│ 2013  │ 5GHz       │ 6.9Gbps  │ MU-MIMO       │   │
│   │  802.11ax│ 2019  │ 2.4/5/6GHz │ 9.6Gbps  │ OFDMA         │   │
│   │  802.11be│ 2024  │ 2.4/5/6GHz │ 46Gbps   │ EHT           │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              WiFi 핵심 기술                                  │   │
│   │                                                              │   │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  • CSMA/CA: 충돌 회피 다중 접속                        │   │ │
│   │   │  • OFDM: 직교 주파수 분할 다중화                        │   │ │
│   │   │  • MIMO: 다중 안테나                                   │   │ │
│   │   │  • 채널 결합: 20/40/80/160MHz                          │   │ │
│   │   │  • WPA3: 보안                                          │   │ │
│   │   │  • WPS: 간편 설정                                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: WiFi는 802.11n부터 MIMO를 도입하여 속도를 크게 향상했습니다.

**💡 비유**: WiFi는 집 안에서 인터넷 선 없이 연결되는 무선 네트워크입니다.

**📢 섹션 요약 비유:** 802.11ax = WiFi 6!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 표준 | 변조 | 채널폭 | 공간 스트림 |
|:---|:---|:---:|:---:|
| **11n** | OFDM | 40MHz | 4 |
| **11ac** | OFDM | 160MHz | 8 |
| **11ax** | OFDMA | 160MHz | 8 |
| **11be** | OFDMA | 320MHz | 16 |

**비교표**

```
+------------+----------+----------+
| 항목       │ 11ac     │ 11ax     │
+------------+----------+----------+
| 다중접속   │ OFDM     │ OFDMA    │
│ MIMO       │ MU-MIMO  │ MU-MIMO  │
│ 보안       │ WPA2     │ WPA3     │
│ 전력       │ 높음     │ TWT      │
+------------+----------+----------+
```

**핵심 알고리즸:** CSMA/CA, RTS/CTS, BSS 컬러링

**📢 섹션 요약 비유:** OFDMA = 11ax 핵심!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: WiFi 보안**

| 프로토콜 | 암호 | 인증 | 연도 |
|:---|:---|:---|:---|
| **WEP** | RC4 | WEP | 1997 |
| **WPA** | TKIP | PSK | 2003 |
| **WPA2** | AES-CCMP | PSK/EAP | 2004 |
| **WPA3** | AES-GCMP | SAE | 2018 |

**📢 섹션 요약 비유:** WPA3 = 최신 보안!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 가정** → 802.11ac, WPA2
**시나리오 2: 기업** → 802.11ax, WPA3-EAP
**시나리오 3: 고밀도** → 802.11ax, OFDMA

**도입 체크리스트:** 사용자 수, 대역폭, 보안

**📢 섹션 요약 비유:** WiFi 6E = 6GHz 대역!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 무선 인터넷 자유

**미래 전망:** WiFi 7 (802.11be), 6GHz 확대

**참고 표준:** IEEE 802.11

**📢 섹션 요약 비유:** WiFi는 무선 인터넷의 표준!
