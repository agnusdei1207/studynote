# OSPF - 개방형 최단 경로 우선

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 링크 상태 라우팅 프로토콜
> 2. **가치**: 빠른 수렴과 확장성
> 3. **융합:** Dijkstra, Area, LSA

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OSPF (Open Shortest Path First)는 링크 상태 기반 라우팅 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OSPF                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OSPF 영역 구조                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Area 계층:                                             │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │              ┌───────────────┐                 ││   │ │
│   │   │  │              │   Area 0      │                 ││   │ │
│   │   │  │              │   Backbone    │                 ││   │ │
│   │   │  │              └───────┬───────┘                 ││   │ │
│   │   │  │                      │                         ││   │ │
│   │   │  │         ┌────────────┼────────────┐            ││   │ │
│   │   │  │         │            │            │            ││   │ │
│   │   │  │    ┌────┴───┐   ┌────┴───┐   ┌────┴───┐       ││   │ │
│   │   │  │    │Area 1  │   │Area 2  │   │Area 3  │       ││   │ │
│   │   │  │    │Normal  │   │Normal  │   │Stub    │       ││   │ │
│   │   │  │    └────────┘   └────────┘   └────────┘       ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  라우터 유형:                                           │   │ │
│   │   │  - IR (Internal Router): 단일 영역                     │   │ │
│   │   │  - ABR (Area Border Router): 영역 경계                  │   │ │
│   │   │  - ASBR (AS Boundary Router): AS 경계                   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: OSPF는 대규모 네트워크의 표준입니다.

**📢 섹션 요약 비유:** OSPF = 네비게이션의 교통정보!
