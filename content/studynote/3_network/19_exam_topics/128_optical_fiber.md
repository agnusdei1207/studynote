# 광섬유 케이블 (Optical Fiber Cable)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 빛을 이용한 초고속 전송 매체
> 2. **가치**: Tbps급 대역폭, 장거리 전송
> 3. **융합**: 백본, 해저 케이블, FTTH와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

광섬유 케이블(Optical Fiber Cable)은 빛의 전반사 원리를 이용하여 데이터를 전송하는 초고속 전송 매체입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    광섬유 케이블 (Optical Fiber Cable)              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              구조                                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  단면도:                                                  │   │ │
│   │   │                                                      │   │ │
│   │   │        외부 피복 (Jacket)                             │   │ │
│   │   │      ┌────────────────────────┐                      │   │ │
│   │   │      │  보강재 (Strength)     │                      │   │ │
│   │   │      │  ┌──────────────────┐  │                      │   │ │
│   │   │      │  │ 클래딩 (Cladding)│  │  ← n₂ (낮은 굴절률)  │   │ │
│   │   │      │  │ ┌──────────────┐ │  │                      │   │ │
│   │   │      │  │ │ 코어 (Core)  │ │  │  ← n₁ (높은 굴절률)  │   │ │
│   │   │      │  │ │   빛 전파    │ │  │                      │   │ │
│   │   │      │  │ └──────────────┘ │  │                      │   │ │
│   │   │      │  └──────────────────┘  │                      │   │ │
│   │   │      └────────────────────────┘                      │   │ │
│   │   │                                                      │   │ │
│   │   │  전반사 원리:                                           │   │ │
│   │   │  n₁ > n₂ → 빛이 코어 내에서 전반사                      │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 초고속 (Tbps 이상)                                  │   │ │
│   │   │  • 장거리 (수십~수백 km)                               │   │ │
│   │   │  • EMI 면역                                            │   │ │
│   │   │  • 가볍고 얇음                                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              광섬유 종류                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  종류      │ 코어     │ 클래딩  │ 용도      │ 거리   │   │ │
│   │   │  ─────────┼─────────┼─────────┼──────────┼─────── │   │ │
│   │   │  멀티모드 │ 50/62.5µm│ 125µm   │ 건물내    │ 2km    │   │ │
│   │   │  단일모드 │ 9µm      │ 125µm   │ 장거리    │ 100km+ │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 광섬유는 전반사 원리로 빛을 코어 내에 가두어 전송합니다.

**💡 비유**: 광섬유는 빛이 터널을 통해 직진하는 것과 같습니다.

**📢 섹션 요약 비유**: 광섬유 = 빛의 고속도로!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **코어** | 빛 전파 | 높은 굴절률 | 터널 |
| **클래딩** | 전반사 | 낮은 굴절률 | 벽면 |
| **피복** | 보호 | 기계적 강도 | 외벽 |

**비교표**

```
+------------+----------+----------+
| 항목       | 멀티모드 | 단일모드 |
+------------+----------+----------+
| 코어       | 50/62.5µm| 9µm      |
| 광원       | LED/VCSEL| LD       |
| 거리       | 2km      | 100km+   |
| 비용       | 낮음     | 높음     |
| 대역폭     | 제한     | 무제한   |
+------------+----------+----------+
```

**핵심 알고리즸**: 전반사, 분산 관리

**📢 섹션 요약 비유**: 단일모드 = 장거리!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데이터센터** → OM3/OM4 멀티모드
**시나리오 2: 백본** → OS1/OS2 단일모드
**시나리오 3: FTTH** → 단일모드

**도입 체크리스트**: 거리, 속도, 예산, 커넥터

**📢 섹션 요약 비유**: 100m 이하 = 멀티모드!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: Tbps급 대역폭, EMI 면역

**미래 전망**: 400G/800G/1.6T 확대

**참고 표준**: ITU-T G.652, G.655

**📢 섹션 요약 비유**: 광섬유는 미래의 매체!
