# PDU - 프로토콜 데이터 단위

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 계층별 데이터 단위
> 2. **가치**: 캡슐화 구조
> 3. **융합:** Frame, Packet, Segment

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PDU (Protocol Data Unit)는 각 계층의 데이터 단위입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PDU (Protocol Data Unit)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              계층별 PDU                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Layer        PDU           예시                      │   │ │
│   │   │  ──────────────────────────────────────────────      │   │ │
│   │   │  L7 App      Data/Message   HTTP Request             │   │ │
│   │   │  L6 Pres     Data           Encrypted Data           │   │ │
│   │   │  L5 Sess     Data           Session Info             │   │ │
│   │   │  L4 Trans    Segment        TCP Segment              │   │ │
│   │   │              Datagram       UDP Datagram             │   │ │
│   │   │  L3 Network  Packet         IP Packet                │   │ │
│   │   │  L2 DataLink Frame          Ethernet Frame           │   │ │
│   │   │  L1 Physical Bit            Electrical Signal        │   │ │
│   │   │                                                      │   │ │
│   │   │  캡슐화:                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ [L2 Header][L3 Header][L4 Header][Data][Trailer]││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: PDU는 캡슐화와 역캡슐화의 기본 단위입니다.

**📢 섹션 요약 비유:** PDU = 택배 상자!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Header** | 헤더 |
| **Payload** | 페이로드 |
| **Trailer** | 트레일러 |
| **SDU** | 서비스 데이터 단위 |
| **PCI** | 프로토콜 제어 정보 |

**핵심 알고리즸:** SDU + PCI = PDU

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**PDU 크기 비교**

| 계층 | PDU | 일반 크기 |
|:---|:---|:---|
| **L4** | Segment | ~64KB |
| **L3** | Packet | ~1500B |
| **L2** | Frame | 64-1518B |
| **L1** | Bit | 1 bit |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** TCP → Segment 분할
**시나리오 2:** IP → Packet 라우팅
**시나리오 3:** Ethernet → Frame 전송
