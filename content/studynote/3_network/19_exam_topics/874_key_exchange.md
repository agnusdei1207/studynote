# Key Exchange - 키 교환

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 안전한 키 공유
> 2. **가치**: 도청 없는 키 전달
> 3. **융합:** Diffie-Hellman, RSA, ECDH

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Key Exchange는 안전하게 키를 공유합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Key Exchange                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Diffie-Hellman 키 교환                            │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  공개 값: p (소수), g (생성자)                          │   │ │
│   │   │                                                      │   │ │
│   │   │  Alice                    Bob                         │   │ │
│   │   │  ┌─────────┐            ┌─────────┐                  │   │ │
│   │   │  │ a (비밀)│            │ b (비밀)│                  │   │ │
│   │   │  │ A=g^a%p│ ── A ────→ │         │                  │   │ │
│   │   │  │         │ ←── B ──── │ B=g^b%p│                  │   │ │
│   │   │  │ K=B^a%p│            │ K=A^b%p│                  │   │ │
│   │   │  └─────────┘            └─────────┘                  │   │ │
│   │   │                                                      │   │ │
│   │   │  결과: K = g^(ab) % p (동일한 키)                       │   │ │
│   │   │  도청자: A, B만 볼 수 있고 K 계산 불가                   │   │ │
│   │   │                                                      │   │ │
│   │   │  변형:                                                 │   │ │
│   │   │  - ECDH: 타원곡선 기반 (더 효율적)                        │   │ │
│   │   │  - RSA: 공개키로 대칭키 암호화                           │   │ │
│   │   │                                                      │   │ │
│   │   │  Perfect Forward Secrecy (PFS):                        │   │ │
│   │   │  - 세션 키가 일회성                                    │   │ │
│   │   │  - 장기 키 노출되어도 과거 세션 안전                     │   │ │
│   │   │  - DHE, ECDHE                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 키 교환은 안전한 통신의 시작점입니다.

**📢 섹션 요약 비유:** Key Exchange = 암호화 열쇠 전달!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **DH** | Diffie-Hellman |
| **ECDH** | 타원곡선 DH |
| **RSA** | RSA 키 교환 |
| **PFS** | Perfect Forward Secrecy |
| **Pre-master Secret** | 예비 마스터 비밀 |

**핵심 알고리즸:** 공개값 교환 → 비밀 키 생성

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**키 교환 방식 비교**

| 방식 | PFS | 속도 | 보안 |
|:---|:---|:---|:---|
| **RSA** | 없음 | 빠름 | 중간 |
| **DH** | 있음 | 느림 | 높음 |
| **ECDH** | 있음 | 빠름 | 높음 |
