# Atomic Operation - 원자적 연산

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 분할 불가능한 연산
> 2. **가치**: 경쟁 상태 방지
> 3. **융합:** CAS, Lock-free, Memory Barrier

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Atomic Operation은 중단 없이 완료되는 연산입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Atomic Operation                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              원자적 연산                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  일반 연산 (비원자적):       원자적 연산:             │   │ │
│   │   │  ┌─────────────────────┐    ┌─────────────────────┐  │   │ │
│   │   │  │ read                │    │                     │  │   │ │
│   │   │  │ ↓                   │    │  Atomic Increment   │  │   │ │
│   │   │  │ modify    ← 간섭    │    │  (한 번에 완료)     │  │   │ │
│   │   │  │ ↓                   │    │                     │  │   │ │
│   │   │  │ write               │    │                     │  │   │ │
│   │   │  └─────────────────────┘    └─────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  CAS (Compare-And-Swap):                             │   │ │
│   │   │  if (value == expected) value = new; return success; │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 원자적 연산은 하드웨어 수준에서 보장됩니다.

**📢 섹션 요약 비유:** Atomic Operation = 단일 동작!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **CAS** | Compare-And-Swap |
| **FAA** | Fetch-And-Add |
| **Test-And-Set** | 테스트-설정 |
| **Load-Linked** | 로드-링크드 |
| **Store-Conditional** | 스토어-컨디셔널 |

**핵심 알고리즸:** 읽기 → 비교 → (일치?) 교체
