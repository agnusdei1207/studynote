# DHCP (Dynamic Host Configuration Protocol) - 동적 호스트 설정 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IP 주소 자동 할당
> 2. **가치**: 관리 편의성, 주소 효율
> 3. **융합:** DORA, Lease, Relay

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DHCP는 IP 주소와 네트워크 설정을 자동 할당합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DHCP                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DORA 프로세스                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                              DHCP Server     │   │ │
│   │   │     │                                      │          │   │ │
│   │   │     │ ── DHCPDISCOVER (Broadcast) ──────→ │          │   │ │
│   │   │     │    "IP 주 있어?"                     │          │   │ │
│   │   │     │                                      │          │   │ │
│   │   │     │ ←── DHCPOFFER (Unicast) ─────────── │          │   │ │
│   │   │     │    "192.168.1.100 쓸래?"            │          │   │ │
│   │   │     │                                      │          │   │ │
│   │   │     │ ── DHCPREQUEST (Broadcast) ───────→ │          │   │ │
│   │   │     │    "그거 쓸게!"                     │          │   │ │
│   │   │     │                                      │          │   │ │
│   │   │     │ ←── DHCPACK (Unicast) ───────────── │          │   │ │
│   │   │     │    "OK! Lease 24시간"               │          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: DHCP는 DORA 4단계로 IP를 할당합니다.

**📢 섹션 요약 비유:** DHCP = 집 주소 자동 배정!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **DORA** | Discover-Offer-Request-Ack |
| **Lease** | 할당 기간 |
| **Scope** | 할당 범위 |
| **Options** | 추가 설정 (GW, DNS) |
| **Relay** | 다른 서브넷 지원 |

**핵심 알고리즸:** Discover → Offer → Request → Ack

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**DHCP Options**

| Code | 설명 |
|:---|:---|
| **1** | Subnet Mask |
| **3** | Default Gateway |
| **6** | DNS Server |
| **51** | Lease Time |
| **53** | Message Type |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 2:** 서버 → Static IP
**시나리오 3:** 다중 서브넷 → DHCP Relay
