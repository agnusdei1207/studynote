# Vector Processor - 벡터 프로세서

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 벡터 연산 전용 프로세서
> 2. **가치**: 고성능 과학 계산

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Vector Processor는 벡터 연산에 최적화된 프로세서입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Vector Processor                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              벡터 프로세서 구조                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Scalar Processor:        Vector Processor:          │   │ │
│   │   │  ┌─────────────────┐      ┌───────────────────────┐ │   │ │
│   │   │  │ A[i] = B[i] + C │      │ V1 = V2 + V3          │ │   │ │
│   │   │  │ (loop 100 times)│      │ (단일 명령)           │ │   │ │
│   │   │  └─────────────────┘      └───────────────────────┘ │   │ │
│   │   │                                                      │   │ │
│   │   │  Vector Register:                                     │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ V1[0] │ V1[1] │ V1[2] │ ... │ V1[N]           │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ V2[0] │ V2[1] │ V2[2] │ ... │ V2[N]           │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │               +                                     │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ V3[0] │ V3[1] │ V3[2] │ ... │ V3[N]           │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 벡터 프로세서는 배열 연산을 효율적으로 처리합니다.

**📢 섹션 요약 비유:** Vector Processor = 대량 계산기!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Scalar vs Vector**

| 항목 | Scalar | Vector |
|:---|:---|:---|
| **연산 단위** | 스칼라 | 벡터 |
| **루프** | 필요 | 불필요 |
| **메모리** | 불연속 | 연속 |
| **용도** | 일반 | 과학 계산 |
