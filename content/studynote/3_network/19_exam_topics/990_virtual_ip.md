# Virtual IP - 가상 IP

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 논리적 IP 주소
> 2. **가치**: 고가용성과 이동성
> 3. **융합:** VIP, VRRP, Floating IP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Virtual IP는 물리적 인터페이스에 종속되지 않는 IP입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Virtual IP                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              가상 IP 사용                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  VIP (Virtual IP) 예시:                                  │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌──────────────┐        ┌──────────────┐        ││   │ │
│   │   │  │  │   Server 1   │        │   Server 2   │        ││   │ │
│   │   │  │  │  Real IP:    │        │  Real IP:    │        ││   │ │
│   │   │  │  │  10.0.0.1    │        │  10.0.0.2    │        ││   │ │
│   │   │  │  └──────┬───────┘        └──────┬───────┘        ││   │ │
│   │   │  │         │                       │                  ││   │ │
│   │   │  │         └───────────┬───────────┘                  ││   │ │
│   │   │  │                     │                              ││   │ │
│   │   │  │              ┌──────┴──────┐                       ││   │ │
│   │   │  │              │   VIP       │                       ││   │ │

│   │   │  │              └─────────────┘                       ││   │ │
│   │   │  │                                                 ││   │ │
```

> **해설**: Virtual IP는 서비스 고가용성의 핵심입니다.

**📢 섹션 요약 비유:** Virtual IP = 대표 전화번호!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VIP** | 가상 IP |
| **Real IP** | 실제 IP |
| **Floating** | 플로팅 |

**핵심 알고리듬:** 마스터 선출 → VIP 할당 → 장애 시 이전
