# 마이크로파 (Microwave) - 1GHz ~ 300GHz

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 1GHz~300GHz 고주파 전파
> 2. **가치**: 광대역, 고용량 통신
> 3. **융합**: 위성, 레이더, 5G와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

마이크로파(Microwave)는 1GHz~300GHz 대역의 전파로, SHF(3-30GHz)와 EHF(30-300GHz)를 포함합니다.

```

│                    마이크로파 (Microwave)                            │
├─────────────────────────────────────────────────────────────────────────┤

│   │   │  대역      │ 주파수       │ 용도                    │   │ │
│   │   │  ─────────┼─────────────┼────────────────────────│   │ │
│   │   │  L-band   │ 1-2GHz      │ GPS, 레이더             │   │ │
│   │   │  S-band   │ 2-4GHz      │ 레이더, 위성            │   │ │
│   │   │  C-band   │ 4-8GHz      │ 위성 TV                 │   │ │
│   │   │  X-band   │ 8-12GHz     │ 레이더                  │   │ │
│   │   │  Ku-band  │ 12-18GHz    │ 위성, VSAT              │   │ │
│   │   │  K-band   │ 18-27GHz    │ 위성                    │   │ │
│   │   │  Ka-band  │ 27-40GHz    │ 위성, 5G                │   │ │
│   │   │  V-band   │ 40-75GHz    │ 60GHz WiFi              │   │ │
│   │   │  E-band   │ 60-90GHz    │ 백홀                    │   │ │

│   │   │  • 작은 안테나                                           │   │ │
```

> **해설**: 마이크로파는 광대역 고용량 통신의 핵심 주파수 대역입니다.

**💡 비유**: 마이크로파는 레이저처럼 직진하는 고주파 전파입니다.

**📢 섹션 요약 비유:** 마이크로파 = 광대역, LOS 필수!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **파라볼라 안테나** | 빔 집중 | 반사경 | 렌즈 |
| **도파관** | 신호 전송 | 중공관 | 파이프 |
| **MAGIC-T** | 신호 분리 | 하이브리드 | 분배기 |

**비교표**

```
+------------+----------+----------+
| 항목       | C-band   | Ka-band  |
+------------+----------+----------+
| 주파수     │ 4-8GHz   │ 27-40GHz │
| 대역폭     │ 좁음     │ 넓음     |
| 강우 감쇠   │ 적음     │ 큼       |
| 안테나     │ 큼       │ 작음     |
+------------+----------+----------+
```

**핵심 알고리즸:** LOS 전파, 빔포밍

**📢 섹션 요약 비유:** Ka-band = 5G mmWave!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 응용 분야**

| 용도 | 대역 | 특징 |
|:---|:---|:---|
| **위성 TV** | C/Ku | DBS |
| **VSAT** | Ku/Ka | 소형 안테나 |
| **5G** | Ka/mmWave | 고속 |

**📢 섹션 요약 비유:** 5G = 마이크로파 + mmWave!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 위성 통신** → Ku/Ka band
**시나리오 2: 지상 링크** → 18/23GHz
**시나리오 3: 5G** → mmWave (28/39GHz)

**도입 체크리스트:** 거리, 대역폭, 기상 조건

**📢 섹션 요약 비유:** 마이크로파는 광대역의 열쇠!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 광대역 고용량 통신

**미래 전망:** 5G/6G mmWave 확대

**참고 표준:** ITU-R F 시리즈

**📢 섹션 요약 비유:** 마이크로파는 통신의 고속도로!
