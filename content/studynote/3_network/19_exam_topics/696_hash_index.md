# Hash Index - 해시 인덱스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 해시 테이블 기반 인덱스
> 2. **가치**: O(1) 등가 검색
> 3. **융합:** Key-Value, Memory, Equality

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Hash Index는 해시 함수로 키를 매핑합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Hash Index                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              해시 인덱스 구조                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Key        Hash Function      Bucket               │   │ │
│   │   │  ┌─────┐    ┌─────────┐       ┌─────────┐           │   │ │
│   │   │  │"ABC"│ →  │ hash()  │  →    │ Bucket 0│ → [Data] │   │ │
│   │   │  └─────┘    └─────────┘       ├─────────┤           │   │ │
│   │   │  ┌─────┐                      │ Bucket 1│ → [Data] │   │ │
│   │   │  │"XYZ"│ →  hash() →          ├─────────┤           │   │ │
│   │   │  └─────┘                      │ Bucket 2│ → [Data] │   │ │
│   │   │                               │         │   [Data]  │   │ │
│   │   │  Collision:                   └─────────┘           │   │ │
│   │   │  - Chaining: 연결 리스트                            │   │ │
│   │   │  - Open Addressing: 다음 버킷                       │   │ │
│   │   │                                                      │   │ │
│   │   │  시간 복잡도: O(1) 평균                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 해시 인덱스는 등가 검색에 최적화됩니다.

**📢 섹션 요약 비유:** Hash Index = 우편 번호!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Hash vs B-Tree**

| 항목 | Hash | B-Tree |
|:---|:---|:---|
| **검색** | O(1) | O(log n) |
| **범위** | 불가능 | 가능 |
| **정렬** | 없음 | 있음 |
| **용도** | 등가 검색 | 범위 검색 |
