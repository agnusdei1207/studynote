# SQL Injection - SQL 인젝션

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 악의적 SQL 주입
> 2. **가치**: 데이터 탈취/변조
> 3. **융합:** OWASP, WAF, Prepared Statement

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SQL Injection은 입력값에 SQL을 주입합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SQL Injection                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              공격 예시                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  정상 입력:                                          │   │ │
│   │   │  SELECT * FROM users WHERE id = '123'               │   │ │
│   │   │                                                      │   │ │
│   │   │  공격 입력: ' OR '1'='1                              │   │ │
│   │   │  SELECT * FROM users WHERE id = '' OR '1'='1'       │   │ │
│   │   │  → 모든 레코드 반환                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  방어:                                               │   │ │
│   │   │  - Prepared Statement (Parameterized Query)         │   │ │
│   │   │  - Input Validation                                  │   │ │
│   │   │  - WAF                                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SQL Injection은 OWASP Top 3 취약점입니다.

**📢 섹션 요약 비유:** SQL Injection = 위조 문서!
