# 테라헤르츠 (THz) - 0.1 ~ 10THz

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 0.1~10THz, 마이크로파와 적외선 사이
> 2. **가치**: 6G, 초고속 무선
> 3. **융합**: 이미징, 센싱과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

테라헤르츠(THz)파는 0.1~10THz 대역의 전자파로, 마이크로파와 적외선 사이 영역입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    테라헤르츠 (Terahertz)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전자파 스펙트럼 위치                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  마이크로파     THz         적외선      가시광       │   │ │
│   │   │  ◀─────────┼──────────┼──────────┼─────────▶        │   │ │
│   │   │           100GHz     1THz      10THa    400THz      │   │ │
│   │   │                                                      │   │ │
│   │   │  특성:                                                  │   │ │
│   │   │  • 전자파와 광파의 중간                                  │   │ │
│   │   │  • 비이온화 (안전)                                      │   │ │
│   │   │  • 많은 물질 투과                                       │   │ │
│   │   │  • 지문/분자 식별 가능                                   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              THz 통신 특성                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 초대역폭 (수십 GHz)                                   │   │ │
│   │   │  • Tbps급 속도 가능                                      │   │ │
│   │   │  • 높은 공간 해상도                                      │   │ │
│   │   │  • 물질 투과/식별                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                   │   │ │
│   │   │  • 매우 짧은 거리 (수m~수십m)                            │   │ │
│   │   │  • 대기 감쇠 심함                                         │   │ │
│   │   │  • 장비 비용 높음                                         │   │ │
│   │   │  • 기술 미성숙                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  응용:                                                   │   │ │
│   │   │  • 6G 이동통신                                           │   │ │
│   │   │  • 보안 이미징                                            │   │ │
│   │   │  • 의료 진단                                              │   │ │
│   │   │  • 화학 분석                                              │   │ │
│   │   │  • 비파괴 검사                                            │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: THz는 6G의 유력 후보 대역으로 Tbps급 통신과 센싱을 가능하게 합니다.

**💡 비유**: THz는 엑스레이처럼 물체를 투과하지만 해롭지 않습니다.

**📢 섹션 요약 비유:** THz = 6G 후보!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **THz 소자** | 발생/검출 | 광전도, QCL | LED/포토다이오드 |
| **변조기** | 신호 변조 | 전광 변조 | 스위치 |
| **안테나** | 신호 방사 | 고이득 안테나 | 혼 안테나 |

**비교표**

```
+------------+----------+----------+
| 항목       | mmWave   | THz      |
+------------+----------+----------+
| 주파수     │ 30-300GHz│ 0.1-10THz│
| 속도       │ 수Gbps   │ Tbps     │
| 거리       │ 수백m    │ 수m~수십m│
| 성숙도     │ 상용화    │ 연구단계  │
+------------+----------+----------+
```

**핵심 알고리즸:** THz 발생, 변조, 검출

**📢 섹션 요약 비유:** THz는 미래 기술!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 주파수 대역**

| 대역 | 주파수 | 속도 | 거리 |
|:---|:---:|:---:|:---:|
| **Sub-6** | <6GHz | 수백Mbps | km |
| **mmWave** | 30-300GHz | Gbps | 수백m |
| **THz** | 0.1-10THz | Tbps | 수m |

**📢 섹션 요약 비유:** THz = 최단 거리 최고 속도!
