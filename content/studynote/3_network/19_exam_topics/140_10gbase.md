# 10GBASE-T / 10GBASE-SR / 10GBASE-LR

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 10Gbps 이더넷, UTP/광섬유 매체
> 2. **가치**: 데이터센터, 서버 표준
> 3. **융합**: PAM-16, 광섬유와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

10GBASE 시리즈는 10Gbps 이더넷 표준으로, 매체에 따라 T(UTP), SR(단파장 광섬유), LR(장파장 광섬유)로 구분됩니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    10GBASE 시리즈                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              10GBASE 비교                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  표준       │ 매체    │ 변조    │ 거리    │ 용도    │   │ │
│   │   │  ──────────┼─────────┼─────────┼─────────┼─────────│   │ │
│   │   │  10GBASE-T │ UTP     │ PAM-16  │ 100m*   │ 서버    │   │ │
│   │   │  10GBASE-SR│ MMF     │ 64b/66b │ 400m    │ DC      │   │ │
│   │   │  10GBASE-LR│ SMF     │ 64b/66b │ 10km    │ 백본    │   │ │
│   │   │  10GBASE-ER│ SMF     │ 64b/66b │ 40km    │ MAN     │   │ │
│   │   │  10GBASE-KR│ Backplane│ 64b/66b│ 1m      │ 라우터  │   │ │
│   │   │                                                      │   │ │
│   │   │  * 10GBASE-T: Cat6a=100m, Cat6=55m                   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              10GBASE-T 상세                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  특징:                                                   │   │ │
│   │   │  • 4페어 전이중                                          │   │ │
│   │   │  • PAM-16 변조 (16레벨)                                  │   │ │
│   │   │  • 400MHz 대역폭                                         │   │ │
│   │   │  • Cat6a 이상 권장 (Cat6: 55m 제한)                      │   │ │
│   │   │  • 높은 전력 소비 (4-6W)                                 │   │ │
│   │   │  • 복잡한 DSP 보정                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  PAM-16:                                                  │   │ │
│   │   │  - 16개 전압 레벨                                        │   │ │
│   │   │  - 심볼당 약 3.5비트                                     │   │ │
│   │   │  - 833Mbaud × 4페어 = 10Gbps                           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              10GBASE-SR/LR 광섬유                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  10GBASE-SR (Short Reach):                              │   │ │
│   │   │  • 파장: 850nm                                           │   │ │
│   │   │  • 광섬유: OM3 (300m), OM4 (400m)                       │   │ │
│   │   │  • 광원: VCSEL                                           │   │ │
│   │   │  • 저비용                                                │   │ │
│   │   │                                                      │   │ │
│   │   │  10GBASE-LR (Long Reach):                                │   │ │
│   │   │  • 파장: 1310nm                                          │   │ │
│   │   │  • 광섬유: SMF (10km)                                    │   │ │
│   │   │  • 광원: DFB-LD                                          │   │ │
│   │   │  • 장거리                                                │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 10GBASE-T는 UTP로 10G를, SR/LR은 광섬유로 10G를 전송합니다.

**💡 비유**: 10GBASE는 초고속도로, T는 일반 도로 확장, SR/LR은 전용 고속도로입니다.

**📢 섹션 요약 비유:** T=UTP 100m, SR=MMF 400m, LR=SMF 10km!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | T | SR | LR |
|:---|:---|:---|:---|
| **매체** | UTP | MMF | SMF |
| **변조** | PAM-16 | 64b/66b | 64b/66b |
| **광원** | - | VCSEL | DFB-LD |
| **전력** | 4-6W | 1W | 1W |

**비교표**

```
+------------+----------+----------+
| 항목       | 10GBASE-T| 10GBASE-SR|
+------------+----------+----------+
| 비용       | 높음     | 낮음     |
| 전력       | 높음     | 낮음     |
| 거리       | 100m     | 400m     |
| 설치       | 쉬움     | 어려움   |
+------------+----------+----------+
```

**핵심 알고리즸**: PAM-16, 64b/66b 코딩

**📢 섹션 요약 비유:** Cat6 = 55m, Cat6a = 100m!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: Cat별 10G 거리**

| Cat | 거리 |
|:---|:---:|
| **Cat6** | 55m |
| **Cat6a** | 100m |
| **Cat7** | 100m |

**📢 섹션 요약 비유:** 10G = Cat6a!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 서버 연결** → 10GBASE-T
**시나리오 2: DC 내부** → 10GBASE-SR
**시나리오 3: 백본** → 10GBASE-LR

**도입 체크리스트**: 거리, 매체, 전력, 비용

**📢 섹션 요약 비유:** 10G는 서버 표준!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 10배 대역폭 향상

**미래 전망**: 25G/40G/100G로 진화

**참고 표준:** IEEE 802.3an (T), 802.3ae (SR/LR)

**📢 섹션 요약 비유:** 10GBASE는 차세대 표준!
