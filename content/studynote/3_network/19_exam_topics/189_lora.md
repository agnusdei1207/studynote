# LoRa / LPWAN (NB-IoT/Sigfox)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 저전력 광역 네트워크
> 2. **가치**: 수km 커버리지, 수년 배터리
> 3. **융합**: IoT, 스마트시티와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

LPWAN(Low Power Wide Area Network)는 저전력으로 수km 커버리지를 제공하는 IoT 통신 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    LPWAN 기술별                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  기술    │ 주파수     │ 거리    │ 속도     │ 전력   │   │
│   │  ───────┼──────────┼───────┼────────┼───────│   │
│   │  LoRa   │ Sub-GHz   │ 10-15km │ 0.3-50kbps│ 매우낮음│   │
│   │  NB-IoT │ LTE 대역   │ 10km    │ 250kbps   │ 낮음   │   │
│   │  Sigfox │ Sub-GHz   │ 10-50km │ 100bps    │ 매우낮음│   │
│   │  LTE-M  │ LTE 대역   │ 10km    │ 1Mbps     │ 낮음   │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              LoRa 특성                                      │   │
│   │                                                              │   │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 장거리 (10-15km)                                      │   │ │
│   │   │  • 저전력 (10+ 년 배터리)                                │   │ │
│   │   │  • 저비용                                                │   │ │
│   │   │  • 비면허 대역                                           │   │ │
│   │   │  • 강한 건물 투과                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                   │   │ │
│   │   │  • 낮은 데이터 속도                                      │   │ │
│   │   │  • 높은 지연                                              │   │ │
│   │   │  • 작은 페이로드                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  응용:                                                   │   │ │
│   │   │  • 스마트 미터링                                          │   │ │
│   │   │  • 스마트 농업                                            │   │ │
│   │   │  • 자산 추적                                              │   │ │
│   │   │  • 스마트시티                                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: LoRa는 CSS(Chirp Spread Spectrum)를 사용하여 장거리 전송을 가능하게 합니다.

**💡 비유**: LoRa는 멀리서 작은 소리를 듣는 것과 같습니다.

**📢 섹션 요약 비유:** LoRa = 장거리 저전력!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소 | 설명 | 역할 |
|:---|:---|:---|
| **LoRa** | 물리 계층 | 변조 |
| **LoRaWAN** | MAC 계층 | 프로토콜 |
| **게이트웨이** | 중계기 | 수집 |
| **네트워크 서버** | 서버 | 관리 |

**비교표**

```
+------------+----------+----------+
| 항목       │ LoRa     │ NB-IoT   │
+------------+----------+----------+
| 대역       │ 비면허   │ LTE      │
│ QoS        │ 없음     │ 있음     │
│ 보안       │ AES-128  │ LTE      │
│ 이동성     │ 제한     │ 지원     │
+------------+----------+----------+
```

**핵심 알고리즸:** CSS 변조, ADR(Adaptive Data Rate)

**📢 섹션 요약 비유:** SF(Spreading Factor) ↑ = 거리 ↑!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: LPWAN 선택**

| 용도 | 추천 기술 | 이유 |
|:---|:---|:---|
| **스마트 미터** | NB-IoT | QoS |
| **농업 센서** | LoRa | 비용 |
| **자산 추적** | NB-IoT | 이동성 |
| **스마트시티** | 혼합 | 용도별 |

**📢 섹션 요약 비유:** 용도에 맞는 LPWAN!
