# 대역폭-지연 곱 - Bandwidth-Delay Product

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 대역폭과 지연의 곱
> 2. **가치**: 윈도우 크기 결정
> 3. **융합:** TCP 윈도우, 파이프 용량

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

BDP는 링크의 전송 중 데이터 양입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BDP 계산                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              BDP 공식                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  BDP = 대역폭 × RTT                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  예: 1Gbps × 100ms                                  │   │ │
│   │   │     = 1000 Mbps × 0.1s                              │   │ │
│   │   │     = 100 Mbit = 12.5 MB                            │   │ │
│   │   │                                                      │   │ │
│   │   │  윈도우 크기 ≥ BDP                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: BDP는 파이프를 채우기 위한 데이터 양입니다.

**📢 섹션 요약 비유:** BDP = 파이프 용량!
