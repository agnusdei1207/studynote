# Grafana - 데이터 시각화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 오픈소스 데이터 시각화 플랫폼

> 3. **융합**: Prometheus, InfluxDB와 통합

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Grafana는 다양한 데이터 소스를 시각화하는 오픈소스 플랫폼입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Grafana                                          │
├─────────────────────────────────────────────────────────────────────────┤

│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Data Sources]                                     │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ Prometheus / InfluxDB / Elasticsearch          │   │ │
│   │   │     │ MySQL / PostgreSQL / CloudWatch                │   │ │
│   │   │     │                                                │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           Grafana Server                     │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • 대시보드                                   │    │   │ │

│   │   │       │                                              │   │ │
│   │   │       │ Web UI                                       │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [사용자]                                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │

│   │   │  +------------+----------+----------+               │   │ │
```

> **해설**: Grafana는 다양한 데이터 소스를 통합하여 시각화합니다.

**💡 비유**: Grafana는 인프라의 계기판입니다. 모든 데이터를 시각화합니다.

**📢 섹션 요약 비유:** Grafana = 시각화!

```
+------------+----------+----------+
| 소스       │ 유형     │ 특징     │

| InfluxDB   │ 시계열   │ IoT      │

| MySQL      │ 관계형   │ 비즈니스 │
+------------+----------+----------+
```
