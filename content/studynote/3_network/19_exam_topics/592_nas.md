# NAS (Network Attached Storage) - 네트워크 결합 스토리지

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 파일 공유 스토리지
> 2. **가치**: 파일 수준 공유
> 3. **융합:** NFS, SMB/CIFS, Storage

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NAS는 네트워크로 연결된 파일 서버입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NAS                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NAS 구성                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client 1         Client 2         Client 3          │   │ │
│   │   │  (NFS/SMB)        (NFS/SMB)        (NFS/SMB)         │   │ │
│   │   │      │                │                │              │   │ │
│   │   │      └────────────────┼────────────────┘              │   │ │
│   │   │                       │                                │   │ │
│   │   │              ┌────────┴────────┐                      │   │ │
│   │   │              │       NAS       │                      │   │ │
│   │   │              │  ┌───────────┐  │                      │   │ │
│   │   │              │  │ File Sys  │  │                      │   │ │
│   │   │              │  ├───────────┤  │                      │   │ │
│   │   │              │  │  RAID     │  │                      │   │ │
│   │   │              │  └───────────┘  │                      │   │ │
│   │   │              └─────────────────┘                      │   │ │
│   │   │                                                      │   │ │
│   │   │  프로토콜:                                           │   │ │
│   │   │  - NFS (Unix/Linux)                                  │   │ │
│   │   │  - SMB/CIFS (Windows)                                │   │ │
│   │   │  - AFP (macOS)                                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NAS는 파일 수준에서 데이터에 접근합니다.

**📢 섹션 요약 비유:** NAS = 공용 폴더!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NFS** | Network File System |
| **SMB** | Server Message Block |
| **CIFS** | Common Internet File System |
| **File Level** | 파일 단위 접근 |

**핵심 알고리즸:** 파일 요청 → 전송 → 저장

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**NAS vs SAN**

| 항목 | NAS | SAN |
|:---|:---|:---|
| **접근** | 파일 | 블록 |
| **프로토콜** | NFS/SMB | FC/iSCSI |
| **성능** | 중간 | 높음 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 2:** 홈 디렉토리 → NAS
**시나리오 3:** DB → SAN
