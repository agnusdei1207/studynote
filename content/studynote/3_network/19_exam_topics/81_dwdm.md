# 고밀도 파장 분할 다중화 (DWDM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 좁은 파장 간격(0.8nm)의 대용량 WDM
> 2. **가치**: Tbps 전송, 장거리 백본, EDFA 증폭
> 3. **융합**: 해저 케이블, 국가망, 5G 백홀과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DWDM(Dense Wavelength Division Multiplexing)은 0.8nm(100GHz) 이하의 좁은 간격으로 파장을 분할하여 대용량 장거리 전송을 구현하는 WDM 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DWDM (고밀도 파장 분할 다중화)                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DWDM 시스템 구조                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  파장 할당 (C-band, 1525-1565nm):                     │   │ │
│   │   │                                                      │   │ │
│   │   │  파장 (nm)                                             │   │ │
│   │   │  1565 ─┬──────────────────────────────────────────┐  │   │ │
│   │   │        │ λ40                                     │  │   │ │
│   │   │        │ λ39                                     │  │   │ │
│   │   │        │ ...   ← 0.8nm (100GHz) 간격             │  │   │ │
│   │   │        │ λ2                                      │  │   │ │
│   │   │  1525 ─┤ λ1                                      │  │   │ │
│   │   │        └──────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  시스템 구성:                                           │   │ │
│   │   │  λ1..λ40 ─→ [OTU] ─→ [MUX] ─→ [EDFA] ─→ 광섬유      │   │ │
│   │   │                                        ↓              │   │ │
│   │   │                                     [EDFA] ─→ 광섬유   │   │ │
│   │   │                                        ↓              │   │ │
│   │   │                           [EDFA] ─→ [DEMUX] ─→ 수신   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              EDFA (Erbium-Doped Fiber Amplifier)              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  구조:                                                  │   │ │
│   │   │  입력 광 ─→ [WDM 결합] ─→ [어빙현 섬유] ─→ 출력 광   │   │ │
│   │   │               ↑                                        │   │ │
│   │   │           펌프 레이저 (980nm 또는 1480nm)              │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • C-band (1530-1565nm)에서 30dB 이득                 │   │ │
│   │   │  • 광-광 증폭 (O-E-O 변환 없음)                         │   │ │
│   │   │  • 저잡음, 광대역                                      │   │ │
│   │   │  • 투명성 (프로토콜 무관)                               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: DWDM + EDFA 조합으로 수천 km 무재생 전송이 가능합니다.

**💡 비유**: DWDM은 좁은 차선의 고속도로입니다.

**📢 섹션 요약 비유**: DWDM + EDFA = 장거리 대용량!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **OTU** | 파장 변환 | 클라이언트→λ | 변환기 |
| **MUX/DEMUX** | 파장 결합/분리 | AWG | 프리즘 |
| **EDFA** | 광 증폭 | 어빙현 | 증폭기 |
| **OADM** | 파장 add/drop | 필터 | 교차로 |

**비교표**

```
+------------+----------+----------+
| 항목       | CWDM     | DWDM     |
+------------+----------+----------+
| 파장 수    | 18       | 40-160   |
| 간격       | 20nm     | 0.8nm    |
| 레이저     | 비냉각   | 냉각     |
| 증폭기     | 없음     | EDFA     |
| 거리       | ~80km    | 수천km   |
| 용량       | ~180G    | ~Tbps    |
+------------+----------+----------+
```

**핵심 알고리즸**: 냉각 레이저 → OTU → MUX → EDFA → 전송 → DEMUX

**📢 섹션 요약 비유**: EDFA는 광 증폭의 핵심!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: DWDM 용량**

| 채널 수 | 채널 속도 | 총 용량 |
|:---|:---:|:---:|
| 40 | 10Gbps | 400Gbps |
| 80 | 10Gbps | 800Gbps |
| 80 | 100Gbps | 8Tbps |
| 160 | 100Gbps | 16Tbps |

**📢 섹션 요약 비유**: 160λ × 100G = 16Tbps!
