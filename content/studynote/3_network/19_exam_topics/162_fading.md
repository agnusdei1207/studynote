# 페이딩 (Fading) - 플랫/주파수 선택적

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 수신 신호 강도의 시간 변화
> 2. **가치**: 통신 품질에 직접 영향
> 3. **융합**: 다이버시티, 등화와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

페이딩(Fading)은 무선 신호가 전파 과정에서 강도가 변하는 현상입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    페이딩 (Fading)                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              페이딩 분류                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  페이딩                                              │   │ │
│   │   │     │                                                │   │ │
│   │   │     ├── 규모별                                       │   │ │
│   │   │     │   ├── 대규모 (Large Scale): 경로 손실          │   │ │
│   │   │     │   └── 소규모 (Small Scale): 다중경로           │   │ │
│   │   │     │                                                │   │ │
│   │   │     └── 주파수 특성별                                 │   │ │
│   │   │         ├── 플랫 페이딩 (Flat Fading)                │   │ │
│   │   │         │   • 전 대역폭 동일 감쇠                    │   │ │
│   │   │         │   • 대역폭 < coherence bandwidth           │   │ │
│   │   │         │                                            │   │ │
│   │   │         └── 주파수 선택적 페이딩                      │   │ │
│   │   │             (Frequency Selective Fading)             │   │ │
│   │   │             • 주파수별 다른 감쇠                      │   │ │
│   │   │             • 대역폭 > coherence bandwidth           │   │ │
│   │   │             • ISI 발생                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              페이딩 원인                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  • 다중경로 (Multipath): 반사, 회절, 산란             │   │ │
│   │   │  • 이동 (Mobility): 도플러 효과                       │   │ │
│   │   │  • 장애물: 음영 영역                                   │   │ │
│   │   │  • 기상: 강우, 대기 감쇠                               │   │ │
│   │   │  • 간섭: 동일 주파수 사용                              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 페이딩은 무선 통신 품질에 가장 큰 영향을 미치는 현상입니다.

**💡 비유**: 페이딩은 라디오에서 신호가 때때로 끊기는 현상입니다.

**📢 섹션 요약 비유:** 페이딩 = 신호 변동!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 종류 | 특성 | 조건 | 대책 |
|:---|:---|:---|:---|
| **플랫** | 전 대역 동일 감쇠 | BW < coherence | AGC |
| **선택적** | 주파수별 다름 | BW > coherence | 등화기 |

**비교표**

```
+------------+----------+----------+
| 항목       | 플랫     | 선택적   │
+------------+----------+----------+
| 대역폭     │ 좁음     │ 넓음     │
| ISI        │ 없음     │ 있음     │
| 등화       │ 불필요   │ 필요     │
│ OFDM       │ 불필요   │ 유리     │
+------------+----------+----------+
```

**핵심 알고리즸:** 레일리 분포, 라이시안 분포

**📢 섹션 요약 비유:** OFDM = 선택적 페이딩 해결!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 페이딩 대책**

| 기법 | 원리 | 효과 |
|:---|:---|:---|
| **다이버시티** | 다중 경로 | 높음 |
| **등화** | ISI 제거 | 중간 |
| **RAKE** | 다중경로 이용 | 높음 |
| **OFDM** | 직렬→병렬 | 높음 |

**📢 섹션 요약 비유:** 다이버시티 = 페이딩 해결사!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 협대역** → 플랫 페이딩, AGC
**시나리오 2: 광대역** → 선택적 페이딩, OFDM
**시나리오 3: 이동 환경** → 빠른 페이딩, 다이버시티

**도입 체크리스트:** 대역폭, 이동 속도, 환경

**📢 섹션 요약 비유:** 대역폭이 넓으면 선택적!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 통신 신뢰성 향상

**미래 전망:** Massive MIMO로 보완

**참고 표준:** 3GPP, IEEE 802.11

**📢 섹션 요약 비유:** 페이딩은 무선의 숙제!
