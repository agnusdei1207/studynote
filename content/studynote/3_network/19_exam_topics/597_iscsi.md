# iSCSI (Internet SCSI) - 인터넷 SCSI

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IP 네트워크 기반 블록 스토리지
> 2. **가치**: FC SAN 대안, 비용 절감
> 3. **융합:** SAN, SCSI, TCP/IP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

iSCSI는 IP로 SCSI 명령을 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    iSCSI                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              iSCSI 구성                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  iSCSI Initiator           IP Network     iSCSI Target│   │ │
│   │   │  ┌───────────┐                           ┌──────────┐│   │ │
│   │   │  │   Server  │ ──── TCP/IP (3260) ────→ │ Storage  ││   │ │
│   │   │  │  iSCSI SW │                           │  LUN     ││   │ │
│   │   │  │   HBA     │                           │ Target   ││   │ │
│   │   │  └───────────┘                           └──────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  캡슐화:                                              │   │ │
│   │   │  SCSI Command → iSCSI PDU → TCP/IP                   │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                               │   │ │
│   │   │  - 기존 IP 인프라 사용                               │   │ │
│   │   │  - FC SAN보다 저렴                                   │   │ │
│   │   │  - 블록 수준 접근                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: iSCSI는 TCP/IP로 SCSI 명령을 전달합니다.

**📢 섹션 요약 비유:** iSCSI = IP 기반 SCSI!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Initiator** | 클라이언트 |
| **Target** | 서버/스토리지 |
| **LUN** | Logical Unit Number |
| **IQN** | iSCSI Qualified Name |
| **Portal** | IP:Port |

**핵심 알고리즸:** SCSI → iSCSI PDU → TCP/IP

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**iSCSI vs FC**

| 항목 | iSCSI | FC |
|:---|:---|:---|
| **네트워크** | IP | FC |
| **비용** | 낮음 | 높음 |
| **성능** | 중간 | 높음 |
| **거리** | 원거리 | 근거리 |
