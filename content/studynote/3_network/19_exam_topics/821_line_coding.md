# Line Coding - 라인 코딩

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디지털 신호 변환
> 2. **가치**: 전송 효율 및 동기화
> 3. **융합:** NRZ, Manchester, 8b/10b

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Line Coding은 디지털 데이터를 전송 신호로 변환합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Line Coding                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              라인 코딩 방식                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  NRZ (Non-Return to Zero):                           │   │ │
│   │   │  1: ────  0: ▁▁▁▁                                   │   │ │
│   │   │  단순, DC 성분                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  Manchester:                                          │   │ │
│   │   │  1: ▔▁   0: ▁▔                                      │   │ │
│   │   │  클럭 복구 용이, 100% 오버헤드                         │   │ │
│   │   │                                                      │   │ │
│   │   │  Differential Manchester:                             │   │ │
│   │   │  전이로 0/1 구분                                      │   │ │
│   │   │  토큰 링                                              │   │ │
│   │   │                                                      │   │ │
│   │   │  AMI (Alternate Mark Inversion):                     │   │ │
│   │   │  1: +V, -V 교대  0: 0V                               │   │ │
│   │   │  DC 성분 제거                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  8b/10b:                                              │   │ │
│   │   │  8비트 → 10비트 매핑                                  │   │ │
│   │   │  DC 밸런스, 클럭 복구                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 라인 코딩은 물리 계층 전송의 핵심입니다.

**📢 섹션 요약 비유:** Line Coding = 디지털 문자체!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NRZ** | 비복귀 제로 |
| **RZ** | 복귀 제로 |
| **Manchester** | 맨체스터 |
| **AMI** | 교대 마크 반전 |
| **Scrambling** | 스크램블링 |

**핵심 알고리즸:** 비트 → 부호화 → 신호 → 복호화 → 비트

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**라인 코딩 비교**

| 방식 | DC 밸런스 | 동기화 | 오버헤드 |
|:---|:---|:---|:---|
| **NRZ** | 없음 | 어려움 | 없음 |
| **Manchester** | 있음 | 쉬움 | 100% |
| **AMI** | 있음 | 중간 | 없음 |
| **8b/10b** | 있음 | 쉬움 | 25% |
