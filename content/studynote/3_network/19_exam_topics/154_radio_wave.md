# 전파 분류: 장파/중파/단파/VHF/UHF

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 주파수 대역별 전파 특성
> 2. **가치**: 용도별 최적 대역 선택
> 3. **융합**: 방송, 통신, 레이더와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

전파는 주파수에 따라 장파(LF), 중파(MF), 단파(HF), 초단파(VHF), 극초단파(UHF) 등으로 분류됩니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    전파 분류 (주파수 대역)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              주파수 대역 분류                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  대역      │ 주파수        │ 파장      │ 용도         │   │ │
│   │   │  ─────────┼──────────────┼──────────┼────────────│   │ │
│   │   │  VLF      │ 3-30kHz      │ 100-10km │ 해군/지진    │   │ │
│   │   │  LF (장파)│ 30-300kHz    │ 10-1km   │ 항해/시보    │   │ │
│   │   │  MF (중파)│ 300k-3MHz    │ 1km-100m │ AM 라디오    │   │ │
│   │   │  HF (단파)│ 3-30MHz      │ 100-10m  │ 국제방송     │   │ │
│   │   │  VHF      │ 30-300MHz    │ 10-1m    │ FM/TV        │   │ │
│   │   │  UHF      │ 300M-3GHz    │ 1m-10cm  │ TV/이동통신  │   │ │
│   │   │  SHF      │ 3-30GHz      │ 10-1cm   │ 위성/레이더  │   │ │
│   │   │  EHF      │ 30-300GHz    │ 1cm-1mm  │ 밀리미터파   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전파 특성                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  저주파 (장파/중파):                                   │   │ │
│   │   │  • 지상파 전파 (지표면 따라)                            │   │ │
│   │   │  • 장거리 통신 가능                                     │   │ │
│   │   │  • 도체에 의한 감쇠 큼                                  │   │ │
│   │   │  • 안테나 크기 큼                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  중주파 (단파):                                          │   │ │
│   │   │  • 전리층 반사 (천파)                                   │   │ │
│   │   │  • 전 세계 통신 가능                                    │   │ │
│   │   │  • 계절/시간대 변동                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  고주파 (VHF/UHF/SHF):                                   │   │ │
│   │   │  • 직진성 강함 (LOS)                                    │   │ │
│   │   │  • 대역폭 넓음                                          │   │ │
│   │   │  • 안테나 작음                                          │   │ │
│   │   │  • 빌딩/지형 영향 큼                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 주파수가 높을수록 직진성이 강하고 대역폭이 넓지만 거리는 짧아집니다.

**💡 비유**: 저주파는 낮은 목소리(멀리 들림), 고주파는 높은 목소리(가까이만 들림)입니다.

**📢 섹션 요약 비유:** VHF = FM라디오, UHF = TV/이동통신!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 대역 | 전파 경로 | 특징 | 용도 |
|:---|:---|:---|:---|
| **LF** | 지상파 | 장거리 | 항해 |
| **MF** | 지상파+천파 | 야간 증가 | AM |
| **HF** | 천파 | 전 세계 | 국제방송 |
| **VHF** | 직진 | LOS | FM/TV |
| **UHF** | 직진 | LOS+관통 | TV/이동 |

**비교표**

```
+------------+----------+----------+
| 항목       | 저주파   | 고주파   |
+------------+----------+----------+
| 거리       │ 긺       │ 짧음     |
| 직진성     │ 약함     │ 강함     |
| 대역폭     │ 좁음     │ 넓음     |
| 안테나     │ 큼       │ 작음     |
+------------+----------+----------+
```

**핵심 알고리즸:** 전리층 반사, 지상파 전파

**📢 섹션 요약 비유:** 단파 = 전리층 반사로 세계 통신!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 방송 대역**

| 방송 | 대역 | 주파수 |
|:---|:---|:---:|
| **AM** | MF | 530-1710kHz |
| **FM** | VHF | 88-108MHz |
| **TV** | VHF/UHF | 54-806MHz |

**📢 섹션 요약 비유:** FM = VHF 대역!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 장거리 통신** → 단파(HF)
**시나리오 2: 지역 방송** → VHF/UHF
**시나리오 3: 이동통신** → UHF/SHF

**도입 체크리스트:** 거리, 대역폭, 환경

**📢 섹션 요약 비유:** 용도에 맞는 대역 선택!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)
