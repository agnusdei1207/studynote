# Proxy Server - 프록시 서버

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 중계 서버
> 2. **가치**: 익명성, 캐싱, 필터링
> 3. **융합:** Forward Proxy, Reverse Proxy, Caching

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Proxy Server는 클라이언트와 서버 사이에서 중계합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Proxy Server                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Proxy 유형                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Forward Proxy:                                      │   │ │
│   │   │  Client → Proxy → Internet → Server                  │   │ │
│   │   │  용도: 익명성, 우회, 캐싱                             │   │ │
│   │   │                                                      │   │ │
│   │   │  Reverse Proxy:                                      │   │ │
│   │   │  Client → Internet → Proxy → Server                  │   │ │
│   │   │  용도: 로드밸런싱, 보안, 캐싱                         │   │ │
│   │   │                                                      │   │ │
│   │   │  Transparent Proxy:                                  │   │ │
│   │   │  클라이언트 인지 없이 동작                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Proxy는 요청을 대신 전달하고 응답을 전달합니다.

**📢 섹션 요약 비유:** Proxy = 대리인!

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Forward** | 클라이언트 대리 |
| **Reverse** | 서버 대리 |
