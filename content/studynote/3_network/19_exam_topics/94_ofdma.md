# OFDMA (직교 주파수 분할 다중 접속)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: OFDM의 부반송파를 사용자에게 동적 할당
> 2. **가치**: 주파수 효율, 다중 경로 내성, 4G/5G 기반
> 3. **융합**: LTE, WiMAX, 5G NR과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OFDMA(Orthogonal Frequency Division Multiple Access)는 OFDM의 부반송파를 여러 사용자에게 동적으로 할당하는 다중 접속 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OFDMA (직교 주파수 분할 다중 접속)               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OFDMA vs OFDM                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  OFDM:                                                  │   │ │
│   │   │  한 사용자가 모든 부반송파 사용                         │   │ │
│   │   │                                                      │   │ │
│   │   │  시간 →                                                 │   │ │
│   │   │  ┌──┬──┬──┬──┬──┬──┬──┬──┐                           │   │ │
│   │   │  │U1│U1│U1│U1│U1│U1│U1│U1│                           │   │ │
│   │   │  └──┴──┴──┴──┴──┴──┴──┴──┘                           │   │ │
│   │   │   ↑ 모든 부반송파 = 한 사용자                          │   │ │
│   │   │                                                      │   │ │
│   │   │  OFDMA:                                                 │   │ │
│   │   │  여러 사용자가 부반송파 분할 사용                        │   │ │
│   │   │                                                      │   │ │
│   │   │  시간 →                                                 │   │ │
│   │   │  ┌──┬──┬──┬──┬──┬──┬──┬──┐                           │   │ │
│   │   │  │U1│U2│U1│U3│U2│U1│U3│U2│                           │   │ │
│   │   │  └──┴──┴──┴──┴──┴──┴──┴──┘                           │   │ │
│   │   │   ↑ 각 부반송파 = 다른 사용자 가능                      │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                 │   │ │
│   │   │  • 다중 사용자 다이버시티                               │   │ │
│   │   │  • 주파수 선택적 스케줄링                               │   │ │
│   │   │  • 효율적인 버스트 트래픽                               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OFDMA 자원 할당                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  자원 격자 (Time-Frequency Grid):                     │   │ │
│   │   │                                                      │   │ │
│   │   │         주파수 (Subcarriers)                          │   │ │
│   │   │         ↑                                            │   │ │
│   │   │    SC ─┼──┬──┬──┬──┬──┬──┬──┬──┐                    │   │ │
│   │   │         │U1│U2│U3│U1│U2│U3│U1│U2│                    │   │ │
│   │   │    SC ─┼──┼──┼──┼──┼──┼──┼──┼──┤                    │   │ │
│   │   │         │U4│U5│U4│U5│U4│U5│U4│U5│                    │   │ │
│   │   │    SC ─┼──┼──┼──┼──┼──┼──┼──┼──┤                    │   │ │
│   │   │         │U1│U2│U3│U1│U2│U3│U1│U2│                    │   │ │
│   │   │         └──┴──┴──┴──┴──┴──┴──┴──┴───→ 시간 (Symbols)│   │ │
│   │   │           ↑                              ↑            │   │ │
│   │   │         RB (Resource Block)                           │   │ │
│   │   │                                                      │   │ │
│   │   │  LTE: RB = 12 SC × 7 symbols = 1 TTI (1ms)           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: OFDMA는 LTE 다운링크와 5G의 핵심 다중 접속 방식입니다.

**💡 비유**: OFDMA는 여러 사람이 피자를 조각으로 나눠 먹는 것입니다.

**📢 섹션 요약 비유**: OFDMA = 다중 사용자 OFDM!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **RB** | 자원 블록 | 12 SC × 7 sym | 조각 |
| **TTI** | 전송 시간 | 1ms | 주기 |
| **스케줄러** | 자원 할당 | PF/Max C/I | 배분자 |

**비교표**

```
+------------+----------+----------+
| 항목       | LTE DL   | LTE UL   |
+------------+----------+----------+
| 방식       | OFDMA    | SC-FDMA  |
| PAPR       | 높음     | 낮음     |
| 전력 효율   | 낮음     | 높음     |
+------------+----------+----------+
```

**핵심 알고리즸**: 채널 상태 → 스케줄러 → RB 할당 → 전송

**📢 섹션 요약 비유**: LTE UL은 SC-FDMA!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 응용 분야**

| 표준 | 다중 접속 | 특징 |
|:---|:---|:---|
| **LTE** | OFDMA (DL) | 4G |
| **WiMAX** | OFDMA | 4G 대안 |
| **5G NR** | OFDMA | 확장 가능 |

**📢 섹션 요약 비유**: 5G도 OFDMA 기반!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: LTE DL** → OFDMA
**시나리오 2: LTE UL** → SC-FDMA
**시나리오 3: 5G** → OFDMA + Scalable numerology

**도입 체크리스트**: 스케줄러, 채널 피드백, PAPR

**📢 섹션 요약 비유**: OFDMA는 4G/5G의 표준!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 높은 스펙트럼 효율, 다중 사용자 지원
