# 케이블 모뎀 (Cable Modem) / DOCSIS

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CATV망으로 고속 데이터 전송
> 2. **가치**: 기존 인프라 활용, 수백 Mbps
> 3. **융합**: HFC, DOCSIS 3.1/4.0과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

케이블 모뎀은 CATV(유선방송) 네트워크를 통해 고속 인터넷 서비스를 제공하는 장치이며, DOCSIS는 그 표준입니다.


> **해설**: 케이블 모뎀은 기존 CATV 인프라로 고속 인터넷을 제공합니다.

**💡 비유**: 케이블 모뎀은 TV 케이블을 인터넷 선으로 바꾸는 마법사입니다.

**📢 섹션 요약 비유:** DOCSIS 3.1 = 10Gbps!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **CMTS** | 헤드엔드 | 관리/변조 | 교환국 |
| **CM** | 가입자 모뎀 | 튜너/복조 | 수신기 |
| **채널 본딩** | 대역폭 확장 | 다중 채널 | 차선 확장 |

**비교표**

```
+------------+----------+----------+
| 항목       | DOCSIS 3.0| DOCSIS 3.1|
+------------+----------+----------+
| 다운       | 1.2Gbps  | 10Gbps   |
| 업         | 200Mbps  | 2Gbps    |
| 변조       | 256QAM   | 4096QAM  |
| 기술       | SC-QAM   | OFDM     |
+------------+----------+----------+
```

**핵심 알고리즸**: 채널 본딩, OFDM/OFDMA

**📢 섹션 요약 비유:** OFDM = DOCSIS 3.1!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
