# WDM - 파장 분할 다중화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 여러 파장 동시 전송
> 2. **가치**: 대역폭 증대
> 3. **융합:** CWDM, DWDM, 광케이블

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

WDM (Wavelength Division Multiplexing)은 여러 파장을 동시에 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    WDM                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              WDM 구조                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  WDM 동작:                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  λ1 (1530nm) ─┐                                 ││   │ │
│   │   │  │  λ2 (1540nm) ─┼── Mux ──→ 광섬유 ──→ Demux ──┤  ││   │ │
│   │   │  │  λ3 (1550nm) ─┤                            λ1  │  ││   │ │
│   │   │  │  λ4 (1560nm) ─┘                            λ2  │  ││   │ │
│   │   │  │                                            λ3  │  ││   │ │
│   │   │  │                                            λ4  │  ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  WDM 유형:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ CWDM (Coarse WDM):                                ││   │ │
│   │   │  │   - 8-18 파장                                       ││   │ │
│   │   │  │   - 파장 간격: 20nm                                  ││   │ │
│   │   │  │   - 용도: 도시권, 기업                               ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │ DWDM (Dense WDM):                                 ││   │ │
│   │   │  │   - 40-160+ 파장                                     ││   │ │
│   │   │  │   - 파장 간격: 0.8nm (100GHz)                       ││   │ │
│   │   │  │   - 용도: 장거리, 백본                               ││   │ │
│   │   │  │                                                 ││   │ │

│                                                                 │
```

> **해설**: WDM은 광케이블 용량을 획기적으로 늘립니다.

**📢 섹션 요약 비유:** WDM = 여러 색의 빛!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Mux** | 멀티플렉서 |
| **Demux** | 디멀티플렉서 |
| **Wavelength** | 파장 |
| **Channel** | 채널 |
| **Filter** | 필터 |

**핵심 알고리듬:** 파장 분리 → 결합 → 전송 → 분리

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**CWDM vs DWDM**

| 항목 | CWDM | DWDM |
|:---|:---|:---|
| **파장 수** | 8-18 | 40-160+ |
| **간격** | 20nm | 0.8nm |
| **비용** | 낮음 | 높음 |
| **거리** | 단거리 | 장거리 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** ISP 백본 → DWDM
**시나리오 2:** 도시권 → CWDM
**시나리오 3:** 데이터센터 → DWDM
