# WAF (Web Application Firewall) - 웹 애플리케이션 방화벽

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: L7 웹 트래픽 보안
> 2. **가치**: OWASP Top 10 방어
> 3. **융합:** XSS, SQLi, CSRF 방어

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

WAF는 웹 애플리케이션 공격을 방어합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    WAF                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              WAF 동작                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                                              │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ ── HTTP Request ────→                         │   │ │
│   │   │     │                       │                        │   │ │
│   │   │     │                    ┌──┴──┐                    │   │ │
│   │   │     │                    │ WAF │ ← 탐지/차단        │   │ │
│   │   │     │                    └──┬──┘                    │   │ │
│   │   │     │                       │                        │   │ │
│   │   │     │                Web Server                      │   │ │
│   │   │                                                      │   │ │
│   │   │  방어 대상:                                          │   │ │
│   │   │  - SQL Injection, XSS, CSRF                         │   │ │
│   │   │  - Path Traversal, LFI/RFI                          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: WAF는 HTTP 페이로드를 검사하여 공격을 탐지합니다.

**📢 섹션 요약 비유:** WAF = 웹 경호원!

|:---|:---|
