# VPN (Virtual Private Network) - 가상 사설망

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 공중망 위 사설망 구축
> 2. **가치**: 보안, 원격 접속
> 3. **융합:** IPsec, TLS, Remote Access

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VPN은 공중망에서 사설망처럼 안전하게 통신합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VPN                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VPN 구성                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Remote                 Internet               Corp │   │ │
│   │   │  User                                           Network│   │ │
│   │   │  ┌────┐              ┌──────┐              ┌────┐   │   │ │
│   │   │  │ PC │──Encrypted──→│  VPN │──Encrypted──→│VPN │   │   │ │
│   │   │  │    │    Tunnel    │ GW   │    Tunnel    │ GW │   │   │ │
│   │   │  └────┘              └──────┘              └────┘   │   │ │
│   │   │       └────────────────────────────────────┘        │   │ │
│   │   │                     VPN Tunnel                       │   │ │
│   │   │                                                      │   │ │
│   │   │  프로토콜:                                           │   │ │
│   │   │  - IPsec (L3)                                       │   │ │
│   │   │  - SSL/TLS (L7)                                     │   │ │
│   │   │  - WireGuard                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: VPN은 터널링과 암호화로 보안 통신을 제공합니다.

**📢 섹션 요약 비유:** VPN = 보안 터널!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Site-to-Site** | 네트워크 간 연결 |
| **Remote Access** | 개인 연결 |
| **Tunneling** | 캡슐화 |
| **Encryption** | 암호화 |
| **Authentication** | 인증 |

**핵심 알고리즸:** 터널 생성 → 암호화 → 전송

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**VPN 프로토콜**

| 프로토콜 | 계층 | 특징 |
|:---|:---|:---|
| **IPsec** | L3 | 표준 |
| **OpenVPN** | L7 | SSL/TLS |
| **WireGuard** | L3 | 경량 |
| **L2TP** | L2 | IPsec과 결합 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 재택근무 → Remote Access VPN
**시나리오 2:** 지사 연결 → Site-to-Site VPN
**시나리오 3:** 클라우드 → VPN Gateway
