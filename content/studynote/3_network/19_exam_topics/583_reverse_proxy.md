# Reverse Proxy - 리버스 프록시

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서버 앞단 요청 대행
> 2. **가치**: 보안, 캐싱, 로드 밸런싱
> 3. **융합:** Load Balancer, Cache, SSL Termination

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Reverse Proxy는 클라이언트 요청을 서버에 대신 전달합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Reverse Proxy                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Reverse Proxy 동작                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                                              │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ Request                                        │   │ │
│   │   │     ↓                                                │   │ │
│   │   │  ┌─────────────────┐                                │   │ │
│   │   │  │  Reverse Proxy  │ ← SSL, Cache, Security         │   │ │
│   │   │  └────────┬────────┘                                │   │ │
│   │   │           │                                          │   │ │
│   │   │           ↓                                          │   │ │
│   │   │  ┌─────────────────┐                                │   │ │
│   │   │  │   Web Server    │                                │   │ │
│   │   │  └─────────────────┘                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Reverse Proxy는 서버를 감추고 추가 기능을 제공합니다.

**📢 섹션 요약 비유:** Reverse Proxy = 비서!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Forward vs Reverse Proxy**

| 항목 | Forward | Reverse |
|:---|:---|:---|
| **대상** | 클라이언트 | 서버 |
| **용도** | 익명, 우회 | 보안, 분산 |
