# 단일모드 / 다중모드 광섬유 (SMF / MMF)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SMF는 단일 경로, MMF는 다중 경로
> 2. **가치**: SMF는 장거리, MMF는 단거리
> 3. **융합**: 백본(SMF) + LAN(MMF) 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

단일모드 광섬유(SMF, Single-mode Fiber)는 하나의 빛 경로만 전파하고, 다중모드 광섬유(MMF, Multi-mode Fiber)는 여러 경로가 전파됩니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    단일모드 / 다중모드 광섬유                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              단일모드 (SMF, Single-mode Fiber)                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  구조:                                                    │   │ │
│   │   │  코어: 약 9µm (매우 작음)                               │   │ │
│   │   │  클래딩: 125µm                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  빛의 경로:                                               │   │ │
│   │   │    ┌─────────────────────────────────┐               │   │ │
│   │   │    │ ─────────────────────────────   │               │   │ │
│   │   │    │      단일 모드만 전파            │               │   │ │
│   │   │    └─────────────────────────────────┘               │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                   │   │ │
│   │   │  • 모드 분산 없음                                        │   │ │
│   │   │  • 무제한 대역폭                                         │   │ │
│   │   │  • 장거리 전송 (수십~수백 km)                            │   │ │
│   │   │  • LD 광원 필요                                         │   │ │
│   │   │  • 높은 비용                                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              다중모드 (MMF, Multi-mode Fiber)                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  구조:                                                    │   │ │
│   │   │  코어: 50µm 또는 62.5µm                                 │   │ │
│   │   │  클래딩: 125µm                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  빛의 경로:                                               │   │ │
│   │   │    ┌─────────────────────────────────┐               │   │ │
│   │   │    │   ╱─────╲    ╱─────╲            │               │   │ │
│   │   │    │  ╱       ╲  ╱       ╲           │               │   │ │
│   │   │    │ ───────────────────────────     │               │   │ │
│   │   │    │        ╲  ╱       ╲  ╱          │               │   │ │
│   │   │    │         ╲╱         ╲╱           │               │   │ │
│   │   │    └─────────────────────────────────┘               │   │ │
│   │   │         여러 모드 전파                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                   │   │ │
│   │   │  • 모드 분산 있음                                        │   │ │
│   │   │  • 제한된 대역폭                                         │   │ │
│   │   │  • 단거리 전송 (수백m~수km)                              │   │ │
│   │   │  • LED/VCSEL 광원 사용                                  │   │ │
│   │   │  • 낮은 비용                                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: SMF는 장거리 백본에, MMF는 단거리 LAN에 주로 사용됩니다.

**💡 비유**: SMF는 1차선 고속도로, MMF는 다차선 도로입니다.

**📢 섹션 요약 비유**: SMF = 장거리, MMF = 단거리!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | SMF | MMF |
|:---|:---|:---|:---|
| **코어** | 빛 전파 | 9µm | 50/62.5µm |
| **광원** | 빛 생성 | LD | LED/VCSEL |
| **거리** | 전송 범위 | 100km+ | 550m |

**비교표**

```
+------------+----------+----------+
| 항목       | SMF      | MMF      |
+------------+----------+----------+
| 코어       | 9µm      | 50µm     |
| 모드       | 1개      | 다수     |
| 모드 분산  | 없음     | 있음     |
| 거리       | 100km+   | 550m     |
| 광원       | LD       | LED      |
| 비용       | 높음     | 낮음     |
+------------+----------+----------+
```

**핵심 알고리즸**: V-수 = 2πa/λ × NA, V < 2.405 = SMF

**📢 섹션 요약 비유**: SMF = 9µm, MMF = 50µm!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 응용 분야**

| 분야 | 추천 | 이유 |
|:---|:---|:---|
| **백본** | SMF | 장거리 |
| **FTTH** | SMF | 장거리 |
| **데이터센터** | MMF | 단거리/저비용 |
| **LAN** | MMF | 단거리 |

**📢 섹션 요약 비유**: 2km 이상 = SMF!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 장거리 백본** → SMF
**시나리오 2: 데이터센터 내부** → MMF (OM4)
**시나리오 3: FTTH** → SMF

**도입 체크리스트**: 거리, 속도, 예산, 광원

**📢 섹션 요약 비유**: 거리가 결정!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 적절한 광섬유 선택으로 비용 최적화

**미래 전망**: SMF 비용 하락으로 MMF 대체 확대

**참고 표준**: ITU-T G.652 (SMF), G.651 (MMF)

**📢 섹션 요약 비유**: SMF/MMF 선택은 거리가 결정!
