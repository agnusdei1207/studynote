# 상호변조 잡음 (Intermodulation Noise)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 비선형 시스템에서 두 신호의 혼합으로 생성된 간섭
> 2. **가치**: 주파수 할당 제한, 통신 품질 저하
> 3. **융합**: IM2, IM3, 선형성, 증폭기와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

상호변조 잡음(Intermodulation Noise)은 비선형 장치에서 두 개 이상의 신호가 혼합되어 새로운 주파수 성분이 생성되는 현상입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    상호변조 잡음 (Intermodulation Noise)             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              상호변조 발생 원리                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  입력 신호: f1, f2                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  비선형 장치 (증폭기, 믹서)                             │   │ │
│   │   │        ┌──────────────┐                              │   │ │
│   │   │  f1 ─→│              │                              │   │ │
│   │   │        │  비선형       │─→ 출력                       │   │ │
│   │   │  f2 ─→│              │                              │   │ │
│   │   │        └──────────────┘                              │   │ │
│   │   │                                                      │   │ │
│   │   │  출력 성분:                                           │   │ │
│   │   │  • 기본파: f1, f2                                    │   │ │
│   │   │  • 고조파: 2f1, 2f3, 3f1, 3f2...                     │   │ │
│   │   │  • 상호변조: f1±f2, 2f1±f2, f1±2f2...               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              상호변조 차수                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  2차 상호변조 (IM2):                                   │   │ │
│   │   │  • f1 + f2 (합)                                       │   │ │
│   │   │  • f1 - f2 (차)                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  3차 상호변조 (IM3) - 가장 문제:                        │   │ │
│   │   │  • 2f1 - f2                                          │   │ │
│   │   │  • 2f2 - f1                                          │   │ │
│   │   │  • 원 신호 대역 내 생성!                               │   │ │
│   │   │                                                      │   │ │
│   │   │  3차 가로채기점 (TOI/IP3): IM3가 기본파와 같아지는점   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: IM3는 원 신호 주파수 대역 내에 생성되어 필터링으로 제거가 어렵습니다.

**💡 비유**: 상호변조는 두 라디오 방송이 섞여 새로운 주파수가 들리는 것입니다.

**📢 섹션 요약 비유**: 3차 상호변조가 가장 치명적!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **IM2** | 2차 | f1±f2 | 합/차 |
| **IM3** | 3차 | 2f1-f2 | 간섭 |
| **IP3** | 가로채기점 | 품질 지표 | 기준 |
| **선형성** | 비왜곡 | 이상적 | 직선 |

**IM3 스펙트럼**

```
주파수 스펙트럼:

전력   │         ▲              ▲
  ↑    │         │              │
  │    │    ▲    │    ▲    ▲    │    ▲
  │    │    │    │    │    │    │    │
  └────┼────┼────┼────┼────┼────┼────┼──→ f
       2f1-f2  f1       f2   2f2-f1
         ↑                    ↑
       IM3                  IM3
    (대역 내)            (대역 내)
```

**핵심 알고리즘**: 입력 → 비선형 → IM 생성 → 필터/선형화

**📢 섹션 요약 비유**: IP3가 높을수록 좋은 증폭기!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 왜곡 유형**

| 왜곡 | 원인 | 특징 |
|:---|:---|:---|
| **고조파** | 단일 신호 | nf |
| **상호변조** | 다중 신호 | mf1±nf2 |
| **교차 변조** | AM 간섭 | 변조 이동 |

**📢 섹션 요약 비유**: IM3는 필터로 안 지워져요!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 셀룰러** → 주파수 할당 시 IM 회피
**시나리오 2: 위성** → 고선형 증폭기 (SSPA)
**시나리오 3: 케이블** → 디지털 선형화

**도입 체크리스트**: IP3, P1dB, 대역폭

**📢 섹션 요약 비유**: 주파수 간격이 좁으면 IM 심각!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 주파수 효율, 간섭 최소화

**미래 전망**: 디지털 프리디스토션 (DPD)

**참고 표준**: 3GPP, IEEE 802.11

**📢 섹션 요약 비유**: 상호변조는 비선형의 산물!
