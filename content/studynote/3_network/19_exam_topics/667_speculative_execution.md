# Speculative Execution - 추측 실행

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 결과 예측 후 실행
> 2. **가치**: 성능 향상
> 3. **융합:** Branch Prediction, Out-of-order, Rollback

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Speculative Execution은 예측에 기반해 실행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Speculative Execution                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              추측 실행 흐름                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Branch: if (x > 0)                                 │   │ │
│   │   │      │                                               │   │ │
│   │   │      ↓                                               │   │ │
│   │   │  Prediction: Taken                                   │   │ │
│   │   │      │                                               │   │ │
│   │   │      ↓                                               │   │ │
│   │   │  ┌─────────────────────────────────────────┐        │   │ │
│   │   │  │ Speculative Execution (if 블록)         │        │   │ │
│   │   │  │ - 결과은 ROB에 저장                     │        │   │ │
│   │   │  │ - 아직 커밋 안 됨                       │        │   │ │
│   │   │  └─────────────────────────────────────────┘        │   │ │
│   │   │      │                                               │   │ │
│   │   │      ↓                                               │   │ │
│   │   │  ┌────────────┐      ┌────────────┐                 │   │ │
│   │   │  │ Prediction │      │ Prediction │                 │   │ │
│   │   │  │   Correct  │      │    Wrong   │                 │   │ │
│   │   │  │   Commit   │      │   Flush    │                 │   │ │
│   │   │  └────────────┘      └────────────┘                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 추측 실행은 예측 실패 시 롤백합니다.

**📢 섹션 요약 비유:** Speculative Execution = 선주문!
