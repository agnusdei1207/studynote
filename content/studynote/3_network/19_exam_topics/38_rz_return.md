# RZ (Return to Zero)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 각 비트 전송 후 전압이 0으로 복귀
> 2. **가치**: 동기화 용이, 자체 클럭 포함
> 3. **융합**: 맨체스터, 대역폭, 이더넷과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

RZ(Return to Zero)는 각 비트 전송 중간이나 끝에서 전압이 0으로 복귀하는 Line Coding 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    RZ (Return to Zero)                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RZ 부호화 특성                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  규칙:                                                 │   │ │
│   │   │  • 1 = 비트 중간까지 +V 후 0으로 복귀                  │   │ │
│   │   │  • 0 = 비트 중간까지 -V 후 0으로 복귀                  │   │ │
│   │   │                                                      │   │ │
│   │   │  파형:                                                 │   │ │
│   │   │  데이터:    1     0     1     1     0                 │   │ │
│   │   │             ┌─┐   ┌─┐   ┌─┐   ┌─┐   ┌─┐             │   │ │
│   │   │           ──┘ └───┘ └───┘ └───┘ └───┘ └───           │   │ │
│   │   │             +V  0  -V 0  +V  0  +V  0  -V  0          │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                               │   │ │
│   │   │  • 각 비트마다 0 복귀 → 동기화 용이                    │   │ │
│   │   │  • 2배 대역폭 필요                                    │   │ │
│   │   │  • 3레벨 신호 (+V, 0, -V)                              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RZ vs NRZ 비교                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  NRZ (Non-Return to Zero):                            │   │ │
│   │   │  데이터:  1     1     0     0                         │   │ │
│   │   │           ┌──────────┐                                │   │ │
│   │   │         ──┘          └──────────                     │   │ │
│   │   │          (0으로 복귀 안함)                             │   │ │
│   │   │                                                      │   │ │
│   │   │  RZ (Return to Zero):                                 │   │ │
│   │   │  데이터:  1     1     0     0                         │   │ │
│   │   │           ┌─┐   ┌─┐   ┌─┐   ┌─┐                     │   │ │
│   │   │         ──┘ └───┘ └───┘ └───┘ └───                  │   │ │
│   │   │          (각 비트 후 0 복귀)                           │   │ │
│   │   │                                                      │   │ │
│   │   │  RZ 장점: 동기화 (자체 클럭)                          │   │ │
│   │   │  RZ 단점: 2배 대역폭                                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: RZ는 동기화가 쉽지만 대역폭이 2배 필요합니다.

**💡 비유**: RZ는 매 글자마다 쉼표를 찍는 것과 같습니다.

**📢 섹션 요약 비유**: RZ = 자체 클럭 내장!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **0 복귀** | 동기화 | 중간 전이 | 휴식 |
| **3레벨** | 신호 | +V, 0, -V | 다단계 |
| **대역폭** | 비용 | 2배 | 비싼 |
| **클럭** | 타이밍 | 내장 | 시계 |

**RZ 파형 상세**

```
비트 구간:

         ← 비트 T →
         ┌──┐
    +V ──┤  │
         │  ├── 0 복귀
    0V ──┤  │
         │  │
    -V ──┴──┘

1의 경우: 전반부 +V, 후반부 0
0의 경우: 전반부 -V, 후반부 0
```

**핵심 알고리즘**: 비트 → 펄스 + 0 복귀 → 전송

**📢 섹션 요약 비유**: Manchester는 RZ의 변형!
