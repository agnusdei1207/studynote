### Ⅰ. 개요 (Context & Background)

**개념 정의**

OLAP는 다차원 데이터 분석을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OLAP                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OLAP 큐브                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │              Time                                    │   │ │
│   │   │               ↑                                      │   │ │
│   │   │              ╱│╲                                     │   │ │
│   │   │             ╱ │ ╲                                    │   │ │
│   │   │            ╱  │  ╲                                   │   │ │
│   │   │           ╱   │   ╲                                  │   │ │
│   │   │ Product ────────────── Region                        │   │ │
│   │   │          │    │    │                                 │   │ │
│   │   │          │  Sales  │                                 │   │ │
│   │   │          │    │    │                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  연산:                                                │   │ │
│   │   │  - Roll-up: 요약                                     │   │ │
│   │   │  - Drill-down: 상세                                  │   │ │
│   │   │  - Slice: 1차원 선택                                 │   │ │
│   │   │  - Dice: 다차원 선택                                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: OLAP은 큐브 구조로 다차원 분석을 지원합니다.

**📢 섹션 요약 비유:** OLAP = 다차원 돋보기!
