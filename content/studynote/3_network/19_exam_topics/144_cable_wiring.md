# 케이블 배선: 다이렉트 vs 크로스오버

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 핀 배열에 따른 케이블 분류
> 2. **가치**: 장비 간 올바른 연결
> 3. **융합**: T568A/B, Auto-MDIX와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

다이렉트 케이블은 양 끝이 같은 핀 배열이고, 크로스오버 케이블은 송수신 핀이 교차된 케이블입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    케이블 배선 (다이렉트 vs 크로스오버)              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              T568A / T568B 핀 배열                            │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  핀 │ T568A        │ T568B        │                 │   │ │
│   │   │  ───┼──────────────┼──────────────┤                 │   │ │
│   │   │  1  │ 녹색/흰      │ 오렌지/흰    │                 │   │ │
│   │   │  2  │ 녹색         │ 오렌지       │                 │   │ │
│   │   │  3  │ 오렌지/흰    │ 녹색/흰      │                 │   │ │
│   │   │  4  │ 파랑         │ 파랑         │                 │   │ │
│   │   │  5  │ 파랑/흰      │ 파랑/흰      │                 │   │ │
│   │   │  6  │ 오렌지       │ 녹색         │                 │   │ │
│   │   │  7  │ 갈색/흰      │ 갈색/흰      │                 │   │ │
│   │   │  8  │ 갈색         │ 갈색         │                 │   │ │
│   │   │                                                      │   │ │
│   │   │  T568B가 상용에서 더 많이 사용됨                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              케이블 타입                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  다이렉트 케이블 (Straight-through):                   │   │ │
│   │   │                                                      │   │ │
│   │   │  A──────────────────────A                            │   │ │
│   │   │  1──────────────────────1                            │   │ │
│   │   │  2──────────────────────2                            │   │ │
│   │   │  3──────────────────────3                            │   │ │
│   │   │  6──────────────────────6                            │   │ │
│   │   │                                                      │   │ │
│   │   │  용도: PC↔스위치, PC↔허브, 라우터↔스위치               │   │ │
│   │   │                                                      │   │ │
│   │   │  크로스오버 케이블 (Crossover):                         │   │ │
│   │   │                                                      │   │ │
│   │   │  A──────────────────────B                            │   │ │
│   │   │  1(TX+)────────────────3(RX+)                        │   │ │
│   │   │  2(TX-)────────────────6(RX-)                        │   │ │
│   │   │  3(RX+)────────────────1(TX+)                        │   │ │
│   │   │  6(RX-)────────────────2(TX-)                        │   │ │
│   │   │                                                      │   │ │
│   │   │  용도: PC↔PC, 스위치↔스위치, 허브↔허브                 │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 다이렉트 케이블은 T568B-T568B, 크로스는 T568B-T568A입니다.

**💡 비유**: 다이렉트는 직진 도로, 크로스오버는 교차로입니다.

**📢 섹션 요약 비유:** 같은 장비 = 크로스, 다른 장비 = 다이렉트!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **T568A** | 표준 A | 녹색 우선 | 미국 정부 |
| **T568B** | 표준 B | 오렌지 우선 | 상용 |
| **교차** | 핀 변경 | 1↔3, 2↔6 | 역전 |

**비교표**

```
+------------+----------+----------+
| 항목       | 다이렉트 | 크로스   |
+------------+----------+----------+
| 양 끝      | 같음     | 다름     |
| TX-RX      | 그대로   | 교차     |
| PC↔스위치  │ O        │ X       |
| PC↔PC      │ X        │ O       |
+------------+----------+----------+
```

**핵심 알고리즸**: TX-RX 매칭, MDI/MDI-X 정합

**📢 섹션 요약 비유:** T568B가 업계 표준!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 연결 매트릭스**

| 장비 A | 장비 B | 케이블 |
|:---|:---|:---|
| **PC** | 스위치 | 다이렉트 |
| **PC** | PC | 크로스 |
| **스위치** | 스위치 | 크로스 |
| **라우터** | 스위치 | 다이렉트 |

**📢 섹션 요약 비유:** Auto-MDIX로 해결!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 일반 연결** → 다이렉트 케이블
**시나리오 2: 같은 장비 간** → 크로스 케이블
**시나리오 3: 최신 장비** → Auto-MDIX 무관

**도입 체크리스트:** 장비 타입, Auto-MDIX 지원

**📢 섹션 요약 비유:** Auto-MDIX 시대에는 무관!
