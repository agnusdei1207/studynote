# 꼬임 쌍선 케이블 (Twisted Pair Cable)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 두 도선을 꼬아 간섭 최소화
> 2. **가치**: 저비용, 이더넷 표준
> 3. **융합**: LAN, 전화선, IoT와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

꼬임 쌍선 케이블(Twisted Pair Cable)은 두 개의 절연된 도선을 서로 꼬아 외부 간섭을 줄이는 전송 매체입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    꼬임 쌍선 케이블 (Twisted Pair Cable)            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              구조 및 원리                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  꼬임 원리:                                              │   │ │
│   │   │                                                      │   │ │
│   │   │  단일 도선:                     꼬임 쌍선:             │   │ │
│   │   │  ───────────────                ╭──╮╭──╮╭──╮           │   │ │
│   │   │    간섭 노출                    │  ││  ││  │            │   │ │
│   │   │                                 ╰──╯╰──╯╰──╯           │   │ │
│   │   │                                   간섭 상쇄              │   │ │
│   │   │                                                      │   │ │
│   │   │  노이즈 상쇄 원리:                                      │   │ │
│   │   │  • 두 선이 같은 노이즈 영향                             │   │ │
│   │   │  • 차동 신호로 노이즈 상쇄                              │   │ │
│   │   │  • 꼬임 간격 = 피치 (Twist Pitch)                       │   │ │
│   │   │                                                      │   │ │
│   │   │  종류:                                                   │   │ │
│   │   │  ┌─────────────────────────────────────┐               │   │ │
│   │   │  │  UTP  │  비차폐  │  저비용, LAN     │               │   │ │
│   │   │  │  STP  │  차폐    │  노이즈 강함     │               │   │ │
│   │   │  │  FTP  │  포일    │  중간            │               │   │ │
│   │   │  │  S/FTP│  이중    │  고성능          │               │   │ │
│   │   │  └─────────────────────────────────────┘               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              4페어 구조                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  이더넷 케이블 (Cat5e/Cat6):                            │   │ │
│   │   │                                                      │   │ │
│   │   │      외피                                            │   │ │
│   │   │    ┌──────────────────────────────┐                  │   │ │
│   │   │    │  ╭─╮    ╭─╮    ╭─╮    ╭─╮  │                  │   │ │
│   │   │    │  │ │    │ │    │ │    │ │  │  4쌍 (8선)        │   │ │
│   │   │    │  ╰─╯    ╰─╯    ╰─╯    ╰─╯  │                  │   │ │
│   │   │    │  오렌지  녹색   파랑  갈색   │                  │   │ │
│   │   │    └──────────────────────────────┘                  │   │ │
│   │   │                                                      │   │ │
│   │   │  T568B 배선:                                           │   │ │
│   │   │  1:오렌지/흰 2:오렌지 3:녹색/흰 4:파랑                  │   │ │
│   │   │  5:파랑/흰 6:녹색 7:갈색/흰 8:갈색                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 꼬임 쌍선은 이더넷 LAN의 표준 전송 매체입니다.

**💡 비유**: 꼬임 쌍선은 두 사람이 팔짱을 끼고 걷는 것처럼 서로 보호합니다.

**📢 섹션 요약 비유**: 꼬임 = 노이즈 상쇄!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **꼬임** | 간섭 방지 | 차동 신호 | 팔짱 |
| **피치** | 꼬임 간격 | cm 단위 | 걸음 |
| **페어** | 신호 쌍 | 4쌍 기본 | 커플 |

**비교표**

```
+------------+----------+----------+
| 항목       | UTP      | STP      |
+------------+----------+----------+
| 차폐       | 없음     | 있음     |
| 비용       | 낮음     | 높음     |
| 노이즈     | 취약     | 강함     |
| 유연성     | 높음     | 낮음     |
+------------+----------+----------+
```

**핵심 알고리즸**: 차동 신호, CMRR (Common Mode Rejection Ratio)

**📢 섹션 요약 비유**: 이더넷 = UTP!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 카테고리**

| 카테고리 | 속도 | 대역폭 |
|:---|:---:|:---:|
| **Cat5e** | 1Gbps | 100MHz |
| **Cat6** | 10Gbps | 250MHz |
| **Cat6a** | 10Gbps | 500MHz |
| **Cat7** | 10Gbps | 600MHz |

**📢 섹션 요약 비유**: Cat6 = 10Gbps 55m!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 일반 사무실** → Cat5e/Cat6
**시나리오 2: 데이터센터** → Cat6a/Cat7
**시나리오 3: 공장** → STP/FTP

**도입 체크리스트**: 속도, 거리, EMI 환경

**📢 섹션 요약 비유**: 100m = UTP 한계!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 저비용, 범용성

**미래 전망**: Cat8 (40Gbps) 확대

**참고 표준**: ANSI/TIA-568, ISO/IEC 11801

**📢 섹션 요약 비유**: UTP는 LAN의 기본!
