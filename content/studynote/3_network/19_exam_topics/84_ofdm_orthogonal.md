# 직교 주파수 분할 다중화 (OFDM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 직교하는 다수 부반송파로 병렬 전송
> 2. **가치**: 주파수 효율, 다중 경로 내성, 채널 적응
> 3. **융합**: WiFi, LTE, 5G, DAB와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OFDM(Orthogonal Frequency Division Multiplexing)은 직교하는 다수의 부반송파를 사용하여 데이터를 병렬로 전송하는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OFDM (직교 주파수 분할 다중화)                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OFDM 원리                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  FDM vs OFDM:                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  FDM (보호대역 필요):                                   │   │ │
│   │   │  ├──┤GB├──┤GB├──┤GB├──┤                             │   │ │
│   │   │  │f1 │ │f2 │ │f3 │ │f4 │                             │   │ │
│   │   │  └──┴──┴──┴──┴──┴──┴──┘                             │   │ │
│   │   │      낭비!                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  OFDM (직교, 보호대역 불필요):                          │   │ │
│   │   │  ├──┼──┼──┼──┼──┼──┼──┤                             │   │ │
│   │   │  │f1│f2│f3│f4│f5│f6│f7│                             │   │ │
│   │   │  └──┴──┴──┴──┴──┴──┴──┘                             │   │ │
│   │   │    ↑ ↑ ↑ ↑                                           │   │ │
│   │   │   직교 (간섭 없음)                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  직교 조건: Δf = 1/Tsymbol                            │   │ │
│   │   │  → 부반송파 간격 = 심볼 주기의 역수                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OFDM 송수신 구조                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  송신:                                                  │   │ │
│   │   │  데이터 → [S/P] → [QAM 변조] → [IFFT] → [CP 추가]    │   │ │
│   │   │           ↓       ↓          ↓         ↓              │   │ │
│   │   │         직렬→병렬  부반송파    시간영역   보호구간      │   │ │
│   │   │                                          ↓              │   │ │
│   │   │                                       [DAC] → 전송      │   │ │
│   │   │                                                      │   │ │
│   │   │  수신:                                                  │   │ │
│   │   │  수신 → [ADC] → [CP 제거] → [FFT] → [QAM 복조] → [P/S]│   │ │
│   │   │                     ↓         ↓         ↓        ↓     │   │ │
│   │   │                  보호구간  주파수영역  심볼복원  병렬→직렬│   │ │
│   │   │                                          ↓              │   │ │
│   │   │                                        데이터           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: OFDM은 FFT/IFFT로 효율적으로 구현됩니다.

**💡 비유**: OFDM은 여러 라디오 채널을 촘촘하게 채우는 것입니다.

**📢 섹션 요약 비유**: OFDM = 직교 부반송파 + IFFT!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **IFFT** | 주파수→시간 | N-point FFT | 역변환 |
| **FFT** | 시간→주파수 | N-point FFT | 변환 |
| **CP** | ISI 방지 | 심볼 끝 복사 | 완충지 |
| **부반송파** | 병렬 전송 | N개 직교 | 채널 |

**비교표**

```
+------------+----------+----------+
| 항목       | FDM      | OFDM     |
+------------+----------+----------+
| 보호대역   | 필요     | 불필요   |
| 스펙트럼   | 낭비     | 효율     |
| 다중경로   | 취약     | 강함     |
| 구현       | 아날로그 | 디지털   |
+------------+----------+----------+
```

**핵심 알고리즸**: S/P → QAM → IFFT → +CP → D/A → 채널 → A/D → -CP → FFT → 복조

**📢 섹션 요약 비유**: FFT/IFFT가 OFDM의 핵심!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: OFDM 응용**

| 표준 | 부반송파 | 대역폭 |
|:---|:---:|:---:|
| **WiFi 802.11a** | 64 | 20MHz |
| **LTE** | 2048 | 20MHz |
| **5G NR** | 4096 | 100MHz |
| **DAB** | 1536 | 1.5MHz |

**📢 섹션 요약 비유**: WiFi 802.11a = 64 부반송파!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: WiFi** → OFDM (64 SC)
**시나리오 2: LTE** → OFDM (DL) / SC-FDMA (UL)
**시나리오 3: 5G** → OFDM + Numerology

**도입 체크리스트**: 부반송파 수, CP 길이, 채널

**📢 섹션 요약 비유**: 다중 경로 환경에 OFDM!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 높은 스펙트럼 효율, 다중 경로 내성

**미래 전망**: OFDMA, FBMC, NOMA 진화

**참고 표준**: IEEE 802.11, 3GPP TS 36.211

**📢 섹션 요약 비유**: OFDM은 무선 광대역의 핵심!
