# sFlow - 샘플링 플로우

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 샘플링 기반 트래픽 모니터링
> 2. **가치**: 낮은 오버헤드
> 3. **융합**: 네트워크 가시성과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

sFlow(Sampled Flow)는 패킷 샘플링으로 네트워크 트래픽을 모니터링합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    sFlow                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              sFlow 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Network Traffic]                                  │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ 샘플링 (예: 1/N)                              │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           sFlow Agent                        │    │   │ │
│   │   │  │  (스위치/라우터)                              │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • 샘플링: 1/N 패킷                           │    │   │ │
│   │   │  │  • 카운터: 인터페이스 통계                    │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ UDP 6343                                     │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           sFlow Collector                      │    │   │ │
│   │   │  │  • 수집, 분석, 시각화                         │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              sFlow vs NetFlow                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 항목       │ sFlow    │ NetFlow  │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 수집 방식  │ 샘플링   │ 플로우    │               │   │ │
│   │   │  │ 오버헤드   │ 낮음     │ 높음     │               │   │ │
│   │   │  │ 정확도     │ 추정     │ 정확     │               │   │ │
│   │   │  │ 확장성     │ 높음     │ 제한     │               │   │ │
│   │   │  | 표준       │ InMon    │ Cisco    │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: sFlow는 샘플링으로 오버헤드를 줄이고 확장성을 높입니다.

**💡 비유**: sFlow는 통계 조사입니다. 일부만 보고 전체를 추정합니다.

**📢 섹션 요약 비유:** sFlow = 샘플링!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Sampling** | 1/N |
| **Counter** | 인터페이스 |
| **Datagram** | 샘플 |

**비교표: 샘플링 레이트**

```
+------------+----------+----------+
| 레이트      │ 오버헤드 │ 정확도 │
+------------+----------+----------+
| 1:100      │ 높음     │ 높음   │
| 1:1000     │ 중간     │ 중간   │
| 1:10000    │ 낮음     │ 낮음   │
+------------+----------+----------+
```

**핵심 알고리즸:** 샘플링, 통계

**📢 섹션 요약 비유:** 1:1000 = 일반적!
