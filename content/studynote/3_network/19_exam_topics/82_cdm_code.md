# 코드 분할 다중화 (CDM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고유 코드로 신호 확산하여 동시 전송
> 2. **가치**: 보안, 간섭 내성, 주파수 효율
> 3. **융합**: CDMA, 3G, GPS와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CDM(Code Division Multiplexing)은 각 신호에 고유한 직교 코드를 곱하여 같은 주파수와 시간에 동시 전송하는 다중화 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CDM (코드 분할 다중화)                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CDM 원리                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  기본 개념:                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  사용자 1: 데이터 × 코드1 → 확산 신호 1               │   │ │
│   │   │  사용자 2: 데이터 × 코드2 → 확산 신호 2               │   │ │
│   │   │  사용자 3: 데이터 × 코드3 → 확산 신호 3               │   │ │
│   │   │                        ↓                              │   │ │
│   │   │                   [결합] → 전송                        │   │ │
│   │   │                        ↓                              │   │ │
│   │   │  수신측:                                               │   │ │
│   │   │  수신 신호 × 코드1 → 사용자 1 데이터 복원              │   │ │
│   │   │  수신 신호 × 코드2 → 사용자 2 데이터 복원              │   │ │
│   │   │  수신 신호 × 코드3 → 사용자 3 데이터 복원              │   │ │
│   │   │                                                      │   │ │
│   │   │  직교성: 코드_i × 코드_j = 0 (i≠j)                    │   │ │
│   │   │           코드_i × 코드_i = 1                         │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Walsh 코드 예시                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  W0 = [+1, +1, +1, +1]                               │   │ │
│   │   │  W1 = [+1, -1, +1, -1]                               │   │ │
│   │   │  W2 = [+1, +1, -1, -1]                               │   │ │
│   │   │  W3 = [+1, -1, -1, +1]                               │   │ │
│   │   │                                                      │   │ │
│   │   │  직교성 확인:                                          │   │ │
│   │   │  W0 × W1 = (+1×+1)+(+1×-1)+(+1×+1)+(+1×-1) = 0       │   │ │
│   │   │  W0 × W0 = (+1×+1)+(+1×+1)+(+1×+1)+(+1×+1) = 4       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: CDM은 CDMA와 GPS의 기반 기술입니다.

**💡 비유**: CDM은 여러 언어로 동시에 대화하는 파티장입니다.

**📢 섹션 요약 비유**: 코드로 나누면 동시 전송!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **확산 코드** | 식별자 | 직교 코드 | 언어 |
| **상관기** | 복원 | 코드 곱셈 | 해독기 |
| **처리 이득** | 이득 | Gp = 10log(N) | 향상 |

**비교표**

```
+------------+----------+----------+----------+
| 항목       | FDM      | TDM      | CDM      |
+------------+----------+----------+----------+
| 분할 자원  | 주파수   | 시간     | 코드     |
| 동시 전송  | O        | O        | O        |
| 간섭 내성  | 낮음     | 낮음     | 높음     |
| 복잡도     | 낮음     | 중간     | 높음     |
+------------+----------+----------+----------+
```

**핵심 알고리즸**: 데이터 × 코드 → 전송 → 수신 × 코드 → 적분 → 복원

**📢 섹션 요약 비유**: 직교 코드 = 간섭 없음!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 3G CDMA** → CDM + DSSS
**시나리오 2: GPS** → CDM (Gold Code)
**시나리오 3: 위성 통신** → CDM

**도입 체크리스트**: 직교성, 처리 이득, 전력 제어

**📢 섹션 요약 비유**: CDM은 보안과 효율의 균형!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 주파수 효율, 보안, 간섭 내성

**미래 전망**: 5G NOMA로 진화

**참고 표준**: IS-95, 3GPP

**📢 섹션 요약 비유**: CDM은 3G의 핵심!
