# Channel - 채널

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 통신 경로
> 2. **가치**: 데이터 전송 매체
> 3. **융합:** 유선, 무선, 논리적

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Channel은 정보가 흐르는 경로입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Channel                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              채널 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  물리적 채널 (Physical Channel):                      │   │ │
│   │   │  - 유선: twisted pair, coax, fiber                   │   │ │
│   │   │  - 무선: radio, microwave, infrared                  │   │ │
│   │   │                                                      │   │ │
│   │   │  논리적 채널 (Logical Channel):                       │   │ │
│   │   │  - 주파수 채널                                       │   │ │
│   │   │  - 시간 슬롯                                         │   │ │
│   │   │  - 코드 채널                                         │   │ │
│   │   │  - 포트 번호                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  채널 용량 (Shannon):                                 │   │ │
│   │   │  C = B × log₂(1 + S/N)                               │   │ │
│   │   │  C: 용량 (bps)                                       │   │ │
│   │   │  B: 대역폭 (Hz)                                      │   │ │
│   │   │  S/N: 신호 대 잡음비                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 채널은 통신 시스템의 기반입니다.

**📢 섹션 요약 비유:** Channel = 정보 고속도로!
