### Ⅰ. 개요 (Context & Background)

**개념 정의**

Zigbee, Z-Wave, Thread는 스마트홈과 IoT를 위한 저전력 무선 메시 네트워크 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    스마트홈 무선 기술                               │

│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  기술    │ 주파수      │ 속도    │ 거리   │ 메시  │   │
│   │  ───────┼───────────┼───────┼──────┼───────│   │
│   │  Zigbee  │ 2.4GHz     │ 250kbps│ 10-100m│ O     │   │
│   │  Z-Wave  │ 908/868MHz │ 100kbps│ 30-100m│ O     │   │
│   │  Thread  │ 2.4GHz     │ 250kbps│ 10-100m│ O     │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              Zigbee 구조                                    │   │
```

> **해설**: Zigbee는 IEEE 802.15.4 기반의 메시 네트워크입니다.

**💡 비유**: Zigbee는 기기들이 서로 손잡고 메시지를 전달하는 것입니다.

**📢 섹션 요약 비유:** Zigbee = 메시 네트워크!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 기술 | 기반 | 주파수 | 특징 |
|:---|:---|:---|:---|
| **Zigbee** | 802.15.4 | 2.4GHz | 범용 |
| **Z-Wave** | 전용 | Sub-GHz | 간섭 적음 |
| **Thread** | 802.15.4 | 2.4GHz | IPv6 |

**비교표**

```
+------------+----------+----------+
| 항목       │ Zigbee   │ Thread   │
+------------+----------+----------+
| IP         │ 아니오   │ IPv6     │
│ 백본       │ 게이트웨이│ Border   │
│ 보안       │ AES-128  │ AES-128  │
│ 상호운용   │ ZCL      │ Matter   │
+------------+----------+----------+
```

**핵심 알고리즸:** AODV 라우팅, CSMA/CA

**📢 섹션 요약 비유:** Thread = IPv6 메시!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 스마트홈 표준**

| 표준 | 기반 | 특징 |
|:---|:---|:---|
| **Matter** | Thread/WiFi | 상호운용 |

| **Z-Wave Plus** | Z-Wave | 향상된 기능 |

**📢 섹션 요약 비유:** Matter = 통합 표준!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
