# TLP (Thread-Level Parallelism) - 스레드 수준 병렬성

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 스레드 간 병렬 실행
> 2. **가치**: 멀티코어 활용
> 3. **융합:** Multi-core, Parallel Programming, Concurrency

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

TLP는 여러 스레드를 병렬로 실행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TLP                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TLP 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │              Application                    │    │   │ │
│   │   │  │  ┌─────────┐ ┌─────────┐ ┌─────────┐       │    │   │ │
│   │   │  │  │Thread 1 │ │Thread 2 │ │Thread 3 │       │    │   │ │
│   │   │  │  │         │ │         │ │         │       │    │   │ │
│   │   │  │  │ Task A  │ │ Task B  │ │ Task C  │       │    │   │ │
│   │   │  │  └────┬────┘ └────┬────┘ └────┬────┘       │    │   │ │
│   │   │  └───────┼───────────┼───────────┼────────────┘    │   │ │
│   │   │          │           │           │                  │   │ │
│   │   │          ↓           ↓           ↓                  │   │ │
│   │   │  ┌──────────────────────────────────────────┐       │   │ │
│   │   │  │ Core 0 │ Core 1 │ Core 2 │ Core 3       │       │   │ │
│   │   │  └──────────────────────────────────────────┘       │   │ │
│   │   │                                                      │   │ │
│   │   │  TLP: 여러 스레드가 여러 코어에서 병렬 실행          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: TLP는 멀티코어 시대의 핵심 병렬성입니다.

**📢 섹션 요약 비유:** TLP = 팀 프로젝트!
