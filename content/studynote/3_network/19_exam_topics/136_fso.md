# 자유 공간 광통신 (FSO / Free Space Optics)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 자유 공간(공기)으로 빛 전송
> 2. **가치**: 무선 광통신, 높은 보안성
> 3. **융합**: 위성-지상, 건물 간, 군사와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

FSO(Free Space Optics)는 광섬유 없이 자유 공간(공기)을 통해 레이저 빛으로 데이터를 전송하는 무선 광통신 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FSO (Free Space Optics)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FSO 시스템 구조                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │   건물 A                          건물 B              │   │ │
│   │   │  ┌─────┐                          ┌─────┐            │   │ │
│   │   │  │ FSO │                          │ FSO │            │   │ │
│   │   │  │송신기├──────────────────────────┤수신기│            │   │ │
│   │   │  │     │     레이저 빔            │     │            │   │ │
│   │   │  └─────┘    (자유 공간)          └─────┘            │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 무선 광통신                                          │   │ │
│   │   │  • 광섬유 없이 빛으로 전송                               │   │ │
│   │   │  • 높은 대역폭 (수 Gbps)                                 │   │ │
│   │   │  • 낮은 지연                                            │   │ │
│   │   │  • 주파수 허가 불필요                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 높은 보안성 (좁은 빔)                                 │   │ │
│   │   │  • 간섭 없음                                            │   │ │
│   │   │  • 빠른 설치                                            │   │ │
│   │   │  • 비용 절감                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                   │   │ │
│   │   │  • 날씨 영향 (비, 안개, 눈)                              │   │ │
│   │   │  • 가시선(LOS) 필요                                     │   │ │
│   │   │  • 거리 제한 (수 km)                                     │   │ │
│   │   │  • 정렬 민감                                            │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              응용 분야                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  • 건물 간 연결 (Last-mile)                            │   │ │
│   │   │  • 위성-지상 통신                                       │   │ │
│   │   │  • 군사 통신                                            │   │ │
│   │   │  • 재난 복구                                            │   │ │
│   │   │  • 임시 네트워크                                         │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: FSO는 광섬유 설치가 어려운 곳에서 무선 광통신으로 대체합니다.

**💡 비유**: FSO는 두 건물 사이에 보이지 않는 광섬유를 설치하는 것과 같습니다.

**📢 섹션 요약 비유**: FSO = 무선 광통신!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **레이저** | 광원 | LD | 손전등 |
| **광학계** | 빔 제어 | 렌즈/거울 | 망원경 |
| **수신기** | 수광 | PIN/APD | 눈 |

**비교표**

```
+------------+----------+----------+
| 항목       | FSO      | RF 무선  |
+------------+----------+----------+
| 대역폭     | 높음     | 낮음     |
| 간섭       | 없음     | 있음     |
| 보안       | 높음     | 낮음     |
| 날씨 영향   | 큼       | 작음     |
| 허가       | 불필요   | 필요     |
+------------+----------+----------+
```

**핵심 알고리즸**: 빔 정렬, 자동 추적

**📢 섹션 요약 비유**: 비/안개는 FSO의 적!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 건물 간 연결** → FSO
**시나리오 2: 재난 복구** → FSO

**📢 섹션 요약 비유**: LOS 필수!

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 광섬유 없이 고속 전송

**참고 표준**: IEEE 802.15.3

**📢 섹션 요약 비유**: FSO는 무선 광섬유!
