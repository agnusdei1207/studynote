# NFS (Network File System) - 네트워크 파일 시스템

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 원격 파일 시스템 접근
> 2. **가치**: Unix/Linux 파일 공유
> 3. **융합:** NAS, RPC, Storage

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NFS는 네트워크로 원격 파일 시스템에 접근합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NFS                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NFS 동작                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                          Server              │   │ │
│   │   │  ┌─────────┐                    ┌─────────┐          │   │ │
│   │   │  │ /mnt/nfs│ ←── Mount ──────── │ /export │          │   │ │
│   │   │  └─────────┘                    └─────────┘          │   │ │
│   │   │      │                                │               │   │ │
│   │   │      │ ── RPC (Remote Procedure Call) │              │   │ │
│   │   │      │ ── NFS Protocol (UDP/TCP 2049) │              │   │ │
│   │   │      │                                │               │   │ │
│   │   │  로컬처럼 원격 파일 접근                               │   │ │
│   │   │                                                      │   │ │
│   │   │  버전:                                               │   │ │
│   │   │  - NFSv3: TCP 지원                                   │   │ │
│   │   │  - NFSv4: 상태 유지, 보안 강화                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NFS는 RPC를 사용해 원격 파일 시스템을 마운트합니다.

**📢 섹션 요약 비유:** NFS = 원격 폴더!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **RPC** | Remote Procedure Call |
| **Port 2049** | NFS 포트 |
| **mountd** | 마운트 데몬 |
| **nfsd** | NFS 데몬 |
| **export** | 공유 디렉토리 |

**핵심 알고리즸:** mount → RPC → file operation

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**NFS vs SMB**

| 항목 | NFS | SMB |
|:---|:---|:---|
| **플랫폼** | Unix/Linux | Windows |
| **포트** | 2049 | 445 |
| **보안** | Kerberos | NTLM/Kerberos |
