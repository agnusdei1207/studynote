# IPsec (IP Security) - IP 보안

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IP 계층 보안 프로토콜
> 2. **가치**: 기밀성, 무결성, 인증
> 3. **융합:** VPN, AH, ESP, IKE

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IPsec은 IP 계층에서 보안을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IPsec                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IPsec 구성요소                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  AH (Authentication Header):                        │   │ │
│   │   │  - 인증, 무결성                                      │   │ │
│   │   │  - 암호화 없음                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  ESP (Encapsulating Security Payload):              │   │ │
│   │   │  - 인증, 무결성, 암호화                              │   │ │
│   │   │                                                      │   │ │
│   │   │  IKE (Internet Key Exchange):                       │   │ │
│   │   │  - 키 교환                                           │   │ │
│   │   │  - SA (Security Association) 관리                    │   │ │
│   │   │                                                      │   │ │
│   │   │  모드:                                               │   │ │
│   │   │  - Transport Mode: 페이로드만 암호화                 │   │ │
│   │   │  - Tunnel Mode: 전체 패킷 암호화                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IPsec은 AH와 ESP로 IP 계층 보안을 제공합니다.

**📢 섹션 요약 비유:** IPsec = 보안 봉투!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **AH** | 인증 헤더 |
| **ESP** | 암호화 페이로드 |
| **IKE** | 키 교환 |
| **SA** | Security Association |
| **SPI** | Security Parameter Index |

**핵심 알고리즸:** IKE → SA 생성 → AH/ESP 적용

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Transport vs Tunnel Mode**

| 항목 | Transport | Tunnel |
|:---|:---|:---|
| **암호화** | 페이로드 | 전체 |
| **용도** | End-to-End | VPN |
| **오버헤드** | 낮음 | 높음 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** Site-to-Site VPN → Tunnel Mode
**시나리오 2:** End-to-End → Transport Mode
**시나리오 3:** IPv6 → IPsec 내장
