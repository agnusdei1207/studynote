### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Fabric Controller** | 중앙 제어기 |
| **Fabric Edge** | 엣지 스위치 |
| **Fabric Core** | 코어 스위치 |
| **VXLAN** | 오버레이 |
| **Automation** | 자동화 |

**핵심 알고리듬:** 자동 발견 → 토폴로지 구성 → 정책 배포 → 모니터링
