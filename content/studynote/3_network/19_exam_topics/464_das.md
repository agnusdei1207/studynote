# DAS (분산 안테나 시스템) - Distributed Antenna System

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 안테나를 분산 배치
> 2. **가치**: 실내 커버리지, 균일 신호
> 3. **융합**: Passive/Active DAS, Small Cell

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

> **해설**: DAS는 대형 건물에 균일한 커버리지를 제공합니다.

**💡 비유**: DAS는 건물의 냉난방 시스템입니다.

**📢 섹션 요약 비유:** DAS = 실내 커버리지!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Headend** | 신호 집합 장치 |
| **RAU** | 원격 안테나 장치 |
| **Fiber** | 광섬유 분배 |
| **Donor** | 신호 소스 |

**비교표: Passive vs Active DAS**

```
+------------+----------+----------+----------+
| 항목       │ Passive  │ Active   │ Hybrid   │
+------------+----------+----------+----------+
| 전원       │ 없음     │ 필요     │ 부분     │
| 거리       │ 제한     │ 확장     │ 중간     │
| 비용       │ 낮음     │ 높음     │ 중간     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 신호 분배, 전력 제어

**📢 섹션 요약 비유:** Active DAS = 대형 건물!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: DAS vs Small Cell**

| 항목 | DAS | Small Cell |
|:---|:---|:---|
| **규모** | 대형 | 소형 |
| **비용** | 높음 | 낮음 |
| **유연성** | 낮음 | 높음 |

**📢 섹션 요약 비유:** 대형 건물 = DAS!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 대형 건물 → Active DAS
**시나리오 2:** 중형 건물 → Hybrid DAS
**시나리오 3:** 소형 공간 → Passive DAS

**도입 체크리스트:** 크기, 예산, 통신사

**📢 섹션 요약 비유:** 다중 통신사 = Neutral Host!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 커버리지 100% 달성

**미래 전망:** 5G DAS, CBRS

**참고 표준:** NFPA, 건축법규

**📢 섹션 요약 비유:** DAS는 실내 커버리지의 표준!
