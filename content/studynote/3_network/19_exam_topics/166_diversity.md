# 다이버시티 (Diversity) - 공간/주파수/시간

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 독립 경로로 신호 수신
> 2. **가치**: 페이딩 완화, 신뢰성 향상
> 3. **융합**: MIMO, 이동통신과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

다이버시티(Diversity)는 여러 독립적인 경로로 신호를 수신하여 페이딩 영향을 줄이는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    다이버시티 (Diversity)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              다이버시티 종류                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 공간 다이버시티 (Space Diversity):                  │   │ │
│   │   │     • 여러 안테나로 수신                               │   │ │
│   │   │     • 안테나 간격 > λ/2                                │   │ │
│   │   │     • 가장 효과적                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 주파수 다이버시티 (Frequency Diversity):            │   │ │
│   │   │     • 여러 주파수로 전송                               │   │ │
│   │   │     • 주파수 간격 > coherence bandwidth                │   │ │
│   │   │     • 대역폭 낭비                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 시간 다이버시티 (Time Diversity):                   │   │ │
│   │   │     • 여러 시간에 재전송                                │   │ │
│   │   │     • 간격 > coherence time                            │   │ │
│   │   │     • 지연 증가                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 편파 다이버시티 (Polarization Diversity):           │   │ │
│   │   │     • 수직/수평 편파 사용                               │   │ │
│   │   │     • 공간 절약                                        │   │ │
│   │   │     • 3dB 손실                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  5. 각도 다이버시티 (Angle Diversity):                  │   │ │
│   │   │     • 다른 각도의 빔 사용                               │   │ │
│   │   │     • 스마트 안테나                                    │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     ││   ┌──────────────────────────────────────────────────────────────┐ │
│   │              합성 기법 (Combining)                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  • 선택 다이버시티 (Selection): 가장 강한 신호 선택     │   │ │
│   │   │  • 등이득 합성 (EGC): 위상 정렬 후 합산                 │   │ │
│   │   │  • 최대비 합성 (MRC): SNR 가중 합산 (최적)              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 다이버시티는 페이딩을 완화하는 가장 효과적인 기술입니다.

**💡 비유**: 다이버시티는 여러 귀로 듣는 것과 같습니다. 하나가 안 들려도 다른 귀로 들립니다.

**📢 섹션 요약 비유:** MRC = 최적 합성!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 종류 | 요구조건 | 장점 | 단점 |
|:---|:---|:---|:---|
| **공간** | 간격 > λ/2 | 효과적 | 공간 필요 |
| **주파수** | 간격 > coh BW | 단순 | 대역 낭비 |
| **시간** | 간격 > coh time | 구현 쉬움 | 지연 |
| **편파** | V/H 편파 | 공간 절약 | 3dB 손실 |

**비교표**

```
+------------+----------+----------+
| 합성 기법   │ 성능     │ 복잡도   │
+------------+----------+----------+
| Selection  │ 낮음     │ 낮음     │
| EGC        │ 중간     │ 중간     │
| MRC        │ 높음     │ 높음     │
+------------+----------+----------+
```

**핵심 알고리즸:** MRC (Maximum Ratio Combining)

**📢 섹션 요약 비유:** 공간 다이버시티 = 가장 좋아!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 다이버시티 이득**

| 안테나 수 | 레일리 환경 이득 | BER 개선 |
|:---:|:---:|:---|
| **2** | ~10dB | 10배 |
| **4** | ~15dB | 100배 |
| **8** | ~18dB | 1000배 |

**📢 섹션 요약 비유:** 안테나 많을수록 좋아!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 기지국** → 공간 다이버시티, MRC
**시나리오 2: 단말기** → 편파 다이버시티
**시나리오 3: 위성** → 주파수 다이버시티

**도입 체크리스트:** 공간, 대역폭, 복잡도

**📢 섹션 요약 비유:** 환경에 맞는 다이버시티!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**참고 표준:** 3GPP, IEEE 802.11
