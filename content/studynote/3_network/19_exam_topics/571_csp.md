# CSP (Content Security Policy) - 콘텐츠 보안 정책

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 리소스 로드 정책
> 2. **가치**: XSS 방지
> 3. **융합:** HTTP Header, XSS, CSP Report

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CSP는 브라우저에서 리소스 로드를 제한합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CSP                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CSP 헤더 예시                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Content-Security-Policy:                            │   │ │
│   │   │    default-src 'self';                               │   │ │
│   │   │    script-src 'self' https://cdn.example.com;        │   │ │
│   │   │    style-src 'self' 'unsafe-inline';                 │   │ │
│   │   │    img-src * data:;                                  │   │ │
│   │   │    connect-src 'self' api.example.com;               │   │ │
│   │   │    report-uri /csp-report                            │   │ │
│   │   │                                                      │   │ │
│   │   │  → 지정된 출처만 로드 허용                             │   │ │
│   │   │  → 인라인 스크립트 차단                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: CSP는 화이트리스트 기반 리소스 제어를 합니다.

**📢 섹션 요약 비유:** CSP = 출입 통제!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 지시어 | 설명 |
|:---|:---|
| **default-src** | 기본 정책 |
| **script-src** | 스크립트 |
| **style-src** | 스타일 |
| **img-src** | 이미지 |
| **connect-src** | XHR/Fetch |
| **report-uri** | 위반 보고 |

**핵심 알고리즸:** 헤더 파싱 → 정책 적용
