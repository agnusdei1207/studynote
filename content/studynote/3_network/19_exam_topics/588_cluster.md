# Cluster - 클러스터

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 여러 노드의 논리적 그룹
> 2. **가치**: 확장성, 가용성
> 3. **융합:** HA, Load Balancing, Distributed

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Cluster는 여러 서버를 하나처럼 동작하게 합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Cluster                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Cluster 구성                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │              ┌─────────────┐                         │   │ │
│   │   │              │   Client    │                         │   │ │
│   │   │              └──────┬──────┘                         │   │ │
│   │   │                     │                                │   │ │
│   │   │  ┌──────────────────┴──────────────────┐            │   │ │
│   │   │  │           Cluster Network            │            │   │ │
│   │   │  │  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐ │            │   │ │
│   │   │  │  │Node1│  │Node2│  │Node3│  │Node4│ │            │   │ │
│   │   │  │  └─────┘  └─────┘  └─────┘  └─────┘ │            │   │ │
│   │   │  └──────────────────────────────────────┘            │   │ │
│   │   │                                                      │   │ │
│   │   │  유형:                                               │   │ │
│   │   │  - HA Cluster (Active-Standby)                       │   │ │
│   │   │  - Load Balancing Cluster (Active-Active)            │   │ │
│   │   │  - Compute Cluster (HPC)                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Cluster는 단일 시스템처럼 보이는 여러 노드입니다.

**📢 섹션 요약 비유:** Cluster = 합동 팀!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Node** | 개별 서버 |
| **Cluster SW** | 관리 소프트웨어 |
| **Shared Storage** | 공유 저장소 |
| **Quorum** | 정족수 |

**핵심 알고리즸:** 노드 관리 → 작업 분산 → 상태 동기화

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Cluster 유형**

| 유형 | 용도 |
|:---|:---|
| **HA** | 고가용성 |
| **LB** | 부하 분산 |
| **HPC** | 고성능 연산 |
| **Storage** | 분산 저장 |
