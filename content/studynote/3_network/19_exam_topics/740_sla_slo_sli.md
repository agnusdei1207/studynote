# SLA/SLO/SLI - 서비스 수준 지표

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서비스 품질 측정
> 2. **가치**: 신뢰성 목표 설정
> 3. **융합:** SRE, Error Budget, Monitoring

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SLA/SLO/SLI는 서비스 수준을 정의합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SLA / SLO / SLI                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              서비스 수준 계층                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  SLI (Service Level Indicator)                       │   │ │
│   │   │  - 측정 지표                                          │   │ │
│   │   │  - 예: 가용성, 지연시간, 에러율                        │   │ │
│   │   │                        ↓                             │   │ │
│   │   │  SLO (Service Level Objective)                       │   │ │
│   │   │  - 목표 값                                            │   │ │
│   │   │  - 예: 가용성 99.9%, 지연 < 100ms                     │   │ │
│   │   │                        ↓                             │   │ │
│   │   │  SLA (Service Level Agreement)                       │   │ │
│   │   │  - 계약상 약속                                        │   │ │
│   │   │  - 예: 99.9% 미달시 환불                              │   │ │
│   │   │                                                      │   │ │
│   │   │  관계: SLI(측정) → SLO(목표) → SLA(계약)              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SLI로 측정하고, SLO를 목표로, SLA를 약속합니다.

**📢 섹션 요약 비유:** SLA/SLO/SLI = 성적표!
