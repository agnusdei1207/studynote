# Elasticsearch - 일래스틱서치

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 분산 검색 엔진
> 2. **가치**: 전문 검색, 실시간 분석
> 3. **융합:** Log Analytics, Kibana, Beats

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Elasticsearch는 분산 검색 엔진입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Elasticsearch                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Elasticsearch 아키텍처                            │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Cluster                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  Node 1        Node 2        Node 3         │    │   │ │
│   │   │  │  ┌─────────┐  ┌─────────┐  ┌─────────┐     │    │   │ │
│   │   │  │  │ Shard 0 │  │ Shard 1 │  │ Shard 2 │     │    │   │ │
│   │   │  │  │ Primary │  │ Primary │  │ Primary │     │    │   │ │
│   │   │  │  ├─────────┤  ├─────────┤  ├─────────┤     │    │   │ │
│   │   │  │  │ Replica │  │ Replica │  │ Replica │     │    │   │ │
│   │   │  │  │   of 2  │  │   of 0  │  │   of 1  │     │    │   │ │
│   │   │  │  └─────────┘  └─────────┘  └─────────┘     │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  Index → Type (deprecated) → Document               │   │ │
│   │   │                                                      │   │ │
```

> **해설**: Elasticsearch는 샤드와 레플리카로 분산 및 고가용성을 제공합니다.

**📢 섹션 요약 비유:** Elasticsearch = 디지털 도서관!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Cluster** | 클러스터 |
| **Node** | 노드 |
| **Index** | 인덱스 |
| **Shard** | 샤드 |
| **Document** | 문서 |

**핵심 알고리즸:** 인덱싱 → 역색인 → 검색 → 랭킹
