# Dynamic IP - 동적 IP

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 자동 할당 IP
> 2. **가치**: 관리 편의성
> 3. **융합:** DHCP, Lease, Pool

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Dynamic IP는 DHCP가 자동으로 할당하는 IP 주소입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Dynamic IP                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              동적 IP 할당                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  DHCP 풀에서 할당:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  DHCP Pool: 192.168.1.100 - 192.168.1.200       ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Client 1 → 192.168.1.100                       ││   │ │
│   │   │  │  Client 2 → 192.168.1.101                       ││   │ │
│   │   │  │  Client 3 → 192.168.1.102                       ││   │ │
│   │   │  │  ...                                            ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                  │   │ │
│   │   │  - 자동 할당                                            │   │ │
│   │   │  - 임대 기간 존재                                        │   │ │
│   │   │  - IP 변경 가능                                          │   │ │
│   │   │  - 관리 편의성                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                  │   │ │
│   │   │  - IP 주소 효율적 사용                                    │   │ │
│   │   │  - 설정 자동화                                           │   │ │
│   │   │  - 중앙 관리                                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Dynamic IP는 네트워크 관리를 단순화합니다.

**📢 섹션 요약 비유:** Dynamic IP = 호텔 방!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Pool** | 주소 풀 |
| **Lease** | 임대 |
| **Scope** | 범위 |
| **Assignment** | 할당 |
| **Release** | 해제 |

**핵심 알고리듬:** 요청 → 풀에서 할당 → 임대 → 갱신/해제

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Static vs Dynamic IP**

| 항목 | Static | Dynamic |
|:---|:---|:---|
| **안정성** | 높음 | 낮음 |
| **관리** | 복잡 | 쉬움 |
| **용도** | 서버 | 클라이언트 |
| **비용** | 높음 | 낮음 |
