# Service Discovery - 서비스 디스커버리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서비스 위치 찾기
> 2. **가치**: 동적 환경 지원
> 3. **융합:** Microservices, Load Balancer, DNS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Service Discovery는 서비스 인스턴스를 찾습니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Service Discovery                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              서비스 디스커버리 패턴                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client-side:                                        │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ Client ──→ Service Registry ──→ Service       │  │   │ │
│   │   │  │         (직접 조회)           (직접 호출)      │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  Server-side:                                        │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ Client ──→ Load Balancer ──→ Service          │  │   │ │
│   │   │  │         (요청)             (프록시)           │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  Service Registry:                                   │   │ │
│   │   │  - Consul, Eureka, Zookeeper, etcd                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 서비스 디스커버리는 동적 환경에서 서비스를 찾습니다.

**📢 섹션 요약 비유:** Service Discovery = 전화번호부!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Service Registry** | 서비스 레지스트리 |
| **Registration** | 등록 |
