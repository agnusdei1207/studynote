# 블루투스 (Bluetooth) - BR/EDR/LE

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단거리 무선 개인 영역 네트워크 (WPAN)
> 2. **가치**: 저전력, 저비용 연결
> 3. **융합**: 웨어러블, IoT, 오디오와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

블루투스(Bluetooth)는 2.4GHz ISM 대역을 사용하는 단거리 무선 통신 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    블루투스 기술                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  버전    │ 속도      │ 거리   │ 특징                       │   │
│   │  ───────┼─────────┼───────┼───────────────────────────│   │
│   │  1.1-2.0 │ 1-3Mbps   │ 10m   │ BR/EDR, 기본              │   │
│   │  3.0     │ 24Mbps    │ 10m   │ HS (High Speed)           │   │
│   │  4.0     │ 1Mbps     │ 100m  │ BLE (Low Energy)          │   │
│   │  5.0     │ 2Mbps     │ 240m  │ LE, 장거리                 │   │
│   │  5.2     │ 2Mbps     │ 240m  │ LE Audio, LC3             │   │
│   │  5.3     │ 2Mbps     │ 240m  │ 향상된 효율                 │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              블루투스 유형                                  │   │
│   │                                                              │   │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. BR/EDR (Basic Rate/Enhanced Data Rate):             │   │ │
│   │   │     • 오디오 스트리밍                                   │   │ │
│   │   │     • 파일 전송                                         │   │ │
│   │   │     • 79채널, 1MHz 간격                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  2. BLE (Bluetooth Low Energy):                          │   │ │
│   │   │     • 센서, 웨어러블                                    │   │ │
│   │   │     • 저전력 (버튼 전지 수년)                            │   │ │
│   │   │     • 40채널, 2MHz 간격                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  3. LE Audio (5.2+):                                     │   │ │
│   │   │     • 멀티 스트림 오디오                                 │   │ │
│   │   │     • Auracast (브로드캐스트)                            │   │ │
│   │   │     • LC3 코덱                                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: BLE는 저전력으로 IoT와 웨어러블의 핵심 기술입니다.

**💡 비유**: 블루투스는 기기끼리 무선으로 악수하는 것입니다.

**📢 섹션 요약 비유:** BLE = 저전력!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소 | 설명 | 역할 |
|:---|:---|:---|
| **GAP** | Generic Access Profile | 연결 관리 |
| **GATT** | Generic Attribute Profile | 데이터 교환 |
| **L2CAP** | Logical Link Control | 프로토콜 다중화 |
| **ATT** | Attribute Protocol | 속성 관리 |

**비교표**

```
+------------+----------+----------+
| 항목       │ BR/EDR   │ BLE      │
+------------+----------+----------+
| 전력       │ 높음     │ 낮음     │
│ 연결 시간   │ 초       │ ms       │
│ 채널       │ 79       │ 40       │
│ 용도       │ 오디오   │ 센서     │
+------------+----------+----------+
```

**핵심 알고리즸:** FHSS, GFSK, CRC

**📢 섹션 요약 비유:** FHSS = 주파수 도약!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 프로파일**

| 프로파일 | 용도 |
|:---|:---|
| **A2DP** | 오디오 스트리밍 |
| **HFP** | 핸즈프리 |
| **SPP** | 시리얼 포트 |
| **HID** | 키보드/마우스 |
| **HOGP** | HID over GATT |

**📢 섹션 요약 비유:** 프로파일 = 용도별 규칙!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 무선 이어폰** → BR/EDR, A2DP
**시나리오 2: 피트니스 밴드** → BLE, GATT
**시나리오 3: 스마트홈** → BLE Mesh

**도입 체크리스트:** 전력, 대역폭, 거리

**📢 섹션 요약 비유:** 오디오 = BR/EDR, 센서 = BLE!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 무선 연결 대중화

**참고 표준:** Bluetooth SIG Core Specification
