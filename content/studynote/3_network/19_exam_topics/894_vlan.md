# VLAN - 가상 LAN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 논리적 네트워크 분리
> 2. **가치**: 브로드캐스트 도메인 분리
> 3. **융합:** 802.1Q, Trunk, Tagging

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VID** | VLAN ID |
| **Tag** | 태그 |
| **Trunk** | 트렁크 |
| **Access** | 액세스 |
| **Native VLAN** | 네이티브 VLAN |
