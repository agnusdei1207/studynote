# Fiber Optic - 광케이블

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 빛을 이용한 전송
> 2. **가치**: 장거리 고속 통신
> 3. **융합:** Single-mode, Multi-mode, WDM

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Fiber Optic은 빛을 이용해 데이터를 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Fiber Optic                                       │|
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              광케이블 구조                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  광섬유 구조:                                             │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │          ┌───────────────────────┐              ││   │ │
│   │   │  │          │      Cladding         │              ││   │ │
│   │   │  │          │    (코팅층, n2)        │              ││   │ │
│   │   │  │          │   ┌───────────────┐   │              ││   │ │
│   │   │  │          │   │    Core       │   │              ││   │ │
│   │   │  │          │   │  (코어, n1)    │   │              ││   │ │
│   │   │  │          │   │   ────────→    │   │              ││   │ │
│   │   │  │          │   │    빛          │   │              ││   │ │
│   │   │  │          │   └───────────────┘   │              ││   │ │
│   │   │  │          └───────────────────────┘              ││   │ │
│   │   │  │                                                 ││   │ │
```

> **해설**: 광케이블은 장거리 고속 통신의 핵심입니다.

**📢 섹션 요약 비유:** Fiber Optic = 빛 파이프!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Core** | 코어 |
| **Cladding** | 코팅 |
| **Jacket** | 재킷 |
| **Connector** | 커넥터 |
| **Transceiver** | 트랜시버 |

**핵심 알고리듬:** 전기 → 빛 변환 → 전송 → 빛 → 전기 변환

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**SMF vs MMF**

| 항목 | SMF | MMF |
|:---|:---|:---|
| **거리** | 수 km | 수백 m |
| **비용** | 높음 | 낮음 |
| **대역폭** | 높음 | 중간 |
| **광원** | 레이저 | LED |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** WAN → SMF
**시나리오 2:** 데이터센터 → MMF
**시나리오 3:** FTTH → SMF
