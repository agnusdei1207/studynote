# 성상도 (Constellation Diagram)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: I/Q 평면에서 변조 신호의 점 표현

### Ⅰ. 개요 (Context & Background)

**개념 정의**

성상도(Constellation Diagram)는 디지털 변조 신호를 I(동위상)-Q(직교위상) 평면에 표시한 것입니다.

> **해설**: 성상도는 변조 품질을 시각적으로 분석하는 도구입니다.

**💡 비유**: 성상도는 밤하늘의 별자리처럼 신호 점들을 배치한 것입니다.

**📢 섹션 요약 비유**: EVM = 오차 벡터 크기!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **이상점** | 기준 | 정확한 위치 | 목표 |
| **실제점** | 수신 | 노이즈 포함 | 결과 |
| **EVM** | 오차 | 거리 차이 | 오류 |
| **클러우드** | 산포 | 잡음 | 흐림 |

**EVM (Error Vector Magnitude)**

```
EVM = |오차 벡터| / |이상 신호| × 100%

이상점 ●
        ╲
         ╲ 오차 벡터
          ╲
           ● 실제점

EVM이 작을수록 좋음
WiFi 6: EVM ≤ -35dB
```

**핵심 알고리즘**: 수신 신호 → I/Q 복조 → 성상도 표시 → EVM 계산

**📢 섹션 요약 비유**: EVM < -30dB가 양호!
