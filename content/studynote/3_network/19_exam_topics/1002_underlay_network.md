# Underlay Network - 언더레이 네트워크

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 물리적 기반 네트워크

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Spine** | 코어 스위치 |
| **Leaf** | 액세스 스위치 |
| **ECMP** | 등가 다중 경로 |
| **BGP** | 라우팅 프로토콜 |
| **IP Fabric** | IP 패브릭 |

**핵심 알고리듬:** IP 라우팅 → ECMP 로드밸런싱 → 목적지 도달
