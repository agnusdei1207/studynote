# Object Pool - 오브젝트 풀

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 객체 재사용
> 2. **가치**: 생성 비용 절감, GC 감소
> 3. **융합:** Flyweight, Memory Management, Performance

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Object Pool은 객체를 재사용합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Object Pool                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              오브젝트 풀 패턴                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Without Pool:                                        │   │ │
│   │   │  new Object() → Use → GC                              │   │ │
│   │   │  (반복 생성/소멸 = 비효율)                              │   │ │
│   │   │                                                      │   │ │
│   │   │  With Pool:                                           │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │          Object Pool                        │    │   │ │
│   │   │  │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐   │    │   │ │
│   │   │  │  │ Obj │ │ Obj │ │ Obj │ │ Obj │ │ Obj │   │    │   │ │
│   │   │  │  │  1  │ │  2  │ │  3  │ │  4  │ │  5  │   │    │   │ │
│   │   │  │  └─────┘ └─────┘ └─────┘ └─────┘ └─────┘   │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       ↑ Borrow      │ Return (reset)                │   │ │
│   │   │       │              ↓                              │   │ │
│   │   │  Client: borrow() → use → return()                  │   │ │
│   │   │                                                      │   │ │
│   │   │  사용 사례:                                           │   │ │
│   │   │  - Database Connection                               │   │ │
│   │   │  - Thread                                           │   │ │
│   │   │  - Buffer                                           │   │ │
│   │   │  - Expensive Objects                                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 오브젝트 풀은 비용이 큰 객체를 재사용합니다.

**📢 섹션 요약 비유:** Object Pool = 도서관!
