# FSK (주파수 편이 변조)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 반송파의 주파수로 디지털 데이터 표현
> 2. **가치**: 잡음 내성 강함, 구현 용이
> 3. **융합**: BFSK, GFSK, 블루투스, 모뎀과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

FSK(Frequency Shift Keying)는 디지털 데이터에 따라 반송파의 주파수를 변화시키는 디지털 변조 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FSK (주파수 편이 변조)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FSK 기본 원리                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  이진 FSK (Binary FSK, BFSK):                          │   │ │
│   │   │  • 0: 주파수 f₁                                       │   │ │
│   │   │  • 1: 주파수 f₂                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  파형:                                                 │   │ │
│   │   │  데이터:    1   0   1   1   0                         │   │ │
│   │   │             ╱╲  ╱╲  ╱╲  ╱╲  ╱╲                        │   │ │
│   │   │           ╱  ╲╱  ╲╱  ╲╱  ╲╱  ╲                       │   │ │
│   │   │          f₂  f₁  f₂  f₂  f₁                          │   │ │
│   │   │          (고주파) (저주파)                             │   │ │
│   │   │                                                      │   │ │
│   │   │  주파수 편이: Δf = |f₂ - f₁|                         │   │ │
│   │   │                                                      │   │ │
│   │   │  변조 지수: h = (f₂ - f₁) / R                        │   │ │
│   │   │  (R = 비트 전송률)                                     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FSK 변형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  GFSK (Gaussian FSK):                                 │   │ │
│   │   │  • 가우시안 필터 적용                                   │   │ │
│   │   │  • 스펙트럼 확산 감소                                   │   │ │
│   │   │  • 블루투스 사용                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  M-FSK (M-ary FSK):                                   │   │ │
│   │   │  • 4-FSK: 4가지 주파수                                │   │ │
│   │   │  • 심볼 당 2비트 전송                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  MSK (Minimum Shift Keying):                          │   │ │
│   │   │  • h = 0.5인 특수 FSK                                 │   │ │
│   │   │  • 위상 연속                                          │   │ │
│   │   │  • GSM 사용                                           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: FSK는 잡음 내성이 강해 저속 무선 통신에 적합합니다.

**💡 비유**: FSK는 두 개의 라디오 방송국을 전환하는 것과 같습니다.

**📢 섹션 요약 비유**: 0은 저주파, 1은 고주파!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **f₁** | 0 주파수 | 저주파 | 낮음 |
| **f₂** | 1 주파수 | 고주파 | 높음 |
| **Δf** | 주파수 편이 | 차이 | 간격 |
| **h** | 변조 지수 | Δf/R | 비율 |

**수학적 표현**

```
s(t) = A·cos(2πf₁t)  (비트 0)
s(t) = A·cos(2πf₂t)  (비트 1)

대역폭 = |f₂ - f₁| + 2R
```

**핵심 알고리즘**: 데이터 → 주파수 선택 → 반송파 생성 → 전송

**📢 섹션 요약 비유**: GFSK는 블루투스 표준!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 디지털 변조 대역폭**

| 변조 | 대역폭 | 잡음 내성 |
|:---|:---:|:---:|
| **ASK** | 좁음 | 약함 |
| **FSK** | 넓음 | 강함 |
| **PSK** | 좁음 | 강함 |

**📢 섹션 요약 비유**: FSK는 대역폭이 넓어요!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 블루투스** → GFSK
**시나리오 2: 모뎀** → FSK (V.21)
**시나리오 3: GSM** → GMSK

**도입 체크리스트**: 대역폭, 잡음 환경, 전력

**📢 섹션 요약 비유**: 저속/저전력은 FSK!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 잡음 내성, 구현 용이

**미래 전망**: IoT/저전력 통신

**참고 표준**: Bluetooth, DECT, GSM

**📢 섹션 요약 비유**: FSK는 무선의 기본!
