# DHCP DORA - DHCP DORA 프로세스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IP 할당 4단계
> 2. **가치**: 자동 IP 설정
> 3. **융합:** Discover, Offer, Request, Ack

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DORA는 DHCP IP 할당의 4단계 과정입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DHCP DORA                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DORA 과정                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  DHCP 할당 과정:                                         │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Client                    DHCP Server          ││   │ │
│   │   │  │    │                            │               ││   │ │
│   │   │  │    │  1. DISCOVER (Broadcast)   │               ││   │ │
│   │   │  │    │  "IP 주소 있나요?"          │               ││   │ │
│   │   │  │    │ ─────────────────────────→│               ││   │ │
│   │   │  │    │                            │               ││   │ │
│   │   │  │    │  2. OFFER (Unicast)        │               ││   │ │
│   │   │  │    │  "192.168.1.100 쓰세요"    │               ││   │ │
│   │   │  │    │ ←───────────────────────── │               ││   │ │
│   │   │  │    │                            │               ││   │ │
│   │   │  │    │  3. REQUEST (Broadcast)    │               ││   │ │
│   │   │  │    │  "192.168.1.100 쓸게요"    │               ││   │ │
│   │   │  │    │ ─────────────────────────→│               ││   │ │
│   │   │  │    │                            │               ││   │ │
│   │   │  │    │  4. ACK (Unicast)          │               ││   │ │
│   │   │  │    │  "확인, 24시간 사용 가능"    │               ││   │ │
│   │   │  │    │ ←───────────────────────── │               ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  할당 정보:                                              │   │ │
│   │   │  - IP 주소                                               │   │ │
│   │   │  - 서브넷 마스크                                          │   │ │
│   │   │  - 게이트웨이                                             │   │ │
│   │   │  - DNS 서버                                               │   │ │
│   │   │  - 임대 시간                                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: DORA는 자동 IP 할당의 표준입니다.

**📢 섹션 요약 비유:** DORA = 집 계약 과정!
