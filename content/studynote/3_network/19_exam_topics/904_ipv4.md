# IPv4 - 인터넷 프로토콜 v4

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 32비트 주소 체계
> 2. **가치**: 인터넷 기반 프로토콜
> 3. **융합:** Header, Fragmentation, TTL

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IPv4는 32비트 주소를 사용하는 인터넷 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IPv4                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IPv4 헤더                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Header 구조 (20 Bytes + Options):                     │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 0-3: Version (4)    4-7: IHL (Header Length)   ││   │ │
│   │   │  │ 8-15: TOS/DS        16-31: Total Length         ││   │ │
│   │   │  │ Identification     Flags  Fragment Offset       ││   │ │
│   │   │  │ TTL                Protocol  Header Checksum    ││   │ │
│   │   │  │ Source IP Address (32 bits)                     ││   │ │
│   │   │  │ Destination IP Address (32 bits)                ││   │ │
│   │   │  │ Options (Variable)  Padding                     ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  주소 형식:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 32비트 = 4 octets                                ││   │ │
│   │   │  │ 예: 192.168.1.1                                  ││   │ │
│   │   │  │     11000000.10101000.00000001.00000001          ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  주소 공간: 2^32 ≈ 43억 개                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IPv4는 인터넷의 기본 프로토콜입니다.

**📢 섹션 요약 비유:** IPv4 = 인터넷의 전화번호!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Version** | 버전 (4) |
| **TTL** | 생존 시간 |
| **Protocol** | 상위 프로토콜 |
| **Checksum** | 오류 검사 |
