# SIMD - SIMD

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단일 명령 다중 데이터
> 2. **가치**: 병렬 데이터 처리
> 3. **융합:** DLP, Vector, AVX

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SIMD는 하나의 명령으로 여러 데이터를 처리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SIMD                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SIMD 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  SISD:               SIMD:                           │   │ │
│   │   │  ┌─────────┐         ┌─────────────────────────────┐│   │ │
│   │   │  │  A + B  │         │ A1+A2+A3+A4 + B1+B2+B3+B4   ││   │ │
│   │   │  │  = C    │         │ = C1+C2+C3+C4               ││   │ │
│   │   │  └─────────┘         └─────────────────────────────┘│   │ │
│   │   │  1 operation         1 operation, 4 results          │   │ │
│   │   │                                                      │   │ │
│   │   │  128-bit SIMD (4 x 32-bit):                          │   │ │
│   │   │  ┌───────┬───────┬───────┬───────┐                  │   │ │
│   │   │  │  32b  │  32b  │  32b  │  32b  │                  │   │ │
│   │   │  └───────┴───────┴───────┴───────┘                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SIMD는 멀티미디어, 과학 계산에 효과적입니다.

**📢 섹션 요약 비유:** SIMD = 한 번에 여러 개!
