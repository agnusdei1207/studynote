# 모노폴 안테나 (Monopole Antenna) - 1/4파장/휘핑

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 접지면 위 단일 도체 안테나
> 2. **가치**: 구조 단순, 소형화 용이
> 3. **융합**: 이동통신, 자동차와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

모노폴 안테나(Monopole Antenna)는 접지면(Ground Plane) 위에 수직으로 세워진 단일 도체 안테나입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    모노폴 안테나 (Monopole Antenna)                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              1/4파장 모노폴 구조                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │                  λ/4                                 │   │ │
│   │   │              ┌──────┐                               │   │ │
│   │   │              │      │                               │   │ │
│   │   │              │      │  ← 수직 도체                   │   │ │
│   │   │              │      │                               │   │ │
│   │   │              │  ○   │  ← 급전점                      │   │ │
│   │   │   ───────────┴──────┴───────────                    │   │ │
│   │   │              ═══════════════                         │   │ │
│   │   │               접지면 (Ground Plane)                  │   │ │
│   │   │                                                      │   │ │
│   │   │  특성:                                                 │   │ │
│   │   │  • 길이 = λ/4                                         │   │ │
│   │   │  • 이득 = 2.15 dBi (이미지 포함 5.15 dBi)             │   │ │
│   │   │  • 임피던스 ≈ 36Ω                                     │   │ │
│   │   │  • 수직 편파                                          │   │ │
│   │   │  • 전방향 패턴 (수평면)                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              모노폴 변형                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 휘핑 안테나 (Whip Antenna):                         │   │ │
│   │   │     • 가요성 모노폴                                     │   │ │
│   │   │     • 자동차, 휴대용                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 안테나 군 (Ground Plane Antenna):                   │   │ │
│   │   │     • 4개 이상의 라디얼                                │   │ │
│   │   │     • 이동통신 기지국                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 로드 안테나 (Rod Antenna):                          │   │ │
│   │   │     • 코일로 길이 단축                                  │   │ │
│   │   │     • 휴대전화 내장                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 안대나 (Inverted-F Antenna):                        │   │ │
│   │   │     • F자 형태                                        │   │ │
│   │   │     • PCB 내장형                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 모노폴은 접지면에 의한 이미지 효과로 반파장 다이폴과 유사한 특성을 가집니다.

**💡 비유**: 모노폴은 접지면이 거울이 되어 다이폴처럼 동작합니다.

**📢 섹션 요약 비유:** 모노폴 = 접지 + 이미지 = 다이폴!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 형태 | 길이 | 이득 | 용도 |
|:---|:---:|:---:|:---|
| **1/4파장** | λ/4 | 2.15 dBi | 일반 |
| **휘핑** | λ/4~λ/2 | 2-5 dBi | 차량 |
| **IFA** | < λ/4 | 0-2 dBi | 모바일 |
| **PIFA** | < λ/4 | 0-3 dBi | 모바일 |

**비교표**

```
+------------+----------+----------+
| 항목       │ 다이폴   │ 모노폴   │
+------------+----------+----------+
| 도체 수    │ 2        │ 1        │
│ 접지면     │ 불필요   │ 필수     │
│ 길이       │ λ/2      │ λ/4      │
│ 임피던스   │ 73Ω      │ 36Ω      │
+------------+----------+----------+
```

**핵심 알고리즸:** 이미지 이론, 임피던스 정합

**📢 섹션 요약 비유:** 이미지 이론 = 거울 효과!
