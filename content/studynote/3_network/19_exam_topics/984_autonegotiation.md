# Auto Negotiation - 자동 협상

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 속도/이중화 자동 설정
> 2. **가치**: 플러그 앤 플레이
> 3. **융합:** Speed, Duplex, NWay

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Auto Negotiation은 링크 속도와 이중화를 자동으로 설정합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Auto Negotiation                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              자동 협상 과정                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  협상 과정:                                               │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Device A                    Device B            ││   │ │
│   │   │  │  (10/100/1000)              (100/1000)           ││   │ │
│   │   │  │     │                           │                ││   │ │
│   │   │  │     │←──── FLP 교환 ────────────│                ││   │ │
│   │   │  │     │     (속도/이중화 정보)       │                ││   │ │
│   │   │  │     │                           │                ││   │ │
│   │   │  │     └─── 최고 공통: 1000M Full ──┘               ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  우선순위 (높은 것부터):                           ││   │ │
│   │   │  │  1. 1000M Full-Duplex                           ││   │ │
│   │   │  │  2. 1000M Half-Duplex                           ││   │ │
│   │   │  │  3. 100M Full-Duplex                            ││   │ │
│   │   │  │  4. 100M Half-Duplex                            ││   │ │
│   │   │  │  5. 10M Full-Duplex                             ││   │ │
│   │   │  │  6. 10M Half-Duplex                             ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  주의사항:                                              │   │ │
│   │   │  - 한쪽만 자동협상 시 이중화 불일치 발생 가능              │   │ │
│   │   │  - 권장: 양쪽 모두 자동협상 또는 수동 설정                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Auto Negotiation은 네트워크 설정을 자동화합니다.

**📢 섹션 요약 비유:** Auto Negotiation = 핸드셰이크!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **FLP** | Fast Link Pulse |
| **Speed** | 속도 |
| **Duplex** | 이중화 |
| **Priority** | 우선순위 |
| **Link** | 링크 |

**핵심 알고리듬:** FLP 교환 → 공통 최고 속도 선택

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**이중화 불일치 문제**

| 설정 | 결과 |
|:---|:---|
| **Auto - Auto** | 정상 |
| **Auto - Manual Full** | 충돌 다발 |
| **Manual - Manual** | 정상 |
