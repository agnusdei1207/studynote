# Multiplexing - 멀티플렉싱

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 다중 신호 전송
> 2. **가치**: 자원 효율성
> 3. **융합:** FDM, TDM, MIMO

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Multiplexing은 하나의 매체로 여러 신호를 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Multiplexing                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              멀티플렉싱 유형                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  FDM (Frequency Division):                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 채널1 ──┐                                        ││   │ │
│   │   │  │ 채널2 ──┼──→ [주파수 분할] ──→ 전송               ││   │ │
│   │   │  │ 채널3 ──┘                                        ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  TDM (Time Division):                                   │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 채널1 ──┐                                        ││   │ │
│   │   │  │ 채널2 ──┼──→ [시간 분할] ──→ 전송                 ││   │ │
│   │   │  │ 채널3 ──┘    슬롯 1 2 3 1 2 3                     ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  WDM (Wavelength Division):                             │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 파장1 (λ1) ──┐                                   ││   │ │
│   │   │  │ 파장2 (λ2) ──┼──→ 광섬유 ──→ 전송                 ││   │ │
│   │   │  │ 파장3 (λ3) ──┘    (DWDM)                         ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  네트워크에서:                                           │   │ │
│   │   │  - 포트 멀티플렉싱: 여러 포트 → 하나의 IP                 │   │ │
│   │   │  - HTTP/2: 스트림 멀티플렉싱                              │   │ │
│   │   │  - MIMO: 다중 안테나                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 멀티플렉싱은 통신 효율성을 높입니다.

**📢 섹션 요약 비유:** Multiplexing = 하나의 도로에 여러 차선!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)
