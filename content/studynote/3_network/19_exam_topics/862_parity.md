# Parity - 패리티

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 비트 수 확인
> 2. **가치**: 간단한 오류 감지
> 3. **융합:** Even Parity, Odd Parity, RAID

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Parity는 비트 수로 오류를 감지합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Parity                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              패리티 방식                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Even Parity (짝수 패리티):                             │   │ │
│   │   │  1의 개수가 짝수가 되도록 패리티 비트 설정               │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ Data: 1 0 1 1 │ Parity: 1 │ (1이 4개 = 짝수)   │  │   │ │
│   │   │  │ Data: 1 0 0 1 │ Parity: 0 │ (1이 2개 = 짝수)   │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  Odd Parity (홀수 패리티):                              │   │ │
│   │   │  1의 개수가 홀수가 되도록 패리티 비트 설정               │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ Data: 1 0 1 1 │ Parity: 0 │ (1이 4개 → 0으로)  │  │   │ │
│   │   │  │ Data: 1 0 0 1 │ Parity: 1 │ (1이 2개 → 1로)    │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  2차원 패리티 (Parity Check Matrix):                    │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │     C1  C2  C3  C4  │ Row Parity               ││   │ │
│   │   │  │ R1   1   0   1   1  │    1                     ││   │ │
│   │   │  │ R2   0   1   0   1  │    0                     ││   │ │
│   │   │  │ Col  1   1   1   0  │                          ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │  → 위치 식별 가능                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  한계:                                                 │   │ │
│   │   │  - 짝수 개 오류 감지 불가                               │   │ │
│   │   │  - 수정 불가 (위치 모름)                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 패리티는 가장 간단한 오류 감지 방법입니다.

**📢 섹션 요약 비유:** Parity = 짝수/홀수 맞추기!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Even** | 짝수 패리티 |
| **Odd** | 홀수 패리티 |
| **XOR** | 배타적 논리합 |
| **Parity Bit** | 패리티 비트 |
| **Check** | 검사 |

**핵심 알고리즸:** 비트 XOR → 패리티 생성 → 검사

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**오류 감지 방식 비교**

| 방식 | 오버헤드 | 검출력 |
|:---|:---|:---|
| **1-bit Parity** | 1비트 | 홀수 개 |
| **2D Parity** | 높음 | 높음 |
| **CRC** | 가변 | 높음 |
| **Checksum** | 16비트 | 중간 |

**시나리오 3:** RAID → Parity
