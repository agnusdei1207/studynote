# 플로우 컨트롤 - Flow Control

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 수신자 속도에 맞춘 전송 제어
> 2. **가치**: 버퍼 오버플로우 방지
> 3. **융합**: TCP 윈도우, 802.3x, XON/XOFF

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

플로우 컨트롤은 수신자를 보호합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    플로우 컨트롤                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              플로우 컨트롤 유형                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 유형       │ 설명     │ 계층     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ TCP Window │ 윈도우   │ 전송     │               │   │ │
│   │   │  │ 802.3x     │ Pause    │ 링크     │               │   │ │
│   │   │  | XON/XOFF   │ 문자     │ 링크     │               │   │ │
│   │   │  │ Buffer     │ 큐       │ 장비     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 플로우 컨트롤은 수신자 버퍼를 보호합니다.

**📢 섹션 요약 비유:** Flow Control = 배수로!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Sliding Window** | TCP 윈도우 |
| **Pause Frame** | 802.3x |
| **Zero Window** | TCP 수신 윈도우 0 |
| **Backpressure** | 역압력 |

**핵심 알고리즸:** 윈도우 크기 조절

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Flow Control vs Congestion Control**

| 항목 | Flow Control | Congestion Control |
|:---|:---|:---|
| **목적** | 수신자 보호 | 네트워크 보호 |
| **위치** | End-to-End | End-to-Network |
| **신호** | 윈도우 | 손실/지연 |
