# Path MTU Discovery (PMTUD) - 경로 MTU 탐색

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 경로상 최소 MTU 탐색
> 2. **가치**: 단편화 방지
> 3. **융합:** ICMP, DF, MTU

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PMTUD는 경로상 최소 MTU를 탐색합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Path MTU Discovery                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PMTUD 동작                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  송신측 (MTU 1500) ──→ 라우터 (MTU 1400) ──→ 수신측 │   │ │
│   │   │         │                    │                       │   │ │
│   │   │         │ ── 패킷 1500 ────→ │                       │   │ │
│   │   │         │    (DF=1)          │ MTU 초과!             │   │ │
│   │   │         │                    │                       │   │ │
│   │   │         │ ←── ICMP "Too Big" │ (MTU=1400)            │   │ │
│   │   │         │                    │                       │   │ │
│   │   │         │ ── 패킷 1400 ────→ │ ──────────────────→   │   │ │
│   │   │         │                    │                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: PMTUD는 ICMP 메시지로 MTU를 조정합니다.

**📢 섹션 요약 비유:** PMTUD = 길 찾기!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **DF=1** | 단편화 금지 |
| **ICMP Type 3** | Destination Unreachable |
| **Code 4** | Fragmentation Needed |
| **Next-Hop MTU** | 제안 MTU |

**핵심 알고리즸:** DF=1 전송 → ICMP 수신 → MTU 조정
