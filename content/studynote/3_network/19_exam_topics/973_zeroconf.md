# ZeroConf - 제로 설정

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 자동 네트워크 구성
> 2. **가치**: 수동 설정 불필요
> 3. **융합:** mDNS, APIPA, Bonjour

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ZeroConf는 수동 설정 없이 네트워크를 구성합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ZeroConf                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ZeroConf 기술                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ZeroConf 요구사항:                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  1. IP 주소 자동 할당                             ││   │ │
│   │   │  │     APIPA (169.254.x.x)                         ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  2. 이름 해석                                    ││   │ │
│   │   │  │     mDNS (Multicast DNS)                        ││   │ │
│   │   │  │     .local 도메인                                ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  3. 서비스 검색                                   ││   │ │
│   │   │  │     DNS-SD (DNS Service Discovery)              ││   │ │
│   │   │  │     프린터, 파일 공유 등                           ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  구현체:                                                 │   │ │
│   │   │  - Apple Bonjour                                       │   │ │
│   │   │  - Avahi (Linux)                                       │   │ │
│   │   │  - mDNSResponder                                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: ZeroConf는 사용자 친화적 네트워킹입니다.

**📢 섹션 요약 비유:** ZeroConf = 플러그 앤 플레이!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **mDNS** | 멀티캐스트 DNS |
| **DNS-SD** | 서비스 검색 |
| **APIPA** | 자동 IP |
| **Bonjour** | 애플 구현 |
| **Avahi** | 리눅스 구현 |

**핵심 알고리듬:** IP 할당 → 이름 등록 → 서비스 광고
