# 평행 2선식 케이블 (Twin-lead Cable)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 두 평행 도선으로 구성된 전송선
> 2. **가치**: 저비용, 간단한 구조
> 3. **융합**: 안테나 급전선, TV 수신

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

평행 2선식 케이블(Twin-lead Cable)은 두 개의 평행한 도선을 일정 간격으로 유지하여 구성된 전송선로입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    평행 2선식 케이블 (Twin-lead Cable)              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              구조 및 특징                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  구조:                                                   │   │ │
│   │   │                                                      │   │ │
│   │   │       절연체                                           │   │ │
│   │   │    ┌──────────────────────────┐                      │   │ │
│   │   │    │    │    │    │    │     │                      │   │ │
│   │   │    │  ● │    │    │  ● │     │  ← 도선 (평행)        │   │ │
│   │   │    │    │    │    │    │     │                      │   │ │
│   │   │    └──────────────────────────┘                      │   │ │
│   │   │         ↑              ↑                             │   │ │
│   │   │       도선1          도선2                            │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 평형 전송선 (Balanced Line)                         │   │ │
│   │   │  • 특성 임피던스: 300Ω (TV용), 450Ω, 600Ω            │   │ │
│   │   │  • 저비용, 가벼움                                      │   │ │
│   │   │  • 노이즈에 취약                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 저비용                                               │   │ │
│   │   │  • 낮은 손실 (VHF/UHF 대역)                             │   │ │
│   │   │  • 유연성                                               │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                   │   │ │
│   │   │  • 전자기 간섭 취약                                     │   │ │
│   │   │  • 임피던스 정합 어려움                                  │   │ │
│   │   │  • 습기에 민감                                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              응용 분야                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  • TV 안테나 급전선 (300Ω 리본 케이블)                 │   │ │
│   │   │  • 아마추어 무선                                        │   │ │
│   │   │  • 단파 수신                                            │   │ │
│   │   │  • 쌍극자 안테나 연결                                    │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 평행 2선식 케이블은 주로 TV 안테나 연결에 사용되는 저비용 전송선입니다.

**💡 비유**: 평행 2선식은 두 발 자전거 도로입니다.

**📢 섹션 요약 비유**: 평행 2선 = 300Ω!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **도선** | 신호 전달 | 구리선 | 차선 |
| **절연체** | 간격 유지 | 플라스틱 | 중앙선 |
| **임피던스** | 정합 | 300Ω | 규격 |

**비교표**

```
+------------+----------+----------+
| 항목       | 평행 2선 | 동축     |
+------------+----------+----------+
| 임피던스   | 300Ω     | 50/75Ω   |
| 차폐       | 없음     | 있음     |
| 노이즈     | 취약     | 강함     |
| 비용       | 낮음     | 높음     |
+------------+----------+----------+
```

**핵심 알고리즸**: 평형 신호 전송, 차동 모드

**📢 섹션 요약 비유**: 평행 2선 = 평형 전송!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 전송선 종류**

| 전송선 | 임피던스 | 용도 |
|:---|:---:|:---|
| **Twin-lead** | 300Ω | TV |
| **동축** | 75Ω | CATV |
| **꼬임쌍선** | 100Ω | LAN |

**📢 섹션 요약 비유**: TV 안테나는 평행 2선!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: TV 안테나** → 평행 2선
**시나리오 2: 노이즈 환경** → 동축 케이블
**시나리오 3: 디지털 전송** → 꼬임쌍선/동축

**도입 체크리스트**: 노이즈 환경, 거리, 임피던스 정합

**📢 섹션 요약 비유**: 노이즈 없는 곳 = 평행 2선!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 저비용 안테나 연결

**미래 전망**: 디지털 방송으로 동축/광섬유로 대체

**참고 표준**: IEC 60728

**📢 섹션 요약 비유**: 평행 2선은 아날로그 시대!
