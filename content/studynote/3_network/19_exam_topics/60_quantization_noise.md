# 양자화 잡음 (Quantization Noise)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 양자화 과정에서 발생하는 필연적 오차
> 2. **가치**: 디지털 시스템 성능의 기본 한계
> 3. **융합**: SNR, 비트 깊이, 컴팬딩과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

양자화 잡음(Quantization Noise)은 연속 신호를 이산 레벨로 근사할 때 발생하는 오차입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    양자화 잡음 (Quantization Noise)                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              양자화 오차 발생                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  원본 값:       3.7                                   │   │ │
│   │   │  양자화 값:     4 (반올림)                             │   │ │
│   │   │  양자화 오차:   |3.7 - 4| = 0.3                        │   │ │
│   │   │                                                      │   │ │
│   │   │  그래프:                                                │   │ │
│   │   │  원본:  ╱╲                                            │   │ │
│   │   │        ╱  ╲                                           │   │ │
│   │   │  양자화: ──── (계단)                                   │   │ │
│   │   │            ↑                                          │   │ │
│   │   │          오차 (에러)                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  특성:                                                 │   │ │
│   │   │  • 최대 오차: ±Δ/2 (Δ = 스텝 크기)                     │   │ │
│   │   │  • 백색 잡음처럼 분포                                  │   │ │
│   │   │  • 제거 불가능 (필연적)                                │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              양자화 SNR                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  선형 양자화 SNR:                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  SNR = 6.02·n + 1.76 (dB)                             │   │ │
│   │   │  (n = 비트 수)                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  경험칙: 비트당 6dB 개선                                │   │ │
│   │   │                                                      │   │ │
│   │   │  예:                                                   │   │ │
│   │   │  • 8비트:  6.02×8 + 1.76 ≈ 50dB                       │   │ │
│   │   │  • 16비트: 6.02×16 + 1.76 ≈ 98dB                      │   │ │
│   │   │  • 24비트: 6.02×24 + 1.76 ≈ 146dB                     │   │ │
│   │   │                                                      │   │ │
│   │   │  비선형 양자화 (μ-law/A-law):                          │   │ │
│   │   │  • 작은 신호 SNR 개선                                  │   │ │
│   │   │  • 전체 범위에서 균등 SNR                               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 양자화 잡음은 줄일 수 있지만 완전히 제거할 수 없습니다.

**💡 비유**: 양자화 잡음은 디지털 사진의 픽셀화와 같습니다.

**📢 섹션 요약 비유**: 비트↑ = 잡음↓ = 6dB/비트!

### Ⅱ. 아키틤텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **Δ** | 스텝 크기 | Vpp/L | 간격 |
| **e** | 오차 | ±Δ/2 | 에러 |
| **SQNR** | 신호 대 잡음 | dB | 품질 |
| **n** | 비트 수 | 해상도 | 정밀도 |

**비교표**

```
+------------+----------+----------+----------+
| 비트       | 레벨     | 최대오차 | SNR      |
+------------+----------+----------+----------+
| 8-bit      | 256      | Vpp/512  | 50dB     |
| 12-bit     | 4096     | Vpp/8192 | 74dB     |
| 16-bit     | 65536    | Vpp/131K | 98dB     |
| 24-bit     | 16M      | Vpp/33M  | 146dB    |
+------------+----------+----------+----------+
```

**핵심 알고리즸**: SNR = 6.02n + 1.76 dB

**📢 섹션 요약 비유**: 16비트 CD는 98dB!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 잡음 유형**

| 잡음 | 원인 | 해결 |
|:---|:---|:---|
| **양자화** | 이산화 | 비트 증가 |
| **열잡음** | 물리 | 냉각 |
| **왜곡** | 비선형 | 선형화 |

**📢 섹션 요약 비유**: 디더링으로 저레벨 개선!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 전화** → 8비트 + μ-law (약 38dB)
**시나리오 2: CD** → 16비트 (98dB)
**시나리오 3: 스튜디오** → 24비트 (146dB)

**도입 체크리스트**: 동적 범위, 잡음 플로어, 비용

**📢 섹션 요약 비유**: 사람 귀는 약 120dB 인지!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 디지털 품질 한계 결정

**미래 전망**: 32비트 float, 하이브리드

**참고 표준**: ITU-T G.711

**📢 섹션 요약 비유**: 양자화 잡음은 디지털의 본질적 한계!
