# 광무선 통신 (OWC) / FSO

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 광대역 무선 광통신
> 2. **가치**: 면허 불필요, 고속, 보안
> 3. **융합**: 위성, 백홀, 우주와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

광무선 통신(OWC, Optical Wireless Communication)은 적외선, 가시광, 자외선을 이용한 무선 통신입니다. FSO(Free Space Optics)는 공간을 통한 광통신입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    광무선 통신 (OWC / FSO)                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OWC 분류                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  OWC (Optical Wireless Communication)                │   │ │
│   │   │     │                                                │   │ │
│   │   │     ├── 적외선 (IR) ──────── IrDA, 리모컨             │   │ │
│   │   │     │                                                │   │ │
│   │   │     ├── 가시광 (VLC) ────── Li-Fi, LED 통신           │   │ │
│   │   │     │                                                │   │ │
│   │   │     └── FSO (Free Space Optics)                      │   │ │
│   │   │          │                                           │   │ │
│   │   │          ├── 지상 FSO: 건물 간 통신                   │   │ │
│   │   │          ├── 위성 FSO: 지구-위성                      │   │ │
│   │   │          └── 우주 FSO: 위성-위성                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FSO 특성                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 고속 (수 Gbps)                                        │   │ │
│   │   │  • 면허 불필요                                           │   │ │
│   │   │  • EMI 없음                                              │   │ │
│   │   │  • 보안성 높음                                            │   │ │
│   │   │  • 설치 용이                                              │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                   │   │ │
│   │   │  • 기상 영향 (안개, 비, 눈)                               │   │ │
│   │   │  • LOS 필수                                               │   │ │
│   │   │  • 정렬 민감                                              │   │ │
│   │   │  • 거리 제한 (수 km)                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  응용:                                                   │   │ │
│   │   │  • 건물 간 백홀                                           │   │ │
│   │   │  • 재난 복구                                              │   │ │
│   │   │  • 위성 광통신                                            │   │ │
│   │   │  • 우주 인터넷                                            │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: OWC는 전파 대역 혼잡을 해결하는 대안 기술입니다.

**💡 비유**: FSO는 두 건물 사이에 보이지 않는 광케이블을 연결하는 것입니다.

**📢 섹션 요약 비유:** FSO = 무선 광케이블!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **광송신기** | 신호 변환 | E-O 변환 | 레이저 |
| **광학계** | 빔 제어 | 렌즈/미러 | 망원경 |
| **광수신기** | 신호 검출 | O-E 변환 | 포토다이오드 |
| **PAT** | 추적 | 포인팅/획득 | 추적 시스템 |

**비교표**

```
+------------+----------+----------+
| 항목       | RF       | FSO      |
+------------+----------+----------+
| 대역폭     │ 수GHz    │ 수THz    │
| 면허       │ 필요     │ 불필요   │
| 기상 영향   │ 적음     │ 큼       │
| 보안       │ 낮음     │ 높음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 빔 정렬, PAT (Pointing, Acquisition, Tracking)

**📢 섹션 요약 비유:** PAT = FSO의 핵심!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 백홀 기술**

| 기술 | 속도 | 거리 | 기상 영향 |
|:---|:---:|:---:|:---:|
| **마이크로파** | 1Gbps | 10km | 작음 |
| **FSO** | 10Gbps | 2km | 큼 |
| **광섬유** | 100Gbps | 80km | 없음 |

**📢 섹션 요약 비유:** FSO + RF = 하이브리드!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 임시 링크** → FSO
**시나리오 2: 재난 복구** → FSO
**시나리오 3: 위성 통신** → FSO + RF 하이브리드

**도입 체크리스트:** LOS, 기상, 대역폭

**📢 섹션 요약 비유:** FSO는 틈새 해결사!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 면허 없는 고속 백홀

**미래 전망:** 우주 인터넷 (Starlink 광링크)

**참고 표준:** IEEE 802.15.7, ITU-T G.9991

**📢 섹션 요약 비유:** FSO는 우주로!
