# XSS (Cross-Site Scripting) - 크로스 사이트 스크립팅

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 악의적 스크립트 주입
> 2. **가치**: 세션 탈취, 피싱
> 3. **융합:** OWASP, CSP, Input Sanitization

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

XSS는 웹페이지에 악성 스크립트를 주입합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    XSS                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              XSS 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Reflected XSS:                                      │   │ │
│   │   │  URL 파라미터 → 즉시 반영                            │   │ │
│   │   │  ?search=<script>alert(1)</script>                  │   │ │
│   │   │                                                      │   │ │
│   │   │  Stored XSS:                                         │   │ │
│   │   │  DB 저장 → 다른 사용자에게 표시                       │   │ │
│   │   │  댓글, 게시판 등                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  DOM-based XSS:                                      │   │ │
│   │   │  클라이언트 사이드 JavaScript 조작                   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: XSS는 쿠키 탈취, 피싱 등에 악용됩니다.

**📢 섹션 요약 비유:** XSS = 위장 우편!

| 유형 | 설명 |
|:---|:---|
| **Reflected** | 즉시 반영 |
| **Stored** | 저장 후 표시 |
