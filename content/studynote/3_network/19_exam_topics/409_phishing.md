# 피싱 - Phishing

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 위장을 통한 정보 탈취
> 2. **가치**: 계정 탈취, 금융 사기
> 3. **융합**: 이메일 보안, MFA, 교육

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

**💡 비유**: 피싱은 낚시입니다. 미끼를 던져 물고기를 낚습니다.

**📢 섹션 요약 비유:** 피싱 = 낚시!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Spoofing** | 발신자 위장 |
| **Clone Phishing** | 기존 메일 복제 |
| **Business Email Compromise** | 비즈니스 이메일 사기 |
| **Credential Harvesting** | 계정 수확 |

**비교표: 탐지 방식**

```
+------------+----------+----------+
| 항목       │ 기술     │ 교육     │
+------------+----------+----------+
| 필터링     │ O        │ -        │
| URL 검사   │ O        │ -        │
| 의심       │ -        │ O        │
+------------+----------+----------+
```

**핵심 알고리즸:** URL 분석, 자연어 처리

**📢 섹션 요약 비유:** MFA = 피싱 방어!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 이메일 보안 → SPF/DKIM/DMARC
**시나리오 2:** 계정 보호 → MFA
**시나리오 3:** 사용자 → 정기 교육 + 모의 훈련

**도입 체크리스트:** 이메일 인증, MFA, 교육, 신고

**📢 섹션 요약 비유:** 사람이 마지막 방어선!
