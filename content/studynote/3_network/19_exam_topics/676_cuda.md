### Ⅰ. 개요 (Context & Background)

**개념 정의**

CUDA는 NVIDIA GPU를 프로그래밍하는 플랫폼입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CUDA                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CUDA 프로그래밍 모델                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Host (CPU)                  Device (GPU)            │   │ │
│   │   │  ┌─────────────────┐         ┌─────────────────────┐│   │ │
│   │   │  │ Main Code       │         │ Kernel              ││   │ │
│   │   │  │                 │  Copy   │ ┌─────────────────┐ ││   │ │
│   │   │  │ cudaMemcpy ────┼────────→│ │ Grid            │ ││   │ │
│   │   │  │                 │         │ │  ┌───────────┐  │ ││   │ │
│   │   │  │ kernel<<< >>> ──┼──Launch→│ │  │ Block     │  │ ││   │ │
│   │   │  │                 │         │ │  │ ┌───────┐  │  │ ││   │ │
│   │   │  │ cudaMemcpy ←───┼─────────│ │  │ │Thread │  │  │ ││   │ │
│   │   │  │                 │  Copy   │ │  │ └───────┘  │  │ ││   │ │
│   │   │  └─────────────────┘         │ │  └───────────┘  │ ││   │ │
│   │   │                              │ └─────────────────┘ ││   │ │
│   │   │                              └─────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: CUDA는 Grid > Block > Thread 계층으로 구성됩니다.

**📢 섹션 요약 비유:** CUDA = GPU 통역사!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Grid** | 그리드 |
| **Block** | 블록 |
| **Thread** | 스레드 |
| **Warp** | 워프 (32 스레드) |
| **Kernel** | 커널 함수 |

**핵심 알고리즸:** Host → 데이터 전송 → Kernel 실행 → 결과 전송
