# Multicast - 멀티캐스트

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 1:Group 통신
> 2. **가치**: 대역폭 효율성
> 3. **융합:** IGMP, PIM, Class D

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Multicast는 하나의 송신자에서 그룹으로 전달합니다.

```

├─────────────────────────────────────────────────────────────────────────┤
```

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Group** | 그룹 주소 |
| **IGMP** | 그룹 관리 |
| **PIM** | 라우팅 |
| **Tree** | 배포 트리 |
| **Join/Leave** | 가입/탈퇴 |

**핵심 알고리듬:** 그룹 가입 → 트리 구성 → 데이터 전달

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**통신 방식 효율성**

| 방식 | 대역폭 | 수신자 수 |
|:---|:---|:---|
| **Unicast × N** | N × 데이터 | N |
| **Broadcast** | 1 × 데이터 | 전체 |
| **Multicast** | 1 × 데이터 | 그룹 |
