# Feature Flags - 기능 플래그

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 기능 토글
> 2. **가치**: 배포와 릴리스 분리
> 3. **융합:** CI/CD, Canary, A/B Testing

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Feature Flags는 런타임에 기능을 제어합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Feature Flags                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              기능 플래그 구조                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Code:                                                │   │ │
│   │   │  if (featureFlag.isEnabled("new-ui", user)) {       │   │ │
│   │   │      showNewUI();                                    │   │ │
│   │   │  } else {                                            │   │ │
│   │   │      showOldUI();                                    │   │ │
│   │   │  }                                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  플래그 유형:                                          │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ Release Toggle: 기능 릴리스 제어               │  │   │ │
│   │   │  │ Experiment Toggle: A/B 테스트                 │  │   │ │
│   │   │  │ Ops Toggle: 운영 설정                         │  │   │ │
│   │   │  │ Permission Toggle: 권한 제어                   │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 기능 플래그는 배포와 릴리스를 분리합니다.

**📢 섹션 요약 비유:** Feature Flags = 조명 스위치!
