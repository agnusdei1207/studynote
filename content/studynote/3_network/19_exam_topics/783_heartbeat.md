### Ⅰ. 개요 (Context & Background)

**개념 정의**

Heartbeat는 노드 생존을 확인합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Heartbeat                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Heartbeat 메커니즘                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Node A              Node B                          │   │ │
│   │   │  ┌─────────┐        ┌─────────┐                     │   │ │
│   │   │  │         │ ── Ping→│         │                     │   │ │
│   │   │  │ Active  │ ←─ Pong─│ Standby │                     │   │ │
│   │   │  │         │        │         │                     │   │ │
│   │   │  └─────────┘        └─────────┘                     │   │ │
│   │   │                                                      │   │ │
│   │   │  정상: Ping/Pong 교환                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  장애:                                                │   │ │
│   │   │  Node A ──×──→ Node B (타임아웃)                       │   │ │
│   │   │          │                                           │   │ │
│   │   │          ↓ 장애 감지                                   │   │ │
│   │   │      Failover → Node B가 Active 인계                   │   │ │
│   │   │                                                      │   │ │
│   │   │  설정:                                                │   │ │
│   │   │  - Interval: 1초                                      │   │ │
│   │   │  - Timeout: 3초                                       │   │ │
│   │   │  - Retry: 3회                                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │

│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Heartbeat는 주기적으로 생존을 확인합니다.
