# MAC Address - MAC 주소 (물리 주소)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터링크 계층 식별자
> 2. **가치**: 로컬 네트워크 고유 식별
> 3. **융합:** Ethernet, ARP, Switch

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MAC 주소는 48비트 하드웨어 주소입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MAC Address                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MAC 주소 구조                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  00:1A:2B:3C:4D:5E                                   │   │ │
│   │   │  ├─────────┤├─────────┤├─────────┤                  │   │ │
│   │   │  OUI (24bit)   NIC (24bit)                          │   │ │
│   │   │  제조사 ID     일련번호                               │   │ │
│   │   │                                                      │   │ │
│   │   │  비트 구조:                                          │   │ │
│   │   │  ┌────────────────────────────────────────┐          │   │ │
│   │   │  │ I/G │ U/L │    OUI (22bit)   │ NIC (24)│          │   │ │
│   │   │  └────────────────────────────────────────┘          │   │ │
│   │   │  I/G: Individual(0)/Group(1)                        │   │ │
│   │   │  U/L: Universally(0)/Locally(1) administered        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MAC 주소는 NIC에 할당된 전역 고유 식별자입니다.

**📢 섹션 요약 비유:** MAC 주소 = 주민등록번호!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **OUI** | IEEE 할당 제조사 ID |
| **UAA** | Universal Administered |
| **LAA** | Locally Administered |
| **Unicast** | I/G=0 |
| **Multicast** | I/G=1 |
| **Broadcast** | FF:FF:FF:FF:FF:FF |

**핵심 알고리즸:** OUI + Serial = Unique

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**주소 유형**
