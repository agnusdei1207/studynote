### Ⅰ. 개요 (Context & Background)

**개념 정의**

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Hash** | 해시 함수 (SHA-256) |
| **Private Key** | 서명용 개인키 |
| **Public Key** | 검증용 공개키 |
| **Signature** | 서명값 |

**비교표: 서명 알고리즘**

```
+------------+----------+----------+
| 항목       │ RSA      │ ECDSA    │
+------------+----------+----------+
| 키 크기    │ 2048+    │ 256      │
| 속도       │ 느림     │ 빠름     │
| 서명 크기   │ 큼       │ 작음     │
| 추세       │ 레거시   │ 현대적   │
+------------+----------+----------+
```

**핵심 알고리즸:** 해시, 비대칭 암호

**📢 섹션 요약 비유:** 무결성 = 디지털 서명!
