### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Presence** | 상태 정보 |
| **IM** | 인스턴트 메시징 |
| **Video** | 비디오 회의 |
| **UM** | 통합 메시징 |

**비교표: UC 솔루션**

```
+------------+----------+----------+
| 항목       │ Teams    │ Slack    │
+------------+----------+----------+
| 음성       │ 지원     │ 제한     │
| 영상       │ 지원     │ 지원     │
| PBX 연동   │ 지원     │ 제한     │
| 엔터프라이즈│ 강함     │ 중간     │
+------------+----------+----------+
```

**핵심 알고리즸:** Presence, 라우팅

**📢 섹션 요약 비유:** Presence = 상태 공유!
