# Queue - 큐

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 순차적 데이터 구조
> 2. **가치**: 비동기 처리
> 3. **융합:** Message Queue, Priority Queue, FIFO

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Queue는 FIFO (First-In First-Out) 순서로 데이터를 처리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Queue                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              큐 유형                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  FIFO Queue:                                           │   │ │
│   │   │  Dequeue ← [A][B][C][D] ← Enqueue                     │   │ │
│   │   │                                                      │   │ │
│   │   │  Priority Queue:                                       │   │ │
│   │   │  Dequeue ← [1][2][3][4][5] ← Enqueue                  │   │ │
│   │   │          (우선순위 기준 정렬)                           │   │ │
│   │   │                                                      │   │ │
│   │   │  네트워킹에서의 큐:                                     │   │ │
│   │   │  - 송신 큐: 전송 대기 패킷                              │   │ │
│   │   │  - 수신 큐: 처리 대기 패킷                              │   │ │
│   │   │  - 라우터 큐: 출력 인터페이스 대기열                    │   │ │
│   │   │                                                      │   │ │
│   │   │  Message Queue:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Producer ──→ [Queue] ──→ Consumer              ││   │ │
│   │   │  │                                                ││   │ │
│   │   │  │ 예시: RabbitMQ, Kafka, SQS, Redis              ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 큐는 비동기 통신과 로드 밸런싱에 필수입니다.

**📢 섹션 요약 비유:** Queue = 줄서기!
