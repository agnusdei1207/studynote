# Hamming Code - 해밍 코드

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 1비트 오류 수정
> 2. **가치**: 오류 수정 + 감지
> 3. **융합:** ECC Memory, Parity, Syndrome

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Hamming Code는 1비트 오류를 수정할 수 있습니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Hamming Code                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              해밍 코드 (7,4)                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  구조:                                                 │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 위치:  1   2   3   4   5   6   7               ││   │ │
│   │   │  │ 종류:  p1  p2  d1  p3  d2  d3  d4               ││   │ │
│   │   │  │       패리티  데이터            데이터           ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  패리티 비트 계산:                                      │   │ │
│   │   │  p1: 위치 1,3,5,7 (1,3,5,7의 2^0=1 비트)              │   │ │
│   │   │  p2: 위치 2,3,6,7 (2,3,6,7의 2^1=2 비트)              │   │ │
│   │   │  p3: 위치 4,5,6,7 (4,5,6,7의 2^2=4 비트)              │   │ │
│   │   │                                                      │   │ │
│   │   │  오류 위치 식별 (Syndrome):                             │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ S = p3 p2 p1 (이진수) = 오류 위치               ││   │ │
│   │   │  │ S = 000 → 오류 없음                              ││   │ │
│   │   │  │ S = 101 → 위치 5 오류                            ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  공식:                                                 │   │ │
│   │   │  2^r ≥ m + r + 1                                      │   │ │
│   │   │  r: 패리티 비트 수, m: 데이터 비트 수                   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 해밍 코드는 ECC 메모리의 기반입니다.

**📢 섹션 요약 비유:** Hamming Code = 스도쿠!
