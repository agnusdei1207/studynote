# sFlow - 샘플드 플로우

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 샘플링 기반 트래픽 분석
> 2. **가치**: 저오버헤드 모니터링
> 3. **융합:** Sampling, Agent, Collector

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

sFlow (Sampled Flow)는 패킷 샘플링 기반 트래픽 분석입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    sFlow                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │

│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌────────────────────────────────────────────┐ ││   │ │
│   │   │  │  │           스위치 (sFlow Agent)              │ ││   │ │
│   │   │  │  │                                            │ ││   │ │
│   │   │  │  │  전체 패킷: ──────────────────────────────── │ ││   │ │
```

**📢 섹션 요약 비유:** sFlow = 설문조사!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Agent** | sFlow 에이전트 |
| **Collector** | 수집기 |
| **Sampling Rate** | 샘플링 비율 |
| **Polling Interval** | 폴링 간격 |
| **Datagram** | 데이터그램 |

**핵심 알고리듬:** N:1 샘플링 → 실시간 전송 → 통계 분석

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**sFlow vs NetFlow**

| 항목 | sFlow | NetFlow |
|:---|:---|:---|
| **방식** | 샘플링 | 플로우 캐시 |
| **오버헤드** | 낮음 | 높음 |
| **정확도** | 통계적 | 정확 |
| **실시간** | 지원 | 지연 가능 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 고속 망 (10G+) → sFlow
**시나리오 2:** 정확한 계량 → NetFlow
**시나리오 3:** DDoS 탐지 → sFlow + 분석
