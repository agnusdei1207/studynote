# 공간 분할 다중화 (SDM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 물리적으로 분리된 경로로 다중화
> 2. **가치**: 단순, 신뢰성, 독립 채널
> 3. **융합**: 케이블 TV, 다중 안테나와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SDM(Space Division Multiplexing)은 물리적으로 분리된 전송 매체를 사용하여 여러 신호를 동시에 전송하는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SDM (공간 분할 다중화)                           │
├─────────────────────────────────────────────────────────────────────────┤

│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SDM 원리                                         │ │
│   │                                                              │ │

│   │   │  ┌───┐      케이블1      ┌───┐                      │   │ │
│   │   │  │CH1├───────────────────►│CH1│                      │   │ │
│   │   │  └───┘                    └───┘                      │   │ │
│   │   │  ┌───┐      케이블2      ┌───┐                      │   │ │
│   │   │  │CH2├───────────────────►│CH2│                      │   │ │
```

> **해설**: SDM은 가장 단순한 다중화 방식입니다.

**💡 비유**: SDM은 여러 차선 도로입니다.

**📢 섹션 요약 비유**: 물리적 분리 = 간섭 없음!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **케이블** | 경로 | 물리적 분리 | 차선 |
| **코어** | 광경로 | 독립 전송 | 파이프 |
| **안테나** | 공간 분리 | 빔 형성 | 방향 |

**비교표**

```
+------------+----------+----------+
| 항목       | SDM      | 타 방식  |
+------------+----------+----------+
| 분리 방식  | 물리     | 논리     |
| 간섭       | 없음     | 가능     |
| 비용       | 높음     | 낮음     |
| 신뢰성     | 높음     | 중간     |
+------------+----------+----------+
```

**핵심 알고리즸**: N개 채널 → N개 물리 경로 → 독립 전송

**📢 섹션 요약 비유**: SDM = 간단하지만 비용↑!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: SDM vs 타 방식**

| 항목 | SDM | FDM | TDM |
|:---|:---:|:---:|:---:|
| 복잡도 | 낮음 | 중간 | 높음 |
| 비용 | 높음 | 중간 | 낮음 |
| 효율 | 낮음 | 중간 | 높음 |

**📢 섹션 요약 비유**: SDM은 기본적 다중화!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 케이블 TV** → SDM (동축 다중)
**시나리오 2: 데이터센터** → SDM (다중 광섬유)
**시나리오 3: 5G** → SDM + MIMO

**도입 체크리스트**: 케이블 수, 비용, 신뢰성

**📢 섹션 요약 비유**: 단순성과 신뢰성의 균형!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 완전한 채널 격리, 높은 신뢰성

**미래 전망**: 멀티 코어 광섬유, Massive MIMO

**참고 표준**: IEEE 802.3, 3GPP

**📢 섹션 요약 비유**: SDM은 물리적 확장!
