# 스냅샷 (Snapshot) - 시점 복사

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 특정 시점 데이터 상태
> 2. **가치**: 빠른 복구, 백업
> 3. **융합**: VM, 스토리지와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

스냅샷은 특정 시점의 데이터 상태를 보존하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    스냅샷 (Snapshot)                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              스냅샷 동작 방식                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  시간축: T0          T1          T2                  │   │ │
│   │   │          │           │           │                   │   │ │
│   │   │          ▼           ▼           ▼                   │   │ │
│   │   │  ┌───────────┐ ┌───────────┐ ┌───────────┐          │   │ │
│   │   │  │ Snapshot 1│ │ Snapshot 2│ │ Snapshot 3│          │   │ │
│   │   │  │   T0      │ │   T1      │ │   T2      │          │   │ │
│   │   │  └───────────┘ └───────────┘ └───────────┘          │   │ │
│   │   │       │             │             │                  │   │ │
│   │   │       ▼             ▼             ▼                  │   │ │
│   │   │  ┌─────────────────────────────────────────┐        │   │ │
│   │   │  │           Original Volume               │        │   │ │
│   │   │  │  ┌───┬───┬───┬───┬───┬───┬───┬───┐    │        │   │ │
│   │   │  │  │ A │ B │ C │ D │ E │ F │ G │ H │    │        │   │ │
│   │   │  │  └───┴───┴───┴───┴───┴───┴───┴───┘    │        │   │ │
│   │   │  └─────────────────────────────────────────┘        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              스냅샷 기술                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. Copy-on-Write (COW):                              │   │ │
│   │   │     • 변경 시 원본 복사                               │   │ │
│   │   │     • 공간 효율                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  2. Redirect-on-Write (ROW):                          │   │ │
│   │   │     • 변경분을 새 위치에                              │   │ │
│   │   │     • 성능 좋음                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  3. Full Clone:                                       │   │ │
│   │   │     • 전체 복사                                       │   │ │
│   │   │     • 공간 많이 사용                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 스냅샷은 CoW 또는 RoW로 공간 효율을 높입니다.

**💡 비유**: 스냅샷은 사진입니다. 그 순간을 그대로 남깁니다.

**📢 섹션 요약 비유:** 스냅샷 = 시점 보존!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **CoW** | 쓰기 시 복사 |
| **RoW** | 쓰기 시 리다이렉트 |
| **Delta** | 변경분 |
| **Revert** | 복원 |

**비교표: 스냅샷 기술**

```
+------------+----------+----------+
| 기술       │ 공간     │ 성능     │
+------------+----------+----------+
| CoW        │ 효율     │ 중간     │
| RoW        │ 효율     │ 높음     │
| Full Clone │ 비효율   │ 높음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 블록 매핑, CoW

**📢 섹션 요약 비유:** CoW = 효율적!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: VM 백업** → 스냅샷 + 백업
