# NetFlow - 넷플로우

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 트래픽 플로우 수집
> 2. **가치**: 트래픽 분석
> 3. **융합:** Flow Record, Export, Collector

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NetFlow는 네트워크 트래픽 플로우를 수집하고 분석합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NetFlow                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NetFlow 구조                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Flow 정의 (7-tuple):                                   │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌────────────────────────────────────────────┐ ││   │ │
│   │   │  │  │ Source IP        Destination IP            │ ││   │ │
│   │   │  │  │ Source Port      Destination Port          │ ││   │ │
│   │   │  │  │ Protocol          ToS                      │ ││   │ │
│   │   │  │  │ Ingress Interface Egress Interface         │ ││   │ │
│   │   │  │  └────────────────────────────────────────────┘ ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  NetFlow 컴포넌트:                                 ││   │ │
│   │   │  │  ┌───────────────┐      ┌───────────────┐       ││   │ │
│   │   │  │  │   Exporter    │      │   Collector   │       ││   │ │
│   │   │  │  │   (라우터)     │─────►│   (서버)      │       ││   │ │
│   │   │  │  │               │ UDP  │               │       ││   │ │
│   │   │  │  │ Flow Cache    │ 2055 │ Flow 저장     │       ││   │ │
│   │   │  │  └───────────────┘      └───────────────┘       ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  용도:                                            ││   │ │
│   │   │  │  - 트래픽 분석                                    ││   │ │
│   │   │  │  - 사용량 계량                                    ││   │ │
│   │   │  │  - 보안 분석 (DDoS 탐지)                           ││   │ │
│   │   │  │  - 용량 계획                                      ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NetFlow는 "누가, 언제, 어디서, 무엇을"을 추적합니다.

**📢 섹션 요약 비유:** NetFlow = 통화 기록!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Flow** | 플로우 (7-tuple) |
| **Exporter** | 내보내기 (라우터) |
| **Collector** | 수집기 |
| **Cache** | 플로우 캐시 |
| **Record** | 플로우 레코드 |

**핵심 알고리듬:** 패킷 수신 → Flow 키 생성 → 캐시 집계 → Export
