# Critical Section - 임계 영역

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 공유 자원 접근 코드
> 2. **가치**: 데이터 무결성
> 3. **융합:** Mutex, Semaphore, Lock

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Critical Section은 공유 자원에 접근하는 코드 영역입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Critical Section                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              임계 영역 구조                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Entry Section          Critical Section             │   │ │
│   │   │  ┌─────────────┐       ┌─────────────────────┐       │   │ │
│   │   │  │ acquire     │──────→│ Shared Resource     │       │   │ │
│   │   │  │ lock        │       │ Access              │       │   │ │
│   │   │  └─────────────┘       └─────────────────────┘       │   │ │
│   │   │                               │                        │   │ │
│   │   │  Exit Section                 │                        │   │ │
│   │   │  ┌─────────────┐              │                        │   │ │
│   │   │  │ release     │←─────────────┘                        │   │ │
│   │   │  │ lock        │                                       │   │ │
│   │   │  └─────────────┘                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  조건:                                                │   │ │
│   │   │  1. Mutual Exclusion (상호 배제)                      │   │ │
│   │   │  2. Progress (진행)                                   │   │ │
│   │   │  3. Bounded Waiting (유한 대기)                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 임계 영역은 한 번에 하나의 스레드만 진입합니다.

**📢 섹션 요약 비유:** Critical Section = 단독 작업실!
