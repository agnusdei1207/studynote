# 10BASE-T / 100BASE-TX (Fast Ethernet)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 10M/100M UTP 이더넷 표준
> 2. **가치**: LAN의 기본, 저비용
> 3. **융합**: 스타 토폴로지, 스위칭과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

10BASE-T와 100BASE-TX는 UTP 케이블을 사용하는 10Mbps와 100Mbps 이더넷 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    10BASE-T / 100BASE-TX                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              표준 비교                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  항목        │  10BASE-T       │  100BASE-TX        │   │ │
│   │   │  ────────────┼─────────────────┼─────────────────   │   │ │
│   │   │  속도        │  10Mbps         │  100Mbps           │   │ │
│   │   │  케이블      │  Cat3+          │  Cat5+             │   │ │
│   │   │  최대 거리   │  100m           │  100m              │   │ │
│   │   │  코딩        │  Manchester     │  4B/5B + MLT-3     │   │ │
│   │   │  라인 속도   │  20Mbaud        │  125Mbaud          │   │ │
│   │   │  페어 사용   │  2 (1,2 / 3,6)  │  2 (1,2 / 3,6)     │   │ │
│   │   │  전이중      │  지원           │  지원              │   │ │
│   │   │  표준 연도   │  1990           │  1995              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              배선 구조                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  RJ-45 핀 배열 (T568B):                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌───┬───┬───┬───┬───┬───┬───┬───┐                  │   │ │
│   │   │  │ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │ 8 │                  │   │ │
│   │   │  ├───┼───┼───┼───┼───┼───┼───┼───┤                  │   │ │
│   │   │  │오/흰│오 │녹/흰│파 │파/흰│녹 │갈/흰│갈│                  │   │ │
│   │   │  └───┴───┴───┴───┴───┴───┴───┴───┘                  │   │ │
│   │   │    │   │   │       │                                   │   │ │
│   │   │    TX+  TX- RX+    RX-                                 │   │ │
│   │   │    (송신)    (수신)                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  사용 페어:                                               │   │ │
│   │   │  • 페어 1-2: 송신 (TX+/TX-)                             │   │ │
│   │   │  • 페어 3-6: 수신 (RX+/RX-)                             │   │ │
│   │   │  • 페어 4-5, 7-8: 미사용 (10/100)                        │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 10BASE-T와 100BASE-TX는 2페어만 사용하며 나머지는 예비입니다.

**💡 비유**: 10BASE-T는 시내도로, 100BASE-TX는 고속도로입니다.

**📢 섹션 요약 비유**: 100M = Cat5 이상!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **Manchester** | 10M 코딩 | 클럭 포함 | 단순 |
| **4B/5B** | 100M 코딩 | 4비트→5비트 | 효율 |
| **MLT-3** | 100M 라인 | 3레벨 | 대역폭 |

**비교표**

```
+------------+----------+----------+
| 항목       | 10BASE-T | 100BASE-TX|
+------------+----------+----------+
| 코딩       | Manchester| 4B/5B    |
| 효율       | 50%      | 80%      |
| 대역폭     | 20MHz    | 31.25MHz |
| Cat 요구   | Cat3     | Cat5     |
+------------+----------+----------+
```

**핵심 알고리즸**: CSMA/CD (반이중), Auto-Negotiation

**📢 섹션 요약 비유**: 2페어 = 송수신 분리!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 토폴로지**

| 토폴로지 | 거리 | 노드 |
|:---|:---:|:---:|
| **스타** | 100m | 1/hub |
| **연속** | 5 segment | 1024 |

**📢 섹션 요약 비유**: 스타 토폴로지 = 스위치!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 레거시** → 10BASE-T
**시나리오 2: IoT/센서** → 100BASE-TX
**시나리오 3: 데스크톱** → 1000BASE-T

**도입 체크리스트**: 속도 요구, 케이블, 거리

**📢 섹션 요약 비유**: 100M는 과도기!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: LAN 보급의 기반

**미래 전망**: 1G로 완전 대체

**참고 표준**: IEEE 802.3i (10BASE-T), 802.3u (100BASE-TX)

**📢 섹션 요약 비유**: 10/100은 역사!
