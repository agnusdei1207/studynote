# Symmetric Key - 대칭키

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 동일 키 사용
> 2. **가치**: 빠른 암호화
> 3. **융합:** AES, DES, ChaCha20

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Symmetric Key는 암호화와 복호화에 같은 키를 사용합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Symmetric Key                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │

│   │   │                                                      │   │ │
│   │   │  과정:                                                 │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Sender                    Receiver              ││   │ │

│   │   │  │  │     │     Key K        │Ciph │               ││   │ │
```

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

| 항목 | 설명 |
|:---|:---|
