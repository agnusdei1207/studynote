# 3G (W-CDMA/HSPA/CDMA2000)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 광대역 멀티미디어 이동통신
> 2. **가치**: 영상 통화, 고속 데이터
> 3. **융합**: 스마트폰 혁명의 기반

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

3G(3rd Generation)는 광대역 멀티미디어 서비스를 제공하는 이동통신입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    3G 기술별                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  표준         │ 지역     │ 기반      │ 속도       │   │
│   │  ───────────┼──────────┼──────────┼───────────│   │
│   │  W-CDMA      │ 유럽/아시아│ GSM 진화  │ 384kbps-2Mbps│   │
│   │  HSPA        │ 유럽/아시아│ W-CDMA    │ 7.2-14Mbps  │   │
│   │  HSPA+       │ 유럽/아시아│ HSPA      │ 21-42Mbps   │   │
│   │  CDMA2000    │ 북미/한국 │ cdmaOne   │ 153kbps-2.4Mbps│   │
│   │  EV-DO       │ 북미/한국 │ CDMA2000  │ 2.4-3.1Mbps │   │
│   │  TD-SCDMA    │ 중국      │ GSM 진화  │ 384kbps-2Mbps│   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              3G 특성                                      │   │
│   │                                                              │   │
│   │  장점:                                                   │   │
│   │  • 광대역 데이터 (384kbps-14Mbps)                         │   │
│   │  • 영상 통화                                              │   │
│   │  • 멀티미디어                                              │   │
│   │  • 글로벌 로밍                                              │   │
│   │  • 동시 음성/데이터                                        │   │
│   │                                                      │   │
│   │  단점:                                                   │   │
│   │  • 4G 대비 낮은 속도                                      │   │
│   │  • 높은 지연                                               │   │
│   │  • 커버리지 비용                                           │   │
│   │                                                      │   │
│   │  서비스:                                                 │   │
│   │  • 영상 통화                                               │   │
│   │  • 모바일 인터넷                                           │   │
│   │  • 모바일 TV                                               │   │
│   │  • 위치 기반 서비스                                        │   │
│   │  • VOD                                                     │   │
│   │                                                      │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 3G는 스마트폰 혁명을 가능하게 한 기술입니다.

**💡 비유**: 3G는 전화기에서 TV를 볼 수 있게 된 것입니다.

**📢 섹션 요약 비유:** HSPA = 3.5G!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 기술 | 기반 | 주파수 | 다운로드 |
|:---|:---|:---|:---|
| **W-CDMA** | GSM | 2.1GHz | 384kbps |
| **HSPA** | W-CDMA | 2.1GHz | 7.2Mbps |
| **HSPA+** | HSPA | 2.1GHz | 21Mbps |
| **CDMA2000** | cdmaOne | 1.8GHz | 2.4Mbps |

**비교표**

```
+------------+----------+----------+
| 항목       │ W-CDMA   │ CDMA2000 │
+------------+----------+----------+
| 대역폭     │ 5MHz     │ 1.25MHz  │
│ 칩레이트   │ 3.84Mcps │ 1.2288Mcps│
│ 진화       │ GSM      │ cdmaOne  │
+------------+----------+----------+
```

**핵심 알고리즸:** 소프트 핸드오프, RAKE 수신기, HSDPA 스케줄링

**📢 섹션 요약 비유:** W-CDMA = GSM 진화!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: HSPA 계층**

| 계층 | 기술 | 속도 |
|:---|:---|:---|
| **HSDPA** | 다운로드 | 14.4Mbps |
| **HSUPA** | 업로드 | 5.76Mbps |
| **HSPA+** | 향상 | 21/42Mbps |

**📢 섹션 요약 비유:** HSPA = 3G에서 4G로!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 도심** → HSPA+ 21Mbps
**시나리오 2: 교외** → W-CDMA 384kbps
**시나리오 3: 음성+데이터** → 동시 지원

**도입 체크리스트:** 커버리지, 용량, 단말기

**📢 섹션 요약 비유:** 3G는 스마트폰 시대!
