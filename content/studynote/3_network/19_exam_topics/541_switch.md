# Switch (L2 Switch) - 스위치

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: MAC 주소 기반 프레임 전달
> 2. **가치**: 충돌 도메인 분리
> 3. **융합:** MAC Table, VLAN, STP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

L2 Switch는 MAC 주소로 프레임을 전달합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    L2 Switch                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              스위치 동작                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Host A   Host B   Host C   Host D                   │   │ │
│   │   │    │        │        │        │                      │   │ │
│   │   │    └────────┴────────┴────────┘                      │   │ │
│   │   │              │                                       │   │ │
│   │   │         ┌────┴────┐                                 │   │ │
│   │   │         │ Switch  │  MAC Table:                      │   │ │
│   │   │         └────┬────┘  ┌──────────────────────┐       │   │ │
│   │   │              │       │ Port 1: AA:BB:CC:11 │       │   │ │
│   │   │              │       │ Port 2: AA:BB:CC:22 │       │   │ │
│   │   │              │       │ Port 3: AA:BB:CC:33 │       │   │ │
│   │   │              │       └──────────────────────┘       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 스위치는 MAC 테이블을 학습하여 유니캐스트 전달합니다.

**📢 섹션 요약 비유:** Switch = 스마트 분배기!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **MAC Table** | 포트-MAC 매핑 |
| **Learning** | 소스 MAC 학습 |
| **Forwarding** | 목적지 MAC 전달 |
| **Flooding** | 알 수 없음 → 브로드캐스트 |
| **Aging** | 캐시 만료 |

**핵심 알고리즸:** Learn → Forward → Flood

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Hub vs Switch**

| 항목 | Hub | Switch |
|:---|:---|:---|
| **전달** | 브로드캐스트 | 유니캐스트 |
| **충돌** | 공유 | 포트별 |
| **속도** | 낮음 | 높음 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** LAN → L2 Switch
**시나리오 2:** VLAN → 논리적 분리
**시나리오 3:** 루프 방지 → STP
