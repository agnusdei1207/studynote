# Envoy - 고성능 프록시

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: L7 프록시 및 서비스 메시 데이터 플레인
> 2. **가치**: 관측성, 동적 설정, 확장성
> 3. **융합**: Istio, Service Mesh와 통합

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Envoy는 고성능 L7 프록시로 서비스 메시의 데이터 플레인입니다.


> **해설**: Envoy는 xDS API로 동적 설정을 지원합니다.

**💡 비유**: Envoy는 프록시의 슈퍼카입니다. 빠르고 스마트합니다.

**📢 섹션 요약 비유:** Envoy = L7 프록시!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Listener** | 리스너 |
| **Filter** | 필터 |
| **Cluster** | 클러스터 |
| **xDS** | 동적 API |

**비교표: xDS API**

```
+------------+----------+----------+
| API        │ 설명     │ 용도     │
+------------+----------+----------+
| LDS        │ Listener │ 리스너   │
| RDS        │ Route    │ 라우트   │
| CDS        │ Cluster  │ 클러스터 │
| EDS        │ Endpoint │ 엔드포인트│
| SDS        │ Secret   │ 인증서   │
+------------+----------+----------+
```

**핵심 알고리즸:** 이벤트 기반, 비동기 I/O

**📢 섹션 요약 비유:** xDS = 동적 설정!
