# CASB - 클라우드 액세스 보안 브로커

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 클라우드 서비스 보안 게이트웨이
> 2. **가치**: 가시성 및 제어
> 3. **융합:** Shadow IT, DLP, API

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CASB (Cloud Access Security Broker)는 클라우드 서비스 접근을 제어합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CASB                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CASB 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  CASB 배치 모드:                                         │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  1. Forward Proxy (순방향 프록시):                  ││   │ │
│   │   │  │     사용자 ──► CASB ──► 클라우드 서비스             ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  2. Reverse Proxy (역방향 프록시):                  ││   │ │
│   │   │  │     사용자 ──► CASB (클라우드 프론트) ──► SaaS      ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  3. API Mode:                                     ││   │ │
│   │   │  │     CASB ◄───API───► SaaS (직접 연동)              ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  주요 기능:                                        ││   │ │
│   │   │  │  - Shadow IT 탐지                                  ││   │ │
│   │   │  │  - DLP (Data Loss Prevention)                     ││   │ │
│   │   │  │  - 암호화                                          ││   │ │
│   │   │  │  - 세밀한 접근 제어                                 ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: CASB는 클라우드 서비스의 보안 가시성을 제공합니다.

**📢 섹션 요약 비유:** CASB = 클라우드 출입국 관리!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Shadow IT** | 미승인 클라우드 탐지 |
| **DLP** | 데이터 유출 방지 |
| **Discovery** | 클라우드 서비스 발견 |
| **Encryption** | 암호화 |
| **Governance** | 거버넌스 |

**핵심 알고리듬:** 트래픽 분석 → 앱 식별 → 정책 적용 → 제어
