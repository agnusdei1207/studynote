# Runbook - 런북

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 운영 절차 문서
> 2. **가치**: 일관된 대응, 지식 전파
> 3. **융합:** Incident Management, Automation, On-call

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Runbook은 운영 절차를 문서화합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Runbook                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              런북 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Title: High CPU Usage Alert                        │   │ │
│   │   │  ────────────────────────────────────────           │   │ │
│   │   │                                                      │   │ │
│   │   │  1. Symptoms:                                        │   │ │
│   │   │     - CPU > 80% for 5 minutes                       │   │ │
│   │   │     - Slow response times                            │   │ │
│   │   │                                                      │   │ │
│   │   │  2. Diagnosis:                                       │   │ │
│   │   │     $ top -c                                         │   │ │
│   │   │     $ ps aux --sort=-%cpu | head -10                 │   │ │
│   │   │                                                      │   │ │
│   │   │  3. Resolution:                                      │   │ │
│   │   │     - If process X: restart service                  │   │ │
│   │   │     - If traffic: scale out                          │   │ │
│   │   │     - If query: kill query                           │   │ │
│   │   │                                                      │   │ │
│   │   │  4. Escalation:                                      │   │ │
│   │   │     - Contact: DBA team                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 런북은 일관된 장애 대응을 제공합니다.

**📢 섹션 요약 비유:** Runbook = 요리책!
