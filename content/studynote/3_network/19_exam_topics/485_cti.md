# CTI (컴퓨터 전화 통합) - Computer Telephony Integration

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 컴퓨터와 전화 시스템 통합
> 2. **가치**: 화면 팝업, 고객 정보 자동 표시
> 3. **융합**: CRM, PBX, 콜센터

---

### Ⅰ. 개요 (Context & Background)

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Screen Pop** | 화면 팝업 |
| **Call Control** | 통화 제어 |
| **Call Logging** | 통화 기록 |
| **ANI/DNIS** | 발신/착신 번호 |

**비교표: CTI 프로토콜**

```
+------------+----------+----------+----------+
| 프로토콜   │ 용도     │ 특징     │ 환경     │
+------------+----------+----------+----------+
| TAPI       │ MS       │ Windows  │ 데스크   │
| CSTA       │ 표준     │ ECMA     │ 범용     │
| JTAPI      │ Java     │ 크로스   │ Java     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 이벤트 처리, 매칭

**📢 섹션 요약 비유:** Screen Pop = 자동 정보!
