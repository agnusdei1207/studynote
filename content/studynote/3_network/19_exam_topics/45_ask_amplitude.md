# ASK (진폭 편이 변조)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 반송파의 진폭으로 디지털 데이터 표현
> 2. **가치**: 구현 간단, 대역폭 효율
> 3. **융합**: OOK, QAM 기반, 광통신과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ASK(Amplitude Shift Keying)는 디지털 데이터에 따라 반송파의 진폭을 변화시키는 디지털 변조 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ASK (진폭 편이 변조)                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ASK 기본 원리                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  이진 ASK (Binary ASK, BASK):                          │   │ │
│   │   │  • 0: 반송파 없음 (진폭 = 0)                           │   │ │
│   │   │  • 1: 반송파 있음 (진폭 = A)                           │   │ │
│   │   │                                                      │   │ │
│   │   │  파형:                                                 │   │ │
│   │   │  데이터:    1   0   1   1   0                         │   │ │
│   │   │             ┌┐       ┌┐┌┐                             │   │ │
│   │   │           ──┘└───────┘└┘└────                         │   │ │
│   │   │            ON  OFF  ON ON OFF                          │   │ │
│   │   │                                                      │   │ │
│   │   │  OOK (On-Off Keying):                                 │   │ │
│   │   │  • ASK의 특수한 형태                                   │   │ │
│   │   │  • 광통신에서 주로 사용                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  M진 ASK (M-ary ASK):                                  │   │ │
│   │   │  • 4-ASK: 4가지 진폭 레벨                              │   │ │
│   │   │  • 비트 당 2비트 전송                                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ASK 장단점                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  장점:                                                 │   │ │
│   │   │  • 구현이 간단함                                       │   │ │
│   │   │  • 대역폭 효율적                                       │   │ │
│   │   │  • 광통신에 적합                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                 │   │ │
│   │   │  • 잡음에 매우 취약                                    │   │ │
│   │   │  • 진폭 변동에 민감                                    │   │ │
│   │   │  • 무선 통신에 부적합                                   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: ASK는 가장 간단하지만 잡음에 가장 취약합니다.

**💡 비유**: ASK는 스위치를 켰다 껐다 하는 것과 같습니다.

**📢 섹션 요약 비유**: ASK = 진폭으로 0/1 표현!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **진폭** | 정보 | A, 0 | 크기 |
| **반송파** | 기반 | cos(2πfct) | 기준 |
| **OOK** | 특수 | ON/OFF | 스위치 |
| **M-ASK** | 다진 | 다중 레벨 | 계단 |

**수학적 표현**

```
s(t) = A·m(t)·cos(2πf_c·t)

m(t) = 0 (비트 0)
m(t) = 1 (비트 1)

대역폭 = 2·R (R = 비트 전송률)
```

**핵심 알고리즘**: 데이터 → 진폭 선택 → 반송파 곱 → 전송

**📢 섹션 요약 비유**: 광통신은 OOK 사용!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 디지털 변조 잡음 내성**

| 변조 | 잡음 내성 | 응용 |
|:---|:---:|:---|
| **ASK** | 약함 | 광통신 |
| **FSK** | 강함 | 블루투스 |
| **PSK** | 강함 | 위성 |

**📢 섹션 요약 비유**: 무선은 ASK 안 써요!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 광통신** → OOK (ASK)
**시나리오 2: RFID** → ASK
**시나리오 3: IR 적외선** → ASK

**도입 체크리스트**: 잡음 환경, 전력, 구현 복잡도

**📢 섹션 요약 비유**: 광섬유는 OOK가 표준!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 구현 단순, 저비용

**미래 전망**: QAM의 기반 (진폭 + 위상)

**참고 표준**: IEEE 802.15.7 (가시광 통신)

**📢 섹션 요약 비유**: ASK는 변조의 기본!
