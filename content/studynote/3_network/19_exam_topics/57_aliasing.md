# 에일리어싱 (Aliasing)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 부족한 샘플링으로 신호가 왜곡되는 현상
> 2. **가치**: 디지털 신호 처리의 한계 이해
> 3. **융합**: Nyquist, 안티에일리어싱 필터와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

에일리어싱(Aliasing)은 샘플링 주파수가 신호 최대 주파수의 2배 미만일 때, 고주파 신호가 저주파로 인식되는 현상입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    에일리어싱 (Aliasing)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              에일리어싱 발생 원리                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  정상 샘플링 (fs ≥ 2fmax):                             │   │ │
│   │   │  원신호:  ╱╲╱╲╱╲                                       │   │ │
│   │   │  샘플:   ●●●●●  → 복원 OK                              │   │ │
│   │   │                                                      │   │ │
│   │   │  에일리어싱 (fs < 2fmax):                               │   │ │
│   │   │  원신호: ╱╲╱╲╱╲╱╲ (고주파)                              │   │ │
│   │   │  샘플:  ●    ●    ●  → 저주파처럼 보임!                │   │ │
│   │   │                (에일리어싱)                              │   │ │
│   │   │                                                      │   │ │
│   │   │  원인:                                                 │   │ │
│   │   │  • 샘플링 주파수 부족                                   │   │ │
│   │   │  • 고주파 신호가 저주파로 "변장" (alias)               │   │ │
│   │   │                                                      │   │ │
│   │   │  결과:                                                 │   │ │
│   │   │  • 원신호 복원 불가                                    │   │ │
│   │   │  • 스펙트럼 겹침                                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              에일리어싱 방지                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. Nyquist 준수:                                     │   │ │
│   │   │     fs ≥ 2·fmax                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 안티에일리어싱 필터 (AAF):                          │   │ │
│   │   │     • 샘플링 전 저역 통과 필터                          │   │ │
│   │   │     • fs/2 이상 주파수 제거                            │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 오버샘플링:                                         │   │ │
│   │   │     • fs >> 2·fmax                                    │   │ │
│   │   │     • 여유 확보                                        │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 에일리어싱은 "다른 이름(alias)으로 불리는" 현상입니다.

**💡 비유**: 에일리어싱은 바퀴가 천천히 돌아가는 것처럼 보이는 현상입니다.

**📢 섹션 요약 비유**: AAF 필터로 고주파 제거!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **fs/2** | 폴딩 주파수 | 경계 | 한계 |
| **AAF** | 필터 | 저역 통과 | 걸러내기 |
| **오버샘플링** | 여유 | fs >> 2fmax | 넉넉히 |
| **주파수 겹침** | 증상 | 스펙트럼 | 혼선 |

**스펙트럼 겹침**

```
정상:                에일리어싱:
┌──────┐            ┌──────┐
│신호  │            │신호  │←─┐겹침
└──────┘            └──────┘  │
      ↑             ┌──────┐  │
     fs/2           │별칭  │──┘
                   └──────┘
```

**핵심 알고리즘**: 신호 → AAF 필터 → 샘플링 → 디지털

**📢 섹션 요약 비유**: ADC 앞에 AAF 필터 필수!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 에일리어싱 방지법**

| 방법 | 원리 | 비용 |
|:---|:---|:---:|
| **Nyquist 준수** | 충분한 fs | 낮음 |
| **AAF 필터** | 고주파 제거 | 중간 |
| **오버샘플링** | 여유 fs | 높음 |

**📢 섹션 요약 비유**: 오디오 CD는 44.1kHz로 20kHz 안전!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 오디오** → AAF 필터 + 44.1kHz

**시나리오 3: 통신** → 수신 필터 설계

**도입 체크리스트**: fs, 대역폭, 필터 설계

**📢 섹션 요약 비유**: 샘플링 전 필터링이 핵심!
