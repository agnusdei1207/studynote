# Nginx - 웹 서버/프록시

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고성능 웹 서버/리버스 프록시
> 2. **가치**: 동시성, 로드밸런싱
> 3. **융합**: K8s, API Gateway와 통합

### Ⅰ. 개요 (Context & Background)

**개념 정의**

> **해설**: Nginx는 이벤트 기반 아키텍처로 높은 동시성을 처리합니다.

**💡 비유**: Nginx는 식당의 주방장입니다. 주문을 효율적으로 처리합니다.

**📢 섹션 요약 비유:** Nginx = 이벤트 기반!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Worker** | 요청 처리 |
| **Upstream** | 백엔드 그룹 |
| **Location** | 경로 매칭 |
| **Server** | 가상 호스트 |

**비교표: Nginx vs Apache**

```
+------------+----------+----------+
| 항목       │ Apache   │ Nginx    │
+------------+----------+----------+
| 아키텍처   │ 프로세스 │ 이벤트   │
| 메모리     │ 높음     │ 낮음     │
| 동시성     │ 제한     │ 높음     │
| 정적       │ 느림     │ 빠름     │
+------------+----------+----------+
```

**핵심 알고리즸:** epoll, kqueue

**📢 섹션 요약 비유:** Event-driven = 비동기!
