# 빔포밍 (Beamforming) - 디지털/아날로그/하이브리드

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 위상/진폭 제어로 빔 형성
> 2. **가치**: 신호 집중, 간섭 감소
> 3. **융합**: 5G, WiFi 6, 레이더와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

빔포밍(Beamforming)은 여러 안테나 소자의 위상과 진폭을 제어하여 신호를 특정 방향으로 집중시키는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    빔포밍 (Beamforming)                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              빔포밍 종류                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 아날로그 빔포밍 (Analog Beamforming):               │   │ │
│   │   │     • RF 단에서 위상 시프터 사용                        │   │ │
│   │   │     • 단일 데이터 스트림                               │   │ │
│   │   │     • 저비용, 저전력                                  │   │ │
│   │   │     • mmWave 5G                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 디지털 빔포밍 (Digital Beamforming):                │   │ │
│   │   │     • 베이스밴드에서 디지털 신호 처리                   │   │ │
│   │   │     • 다중 데이터 스트림 (MU-MIMO)                     │   │ │
│   │   │     • 높은 유연성                                      │   │ │
│   │   │     • 높은 비용/전력                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 하이브리드 빔포밍 (Hybrid Beamforming):             │   │ │
│   │   │     • 디지털 + 아날로그 결합                           │   │ │
│   │   │     • 비용과 성능 균형                                 │   │ │
│   │   │     • 5G mmWave 표준                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 컨포멀 빔포밍 (Conformal Beamforming):              │   │ │
│   │   │     • 곡면 배열                                       │   │ │
│   │   │     • 항공기, 차량                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              빔포밍 원리                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │     w₁    w₂    w₃    w₄    (가중치)                  │   │ │
│   │   │      ↓     ↓     ↓     ↓                             │   │ │
│   │   │     ┌───────────────────┐                            │   │ │
│   │   │     │  ○   ○   ○   ○    │  ← 안테나 배열             │   │ │
│   │   │     └───────────────────┘                            │   │ │
│   │   │              │                                       │   │ │
│   │   │              ▼                                       │   │ │
│   │   │        ▒▒▒▒▒▒▒▒  ← 집중된 빔                         │   │ │
│   │   │                                                      │   │ │
│   │   │  y(t) = Σ wₙ·xₙ(t)  (가중 합)                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 빔포밍은 5G mmWave와 MU-MIMO의 핵심 기술입니다.

**💡 비유**: 빔포밍은 손전등 빛을 한 곳에 집중하는 것과 같습니다.

**📢 섹션 요약 비유:** 위상 제어 = 빔 조향!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 유형 | 제어 단계 | 스트림 | 비용 |
|:---|:---|:---:|:---:|
| **아날로그** | RF | 1 | 낮음 |
| **디지털** | BB | N | 높음 |
| **하이브리드** | RF+BB | N | 중간 |

**비교표**

```
+------------+----------+----------+
| 항목       │ 아날로그 │ 디지털   │
+------------+----------+----------+
| 스트림     │ 1        │ N        │
│ 유연성     │ 낮음     │ 높음     │
│ 전력       │ 낮음     │ 높음     │
│ 복잡도     │ 낮음     │ 높음     │
+------------+----------+----------+
```

**핵심 알고리즸:** MVDR, ZF, MMSE

**📢 섹션 요약 비유:** 하이브리드 = 실용적 선택!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 빔포밍 이득**

| 소자 수 | 빔 이득 | 커버리지 |
|:---:|:---:|:---:|
| **4** | 6 dB | 2배 |
| **8** | 9 dB | 3배 |
| **16** | 12 dB | 4배 |
| **64** | 18 dB | 8배 |

**📢 섹션 요약 비유:** 소자 많을수록 이득 높음!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: Sub-6GHz** → 디지털 빔포밍
**시나리오 2: mmWave** → 하이브리드 빔포밍
**시나리오 3: WiFi 6** → 디지털 빔포밍

**도입 체크리스트:** 주파수, 스트림 수, 전력

**📢 섹션 요약 비유:** 주파수 높을수록 아날로그 필요!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 커버리지 2-8배, 용량 2-10배
