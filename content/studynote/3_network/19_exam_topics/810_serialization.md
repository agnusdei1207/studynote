# Serialization - 직렬화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 객체 → 바이트 변환
> 2. **가치**: 전송 및 저장 가능
> 3. **융합:** JSON, Protobuf, MessagePack

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Serialization은 객체를 바이트 스트림으로 변환합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Serialization                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              직렬화 프로세스                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  송신측:                                              │   │ │
│   │   │  ┌──────────┐    직렬화    ┌──────────┐              │   │ │
│   │   │  │  Object  │ ──────────→ │  Bytes   │              │   │ │
│   │   │  │ (Memory) │             │(Stream)  │              │   │ │
│   │   │  └──────────┘             └──────────┘              │   │ │
│   │   │       │                        │                    │   │ │
│   │   │       └────────────────────────┘                    │   │ │
│   │   │                 │                                   │   │ │
│   │   │           네트워크 전송                              │   │ │
│   │   │                 ↓                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  수신측:                                              │   │ │
│   │   │  ┌──────────┐   역직렬화   ┌──────────┐              │   │ │
│   │   │  │  Object  │ ←────────── │  Bytes   │              │   │ │
│   │   │  │ (Memory) │             │(Stream)  │              │   │ │
│   │   │  └──────────┘             └──────────┘              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 직렬화는 이기종 시스템 간 데이터 교환을 가능하게 합니다.

**📢 섹션 요약 비유:** Serialization = 포장 이사!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Encoder** | 인코더 |
| **Decoder** | 디코더 |
| **Schema** | 스키마 |
