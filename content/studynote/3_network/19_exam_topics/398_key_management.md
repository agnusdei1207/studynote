# 키 관리 - Key Management

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 암호키 수명주기 관리
> 2. **가치**: 키 유출 방지, 규정 준수
> 3. **융합**: KMS, HSM, 클라우드

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

키 관리는 암호키의 생성부터 폐기까지 전 과정을 관리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    키 관리                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              키 수명주기                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [생성] ──▶ [분배] ──▶ [저장] ──▶ [사용] ──▶ [교체] │   │ │
│   │   │                                        │              │   │ │
│   │   │                                        ▼              │   │ │
│   │   │                                    [폐기/파기]        │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐   │   │ │
│   │   │  │ 난수    │  │ 암호화  │  │ HSM/   │  │ 로테이션│   │   │ │
│   │   │  │ 생성    │  │ 전송    │  │ KMS    │  │ 정책   │   │   │ │
│   │   │  └────────┘  └────────┘  └────────┘  └────────┘   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              키 관리 요소                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 요소       │ 설명     │ 도구     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │

│   │   │  │ 분배       │ 암호화   │ PKCS#12  │               │   │ │
│   │   │  | 저장       │ 암호화   │ KMS      │               │   │ │
│   │   │  │ 교체       │ 정기적   │ 정책     │               │   │ │
│   │   │  │ 폐기       │ 안전파기 │ 영구삭제 │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
```

> **해설**: 키 관리는 암호 시스템의 가장 취약한 부분입니다.

**💡 비유**: 키 관리는 열쇠 관리입니다. 누가, 언제, 어떤 키를 쓰는지 관리합니다.

**📢 섹션 요약 비유:** 키 관리 = 보안의 핵심!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Key Generation** | 키 생성 |
| **Key Distribution** | 키 분배 |
| **Key Storage** | 키 저장 |
| **Key Rotation** | 키 교체 |
| **Key Destruction** | 키 파기 |

**비교표: 키 저장 방식**

```
+------------+----------+----------+
| 항목       │ HSM      │ KMS      │
+------------+----------+----------+
| 위치       │ 온프레미스│ 클라우드 │
| 관리       │ 자체     │ 제공자   │
| 비용       │ CAPEX    │ OPEX     │
| 확장성     │ 제한     │ 높음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 키 파생, 키 래핑

**📢 섹션 요약 비유:** KMS = 클라우드 키 관리!
