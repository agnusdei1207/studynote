# Branch Prediction - 분기 예측

> 2. **가치**: 파이프라인 효율

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Branch Prediction은 분기 결과를 예측합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Branch Prediction                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              분기 예측 구조                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Branch Instruction                                  │   │ │
│   │   │       │                                              │   │ │
│   │   │       ↓                                              │   │ │
│   │   │  ┌─────────────────────────────────────┐            │   │ │
│   │   │  │         Branch Predictor            │            │   │ │
│   │   │  │  ┌───────────────────────────────┐ │            │   │ │
│   │   │  │  │ BTB (Branch Target Buffer)    │ │            │   │ │
│   │   │  │  │ - 주소 → 타겟                 │ │            │   │ │
│   │   │  │  │ - 예측 히스토리               │ │            │   │ │
│   │   │  │  └───────────────────────────────┘ │            │   │ │
│   │   │  └─────────────────────────────────────┘            │   │ │
│   │   │       │                                              │   │ │
│   │   │       ↓                                              │   │ │
│   │   │  Predicted Path → Speculative Execution              │   │ │
│   │   │       │                                              │   │ │
│   │   │       ↓                                              │   │ │
│   │   │  (Correct) Continue / (Wrong) Flush                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 분기 예측은 예측 실패 시 페널티가 발생합니다.

**📢 섹션 요약 비유:** Branch Prediction = 날씨 예측!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **BTB** | Branch Target Buffer |
| **BHT** | Branch History Table |
| **PHT** | Pattern History Table |
| **RAS** | Return Address Stack |
