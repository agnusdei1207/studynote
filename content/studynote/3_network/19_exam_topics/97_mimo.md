# MIMO (다중 안테나 기술)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 송수신 다중 안테나로 전송 용량/신뢰성 향상
> 2. **가치**: 전송률 증대, 다중 경로 활용
> 3. **융합**: WiFi, LTE, 5G와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MIMO(Multiple-Input Multiple-Output)는 송신과 수신에 여러 안테나를 사용하여 공간 다중화와 다이버시티를 통해 전송 성능을 향상시키는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MIMO (다중 안테나 기술)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MIMO 구조                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  기본 형태 (2×2 MIMO):                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  송신측                    수신측                      │   │ │
│   │   │  ┌───┐                    ┌───┐                      │   │ │
│   │   │  │TX1├──── h11 ──────────►│RX1│                      │   │ │
│   │   │  │   ├──── h12 ────────┐  │   │                      │   │ │
│   │   │  └───┘                 │  └───┘                      │   │ │
│   │   │  ┌───┐                 │  ┌───┐                      │   │ │
│   │   │  │TX2├──── h21 ─────┐  └─►│RX1│                      │   │ │
│   │   │  │   ├──── h22 ─────┴────►│   │                      │   │ │
│   │   │  └───┘                    └───┘                      │   │ │
│   │   │                                                      │   │ │
│   │   │  채널 행렬 H:                                          │   │ │
│   │   │  ┌         ┐                                          │   │ │
│   │   │  │ h11 h12 │                                          │   │ │
│   │   │  │ h21 h22 │                                          │   │ │
│   │   │  └         ┘                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  MIMO 이득:                                            │   │ │
│   │   │  • 공간 다중화: 전송률 ↑                               │   │ │
│   │   │  • 다이버시티: 신뢰성 ↑                                │   │ │
│   │   │  • 빔포밍: SINR ↑                                     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MIMO 분류                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  SISO: 1×1 (기본)                                     │   │ │
│   │   │  SIMO: 1×N (수신 다이버시티)                           │   │ │
│   │   │  MISO: N×1 (송신 다이버시티/빔포밍)                     │   │ │
│   │   │  MIMO: N×M (공간 다중화 + 다이버시티)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  안테나 수에 따른 용량:                                 │   │ │
│   │   │  min(N_tx, N_rx) × log₂(1+SNR)                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: MIMO는 WiFi 4(802.11n)부터 도입된 핵심 기술입니다.

**💡 비유**: MIMO는 여러 차선 도로에서 동시에 운전하는 것입니다.

**📢 섹션 요약 비유**: MIMO = 안테나 수 = 용량!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **공간 다중화** | 전송률 | 독립 스트림 | 차선 |
| **다이버시티** | 신뢰성 | 중복 전송 | 보험 |
| **채널 추정** | H 파악 | 파일럿 | 측량 |
| **프리코딩** | 송신 최적화 | 가중치 | 조향 |

**비교표**

```
+------------+----------+----------+
| 항목       | SISO     | 2×2 MIMO |
+------------+----------+----------+
| 안테나     | 1+1      | 2+2      |
| 전송률     | 1×       | 2×       |
| 신뢰성     | 낮음     | 높음     |
| 복잡도     | 낮음     | 높음     |
+------------+----------+----------+
```

**핵심 알고리즸**: 채널 추정 → 프리코딩 → 다중 스트림 → 디코딩

**📢 섹션 요약 비유**: MIMO는 안테나가 많을수록 좋다!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: MIMO 응용**

| 표준 | MIMO 형태 | 이득 |
|:---|:---:|:---:|
| **WiFi 4** | 4×4 | 600Mbps |
| **WiFi 5** | 8×8 | 6.9Gbps |
| **WiFi 6** | 8×8 MU | 9.6Gbps |
| **LTE** | 4×4 | 300Mbps |
| **5G** | 64×64 | Gbps |

**📢 섹션 요약 비유**: 5G = Massive MIMO!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: WiFi** → SU-MIMO
**시나리오 2: LTE** → 2×2 또는 4×4
**시나리오 3: 5G** → Massive MIMO

**도입 체크리스트**: 안테나 수, 채널 환경, 복잡도

**📢 섹션 요약 비유**: MIMO는 무선의 필수!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 전송률 N배, 신뢰성 향상

**미래 전망**: Massive MIMO, Distributed MIMO

**참고 표준**: IEEE 802.11n/ac/ax, 3GPP

**📢 섹션 요약 비유**: MIMO는 무선 통신의 혁명!
