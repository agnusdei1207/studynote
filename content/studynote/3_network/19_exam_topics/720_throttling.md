# Throttling - 스로틀링

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 처리 속도 조절
> 2. **가치**: 시스템 보호, 자원 관리
> 3. **융합:** Rate Limiting, Backpressure, Flow Control

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Throttling은 처리 속도를 조절합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Throttling                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              스로틀링 유형                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Hard Throttling:                                    │   │ │
│   │   │  제한 초과 → 즉시 거부                                │   │ │
│   │   │                                                      │   │ │
│   │   │  Soft Throttling:                                    │   │ │
│   │   │  제한 초과 → 대기 후 처리                              │   │ │
│   │   │                                                      │   │ │
│   │   │  Dynamic Throttling:                                 │   │ │
│   │   │  시스템 상태에 따라 동적 조절                          │   │ │
│   │   │                                                      │   │ │
│   │   │  Elastic Throttling:                                 │   │ │
│   │   │  부하에 따라 탄력적 조절                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 스로틀링은 시스템 안정성을 유지합니다.

**📢 섹션 요약 비유:** Throttling = 속도 제어장치!
