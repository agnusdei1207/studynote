# PCM (펄스 부호 변조)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 아날로그 신호를 디지털로 변환하는 표준 방식
> 2. **가치**: 디지털 통신의 기반, CD 품질
> 3. **융합**: 샘플링, 양자화, 부호화와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PCM(Pulse Code Modulation)은 아날로그 신호를 샘플링, 양자화, 부호화하여 디지털 신호로 변환하는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PCM (펄스 부호 변조)                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PCM 변환 과정                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  아날로그      샘플링      양자화      부호화          │   │ │
│   │   │    신호      ─────→     ─────→     ─────→           │   │ │
│   │   │                                                      │   │ │
│   │   │  ╱╲╱╲          ●●●●        5 6 7      0101          │   │ │
│   │   │ ╱    ╲    →    ●  ●   →   3 4 5  →   0011          │   │ │
│   │   │                                                     │   │ │
│   │   │  1단계: 샘플링 (Sampling)                              │   │ │
│   │   │     • 8kHz (전화) / 44.1kHz (CD)                      │   │ │
│   │   │     • PAM 신호 생성                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  2단계: 양자화 (Quantization)                          │   │ │
│   │   │     • 8비트 (전화) / 16비트 (CD)                       │   │ │
│   │   │     • 256 / 65536 레벨                                │   │ │
│   │   │                                                      │   │ │
│   │   │  3단계: 부호화 (Encoding)                              │   │ │
│   │   │     • 이진 코드로 변환                                 │   │ │
│   │   │     • 직렬 전송                                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PCM 표준                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  전화 (G.711):                                          │   │ │
│   │   │  • 샘플링: 8kHz                                        │   │ │
│   │   │  • 양자화: 8비트 (μ-law/A-law)                         │   │ │
│   │   │  • 비트율: 64kbps                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  CD (Red Book):                                         │   │ │
│   │   │  • 샘플링: 44.1kHz                                     │   │ │
│   │   │  • 양자화: 16비트 선형                                  │   │ │
│   │   │  • 채널: 스테레오 (2채널)                               │   │ │
│   │   │  • 비트율: 1.411Mbps                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  DVD-Audio:                                             │   │ │
│   │   │  • 샘플링: 96kHz / 192kHz                              │   │ │
│   │   │  • 양자화: 24비트                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: PCM은 디지털 오디오의 기초입니다.

**💡 비유**: PCM은 음악을 0과 1의 숫자로 적어두는 것입니다.

**📢 섹션 요약 비유**: 전화는 64kbps PCM!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 전화 | CD | DVD |
|:---|:---:|:---:|:---:|
| **샘플링** | 8kHz | 44.1kHz | 96kHz |
| **양자화** | 8비트 | 16비트 | 24비트 |
| **비트율** | 64kbps | 1.4Mbps | 4.6Mbps |
| **대역폭** | 4kHz | 20kHz | 48kHz |

**비교표**

```
+------------+----------+----------+----------+
| 항목       | 전화     | CD       | DVD-A    |
+------------+----------+----------+----------+
| fs         | 8kHz     | 44.1kHz  | 96kHz    |
| 비트       | 8        | 16       | 24       |
| 채널       | 1        | 2        | 2~6      |
| 비트율     | 64kbps   | 1.4Mbps  | 4.6Mbps  |
+------------+----------+----------+----------+
```

**핵심 알고리즸**: 아날로그 → 샘플링 → 양자화 → 부호화 → 디지털

**📢 섹션 요약 비유**: CD = 44.1kHz × 16비트 × 2채널!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 음성 부호화**

| 방식 | 비트율 | 품질 |
|:---|:---:|:---:|
| **PCM** | 64kbps | 최고 |
| **ADPCM** | 32kbps | 좋음 |
| **MP3** | 128kbps | 좋음 |

**📢 섹션 요약 비유**: PCM은 무압축 최고 품질!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 전화망** → G.711 PCM (64kbps)
**시나리오 2: CD** → 44.1kHz/16비트 PCM
**시나리오 3: VoIP** → G.711 또는 압축

**도입 체크리스트**: 대역폭, 품질, 지연

**📢 섹션 요약 비유**: PCM은 품질 기준!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 무손실 디지털 변환

**미래 전망**: Hi-Res Audio (24비트/192kHz)

**참고 표준**: ITU-T G.711, IEC 60908 (CD)

**📢 섹션 요약 비유**: PCM은 디지털 오디오의 아버지!
