# NATS - 네이츠

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 경량 메시징 시스템
> 2. **가치**: 높은 성능, 단순함
> 3. **융합:** Pub/Sub, Request/Reply, Streaming

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NATS는 경량 메시징 시스템입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NATS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NATS 통신 모델                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Pub/Sub:                                             │   │ │
│   │   │  Publisher ──→ [Subject] ──→ Subscribers             │   │ │
│   │   │                (topic)                               │   │ │
│   │   │                                                      │   │ │
│   │   │  Request/Reply:                                       │   │ │
│   │   │  Client ──→ [Subject] ──→ Service                    │   │ │
│   │   │  Client ←── [Reply] ←── Service                      │   │ │
│   │   │                                                      │   │ │
│   │   │  Queue Groups:                                        │   │ │
│   │   │  Publisher ──→ [Subject] ──→ Queue Group             │   │ │
│   │   │                               ├── Subscriber 1        │   │ │
│   │   │                               └── Subscriber 2        │   │ │
│   │   │                             (로드밸런싱)               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NATS는 단순하고 빠른 메시징을 제공합니다.

**📢 섹션 요약 비유:** NATS = 공지방송!

| 항목 | 설명 |
|:---|:---|
