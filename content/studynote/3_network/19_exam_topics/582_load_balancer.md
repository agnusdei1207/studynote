# Load Balancer - 로드 밸런서

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 트래픽 분산 장치
> 2. **가치**: 고가용성, 확장성
> 3. **융합:** L4/L7, Health Check, HA

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Load Balancer는 트래픽을 여러 서버로 분산합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Load Balancer                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Load Balancer 동작                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                                              │   │ │
│   │   │     │                                                │   │ │
│   │   │     ↓                                                │   │ │
│   │   │  ┌─────────────────┐                                │   │ │
│   │   │  │  Load Balancer  │ ← Health Check                 │   │ │
│   │   │  └────────┬────────┘                                │   │ │
│   │   │       /   |   \                                     │   │ │
│   │   │     /    |    \                                     │   │ │
│   │   │   ↓     ↓     ↓                                     │   │ │
│   │   │ Server1 Server2 Server3                             │   │ │
│   │   │                                                      │   │ │
│   │   │  알고리즘: Round Robin, Least Connection, IP Hash    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Load Balancer는 서버 부하를 균등하게 분산합니다.

**📢 섹션 요약 비유:** Load Balancer = 교통 정리!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **L4 LB** | IP/Port 기반 |
| **L7 LB** | HTTP 내용 기반 |
| **Health Check** | 서버 상태 확인 |
| **Sticky Session** | 세션 유지 |
| **Failover** | 장애 조치 |

**핵심 알고리즸:** 알고리즘 선택 → 분산 → 모니터링

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**분산 알고리즘**

| 알고리즘 | 설명 |
|:---|:---|
| **Round Robin** | 순차 |
| **Least Connection** | 연결 적은 곳 |
| **IP Hash** | IP 기반 |
| **Weighted** | 가중치 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
