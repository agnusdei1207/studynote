# MIMO (Multiple Input Multiple Output)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 다중 안테나로 공간 다중화
> 2. **가치**: 용량 증대, 신뢰성 향상
> 3. **융합**: 4G/5G, WiFi, 레이더와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MIMO는 송수신 양단에 다중 안테나를 사용하여 공간 다중화와 다이버시티를 제공하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MIMO (Multiple Input Multiple Output)             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MIMO 종류                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. SU-MIMO (Single-User MIMO):                        │   │ │
│   │   │     • 단일 사용자에게 모든 스트림                      │   │ │
│   │   │     • 공간 다중화 (Spatial Multiplexing)              │   │ │
│   │   │     • 용량 증대                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  2. MU-MIMO (Multi-User MIMO):                         │   │ │
│   │   │     • 여러 사용자에게 동시 전송                        │   │ │
│   │   │     • 빔포밍 필수                                     │   │ │
│   │   │     • 시스템 용량 극대화                               │   │ │
│   │   │                                                      │   │ │

│   │   │                                                      │   │ │

│   │   │     • 여러 기지국 협력                                 │   │ │

│   │   │                                                      │   │ │

│   │                                                              │ │

│   │   │                                                      │   │ │
│   │   │  ┌───┐                      ┌───┐                   │   │ │

│   │   │                                                      │   │ │
```

> **해설**: MIMO 용량은 안테나 수에 비례하여 증가합니다.

**💡 비유**: MIMO는 도로를 여러 차선으로 확장하는 것과 같습니다.

**📢 섹션 요약 비유:** MIMO = 용량 증대의 핵심!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 유형 | 안테나 수 | 스트림 | 용도 |
|:---|:---:|:---:|:---|
| **SU-MIMO** | 2-8 | 2-8 | WiFi |
| **MU-MIMO** | 4-16 | 사용자당 1-2 | LTE-A |
| **Massive** | 64-256 | 16-32 | 5G |

**비교표**

```
+------------+----------+----------+
| 항목       │ SU-MIMO  │ MU-MIMO  │
+------------+----------+----------+
| 사용자     │ 1        │ 다수     │
│ 스케줄링   │ 단순     │ 복잡     │
│ 용량       │ 개인     │ 시스템   │
│ 빔포밍     │ 선택적   │ 필수     │
+------------+----------+----------+
```

**핵심 알고리즸:** SVD, ZF, MMSE, MRC

**📢 섹션 요약 비유:** 용량 = min(M,N) × log₂(1+SNR)!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 용량 증대**

| 안테나 | 이론 용량 | 실제 효율 |
|:---:|:---:|:---:|
| **2×2** | 2배 | 1.6배 |
| **4×4** | 4배 | 3배 |
| **8×8** | 8배 | 5배 |
| **64×64** | 64배 | 20배 |

**📢 섹션 요약 비유:** Massive MIMO = 5G 용량의 핵심!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: WiFi 6** → 8×8 MU-MIMO
**시나리오 2: LTE-A** → 8×8 MU-MIMO
**시나리오 3: 5G** → 64T64R Massive MIMO

**도입 체크리스트:** 채널 환경, 사용자 수, 비용

**📢 섹션 요약 비유:** 안테나 많을수록 용량 높음!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 용량 2-20배 증대

**미래 전망:** 256+ 안테나, RIS (Reconfigurable Intelligent Surface)

**참고 표준:** 3GPP TS 38.211, IEEE 802.11ax

**📢 섹션 요약 비유:** MIMO는 무선 용량의 해법!
