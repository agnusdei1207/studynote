# Latency - 지연 시간

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 전송 소요 시간
> 2. **가치**: 실시간 서비스 품질
> 3. **융합:** RTT, Jitter, Propagation Delay

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Latency는 데이터가 출발지에서 목적지까지 도달하는 시간입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Latency                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              지연 시간 구성요소                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  지연 시간 구성:                                          │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Total Latency =                                ││   │ │
│   │   │  │    Propagation + Transmission +                 ││   │ │
│   │   │  │    Processing + Queuing                         ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  1. Propagation Delay: 전파 지연                 ││   │ │
│   │   │  │     거리 / 빛의 속도                              ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  2. Transmission Delay: 전송 지연                 ││   │ │
│   │   │  │     패킷 크기 / 대역폭                            ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  3. Processing Delay: 처리 지연                   ││   │ │
│   │   │  │     라우터 처리 시간                              ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  4. Queuing Delay: 큐잉 지연                      ││   │ │
│   │   │  │     대기열 대기 시간                              ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Latency는 사용자 경험에 직접 영향을 미칩니다.

**📢 섹션 요약 비유:** Latency = 배달 소요 시간!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **RTT** | 왕복 시간 |
| **One-way** | 단방향 지연 |
| **Jitter** | 지연 변동 |
| **Ping** | 측정 도구 |
| **Hops** | 경유 홉 수 |

**핵심 알고리듬:** Ping → RTT 측정 → Latency 계산
