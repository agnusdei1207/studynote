# SD-WAN (Software-Defined WAN) - 소프트웨어 정의 WAN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SDN 기반 WAN 최적화
> 2. **가치**: 비용 절감, 성능 향상
> 3. **융합:** SDN, WAN, MPLS 대체

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SD-WAN은 SDN 기술로 WAN을 최적화합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SD-WAN                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SD-WAN 구성                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │              SD-WAN Controller                       │   │ │
│   │   │              (Central Management)                    │   │ │
│   │   │                      │                               │   │ │
│   │   │        ┌─────────────┼─────────────┐                │   │ │
│   │   │        │             │             │                │   │ │
│   │   │   ┌────┴───┐    ┌────┴───┐    ┌────┴───┐           │   │ │
│   │   │   │ Branch │    │ Branch │    │ Branch │           │   │ │
│   │   │   │  Edge  │    │  Edge  │    │  Edge  │           │   │ │
│   │   │   └────┬───┘    └────┬───┘    └────┬───┘           │   │ │
│   │   │        │             │             │                │   │ │
│   │   │   ┌────┴─────────────┴─────────────┴────┐          │   │ │
│   │   │   │     Internet / MPLS / 4G/5G         │          │   │ │
│   │   │   └─────────────────────────────────────┘          │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                               │   │ │
│   │   │  - 경로 최적화 (Application-aware)                    │   │ │
│   │   │  - MPLS + Internet 혼용                              │   │ │
│   │   │  - 중앙 관리                                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SD-WAN은 애플리케이션 인식 경로 선택을 제공합니다.

**📢 섹션 요약 비유:** SD-WAN = 스마트 교통!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Controller** | 중앙 제어 |
| **Edge** | 지사 장비 |
| **Gateway** | 데이터센터 |
| **Transport** | MPLS, Internet, LTE |

**핵심 알고리즸:** 앱 식별 → 경로 선택 → 전송

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**MPLS vs SD-WAN**

| 항목 | MPLS | SD-WAN |
|:---|:---|:---|
| **비용** | 높음 | 낮음 |
| **유연성** | 낮음 | 높음 |
| **배포** | 느림 | 빠름 |
