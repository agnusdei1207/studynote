# Gateway - 게이트웨이

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 출구
> 2. **가치**: 다른 네트워크 연결
> 3. **융합:** Default Gateway, Router, Protocol Conversion

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Gateway는 서로 다른 네트워크를 연결합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Gateway                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              게이트웨이 역할                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Default Gateway:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │    Local Network              Internet           ││   │ │
│   │   │  │    ┌──────────┐              ┌──────────┐       ││   │ │
│   │   │  │    │192.168.1.x│              │External  │       ││   │ │
│   │   │  │    │          │    Gateway   │Networks  │       ││   │ │
│   │   │  │    └────┬─────┘ ─────────→  └──────────┘       ││   │ │
│   │   │  │         │         192.168.1.1                   ││   │ │
│   │   │  │         │                                        ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  게이트웨이 유형:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Default Gateway: 기본 출구                       ││   │ │
│   │   │  │ Protocol Gateway: 프로토콜 변환                   ││   │ │
│   │   │  │ Application Gateway: 애플리케이션 계층            ││   │ │
```

> **해설**: 게이트웨이는 네트워크의 출입구입니다.

**📢 섹션 요약 비유:** Gateway = 국경!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

| 항목 | 설명 |
|:---|:---|
