# 경로 손실 (Path Loss) - 자유공간/대수

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 거리에 따른 신호 감쇠
> 2. **가치**: 커버리지 설계의 기준
> 3. **융합**: 링크 예산, 안테나와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

경로 손실(Path Loss)은 신호가 거리에 따라 약해지는 현상으로, dB로 표현합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    경로 손실 (Path Loss)                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              자유공간 경로 손실 (FSPL)                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  FSPL (Free Space Path Loss):                         │   │ │
│   │   │                                                      │   │ │
│   │   │  PL(dB) = 20log₁₀(d) + 20log₁₀(f) + 32.44           │   │ │
│   │   │                                                      │   │ │
│   │   │  여기서:                                               │   │ │
│   │   │  • d = 거리 (km)                                      │   │ │
│   │   │  • f = 주파수 (MHz)                                   │   │ │
│   │   │  • 32.44 = 상수 (단위 변환 포함)                       │   │ │
│   │   │                                                      │   │ │
│   │   │  특성:                                                 │   │ │
│   │   │  • 거리 2배 → 6dB 증가                                │   │ │
│   │   │  • 주파수 2배 → 6dB 증가                              │   │ │
│   │   │  • 이상적인 자유공간 기준                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              경로 손실 모델                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 자유공간 (Free Space):                             │   │ │
│   │   │     PL = 20log(d) + 20log(f) + 32.44                 │   │ │
│   │   │     • 이상적 환경                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 대수 경로 손실 (Log-distance):                      │   │ │
│   │   │     PL = PL(d₀) + 10n·log(d/d₀) + Xσ                │   │ │
│   │   │     • n = 경로 손실 지수 (2~6)                         │   │ │
│   │   │     • Xσ = 그림자 페이딩                              │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 실내 경로 손실 (Indoor):                            │   │ │
│   │   │     • ITU-R P.1238                                    │   │ │
│   │   │     • 벽/층 손실 포함                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 옥외 경로 손실 (Outdoor):                           │   │ │
│   │   │     • COST-231 Hata                                   │   │ │
│   │   │     • Okumura-Hata                                    │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 경로 손실은 무선 네트워크 설계의 기본이 됩니다.

**💡 비유**: 경로 손실은 소리가 멀어질수록 작아지는 것과 같습니다.

**📢 섹션 요약 비유:** 거리 2배 = 6dB 손실!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 모델 | 공식 | 환경 | n 값 |
|:---|:---|:---|:---:|
| **자유공간** | 20log(d) | 이상적 | 2 |
| **도심** | 10n·log(d) | 빌딩 밀집 | 3.5-4.5 |
| **실내** | ITU 모델 | 사무실 | 3-6 |
| **지하** | 매우 높음 | 지하철 | > 4 |

**비교표**

```
+------------+----------+----------+
| 환경       │ n 값     │ 특징     │
+------------+----------+----------+
| 자유공간   │ 2        │ 최소     │
| 교외       │ 2-3      │ 낮음     │
| 도심       │ 3-4      │ 높음     │
| 실내       │ 4-6      │ 매우 높음│
+------------+----------+----------+
```

**핵심 알고리즸:** FSPL, Okumura-Hata, COST-231

**📢 섹션 요약 비유:** n = 환경 복잡도!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 주파수별 손실**

| 주파수 | 1km 손실 | 10km 손실 | 용도 |
|:---:|:---:|:---:|:---|
| **900MHz** | 91.5dB | 111.5dB | GSM |
| **2.4GHz** | 100dB | 120dB | WiFi |
| **5GHz** | 106dB | 126dB | WiFi 5G |

**📢 섹션 요약 비유:** 높은 주파수 = 높은 손실!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
