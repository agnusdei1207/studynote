# Circuit Breaker - 서킷 브레이커

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 장애 전파 방지
> 2. **가치**: 복원력, 빠른 실패
> 3. **융합:** Microservices, Resilience, Hystrix

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Circuit Breaker는 장애 전파를 방지합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Circuit Breaker                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              서킷 브레이커 상태                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │              ┌─────────────┐                        │   │ │
│   │   │              │   CLOSED    │                        │   │ │
│   │   │              │  (정상)     │                        │   │ │
│   │   │              └──────┬──────┘                        │   │ │
│   │   │                     │ 실패 임계치 초과               │   │ │
│   │   │                     ↓                                │   │ │
│   │   │              ┌─────────────┐                        │   │ │
│   │   │   Timeout ──→│    OPEN     │                        │   │ │
│   │   │              │  (차단)     │                        │   │ │
│   │   │              └──────┬──────┘                        │   │ │
│   │   │                     │ 시간 경과                      │   │ │
│   │   │                     ↓                                │   │ │
│   │   │              ┌─────────────┐                        │   │ │
│   │   │  실패 ──────→│ HALF-OPEN   │────── 성공 ────→ CLOSED│   │ │
│   │   │              │  (반열림)   │                        │   │ │
│   │   │              └─────────────┘                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 서킷 브레이커는 실패를 감지해 빠르게 차단합니다.

**📢 섹션 요약 비유:** Circuit Breaker = 두꺼비집!

| 항목 | 설명 |
|:---|:---|
| **Closed** | 닫힘 (정상) |
