# MSS - 최대 세그먼트 크기

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: TCP 데이터 최대 크기
> 2. **가치**: 효율적 전송
> 3. **융합:** MTU, TCP Option, SYN

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MSS (Maximum Segment Size)는 TCP 데이터의 최대 크기입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MSS                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MSS 계산                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  MSS 계산:                                               │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  MSS = MTU - IP Header - TCP Header             ││   │ │
│   │   │  │       = 1500 - 20 - 20                          ││   │ │
│   │   │  │       = 1460 bytes                              ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌────────────────────────────────────────┐     ││   │ │
│   │   │  │  │ IP Header │ TCP Header │ TCP Data     │     ││   │ │
│   │   │  │  │   20B     │    20B     │   1460B      │     ││   │ │
│   │   │  │  └────────────────────────────────────────┘     ││   │ │
│   │   │  │       ←────────── 1500 bytes ───────────→       ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  MSS 협상:                                              │   │ │
│   │   │  SYN 패킷의 TCP Option으로 교환                          │   │ │
│   │   │  양쪽이 각자 MSS 제안 → 작은 값 선택                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MSS는 TCP 성능 최적화의 핵심입니다.

**📢 섹션 요약 비유:** MSS = 박스 내용물 크기!
