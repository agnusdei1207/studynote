### Ⅰ. 개요 (Context & Background)

**개념 정의**

Fencing은 문제 노드를 격리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Fencing                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Fencing 유형                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Storage Fencing:                                     │   │ │
│   │   │  - 스토리지 접근 차단                                   │   │ │
│   │   │  - SCSI Reservation                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  Network Fencing:                                     │   │ │
│   │   │  - 네트워크 격리                                       │   │ │
│   │   │  - VLAN 차단                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  Power Fencing (STONITH):                              │   │ │
│   │   │  - 전원 차단                                           │   │ │
│   │   │  - IPMI, iLO, DRAC                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  시나리오:                                              │   │ │
│   │   │  Node1 장애 → Fence → Node2가 리더 인계                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Fencing은 데이터 무결성을 보호합니다.

**📢 섹션 요약 비유:** Fencing = 격리 조치!
