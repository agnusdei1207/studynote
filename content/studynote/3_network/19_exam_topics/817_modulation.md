# Modulation - 변조

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 변환
> 2. **가치**: 전송 효율
> 3. **융합:** AM, FM, QAM

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Modulation은 정보를 반송파에 실습니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Modulation                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              변조 방식                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  AM (Amplitude Modulation):                          │   │ │
│   │   │  진폭 변조                                            │   │ │
│   │   │  ┌─────────────────────────────┐                    │   │ │
│   │   │  │ ∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿∿ │ ← 크기 변화               │   │ │
│   │   │  └─────────────────────────────┘                    │   │ │
│   │   │  라디오 AM                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  FM (Frequency Modulation):                          │   │ │
│   │   │  주파수 변조                                          │   │ │
│   │   │  ┌─────────────────────────────┐                    │   │ │
│   │   │  │ ∿∾∾∾∿∿∿∾∾∾∿∿∿∾∾∾ │ ← 주파수 변화               │   │ │
│   │   │  └─────────────────────────────┘                    │   │ │
│   │   │  라디오 FM, TV                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  PM (Phase Modulation):                              │   │ │
│   │   │  위상 변조                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  QAM (Quadrature AM):                                │   │ │
│   │   │  진폭 + 위상 변조                                     │   │ │
```

**📢 섹션 요약 비유:** Modulation = 신호 포장!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

| 방식 | 특징 | 용도 |
|:---|:---|:---|
| **AM** | 단순 | 라디오 |
| **FM** | 잡음 강함 | FM 라디오 |
