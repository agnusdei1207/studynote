# Auto-MDIX (크로스 케이블 자동 인식)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: MDI/MDI-X 자동 감지 및 전환
> 2. **가치**: 케이블 타입 무관 연결
> 3. **융합**: Auto-Negotiation, 기가비트과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Auto-MDIX(Automatic Medium Dependent Interface Crossover)는 연결된 케이블 타입을 자동 감지하여 MDI/MDI-X를 자동으로 전환하는 기능입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Auto-MDIX                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Auto-MDIX 동작                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  문제 상황 (Auto-MDIX 없이):                            │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌───┐              ┌───┐                            │   │ │
│   │   │  │PC │─다이렉트케이블─│PC │  → 연결 안 됨!              │   │ │
│   │   │  │MDI│              │MDI│                            │   │ │
│   │   │  └───┘              └───┘                            │   │ │
│   │   │    ↑                    ↑                             │   │ │
│   │   │   TX+                  TX+ (교차 필요)                 │   │ │
│   │   │                                                      │   │ │
│   │   │  Auto-MDIX 동작:                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌───┐              ┌───┐                            │   │ │
│   │   │  │PC │─아무케이블─────│PC │  → 자동 연결!              │   │ │
│   │   │  │   │              │   │                            │   │ │
│   │   │  │자동│   감지→전환   │자동│                            │   │ │
│   │   │  │MDI↔MDI-X          │MDI↔MDI-X                       │   │ │
│   │   │  └───┘              └───┘                            │   │ │
│   │   │                                                      │   │ │
│   │   │  작동 원리:                                             │   │ │
│   │   │  1. 상대방 신호 감지                                    │   │ │
│   │   │  2. TX/RX 핀 교차 여부 판단                             │   │ │
│   │   │  3. 필요시 내부적으로 교차                              │   │ │
│   │   │  4. 연결 성공                                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Auto-MDIX 지원                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  지원 속도:                                               │   │ │
│   │   │  • 100BASE-TX: 지원                                      │   │ │
│   │   │  • 1000BASE-T: 기본 지원                                 │   │ │
│   │   │  • 10GBASE-T: 기본 지원                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 케이블 타입 고민 해결                                  │   │ │
│   │   │  • 크로스 케이블 필요 없음                                │   │ │
│   │   │  • 설치 실수 방지                                        │   │ │
│   │   │  • 관리 편의성                                           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: Auto-MDIX는 기가비트 이더넷에서 기본적으로 지원됩니다.

**💡 비유**: Auto-MDIX는 자동 변속 자동차처럼 상황에 맞게 자동 조정합니다.

**📢 섹션 요약 비유:** Auto-MDIX = 케이블 고민 끝!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 레거시 100M** → Auto-MDIX 확인
**시나리오 2: 1G/10G** → 자동 지원
**시나리오 3: 불량 케이블** → Auto-MDIX로도 해결 안 됨

**도입 체크리스트**: 속도, 장비 지원 여부

**📢 섹션 요약 비유:** Auto-MDIX는 설치의 친구!
