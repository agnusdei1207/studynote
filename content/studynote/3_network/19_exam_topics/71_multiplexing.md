# 다중화 (Multiplexing)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 하나의 전송로에 여러 신호 동시 전송
> 2. **가치**: 전송 효율, 비용 절감, 대역폭 활용
> 3. **융합**: FDM, TDM, WDM, CDM으로 확장

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

다중화(Multiplexing)는 하나의 전송 매체를 통해 여러 신호를 동시에 전송하는 기술입니다.

```

│   │   │  입력2 ──┼──→ [다중화기 MUX] ──→ 전송로 ──→         │   │ │
│   │   │  입력3 ──┤                                           │   │ │
│   │   │  입력4 ──┘                                           │   │ │
```

> **해설**: 다중화는 전송 효율을 높이는 핵심 기술입니다.

**💡 비유**: 다중화는 하나의 도로에 여러 차선을 만드는 것입니다.

**📢 섹션 요약 비유**: MUX = 여러 신호를 하나로!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **MUX** | 다중화 | 신호 결합 | 병합기 |
| **DEMUX** | 역다중화 | 신호 분리 | 분배기 |
| **전송로** | 매체 | 광케이블, 동축 | 도로 |
| **채널** | 논리 경로 | 시간/주파수 분할 | 차선 |

**비교표**

```
+------------+----------+----------+----------+
| 방식       | 자원     | 아날로그 | 디지털   |
+------------+----------+----------+----------+
| FDM        | 주파수   | O        | O        |
| TDM        | 시간     | X        | O        |
| WDM        | 파장     | O        | O        |
| CDM        | 코드     | X        | O        |
| SDM        | 공간     | O        | O        |
+------------+----------+----------+----------+
```

**핵심 알고리즘**: N개 입력 → 자원 분할 → 전송 → 자원 역분할 → N개 출력

**📢 섹션 요약 비유**: n:1 MUX → 1:n DEMUX!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 다중화 효율**

| 방식 | 장점 | 단점 | 응용 |
|:---|:---|:---|:---|
| **FDM** | 단순 | 보호대역 필요 | 라디오 |
| **TDM** | 효율 | 동기 필요 | 전화망 |
| **WDM** | 대용량 | 광케이블 | 백본 |
| **CDM** | 보안 | 복잡 | CDMA |

**📢 섹션 요약 비유**: 다중화는 전송 효율의 핵심!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 전화망** → TDM (E1/T1)
**시나리오 2: 광통신** → WDM (DWDM)
**시나리오 3: 이동통신** → CDMA/OFDMA

**도입 체크리스트**: 대역폭, 신호 수, 동기, 비용

**📢 섹션 요약 비유**: 목적에 맞는 다중화 선택!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 전송 효율 N배 증가, 비용 절감

**미래 전망**: FlexE, OTN 진화

**참고 표준**: ITU-T G.7xx, IEEE 802.3

**📢 섹션 요약 비유**: 다중화는 통신 경제성의 열쇠!
