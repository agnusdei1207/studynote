# 매체 구분: 유도 매체 vs 비유도 매체

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전송 매체의 물리적 구분
> 2. **가치**: 유도는 케이블, 비유도는 무선
> 3. **융합**: 유선/무선 하이브리드 네트워크

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

전송 매체는 데이터를 전송하는 물리적 경로로, 유도 매체(Guided)는 케이블, 비유도 매체(Unguided)는 무선 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    매체 구분 (Media Types)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전송 매체 분류                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                    전송 매체                          │   │ │
│   │   │                        │                               │   │ │
│   │   │         ┌──────────────┴──────────────┐               │   │ │
│   │   │         ▼                              ▼               │   │ │
│   │   │    유도 매체                       비유도 매체          │   │ │
│   │   │   (Guided)                       (Unguided)           │   │ │
│   │   │        │                              │               │   │ │
│   │   │   ┌────┼────┐                    ┌───┼───┐            │   │ │
│   │   │   ▼    ▼    ▼                    ▼   ▼   ▼            │   │ │
│   │   │ 꼬임  동축  광섬유               전파  마이크로파  위성  │   │ │
│   │   │ 쌍선  케이블  케이블                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  특징 비교:                                             │   │ │
│   │   │  ┌─────────────┬─────────────┬─────────────┐          │   │ │
│   │   │  │   항목      │  유도 매체   │ 비유도 매체  │          │   │ │
│   │   │  ├─────────────┼─────────────┼─────────────┤          │   │ │
│   │   │  │ 물리 경로   │   있음      │   없음      │          │   │ │
│   │   │  │ 전파 방식   │   전도      │   복사      │          │   │ │
│   │   │  │ 대역폭      │   높음      │   제한      │          │   │ │
│   │   │  │ 보안        │   높음      │   낮음      │          │   │ │
│   │   │  │ 이동성      │   낮음      │   높음      │          │   │ │
│   │   │  │ 비용        │   설치 비용  │   라이선스  │          │   │ │
│   │   │  └─────────────┴─────────────┴─────────────┘          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 유도 매체는 신호가 물리적 매체를 따라 전달되고, 비유도 매체는 자유 공간으로 전파됩니다.

**💡 비유**: 유도 매체는 파이프를 통한 물 흐름, 비유도 매체는 공기 중 물 뿌리기입니다.

**📢 섹션 요약 비유**: 유도 = 케이블, 비유도 = 무선!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **유도 매체** | 유선 전송 | 전도/굴절 | 파이프 |
| **비유도 매체** | 무선 전송 | 복사/산란 | 공기 |
| **감쇠** | 신호 약화 | 거리 비례 | 마찰 |

**비교표**

```
+------------+----------+----------+
| 항목       | 유도     | 비유도   |
+------------+----------+----------+
| 경로       | 고정     | 자유     |
| 간섭       | 낮음     | 높음     |
| 설치       | 복잡     | 간단     |
| 확장       | 어려움   | 쉬움     |
+------------+----------+----------+
```

**핵심 알고리즸**: 신호 생성 → 매체 전달 → 수신 복원

**📢 섹션 요약 비유**: 유선은 안정, 무선은 유연!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 유도 매체**

|:---|:---:|:---:|

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 고정 설치** → 유도 매체
**시나리오 2: 이동성 필요** → 비유도 매체
**시나리오 3: 장거리** → 광섬유 + 무선

**도입 체크리스트**: 이동성, 대역폭, 보안, 비용

**📢 섹션 요약 비유**: 상황에 맞는 매체!
