# OUI - 조직 고유 식별자

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: MAC 주소 제조사 코드
> 2. **가치**: 장비 제조사 식별
> 3. **융합:** IEEE, MAC Prefix, Vendor

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OUI (Organizationally Unique Identifier)는 MAC 주소의 제조사 식별자입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OUI                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OUI 구조                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  MAC 주소와 OUI:                                         │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  MAC: AA:BB:CC:DD:EE:FF                          ││   │ │
│   │   │  │       └────┬────┘                               ││   │ │
│   │   │  │            │                                    ││   │ │
│   │   │  │          OUI (24비트)                            ││   │ │
│   │   │  │       제조사 식별 코드                            ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  예시:                                          ││   │ │
│   │   │  │  - 00:1A:2B:xx:xx:xx → Cisco                   ││   │ │
│   │   │  │  - 00:50:56:xx:xx:xx → VMware                  ││   │ │
│   │   │  │  - 00:03:93:xx:xx:xx → Apple                   ││   │ │
│   │   │  │  - 00:0C:29:xx:xx:xx → VMware                  ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  용도:                                                  │   │ │
│   │   │  - 장비 제조사 확인                                     │   │ │
│   │   │  - 네트워크 문제 해결                                    │   │ │
│   │   │  - 보안 분석                                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: OUI는 장비 제조사를 식별합니다.

**📢 섹션 요약 비유:** OUI = 제조사 브랜드!
