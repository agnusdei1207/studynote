# ARP (Address Resolution Protocol) - 주소 결정 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IP → MAC 주소 변환
> 2. **가치**: 로컬 네트워크 통신
> 3. **융합:** Ethernet, MAC Table, ARP Cache

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ARP는 IP 주소로 MAC 주소를 찾습니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ARP                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ARP 동작                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Host A (IP: 192.168.1.10, MAC: aa:bb:cc:11:22:33)  │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ "192.168.1.20의 MAC이 뭐야?"                 │   │ │
│   │   │       │ ── ARP Request (Broadcast) ──────────────→   │   │ │
│   │   │       │                                              │   │ │
│   │   │  Host B (IP: 192.168.1.20, MAC: dd:ee:ff:44:55:66)  │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ ←── ARP Reply (Unicast) ──────────────────   │   │ │
│   │   │       │ "내 MAC은 dd:ee:ff:44:55:66이야"             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: ARP는 로컬 네트워크에서 MAC 주소를 획득합니다.

**📢 섹션 요약 비유:** ARP = 전화번호부!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **ARP Request** | 브로드캐스트 |
| **ARP Reply** | 유니캐스트 |
| **ARP Cache** | 캐시 테이블 |
| **ARP Timeout** | 캐시 만료 |

**핵심 알고리즸:** Request → Reply → Cache 저장

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**ARP 관련 프로토콜**

| 프로토콜 | 방향 |
|:---|:---|
| **ARP** | IP → MAC |
| **RARP** | MAC → IP |
| **Gratuitous ARP** | 자신의 IP 알림 |
| **Proxy ARP** | 대신 응답 |
