# HTTP Status Codes - HTTP 상태 코드

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: HTTP 응답 결과 분류
> 2. **가치**: 오류 진단, 클라이언트 처리
> 3. **융합:** HTTP Response, Error Handling

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

HTTP Status Code는 3자리 숫자로 응답 결과를 분류합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    HTTP Status Codes                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              상태 코드 분류                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1xx Informational: 요청 진행 중                     │   │ │
│   │   │  2xx Success: 요청 성공                              │   │ │
│   │   │  3xx Redirection: 추가 조치 필요                     │   │ │
│   │   │  4xx Client Error: 클라이언트 오류                   │   │ │
│   │   │  5xx Server Error: 서버 오류                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 상태 코드 첫 자리는 클래스를 나타냅니다.

**📢 섹션 요약 비유:** Status Code = 주문 결과!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 코드 | 설명 |
|:---|:---|
| **200** | OK |
| **201** | Created |
| **301** | Moved Permanently |
| **302** | Found |
| **304** | Not Modified |
| **400** | Bad Request |
| **401** | Unauthorized |
| **403** | Forbidden |
| **404** | Not Found |
| **500** | Internal Server Error |
| **502** | Bad Gateway |
| **503** | Service Unavailable |

**핵심 알고리즸:** 응답 생성 → 상태 코드 할당

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**클래스별 주요 코드**

|:---|:---|
