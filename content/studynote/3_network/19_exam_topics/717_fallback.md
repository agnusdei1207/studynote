# Fallback - 폴백

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 대체 동작
> 2. **가치**: 서비스 연속성
> 3. **융합:** Circuit Breaker, Cache, Default Value

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Fallback은 실패 시 대체 동작을 수행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Fallback                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              폴백 전략                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client ──→ Primary Service                          │   │ │
│   │   │              │                                       │   │ │
│   │   │              ↓ (실패)                                │   │ │
│   │   │         ┌─────────────────────────────────────┐     │   │ │
│   │   │         │  Fallback Options:                   │     │   │ │
│   │   │         │  - Cached Data                       │     │   │ │
│   │   │         │  - Default Value                     │     │   │ │
│   │   │         │  - Alternative Service               │     │   │ │
│   │   │         │  - Graceful Degradation              │     │   │ │
│   │   │         │  - Error Message                     │     │   │ │
│   │   │         └─────────────────────────────────────┘     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 폴백은 서비스 중단을 방지합니다.

**📢 섹션 요약 비유:** Fallback = 비상구!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Cache** | 캐시된 데이터 |
| **Default** | 기본값 |
