# IP Fragmentation - IP 단편화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 큰 패킷을 MTU에 맞게 분할
> 2. **가치**: 이기종 네트워크 연결
> 3. **융합:** MTU, DF, Reassembly

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IP Fragmentation은 큰 패킷을 작은 조각으로 나눕니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IP Fragmentation                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              단편화 과정                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  원본 IP 패킷 (4000바이트)                           │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ IP Hdr │         Data (3980바이트)          │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                        ↓                             │   │ │
│   │   │                   단편화 (MTU 1500)                  │   │ │
│   │   │                        ↓                             │   │ │
│   │   │  ┌──────────┐ ┌──────────┐ ┌──────────┐            │   │ │
│   │   │  │IP|Data   │ │IP|Data   │ │IP|Data   │            │   │ │
│   │   │  │ (1500)   │ │ (1500)   │ │ (1000)   │            │   │ │
│   │   │  │ Offset=0 │ │ Offset=1 │ │ Offset=2 │            │   │ │
│   │   │  │ MF=1     │ │ MF=1     │ │ MF=0     │            │   │ │
│   │   │  └──────────┘ └──────────┘ └──────────┘            │   │ │
│   │   │                                                      │   │ │
```

> **해설**: 단편화는 중간 라우터가 아닌 최종 목적지에서 재조립됩니다.

**📢 섹션 요약 비유:** Fragmentation = 짐 나누기!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 필드 | 설명 |
|:---|:---|
| **Identification** | 같은 패킷 식별 |
| **Flags** | DF, MF |
| **Fragment Offset** | 조각 위치 |
| **DF=1** | 단편화 금지 |
| **MF=1** | 더 있음 |

**핵심 알고리즸:** size > MTU && DF=0 → fragment
