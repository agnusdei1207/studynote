# 패킷 캡처 - Packet Capture

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 패킷 수집
> 2. **가치**: 심층 분석, 보안
> 3. **융합**: 트러블슈팅, 모니터링과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

패킷 캡처는 네트워크를 지나는 패킷을 수집하고 분석합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    패킷 캡처                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              패킷 캡처 방식                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. Port Mirroring (SPAN):                            │   │ │
│   │   │                                                      │   │ │
│   │   │     [Traffic] ──▶ [Switch Port 1]                    │   │ │
│   │   │                          │                            │   │ │
│   │   │                          │ Mirror                     │   │ │
│   │   │                          ▼                            │   │ │
│   │   │                    [Capture Port] ──▶ [Analyzer]      │   │ │
│   │   │                                                      │   │ │
│   │   │  2. TAP (Test Access Point):                          │   │ │
│   │   │                                                      │   │ │
│   │   │     [Traffic] ──▶ [TAP] ──▶ [Analyzer]               │   │ │
│   │   │                        │                              │   │ │
│   │   │                        └──▶ [Network] (계속)          │   │ │
│   │   │                                                      │   │ │
│   │   │  3. In-line:                                          │   │ │
│   │   │                                                      │   │ │
│   │   │     [Traffic] ──▶ [Capture Device] ──▶ [Network]     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              캡처 도구                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 도구       │ 플랫폼   │ 특징     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ tcpdump    │ Linux    │ CLI      │               │   │ │
│   │   │  │ Wireshark  │ GUI      │ 분석     │               │   │ │
│   │   │  │ tshark     │ CLI      │ 스크립트 │               │   │ │
│   │   │  │ SolarWinds │ 상용     │ 엔터프라이즈│              │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 패킷 캡처는 미러링, TAP, 인라인 방식으로 수행합니다.

**💡 비유**: 패킷 캡처는 우편물을 몰래 복사하는 것입니다.

**📢 섹션 요약 비유:** 패킷 캡처 = 심층 분석!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **libpcap** | 캡처 라이브러리 |
| **BPF** | 필터 |
| **Promiscuous** | 무차별 모드 |
| **Ring Buffer** | 버퍼 |

**비교표: 캡처 방식**

```
+------------+----------+----------+
| 방식       │ 장점     │ 단점     │
+------------+----------+----------+
| SPAN       │ 간편     │ 부하     │
| TAP        │ 정확     │ 비용     │
| In-line    │ 실시간   │ 지연     │
+------------+----------+----------+
```

**핵심 알고리즸:** BPF 필터링

**📢 섹션 요약 비유:** BPF = 필터!
