# RADIUS - 원격 인증 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: AAA 인증/인가/계정 프로토콜
> 2. **가치**: 중앙 집중식 사용자 관리
> 3. **융합**: VPN, WiFi, 네트워크 장비

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

RADIUS(Remote Authentication Dial-In User Service)는 AAA 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    RADIUS                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RADIUS AAA                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Client]                                           │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ 1. 접속 요청                                   │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  [NAS] ──────▶ [RADIUS Server]                       │   │ │
│   │   │     │              │                                 │   │ │
│   │   │     │              │ Authentication                  │   │ │
│   │   │     │              │ Authorization                   │   │ │
│   │   │     │              │ Accounting                      │   │ │
│   │   │     │              ▼                                 │   │ │
│   │   │  [Access-Accept/Reject]                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RADIUS 패킷                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 패킷 타입  │ 용도     │ 코드     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ Access-Req│ 인증요청 │ 1        │               │   │ │
│   │   │  │ Access-Acc│ 인증성공 │ 2        │               │   │ │
│   │   │  | Access-Rej│ 인증거부 │ 3        │               │   │ │

│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: RADIUS는 UDP 1812/1813 포트를 사용합니다.

**💡 비유**: RADIUS는 출입관리소입니다. 들어오는 사람을 확인합니다.

**📢 섹션 요약 비유:** RADIUS = AAA!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NAS** | Network Access Server |
| **RADIUS Server** | 인증 서버 |
| **Shared Secret** | 공유 비밀키 |
| **Attributes** | 사용자 속성 |

**비교표: RADIUS vs TACACS+**

```
+------------+----------+----------+
| 항목       │ RADIUS   │ TACACS+  │
+------------+----------+----------+
| 프로토콜    │ UDP      │ TCP      │
| 암호화     │ 비밀번호만│ 전체     │
| 용도       │ 네트워크 │ 장비     │
| 포트       │ 1812/1813│ 49       │
+------------+----------+----------+
```

**핵심 알고리즸:** MD5 해시, 속성 검증

**📢 섹션 요약 비유:** WiFi = RADIUS!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: AAA 프로토콜**

| 항목 | RADIUS | TACACS+ | Diameter |
|:---|:---|:---|:---|
| **프로토콜** │ UDP │ TCP │ TCP/SCTP |
| **확장성** │ 제한 │ 중간 │ 높음 |
| **모바일** │ 제한 │ 아니오 │ 적합 |

**📢 섹션 요약 비유:** WiFi/VPN = RADIUS!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 기업 WiFi → RADIUS (802.1X)
**시나리오 2:** VPN → RADIUS
**시나리오 3:** 네트워크 장비 → TACACS+

**도입 체크리스트:** 서버 선정, 백엔드 연동, 고가용성

**📢 섹션 요약 비유:** 네트워크 인증 = RADIUS!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 중앙 집중식 AAA

**미래 전망:** Diameter로 점진적 전환

**참고 표준:** RFC 2865, RFC 2866

**📢 섹션 요약 비유:** RADIUS는 AAA 표준!
