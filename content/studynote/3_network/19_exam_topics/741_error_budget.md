### Ⅰ. 개요 (Context & Background)

**개념 정의**

Error Budget은 허용 가능한 장애 시간입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Error Budget                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              에러 예산 계산                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  SLO: 99.9% 가용성                                    │   │ │
│   │   │  기간: 30일                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  Error Budget 계산:                                   │   │ │
│   │   │  30일 × 24시간 × 60분 × (1 - 0.999)                  │   │ │
│   │   │  = 43,200분 × 0.001 = 43.2분                         │   │ │
│   │   │                                                      │   │ │
│   │   │  활용:                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 예산 있음 → 새 기능 개발                         ││   │ │
│   │   │  │ 예산 없음 → 안정성 개선                          ││   │ │
│   │   │  │ 예산 초과 → 기능 동결                            ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 에러 예산은 혁신과 안정성의 균형을 맞춥니다.

**📢 섹션 요약 비유:** Error Budget = 연차!
