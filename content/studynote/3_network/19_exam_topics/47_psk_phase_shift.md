# PSK (위상 편이 변조)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 반송파의 위상으로 디지털 데이터 표현
> 2. **가치**: 잡음 내성 강함, 대역폭 효율
> 3. **융합**: BPSK, QPSK, 8PSK, 위성/WiFi와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PSK(Phase Shift Keying)는 디지털 데이터에 따라 반송파의 위상을 변화시키는 디지털 변조 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PSK (위상 편이 변조)                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              BPSK (Binary PSK)                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  위상 매핑:                                             │   │ │
│   │   │  • 0: 위상 0°                                           │   │ │
│   │   │  • 1: 위상 180°                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  성상도 (Constellation):                                │   │ │
│   │   │         0°                                             │   │ │
│   │   │          ●────────────→                                │   │ │
│   │   │          │                                             │   │ │
│   │   │          │                                             │   │ │
│   │   │          ←────────────●                                │   │ │
│   │   │         180°                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 1 비트/심볼                                         │   │ │
│   │   │  • 가장 강건한 PSK                                     │   │ │
│   │   │  • 대역폭 효율 낮음                                    │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              QPSK (Quadrature PSK)                            │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  위상 매핑:                                             │   │ │
│   │   │  • 00: 45° (π/4)                                       │   │ │
│   │   │  • 01: 135° (3π/4)                                     │   │ │
│   │   │  • 11: 225° (5π/4)                                     │   │ │
│   │   │  • 10: 315° (7π/4)                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  성상도:                                                │   │ │
│   │   │        135° ●                                          │   │ │
│   │   │            │╲    ╱│                                    │   │ │
│   │   │            │  ╲╱  │                                    │   │ │
│   │   │    180° ───┼───┼─── 0°                                │   │ │
│   │   │            │  ╱╲  │                                    │   │ │
│   │   │            │╱    ╲│                                    │   │ │
│   │   │         225° ●    ● 45°                                │   │ │
│   │   │              ●                                         │   │ │
│   │   │             315°                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 2 비트/심볼                                         │   │ │
│   │   │  • BPSK 대비 2배 효율                                  │   │ │
│   │   │  • 위성, LTE 다운링크                                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: PSK는 잡음 내성이 강하고 대역폭 효율이 좋습니다.

**💡 비유**: PSK는 시계 바늘의 위치로 정보를 전달합니다.

**📢 섹션 요약 비유**: BPSK = 1비트, QPSK = 2비트!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | BPSK | QPSK | 8PSK |
|:---|:---:|:---:|:---:|
| **위상 수** | 2 | 4 | 8 |
| **비트/심볼** | 1 | 2 | 3 |
| **위상 간격** | 180° | 90° | 45° |
| **BER** | 최저 | 낮음 | 중간 |

**PSK 비교표**

```
+------------+----------+----------+----------+
| 항목       | BPSK     | QPSK     | 8PSK     |
+------------+----------+----------+----------+
| 심볼       | 1비트    | 2비트    | 3비트    |
| 위상       | 2        | 4        | 8        |
| Eb/N0      | 9.6dB    | 9.6dB    | 12.8dB   |
| 응용       | GPS      | 위성     | 위성     |
+------------+----------+----------+----------+
```

**핵심 알고리즘**: 비트 → 위상 매핑 → 반송파 생성 → 전송

**📢 섹션 요약 비유**: 위성 통신은 QPSK가 표준!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 변조 효율**

| 변조 | 비트/심볼 | BER 성능 |
|:---|:---:|:---:|
| **BPSK** | 1 | 최고 |
| **QPSK** | 2 | 좋음 |
| **8PSK** | 3 | 보통 |
| **16QAM** | 4 | 낮음 |

**📢 섹션 요약 비유**: 심볼당 비트 ↑ = 잡음 내성 ↓!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: GPS** → BPSK
**시나리오 2: 위성 DVB** → QPSK/8PSK
**시나리오 3: LTE** → QPSK~256QAM 적응형

**도입 체크리스트**: SNR, 대역폭, 전력

**📢 섹션 요약 비유**: 좋은 채널 = 높은 변조 차수!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 잡음 내성 + 대역폭 효율

**미래 전망**: 적응형 변조 (AMC)

**참고 표준**: DVB-S2, LTE, GPS

**📢 섹션 요약 비유**: PSK는 무선 통신의 척추!
