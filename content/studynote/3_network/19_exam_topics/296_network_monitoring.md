# 네트워크 모니터링 (Network Monitoring)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 상태 관찰

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **SNMP** | 장비 상태 |
| **NetFlow** | 트래픽 |
| **syslog** | 로그 |
| **ICMP** | 연결성 |

**비교표: 모니터링 방식**

```
+------------+----------+----------+
| 방식       │ 특징     │ 용도     │
+------------+----------+----------+
| Polling    │ 능동     │ 상태     │
| Trapping   │ 수동     │ 이벤트   │
| Flow       │ 샘플링   │ 트래픽   │
| Streaming  │ 실시간   │ 분석     │
+------------+----------+----------+
```

**핵심 알고리즸:** Polling 간격, 임계값

**📢 섹션 요약 비유:** Polling vs Trap!
