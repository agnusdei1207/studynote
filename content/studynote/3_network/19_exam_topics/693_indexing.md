# Indexing - 인덱싱

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 빠른 검색 구조
> 2. **가치**: 조회 성능 향상
> 3. **융합:** B-Tree, Hash, Full-text

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Indexing은 데이터를 빠르게 검색하는 구조입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Indexing                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              B-Tree 인덱스                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │                    [50]                              │   │ │
│   │   │                   ╱    ╲                             │   │ │
│   │   │               [25]      [75]                         │   │ │
│   │   │              ╱   ╲     ╱   ╲                         │   │ │
│   │   │          [10]   [40] [60]  [90]                      │   │ │
│   │   │            │     │    │     │                        │   │ │
│   │   │            ↓     ↓    ↓     ↓                        │   │ │
│   │   │          Data  Data Data  Data                       │   │ │
│   │   │                                                      │   │ │
│   │   │  시간 복잡도: O(log n)                               │   │ │
│   │   │  정렬된 순서 유지                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 인덱스는 검색 속도를 O(n)에서 O(log n)로 향상시킵니다.

**📢 섹션 요약 비유:** Indexing = 색인!

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **B-Tree** | 균형 트리 |
| **Hash** | 해시 테이블 |
