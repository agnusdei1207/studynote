# Message Queue - 메시지 큐

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 비동기 메시지 전달
> 2. **가치**: 결합도 감소, 확장성
> 3. **융합:** RabbitMQ, Kafka, SQS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Message Queue는 비동기 메시지 전달을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Message Queue                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MQ 구조                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Producer                Queue                Consumer│   │ │
│   │   │  ┌─────┐               ┌─────────┐            ┌─────┐│   │ │
│   │   │  │     │   Message     │ M1 M2 M3│   Consume  │     ││   │ │
│   │   │  │ App │──────────────→│ ↓ ↓ ↓   │───────────→│ App ││   │ │
│   │   │  │     │               │         │            │     ││   │ │
│   │   │  └─────┘               └─────────┘            └─────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                               │   │ │
│   │   │  - 비동기 (Asynchronous)                             │   │ │
│   │   │  - 비동기 (Decoupling)                               │   │ │
│   │   │  - 버퍼링 (Buffering)                                │   │ │
│   │   │  - 확장성 (Scalability)                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MQ는 생산자와 소비자를 분리합니다.

**📢 섹션 요약 비유:** Message Queue = 우편함!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Producer** | 메시지 생산자 |
| **Consumer** | 메시지 소비자 |
| **Queue** | 메시지 저장소 |
| **Exchange** | 라우팅 (RabbitMQ) |
| **Broker** | MQ 서버 |

**핵심 알고리즸:** Produce → Queue → Consume
