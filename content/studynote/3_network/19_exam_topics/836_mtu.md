# MTU - 최대 전송 단위

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 최대 프레임 크기
> 2. **가치**: 단편화 방지
> 3. **융합:** Ethernet, Jumbo Frame, Path MTU

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MTU (Maximum Transmission Unit)는 한 번에 전송 가능한 최대 크기입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MTU (Maximum Transmission Unit)                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MTU 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Ethernet Frame (MTU = 1500 bytes):                   │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Dst │ Src │ Type │   Payload (1500B max)   │FCS ││   │ │
│   │   │  │  6  │  6  │  2   │                         │ 4  ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │  ←────────── 1518 bytes (표준) ────────────────→     │   │ │
│   │   │                                                      │   │ │
│   │   │  네트워크별 MTU:                                       │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Ethernet:        1500 bytes                      ││   │ │
│   │   │  │ PPPoE:           1492 bytes                      ││   │ │
│   │   │  │ VPN/GRE:         1476 bytes                      ││   │ │
│   │   │  │ Jumbo Frame:     9000 bytes                      ││   │ │
│   │   │  │ IPv6 최소:       1280 bytes                      ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  MSS (Maximum Segment Size):                          │   │ │
│   │   │  MSS = MTU - IP Header - TCP Header                   │   │ │
│   │   │  MSS = 1500 - 20 - 20 = 1460 bytes                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MTU는 네트워크 효율과 단편화에 직접적 영향을 미칩니다.

**📢 섹션 요약 비유:** MTU = 택배 최대 크기!
