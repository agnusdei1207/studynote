# State Machine - 상태 머신

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 상태 전이 모델
> 2. **가치**: 프로토콜 동작 정의
> 3. **융합:** TCP States, FSM, Protocol Design

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

State Machine은 상태와 전이로 동작을 모델링합니다.

```

│   │   │             │                                        │   │ │

│   │   │     ┌─────────────┐         ┌──────────┐             │   │ │
```

> **해설**: 상태 머신은 프로토콜의 정확한 동작을 정의합니다.

**📢 섹션 요약 비유:** State Machine = 교통신호!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

| 항목 | 설명 |
|:---|:---|
| **State** | 상태 |
| **Event** | 이벤트 |

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**TCP 상태 종류**

| 상태 | 설명 |
|:---|:---|
| **CLOSED** | 연결 없음 |
| **LISTEN** | 대기 |
| **SYN_SENT** | SYN 전송 |
| **ESTABLISHED** | 연결됨 |
| **FIN_WAIT** | 종료 대기 |
