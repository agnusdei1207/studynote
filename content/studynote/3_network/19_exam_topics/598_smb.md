# SMB (Server Message Block) - 서버 메시지 블록

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 윈도우 파일 공유 프로토콜
> 2. **가치**: 파일/프린터 공유
> 3. **융합:** NAS, CIFS, Windows

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SMB는 윈도우 파일/프린터 공유 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SMB                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SMB 동작                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Windows Client                    SMB Server        │   │ │
│   │   │  ┌───────────┐                    ┌───────────┐      │   │ │
│   │   │  │   PC      │ ── SMB (TCP 445) ─→│ File Share│      │   │ │
│   │   │  │ Z:\Drive  │                    │ Printer   │      │   │ │
│   │   │  └───────────┘                    └───────────┘      │   │ │
│   │   │                                                      │   │ │
│   │   │  기능:                                               │   │ │
│   │   │  - 파일/폴더 접근                                    │   │ │
│   │   │  - 프린터 공유                                       │   │ │
│   │   │  - 인증 (NTLM/Kerberos)                              │   │ │
│   │   │                                                      │   │ │
│   │   │  버전:                                               │   │ │
│   │   │  - SMB 1.0: 레거시 (취약)                            │   │ │
│   │   │  - SMB 2.0: Vista+                                  │   │ │
│   │   │  - SMB 3.0: Win8+ (암호화)                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SMB는 윈도우 환경 파일 공유의 표준입니다.

**📢 섹션 요약 비유:** SMB = 윈도우 공유 폴더!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Port 445** | SMB Direct |
| **Port 139** | NetBIOS (구형) |
| **NTLM** | 인증 |
| **Kerberos** | AD 인증 |
| **DFS** | 분산 파일 시스템 |

**핵심 알고리즸:** 인증 → 세션 → 파일 작업

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**SMB 버전**

| 버전 | 특징 |
|:---|:---|
| **SMB 1.0** | 레거시, 취약 |
| **SMB 2.0** | 성능 개선 |
| **SMB 3.0** | 암호화, 멀티채널 |
