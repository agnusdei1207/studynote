# 멀티모드 계단형 광섬유 (Multi-mode Step-index)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 균일한 코어 굴절률, 다중 경로
> 2. **가치**: 저비용, 단거리 전송
> 3. **융합**: LAN, 데이터센터와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

멀티모드 계단형 광섬유(Multi-mode Step-index Fiber)는 코어의 굴절률이 균일하고 클래딩과 계단식으로 변하는 구조입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    멀티모드 계단형 광섬유 (Step-index)               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              구조 및 원리                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  굴절률 분포:                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  n (굴절률)                                            │   │ │
│   │   │    ▲                                                   │   │ │
│   │   │    │  ┌────────────────┐                              │   │ │
│   │   │    │  │     코어        │ n₁ (균일)                    │   │ │
│   │   │    │  │                │                              │   │ │
│   │   │    │  └────────────────┘                              │   │ │
│   │   │    │─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ← 계단식 변화           │   │ │
│   │   │    │  ┌────────────────┐                              │   │ │
│   │   │    │  │   클래딩       │ n₂ (균일)                    │   │ │
│   │   │    │  │                │                              │   │ │
│   │   │    │  └────────────────┘                              │   │ │
│   │   │    └───┴────────────────┴───→ 반경                     │   │ │
│   │   │         코어      클래딩                               │   │ │
│   │   │                                                      │   │ │
│   │   │  빛의 경로:                                             │   │ │
│   │   │                                                      │   │ │
│   │   │    ┌─────────────────────────────────┐               │   │ │
│   │   │    │   ╱─────╲      ╱─────╲          │               │   │ │
│   │   │    │  ╱       ╲    ╱       ╲         │  ← 모드 1     │   │ │
│   │   │    │ ───────────────────────────    │  ← 모드 2     │   │ │
│   │   │    │        ╲  ╱       ╲  ╱         │  ← 모드 3     │   │ │
│   │   │    │         ╲╱         ╲╱          │               │   │ │
│   │   │    └─────────────────────────────────┘               │   │ │
│   │   │                                                      │   │ │
│   │   │  문제: 모드 분산 (경로차 → 지연)                        │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              특징                                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  장점:                                                   │   │ │
│   │   │  • 구조 단순                                             │   │ │
│   │   │  • 저비용                                                │   │ │
│   │   │  • LED 광원 사용 가능                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                   │   │ │
│   │   │  • 모드 분산 (대역폭 제한)                               │   │ │
│   │   │  • 거리 제한 (2km 이내)                                  │   │ │
│   │   │  • 속도 제한                                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 계단형은 구조가 단순하지만 모드 분산으로 인해 대역폭이 제한됩니다.

**💡 비유**: 계단형은 여러 차선이 있지만 각각 다른 속도로 달리는 도로입니다.

**📢 섹션 요약 비유**: 계단형 = 모드 분산 문제!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **코어** | 빛 전파 | 균일 굴절률 | 도로 |
| **계단** | 굴절률 변화 | 급격한 변화 | 차선 |
| **모드** | 빛 경로 | 다수 경로 | 차량 |

**비교표**

```
+------------+----------+----------+
| 항목       | 계단형   | 언덕형   |
+------------+----------+----------+
| 굴절률     | 균일     | 점진적   |
| 모드 분산  | 큼       | 작음     |
| 대역폭     | 낮음     | 높음     |
| 제작       | 쉬움     | 어려움   |
+------------+----------+----------+
```

**핵심 알고리즸**: 다중 모드 전파, 모드 분산

**📢 섹션 요약 비유**: 모드 분산 = 대역폭 제한!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 단거리 LAN** → 계단형 멀티모드
**시나리오 2: 고속 전송** → 언덕형 멀티모드
**시나리오 3: 장거리** → 단일모드

**도입 체크리스트**: 거리, 속도, 비용

**📢 섹션 요약 비유**: 저비용 단거리 = 계단형!
