### Ⅰ. 개요 (Context & Background)

**개념 정의**

Virtual Memory는 물리 메모리를 논리적으로 확장합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Virtual Memory                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              가상 메모리 구조                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Virtual Address Space      Physical Memory          │   │ │
│   │   │  ┌─────────────────┐       ┌─────────────────┐      │   │ │
│   │   │  │ Page 0          │       │ Frame 0         │      │   │ │
│   │   │  │ Page 1   ───────┼──→    │ Frame 1         │      │   │ │
│   │   │  │ Page 2          │       │ Frame 2         │      │   │ │
│   │   │  │ Page 3   ───────┼──→    │ Frame 3         │      │   │ │
│   │   │  │ ...             │       │ ...             │      │   │ │
│   │   │  └─────────────────┘       └─────────────────┘      │   │ │
│   │   │         ↑                                              │   │ │
│   │   │      MMU (Memory Management Unit)                     │   │ │
│   │   │         ↑                                              │   │ │
│   │   │      Page Table                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 가상 메모리는 논리 주소를 물리 주소로 변환합니다.

**📢 섹션 요약 비유:** Virtual Memory = 가상 주소!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Virtual Address** | 가상 주소 |
| **Physical Address** | 물리 주소 |
| **Page** | 가상 페이지 |
| **Frame** | 물리 프레임 |
| **MMU** | 메모리 관리 유닛 |
| **Page Table** | 페이지 테이블 |

**핵심 알고리즸:** 가상 주소 → MMU → 페이지 테이블 → 물리 주소
