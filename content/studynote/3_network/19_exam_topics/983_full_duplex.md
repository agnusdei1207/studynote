# Full Duplex - 전이중

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 양방향 동시 통신
> 2. **가치**: 최대 성능
> 3. **융합:** Switch, No Collision, Point-to-Point

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Full Duplex는 동시에 양방향 통신합니다.


### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **TX** | 송신 |
| **RX** | 수신 |
| **Simultaneous** | 동시 |
| **Dedicated** | 전용 |
| **No Collision** | 충돌 없음 |

**핵심 알고리듬:** 독립 채널로 동시 송수신
