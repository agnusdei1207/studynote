# Packet Loss - 패킷 손실

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 전송 실패
> 2. **가치**: 네트워크 상태 지표
> 3. **융합:** Retransmission, Congestion, Error

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Loss Rate** | 손실률 |
| **Retransmit** | 재전송 |
| **Timeout** | 타임아웃 |
| **Duplicate** | 중복 수신 |
| **Gap** | 시퀀스 간격 |

**핵심 알고리듬:** 손실 감지 → 재전송 요청

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**TCP vs UDP 손실 처리**

| 항목 | TCP | UDP |
|:---|:---|:---|
| **손실 감지** | 자동 | 없음 |
| **재전송** | 있음 | 없음 |
| **지연** | 증가 | 유지 |
| **용도** | 데이터 | 실시간 |
