# 모바일 IP - Mobile IP

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이동 중 IP 주소 유지
> 2. **가치**: 세션 연속성, 투명성
> 3. **융합**: MIP, PMIP, DSMIP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

> **해설**: Mobile IP는 터널링으로 패킷을 전달합니다.

**💡 비유**: Mobile IP는 우편전송입니다.

**📢 섹션 요약 비유:** Mobile IP = IP 연속성!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **MN** | Mobile Node |
| **HA** | Home Agent |
| **FA** | Foreign Agent |
| **Binding** | 주소 매핑 |

**비교표: MIP vs PMIP**

```
+------------+----------+----------+----------+
| 항목       │ MIPv4    │ MIPv6    │ PMIPv6   │
+------------+----------+----------+----------+
| 클라이언트 │ 있음     │ 있음     │ 없음     │
| 터널       │ IP-in-IP │ IPv6     │ IPv6     │
| 네트워크   │ IPv4     │ IPv6     │ IPv6     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 등록, 터널링

**📢 섹션 요약 비유:** PMIP = 네트워크 관리!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 이동성 관리**

| 기술 | 계층 | 용도 |
|:---|:---|:---|
| **Mobile IP** | L3 | IP 연속성 |
| **GTP** | L3 | 3GPP |
| **MPTCP** | L4 | 다중 경로 |

**📢 섹션 요약 비유:** L2 = L2 이동성!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** Wi-Fi → PMIPv6
**시나리오 2:** LTE → GTP
**시나리오 3:** IMS → MIPv6

**도입 체크리스트:** 네트워크, 단말, 오버헤드

**📢 섹션 요약 비유:** 트라이앵글 라우팅 = 오버헤드!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 세션 연속성

**미래 전망:** 5G 이동성 관리

**참고 표준:** RFC 3344 (MIPv4), RFC 6275 (MIPv6)

**📢 섹션 요약 비유:** Mobile IP는 이동성의 기반!
