# NUMA (Non-Uniform Memory Access) - NUMA

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 불균등 메모리 접근
> 2. **가치**: 확장성, 성능
> 3. **융합:** Multi-socket, Memory Locality, SMP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NUMA는 CPU마다 메모리 접근 속도가 다릅니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NUMA                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NUMA 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  NUMA Node 0              NUMA Node 1                │   │ │
│   │   │  ┌─────────────────┐     ┌─────────────────┐         │   │ │
│   │   │  │ CPU 0 │ CPU 1   │     │ CPU 2 │ CPU 3   │         │   │ │
│   │   │  │       │         │     │       │         │         │   │ │
│   │   │  │ Memory (Local)  │     │ Memory (Local)  │         │   │ │
│   │   │  │    Fast         │     │    Fast         │         │   │ │
│   │   │  └────────┬────────┘     └────────┬────────┘         │   │ │
│   │   │           │                       │                   │   │ │
│   │   │           └───────────┬───────────┘                   │   │ │
│   │   │                       │                               │   │ │
│   │   │                 Interconnect                          │   │ │
│   │   │                  (Slow Access)                        │   │ │
│   │   │                                                      │   │ │
│   │   │  Local: 빠름, Remote: 느림                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NUMA는 로컬 메모리 접근이 원격보다 빠릅니다.

**📢 섹션 요약 비유:** NUMA = 지역 창고!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NUMA Node** | 노드 |
| **Local Memory** | 로컬 메모리 |
| **Remote Memory** | 원격 메모리 |
| **Interconnect** | 연결망 |
| **Memory Locality** | 메모리 지역성 |

**핵심 알고리즸:** CPU → (로컬?) 빠름 / (원격?) 느림

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**NUMA vs UMA**

| 항목 | UMA | NUMA |
|:---|:---|:---|
| **접근 속도** | 균일 | 불균일 |
| **확장성** | 제한 | 높음 |
| **복잡도** | 낮음 | 높음 |
| **용도** | 소규모 | 대규모 |
