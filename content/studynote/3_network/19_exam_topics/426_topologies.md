# 네트워크 토폴로지 - Network Topologies

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 연결 형태
> 2. **가치**: 확장성, 내결함성, 비용
> 3. **융합**: 스타, 메시, 버스, 링

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

네트워크 토폴로지는 장비 간 연결 구조입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    네트워크 토폴로지                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              토폴로지 유형                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  스타 (Star)                                        │   │ │
│   │   │         ◎ 중앙                                      │   │ │
│   │   │       / | \                                         │   │ │
│   │   │      ○ ○ ○                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  버스 (Bus)                                         │   │ │
│   │   │  ○────○────○────○                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  링 (Ring)                                          │   │ │
│   │   │     ○───○                                           │   │ │
│   │   │     │   │                                           │   │ │
│   │   │     ○───○                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  메시 (Mesh)                                        │   │ │
│   │   │     ○───○                                           │   │ │
│   │   │    /│\ /│\                                          │   │ │
│   │   │   ○─○─○─○                                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              토폴로지 비교                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │  │ 항목       │ 스타     │ 버스     │ 메시     │     │   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │  │ 내결함성   │ 중간     │ 낮음     │ 높음     │     │   │ │
│   │   │  │ 확장성     │ 높음     │ 낮음     │ 높음     │     │   │ │
│   │   │  | 비용       │ 중간     │ 낮음     │ 높음     │     │   │ │
│   │   │  │ 관리       │ 쉬움     │ 쉬움     │ 어려움   │     │   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 현대 네트워크는 주로 스타와 메시가 혼합된 형태입니다.

**💡 비유**: 토폴로지는 도시의 도로망입니다. 어떻게 연결되느냐가 중요합니다.

**📢 섹션 요약 비유:** 토폴로지 = 연결 구조!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Physical Topology** | 물리적 연결 |
| **Logical Topology** | 논리적 흐름 |
| **Hybrid** | 혼합 토폴로지 |
| **Tree** | 트리 구조 |

**비교표: 물리 vs 논리**

```
+------------+----------+----------+
| 항목       │ 물리     │ 논리     │
+------------+----------+----------+
| 예시       │ 스타     │ 링       │
| 참조       │ 케이블   │ 흐름     │
| 변경       │ 어려움   │ 쉬움     │
+------------+----------+----------+
```

**핵심 알고리즸:** 경로 선택, 장애 감지

**📢 섹션 요약 비유:** LAN = 스타!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 2:** WAN 백본 → 메시
**시나리오 3:** 데이터센터 → 스파인-리프

**📢 섹션 요약 비유:** 요구사항 = 토폴로지 선택!
