# IPv6 Transition - IPv6 전환 기술

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IPv4에서 IPv6로 마이그레이션
> 2. **가치**: 점진적 전환, 호환성
> 3. **융합:** Dual Stack, Tunneling, NAT64

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IPv6 전환 기술은 IPv4와 IPv6 공존을 지원합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IPv6 Transition                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전환 기술                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. Dual Stack                                       │   │ │
│   │   │     IPv4와 IPv6 동시 운영                             │   │ │
│   │   │                                                      │   │ │
│   │   │  2. Tunneling                                        │   │ │
│   │   │     IPv6 in IPv4 캡슐화                               │   │ │
│   │   │                                                      │   │ │
│   │   │  3. Translation                                      │   │ │
│   │   │     NAT64, DNS64                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 전환 기술로 서비스 중단 없이 IPv6로 이전합니다.

**📢 섹션 요약 비유:** IPv6 전환 = 이사!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 기술 | 설명 |
|:---|:---|
| **Dual Stack** | 두 스택 동시 실행 |
| **6to4** | IPv4 네트워크 통해 터널 |
| **Teredo** | NAT 통과 터널링 |
| **ISATAP** | 인트라넷 터널링 |
| **NAT64** | IPv6 → IPv4 변환 |
| **DNS64** | IPv4 응답을 IPv6로 변환 |

**핵심 알고리즸:** 프로토콜 변환/캡슐화

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)
