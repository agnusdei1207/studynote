# BPSK / QPSK / OQPSK

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 위상 편이 변조의 기본 형태들
> 2. **가치**: 신뢰성과 효율의 균형
> 3. **융합**: GPS, 위성, CDMA, Zigbee와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

BPSK, QPSK, OQPSK는 위상 편이 변조(PSK)의 대표적인 형태로, 각각 1비트, 2비트, 2비트(오프셋)를 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BPSK / QPSK / OQPSK 비교                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              BPSK (Binary PSK)                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  위상: 0°, 180° (2가지)                               │   │ │
│   │   │  전송: 1 비트/심볼                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  성상도:        I                                     │   │ │
│   │   │                  ●────────────→                       │   │ │
│   │   │            180°  │                                    │   │ │
│   │   │                  │                                    │   │ │
│   │   │                  ←────────────●                       │   │ │
│   │   │                                                      │   │ │
│   │   │  특징: 가장 강건, 낮은 대역폭 효율                      │   │ │
│   │   │  응용: GPS, 깊은 우주 통신                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              QPSK (Quadrature PSK)                            │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  위상: 45°, 135°, 225°, 315° (4가지)                  │   │ │
│   │   │  전송: 2 비트/심볼                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  성상도:        I                                     │   │ │
│   │   │            135° ●                                     │   │ │
│   │   │                 │╲    ╱│                              │   │ │
│   │   │                 │  ╲╱  │                              │   │ │
│   │   │       180° ─────┼───────┼──── 0°                      │   │ │
│   │   │                 │  ╱╲  │                              │   │ │
│   │   │                 │╱    ╲│                              │   │ │
│   │   │            225° ●      ● 45°                          │   │ │
│   │   │                      ●                                │   │ │
│   │   │                     315°                              │   │ │
│   │   │                                                      │   │ │
│   │   │  특징: BPSK의 2배 효율                                │   │ │
│   │   │  응용: 위성, CDMA, LTE                                │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OQPSK (Offset QPSK)                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  원리: Q성분을 T/2만큼 지연                            │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                               │   │ │
│   │   │  • 위상 전이가 90° 이하로 제한                         │   │ │
│   │   │  • 180° 전이 사라짐                                   │   │ │
│   │   │  • 비선형 증폭기에 유리                                │   │ │
│   │   │                                                      │   │ │
│   │   │  QPSK: 0°↔180° 전이 가능 (큰 진폭 변화)               │   │ │
│   │   │  OQPSK: 최대 90° 전이만 (진폭 변화 작음)              │   │ │
│   │   │                                                      │   │ │
│   │   │  응용: CDMA, Zigbee, 블루투스                         │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: OQPSK는 비선형 증폭기 사용 시 유리합니다.

**💡 비유**: OQPSK는 QPSK의 부드러운 버전입니다.

**📢 섹션 요약 비유**: OQPSK는 180° 전이 방지!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | BPSK | QPSK | OQPSK |
|:---|:---:|:---:|:---:|
| **비트/심볼** | 1 | 2 | 2 |
| **위상 수** | 2 | 4 | 4 |
| **최대 전이** | 180° | 180° | 90° |
| **포락선 변화** | 큼 | 큼 | 작음 |

**비교표**

```
+------------+----------+----------+----------+
| 항목       | BPSK     | QPSK     | OQPSK    |
+------------+----------+----------+----------+
| BER 성능   | 최고     | 동일     | 동일     |
| 대역폭 효율 | 1x       | 2x       | 2x       |
| 포락선 변동 | 큼       | 큼       | 작음     |
| 복잡도     | 낮음     | 중간     | 중간     |
+------------+----------+----------+----------+
```

**핵심 알고리즘**: I/Q 변조 → 위상 매핑 → 전송

**📢 섹션 요약 비유**: π/4-QPSK도 자주 사용!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 응용 분야**

| 변조 | 응용 |
|:---|:---|
| **BPSK** | GPS, 심우주 |
| **QPSK** | DVB-S, LTE |
| **OQPSK** | CDMA, Zigbee |

**📢 섹션 요약 비유**: 비선형 증폭 = OQPSK!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: GPS** → BPSK
**시나리오 2: 위성 TV** → QPSK
**시나리오 3: 센서 네트워크** → OQPSK

**도입 체크리스트**: SNR, 증폭기 선형성, 전력

**📢 섹션 요약 비유**: 전력 효율 = BPSK!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 신뢰성 vs 효율 균형

**미래 전망**: 5G는 QPSK~256QAM 적응형

**참고 표준**: IEEE 802.15.4, 3GPP

**📢 섹션 요약 비유**: PSK 계열은 무선의 기본!
