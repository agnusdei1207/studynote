# 지연 왜곡 (Delay Distortion)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 주파수별 전파 속도 차이로 인한 신호 변형
> 2. **가치**: 대역폭 제한 원인, 디지털 통신 품질 저하
> 3. **융합**: 등화기, 채널 보정, OFDM과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

지연 왜곡(Delay Distortion)은 신호의 각 주파수 성분이 매체를 통과할 때 서로 다른 속도로 전파되어 발생하는 신호 변형 현상입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    지연 왜곡 (Delay Distortion)                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              지연 왜곡 발생 원리                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  입력 신호 (복합 신호):                                 │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │  f1 ────→                                    │    │   │ │
│   │   │  │  f2 ────→  (동일 위상)                        │    │   │ │
│   │   │  │  f3 ────→                                    │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  매체 통과 후 (주파수별 속도 차이):                    │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │  f1 ──────────────────→  (느림)              │    │   │ │
│   │   │  │  f2 ───────────→        (중간)               │    │ │ │
│   │   │  │  f3 ──────→             (빠름)               │    │ │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  결과: 위상 차이 → 신호 왜곡                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              군지연 (Group Delay)                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  군지연 = dφ/dω (위상-주파수 미분)                     │   │ │
│   │   │                                                      │   │ │
│   │   │  이상적: 모든 주파수에서 일정                          │   │ │
│   │   │  실제: 주파수마다 다름 → 왜곡 발생                      │   │ │
│   │   │                                                      │   │ │
│   │   │  영향:                                               │   │ │
│   │   │  • 음성 통신: 큰 영향 없음                            │   │ │
│   │   │  • 디지털 통신: 비트 에러 증가                         │   │ │
│   │   │  • 비디오: 화질 저하                                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 지연 왜곡은 주로 음성 주파수 대역 전화선에서 심각하게 발생합니다.

**💡 비유**: 지연 왜곡은 오케스트라에서 각 악기가 도착하는 시간이 달라 소리가 어긋나는 것입니다.

**📢 섹션 요약 비유**: 주파수마다 속도가 달라요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **위상** | Phase | 각도 | 시계 |
| **군지연** | Group Delay | 지연 시간 | 도착 |
| **대역폭** | Bandwidth | 주파수 범위 | 폭 |
| **등화기** | Equalizer | 보정 | 조정 |

**지연 왜곡 영향**

```
주파수별 지연 특성:

지연   │     ╱╲
시간   │    ╱  ╲
  ↑    │   ╱    ╲
  │    │  ╱      ╲
  │    │ ╱        ╲
  └────┼──────────────→ 주파수
       f1   f2   f3

중심 주파수에서 최소 지연
가장자리에서 증가
```

**핵심 알고리즘**: 채널 측정 → 등화기 설계 → 왜곡 보정

**📢 섹션 요약 비유**: 등화기로 보정!
