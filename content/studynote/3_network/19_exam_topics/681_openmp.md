# OpenMP (Open Multi-Processing) - OpenMP

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 공유 메모리 병렬 프로그래밍
> 2. **가치**: 간단한 병렬화
> 3. **융합:** Multi-core, Thread, Fork-Join

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OpenMP는 공유 메모리 병렬 프로그래밍 API입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OpenMP                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OpenMP 실행 모델                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │              Master Thread                           │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │                                               │  │   │ │
│   │   │  │          Sequential Region                    │  │   │ │
│   │   │  │                                               │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                        │                             │   │ │
│   │   │                        ↓ #pragma omp parallel        │   │ │
│   │   │  ┌─────────┬─────────┬─────────┬─────────┐          │   │ │
│   │   │  │ Thread  │ Thread  │ Thread  │ Thread  │          │   │ │
│   │   │  │   0     │   1     │   2     │   3     │          │   │ │
│   │   │  │         │         │         │         │          │   │ │
│   │   │  │ Parallel Region                        │          │   │ │
│   │   │  └─────────┴─────────┴─────────┴─────────┘          │   │ │
│   │   │                        │                             │   │ │
│   │   │                        ↓ Implicit Barrier            │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │          Sequential Region                    │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
```

> **해설**: OpenMP는 Fork-Join 모델로 병렬화합니다.

**📢 섹션 요약 비유:** OpenMP = 병렬화 지시어!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **parallel** | 병렬 영역 |
| **for** | 루프 병렬화 |
| **sections** | 섹션 병렬화 |
| **critical** | 임계 영역 |
| **reduction** | 리덕션 |

**핵심 알고리즸:** 지시어 → 스레드 생성 → 병렬 실행 → 동기화

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**OpenMP vs MPI**

| 항목 | OpenMP | MPI |
|:---|:---|:---|
| **메모리** | 공유 | 분산 |
| **통신** | 암시적 | 명시적 |
| **확장성** | 단일 노드 | 다중 노드 |
| **복잡도** | 낮음 | 높음 |
