# NOMA (비직교 다중 접속)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전력 영역에서 비직교 다중화
> 2. **가치**: 스펙트럼 효율 증대, 대용량 지원
> 3. **융합**: 5G, 6G, IoT와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NOMA(Non-Orthogonal Multiple Access)는 여러 사용자가 같은 시간-주파수 자원을 공유하되, 전력 영역에서 중첩하여 전송하고 수신측에서 SIC(Successive Interference Cancellation)로 분리하는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NOMA (비직교 다중 접속)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NOMA 원리                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  OMA (직교):                                            │   │ │
│   │   │  ┌────┬────┬────┬────┐                               │   │ │
│   │   │  │ U1 │ U2 │ U3 │ U4 │  각 사용자 분리된 자원        │   │ │
│   │   │  └────┴────┴────┴────┘                               │   │ │
│   │   │                                                      │   │ │
│   │   │  NOMA (비직교):                                        │   │ │
│   │   │  ┌────────────────────┐                               │   │ │
│   │   │  │    U1+U2+U3+U4     │  모든 사용자 같은 자원        │   │ │
│   │   │  │  (전력으로 구분)    │                               │   │ │
│   │   │  └────────────────────┘                               │   │ │
│   │   │                                                      │   │ │
│   │   │  전력 분배:                                            │   │ │
│   │   │  P                                    │   │ │
│   │   │  ▲                                    │   │ │
│   │   │  │ ████ U1 (높은 전력, 멀리)          │   │ │
│   │   │  │ ████ ████ U2                       │   │ │
│   │   │  │ ████ ████ ████ U3                  │   │ │
│   │   │  │ ████ ████ ████ ████ U4 (낮은 전력, 가까이)       │   │ │
│   │   │  └──────────────────────→ 시간-주파수                │   │ │
│   │   │                                                      │   │ │
│   │   │  원칙:                                                 │   │ │
│   │   │  • 멀리 있는 사용자: 높은 전력                         │   │ │
│   │   │  • 가까이 있는 사용자: 낮은 전력                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SIC (연속 간섭 제거)                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  수신 신호: U1 + U2 + U3 + U4                        │   │ │
│   │   │                                                      │   │ │
│   │   │  단계 1: 가장 강한 U1 디코딩 → U1 복원               │   │ │
│   │   │  단계 2: U1 신호 재생성 후 제거 → U2+U3+U4 남음       │   │ │
│   │   │  단계 3: U2 디코딩 → 제거 → U3+U4 남음               │   │ │
│   │   │  단계 4: U3 디코딩 → 제거 → U4 남음                  │   │ │
│   │   │  단계 5: U4 디코딩 → 완료                            │   │ │
│   │   │                                                      │   │ │
│   │   │  조건:                                                │   │ │
│   │   │  • 전력 차이가 충분해야 함                             │   │ │
│   │   │  • 채널 정보가 정확해야 함                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: NOMA는 5G/6G의 핵심 기술로 스펙트럼 효율을 크게 높입니다.

**💡 비유**: NOMA는 여러 사람이 동시에 말하는데 큰 소리부터 순서대로 이해하는 것입니다.

**📢 섹션 요약 비유**: NOMA = 전력 영역 다중화!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **전력 분배** | 사용자 간 전력 할당 | 최적화 | 배분 |
| **SIC** | 간섭 제거 | 순차 디코딩 | 필터 |
| **CSI** | 채널 정보 | 피드백 | 정보 |

**비교표**

```
+------------+----------+----------+
| 항목       | OMA      | NOMA     |
+------------+----------+----------+
| 직교성     | 있음     | 없음     |
| 스펙트럼   | 낭비     | 효율     |
| 복잡도     | 낮음     | 높음     |
| 용량       | 제한     | 증대     |
+------------+----------+----------+
```

**핵심 알고리즸**: 전력 분배 → 중첩 전송 → SIC 수신 → 순차 복원

**📢 섹션 요약 비유**: SIC가 NOMA의 핵심!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: NOMA 장단점**

| 장점 | 단점 |
|---|---|
| 높은 스펙트럼 효율 | 높은 수신 복잡도 |
| 대용량 지원 | SIC 오류 전파 |
| 지연 민감 트래픽 | 채널 정보 의존 |

**📢 섹션 요약 비유**: NOMA는 복잡하지만 효율적!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 5G mMTC** → NOMA
**시나리오 2: 6G** → NOMA 확장
**시나리오 3: IoT** → NOMA

**도입 체크리스트**: SIC 능력, 전력 분배, CSI 정확도

**📢 섹션 요약 비유**: 5G/6G의 핵심 기술!
