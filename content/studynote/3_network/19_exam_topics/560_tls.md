# TLS (Transport Layer Security) - 전송 계층 보안

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전송 계층 암호화 프로토콜
> 2. **가치**: 기밀성, 무결성, 인증
> 3. **융합:** HTTPS, 인증서, Handshake

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

TLS는 전송 계층에서 보안을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TLS                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TLS Handshake (1.3)                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                              Server          │   │ │
│   │   │     │                                   │            │   │ │
│   │   │     │ ── ClientHello + Key Share ────→ │            │   │ │
│   │   │     │                                   │            │   │ │
│   │   │     │ ←── ServerHello + Encrypted Ext ─ │            │   │ │
│   │   │     │                                   │            │   │ │
│   │   │     │ ── Finished ───────────────────→ │            │   │ │
│   │   │     │                                   │            │   │ │
│   │   │  → 1-RTT Handshake 완료                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: TLS 1.3은 1-RTT로 빠른 Handshake를 수행합니다.

**📢 섹션 요약 비유:** TLS = 보안 터널!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Record Protocol** | 데이터 분할/암호화 |
| **Handshake Protocol** | 키 교환 |
| **Cipher Suite** | 암호화 알고리즘 |
| **MAC** | 무결성 검사 |
| **Certificate** | 인증 |

**핵심 알고리즸:** Handshake → 키 교환 → 암호화 전송

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**TLS 버전**

| 버전 | 특징 |
|:---|:---|
| **TLS 1.0** | SSL 3.0 후속 |
| **TLS 1.1** | 보안 강화 |
| **TLS 1.2** | AES-GCM 추가 |
| **TLS 1.3** | 1-RTT, 레거시 제거 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
