# IDS/IPS (Intrusion Detection/Prevention System) - 침입 탐지/방지 시스템

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 침입 탐지 및 차단
> 2. **가치**: 실시간 위협 대응
> 3. **융합:** Signature, Anomaly, SIEM

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IDS는 침입을 탐지하고, IPS는 탐지 후 차단까지 수행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IDS/IPS                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IDS vs IPS                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  IDS (Detection):                                    │   │ │
│   │   │  - 트래픽 미러링 (Inline 아님)                        │   │ │
│   │   │  - 탐지 → 알림만                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  IPS (Prevention):                                   │   │ │
│   │   │  - Inline 배치                                       │   │ │
│   │   │  - 탐지 → 차단 + 알림                                │   │ │
│   │   │                                                      │   │ │
│   │   │  탐지 방식:                                          │   │ │
│   │   │  - Signature-based: 알려진 패턴                      │   │ │
│   │   │  - Anomaly-based: 이상 행위                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IDS는 관찰만, IPS는 적극적 차단을 수행합니다.

**📢 섹션 요약 비유:** IDS = CCTV, IPS = 경호원!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NIDS/NIPS** | Network 기반 |
| **HIDS/HIPS** | Host 기반 |
| **Signature** | 패턴 매칭 |
| **Anomaly** | 행위 분석 |
| **Heuristic** | 휴리스틱 |

**핵심 알고리즸:** 패턴 매칭 또는 통계 분석 → 판정
