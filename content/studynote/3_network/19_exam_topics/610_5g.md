# 5G (5th Generation) - 5세대 이동통신

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 차세대 이동통신
> 2. **가치**: 초고속, 초저지연, 대규모 연결
> 3. **융합:** IoT, Edge, Network Slicing

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

5G는 초고속, 초저지연, 대규모 연결을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    5G                                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              5G 특성                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  eMBB (Enhanced Mobile Broadband):                   │   │ │
│   │   │  - 20 Gbps 다운로드                                  │   │ │
│   │   │  - 4K/8K 스트리밍, VR/AR                             │   │ │
│   │   │                                                      │   │ │
│   │   │  URLLC (Ultra-Reliable Low Latency):                 │   │ │
│   │   │  - 1ms 이하 지연                                     │   │ │
│   │   │  - 자율주행, 원격수술                                │   │ │
│   │   │                                                      │   │ │
│   │   │  mMTC (Massive Machine Type):                        │   │ │
│   │   │  - 100만 대/km² 연결                                 │   │ │
│   │   │  - 스마트시티, IoT                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  기술:                                               │   │ │
│   │   │  - mmWave, Massive MIMO                             │   │ │
│   │   │  - Network Slicing, Beamforming                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 5G는 세 가지 사용 시나리오(eMBB, URLLC, mMTC)를 제공합니다.

**📢 섹션 요약 비유:** 5G = 초고속도로!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**4G vs 5G**

| 항목 | 4G | 5G |
|:---|:---|:---|
| **속도** | 1 Gbps | 20 Gbps |
| **지연** | 10ms | 1ms |
| **연결** | 10만/km² | 100만/km² |
