# 변조 (Modulation) 개요

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 반송파의 특성을 신호로 변화시켜 정보 전송
> 2. **가치**: 안테나 크기, 주파수 다중화, 잡음 내성
> 3. **융합**: ASK, FSK, PSK, QAM과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

변조(Modulation)는 정보 신호를 반송파(Carrier)에 실어 전송하기 위해 반송파의 진폭, 주파수, 또는 위상을 변화시키는 과정입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    변조 (Modulation)                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              변조의 필요성                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 안테나 크기:                                       │   │ │
│   │   │     • 안테나 크기 ≈ 파장의 1/4 ~ 1/2                   │   │ │
│   │   │     • 저주파는 안테나가 너무 커짐                       │   │ │
│   │   │     • 고주파 반송파 사용 → 안테나 소형화                │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 주파수 다중화:                                     │   │ │
│   │   │     • 여러 신호를 서로 다른 주파수로 전송               │   │ │
│   │   │     • FDM/FDMA의 기반                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 전파 특성:                                         │   │ │
│   │   │     • 주파수별 전파 특성 다름                          │   │ │
│   │   │     • 원거리: 저주파, 단거리: 고주파                   │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 잡음 내성:                                         │   │ │
│   │   │     • 디지털 변조는 잡음에 강함                        │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              변조 종류                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  아날로그 변조:                                         │   │ │
│   │   │  • AM (Amplitude Modulation)                          │   │ │
│   │   │  • FM (Frequency Modulation)                          │   │ │
│   │   │  • PM (Phase Modulation)                              │   │ │
│   │   │                                                      │   │ │
│   │   │  디지털 변조:                                           │   │ │
│   │   │  • ASK (Amplitude Shift Keying)                       │   │ │
│   │   │  • FSK (Frequency Shift Keying)                       │   │ │
│   │   │  • PSK (Phase Shift Keying)                           │   │ │
│   │   │  • QAM (Quadrature Amplitude Modulation)              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 변조는 신호를 전송 매체에 적합한 형태로 변환합니다.

**💡 비유**: 변조는 화물을 트럭에 싣는 것과 같습니다.

**📢 섹션 요약 비유**: 진폭, 주파수, 위상이 3대 요소!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **반송파** | Carrier | 고주파 | 트럭 |
| **변조 신호** | Modulating | 정보 | 화물 |
| **변조된 파** | Modulated | 전송 | 적재된 트럭 |
| **복조** | Demodulation | 추출 | 하차 |

**반송파 수식**

```
반송파: c(t) = A·cos(2πf_c·t + φ)

변조 가능한 매개변수:
• A (진폭): AM/ASK
• f (주파수): FM/FSK
• φ (위상): PM/PSK
```

**핵심 알고리즘**: 신호 → 매개변수 변화 → 반송파 → 전송

**📢 섹션 요약 비유**: QAM은 진폭+위상 동시 변조!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 디지털 변조**

| 변조 | 변화 요소 | 잡음 내성 |
|:---|:---|:---:|
| **ASK** | 진폭 | 약함 |
| **FSK** | 주파수 | 강함 |
| **PSK** | 위상 | 강함 |
| **QAM** | 진폭+위상 | 중간 |

**📢 섹션 요약 비유**: PSK > FSK > ASK 잡음 내성!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: WiFi** → QAM (64-QAM, 256-QAM)
**시나리오 2: 블루투스** → GFSK
**시나리오 3: 위성** → QPSK, 8PSK

**도입 체크리스트**: 대역폭, 전력, 잡음 환경

**📢 섹션 요약 비유**: 높은 전송률 = QAM!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 대역폭 효율, 잡음 내성

**미래 전망**: OFDM + QAM (WiFi, LTE, 5G)

**참고 표준**: IEEE 802.11, 3GPP

**📢 섹션 요약 비유**: 변조는 무선 통신의 핵심!
