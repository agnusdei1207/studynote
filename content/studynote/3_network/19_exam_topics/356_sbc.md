# SBC - 세션 보더 컨트롤러

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: VoIP 세션 경계 제어 장비
> 2. **가치**: 보안, NAT 통과, QoS
> 3. **융합**: SIP, RTP, 방화벽

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SBC(Session Border Controller)는 VoIP 네트워크 경계를 제어합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SBC                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SBC 위치                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Enterprise Network]                               │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ SIP/RTP                                        │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           SBC                               │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • 시그널링 제어 (SIP)                        │    │   │ │
│   │   │  │  • 미디어 처리 (RTP)                          │    │   │ │
│   │   │  │  • NAT 통과                                   │    │   │ │
│   │   │  │  • 보안                                       │    │   │ │
│   │   │  │  • QoS                                        │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ SIP/RTP                                      │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [Service Provider Network]                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SBC 기능                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 기능       │ 설명     │ 가치     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 보안       │ 방화벽   │ 보호     │               │   │ │
│   │   │  │ NAT        │ 통과     │ 연결     │               │   │ │
│   │   │  | QoS        │ 품질     │ 품질     │               │   │ │
│   │   │  │ Transcoding│ 변환     │ 호환성   │               │   │ │
│   │   │  │ Monitoring │ 감시     │ 관리     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SBC는 VoIP 네트워크 경계에서 보안과 연결성을 제공합니다.

**💡 비유**: SBC는 VoIP의 국경 검문소입니다. 모든 트래픽을 검사합니다.

**📢 섹션 요약 비유:** SBC = VoIP 보안!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Signaling** | 시그널링 처리 |
| **Media** | 미디어 처리 |
| **NAT** | NAT 바인딩 |
| **Topology** | 토폴로지 숨김 |

**비교표: SBC vs 방화벽**

```
+------------+----------+----------+
| 항목       │ SBC      │ Firewall │
+------------+----------+----------+
| SIP        │ 이해     │ 포트     │
| RTP        │ 처리     │ 포트     │
| NAT        │ 지원     │ 지원     │
| 코덱 변환  │ 지원     │ 미지원   │
+------------+----------+----------+
```

**핵심 알고리즸:** SIP 파싱, 미디어 릴레이

**📢 섹션 요약 비유:** 토폴로지 숨김 = 보안!
