# Redis - 레디스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 인메모리 데이터 구조 저장소
> 2. **가치**: 고속 캐싱, 세션 저장소
> 3. **융합:** Cache, Pub/Sub, Message Queue

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Redis는 메모리 기반의 키-값 데이터 저장소입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Redis                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Redis 구조                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Data Types:                                         │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ String │ List │ Set │ Hash │ Sorted Set    │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  Features:                                           │   │ │
│   │   │  - In-Memory (빠른 속도)                             │   │ │
│   │   │  - Persistence (RDB, AOF)                           │   │ │
│   │   │  - Replication (Master-Slave)                       │   │ │
│   │   │  - Clustering                                        │   │ │
│   │   │  - Pub/Sub                                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Redis는 다양한 자료구조를 지원하는 고속 저장소입니다.

**📢 섹션 요약 비유:** Redis = 메모리 메모장!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **String** | 문자열 |
| **List** | 리스트 |

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Redis vs Memcached**

| 항목 | Redis | Memcached |
|:---|:---|:---|
| **데이터 타입** | 다양 | String만 |
| **Persistence** | 지원 | 미지원 |
| **클러스터링** | 지원 | 제한적 |
