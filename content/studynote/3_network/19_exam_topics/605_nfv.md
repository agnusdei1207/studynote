# NFV (Network Functions Virtualization) - 네트워크 기능 가상화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 기능의 가상화

> 3. **융합:** SDN, VNF, Cloud

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NFV는 네트워크 기능을 가상머신으로 구현합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NFV                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NFV 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  VNF (Virtual Network Functions):                    │   │ │
│   │   │  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐        │   │ │
│   │   │  │ vFire- │ │ vLoad  │ │ vRouter│ │ vCPE   │        │   │ │
│   │   │  │ wall   │ │Balanc- │ │        │ │        │        │   │ │
│   │   │  └────────┘ │ er     │ └────────┘ └────────┘        │   │ │
│   │   │             └────────┘                               │   │ │
│   │   │                    ↓                                 │   │ │
│   │   │  NFVI (NFV Infrastructure):                          │   │ │
│   │   │  ┌──────────────────────────────────────────┐       │   │ │
│   │   │  │  Hypervisor / Container Platform          │       │   │ │
│   │   │  │  Compute │ Network │ Storage              │       │   │ │
│   │   │  └──────────────────────────────────────────┘       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NFV는 전용 하드웨어를 소프트웨어로 대체합니다.

**📢 섹션 요약 비유:** NFV = 네트워크 앱!
