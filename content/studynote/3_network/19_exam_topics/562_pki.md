# PKI (Public Key Infrastructure) - 공개키 기반 구조

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 공개키 관리 인프라
> 2. **가치**: 신뢰 체계, 인증
> 3. **융합:** CA, 인증서, CRL, OCSP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PKI는 공개키 암호의 신뢰 체계를 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PKI                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PKI 구성요소                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │         Root CA                                      │   │ │
│   │   │            │                                         │   │ │
│   │   │      ┌─────┴─────┐                                  │   │ │
│   │   │   Intermediate  Intermediate                        │   │ │
│   │   │      │             │                                │   │ │
│   │   │   Leaf CA      Leaf CA                              │   │ │
│   │   │      │             │                                │   │ │
│   │   │   인증서       인증서                                │   │ │
│   │   │                                                      │   │ │
│   │   │  → 신뢰 체인 (Chain of Trust)                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: PKI는 계층적 CA 구조로 신뢰를 전파합니다.

**📢 섹션 요약 비유:** PKI = 공증 시스템!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **CA** | 인증 기관 |
| **RA** | 등록 기관 |
| **Certificate** | 인증서 |
| **CRL** | 폐기 목록 |
| **OCSP** | 실시간 폐기 확인 |

**핵심 알고리즸:** Root → Intermediate → Leaf → 인증서

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**폐기 확인 방식**

| 방식 | 설명 |
|:---|:---|
| **CRL** | 목록 다운로드 |
| **OCSP** | 실시간 조회 |
| **OCSP Stapling** | 서버가 대신 조회 |
