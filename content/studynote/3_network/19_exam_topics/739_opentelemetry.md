# OpenTelemetry - 오픈텔레메트리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 관측성 표준 프레임워크
> 2. **가치**: 벤더 중립, 통합 수집
> 3. **융합:** Tracing, Metrics, Logging

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OpenTelemetry는 관측성 데이터 수집 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OpenTelemetry                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OpenTelemetry 구조                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Signals:                                             │   │ │
│   │   │  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐   │   │ │
│   │   │  │   Traces    │ │   Metrics   │ │    Logs     │   │   │ │
│   │   │  │  (추적)     │ │  (메트릭)   │ │   (로그)    │   │   │ │
│   │   │  └─────────────┘ └─────────────┘ └─────────────┘   │   │ │
│   │   │                                                      │   │ │
│   │   │  API + SDK:                                           │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Instrumentation API                              ││   │ │
│   │   │  │ ─────────────────────────────────────────────── ││   │ │
│   │   │  │ OTLP Exporter → Backend (Jaeger, Prometheus)    ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  Collector:                                           │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Receiver → Processor → Exporter                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: OpenTelemetry는 통합 관측성을 제공합니다.

**📢 섹션 요약 비유:** OpenTelemetry = USB-C 포트!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **API** | 계측 API |
| **SDK** | 구현체 |
| **Collector** | 수집기 |
| **OTLP** | 프로토콜 |
| **Exporter** | 내보내기 |

**핵심 알고리즸:** 계측 → 수집 → 처리 → 내보내기
