# NoSQL - NoSQL 데이터베이스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 비관계형 데이터베이스
> 2. **가치**: 확장성, 유연성
> 3. **융합:** Document, Key-Value, Graph, Column

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NoSQL은 비관계형 데이터베이스입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NoSQL                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NoSQL 종류                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Key-Value:          Document:                       │   │ │
│   │   │  ┌─────────────┐     ┌─────────────────────────────┐│   │ │
│   │   │  │ K1 │ V1     │     │ {id: 1, name: "John", ...}  ││   │ │
│   │   │  │ K2 │ V2     │     │ {id: 2, name: "Jane", ...}  ││   │ │
│   │   │  │ Redis, DynDB │     │ MongoDB, CouchDB           ││   │ │
│   │   │  └─────────────┘     └─────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  Column:             Graph:                          │   │ │
│   │   │  ┌─────────────────┐ ┌─────────────────────────────┐│   │ │
│   │   │  │ Row │ Col │ Val │ │ Node ──Edge──→ Node        ││   │ │
│   │   │  │  1  │  A  │ v1  │ │ Neo4j, Neptune             ││   │ │
│   │   │  │ Cassandra       │ └─────────────────────────────┘│   │ │
│   │   │  └─────────────────┘                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NoSQL은 다양한 데이터 모델을 지원합니다.

**📢 섹션 요약 비유:** NoSQL = 자유 형식!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Key-Value** | 키-값 저장 |
| **Document** | 문서 저장 |
| **Column** | 컬럼 저장 |
