# W-CDMA (Wideband CDMA) / HSPA

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 5MHz 광대역 CDMA, 3GPP 표준
> 2. **가치**: 글로벌 3G 표준, 음성+데이터 통합
> 3. **융합**: GSM 진화, LTE 전환

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

W-CDMA(Wideband CDMA)는 3GPP가 정의한 3세대 이동통신 표준으로, 5MHz 대역폭을 사용하며 HSPA로 고속 데이터를 지원합니다.

```

├─────────────────────────────────────────────────────────────────────────┤

│   │              W-CDMA/HSPA 계보                                 │ │

│   │   │  진화 과정:                                              │   │ │
│   │   │                                                      │   │ │
│   │   │  GSM (2G)                                             │   │ │
│   │   │     │                                                  │   │ │
│   │   │     ▼                                                  │   │ │

│   │   │  • 최대 14.4Mbps                                       │   │ │
│   │   │                                                      │   │ │
```

> **해설**: W-CDMA는 GSM 진화 경로의 3G 표준으로, 전 세계적으로 가장 널리 사용됩니다.

**💡 비유**: W-CDMA는 GSM 도로를 4차선에서 8차선으로 확장한 것입니다.

**📢 섹션 요약 비유**: W-CDMA = 5MHz 광대역!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **NodeB** | 기지국 | 무선 송수신 | 기지국 |
| **RNC** | 제어기 | 무선 자원 관리 | 교환국 |
| **UE** | 단말 | 사용자 장비 | 폰 |

**비교표**

```
+------------+----------+----------+
| 항목       | CDMA2000 | W-CDMA   |
+------------+----------+----------+
| 대역폭     | 1.25MHz  | 5MHz     |
| 칩레이트   | 1.2288Mcps| 3.84Mcps|
| 주파수     | 800/1900 | 2100MHz  |
| 조직       | 3GPP2    | 3GPP     |
+------------+----------+----------+
```

**핵심 알고리즸**: 소프트 핸드오버, 파워 컨트롤, RAKE 수신기

**📢 섹션 요약 비유**: W-CDMA는 GSM의 3G 진화!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 데이터 속도**

| 기술 | 다운링크 | 업링크 |
|:---|:---:|:---:|
| **W-CDMA** | 384kbps | 128kbps |
| **HSDPA** | 14.4Mbps | 384kbps |
| **HSUPA** | 14.4Mbps | 5.76Mbps |
| **HSPA+** | 42Mbps | 22Mbps |

**📢 섹션 요약 비유**: HSPA = 3.5G!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 음성+데이터** → W-CDMA
**시나리오 2: 고속 데이터** → HSPA+
**시나리오 3: 대용량** → LTE

**도입 체크리스트**: 커버리지, 사용자 밀도, 트래픽

**📢 섹션 요약 비유**: 3G→4G 진화!
