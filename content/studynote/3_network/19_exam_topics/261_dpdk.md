# DPDK - Data Plane Development Kit

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 커널 우회 패킷 처리
> 2. **가치**: 사용자 공간 고속 처리
> 3. **융합**: NFV, 가상화와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DPDK(Data Plane Development Kit)는 사용자 공간에서 고속 패킷 처리를 위한 라이브러리 세트입니다.

**📢 섹션 요약 비유:** DPDK = 커널 우회!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **PMD** | 폴링 모드 드라이버 |
| **Ring** | Lock-free 큐 |
| **Mempool** | 메모리 풀 |
| **mbuf** | 패킷 버퍼 |

**비교표: DPDK vs 커널 네트워킹**

```
+------------+----------+----------+
| 항목       │ 커널     │ DPDK     │
+------------+----------+----------+
| 인터럽트   │ 있음     │ 없음     │
| 컨텍스트   │ 빈번     │ 최소     │
| 메모리     │ 복사     │ Zero-copy│
| 처리량     │ ~1M pps  │ ~10M+ pps│
+------------+----------+----------+
```

**핵심 알고리즸:** 폴링, 배치 처리

**📢 섹션 요약 비유:** Zero-copy = 복사 없음!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 고속 패킷 처리**

| 기술 | 위치 | 특징 |
|:---|:---|:---|

| **PF_RING** | 커널 모듈 | 간단 |

**📢 섹션 요약 비유:** DPDK = 사용자 공간!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: NFV** → DPDK + OVS
**시나리오 2: 라우터** → DPDK + VPP
**시나리오 3: 스위치** → DPDK + OVS

**도입 체크리스트:** NIC 지원, CPU 코어, Hugepages

**📢 섹션 요약 비유:** Intel NIC = DPDK 최적!
