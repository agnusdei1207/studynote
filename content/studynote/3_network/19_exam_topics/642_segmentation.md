# Segmentation - 세그먼테이션

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 논리적 메모리 분할
> 2. **가치**: 모듈성, 보호
> 3. **융합:** Paging, Virtual Memory, Protection

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Segmentation은 메모리를 논리적 단위로 분할합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Segmentation                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              세그먼테이션 구조                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Logical View              Physical Memory           │   │ │
│   │   │  ┌─────────────┐           ┌─────────────────┐       │   │ │
│   │   │  │    Code     │           │    Segment     │       │   │ │
│   │   │  ├─────────────┤           │      Code      │       │   │ │
│   │   │  │    Data     │   ──→     ├────────────────┤       │   │ │
│   │   │  ├─────────────┤           │    Segment     │       │   │ │
│   │   │  │    Stack    │           │      Data      │       │   │ │
│   │   │  └─────────────┘           ├────────────────┤       │   │ │
│   │   │                            │    Segment     │       │   │ │
│   │   │                            │     Stack      │       │   │ │
│   │   │                            └─────────────────┘       │   │ │
│   │   │                                                      │   │ │
│   │   │  Segment Table: Base + Limit                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 세그먼테이션은 프로그램의 논리적 구조를 반영합니다.

**📢 섹션 요약 비유:** Segmentation = 챕터!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Segment** | 논리적 메모리 단위 |
| **Base** | 시작 주소 |
| **Limit** | 크기 |
| **Segment Table** | 세그먼트 정보 |
| **Segment Selector** | 세그먼트 선택자 |

**핵심 알고리즸:** 논리 주소 → (Segment#, Offset) → 테이블 → 물리 주소

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Paging vs Segmentation**

| 항목 | Paging | Segmentation |
|:---|:---|:---|
| **분할 단위** | 고정 | 가변 |
| **단편화** | 내부 | 외부 |
| **논리적 의미** | 없음 | 있음 |
| **공유/보호** | 어려움 | 쉬움 |
