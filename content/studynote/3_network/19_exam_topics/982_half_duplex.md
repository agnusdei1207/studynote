# Half Duplex - 반이중

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단방향 교대 통신
> 2. **가치**: 간단한 구현
> 3. **융합:** CSMA/CD, Hub, Collision

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Half Duplex는 한 번에 한 방향으로만 통신합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Half Duplex                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              반이중 통신                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  반이중 통신:                                            │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Device A                    Device B            ││   │ │
│   │   │  │     │                           │                ││   │ │
│   │   │  │     │───── 전송 ─────────────→│                ││   │ │
│   │   │  │     │                           │                ││   │ │
│   │   │  │     │←──── 전송 ───────────────│                ││   │ │
│   │   │  │     │                           │                ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  특징:                                          ││   │ │
│   │   │  │  - 한 번에 한 방향만                              ││   │ │
│   │   │  │  - 송수신 교대                                   ││   │ │
│   │   │  │  - 충돌 가능                                     ││   │ │
│   │   │  │  - CSMA/CD 필요                                  ││   │ │
│   │   │  │                                                 ││   │ │

│   │   │                                                      │   │ │
```

**📢 섹션 요약 비유:** Half Duplex = 무전기!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **TX** | 송신 |
| **RX** | 수신 |
| **CSMA/CD** | 충돌 감지 |
| **Collision** | 충돌 |
| **Backoff** | 백오프 |

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Half vs Full Duplex**

| 항목 | Half Duplex | Full Duplex |
|:---|:---|:---|
| **방향** | 교대 | 동시 |
| **속도** | 낮음 | 높음 |
| **충돌** | 있음 | 없음 |
| **매체** | 공유 | 전용 |
