# PKI - 공개키 기반 구조

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 공개키 인증 체계
> 2. **가치**: 신뢰 체인 구축
> 3. **융합:** CA, Certificate, Chain of Trust

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PKI (Public Key Infrastructure)는 공개키 관리 체계입니다.

**📢 섹션 요약 비유:** PKI = 디지털 여권 시스템!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **CA** | 인증 기관 |
| **Certificate** | 인증서 |
| **CRL** | 폐기 목록 |
| **OCSP** | 온라인 검증 |
| **Keystore** | 키 저장소 |

**핵심 알고리듬:** 신원 확인 → 인증서 발급 → 검증
