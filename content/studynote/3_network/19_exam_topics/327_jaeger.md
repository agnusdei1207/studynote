### Ⅰ. 개요 (Context & Background)

**개념 정의**

Jaeger는 마이크로서비스 환경에서 분산 추적을 제공하는 오픈소스 플랫폼입니다.

**📢 섹션 요약 비유:** Jaeger = 분산 추적!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Agent** | 사이드카 |
| **Collector** | 수집기 |
| **Query** | 조회 서비스 |
| **UI** | 웹 인터페이스 |

**비교표: Jaeger vs Zipkin**

```
+------------+----------+----------+
| 항목       │ Jaeger   │ Zipkin   │
+------------+----------+----------+
| 개발사     │ Uber     │ Twitter  │
| 저장소     │ 다양     │ 제한     │
| UI         │ 우수     │ 기본     │
| 확장성     │ 높음     │ 중간     │
+------------+----------+----------+
```

**핵심 알고리즸:** 컨텍스트 전파, 샘플링

**📢 섹션 요약 비유:** Span = 작업 단위!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 추적 백엔드**

| 항목 | Jaeger | Zipkin | Tempo |
|:---|:---|:---|:---|
| **저장소** | Cassandra | 자체 | S3 |
| **비용** | 무료 | 무료 | 무료 |
| **규모** | 대형 | 중소 | 대형 |

**📢 섹션 요약 비유:** 대규모 = Jaeger!
