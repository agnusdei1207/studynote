# Network Slicing - 네트워크 슬라이싱

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 물리망을 논리망으로 분리
> 2. **가치**: 서비스별 최적화
> 3. **융합:** 5G, SDN, NFV, QoS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Network Slicing은 하나의 물리망을 여러 논리망으로 분리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Network Slicing                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Network Slicing 구성                              │ │

│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  논리 슬라이스:                                       │   │ │
│   │   │  ┌─────────┐ ┌─────────┐ ┌─────────┐                │   │ │
│   │   │  │  eMBB   │ │ URLLC   │ │  mMTC   │                │   │ │
│   │   │  │ (속도)  │ │ (지연)  │ │ (연결)  │                │   │ │
│   │   │  └─────────┘ └─────────┘ └─────────┘                │   │ │
│   │   │       │            │            │                    │   │ │

│   │   │  │              Physical Network                  │   │   │ │
│   │   │  │         (공유 인프라)                          │   │   │ │

│   │   │                                                      │   │ │

│   │   │  - 서비스별 격리                                     │   │ │
│   │   │  - 독립적 자원 할당                                  │   │ │
│   │   │  - SLA 보장                                          │   │ │
│   │   │                                                      │   │ │
```

> **해설**: 슬라이싱은 서비스별로 격리된 네트워크를 제공합니다.

**📢 섹션 요약 비유:** Network Slicing = 전용 차선!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **RAN Slicing** | 무선 접속 슬라이싱 |
| **Core Slicing** | 코어 슬라이싱 |
| **Transport Slicing** | 전송 슬라이싱 |
| **NSMF** | 슬라이스 관리 |
