# Reed-Solomon - 리드-솔로몬

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 블록 단위 오류 수정
> 2. **가치**: 버스트 오류 수정
> 3. **융합:** CD, DVD, QR Code, Erasure

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Reed-Solomon은 블록 단위로 오류를 수정합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Reed-Solomon                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RS 코드 구조                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  파라미터:                                              │   │ │
│   │   │  - n: 코드워드 길이 (최대 2^m - 1)                       │   │ │
│   │   │  - k: 데이터 심볼 수                                    │   │ │
│   │   │  - n-k: 패리티 심볼 수                                  │   │ │
│   │   │  - t: 수정 가능한 심볼 수 = (n-k)/2                      │   │ │
│   │   │                                                      │   │ │
│   │   │  예시 (CD):                                             │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ RS(255, 223)                                    ││   │ │
│   │   │  │ n=255 심볼, k=223 데이터 심볼                    ││   │ │
│   │   │  │ 패리티: 32 심볼                                  ││   │ │
│   │   │  │ 수정 가능: 16 심볼                               ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  버스트 오류 수정:                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ [D][D][D][P][P][P][P] → [D][X][X][X][P][P][P]   ││   │ │
│   │   │  │                         ↑____ 수정 가능 ____↑   ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  응용:                                                 │   │ │
│   │   │  - CD/DVD: 스크래치 복구                                │   │ │
│   │   │  - QR Code: 손상 복구                                   │   │ │
│   │   │  - 위성 통신: 페이딩 보정                                │   │ │
│   │   │  - 스토리지: RAID 6                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Reed-Solomon은 연속적인 오류 수정에 탁월합니다.

**📢 섹션 요약 비유:** Reed-Solomon = 퍼즐 조각!
