# MIMO (다중 입력 다중 출력) - Multiple Input Multiple Output

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 다중 안테나로 용량 증가
> 2. **가치**: 속도 향상, 커버리지 개선
> 3. **융합**: SU-MIMO, MU-MIMO, Massive MIMO

---

**📢 섹션 요약 비유:** MIMO = 다중 안테나!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Spatial Multiplexing** | 공간 다중화 |
| **Diversity** | 다이버시티 |
| **Beamforming** | 빔포밍 |
| **Channel Matrix** | 채널 행렬 |

**비교표: MIMO 유형**

```
+------------+----------+----------+----------+
| 유형       │ 안테나   │ 사용자   │ 용도     │
+------------+----------+----------+----------+
| SU-MIMO    │ 2-4      │ 1        │ 가정     │
| MU-MIMO    │ 4-8      │ 다수     │ 기업     │
| Massive    │ 64-256   │ 다수     │ 5G       │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 채널 추정, 프리코딩

**📢 섹션 요약 비유:** Massive MIMO = 5G!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 4x4 vs 8x8 MIMO**

| 항목 | 4x4 | 8x8 |
|:---|:---|:---|
| **스트림** | 4 | 8 |
| **속도** | 2.4Gbps | 4.8Gbps |
| **복잡성** | 중간 | 높음 |

**📢 섹션 요약 비유:** 스트림 = 속도!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 가정 공유기 → 2x2 MIMO
**시나리오 2:** 기업 AP → 4x4 MU-MIMO
**시나리오 3:** 5G 기지국 → Massive MIMO

**도입 체크리스트:** 안테나, 단말, 환경

**📢 섹션 요약 비유:** MU-MIMO = 다중 사용자!
