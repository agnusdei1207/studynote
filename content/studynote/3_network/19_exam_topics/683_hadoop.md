# Hadoop - 하둡

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 분산 컴퓨팅 프레임워크
> 2. **가치**: 대용량 데이터 처리
> 3. **융합:** HDFS, MapReduce, YARN

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Hadoop은 대용량 분산 데이터 처리 프레임워크입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Hadoop                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Hadoop 에코시스템                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │            Applications                      │    │   │ │
│   │   │  │   Hive │ Pig │ HBase │ Spark │ ...          │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                        │                             │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           YARN (Resource Manager)            │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │

│   │   │  │   (Processing)    │     (Processing)      │      │   │ │
```

**📢 섹션 요약 비유:** Hadoop = 분산 데이터 센터!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **HDFS** | 분산 파일 시스템 |
| **YARN** | 리소스 관리자 |
| **MapReduce** | 처리 엔진 |
| **NameNode** | 메타데이터 관리 |
| **DataNode** | 데이터 저장 |

**핵심 알고리즸:** 데이터 분산 → 병렬 처리 → 결과 취합

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Hadoop vs Traditional RDBMS**

| 항목 | Hadoop | RDBMS |
|:---|:---|:---|
| **데이터 크기** | PB | TB |
| **스키마** | 유연 | 고정 |
| **확장** | 수평 | 수직 |
| **처리** | 일괄 | 실시간 |
