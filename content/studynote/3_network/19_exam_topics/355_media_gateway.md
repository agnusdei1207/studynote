# Media Gateway - 미디어 게이트웨이

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 간 미디어 변환
> 2. **가치**: 이기종 네트워크 연동
> 3. **융합**: VoIP, PSTN, WebRTC

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Media Gateway는 서로 다른 네트워크 간 미디어를 변환합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Media Gateway                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Media Gateway 구조                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [IP Network]                                       │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ SIP / RTP                                      │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           Media Gateway                     │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • 코덱 변환 (Transcoding)                   │    │   │ │
│   │   │  │  • 프로토콜 변환                              │    │   │ │
│   │   │  │  • 미디어 처리                                │    │   │ │
│   │   │  │  • 포맷 변환                                  │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ TDM / ISDN / SS7                             │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [PSTN Network]                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Gateway 유형                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 유형       │ 변환     │ 용도     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ VoIP-PSTN  │ IP↔TDM  │ 전화망   │               │   │ │
│   │   │  │ Trunking   │ SIP↔ISUP│ 중계     │               │   │ │
│   │   │  | Access     │ Analog↔IP│ 가입자   │               │   │ │
│   │   │  │ WebRTC     │ Web↔SIP │ 브라우저 │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Media Gateway는 코덱, 프로토콜, 포맷을 변환합니다.

**💡 비유**: Media Gateway는 통역사입니다. 다른 언어를 번역합니다.

**📢 섹션 요약 비유:** Media Gateway = 네트워크 번역!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Transcoding** | 코덱 변환 |
| **Protocol** | 프로토콜 변환 |
| **DSP** | 디지털 신호 처리 |
| **MGC** | 게이트웨이 컨트롤러 |

**비교표: 코덱 변환**

```
+------------+----------+----------+
| 원본       │ 변환     │ 용도     │
+------------+----------+----------+
| G.711      │ G.729    │ 대역폭   │
| Opus       │ G.711    │ PSTN     │
| H.264      │ H.265    │ 효율     │
+------------+----------+----------+
```

**핵심 알고리즸:** DSP, 코덱

**📢 섹션 요약 비유:** DSP = 신호 처리!
