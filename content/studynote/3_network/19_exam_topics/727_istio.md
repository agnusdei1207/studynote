# Istio - 이스티오

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서비스 메시 플랫폼
> 2. **가치**: 트래픽 관리, 보안, 관측성
> 3. **융합:** Kubernetes, Envoy, Service Mesh

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Istio는 서비스 메시 플랫폼입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Istio                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Istio 아키텍처                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Control Plane (Istiod):                              │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ Pilot        - 트래픽 관리                    │    │   │ │
│   │   │  │ Citadel      - 보안 (mTLS)                    │    │   │ │
│   │   │  │ Galley       - 설정 검증                      │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  Data Plane (Envoy Sidecars):                         │   │ │
│   │   │  ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐           │   │ │
│   │   │  │Envoy  │ │Envoy  │ │Envoy  │ │Envoy  │           │   │ │
│   │   │  │ Svc A │ │ Svc B │ │ Svc C │ │ Svc D │           │   │ │
│   │   │  └───────┘ └───────┘ └───────┘ └───────┘           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Istio는 마이크로서비스 통신을 관리합니다.

**📢 섹션 요약 비유:** Istio = 교통 통제 센터!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Istiod** | 제어 평면 |
| **Envoy** | 데이터 평면 |
| **VirtualService** | 가상 서비스 |
| **DestinationRule** | 대상 규칙 |
