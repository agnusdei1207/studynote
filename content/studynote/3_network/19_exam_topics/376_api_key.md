# API Key - API 인증 키

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단순 API 인증 토큰
> 2. **가치**: 구현 용이, 광범위 사용
> 3. **융합**: Gateway, Rate Limiting

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

API Key는 API 접근을 위한 식별자 토큰입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    API Key                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              API Key 인증 흐름                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Client]                                           │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ API Key: abc123                                │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  [API Gateway] ──────▶ [인증 확인]                  │   │ │
│   │   │                            │                          │   │ │
│   │   │                            │ 유효                     │   │ │
│   │   │                            ▼                          │   │ │
│   │   │  [Backend API] ◀───────────┘                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              API Key 전송 방식                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 방식       │ 위치     │ 예시     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ Header     │ X-API-Key│ 헤더     │               │   │ │
│   │   │  │ Query      │ ?key=    │ URL      │               │   │ │
│   │   │  | Cookie     │ api_key  │ 쿠키     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: API Key는 클라이언트를 식별하고 접근을 제어합니다.

**💡 비유**: API Key는 출입증입니다. 건물에 들어갈 때 보여줍니다.

**📢 섹션 요약 비유:** API Key = 출입증!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Public Key** | 식별자 |
| **Secret Key** | 비밀키 |
| **Scope** | 권한 범위 |
| **Expiry** | 만료일 |

**비교표: API Key vs JWT vs OAuth**

```
+------------+----------+----------+
| 항목       │ API Key  │ JWT/OAuth│
+------------+----------+----------+
| 복잡도     │ 낮음     │ 높음     │
| 보안       │ 기본     │ 강력     │
| 사용자     │ 없음     │ 있음     │
| 세션       │ 없음     │ 있음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 키 검증, 스코프 확인

**📢 섹션 요약 비유:** 간단 = API Key!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 인증 방식**

| 항목 | API Key | Basic Auth | Bearer |
|:---|:---|:---|:---|
| **형식** │ 문자열 │ Base64 │ 토큰 |
| **보안** │ 중간 │ 낮음 │ 높음 |
| **용도** │ API │ 레거시 │ OAuth |

**📢 섹션 요약 비유:** 서비스 = API Key!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 공개 API → API Key
**시나리오 2:** M2M 통신 → API Key + Secret
**시나리오 3:** 사용자 인증 → OAuth/JWT

**도입 체크리스트:** 키 생성, 저장, 로테이션

**📢 섹션 요약 비유:** 공개 API = API Key!
