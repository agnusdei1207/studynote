# CDMA (코드 분할 다중 접속)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고유 코드로 사용자 식별하여 동시 전송
> 2. **가치**: 소프트 핸드오버, 주파수 효율, 3G 기반
> 3. **융합**: IS-95, WCDMA, CDMA2000과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CDMA(Code Division Multiple Access)는 각 사용자에게 고유한 확산 코드(Walsh Code)를 할당하여 같은 주파수와 시간에 동시 전송하는 다중 접속 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CDMA (코드 분할 다중 접속)                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CDMA 원리                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  기본 개념:                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  사용자1: 데이터1 × Walsh코드1 → 확산 신호           │   │ │
│   │   │  사용자2: 데이터2 × Walsh코드2 → 확산 신호   ─→ 결합  │   │ │
│   │   │  사용자3: 데이터3 × Walsh코드3 → 확산 신호           │   │ │
│   │   │                        ↓                              │   │ │
│   │   │                   공통 주파수                          │   │ │
│   │   │                        ↓                              │   │ │
│   │   │  수신측:                                                │   │ │
│   │   │  수신 신호 × Walsh코드1 → 사용자1 데이터              │   │ │
│   │   │  수신 신호 × Walsh코드2 → 사용자2 데이터              │   │ │
│   │   │  수신 신호 × Walsh코드3 → 사용자3 데이터              │   │ │
│   │   │                                                      │   │ │
│   │   │  핵심: 직교 코드 (Walsh Code)                         │   │ │
│   │   │  W₁ × W₂ = 0 (다른 사용자 간섭 없음)                  │   │ │
│   │   │  W₁ × W₁ = 1 (자신의 데이터 복원)                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Walsh 코드 예시                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  W0 = [+1 +1 +1 +1]                                 │   │ │
│   │   │  W1 = [+1 -1 +1 -1]                                 │   │ │
│   │   │  W2 = [+1 +1 -1 -1]                                 │   │ │
│   │   │  W3 = [+1 -1 -1 +1]                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  직교성 확인:                                          │   │ │
│   │   │  W0·W1 = (+1)(+1)+(+1)(-1)+(+1)(+1)+(+1)(-1) = 0     │   │ │
│   │   │  W0·W0 = (+1)(+1)+(+1)(+1)+(+1)(+1)+(+1)(+1) = 4     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: CDMA는 3G 이동통신의 핵심 기술입니다.

**💡 비유**: CDMA는 파티장에서 각자 다른 언어로 대화하는 것입니다.

**📢 섹션 요약 비유**: CDMA = 코드로 구분!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **Walsh 코드** | 사용자 식별 | 직교 코드 | 언어 |
| **PN 코드** | 기지국 식별 | 스크램블링 | 사투리 |
| **RAKE 수신기** | 다중경로 수신 | 경로 결합 | 다중 안테나 |
| **전력 제어** | Near-Far 해결 | 1500Hz | 음량 조절 |

**비교표**

```
+------------+----------+----------+
| 항목       | IS-95    | WCDMA    |
+------------+----------+----------+
| 대역폭     | 1.25MHz  | 5MHz     |
| 칩율       | 1.2288Mcps| 3.84Mcps|
| 동기       | 동기식   | 비동기식 |
| 음성 코덱  | EVRC     | AMR      |
+------------+----------+----------+
```

**핵심 알고리즸**: 데이터 × Walsh → PN 스크램블 → 전송 → 역확산 → 수신

**📢 섹션 요약 비유**: IS-95 = 동기식 CDMA!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: CDMA 특징**

| 장점 | 단점 |
|:---|:---|
| 소프트 핸드오버 | 전력 제어 필수 |
| 주파수 효율 | Near-Far 문제 |
| 다중경로 내성 | 복잡한 수신기 |
| 보안성 | 동기 어려움 |

**📢 섹션 요약 비유**: CDMA는 전력 제어가 핵심!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: IS-95** → 동기식 CDMA (GPS 동기)
**시나리오 2: WCDMA** → 비동기식 CDMA
**시나리오 3: CDMA2000** → 3x 확장

**도입 체크리스트**: 전력 제어, RAKE 수신기, 동기

**📢 섹션 요약 비유**: WCDMA는 비동기식!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 소프트 핸드오버, 높은 용량

**미래 전망**: OFDMA로 진화 (4G)

**참고 표준**: IS-95, 3GPP

**📢 섹션 요약 비유**: CDMA는 3G의 혁명!
