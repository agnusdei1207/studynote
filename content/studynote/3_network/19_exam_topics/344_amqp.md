# AMQP - 메시지 큐 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 엔터프라이즈 메시징 프로토콜

> 3. **융합**: RabbitMQ, Kafka와 통합

### Ⅰ. 개요 (Context & Background)

**개념 정의**

AMQP는 엔터프라이즈급 메시지 큐 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    AMQP                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              AMQP 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Producer]                                         │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ Publish (Exchange)                             │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           AMQP Broker (RabbitMQ)            │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  ┌─────────┐    ┌─────────┐    ┌─────────┐ │    │   │ │
│   │   │  │  │Exchange │───▶│ Binding │───▶│ Queue   │ │    │   │ │

│   │   │  │       ▼                              ▼      │    │   │ │
│   │   │  │    [라우팅]                      [Consumer]│    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
```

> **해설**: AMQP는 Exchange와 Queue로 유연한 라우팅을 제공합니다.

**💡 비유**: AMQP는 우편 분류소입니다. 편지를 주소별로 분류합니다.

**📢 섹션 요약 비유:** AMQP = 엔터프라이즈 메시징!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Exchange** | 교환기 |
| **Queue** | 큐 |
| **Binding** | 바인딩 |
| **Routing Key** | 라우팅 키 |

**비교표: AMQP vs MQTT**

```
+------------+----------+----------+
| 항목       │ AMQP     │ MQTT     │
+------------+----------+----------+
| 모델       │ 큐       │ Pub/Sub  │
| 라우팅     │ 복잡     │ 단순     │
| 트랜잭션   │ 지원     │ 제한     │
| 용도       │ 엔터프라이즈│ IoT    │
+------------+----------+----------+
```

**핵심 알고리즸:** 라우팅, 트랜잭션

**📢 섹션 요약 비유:** Exchange = 분류기!
