# OpenFlow - 오픈플로우

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SDN Southbound 프로토콜
> 2. **가치**: 스위치 프로그래밍
> 3. **융합:** SDN, Flow Table, Controller

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OpenFlow는 컨트롤러와 스위치 간 통신 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OpenFlow                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OpenFlow 구조                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │              OpenFlow Controller                     │   │ │
│   │   │                      │                               │   │ │
│   │   │                      │ OpenFlow Protocol             │   │ │
│   │   │                      ↓                               │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │            OpenFlow Switch                   │    │   │ │
│   │   │  │  ┌───────────────────────────────────────┐  │    │   │ │
│   │   │  │  │ Flow Table                            │  │    │   │ │
│   │   │  │  │ Match Fields → Actions                │  │    │   │ │
│   │   │  │  │ - Ingress Port                        │  │    │   │ │
│   │   │  │  │ - MAC, IP, TCP/UDP                    │  │    │   │ │
│   │   │  │  │ - VLAN, Metadata                     │  │    │   │ │
│   │   │  │  └───────────────────────────────────────┘  │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: OpenFlow는 Flow Table을 통해 패킷 처리를 제어합니다.

**📢 섹션 요약 비유:** OpenFlow = 스위치 프로그래밍 언어!
