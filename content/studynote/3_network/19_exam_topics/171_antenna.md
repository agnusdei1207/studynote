# 안테나 (Antenna) - 이득/빔폭/편파

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전자파를 방사/수신하는 장치
> 2. **가치**: 지향성, 이득으로 효율화
> 3. **융합**: 무선 통신의 핵심

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

안테나(Antenna)는 전기 신호를 전자파로 변환하여 방사하거나, 전자파를 전기 신호로 수신하는 장치입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    안테나 (Antenna)                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              안테나 파라미터                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 이득 (Gain):                                       │   │ │
│   │   │     • 등방성 안테나 대비 전력 밀도                      │   │ │
│   │   │     • 단위: dBi (등방성 기준)                          │   │ │
│   │   │     • 단위: dBd (다이폴 기준)                          │   │ │
│   │   │     • 0 dBi = 2.15 dBd                               │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 빔폭 (Beamwidth):                                  │   │ │
│   │   │     • 반전력 빔폭 (HPBW): -3dB 지점 폭                 │   │ │
│   │   │     • 이득↑ = 빔폭↓                                   │   │ │
│   │   │     • 좁은 빔 = 높은 지향성                            │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 편파 (Polarization):                               │   │ │
│   │   │     • 전계 벡터 방향                                   │   │ │
│   │   │     • 수직/수평/원형/타원                              │   │ │
│   │   │     • 송수신 편파 일치 필요                            │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 임피던스 (Impedance):                               │   │ │
│   │   │     • 일반적으로 50Ω 또는 75Ω                          │   │ │
│   │   │     • 정합 불일치 → 반사                               │   │ │
│   │   │     • VSWR로 측정                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  5. 대역폭 (Bandwidth):                                 │   │ │
│   │   │     • 정상 동작 주파수 범위                             │   │ │
│   │   │     • VSWR < 2:1 기준                                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              프리스 전송 공식                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Pr = Pt × Gt × Gr × λ² / (4πd)²                    │   │ │
│   │   │                                                      │   │ │
│   │   │  Pr: 수신 전력     Pt: 송신 전력                     │   │ │
│   │   │  Gt: 송신 안테나 이득  Gr: 수신 안테나 이득          │   │ │
│   │   │  λ: 파장          d: 거리                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 안테나 이득이 높을수록 빔이 좁아지고 전력이 집중됩니다.

**💡 비유**: 안테나는 손전등입니다. 반사경이 클수록 빛이 멀리 갑니다.

**📢 섹션 요약 비유:** 이득↑ = 빔폭↓!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 파라미터 | 단위 | 의미 | 전형값 |
|:---|:---|:---|:---|
| **이득** | dBi | 집중도 | 0-50 dBi |
| **빔폭** | 도 | 지향각 | 1-360° |
| **VSWR** | 비 | 정합도 | 1.0-2.0 |
| **효율** | % | 손실 | 50-99% |

**비교표**

```
+------------+----------+----------+
| 안테나     │ 이득     │ 빔폭     │
+------------+----------+----------+
| 등방성     │ 0 dBi    │ 360°     │
| 다이폴     │ 2.15 dBi │ 78°      │
| 파라볼라   │ 30+ dBi  │ 1-5°     │
│ 패치       │ 6-9 dBi  │ 60-90°   │
+------------+----------+----------+
```

**핵심 알고리즸:** 프리스 공식, 안테나 배열

**📢 섹션 요약 비유:** dBi = 등방성 기준!
