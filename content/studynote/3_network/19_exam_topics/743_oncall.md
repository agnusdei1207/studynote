#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 대기 근무
> 2. **가치**: 24/7 대응

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

On-call은 대기 근무를 의미합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    On-call                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              온콜 로테이션                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Primary:    [ Alice ]                              │   │ │
│   │   │  Secondary:  [ Bob ]                                │   │ │
│   │   │  Manager:    [ Carol ]                              │   │ │
│   │   │                                                      │   │ │
│   │   │  Escalation Path:                                    │   │ │
│   │   │  Alert ──→ Primary (5분)                            │   │ │
│   │   │              ↓ 응답 없음                             │   │ │
│   │   │         Secondary (10분)                             │   │ │
│   │   │              ↓ 응답 없음                             │   │ │
│   │   │            Manager                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  로테이션:                                            │   │ │
│   │   │  Week 1: Alice (Primary), Bob (Secondary)           │   │ │
│   │   │  Week 2: Bob (Primary), Carol (Secondary)           │   │ │
│   │   │  Week 3: Carol (Primary), Alice (Secondary)         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 온콜은 24/7 서비스 대응을 보장합니다.

**📢 섹션 요약 비유:** On-call = 당직!
