# BOOTP (Bootstrap Protocol) - 부트스트랩 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디스크리스 워크스테이션 부팅 지원
> 2. **가치**: IP 할당 + 부팅 이미지
> 3. **융합:** DHCP, TFTP, RARP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

BOOTP는 IP 할당과 부팅 정보를 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BOOTP                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              BOOTP 동작                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Diskless Client           BOOTP Server             │   │ │
│   │   │     │                           │                    │   │ │
│   │   │     │ ── BOOTP Request ───────→ │                    │   │ │
│   │   │     │    (MAC 알림)             │                    │   │ │
│   │   │     │                           │                    │   │ │
│   │   │     │ ←── BOOTP Reply ───────── │                    │   │ │
│   │   │     │    IP, GW, TFTP Server    │                    │   │ │
│   │   │     │                           │                    │   │ │
│   │   │     │ ── TFTP 이미지 요청 ────→ │ TFTP Server        │   │ │
│   │   │     │                           │                    │   │ │
│   │   │     │ ←── 부팅 이미지 ───────── │                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: BOOTP는 정적 매핑으로 IP를 할당합니다.

**📢 섹션 요약 비유:** BOOTP = 원격 부팅 도우미!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Static Mapping** | MAC-IP 고정 |
| **Boot File** | 부팅 이미지 경로 |
| **TFTP** | 이미지 전송 |
| **UDP 67/68** | 포트 |

**핵심 알고리즸:** MAC → IP 매핑 → 부팅 정보 제공

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**BOOTP vs DHCP**

| 항목 | BOOTP | DHCP |
|:---|:---|:---|
| **할당** | 정적 | 동적 |
| **Lease** | 없음 | 있음 |
| **Options** | 제한 | 풍부 |
| **용도** | 부팅 | 일반 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 2:** 일반 → DHCP
**시나리오 3:** PXE → DHCP + TFTP
