# TCP Timers - TCP 타이머

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: TCP 연결 관리 시간 제어
> 2. **가치**: 신뢰성, 자원 정리
> 3. **융합:** RTO, Keepalive, TIME_WAIT

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

TCP Timers는 연결 상태를 시간 기반으로 관리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TCP Timers                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              주요 TCP 타이머                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. Retransmission Timer (RTO)                      │   │ │
│   │   │     - 재전송 대기 시간                                │   │ │
│   │   │                                                      │   │ │
│   │   │  2. Persist Timer                                    │   │ │
│   │   │     - 윈도우 프로브                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  3. Keepalive Timer                                  │   │ │
│   │   │     - 연결 활성 확인                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  4. TIME_WAIT Timer (2MSL)                           │   │ │
│   │   │     - 연결 종료 대기                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 각 타이머는 특정 상황에서 TCP의 신뢰성을 보장합니다.

**📢 섹션 요약 비유:** TCP Timers = 시간 관리자!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 타이머 | 기본값 | 용도 |
|:---|:---|:---|
| **RTO** | 동적 계산 | 재전송 |
| **Persist** | 60초 | 윈도우 0 |
| **Keepalive** | 2시간 | 연결 확인 |
| **2MSL** | 60초 | 종료 대기 |
