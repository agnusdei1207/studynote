# SDN (Software-Defined Networking) - 소프트웨어 정의 네트워킹

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 제어/데이터 평면 분리
> 2. **가치**: 중앙 제어, 프로그래밍 가능
> 3. **융합**: OpenFlow, NFV와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SDN(Software-Defined Networking)은 네트워크의 제어 평면과 데이터 평면을 분리하여 중앙에서 제어하는 아키텍처입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SDN (Software-Defined Networking)                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SDN 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │         Application Layer                   │    │   │ │
│   │   │  │  (네트워크 앱: 방화벽, 로드밸런서)          │    │   │ │
│   │   │  └───────────────────┬─────────────────────────┘    │   │ │
│   │   │                      │ NB API                       │   │ │
│   │   │  ┌───────────────────┴─────────────────────────┐    │   │ │
│   │   │  │         Control Layer (SDN Controller)      │    │   │ │
│   │   │  │  (중앙 제어: 경로 계산, 정책)               │    │   │ │
│   │   │  └───────────────────┬─────────────────────────┘    │   │ │
│   │   │                      │ SB API (OpenFlow)            │   │ │
│   │   │  ┌───────────────────┴─────────────────────────┐    │   │ │
│   │   │  │         Infrastructure Layer                │    │   │ │
│   │   │  │  (스위치: 패킷 전달)                        │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SDN 특징                                          │ │

│   │   │                                                      │   │ │
│   │   │  1. 분리 (Separation):                               │   │ │
│   │   │     • 제어 평면과 데이터 평면 분리                   │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 중앙화 (Centralization):                         │   │ │
│   │   │     • 논리적 중앙 제어                               │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 프로그래밍 (Programmability):                    │   │ │
│   │   │     • API를 통한 자동화                              │   │ │
│   │   │                                                      │   │ │

│   │                                                              │ │

│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SDN은 제어 평면을 중앙화하여 네트워크를 프로그래밍합니다.

**💡 비유**: SDN은 네트워크의 운영체제입니다. 앱이 하드웨어를 제어합니다.

**📢 섹션 요약 비유:** SDN = 제어/데이터 분리!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Controller** | 중앙 제어 |
| **OpenFlow** | 남북 API |
| **OVS** | Open vSwitch |
| **Flow Table** | 규칙 테이블 |

**비교표: 전통 vs SDN**

```
+------------+----------+----------+
| 항목       │ 전통     │ SDN      │
+------------+----------+----------+
| 제어       │ 분산     │ 중앙     │
| 설정       │ CLI      │ API      │
| 확장       │ 제한     │ 유연     │
| 자동화     │ 어려움   │ 쉬움     │
+------------+----------+----------+
```

**핵심 알고리즸:** Flow 매칭, 액션 실행

**📢 섹션 요약 비유:** OpenFlow = SDN 프로토콜!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데이터센터** → SDN + NFV

**📢 섹션 요약 비유:** SDN = 네트워크 자동화의 핵심!
