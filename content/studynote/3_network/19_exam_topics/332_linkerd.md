### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Proxy** | Rust 프록시 |
| **Control Plane** | 제어 플레인 |
| **CA** | 인증 기관 |
| **Viz** | 관측성 |

**비교표: Linkerd vs Istio**

```
+------------+----------+----------+
| 항목       │ Linkerd  │ Istio    │
+------------+----------+----------+
| 프록시     │ Rust     │ Envoy    │
| 메모리     │ ~10MB    │ ~50MB    │
| 설정       │ 간단     │ 복잡     │
| 기능       │ 기본     │ 풍부     │
+------------+----------+----------+
```

**핵심 알고리즸:** Rust 프록시, 자동 mTLS

**📢 섹션 요약 비유:** Rust = 빠름!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)
