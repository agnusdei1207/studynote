# MSK / GMSK (최소 편이 변조)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 위상 연속성을 유지하는 FSK 변형
> 2. **가치**: 스펙트럼 효율, 포락선 일정
> 3. **융합**: GSM, 위성, DECT와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MSK(Minimum Shift Keying)는 변조 지수 h=0.5인 특수 FSK로, 위상 연속성을 유지합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MSK / GMSK                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MSK (Minimum Shift Keying)                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 변조 지수 h = 0.5                                   │   │ │
│   │   │  • 주파수 편이 = 비트율/2                               │   │ │
│   │   │  • 위상 연속 (Phase Continuous)                        │   │ │
│   │   │  • 포락선 일정 (Constant Envelope)                     │   │ │
│   │   │                                                      │   │ │
│   │   │  파형:                                                 │   │ │
│   │   │  데이터:    1   0   1   1   0                         │   │ │
│   │   │              ╱╲ ╱╲╱╲╱╲╱╲                              │   │ │
│   │   │            ╱  ╲╱      ╲  ╲                            │   │ │
│   │   │           f₂  f₁      f₂ f₂ f₁                        │   │ │
│   │   │          (위상 연속)                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                               │   │ │
│   │   │  • 좁은 스펙트럼                                      │   │ │
│   │   │  • 비선형 증폭기 사용 가능                             │   │ │
│   │   │  • BER 성능 우수                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              GMSK (Gaussian MSK)                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  원리: MSK + 가우시안 필터                             │   │ │
│   │   │                                                      │   │ │
│   │   │  가우시안 필터 효과:                                   │   │ │
│   │   │  • 급격한 주파수 전이 완화                              │   │ │
│   │   │  • 스펙트럼 폭 더 좁아짐                                │   │ │
│   │   │  • 인접 채널 간섭 감소                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  BT (Bandwidth-Time product):                          │   │ │
│   │   │  • GSM: BT = 0.3                                      │   │ │
│   │   │  • BT ↓ = 스펙트럼 좁음, ISI ↑                        │   │ │
│   │   │                                                      │   │ │
│   │   │  비교:                                                 │   │ │
│   │   │  MSK:  ───────╱╲───────  (급격한 전이)                 │   │ │
│   │   │  GMSK: ───────╱╲───────  (부드러운 전이)               │   │ │
│   │   │                ↑ 가우시안 필터                         │   │ │
│   │   │                                                      │   │ │
│   │   │  응용: GSM, GPRS, DECT                                │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: GMSK는 GSM의 표준 변조 방식입니다.

**💡 비유**: GMSK는 MSK에 스무딩 필터를 추가한 것입니다.

**📢 섹션 요약 비유**: GSM = GMSK!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | MSK | GMSK | 비고 |
|:---|:---|:---|:---|
| **변조 지수** | 0.5 | 0.5 | 최소 |
| **필터** | 없음 | 가우시안 | 차이 |
| **스펙트럼** | 좁음 | 더 좁음 | GMSK 우수 |
| **ISI** | 없음 | 약간 있음 | 트레이드오프 |

**비교표**

```
+------------+----------+----------+----------+
| 항목       | FSK      | MSK      | GMSK     |
+------------+----------+----------+----------+
| 위상 연속  | 아님     | 연속     | 연속     |
| 스펙트럼   | 넓음     | 좁음     | 더 좁음  |
| 포락선     | 일정     | 일정     | 일정     |
| 복잡도     | 낮음     | 중간     | 높음     |
+------------+----------+----------+----------+
```

**핵심 알고리즘**: 데이터 → 가우시안 필터 → FSK(h=0.5) → 전송

**📢 섹션 요약 비유**: BT=0.3이 GSM 표준!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 셀룰러 변조**

| 세대 | 변조 | 이유 |
|:---|:---|:---|
| **2G GSM** | GMSK | 스펙트럼 효율 |
| **3G WCDMA** | QPSK | 코드 도메인 |
| **4G LTE** | QPSK~256QAM | 적응형 |

**📢 섹션 요약 비유**: GMSK는 2G의 변조!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: GSM** → GMSK (BT=0.3)
**시나리오 2: DECT** → GMSK
**시나리오 3: 위성** → GMSK

**도입 체크리스트**: 스펙트럼, 전력, ISI

**📢 섹션 요약 비유**: 일정 포락선 = 비선형 증폭기 OK!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 스펙트럼 효율 + 전력 효율

**미래 전망**: IoT 저전력 통신

**참고 표준**: GSM, DECT

**📢 섹션 요약 비유**: MSK/GMSK는 효율의 극치!
