# PBX (사설 교환기) - Private Branch Exchange

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 기업 내 전화 교환 시스템
> 2. **가치**: 내선 통화, 비용 절감
> 3. **융합**: IP-PBX, SIP, VoIP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PBX는 기업 내 전화 통화를 관리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PBX 구성                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PBX 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ PSTN (공중망)                                │    │   │ │
│   │   │  └────────────────────┬────────────────────────┘    │   │ │
│   │   │                       │                             │   │ │
│   │   │                       ▼                             │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ PBX (사설 교환기)                           │    │   │ │
│   │   │  │                                             │    │   │ │
│   │   │  │  - 트렁크 연결 (외선)                       │    │   │ │
│   │   │  │  - 내선 연결 (내부)                         │    │   │ │
│   │   │  │  - 통화 라우팅                              │    │   │ │
│   │   │  └────────────────────┬────────────────────────┘    │   │ │
│   │   │                       │                             │   │ │
│   │   │        ┌──────────────┼──────────────┐             │   │ │
│   │   │        ▼              ▼              ▼             │   │ │
│   │   │    ┌──────┐       ┌──────┐       ┌──────┐         │   │ │
│   │   │    │내선 1│       │내선 2│       │내선 3│         │   │ │
│   │   │    └──────┘       └──────┘       └──────┘         │   │ │
│   │   │                                                      │   │ │

│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PBX vs IP-PBX                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 항목       │ PBX      │ IP-PBX   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 네트워크   │ 전용선   │ IP망     │               │   │ │
│   │   │  │ 전화       │ 아날로그 │ IP전화   │               │   │ │
│   │   │  | 확장       │ 하드웨어 │ 소프트웨어│              │   │ │

│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: PBX는 내선 간 무료 통화와 외선 연결을 제공합니다.

**💡 비유**: PBX는 기업의 전화 교환실입니다.

**📢 섹션 요약 비유:** PBX = 기업 전화망!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Trunk** | 외선 연결 |
| **Extension** | 내선 |
| **Operator** | 교환원 |
| **DID** | 직접 내선 연결 |

**비교표: PBX 기능**

```
+------------+----------+----------+----------+
| 기능       │ 설명     │ 전통     │ IP-PBX   │
+------------+----------+----------+----------+
| 내선통화   │ 무료     │ O        │ O        │
| 전송       │ 착신전환 │ O        │ O        │
| 회의       │ 3자통화  │ O        │ O        │
| 음성사서함 │ VM       │ 추가     │ 기본     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 다이얼 플랜, 라우팅

**📢 섹션 요약 비유:** DID = 직접 연결!
