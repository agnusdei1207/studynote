# Latency - 지연 시간

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 요청-응답 간 시간
> 2. **가치**: 사용자 경경험
> 3. **융합:** Network, Disk, Processing

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Latency는 요청에서 응답까지의 시간입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Latency                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              지연 시간 구성                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Total Latency =                                      │   │ │
│   │   │    Network Latency                                    │   │ │
│   │   │  + Processing Latency                                 │   │ │
│   │   │  + Queueing Latency                                   │   │ │
│   │   │  + Serialization Latency                              │   │ │
│   │   │                                                      │   │ │
│   │   │  측정 지표:                                            │   │ │
│   │   │  - Average: 평균                                       │   │ │
│   │   │  - P50: 중간값                                         │   │ │
│   │   │  - P95: 95%가 이하                                      │   │ │
│   │   │  - P99: 99%가 이하                                      │   │ │
│   │   │  - Max: 최대값                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  목표:                                                 │   │ │
│   │   │  - 웹: < 100ms                                        │   │ │
│   │   │  - API: < 50ms                                        │   │ │
│   │   │  - 실시간: < 10ms                                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: P99가 사용자 경험을 결정합니다.

**📢 섹션 요약 비유:** Latency = 대기 시간!
