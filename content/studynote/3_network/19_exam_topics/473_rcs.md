# RCS (리치 커뮤니케이션 서비스) - Rich Communication Services

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SMS/MMS의 진화
> 2. **가치**: 그룹 채팅, 파일 전송, 읽음 확인
> 3. **융합**: IMS, 챗봇, Google Messages

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

RCS는 SMS/MMS의 기능을 확장한 메시징 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    RCS 기능                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RCS vs SMS/MMS                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────┐ ┌─────────┐ ┌─────────┐              │   │ │
│   │   │  │ SMS     │ │ MMS     │ │ RCS     │              │   │ │
│   │   │  │ 텍스트  │ │ 이미지  │ │ 모든것  │              │   │ │
│   │   │  │ 160자   │ │ 제한    │ │ 무제한  │              │   │ │
│   │   │  └─────────┘ └─────────┘ └─────────┘              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RCS 기능                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 기능       │ SMS/MMS  │ RCS      │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 그룹 채팅   │ X        │ O        │               │   │ │
│   │   │  │ 파일 전송   │ 제한     │ 대용량   │               │   │ │
│   │   │  | 읽음 확인   │ X        │ O        │               │   │ │
│   │   │  │ 위치 공유   │ X        │ O        │               │   │ │
│   │   │  │ 챗봇       │ X        │ O        │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: RCS는 카카오톡 같은 기능을 기본 메시지에 제공합니다.

**💡 비유**: RCS는 메시지 앱의 진화입니다.

**📢 섹션 요약 비유:** RCS = SMS 진화!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **RCS Client** | 단말 앱 |
| **RCS Server** | 메시징 서버 |
| **IMS** | 세션 관리 |
| **Presence** | 현재 상태 |

**비교표: RCS 프로파일**

```
+------------+----------+----------+----------+
| 프로파일   │ 기능     │ 버전     │ 특징     │
+------------+----------+----------+----------+
| RCS 1.0    │ 기본     │ 초기     │ 제한     │
| RCS 2.0    │ 확장     │ 개선     │ 파일     │
| UP 1.0     │ 통합     │ GSMA     │ 표준     │
| UP 2.0     │ 챗봇     │ 최신     │ RBM     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 메시지 동기화, 상태 관리

**📢 섹션 요약 비유:** Universal Profile = 표준!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: RCS vs OTT**

| 항목 | RCS | OTT (카톡) |
|:---|:---|:---|
| **설치** | 기본 | 앱 |
| **번호** | 전화번호 | ID |
| **연동** | 통신사 | 독립 |

**📢 섹션 요약 비유:** Google Messages = RCS 클라이언트!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**📢 섹션 요약 비유:** RBM = RCS 챗봇!
