# OpenFlow - SDN 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SDN 컨트롤러-스위치 통신 프로토콜
> 2. **가치**: 프로그래밍 가능한 네트워크
> 3. **융합**: SDN, NFV와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OpenFlow는 SDN 컨트롤러와 스위치 간의 통신을 위한 표준 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OpenFlow                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OpenFlow 구조                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Controller]                                        │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ OpenFlow Protocol                           │   │ │
│   │   │       │ (TCP 6653)                                  │   │ │
│   │   │       ↓                                              │   │ │
│   │   │  [OpenFlow Switch]                                  │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ Flow Table                                  │   │ │
│   │   │       │ ┌─────────────────────────────────────┐    │   │ │
│   │   │       │ │ Match    │ Actions   │ Counters    │    │   │ │
│   │   │       │ │ In_Port  │ Output    │ Packets     │    │   │ │
│   │   │       │ │ Eth_Src  │ Drop      │ Bytes       │    │   │ │
│   │   │       │ │ Eth_Dst  │ Modify    │             │    │   │ │
│   │   │       │ │ IP_Src   │ Group     │             │    │   │ │
│   │   │       │ │ IP_Dst   │           │             │    │   │ │
│   │   │       │ └─────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              OpenFlow 메시지                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  유형        │ 설명                                  │   │ │
│   │   │  ──────────┼──────────────────────────────────────│   │ │
│   │   │  Controller→Switch:                                  │   │ │
│   │   │  • Features  │ 기능 조회                              │   │ │
│   │   │  • Flow-Mod  │ Flow 규칙 추가/수정/삭제               │   │ │
│   │   │  • Packet-Out│ 패킷 전송                             │   │ │
│   │   │                                                      │   │ │
│   │   │  Switch→Controller:                                  │   │ │
│   │   │  • Packet-In │ 패킷 전송 (매칭 실패)                  │   │ │
│   │   │  • Flow-Removed│ Flow 제거                          │   │ │
│   │   │  • Port-Status│ 포트 상태                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: OpenFlow는 Flow Table을 프로그래밍합니다.

**💡 비유**: OpenFlow는 교통경찰의 신호등 제어 시스템입니다.

**📢 섹션 요약 비유:** OpenFlow = SDN 프로토콜!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Match Fields** | 헤더 필드 |
| **Actions** | 액션 리스트 |
| **Counters** | 통계 |
| **Groups** | 그룹 테이블 |

**비교표: OpenFlow 버전**

```
+------------+----------+----------+
| 버전       │ 특징     │ 추가     │
+------------+----------+----------+
| 1.0        │ 기본     │ Flow Table│
| 1.3        │ 확장     │ Groups   │
| 1.5        │ 고급     │ Egress   │
+------------+----------+----------+
```

**핵심 알고리즸:** Flow 매칭, 액션 실행

**📢 섹션 요약 비유:** Flow-Mod = 규칙 설정!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 액션**

| 액션 | 설명 |
