# 네트워크 자동화 - Network Automation

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 수동 작업을 자동으로 수행
> 2. **가치**: 오류 감소, 속도 향상, 일관성
> 3. **융합**: Ansible, Python, CI/CD
