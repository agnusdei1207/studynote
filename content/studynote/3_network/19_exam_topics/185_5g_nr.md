# 5G NR (New Radio)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 5G 무선 접속 기술
> 2. **가치**: eMBB, uRLLC, mMTC
> 3. **융합**: mmWave, Massive MIMO와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

5G NR(New Radio)는 5G 이동통신의 무선 접속 기술 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    5G NR 기술                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  주파수 대역  │ 범위         │ 용도              │   │
│   │  ───────────┼──────────────┼──────────────────│   │
│   │  FR1 (Sub-6)│ 410MHz-7.125GHz│ 광역 커버리지      │   │
│   │  FR2 (mmWave)│ 24.25-52.6GHz │ 초고속/초저지연    │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              5G 서비스 유형 (ITU)                           │   │
│   │                                                              │   │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. eMBB (enhanced Mobile Broadband):                    │   │ │
│   │   │     • 초고속 모바일 광대역                               │   │ │
│   │   │     • 1-20 Gbps                                         │   │ │
│   │   │     • 4K/8K 스트리밍, AR/VR                              │   │ │
│   │   │                                                      │   │ │
│   │   │  2. uRLLC (ultra-Reliable Low Latency Communications): │   │ │
│   │   │     • 초신뢰 저지연 통신                                 │   │ │
│   │   │     • 지연 < 1ms                                        │   │ │
│   │   │     • 자율주행, 원격 수술, 공장 자동화                    │   │ │
│   │   │                                                      │   │ │
│   │   │  3. mMTC (massive Machine Type Communications):         │   │ │
│   │   │     • 대규모 사물통신                                   │   │ │
│   │   │     • 100만 대/km²                                      │   │ │
│   │   │     • 스마트시티, 스마트팜, IoT                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              5G 핵심 기술                                  │   │
│   │                                                              │   │
│   │   • Massive MIMO (64-256 안테나)                              │   │
│   │   • Beamforming (빔포밍)                                      │   │
│   │   • mmWave (28/39GHz)                                        │   │
│   │   • Network Slicing (네트워크 슬라이싱)                        │   │
│   │   • CU/DU 분리 (Cloud RAN)                                   │   │
│   │   • D2D (Device-to-Device)                                   │   │
│   │                                                              │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 5G NR은 Sub-6GHz와 mmWave 대역을 모두 지원합니다.

**💡 비유**: 5G는 LTE 고속도로를 자율주행 자동차 전용 도로로 업그레이드한 것입니다.

**📢 섹션 요약 비유:** 5G = eMBB + uRLLC + mMTC!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소 | 역할 | 설명 |
|:---|:---|:---|
| **gNB** | 기지국 | 5G NodeB |
| **AMF** | 제어 | 접속/이동성 |
| **UPF** | 데이터 | 사용자 평면 |
| **SMF** | 제어 | 세션 관리 |
| **UDM** | 데이터베이스 | 통합 데이터 |

**비교표**

```
+------------+----------+----------+
| 항목       │ LTE      │ 5G NR    │
+------------+----------+----------+
| 최대 속도   │ 1Gbps    │ 20Gbps   │
│ 지연       │ 10ms     │ < 1ms    │
│ 주파수     │ <6GHz    │ <52GHz   │
│ MIMO       │ 8×8      │ 64×64    │
+------------+----------+----------+
```

**핵심 알고리즸:** CP-OFDM, DFT-s-OFDM, LDPC/Polar

**📢 섹션 요약 비유:** FR1 = 커버리지, FR2 = 속도!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: NUM(Non-Orthogonal) 기법**

| 기법 | 설명 | 효과 |
|:---|:---|:---|
| **NOMA** | 전력 영역 다중화 | 용량 ↑ |
| **SCMA** | 코드북 다중화 | 용량 ↑ |
| **MUSA** | 시퀀스 다중화 | 용량 ↑ |

**📢 섹션 요약 비유:** Massive MIMO = 5G 핵심!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 도심 eMBB** → FR2 mmWave, 64T64R
**시나리오 2: 공장 uRLLC** → FR1, TSN
**시나리오 3: IoT mMTC** → FR1, NB-IoT

**도입 체크리스트:** 서비스 유형, 커버리지, 단말기

**📢 섹션 요약 비유:** 5G = 서비스 맞춤형!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)
