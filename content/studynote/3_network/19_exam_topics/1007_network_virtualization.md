# Network Virtualization - 네트워크 가상화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 논리적 네트워크 분리
> 2. **가치**: 멀티테넌시
> 3. **융합:** SDN, NFV, Overlay

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Network Virtualization은 물리 네트워크를 논리적으로 분리합니다.


**📢 섹션 요약 비유:** Network Virtualization = 가상 벽!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Tenant** | 테넌트 |
| **VNI** | Virtual Network ID |
| **Overlay** | 오버레이 |
| **Isolation** | 격리 |
| **Multi-tenancy** | 멀티테넌시 |

**핵심 알고리듬:** 테넌트 식별 → VNI 할당 → 캡슐화 → 격리된 전송
