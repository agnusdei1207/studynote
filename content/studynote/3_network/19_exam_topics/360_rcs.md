# RCS - 리치 커뮤니케이션 서비스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SMS/MMS 진화형 메시징
> 2. **가치**: 그룹 채팅, 파일 공유, 읽음 확인
> 3. **융합**: IMS, 4G/5G, 안드로이드

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

RCS(Rich Communication Services)는 SMS/MMS의 진화형 메시징입니다.


> **해설**: RCS는 앱 없이 기본 메시지 앱에서 풍부한 기능을 제공합니다.

**💡 비유**: RCS는 진화된 문자입니다. 카카오톡 같은 기능이 기본입니다.

**📢 섹션 요약 비유:** RCS = 진화형 SMS!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **RCS Client** | 단말 클라이언트 |
| **RCS Server** | RCS 서버 |
| **IMS** | IMS 코어 |
| **Chatbot** | AI 봇 |

**비교표: RCS 버전**

```
+------------+----------+----------+
| 버전       │ 기능     │ 특징     │
+------------+----------+----------+
| RCS 1.0    │ 기본     │ 최소     │
| RCS 2.0    │ 확장     │ 파일     │
| RCS UP     │ 통합     │ 안드로이드│
+------------+----------+----------+
```

**핵심 알고리즸:** SIP, CPIM

**📢 섹션 요약 비유:** RCS UP = 표준!
