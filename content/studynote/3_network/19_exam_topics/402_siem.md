# SIEM - 보안 정보 이벤트 관리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 로그 수집, 분석, 경고
> 2. **가치**: 통합 가시성, 규정 준수
> 3. **융합**: SOAR, EDR, 위협 탐지

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SIEM(Security Information and Event Management)은 보안 로그를 통합 관리합니다.


> **해설**: SIEM은 다양한 로그를 통합하여 보안 인사이트를 제공합니다.

**💡 비유**: SIEM은 보안 관제 센터입니다. 모든 CCTV를 한곳에서 봅니다.

**📢 섹션 요약 비유:** SIEM = 로그 통합!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Collectors** | 로그 수집기 |
| **Parser** | 파서/정규화 |
| **Correlation Engine** | 상관관계 엔진 |
| **Dashboard** | 대시보드 |

**비교표: SIEM 제품**

```
+------------+----------+----------+
| 항목       │ Splunk   │ QRadar   │
+------------+----------+----------+
| 타입       │ 로그플랫폼│ SIEM     │
| 규모       │ 대기업   │ 중대기업 │
| 비용       │ 높음     │ 높음     │
| 확장성     │ 높음     │ 중간     │
+------------+----------+----------+
```

**핵심 알고리즸:** 상관관계 규칙, 통계 분석

**📢 섹션 요약 비유:** 상관관계 = 핵심!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 보안 모니터링**

| 항목 | SIEM | SOAR | EDR |
|:---|:---|:---|:---|
| **범위** │ 전체 | 자동화 | 엔드포인트 |
| **기능** │ 분석 | 대응 | 탐지 |
| **통합** │ 허브 | 연동 | 소스 |

**📢 섹션 요약 비유:** SOC = SIEM!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 규정 준수 → SIEM + 리포트
**시나리오 2:** 위협 탐지 → SIEM + SOAR
**시나리오 3:** 클라우드 → Cloud SIEM

**도입 체크리스트:** 로그 소스, 보존 정책, 규칙 튜닝

**📢 섹션 요약 비유:** 로그 분석 = SIEM!
