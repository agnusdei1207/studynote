---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

etcd는 분산 키-값 저장소입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    etcd                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              etcd 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  etcd Cluster (Raft)                                 │   │ │
│   │   │  ┌─────────┐ ┌─────────┐ ┌─────────┐               │   │ │
│   │   │  │ Leader  │ │Follower │ │Follower │               │   │ │
│   │   │  │         │ │         │ │         │               │   │ │
│   │   │  │  etcd1  │ │  etcd2  │ │  etcd3  │               │   │ │
│   │   │  └─────────┘ └─────────┘ └─────────┘               │   │ │
│   │   │       ↑                                       │   │ │
│   │   │       │ gRPC API                              │   │ │
│   │   │       ↓                                       │   │ │
│   │   │  ┌─────────────────────────────────────────┐   │   │ │
│   │   │  │           Kubernetes API Server          │   │   │ │
│   │   │  └─────────────────────────────────────────┘   │   │ │
│   │   │                                                      │   │ │
│   │   │  Key-Value:                                           │   │ │
│   │   │  /registry/pods/default/my-pod → {pod spec}         │   │ │
│   │   │  /registry/services/default/my-svc → {svc spec}      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: etcd는 Kubernetes의 상태를 저장합니다.

**📢 섹션 요약 비유:** etcd = 분산 데이터베이스!
