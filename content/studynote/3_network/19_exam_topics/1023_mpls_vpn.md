# MPLS VPN - MPLS 가상사설망

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: MPLS 기반 VPN
> 2. **가치**: 격리된 프라이빗 네트워크
> 3. **융합:** L3VPN, L2VPN, VRF

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MPLS VPN은 MPLS 기반의 가상사설망 서비스입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MPLS VPN                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MPLS VPN 유형                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  L3VPN (Layer 3 VPN):                                  │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  고객A ◄───────► PE ◄──────► PE ◄───────► 고객A ││   │ │
│   │   │  │          VRF-A    │         │    VRF-A           ││   │ │
│   │   │  │                   │ MPLS    │                    ││   │ │
│   │   │  │  고객B ◄───────► PE ◄──────► PE ◄───────► 고객B ││   │ │
│   │   │  │          VRF-B    │ Core    │    VRF-B           ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  L2VPN (Layer 2 VPN):                                  │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  고객 ◄──L2 터널──► MPLS Core ◄──L2 터널──► 고객 ││   │ │
│   │   │  │        (VLL/VPLS)                               ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MPLS VPN은 고객 간 완전한 격리를 제공합니다.

**📢 섹션 요약 비유:** MPLS VPN = 전용 도로!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **PE** | Provider Edge |
| **CE** | Customer Edge |
| **P** | Provider Core |
| **VRF** | Virtual Routing and Forwarding |
| **RD/RT** | Route Distinguisher/Target |

**핵심 알고리듬:** CE 연결 → VRF 할당 → MP-BGP → 라우트 교환

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**L3VPN vs L2VPN**

| 항목 | L3VPN | L2VPN |
|:---|:---|:---|
| **계층** | L3 | L2 |
| **라우팅** | PE 수행 | CE 수행 |
| **브로드캐스트** | PE 차단 | 전달 |
| **유형** | BGP/MPLS | VLL, VPLS |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 기업 WAN → L3VPN
**시나리오 2:** 데이터센터 연결 → L2VPN
**시나리오 3:** 멀티사이트 → VPLS
