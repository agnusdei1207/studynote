# SASE - 보안 액세스 서비스 엣지

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 + 보안 통합
> 2. **가치**: 클라우드 기반 보안
> 3. **융합:** SD-WAN, CASB, ZTNA

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SASE (Secure Access Service Edge)는 네트워크와 보안을 클라우드로 통합합니다.

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **SD-WAN** | WAN 최적화 |
| **SWG** | Secure Web Gateway |
| **CASB** | Cloud Access Security Broker |
| **ZTNA** | Zero Trust Network Access |
| **FWaaS** | Firewall as a Service |

**핵심 알고리듬:** 사용자 인증 → 보안 검사 → 정책 적용 → 접근

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**SASE vs 전통적 보안**

| 항목 | SASE | 전통적 |
|:---|:---|:---|
| **위치** | 클라우드 | 온프레미스 |
| **확장** | 탄력적 | 제한적 |
| **지연** | 낮음 (PoP) | 높음 (Backhaul) |
