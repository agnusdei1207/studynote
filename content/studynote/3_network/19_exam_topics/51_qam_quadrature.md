# QAM (직교 진폭 변조)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: ASK(진폭)와 PSK(위상)를 결합한 변조
> 2. **가치**: 대역폭 효율 극대화, 고속 통신 핵심
> 3. **융합**: WiFi, LTE, 5G, DOCSIS와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

QAM(Quadrature Amplitude Modulation)은 두 개의 직교 반송파(I, Q)에 진폭 변조를 적용하여 위상과 진폭을 동시에 변조하는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    QAM (직교 진폭 변조)                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              QAM 기본 원리                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  직교 반송파:                                           │   │ │
│   │   │  • I (In-phase): cos(2πf_c·t)                         │   │ │
│   │   │  • Q (Quadrature): sin(2πf_c·t)                       │   │ │
│   │   │  • 위상차 90°로 직교                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  변조 신호:                                             │   │ │
│   │   │  s(t) = I·cos(2πf_c·t) + Q·sin(2πf_c·t)               │   │ │
│   │   │                                                      │   │ │
│   │   │  I/Q 평면:                                              │   │ │
│   │   │                    Q                                   │   │ │
│   │   │                    ↑                                   │   │ │
│   │   │                    │                                   │   │ │
│   │   │            ●       │       ●                           │   │ │
│   │   │                    │                                   │   │
│   │   │    ────────────────┼──────────────→ I                  │   │ │
│   │   │                    │                                   │   │ │
│   │   │            ●       │       ●                           │   │ │
│   │   │                    │                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  16-QAM: 4×4 격자 = 16 점 = 4비트/심볼                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              QAM 계층                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  16-QAM:  4비트/심볼, WiFi 802.11b                   │   │ │
│   │   │  64-QAM:  6비트/심볼, WiFi 802.11n, LTE              │   │ │
│   │   │  256-QAM: 8비트/심볼, WiFi 802.11ac, LTE-A           │   │ │
│   │   │  1024-QAM: 10비트/심볼, WiFi 802.11ax                │   │ │
│   │   │  4096-QAM: 12비트/심볼, WiFi 7 (802.11be)            │   │ │
│   │   │                                                      │   │ │
│   │   │  계산: 비트/심볼 = log₂(M)                            │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: QAM은 현대 디지털 통신의 핵심 변조 기술입니다.

**💡 비유**: QAM은 2차원 평면에서 위치로 정보를 전달합니다.

**📢 섹션 요약 비유**: I/Q 변조 = 직교 진폭 변조!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **I** | 동위상 | cos | X축 |
| **Q** | 직교위상 | sin | Y축 |
| **성상도** | 점 배치 | 격자 | 지도 |
| **심볼** | 정보 단위 | log₂(M)비트 | 패킷 |

**비교표**

```
+------------+----------+----------+----------+
| QAM        | 비트/심볼| SNR 요구 | 응용     |
+------------+----------+----------+----------+
| 16-QAM     | 4        | 12.4dB   | 기본     |
| 64-QAM     | 6        | 16.5dB   | WiFi 5   |
| 256-QAM    | 8        | 20.6dB   | LTE-A    |
| 1024-QAM   | 10       | 24.5dB   | WiFi 6   |
+------------+----------+----------+----------+
```

**핵심 알고리즘**: 비트 → I/Q 매핑 → 직교 변조 → 합성 → 전송

**📢 섹션 요약 비유**: QAM은 ASK + PSK의 결합!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 변조 효율**

| 변조 | 비트/심볼 | 대역폭 효율 |
|:---|:---:|:---:|
| **QPSK** | 2 | 2 bps/Hz |
| **16QAM** | 4 | 4 bps/Hz |
| **64QAM** | 6 | 6 bps/Hz |
| **256QAM** | 8 | 8 bps/Hz |

**📢 섹션 요약 비유**: QAM 차수 ↑ = 대역폭 효율 ↑!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: WiFi 6** → 1024-QAM
**시나리오 2: 5G NR** → 최대 256-QAM
**시나리오 3: 케이블 모뎀** → DOCSIS 3.1 4096-QAM

**도입 체크리스트**: SNR, 선형성, 이퀄라이저

**📢 섹션 요약 비유**: 좋은 채널 = 높은 QAM!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 대역폭 효율 극대화

**미래 전망**: 4096-QAM, 16384-QAM

**참고 표준**: IEEE 802.11, 3GPP, DOCSIS

**📢 섹션 요약 비유**: QAM은 디지털 통신의 엔진!
