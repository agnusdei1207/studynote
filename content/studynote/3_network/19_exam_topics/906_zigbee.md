# Zigbee - 지그비

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 저전력 무선 메시 네트워크
> 2. **가치**: IoT 통신 표준
> 3. **융합:** Mesh, Coordinator, End Device

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Zigbee는 저전력 무선 통신 프로토콜입니다.

```

│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
```

**📢 섹션 요약 비유:** Zigbee = IoT의 신경망!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Coordinator** | 코디네이터 |
| **Router** | 라우터 |
| **End Device** | 엔드 디바이스 |
