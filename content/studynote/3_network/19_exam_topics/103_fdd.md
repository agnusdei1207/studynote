# FDD (주파수 분할 이중화)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 업링크와 다운링크에 별도 주파수 할당
> 2. **가치**: 저지연, 연속 전송, 음성에 적합
> 3. **융합**: LTE, 3G, 2G와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

FDD(Frequency Division Duplexing)는 업링크(UL)와 다운링크(DL)에 각각 다른 주파수 대역을 할당하여 동시 전송하는 이중화 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FDD (주파수 분할 이중화)                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FDD 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  주파수 사용:                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  주파수                                                 │   │ │
│   │   │    ▲ ┌────────────────────────────┐                  │   │ │
│   │   │    │ │ DL ↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓   │                  │   │ │
│   │   │    │ │    연속 다운링크 전송        │                  │   │ │
│   │   │    │ ├────────────────────────────┤ ← 이중 주파수    │   │ │
│   │   │    │ │ UL ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑   │                  │   │ │
│   │   │    │ │    연속 업링크 전송          │                  │   │ │
│   │   │    │ └────────────────────────────┘                  │   │ │
│   │   │    └──────────────────────────────→ 시간             │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 두 개의 주파수 대역 필요                             │   │ │
│   │   │  • UL/DL 동시 전송 가능                                 │   │ │
│   │   │  • 낮은 지연                                           │   │ │
│   │   │  • 보호 대역 필요                                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FDD 주파수 예시                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  LTE FDD Band 1:                                        │   │ │
│   │   │  • DL: 2110-2170 MHz (60MHz)                           │   │ │
│   │   │  • UL: 1920-1980 MHz (60MHz)                           │   │ │
│   │   │  • 듀플렉스 간격: 130MHz                                │   │ │
│   │   │                                                      │   │ │
│   │   │  장점:                                                 │   │ │
│   │   │  • 연속 전송                                           │   │ │
│   │   │  • 낮은 레이턴시                                       │   │ │
│   │   │  • 동기 불필요                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  단점:                                                 │   │ │
│   │   │  • 주파수 자원 많이 소모                                │   │ │
│   │   │  • UL/DL 비율 고정                                     │   │ │
│   │   │  • 비대칭 트래픽 비효율                                 │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: FDD는 음성 중심의 2G/3G와 LTE에서 널리 사용됩니다.

**💡 비유**: FDD는 왕복 2차선 도로입니다.

**📢 섹션 요약 비유**: FDD = 두 주파수, 동시 전송!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **DL 주파수** | 다운링크 | 고정 대역 | 하행 |
| **UL 주파수** | 업링크 | 고정 대역 | 상행 |
| **듀플렉스 간격** | 분리 | 주파수 차이 | 중앙선 |

**비교표**

```
+------------+----------+----------+
| 항목       | FDD      | TDD      |
+------------+----------+----------+
| 주파수     | 2개      | 1개      |
| UL/DL      | 동시     | 교대     |
| 지연       | 낮음     | 높음     |
| 상호성     | X        | O        |
| 동기       | 불필요   | 필요     |
+------------+----------+----------+
```

**핵심 알고리즸**: DL 주파수 수신 ↔ UL 주파수 송신 (동시)

**📢 섹션 요약 비유**: FDD는 레이턴시가 낮다!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: FDD 장단점**

| 장점 | 단점 |
|:---|:---|
| 낮은 지연 | 주파수 낭비 |

**📢 섹션 요약 비유**: 음성은 FDD!
