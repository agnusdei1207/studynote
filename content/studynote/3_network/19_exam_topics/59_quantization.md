# 양자화 (Quantization)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 연속 진폭을 이산 레벨로 변환
> 2. **가치**: 디지털 표현, PCM의 핵심
> 3. **융합**: 선형/비선형, μ-law, A-law와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

양자화(Quantization)는 샘플링된 신호의 연속적인 진폭을 유한한 레벨로 근사하는 과정입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    양자화 (Quantization)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              양자화 과정                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  PAM 샘플 (연속 진폭):                                  │   │ │
│   │   │         ●                                             │   │ │
│   │   │       ●   ●                                           │   │ │
│   │   │     ●       ●                                         │   │ │
│   │   │   ●           ●                                       │   │ │
│   │   │ ───────────────────                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  양자화 후 (이산 레벨):                                 │   │ │
│   │   │  7 ────────────────────                               │   │ │
│   │   │  6 ──────●─────────────                               │   │ │
│   │   │  5 ────────────────────                               │   │ │
│   │   │  4 ─●───────●──────────                               │   │ │
│   │   │  3 ────────────────────                               │   │ │
│   │   │  2 ──────────●─────────                               │   │ │
│   │   │  1 ────────────────────                               │   │ │
│   │   │  0 ─●─────────────────●─                               │   │ │
│   │   │     (가장 가까운 레벨로 근사)                           │   │ │
│   │   │                                                      │   │ │
│   │   │  양자화 스텝: Δ = Vpp / L                              │   │ │
│   │   │  (L = 레벨 수, Vpp = 전체 범위)                         │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              양자화 종류                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 선형 양자화 (Linear Quantization):                  │   │ │
│   │   │     • 균일한 스텝 크기                                  │   │ │
│   │   │     • 단순하지만 SNR 낮음                               │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 비선형 양자화 (Non-linear Quantization):            │   │ │
│   │   │     • 작은 신호: 작은 스텝                              │   │ │
│   │   │     • 큰 신호: 큰 스텝                                  │   │ │
│   │   │     • 컴팬딩 (Companding)                              │   │ │
│   │   │                                                      │   │ │
│   │   │  컴팬딩 법칙:                                            │   │ │
│   │   │  • μ-law (북미, 일본): μ = 255                         │   │ │
│   │   │  • A-law (유럽): A = 87.6                              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 양자화는 필연적으로 오류(양자화 잡음)를 발생시킵니다.

**💡 비유**: 양자화는 키를 반음 단위로 근사하는 것입니다.

**📢 섹션 요약 비유**: 8비트 = 256 레벨!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **레벨 L** | 단계 수 | 2^n | 계단 |
| **스텝 Δ** | 간격 | Vpp/L | 폭 |
| **비트 n** | 해상도 | log₂L | 정밀도 |
| **SNR** | 품질 | 6.02n dB | 잡음비 |

**선형 vs 비선형**

```
+------------+----------+----------+
| 항목       | 선형     | 비선형   |
+------------+----------+----------+
| 스텝       | 균일     | 불균일   |
| SNR        | 낮음     | 높음     |
| 복잡도     | 낮음     | 높음     |
| 응용       | 오디오   | 전화     |
+------------+----------+----------+
```

**핵심 알고리즘**: 샘플 → 반올림 → 레벨 → 부호화

**📢 섹션 요약 비유**: 비트당 6dB SNR 향상!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 비트 깊이별 SNR**

| 비트 | 레벨 | SNR |
|:---|:---:|:---:|
| **8** | 256 | 48dB |
| **12** | 4096 | 72dB |
| **16** | 65536 | 96dB |
| **24** | 16M | 144dB |

**📢 섹션 요약 비유**: CD는 16비트!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 전화** → 8비트 + μ-law
**시나리오 2: CD** → 16비트 선형
**시나리오 3: 전문 오디오** → 24비트

**도입 체크리스트**: SNR, 대역폭, 저장 용량

**📢 섹션 요약 비유**: 전화는 비선형 양자화!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 디지털 표현 기반

**미래 전망**: 32비트 float, 고해상도

**참고 표준**: ITU-T G.711 (μ/A-law)

**📢 섹션 요약 비유**: 양자화는 아날로그→디지털의 다리!
