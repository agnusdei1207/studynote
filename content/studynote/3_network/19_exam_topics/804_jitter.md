# Jitter - 지터

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 지연 시간의 변동성

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Jitter는 지연 시간의 변동성입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Jitter                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              지터 시각화                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  지연 없음:                                            │   │ │
│   │   │  |----|----|----|----| (일정)                         │   │ │
│   │   │                                                      │   │ │
│   │   │  높은 지터:                                            │   │ │
│   │   │  |--|------|-|-----|--| (불규칙)                       │   │ │
│   │   │                                                      │   │ │
│   │   │  지터 버퍼:                                            │   │ │
│   │   │  ┌───────────────────────────────────────────────┐   │   │ │
│   │   │  │ 수신된 패킷                                    │   │   │ │
│   │   │  │ [1][2][  3][4][    5][6]                       │   │   │ │
│   │   │  │       ↓ 지터 버퍼                              │   │   │ │
│   │   │  │ [1][2][3][4][5][6]                             │   │   │ │
│   │   │  │       ↓ 재생                                   │   │   │ │
│   │   │  │ 일정한 간격으로 출력                             │   │   │ │
│   │   │  └───────────────────────────────────────────────┘   │   │ │
│   │   │                                                      │   │ │
│   │   │  영향:                                                │   │ │
│   │   │  - VoIP: 음질 저하                                    │   │ │
│   │   │  - Video: 화면 끊김                                   │   │ │
│   │   │  - Gaming: 렉                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 지터는 실시간 통신 품질에 영향합니다.

**📢 섹션 요약 비유:** Jitter = 버스 지연 변동!
