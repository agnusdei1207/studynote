# Digital Signature - 디지털 서명

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전자 서명
> 2. **가치**: 무결성 + 인증 + 부인방지
> 3. **융합:** RSA, ECDSA, Certificate

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Digital Signature는 문서의 진위를 증명합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Digital Signature                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              서명 과정                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  서명 생성:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Message ──→ Hash ──→ Encrypt with ──→ Signature││   │ │
│   │   │  │                          Private Key             ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  서명 검증:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Message ──→ Hash                 Compare        ││   │ │
│   │   │  │              └──────────────────────┬───→ ←───┐  ││   │ │
│   │   │  │                                     │          │  ││   │ │
│   │   │  │  Signature ──→ Decrypt with ────────┘          │  ││   │ │
│   │   │  │                 Public Key                     │  ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  특성:                                                 │   │ │
│   │   │  - 인증 (Authentication): 서명자 확인                   │   │ │
│   │   │  - 무결성 (Integrity): 변조 감지                        │   │ │
│   │   │  - 부인방지 (Non-repudiation): 서명 사실 증명            │   │ │
│   │   │                                                      │   │ │
│   │   │  알고리즘:                                              │   │ │
│   │   │  - RSA-PSS, RSA-PKCS#1                               │   │ │
│   │   │  - ECDSA (타원곡선)                                    │   │ │
│   │   │  - EdDSA (Ed25519)                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 디지털 서명은 전자 문서의 신뢰성을 보장합니다.

**📢 섹션 요약 비유:** Digital Signature = 도장!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Hash** | 해시 함수 |
| **Private Key** | 개인키 |
| **Public Key** | 공개키 |
| **Signature** | 서명값 |
| **Certificate** | 인증서 |

**핵심 알고리즸:** 해시 → 개인키 서명 → 공개키 검증

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**서명 알고리즘 비교**

| 알고리즘 | 키 길이 | 서명 크기 | 속도 |
|:---|:---|:---|:---|
| **RSA-2048** | 2048비트 | 256바이트 | 느림 |
| **ECDSA-256** | 256비트 | 64바이트 | 빠름 |
| **Ed25519** | 256비트 | 64바이트 | 매우 빠름 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 코드 서명 → RSA/ECDSA
**시나리오 2:** TLS 인증서 → ECDSA
**시나리오 3:** 블록체인 → Ed25519
