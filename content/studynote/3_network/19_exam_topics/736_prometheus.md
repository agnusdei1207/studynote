# Prometheus - 프로메테우스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 메트릭 수집 및 경고
> 2. **가치**: 시계열 데이터, 모니터링
> 3. **융합:** Grafana, AlertManager, Kubernetes

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Prometheus는 메트릭을 수집하고 경고를 발생합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Prometheus                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Prometheus 아키텍처                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Targets              Prometheus            Grafana  │   │ │
│   │   │  ┌─────────┐         ┌─────────┐          ┌─────────┐│   │ │
│   │   │  │ App     │ ── Pull→│ Retrieval│ ── Query→│Dashboard││   │ │
│   │   │  │/metrics │         │         │          │         ││   │ │
│   │   │  └─────────┘         │  TSDB   │          └─────────┘│   │ │
│   │   │  ┌─────────┐         │         │                      │   │ │
│   │   │  │Node Exp.│ ───────→│  ┌─────┐│                      │   │ │
│   │   │  └─────────┘         │  │Alert││                      │   │ │
│   │   │  ┌─────────┐         │  │Mgr  ││                      │   │ │
│   │   │  │Kubelets │ ───────→│  └─────┘│                      │   │ │
│   │   │  └─────────┘         └─────────┘                      │   │ │
│   │   │                            │                          │   │ │
│   │   │                            ↓ Alert                    │   │ │
│   │   │                       ┌─────────┐                     │   │ │
│   │   │                       │Notifier │                     │   │ │
│   │   │                       └─────────┘                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Prometheus는 Pull 방식으로 메트릭을 수집합니다.

**📢 섹션 요약 비유:** Prometheus = 건강 검진 기록!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Retrieval** | 수집 |
| **TSDB** | 시계열 DB |
| **PromQL** | 쿼리 언어 |
| **Alertmanager** | 알림 관리 |
| **Exporter** | 익스포터 |

**핵심 알고리즸:** Pull → TSDB 저장 → PromQL 쿼리 → Alert
