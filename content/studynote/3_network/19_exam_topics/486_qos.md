# QoS (서비스 품질) - Quality of Service

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 트래픽 우선순위 관리
> 2. **가치**: 지연, 손실, 지터 제어
> 3. **융합**: DiffServ, IntServ, CoS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

QoS는 네트워크 서비스 품질을 보장합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    QoS 매개변수                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              QoS 지표                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │  │ 지표       │ 설명     │ 단위     │ 목표     │     │   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │  │ 지연       │ Latency  │ ms       │ <150ms   │     │   │ │
│   │   │  │ 지터       │ Jitter   │ ms       │ <30ms    │     │   │ │
│   │   │  | 손실       │ Loss     │ %        │ <1%      │     │   │ │
│   │   │  │ 대역폭     │ Bandwidth│ Mbps     │ 확보     │     │   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: QoS는 실시간 트래픽에 우선순위를 부여합니다.

**💡 비유**: QoS는 긴급 차선입니다.

**📢 섹션 요약 비유:** QoS = 품질 보장!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **DiffServ** | 차등화 서비스 |
| **IntServ** | 통합 서비스 |
| **DSCP** | Differentiated Services Code Point |
| **CoS** | Class of Service |

**핵심 알고리즸:** 분류, 마킹, 폴리싱

**📢 섹션 요약 비유:** DSCP = QoS 마킹!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**DiffServ vs IntServ**

| 항목 | DiffServ | IntServ |
|:---|:---|:---|
| **상태** | 무상태 | 상태 유지 |
| **확장성** | 높음 | 낮음 |
| **복잡성** | 낮음 | 높음 |
