# 가시광 통신 (VLC / Li-Fi)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 가시광(380-750nm)으로 데이터 전송
> 2. **가치**: 면허 불필요, EMI 없음, 보안성
> 3. **융합**: LED 조명, IoT와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

가시광 통신(VLC, Visible Light Communication)은 가시광을 이용한 무선 통신으로, Li-Fi(Light Fidelity)는 이의 상용화 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    가시광 통신 (VLC / Li-Fi)                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VLC 시스템 구조                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [송신기]              [채널]           [수신기]      │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────┐  디지털    ┌─────┐  가시광   ┌─────┐        │   │ │
│   │   │  │변조 │───────►│ LED │─────────►│포토 │        │   │ │
│   │   │  │기   │  신호     │조명 │  신호     │다이오드│       │   │ │
│   │   │  └─────┘           └─────┘           └─────┘        │   │ │
│   │   │                                                      │   │ │
│   │   │  원리:                                                   │   │ │
│   │   │  • LED를 초고속으로 깜빡임 (인식 불가)                   │   │ │
│   │   │  • ON/OFF = 1/0 (OOK 변조)                             │   │ │
│   │   │  • 조명과 통신 동시 가능                                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VLC vs Wi-Fi                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  항목        │ VLC (Li-Fi)      │ Wi-Fi             │   │ │
│   │   │  ───────────┼──────────────────┼──────────────────│   │ │
│   │   │  매체        │ 가시광            │ 전파              │   │ │
│   │   │  주파수      │ 400-790THz       │ 2.4/5GHz          │   │ │
│   │   │  속도        │ 수Gbps           │ 수백Mbps          │   │ │
│   │   │  거리        │ 수m              │ 수십m             │   │ │
│   │   │  투과        │ 불가 (벽)         │ 가능              │   │ │
│   │   │  간섭        │ 없음              │ 있음              │   │ │
│   │   │  보안        │ 높음              │ 낮음              │   │ │
│   │   │  면허        │ 불필요            │ 필요              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: VLC는 조명 인프라를 활용한 새로운 무선 통신 방식입니다.

**💡 비유**: VLC는 손전등으로 모스 부호를 보내는 것과 같습니다.

**📢 섹션 요약 비유:** Li-Fi = 조명으로 통신!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **LED** | 광원/송신 | 고속 변조 | 전구 |
| **포토다이오드** | 수신 | 광-전 변환 | 눈 |
| **변조기** | 신호 변조 | OOK, OFDM | 인코더 |

**비교표**

```
+------------+----------+----------+
| 항목       | Wi-Fi    | Li-Fi    |
+------------+----------+----------+
| 대역       │ 전파     │ 가시광   │
| 커버리지   │ 넓음     │ 좁음     │
| 보안       │ 낮음     │ 높음     │
| EMI        │ 있음     │ 없음     │
+------------+----------+----------+
```

**핵심 알고리즸:** OOK, VPPM, OFDM

**📢 섹션 요약 비유:** LED = 조명 + 통신!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 광통신 기술**

| 기술 | 파장 | 용도 |
|:---|:---:|:---|
| **적외선 (IrDA)** | 850nm | 단거리 |
| **VLC** | 380-750nm | 조명+통신 |
| **자외선** | <380nm | 살균+통신 |

**📢 섹션 요약 비유:** VLC는 조명 혁명!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 병원** → EMI 없는 VLC
**시나리오 2: 비행기** → 안전한 통신
**시나리오 3: 보안 구역** → 투과 불가 VLC

**도입 체크리스트:** LOS 확보, 조명 인프라, 데이터 속도

**📢 섹션 요약 비유:** VLC는 틈새시장!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 전파 혼잡 해결, 보안 통신

**미래 전망:** IoT, 스마트홈 확대

**참고 표준:** IEEE 802.15.7

**📢 섹션 요약 비유:** Li-Fi는 조명의 미래!
