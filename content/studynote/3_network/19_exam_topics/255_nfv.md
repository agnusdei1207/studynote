# NFV (Network Functions Virtualization) - 네트워크 기능 가상화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 기능을 가상화
> 2. **가치**: 비용 절감, 유연성
> 3. **융합**: SDN, 클라우드와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NFV(Network Functions Virtualization)는 네트워크 기능을 전용 하드웨어에서 범용 서버의 가상화로 이동하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NFV (Network Functions Virtualization)            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NFV 변화                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Before (전통):                                      │   │ │
│   │   │  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                │   │ │
│   │   │  │라우터│  │방화벽│  │LB   │  │DPI  │                │   │ │
│   │   │  │(HW) │  │(HW) │  │(HW) │  │(HW) │                │   │ │
│   │   │  └─────┘  └─────┘  └─────┘  └─────┘                │   │ │
│   │   │  전용 하드웨어                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  After (NFV):                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │         범용 서버 (x86)                     │    │   │ │
│   │   │  │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐          │    │   │ │
│   │   │  │  │vRouter│ │vFW  │ │vLB │ │vDPI│          │    │   │ │
│   │   │  │  │(VM) │ │(VM) │ │(VM)│ │(VM)│           │    │   │ │
│   │   │  │  └─────┘ └─────┘ └─────┘ └─────┘          │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │  가상화 (VM/Container)                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VNF 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  VNF        │ 설명                                   │   │ │
│   │   │  ──────────┼──────────────────────────────────────│   │ │
│   │   │  vRouter   │ 가상 라우터                            │   │ │
│   │   │  vFW       │ 가상 방화벽                            │   │ │
│   │   │  vLB       │ 가상 로드밸런서                        │   │ │
│   │   │  vCPE      │ 가상 CPE                              │   │ │
│   │   │  vEPC      │ 가상 EPC (4G)                         │   │ │
│   │   │  vIMS      │ 가상 IMS                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NFV는 전용 하드웨어를 범용 서버의 소프트웨어로 대체합니다.

**💡 비유**: NFV는 스마트폰 앱입니다. 필요한 기능을 앱으로 설치합니다.

**📢 섹션 요약 비유:** NFV = 네트워크 가상화!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NFVI** | Infrastructure |
| **VNF** | Virtual Network Function |
| **MANO** | Management & Orchestration |
| **VNFM** | VNF Manager |

**비교표: NFV vs SDN**

```
+------------+----------+----------+
| 항목       │ NFV      │ SDN      │
+------------+----------+----------+
| 목적       │ 가상화   │ 제어 분리│
| 대상       │ 네트워크 기능│ 네트워크 │
| 접근       │ 하드웨어 │ 소프트웨어│
| 관계       │ 독립     │ 독립     │
+------------+----------+----------+
```

**핵심 알고리즸:** 오케스트레이션, 라이프사이클 관리

**📢 섹션 요약 비유:** NFV + SDN = 완벽한 조합!
