### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Server** | 중앙 서버 |
| **Proxy** | 분산 프록시 |
| **Agent** | 에이전트 |
| **Template** | 템플릿 |

**비교표: Zabbix vs Prometheus**

```
+------------+----------+----------+
| 항목       │ Zabbix   │ Prometheus│
+------------+----------+----------+
| 데이터     │ 폴링/풀  │ 풀       │
| 저장소     │ SQL      │ TSDB     │
| 알림       │ 내장     │ Alertmgr │
| UI         │ 내장     │ Grafana  │
+------------+----------+----------+
```

**핵심 알고리즸:** 폴링, 트리거, 템플릿

**📢 섹션 요약 비유:** Template = 재사용!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
