# Buffer - 버퍼

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 임시 저장 공간
> 2. **가치**: 속도 차이 흡수
> 3. **융합:** TCP Buffer, Ring Buffer, Zero-Copy

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Buffer는 데이터를 임시로 저장하는 공간입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Buffer                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              버퍼 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  TCP Buffer:                                           │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │              Send Buffer                        ││   │ │
│   │   │  │  ┌───┬───┬───┬───┬───┬───┬───┬───┐            ││   │ │
│   │   │  │  │ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │   │            ││   │ │
│   │   │  │  └───┴───┴───┴───┴───┴───┴───┴───┘            ││   │ │
│   │   │  │    ↑───────────────↑                          ││   │ │
│   │   │  │    전송 대기 중                                  ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │              Receive Buffer                     ││   │ │
│   │   │  │  ┌───┬───┬───┬───┬───┬───┬───┬───┐            ││   │ │
│   │   │  │  │ 1 │ 2 │ 3 │ 4 │   │   │   │   │            ││   │ │
```

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Send Buffer** | 송신 버퍼 |
| **Recv Buffer** | 수신 버퍼 |
| **Ring Buffer** | 환형 버퍼 |
| **Zero-Copy** | 제로 카피 |
| **Bufferbloat** | 버퍼블로트 |
