# 트래픽 폴리싱 - Traffic Policing

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 트래픽 속도 제한
> 2. **가치**: 네트워크 보호, 공정성
> 3. **융합**: Token Bucket, CAR, CoPP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

트래픽 폴리싱은 속도를 제한하고 초과 트래픽을 폐기합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    트래픽 폴리싱                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              폴리싱 vs 쉐이핑                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 항목       │ 폴리싱   │ 쉐이핑   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 동작       │ 폐기     │ 지연     │               │   │ │
│   │   │  │ 버퍼       │ 없음     │ 있음     │               │   │ │
│   │   │  | 지연       │ 없음     │ 발생     │               │   │ │
│   │   │  │ 용도       │ 속도제한 │ 평탄화   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 폴리싱은 초과 트래픽을 버립니다.

**📢 섹션 요약 비유:** Policing = 속도 제한!
