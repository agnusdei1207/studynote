# Join - 조인

> 1. **본질**: 테이블 결합
> 2. **가치**: 관계형 데이터 조회
> 3. **융합:** SQL, Normalization, Query

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Join은 두 테이블을 결합합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Join                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              조인 종류                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  INNER JOIN:            LEFT JOIN:                   │   │ │
│   │   │  ┌───┐   ┌───┐         ┌───┐   ┌───┐                │   │ │
│   │   │  │ A │ ∩ │ B │         │ A │←──│ B │                │   │ │
│   │   │  └───┘   └───┘         └───┘   └───┘                │   │ │
│   │   │  (교집합)               (A 전체 + B 일치)             │   │ │
│   │   │                                                      │   │ │
│   │   │  RIGHT JOIN:           FULL OUTER JOIN:              │   │ │
│   │   │  ┌───┐   ┌───┐         ┌───┐   ┌───┐                │   │ │
│   │   │  │ A │──→│ B │         │ A │ ∪ │ B │                │   │ │
│   │   │  └───┘   └───┘         └───┘   └───┘                │   │ │
│   │   │  (B 전체 + A 일치)      (합집합)                      │   │ │
│   │   │                                                      │   │ │
│   │   │  CROSS JOIN:          SELF JOIN:                     │   │ │
│   │   │  A × B (카티션 곱)     자기 자신과 조인               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 조인은 관계형 데이터베이스의 핵심 연산입니다.

**📢 섹션 요약 비유:** Join = 두 문서 합치기!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Inner Join** | 내부 조인 |
| **Left Join** | 좌측 외부 조인 |
| **Right Join** | 우측 외부 조인 |
| **Full Join** | 전체 외부 조인 |
| **Cross Join** | 교차 조인 |

**핵심 알고리즸:** 키 매칭 → 결과 생성

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
