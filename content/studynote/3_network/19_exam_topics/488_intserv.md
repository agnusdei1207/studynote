# IntServ (통합 서비스) - Integrated Services

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 흐름 기반 자원 예약
> 2. **가치**: 엄격한 QoS 보장
> 3. **융합**: RSVP, Guaranteed Rate

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IntServ는 흐름별로 자원을 예약합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IntServ 구조                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IntServ 서비스 클래스                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 클래스     │ 설명     │ 보장     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ Guaranteed │ 엄격     │ 지연/대역폭│             │   │ │
│   │   │  │ Controlled | 적당     │ 대역폭   │               │   │ │
│   │   │  | Best Effort│ 없음     │ 없음     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IntServ는 RSVP로 자원을 예약합니다.

**📢 섹션 요약 비유:** IntServ = 흐름별 예약!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**IntServ vs DiffServ**

| 항목 | IntServ | DiffServ |
|:---|:---|:---|
| **상태** | 유지 | 무상태 |
| **확장성** | 낮음 | 높음 |
| **보장** | 엄격 | 클래스별 |
