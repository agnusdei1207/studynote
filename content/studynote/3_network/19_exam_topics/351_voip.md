# VoIP - 인터넷 전화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IP 네트워크 기반 음성 통신
> 2. **가치**: 비용 절감, 융합 서비스
> 3. **융합**: SIP, RTP, 코덱과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VoIP(Voice over IP)는 IP 네트워크로 음성 통신을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VoIP                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VoIP 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [IP Phone]                                         │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ SIP 시그널링                                   │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           VoIP Server (PBX)                 │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • SIP Proxy                                 │    │   │ │
│   │   │  │  • Registrar                                 │    │   │ │
│   │   │  │  • 미디어 서버                                │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ RTP 미디어 스트림                            │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [IP Phone / PSTN Gateway]                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VoIP 프로토콜 스택                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 계층       │ 프로토콜 │ 기능     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 시그널링   │ SIP      │ 통화 설정│               │   │ │
│   │   │  │ 미디어     │ RTP      │ 음성 전송│               │   │ │
│   │   │  | 제어       │ RTCP     │ 품질     │               │   │ │
│   │   │  │ 보안       │ SRTP     │ 암호화   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: VoIP는 SIP로 시그널링하고 RTP로 미디어를 전송합니다.

**💡 비유**: VoIP는 인터넷으로 전화하는 것입니다. 전화선 대신 인터넷선입니다.

**📢 섹션 요약 비유:** VoIP = 인터넷 전화!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **IP Phone** | IP 전화기 |
| **PBX** | 사설 교환기 |
| **Gateway** | PSTN 게이트웨이 |
| **SBC** | 세션 보더 컨트롤러 |

**비교표: VoIP vs PSTN**

```
+------------+----------+----------+
| 항목       │ VoIP     │ PSTN     │
+------------+----------+----------+
| 네트워크   │ IP       │ TDM      │
| 비용       │ 낮음     │ 높음     │
| 확장성     │ 높음     │ 낮음     │
| 품질       │ 변동     │ 일정     │
+------------+----------+----------+
```

**핵심 알고리즸:** 코덱, 지터 버퍼

**📢 섹션 요약 비유:** QoS = VoIP 필수!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 음성 코덱**

| 항목 | G.711 | G.729 | Opus |
|:---|:---|:---|:---|
| **대역폭** | 64kbps | 8kbps | 가변 |
| **품질** | 높음 | 중간 | 높음 |
| **복잡도** | 낮음 | 높음 | 중간 |

**📢 섹션 요약 비유:** 기업 = VoIP!
