### Ⅰ. 개요 (Context & Background)

**개념 정의**

Partitioning은 데이터를 논리적/물리적으로 분할합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Partitioning                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              파티셔닝 유형                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Horizontal Partitioning:                            │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ Partition 1 │ Partition 2 │ Partition 3       │  │   │ │
│   │   │  │ Row 1-1000  │ Row 1001-   │ Row 2001-         │  │   │ │
│   │   │  │             │ 2000        │ 3000              │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  Vertical Partitioning:                              │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ Partition 1  │ Partition 2                    │  │   │ │
│   │   │  │ ID, Name     │ ID, Email, Phone               │  │   │ │
│   │   │  │ (자주 사용)  │ (가끔 사용)                    │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  종류: Range, List, Hash, Composite                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 파티셔닝은 관리와 성능을 위해 분할합니다.

**📢 섹션 요약 비유:** Partitioning = 서랍 정리!
