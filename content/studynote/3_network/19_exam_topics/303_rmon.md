# RMON - 원격 모니터링

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 원격 네트워크 모니터링 MIB
> 2. **가치**: 트래픽 분석, 프로브
> 3. **융합**: SNMP, NetFlow와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

RMON(Remote Monitoring)은 원격 네트워크 세그먼트를 모니터링하는 MIB 확장입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    RMON                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RMON 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [NMS]                                              │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ SNMP                                        │   │ │
│   │   │       │                                              │   │ │
│   │   │       ├──────────────┬──────────────┐                │   │ │
│   │   │       ↓              ↓              ↓                │   │ │
│   │   │  ┌──────────┐  ┌──────────┐  ┌──────────┐            │   │ │
│   │   │  │ RMON     │  │ RMON     │  │ RMON     │            │   │ │
│   │   │  │ Probe 1  │  │ Probe 2  │  │ Probe 3  │            │   │ │
│   │   │  │          │  │          │  │          │            │   │ │
│   │   │  │ 세그먼트1 │  │ 세그먼트2 │  │ 세그먼트3 │            │   │ │
│   │   │  └──────────┘  └──────────┘  └──────────┘            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RMON 그룹                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  RMON-1 (RFC 2819):                                   │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 그룹       │ 번호     │ 기능     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ statistics │ 1        │ 통계     │               │   │ │
│   │   │  │ history    │ 2        │ 이력     │               │   │ │
│   │   │  │ alarm      │ 3        │ 알림     │               │   │ │
│   │   │  │ host       │ 4        │ 호스트   │               │   │ │
│   │   │  │ matrix     │ 5        │ 매트릭스 │               │   │ │
│   │   │  │ filter     │ 6        │ 필터     │               │   │ │
│   │   │  │ capture    │ 7        │ 캡처     │               │   │ │
│   │   │  │ event      │ 8        │ 이벤트   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   │  RMON-2 (RFC 2021):                                   │   │ │
│   │   │  • 프로토콜 분석 (L3-L7)                              │   │ │
│   │   │  • 애플리케이션 가시성                                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: RMON은 프로브가 로컬에서 데이터를 수집하고 NMS에 보고합니다.

**💡 비유**: RMON은 CCTV 카메라입니다. 현장에서 녹화하고 필요시 전송합니다.

**📢 섹션 요약 비유:** RMON = 분산 모니터링!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Probe** | RMON 에이전트 |
| **Statistics** | 통계 |
| **History** | 이력 |
| **Alarm** | 임계값 알림 |

**비교표: RMON-1 vs RMON-2**

```
+------------+----------+----------+
| 항목       │ RMON-1   │ RMON-2   │
+------------+----------+----------+
| 계층       │ L1-L2    │ L3-L7    │
| 분석       │ MAC      │ IP/App   │
| 용도       │ 기본     │ 상세     │
+------------+----------+----------+
```

**핵심 알고리즸:** 통계, 알림, 캡처

**📢 섹션 요약 비유:** Alarm = 임계값!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: RMON vs NetFlow**

| 항목 | RMON | NetFlow |
|:---|:---|:---|
| **방식** | 샘플링/전체 | 플로우 |
| **위치** | 세그먼트 | 라우터 |
| **상세도** | 높음 | 중간 |

**📢 섹션 요약 비유:** RMON = 세그먼트 분석!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: LAN 모니터링** → RMON-1
**시나리오 2: 애플리케이션** → RMON-2
**시나리오 3: 대규모** → NetFlow + RMON

**도입 체크리스트:** 프로브, 그룹, 비용

**📢 섹션 요약 비유:** Probe = RMON 장비!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 상세 분석
