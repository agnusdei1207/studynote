# 확산 스펙트럼 (Spread Spectrum) - DSSS/FHSS

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호를 넓은 대역으로 확산
> 2. **가치**: 간섭 저항, 보안, 다중 접속
> 3. **융합**: CDMA, WiFi, GPS와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

확산 스펙트럼(Spread Spectrum)은 신호를 원래 대역폭보다 훨씬 넓은 대역으로 확산시키는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    확산 스펙트럼 (Spread Spectrum)                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              확산 스펙트럼 종류                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. DSSS (Direct Sequence Spread Spectrum):            │   │ │
│   │   │     • PN 코드로 직접 확산                              │   │ │
│   │   │     • 데이터 × PN 코드 → 확산 신호                     │   │ │
│   │   │     • GPS, CDMA, 802.11b                              │   │ │
│   │   │     • Processing Gain = 10·log(BW/Data)               │   │ │
│   │   │                                                      │   │ │
│   │   │  2. FHSS (Frequency Hopping Spread Spectrum):          │   │ │
│   │   │     • 주파수를 빠르게 변경                              │   │ │
│   │   │     • 블루투스, 802.11 구형                            │   │ │
│   │   │     • 간섭 회피에 효과적                                │   │ │
│   │   │                                                      │   │ │
│   │   │  3. THSS (Time Hopping Spread Spectrum):               │   │ │
│   │   │     • 시간 영역에서 확산                                │   │ │
│   │   │     • UWB에서 사용                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  4. Chirp Spread Spectrum:                             │   │ │
│   │   │     • 주파수 선형 변조                                  │   │ │
│   │   │     • LoRa에서 사용                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DSSS 원리                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [데이터] × [PN 코드] = [확산 신호]                   │   │ │
│   │   │                                                      │   │ │
│   │   │  Data:    1   0   1   1   0                         │   │ │
│   │   │  PN:     +1  -1  +1  +1  -1                         │   │ │
│   │   │  Output: +1  -1  +1  +1  -1  (chip rate)            │   │ │
│   │   │                                                      │   │ │
│   │   │  Processing Gain (dB) = 10·log(Chip Rate / Data Rate)│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 확산 스펙트럼은 간섭 저항과 보안성을 동시에 제공합니다.

**💡 비유**: 확산 스펙트럼은 한 사람이 크게 말하는 대신 여러 사람이 작게 말하는 것입니다.

**📢 섹션 요약 비유:** Processing Gain = 확산 이득!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 기술 | 확산 방식 | 대표 용도 | 장점 |
|:---|:---|:---|:---|
| **DSSS** | PN 코드 곱셈 | GPS, CDMA | 간섭 저항 |
| **FHSS** | 주파수 도약 | 블루투스 | 간섭 회피 |
| **THSS** | 시간 도약 | UWB | 저전력 |
| **Chirp** | 주파수 변조 | LoRa | 장거리 |

**비교표**

```
+------------+----------+----------+
| 항목       │ DSSS     │ FHSS     │
+------------+----------+----------+
| 확산 방식   │ 코드 곱   │ 주파수 변경│
│ 동기       │ 필요     │ 선택적   │
│ 간섭 대응   │ 저항     │ 회피     │
│ 구현       │ 복잡     │ 단순     │
+------------+----------+----------+
```

**핵심 알고리즸:** PN 코드, 상관기

**📢 섹션 요약 비유:** PN 코드 = 암호 키!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: Processing Gain**

| 시스템 | Chip Rate | Data Rate | PG |
|:---|:---:|:---:|:---:|
| **GPS** | 1.023 Mcps | 50 bps | 43 dB |
| **802.11b** | 11 Mcps | 1 Mbps | 10 dB |
| **CDMA** | 1.2288 Mcps | 9.6 kbps | 21 dB |

**📢 섹션 요약 비유:** PG 높을수록 간섭 저항!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: GPS** → DSSS, 높은 PG
**시나리오 2: 블루투스** → FHSS, 1600 hops/s
**시나리오 3: LoRa IoT** → Chirp, 장거리

**도입 체크리스트:** 간섭 환경, 대역폭, 보안 요구

**📢 섹션 요약 비유:** 확산 스펙트럼은 잡음 속에서도 들려!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 간섭 저항, LPI/LPD

**미래 전망:** UWB, LoRa로 확장

**참고 표준:** IEEE 802.11, 3GPP2

**📢 섹션 요약 비유:** 확산 스펙트럼은 통신의 위장술!
