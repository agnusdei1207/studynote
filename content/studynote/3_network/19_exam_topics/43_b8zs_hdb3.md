# B8ZS / HDB3 (Zero Substitution)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 연속 0을 특수 패턴으로 대체하여 동기화 유지
> 2. **가치**: AMI의 연속 0 문제 해결
> 3. **융합**: T1(B8ZS), E1(HDB3), 전화망과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

B8ZS와 HDB3는 AMI 부호화에서 연속 0이 발생할 때 동기화를 유지하기 위해 특수 패턴으로 대체하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    B8ZS / HDB3 (Zero Substitution)                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              B8ZS (Bipolar with 8-Zero Substitution)          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  문제: AMI에서 8개 연속 0 → 동기화 불가               │   │ │
│   │   │                                                      │   │ │
│   │   │  해결: 8개 연속 0 → 특수 패턴 대체                     │   │ │
│   │   │                                                      │   │ │
│   │   │  대체 패턴: 000VB0VB                                  │   │ │
│   │   │  • V (Violation): AMI 위반 (이전 1과 같은 극성)        │   │ │
│   │   │  • B (Bipolar): 정상 AMI 규칙                         │   │ │
│   │   │                                                      │   │ │
│   │   │  예시:                                               │   │ │
│   │   │  원본: 00000000                                      │   │ │
│   │   │        ↓                                             │   │ │
│   │   │  대체: 000V B0V B                                    │   │ │
│   │   │        ↑  ↑  ↑  ↑                                    │   │ │
│   │   │        0  위반 0 위반                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  응용: 북미 T1 회선 (DS1)                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              HDB3 (High-Density Bipolar 3 zeros)              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  문제: AMI에서 4개 연속 0 → 동기화 어려움              │   │ │
│   │   │                                                      │   │ │
│   │   │  해결: 4개 연속 0 → 특수 패턴 대체                     │   │ │
│   │   │                                                      │   │ │
│   │   │  대체 규칙:                                          │   │ │
│   │   │  • 이후 1 개수가 홀수: 000V                          │   │ │
│   │   │  • 이후 1 개수가 짝수: B00V                          │   │ │
│   │   │                                                      │   │ │
│   │   │  V: 위반 (같은 극성)                                  │   │ │
│   │   │  B: 정상 AMI (교대 극성)                              │   │ │
│   │   │                                                      │   │ │
│   │   │  예시:                                               │   │ │
│   │   │  데이터: 1 0 0 0 0 1                                  │   │ │
│   │   │  이후 1: 2개 (짝수) → B00V                           │   │ │
│   │   │  결과: + 0 0 0 V +  (V은 +로 위반)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  응용: 유럽 E1 회선                                   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: B8ZS는 8개, HDB3는 4개 연속 0을 대체합니다.

**💡 비유**: B8ZS/HDB3는 긴 침묵을 특수 신호로 채우는 것입니다.

**📢 섹션 요약 비유**: 북미 B8ZS, 유럽 HDB3!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | B8ZS | HDB3 | 비고 |
|:---|:---:|:---:|:---|
| **대체 조건** | 8개 0 | 4개 0 | 임계값 |
| **패턴** | 000VB0VB | 000V/B00V | 위반 |
| **지역** | 북미 | 유럽 | 표준 |
| **회선** | T1 | E1 | 적용 |

**위반(Violation)의 의미**

```
정상 AMI: 1은 ±V 교대
         +  -  +  -

위반(V): 이전 1과 같은 극성
         +  +  (위반!)
         또는
         -  -  (위반!)

수신측: 위반 감지 → 특수 패턴 → 원본 복원
```

**핵심 알고리즘**: 연속 0 감지 → 패턴 대체 → 수신측 복원

**📢 섹션 요약 비유**: V는 의도적 AMI 위반!
