# HPC (High Performance Computing) - 고성능 컴퓨팅

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 슈퍼컴퓨팅
> 2. **가치**: 대규모 계산
> 3. **융합:** Cluster, MPI, Parallel Computing

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

HPC는 대규모 계산을 위한 고성능 컴퓨팅입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    HPC                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              HPC 클러스터 구조                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │              Head Node                        │  │   │ │
│   │   │  │         (Job Scheduler)                       │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                        │                             │   │ │
│   │   │           ┌────────────┼────────────┐                │   │ │
│   │   │           ↓            ↓            ↓                │   │ │
│   │   │   ┌───────────┐ ┌───────────┐ ┌───────────┐          │   │ │
│   │   │   │ Compute   │ │ Compute   │ │ Compute   │          │   │ │
│   │   │   │ Node 1    │ │ Node 2    │ │ Node N    │          │   │ │
│   │   │   │ CPU+GPU   │ │ CPU+GPU   │ │ CPU+GPU   │          │   │ │
│   │   │   └─────┬─────┘ └─────┬─────┘ └─────┬─────┘          │   │ │
│   │   │         │             │             │                  │   │ │
│   │   │         └─────────────┼─────────────┘                  │   │ │
│   │   │                       │                                │   │ │
│   │   │               High-Speed Network                       │   │ │
│   │   │                   (InfiniBand)                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: HPC는 여러 노드가 고속 네트워크로 연결됩니다.

**📢 섹션 요약 비유:** HPC = 슈퍼컴퓨터!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Compute Node** | 계산 노드 |
| **Head Node** | 관리 노드 |
| **Interconnect** | 노드 간 연결 |
| **Storage** | 스토리지 |
| **Scheduler** | 작업 스케줄러 |

**핵심 알고리즸:** 작업 분할 → 노드 할당 → 병렬 실행 → 결과 병합
