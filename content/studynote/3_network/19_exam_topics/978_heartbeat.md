### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Interval** | 전송 간격 |
| **Timeout** | 대기 시간 |
| **Dead** | 사망 판정 |
| **Split-brain** | 분할 뇌 |
| **Quorum** | 정족수 |

**핵심 알고리듬:** 전송 → 대기 → 응답 없으면 장애
