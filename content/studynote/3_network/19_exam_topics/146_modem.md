# 모뎀 (Modem)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디지털↔아날로그 신호 변환
> 2. **가치**: 전화선 데이터 통신
> 3. **융합**: DSL, 케이블, 무선과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

모뎀(Modem, Modulator/Demodulator)은 디지털 신호를 아날로그로 변조(Modulation)하고, 아날로그 신호를 디지털로 복조(Demodulation)하는 장치입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    모뎀 (Modem)                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              모뎀 동작 원리                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [디지털 장비]      [모뎀]        [아날로그 회선]      │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌───────┐      ┌──────────┐      ┌─────────┐       │   │ │
│   │   │  │컴퓨터 │──────│변조 복조│──────│전화선   │       │   │ │
│   │   │  │(DTE)  │디지털│(Mod/Demod)│아날로그│(PSTN)   │       │   │ │
│   │   │  └───────┘      └──────────┘      └─────────┘       │   │ │
│   │   │                   │                                 │   │ │
│   │   │            ┌──────┴──────┐                          │   │ │
│   │   │            │             │                          │   │ │
│   │   │          변조           복조                         │   │ │
│   │   │    (Digital→Analog)  (Analog→Digital)                │   │ │
│   │   │                                                      │   │ │
│   │   │  변조 방식:                                             │   │ │
│   │   │  • ASK (진폭 편이): 진폭 변화                          │   │ │
│   │   │  • FSK (주파수 편이): 주파수 변화                      │   │ │
│   │   │  • PSK (위상 편이): 위상 변화                          │   │ │
│   │   │  • QAM (직교 진폭): 진폭+위상                          │   │ │
│   │   │                                                      │   │ │
│   │   │  모뎀 속도 역사:                                        │   │ │
│   │   │  300bps → 1200bps → 2400bps → 9600bps → 14.4K →     │   │ │
│   │   │  28.8K → 33.6K → 56K (V.90/V.92)                     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              모뎀 종류                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  • 다이얼업 모뎀: 전화선, 최대 56Kbps                   │   │ │
│   │   │  • DSL 모뎀: ADSL/VDSL, 수Mbps~수십Mbps               │   │ │
│   │   │  • 케이블 모뎀: CATV, 수십~수백Mbps                    │   │ │
│   │   │  • 광 모뎀: FTTH, 수백Mbps~Gbps                        │   │ │
│   │   │  • 무선 모뎀: 셀룰러, WiFi                              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 모뎀은 디지털 통신의 시작, 아날로그 회선을 통한 데이터 전송을 가능하게 합니다.

**💡 비유**: 모뎀은 번역기입니다. 디지털어를 아날로그어로, 아날로그어를 디지털어로 번역합니다.

**📢 섹션 요약 비유:** 모뎀 = Modulator + Demodulator!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **변조기** | D→A 변환 | 반송파 변조 | 번역 |
| **복조기** | A→D 변환 | 신호 검출 | 역번역 |
| **등화기** | 신호 보정 | 채널 보상 | 교정 |

**비교표**

```
+------------+----------+----------+
| 항목       | 다이얼업 | DSL      |
+------------+----------+----------+
| 속도       | 56Kbps   | 100Mbps+ |
| 주파수     | 음성대역 | 고주파   |
| 동시       | 안 됨    | 됨       |
| 용도       | 레거시   | 현재     |
+------------+----------+----------+
```

**핵심 알고리즸**: QAM 변조, 등화, 에러 정정

**📢 섹션 요약 비유:** 56K = 다이얼업 한계!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 모뎀 표준**

| 표준 | 속도 | 연도 |
|:---|:---:|:---:|
| **V.32** | 9.6K | 1984 |
| **V.34** | 28.8K | 1994 |
| **V.90** | 56K | 1998 |

**📢 섹션 요약 비유:** V.90 = 56K 다이얼업!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 레거시 연결** → 다이얼업 모뎀
**시나리오 2: 가정용 인터넷** → DSL/케이블 모뎀
**시나리오 3: 기업** → 광 모뎀/전용선

**도입 체크리스트**: 속도 요구, 회선 타입, 비용

**📢 섹션 요약 비유:** 다이얼업은 역사, DSL/케이블은 현재!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 아날로그 회선 디지털화

**미래 전망:** 광 모뎀/무선 모뎀으로 진화

**참고 표준:** ITU-T V 시리즈

**📢 섹션 요약 비유:** 모뎀은 통신의 교량!
