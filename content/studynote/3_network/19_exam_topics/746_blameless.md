# Blameless Culture - 비난 없는 문화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시스템 중심 사고
> 2. **가치**: 심리적 안전, 솔직한 보고
> 3. **융합:** Postmortem, SRE, Psychological Safety

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Blameless Culture는 개인이 아닌 시스템을 개선합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Blameless Culture                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              비난 없는 문화                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Blameful:                                            │   │ │
│   │   │  "Who broke the production?"                         │   │ │
│   │   │  → 개인 비난                                          │   │ │
│   │   │  → 정보 은폐                                          │   │ │
│   │   │  → 두려움                                             │   │ │
│   │   │                                                      │   │ │
│   │   │  Blameless:                                           │   │ │
│   │   │  "What allowed this to happen?"                      │   │ │
│   │   │  → 시스템 개선                                        │   │ │
│   │   │  → 솔직한 공유                                        │   │ │
│   │   │  → 학습                                               │   │ │
│   │   │                                                      │   │ │
│   │   │  원칙:                                                │   │ │
│   │   │  1. 인간은 실수한다                                    │   │ │
│   │   │  2. 시스템이 실수를 방지해야                            │   │ │
│   │   │  3. 질문은 "왜"가 아니라 "어떻게"                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 비난 없는 문화는 심리적 안전을 보장합니다.

**📢 섹션 요약 비유:** Blameless = 안전망!
