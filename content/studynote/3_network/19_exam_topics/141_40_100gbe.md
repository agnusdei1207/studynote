# 40GbE / 100GbE / 400GbE / 800GbE 이더넷

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 초고속 이더넷, 다중 레인 구조
> 2. **가치**: 데이터센터, 클라우드 백본
> 3. **융합**: 래빗, QSFP, PAM4와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

40G/100G/400G/800G 이더넷은 다중 레인(Multi-lane) 구조로 초고속 전송을 달성하는 이더넷 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    초고속 이더넷 (40G~800G)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              표준 진화                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  속도   │ 레인 구성        │ 표준        │ 연도     │   │ │
│   │   │  ──────┼─────────────────┼────────────┼──────────│   │ │
│   │   │  40G   │ 4×10G           │ 802.3ba    │ 2010     │   │ │
│   │   │  100G  │ 4×25G / 10×10G  │ 802.3bj    │ 2010/14  │   │ │
│   │   │  200G  │ 4×50G (PAM4)    │ 802.3cd    │ 2019     │   │ │
│   │   │  400G  │ 8×50G (PAM4)    │ 802.3cm/ck │ 2017/20 │   │ │
│   │   │  800G  │ 8×100G (PAM4)   │ 802.3df    │ 2024     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │

│   │   │                                                      │   │ │

│   │   │  ┌─────┬─────┬─────┬─────┐                            │   │ │
│   │   │  │레인0│레인1│레인2│레인3│  각 25Gbps                 │   │ │

│   │                                                              │ │
```

> **해설**: 초고속 이더넷은 다중 레인 구조로 대역폭을 확장합니다.

**💡 비유**: 다중 레인은 4차선→8차선→16차선 도로로 확장하는 것입니다.

**📢 섹션 요약 비유:** 100G = 4×25G, 400G = 8×50G PAM4!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **레인** | 전송 채널 | 직렬화 | 차선 |
| **PAM4** | 변조 | 4레벨 | 멀티레벨 |
| **FEC** | 오류 정정 | RS/Reed-Solomon | 안전망 |

**비교표**

```
+------------+----------+----------+
| 항목       | 40/100G  | 400G     |
+------------+----------+----------+
| 레인       | 4/10     | 8        |
| 변조       | NRZ      | PAM4     |
| FEC        | Baseline | RS-FEC   |
| 전력       | 중간     | 높음     |
+------------+----------+----------+
```

**핵심 알고리즸**: PAM4, RS-FEC, Auto-Negotiation

**📢 섹션 요약 비유:** PAM4 = 2배 효율!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 커넥터별 속도**

| 커넥터 | 레인 | 속도 |
|:---|:---:|:---:|
| **QSFP28** | 4 | 100G |
| **QSFP56** | 8 | 200/400G |
| **QSFP-DD** | 8 | 400/800G |

**📢 섹션 요약 비유:** QSFP-DD = 800G!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데이터센터** → 100G/400G
**시나리오 2: 백본** → 400G/800G
**시나리오 3: 서버** → 25G/100G

**도입 체크리스트**: 대역폭, 거리, 전력, 비용

**📢 섹션 요약 비유:** DC는 400G 시대!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 대역폭 10배 향상

**미래 전망:** 1.6T 이더넷 개발

**참고 표준:** IEEE 802.3ba/bj/cm/ck/df

**📢 섹션 요약 비유:** 이더넷은 끝없이 진화!
