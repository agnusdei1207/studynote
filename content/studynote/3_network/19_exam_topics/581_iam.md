# IAM (Identity and Access Management) - 식별 및 접근 관리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 사용자/권한 통합 관리
> 2. **가치**: 보안, 컴플라이언스
> 3. **융합:** RBAC, MFA, SSO, Audit

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IAM은 사용자 식별과 접근 권한을 관리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IAM                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IAM 구성요소                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Identity Management:                                │   │ │
│   │   │  - 사용자 계정 생성/수정/삭제                         │   │ │
│   │   │  - 그룹 관리                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  Access Management:                                  │   │ │
│   │   │  - 권한 부여 (RBAC/ABAC)                             │   │ │
│   │   │  - 정책 관리                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  Authentication:                                     │   │ │
│   │   │  - MFA, SSO, Password Policy                         │   │ │
│   │   │                                                      │   │ │
│   │   │  Audit:                                              │   │ │
│   │   │  - 로그, 모니터링                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IAM은 보안과 컴플라이언스의 핵심입니다.

**📢 섹션 요약 비유:** IAM = 인사부 + 경비실!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

|:---|:---|
