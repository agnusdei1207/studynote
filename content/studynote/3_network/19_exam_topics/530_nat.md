# NAT (Network Address Translation) - 네트워크 주소 변환

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IP 주소 변환
> 2. **가치**: 공인 IP 절약, 보안
> 3. **융합:** Private IP, PAT, Firewall

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NAT는 사설 IP를 공인 IP로 변환합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NAT                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NAT 동작                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  내부망 (사설 IP)          NAT           인터넷      │   │ │
│   │   │                                                      │   │ │
│   │   │  192.168.1.10          ┌────┐         203.0.113.1   │   │ │
│   │   │       │                │    │              │         │   │ │
│   │   │       │ ── Src: 192.168│NAT │── Src: 203──→         │   │ │
│   │   │       │                │    │                        │   │ │
│   │   │       │ ←── Dst: 203   │    │←── Dst: 192.168 ──    │   │ │
│   │   │       │                │    │              │         │   │ │
│   │   │       │                └────┘                        │   │ │
│   │   │                                                      │   │ │
│   │   │  NAT 테이블로 매핑 관리                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: NAT는 내부 네트워크를 외부에서 숨깁니다.

**📢 섹션 요약 비유:** NAT = 우편 배달원!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 유형 | 설명 |
|:---|:---|
| **Static NAT** | 1:1 매핑 |
| **Dynamic NAT** | 풀에서 할당 |
| **PAT/NAPT** | 포트 기반 다중 매핑 |
| **NAT64** | IPv6 ↔ IPv4 |

**핵심 알고리즸:** Src IP:Port → NAT Table → 변환

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**NAT 유형 비교**

| 유형 | 매핑 | 용도 |
|:---|:---|:---|
| **Static** | 1:1 | 서버 공개 |
| **Dynamic** | N:M | 일반 사용 |
| **PAT** | N:1 | 가정/소호 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 가정 → PAT
**시나리오 2:** 서버 → Static NAT
**시나리오 3:** 기업 → Dynamic + PAT
