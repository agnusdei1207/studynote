# TACACS+ - 터미널 접근 제어 시스템 플러스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: Cisco AAA 프로토콜
> 2. **가치**: 관리자 접근 제어
> 3. **융합:** TCP 49, 전체 암호화

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

TACACS+ (Terminal Access Controller Access-Control System Plus)는 AAA 프로토콜입니다.


> **해설**: TACACS+는 네트워크 장비 관리에 최적화되어 있습니다.

**📢 섹션 요약 비유:** TACACS+ = 장비 관리자 신분증!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Authentication** | 인증 |
| **Authorization** | 인가 |
| **Accounting** | 계정 |
| **Session** | 세션 |
| **Encryption** | 전체 암호화 |

**핵심 알고리듬:** TCP 연결 → 인증 → 인가 → 계정 기록

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**TACACS+ vs RADIUS**

| 항목 | TACACS+ | RADIUS |
|:---|:---|:---|
| **전송** | TCP | UDP |
| **암호화** | 전체 | 비밀번호만 |
| **AAA** | 분리 | 통합 |
| **용도** | 장비 관리 | 사용자 인증 |
