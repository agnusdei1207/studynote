# 저밀도 파장 분할 다중화 (CWDM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 넓은 파장 간격(20nm)의 저비용 WDM
> 2. **가치**: 메트로망, 접속망에 경제적
> 3. **융합**: FTTH, 기업망, 데이터센터와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CWDM(Coarse Wavelength Division Multiplexing)은 20nm 간격으로 파장을 분할하여 저비용으로 구현한 WDM 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CWDM (저밀도 파장 분할 다중화)                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CWDM 스펙트럼                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  파장 할당 (ITU-T G.694.2):                           │   │ │
│   │   │                                                      │   │ │
│   │   │  파장 (nm)                                             │   │ │
│   │   │  1610 ─┬─────────────────────────────────────────┐   │   │ │
│   │   │  1590 ─┤ λ18                                      │   │   │ │
│   │   │  1570 ─┤ λ17                                      │   │   │ │
│   │   │  ...   │ ...                                       │   │   │ │
│   │   │  1310 ─┤ λ1                                        │   │   │ │
│   │   │  1270 ─┴─────────────────────────────────────────┘   │   │ │
│   │   │         │←── 20nm 간격 ──→│                         │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 1270nm ~ 1610nm (18파장)                          │   │ │
│   │   │  • 20nm 간격 (DWDM 대비 넓음)                         │   │ │
│   │   │  • 비냉각 레이저 사용 가능                             │   │ │
│   │   │  • EDFA 없이 ~80km 전송                               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CWDM vs DWDM 비교                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  파장 스펙트럼 비교:                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  CWDM:                                                 │   │ │
│   │   │  ├────┼────┼────┼────┼────┤                         │   │ │
│   │   │  │λ1  │λ2  │λ3  │λ4  │λ5  │  20nm 간격              │   │ │
│   │   │  └────┴────┴────┴────┴────┘                         │   │ │
│   │   │                                                      │   │ │
│   │   │  DWDM:                                                 │   │ │
│   │   │  ├──┼──┼──┼──┼──┼──┼──┼──┼──┤                       │   │ │
│   │   │  │λ │λ │λ │λ │λ │λ │λ │λ │λ │  0.8nm 간격           │   │ │
│   │   │  └──┴──┴──┴──┴──┴──┴──┴──┴──┘                       │   │ │
│   │   │                                                      │   │ │
│   │   │  CWDM 장점:                                           │   │ │
│   │   │  • 저비용 (비냉각 레이저)                              │   │ │
│   │   │  • 단순한 구조                                        │   │ │
│   │   │  • 저전력                                             │   │ │
│   │   │  • 메트로망에 적합                                     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: CWDM은 비용 민감한 메트로망에 최적입니다.

**💡 비유**: CWDM은 넓은 차선의 일반 도로입니다.

**📢 섹션 요약 비유**: CWDM = 저비용 WDM!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **비냉각 레이저** | 광원 | 온도 무보정 | LED |
| **필터** | 파장 분리 | 박막 | 프리즘 |
| **광섬유** | 전송 | SMF | 도로 |

**비교표**

```
+------------+----------+----------+
| 항목       | CWDM     | DWDM     |
+------------+----------+----------+
| 파장 수    | 18       | 40-160   |
| 간격       | 20nm     | 0.8nm    |
| 레이저     | 비냉각   | 냉각     |
| 증폭기     | 없음     | EDFA     |
| 거리       | ~80km    | 수천km   |
| 비용       | 낮음     | 높음     |
+------------+----------+----------+
```

**핵심 알고리즸**: 비냉각 레이저 → MUX → 광섬유 → DEMUX → 수신

**📢 섹션 요약 비유**: 냉각 없이 = 저비용!

| **접속망** | O | X |

**📢 섹션 요약 비유**: 메트로망은 CWDM!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 메트로망** → CWDM
**시나리오 2: 기업 접속** → CWDM
**시나리오 3: FTTH** → CWDM/PON

**도입 체크리스트**: 거리 <80km, 비용 중시, 채널 수 적음

**📢 섹션 요약 비유**: 단거리 저비용은 CWDM!

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: DWDM 대비 50% 비용 절감

**미래 전망**: CWDM/DWDM 혼합 시스템

**참고 표준**: ITU-T G.694.2, G.695

**📢 섹션 요약 비유**: CWDM은 메트로망의 경제적 선택!
