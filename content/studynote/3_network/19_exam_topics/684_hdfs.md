# HDFS (Hadoop Distributed File System) - HDFS

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 분산 파일 시스템
> 2. **가치**: 대용량 저장, 내결함성
> 3. **융합:** Hadoop, Block Storage, Replication

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

HDFS는 대용량 파일을 분산 저장합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    HDFS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              HDFS 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │              NameNode                         │  │   │ │
│   │   │  │         (Metadata Management)                 │  │   │ │
│   │   │  │  - File → Block 매핑                          │  │   │ │
│   │   │  │  - DataNode 상태                              │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                        │                             │   │ │
│   │   │           ┌────────────┼────────────┐                │   │ │
│   │   │           ↓            ↓            ↓                │   │ │
│   │   │   ┌───────────┐ ┌───────────┐ ┌───────────┐          │   │ │
│   │   │   │ DataNode  │ │ DataNode  │ │ DataNode  │          │   │ │
│   │   │   │ Block 1   │ │ Block 2   │ │ Block 3   │          │   │ │
│   │   │   │ Block 4   │ │ Block 1   │ │ Block 2   │          │   │ │
│   │   │   │ (Replica) │ │ (Replica) │ │ (Replica) │          │   │ │
│   │   │   └───────────┘ └───────────┘ └───────────┘          │   │ │
│   │   │                                                      │   │ │
│   │   │  Block Size: 128MB (default)                         │   │ │
│   │   │  Replication Factor: 3 (default)                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: HDFS는 블록 단위로 복제 저장합니다.

**📢 섹션 요약 비유:** HDFS = 분산 창고!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NameNode** | 메타데이터 관리 |
| **DataNode** | 데이터 저장 |
| **Block** | 저장 단위 |
| **Replication** | 복제 |
| **Rack Awareness** | 랙 인식 |

**핵심 알고리즸:** 파일 분할 → 블록 복제 → 분산 저장

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**HDFS vs 일반 파일 시스템**

| 항목 | HDFS | 일반 FS |
|:---|:---|:---|
| **블록 크기** | 128MB | 4KB |
| **복제** | 기본 | 없음 |
| **순차 읽기** | 최적화 | 무작위 |
