# SMS - 단문 메시지 서비스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이동통신망 텍스트 메시지
> 2. **가치**: 보편성, 신뢰성
> 3. **융합**: 2G/3G/4G/5G, RCS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SMS(Short Message Service)는 이동통신망에서 텍스트 메시지를 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SMS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SMS 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [발신 단말]                                         │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ SMS 메시지 (160자)                              │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           MSC (Mobile Switching Center)     │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • 메시지 저장                                │    │   │ │
│   │   │  │  • 라우팅                                     │    │   │ │
│   │   │  │  • 전달                                       │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [SMSC]                                             │   │ │
│   │   │  (Short Message Service Center)                     │   │ │
│   │   │       │                                              │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [수신 단말]                                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SMS 특징                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │

│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 내용       │ 텍스트   │ 미디어   │               │   │ │
│   │   │  │ 크기       │ 160자    │ 300KB    │               │   │ │
│   │   │  | 비용       │ 낮음     │ 중간     │               │   │ │
│   │   │  │ 전송       │ 시그널링 │ 데이터   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
```

> **해설**: SMS는 시그널링 채널로 160자 텍스트를 전송합니다.

**💡 비유**: SMS는 엽서입니다. 짧고 간단하게 전합니다.

**📢 섹션 요약 비유:** SMS = 160자!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **SMSC** | 메시지 센터 |
| **MSC** | 교환기 |
| **HLR** | 위치 등록 |
| **SRI** | 라우팅 정보 |

**비교표: SMS 전송 방식**

```
+------------+----------+----------+
| 네트워크   │ SMS      │ 비고     │
+------------+----------+----------+
| 2G         │ 지원     │ 원조     │
| 3G         │ 지원     │ 확장     │
| 4G         │ SMS over IMS │ RCS │
| 5G         │ SMS over NAS │ IMS │
+------------+----------+----------+
```

**핵심 알고리즸:** 저장 후 전달

**📢 섹션 요약 비유:** SMSC = 우체국!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 메시징**

| 항목 | SMS | RCS | OTT |
|:---|:---|:---|:---|

**📢 섹션 요약 비유:** 보편성 = SMS!
