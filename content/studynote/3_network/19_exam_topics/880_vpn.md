# VPN - 가상 사설망

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 암호화된 터널
> 2. **가치**: 안전한 원격 접속
> 3. **융합:** IPsec, OpenVPN, WireGuard

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VPN (Virtual Private Network)은 암호화된 터널을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VPN                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VPN 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Remote                    VPN                    Corp │   │ │
│   │   │  Client                   Tunnel                Network │   │ │
│   │   │  ┌─────┐  ═══════════════════════════════════  ┌─────┐│   │ │
│   │   │  │     │  Encrypted Tunnel (암호화된 터널)     │     ││   │ │
│   │   │  │     │══════════════════════════════════════│     ││   │ │
│   │   │  └─────┘                                        └─────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  VPN 프로토콜:                                          │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ IPsec: L3, 표준, 복잡                             ││   │ │
│   │   │  │ OpenVPN: SSL/TLS 기반, 유연                        ││   │ │
│   │   │  │ WireGuard: 최신, 가볍, 빠름                        ││   │ │
│   │   │  │ L2TP: L2 터널링                                   ││   │ │
│   │   │  │ PPTP: 구식, 취약                                  ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  VPN 유형:                                              │   │ │
│   │   │  - Site-to-Site: 네트워크 간                           │   │ │
│   │   │  - Remote Access: 개별 사용자                          │   │ │
│   │   │  - Client-to-Site: 재택근무                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: VPN은 공중망에서 사설망 보안을 제공합니다.

**📢 섹션 요약 비유:** VPN = 비밀 터널!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Tunnel** | 터널 |
| **Encryption** | 암호화 |
| **Authentication** | 인증 |
| **Gateway** | 게이트웨이 |
| **Client** | 클라이언트 |

**핵심 알고리즸:** 패킷 → 캡슐화 → 암호화 → 전송

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**VPN 프로토콜 비교**

| 프로토콜 | 계층 | 속도 | 보안 |
|:---|:---|:---|:---|
| **IPsec** | L3 | 빠름 | 높음 |
| **OpenVPN** | L7 | 중간 | 높음 |
| **WireGuard** | L3 | 매우 빠름 | 높음 |
| **PPTP** | L2 | 빠름 | 낮음 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 재택근무 → WireGuard
**시나리오 2:** 기업 연결 → IPsec
**시나리오 3:** 개인 → OpenVPN
