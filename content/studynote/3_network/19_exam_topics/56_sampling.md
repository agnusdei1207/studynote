# 표본화 (Sampling)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 연속 아날로그 신호를 이산 샘플로 변환
> 2. **가치**: 디지털 신호 처리의 첫 단계
> 3. **융합**: PCM, ADC, Nyquist 정리와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

표본화(Sampling)는 연속적인 아날로그 신호를 일정 간격으로 측정하여 이산적인 샘플로 변환하는 과정입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    표본화 (Sampling)                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              표본화 과정                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  아날로그 신호:                                         │   │ │
│   │   │         ╱╲      ╱╲                                    │   │ │
│   │   │        ╱  ╲    ╱  ╲                                   │   │ │
│   │   │       ╱    ╲  ╱    ╲                                  │   │ │
│   │   │  ───╱──────╲╱──────╲───                              │   │ │
│   │   │     │  │  │  │  │  │                                 │   │ │
│   │   │     ↓  ↓  ↓  ↓  ↓  ↓  ← 샘플링 주기 Ts               │   │ │
│   │   │     ●  ●  ●  ●  ●  ●  ← 샘플 값                       │   │ │
│   │   │                                                      │   │ │
│   │   │  샘플링 주파수: fs = 1/Ts                             │   │ │
│   │   │  샘플 개수: 무한대 (실제로는 유한)                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Nyquist 표본화 정리                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  정리:                                                  │   │ │
│   │   │  fs ≥ 2·fmax                                           │   │ │
│   │   │  (샘플링 주파수 ≥ 신호 최대 주파수의 2배)               │   │ │
│   │   │                                                      │   │ │
│   │   │  Nyquist 속도: fN = 2·fmax                            │   │ │
│   │   │  Nyquist 주기: TN = 1/(2·fmax)                        │   │ │
│   │   │                                                      │   │ │
│   │   │  예: 음성 신호 (4kHz)                                  │   │ │
│   │   │  fs ≥ 8kHz (전화는 8kHz 사용)                         │   │ │
│   │   │                                                      │   │ │
│   │   │  예: 오디오 CD (20kHz)                                │   │ │
│   │   │  fs ≥ 40kHz (CD는 44.1kHz 사용)                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: Nyquist 정리를 만족하면 원 신호를 완벽히 복원할 수 있습니다.

**💡 비유**: 표본화는 영화의 각 프레임을 찍는 것과 같습니다.

**📢 섹션 요약 비유**: fs ≥ 2fmax!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **fs** | 샘플링 주파수 | Hz | 속도 |
| **Ts** | 샘플링 주기 | 초 | 간격 |
| **fmax** | 최대 주파수 | Hz | 한계 |
| **Nyquist** | 기준 | 2fmax | 임계 |

**표본화 방식**

```
+------------+----------+----------+
| 방식       | 특징     | 응용     |
+------------+----------+----------+
| 이상 샘플링| 순간값   | 이론     |
| 자연 샘플링| 펄스 폭  | 실제     |
| 평균 샘플링| 적분값   | ADC      |
+------------+----------+----------+
```

**핵심 알고리즘**: 아날로그 → 샘플링 → PAM → 양자화

**📢 섹션 요약 비유**: CD는 44.1kHz!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 응용별 샘플링 주파수**

| 응용 | 대역폭 | 샘플링 |
|:---|:---:|:---:|
| **전화** | 4kHz | 8kHz |
| **CD** | 20kHz | 44.1kHz |
| **DVD** | 20kHz | 48kHz |
| **Hi-Res** | 96kHz | 192kHz |

**📢 섹션 요약 비유**: 전화는 8kHz, CD는 44.1kHz!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 전화망** → 8kHz 샘플링
**시나리오 2: 오디오** → 44.1kHz 이상
**시나리오 3: 영상** → 2×해상도

**도입 체크리스트**: 대역폭, 저장 용량, 처리 능력

**📢 섹션 요약 비유**: 샘플링 높을수록 품질 ↑ 용량 ↑!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 아날로그→디지털 변환 기반

**미래 전망**: 초고해상도 오디오/비디오

**참고 표준**: ITU-T G.711, CD Red Book

**📢 섹션 요약 비유**: 표본화는 디지털의 시작!
