# Firewall - 방화벽

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 접근 제어
> 2. **가치**: 보안 경계
> 3. **융합:** Packet Filter, Stateful, Next-Gen

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Firewall은 네트워크 트래픽을 제어합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Firewall                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              방화벽 유형                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Packet Filtering:                                     │   │ │
│   │   │  - IP/Port 기반 필터링                                  │   │ │
│   │   │  - 상태 비저장 (Stateless)                              │   │ │
│   │   │  - 규칙 기반                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  Stateful Inspection:                                   │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 연결 상태 추적 (Connection State)                ││   │ │
│   │   │  │ SYN, SYN-ACK, ACK 상태 관리                       ││   │ │
│   │   │  │ ESTABLISHED 연결만 응답 허용                      ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  Next-Generation Firewall (NGFW):                      │   │ │
│   │   │  - DPI (Deep Packet Inspection)                       │   │ │
│   │   │  - 애플리케이션 인식                                    │   │ │
│   │   │  - IPS/IDS 통합                                        │   │ │
│   │   │  - 사용자 인증                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  배치 위치:                                              │   │ │
│   │   │  - 네트워크 경계                                        │   │ │
│   │   │  - 호스트 기반 (Host-based)                             │   │ │
│   │   │  - 클라우드 (WAF, Security Groups)                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 방화벽은 네트워크 보안의 첫 번째 방어선입니다.

**📢 섹션 요약 비유:** Firewall = 출입구 검문소!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Rule** | 규칙 |
| **ACL** | 접근 제어 목록 |
| **State** | 상태 |
| **Zone** | 영역 |
| **Policy** | 정책 |

**핵심 알고리즸:** 패킷 → 규칙 매칭 → 허용/차단

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**방화벽 유형 비교**

| 유형 | 계층 | 기능 |
|:---|:---|:---|
| **Packet Filter** | L3/L4 | 기본 |
| **Stateful** | L3/L4 | 상태 추적 |
| **NGFW** | L3-L7 | 통합 보안 |
| **WAF** | L7 | 웹 전용 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 네트워크 경계 → NGFW
**시나리오 2:** 웹 서버 → WAF
**시나리오 3:** 서버 → Host Firewall
