# BGP - 경계 게이트웨이 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 경로 벡터 라우팅
> 2. **가치**: 인터넷 백본 프로토콜
> 3. **융합:** AS, Peering, Policy

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

BGP (Border Gateway Protocol)는 자율 시스템 간 라우팅 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BGP                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              BGP互联                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  AS 간 연결:                                            │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │         ┌─────────┐         ┌─────────┐        ││   │ │
│   │   │  │         │  AS100  │─────────│  AS200  │        ││   │ │
│   │   │  │         │ (ISP A) │  eBGP   │ (ISP B) │        ││   │ │
│   │   │  │         └────┬────┘         └────┬────┘        ││   │ │
│   │   │  │              │                   │              ││   │ │
│   │   │  │         ┌────┴────┐         ┌────┴────┐        ││   │ │
│   │   │  │         │  AS300  │─────────│  AS400  │        ││   │ │
│   │   │  │         │ (ISP C) │  eBGP   │ (ISP D) │        ││   │ │
│   │   │  │         └─────────┘         └─────────┘        ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  BGP 유형:                                              │   │ │
│   │   │  - eBGP: 외부 BGP (AS 간)                               │   │ │
│   │   │  - iBGP: 내부 BGP (AS 내)                               │   │ │
│   │   │                                                      │   │ │
│   │   │  경로 선택 속성:                                          │   │ │
│   │   │  - AS_PATH: AS 경로                                    │   │ │
│   │   │  - NEXT_HOP: 다음 홉                                    │   │ │
│   │   │  - LOCAL_PREF: 로컬 선호도                               │   │ │
│   │   │  - MED: 다중 출구 판별자                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: BGP는 인터넷의 라우팅 기반입니다.

**📢 섹션 요약 비유:** BGP = 인터넷의 우편 배달 시스템!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **AS** | 자율 시스템 |
| **Peer** | 피어링 |
| **Attribute** | 경로 속성 |
| **Policy** | 정책 |
| **Community** | 커뮤니티 |

**핵심 알고리듬:** 경로 벡터 + 정책 기반 선택

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**IGP vs EGP**

| 항목 | IGP (OSPF) | EGP (BGP) |
|:---|:---|:---|
| **범위** | AS 내부 | AS 간 |
| **정책** | 없음 | 있음 |
| **수렴** | 빠름 | 느림 |
| **규모** | 제한적 | 인터넷 전체 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** ISP → BGP 피어링
**시나리오 2:** 멀티홈 → BGP 다중 경로
**시나리오 3:** 엔터프라이즈 → BGP + IGP
