# ARP 스푸핑 - ARP Spoofing

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: ARP 캐시 조작 공격
> 2. **가치**: 중간자 공격, 스니핑
> 3. **융합**: DAI, 정적 ARP, 포트 보안

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ARP 스푸핑은 ARP 캐시를 조작해 트래픽을 가로챕니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ARP 스푸핑                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              정상 ARP                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [PC] ◀────────────────────▶ [Gateway]              │   │ │
│   │   │  IP: 192.168.1.10          IP: 192.168.1.1          │   │ │
│   │   │  MAC: AA:AA               MAC: BB:BB                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ARP 스푸핑 공격                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [공격자] "나가 Gateway야" (거짓 ARP)                │   │ │
│   │   │     │                                                │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  [PC] ──── 트래픽 ────▶ [공격자] ────▶ [Gateway]    │   │ │
│   │   │                          │                           │   │ │
│   │   │                          ▼                           │   │ │
│   │   │                     [스니핑]                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: ARP는 신원 검증 없이 MAC-IP 매핑을 수락합니다.

**💡 비유**: ARP 스푸핑은 우편 배달부 사칭입니다. 편지를 가로챕니다.

**📢 섹션 요약 비유:** ARP 스푸핑 = 중간자 공격!
