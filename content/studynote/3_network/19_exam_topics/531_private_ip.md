# Private IP - 사설 IP 주소

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 내부망 전용 IP 주소
> 2. **가치**: 공인 IP 절약, 보안
> 3. **융합:** NAT, RFC 1918, VPC

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Private IP는 인터넷에서 라우팅되지 않는 주소입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Private IP                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RFC 1918 사설 IP 대역                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Class A: 10.0.0.0/8                                 │   │ │
│   │   │           10.0.0.0 - 10.255.255.255 (16M개)          │   │ │
│   │   │                                                      │   │ │
│   │   │  Class B: 172.16.0.0/12                              │   │ │
│   │   │           172.16.0.0 - 172.31.255.255 (1M개)         │   │ │
│   │   │                                                      │   │ │
│   │   │  Class C: 192.168.0.0/16                             │   │ │
│   │   │           192.168.0.0 - 192.168.255.255 (65K개)      │   │ │
│   │   │                                                      │   │ │
│   │   │  → 인터넷 라우터에서 폐기됨                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 사설 IP는 NAT와 함께 사용되어 인터넷 접속을 가능케 합니다.

**📢 섹션 요약 비유:** Private IP = 내부 전화번호!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 대역 | 크기 | 용도 |
|:---|:---|:---|
| **10.x.x.x** | 16M | 대형 기업 |
| **172.16-31.x.x** | 1M | 중형 기업 |
| **192.168.x.x** | 65K | 소호/가정 |

**핵심 알고리즸:** 사설 IP → NAT → 공인 IP

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**사설 IP vs 공인 IP**

| 항목 | 사설 | 공인 |
|:---|:---|:---|
| **라우팅** | 안됨 | 됨 |
| **유일성** | 내부만 | 전역 |
| **비용** | 무료 | 유료 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 가정 → 192.168.x.x
**시나리오 2:** 기업 → 10.x.x.x
**시나리오 3:** 클라우드 VPC → 사용자 지정
