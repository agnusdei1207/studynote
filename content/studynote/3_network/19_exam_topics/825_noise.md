# Noise - 잡음

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 원치 않는 신호
> 2. **가치**: 통신 품질 저하
> 3. **융합:** SNR, Thermal, Interference

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Noise는 원하지 않는 신호 간섭입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Noise                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              잡음 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Thermal Noise (열 잡음):                             │   │ │
│   │   │  전자의 무작위 운동                                    │   │ │
│   │   │  N = k × T × B (k: 볼츠만 상수)                       │   │ │
│   │   │  항상 존재, 제거 불가                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  Intermodulation Noise:                               │   │ │
│   │   │  여러 신호 혼합으로 발생                                │   │ │
│   │   │                                                      │   │ │
│   │   │  Crosstalk (누화):                                    │   │ │
│   │   │  인접 회로 간 간섭                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  Impulse Noise (충격 잡음):                           │   │ │
│   │   │  번개, 전원 스위치 등                                  │   │ │
│   │   │  짧고 강한 간섭                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  SNR (Signal-to-Noise Ratio):                         │   │ │
│   │   │  SNR = 신호 전력 / 잡음 전력                           │   │ │
│   │   │  높을수록 좋음                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 잡음은 통신 시스템 성능의 주요 제약입니다.

**📢 섹션 요약 비유:** Noise = 배경 소음!
