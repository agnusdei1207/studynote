# Container - 컨테이너

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 격리된 실행 환경

> 3. **융합:** Docker, Kubernetes, Microservices

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Container는 격리된 실행 환경을 제공합니다.

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Image** | 이미지 |
| **Runtime** | 런타임 |
| **Namespace** | 네임스페이스 |
| **Cgroups** | 컨트롤 그룹 |
| **Union FS** | 유니온 파일시스템 |

**핵심 알고리즸:** 이미지 빌드 → 실행 → 격리

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**격리 기술**
