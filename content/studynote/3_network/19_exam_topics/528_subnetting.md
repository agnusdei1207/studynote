# Subnetting - 서브네팅

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 분할
> 2. **가치**: IP 효율, 보안, 관리
> 3. **융합:** Subnet Mask, CIDR, VLSM

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Subnetting은 네트워크를 더 작은 단위로 분할합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Subnetting                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              서브네팅 예시                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  원본: 192.168.1.0/24 (256개)                        │   │ │
│   │   │                                                      │   │ │
│   │   │  서브네팅 (/26):                                     │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ 192.168.1.0/26   │ 호스트: .1-.62    (62개) │    │   │ │
│   │   │  │ 192.168.1.64/26  │ 호스트: .65-.126  (62개) │    │   │ │
│   │   │  │ 192.168.1.128/26 │ 호스트: .129-.190 (62개) │    │   │ │
│   │   │  │ 192.168.1.192/26 │ 호스트: .193-.254 (62개) │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 서브네팅은 IP 주소를 효율적으로 활용합니다.

**📢 섹션 요약 비유:** Subnetting = 땅 분할!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Subnet Mask** | 네트워크/호스트 구분 |
| **Network ID** | 네트워크 식별 |
| **Host ID** | 호스트 식별 |
| **Broadcast** | 마지막 주소 |

**핵심 알고리즸:** IP & Mask = Network ID

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**서브넷 크기 계산**

| CIDR | 마스크 | 호스트 |
|:---|:---|:---|
| **/24** | 255.255.255.0 | 254 |
| **/25** | 255.255.255.128 | 126 |
| **/26** | 255.255.255.192 | 62 |
| **/27** | 255.255.255.224 | 30 |
| **/28** | 255.255.255.240 | 14 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
