# 1001-1120. 네트워크 성능 평가 및 심화 기술 마스터 요약

> **핵심 인사이트**: 미래의 네트워크는 '단순 전송'을 넘어 '지능형 서비스'로 진화하고 있다. 데이터센터의 초저지연 기술(RDMA), 무선망의 한계를 깨는 6G/위성 통신, 그리고 하드웨어를 소프트웨어로 제어하는 클라우드 네이티브 기술의 융합이 디지털 전환의 핵심 동력이다.

---

## Ⅰ. 네트워크 성능 및 신뢰성 평가지표
1. **QoS vs QoE**: 기술적 전송 품질(지연, 지터 등)과 사용자가 체감하는 주관적 품질(만족도)의 차이.
2. **Throughput (처리량)**: 단위 시간당 전송되는 유효 데이터 양. (Goodput은 오버헤드 제외 순수 데이터)
3. **Availability (가용성)**: MTBF(평균 무고장 시간) / (MTBF + MTTR(평균 수리 시간)). '99.999% (Five Nines)'가 목표.
4. **Erlang (얼랑)**: 통신 트래픽의 부하 단위. 1시간 동안 채널이 점유된 비율.

---

## Ⅱ. 차세대 무선 및 입체 통신
1. **mmWave / THz**: 5G/6G의 초고속 전송을 위한 고주파 대역. (강한 감쇠와 직진성 극복이 과제)
2. **NTN (Non-Terrestrial Network)**: 저궤도 위성(LEO)과 지상망을 통합하여 전 지구적 커버리지 확보.
3. **RIS (지능형 반사 표면)**: 전파 음영 지역을 해소하기 위한 소프트웨어 제어형 반사판.
4. **V2X (C-V2X / WAVE)**: 자율주행을 위한 차량 간(V2V), 차량-인프라 간(V2I) 저지연 통신.

---

## Ⅲ. 클라우드 및 하드웨어 가속
1. **RDMA / RoCE**: CPU를 우회하여 서버 메모리 간 직접 데이터를 복사하는 초고속 전송 기술.
2. **SmartNIC / DPU**: 네트워크 및 보안 처리를 전담하여 서버 CPU의 부하를 덜어주는 전용 프로세서.
3. **Service Mesh (Istio)**: 사이드카 프록시를 통해 마이크로서비스 간 통신을 제어/보안/관측하는 계층.
4. **eBPF**: 리눅스 커널 소스를 수정하지 않고도 동적으로 기능을 추가하여 고성능 네트워킹 및 보안 구현.

---

## Ⅳ. 미래 보안 및 지능형 운영
1. **Zero Trust**: 경계 중심 보안을 버리고 '모든 접속을 매번 검증'하는 차세대 보안 모델.
2. **QKD (양자 키 분배)**: 양자 역학 원리를 이용해 해킹이 원천적으로 불가능한 암호키를 공유하는 기술.
3. **AIOps / ADN**: AI가 트래픽을 분석하여 스스로 장애를 복구하고 경로를 최적화하는 자율 주행 네트워크.
4. **Digital Twin Network**: 실제 망과 똑같은 가상 모델을 구축하여 장애를 예측하고 변화를 시뮬레이션.

---

## Ⅴ. 개념 맵 및 요약

```ascii
[네트워크 기술의 미래 지향점]

  Intelligent (지능형) <───  AIOps, ADN, Digital Twin
      │
  Ubiquitous (초공간)  <───  6G, LEO Satellite, NTN
      │
  Ultra-Fast (초고속)  <───  THz, RDMA, DPU, 800G Ethernet
      │
  Trustworthy (고신뢰) <───  Zero Trust, QKD, RPKI
```

📢 **섹션 요약 비유**: **네트워크 성능 평가**는 건강검진(QoS)과 환자의 기분(QoE)을 모두 체크하는 것입니다. **차세대 통신**은 도시에만 있던 전기를 산꼭대기(위성)와 바닷속까지 연결하는 전력화 사업이며, **하드웨어 가속**은 사장님이 직접 하던 잡일을 유능한 비서(DPU)에게 맡겨 회사의 효율을 높이는 경영 혁신입니다. **미래 보안**은 성문을 굳게 닫는 대신, 건물 내 모든 방에 지문 인식기(Zero Trust)를 달고 위조 불가능한 신분증(양자 암호)을 만드는 철저한 보안 시스템입니다.
