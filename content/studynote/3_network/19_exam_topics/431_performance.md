# 네트워크 성능 지표 - Network Performance Metrics

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 성능 측정 기준
> 2. **가치**: SLA, 용량 계획, 문제 진단
> 3. **융합**: 지연 시간, 처리량, 패킷 손실

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

네트워크 성능 지표는 네트워크 품질을 측정합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    네트워크 성능 지표                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              주요 성능 지표                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ Bandwidth (대역폭)                          │    │   │ │
│   │   │  │ 최대 전송 용량 (Mbps/Gbps)                   │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ Throughput (처리량)                         │    │   │ │
│   │   │  │ 실제 전송 속도 (Mbps/Gbps)                   │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ Latency (지연 시간)                         │    │   │ │
│   │   │  │ 왕복 시간 (ms)                               │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ Packet Loss (패킷 손실)                     │    │   │ │
│   │   │  │ 손실률 (%)                                    │    │ │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              지표별 목표                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 지표       │ 일반     │ 실시간   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ Latency    │ <100ms   │ <20ms    │               │   │ │
│   │   │  │ Jitter     │ <30ms    │ <5ms     │               │   │ │
│   │   │  | Packet Loss│ <1%      │ <0.1%    │               │   │ │
│   │   │  │ Throughput │ 높을수록 │ 안정적   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 성능 지표는 용도에 따라 다른 목표를 가집니다.

**💡 비유**: 성능 지표는 자동차의 속도계, 연료계입니다.

**📢 섹션 요약 비유:** 성능 지표 = 네트워크 건강!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Bandwidth** | 최대 용량 |
| **Throughput** | 실제 전송량 |
| **Latency** | 왕복 지연 |
| **Goodput** | 유용한 데이터량 |

**비교표: 측정 도구**

```
+------------+----------+----------+
| 항목       │ 도구     │ 용도     │
+------------+----------+----------+
| Latency    │ Ping     │ 기본     │
| Throughput │ iPerf    │ 정밀     │
| 경로       │ Traceroute│ 진단    │
+------------+----------+----------+
```

**핵심 알고리즸:** 측정, 분석, 임계값

**📢 섹션 요약 비유:** iPerf = 성능 측정 표준!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 애플리케이션별 요구사항**

| 앱 | Latency | Jitter | Loss |
|:---|:---|:---|:---|
| **VoIP** │ <150ms │ <30ms | <1% |
| **Video** │ <200ms | <50ms | <0.1% |
| **Gaming** │ <50ms | <10ms | <0.5% |

**📢 섹션 요약 비유:** SLA = 성능 목표!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 일반 업무 → 기본 지표
