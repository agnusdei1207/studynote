# API Gateway - API 게이트웨이

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: API 진입점
> 2. **가치**: 통합 관리, 보안, 라우팅
> 3. **융합:** Microservices, Load Balancer, Authentication

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Routing** | 라우팅 |
| **Authentication** | 인증 |
| **Rate Limiting** | 속도 제한 |
| **Transformation** | 변환 |
| **Caching** | 캐싱 |

**핵심 알고리즸:** 요청 → 전처리 → 라우팅 → 후처리 → 응답
