# OAuth 2.0 - 오픈 인증 2.0

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 권한 위임 프레임워크
> 2. **가치**: 안전한 접근 권한 위임
> 3. **융합:** OpenID Connect, JWT, API Security

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OAuth 2.0은 권한을 위임합니다.


**📢 섹션 요약 비유:** OAuth 2.0 = 출입증!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Resource Owner** | 자원 소유자 |
| **Client** | 클라이언트 |
| **Authorization Server** | 인증 서버 |
| **Resource Server** | 자원 서버 |
| **Access Token** | 접근 토큰 |

**핵심 알고리즸:** 인증 → 권한 → 토큰 → 접근
