# Congestion - 혼잡

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 과부하
> 2. **가치**: 성능 저하 원인 파악
> 3. **융합:** TCP Congestion Control, AQM, QoS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Congestion은 네트워크 과부하 상태입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Congestion                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              혼잡 발생 및 제어                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  발생 원인:                                            │   │ │
│   │   │  - 대역폭 초과                                         │   │ │
│   │   │  - 버퍼 오버플로우                                     │   │ │
│   │   │  - 라우터 처리 한계                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  영향:                                                 │   │ │
│   │   │  - 패킷 손실                                          │   │ │
│   │   │  - 지연 증가                                          │   │ │
│   │   │  - 처리량 감소                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  TCP 혼잡 제어:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Slow Start: 1 → 2 → 4 → 8 → ...                ││   │ │
│   │   │  │ Congestion Avoidance: 선형 증가                  ││   │ │
│   │   │  │ Fast Retransmit: 3 ACK → 재전송                 ││   │ │
│   │   │  │ Fast Recovery: 손실 후 복구                      ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 혼잡 제어는 네트워크 안정성을 유지합니다.

**📢 섹션 요약 비유:** Congestion = 교통 체증!
