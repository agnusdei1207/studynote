# ECN (명시적 혼잡 알림) - Explicit Congrosion Notification

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 패킷 폐기 없이 혼잡 알림
> 2. **가치**: 지연 감소, 처리량 향상
> 3. **융합**: TCP, IP, DCTCP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ECN은 패킷을 폐기하지 않고 혼잡을 알립니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ECN 동작                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ECN 필드                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ ECT        │ CE       │ 의미     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 00         │ 0        │ 비ECN    │               │   │ │
│   │   │  │ 01/10      │ 0        │ ECN 지원 │               │   │ │
│   │   │  | XX         │ 1        │ 혼잡     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: ECN은 IP 헤더와 TCP 헤더를 사용합니다.

**📢 섹션 요약 비유:** ECN = 혼잡 신호등!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **ECT** | ECN-Capable Transport |
| **CE** | Congestion Experienced |
| **CWR** | Congestion Window Reduced |
| **ECE** | ECN-Echo |

**핵심 알고리즸:** 마킹, 신호

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**ECN vs Drop**

| 항목 | Drop | ECN |
|:---|:---|:---|
| **손실** | 있음 | 없음 |
| **지연** | 높음 | 낮음 |
| **호환** | 모두 | 지원 필요 |
