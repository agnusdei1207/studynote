# IPv6 Addressing - IPv6 주소 체계

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 128비트 계층적 주소
> 2. **가치**: 효율적 라우팅, 자동 설정
> 3. **융합:** SLAAC, EUI-64, Prefix

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IPv6 주소는 128비트로 구성됩니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IPv6 Addressing                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IPv6 주소 구조                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  2001:0db8:85a3:0000:0000:8a2e:0370:7334             │   │ │
│   │   │  ├───────────────┤├─────────────┤├──────────────┤   │   │ │
│   │   │   Prefix (48비트)  Subnet(16)   Interface ID(64)     │   │ │
│   │   │                                                      │   │ │
│   │   │  Prefix: ISP 할당                                    │   │ │
│   │   │  Subnet: 조직 내부 구분                               │   │ │
│   │   │  Interface ID: 호스트 식별 (EUI-64 또는 랜덤)          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IPv6는 Prefix, Subnet, Interface ID로 구성됩니다.

**📢 섹션 요약 비유:** IPv6 주소 = 대륙-국가-도시-번지!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 유형 | Prefix | 설명 |
|:---|:---|:---|
| **Unicast** | - | 1:1 |
| **Global** | 2000::/3 | 공인 |
| **Link-Local** | fe80::/10 | 링크 내 |
| **Unique Local** | fc00::/7 | 사설 |
| **Multicast** | ff00::/8 | 그룹 |
| **Anycast** | - | 가까운 1개 |

**핵심 알고리즸:** Prefix로 주소 유형 식별

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**주소 할당 방식**

| 방식 | 설명 |
|:---|:---|
| **수동** | 관리자 설정 |
| **SLAAC** | 라우터 RA 기반 |
| **DHCPv6** | 서버 할당 |
| **EUI-64** | MAC 기반 |
