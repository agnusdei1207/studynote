# Heartbeat - 하트비트

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 주기적 생존 신호
> 2. **가치**: 장애 탐지, 고가용성
> 3. **융합:** HA, Clustering, Load Balancer

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Heartbeat는 주기적으로 생존을 알립니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Heartbeat                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Heartbeat 동작                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  서버 A                    서버 B                    │   │ │
│   │   │     │                         │                      │   │ │
│   │   │     │ ─── Heartbeat ────────→ │                      │   │ │
│   │   │     │ ←── ACK ─────────────── │                      │   │ │
│   │   │     │                         │                      │   │ │
│   │   │  (1초마다 반복)                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  3회 연속 실패 → 장애 간주 → Failover               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Heartbeat 실패 시 장애로 간주하고 조치합니다.

**📢 섹션 요약 비유:** Heartbeat = 맥박 체크!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Interval** | 전송 주기 |
| **Timeout** | 응답 대기 |
| **Threshold** | 실패 임계값 |
| **Action** | Failover |

**핵심 알고리즸:** failures > threshold → failover
