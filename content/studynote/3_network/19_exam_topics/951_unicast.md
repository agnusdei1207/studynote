# Unicast - 유니캐스트

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 1:1 통신
> 2. **가치**: 개별 전달
> 3. **융합:** IP 주소, MAC, Point-to-Point

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Unicast는 하나의 송신자에서 하나의 수신자로 전달합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Unicast                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              유니캐스트 통신                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1:1 통신:                                               │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │    ┌─────────┐              ┌─────────┐         ││   │ │
│   │   │  │    │ Sender  │──────────────│Receiver │         ││   │ │
│   │   │  │    │ 192.168.│   Unicast    │ 192.168.│         ││   │ │
│   │   │  │    │  1.10   │──────────────│  1.20   │         ││   │ │
│   │   │  │    └─────────┘              └─────────┘         ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                  │   │ │
│   │   │  - 목적지: 단일 호스트                                   │   │ │
│   │   │  - 주소: 일반 IP 주소 (Class A/B/C)                      │   │ │
│   │   │  - MAC: 유니캐스트 MAC (첫 바이트 최하위 비트 0)            │   │ │
│   │   │                                                      │   │ │
│   │   │  예시:                                                  │   │ │
│   │   │  - 웹 브라우징                                          │   │ │
│   │   │  - 이메일 전송                                          │   │ │
│   │   │  - 파일 다운로드                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 유니캐스트는 가장 일반적인 통신 방식입니다.

**📢 섹션 요약 비유:** Unicast = 1:1 전화!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**통신 방식 비교**

| 방식 | 송신자 | 수신자 | 용도 |
|:---|:---|:---|:---|
| **Unicast** | 1 | 1 | 개별 전달 |
| **Broadcast** | 1 | 전체 | 네트워크 전체 |
| **Multicast** | 1 | 그룹 | 그룹 전달 |
| **Anycast** | 1 | 가장 가까운 1 | 부하 분산 |
