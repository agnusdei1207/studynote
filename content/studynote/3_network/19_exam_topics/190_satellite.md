# 위성 통신 (Satellite) - GEO/MEO/LEO

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 위성을 이용한 광역 통신
> 2. **가치**: 지상국 없는 지역 연결
> 3. **융합**: 방송, 인터넷, GPS와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

위성 통신은 지구 궤도의 위성을 이용하여 통신하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    위성 궤도별                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  궤도  │ 고도       │ 커버리지 │ 지연   │ 용도           │   │
│   │  ─────┼───────────┼─────────┼───────┼───────────────│   │
│   │  GEO  │ 35,786km  │ 1/3지구  │ 250ms │ 방송, 통신     │   │
│   │  MEO  │ 5,000-20,000km│ 지역  │ 100ms │ GPS           │   │
│   │  LEO  │ 300-2,000km│ 좁음    │ 20ms  │ Starlink, IoT │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              위성 통신 시스템                                │   │
│   │                                                              │   │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │      [지구국]                                        │   │ │
│   │   │          │                                           │   │ │
│   │   │          │ 업링크                                    │   │ │
│   │   │          ▼                                           │   │ │
│   │   │      ┌───────────┐                                   │   │ │
│   │   │      │  위성     │                                   │   │ │
│   │   │      │ (트랜스폰더)│                                   │   │ │
│   │   │      └───────────┘                                   │   │ │
│   │   │          │                                           │   │ │
│   │   │          │ 다운링크                                  │   │ │
│   │   │          ▼                                           │   │ │
│   │   │      [수신국]                                        │   │ │

│   │   │  • C-band: 6/4 GHz                                   │   │ │

│   │   │                                                      │   │ │

│   │                                                              │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: LEO 위성(Starlink)은 저지연으로 지상망 수준의 인터넷을 제공합니다.

**💡 비유**: 위성은 하늘에 떠 있는 거대한 중계기입니다.

**📢 섹션 요약 비유:** LEO = 저지연!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 궤도 | 고도 | 위성 수 | 지연 |
|:---|:---:|:---:|:---:|
| **GEO** | 35,786km | 3 | 250ms |
| **MEO** | 20,000km | 24-30 | 100ms |
| **LEO** | 550km | 1000s | 20ms |

**비교표**

```
+------------+----------+----------+
| 항목       │ GEO      │ LEO      │
+------------+----------+----------+
| 지연       │ 높음     │ 낮음     │
│ 위성 수    │ 3        │ 수천     │
│ 핸드오프    │ 없음     │ 빈번     │
│ 용도       │ 방송     │ 인터넷   │
+------------+----------+----------+
```

**핵심 알고리즸:** 링크 예산, 트랜스폰더, 빔포밍

**📢 섹션 요약 비유:** Starlink = 42,000개 LEO!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 위성 서비스**

| 서비스 | 위성 | 용도 |
|:---|:---|:---|
| **DTH** | GEO | 직접 방송 |
| **VSAT** | GEO | 기업망 |
| **Starlink** | LEO | 인터넷 |
| **GPS** | MEO | 위치 |

**📢 섹션 요약 비유:** LEO 인터넷 = 지상망 대체!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 방송** → GEO, Ku-band
**시나리오 2: 인터넷** → LEO, Ka-band
**시나리오 3: IoT** → LEO, 광대역

**도입 체크리스트:** 커버리지, 지연, 비용

**📢 섹션 요약 비유:** 위성은 어디서나 연결!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 전 지구 커버리지

**미래 전망:** LEO 인터넷 확대, 6G 통합

**참고 표준:** ITU-R, 3GPP NTN
