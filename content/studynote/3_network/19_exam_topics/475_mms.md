# MMS (멀티미디어 메시지 서비스) - Multimedia Messaging Service

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SMS에 멀티미디어 추가
> 2. **가치**: 이미지, 비디오, 오디오 전송
> 3. **융합:** WAP, MMSC, RCS로 진화

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MMS는 멀티미디어 메시지를 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MMS 아키텍처                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MMS 구성                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────┐    ┌─────────┐    ┌─────────┐         │   │ │
│   │   │  │ Sender  │───►│ MMSC    │───►│ Receiver│         │   │ │
│   │   │  │ MMS     │    │ (중계)  │    │ MMS     │         │   │ │
│   │   │  │ Client  │    │         │    │ Client  │         │   │ │
│   │   │  └─────────┘    └─────────┘    └─────────┘         │   │ │
│   │   │                                                      │   │ │
│   │   │  MMS 지원:                                           │   │ │
│   │   │  - 이미지 (JPEG, GIF)                               │   │ │
│   │   │  - 비디오 (3GP, MP4)                                │   │ │
│   │   │  - 오디오 (AMR, MP3)                                │   │ │
│   │   │  - 텍스트                                           │   │ │
│   │   │  - 최대 300KB (일반적)                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MMS vs SMS                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 항목       │ SMS      │ MMS      │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 미디어     │ 텍스트   │ 멀티미디어│              │   │ │
│   │   │  │ 크기       │ 160자    │ 300KB    │               │   │ │
│   │   │  | 비용       │ 낮음     │ 높음     │               │   │ │
│   │   │  │ 네트워크   │ CS       │ PS       │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MMS는 SMS를 확장하여 미디어를 지원합니다.

**💡 비유**: MMS는 엽서에서 사진 편지로 진화입니다.

**📢 섹션 요약 비유:** MMS = 멀티미디어 메시지!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **MMSC** | MMS Center |
| **MMS Client** | 단말 앱 |
| **WAP** | Wireless Application Protocol |
| **PDU** | Protocol Data Unit |

**비교표: MMS 규격**

```
+------------+----------+----------+----------+
| 규격       │ 크기     │ 미디어   │ 버전     │
+------------+----------+----------+----------+
| MMS 1.0    │ 30KB     │ 기본     │ 초기     │
| MMS 1.2    │ 100KB    │ 확장     │ 2G       │
| MMS 1.3    │ 300KB    │ 풀       │ 3G       │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 인코딩, MIME

**📢 섹션 요약 비유:** MMSC = 멀티미디어 우체국!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 메시징 진화**

| 세대 | 기술 | 특징 |
|:---|:---|:---|
| **1G** | 음성 | 통화 |
| **2G** | SMS | 텍스트 |
