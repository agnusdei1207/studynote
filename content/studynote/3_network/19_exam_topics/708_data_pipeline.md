# Data Pipeline - 데이터 파이프라인

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 이동/처리 흐름
> 2. **가치**: 자동화, 신뢰성
> 3. **융합:** ETL, Stream Processing, Batch

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Data Pipeline은 데이터를 자동으로 이동/처리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Data Pipeline                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              데이터 파이프라인 구조                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Source          Pipeline                 Target    │   │ │
│   │   │  ┌─────────┐     ┌─────────────────────┐ ┌─────────┐│   │ │
│   │   │  │  DB     │     │ Ingest │ Transform  │ │  DW     ││   │ │
│   │   │  │         │────→│        │            │─→│         ││   │ │
│   │   │  └─────────┘     └─────────────────────┘ └─────────┘│   │ │
│   │   │  ┌─────────┐     ┌─────────────────────┐ ┌─────────┐│   │ │
│   │   │  │  API    │────→│ Parse │ Validate   │─→│  Lake   ││   │ │
│   │   │  └─────────┘     └─────────────────────┘ └─────────┘│   │ │
│   │   │  ┌─────────┐     ┌─────────────────────┐ ┌─────────┐│   │ │
│   │   │  │ Stream  │────→│ Real-time Process   │─→│  DB     ││   │ │
│   │   │  └─────────┘     └─────────────────────┘ └─────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                │   │ │
│   │   │  - 자동화 (Automation)                               │   │ │
│   │   │  - 스케줄링 (Scheduling)                             │   │ │
│   │   │  - 모니터링 (Monitoring)                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 데이터 파이프라인은 데이터를 자동으로 처리합니다.

**📢 섹션 요약 비유:** Data Pipeline = 데이터 컨베이어!
