# MTU - 최대 전송 단위

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 패킷 최대 크기
> 2. **가치**: 단편화 방지
> 3. **융합:** MSS, Path MTU, Fragmentation

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MTU (Maximum Transmission Unit)는 한 번에 전송 가능한 최대 패킷 크기입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MTU                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MTU 구조                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  이더넷 MTU (1500 bytes):                                │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌──────────────────────────────────────────┐   ││   │ │
│   │   │  │  │ IP Header (20B) │ TCP Header (20B) │ Data │   ││   │ │
│   │   │  │  └──────────────────────────────────────────┘   ││   │ │
│   │   │  │           ←────────── 1500 bytes ──────────→    ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  MSS = MTU - IP Header - TCP Header             ││   │ │
│   │   │  │      = 1500 - 20 - 20 = 1460 bytes              ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  MTU 값:                                                 │   │ │
│   │   │  - Ethernet: 1500 bytes                               │   │ │
│   │   │  - PPPoE: 1492 bytes                                   │   │ │
│   │   │  - VPN: 1400 bytes (오버헤드)                           │   │ │
│   │   │  - Jumbo Frame: 9000 bytes                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MTU는 네트워크 성능에 큰 영향을 미칩니다.

**📢 섹션 요약 비유:** MTU = 트럭 적재 용량!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **MTU** | 최대 전송 단위 |

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**MTU vs MSS**

| 항목 | MTU | MSS |
|:---|:---|:---|
| **계층** | L2/L3 | L4 |
| **포함** | 전체 | 데이터만 |
| **크기** | 1500 | 1460 |
| **TCP** | 관련 없음 | 직접 관련 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 일반 네트워크 → 1500
**시나리오 2:** VPN → 1400
**시나리오 3:** 데이터센터 → 9000 (Jumbo)
