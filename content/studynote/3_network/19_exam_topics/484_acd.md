# ACD (자동 착신 분배) - Automatic Call Distributor

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 콜을 상담원에게 자동 분배
> 2. **가치**: 대기 시간 최소화, 효율적 배분
> 3. **융합**: IVR, CTI, Skills-based Routing

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Queue** | 대기열 |
| **Agent** | 상담원 |
| **Skills** | 기술 그룹 |
| **Priority** | 우선순위 |

**비교표: 분배 전략**

```
+------------+----------+----------+----------+
| 전략       │ 장점     │ 단점     │ 용도     │
+------------+----------+----------+----------+
| 순환       │ 공평     │ 단순     │ 일반     │
| 스킬기반   │ 전문성   │ 복잡     │ 기술지원 │
| 우선순위   │ VIP      │ 불공평   │ 프리미엄 │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 대기열, 상담원 상태
