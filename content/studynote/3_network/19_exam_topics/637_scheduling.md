# Scheduling - 스케줄링

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU 시간 분배
> 2. **가치**: 공정성, 효율성
> 3. **융합:** OS, Process, Priority

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Scheduling은 CPU 시간을 프로세스에 분배합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Scheduling                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              스케줄링 큐                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Ready Queue                                         │   │ │
│   │   │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐           │   │ │
│   │   │  │ P1  │→│ P2  │→│ P3  │→│ P4  │→│ P5  │           │   │ │
│   │   │  └─────┘ └─────┘ └─────┘ └─────┘ └─────┘           │   │ │
│   │   │      ↓                                             │   │ │
│   │   │  Scheduler                                         │   │ │
│   │   │      ↓                                             │   │ │
│   │   │  ┌─────────┐                                       │   │ │
│   │   │  │   CPU   │ ← Running                             │   │ │
│   │   │  └─────────┘                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  스케줄링 알고리즘:                                  │   │ │
│   │   │  FCFS, SJF, Priority, Round Robin, Multilevel       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 스케줄러는 CPU 할당을 결정합니다.

**📢 섹션 요약 비유:** Scheduling = 식당 줄 관리!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Scheduler** | 스케줄러 |
| **Ready Queue** | 대기 큐 |
