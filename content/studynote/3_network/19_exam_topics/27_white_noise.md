# 백색 잡음 (White Noise)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 모든 주파수에서 균일한 전력 밀도를 가진 무작위 신호
> 2. **가치**: 통신 시스템 성능 한계 결정, Shannon 용량 기반
> 3. **융합**: 열 잡음, 가우스 분포, SNR 계산과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

백색 잡음(White Noise)은 가시광선의 백색광처럼 모든 주파수 대역에서 균일한 전력을 가진 무작위 신호입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    백색 잡음 (White Noise)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              백색 잡음 특성                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  전력 스펙트럼 밀도 (PSD):                              │   │ │
│   │   │                                                      │   │ │
│   │   │  전력   │────────────────────────────────             │   │ │
│   │   │  밀도   │   일정한 전력 밀도 (Flat Spectrum)           │   │ │
│   │   │    ↑    │                                              │   │ │
│   │   │    │    │                                              │   │ │
│   │   │    └────┼──────────────────────────────→ 주파수       │   │ │
│   │   │         0     f1    f2    f3                          │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                               │   │ │
│   │   │  • 모든 주파수에서 동일한 세기                         │   │ │
│   │   │  • "하얀" = 프리즘처럼 모든 파장 포함                  │   │ │
│   │   │  • 시간상 무작위 (Random)                              │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              열 잡음 (Thermal Noise)                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  열 잡음 = 백색 잡음의 주요 원인                        │   │ │
│   │   │                                                      │   │ │
│   │   │  전력: N = kTB                                       │   │ │
│   │   │  • k = 볼츠만 상수 (1.38×10⁻²³ J/K)                  │   │ │
│   │   │  • T = 절대 온도 (Kelvin)                             │   │ │
│   │   │  • B = 대역폭 (Hz)                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  예: 상온(290K), 1MHz 대역폭                          │   │ │
│   │   │  N = 1.38×10⁻²³ × 290 × 10⁶ = 4×10⁻¹⁵ W             │   │ │
│   │   │    = -114 dBm                                        │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 백색 잡음은 피할 수 없는 물리적 현상으로, 통신 시스템의 근본적 한계입니다.

**💡 비유**: 백색 잡음은 라디오의 지직거리는 소리, TV의 화면 노이즈와 같습니다.

**📢 섹션 요약 비유**: 모든 주파수에 고른 잡음!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **PSD** | 전력 밀도 | 일정 | 평평함 |
| **Gaussian** | 확률 분포 | 정규 분포 | 종 모양 |
| **AWGN** | 채널 모델 | 가산 백색 | 기준 |
| **SNR** | 품질 지표 | 신호/잡음 | 비율 |

**가우스 분포**

```
확률 밀도 함수:

      │      ╱╲
  P   │     ╱  ╲
  (x) │    ╱    ╲
      │   ╱      ╲
      │  ╱        ╲
      └─┴────────┴──→ x
       -σ  0  +σ

평균 = 0, 표준편차 = σ
68%: ±1σ, 95%: ±2σ, 99.7%: ±3σ
```

**핵심 알고리즘**: kTB → 열 잡음 전력 계산 → SNR 산출

**📢 섹션 요약 비유**: AWGN이 통신의 기준!
