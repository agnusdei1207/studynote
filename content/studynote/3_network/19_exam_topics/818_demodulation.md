# Demodulation - 복조

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 복원
> 2. **가치**: 정보 추출

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Demodulation은 변조된 신호에서 원신호를 복원합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Demodulation                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              복조 과정                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  송신측:             수신측:                          │   │ │
│   │   │  ┌────┐  변조  ┌────┐  전송  ┌────┐  복조  ┌────┐  │   │ │
│   │   │  │정보│ ───→ │변조│ ───→  │수신│ ───→ │정보│  │   │ │
│   │   │  │    │       │신호│       │신호│       │    │  │   │ │
│   │   │  └────┘       └────┘       └────┘       └────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  AM 복조:                                             │   │ │
│   │   │  - Envelope Detector                                  │   │ │
│   │   │  - 다이오드 + RC 필터                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  FM 복조:                                             │   │ │
│   │   │  - Frequency Discriminator                            │   │ │
│   │   │  - PLL (Phase Locked Loop)                            │   │ │
│   │   │                                                      │   │ │
│   │   │  디지털 복조:                                          │   │ │
│   │   │  - IQ Demodulator                                     │   │ │
│   │   │  - Constellation Mapping                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 복조는 수신기의 핵심 기능입니다.

**📢 섹션 요약 비유:** Demodulation = 편지 개봉!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Detector** | 검파기 |
| **PLL** | 위상 고정 루프 |
| **Mixer** | 믹서 |
| **Filter** | 필터 |
| **ADC** | 아날로그-디지털 |
