# VoWiFi - WiFi 음성

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: WiFi 네트워크 기반 음성 통화
> 2. **가치**: 실내 커버리지, 해외 로밍
> 3. **융합**: IMS, VoLTE, EPC

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VoWiFi(Voice over WiFi)는 WiFi 네트워크로 음성 통화를 제공합니다.

> **해설**: VoWiFi는 ePDG를 통해 WiFi에서 IMS 음성 서비스를 제공합니다.

**💡 비유**: VoWiFi는 WiFi로 전화하는 것입니다. 실내에서도 잘 돼요.

**📢 섹션 요약 비유:** VoWiFi = WiFi 통화!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **ePDG** | 패킷 데이터 게이트웨이 |
| **IPsec** | 보안 터널 |
| **IKEv2** | 키 교환 |
| **EAP-AKA** | 인증 |

**비교표: VoWiFi vs VoLTE**

```
+------------+----------+----------+
| 항목       │ VoWiFi   │ VoLTE    │
+------------+----------+----------+
| 네트워크   │ WiFi     │ LTE      │
| QoS        │ Best Effort│ 보장   │
| 보안       │ IPsec    │ 네이티브 │
| 커버리지   │ WiFi     │ LTE      │
+------------+----------+----------+
```

**핵심 알고리즸:** ePDG 터널, IMS

**📢 섹션 요약 비유:** ePDG = WiFi 게이트!
