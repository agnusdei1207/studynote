# 반송파 (Carrier Wave)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 정보를 실어 나르는 고주파 신호
> 2. **가치**: 안테나 크기, 주파수 다중화 기반
> 3. **융합**: AM, FM, PM, 디지털 변조와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

반송파(Carrier Wave)는 정보 신호를 전송하기 위해 사용되는 고주파 사인파입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    반송파 (Carrier Wave)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              반송파 기본                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  수식: c(t) = A·cos(2πf_c·t + φ)                      │   │ │
│   │   │                                                      │   │ │
│   │   │  매개변수:                                             │   │ │
│   │   │  • A (진폭): Amplitude                               │   │ │
│   │   │  • f_c (주파수): Carrier Frequency                   │   │ │
│   │   │  • φ (위상): Phase                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  파형:                                                 │   │ │
│   │   │         ╱╲    ╱╲    ╱╲    ╱╲                         │   │ │
│   │   │        ╱  ╲  ╱  ╲  ╱  ╲  ╱  ╲                        │   │ │
│   │   │  ────╱────╲╱────╲╱────╲╱────╲───                    │   │ │
│   │   │       ← 주기 T = 1/f_c →                             │   │ │
│   │   │                                                      │   │ │
│   │   │  파장: λ = c/f_c (c = 빛의 속도)                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              변조 가능한 매개변수                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 진폭 (Amplitude):                                  │   │ │
│   │   │     • AM (아날로그)                                    │   │ │
│   │   │     • ASK (디지털)                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 주파수 (Frequency):                                 │   │ │
│   │   │     • FM (아날로그)                                    │   │ │
│   │   │     • FSK (디지털)                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 위상 (Phase):                                       │   │ │
│   │   │     • PM (아날로그)                                    │   │ │
│   │   │     • PSK (디지털)                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 진폭 + 위상:                                        │   │ │
│   │   │     • QAM (디지털)                                     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 반송파 주파수는 응용에 따라 다릅니다.

**💡 비유**: 반송파는 화물을 싣는 트럭입니다.

**📢 섹션 요약 비유**: 진폭, 주파수, 위상이 3대 요소!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **주파수** | 채널 | f_c | 라디오 국 |
| **진폭** | 세기 | A | 볼륨 |
| **위상** | 타이밍 | φ | 시간 |
| **파장** | 크기 | λ = c/f | 안테나 |

**주파수 대역별 응용**

```
+------------+----------+------------------+
| 대역       | 주파수   | 응용             |
+------------+----------+------------------+
| LF         | 30-300kHz| 장파 라디오      |
| MF         | 0.3-3MHz | AM 라디오        |
| HF         | 3-30MHz  | 단파             |
| VHF        | 30-300MHz| FM, TV           |
| UHF        | 0.3-3GHz | TV, LTE          |
| SHF        | 3-30GHz  | WiFi, 위성       |
+------------+----------+------------------+
```

**핵심 알고리즘**: 베이스밴드 → 반송파 변조 → 전송

**📢 섹션 요약 비유**: 안테나 크기 ≈ 파장/2!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 반송파 주파수와 안테나**

| 주파수 | 파장 | 안테나 |
|:---|:---|:---|
| **1MHz** | 300m | 75m (λ/4) |
| **100MHz** | 3m | 0.75m |
| **1GHz** | 30cm | 7.5cm |
| **10GHz** | 3cm | 0.75cm |

**📢 섹션 요약 비유**: 고주파 = 소형 안테나!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 라디오** → MF/HF 반송파
**시나리오 2: WiFi** → 2.4/5GHz 반송파
**시나리오 3: 5G** → mmWave (28/39GHz)

**도입 체크리스트**: 커버리지, 대역폭, 안테나 크기

**📢 섹션 요약 비유**: 5G mmWave = 초고주파 반송파!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 정보 전송의 기반

**미래 전망**: THz 통신 (6G)

**참고 표준**: ITU-R 주파수 대역

**📢 섹션 요약 비유**: 반송파는 통신의 그릇!
