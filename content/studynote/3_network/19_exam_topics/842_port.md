# Port - 포트

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스 식별자
> 2. **가치**: 다중 서비스 구분
> 3. **융합:** Well-Known, Ephemeral, Socket

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Port는 호스트 내 프로세스를 식별하는 번호입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Port                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              포트 번호 범위                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  범위                    용도                         │   │ │
│   │   │  ────────────────────────────────────────────        │   │ │
│   │   │  0-1023        Well-Known Ports                      │   │ │
│   │   │                (시스템 포트, root 필요)                │   │ │
│   │   │                20/21: FTP                             │   │ │
│   │   │                22: SSH                                │   │ │
│   │   │                23: Telnet                             │   │ │
│   │   │                25: SMTP                               │   │ │
│   │   │                53: DNS                                │   │ │
│   │   │                80: HTTP                               │   │ │
│   │   │                443: HTTPS                             │   │ │
│   │   │                                                      │   │ │
│   │   │  1024-49151  Registered Ports                         │   │ │
│   │   │                (등록된 포트)                           │   │ │
│   │   │                3306: MySQL                            │   │ │
│   │   │                5432: PostgreSQL                       │   │ │
│   │   │                6379: Redis                            │   │ │
│   │   │                                                      │   │ │
│   │   │  49152-65535 Dynamic/Ephemeral Ports                   │   │ │
│   │   │                (동적/임시 포트)                         │   │ │
│   │   │                클라이언트 연결용                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 포트는 다중 애플리케이션 통신을 가능하게 합니다.

**📢 섹션 요약 비유:** Port = 건물 호수!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Well-Known** | 잘 알려진 포트 |
| **Registered** | 등록된 포트 |
| **Ephemeral** | 임시 포트 |
| **Socket** | IP:Port 조합 |
| **NAT** | 포트 변환 |

**핵심 알고리즸:** IP + Port = Socket
