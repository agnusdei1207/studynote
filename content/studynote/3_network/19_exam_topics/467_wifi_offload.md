# Wi-Fi 오프로드 - Wi-Fi Offload

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 모바일 트래픽을 Wi-Fi로 분산
> 2. **가치**: 비용 절감, 용량 증대
> 3. **융합:** ANDSF, Hotspot 2.0, Passpoint

---

### Ⅰ. 개요 (Context & Background)

**💡 비유**: Wi-Fi 오프로드는 고속도로 우회도로입니다.

**📢 섹션 요약 비유:** Offload = 비용 절감!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **ANDSF** | Access Network Discovery |
| **Hotspot 2.0** | 자동 연결 |
| **Passpoint** | 인증 표준 |
| **EAP-SIM/AKA** | 통신사 인증 |

**비교표: 오프로드 기술**

```
+------------+----------+----------+----------+
| 기술       │ 기능     │ 표준     │ 용도     │
+------------+----------+----------+----------+
| ANDSF      │ 정책     │ OMA     │ 안드로이드│
| Hotspot 2.0│ 자동연결 │ WFA     │ 공용Wi-Fi│
| MPTCP      │ 다중경로 │ IETF     │ 동시사용 │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 정책 기반 전환

**📢 섹션 요약 비유:** Hotspot 2.0 = 자동 연결!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: VoWiFi vs VoLTE**

| 항목 | VoWiFi | VoLTE |
|:---|:---|:---|
| **커버리지** | 실내 우수 | 실외 우수 |
| **품질** | 가변 | 안정 |
| **비용** | 낮음 | 높음 |

**📢 섹션 요약 비유:** VoWiFi = 실내 커버리지!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 실내 → Wi-Fi 오프로드
**시나리오 2:** 혼잡 → 자동 오프로드
**시나리오 3:** 해외 → VoWiFi

**도입 체크리스트:** 정책, 인증, 품질

**📢 섹션 요약 비유:** 30-70% 오프로드!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 데이터 비용 30% 절감

**미래 전망:** ATSSS (5G)

**참고 표준:** IEEE 802.11u, WFA Passpoint

**📢 섹션 요약 비유:** Wi-Fi 오프로드는 통신사 필수!
