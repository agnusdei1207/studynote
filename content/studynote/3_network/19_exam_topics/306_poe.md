# PoE - Power over Ethernet

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이더넷 케이블로 전력 공급
> 2. **가치**: 배선 단순화, IP 장비
> 3. **융합**: VoIP, 무선 AP와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PoE(Power over Ethernet)는 이더넷 케이블을 통해 데이터와 전력을 동시에 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PoE                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PoE 구조                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [PoE Switch]                                       │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ UTP 케이블 (데이터 + 전력)                    │   │ │
│   │   │       │ 1,2,3,6: 데이터                               │   │ │
│   │   │       │ 4,5,7,8: 전력 (PoE+)                         │   │ │
│   │   │       │                                              │   │ │
│   │   │       ├──────────────┬──────────────┐                │   │ │
│   │   │       ↓              ↓              ↓                │   │ │
│   │   │  ┌──────────┐  ┌──────────┐  ┌──────────┐            │   │ │
│   │   │  │ IP Phone │  │ Wireless │  │ IP Camera│            │   │ │
│   │   │  │ (VoIP)   │  │ AP       │  │ (CCTV)   │            │   │ │
│   │   │  │          │  │          │  │          │            │   │ │
│   │   │  │ Powered  │  │ Powered  │  │ Powered  │            │   │ │
│   │   │  │ Device   │  │ Device   │  │ Device   │            │   │ │
│   │   │  └──────────┘  └──────────┘  └──────────┘            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PoE 표준                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │  │ 표준       │ IEEE     │ 전력     │ 용도     │     │   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │  │ PoE        │ 802.3af  │ 15.4W    │ 기본     │     │   │ │
│   │   │  │ PoE+       │ 802.3at  │ 30W      │ AP, PTZ │     │   │ │
│   │   │  │ PoE++      │ 802.3bt  │ 60W/100W │ 고전력   │     │   │ │
│   │   │  +------------+----------+----------+----------+     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: PoE는 스위치에서 전력을 공급하고 PD가 소비합니다.

**💡 비유**: PoE는 USB 충전과 같습니다. 데이터와 전력을 한 케이블로 전송합니다.

**📢 섹션 요약 비유:** PoE = 전력 + 데이터!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **PSE** | 전력 공급 (Switch) |
| **PD** | 전력 소비 (Device) |
| **Class** | 전력 등급 |
| **Detection** | 감지 |

**비교표: PoE Class**

```
+------------+----------+----------+
| Class      │ 전력     │ 용도     │
+------------+----------+----------+
| 0          │ 15.4W    │ 기본     │
| 1          │ 4W       │ 저전력   │
| 2          │ 7W       │ 중간     │
| 3          │ 15.4W    │ 표준     │
| 4          │ 30W      │ PoE+     │
| 5-8        │ 60-100W  │ PoE++    │
+------------+----------+----------+
```

**핵심 알고리즸:** Detection, Classification

**📢 섹션 요약 비유:** Class = 전력 등급!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: VoIP** → PoE Class 3
**시나리오 2: AP** → PoE+ (30W)
**시나리오 3: PTZ 카메라** → PoE++ (60W+)

**도입 체크리스트:** 전력 예산, Class, 케이블

**📢 섹션 요약 비유:** 전력 예산 = 총 전력!
