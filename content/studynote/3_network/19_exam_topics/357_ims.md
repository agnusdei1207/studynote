# IMS - IP 멀티미디어 서브시스템

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 통신사 IP 멀티미디어 아키텍처
> 2. **가치**: 서비스 통합, FMC
> 3. **융합**: VoLTE, VoWiFi, RCS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IMS(IP Multimedia Subsystem)는 통신사의 IP 기반 멀티미디어 아키텍처입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IMS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IMS 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [UE - User Equipment]                              │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ SIP                                            │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           P-CSCF (Proxy)                    │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           I-CSCF (Interrogating)            │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           S-CSCF (Serving)                  │    │   │ │
│   │   │  │  • 등록, 라우팅, 세션 제어                    │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [Application Servers]                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IMS 서비스                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 서비스     │ 설명     │ 네트워크 │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ VoLTE      │ LTE 음성 │ 4G       │               │   │ │
│   │   │  │ VoWiFi     │ WiFi 음성│ WiFi     │               │   │ │
│   │   │  | RCS        │ 메신저   │ 4G/5G    │               │   │ │
│   │   │  │ SMS        │ 문자     │ 2G+      │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IMS는 SIP 기반으로 통신사 멀티미디어 서비스를 제공합니다.

**💡 비유**: IMS는 통신사의 앱 스토어 인프라입니다. 모든 서비스를 관리합니다.

**📢 섹션 요약 비유:** IMS = 통신사 IP 플랫폼!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **P-CSCF** | 프록시 CSCF |
| **I-CSCF** | 조회 CSCF |
| **S-CSCF** | 서빙 CSCF |
| **HSS** | 홈 구독자 서버 |

**비교표: CSCF 역할**

```
+------------+----------+----------+
| CSCF       │ 역할     │ 위치     │
+------------+----------+----------+
| P-CSCF     │ 진입점   │ 가장자리 │
| I-CSCF     │ 조회     │ 홈       │
| S-CSCF     │ 세션     │ 홈       │
+------------+----------+----------+
```

**핵심 알고리즸:** SIP 라우팅, 인증

**📢 섹션 요약 비유:** HSS = 사용자 DB!
