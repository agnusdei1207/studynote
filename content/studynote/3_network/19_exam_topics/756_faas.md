# FaaS (Function as a Service) - 함수형 서비스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 함수 단위 실행
> 2. **가치**: 이벤트 기반, 사용량 과금
> 3. **융합:** Serverless, Event Triggers, Stateless

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

FaaS는 함수를 서비스로 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    FaaS (Function as a Service)                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              FaaS 실행 모델                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Cold Start:                                         │   │ │
│   │   │  이벤트 ──→ [환경 생성] ──→ [초기화] ──→ [실행]       │   │ │
│   │   │         (수백 ms ~ 수 초)                             │   │ │
│   │   │                                                      │   │ │
│   │   │  Warm Start:                                         │   │ │
│   │   │  이벤트 ──→ [실행]                                   │   │ │
│   │   │         (수 ms)                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                │   │ │
│   │   │  - Stateless (상태 없음)                              │   │ │
│   │   │  - Ephemeral (일시적)                                 │   │ │
│   │   │  - Event-driven (이벤트 기반)                          │   │ │
│   │   │  - Auto-scaling (자동 확장)                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: FaaS는 이벤트에 반응하여 함수를 실행합니다.

**📢 섹션 요약 비유:** FaaS = 자판기!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Function** | 함수 |
| **Trigger** | 트리거 |
| **Runtime** | 런타임 |
| **Execution** | 실행 |
| **Timeout** | 타임아웃 |

**핵심 알고리즸:** 이벤트 → 함수 호출 → 실행 → 응답
