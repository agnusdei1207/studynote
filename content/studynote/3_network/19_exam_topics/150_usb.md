# USB / IEEE 1394 (FireWire)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고속 직렬 버스, 플러그 앤 플레이
> 2. **가치**: 범용 연결, 핫 스왑
> 3. **융합**: PC, 모바일, IoT와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

USB(Universal Serial Bus)는 PC와 주변장치를 연결하는 범용 인터페이스이고, IEEE 1394(FireWire)는 Apple이 개발한 고속 시리얼 버스입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    USB / IEEE 1394                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              USB 버전 진화                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  버전   │ 속도       │ 명칭        │ 연도  │ 전력   │   │ │
│   │   │  ──────┼───────────┼────────────┼──────┼───────│   │ │
│   │   │  1.0   │ 1.5/12Mbps │ Low/Full   │ 1996  │ 500mA │   │ │
│   │   │  1.1   │ 12Mbps     │ Full Speed │ 1998  │ 500mA │   │ │
│   │   │  2.0   │ 480Mbps    │ Hi-Speed   │ 2000  │ 500mA │   │ │
│   │   │  3.0   │ 5Gbps      │ SuperSpeed │ 2008  │ 900mA │   │ │
│   │   │  3.1   │ 10Gbps     │ SS+        │ 2013  │ 900mA │   │ │
│   │   │  3.2   │ 20Gbps     │ SS++       │ 2017  │ 900mA │   │ │
│   │   │  4.0   │ 40Gbps     │ USB4       │ 2019  │ 100W+ │   │ │
│   │   │  4 v2  │ 80Gbps     │ USB4 v2    │ 2022  │ 240W  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              USB 커넥터                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  타입   │ 특징                    │ 용도            │   │ │
│   │   │  ──────┼─────────────────────────┼────────────────│   │ │
│   │   │  Type-A│ 기존 큰 커넥터          │ PC, 허브       │   │ │
│   │   │  Type-B│ 프린터용                │ 프린터         │   │ │
│   │   │  Mini-B│ 구형 소형               │ 구형 기기      │   │ │
│   │   │  Micro-B│ 안드로이드 충전        │ 모바일         │   │ │
│   │   │  Type-C│ 양면 삽입, 고속         │ 범용           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IEEE 1394 (FireWire)                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  버전   │ 속도    │ 명칭       │ 용도              │   │ │
│   │   │  ──────┼────────┼───────────┼──────────────────│   │ │
│   │   │  1394a │ 400Mbps │ FireWire 400│ 비디오, 오디오   │   │ │
│   │   │  1394b │ 800Mbps │ FireWire 800│ 전문가용         │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                   │   │ │
│   │   │  • 등시성 전송 (실시간 보장)                              │   │ │
│   │   │  • 피어 투 피어                                          │   │ │
│   │   │  • 데이지 체인 (최대 63노드)                              │   │ │
│   │   │  • Apple 전유 → USB에 밀려 사양                          │   │ │
```

> **해설**: USB는 PC 주변장치의 표준, IEEE 1394는 전문가용 비디오/오디오에 사용되었습니다.

**💡 비유**: USB는 만능 콘센트, IEEE 1394는 전문가용 고속 도로입니다.

**📢 섹션 요약 비유:** USB4 = 40Gbps, Type-C = 미래!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **호스트** | 컨트롤러 | PC | 서버 |
| **디바이스** | 주변장치 | 슬레이브 | 클라이언트 |
| **허브** | 확장 | 포트 확장 | 분배기 |

**비교표**

```
+------------+----------+----------+
| 항목       | USB      | FireWire |
+------------+----------+----------+
| 구조       | 호스트 중심| 피어투피어|
| 실시간     | 제한      | 보장     |
| 전력       │ 제공      │ 제공     |
| 승리       │ O        │ X       |
+------------+----------+----------+
```

**핵심 알고리즸:** 토큰 버킷, 등시성 전송

**📢 섹션 요약 비유:** Type-C = 양면 삽입!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 전력 공급**

| 타입 | 전력 |
|:---|:---:|
| **USB 2.0** | 2.5W |
| **USB 3.0** | 4.5W |
| **USB PD** | 100W+ |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 일반 주변장치** → USB 3.0+
**시나리오 2: 모바일** → USB Type-C
**시나리오 3: 고속 외장** → USB4/Thunderbolt

**도입 체크리스트:** 속도, 전력, 커넥터

**📢 섹션 요약 비유:** USB는 모든 것을 연결!
