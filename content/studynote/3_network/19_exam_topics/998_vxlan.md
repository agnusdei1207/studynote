# VXLAN - 가상 확장 가능 LAN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: L2 오버레이 on L3
> 2. **가치**: 16M 세그먼트
> 3. **융합:** VNI, VTEP, Underlay

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VXLAN (Virtual Extensible LAN)은 L2 네트워크를 L3 위에 오버레이합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VXLAN                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VXLAN 구조                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Overlay Network (L2 가상 네트워크):                     │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │     VM1          VM2          VM3               ││   │ │
│   │   │  │   ┌───┐        ┌───┐        ┌───┐              ││   │ │
│   │   │  │   │   │        │   │        │   │              ││   │ │
│   │   │  │   └─┬─┘        └─┬─┘        └─┬─┘              ││   │ │
│   │   │  │     │            │            │                 ││   │ │
│   │   │  │  ┌──┴──┐      ┌──┴──┐      ┌──┴──┐             ││   │ │
│   │   │  │  │VTEP1│      │VTEP2│      │VTEP3│             ││   │ │
│   │   │  │  └──┬──┘      └──┬──┘      └──┬──┘             ││   │ │
│   │   │  └─────┼─────────────┼─────────────┼───────────────┘│   │ │
│   │   │        │             │             │                 │   │ │
│   │   │  Underlay Network (L3 IP 네트워크):                   │   │ │
│   │   │  ┌─────┼─────────────┼─────────────┼───────────────┐│   │ │
│   │   │  │  ┌──┴──┐      ┌──┴──┐      ┌──┴──┐             ││   │ │
│   │   │  │  │Router│      │Router│      │Router│            ││   │ │
│   │   │  │  └─────┘      └─────┘      └─────┘             ││   │ │
│   │   │  │    VXLAN 패킷: UDP 4789                        ││   │ │
│   │   │  │    VNI: 24-bit (16M 세그먼트)                   ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: VXLAN은 VLAN의 4096 제한을 16M로 확장합니다.

**📢 섹션 요약 비유:** VXLAN = L2 터널링!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VNI** | VXLAN Network Identifier (24-bit) |
| **VTEP** | VXLAN Tunnel Endpoint |
| **Underlay** | 물리적 L3 네트워크 |
| **Overlay** | 가상 L2 네트워크 |
| **Multicast** | BUM 트래픽 처리 |

**핵심 알고리듬:** 캡슐화 → VTEP 전송 → 디캡슐레이션 → VM 전달

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**VXLAN vs VLAN**

| 항목 | VXLAN | VLAN |
|:---|:---|:---|
| **세그먼트** | 16M | 4096 |
| **전송** | L3 | L2 |
| **범위** | 데이터센터 전체 | 로컬 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 데이터센터 → VXLAN
**시나리오 2:** 멀티테넌시 → VXLAN + VNI
**시나리오 3:** 클라우드 → VXLAN + EVPN
