# 다이폴 안테나 (Dipole Antenna) - 반파장/1/4파장

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 두 도체로 구성된 기본 안테나
> 2. **가치**: 기준 안테나, 다양한 변형
> 3. **융합**: 라디오, TV, WiFi와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

다이폴 안테나(Dipole Antenna)는 두 개의 도체 요소로 구성된 가장 기본적인 안테나 형태입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    다이폴 안테나 (Dipole Antenna)                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              반파장 다이폴 (Half-wave Dipole)                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │           λ/4            λ/4                         │   │ │
│   │   │        ┌──────┐      ┌──────┐                       │   │ │
│   │   │        │      │      │      │                       │   │ │
│   │   │        │  ←───┴──────┴───→  │                       │   │ │
│   │   │        │      급전점      │                       │   │ │
│   │   │        │                  │                       │   │ │
│   │   │        └──────┘      └──────┘                       │   │ │
│   │   │           │              │                           │   │ │
│   │   │           └──────┬───────┘                           │   │ │
│   │   │                  │                                   │   │ │
│   │   │              [급전선]                                │   │ │
│   │   │                                                      │   │ │
│   │   │  총 길이 = λ/2 (반파장)                               │   │ │
│   │   │  이득 = 2.15 dBi                                     │   │ │
│   │   │  임피던스 ≈ 73Ω                                      │   │ │
│   │   │  빔폭 = 78°                                          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              다이폴 변형                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 단파장 다이폴 (Short Dipole):                       │   │ │
│   │   │     • 길이 << λ/2                                      │   │ │
│   │   │     • 이득 낮음, 임피던스 높음                          │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 접힌 다이폴 (Folded Dipole):                        │   │ │
│   │   │     • 평행 도체로 구성                                  │   │ │
│   │   │     • 임피던스 ≈ 300Ω                                  │   │ │
│   │   │     • TV 안테나                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 바이커닉스 (Biconical):                              │   │ │
│   │   │     • 광대역 다이폴                                     │   │ │
│   │   │     • EMI 측정용                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 로그 주기 (Log-Periodic):                           │   │ │
│   │   │     • 다중 다이폴 배열                                  │   │ │
│   │   │     • 광대역                                            │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 반파장 다이폴은 안테나 이득의 기준(dBd)이 됩니다.

**💡 비유**: 다이폴은 가장 기본적인 안테나로, 모든 안테나의 출발점입니다.

**📢 섹션 요약 비유:** 다이폴 = 안테나 기준!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 파라미터 | 반파장 | 단파장 | 접힌 |
|:---|:---:|:---:|:---:|
| **길이** | λ/2 | << λ/2 | λ/2 |
| **이득** | 2.15 dBi | < 2 dBi | 2.15 dBi |
| **임피던스** | 73Ω | 높음 | 300Ω |
| **대역폭** | 좁음 | 좁음 | 넓음 |

**비교표**

```
+------------+----------+----------+
| 항목       │ 다이폴   │ 모노폴   │
+------------+----------+----------+
| 도체 수    │ 2        │ 1        │
│ 길이       │ λ/2      │ λ/4      │
│ 접지       │ 불필요   │ 필요     │
│ 이득       │ 2.15dBi  │ 2.15dBi  │
+------------+----------+----------+
```

**핵심 알고리즸:** 공진 조건, 임피던스 정합

**📢 섹션 요약 비유:** λ/2 = 공진!
