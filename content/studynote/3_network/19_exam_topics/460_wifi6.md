# Wi-Fi 6/6E - 차세대 무선 LAN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: IEEE 802.11ax 무선 표준
> 2. **가치**: 고속, 고밀도, 저지연
> 3. **융합**: OFDMA, MU-MIMO, 6GHz

---

**개념 정의**

**💡 비유**: Wi-Fi 6는 8차선 하이웨이입니다.

**📢 섹션 요약 비유:** Wi-Fi 6 = 고밀도!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **OFDMA** | Orthogonal FDMA |
| **MU-MIMO** | Multi-User MIMO |
| **TWT** | Target Wake Time |
| **BSS Coloring** | 간섭 감소 |

**비교표: OFDMA vs OFDM**

```
+------------+----------+----------+----------+
| 항목       │ OFDM     │ OFDMA    │ 이점     │
+------------+----------+----------+----------+
| 자원       │ 전체     │ 서브채널  │ 효율     │
| 다중접속   │ 시간분할 │ 주파수   │ 지연     │
| 밀집       │ 낮음     │ 높음     │ 성능     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 스케줄링, 빔포밍

**📢 섹션 요약 비유:** OFDMA = 주파수 분할!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 주파수 대역**

| 대역 | 범위 | 특징 | 용도 |
|:---|:---|:---|:---|
| **2.4GHz** | 80MHz | 커버리지 | IoT |
| **5GHz** | 160MHz | 속도 | 일반 |
| **6GHz** | 160MHz | 간섭 없음 | 고성능 |

**📢 섹션 요약 비유:** 6GHz = 청정 대역!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 가정 → Wi-Fi 6 공유기

**시나리오 3:** IoT → TWT 활용

**도입 체크리스트:** 단말, AP, 간섭

**📢 섹션 요약 비유:** WPA3 = 보안!
