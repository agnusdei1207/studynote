# Congestion Control - 혼잡 제어

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 과부하 방지
> 2. **가치**: 공정한 자원 분배
> 3. **융합:** Slow Start, AIMD, ECN

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Congestion Control은 네트워크 과부하를 방지합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Congestion Control                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TCP 혼잡 제어                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  혼잡 윈도우 변화:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  cwnd                                           ││   │ │
│   │   │  │    ↑                                            ││   │ │
│   │   │  │    │         ██████████                         ││   │ │
│   │   │  │    │       ██          ████████                 ││   │ │
│   │   │  │    │     ██                  ████████           ││   │ │
│   │   │  │    │   ██                        ████████       ││   │ │
│   │   │  │    │ ██                              ████████   ││   │ │
│   │   │  │    └─────────────────────────────────────────→  ││   │ │
│   │   │  │        Slow Start  →   Congestion Avoidance     ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  알고리즘:                                              │   │ │
│   │   │  - Slow Start: 지수적 증가                              │   │ │
│   │   │  - AIMD: 선형 증가, 반으로 감소                          │   │ │
│   │   │  - Fast Retransmit: 3중 ACK                            │   │ │
│   │   │  - Fast Recovery: 손실 후 빠른 복구                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 혼잡 제어는 인터넷 안정성의 핵심입니다.

**📢 섹션 요약 비유:** Congestion Control = 교통 체증 해소!
