# NVGRE - 가상 라우팅 그리스 이더넷

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: GRE 기반 L2 오버레이
> 2. **가치**: 16M 테넌트
> 3. **융합:** VSID, GRE, Hyper-V

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NVGRE (Network Virtualization using GRE)는 GRE를 이용한 L2 오버레이입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NVGRE                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NVGRE 구조                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  NVGRE 캡슐화:                                          │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌─────────────────────────────────────────┐    ││   │ │
│   │   │  │  │            Outer IP Header              │    ││   │ │
│   │   │  │  │            (Underlay IP)                │    ││   │ │
│   │   │  │  └─────────────────────────────────────────┘    ││   │ │
│   │   │  │  ┌─────────────────────────────────────────┐    ││   │ │
│   │   │  │  │            GRE Header                   │    ││   │ │

│   │   │  │  └─────────────────────────────────────────┘    ││   │ │
│   │   │  │  ┌─────────────────────────────────────────┐    ││   │ │
│   │   │  │  │            Inner Ethernet Frame          │    ││   │ │
│   │   │  │  │            (Original L2 Frame)           │    ││   │ │

│   │   │  - 24-bit VSID (16M 테넌트)                            │   │ │
│   │   │  - Hyper-V 통합                                       │   │ │
```

> **해설**: NVGRE는 Microsoft 환경에서 주로 사용됩니다.

**📢 섹션 요약 비유:** NVGRE = Microsoft의 VXLAN!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VSID** | Virtual Subnet ID (24-bit) |
| **GRE** | Generic Routing Encapsulation |
| **PA** | Provider Address |
| **CA** | Customer Address |
| **NVGRE Endpoint** | 터널 종단 |

**핵심 알고리듬:** GRE 캡슐화 → PA 라우팅 → 디캡슐레이션 → CA 전달

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**NVGRE vs VXLAN**

| 항목 | NVGRE | VXLAN |
|:---|:---|:---|
| **캡슐화** | GRE | UDP |
| **포트** | IP 프로토콜 47 | UDP 4789 |
| **주도** | Microsoft | Cisco/VMware |
| **로드밸런싱** | 제한적 | 우수 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 3:** Azure → NVGRE
