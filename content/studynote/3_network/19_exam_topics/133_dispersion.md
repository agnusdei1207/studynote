# 분산 (Dispersion) - 모드 분산, 파장 분산

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 빛 신호의 시간적 퍼짐
> 2. **가치**: 대역폭 제한의 주요 원인
> 3. **융합**: 광섬유 설계, DWDM과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

분산(Dispersion)은 광섬유에서 빛 신호가 시간적으로 퍼지는 현상으로, 모드 분산과 파장 분산이 있습니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    분산 (Dispersion)                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              분산 종류                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  분산 종류:                                              │   │ │
│   │   │                                                      │   │ │
│   │   │  1. 모드 분산 (Modal Dispersion):                       │   │ │
│   │   │     • MMF에서 발생                                      │   │ │
│   │   │     • 여러 모드의 경로 차이                              │   │ │
│   │   │     • 언덕형에서 감소                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 파장 분산 (Chromatic Dispersion):                   │   │ │
│   │   │     • SMF/MMF 모두 발생                                 │   │ │
│   │   │     • 파장별 속도 차이                                   │   │ │
│   │   │     • 재료 분산 + 파도로 분산                            │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 편광 모드 분산 (PMD):                                │   │ │
│   │   │     • 편광 상태 차이                                     │   │ │
│   │   │     • 초장거리에서 문제                                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              분산 효과                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  입력 신호:          출력 신호 (분산 후):              │   │ │
│   │   │                                                      │   │ │
│   │   │    ┌──┐  ┌──┐           ┌─╲  ╱─┐                    │   │ │
│   │   │    │  │  │  │           │  ╲╱  │                    │   │ │
│   │   │  ──┘  └──┘  └──      ──┘    └──┘                    │   │ │
│   │   │    명확한 펄스        퍼진 펄스 (ISI)                  │   │ │
│   │   │                                                      │   │ │
│   │   │  영향:                                                   │   │ │
│   │   │  • 펄스 폭 증가 → ISI                                   │   │ │
│   │   │  • 비트 에러율 증가                                      │   │ │
│   │   │  • 최대 전송 거리 제한                                   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 분산은 광섬유 대역폭의 주요 제한 요소입니다.

**💡 비유**: 분산은 여러 사람이 다른 속도로 달려서 도착 시간이 퍼지는 것입니다.

**📢 섹션 요약 비유**: 분산 = 신호 퍼짐!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **모드 분산** | MMF 제한 | 경로차 | 차선차이 |
| **재료 분산** | 파장별 속도 | 굴절률 | 기울기 |
| **파도로 분산** | 구조 영향 | 코어 직경 | 터널 |

**비교표**

```
+------------+----------+----------+
| 항목       | 모드 분산| 파장 분산|
+------------+----------+----------+
| 발생       | MMF      | 모두     |
| 원인       | 경로차   | 파장차   |
| 보정       | 언덕형   | DCF      |
| 단위       | ns/km    | ps/nm·km |
+------------+----------+----------+
```

**핵심 알고리즸**: D = L × (Δt/Δλ), B × L = 상수

**📢 섹션 요약 비유**: 제로 분산 파장 = 1310nm!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 분산 보정**

| 방법 | 대상 | 방식 |
|:---|:---|:---|
| **DCF** | 파장 분산 | 음의 분산 섬유 |
| **DSP** | PMD | 전자 보정 |
| **언덕형** | 모드 분산 | 구조적 |

**📢 섹션 요약 비유**: 1550nm = 양의 분산!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: MMF 단거리** → 모드 분산 고려
**시나리오 2: SMF 장거리** → 파장 분산 보정
**시나리오 3: 40G/100G** → DCF 필수

**도입 체크리스트**: 거리, 파장, 속도, 비용

**📢 섹션 요약 비유**: 분산은 대역폭의 적!
