# 이더넷 물리 계층 표준 (IEEE 802.3 PHY)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이더넷의 물리적 전송 규격
> 2. **가치**: 호환성, 속도/매체 표준화
> 3. **융합**: 10M~400G 이더넷과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IEEE 802.3 PHY는 이더넷의 물리 계층 표준으로, 속도, 매체, 코딩 방식을 정의합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    이더넷 물리 계층 표준 (IEEE 802.3 PHY)           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              네이밍 규칙                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  형식: [속도][BASE/BROAD]-[매체타입]                  │   │ │
│   │   │                                                      │   │ │
│   │   │  예시: 1000BASE-T                                     │   │ │
│   │   │        │    │    │                                    │   │ │
│   │   │        │    │    └── T: Twisted Pair (꼬임쌍선)        │   │ │
│   │   │        │    │        S: Short Reach (단파장 광섬유)    │   │ │
│   │   │        │    │        L: Long Reach (장파장 광섬유)     │   │ │
│   │   │        │    │        K: Backplane                     │   │ │
│   │   │        │    └── BASE: Baseband (기저대역)              │   │ │
│   │   │        │        BROAD: Broadband (대역통과)            │   │ │
│   │   │        └── 1000: 1000Mbps = 1Gbps                     │   │ │
│   │   │                                                      │   │ │
│   │   │  주요 표준:                                              │   │ │
│   │   │  ┌────────────────────────────────────────┐           │   │ │
│   │   │  │ 표준        │ 속도   │ 매체    │ 거리  │           │   │ │
│   │   │  ├────────────────────────────────────────┤           │   │ │
│   │   │  │ 10BASE-T   │ 10M    │ UTP     │ 100m  │           │   │ │
│   │   │  │ 100BASE-TX │ 100M   │ UTP     │ 100m  │           │   │ │
│   │   │  │ 1000BASE-T │ 1G     │ UTP     │ 100m  │           │   │ │
│   │   │  │ 10GBASE-T  │ 10G    │ UTP     │ 100m  │           │   │ │
│   │   │  │ 10GBASE-SR │ 10G    │ MMF     │ 400m  │           │   │ │
│   │   │  │ 10GBASE-LR │ 10G    │ SMF     │ 10km  │           │   │ │
│   │   │  │ 40GBASE-T  │ 40G    │ Cat8    │ 30m   │           │   │ │
│   │   │  │ 100GBASE-LR│ 100G   │ SMF     │ 10km  │           │   │ │
│   │   │  └────────────────────────────────────────┘           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PHY 계층 구조                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  RS (Reconciliation Sublayer)                          │   │ │
│   │   │           │                                             │   │ │
│   │   │  ┌────────┴────────┐                                   │   │ │
│   │   │  │ PCS │ PMA │ PMD │  PHY (Physical Coding Sublayer) │   │ │
│   │   │  │(코딩)│(연결)│(매체)│                                │   │ │
│   │   │  └────────┴────────┘                                   │   │ │
│   │   │           │                                             │   │ │
│   │   │       MDI (Medium Dependent Interface)                 │   │ │
│   │   │           │                                             │   │ │
│   │   │       전송 매체 (UTP, 광섬유 등)                         │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: PHY 표준은 속도와 매체에 따라 다양한 규격이 있습니다.

**💡 비유**: PHY는 자동차의 엔진과 타이어 규격과 같습니다.

**📢 섹션 요약 비유**: BASE = 기저대역!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **PCS** | 코딩 | 64b/66b | 변환기 |
| **PMA** | 연결 | 직렬화 | 변속기 |
| **PMD** | 매체 | 전송 | 타이어 |
| **MDI** | 인터페이스 | 커넥터 | 휠 |

**비교표**

```
+------------+----------+----------+
| 항목       | 100M     | 1G       |
+------------+----------+----------+
| 코딩       | 4B/5B    | 8B/10B   |
| 라인속도   | 125Mbps  | 1.25Gbps |
| 페어       | 2        | 4        |
+------------+----------+----------+
```

**핵심 알고리즸**: 라인 코딩,Auto-Negotiation

**📢 섹션 요약 비유**: 1G = 8B/10B!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 매체별 표준**

| 매체 | 1G | 10G |
|:---|:---|:---|
| **UTP** | 1000BASE-T | 10GBASE-T |
| **MMF** | 1000BASE-SX | 10GBASE-SR |
| **SMF** | 1000BASE-LX | 10GBASE-LR |

**📢 섹션 요약 비유**: T=UTP, S=단파장, L=장파장!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데스크톱** → 1000BASE-T
**시나리오 2: 서버** → 10GBASE-T
**시나리오 3: 백본** → 100GBASE-LR

**도입 체크리스트**: 속도, 거리, 매체, 비용

**📢 섹션 요약 비유**: PHY 선택은 속도와 거리!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 호환성, 상호 운용성

**미래 전망**: 800G/1.6T 이더넷

**참고 표준**: IEEE 802.3-2022

**📢 섹션 요약 비유**: PHY는 이더넷의 물리적 기반!
