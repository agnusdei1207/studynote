# 2G (GSM/GPRS/EDGE)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디지털 이동통신, 패킷 데이터
> 2. **가치**: 음성 + 데이터 동시
> 3. **융합**: SMS, MMS, 인터넷과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

2G(2nd Generation)는 디지털 이동통신으로, GSM을 기반으로 음성과 데이터 서비스를 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    2G 기술별                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  기술         │ 전송     │ 속도       │ 서비스     │   │
│   │  ───────────┼──────────┼───────────┼───────────│   │
│   │  GSM         │ 회선/패킷│ 9.6-14.4kbps│ 음성/데이터 │   │
│   │  GPRS        │ 패킷      │ 171kbps    │ 인터넷    │   │
│   │  EDGE        │ 패킷      │ 384kbps    │ 인터넷    │   │
│   │  EDGE Evolution│ 패킷    │ 1Mbps      │ 인터넷    │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              2G 특성                                      │   │
│   │                                                              │   │
│   │  장점:                                                   │   │
│   │  • 디지털 음성 품질 향상                                │   │
│   │  • SMS/MMS 지원                                         │   │
│   │  • 패킷 데이터 (GPRS/EDGE)                              │   │
│   │  • 글로벌 로밍                                          │   │
│   │  • 넓은 커버리지                                        │   │
│   │                                                      │   │
│   │  단점:                                                   │   │
│   │  • 낮은 데이터 속도                                     │   │
│   │  • 멀티미디어 제한                                      │   │
```

> **해설**: 2G는 디지털 이동통신의 시작으로 데이터 서비스를 가능하게 했습니다.

**💡 비유**: 2G는 전화기에서 문자 메시지를 보낼 수 있게 된 것입니다.

**📢 섹션 요약 비유:** GPRS = 2.5G!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 기술 | 기반 | 변조 | 속도 |
|:---|:---|:---|:---|
| **GSM** | TDMA | GMSK | 14.4kbps |
| **GPRS** | GSM | GMSK | 171kbps |
| **EDGE** | GSM | 8PSK | 384kbps |

**비교표**

```
+------------+----------+----------+
| 항목       │ GSM      │ GPRS     │
+------------+----------+----------+
| 전송       │ 회선     │ 패킷     │
│ 과금       │ 시간     │ 데이터   │
│ 항상연결   │ 아니오   │ 예       │
+------------+----------+----------+
```

**핵심 알고리즸:** GMSK 변조, 시간 슬롯, 주파수 홉핑

**📢 섹션 요약 비유:** EDGE = 3G 못가요!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 네트워크 구조**

| 요소 | 설명 |
|:---|:---|
| **HLR** | 홈 위치 등록기 |
| **VLR** | 방문 위치 등록기 |
| **MSC** | 이동 교환국 |
| **BSC** | 기지국 제어기 |
| **BTS** | 기지국 송수신기 |

**📢 섹션 요약 비유:** GSM = 글로벌 표준!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 음성 위주** → GSM
**시나리오 2: 데이터** → EDGE
**시나리오 3: M2M/IoT** → GPRS (저전력)

**도입 체크리스트:** 커버리지, 단말기, 서비스

**📢 섹션 요약 비유:** 2G는 신뢰성!
