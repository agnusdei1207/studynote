# OpenFlow - 오픈플로우

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SDN 남향 인터페이스
> 2. **가치**: 프로그래밍 가능한 스위치
> 3. **융합:** Flow Table, Match, Action

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OpenFlow는 SDN 컨트롤러와 스위치 간 통신 프로토콜입니다.

```

│   │   │                                                      │   │ │

│   │   │  │          SDN Controller                          ││   │ │
```

> **해설**: OpenFlow는 스위치 동작을 프로그래밍합니다.

**📢 섹션 요약 비유:** OpenFlow = 스위치의 API!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Flow Table** | 플로우 테이블 |
| **Match Fields** | 매치 필드 |
| **Actions** | 액션 |
| **Counters** | 카운터 |
| **Group Table** | 그룹 테이블 |

**핵심 알고리듬:** 패킷 수신 → 매치 → 액션 실행 → 카운터 갱신

**OpenFlow 버전**
