### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **IoC** | 침입 지표 (IP, 해시, 도메인) |
| **TTP** | 전술, 기법, 절차 |
| **APT** | 지능형 지속 위협 |
| **STIX/TAXII** | 표준 형식/프로토콜 |

**비교표: TI 소스**

```
+------------+----------+----------+
| 항목       │ OSINT    │ 상용     │
+------------+----------+----------+
| 비용       │ 무료     │ 유료     │
| 품질       │ 변동     │ 일관     │
| 신속성     │ 빠름     │ 중간     │
| 깊이       │ 얕음     │ 깊음     │
+------------+----------+----------+
```

**핵심 알고리즸:** IoC 매칭, 행동 분석

**📢 섹션 요약 비유:** IoC = 위협 지문!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
