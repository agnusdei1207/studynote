# GPGPU (General Purpose GPU) - GPGPU

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 범용 GPU 컴퓨팅
> 2. **가치**: 대규모 병렬 처리
> 3. **융합:** CUDA, OpenCL, Deep Learning

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

GPGPU는 그래픽스 외 목적으로 GPU를 활용합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    GPGPU                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              GPGPU 발전                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  전통적 GPU:              GPGPU:                     │   │ │
│   │   │  ┌─────────────────┐      ┌───────────────────────┐ │   │ │
│   │   │  │ Graphics Only   │      │ General Purpose       │ │   │ │
│   │   │  │ - Rendering     │      │ - Scientific Calc     │ │   │ │
│   │   │  │ - 3D Graphics   │  →   │ - Machine Learning    │ │   │ │
│   │   │  │ - Video         │      │ - Cryptography        │ │   │ │
│   │   │  │                 │      │ - Data Processing     │ │   │ │
│   │   │  └─────────────────┘      └───────────────────────┘ │   │ │
│   │   │                                                      │   │ │
│   │   │  GPGPU 플랫폼:                                        │   │ │
│   │   │  - CUDA (NVIDIA)                                     │   │ │
│   │   │  - OpenCL (Khronos)                                  │   │ │
│   │   │  - ROCm (AMD)                                        │   │ │
│   │   │  - Metal (Apple)                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: GPGPU는 GPU의 방대한 연산 능력을 범용으로 활용합니다.

**📢 섹션 요약 비유:** GPGPU = GPU의 재발견!
