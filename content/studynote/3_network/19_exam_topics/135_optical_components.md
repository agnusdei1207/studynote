# 광전송 용어 - 광원 / 수광소자

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전기→빛 변환(광원), 빛→전기 변환(수광)
> 2. **가치**: 광통신의 송수신 핵심
> 3. **융합**: 이더넷, WDM, FTTH와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

광원(Light Source)은 전기 신호를 빛으로 변환하고, 수광소자(Photodetector)는 빛을 전기 신호로 변환합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    광원 / 수광소자                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              광원 (Light Source)                              │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. LED (Light Emitting Diode):                         │   │ │
│   │   │     • 저비용                                              │   │ │
│   │   │     • 넓은 스펙트럼 (~100nm)                             │   │ │
│   │   │     • MMF용 (850nm, 1300nm)                             │   │ │
│   │   │     • 단거리에 적합                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  2. VCSEL (Vertical Cavity Surface Emitting Laser):     │   │ │
│   │   │     • 표면 방출 레이저                                    │   │ │
│   │   │     • 좁은 스펙트럼                                       │   │ │
│   │   │     • MMF용 (850nm)                                      │   │ │
│   │   │     • 저비용, 고효율                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  3. LD (Laser Diode):                                    │   │ │
│   │   │     • 매우 좁은 스펙트럼 (<1nm)                           │   │ │
│   │   │     • SMF용 (1310nm, 1550nm)                            │   │ │
│   │   │     • 장거리에 적합                                       │   │ │
│   │   │     • 고비용                                              │   │ │
│   │   │     • DFB-LD, DBR-LD (단일 파장)                         │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              수광소자 (Photodetector)                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. PIN 다이오드:                                        │   │ │
│   │   │     • P-I-N 구조                                          │   │ │
│   │   │     • 빠른 응답 (수 GHz)                                  │   │ │
│   │   │     • 낮은 비용                                           │   │ │
│   │   │     • 일반적인 용도                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  2. APD (Avalanche Photodiode):                          │   │ │
│   │   │     • 눈사태 이득 (10~100배)                              │   │ │
│   │   │     • 높은 감도                                           │   │ │
│   │   │     • 장거리/저신호에 적합                                 │   │ │
│   │   │     • 높은 비용                                           │   │ │
│   │   │     • 높은 전압 필요                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 광원과 수광소자는 광통신 시스템의 핵심 소자입니다.

**💡 비유**: 광원은 마이크(전기→음향), 수광소자는 스피커(음향→전기) 역할입니다.

**📢 섹션 요약 비유**: LED = MMF, LD = SMF!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **LED** | 저가 광원 | 자연 방출 | 전구 |
| **LD** | 고성능 광원 | 유도 방출 | 레이저 |
| **PIN** | 수광 | 광전 효과 | 태양전지 |
| **APD** | 고감도 수광 | 눈사태 이득 | 증폭기 |

**비교표**

```
+------------+----------+----------+
| 항목       | LED      | LD       |
+------------+----------+----------+
| 스펙트럼   | 넓음     | 좁음     |
| 거리       | 단거리   | 장거리   |
| 비용       | 낮음     | 높음     |
| 파장       | MMF      | SMF      |
+------------+----------+----------+
```

**핵심 알고리즸**: 광전 효과, 여기-방출

**📢 섹션 요약 비유**: APD = 높은 감도!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 파장 대역**

| 대역 | 파장 | 광원 |
|:---|:---:|:---|
| **850nm** | MMF | LED/VCSEL |
| **1310nm** | SMF/MMF | LD |
| **1550nm** | SMF | LD |

**📢 섹션 요약 비유**: 850nm = MMF!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: MMF 단거리** → LED/VCSEL + PIN
**시나리오 2: SMF 장거리** → LD + APD
**시나리오 3: DWDM** → DFB-LD + PIN

**도입 체크리스트**: 거리, 속도, 예산, 파장

**📢 섹션 요약 비유**: 거리/속도가 광원 결정!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 광통신 시스템 구현

**미래 전망**: 실리존 포토닉스, 집소화

**참고 표준**: IEEE 802.3, ITU-T G.957

**📢 섹션 요약 비유**: 광원/수광은 광통신의 눈과 귀!
