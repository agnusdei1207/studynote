# TDMA (시분할 다중 접속)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 각 사용자에게 시간 슬롯을 할당하여 전송
> 2. **가치**: 주파수 효율, 디지털, 2G 기반
> 3. **융합**: GSM, IS-136, 위성과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

TDMA(Time Division Multiple Access)는 하나의 주파수 채널을 시간 슬롯으로 나누어 여러 사용자가 교대로 전송하는 다중 접속 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TDMA (시분할 다중 접속)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TDMA 원리                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  시간 슬롯 구조:                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  시간                                                 │   │ │
│   │   │    ├────┼────┼────┼────┼────┼────┼────┼────┤       │   │ │
│   │   │    │U1  │U2  │U3  │U4  │U5  │U6  │U7  │U8  │       │   │ │
│   │   │    │TS0 │TS1 │TS2 │TS3 │TS4 │TS5 │TS6 │TS7 │       │   │ │
│   │   │    └────┴────┴────┴────┴────┴────┴────┴────┘       │   │ │
│   │   │         │←─────── TDMA 프레임 ────────→│            │   │ │
│   │   │                                                      │   │ │
│   │   │  GSM 예시:                                             │   │ │
│   │   │  • 주파수 채널: 200kHz                                │   │ │
│   │   │  • 슬롯/프레임: 8                                     │   │ │
│   │   │  • 슬롯 길이: 0.577ms                                 │   │ │
│   │   │  • 프레임 길이: 4.615ms                               │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • 한 주파수에 8명 동시 사용                           │   │ │
│   │   │  • 버스트 전송                                        │   │ │
│   │   │  • 정밀 동기 필요                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TDMA vs FDMA                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  FDMA: 1 주파수 = 1 사용자                             │   │ │
│   │   │  TDMA: 1 주파수 = N 사용자 (슬롯 분할)                  │   │ │
│   │   │                                                      │   │ │
│   │   │  주파수 효율:                                          │   │ │
│   │   │  FDMA: 30kHz/사용자                                   │   │ │
│   │   │  TDMA (GSM): 200kHz/8 = 25kHz/사용자                  │   │ │
│   │   │  → TDMA가 더 효율적!                                   │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: TDMA는 GSM(2G)의 핵심 기술입니다.

**💡 비유**: TDMA는 회의에서 발언권을 돌아가며 갖는 것입니다.

**📢 섹션 요약 비유**: TDMA = 시간 슬롯 할당!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **타임슬롯** | 전송 단위 | 0.577ms (GSM) | 발언권 |
| **프레임** | 슬롯 세트 | 8슬롯 (GSM) | 라운드 |
| **동기** | 정렬 | 타이밍 어드밴스 | 박자 |
| **버스트** | 전송 형태 | 짧은 펄스 | 외침 |

**비교표**

```
+------------+----------+----------+
| 항목       | GSM      | IS-136   |
+------------+----------+----------+
| 채널 폭    | 200kHz   | 30kHz    |
| 슬롯/프레임| 8        | 3        |
| 슬롯 길이  | 0.577ms  | 6.67ms   |
| 비트/슬롯  | 156.25   | 260      |
+------------+----------+----------+
```

**핵심 알고리즸**: 슬롯 할당 → 버스트 전송 → 동기 유지 → 수신 분리

**📢 섹션 요약 비유**: GSM = 8 슬롯 TDMA!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: TDMA 장단점**

| 장점 | 단점 |
|:---|:---|
| 주파수 효율 | 동기 복잡 |
| 디지털 친화적 | 버퍼링 필요 |
| 유연한 할당 | 지연 존재 |

**📢 섹션 요약 비유**: TDMA는 디지털 시대의 시작!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: GSM** → FDMA + TDMA
**시나리오 2: IS-136** → TDMA
**시나리오 3: 위성** → TDMA

**도입 체크리스트**: 동기 정밀도, 지연 허용, 슬롯 수

**📢 섹션 요약 비유**: GSM은 FDMA+TDMA!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: FDMA 대비 8배 효율 (GSM)

**미래 전망**: OFDMA, SC-FDMA로 진화

**참고 표준**: GSM, IS-136

**📢 섹션 요약 비유**: TDMA는 2G의 핵심!
