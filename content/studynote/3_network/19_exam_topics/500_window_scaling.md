# 윈도우 스케일링 - Window Scaling

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: TCP 윈도우 크기 확장
> 2. **가치**: 고속 네트워크 활용
> 3. **융합:** TCP Options, BDP, RFC 1323

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

윈도우 스케일링은 16비트 윈도우를 확장합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    윈도우 스케일링                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              윈도우 크기 확장                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  기본: 16bit = 65,535 bytes                         │   │ │
│   │   │                                                      │   │ │
│   │   │  스케일링: 16bit × 2^scale                          │   │ │
│   │   │                                                      │   │ │
│   │   │  예: scale=7 → 65,535 × 128 = 8,388,480 bytes      │   │ │
│   │   │                                                      │   │ │
│   │   │  최대: scale=14 → 약 1GB                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 윈도우 스케일링은 TCP 옵션으로 협상됩니다.

**📢 섹션 요약 비유:** Window Scaling = 윈도우 확장!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Scale Factor** | 스케일 팩터 (0-14) |
| **TCP Option** | 옵션 3 |
| **SYN** | 연결 설정 시 협상 |
| **Max Window** | 최대 윈도우 |

**핵심 알고리즸:** shift 연산

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**스케일 팩터별 윈도우**

| Scale | 배수 | 최대 윈도우 |
|:---|:---|:---|
| **0** | 1 | 64KB |
| **7** | 128 | 8MB |
| **14** | 16384 | 1GB |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 1Gbps LAN → scale=7
**시나리오 2:** 10Gbps WAN → scale=14
**시나리오 3:** 레거시 → scale=0
