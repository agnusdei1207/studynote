# Consul - 컨설

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서비스 메시 및 디스커버리
> 2. **가치**: 서비스 검색, 설정 관리
> 3. **융합:** Service Mesh, KV Store, Health Check

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Consul은 서비스 디스커버리와 설정을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Consul                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Consul 아키텍처                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Consul Servers                                      │   │ │
│   │   │  ┌─────────┐ ┌─────────┐ ┌─────────┐               │   │ │
│   │   │  │ Leader  │ │Follower │ │Follower │               │   │ │
│   │   │  │ (RAFT)  │ │         │ │         │               │   │ │
│   │   │  └─────────┘ └─────────┘ └─────────┘               │   │ │
│   │   │       │                                       │   │ │
│   │   │       │ Gossip Protocol                       │   │ │
│   │   │       ↓                                       │   │ │
│   │   │  Consul Agents                                      │   │ │
│   │   │  ┌─────────┐ ┌─────────┐ ┌─────────┐            │   │ │
│   │   │  │ Service │ │ Service │ │ Service │            │   │ │
│   │   │  │    A    │ │    B    │ │    C    │            │   │ │
│   │   │  └─────────┘ └─────────┘ └─────────┘            │   │ │
│   │   │                                                      │   │ │
│   │   │  기능:                                                │   │ │
│   │   │  - Service Discovery                                 │   │ │
│   │   │  - Health Checking                                   │   │ │
│   │   │  - KV Store                                          │   │ │
│   │   │  - Service Mesh (Connect)                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Consul은 서비스 검색과 설정을 통합합니다.

**📢 섹션 요약 비유:** Consul = 동네 게시판!
