# Normalization - 정규화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 중복 제거
> 2. **가치**: 무결성, 저장 공간
> 3. **융합:** Database Design, FD, Normal Form

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Normalization은 데이터 중복을 최소화합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Normalization                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              정규화 단계                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1NF (제1정규형): 원자성                              │   │ │
│   │   │  - 모든 컬럼이 원자값                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  2NF (제2정규형): 부분 함수 종속 제거                 │   │ │
│   │   │  - 복합키의 일부에 종속된 속성 제거                   │   │ │
│   │   │                                                      │   │ │
│   │   │  3NF (제3정규형): 이행 함수 종속 제거                 │   │ │
│   │   │  - 비키 속성에 종속된 속성 제거                       │   │ │
│   │   │                                                      │   │ │
│   │   │  BCNF (보이스-코드 정규형):                           │   │ │
│   │   │  - 모든 결정자가 후보키                               │   │ │
│   │   │                                                      │   │ │
│   │   │  4NF, 5NF: 다중값 종속, 조인 종속                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 정규화는 이상 현상을 방지합니다.

**📢 섹션 요약 비유:** Normalization = 정리 정돈!
