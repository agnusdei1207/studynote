# WAP (무선 애플리케이션 프로토콜) - Wireless Application Protocol

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 무선 인터넷 접속 프로토콜
> 2. **가치**: 초기 모바일 웹
> 3. **융합**: WML, WAP Gateway

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

WAP은 초기 무선 인터넷 접속 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    WAP 아키텍처                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              WAP 스택                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ WAE (Wireless Application Environment)     │    │   │ │
│   │   │  │ WML, WMLScript                             │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ WSP (Wireless Session Protocol)            │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ WTP (Wireless Transaction Protocol)        │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ WTLS (Wireless TLS)                        │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ WDP (Wireless Datagram Protocol)           │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              WAP Gateway                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 구성       │ 설명     │ 역할     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ WAP Client │ 단말     │ WML      │               │   │ │
│   │   │  │ WAP Gateway│ 변환     │ WML↔HTML │               │   │ │
│   │   │  | Web Server │ 서버     │ HTML     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: WAP은 유선 웹을 무선으로 연결했습니다.

**💡 비유**: WAP은 모바일 웹의 시초입니다.

**📢 섹션 요약 비유:** WAP = 초기 모바일 웹!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **WML** | Wireless Markup Language |
| **WAP Gateway** | 프로토콜 변환 |
| **WTLS** | 보안 |
| **WDP** | 데이터그램 |

**비교표: WAP vs WWW**

```
+------------+----------+----------+----------+
| 항목       │ WWW      │ WAP      │ 비고     │
+------------+----------+----------+----------+
| 마크업     │ HTML     │ WML      │ 언어     │
| 스크립트   │ JS       │ WMLScript│ 스크립트 │
| 보안       │ TLS      │ WTLS     │ 암호화   │
| 대역폭     │ 높음     │ 낮음     │ 무선     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 코덱 변환, 압축

**📢 섹션 요약 비유:** WML = 무선 HTML!
