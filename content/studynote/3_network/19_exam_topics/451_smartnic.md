# SmartNIC (스마트 네트워크 인터페이스 카드) - Smart Network Interface Card

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로그래밍 가능한 네트워크 카드
> 2. **가치**: CPU 오프로드, 성능 향상
> 3. **융합**: FPGA, ASIC, P4

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **FPGA** | 프로그래밍 가능 로직 |
| **ASIC** | 고정 기능 가속 |
| **ARM Core** | 관리 프로세서 |
| **Memory** | 온보드 메모리 |

**비교표: SmartNIC 유형**

```
+------------+----------+----------+----------+
| 유형       │ 유연성   │ 성능     │ 비용     │

| FPGA       │ 높음     │ 중간     │ 높음     │
| ASIC       │ 낮음     │ 높음     │ 중간     │
| SoC        │ 중간     │ 중간     │ 낮음     │
+------------+----------+----------+----------+
```

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** OVS 오프로드 → Mellanox
