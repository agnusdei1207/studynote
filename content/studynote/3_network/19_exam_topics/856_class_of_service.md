# Class of Service - 서비스 클래스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 트래픽 분류
> 2. **가치**: 우선순위 관리
> 3. **융합:** CoS, DSCP, 802.1Q

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Class of Service (CoS)는 트래픽을 클래스별로 분류합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Class of Service (CoS)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CoS 구현                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Ethernet 802.1Q (Layer 2):                            │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │  Tag: [TPID][TCI] = [0x8100][PRI|CFI|VID]      ││   │ │
│   │   │  │                    ↑                            ││   │ │
│   │   │  │              3-bit PRI (0-7)                    ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │  - 8개 클래스 (0=최저, 7=최고)                         │   │ │
│   │   │                                                      │   │ │
│   │   │  IP DSCP (Layer 3):                                   │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Class  DSCP    이름        설명                  ││   │ │
│   │   │  │ ────────────────────────────────────────────    ││   │ │
│   │   │  │ EF     46      Expedite    음성                 ││   │ │
│   │   │  │ AF41   34      Assured     화상회의             ││   │ │
│   │   │  │ AF31   26      Assured     중요 데이터          ││   │ │
│   │   │  │ AF21   18      Assured     일반 데이터          ││   │ │
│   │   │  │ BE     0       Best Effort 일반                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  CoS vs QoS:                                           │   │ │
│   │   │  - CoS: 분류/마킹 (Layer 2/3)                          │   │ │
│   │   │  - QoS: 종합적 서비스 품질                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: CoS는 L2/L3에서 트래픽 식별과 우선순위를 부여합니다.

**📢 섹션 요약 비유:** CoS = 비즈니스 클래스!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **PRI** | 우선순위 (802.1Q) |
| **DSCP** | DiffServ Code Point |
| **EF** | Expedited Forwarding |
| **AF** | Assured Forwarding |
| **BE** | Best Effort |

**핵심 알고리즸:** 분류 → 마킹 → 큐 할당

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**L2 vs L3 CoS**

| 계층 | 기술 | 비트 |
|:---|:---|:---|
| **L2** | 802.1Q PRI | 3 bit (8 클래스) |
| **L3** | DSCP | 6 bit (64 클래스) |
| **MPLS** | EXP | 3 bit (8 클래스) |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** VoIP → PRI 7 / DSCP EF
**시나리오 2:** 비디오 → PRI 5 / DSCP AF41
**시나리오 3:** 데이터 → PRI 0 / DSCP BE
