# OpenCL (Open Computing Language) - OpenCL

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이기종 병렬 컴퓨팅 프레임워크
> 2. **가치**: 플랫폼 독립적 GPGPU
> 3. **융합:** GPU, CPU, FPGA

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

OpenCL은 다양한 플랫폼에서 병렬 컴퓨팅을 제공합니다.

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Platform** | 플랫폼 |
| **Device** | 장치 |
| **Context** | 컨텍스트 |
| **Command Queue** | 명령 큐 |
| **Kernel** | 커널 |

**핵심 알고리즸:** 컨텍스트 생성 → 큐 생성 → 커널 실행

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**OpenCL vs CUDA**

| 항목 | OpenCL | CUDA |
|:---|:---|:---|
| **플랫폼** | 다양 | NVIDIA만 |
| **성능** | 보통 | 최적화 |
| **학습 곡선** | 높음 | 낮음 |
| **생태계** | 작음 | 큼 |

**시나리오 3:** FPGA → OpenCL
