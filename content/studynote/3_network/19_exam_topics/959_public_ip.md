# Public IP - 공인 IP

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 인터넷 고유 주소
> 2. **가치**: 전역 라우팅
> 3. **융합:** ISP, IANA, Global Unicast

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **IANA** | 국제 관리 |
| **RIR** | 지역 등록기관 |
| **ISP** | 인터넷 제공자 |
| **Assignment** | 할당 |
| **Allocation** | 배정 |

**핵심 알고리듬:** IANA → RIR → ISP → 사용자

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**IP 주소 유형**
