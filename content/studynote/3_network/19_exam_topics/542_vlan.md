# VLAN (Virtual LAN) - 가상 LAN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 논리적 네트워크 분리
> 2. **가치**: 브로드캐스트 도메인 분리
> 3. **융합:** Tagging, Trunk, Access

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VLAN은 물리적 배치와 무관하게 논리적 네트워크를 구성합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VLAN                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VLAN 구성                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Port 1-4:  VLAN 10 (영업팀)                         │   │ │
│   │   │  Port 5-8:  VLAN 20 (개발팀)                         │   │ │
│   │   │  Port 9-12: VLAN 30 (인사팀)                         │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │              Switch                         │    │   │ │
│   │   │  │  VLAN10 │ VLAN20 │ VLAN30 │ Trunk          │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  → 서로 다른 VLAN은 라우터 필요                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: VLAN은 브로드캐스트 도메인을 논리적으로 분리합니다.

**📢 섹션 요약 비유:** VLAN = 방 나누기!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VLAN ID** | 1-4094 |
| **Access Port** | 단일 VLAN |
| **Trunk Port** | 다중 VLAN |
| **802.1Q** | VLAN 태깅 표준 |
| **Native VLAN** | 태그 없는 프레임 |

**핵심 알고리즸:** Tag 추가/제거 → VLAN 구분

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**VLAN 타입**

| 타입 | 설명 |
|:---|:---|
| **Port-based** | 포트 기반 |
| **Tag-based** | 802.1Q 태그 |
| **Protocol-based** | 프로토콜 기반 |
| **MAC-based** | MAC 기반 |
