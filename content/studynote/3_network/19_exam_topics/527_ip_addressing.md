# IP Addressing - IP 주소 체계

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 장치 식별자
> 2. **가치**: 계층적 라우팅
> 3. **융합:** Subnet, CIDR, Class

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IP 주소는 네트워크에서 장치를 식별합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IP Addressing                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IPv4 주소 구조                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  192.168.1.100                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  32비트 = 4옥텟 = 4바이트                            │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────┬─────────┬─────────┬─────────┐          │   │ │
│   │   │  │  192    │   168   │    1    │   100   │          │   │ │
│   │   │  │11000000 │10101000 │00000001 │01100100 │          │   │ │
│   │   │  └─────────┴─────────┴─────────┴─────────┘          │   │ │
│   │   │     ↑                ↑            ↑                  │   │ │
│   │   │   네트워크 ID                   호스트 ID             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IP 주소는 네트워크 ID와 호스트 ID로 구성됩니다.

**📢 섹션 요약 비유:** IP 주소 = 집 주소!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 클래스 | 범위 | 네트워크 | 호스트 |
|:---|:---|:---|:---|
| **A** | 1-126 | 8bit | 24bit |
| **B** | 128-191 | 16bit | 16bit |
| **C** | 192-223 | 24bit | 8bit |
| **D** | 224-239 | 멀티캐스트 | - |
| **E** | 240-255 | 예약 | - |

**핵심 알고리즸:** 클래스/서브넷 마스크로 구분

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**특수 IP 주소**

| 주소 | 용도 |
|:---|:---|
| **0.0.0.0** | 기본 경로 |
| **127.0.0.1** | 루프백 |
| **255.255.255.255** | 브로드캐스트 |
| **169.254.x.x** | APIPA |
