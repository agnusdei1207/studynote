# 802.1X - 포트 기반 네트워크 액세스 제어

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 포트 기반 인증 프로토콜
> 2. **가치**: 무단 접속 차단, NAC
> 3. **융합**: RADIUS, WiFi, 유선

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

802.1X는 네트워크 포트에 대한 인증을 수행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    802.1X                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              802.1X 구성요소                                   │ │

│   │   │  [Supplicant]                                       │   │ │
│   │   │     │                                                │   │ │

│   │   │     ▼                                                │   │ │

│   │   │     │                      ▼                         │   │ │

│   │              802.1X 용어                                       │ │

│   │   │  +------------+----------+----------+               │   │ │
```

> **해설**: 802.1X는 인증 전까지 포트를 차단합니다.

**💡 비유**: 802.1X는 출입문입니다. 인증해야 들어갑니다.

**📢 섹션 요약 비유:** 802.1X = 포트 인증!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Supplicant** | 인증 요청 클라이언트 |
| **Authenticator** | 인증 장비 (스위치/AP) |
| **Auth Server** | 인증 서버 (RADIUS) |
| **EAP** | Extensible Authentication Protocol |

**비교표: EAP 방식**

```
+------------+----------+----------+
| 항목       │ EAP-TLS  │ PEAP     │
+------------+----------+----------+
| 인증서     │ 클라이언트│ 서버만   │
| 보안       │ 높음     │ 중간     │
| 복잡도     │ 높음     │ 낮음     │
| 배포       │ 어려움   │ 쉬움     │
+------------+----------+----------+
```

**핵심 알고리즸:** EAPOL 프레임, RADIUS 통신

**📢 섹션 요약 비유:** NAC = 802.1X!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 네트워크 인증**

| 항목 | 802.1X | MAC Filtering | Port Security |
|:---|:---|:---|:---|
| **보안** │ 높음 │ 낮음 │ 중간 |
| **인증** │ 사용자 | MAC | MAC |
| **관리** │ 중앙 │ 분산 | 로컬 |

**📢 섹션 요약 비유:** WiFi = 802.1X!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 기업 WiFi → 802.1X + RADIUS
**시나리오 2:** 유선 네트워크 → 802.1X
**시나리오 3:** IoT → MAC Filtering

**도입 체크리스트:** RADIUS 구성, EAP 방식, 인증서

**📢 섹션 요약 비유:** 기업 보안 = 802.1X!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 무단 접속 차단

**미래 전망:** NAC와 통합

**참고 표준:** IEEE 802.1X-2010

**📢 섹션 요약 비유:** 802.1X는 NAC 표준!
