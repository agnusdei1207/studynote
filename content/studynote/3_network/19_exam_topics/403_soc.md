# SOC - 보안 운영 센터

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 보안 모니터링 및 대응 조직
> 2. **가치**: 24/7 위협 감시
> 3. **융합**: SIEM, SOAR, 위협 인텔리전스

---

### Ⅰ. 개요 (Context & Background)

**💡 비유**: SOC는 119 상황실입니다. 항상 감시하고 출동합니다.

**📢 섹션 요약 비유:** SOC = 보안 관제!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **People** | 분석가, 관리자 |
| **Process** | 절차, 플레이북 |
| **Technology** | SIEM, SOAR, EDR |
| **Intelligence** | 위협 인텔리전스 |

**비교표: SOC vs CSIRT**

```
+------------+----------+----------+
| 항목       │ SOC      │ CSIRT    │
+------------+----------+----------+
| 초점       │ 모니터링 │ 사고대응 │
| 시점       │ 예방/탐지│ 사후대응 │
| 운영       │ 상시     │ 필요시   │
| 조직       │ 운영     │ 대응     │
+------------+----------+----------+
```

**핵심 알고리즸:** 알림 분류, 사고 분석

**📢 섹션 요약 비유:** 24/7 = SOC!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: SOC 도구**

| 항목 | SIEM | SOAR | TIP |
|:---|:---|:---|:---|
| **용도** │ 분석 │ 자동화 │ 인텔 |
| **필요성** │ 필수 | 권장 | 권장 |
