# 비대칭키 암호 - Asymmetric Encryption

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 공개키/개인키 쌍 사용
> 2. **가치**: 키 분배 문제 해결

### Ⅰ. 개요 (Context & Background)

**개념 정의**

비대칭키 암호는 공개키와 개인키 쌍을 사용합니다.

```

├─────────────────────────────────────────────────────────────────────────┤
```

> **해설**: 비대칭키 암호는 키 분배 문제를 해결합니다.

**💡 비유**: 비대칭키는 우편함입니다. 누구나 넣을 수 있지만 주인만 열 수 있습니다.

**📢 섹션 요약 비유:** 비대칭키 = 다른 키!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Public Key** | 공개키 (암호화용) |
| **Private Key** | 개인키 (복호화용) |
| **Key Pair** | 키 쌍 |
| **Key Exchange** | 키 교환 |

**비교표: RSA vs ECC**

```
+------------+----------+----------+
| 항목       │ RSA      │ ECC      │
+------------+----------+----------+
| 키 크기    │ 2048+    │ 256      │
| 속도       │ 느림     │ 빠름     │
| 서명 크기   │ 큼       │ 작음     │
| 추세       │ 레거시   │ 현대적   │
+------------+----------+----------+
```

**핵심 알고리즸:** 모듈러 거듭제곱, 타원곡선 연산

**📢 섹션 요약 비유:** 키 교환 = 비대칭키!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 용도**

| 항목 | 암호화 | 서명 | 키 교환 |
|:---|:---|:---|:---|
| **RSA** │ 가능 │ 가능 │ 가능 |
| **ECDSA** │ 불가 │ 가능 │ 불가 |
| **ECDH** │ 불가 │ 불가 │ 가능 |

**📢 섹션 요약 비유:** TLS = 비대칭키 + 대칭키!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** TLS 키 교환 → ECDHE
**시나리오 2:** 디지털 서명 → RSA/ECDSA
**시나리오 3:** SSH 인증 → Ed25519

**도입 체크리스트:** 알고리즘 선택, 키 크기, 키 관리

**📢 섹션 요약 비유:** 보안 = 비대칭키!
