# Firewall - 방화벽

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 트래픽 필터링
> 2. **가치**: 접근 제어, 보안
> 3. **융합:** ACL, Stateful, NGFW

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Firewall은 네트워크 경계에서 트래픽을 필터링합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Firewall                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Firewall 배치                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  인터넷                                              │   │ │
│   │   │      │                                               │   │ │
│   │   │      ↓                                               │   │ │
│   │   │  ┌─────────┐                                        │   │ │
│   │   │  │Firewall │ ← Allow/Deny                           │   │ │
│   │   │  └─────────┘                                        │   │ │
│   │   │      │                                               │   │ │
│   │   │      ↓                                               │   │ │
│   │   │  내부망 (192.168.x.x)                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Firewall은 정책에 따라 트래픽을 허용/차단합니다.

**📢 섹션 요약 비유:** Firewall = 경비실!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 유형 | 설명 |
|:---|:---|
| **Packet Filter** | 패킷 필터링 |
| **Stateful** | 상태 추적 |
| **Application** | L7 필터링 |
| **NGFW** | 차세대 방화벽 |
| **WAF** | 웹 방화벽 |

**핵심 알고리즸:** Rule 매칭 → Allow/Deny
