# VoLTE (Voice over LTE) - LTE 음성

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: LTE 네트워크 기반 음성 통화
> 2. **가치**: HD 음질, 빠른 연결, 동시 데이터
> 3. **융합**: IMS, SIP, SRVCC

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VoLTE는 LTE 네트워크로 HD 음성 통화를 제공합니다.

> **해설**: VoLTE는 IMS 기반으로 HD 음성을 제공합니다.

**💡 비유**: VoLTE는 LTE 위에서 도는 통화 앱입니다.

**📢 섹션 요약 비유:** VoLTE = HD 음성!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **IMS** | IP Multimedia Subsystem |
| **SIP** | Session Initiation Protocol |
| **RTP** | Real-time Transport Protocol |
| **QCI=1** | 음성 QoS |

**비교표: 코덱 비교**

```
+------------+----------+----------+----------+
| 코덱       │ 대역폭   │ 품질     │ 용도     │
+------------+----------+----------+----------+
| AMR-NB     │ 12.2kbps │ 4.0 MOS │ 2G/3G    │
| AMR-WB     │ 23.85kbps│ 4.5 MOS │ VoLTE    │
| EVS        │ 24.4kbps │ 4.8 MOS │ VoLTE-A  │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** SIP 세션, QoS 보장

**📢 섹션 요약 비유:** AMR-WB = HD 음성!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: SRVCC**

| 항목 | 설명 | 용도 |
|:---|:---|:---|
| **SRVCC** | LTE → 2G/3G | 커버리지 |

**📢 섹션 요약 비유:** SRVCC = 끊김 없는 전환!
