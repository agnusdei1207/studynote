# Event Sourcing - 이벤트 소싱

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이벤트로 상태 저장
> 2. **가치**: 감사, 복구, 시간 여행
> 3. **융합:** CQRS, Event Store, Kafka

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Event Sourcing은 모든 상태 변화를 이벤트로 저장합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Event Sourcing                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Event Sourcing 구조                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  传统 방식:                   Event Sourcing:        │   │ │
│   │   │  ┌─────────────┐             ┌─────────────────┐    │   │ │
│   │   │  │ State       │             │ Event Stream    │    │   │ │
│   │   │  │ Balance:100 │             │ E1→E2→E3→E4→... │    │   │ │
│   │   │  └─────────────┘             └─────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  Event Stream:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ AccountCreated {balance: 0}                 │    │   │ │
│   │   │  │ MoneyDeposited {amount: 100}                │    │   │ │
│   │   │  │ MoneyWithdrawn {amount: 50}                 │    │   │ │
│   │   │  │ → 현재 상태: 50 (재생으로 계산)              │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 현재 상태 대신 이벤트 스트림을 저장합니다.

**📢 섹션 요약 비유:** Event Sourcing = 장부!
