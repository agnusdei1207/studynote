# VXLAN (Virtual Extensible LAN) - 가상 확장 LAN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: L2 오버레이 네트워크
> 2. **가치**: VLAN 한계 극복
> 3. **융합:** Data Center, SDN, Overlay

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

VXLAN은 UDP로 L2를 캡슐화합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VXLAN                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              VXLAN 구조                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Original L2 Frame:                                  │   │ │
│   │   │  ┌─────────────────────────────────────┐            │   │ │
│   │   │  │ L2 Header │ Payload                 │            │   │ │
│   │   │  └─────────────────────────────────────┘            │   │ │
│   │   │                          ↓                           │   │ │
│   │   │  VXLAN Packet:                                        │   │ │
│   │   │  ┌──────────────────────────────────────────────┐   │   │ │
│   │   │  │Outer IP│UDP│VXLAN│ L2 Header │   Payload    │   │   │ │
│   │   │  │        │   │ Hdr │            │              │   │   │ │
│   │   │  └──────────────────────────────────────────────┘   │   │ │
│   │   │            VNI (24-bit, 16M VLAN)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                               │   │ │
│   │   │  - VLAN (4096) 한계 극복                             │   │ │
│   │   │  - UDP 4789 포트                                     │   │ │
│   │   │  - 데이터센터 오버레이                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: VXLAN은 24비트 VNI로 1600만 개 네트워크를 지원합니다.

**📢 섹션 요약 비유:** VXLAN = 가상 LAN 확장!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VTEP** | VXLAN Tunnel Endpoint |
| **VNI** | VXLAN Network Identifier |
| **Overlay** | 논리 네트워크 |
| **Underlay** | 물리 네트워크 |

**핵심 알고리즸:** 캡슐화 → Underlay 전송 → 역캡슐화

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**VLAN vs VXLAN**

| 항목 | VLAN | VXLAN |
|:---|:---|:---|
| **ID 범위** | 12-bit (4096) | 24-bit (16M) |
| **계층** | L2 | L2 over L3 |
| **용도** | LAN | Data Center |
