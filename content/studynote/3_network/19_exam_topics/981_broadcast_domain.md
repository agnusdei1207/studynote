# Broadcast Domain - 브로드캐스트 도메인

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 브로드캐스트 전파 영역
> 2. **가치**: 네트워크 효율성
> 3. **융합:** Router, VLAN, Subnet

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Broadcast Domain은 브로드캐스트가 전파되는 영역입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Broadcast Domain                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              브로드캐스트 도메인 분리                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Router로 분리:                                          │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌─────────────┐         ┌─────────────┐        ││   │ │
│   │   │  │  │ Broadcast   │         │ Broadcast   │        ││   │ │
│   │   │  │  │ Domain 1    │         │ Domain 2    │        ││   │ │
│   │   │  │  │ 192.168.1.0 │         │ 192.168.2.0 │        ││   │ │
│   │   │  │  │    /24      │         │    /24      │        ││   │ │
│   │   │  │  └──────┬──────┘         └──────┬──────┘        ││   │ │
│   │   │  │         │                       │                ││   │ │
│   │   │  │         └─────────┬─────────────┘                ││   │ │
│   │   │  │                   │                              ││   │ │
│   │   │  │              ┌────┴────┐                         ││   │ │
│   │   │  │              │ Router  │                         ││   │ │
│   │   │  │              │(L3 경계)│                         ││   │ │
│   │   │  │              └─────────┘                         ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  브로드캐스트는 라우터를 넘지 않음                    ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  분리 방법:                                              │   │ │
│   │   │  - Router: L3 장비로 분리                               │   │ │
│   │   │  - VLAN: 가상 LAN으로 분리                               │   │ │
│   │   │  - Subnet: 서브넷으로 분리                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 브로드캐스트 도메인 분리는 네트워크 성능과 보안을 향상시킵니다.

**📢 섹션 요약 비유:** Broadcast Domain = 방송 청취 범위!
