# MSS (Maximum Segment Size) - 최대 세그먼트 크기

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: TCP 데이터 최대 크기
> 2. **가치**: 단편화 방지
> 3. **융합:** MTU, TCP Options, PMTUD

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MSS는 TCP 페이로드의 최대 크기입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MSS (Maximum Segment Size)                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MSS vs MTU                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  IP 패킷 (MTU 1500)                                  │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │ IP Header (20) │ TCP Header (20) │ Data     │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │         ↓                  ↓               ↓         │   │ │
│   │   │        20                 20           MSS=1460      │   │ │
│   │   │                                                      │   │ │
│   │   │  MSS = MTU - IP Header - TCP Header                  │   │ │
│   │   │  MSS = 1500 - 20 - 20 = 1460                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MSS는 MTU에서 헤더를 제외한 데이터 크기입니다.

**📢 섹션 요약 비유:** MSS = 실제 짐 크기!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**MSS vs MTU**

| 항목 | MSS | MTU |
|:---|:---|:---|
| **계층** | TCP | IP |
| **크기** | 페이로드 | 전체 |
| **단편화** | 없음 | 가능 |
