# 굴절률 / 전반사 (Refractive Index / Total Internal Reflection)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 빛의 경로 변화와 완전 반사
> 2. **가치**: 광섬유 전송의 기본 원리
> 3. **융합**: 광섬유, 프리즘, 다이아몬드와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

굴절률(Refractive Index)은 빛의 속도 비율이고, 전반사(Total Internal Reflection)는 빛이 경계면에서 완전히 반사되는 현상입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    굴절률 / 전반사                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              굴절률 (Refractive Index)                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  정의:                                                    │   │ │
│   │   │  n = c / v  (빛의 속도 비율)                             │   │ │
│   │   │                                                      │   │ │
│   │   │  n = 굴절률                                             │   │ │
│   │   │  c = 진공에서 빛의 속도 (3×10⁸ m/s)                      │   │ │
│   │   │  v = 매질에서 빛의 속도                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  스넬의 법칙:                                             │   │ │
│   │   │  n₁ × sin(θ₁) = n₂ × sin(θ₂)                           │   │ │
│   │   │                                                      │   │ │
│   │   │  예시:                                                   │   │ │
│   │   │  • 공기: 1.0003 ≈ 1                                    │   │ │
│   │   │  • 물: 1.33                                             │   │ │
│   │   │  • 유리: 1.5                                            │   │ │
│   │   │  • 코어: 1.46~1.48                                      │   │ │
│   │   │  • 클래딩: 1.44~1.46                                    │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              전반사 (Total Internal Reflection)               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  조건:                                                   │   │ │
│   │   │  1. n₁ > n₂ (밀도 높은→낮은 매질)                        │   │ │
│   │   │  2. θ > θc (임계각 이상)                                │   │ │
│   │   │                                                      │   │ │
│   │   │  임계각: sin(θc) = n₂ / n₁                             │   │ │
│   │   │                                                      │   │ │
│   │   │  광섬유 적용:                                             │   │ │
│   │   │         코어 (n₁)                                       │   │ │
│   │   │    ╱───────────────────╲                               │   │ │
│   │   │   ╱    ──────────────    ╲  클래딩 (n₂)                │   │ │
│   │   │   │   ╱             ╲    │                             │   │ │
│   │   │   │  ╱               ╲   │                             │   │ │
│   │   │   │ ╱                 ╲  │                             │   │ │
│   │   │   ╱                    ╲ │                             │   │ │
│   │   │      전반사로 빛 유지                                   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 광섬유는 코어(높은 굴절률)와 클래딩(낮은 굴절률)의 전반사로 빛을 전파합니다.

**💡 비유**: 전반사는 물속에서 수면을 보면 물 밖이 안 보이는 현상입니다.

**📢 섹션 요약 비유**: n₁ > n₂ + θ > θc = 전반사!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **굴절률** | 빛 속도 비율 | n = c/v | 밀도 |
| **임계각** | 전반사 조건 | θc | 기울기 |
| **수치구경** | 수광 능력 | NA = √(n₁²-n₂²) | 시야각 |

**비교표**

```
+------------+----------+----------+
| 항목       | 코어     | 클래딩   |
+------------+----------+----------+
| 굴절률     | 높음     | 낮음     |
| 역할       | 빛 전파   | 전반사   |
| 직경       | 작음     | 큼       |
+------------+----------+----------+
```

**핵심 알고리즸**: 스넬의 법칙, NA 계산

**📢 섹션 요약 비유**: NA = 수광 능력!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)
