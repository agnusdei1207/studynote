# Geneve - 제네릭 네트워크 가상화 캡슐화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 유연한 오버레이 캡슐화
> 2. **가치**: 확장 가능한 옵션
> 3. **융합:** VNI, Options, UDP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Geneve (Generic Network Virtualization Encapsulation)는 확장 가능한 오버레이 캡슐화입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Geneve                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Geneve 구조                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Geneve 캡슐화:                                         │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌─────────────────────────────────────────┐    ││   │ │
│   │   │  │  │         Outer IP Header                 │    ││   │ │
│   │   │  │  └─────────────────────────────────────────┘    ││   │ │
│   │   │  │  ┌─────────────────────────────────────────┐    ││   │ │
│   │   │  │  │         Outer UDP Header                │    ││   │ │

│   │   │  │  ┌─────────────────────────────────────────┐    ││   │ │
│   │   │  │  │         Geneve Header                   │    ││   │ │
│   │   │  │  │  ┌───────────────────────────────────┐  │    ││   │ │
│   │   │  │  │  │ VNI (24-bit) │ Options (가변)     │  │    ││   │ │
│   │   │  │  │  └───────────────────────────────────┘  │    ││   │ │
│   │   │  │  └─────────────────────────────────────────┘    ││   │ │

│   │   │  - OVS (Open vSwitch) 지원                             │   │ │
```

> **해설**: Geneve는 다양한 오버레이 프로토콜의 통합을 목표로 합니다.

**📢 섹션 요약 비유:** Geneve = 만능 캡슐화!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VNI** | Virtual Network Identifier |
| **Options** | 확장 옵션 필드 |
| **UDP Port** | 6081 |
