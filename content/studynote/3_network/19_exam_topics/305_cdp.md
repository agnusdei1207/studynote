# CDP - 시스코 디스커버리 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시스코 장비 디스커버리
> 2. **가치**: 토폴로지 자동 매핑
> 3. **융합**: VoIP, PoE와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CDP(Cisco Discovery Protocol)는 시스코 전용 L2 디스커버리 프로토콜입니다.

```

│                    CDP                                              │

│   │   │                                                      │   │ │

│   │   │     │                                │                │   │ │
│   │   │     │ 정보:                          │ 정보:          │   │ │
```

> **해설**: CDP는 시스코 장비 간 상세 정보를 교환합니다.

**💡 비유**: CDP는 시스코 장비만의 비밀 대화입니다.

**📢 섹션 요약 비유:** CDP = Cisco 전용!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Device ID** | 장비 식별 |
| **Capability** | 기능 (R, S, B) |
| **Platform** | 모델 |
| **Holdtime** | 유효 시간 |

**비교표: CDP Capability**

```
+------------+----------+
| 코드       │ 기능     │
+------------+----------+
| R          │ Router   │
| S          │ Switch   │
| B          │ Bridge   │
| H          │ Host     │
| I          │ IGMP     │
+------------+----------+
```

**핵심 알고리즸:** 멀티캐스트 01:00:0C:CC:CC:CC

**📢 섹션 요약 비유:** CDP = 상세 정보!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: CDP 정보**

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: Cisco 환경** → CDP
**시나리오 2: 멀티벤더** → LLDP
**시나리오 3: 보안** → CDP 비활성화

**도입 체크리스트:** 벤더, 보안, 토폴로지

**📢 섹션 요약 비유:** 보안 = CDP 끄기!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 자동 발견

**미래 전망:** LLDP로 전환

**참고 표준:** Cisco 전용

**📢 섹션 요약 비유:** CDP는 Cisco 필수!
