# API Gateway - API 게이트웨이

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: API 단일 진입점
> 2. **가치**: 인증, 라우팅, 제한
> 3. **융합:** Microservices, Load Balancer, Security

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

API Gateway는 모든 API 요청의 단일 진입점입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    API Gateway                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              API Gateway 구성                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Client                                              │   │ │
│   │   │     │                                                │   │ │
│   │   │     ↓                                                │   │ │
│   │   │  ┌─────────────────────────────────────┐            │   │ │
│   │   │  │          API Gateway                │            │   │ │
│   │   │  │  ┌─────────────────────────────┐   │            │   │ │
│   │   │  │  │ Auth │ Rate Limit │ Logging │   │            │   │ │
│   │   │  │  └─────────────────────────────┘   │            │   │ │
│   │   │  │  ┌─────────────────────────────┐   │            │   │ │
│   │   │  │  │ Routing │ Transform │ Cache │   │            │   │ │
│   │   │  │  └─────────────────────────────┘   │            │   │ │
│   │   │  └─────────────────────────────────────┘            │   │ │
│   │   │     │           │           │                        │   │ │
│   │   │     ↓           ↓           ↓                        │   │ │
│   │   │  ┌─────┐    ┌─────┐    ┌─────┐                     │   │ │
│   │   │  │ Svc │    │ Svc │    │ Svc │                     │   │ │
│   │   │  │  A  │    │  B  │    │  C  │                     │   │ │
│   │   │  └─────┘    └─────┘    └─────┘                     │   │ │
```

> **해설**: API Gateway는 인증, 라우팅, 제한 등을 중앙에서 처리합니다.

**📢 섹션 요약 비유:** API Gateway = 프론트 데스크!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Authentication** | 인증 |
| **Rate Limiting** | 속도 제한 |
| **Routing** | 경로 설정 |
| **Transformation** | 데이터 변환 |
| **Logging** | 로깅 |

**핵심 알고리즸:** 요청 → 검증 → 라우팅 → 응답

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**API Gateway 기능**
