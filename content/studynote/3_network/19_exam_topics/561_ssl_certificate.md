# SSL Certificate (Digital Certificate) - SSL 인증서

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 공개키 소유 증명
> 2. **가치**: 신원 검증, 암호화
> 3. **융합:** X.509, CA, PKI

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SSL 인증서는 공개키와 소유자 정보를 CA가 서명합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SSL Certificate                                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              인증서 구조 (X.509)                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Version: v3                                         │   │ │
│   │   │  Serial Number: 01:23:45:67                          │   │ │
│   │   │  Issuer: DigiCert Inc.                               │   │ │
│   │   │  Validity: 2024-01-01 ~ 2025-01-01                   │   │ │
│   │   │  Subject: www.example.com                            │   │ │
│   │   │  Public Key: RSA 2048-bit                            │   │ │
│   │   │  Signature: CA's Private Key                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 인증서는 CA의 서명으로 신뢰를 보장합니다.

**📢 섹션 요약 비유:** SSL 인증서 = 신분증!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **DV** | Domain Validation |
| **OV** | Organization Validation |
| **EV** | Extended Validation |
| **Wildcard** | 서브도메인 포함 |
| **SAN** | Subject Alternative Name |
| **Chain** | 인증서 체인 |

**핵심 알고리즸:** CA 서명 검증 → 신뢰

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**인증서 유형**

| 유형 | 검증 | 용도 |
|:---|:---|:---|
| **DV** | 도메인만 | 개인/소호 |
| **OV** | 조직 | 기업 |
| **EV** | 엄격 | 금융 |

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 웹사이트 → DV 인증서
**시나리오 2:** 금융 → EV 인증서
**시나리오 3:** 다중 도메인 → SAN
