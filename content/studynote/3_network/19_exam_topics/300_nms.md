# NMS - 네트워크 관리 시스템

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 통합 네트워크 관리 플랫폼
> 2. **가치**: 중앙 관리, 자동화
> 3. **융합**: SNMP, 모니터링과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

NMS(Network Management System)는 네트워크 장비를 중앙에서 관리하는 시스템입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    NMS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NMS 구조                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ┌────────────────────────────────────────────┐     │   │ │
│   │   │  │              NMS Server                     │     │   │ │
│   │   │  │                                              │     │   │ │
│   │   │  │  ┌────────┐ ┌────────┐ ┌────────┐          │     │   │ │
│   │   │  │  │모니터링│ │구성   │ │성능   │          │     │   │ │
│   │   │  │  │        │ │관리   │ │분석   │          │     │   │ │
│   │   │  │  └────────┘ └────────┘ └────────┘          │     │   │ │
│   │   │  │                                              │     │   │ │
│   │   │  │  ┌────────┐ ┌────────┐ ┌────────┐          │     │   │ │
│   │   │  │  │장애   │ │보고서 │ │자동화 │          │     │   │ │
│   │   │  │  │관리   │ │       │ │       │          │     │   │ │
│   │   │  │  └────────┘ └────────┘ └────────┘          │     │   │ │
│   │   │  └────────────────────────────────────────────┘     │   │ │
│   │   │       │              │              │                 │   │ │
│   │   │       │ SNMP         │ CLI/NETCONF  │ syslog         │   │ │
│   │   │       ▼              ▼              ▼                 │   │ │
│   │   │  [Router]      [Switch]       [Firewall]             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              NMS 기능                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. FCAPS (ITU-T 모델):                               │   │ │
│   │   │     • F: Fault (장애)                                 │   │ │
│   │   │     • C: Configuration (구성)                        │   │ │
│   │   │     • A: Accounting (계정)                           │   │ │
│   │   │     • P: Performance (성능)                          │   │ │
│   │   │     • S: Security (보안)                             │   │ │

│   │   │     • 네트워크 맵                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 이벤트 관리:                                       │   │ │
│   │   │     • 알림, 상관관계 분석                             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
```

> **해설**: NMS는 FCAPS 모델을 기반으로 네트워크를 종합 관리합니다.

**💡 비유**: NMS는 네트워크 관제실입니다. 모든 것을 한 곳에서 봅니다.

**📢 섹션 요약 비유:** NMS = 중앙 관리!
