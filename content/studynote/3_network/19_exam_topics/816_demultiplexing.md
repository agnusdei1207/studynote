# Demultiplexing - 디멀티플렉싱

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 분리
> 2. **가치**: 개별 채널 복원
> 3. **융합:** TDM, FDM, 포트 기반

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Demultiplexing은 결합된 신호를 개별로 분리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Demultiplexing                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              디멀티플렉싱 과정                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  복합 신호                                             │   │ │
│   │   │  ┌───────────────────────────────┐                  │   │ │
│   │   │  │ [Ch1][Ch2][Ch3][Ch1][Ch2][Ch3]│                  │   │ │
│   │   │  └───────────────────────────────┘                  │   │ │
│   │   │                     │                                │   │ │
│   │   │                     ↓                                │   │ │
│   │   │            ┌─────────────────┐                       │   │ │
│   │   │            │  Demultiplexer  │                       │   │ │
│   │   │            └─────────────────┘                       │   │ │
│   │   │              │     │     │                           │   │ │
│   │   │              ↓     ↓     ↓                           │   │ │
│   │   │           ┌───┐ ┌───┐ ┌───┐                         │   │ │
│   │   │           │Ch1│ │Ch2│ │Ch3│                         │   │ │
│   │   │           └───┘ └───┘ └───┘                         │   │ │
│   │   │                                                      │   │ │
│   │   │  TCP/UDP Demux:                                       │   │ │
│   │   │  IP 패킷 → 포트 번호로 소켓 분리                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 디멀티플렉싱은 수신측에서 필수적입니다.

**📢 섹션 요약 비유:** Demultiplexing = 우편 분류!
