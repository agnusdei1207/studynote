# B-Tree - B-트리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 균형 다진 트리
> 2. **가치**: 효율적인 검색/삽입/삭제
> 3. **융합:** Database Index, File System, Storage

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

B-Tree는 자식이 여러 개인 균형 트리입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    B-Tree                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              B-Tree 구조                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │              [30 | 70]                               │   │ │
│   │   │             ╱    |    ╲                              │   │ │
│   │   │          [10|20] [50|60] [80|90]                     │   │ │
│   │   │          ╱  |  ╲   ╱  |  ╲   ╱  |  ╲                │   │ │
│   │   │        [1] [15] [25] [40] [55] [65] [75] [85] [95]   │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                │   │ │
│   │   │  - 모든 리프가 같은 레벨                              │   │ │
│   │   │  - 각 노드는 여러 키를 가짐                           │   │ │
│   │   │  - 최소 50% 채워짐                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  시간 복잡도: O(log n)                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: B-Tree는 디스크 I/O를 최소화합니다.

**📢 섹션 요약 비유:** B-Tree = 멀티 레벨 색인!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

| 항목 | 설명 |
|:---|:---|
| **Node** | 노드 |

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**B-Tree vs B+Tree**

| 항목 | B-Tree | B+Tree |
|:---|:---|:---|
| **데이터 위치** | 모든 노드 | 리프만 |
| **링크** | 없음 | 리프 연결 |
| **용도** | 일반 | DB 인덱스 |
