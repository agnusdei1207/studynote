# MPLS - 멀티프로토콜 라벨 스위칭

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 라벨 기반 포워딩
> 2. **가치**: 고속 L2.5 스위칭
> 3. **융합:** LDP, LSR, LER

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MPLS (Multi-Protocol Label Switching)는 라벨을 이용한 패킷 전송 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MPLS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MPLS 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  MPLS 헤더:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌─────────┬──────────┬───────────┬──────────┐ ││   │ │
│   │   │  │  │ Label   │ EXP (3b) │ S (1b)    │ TTL (8b) │ ││   │ │
│   │   │  │  │ (20b)   │ QoS      │ Bottom of │ Hop Limit│ ││   │ │
│   │   │  │  │         │          │ Stack     │          │ ││   │ │
│   │   │  │  └─────────┴──────────┴───────────┴──────────┘ ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  MPLS 네트워크:                                    ││   │ │
│   │   │  │  ┌─────┐    ┌─────┐    ┌─────┐    ┌─────┐      ││   │ │
│   │   │  │  │ LER │────│ LSR │────│ LSR │────│ LER │      ││   │ │
│   │   │  │  │입구 │ L1  │ L2  │ L2  │ L2  │ L1  │출구 │      ││   │ │
│   │   │  │  └──┬──┘    └─────┘    └─────┘    └──┬──┘      ││   │ │
│   │   │  │     │                               │           ││   │ │
│   │   │  │  IP패킷                            IP패킷        ││   │ │
│   │   │  │  +라벨                            -라벨          ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  LER: Label Edge Router (라벨 추가/제거)          ││   │ │
│   │   │  │  LSR: Label Switch Router (라벨 교체)             ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MPLS는 L2와 L3 사이의 L2.5 계층에서 동작합니다.

**📢 섹션 요약 비유:** MPLS = 우편물 라벨!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Label** | 20-bit 라벨 |
| **LER** | Label Edge Router |
| **LSR** | Label Switch Router |
| **LSP** | Label Switched Path |
| **FEC** | Forwarding Equivalence Class |

**핵심 알고리듬:** 라벨 부여 → 라벨 교체 → 라벨 제거
