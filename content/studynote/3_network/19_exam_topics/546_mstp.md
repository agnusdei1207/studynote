# MSTP (Multiple STP) - 다중 스패닝 트리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: VLAN 그룹별 STP 인스턴스
> 2. **가치**: 로드 밸런싱, 확장성
> 3. **융합:** RSTP, VLAN, MST Region

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MSTP는 여러 VLAN을 그룹으로 묶어 STP를 실행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MSTP                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MSTP 구성                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  PVST+: VLAN마다 STP (Cisco)                        │   │ │
│   │   │  → 1000개 VLAN = 1000개 STP 인스턴스 (비효율)         │   │ │
│   │   │                                                      │   │ │
│   │   │  MSTP: VLAN 그룹별 STP                               │   │ │
│   │   │  Instance 1: VLAN 10, 20, 30                         │   │ │
│   │   │  Instance 2: VLAN 40, 50, 60                         │   │ │
│   │   │  → 로드 밸런싱 가능                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  Switch A ──────── Switch B                          │   │ │
│   │   │    │ Inst1 Forward   │ Inst2 Forward                 │   │ │
│   │   │    │ Inst2 Block     │ Inst1 Block                   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MSTP는 VLAN 그룹화로 STP 오버헤드를 줄입니다.

**📢 섹션 요약 비유:** MSTP = 차선 변경!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **MST Region** | MSTP 도메인 |
| **MST Instance** | VLAN 그룹 STP |
| **IST** | Internal Spanning Tree |
| **CST** | Common Spanning Tree |

**핵심 알고리즸:** VLAN → Instance 매핑 → 독립 STP

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**STP 변형 비교**

| 프로토콜 | 인스턴스 | 용도 |
|:---|:---|:---|
| **STP** | 1개 | 기본 |
| **PVST+** | VLAN당 | Cisco |
| **RSTP** | 1개 | 빠른 수렴 |
| **MSTP** | 그룹 | 확장 |
