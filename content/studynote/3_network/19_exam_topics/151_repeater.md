# 베이스밴드 중계기 / 리피터 (Repeater)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 증폭 및 재생
> 2. **가치**: 전송 거리 연장
> 3. **융합**: 광섬유, 무선과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

리피터(Repeater)는 약해진 신호를 증폭하거나 재생하여 전송 거리를 연장하는 장치입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    리피터 (Repeater)                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              리피터 동작                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [송신측]        [리피터]        [수신측]              │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────┐    약한 신호   ┌──────────┐   강한 신호 ┌─────┐│   │ │
│   │   │  │     │──────────────►│증폭/재생 │──────────────►│     ││   │ │
│   │   │  │     │    ~~~→       │   ▲▲▲   │    ──────►   │     ││   │ │
│   │   │  └─────┘               └──────────┘              └─────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  동작 과정:                                             │   │ │
│   │   │  1. 약해진 신호 수신                                    │   │ │
│   │   │  2. 신호 증폭 또는 재생                                  │   │ │
│   │   │  3. 잡음 제거 (디지털)                                  │   │ │
│   │   │  4. 재전송                                              │   │ │
│   │   │                                                      │   │ │
│   │   │  종류:                                                   │   │ │
│   │   │  • 아날로그 리피터: 증폭만 (잡음도 증폭)                  │   │ │
│   │   │  • 디지털 리피터: 재생 (잡음 제거)                        │   │ │
│   │   │  • 광 리피터: O-E-O 변환                                │   │ │
│   │   │                                                      │   │ │
│   │   │  OSI 계층: 물리 계층 (1계층)                             │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              베이스밴드 vs 브로드밴드                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  베이스밴드 중계기:                                     │   │ │
│   │   │  • 디지털 신호 그대로 전송                              │   │ │
│   │   │  • 이더넷(10BASE-T, 100BASE-TX)에서 사용               │   │ │
│   │   │  • 신호 재생                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  브로드밴드 중계기:                                      │   │ │
│   │   │  • 아날로그 신호 증폭                                    │   │ │
│   │   │  • CATV에서 사용                                        │   │ │
│   │   │  • 잡음도 증폭                                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 리피터는 1계층 장비로 신호를 증폭/재생하여 거리를 연장합니다.

**💡 비유**: 리피터는 마라톤에서 달리는 주자에게 물을 주는 급수소입니다.

**📢 섹션 요약 비유:** 리피터 = 1계층 신호 증폭!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **수신부** | 신호 수신 | 증폭기 | 귀 |
| **재생부** | 신호 복원 | 클럭 복구 | 뇌 |
| **송신부** | 신호 재전송 | 드라이버 | 입 |

**비교표**

```
+------------+----------+----------+
| 항목       | 아날로그 | 디지털   |
+------------+----------+----------+
| 잡음       | 증폭     | 제거     |
| 신호       | 증폭만   | 재생     |
| 품질       | 저하     | 유지     |
+------------+----------+----------+
```

**핵심 알고리즸:** 클럭 복구, 신호 재생

**📢 섹션 요약 비유:** 디지털 리피터 = 잡음 제거!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 거리 연장** → 리피터 추가
**시나리오 2: 광섬유** → 광 증폭기(EDFA)
**시나리오 3: 무선** → 중계기

**도입 체크리스트:** 거리, 신호 품질, 지연

**📢 섹션 요약 비유:** 100m마다 리피터!
