# Sharding - 샤딩

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 수평 분할
> 2. **가치**: 확장성, 성능
> 3. **융합:** Partitioning, Distributed Database, Scaling

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Sharding은 데이터를 수평으로 분할합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Sharding                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              샤딩 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Before (단일 DB):                                    │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │         All Data (1TB)                       │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │                                                      │   │ │
│   │   │  After (샤딩):                                        │   │ │
│   │   │  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐   │   │ │
│   │   │  │ Shard 1 │ │ Shard 2 │ │ Shard 3 │ │ Shard 4 │   │   │ │
│   │   │  │ User    │ │ User    │ │ User    │ │ User    │   │   │ │
│   │   │  │ 1-250   │ │ 251-500 │ │ 501-750 │ │ 751-1000│   │   │ │
│   │   │  └─────────┘ └─────────┘ └─────────┘ └─────────┘   │   │ │
│   │   │                                                      │   │ │
│   │   │  샤딩 키 전략:                                         │   │ │
│   │   │  - Range: 범위                                        │   │ │
│   │   │  - Hash: 해시                                         │   │ │
│   │   │  - Directory: 디렉토리                                 │   │ │
│   │   │  - Geo: 지역                                          │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 샤딩은 데이터를 여러 노드에 분산합니다.

**📢 섹션 요약 비유:** Sharding = 책장 나누기!
