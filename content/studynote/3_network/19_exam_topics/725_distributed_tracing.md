# Distributed Tracing - 분산 추적

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 요청 흐름 추적
> 2. **가치**: 문제 진단, 성능 분석
> 3. **융합:** Observability, OpenTelemetry, Jaeger

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Distributed Tracing은 분산 요청을 추적합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Distributed Tracing                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              분산 추적 구조                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Trace (트레이스): 전체 요청 흐름                      │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Trace ID: abc123                                ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │ Span 1: API Gateway (10ms)                      ││   │ │
│   │   │  │   ├── Span 2: Auth Service (5ms)               ││   │ │
│   │   │  │   ├── Span 3: Order Service (20ms)             ││   │ │
│   │   │  │   │     ├── Span 4: DB Query (8ms)            ││   │ │
│   │   │  │   │     └── Span 5: Cache (2ms)               ││   │ │
│   │   │  │   └── Span 6: Response (1ms)                   ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 분산 추적은 마이크로서비스 간 요청을 추적합니다.

**📢 섹션 요약 비유:** Distributed Tracing = 배송 추적!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Trace** | 전체 요청 |
| **Span** | 개별 작업 |
| **Context** | 컨텍스트 전파 |
| **Sampler** | 샘플링 |
| **Collector** | 수집기 |

**핵심 알고리즸:** 요청 시작 → Trace ID 생성 → Span 기록 → 전파
