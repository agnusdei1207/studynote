# MMS - 멀티미디어 메시지

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: SMS 확장형 멀티미디어 메시징
> 2. **가치**: 이미지, 비디오, 오디오 전송
> 3. **융합:** 3G/4G, WAP, HTTP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MMS(Multimedia Messaging Service)는 이미지, 비디오, 오디오를 전송합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MMS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MMS 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [발신 단말]                                         │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ MMS 메시지 (이미지/비디오/오디오)               │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           MMSC (MMS Center)                 │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • 메시지 저장                                │    │   │ │
│   │   │  │  • 형식 변환                                  │    │   │ │
│   │   │  │  • 전달                                       │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ 데이터 채널 (GPRS/3G/4G)                     │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [수신 단말]                                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MMS 지원 미디어                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 미디어     │ 형식     │ 크기     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 이미지     │ JPEG     │ 300KB    │               │   │ │
│   │   │  │ 오디오     │ AMR      │ 100KB    │               │   │ │
│   │   │  | 비디오     │ 3GP      │ 300KB    │               │   │ │
│   │   │  │ 텍스트     │ UTF-8    │ 무제한   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MMS는 데이터 채널로 멀티미디어 메시지를 전송합니다.

**💡 비유**: MMS는 사진 엽서입니다. 그림과 글을 함께 보냅니다.

**📢 섹션 요약 비유:** MMS = 멀티미디어!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **MMSC** | MMS 센터 |
| **WAP** | 무선 애플리케이션 |
| **PAP** | 푸시 프로토콜 |
| **MIME** | 미디어 형식 |

**비교표: SMS vs MMS**

```
+------------+----------+----------+
| 항목       │ SMS      │ MMS      │
+------------+----------+----------+
| 내용       │ 텍스트   │ 미디어   │
| 채널       │ 시그널링 │ 데이터   │
| 비용       │ 낮음     │ 높음     │
| 크기       │ 160자    │ 300KB    │
+------------+----------+----------+
```

**핵심 알고리즸:** MIME, 트랜스코딩

**📢 섹션 요약 비유:** MMSC = MMS 우체국!
