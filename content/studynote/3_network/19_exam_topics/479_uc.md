### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |

**비교표: UC 기능**

```
+------------+----------+----------+----------+
| 기능       │ 설명     │ 프로토콜 │ 용도     │
+------------+----------+----------+----------+
| 음성       │ VoIP     │ SIP/RTP  │ 통화     │
| 영상       │ 화상     │ WebRTC   │ 회의     │
| 메시징     │ IM       │ XMPP     │ 채팅     │
| 상태       │ Presence │ SIP      │ 가용성   │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 상태 동기화, 라우팅
