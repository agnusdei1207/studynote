# Protocol - 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 통신 규칙
> 2. **가치**: 상호 운용성
> 3. **융합:** TCP/IP, HTTP, Standards

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Protocol은 통신을 위한 규칙과 형식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Protocol                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              프로토콜 구성 요소                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Syntax (구문):                                       │   │ │
│   │   │  - 데이터 형식                                        │   │ │
│   │   │  - 신호 레벨                                          │   │ │
│   │   │  - 인코딩                                             │   │ │
│   │   │                                                      │   │ │
│   │   │  Semantics (의미):                                    │   │ │
│   │   │  - 제어 정보                                          │   │ │
│   │   │  - 오류 처리                                          │   │ │
│   │   │  - 동작 정의                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  Timing (타이밍):                                      │   │ │
│   │   │  - 속도 조절                                          │   │ │
│   │   │  - 순서 제어                                          │   │ │
│   │   │  - 동기화                                             │   │ │
│   │   │                                                      │   │ │
│   │   │  계층별 프로토콜:                                      │   │ │
│   │   │  App: HTTP, FTP, SMTP, DNS                            │   │ │
│   │   │  Transport: TCP, UDP, SCTP                            │   │ │
│   │   │  Network: IP, ICMP, OSPF                              │   │ │
│   │   │  Link: Ethernet, WiFi, PPP                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 프로토콜은 이기종 시스템 간 통신을 가능하게 합니다.

**📢 섹션 요약 비유:** Protocol = 통신 언어!
