# Timeout - 타임아웃

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 대기 시간 제한
> 2. **가치**: 빠른 실패, 자원 보호
> 3. **융합:** Circuit Breaker, Retry, Resilience

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Timeout은 대기 시간을 제한합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Timeout                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              타임아웃 종류                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Connection Timeout:                                 │   │ │
│   │   │  Client ──→ Server (연결 대기 시간)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  Read Timeout:                                       │   │ │
│   │   │  Client ──→ Server (응답 대기 시간)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  Write Timeout:                                      │   │ │
│   │   │  Client ──→ Server (전송 대기 시간)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  Idle Timeout:                                       │   │ │
│   │   │  연결 유지 시간                                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 타임아웃은 무한 대기를 방지합니다.

**📢 섹션 요약 비유:** Timeout = 알람 시계!
