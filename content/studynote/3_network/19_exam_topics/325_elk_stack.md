# ELK Stack - 오픈소스 로그 분석

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 오픈소스 로그 분석 스택
> 2. **가치**: 검색, 시각화, 확장성
> 3. **융합**: Beats, Kibana와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ELK Stack은 Elasticsearch, Logstash, Kibana로 구성된 로그 분석 플랫폼입니다.


> **해설**: ELK Stack은 오픈소스로 로그 수집부터 시각화까지 제공합니다.

**💡 비유**: ELK Stack은 데이터 정수 공장입니다. 원석을 보석으로 만듭니다.

**📢 섹션 요약 비유:** ELK = 로그 분석!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Elasticsearch** | 검색 엔진 |
| **Logstash** | 파이프라인 |
| **Kibana** | 시각화 |
| **Beats** | 경량 수집 |

**비교표: ELK vs Splunk**

```
+------------+----------+----------+
| 항목       │ ELK      │ Splunk  │
+------------+----------+----------+
| 비용       │ 무료     │ 유료     │
| 운영       │ 자체     │ 자체/SaaS│
| 확장성     │ 높음     │ 높음     │
| 복잡도     │ 높음     │ 중간     │
+------------+----------+----------+
```

**핵심 알고리즸:** 인덱싱, Lucene 검색

**📢 섹션 요약 비유:** Elasticsearch = 검색 엔진!
