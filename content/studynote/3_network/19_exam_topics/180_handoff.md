# 핸드오프 (Handoff / Handover)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 셀 간 이동 시 연결 유지
> 2. **가치**: 끊김 없는 통신
> 3. **융합**: 이동성 관리의 핵심

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

핸드오프(Handoff/Handover)는 이동국이 셀 경계를 넘을 때 통화 연결을 유지하며 기지국을 변경하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    핸드오프 (Handoff / Handover)                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              핸드오프 종류                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 하드 핸드오프 (Hard Handoff):                       │   │ │
│   │   │     • 기존 연결 끊고 새 연결                           │   │ │
│   │   │     • 끊김 있음 (break-before-make)                   │   │ │
│   │   │     • GSM                                             │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 소프트 핸드오프 (Soft Handoff):                     │   │ │
│   │   │     • 기존+새 연결 동시 유지                           │   │ │
│   │   │     • 끊김 없음 (make-before-break)                   │   │ │
│   │   │     • CDMA                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 소fter 핸드오프 (Softer Handoff):                  │   │ │
│   │   │     • 같은 기지국 내 섹터 간                            │   │ │
│   │   │     • CDMA                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  4. 수직 핸드오프 (Vertical Handoff):                   │   │ │
│   │   │     • 다른 네트워크 간 (WiFi↔LTE)                      │   │ │
│   │   │     • 이기종 네트워크                                  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              핸드오프 절차                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [BS1]              [이동국]             [BS2]       │   │ │
│   │   │    │                   │                   │         │   │ │
│   │   │    │  1. 측정 보고 ────►│                   │         │   │ │
│   │   │    │                   │  2. 인접 BS 탐색  │         │   │ │
│   │   │    │◄──────────────────┼──────────────────►│         │   │ │
│   │   │    │  3. 핸드오프 결정  │                   │         │   │ │
│   │   │    │                   │  4. 핸드오프 실행  │         │   │ │
│   │   │    │                   ├──────────────────►│         │   │ │
│   │   │    │  5. 연결 해제     │                   │         │   │ │
│   │   │    │◄──────────────────┤                   │         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 소프트 핸드오프는 CDMA의 핵심 장점입니다.

**💡 비유**: 핸드오프는 달리면서 공을 패스하는 것과 같습니다.

**📢 섹션 요약 비유:** 소프트 핸드오프 = 끊김 없음!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 유형 | 연결 상태 | 지연 | 용도 |
|:---|:---:|:---:|:---|
| **하드** | 끊김 | 있음 | GSM, LTE |
| **소프트** | 이중 | 없음 | CDMA |
| **수직** | 네트워크 변경 | 있음 | WiFi↔LTE |

**비교표**

```
+------------+----------+----------+
| 항목       │ 하드     │ 소프트   │
+------------+----------+----------+
| 끊김       │ 있음     │ 없음     │
│ 자원       │ 단일     │ 이중     │
│ 복잡도     │ 낮음     │ 높음     │
│ 품질       │ 낮음     │ 높음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 핸드오프 결정, 핑퐁 방지

**📢 섹션 요약 비유:** 핑퐁 = 잦은 핸드오프!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 핸드오프 기준**

| 기준 | 설명 | 임계값 |
|:---|:---|:---|
| **RSSI** | 수신 신호 세기 | -100dBm |

| **거리** | 기지국 거리 | 셀 반경 |

**📢 섹션 요약 비유:** 히스테리시스 = 핑퐁 방지!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 고속 이동** → 하드 핸드오프, 빠른 결정
**시나리오 2: 셀 경계** → 소프트 핸드오프
**시나리오 3: WiFi↔LTE** → 수직 핸드오프

**도입 체크리스트:** 이동 속도, 지연 요구, 자원

**📢 섹션 요약 비유:** 이동성 = 핸드오프!
