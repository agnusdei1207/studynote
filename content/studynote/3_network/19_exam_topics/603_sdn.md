# SDN (Software-Defined Networking) - 소프트웨어 정의 네트워킹

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 컨트롤/데이터 평면 분리
> 2. **가치**: 프로그래머블 네트워크
> 3. **융합:** OpenFlow, NFV, Cloud

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SDN은 컨트롤 평면과 데이터 평면을 분리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SDN                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SDN 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Application Layer:                                  │   │ │
│   │   │  ┌──────────────────────────────────────────┐       │   │ │
│   │   │  │  Network Apps (Firewall, LB, Monitoring) │       │   │ │
│   │   │  └──────────────────────────────────────────┘       │   │ │
│   │   │                    ↑ NBI                            │   │ │
│   │   │  Control Layer:                                      │   │ │
│   │   │  ┌──────────────────────────────────────────┐       │   │ │
│   │   │  │         SDN Controller                    │       │   │ │
│   │   │  │    (OpenFlow, ODL, ONOS)                  │       │   │ │
│   │   │  └──────────────────────────────────────────┘       │   │ │
│   │   │                    ↓ SBI                            │   │ │
│   │   │  Infrastructure Layer:                               │   │ │
│   │   │  ┌──────────────────────────────────────────┐       │   │ │
│   │   │  │  Switch  │  Switch  │  Switch  │ Switch  │       │   │ │
│   │   │  └──────────────────────────────────────────┘       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SDN은 중앙 집중식 컨트롤러로 네트워크를 제어합니다.

**📢 섹션 요약 비유:** SDN = 네트워크 OS!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Control Plane** | 제어 로직 |
| **Data Plane** | 패킷 전송 |
| **NBI** | Northbound API |
| **SBI** | Southbound API |
| **OpenFlow** | 표준 프로토콜 |

**핵심 알고리즸:** 앱 → 컨트롤러 → 스위치

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기존 네트워크 vs SDN**

| 항목 | 기존 | SDN |
|:---|:---|:---|
| **제어** | 분산 | 중앙 |
| **설정** | 개별 | 자동화 |
| **유연성** | 낮음 | 높음 |
