# IoT (Internet of Things) - 사물인터넷

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 사물의 인터넷 연결
> 2. **가치**: 데이터 수집, 자동화
> 3. **융합:** MQTT, CoAP, 5G, Edge

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

IoT는 사물이 인터넷에 연결되어 통신합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    IoT                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              IoT 아키텍처                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  IoT Devices          Gateway          Cloud         │   │ │
│   │   │  ┌─────┐              ┌─────┐         ┌─────┐       │   │ │
│   │   │  │Sensor│───┐         │     │         │     │       │   │ │
│   │   │  └─────┘   │         │ API │────────→│ IoT │       │   │ │
│   │   │  ┌─────┐   │────────→│ GW  │         │ Plat│       │   │ │
│   │   │  │Actua-│───┘         │     │         │form │       │   │ │
│   │   │  │tor  │              └─────┘         └─────┘       │   │ │
│   │   │  └─────┘                                             │   │ │
│   │   │                                                      │   │ │
│   │   │  프로토콜:                                           │   │ │
│   │   │  - MQTT (경량 Pub/Sub)                               │   │ │
│   │   │  - CoAP (Constrained App Protocol)                   │   │ │
│   │   │  - LwM2M (Device Management)                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: IoT는 경량 프로토콜로 제약 있는 장치를 지원합니다.

**📢 섹션 요약 비유:** IoT = 사물의 언어!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Sensor** | 데이터 수집 |
| **Actuator** | 동작 수행 |
| **Gateway** | 중계 장치 |
| **IoT Platform** | 관리 플랫폼 |

**핵심 알고리즸:** 수집 → 전송 → 처리 → 제어

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**IoT 프로토콜**

| 프로토콜 | 계층 | 특징 |
|:---|:---|:---|
| **MQTT** | App | Pub/Sub |
| **CoAP** | App | REST |
| **Zigbee** | L2/L3 | Mesh |
| **LoRa** | L1/L2 | 장거리 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
