# 직교성 (Orthogonality)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 간 간섭 없이 독립 전송 가능한 성질
> 2. **가치**: CDM, OFDM, QAM의 수학적 기반
> 3. **융합**: CDMA, WiFi, LTE와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

직교성(Orthogonality)은 두 신호의 내적이 0이 되어 서로 간섭하지 않는 성질입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    직교성 (Orthogonality)                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              직교성 정의                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  수학적 정의:                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  신호 f(t)와 g(t)가 직교:                              │   │ │
│   │   │  ∫ f(t) × g(t) dt = 0                                │   │ │
│   │   │                                                      │   │ │
│   │   │  이산 신호의 경우:                                      │   │ │
│   │   │  Σ f[n] × g[n] = 0                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  자기 상관:                                             │   │ │
│   │   │  Σ f[n] × f[n] = E (에너지)                           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              직교성 응용                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  1. 주파수 직교 (OFDM):                                │   │ │
│   │   │     sin(2πf₁t) × sin(2πf₂t) = 0  (f₁ ≠ f₂)          │   │ │
│   │   │     → 부반송파 간 간섭 없음                             │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 코드 직교 (CDM):                                    │   │ │
│   │   │     Walsh 코드: W₁ × W₂ = 0                           │   │ │
│   │   │     → 사용자 간 간섭 없음                               │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 위상 직교 (QAM):                                    │   │ │
│   │   │     cos(ωt) × sin(ωt) = 0                             │   │ │
│   │   │     → I/Q 채널 독립 전송                                │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 직교성은 현대 통신의 수학적 기반입니다.

**💡 비유**: 직교성은 서로 다른 언어로 대화하여 간섭이 없는 것입니다.

**📢 섹션 요약 비유**: 직교 = 내적이 0!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 유형 | 조건 | 응용 | 비유 |
|:---|:---|:---|:---|
| **주파수 직교** | f 간격 = 1/T | OFDM | 파장 |
| **코드 직교** | 내적 = 0 | CDMA | 암호 |
| **위상 직교** | 90° 위상차 | QAM | 축 |

**비교표**

```
+------------+----------+----------+
| 직교 유형  | 조건     | 응용     |
+------------+----------+----------+
| 주파수     | Δf=1/T   | OFDM     |
| 코드       | W₁×W₂=0  | CDMA     |
| 위상       | 90°차이  | QAM/PSK  |
| 시간       | 시분할   | TDM      |
+------------+----------+----------+
```

**핵심 알고리즸**: 내적 계산 → 0이면 직교 → 간섭 없음

**📢 섹션 요약 비유**: cos × sin = 0 (90° 직교)!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 직교성 활용**

| 기술 | 직교 유형 | 장점 |
|:---|:---|:---|
| **OFDM** | 주파수 | 스펙트럼 효율 |
| **CDMA** | 코드 | 보안, 용량 |
| **QAM** | 위상 | 2배 전송 |

**📢 섹션 요약 비유**: OFDM은 주파수 직교!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: OFDM** → 주파수 직교
**시나리오 2: CDMA** → 코드 직교
**시나리오 3: QPSK/QAM** → 위상 직교

**도입 체크리스트**: 직교 조건, 동기, 왜곡

**📢 섹션 요약 비유**: 직교성은 통신의 기하학!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 간섭 없는 동시 전송

**미래 전망**: 비직교(NOMA)와 공존

**참고 표준**: IEEE 802.11, 3GPP

**📢 섹션 요약 비유**: 직교성은 독립의 수학!
