# Splunk - 로그 분석 플랫폼

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 머신 데이터 분석 플랫폼
> 2. **가치**: 검색, 시각화, 보안
> 3. **융합**: SIEM, IT 서비스와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Splunk는 머신 데이터를 수집, 분석, 시각화하는 플랫폼입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Splunk                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │

│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Data Sources]                                     │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ Logs / Metrics / Events                        │   │ │
│   │   │     │                                                │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │

│   │   │  │  • 데이터 수집                                 │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
```

> **해설**: Splunk는 머신 데이터를 검색 가능한 지능으로 변환합니다.

**💡 비유**: Splunk는 데이터의 구글입니다. 모든 로그를 검색합니다.

**📢 섹션 요약 비유:** Splunk = 로그 검색!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Forwarder** | 수집 |
| **Indexer** | 인덱싱 |
| **Search Head** | 검색 |
| **SPL** | 검색 언어 |

**비교표: Splunk vs ELK**

```
+------------+----------+----------+
| 항목       │ Splunk   │ ELK      │
+------------+----------+----------+
| 비용       │ 유료     │ 오픈소스 │
| 검색       │ SPL      │ Lucene   │
| 규모       │ 대형     │ 중대형   │
| 지원       │ 기술     │ 커뮤니티 │
+------------+----------+----------+
```

**핵심 알고리즸:** 인덱싱, SPL 검색

**📢 섹션 요약 비유:** SPL = 강력한 검색!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 로그 분석**

| 항목 | Splunk | ELK | Datadog |
|:---|:---|:---|:---|
| **검색** | 강함 | 강함 | 중간 |
| **SIEM** | 강함 | 약함 | 중간 |
| **비용** | 높음 | 낮음 | 중간 |

**📢 섹션 요약 비유:** SIEM = Splunk!
