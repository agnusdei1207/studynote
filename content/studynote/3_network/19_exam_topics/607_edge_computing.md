# Edge Computing - 엣지 컴퓨팅

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 소스 근처 처리
> 2. **가치**: 저지연, 대역폭 절약
> 3. **융합:** IoT, 5G, Cloud

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Edge Computing은 데이터 생성지 근처에서 처리합니다.

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Cloud vs Edge**

| 항목 | Cloud | Edge |
|:---|:---|:---|
| **지연** | 높음 | 낮음 |
| **용량** | 무제한 | 제한 |
| **실시간** | 어려움 | 가능 |
