# TShark - 와이어샤크 CLI

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: Wireshark CLI 분석 도구
> 2. **가치**: 자동화, 스크립팅
> 3. **융합**: CI/CD, 자동화와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

TShark는 Wireshark의 CLI 버전으로 터미널에서 패킷을 분석합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TShark                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TShark 사용법                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  $ tshark -r capture.pcap                            │   │ │
│   │   │                                                      │   │ │
│   │   │  옵션:                                              │   │ │
│   │   │  • -r: pcap 읽기                                   │   │ │
│   │   │  • -i: 인터페이스                                 │   │ │
│   │   │  • -Y: 디스플레이 필터                             │   │ │
│   │   │  • -T: 출력 형식 (json, text)                       │   │ │
│   │   │  • -c: 패킷 수 제한                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              TShark vs Wireshark                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 항목       │ TShark   │ Wireshark│               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 인터페이스 │ CLI      │ GUI      │               │   │ │
│   │   │  │ 자동화     │ 가능     │ 어려움   │               │   │ │
│   │   │  │ 스크립팅   │ 가능     │ 제한     │               │   │ │
│   │   │  │ 시각화     │ 제한     │ 우수     │               │   │ │
│   │   │  | 리소스     │ 적음      │ 많음     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: TShark는 Wireshark의 디스섬블리 필터를 CLI에서 사용할 수 있습니다.

**💡 비유**: TShark는 터미널용 Wireshark입니다.

**📢 섹션 요약 비유:** TShark = 자동화!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Display Filter** | -Y 옵션 |
| **Output Format** | -T 옵션 |
| **JSON** | 구조화 출력 |

**비교표: 출력 형식**

```
+------------+----------+
| 형식       │ 설명     │
+------------+----------+
| text       │ 기본     │
| json       │ 구조화   │
| pdml       │ XML      │
| psml       │ XML      │
+------------+----------+
```

**핵심 알고리즸:** 디스섬블리 필터

**📢 섹션 요약 비유:** -Y http = HTTP 필터!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 자동화** → tshark -T json
**시나리오 2: HTTP 분석** → tshark -Y http
**시나리오 3: 통계** → tshark -q io,ip

**도입 체크리스트:** tshark 설치, 자동화 스크립트

**📢 섹션 요약 비유:** tshark -T json = 구조화!
