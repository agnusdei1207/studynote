# Static IP - 고정 IP

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 수동 할당 IP
> 2. **가치**: 일관성과 안정성
> 3. **융합:** Manual, Server, Reservation

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Static IP는 수동으로 설정하는 고정 IP 주소입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Static IP                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              고정 IP 설정                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  설정 정보:                                               │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  IP Address:     192.168.1.10                   ││   │ │
│   │   │  │  Subnet Mask:    255.255.255.0                  ││   │ │
│   │   │  │  Default Gateway: 192.168.1.1                   ││   │ │
│   │   │  │  DNS Server:     8.8.8.8                        ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  설정 방법:                                              │   │ │
│   │   │  1. 수동 설정 (직접 입력)                                 │   │ │
│   │   │  2. DHCP 예약 (MAC 기반 고정)                            │   │ │
│   │   │                                                      │   │ │
│   │   │  사용 용도:                                              │   │ │
│   │   │  - 서버 (웹, DB, 파일)                                   │   │ │
│   │   │  - 네트워크 장비 (라우터, 스위치)                          │   │ │
│   │   │  - 프린터                                                │   │ │
│   │   │  - 보안 시스템                                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Static IP는 중요 서비스의 안정성을 보장합니다.

**📢 섹션 요약 비유:** Static IP = 고정 전화번호!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Manual** | 수동 설정 |
| **Reservation** | DHCP 예약 |
| **Consistency** | 일관성 |
| **Documentation** | 문서화 |
| **Conflict** | 충돌 방지 |

**핵심 알고리듬:** 수동 설정 → 검증 → 적용

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Static vs Dynamic IP**

| 항목 | Static | Dynamic |
|:---|:---|:---|
| **할당** | 수동 | 자동 |
| **변경** | 없음 | 있음 |
| **관리** | 복잡 | 쉬움 |
| **용도** | 서버 | 클라이언트 |
