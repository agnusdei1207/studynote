# Crosstalk - 누화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 간섭
> 2. **가치**: 통신 품질 저하
> 3. **융합:** NEXT, FEXT, Twisted Pair

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Crosstalk는 인접 회선 간 신호 간섭입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Crosstalk                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              누화 현상                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  누화 유형:                                               │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  NEXT (Near End Crosstalk):                     ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  ┌────────┐         ┌────────┐                  ││   │ │
│   │   │  │  │ TX  A  │─────────│ TX  A  │                  ││   │ │
│   │   │  │  └────────┘         └────────┘                  ││   │ │
│   │   │  │       │↘                 ↙│                      ││   │ │
│   │   │  │       │  ╲             ╱  │                      ││   │ │
│   │   │  │       │    ╲         ╱    │                      ││   │ │
│   │   │  │  ┌────┴──┐  ╲     ╱  ┌────┴──┐                  ││   │ │
│   │   │  │  │ RX  A │   ╲   ╱   │ RX  A │                  ││   │ │
│   │   │  │  └───────┘    ╲ ╱    └───────┘                  ││   │ │
│   │   │  │       ↑        X        ↑                       ││   │ │
│   │   │  │       │       ╱ ╲       │                       ││   │ │
│   │   │  │  (같은 쪽에서 발생)                               ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  FEXT (Far End Crosstalk):                      ││   │ │
│   │   │  │  (반대쪽에서 발생)                                ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  완화 방법:                                             │   │ │
│   │   │  - 트위스티드 페어 (꼬임선)                               │   │ │
│   │   │  - 차폐 (STP, FTP)                                      │   │ │
│   │   │  - 적절한 케이블 간격                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Crosstalk는 케이블 성능에 영향을 미칩니다.

**📢 섹션 요약 비유:** Crosstalk = 옆 방 소음!
