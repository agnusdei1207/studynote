# Error Correction - 오류 수정

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 오류 자동 복구
> 2. **가치**: 재전송 없는 신뢰성
> 3. **융합:** FEC, ECC, Hamming Code

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Error Correction은 오류를 자동으로 수정합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Error Correction                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              오류 수정 방법                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ARQ (Automatic Repeat reQuest):                      │   │ │
│   │   │  오류 감지 → 재전송 요청                                │   │ │
│   │   │  Stop-and-Wait, Go-Back-N, Selective Repeat          │   │ │
│   │   │                                                      │   │ │
│   │   │  FEC (Forward Error Correction):                      │   │ │
│   │   │  오류 수정 정보를 미리 포함                              │   │ │
│   │   │  재전송 없이 수신측에서 수정                              │   │ │
│   │   │                                                      │   │ │
│   │   │  Hamming Code:                                        │   │ │
│   │   │  Data: 4bit + Parity: 3bit = 7bit                    │   │ │
│   │   │  1비트 오류 수정 가능                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  Reed-Solomon:                                        │   │ │
│   │   │  블록 단위 오류 수정                                    │   │ │
│   │   │  CD, DVD, QR Code                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  ECC Memory:                                          │   │ │
│   │   │  서버용 RAM                                           │   │ │
│   │   │  1비트 오류 수정, 2비트 오류 감지                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 오류 수정은 재전송 지연을 피합니다.

**📢 섹션 요약 비유:** Error Correction = 자동 수정 펜!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **FEC** | 전방 오류 수정 |
| **ARQ** | 자동 재전송 요청 |
| **ECC** | 오류 수정 코드 |
| **Hamming** | 해밍 코드 |
| **Parity** | 패리티 |

**핵심 알고리즸:** 데이터 + 중복 → 전송 → 오류 수정 → 복구
