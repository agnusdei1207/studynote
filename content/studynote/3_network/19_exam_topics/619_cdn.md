# CDN (Content Delivery Network) - 콘텐츠 전송 네트워크

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 분산 캐시 서버
> 2. **가치**: 지연 감소, 대역폭 절약
> 3. **융합:** Cache, Edge, DNS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CDN은 전 세계 엣지 서버에서 콘텐츠를 캐시합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CDN                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CDN 구성                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  User (Korea)                                       │   │ │
│   │   │       │                                              │   │ │
│   │   │       ↓                                              │   │ │
│   │   │  ┌─────────┐                                        │   │ │
│   │   │  │Edge (KR)│ ← Cache Hit                            │   │ │
│   │   │  └─────────┘                                        │   │ │
│   │   │       ↑                                              │   │ │
│   │   │       │ (Cache Miss)                                 │   │ │
│   │   │  Origin Server (US)                                 │   │ │
│   │   │  ┌─────────┐                                        │   │ │
│   │   │  │ Origin  │                                        │   │ │
│   │   │  └─────────┘                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  Edge 서버 위치:                                     │   │ │
│   │   │  - Cloudflare, Akamai, AWS CloudFront               │   │ │
│   │   │  - 전 세계 분산                                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: CDN은 사용자와 가까운 엣지에서 콘텐츠를 제공합니다.

**📢 섹션 요약 비유:** CDN = 지역 창고!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Edge Server** | 엣지 캐시 |
| **Origin Server** | 원본 서버 |
| **PoP** | Point of Presence |
| **Cache Policy** | 캐시 정책 |

**핵심 알고리즸:** 요청 → Edge → (Miss) Origin → Cache → 응답
