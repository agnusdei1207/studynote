# 핸드오버 - Handover

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이동 중 연결 유지
> 2. **가치**: 끊김 없는 통신
> 3. **융합**: Hard/Soft, X2/S1 Handover

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

핸드오버는 이동 중 셀 간 연결을 전환합니다.

```

│   │   │      │    1. 연결 해제              │                │   │ │
```

**💡 비유**: 핸드오버는 택시의 무전기 기지국 전환입니다.
