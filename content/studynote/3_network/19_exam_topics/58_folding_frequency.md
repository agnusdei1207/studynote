# 폴딩 주파수 (Folding Frequency)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 샘플링 주파수의 절반 (fs/2)
> 2. **가치**: 에일리어싱 발생 기준점
> 3. **융합**: Nyquist 주파수, 대역폭과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

폴딩 주파수(Folding Frequency)는 샘플링 주파수의 절반으로, 이 주파수를 넘는 신호는 에일리어싱이 발생합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    폴딩 주파수 (Folding Frequency)                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              폴딩 주파수 정의                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  폴딩 주파수: fN = fs/2                               │   │ │
│   │   │  (= Nyquist 주파수)                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  예: fs = 44.1kHz (CD)                                │   │ │
│   │   │      fN = 22.05kHz                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                 │   │ │
│   │   │  • fs/2 이하 주파수: 정상 복원                         │   │ │
│   │   │  • fs/2 초과 주파수: 에일리어싱 발생                    │   │ │
│   │   │  • 스펙트럼이 fs/2에서 "접힘" (folding)                │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              주파수 접힘 현상                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  스펙트럼:                                               │   │ │
│   │   │                                                      │   │ │
│   │   │  f=0    fN           fs    fs+fN                    │   │ │
│   │   │  ↓      ↓            ↓     ↓                         │   │ │
│   │   │  ├──────┼────────────┼─────┼───→ f                   │   │ │
│   │   │  │신호  │  (에일리어싱) │     │                        │   │ │
│   │   │  │      │←─── 접힘 ───→│     │                        │   │ │
│   │   │  │      │              │     │                        │   │ │
│   │   │        fN을 넘으면 fN 아래로 접혀서 나타남             │   │ │
│   │   │                                                      │   │ │
│   │   │  별칭 주파수: f_alias = |f - k·fs|                   │   │ │
│   │   │  (k는 정수)                                            │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 폴딩 주파수는 디지털 신호 처리의 "마지노선"입니다.

**💡 비유**: 폴딩 주파수는 종이를 반으로 접는 선입니다.

**📢 섹션 요약 비유**: fN = fs/2!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **fN** | 폴딩 주파수 | fs/2 | 경계선 |
| **fs** | 샘플링 주파수 | 기준 | 속도 |
| **별칭** | 에일리어싱 | 접힌 신호 | 가짜 |
| **대역폭** | 유효 범위 | fN | 폭 |

**비교표**

```
+------------+----------+----------+
| 시스템     | fs       | fN       |
+------------+----------+----------+
| 전화       | 8kHz     | 4kHz     |
| CD         | 44.1kHz  | 22.05kHz |
| DVD        | 48kHz    | 24kHz    |
| 전문 오디오| 96kHz    | 48kHz    |
+------------+----------+----------+
```

**핵심 알고리즘**: fs 설정 → fN = fs/2 → 필터 설계

**📢 섹션 요약 비유**: 신호 대역폭 < fN 이어야 함!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 주파수 용어**

| 용어 | 정의 | 관계 |
|:---|:---|:---|
| **fs** | 샘플링 주파수 | 기준 |
| **fN** | Nyquist 주파수 | fs/2 |
| **fmax** | 신호 최대 주파수 | ≤ fN |
| **2fmax** | Nyquist 속도 | 최소 fs |

**📢 섹션 요약 비유**: Nyquist 주파수 = 폴딩 주파수!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: CD** → fN = 22.05kHz, 오디오 20kHz
**시나리오 2: 전화** → fN = 4kHz, 음성 3.4kHz
**시나리오 3: 영상** → fN 고려 필터 설계

**도입 체크리스트**: fs, 대역폭, 필터

**📢 섹션 요약 비유**: 여유 있게 fmax < fN!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 에일리어싱 방지 기준

**미래 전망**: 고해상도 오디오/영상

**참고 표준**: Shannon-Nyquist

**📢 섹션 요약 비유**: 폴딩 주파수는 디지털의 한계!
