# 1000BASE-T (Gigabit Ethernet)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 1Gbps UTP 이더넷, 4페어 전이중
> 2. **가치**: 데스크톱 표준, Cat5e 이상
> 3. **융합**: PAM-5, DSP 기술과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

1000BASE-T는 UTP 케이블로 1Gbps를 전송하는 이더넷 표준으로, 4페어 모두 사용하고 PAM-5 변조를 합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    1000BASE-T (Gigabit Ethernet)                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              기술 특징                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  항목        │  1000BASE-T                           │   │ │
│   │   │  ────────────┼─────────────────────────────────────  │   │ │
│   │   │  속도        │  1000Mbps (1Gbps)                     │   │ │
│   │   │  케이블      │  Cat5e 이상                            │   │ │
│   │   │  거리        │  100m                                 │   │ │
│   │   │  페어 사용   │  4페어 모두 사용                       │   │ │
│   │   │  변조        │  PAM-5 (5단계 펄스 진폭 변조)          │   │ │
│   │   │  코딩        │  8B/10B                               │   │ │
│   │   │  전송 방식   │  전이중 (각 페어가 송수신 동시)        │   │ │
│   │   │  라인 속도   │  125Mbaud/페어 × 4페어 = 1Gbps       │   │ │
│   │   │  표준        │  IEEE 802.3ab (1999)                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PAM-5 변조                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  PAM-5 (Pulse Amplitude Modulation - 5 levels):        │   │ │
│   │   │                                                      │   │ │
│   │   │  전압 레벨:                                            │   │ │
│   │   │      +2 ─────────────────                             │   │ │
│   │   │      +1 ─────────────────                             │   │ │
│   │   │       0 ─────────────────  ← 5개 레벨                 │   │ │
│   │   │      -1 ─────────────────                             │   │ │
│   │   │      -2 ─────────────────                             │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                   │   │ │
│   │   │  • 심볼당 2비트 전송 (4레벨) + 오류검출(1레벨)           │   │ │
│   │   │  • 125Mbaud × 2비트 × 4페어 = 1Gbps                    │   │ │
│   │   │  • DSP로 에코/NEXT 보정                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  4페어 구조:                                             │   │ │
│   │   │  ┌─────────────────────────────────────┐               │   │ │
│   │   │  │ 페어 1-2: 송신 + 수신 (전이중)       │               │   │ │
│   │   │  │ 페어 3-6: 송신 + 수신 (전이중)       │               │   │ │
│   │   │  │ 페어 4-5: 송신 + 수신 (전이중)       │               │   │ │
│   │   │  │ 페어 7-8: 송신 + 수신 (전이중)       │               │   │ │
│   │   │  └─────────────────────────────────────┘               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 1000BASE-T는 4페어 모두 전이중으로 사용하여 1Gbps를 달성합니다.

**💡 비유**: 1000BASE-T는 4차선 도로에서 양방향 통행을 하는 것입니다.

**📢 섹션 요약 비유**: 4페어 + PAM-5 = 1Gbps!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **PAM-5** | 변조 | 5레벨 | 멀티레벨 |
| **DSP** | 보정 | 에코 제거 | 노이즈캔슬 |
| **4페어** | 대역폭 | 동시 송수신 | 4차선 |

**비교표**

```
+------------+----------+----------+
| 항목       | 100M     | 1000M    |
+------------+----------+----------+
| 페어       | 2        | 4        |
| 변조       | MLT-3    | PAM-5    |
| Cat 요구   | Cat5     | Cat5e    |
| 전이중     | 페어 분리 | 하이브리드|
+------------+----------+----------+
```

**핵심 알고리즸**: 에코 제거, NEXT 보정,Auto-Negotiation

**📢 섹션 요약 비유**: Cat5e = 1Gbps 100m!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: Cat별 성능**

| Cat | 1G 거리 | 10G 거리 |
|:---|:---:|:---:|
| **Cat5e** | 100m | - |
| **Cat6** | 100m | 55m |
| **Cat6a** | 100m | 100m |

**📢 섹션 요약 비유**: Cat5e는 1G 한계!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데스크톱** → 1000BASE-T
**시나리오 2: 서버** → 10GBASE-T
**시나리오 3: 레거시 Cat5** → 100M 또는 케이블 교체

**도입 체크리스트**: Cat5e+, 스위치 용량, 전력

**📢 섹션 요약 비유:** 1G는 현재 표준!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 10배 속도 향상

**미래 전망**: 2.5G/5G로 진화

**참고 표준**: IEEE 802.3ab

**📢 섹션 요약 비유**: 1000BASE-T는 데스크톱 표준!
