# 로밍 - Roaming

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 타 통신사 네트워크 사용
> 2. **가치**: 해외 통신, 서비스 연속성
> 3. **융합**: Home/Visited, CAMEL, LTE Roaming

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **HPLMN** | Home PLMN |
| **VPLMN** | Visited PLMN |
| **GRX/IPX** | 로밍 네트워크 |
| **S6a/d** | 인터페이스 |

**비교표: 로밍 인터페이스**

```
+------------+----------+----------+----------+
| 세대       │ 인터페이스│ 프로토콜 │ 용도     │
+------------+----------+----------+----------+
| 2G         │ C/D      │ MAP      │ 음성     │
| 3G         │ Gr       │ MAP      │ 데이터   │
| LTE        │ S6a      │ Diameter │ 4G       │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 인증, 과금

**📢 섹션 요약 비유:** GRX/IPX = 로밍 백본!
