### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **SM** | Streaming Multiprocessor |
| **CUDA Core** | CUDA 코어 |
| **Warp** | 워프 (32 스레드) |
| **Shared Memory** | 공유 메모리 |
| **Global Memory** | 글로벌 메모리 |

**핵심 알고리즸:** 데이터 전송 → 커널 실행 → 결과 전송

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**CPU vs GPU**

| 항목 | CPU | GPU |
|:---|:---|:---|
| **코어 수** | 적음 | 많음 |
| **코어 성능** | 높음 | 낮음 |
| **메모리** | 적음 | 많음 |
| **용도** | 범용 | 병렬 |
