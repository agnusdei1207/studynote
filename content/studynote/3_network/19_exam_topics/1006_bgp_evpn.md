# BGP EVPN - BGP 이더넷 VPN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: MP-BGP 기반 EVPN
> 2. **가치**: 분산형 MAC 학습
> 3. **융합:** NLRI, Route Types, MP-BGP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

BGP EVPN은 MP-BGP 확장을 이용한 EVPN 구현입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BGP EVPN                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              BGP EVPN 동작                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  MP-BGP EVPN:                                           │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  NLRI (Network Layer Reachability Info):         ││   │ │
│   │   │  │  ┌─────────────────────────────────────────────┐││   │ │
│   │   │  │  │ AFI: 25 (L2VPN)                             │││   │ │
│   │   │  │  │ SAFI: 70 (EVPN)                             │││   │ │
│   │   │  │  └─────────────────────────────────────────────┘││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  경로 교환:                                        ││   │ │
│   │   │  │  ┌───────────────┐       ┌───────────────┐      ││   │ │
│   │   │  │  │   VTEP 1      │       │   VTEP 2      │      ││   │ │
│   │   │  │  │ MAC: AA:BB:CC │◄─────►│ MAC: DD:EE:FF │      ││   │ │
│   │   │  │  │ IP: 10.1.1.1  │  BGP  │ IP: 10.1.1.2  │      ││   │ │
│   │   │  │  │ VNI: 10010    │ EVPN  │ VNI: 10010    │      ││   │ │
│   │   │  │  └───────────────┘       └───────────────┘      ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  장점:                                            ││   │ │
│   │   │  │  - 제어 평면 분리                                  ││   │ │
│   │   │  │  - 플러딩 제거                                     ││   │ │
│   │   │  │  - 멀티패스 지원                                   ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: BGP EVPN은 기존 BGP 인프라를 활용합니다.

**📢 섹션 요약 비유:** BGP EVPN = L2를 위한 라우팅!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **AFI 25** | L2VPN 주소 패밀리 |
| **SAFI 70** | EVPN |
| **NLRI** | Network Layer Reachability |
| **Route Distinguisher** | 경로 식별자 |
| **Route Target** | 경로 타겟 |

**핵심 알고리듬:** MAC 학습 → BGP 업데이트 → 피어 전파 → FIB 갱신

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**BGP EVPN vs 전통적 L2**

| 항목 | BGP EVPN | 전통적 L2 |
|:---|:---|:---|
| **MAC 학습** | 제어 평면 | 데이터 평면 |
| **플러딩** | 없음 | 있음 |
| **루프 방지** | BGP | STP |
