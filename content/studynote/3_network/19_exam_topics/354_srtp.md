# SRTP - 보안 RTP

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: RTP 미디어 암호화 프로토콜

> 3. **융합**: VoIP, WebRTC, 영상회의

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SRTP(Secure RTP)는 RTP 미디어를 암호화합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SRTP                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SRTP 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Sender]                                           │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ RTP Packet                                     │   │ │
│   │   │     │                                                │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           SRTP Encryption                   │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • 암호화 (AES)                              │    │   │ │
│   │   │  │  • 인증 (HMAC-SHA1)                          │    │   │ │
│   │   │  │  • 키 관리 (DTLS-SRTP)                       │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ SRTP Packet                                  │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [Network]                                          │   │ │
│   │   │     │                                                │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  [Receiver]                                         │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ 복호화 + 인증 검증                              │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  [재생]                                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SRTP 보안 기능                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 기능       │ 알고리즘 │ 가치     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 기밀성     │ AES      │ 암호화   │               │   │ │
│   │   │  │ 무결성     │ HMAC     │ 변조방지 │               │   │ │
│   │   │  | 재생 방지  │ ROC      │ 순서     │               │   │ │
│   │   │  │ 인증       │ MKI      │ 키 식별  │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SRTP는 AES 암호화와 HMAC 인증으로 미디어를 보호합니다.

**💡 비유**: SRTP는 통화의 금고입니다. 내용을 안전하게 보관합니다.

**📢 섹션 요약 비유:** SRTP = 미디어 암호화!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Master Key** | 마스터 키 |
| **Session Key** | 세션 키 |
| **ROC** | 롤오버 카운터 |
| **MKI** | 마스터 키 식별자 |

**비교표: SRTP vs RTP**

```
+------------+----------+----------+
| 항목       │ RTP      │ SRTP     │
+------------+----------+----------+
| 암호화     │ 없음     │ AES      │
| 인증       │ 없음     │ HMAC     │
| 오버헤드   │ 없음     │ 있음     │
| 보안       │ 없음     │ 있음     │
+------------+----------+----------+
```

**핵심 알고리즸:** AES-CTR, HMAC-SHA1

**📢 섹션 요약 비유:** DTLS-SRTP = 키 교환!
