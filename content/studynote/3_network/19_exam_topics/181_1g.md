# 1G 이동통신 (1G)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이동통신의 시작
> 2. **가치**: 광대역, 고속
> 3. **융합**: 4G/5G, M2M, IoT와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

1G(1st Generation)은 1990년대 초기 이동통신 서비스로, FDMA, TDMA, cdmaOne, cdma2000 1x EV-DO, GSM, GPRS, EDGE

```
┌─────────────────────────────────────────────────────────────────────┐
│                    1G 기술별                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │  기술         │ 전송     │ 다중접속    │ 데이터     │   │
│   │  ───────────┼──────────┼──────────────┼────────────│   │
│   │  FDMA        │ 순방향   │ 없음         │ 음성        │   │
│   │  TDMA        │ 순방향   │ 있음         │ 음성/저속   │   │
│   │  cdmaOne     │ 순방향   │ 없음         │ 14.4kbps     │   │
│   │  cdma2000   │ 순방향   │ 있음         │ 153kbps     │   │
│   │  GPRS        │ 패킷      │ 있음         │ 171kbps     │   │
│   │  EDGE        │ 패킷      │ 있음         │ 384kbps     │   │
│   │  EV-DO        │ 패킷      │ 있음         │ 384kbps     │   │
│   │  GSM         │ 회선/패킷│ 있음         │ 9.6-14.4kbps │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────────┐   │
│   │              1G 특성                                      │   │
│   │                                                              │   │
│   │  장점:                                                   │   │
│   │  • 최초의 패킷 데이터 서비스                           │   │
│   │  • 인터넷 접속 가능                                 │   │
│   │  • SMS/MMS 통합                                 │   │
│   │                                                      │   │
│   │  단점:                                                   │   │
│   │  • 낮은 전송 속도                                  │   │
│   │  • 음성 위주 설계                                │   │
│   │  • 보안 취약                                        │   │
│   │                                                      │   │
│   └─────────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 1G은 이동통신의 시작으로, 음성 위주 설계로 인터넷 접속을 가능하게 했습니다.

**💡 비유**: 1G는 전화기로, 처음으로 무선 전화가 가능해졌습니다.

**📢 섹션 요약 비유:** 1G = 이동통신의 태동!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 기술 | 전송 방식 | 다중접속 | 최고 속도 |
|:---|:---|:---:|:---|
| **FDMA** | 순방향 | 없음 | 14.4kbps |
| **TDMA** | 순방향 | 있음 | 14.4kbps |
| **GSM** | 회선/패킷 | 있음 | 14.4kbps |
| **GPRS** | 패킷 | 있음 | 171kbps |
| **EDGE** | 패킷 | 있음 | 384kbps |

**비교표**

```
+------------+----------+----------+
| 항목       │ 1G       │ 2G       │
+------------+----------+----------+
| 전송       │ 음성/데이터 │ 데이터   │
│ 속도       │ <14.4kbps│ 64-144kbps│
│ 서비스     │ 음성/문자 │ 멀티미디어│
+------------+----------+----------+
```

**핵심 알고리즸:** 셀 구조, 핸드오프. 음성 부호화

**📢 섹션 요약 비유:** GSM = 2G 전 세계 표준!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 레거시 음성** → GSM
**시나리오 2: 간단한 데이터** → GPRS
**시나리오 3: 데이터** → EDGE

**도입 체크리스트:** 커버리지, 단말기 호환성, 서비스 요구

**📢 섹션 요약 비유:** 1G는 역사!
