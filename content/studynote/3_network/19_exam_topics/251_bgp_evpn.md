# BGP EVPN - 이더넷 VPN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: BGP 기반 L2/L3 VPN
> 2. **가치**: 데이터센터互联, 멀티테넌시
> 3. **융합**: VXLAN, Spine-Leaf와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

BGP EVPN(Ethernet VPN)은 BGP를 사용하여 L2/L3 VPN 서비스를 제공하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    BGP EVPN                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              EVPN 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Spine]              [Spine]                        │   │ │
│   │   │      │                   │                            │   │ │
│   │   │      │   BGP EVPN       │                            │   │ │
│   │   │      ↓                   ↓                            │   │ │
│   │   │  [Leaf 1]           [Leaf 2]                         │   │ │
│   │   │      │                   │                            │   │ │
│   │   │      ↓                   ↓                            │   │ │
│   │   │  [서버]              [서버]                          │   │ │
│   │   │  VLAN 10             VLAN 10                         │   │ │
│   │   │                                                      │   │ │
│   │   │  특징:                                                │   │ │
│   │   │  • BGP Type 2: MAC/IP 경로                           │   │ │
│   │   │  • BGP Type 3: 멀티캐스트                            │   │ │
│   │   │  • BGP Type 5: Prefix 경로                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              EVPN Route Types                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Type │ 이름              │ 용도                      │   │ │
│   │   │  ─────┼──────────────────┼─────────────────────────│   │ │
│   │   │  1    │ Ethernet Auto-Discovery │ 멀티홈          │   │ │
│   │   │  2    │ MAC/IP Advertisement   │ MAC/IP 광고      │   │ │
│   │   │  3    │ Inclusive Multicast    │ BUM 트래픽        │   │ │
│   │   │  4    │ Ethernet Segment       │ 멀티홈            │   │ │
│   │   │  5    │ IP Prefix              │ L3 라우팅         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: EVPN은 BGP로 MAC/IP 정보를 분배합니다.

**💡 비유**: EVPN은 스마트한 주소록입니다. 자동으로 업데이트됩니다.

**📢 섹션 요약 비유:** EVPN = BGP + L2VPN!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VTEP** | VXLAN Tunnel Endpoint |
| **VNI** | VXLAN Network Identifier |
| **ESI** | Ethernet Segment ID |
| **RD** | Route Distinguisher |
| **RT** | Route Target |

**비교표: EVPN vs 传统 L2VPN**

```
+------------+----------+----------+
| 항목       │ 传统 L2  │ EVPN     │
+------------+----------+----------+
| 제어 평면  │ 플러딩   │ BGP      │
| MAC 학습   │ 데이터   │ 컨트롤   │
| 멀티캐스트 │ 필요     │ Type 3   │
| 멀티테넌시 │ 제한     │ RT 기반  │
+------------+----------+----------+
```

**핵심 알고리즸:** BGP 기반 MAC 학습

**📢 섹션 요약 비유:** Type 2 = MAC/IP 광고!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 오버레이**

| 기술 | L2/L3 | 용도 |
|:---|:---:|:---|
| **VXLAN** | L2 | 오버레이 |
| **EVPN** | L2/L3 | 컨트롤 플레인 |
| **Geneve** | L2 | 일반화 |

**📢 섹션 요약 비유:** EVPN + VXLAN = 데이터센터 표준!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 데이터센터** → EVPN-VXLAN

**도입 체크리스트:** Spine-Leaf, BGP, VXLAN

**📢 섹션 요약 비유:** EVPN은 데이터센터의 표준!
