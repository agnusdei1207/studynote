# MapReduce - 맵리듀스

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 대용량 데이터 병렬 처리
> 2. **가치**: 확장성, 내결함성
> 3. **융합:** Hadoop, Big Data, Distributed Computing

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MapReduce는 대용량 데이터를 병렬로 처리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MapReduce                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MapReduce 흐름                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Input Data                                          │   │ │
│   │   │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐                   │   │ │
│   │   │  │ D1  │ │ D2  │ │ D3  │ │ D4  │                   │   │ │
│   │   │  └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘                   │   │ │
│   │   │     │       │       │       │                        │   │ │
│   │   │     ↓       ↓       ↓       ↓                        │   │ │
│   │   │  ┌─────────────────────────────────────┐            │   │ │
│   │   │  │            Map Phase                 │            │   │ │
│   │   │  │  Mapper 1 │ Mapper 2 │ Mapper 3     │            │   │ │
│   │   │  │  (K1,V1)  │ (K2,V2)  │ (K3,V3)      │            │   │ │
│   │   │  └─────────────────────────────────────┘            │   │ │
│   │   │                    ↓                                 │   │ │
│   │   │  ┌─────────────────────────────────────┐            │   │ │
│   │   │  │         Shuffle & Sort               │            │   │ │
│   │   │  └─────────────────────────────────────┘            │   │ │
│   │   │                    ↓                                 │   │ │
│   │   │  ┌─────────────────────────────────────┐            │   │ │
│   │   │  │           Reduce Phase               │            │   │ │
│   │   │  │  Reducer 1 │ Reducer 2               │            │   │ │
│   │   │  │  (K1, [V1..]) → Result              │            │   │ │
│   │   │  └─────────────────────────────────────┘            │   │ │
│   │   │                    ↓                                 │   │ │
│   │   │              Output Data                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MapReduce는 Map → Shuffle → Reduce로 처리합니다.

**📢 섹션 요약 비유:** MapReduce = 분업 처리!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

|:---|:---|
| **Map** | 데이터 변환 |
