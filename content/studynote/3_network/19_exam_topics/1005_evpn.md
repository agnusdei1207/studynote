# EVPN - 이더넷 VPN

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: BGP 기반 L2 VPN
> 2. **가치**: VXLAN 제어 평면
> 3. **융합:** BGP, VXLAN, MP-BGP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

EVPN (Ethernet VPN)은 BGP를 이용한 L2 VPN 기술입니다.

> **해설**: EVPN은 VXLAN의 제어 평면으로 플러딩 문제를 해결합니다.

**📢 섹션 요약 비유:** EVPN = L2를 위한 BGP!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **EVPN Route** | EVPN 경로 타입 |
| **VTEP** | VXLAN 터널 종단 |
| **BGP RR** | Route Reflector |
| **MAC-VRF** | MAC 가상 라우팅 |
| **ESI** | Ethernet Segment ID |

**핵심 알고리듬:** MAC 학습 → BGP 광고 → VTEP 동기화 → 유니캐스트

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**EVPN Route Types**

| Type | 이름 | 용도 |
|:---|:---|:---|
| **1** | Ethernet Auto-Discovery | 다중 홈 |
| **2** | MAC/IP Advertisement | MAC/IP 학습 |
| **3** | Inclusive Multicast | BUM 트래픽 |
| **4** | Ethernet Segment | 다중 홈 |
| **5** | IP Prefix | L3 라우팅 |
