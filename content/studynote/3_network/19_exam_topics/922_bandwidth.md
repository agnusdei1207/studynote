# Bandwidth - 대역폭

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 전송 용량
> 2. **가치**: 네트워크 성능 지표
> 3. **융합:** Throughput, Latency, Bottleneck

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Bandwidth는 단위 시간당 전송 가능한 데이터 양입니다.


### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Capacity** | 용량 |
| **Utilization** | 사용률 |
| **Bottleneck** | 병목 |
| **Overhead** | 오버헤드 |
| **MTU** | 최대 전송 단위 |
