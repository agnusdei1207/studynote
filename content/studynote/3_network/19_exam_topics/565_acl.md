# ACL (Access Control List) - 접근 제어 목록

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 트래픽 허용/차단 규칙 목록
> 2. **가치**: 세밀한 접근 제어
> 3. **융합:** Firewall, Router, Security Group

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ACL은 트래픽 필터링 규칙의 목록입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ACL                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ACL 예시                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Rule  Action  Source          Dest          Port    │   │ │
│   │   │  ─────────────────────────────────────────────────── │   │ │
│   │   │  10    PERMIT  192.168.1.0/24  10.0.0.1     80      │   │ │
│   │   │  20    PERMIT  192.168.1.0/24  10.0.0.1     443     │   │ │
│   │   │  30    DENY    any             any          any     │   │ │
│   │   │                                                      │   │ │
│   │   │  → 위에서 아래로 순차적 매칭                          │   │ │
│   │   │  → 첫 매칭 Rule 적용                                 │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: ACL은 순차적으로 평가되며 첫 매칭이 적용됩니다.

**📢 섹션 요약 비유:** ACL = 출입 명단!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Standard ACL** | 출발지 IP만 |
| **Extended ACL** | 출발지/목적지/포트 |
| **Named ACL** | 이름 기반 |
| **Implicit Deny** | 암시적 거부 |

**핵심 알고리즸:** Top-down 매칭 → Action 수행

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**ACL 유형**

| 유형 | 조건 | 용도 |
|:---|:---|:---|
| **Standard** | Source | 기본 |
| **Extended** | Src/Dst/Port | 세밀 |
| **Reflexive** | 응답 | Stateful |
