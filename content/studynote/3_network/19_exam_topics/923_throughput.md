### Ⅰ. 개요 (Context & Background)

**개념 정의**

**📢 섹션 요약 비유:** Throughput = 실제 도로 통행량!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Goodput** | 유효 데이터 |
| **Overhead** | 오버헤드 |
| **Efficiency** | 효율성 |
| **Loss** | 손실 |
| **Retransmit** | 재전송 |

**핵심 알고리듬:** Throughput = (총 데이터 - 오버헤드) / 시간
