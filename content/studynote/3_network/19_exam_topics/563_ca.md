# CA (Certificate Authority) - 인증 기관

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디지털 인증서 발급 기관
> 2. **가치**: 신뢰 보증
> 3. **융합:** PKI, Root CA, Intermediate CA

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CA는 인증서를 발급하고 서명하는 신뢰 기관입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CA                                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CA 역할                                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 신원 검증 (DV/OV/EV)                             │   │ │
│   │   │  2. 인증서 발급                                      │   │ │
│   │   │  3. 인증서 서명 (개인키)                             │   │ │
│   │   │  4. 폐기 관리 (CRL/OCSP)                             │   │ │
│   │   │                                                      │   │ │
│   │   │  주요 CA:                                            │   │ │
│   │   │  - DigiCert, Let's Encrypt, GlobalSign              │   │ │
│   │   │  - Comodo, Sectigo, GoDaddy                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: CA는 브라우저/OS에 기본 신뢰되어 있습니다.

**📢 섹션 요약 비유:** CA = 공증 사무소!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Root CA** | 최상위 CA |
| **Intermediate CA** | 중간 CA |
| **Leaf CA** | 말단 CA |
| **Trust Store** | 신뢰 저장소 |

**핵심 알고리즸:** Root 신뢰 → 체인 검증

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**CA 유형**

| 유형 | 설명 |
|:---|:---|
| **Public CA** | 공인 CA |
| **Private CA** | 사설 CA |
| **Root** | 자체 서명 |
| **Intermediate** | 위임 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
