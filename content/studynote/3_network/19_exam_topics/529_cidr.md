# CIDR (Classless Inter-Domain Routing) - 클래스 없는 도메인 간 라우팅

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 클래스 없는 주소 할당
> 2. **가치**: IP 주소 절약, 라우팅 효율
> 3. **융합:** Subnet, Supernet, Route Aggregation

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

CIDR은 클래스 경계 없이 주소를 할당합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    CIDR                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              CIDR 표기법                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Classful:                                           │   │ │
│   │   │  - Class A: /8                                       │   │ │
│   │   │  - Class B: /16                                      │   │ │
│   │   │  - Class C: /24                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  CIDR:                                               │   │ │
│   │   │  - 192.168.0.0/18 (18비트 네트워크)                  │   │ │
│   │   │  - 10.0.0.0/12 (12비트 네트워크)                     │   │ │
│   │   │  - 172.16.0.0/20 (20비트 네트워크)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  유연한 크기 할당 가능                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: CIDR은 클래스의 제한을 없애고 유연한 할당을 가능하게 합니다.

**📢 섹션 요약 비유:** CIDR = 자유로운 분할!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Prefix** | 네트워크 비트 수 |
| **Notation** | IP/Prefix |
| **Block** | 연속된 주소 공간 |
| **Aggregation** | 경로 요약 |

**핵심 알고리즸:** prefix 일치 → 같은 네트워크

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**Classful vs CIDR**

| 항목 | Classful | CIDR |
|:---|:---|:---|
| **할당** | 고정 | 유연 |
| **낭비** | 많음 | 적음 |
| **라우팅** | 큰 테이블 | 작은 테이블 |
