# WFQ (Weighted Fair Queuing) - 가중치 공정 큐잉

#### 핵심 인사이트 (3줄 요약)

### Ⅰ. 개요 (Context & Background)

**개념 정의**

WFQ(Weighted Fair Queuing)는 각 플로우에 가중치를 부여하여 공정하게 대역폭을 분배하는 스케줄링 알고리즘입니다.

> **해설**: WFQ는 IP Precedence 값을 기반으로 가중치를 자동 계산합니다.

**💡 비유**: WFQ는 버스 좌석 배정입니다. 우등 고객이 더 많은 좌석을 얻습니다.

**📢 섹션 요약 비유:** WFQ = 자동 공정 분배!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Flow** | 5-tuple 기반 |
| **Weight** | Precedence + 1 |
| **Finish Time** | 전송 완료 시간 |
| **BQ** | Busy Queue |

**비교표: 큐잉 방식 비교**

```
+------------+----------+----------+
| 방식       │ 분류     │ 공정성   │
+------------+----------+----------+
| FIFO       │ 없음     │ 없음     │
| PQ         │ 우선순위 │ 없음     │
| CQ         │ 클래스   │ 제한     │
| WFQ        │ 플로우   │ O        │
+------------+----------+----------+
```

**핵심 알고리즸:** Finish Time 계산, 가중치 스케줄링

**📢 섹션 요약 비유:** WFQ는 기아 현상 없음!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: WFQ vs CBWFQ**

| 항목 | WFQ | CBWFQ |
|:---|:---|:---|
| **분류** | 자동 | 수동 |
| **가중치** | 자동 | 수동 |
| **제어** | 제한 | 강력 |
| **확장** | 낮음 | 높음 |

**📢 섹션 요약 비유:** CBWFQ = WFQ + 제어!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 소규모** → WFQ
