# TCP URG (Urgent) - 어전트 플래그

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 긴급 데이터 표시
> 2. **가치**: 대역폭 데이터 우선 처리
> 3. **융합:** TCP Flags, Urgent Pointer

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

URG 플래그는 긴급 데이터가 있음을 표시합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TCP URG (Urgent)                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              URG 동작                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  TCP 헤더:                                           │   │ │
│   │   │  ┌─────────────────────────────────────────┐        │   │ │
│   │   │  │ URG=1 │ Urgent Pointer = n            │        │   │ │
│   │   │  └─────────────────────────────────────────┘        │   │ │
│   │   │                                                      │   │ │
│   │   │  데이터 스트림:                                      │   │ │
│   │   │  [일반 데이터...][긴급 데이터][일반 데이터...]        │   │ │
│   │   │                      ↑                               │   │ │
│   │   │               Urgent Pointer가 가리킴                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: URG는 긴급 데이터의 위치를 Urgent Pointer로 알립니다.

**📢 섹션 요약 비유:** URG = 긴급 택배!
