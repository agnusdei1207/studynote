# MDI / MDI-X (Medium Dependent Interface)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이더넷 매체 연결 인터페이스
> 2. **가치**: 장비 간 연결 표준화
> 3. **융합**: Auto-MDIX, 스위치와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MDI(Medium Dependent Interface)는 DTE와 전송 매체 간의 연결 인터페이스이고, MDI-X는 송수신 핀이 교차된 인터페이스입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MDI / MDI-X                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MDI vs MDI-X                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  MDI (Normal):                                           │   │ │
│   │   │                                                      │   │ │
│   │   │  RJ-45 핀:                                              │   │ │
│   │   │  1: TX+ (송신+)                                         │   │ │
│   │   │  2: TX- (송신-)                                         │   │ │
│   │   │  3: RX+ (수신+)                                         │   │ │
│   │   │  6: RX- (수신-)                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  사용 장비: PC, 서버, 라우터 (DTE)                      │   │ │
│   │   │                                                      │   │ │
│   │   │  MDI-X (Crossover):                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  RJ-45 핀:                                              │   │ │
│   │   │  1: RX+ (수신+)  ← 교차                                 │   │ │
│   │   │  2: RX- (수신-)  ← 교차                                 │   │ │
│   │   │  3: TX+ (송신+)  ← 교차                                 │   │ │
│   │   │  6: TX- (송신-)  ← 교차                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  사용 장비: 허브, 스위치 (DCE)                           │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              연결 규칙                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  다이렉트 케이블:                                        │   │ │
│   │   │  MDI ←───────────────→ MDI-X                          │   │ │
│   │   │  (PC)   다이렉트 케이블   (스위치)                       │   │ │
│   │   │                                                      │   │ │
│   │   │  크로스 케이블:                                          │   │ │
│   │   │  MDI ←───────────────→ MDI                            │   │ │
│   │   │  (PC)   크로스 케이블    (PC)                           │   │ │
│   │   │                                                      │   │ │
│   │   │  또는                                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  MDI-X ←──────────────→ MDI-X                         │   │ │
│   │   │  (스위치)  크로스 케이블  (스위치)                       │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: MDI와 MDI-X는 송수신 핀 배열이 교차되어 있습니다.

**💡 비유**: MDI/MDI-X는 왼손잡이와 오른손잡이가 악수하는 것과 같습니다.

**📢 섹션 요약 비유:** DTE = MDI, DCE = MDI-X!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **MDI** | DTE 인터페이스 | TX-RX 순서 | 왼손 |
| **MDI-X** | DCE 인터페이스 | RX-TX 교차 | 오른손 |
| **커넥터** | 물리 연결 | RJ-45 | 악수 |

**비교표**

```
+------------+----------+----------+
| 항목       | MDI      | MDI-X    |
+------------+----------+----------+
| 1번 핀     | TX+      | RX+      |
| 2번 핀     | TX-      | RX-      |
| 3번 핀     | RX+      | TX+      |
| 6번 핀     | RX-      | TX-      |
| 장비       | DTE      | DCE      |
+------------+----------+----------+
```

**핵심 알고리즸**: 송수신 교차, 신호 정합

**📢 섹션 요약 비유:** 같은 타입 = 크로스!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 케이블 선택**

| 연결 | 케이블 |
|:---|:---|
| **PC↔스위치** | 다이렉트 |
| **PC↔PC** | 크로스 |
| **스위치↔스위치** | 크로스 |

**📢 섹션 요약 비유:** Auto-MDIX = 자동 해결!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: PC↔스위치** → 다이렉트 케이블
**시나리오 2: PC↔PC** → 크로스 케이블 (또는 Auto-MDIX)
**시나리오 3: 최신 장비** → Auto-MDIX 자동

**도입 체크리스트:** 장비 타입, 케이블 타입, Auto-MDIX 지원

**📢 섹션 요약 비유:** Auto-MDIX로 고민 해결!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**미래 전망:** Auto-MDIX 표준화
