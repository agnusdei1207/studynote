# DLP (Data-Level Parallelism) - 데이터 수준 병렬성

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 동일 연산 병렬 적용
> 2. **가치**: 벡터/SIMD 처리
> 3. **융합:** SIMD, GPU, Vector Processor

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DLP는 여러 데이터에 동일 연산을 병렬 적용합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DLP                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DLP 구조                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Scalar:                 SIMD (DLP):                │   │ │
│   │   │  for i in 0..4:         A[0..3] = B[0..3] + C[0..3]│   │ │
│   │   │    A[i] = B[i] + C[i]   (한 번에 4개 연산)          │   │ │
│   │   │                                                      │   │ │
│   │   │  SIMD Register (128-bit):                           │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ B[0] │ B[1] │ B[2] │ B[3] │                   │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ C[0] │ C[1] │ C[2] │ C[3] │        +          │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │  ───────────────────────────────────────────────────│   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │ A[0] │ A[1] │ A[2] │ A[3] │                   │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: DLP는 SIMD로 대량 데이터를 효율적으로 처리합니다.

**📢 섹션 요약 비유:** DLP = 일괄 처리!
