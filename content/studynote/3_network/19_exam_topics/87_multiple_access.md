# 다중 접속 (Multiple Access)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 다수 사용자가 공유 매체에 동시 접속
> 2. **가치**: 자원 효율, 사용자 수용, 충돌 관리
> 3. **융합**: FDMA, TDMA, CDMA, OFDMA로 확장

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

다중 접속(Multiple Access)은 다수의 사용자가 제한된 통신 자원(주파수, 시간, 코드)을 공유하여 동시에 통신하는 기술입니다.

> **해설**: 다중 접속은 셀룰러와 무선 네트워크의 기반입니다.

**💡 비유**: 다중 접속은 여러 사람이 회의실에서 발언하는 방식입니다.

**📢 섹션 요약 비유**: FDMA/TDMA/CDMA = 3대 접속 방식!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **자원** | 공유 매체 | 주파수/시간/코드 | 회의실 |
| **할당** | 분배 | 고정/동적 | 예약 |
| **간섭** | 관리 | 격리 | 규칙 |

**비교표**

```
+------------+----------+----------+----------+
| 항목       | FDMA     | TDMA     | CDMA     |
+------------+----------+----------+----------+
| 분할 자원  | 주파수   | 시간     | 코드     |
| 대역폭     | 분할     | 공유     | 공유     |
| 동기       | 불필요   | 필요     | 필요     |
| 용량       | 제한     | 제한     | 유연     |
| 복잡도     | 낮음     | 중간     | 높음     |
+------------+----------+----------+----------+
```

**핵심 알고리즸**: 사용자 요청 → 자원 할당 → 분할 전송 → 수신 분리

**📢 섹션 요약 비유**: 다중화는 1:N, 다중접속은 N:N!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 세대별 접속**

| 세대 | 방식 | 특징 |
|:---|:---|:---|
| **1G** | FDMA | 아날로그 |
| **2G** | TDMA | 디지털 |
| **3G** | CDMA | 코드 |
| **4G** | OFDMA | 직교 |
| **5G** | OFDMA+NOMA | 비직교 추가 |

**📢 섹션 요약 비유**: 세대별 다중 접속 진화!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 셀룰러** → OFDMA (DL) / SC-FDMA (UL)
**시나리오 2: WiFi** → CSMA/CA
**시나리오 3: 위성** → FDMA/TDMA

**도입 체크리스트**: 사용자 수, 트래픽, 이동성

**📢 섹션 요약 비유**: 환경에 맞는 접속 방식!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 자원 효율, 다중 사용자 지원

**미래 전망**: NOMA, 비직교 확장

**참고 표준**: 3GPP, IEEE 802.11

**📢 섹션 요약 비유**: 다중 접속은 무선의 필수!
