# 변조 (Modulation) - ASK/FSK/PSK/QAM

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호를 반송파에 실음
> 2. **가치**: 전송 효율, 대역폭 활용
> 3. **융합**: 통신 시스템의 핵심

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

변조(Modulation)는 정보 신호를 반송파(Carrier)에 실어 전송하는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    변조 (Modulation)                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              디지털 변조 종류                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  반송파: s(t) = A·cos(2πf_c·t + φ)                   │   │ │
│   │   │                                                      │   │ │
│   │   │  1. ASK (Amplitude Shift Keying):                      │   │ │
│   │   │     • 진폭으로 데이터 표현                             │   │ │
│   │   │     • A₁ = 1, A₀ = 0 (OOK)                           │   │ │
│   │   │     • 단순, 잡음에 취약                                │   │ │
│   │   │                                                      │   │ │
│   │   │  2. FSK (Frequency Shift Keying):                      │   │ │
│   │   │     • 주파수로 데이터 표현                             │   │ │
│   │   │     • f₁ = 1, f₀ = 0                                  │   │ │
│   │   │     • 잡음에 강함, 대역폭 넓음                          │   │ │
│   │   │                                                      │   │ │
│   │   │  3. PSK (Phase Shift Keying):                          │   │ │
│   │   │     • 위상으로 데이터 표현                             │   │ │
│   │   │     • BPSK: 0°/180° (1 bit)                          │   │ │
│   │   │     • QPSK: 0°/90°/180°/270° (2 bits)                │   │ │
│   │   │     • 8PSK: 3 bits                                   │   │ │
│   │   │     • 잡음에 강함, 효율 좋음                            │   │ │
│   │   │                                                      │   │ │
│   │   │  4. QAM (Quadrature Amplitude Modulation):             │   │ │
│   │   │     • 진폭 + 위상 동시 사용                            │   │ │
│   │   │     • 16-QAM: 4 bits                                 │   │ │
│   │   │     • 64-QAM: 6 bits                                 │   │ │
│   │   │     • 256-QAM: 8 bits                                │   │ │
│   │   │     • 최고 효율, 잡음에 민감                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              성좌도 (Constellation)                           │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  BPSK:      QPSK:         16-QAM:                    │   │ │
│   │   │    ○   ●     ●     ●       ●  ○  ○  ●               │   │ │
│   │   │              │              ○  ○  ○  ●               │   │ │
│   │   │    0    1  ──┼──          ●  ●  ●  ●               │   │ │
│   │   │              │              ○  ●  ●  ○               │   │ │
│   │   │              ●     ●       (4 bits/symbol)           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: QAM은 높은 스펙트럼 효율로 현대 통신의 핵심입니다.

**💡 비유**: 변조는 편지를 우편물에 넣는 것과 같습니다.

**📢 섹션 요약 비유:** QAM = 진폭 + 위상!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 변조 | bits/symbol | 복잡도 | 잡음 내성 |
|:---|:---:|:---:|:---:|
| **BPSK** | 1 | 낮음 | 높음 |
| **QPSK** | 2 | 중간 | 높음 |
| **16-QAM** | 4 | 높음 | 중간 |
| **64-QAM** | 6 | 높음 | 낮음 |
| **256-QAM** | 8 | 높음 | 낮음 |

**비교표**

```
+------------+----------+----------+
| 항목       │ PSK      │ QAM      │
+------------+----------+----------+
| 진폭       │ 일정     │ 변함     │
│ 위상       │ 변함     │ 변함     │
│ 효율       │ 낮음     │ 높음     │
│ 잡음 내성   │ 높음     │ 낮음     │
+------------+----------+----------+
```

**핵심 알고리즸:** I/Q 변조, 성좌도 매핑

**📢 섹션 요약 비유:** QAM = 용량 vs 품질 트레이드오프!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 스펙트럼 효율**

| 변조 | bps/Hz | 필요 SNR | 용도 |
|:---:|:---:|:---:|:---|
| **BPSK** | 1 | 낮음 | 위성 |
| **QPSK** | 2 | 중간 | WiFi |
| **16-QAM** | 4 | 높음 | LTE |
| **64-QAM** | 6 | 높음 | LTE-A |
| **256-QAM** | 8 | 매우 높음 | 5G |

**📢 섹션 요약 비유:** SNR 높으면 고차 변조!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 열악한 채널** → BPSK/QPSK
**시나리오 2: 양호한 채널** → 16/64-QAM
**시나리오 3: 매우 양호** → 256/1024-QAM

**도입 체크리스트:** SNR, 대역폭, BER 요구사항

**📢 섹션 요약 비유:** 적응 변조 = 환경에 맞춤!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 스펙트럼 효율 극대화

**미래 전망:** 1024-QAM, 4096-QAM 확대

**참고 표준:** 3GPP, IEEE 802.11

**📢 섹션 요약 비유:** 변조는 통신의 핵심!
