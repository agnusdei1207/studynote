# DNS Resolver - DNS 리졸버

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: DNS 질의 수행
> 2. **가치**: 도메인 해석
> 3. **융합:** Recursive, Iterative, Stub

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DNS Resolver는 DNS 질의를 수행합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DNS Resolver                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DNS 해석 과정                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Recursive Resolution:                                  │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Client    Resolver    Root    TLD    Auth      ││   │ │
│   │   │  │    │          │         │       │       │        ││   │ │
│   │   │  │    │──query──→│         │       │       │        ││   │ │
│   │   │  │    │          │──query─→│       │       │        ││   │ │
│   │   │  │    │          │←─.com───│       │       │        ││   │ │
│   │   │  │    │          │──query─────────→│       │        ││   │ │
│   │   │  │    │          │←example.com─────│       │        ││   │ │
│   │   │  │    │          │──query─────────────────→│        ││   │ │
│   │   │  │    │          │←───────IP───────────────│        ││   │ │
│   │   │  │    │←──IP─────│                                 ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  Resolver 유형:                                         │   │ │
│   │   │  - Recursive: 대신 모든 질의 수행                        │   │ │
│   │   │  - Iterative: 각 단계에서 클라이언트에게 응답              │   │ │
│   │   │  - Stub: 로컬 리졸버                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: DNS Resolver는 도메인을 IP로 변환합니다.

**📢 섹션 요약 비유:** DNS Resolver = 도서관 사서!
