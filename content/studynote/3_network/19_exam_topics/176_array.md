# 안테나 배열 (Antenna Array) - 선형/평면/원형

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 여러 안테나를 배열하여 이득 향상
> 2. **가치**: 빔 제어, 지향성 향상
> 3. **융합**: 5G, 레이더, 위성과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

안테나 배열(Antenna Array)은 여러 개의 안테나 소자를 공간적으로 배치하여 원하는 지향성을 얻는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    안테나 배열 (Antenna Array)                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              배열 형태                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 선형 배열 (Linear Array):                          │   │ │
│   │   │                                                      │   │ │
│   │   │     ○───○───○───○───○     (1차원)                    │   │ │
│   │   │     d (소자 간격)                                     │   │ │
│   │   │     • 구현 단순                                        │   │ │
│   │   │     • 1차원 빔 제어                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 평면 배열 (Planar Array):                          │   │ │
│   │   │                                                      │   │ │
│   │   │     ○───○───○───○                                     │   │ │
│   │   │     │    │    │    │     (2차원)                      │   │ │
│   │   │     ○───○───○───○                                     │   │ │
│   │   │     │    │    │    │                                  │   │ │
│   │   │     ○───○───○───○                                     │   │ │
│   │   │     • 2차원 빔 제어                                    │   │ │
│   │   │     • 고이득                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  3. 원형 배열 (Circular Array):                        │   │ │
│   │   │                                                      │   │ │
│   │   │           ○   ○                                       │   │ │
│   │   │        ○         ○                                    │   │ │
│   │   │        ○    ○    ○    (원형 배치)                     │   │ │
│   │   │           ○   ○                                       │   │ │
│   │   │     • 360° 커버리지                                   │   │ │
│   │   │     • 방위각 제어                                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              배열 팩터 (Array Factor)                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  AF = Σ aₙ·e^(j·n·kd·cosθ)                          │   │ │
│   │   │                                                      │   │ │
│   │   │  aₙ: n번째 소자의 복소 가중치                          │   │ │
│   │   │  k: 위상 상수 (2π/λ)                                 │   │ │
│   │   │  d: 소자 간격                                        │   │ │
│   │   │  θ: 빔 각도                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  전체 패턴 = 소자 패턴 × 배열 팩터                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 배열 이득은 소자 수에 비례합니다 (10·log₁₀(N)).

**💡 비유**: 안테나 배열은 여러 사람이 같은 방향을 보며 소리치는 것과 같습니다.

**📢 섹션 요약 비유:** N개 소자 = 10·log₁₀(N) dB 이득!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 배열 형태 | 차원 | 빔 제어 | 응용 |
|:---|:---:|:---:|:---|
| **선형** | 1D | 1D | 기지국 |
| **평면** | 2D | 2D | 5G mmWave |
| **원형** | 2D | 방위각 | 전방향 |

**비교표**

```
+------------+----------+----------+
| 소자 수    │ 이득     │ 빔폭     │
| N          │ (dB)     │ (도)     │
+------------+----------+----------+
| 2          │ 3        │ 60       │
| 4          │ 6        │ 30       │
| 8          │ 9        │ 15       │
| 16         │ 12       │ 7.5      │
| 64         │ 18       │ 2        │
+------------+----------+----------+
```

**핵심 알고리즸:** 위상 배열, 빔 조향

**📢 섹션 요약 비유:** 위상 제어 = 빔 조향!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 급전 방식**

| 방식 | 원리 | 장점 |
|:---|:---|:---|
| **직렬** | 직렬 연결 | 단순 |
| **병렬** | 병렬 분배 | 대역폭 |
| **상관성** | 능동 제어 | 빔 제어 |

**📢 섹션 요약 비유:** 평면 배열 = 2D 빔 제어!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 기지국** → 선형 배열, 수직 빔
**시나리오 2: 5G mmWave** → 평면 배열 64-256소자
**시나리오 3: 레이더** → 능동 위상 배열 (AESA)

**도입 체크리스트:** 이득 요구, 빔 제어 범위, 비용

**📢 섹션 요약 비유:** 배열 = 이득 + 제어!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 고이득, 빔 제어

**미래 전망:** Massive MIMO (64-256 소자)

**참고 표준:** 3GPP TR 38.214

**📢 섹션 요약 비유:** 배열은 안테나의 파워업!
