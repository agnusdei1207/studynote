### Ⅰ. 개요 (Context & Background)

**개념 정의**

RTT Estimation은 패킷 왕복 시간을 측정합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    RTT Estimation                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              RTT 측정                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  송신측                    수신측                    │   │ │
│   │   │     │                         │                      │   │ │
│   │   │     │ ─── Data (T1) ────────→ │                      │   │ │
│   │   │     │                         │                      │   │ │
│   │   │     │ ←── ACK (T2) ────────── │                      │   │ │
│   │   │     │                         │                      │   │ │
│   │   │  RTT = T2 - T1                                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: RTT는 네트워크 상태에 따라 동적으로 변합니다.

**📢 섹션 요약 비유:** RTT = 왕복 시간!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Sample RTT** | 측정값 |
| **Estimated RTT** | 지수 이동 평균 |
| **Dev RTT** | 편차 |
| **RTO** | 재전송 타임아웃 |

**핵심 알고리즘:** Jacobson/Karels Algorithm

```
EstimatedRTT = (1-α) × EstimatedRTT + α × SampleRTT
DevRTT = (1-β) × DevRTT + β × |SampleRTT - EstimatedRTT|
RTO = EstimatedRTT + 4 × DevRTT
```
