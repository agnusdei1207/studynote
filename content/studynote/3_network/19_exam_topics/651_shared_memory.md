# Shared Memory - 공유 메모리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스 간 메모리 공유
> 2. **가치**: 최고 속도 IPC
> 3. **융합:** IPC, Synchronization, mmap

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Shared Memory는 여러 프로세스가 메모리를 직접 공유합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Shared Memory                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              공유 메모리 구조                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Process A                  Process B                │   │ │
│   │   │  ┌─────────────┐            ┌─────────────┐          │   │ │
│   │   │  │ Private     │            │ Private     │          │   │ │
│   │   │  │ Memory      │            │ Memory      │          │   │ │
│   │   │  └─────────────┘            └─────────────┘          │   │ │
│   │   │        │                          │                   │   │ │
│   │   │        │                          │                   │   │ │
│   │   │        └──────────┬───────────────┘                   │   │ │
│   │   │                   │                                   │   │ │
│   │   │                   ↓                                   │   │ │
│   │   │        ┌─────────────────────┐                        │   │ │
│   │   │        │   Shared Memory     │                        │   │ │
│   │   │        │  ┌───────────────┐  │                        │   │ │
│   │   │        │  │ Data sharing  │  │                        │   │ │
│   │   │        │  └───────────────┘  │                        │   │ │
│   │   │        │  (빠른 접근)        │                        │   │ │
│   │   │        └─────────────────────┘                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 공유 메모리는 커널 개입 없이 직접 접근합니다.

**📢 섹션 요약 비유:** Shared Memory = 공용 게시판!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **shmget** | 공유 메모리 생성 |
| **shmat** | 메모리 부착 |
| **shmdt** | 메모리 분리 |
| **shmctl** | 제어 |
| **mmap** | 메모리 매핑 |

**핵심 알고리즸:** 생성 → 부착 → 읽기/쓰기 → 분리
