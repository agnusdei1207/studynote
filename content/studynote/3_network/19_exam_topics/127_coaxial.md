# 동축 케이블 (Coaxial Cable)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 중심 도체와 차폐층이 동축 구조
> 2. **가치**: 높은 대역폭, EMI 강함
> 3. **융합**: CATV, RF, 이더넷과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

동축 케이블(Coaxial Cable)은 중심 도체와 외부 차폐층이 같은 축을 공유하는 구조로, 높은 주파수 전송에 적합한 케이블입니다.

```

│                    동축 케이블 (Coaxial Cable)                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              구조                                             │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  단면도:                                                  │   │ │
│   │   │                                                      │   │ │
│   │   │           외부 피복 (Jacket)                          │   │ │
│   │   │         ┌──────────────────────────┐                │   │ │
│   │   │         │  외부 차폐층 (Shield)     │                │   │ │
│   │   │         │  ┌────────────────────┐  │                │   │ │
│   │   │         │  │  절연체 (Dielectric)│  │                │   │ │
│   │   │         │  │  ┌──────────────┐  │  │                │   │ │

│   │   │  특징:                                                 │   │ │

│   │   │  RG-6  │ 75Ω     │ CATV           │ 6.9mm         │   │ │
```

> **해설**: 동축 케이블은 CATV, RF 통신, 초기 이더넷에서 사용됩니다.

**💡 비유**: 동축 케이블은 파이프 안에 작은 파이프가 있는 구조입니다.

**📢 섹션 요약 비유**: 동축 = 75Ω CATV!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **중심 도체** | 신호 전송 | 구리선 | 내관 |
| **절연체** | 간격 유지 | 폼/PE | 공기 |
| **차폐층** | EMI 차폐 | 땜납/알루미늄 | 외관 |

**비교표**

```
+------------+----------+----------+
| 항목       | RG-6     | RG-58    |
+------------+----------+----------+
| 임피던스   | 75Ω      | 50Ω      |
| 용도       | CATV     | LAN/RF   |
| 감쇠       | 낮음     | 높음     |
| 거리       | 길다     | 짧다     |
+------------+----------+----------+
```

**핵심 알고리즸**: TEM 모드 전파, 차폐 효과

**📢 섹션 요약 비유**: 50Ω = RF, 75Ω = CATV!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 커넥터**

| 커넥터 | 용도 | 크기 |
|:---|:---|:---:|
| **BNC** | RF/비디오 | 작음 |
| **F-type** | CATV | 중간 |
| **N-type** | 무선 | 큼 |

**📢 섹션 요약 비유**: BNC = 10BASE2!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: CATV** → RG-6/RG-11 (75Ω)
**시나리오 2: RF 안테나** → LMR/RG-58 (50Ω)
**시나리오 3: 고속 LAN** → UTP/광섬유

**도입 체크리스트**: 주파수, 거리, 임피던스 정합

**📢 섹션 요약 비유**: CATV = 75Ω 동축!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 높은 대역폭, EMI 강함

**미래 전망**: 광섬유로 대체 중

**참고 표준**: MIL-C-17, IEC 61196

**📢 섹션 요약 비유**: 동축은 RF의 기본!
