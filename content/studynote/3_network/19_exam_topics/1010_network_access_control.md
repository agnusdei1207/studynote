# Network Access Control - 네트워크 접근 제어

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단말 접속 보안
> 2. **가치**: 무결성 검증
> 3. **융합:** 802.1X, RADIUS, NAC

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Supplicant** | 인증 요청자 |
| **Authenticator** | 인증자 (스위치) |
| **Auth Server** | 인증 서버 |
| **Posture** | 단말 상태 |
| **Quarantine** | 격리 |

**핵심 알고리듬:** 접속 → 인증 → 상태 검사 → 정책 적용 → 접근
