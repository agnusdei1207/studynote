# ARP Spoofing - ARP 스푸핑

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: ARP 테이블 조작
> 2. **가치**: MITM 공격 벡터
> 3. **융합:** Poisoning, MITM, Dynamic ARP

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

ARP Spoofing은 ARP 테이블을 조작하는 공격입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    ARP Spoofing                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              ARP 스푸핑 공격                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  정상 통신:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Victim                    Gateway               ││   │ │
│   │   │  │ 192.168.1.10              192.168.1.1           ││   │ │
│   │   │  │ MAC: AA:AA:AA             MAC: GG:GG:GG         ││   │ │
│   │   │  │        ←────────────────────────→               ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  ARP 스푸핑:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Victim        Attacker        Gateway           ││   │ │
│   │   │  │ 192.168.1.10  192.168.1.100   192.168.1.1       ││   │ │
│   │   │  │     │              │              │              ││   │ │
│   │   │  │     │←──"게이트웨이 MAC=공격자MAC"──→│              ││   │ │
│   │   │  │     │              │              │              ││   │ │
│   │   │  │     ←──모든 트래픽──→공격자──→게이트웨이──→          ││   │ │
│   │   │  │                    (MITM)                        ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  방어:                                                  │   │ │
│   │   │  - Static ARP: 정적 ARP 등록                            │   │ │
│   │   │  - ARP Inspection: 스위치 검사                          │   │ │
│   │   │  - 암호화: HTTPS, VPN                                   │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: ARP 스푸핑은 신뢰 없는 프로토콜의 약점입니다.

**📢 섹션 요약 비유:** ARP Spoofing = 신분 사칭!
