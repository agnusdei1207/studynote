# QoS - 서비스 품질

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 트래픽 우선순위 관리
> 2. **가치**: 중요 트래픽 보장

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

QoS (Quality of Service)는 네트워크 트래픽 우선순위를 관리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    QoS                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              QoS 메커니즘                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  트래픽 분류 및 우선순위:                                 │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Priority   Traffic Type      DSCP              ││   │ │
│   │   │  │  ────────   ───────────      ─────              ││   │ │
│   │   │  │  High       Voice (VoIP)     EF (46)            ││   │ │
│   │   │  │  Medium     Video           AF41 (34)           ││   │ │
│   │   │  │  Normal     Data            BE (0)              ││   │ │
│   │   │  │  Low        Bulk Transfer   CS1 (8)             ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  QoS 파라미터:                                          │   │ │
│   │   │  - Bandwidth: 대역폭 보장                                │   │ │
│   │   │  - Latency: 지연 시간                                    │   │ │
│   │   │  - Jitter: 지연 변동                                     │   │ │
│   │   │  - Packet Loss: 패킷 손실률                              │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: QoS는 실시간 트래픽 품질을 보장합니다.

**📢 섹션 요약 비유:** QoS = 응급차 우선 통행!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**DiffServ vs IntServ**

| 항목 | DiffServ | IntServ |
|:---|:---|:---|
| **예약** | 없음 | RSVP |
| **확장성** | 높음 | 낮음 |
| **복잡도** | 낮음 | 높음 |
| **용도** | 인터넷 | 소규모 |
