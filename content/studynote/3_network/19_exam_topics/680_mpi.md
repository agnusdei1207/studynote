# MPI (Message Passing Interface) - MPI

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 분산 메모리 통신 표준
> 2. **가치**: HPC, 클러스터 컴퓨팅
> 3. **융합:** HPC, Parallel Computing, Cluster

### Ⅰ. 개요 (Context & Background)

**개념 정의**

MPI는 분산 메모리 시스템에서 통신을 위한 표준입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    MPI                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              MPI 통신 모델                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Process 0              Process 1              Process 2│   │ │
│   │   │  ┌─────────┐           ┌─────────┐           ┌─────────┐│   │ │
│   │   │  │ Rank 0  │           │ Rank 1  │           │ Rank 2  ││   │ │
│   │   │  │ Memory  │           │ Memory  │           │ Memory  ││   │ │
│   │   │  └────┬────┘           └────┬────┘           └────┬────┘│   │ │
│   │   │       │                     │                     │      │   │ │
│   │   │       │   MPI_Send/Recv     │                     │      │   │ │
│   │   │       │◄───────────────────►│                     │      │   │ │
│   │   │       │                     │                     │      │   │ │
│   │   │       │   MPI_Bcast         │                     │      │   │ │
│   │   │       ├────────────────────►│                     │      │   │ │
│   │   │       │                     │                     │      │   │ │
│   │   │       │   MPI_Reduce        │                     │      │   │ │
│   │   │       │◄────────────────────┼─────────────────────┤      │   │ │
│   │   │       │                     │                     │      │   │ │
│   │   │  Collective Communication:                          │      │   │ │
│   │   │  - Broadcast, Scatter, Gather, Reduce              │      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: MPI는 분산 메모리 시스템의 사실상 표준입니다.

**📢 섹션 요약 비유:** MPI = 분산 메신저!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Rank** | 프로세스 식별자 |
| **Communicator** | 통신 그룹 |
| **Point-to-Point** | 점대점 통신 |
| **Collective** | 집합 통신 |
| **Tag** | 메시지 태그 |

**핵심 알고리즸:** Send/Recv → 계산 → 통신 → 동기화

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**MPI 통신 방식**

| 방식 | 설명 |
|:---|:---|

| **Bcast** | 브로드캐스트 |
