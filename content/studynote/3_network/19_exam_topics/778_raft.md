### Ⅰ. 개요 (Context & Background)

**개념 정의**

Raft는 분산 시스템에서 합의를 이룹니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Raft                                              │
├─────────────────────────────────────────────────────────────────────────┤

│   │   │  │   Election  │   LEADER    │                 │    │   │ │
│   │   │  │   Timeout   └──────┬──────┘                 │    │   │ │
│   │   │  │              ↑     │ Append Entries          │    │   │ │
```
