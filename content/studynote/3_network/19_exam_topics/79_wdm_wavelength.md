# 광파장 분할 다중화 (WDM)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서로 다른 파장(색상)으로 다중화
> 2. **가치**: 광케이블 대용량화, Tbps 전송
> 3. **융합**: 백본망, 해저 케이블, 5G와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

WDM(Wavelength Division Multiplexing)은 광섬유에서 서로 다른 파장(색상)의 빛을 사용하여 여러 신호를 동시에 전송하는 방식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    WDM (광파장 분할 다중화)                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              WDM 원리                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  파장 분할:                                            │   │ │
│   │   │                                                      │   │ │
│   │   │  파장 (λ)                                              │   │ │
│   │   │    ▲                                                  │   │ │
│   │   │    │ ┌───┐GB┌───┐GB┌───┐GB┌───┐                    │   │ │
│   │   │    │ │λ1 │ │λ2 │ │λ3 │ │λ4 │                      │   │ │
│   │   │    │ │빨강│ │초록│ │파랑│ │보라│                      │   │ │
│   │   │    │ └───┘ └───┘ └───┘ └───┘                      │   │ │
│   │   │    ├──────────────────────────────→                │   │ │
│   │   │    │  1310nm    1490nm   1550nm   1590nm           │   │ │
│   │   │                                                      │   │ │
│   │   │  WDM 시스템:                                           │   │ │
│   │   │  λ1 ─→ [변환] ─┐                                      │   │ │
│   │   │  λ2 ─→ [변환] ─┼─→ [결합 MUX] ─→ 광섬유 ─→           │   │ │
│   │   │  λ3 ─→ [변환] ─┤                   [분리 DEMUX]      │   │ │
│   │   │  λ4 ─→ [변환] ─┘                        │           │   │ │
│   │   │                                          ├─→ λ1     │   │ │
│   │   │                                          ├─→ λ2     │   │ │
│   │   │                                          ├─→ λ3     │   │ │
│   │   │                                          └─→ λ4     │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              WDM 분류                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  CWDM (Coarse WDM):                                   │   │ │
│   │   │  • 채널 수: 8-18개                                    │   │ │
│   │   │  • 채널 간격: 20nm                                    │   │ │
│   │   │  • 거리: ~80km                                        │   │ │
│   │   │  • 비용: 낮음                                         │   │ │
│   │   │                                                      │   │ │
│   │   │  DWDM (Dense WDM):                                    │   │ │
│   │   │  • 채널 수: 40-160개                                  │   │ │
│   │   │  • 채널 간격: 0.8nm (100GHz)                          │   │ │
│   │   │  • 거리: 수천 km (EDFA 사용)                          │   │ │
│   │   │  • 비용: 높음                                         │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: WDM은 광통신의 대용량화 핵심 기술입니다.

**💡 비유**: WDM은 프리즘이 빛을 여러 색으로 나누는 것입니다.

**📢 섹션 요약 비유**: WDM = 빛의 색상 분할!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **MUX** | 파장 결합 | 프리즘 | 병합 |
| **DEMUX** | 파장 분리 | 격자 | 분배 |
| **EDFA** | 광 증폭 | 어빙현 | 증폭기 |
| **광섬유** | 전송 매체 | 전반사 | 도로 |

**비교표**

```
+------------+----------+----------+
| 항목       | CWDM     | DWDM     |
+------------+----------+----------+
| 채널 수    | 8-18     | 40-160   |
| 간격       | 20nm     | 0.8nm    |
| 전송률     | ~10Gbps  | ~100Tbps |
| 증폭기     | 불필요   | EDFA     |
| 비용       | 저렴     | 비쌈     |
+------------+----------+----------+
```

**핵심 알고리즸**: N파장 → 변환 → 결합 → 광섬유 → 분리 → 수신

**📢 섹션 요약 비유**: DWDM = Tbps 전송!

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 백본망** → DWDM
**시나리오 2: 메트로** → CWDM
**시나리오 3: 해저 케이블** → DWDM + EDFA

**도입 체크리스트**: 거리, 용량, 비용

**📢 섹션 요약 비유**: 장거리는 DWDM!
