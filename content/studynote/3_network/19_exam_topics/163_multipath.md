# 다중경로 (Multipath) - 레일리/라이시안

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 여러 경로로 도달하는 신호
> 2. **가치**: ISI, 페이딩의 원인
> 3. **융합**: RAKE, MIMO와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

다중경로(Multipath)는 신호가 반사, 회절, 산란 등을 통해 여러 경로로 수신되는 현상입니다.

```

│                    다중경로 (Multipath)                              │
├─────────────────────────────────────────────────────────────────────────┤

│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │          [건물]                                      │   │ │
│   │   │            │                                         │   │ │
│   │   │      반사파│         [산]                            │   │ │
│   │   │      ↗     │      ↗    산란파                       │   │ │
│   │   │     /      │     /       ↘                          │   │ │
│   │   │    /       │    /          \                         │   │ │
│   │   │   /        │   /            \                        │   │ │
│   │   │  ○ 송신 ───┼── 직진파 ────────○ 수신                 │   │ │
```

> **해설**: 다중경로는 페이딩과 ISI의 주요 원인입니다.

**💡 비유**: 다중경로는 산 속에서 메아리가 여러 번 들리는 것과 같습니다.

**📢 섹션 요약 비유:** 레일리 = NLOS, 라이시안 = LOS!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 분포 | 환경 | 특징 | 파라미터 |
|:---|:---|:---|:---|
| **레일리** | NLOS | 직진파 없음 | σ (표준편차) |
| **라이시안** | LOS | 직진파 존재 | K-factor |
| **나카가미** | 일반 | m 파라미터 | m |

**비교표**

```
+------------+----------+----------+
| 항목       | 레일리   │ 라이시안 │
+------------+----------+----------+
| LOS        │ 없음     │ 있음     │
│ 환경       │ 도심     │ 위성     │
│ K-factor   │ 0        │ > 0      │
│ 품질       │ 낮음     │ 높음     │
+------------+----------+----------+
```

**핵심 알고리즸:** 채널 추정, RAKE 수신기

**📢 섹션 요약 비유:** K-factor = 직진파 세기!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 다중경로 대책**

| 기법 | 원리 | 장점 |
|:---|:---|:---|
| **RAKE** | 다중경로 이용 | 에너지 수집 |
| **OFDM** | 심볼 연장 | ISI 제거 |
| **MIMO** | 공간 다중화 | 용량 증가 |

**📢 섹션 요약 비유:** RAKE = 다중경로를 친구로!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: 도심 NLOS** → 레일리 페이딩, OFDM
**시나리오 2: 위성 LOS** → 라이시안, 높은 K
**시나리오 3: 실내** → 레일리, 다이버시티

**도입 체크리스트:** LOS 유무, 지연 확산, 대역폭

**📢 섹션 요약 비유:** 환경 분석 = 분포 선택!

---
