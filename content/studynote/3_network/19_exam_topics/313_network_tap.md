# Network TAP - 트래픽 복제

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 트래픽 복제 장비
> 2. **가치**: 무손실 패킷 캡처
> 3. **융합**: IDS, 모니터링과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Network TAP(Test Access Point)는 네트워크 트래픽을 복제하여 모니터링 장비로 전달합니다.

**💡 비유**: TAP는 Y 케이블 분배기입니다. 하나를 여러 개로 복사합니다.

**📢 섹션 요약 비유:** TAP = 무손실!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Inline** | 인라인 |
| **Breakout** | 복제 |
| **Aggregation** | 통합 |

**비교표: TAP 유형**

```
+------------+----------+----------+
| 유형       │ 설명     │ 용도     │
+------------+----------+----------+
| Passive    │ 수동     │ 기본     │
| Active     │ 능동     | 재생성   │
| Aggregator │ 통합     | 멀티     │
+------------+----------+----------+
```

**핵심 알고리즸:** 신호 복제

**📢 섹션 요약 비유:** Aggregator = 통합!
