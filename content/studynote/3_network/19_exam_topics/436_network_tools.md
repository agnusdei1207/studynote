### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Ping** | ICMP Echo Request/Reply |
| **Traceroute** | TTL 기반 경로 추적 |
| **Netstat** | 네트워크 통계 |
| **Tcpdump** | 패킷 캡처 |

**비교표: 도구별 기능**

```
+------------+----------+----------+----------+
| 도구       │ 연결     │ 경로     │ 상세     │
+------------+----------+----------+----------+
| ping       │ O        │ X        │ X        │
| traceroute │ O        │ O        │ X        │
| tcpdump    │ O        │ X        │ O        │
| Wireshark  │ O        │ X        │ O        │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** ICMP, TTL

**📢 섹션 요약 비유:** Traceroute = 경로 추적!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 심화 도구**

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 연결 안 됨 → ping → traceroute
**시나리오 2:** DNS 문제 → nslookup → dig
**시나리오 3:** 성능 문제 → iPerf → tcpdump

**도입 체크리스트:** 도구 숙지, 권한, 로그

**📢 섹션 요약 비유:** 체계적 접근 = 문제 해결!
