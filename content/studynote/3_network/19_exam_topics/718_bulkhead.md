# Bulkhead - 벌크헤드

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 격리 구획
> 2. **가치**: 장애 격리, 자원 보호

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Bulkhead는 자원을 격리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Bulkhead                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              벌크헤드 패턴                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  격리된 자원 풀:                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐   │   │ │
│   │   │  │ Service A   │ │ Service B   │ │ Service C   │   │   │ │
│   │   │  │ Thread Pool │ │ Thread Pool │ │ Thread Pool │   │   │ │
│   │   │  │   [10]      │ │   [10]      │ │   [10]      │   │   │ │
│   │   │  └─────────────┘ └─────────────┘ └─────────────┘   │   │ │
│   │   │        │                │                │          │   │ │
│   │   │        ↓                ↓                ↓          │   │ │
│   │   │   (독립적 실행, 장애 격리)                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 벌크헤드는 한 구역의 장애가 전파되지 않게 합니다.

**📢 섹션 요약 비유:** Bulkhead = 선박의 격벽!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Thread Pool** | 스레드 풀 격리 |
| **Connection Pool** | 연결 풀 격리 |
| **Process** | 프로세스 격리 |
| **Container** | 컨테이너 격리 |
| **Circuit** | 서킷 격리 |
