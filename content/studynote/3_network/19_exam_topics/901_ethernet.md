# Ethernet - 이더넷

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 유선 LAN 표준
> 2. **가치**: 높은 대역폭과 안정성
> 3. **융합:** CSMA/CD, Frame, MAC Address

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Ethernet은 가장 널리 사용되는 유선 LAN 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Ethernet                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              이더넷 프레임                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Frame 구조:                                            │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Preamble │ SFD │ DA │ SA │ Type │ Data │ FCS   ││   │ │
│   │   │  │  7B      │ 1B  │ 6B │ 6B │ 2B   │46-1500B│ 4B   ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  속도 표준:                                             │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ 10BASE-T: 10 Mbps     (Cat 3)                  ││   │ │
│   │   │  │ 100BASE-TX: 100 Mbps   (Cat 5)                 ││   │ │
│   │   │  │ 1000BASE-T: 1 Gbps    (Cat 5e/6)               ││   │ │
│   │   │  │ 10GBASE-T: 10 Gbps    (Cat 6a/7)               ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  CSMA/CD:                                               │   │ │
│   │   │  1. Carrier Sense (통신 중인지 감지)                      │   │ │
│   │   │  2. Multiple Access (다중 접속)                          │   │ │
│   │   │  3. Collision Detection (충돌 감지)                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 이더넷은 현대 네트워크의 기반 기술입니다.

**📢 섹션 요약 비유:** Ethernet = 유선 통신의 공용 언어!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Frame** | 데이터 단위 |
| **MAC** | 하드웨어 주소 |
| **CSMA/CD** | 접속 제어 |
| **MTU** | 최대 전송 단위 |
| **PHY** | 물리 계층 |

**핵심 알고리듬:** 감지 → 전송 → 충돌 시 재전송

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**이더넷 속도 비교**
