# 근거리-원거리 문제 (Near-Far Problem)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 가까운 사용자 신호가 먼 사용자 신호를 간섭
> 2. **가치**: CDMA 전력 제어의 핵심 이슈
> 3. **융합**: 3G, 전력 제어 알고리즘과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

근거리-원거리 문제(Near-Far Problem)는 CDMA에서 가까운 사용자의 강한 신호가 먼 사용자의 약한 신호를 압도하여 간섭을 유발하는 현상입니다.

```

│   ┌──────────────────────────────────────────────────────────────┐ │

│   │   │   사용자1  사용자2  사용자3                           │   │ │
│   │   │   (가까움) (중간)   (멀음)                            │   │ │

│   │   │                                                      │   │ │
```

> **해설**: Near-Far 문제는 CDMA의 고유한 문제로 전력 제어로 해결합니다.

**💡 비유**: Near-Far 문제는 큰 소리로 말하는 사람이 작은 소리를 지워버리는 것입니다.

**📢 섹션 요약 비유**: 전력 제어 = 1500Hz!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **개루프** | 추정 | 경로 손실 | 예측 |
| **폐루프** | 제어 | BS 명령 | 피드백 |
| **TPC** | 전력 제어 | 1500Hz | 음량 조절 |

**비교표**

```
+------------+----------+----------+
| 항목       | 개루프   | 폐루프   |
+------------+----------+----------+
| 기준       | 경로손실 | BS 수신  |
| 정확도     | 낮음     | 높음     |
| 속도       | 빠름     | 느림     |
| 오버헤드   | 없음     | 있음     |
+------------+----------+----------+
```

**핵심 알고리즸**: BS 수신 전력 측정 → TPC 명령 → 단말 전력 조절

**📢 섹션 요약 비유**: WCDMA는 개루프+폐루프 결합!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 시스템별 전력 제어**

| 시스템 | 주기 | 방식 |
|:---|:---:|:---|
| **IS-95** | 800Hz | 폐루프 |
| **WCDMA** | 1500Hz | 개+폐루프 |
| **LTE** | 1000Hz | 폐루프 |

**📢 섹션 요약 비유**: 전력 제어 주기 = 시스템 용량!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: CDMA** → 필수 전력 제어
**시나리오 2: TDMA** → 덜 중요
**시나리오 3: OFDMA** → 부분적 필요

**도입 체크리스트**: 전력 제어 알고리즘, 주기, 정확도

**📢 섹션 요약 비유**: CDMA는 전력 제어가 생명!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 시스템 용량 증대, 통화 품질 향상

**미래 전망**: OFDMA에서도 전력 제어 유지

**참고 표준**: 3GPP TS 25.214

**📢 섹션 요약 비유**: Near-Far 해결 = 전력 제어!
