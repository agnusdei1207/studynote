# Kong - API Gateway 플랫폼

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 오픈소스 API Gateway
> 2. **가치**: 플러그인 생태계, 확장성
> 3. **융합**: Kubernetes, Service Mesh와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Kong은 Nginx 기반의 오픈소스 API Gateway 플랫폼입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Kong                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Kong 아키텍처                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Client]                                           │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ HTTP/HTTPS                                     │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           Kong Gateway                      │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • Nginx 기반                                │    │   │ │
│   │   │  │  • Lua 플러그인                              │    │   │ │
│   │   │  │  • 데이터스토어 (PostgreSQL/Cassandra)        │    │   │ │
│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ 플러그인 체인                                 │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [Upstream Services]                                │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Kong 플러그인                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 플러그인    │ 기능     │ 가치     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ JWT        │ 인증     │ 보안     │               │   │ │
│   │   │  │ Rate Limiting│ 속도제한│ 보호     │               │   │ │
│   │   │  | Caching    │ 캐시     │ 성능     │               │   │ │
│   │   │  │ Logging    │ 로그     │ 관측성   │               │   │ │
│   │   │  │ CORS       │ 크로스   │ 호환성   │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Kong은 플러그인 기반으로 확장 가능한 API Gateway입니다.

**💡 비유**: Kong은 API의 스위스 아미 나이프입니다. 모든 기능을 플러그인으로 추가합니다.

**📢 섹션 요약 비유:** Kong = 플러그인 생태계!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Kong Gateway** | 게이트웨이 |
| **Datastore** | 데이터 저장소 |
| **Admin API** | 관리 API |
| **Kong Manager** | 관리 UI |

**비교표: Kong 에디션**

```
+------------+----------+----------+
| 항목       │ OSS      │ Enterprise│
+------------+----------+----------+
| 비용       │ 무료     │ 유료     │
| 플러그인   │ 기본     │ 전체     │
| 지원       │ 커뮤니티 │ 기술     │
| UI         │ KIC      │ Manager  │
+------------+----------+----------+
```

**핵심 알고리즸:** Nginx, Lua

**📢 섹션 요약 비유:** Lua = 확장!
