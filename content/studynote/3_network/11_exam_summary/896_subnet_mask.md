# Subnet Mask - 서브넷 마스크

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크/호스트 구분
> 2. **가치**: 주소 범위 결정
> 3. **융합:** CIDR, Binary AND, Network ID

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Subnet Mask는 IP 주소에서 네트워크 부분을 식별합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Subnet Mask                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              마스크 동작                                       │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Binary AND 연산:                                       │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ IP:    192.168.1.100                           ││   │ │
│   │   │  │        11000000.10101000.00000001.01100100      ││   │ │
│   │   │  │ Mask:  255.255.255.0                             ││   │ │
│   │   │  │        11111111.11111111.11111111.00000000      ││   │ │
│   │   │  │ ─────────────────────────────────────────       ││   │ │
│   │   │  │ Net:   192.168.1.0                              ││   │ │
│   │   │  │        11000000.10101000.00000001.00000000      ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  일반적인 마스크:                                        │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Class A: 255.0.0.0        /8                    ││   │ │
│   │   │  │ Class B: 255.255.0.0      /16                   ││   │ │
│   │   │  │ Class C: 255.255.255.0    /24                   ││   │ │
│   │   │  │ Custom: 255.255.255.192  /26                    ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  주소 범위 계산:                                          │   │ │
│   │   │  Network: 192.168.1.0/24                               │   │ │
│   │   │  First Host: 192.168.1.1                                │   │ │
│   │   │  Last Host: 192.168.1.254                               │   │ │
│   │   │  Broadcast: 192.168.1.255                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 서브넷 마스크는 네트워크 범위를 결정합니다.

**📢 섹션 요약 비유:** Subnet Mask = 주소 필터!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Network Bits** | 네트워크 비트 (1) |
| **Host Bits** | 호스트 비트 (0) |
| **AND** | 논리곱 연산 |
| **Prefix** | 프리픽스 길이 |
| **Wildcard** | 와일드카드 마스크 |

**핵심 알고리즸:** IP AND Mask = Network ID

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**마스크 vs 호스트 수**
