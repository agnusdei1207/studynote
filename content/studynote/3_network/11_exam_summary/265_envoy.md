# Envoy - 고성능 프록시

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: L3/L4/L7 프록시
> 2. **가치**: 관찰 가능성, 확장성
> 3. **융합**: Istio, Service Mesh와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Envoy는 Lyft에서 개발한 고성능 L3/L4/L7 프록시입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Envoy                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Envoy 구조                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Downstream]                                       │   │ │
│   │   │       │                                              │   │ │
│   │   │       ↓                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           Envoy                              │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  ┌─────────────────────────────────────┐    │    │   │ │
│   │   │  │  │  Listener                           │    │    │   │ │
│   │   │  │  │  (수신)                             │    │    │   │ │
│   │   │  │  └─────────────────────────────────────┘    │    │   │ │
│   │   │  │       │                                      │    │   │ │
│   │   │  │       ↓                                      │    │   │ │
│   │   │  │  ┌─────────────────────────────────────┐    │    │   │ │
│   │   │  │  │  Filter Chain                       │    │    │   │ │
│   │   │  │  │  (인증, 로깅, 라우팅)               │    │    │   │ │
│   │   │  │  └─────────────────────────────────────┘    │    │   │ │
│   │   │  │       │                                      │    │   │ │

│   │   │       │                                              │   │ │

│                                                                     │

│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  1. 프로토콜 지원:                                     │   │ │
│   │   │     • HTTP/1.1, HTTP/2, HTTP/3                       │   │ │
│   │   │     • gRPC, WebSocket, TCP                           │   │ │
│   │   │                                                      │   │ │
│   │   │  2. 관찰 가능성:                                       │   │ │
│   │   │     • 통계, 분산 추적                                 │   │ │
```

> **해설**: Envoy는 Listener → Filter → Cluster 구조로 동작합니다.

**💡 비유**: Envoy는 스마트 우체국입니다. 모든 편지를 검사하고 전달합니다.

**📢 섹션 요약 비유:** Envoy = 프록시!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Listener** | 수신 지점 |
| **Filter** | 처리 체인 |
| **Cluster** | 업스트림 그룹 |
| **Route** | 라우팅 규칙 |

**비교표: Envoy xDS API**

```
+------------+----------+----------+
| API        │ 설명     │ 용도     │
+------------+----------+----------+
| LDS        │ Listener │ 수신     │
| RDS        │ Route    │ 라우팅  │
| CDS        │ Cluster  │ 서비스  │
| EDS        │ Endpoint │ 주소    │
+------------+----------+----------+
```

**핵심 알고리즸:** 로드밸런싱, 서킷 브레이커

**📢 섹션 요약 비유:** xDS = 동적 구성!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 로드밸런싱**

| 알고리즘 | 설명 |
|:---|:---|
| **Round Robin** | 순차 |
| **Random** | 무작위 |
| **Least Request** | 최소 요청 |
| **Ring Hash** | 일관성 해시 |

**📢 섹션 요약 비유:** Least Request = 분산!
