# Compression - 압축

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 크기 감소
> 2. **가치**: 대역폭 절약
> 3. **융합:** GZIP, LZ4, ZSTD

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Compression은 데이터 크기를 줄입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Compression                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              압축 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  무손실 (Lossless):                                   │   │ │
│   │   │  - GZIP: 웹, 로그                                     │   │ │
│   │   │  - LZ4: 고속 압축                                     │   │ │
│   │   │  - ZSTD: 균형                                         │   │ │
│   │   │  - Brotli: 웹 최적화                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  손실 (Lossy):                                         │   │ │
│   │   │  - JPEG: 이미지                                       │   │ │
│   │   │  - MP3: 오디오                                        │   │ │
│   │   │  - H.264: 비디오                                      │   │ │
│   │   │                                                      │   │ │
│   │   │  압축 과정:                                            │   │ │
│   │   │  원본: A A A B B B B C C                              │   │ │
│   │   │  RLE:  3A 4B 2C                                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 압축은 저장 비용과 전송 시간을 줄입니다.

**📢 섹션 요약 비유:** Compression = 짐 싸기!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **RLE** | 런 렝코드 부호화 |
| **Huffman** | 허프만 코딩 |
| **LZ77** | 사전 기반 |
| **DEFLATE** | GZIP 기반 |

**핵심 알고리즸:** 중복 제거 → 인코딩 → 압축
