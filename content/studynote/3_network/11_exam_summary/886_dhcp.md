# DHCP - 동적 호스트 구성 프로토콜

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 자동 IP 할당
> 2. **가치**: 네트워크 설정 자동화
> 3. **융합:** DORA, Lease, Scope

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

DHCP (Dynamic Host Configuration Protocol)는 IP를 자동 할당합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DHCP                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              DORA 프로세스                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  DHCP DORA:                                              │   │ │
│   │   │  Client                    Server                      │   │ │
│   │   │  ┌─────┐                  ┌─────┐                     │   │ │
│   │   │  │     │ ── Discover ───→ │     │ (Broadcast)         │   │ │
│   │   │  │     │    "IP 있어?"     │     │                     │   │ │
│   │   │  │     │                  │     │                     │   │ │
│   │   │  │     │ ←─ Offer ─────── │     │                     │   │ │
│   │   │  │     │    "이거 써"      │     │                     │   │ │
│   │   │  │     │                  │     │                     │   │ │
│   │   │  │     │ ── Request ────→ │     │                     │   │ │
│   │   │  │     │    "이거 쓸게"    │     │                     │   │ │
│   │   │  │     │                  │     │                     │   │ │
│   │   │  │     │ ←─ Ack ───────── │     │                     │   │ │
│   │   │  │     │    "확인"         │     │                     │   │ │
│   │   │  └─────┘                  └─────┘                     │   │ │
│   │   │                                                      │   │ │
│   │   │  할당 정보:                                              │   │ │
│   │   │  - IP 주소                                               │   │ │
│   │   │  - 서브넷 마스크                                          │   │ │
│   │   │  - 기본 게이트웨이                                        │   │ │
│   │   │  - DNS 서버                                              │   │ │
│   │   │  - Lease Time (임대 시간)                                 │   │ │
│   │   │                                                      │   │ │
│   │   │  포트:                                                    │   │ │
│   │   │  - UDP 67 (Server)                                        │   │ │
│   │   │  - UDP 68 (Client)                                        │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: DHCP는 네트워크 설정을 자동화합니다.

**📢 섹션 요약 비유:** DHCP = IP 자동 발급기!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **DORA** | Discover-Offer-Request-Ack |
| **Lease** | 임대 시간 |
| **Scope** | 범위 |
| **Pool** | 풀 |
| **Reservation** | 예약 |

**핵심 알고리즸:** Discover → Offer → Request → Ack

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**정적 vs 동적 IP**

| 항목 | 정적 | 동적 (DHCP) |
|:---|:---|:---|
| **할당** | 수동 | 자동 |
| **관리** | 복잡 | 간단 |
| **용도** | 서버 | 클라이언트 |
| **변경** | 어려움 | 쉬움 |
