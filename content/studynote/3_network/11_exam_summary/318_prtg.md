# PRTG - 네트워크 모니터링

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 통합 인프라 모니터링 도구
> 2. **가치**: 간편한 설정, 직관적 UI
> 3. **융합**: 네트워크, 서버, 애플리케이션 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

PRTG(Paessler Router Traffic Grapher)는 통합 인프라 모니터링 도구입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    PRTG                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              PRTG 아키텍처                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [Infrastructure]                                   │   │ │
│   │   │     │                                                │   │ │
│   │   │     │ SNMP / WMI / NetFlow / Ping                    │   │ │
│   │   │     │                                                │   │ │
│   │   │     ▼                                                │   │ │
│   │   │  ┌─────────────────────────────────────────────┐    │   │ │
│   │   │  │           PRTG Core Server                  │    │   │ │
│   │   │  │                                              │    │   │ │
│   │   │  │  • Sensors (모니터링 항목)                    │    │   │ │
│   │   │  │  • 프로브 (원격 수집)                         │    │   │ │

│   │   │  └─────────────────────────────────────────────┘    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ Web UI / API                                 │   │ │
│   │   │       ▼                                              │   │ │
│   │   │  [관리자]                                           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │

│   │              PRTG 센서 유형                                    │ │

│   │   │  │ Ping       │ ICMP     │ 연결     │               │   │ │

│   │   │  | WMI        │ WMI      │ Windows  │               │   │ │
│   │   │  │ NetFlow    │ NetFlow  │ 트래픽   │               │   │ │

│   │   │                                                      │   │ │
```

> **해설**: PRTG는 센서 기반으로 다양한 인프라를 모니터링합니다.

**💡 비유**: PRTG는 인프라의 통합 계기판입니다. 모든 상태를 한눈에 봅니다.

**📢 섹션 요약 비유:** PRTG = 올인원!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Core Server** | 중앙 서버 |
| **Probe** | 원격 수집 |
| **Sensor** | 모니터링 항목 |
| **Device** | 장비 |

**비교표: 라이선스**

```
+------------+----------+----------+
| 버전       │ 센서 수   │ 비용     │
+------------+----------+----------+
| Free       │ 100      │ 무료     │
| 500        │ 500      │ 유료     │
| 1000       │ 1000     │ 유료     │
| Unlimited  │ 무제한   │ 유료     │
+------------+----------+----------+
```

**핵심 알고리즸:** 폴링, 임계값

**📢 섹션 요약 비유:** 센서 = 모니터링 단위!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 모니터링 도구**

| 항목 | PRTG | Nagios | Zabbix |
|:---|:---|:---|:---|
| **설정** | 쉬움 | 복잡 | 중간 |
| **UI** | 우수 | 제한 | 좋음 |

**📢 섹션 요약 비유:** 중소 기업 = PRTG!
