# Interference - 간섭

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 간 충돌
> 2. **가치**: 품질 저하 원인
> 3. **융합:** ISI, ACI, CCI

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Interference는 외부 신호의 영향입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Interference                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              간섭 유형                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  ISI (Inter-Symbol Interference):                    │   │ │
│   │   │  심볼 간 간섭                                        │   │ │
│   │   │  ┌───┬───┬───┐                                      │   │ │
│   │   │  │ 1 │ 2 │ 3 │  ← 심볼이 서로 겹침                    │   │ │
│   │   │  └───┴───┴───┘                                      │   │ │
│   │   │  원인: 다중 경로, 대역 제한                            │   │ │
│   │   │                                                      │   │ │
│   │   │  ACI (Adjacent Channel Interference):                │   │ │
│   │   │  인접 채널 간섭                                       │   │ │
│   │   │  ┌─────┬─────┬─────┐                                │   │ │
│   │   │  │ Ch1 │ Ch2 │ Ch3 │  ← Ch2가 Ch1/Ch3에 영향         │   │ │
│   │   │  └─────┴─────┴─────┘                                │   │ │
│   │   │                                                      │   │ │
│   │   │  CCI (Co-Channel Interference):                      │   │ │
│   │   │  동일 채널 간섭                                       │   │ │
│   │   │  셀룰러에서 동일 주파수 재사용                          │   │ │
│   │   │                                                      │   │ │
│   │   │  대응:                                                │   │ │
│   │   │  - Equalizer (등화기)                                │   │ │
│   │   │  - Guard Band                                       │   │ │
│   │   │  - Frequency Planning                               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 간섭은 통신 품질에 직접적 영향을 미칩니다.

**📢 섹션 요약 비유:** Interference = 잡담 방해!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **ISI** | 심볼 간 간섭 |
| **ACI** | 인접 채널 간섭 |
| **CCI** | 동일 채널 간섭 |
| **EMI** | 전자기 간섭 |
| **RFI** | 무선 주파수 간섭 |

**핵심 알고리즸:** 신호 → 간섭 추가 → 수신 → 보정

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**간섭 대응 기법**
