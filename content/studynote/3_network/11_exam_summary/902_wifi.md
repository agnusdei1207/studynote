# WiFi - 와이파이

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 무선 LAN 표준

### Ⅰ. 개요 (Context & Background)

**개념 정의**

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **AP** | 액세스 포인트 |
| **SSID** | 네트워크 이름 |
| **Channel** | 주파수 채널 |
| **CSMA/CA** | 충돌 회피 |
| **WPA** | 보안 프로토콜 |

**핵심 알고리듬:** 감지 → 대기 → 전송 → ACK 대기

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)
