# Throughput - 처리량

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단위 시간당 처리 수
> 2. **가치**: 시스템 용량
> 3. **융합:** Bandwidth, QPS, TPS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Throughput은 단위 시간당 처리 수입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Throughput                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              처리량 지표                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  QPS (Queries Per Second):                            │   │ │
│   │   │  초당 쿼리 수                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  RPS (Requests Per Second):                           │   │ │
│   │   │  초당 요청 수                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  TPS (Transactions Per Second):                        │   │ │
│   │   │  초당 트랜잭션 수                                       │   │ │
│   │   │                                                      │   │ │
│   │   │  Bandwidth:                                            │   │ │
│   │   │  초당 전송 데이터 양 (Mbps, Gbps)                        │   │ │
│   │   │                                                      │   │ │
│   │   │  Little's Law:                                         │   │ │
│   │   │  L = λ × W                                            │   │ │
│   │   │  L: 시스템 내 요청 수                                   │   │ │
│   │   │  λ: 도착률 (Throughput)                                │   │ │
│   │   │  W: 평균 대기 시간 (Latency)                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Throughput과 Latency는 상충 관계일 수 있습니다.

**📢 섹션 요약 비유:** Throughput = 파이프 용량!
