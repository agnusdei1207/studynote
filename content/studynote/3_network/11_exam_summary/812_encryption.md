# Encryption - 암호화

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 기밀성
> 2. **가치**: 무단 접근 방지
> 3. **융합:** AES, RSA, TLS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Encryption은 데이터를 암호화합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Encryption                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              암호화 유형                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  대칭키 (Symmetric):                                  │   │ │
│   │   │  ┌───────────┐    Key    ┌───────────┐              │   │ │
│   │   │  │ Plaintext │ ────────→ │ Ciphertext│              │   │ │
│   │   │  └───────────┘  Encrypt  └───────────┘              │   │ │
│   │   │  AES, DES, ChaCha20                                   │   │ │
│   │   │  빠름, 키 분배 문제                                     │   │ │
│   │   │                                                      │   │ │
│   │   │  비대칭키 (Asymmetric):                                │   │ │
│   │   │  ┌───────────┐  Public   ┌───────────┐              │   │ │
│   │   │  │ Plaintext │ ────────→ │ Ciphertext│              │   │ │
│   │   │  └───────────┘  Encrypt  └───────────┘              │   │ │
│   │   │                    │                                │   │ │
│   │   │              Private Key                             │   │ │
│   │   │                    ↓                                │   │ │
│   │   │  RSA, ECC, DSA                                        │   │ │
│   │   │  느림, 키 분배 용이                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 암호화는 데이터 보안의 핵심입니다.

**📢 섹션 요약 비유:** Encryption = 비밀 자물쇠!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **AES** | 대칭키 표준 |
| **RSA** | 비대칭키 |
| **ECC** | 타원곡선 |
| **Key** | 키 |
| **IV** | 초기화 벡터 |

**핵심 알고리즸:** 평문 → 암호화 → 암호문 → 복호화 → 평문

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)
