# Partitioning - 파티셔닝

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 논리적 분할
> 2. **가치**: 관리성, 성능
> 3. **융합:** Sharding, Database, Indexing

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Partitioning은 데이터를 논리적으로 분할합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Partitioning                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              파티셔닝 유형                                      │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Horizontal (수평):                                   │   │ │
│   │   │  Row 기준 분할                                        │   │ │
│   │   │  ┌─────────────┬─────────────┐                      │   │ │
│   │   │  │ Partition 1 │ Partition 2 │                      │   │ │
│   │   │  │ Rows 1-1000 │ Rows 1001+  │                      │   │ │
│   │   │  └─────────────┴─────────────┘                      │   │ │
│   │   │                                                      │   │ │
│   │   │  Vertical (수직):                                     │   │ │
│   │   │  Column 기준 분할                                     │   │ │
│   │   │  ┌─────────────┬─────────────┐                      │   │ │
│   │   │  │ ID, Name    │ ID, Email   │                      │   │ │
│   │   │  └─────────────┴─────────────┘                      │   │ │
│   │   │                                                      │   │ │
│   │   │  파티셔닝 전략:                                        │   │ │
│   │   │  - Range: 날짜, ID 범위                                │   │ │
│   │   │  - List: 특정 값 목록                                  │   │ │
│   │   │  - Hash: 해시 함수                                     │   │ │
│   │   │  - Composite: 조합                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 파티셔닝은 단일 서버 내 데이터 분할입니다.

**📢 섹션 요약 비유:** Partitioning = 폴더 정리!
