# 링크 예산 (Link Budget) - 이득/손실

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 송수신 간 신호 세기 계산
> 2. **가치**: 통신 가능 여부 판단
> 3. **융합**: 안테나, 경로 손실과 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

링크 예산(Link Budget)은 송신부터 수신까지의 모든 이득과 손실을 합산하여 수신 신호 세기를 계산하는 것입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    링크 예산 (Link Budget)                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              링크 예산 구성                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [송신기]                                           │   │ │
│   │   │     │                                               │   │ │
│   │   │  Tx Power ────► (+) 송신 이득                       │   │ │
│   │   │     │                                               │   │ │
│   │   │  Tx Antenna Gain (+) 송신 안테나 이득                │   │ │
│   │   │     │                                               │   │ │
│   │   │  [전파 경로]                                        │   │ │
│   │   │     │                                               │   │ │
│   │   │  Path Loss ────► (-) 경로 손실                      │   │ │
│   │   │  Fade Margin ──► (-) 페이딩 여유                    │   │ │
│   │   │  Cable Loss ───► (-) 케이블 손실                    │   │ │
│   │   │     │                                               │   │ │
│   │   │  [수신기]                                           │   │ │
│   │   │     │                                               │   │ │
│   │   │  Rx Antenna Gain (+) 수신 안테나 이득                │   │ │
│   │   │     │                                               │   │ │
│   │   │  Rx Sensitivity ──► 수신 감도                       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              링크 예산 공식                                    │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  Rx Power = Tx Power                                 │   │ │
│   │   │            + Tx Antenna Gain                         │   │ │
│   │   │            + Rx Antenna Gain                         │   │ │
│   │   │            - Path Loss                               │   │ │
│   │   │            - Cable Loss                              │   │ │
│   │   │            - Other Losses                            │   │ │
│   │   │            - Fade Margin                             │   │ │
│   │   │                                                      │   │ │
│   │   │  Link Margin = Rx Power - Rx Sensitivity             │   │ │
│   │   │                                                      │   │ │
│   │   │  • Link Margin > 0: 통신 가능                         │   │ │
│   │   │  • Link Margin < 0: 통신 불가                         │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 링크 예산은 통신 가능 여부를 판단하는 핵심 계산입니다.

**💡 비유**: 링크 예산은 가계부와 같습니다. 수입(이득)에서 지출(손실)을 뺀 것입니다.

**📢 섹션 요약 비유:** Margin > 0 = 통신 가능!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 부호 | 설명 | 전형값 |
|:---|:---:|:---|:---|
| **Tx Power** | + | 송신 전력 | 20-46dBm |
| **Tx Ant Gain** | + | 송신 안테나 | 0-30dBi |
| **Path Loss** | - | 경로 손실 | 100-150dB |
| **Rx Ant Gain** | + | 수신 안테나 | 0-30dBi |
| **Rx Sensitivity** | - | 수신 감도 | -100dBm |

**비교표**

```
+------------+----------+----------+
| 항목       │ 이득     │ 손실     │
+------------+----------+----------+
| 안테나     │ O        │ -        │
│ 증폭기     │ O        │ -        │
│ 케이블     │ -        │ O        │
│ 경로       │ -        │ O        │
│ 커넥터     │ -        │ O        │
+------------+----------+----------+
```

**핵심 알고리즸:** EIRP, EBN0, SNR

**📢 섹션 요약 비유:** EIRP = 송신 전력 + 안테나 이득!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 링크 여유**

| 시스템 | Fade Margin | 용도 |
|:---|:---:|:---|
| **이동통신** | 10-20dB | 실외 |
| **위성** | 3-6dB | LOS |
| **마이크로웨이브** | 20-40dB | 신뢰성 |
| **WiFi** | 10-15dB | 실내/실외 |

**📢 섹션 요약 비유:** 여유 = 안전 마진!
