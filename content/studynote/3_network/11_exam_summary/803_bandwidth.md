# Bandwidth - 대역폭

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 최대 전송 용량
> 2. **가치**: 데이터 전송 속도
> 3. **융합:** Throughput, Network, QoS

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Bandwidth는 최대 전송 용량입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Bandwidth                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              대역폭 개념                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  파이프 비유:                                          │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │                                                │  │   │ │
│   │   │  │  Bandwidth = 파이프 직경                        │  │   │ │
│   │   │  │  Throughput = 실제 흐르는 물의 양               │  │   │ │
│   │   │  │  Latency = 물이 파이프를 통과하는 시간            │  │   │ │
│   │   │  │                                                │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  단위:                                                │   │ │
│   │   │  - bps (bits per second)                             │   │ │
│   │   │  - Kbps (Kilobits)                                   │   │ │
│   │   │  - Mbps (Megabits)                                   │   │ │
│   │   │  - Gbps (Gigabits)                                   │   │ │
│   │   │                                                      │   │ │
│   │   │  예시:                                                │   │ │
│   │   │  - 가정용 인터넷: 100Mbps - 1Gbps                     │   │ │
│   │   │  - 데이터센터: 10Gbps - 100Gbps                       │   │ │
│   │   │  - 백본: 400Gbps+                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Bandwidth는 최대치, Throughput은 실제 사용량입니다.

**📢 섹션 요약 비유:** Bandwidth = 도로 차선 수!
