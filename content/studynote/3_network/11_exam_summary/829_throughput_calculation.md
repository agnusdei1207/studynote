# Throughput Calculation - 처리량 계산

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 실제 전송 속도
> 2. **가치**: 성능 측정
> 3. **융합:** Efficiency, Overhead, Goodput

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Throughput Calculation은 실제 데이터 전송률을 계산합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Throughput Calculation                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              처리량 계산 공식                                   │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  기본 공식:                                            │   │ │
│   │   │  Throughput = 전송 데이터 양 / 시간                     │   │ │
│   │   │             = (파일 크기) / (전송 시간)                 │   │ │
│   │   │                                                      │   │ │
│   │   │  TCP 처리량 (간단):                                    │   │ │
│   │   │  Throughput = Window Size / RTT                       │   │ │
│   │   │                                                      │   │ │
│   │   │  TCP 처리량 (상세):                                    │   │ │
│   │   │  Throughput = (MSS × 8) / (RTT × √p)                  │   │ │
│   │   │  MSS: Maximum Segment Size                            │   │ │
│   │   │  p: 패킷 손실률                                        │   │ │
│   │   │                                                      │   │ │
│   │   │  Goodput vs Throughput:                                │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ Throughput: 전체 전송 (헤더 포함)                  ││   │ │
│   │   │  │ Goodput: 유용한 데이터만                          ││   │ │
│   │   │  │ Goodput = Throughput × (1 - Overhead)            ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  효율:                                                │   │ │
│   │   │  Efficiency = Goodput / Bandwidth × 100%             │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 처리량 계산은 네트워크 성능 분석의 기초입니다.

**📢 섹션 요약 비유:** Throughput = 실제 속도계!
