# 스몰 셀 - Small Cell

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 소형 기지국
> 2. **가치**: 커버리지, 용량 증대
> 3. **융합**: HetNet, 5G, 실내 커버리지

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Femtocell** | 가정용 소형 셀 |
| **Picocell** | 기업용 소형 셀 |
| **Microcell** | 실외 소형 셀 |
| **DAS** | 분산 안테나 시스템 |

**비교표: 백홀 옵션**

```
+------------+----------+----------+----------+
| 백홀       │ 속도     │ 지연     │ 용도     │
+------------+----------+----------+----------+
| 광케이블   │ 높음     │ 낮음     │ 기업     │
| 이더넷     │ 중간     │ 낮음     │ 실내     │
| 무선       │ 중간     │ 중간     │ 실외     │
+------------+----------+----------+----------+
```

**핵심 알고리즸:** 핸드오버, 간섭 관리

**📢 섹션 요약 비유:** Femto = 가정용!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 스몰 셀 vs DAS**
