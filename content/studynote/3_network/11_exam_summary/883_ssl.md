# SSL - 보안 소켓 계층

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: TLS의 전신
> 2. **가치**: 웹 보안 표준
> 3. **융합:** HTTPS, TLS, Deprecation

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SSL (Secure Sockets Layer)은 TLS의 전신입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SSL                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SSL 역사                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  버전 역사:                                              │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │ SSL 1.0: 미발표 (보안 취약)                       ││   │ │
│   │   │  │ SSL 2.0: 1995, 취약, 사용 중단                    ││   │ │
│   │   │  │ SSL 3.0: 1996, POODLE 취약점, 사용 중단           ││   │ │
│   │   │  │ TLS 1.0: 1999, SSL 3.0 개선                       ││   │ │
│   │   │  │ TLS 1.1: 2006                                    ││   │ │
│   │   │  │ TLS 1.2: 2008, 현재 표준                          ││   │ │
│   │   │  │ TLS 1.3: 2018, 최신 표준                          ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  주요 취약점:                                            │   │ │
│   │   │  - POODLE: SSL 3.0 패딩 오라클                         │   │ │
│   │   │  - DROWN: SSL 2.0 공격                                 │   │ │
│   │   │  - BEAST: TLS 1.0 CBC                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  현황:                                                   │   │ │
│   │   │  - SSL 2.0/3.0: 사용 금지                               │   │ │
│   │   │  - TLS 1.0/1.1: 사용 중단 (2020)                        │   │ │
│   │   │  - TLS 1.2/1.3: 권장                                    │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SSL은 역사적 용어로 TLS가 정확한 명칭입니다.

**📢 섹션 요약 비유:** SSL = 구형 보안 표준!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Handshake** | 핸드셰이크 |
| **Record** | 레코드 |
| **Cipher** | 암호화 |
| **MAC** | 메시지 인증 |
| **Alert** | 알림 |

**핵심 알고리즸:** SSL → TLS 마이그레이션

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**SSL vs TLS**

| 항목 | SSL | TLS |
|:---|:---|:---|
| **상태** | 폐지 | 현재 |
| **보안** | 취약 | 안전 |
| **버전** | 2.0/3.0 | 1.2/1.3 |
| **사용** | 금지 | 권장 |
