# SNMP - Simple Network Management Protocol

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 장비 관리 프로토콜
> 2. **가치**: 모니터링, 장애 관리
> 3. **융합**: NMS, MIB, OID

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SNMP는 네트워크 장비를 모니터링하고 관리합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SNMP                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SNMP 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [NMS - Manager]                                    │   │ │
│   │   │        │                                             │   │ │
│   │   │        │ SNMP (UDP 161/162)                          │   │ │
│   │   │        │                                             │   │ │
│   │   │        ├──────────┬──────────┬──────────┐            │   │ │
│   │   │        ▼          ▼          ▼          ▼            │   │ │
│   │   │  [Agent]     [Agent]     [Agent]     [Agent]         │   │ │
│   │   │  (Router)    (Switch)    (Server)    (AP)            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SNMP 버전                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 버전       │ 보안     │ 특징     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ SNMPv1     │ 커뮤니티 │ 기본     │               │   │ │
│   │   │  │ SNMPv2c    │ 커뮤니티 │ 향상     │               │   │ │
│   │   │  | SNMPv3     │ 암호화   │ 보안     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SNMP는 Manager-Agent 구조로 동작합니다.

**💡 비유**: SNMP는 장비 원격 리모컨입니다. 상태를 보고 설정을 바꿉니다.

**📢 섹션 요약 비유:** SNMP = 장비 관리!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **NMS** | Network Management System |
| **Agent** | 장비 내 에이전트 |
| **MIB** | Management Information Base |
| **OID** | Object Identifier |

**비교표: SNMP 메시지**

```
+------------+----------+----------+
| 항목       │ 용도     │ 방향     │
+------------+----------+----------+
| GET        │ 조회     │ M→A      │
| SET        │ 설정     │ M→A      │
| TRAP       │ 알림     │ A→M      │
| WALK       │ 순회     │ M→A      │
+------------+----------+----------+
```

**핵심 알고리즸:** OID 순회, 커뮤니티 스트링

**📢 섹션 요약 비유:** v3 = 보안 SNMP!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 관리 프로토콜**

| 항목 | SNMP | Netconf | RESTCONF |
|:---|:---|:---|:---|
| **형식** │ 바이너리 │ XML | JSON |
| **보안** │ v3 | SSH | HTTPS |
| **현대성** │ 레거시 | 현대적 | 현대적 |

**📢 섹션 요약 비유:** 모니터링 = SNMP!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 장비 모니터링 → SNMPv3
**시나리오 2:** 레거시 장비 → SNMPv2c (격리)
**시나리오 3:** 자동화 → Netconf/RESTCONF

**도입 체크리스트:** 커뮤니티/인증, ACL, 버전 선택

**📢 섹션 요약 비유:** 보안 = SNMPv3!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: 중앙 관리 효율

**미래 전망:** 모델 기반 관리 (YANG)

**참고 표준:** RFC 1157 (v1), RFC 3414 (v3)

**📢 섹션 요약 비유:** SNMP는 네트워크 관리 표준!
