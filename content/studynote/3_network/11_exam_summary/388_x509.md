# X.509 - 인증서 표준

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디지털 인증서 표준 형식
> 2. **가치**: 상호 운용성, 신뢰
> 3. **융합**: TLS, S/MIME, 코드 서명

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

X.509는 디지털 인증서의 표준 형식입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    X.509                                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              X.509 인증서 구조                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  Certificate:                                       │   │ │
│   │   │  +------------------------------------------+       │   │ │
│   │   │  │ Version: V3                              │       │   │ │
│   │   │  │ Serial Number: 01:23:45:67              │       │   │ │
│   │   │  │ Signature Algorithm: SHA256withRSA      │       │   │ │
│   │   │  │ Issuer: CN=Example CA                   │       │   │ │
│   │   │  │ Validity: 2024-01-01 ~ 2025-01-01      │       │   │ │
│   │   │  │ Subject: CN=example.com                 │       │   │ │
│   │   │  │ Public Key: RSA 2048-bit                │       │   │ │
│   │   │  │ Extensions: Key Usage, SAN              │       │   │ │
│   │   │  │ Signature: [CA 서명]                    │       │   │ │
│   │   │  +------------------------------------------+       │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              X.509 확장                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 확장       │ 용도     │ 설명     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ Key Usage  │ 키용도   │ 암호/서명│               │   │ │
│   │   │  │ SAN        │ 대체이름 │ DNS/IP   │               │   │ │
│   │   │  | Basic Constr│ 기본    │ CA 여부  │               │   │ │
│   │   │  │ CRL Dist   │ 폐기    │ CRL 위치 │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: X.509는 버전 3가 가장 널리 사용됩니다.

**💡 비유**: X.509는 디지털 신분증입니다. 형식이 표준화되어 있습니다.

**📢 섹션 요약 비유:** X.509 = 인증서 표준!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Version** | 인증서 버전 (v3) |
| **Serial Number** | 일련번호 |
| **Issuer** | 발급자 |
| **Subject** | 소유자 |
| **Validity** | 유효기간 |
| **Public Key** | 공개키 |

**비교표: X.509 버전**

```
+------------+----------+----------+
| 항목       │ V1/V2    │ V3       │
+------------+----------+----------+
| 확장       │ 없음     │ 있음     │
| SAN        │ 없음     │ 있음     │
| 용도       │ 기본     │ 다양     │
| 현대       │ 레거시   │ 표준     │
+------------+----------+----------+
```

**핵심 알고리즸:** ASN.1 DER 인코딩, 서명 검증

**📢 섹션 요약 비유:** TLS = X.509!

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 인증서 형식**

| 항목 | X.509 | PGP | SSH Key |
|:---|:---|:---|:---|
| **형식** │ ASN.1 │ 바이너리 │ OpenSSH |

| **신뢰** │ 계층 | 신뢰망 | 직접 |

**📢 섹션 요약 비유:** HTTPS = X.509!
