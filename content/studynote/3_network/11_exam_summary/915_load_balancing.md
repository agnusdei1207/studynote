# Load Balancing - 부하 분산

### Ⅰ. 개요 (Context & Background)

**개념 정의**

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **VIP** | 가상 IP |
| **Pool** | 서버 풀 |
| **Member** | 멤버 서버 |
| **Health** | 상태 확인 |
| **Session** | 세션 유지 |

**핵심 알고리듬:** 알고리즘 선택 → 분산 → 모니터링

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**L4 vs L7 로드밸런싱**

| 항목 | L4 | L7 |
|:---|:---|:---|
| **계층** | 전송 | 응용 |
| **기준** | IP:Port | URL/Header |
| **속도** | 빠름 | 느림 |
| **기능** | 단순 | 지능형 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1:** 웹서버 → Round Robin
**시나리오 2:** DB → Least Connections
**시나리오 3:** 세션 → IP Hash
