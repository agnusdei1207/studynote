# Split-Brain - 스플릿 브레인

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 분할로 인한 다중 리더
> 2. **가치**: 인지 및 방지 필요
> 3. **융합:** Quorum, Raft, HA

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Split-Brain은 네트워크 분할 시 발생합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Split-Brain                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              Split-Brain 시나리오                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  정상:                                                │   │ │
│   │   │  ┌───────────────────────────────────────────────┐  │   │ │
│   │   │  │   Node1 ←────→ Node2 ←────→ Node3             │  │   │ │
│   │   │  │         Leader: Node1                         │  │   │ │
│   │   │  └───────────────────────────────────────────────┘  │   │ │
│   │   │                                                      │   │ │
│   │   │  Split-Brain:                                         │   │ │
│   │   │  ┌─────────────────┐    ┌─────────────────┐        │   │ │
│   │   │  │ Node1   Node2   │ X  │   Node3         │        │   │ │
│   │   │  │ Leader  Follower│    │   Leader (?)    │        │   │ │
│   │   │  └─────────────────┘    └─────────────────┘        │   │ │
│   │   │                                                      │   │ │
│   │   │  문제: 두 파티션이 각자 리더를 선출                     │   │ │
│   │   │  → 데이터 불일치 발생                                  │   │ │
│   │   │                                                      │   │ │
│   │   │  해결:                                                │   │ │
│   │   │  - Quorum (과반수 필요)                               │   │ │
│   │   │  - Fencing (STONITH)                                 │   │ │
│   │   │  - Witness Node                                      │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Split-Brain은 쿼럼으로 방지합니다.

**📢 섹션 요약 비유:** Split-Brain = 양두마차!
