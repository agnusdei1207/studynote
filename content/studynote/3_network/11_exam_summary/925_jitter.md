# Jitter - 지터

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 지연 시간 변동
> 2. **가치**: 실시간 서비스 품질 지표
> 3. **융합:** VoIP, Video, Buffer

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Jitter는 패킷 간 지연 시간의 변동을 의미합니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Jitter                                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              지터 발생                                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  지연 시간 변동:                                          │   │ │
│   │   │  ┌─────────────────────────────────────────────────┐│   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  패킷 1: 20ms ─────────────────────→ 도착       ││   │ │
│   │   │  │  패킷 2: 25ms ─────────────────────────→ 도착    ││   │ │
│   │   │  │  패킷 3: 15ms ────────────────→ 도착             ││   │ │
│   │   │  │  패킷 4: 30ms ─────────────────────────────→ 도착││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  │  Jitter = 지연 시간 편차                         ││   │ │
│   │   │  │                                                 ││   │ │
│   │   │  └─────────────────────────────────────────────────┘│   │ │
│   │   │                                                      │   │ │
│   │   │  영향:                                                  │   │ │
│   │   │  - VoIP: 음성 끊김                                     │   │ │
│   │   │  - Video: 화면 끊김                                    │   │ │
│   │   │  - Gaming: 랙                                          │   │ │
│   │   │                                                      │   │ │
│   │   │  해결책:                                                 │   │ │
│   │   │  - Jitter Buffer: 지터 완충                             │   │ │
│   │   │  - QoS: 서비스 품질                                     │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: Jitter는 실시간 통신 품질에 큰 영향을 미칩니다.

**📢 섹션 요약 비유:** Jitter = 도로 교통 상황 변동!
