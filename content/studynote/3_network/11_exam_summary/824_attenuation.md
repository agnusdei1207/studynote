# Attenuation - 감쇠

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 신호 세기 감소
> 2. **가치**: 전송 거리 한계
> 3. **융합:** dB, Amplifier, Repeater

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

Attenuation은 전송 중 신호 세기 감소입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Attenuation                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              신호 감쇠 현상                                     │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  송신                      거리                     수신 │   │ │
│   │   │  ┌───┐                                            ┌───┐│   │ │
│   │   │  │   │ ▔▔▔▔▔▔▔▔ ▔▔▔▔ ▔▔ ▔                          │   ││   │ │
│   │   │  │ S │                                          │ R ││   │ │
│   │   │  │   │                                          │   ││   │ │
│   │   │  └───┘                                            └───┘│   │ │
│   │   │   ↑                    ↓                           ↑   │   │ │
│   │   │  강함              점점 약함                      약함  │   │ │
│   │   │                                                      │   │ │
│   │   │  원인:                                                │   │ │
│   │   │  - 거리 (Distance)                                   │   │ │
│   │   │  - 매질 손실 (Medium Loss)                           │   │ │
│   │   │  - 주파수 (Frequency Dependent)                      │   │ │
│   │   │                                                      │   │ │
│   │   │  측정:                                                │   │ │
│   │   │  Attenuation (dB) = 10 × log₁₀(P₁/P₂)               │   │ │
│   │   │  또는 = 20 × log₁₀(V₁/V₂)                            │   │ │
│   │   │                                                      │   │ │
│   │   │  대응: 증폭기 (Amplifier), 중계기 (Repeater)           │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └────────────────────────────────────────ational──────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: 감쇠는 전송 거리 제한의 주원인입니다.

**📢 섹션 요약 비유:** Attenuation = 멀어지는 소리!
