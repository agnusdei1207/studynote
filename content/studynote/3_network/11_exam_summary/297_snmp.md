# SNMP - Simple Network Management Protocol

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 관리 프로토콜
> 2. **가치**: 장비 상태 수집/제어
> 3. **융합**: NMS, MIB와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

SNMP(Simple Network Management Protocol)는 네트워크 장비를 관리하고 모니터링하는 프로토콜입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    SNMP                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SNMP 구조                                         │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  [NMS - Manager]                                    │   │ │
│   │   │       │                                              │   │ │
│   │   │       │ SNMP (UDP 161/162)                          │   │ │
│   │   │       │                                              │   │ │
│   │   │       ├──────────────┬──────────────┐                │   │ │
│   │   │       ↓              ↓              ↓                │   │ │
│   │   │  ┌──────────┐  ┌──────────┐  ┌──────────┐            │   │ │
│   │   │  │ Router   │  │ Switch   │  │ Server   │            │   │ │
│   │   │  │ (Agent)  │  │ (Agent)  │  │ (Agent)  │            │   │ │
│   │   │  │          │  │          │  │          │            │   │ │
│   │   │  │ MIB      │  │ MIB      │  │ MIB      │            │   │ │
│   │   │  │ OID      │  │ OID      │  │ OID      │            │   │ │
│   │   │  └──────────┘  └──────────┘  └──────────┘            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              SNMP 메시지                                        │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │                                                      │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ 메시지     │ 방향     │ 설명     │               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  │ GET        │ M→A      │ 값 요청  │               │   │ │
│   │   │  │ GET-NEXT   │ M→A      │ 다음 값  │               │   │ │
│   │   │  │ GET-BULK   │ M→A      │ 대량 요청│               │   │ │
│   │   │  │ SET        │ M→A      │ 값 설정  │               │   │ │
│   │   │  │ RESPONSE   │ A→M      │ 응답     │               │   │ │
│   │   │  │ TRAP       │ A→M      │ 이벤트   │               │   │ │
│   │   │  │ INFORM     │ A→M      │ 확인 TRAP│               │   │ │
│   │   │  +------------+----------+----------+               │   │ │
│   │   │  (M: Manager, A: Agent)                            │   │ │
│   │   │                                                      │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────────────┘
```

> **해설**: SNMP는 Manager가 Agent에게 정보를 요청하거나 설정합니다.

**💡 비유**: SNMP는 원격 리모컨입니다. 장비를 제어하고 상태를 확인합니다.

**📢 섹션 요약 비유:** SNMP = 네트워크 관리!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 항목 | 설명 |
|:---|:---|
| **Manager** | NMS |
| **Agent** | 장비 |
| **MIB** | 관리 정보 |
| **OID** | 객체 식별자 |

**비교표: SNMP 버전**

```
+------------+----------+----------+
| 버전       │ 보안     │ 특징     │
+------------+----------+----------+
| SNMPv1     │ Community│ 기본     │
| SNMPv2c    │ Community│ GetBulk  │
| SNMPv3     │ USM/VACM │ 강력     │
+------------+----------+----------+
```

**핵심 알고리즸:** OID 트리, Community String

**📢 섹션 요약 비유:** SNMPv3 = 보안 필수!
