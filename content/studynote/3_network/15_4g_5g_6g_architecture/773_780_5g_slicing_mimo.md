# 773-780. 5G 슬라이싱과 무선 혁신 (Massive MIMO)

> **핵심 인사이트**: 5G의 진짜 힘은 '유연함'에서 나온다. 하나의 거대한 망을 서비스 성격에 따라 수천 개의 독립된 가상망으로 쪼개는 슬라이싱 기술과, 전파를 사용자 방향으로 정확히 조준해 쏘는 빔포밍 기술은 한정된 주파수 자원의 한계를 뛰어넘게 해 준다.

---

## Ⅰ. 네트워크 슬라이싱 (Network Slicing)
하나의 물리적인 네트워크 인프라를 다수의 가상 네트워크(Slice)로 분리하여, 각각의 서비스 요구사항(QoS)에 맞게 자원을 할당하는 기술입니다.

* **핵심 기술**: **SDN (소프트웨어 정의 네트워킹)**과 **NFV (네트워크 기능 가상화)**가 필수적입니다.
* **격리 (Isolation)**: 각 슬라이스는 서로 간섭하지 않도록 철저히 격리되어 보안과 성능을 보장합니다.
* **사례**: 자율주행용 슬라이스(초저지연), 스마트 계량기용 슬라이스(저전력/초연결), 유튜브용 슬라이스(초고속).

---

## Ⅱ. 무선 인터페이스의 진화

### 1. Massive MIMO (대규모 다중 입출력)
* **개념**: 기지국에 수백 개의 안테나 소자를 장착하여 수많은 사용자에게 데이터를 동시에 전송하는 기술입니다.
* **효과**: 주파수 효율을 수십 배 높이고, 전체 시스템 용량을 극대화합니다.

### 2. 빔포밍 (Beamforming) & 빔 트래킹
* **빔포밍**: 전파를 모든 방향으로 낭비하지 않고, **사용자의 위치를 향해 가늘고 날카로운 빔**을 만들어 집중적으로 쏘는 기술입니다.
* **빔 트래킹**: 이동 중인 사용자의 위치를 핀포인트로 추적하며 전파 빔의 방향을 실시간으로 수정하는 기술입니다.

---

## Ⅲ. BSS Coloring (Wi-Fi 6/7 연계 기술)
* **문제**: 주변에 무선 AP가 너무 많으면 신호가 겹쳐 간섭이 발생합니다.
* **해결**: 각 무선 신호에 '색상(Color)' 코드를 부여하여, 내 신호와 남의 신호를 즉시 구분하고 간섭을 무시하는 지능형 채널 관리 기술입니다. (5G 고밀도망에서도 유사한 개념 적용)

---

## Ⅳ. 개념 맵 및 요약

```ascii
[Massive MIMO와 빔포밍]

  [ 기지국 (수백개 안테나) ]
       │
       ├────── ( 빔 1 ) ──────> [ 스마트폰 A ]
       │
       ├────── ( 빔 2 ) ──────> [ 자율주행차 B ]
       │
       └────── ( 빔 3 ) ──────> [ VR 기기 C ]
```

📢 **섹션 요약 비유**: **네트워크 슬라이싱**은 뷔페 식당의 '개별 룸'과 같습니다. 누구나 음식을 먹을 수 있지만, 조용한 대화가 필요한 손님에게는 방음벽이 쳐진 전용 룸(슬라이스)을 내어주는 것입니다. **빔포밍**은 어두운 광장에서 전체 전등을 다 켜는 대신, 사람 한 명 한 명에게만 '레이저 포인터'를 비춰주는 정교한 조명 시스템입니다. **BSS Coloring**은 시끄러운 파티장에서 나와 똑같은 파란색 옷을 입은 사람의 말만 골라 듣는 '선택적 집중' 기술과 같습니다.
