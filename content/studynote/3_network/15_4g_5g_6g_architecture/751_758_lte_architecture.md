# 751-758. 4G LTE 아키텍처와 핵심 기술

> **핵심 인사이트**: 4G LTE는 음성과 데이터를 나누던 과거의 이중 생활을 청산하고, 모든 것을 IP 패킷으로 통일한(All-IP) 혁명적 세대다. 복잡했던 코어망을 EPC로 단순화하고, 주파수 대역을 묶어 속도를 높이는 CA 기술을 통해 진정한 모바일 브로드밴드 시대를 열었다.

---

## Ⅰ. LTE 네트워크 구성 요소 (EPC)
기존의 복잡한 회선 교환망을 없애고 패킷 교환에 최적화된 **EPC (Evolved Packet Core)** 구조를 채택했습니다.

1. **MME (Mobility Management Entity)**: 코어망의 컨트롤 타워. 가입자 인증, 핸드오버 제어 등 '신호(Signaling)' 처리를 전담합니다.
2. **S-GW (Serving Gateway)**: 기지국과 코어망 사이의 징검다리. 패킷을 전송하는 데이터 평면(User Plane) 역할을 합니다.
3. **P-GW (PDN Gateway)**: 외부 인터넷망과 연결되는 최종 관문. IP 주소를 할당하고 QoS 정책을 적용합니다.
4. **HSS (Home Subscriber Server)**: 모든 가입자의 마스터 정보(전화번호, 요금제, 현재 위치 등)가 담긴 거대 데이터베이스입니다.

---

## Ⅱ. 기지국과 무선 기술 (eNodeB)
* **eNodeB**: LTE의 지능형 기지국입니다. 과거와 달리 무선 자원 관리와 라우팅 기능을 직접 수행하여 지연 시간을 줄였습니다.
* **OFDMA**: 주파수를 아주 잘게 쪼개어 수많은 사용자에게 효율적으로 배분하는 LTE의 핵심 변조 방식입니다.

---

## Ⅲ. LTE-Advanced의 핵심: CA (Carrier Aggregation)
서로 떨어진 여러 개의 주파수 대역을 하나로 묶어서 마치 하나의 넓은 대역폭처럼 사용하는 기술입니다.

* **효과**: 도로 차선을 합치는 것과 같아서, 대역폭이 넓어진 만큼 데이터 전송 속도가 정비례하여 상승합니다. (예: 10MHz + 20MHz = 30MHz 효과)

---

## Ⅳ. VoLTE (Voice over LTE)
음성 통화도 2G/3G 망으로 우회하지 않고, **LTE 데이터망(IP) 위에서 패킷 방식**으로 처리하는 기술입니다.

* **장점**: 서킷망보다 훨씬 넓은 대역폭을 사용하여 음질이 깨끗하고(HD Voice), 통화 연결 시간이 매우 짧습니다.
* **프로토콜**: **SIP**와 **RTP**를 기반으로 동작합니다.

---

## Ⅴ. 개념 맵 및 요약

```ascii
[LTE 전체 아키텍처]

  [ Mobile ] <──(무선)──> [ eNodeB ] <──(S1-U)──> [ S-GW ] <───> [ P-GW ] ──> [ Internet ]
                             │                      │             │
                             └────────(S1-MME)───> [ MME ] <───> [ HSS ]
                                                  (제어부)       (가입자DB)
```

📢 **섹션 요약 비유**: **EPC**는 간소화된 '스마트 우체국'입니다. **MME**는 편지가 어디로 갈지 분류하고 주소를 확인하는 '관리 소장님'이고, **S-GW/P-GW**는 실제로 택배 상자를 나르는 '운송 트럭'입니다. **CA**는 좁은 1차선 도로 여러 개를 합쳐서 널찍한 '8차선 고속도로'를 만드는 공법이며, **VoLTE**는 무전기 대화(3G) 대신 고음질 인터넷 전화(IP)로 수다를 떠는 세련된 통화 방식입니다.
