# 901-904. 지능형 네트워크 운영 (AIOps, ADN)

> **핵심 인사이트**: 네트워크는 이제 스스로 숨 쉬고 생각한다. 사람이 로그를 뒤지는 시대는 끝났다. AI가 수조 개의 데이터를 실시간으로 분석해 장애를 예견하고(AIOps), 스스로 경로를 바꾸는 자율 네트워크(ADN)가 미래의 표준이다.

---

## Ⅰ. AIOps (AI for IT Operations)
인공지능과 머신러닝을 활용하여 IT 인프라(네트워크, 서버 등) 운영을 자동화하는 기술입니다.

* **핵심 기능**:
  - **이상 징후 탐지**: 평소와 다른 아주 미세한 트래픽 변화를 감지하여 장애를 미리 알림.
  - **근본 원인 분석 (RCA)**: 수천 개의 경고 메시지 중 진짜 원인이 무엇인지 AI가 짚어줌.
  - **자동 조치**: 간단한 장애는 사람이 개입하기 전 AI가 스스로 재부팅하거나 경로 변경.

---

## Ⅱ. ADN (Autonomous Driving Network)
네트워크가 사람의 개입 없이 스스로 계획, 구성, 운영, 최적화하는 **자율 주행 네트워크**입니다.

* **단계 (Level 0 ~ 5)**:
  - **L0**: 수동 운영.
  - **L2**: 부분 자동화 (특정 상황만 자동).
  - **L5**: 완전 자율 (사람은 목적지만 말하면 AI가 전체 망을 관리).

---

## Ⅲ. 디지털 트윈 네트워크 (DTN)
실제 네트워크와 똑같은 가상의 복제본(Digital Twin)을 만드는 기술입니다.

* **용도**: 
  - **사전 검증**: 새로운 설정을 적용하기 전, 가상 망에서 미리 돌려보고 문제가 없는지 확인.
  - **영향도 분석**: 특정 회선이 끊어졌을 때 전체 망에 어떤 영향이 올지 실시간 시뮬레이션.

---

## Ⅳ. 가시성 확보 기술 (Observability)
* **퍼시스턴트 토폴로지**: 컨테이너나 가상 장비가 수시로 생겼다 사라지는 환경에서, 그들 간의 연결 관계를 그래프 DB 등으로 끊임없이 추적하고 기록하는 기술입니다.

---

## Ⅴ. 개념 맵 및 요약

```ascii
[지능형 운영 루프]

  (수집) Telemetry / Log ──> (분석) AI / ML ──> (판단) Policy Engine
           ^                                          │
           └────────── ( 조치 ) ── [ Infra ] <────────┘
```

📢 **섹션 요약 비유**: **AIOps**는 환자의 상태를 24시간 감시하며 발병 전에 위험 신호를 보내주는 'AI 간병인'입니다. **ADN**은 운전자가 핸들을 놓아도 목적지까지 안전하게 데려다주는 '자율 주행 자동차'와 같으며, **디지털 트윈**은 실제 수술 전 의사가 가상 현실에서 미리 연습해 보는 '수술 시뮬레이터'입니다. **가시성 기술**은 쉴 새 없이 움직이는 개미 떼(컨테이너)의 이동 경로를 놓치지 않고 쫓는 '추적 카메라'와 같습니다.
