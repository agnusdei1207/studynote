# OFDM (직교 주파수 분할 다중화) 및 부반송파, CP/GI

> 1. **본질**: 전체 대역폭을 서로 직교(Orthogonal)하는 수많은 저속의 **부반송파(Subcarriers)**로 나누어 데이터를 병렬 전송함으로써, 주파수 간섭 없이 스펙트럼을 중첩하여 대역폭 효율을 극대화하는 멀티캐리어(Multi-carrier) 변조 방식이다.
> 2. **가치**: 무선 환경의 고질적 문제인 다중 경로 페이딩(Multipath Fading)과 심볼 간 간섭(ISI)을 **보호 구간(CP/GI)** 삽입을 통해 효과적으로 극복하며, 현대 초고속 무선망의 속도 한계를 결정짓는 핵심 물리 계층 기술이다.
> 3. **융합**: Wi-Fi(802.11a/g/n/ac/ax), 4G LTE, 5G NR, 그리고 디지털 방송(DMB/DVB)의 표준 전송 기술이며, 대중적인 초고속 인터넷 서비스의 기술적 근간이다.

+++

## Ⅰ. 개요 (Context & Background)

- **개념**: **OFDM**은 주파수를 아주 잘게 쪼개서(부반송파) 데이터를 나누어 담는 기술이다. 기존 FDM은 채널 사이를 띄워야(Guard Band) 했지만, OFDM은 수학적 '직교성'을 이용해 채널을 겹쳐서(Overlap) 사용한다. 여기에 **CP(Cyclic Prefix)**라는 안전장치를 더해 신호가 겹치는 부작용을 막는다.

- **💡 비유**: 좁은 문을 통과하는 거대한 짐차에 비유할 수 있습니다. 한 번에 큰 짐을 실은 차(Single Carrier)는 문에 걸리기 쉽지만, 짐을 아주 작은 수백 개의 상자로 나눠서 수백 명의 사람(부반송파)이 동시에 들고 지나가면 훨씬 빠르고 안전하게 통과할 수 있는 것과 같습니다.

- **등장 배경**:
  1. **대역폭 효율성 한계**: 기존 FDM은 보호 대역(Guard Band)으로 인한 주파수 낭비가 너무 심했다.
  2. **고속 전송의 ISI 문제**: 데이터 속도가 빨라지면 심볼 주기가 짧아져, 반사되어 늦게 온 이전 신호가 다음 신호를 덮어쓰는 간섭(ISI)이 심해졌다.
  3. **디지털 연산 능력 향상**: 복잡한 직교 연산을 실시간으로 처리할 수 있는 **IFFT/FFT** 칩셋이 상용화되면서 OFDM 시대가 열렸다.

- **📢 섹션 요약 비유**: 주파수라는 영토를 낭비 없이 촘촘하게 메우고, "시간의 마진(CP)"을 두어 전파의 반사 공격으로부터 정보를 지켜내는 현대 무선 통신의 꽃입니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 1. 직교성(Orthogonality)과 부반송파(Subcarrier)

OFDM의 핵심은 특정 부반송파의 정점(Peak) 지점에서 다른 부반송파의 값이 정확히 '0'이 되게 설계하는 것이다.

```text
 [ OFDM 스펙트럼 중첩 시각화 ]
 
 Energy ^
        │    / \     / \     / \
        │   /   \   /   \   /   \
        │  /     \ /     \ /     \
        └─┴───────●───────●───────●──────▶ Frequency
                 f1      f2      f3
```

**[다이어그램 해설]** $f1$ 주파수의 에너지가 최대일 때, 이웃한 $f2, f3$의 에너지는 바닥(Null)을 친다. 이 수학적 약속 덕분에 필터로 채널을 멀리 떼어놓지 않고도 서로 간섭 없이 데이터를 뽑아낼 수 있다. 이를 통해 기존 FDM 대비 약 **50% 이상의 주파수 절감** 효과를 얻는다.

+++

### 2. CP (Cyclic Prefix)와 GI (Guard Interval)

다중 경로 페이딩에 의한 **ISI(심볼 간 간섭)**를 막기 위한 기술이다.

- **GI (Guard Interval)**: 심볼과 심볼 사이에 두는 빈 시간 간격. (반사파가 들어올 때까지 기다려주는 시간)
- **CP (Cyclic Prefix)**: 심볼의 뒷부분을 복사하여 GI 구간에 채워 넣는 것. 단순히 비워두는 것보다 위상 연속성을 유지해주어 **ICI(채널 간 간섭)**까지 방지한다.

```text
 [ OFDM 심볼 구조 ]
 
 ┌───────────┬────────────────────────────┐
 │  CP (복사본) │      원본 유효 심볼 (Data)     │
 └───────────┴────────────────────────────┘
 <── GI 구간 ──>
```

+++

### 3. IFFT / FFT 연산

OFDM의 물리적 구현은 복잡한 하드웨어 대신 수학적 변환으로 이루어진다.
- **송신 (IFFT)**: 주파수 영역의 병렬 데이터를 시간 영역의 파형으로 합침.
- **수신 (FFT)**: 시간 영역의 파형에서 다시 주파수별 데이터를 분리해냄.

- **📢 섹션 요약 비유**: 수백 명의 합창단원(부반송파)이 각자의 음을 내도, 완벽한 화음(직교성)을 이루어 소리가 섞이지 않고, 한 곡이 끝날 때마다 짧은 휴식(CP)을 주어 에코(반사파)가 사라지길 기다리는 정교한 공연입니다.

## Ⅲ. 융합 비교 및 다각도 분석

### 일반 FDM vs OFDM 비교

| 비교 항목 | 일반 FDM (Single Carrier) | OFDM (Multi-carrier) |
|:---|:---|:---|
| **스펙트럼 활용** | 채널 간 분리 (비효율) | **채널 간 중첩 (고효율)** |
| **ISI 내성** | 약함 (이퀄라이저 필수) | **매우 강함 (CP로 해결)** |
| **대역폭 효율** | 낮음 | **매우 높음** |
| **구현 방식** | 아날로그 필터 위주 | **디지털 IFFT/FFT 연산** |
| **최대 단점** | 주파수 낭비 | **PAPR (전력 변동폭)이 큼** |

### 과목 융합 관점

- **컴퓨터 구조 (CA)**: FFT 연산은 막대한 부동소수점 연산량을 요구한다. 따라서 스마트폰 AP 내의 DSP나 통신 모뎀에는 FFT 전용 하드웨어 가속기가 반드시 탑재된다.
- **물리 (전자기학)**: OFDM은 주파수 선택적 페이딩(Frequency Selective Fading)에 강하다. 특정 부반송파가 노이즈로 죽어도, 다른 수많은 부반송파가 살아있어 에러 정정(FEC)을 통해 데이터 복구가 가능하기 때문이다.

- **📢 섹션 요약 비유**: FDM이 "각자 다른 방에서 소리 지르기"라면, OFDM은 "같은 방에서 주파수라는 서로 다른 악기로 오케스트라 연주하기"입니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 지하철 Wi-Fi 끊김 현상**: 지하철이 빠르게 이동하면서 금속 벽면에 반사된 전파들이 시간차를 두고 들어온다. 기술사는 **OFDM의 CP 길이**를 점검한다. 반사파의 지연 시간(Delay Spread)보다 CP 길이가 짧으면 ISI가 발생하여 통신이 끊긴다. 기술사는 CP 길이를 확장하는 파라미터 튜닝을 통해 전송 속도는 약간 희생하되 연결 안정성을 확보한다.

2. **시나리오 — 5G 기지국 전력 효율 저하**: 5G 기지국의 전기 요금이 너무 많이 나온다. 분석 결과 OFDM 신호의 높은 **PAPR(Peak-to-Average Power Ratio)**이 원인이었다. 수많은 부반송파가 겹치면서 순간 전력이 튀어 증폭기가 비효율적으로 동작하고 있었다. 기술사는 **CFR(Crest Factor Reduction)** 기술과 디지털 왜곡 보상(DPD)을 적용하여 전력 소모를 20% 이상 절감한다.

### 도입 체크리스트
- **CP 길이 설정**: 설치 환경의 최대 지연 확산(Delay Spread)을 감안하여 CP가 충분히 긴가?
- **부반송파 간격 ($\Delta f$)**: 이동 속도에 의한 도플러 편이를 견딜 만큼 간격이 넓은가? (5G는 이를 위해 가변 부반송파 간격 적용)
- **IFFT/FFT 해상도**: 채널 수에 따른 연산 정밀도가 시스템의 에러 요구사항(EVM)을 충족하는가?

### 안티패턴
- **주파수 동기화 무시**: OFDM은 직교성이 생명이다. 송수신기의 주파수가 단 1%만 틀어져도 직교성이 무너져 모든 부반송파가 서로를 간섭하는 대참사가 벌어진다.
- **과도하게 긴 CP**: ISI를 잡겠다고 CP를 너무 길게 잡으면, 실제 데이터를 보내는 시간이 줄어들어 유효 전송 속도(Goodput)가 반 토막 난다.

- **📢 섹션 요약 비유**: 아무리 좋은 엔진(OFDM)을 달았어도, 바퀴(주파수 동기)가 정렬되지 않거나 연료(전력) 관리 효율이 엉망이면 슈퍼카는 굴러가지 않습니다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과

| 구분 | 도입 전 (Single Carrier) | 도입 후 (OFDM) | 개선 효과 |
|:---|:---|:---|:---|
| **전송 속도** | 수 Mbps 한계 | **Gbps급 초고속 구현** | 속도 **100배 이상** 향상 |
| **망 안정성** | 장애물/반사파에 취약 | 다중 경로 환경에 최적화 | 가동률 및 신뢰성 극대화 |
| **경제성** | - | 좁은 주파수로 더 많은 데이터 | 주파수 경매 가치 극대화 |

### 미래 전망
OFDM은 이제 단순한 전송 기술을 넘어, 각 사용자에게 부반송파 묶음을 나누어주는 **OFDMA**로 발전하여 5G와 Wi-Fi 6의 심장이 되었다. 미래의 6G에서는 인공지능이 채널 상황에 맞춰 부반송파의 개수와 직교성 파라미터를 실시간으로 빚어내는 **지능형 메타-변조**로 진화하여, 물리적 한계인 섀넌의 채널 용량에 더욱 근접할 것이다.

### 참고 표준
- **IEEE 802.11ax**: Wi-Fi 6 (OFDMA 기술 적용)
- **3GPP Release 15/16**: 5G NR의 가변 부반송파 간격(SCS) 규격

- **📢 섹션 요약 비유**: 좁은 공간에 수백 개의 바늘을 세우는 정교함과, 거친 파도를 기다려주는 여유(CP)가 만난 무선 통신의 마스터피스입니다.

## 📌 관련 개념 맵 (Knowledge Graph)
- **직교성 (Orthogonality)**: OFDM이 성립할 수 있는 유일한 물리적 원리.
- **OFDMA**: OFDM을 여러 사용자가 나누어 쓰는 다중 접속 기술.
- **PAPR (전력 대 평균 전력비)**: OFDM 설계 시 반드시 해결해야 할 최대의 숙제.
- **IFFT / FFT**: OFDM을 디지털로 구현하는 마법의 수학 도구.
