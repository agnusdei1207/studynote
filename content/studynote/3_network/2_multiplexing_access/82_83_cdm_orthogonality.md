# 코드 분할 다중화 (CDM) 및 직교성 (Orthogonality)

> 1. **본질**: 다수의 사용자가 동일한 주파수 대역과 시간 슬롯을 동시에 공유하되, 각 사용자에게 서로 **직교(Orthogonal)**하는 고유한 확산 코드(Spreading Code)를 부여하여 논리적으로 채널을 분리하는 다중화 기술이다.
> 2. **가치**: 주파수와 시간 자원의 물리적 분할 한계를 극복하여 망 수용량을 극대화하며, 직교성 원리를 통해 상호 간섭(Interference) 없이 자신의 신호만을 정확히 추출할 수 있는 수학적 무결성을 제공한다.
> 3. **융합**: 3G 이동통신(WCDMA)의 근간인 CDMA 기술의 핵심이며, 현대 무선 통신의 보안성 확보와 GPS 위성 항법 시스템의 정밀도 구현을 위한 필수 원리이다.

+++

## Ⅰ. 개요 (Context & Background)

- **개념**: **CDM**은 모든 사람이 같은 방에서 동시에 말하는 것과 같다. 대신 각자 다른 '언어(Code)'로 말하기 때문에, 내가 아는 언어만 귀담아들으면 다른 사람의 말은 단순한 소음으로 처리되는 원리이다. 이때 언어들이 서로 전혀 겹치지 않아야 하는데, 이 성질을 **직교성(Orthogonality)**이라고 한다.

- **💡 비유**: 칵테일 파티 효과에 비유할 수 있습니다. 수많은 사람이 떠들어도 내가 아는 친구의 목소리(코드)에만 집중하면 대화가 가능한 것과 같습니다. 수학적으로는 두 벡터를 내적했을 때 결과가 0이 되는 '수직' 관계를 통신에 이용하는 것입니다.

- **등장 배경**:
  1. **자원 고갈**: 주파수(FDM)와 시간(TDM) 자원은 이미 포화 상태였다. 같은 자원을 중첩해서 쓸 새로운 차원이 필요했다.
  2. **보안 및 은밀성**: 특정 코드를 모르면 도청이 불가능한 은닉 통신에 대한 요구가 컸다.
  3. **다중 경로 페이딩 저항**: 전파가 반사되어 생기는 간섭을 오히려 신호 강화에 활용(Rake Receiver)할 수 있는 기술이 필요했다.

- **📢 섹션 요약 비유**: 주파수라는 영토와 시간이라는 기회를 공평하게 나눠 갖는 대신, "코드라는 암호"를 통해 각자의 비밀 통로를 만드는 수학적 다중화 기술입니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 직교성 (Orthogonality)의 수학적 정의

두 신호(또는 코드) $C_1$과 $C_2$가 직교한다는 것은, 한 주기 동안 두 신호를 곱해서 합산(적분)한 값이 0이 됨을 의미한다.

- **수식**: $\int_{0}^{T} C_1(t) \cdot C_2(t) dt = 0$
- **의미**: 수신측에서 자기 코드($C_1$)를 곱하면 정보가 살아나지만, 남의 코드($C_2$)를 곱하면 결과가 0이 되어 사라진다.

+++

### CDM 전송 메커니즘 (Walsh Code 예시)

송신자는 데이터 비트에 직교 코드를 곱해 확산시키고, 수신자는 동일 코드를 곱해 역확산한다.

```text
 [ CDM 송수신 시뮬레이션 ]
 
 1. 사용자 A: 데이터(1) x 코드(1, 1, -1, -1) = 전송(1, 1, -1, -1)
 2. 사용자 B: 데이터(-1) x 코드(1, -1, 1, -1) = 전송(-1, 1, -1, 1)
 
 3. 공중(합산): (1-1, 1+1, -1-1, -1+1) = (0, 2, -2, 0) <-- 합쳐진 신호
 
 4. 수신자 A의 복원: (0, 2, -2, 0) 내적 (1, 1, -1, -1)
    => (0*1) + (2*1) + (-2*-1) + (0*-1) = 0 + 2 + 2 + 0 = 4 (양수 -> 1 복원!)
 
 5. 수신자 A가 본 B의 신호: (0, 2, -2, 0) 내적 (1, -1, 1, -1)
    => (0*1) + (2*-1) + (-2*1) + (0*-1) = 0 - 2 - 2 + 0 = -4 (음수 -> -1 복원!)
```

**[다이어그램 해설]** 신호들이 공중에서 뒤섞여 엉망이 된 것 같지만, 각자의 직교 코드로 내적(Inner Product) 연산을 수행하면 신기하게도 자기 데이터만 정확한 크기로 솟아오른다. 다른 사용자의 신호는 직교성 덕분에 연산 결과가 0이 되어 완벽히 제거된다.

+++

### 주요 직교 코드 유형

| 코드 명칭 | 특징 | 주요 용도 | 비유 |
|:---|:---|:---|:---|
| **왈시 코드 (Walsh)** | 완벽한 직교성, 동기식에 최적 | CDMA 채널 구분 (순방향) | 똑바른 격자무늬 |
| **PN 코드 (Pseudo Noise)** | 잡음과 유사, 자기 상관 우수 | 사용자 식별, 확산 (역방향) | 복잡한 미로 패턴 |
| **골드 코드 (Gold)** | 다수의 가용 코드 생성 가능 | GPS, WCDMA | 수많은 열쇠 꾸러미 |

- **📢 섹션 요약 비유**: 서로 다른 언어를 쓰는 사람들이 동시에 떠들어도, 내가 배운 단어장(코드)만 가지고 필터링하면 오직 친구의 목소리만 들리는 "수학적 칵테일 파티"입니다.

## Ⅲ. 융합 비교 및 다각도 분석

### FDM, TDM vs CDM (다중화의 패러다임 쉬프트)

| 비교 항목 | FDM (주파수) | TDM (시간) | CDM (코드) |
|:---|:---|:---|:---|
| **분할 방식** | 주파수 대역 분할 | 시간 슬롯 분할 | **수학적 코드 분할** |
| **자원 공유** | 동시 사용 불가 | 동시 사용 불가 | **주파수/시간 동시 사용** |
| **간섭 방지** | 보호 대역 (Band) | 보호 시간 (Slot) | **직교성 (Orthogonality)** |
| **보안성** | 낮음 | 낮음 | **매우 높음 (LPI/LPD)** |
| **수용 수단** | 하드웨어 필터 | 타이밍 제어 로직 | **디지털 상관기 (Correlator)** |

### 과목 융합 관점

- **컴퓨터 구조 (CA)**: CDM 연산을 실시간으로 처리하기 위해 DSP(Digital Signal Processor) 내부에 곱셈-누산(MAC, Multiply-Accumulate) 연산기가 수천 개씩 병렬로 배치된다.
- **물리 (전자기학)**: 전파의 위상이 틀어지면 직교성이 무너진다. 이를 방지하기 위해 송수신기의 위상을 나노초($ns$) 단위로 정렬하는 위상 동기화 기술이 필수적이다.

- **📢 섹션 요약 비유**: FDM이 "칸막이 방", TDM이 "시간제 대여"라면, CDM은 "투명한 공기 속에서 서로 다른 색깔의 안경을 쓰고 대화하는 것"입니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 이동통신 기지국 용량 부족 해결**: 특정 지역에 사용자가 몰려 주파수 자원이 부족하다. 기술사는 **CDMA(코드 분할 다중 접속)**의 코드 할당 효율을 점검한다. 사용자별로 할당된 왈시 코드의 길이를 조정(가변 확산 인자, OVSF)하여, 저속 사용자는 짧은 코드를, 고속 사용자는 긴 코드를 쓰게 함으로써 제한된 코드 공간 내에서 수용 인원을 2배 이상 늘리는 소프트웨어적 최적화를 수행한다.

2. **시나리오 — 군사 기지 보안 통신망 구축**: 적의 강력한 감청과 재밍이 우려된다. 기술사는 **CDM 기반의 확산 통신**을 제안한다. 신호 에너지를 잡음 바닥(Noise Floor)보다 낮게 넓게 펴서 전송하므로, 적은 통신이 일어나는지조차 알기 힘들고(Low Probability of Intercept), 설령 알더라도 수조 개의 코드 조합을 풀지 못하면 내용을 읽을 수 없는 철통 보안을 실현한다.

### 도입 체크리스트
- **Near-Far 문제 해결**: 기지국 근처 단말기가 너무 세게 쏘면 먼 단말기의 신호가 묻힌다. 초당 800번 이상의 정밀한 **전력 제어(Power Control)**가 구현되어 있는가?
- **코드 자동 상관(Auto-correlation)**: 수신측에서 코드의 시작점을 정확히 찾을 수 있도록 코드 자체의 타이밍 특성이 우수한가?
- **동기화 정밀도**: 다중 경로 반사파를 합치기 위한 레이크 수신기(Rake Receiver)의 핑거(Finger) 개수가 충분한가?

### 안티패턴
- **비직교 코드 남용**: 코드가 서로 완벽히 수직이지 않으면 '다중 사용자 간섭(MAI)'이 발생하여, 사용자가 늘어날수록 통화 품질이 급격히 떨어진다. (셀 호흡 현상의 원인)
- **전력 제어 실패**: 단 한 명의 사용자가 전력 제어에 실패해 신호를 너무 세게 쏘면, 해당 섹터 내의 수백 명의 통신이 일시에 마비될 수 있다.

- **📢 섹션 요약 비유**: 모두가 똑똑하게 자기 목소리 크기를 조절(전력 제어)하고 정해진 약속(직교 코드)을 지켜야만 성립되는 "고도의 사회적 협약" 기술입니다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과

| 구분 | 도입 전 (FDM/TDM) | 도입 후 (CDM) | 개선 효과 |
|:---|:---|:---|:---|
| **수용 용량** | 물리적 채널 수 제한 | 코드 수에 의한 유연한 확장 | 수용량 **3~5배** 증대 |
| **통화 품질** | 핸드오버 시 끊김 | 소프트 핸드오버 가능 | 호 절단율 **90%** 감소 |
| **보안 등급** | 단순 암호화 의존 | 물리 계층 자체 은닉 | 군사급 보안 수준 달성 |

### 미래 전망
순수 CDM은 현대의 OFDM 기술에 주도권을 내주었으나, 그 핵심인 **직교성(Orthogonality)** 원리는 5G/6G의 기초인 OFDMA와 MIMO 기술 속에 더욱 정교하게 녹아들었다. 특히 수조 개의 기기가 연결되는 초연결 IoT 시대에는, 각 기기를 식별하고 간섭을 제어하는 '코드'의 역할이 다시 한번 중요해질 것으로 보인다.

### 참고 표준
- **3GPP IS-95**: 세계 최초의 상용 CDMA 표준
- **ITU-T Q.1741**: IMT-2000(3G)을 위한 무선 인터페이스 규격

- **📢 섹션 요약 비유**: 보이지 않는 수학의 선(직교성)이 전파의 혼돈을 질서로 바꾸었습니다. 숫자가 어떻게 물리적 공간을 지배하는지 보여주는 통신 공학의 걸작입니다.
