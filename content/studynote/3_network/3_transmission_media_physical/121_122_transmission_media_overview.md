# 121-122. 전송 매체의 분류와 평행 2선식 케이블

> **핵심 인사이트 (3줄 요약)**
> 1. **본질**= 전송 매체(Transmission Medium)는 OSI 7계층 중 **물리 계층(Physical Layer)**의 구성 요소로, 정보를 전기 신호나 광 신호로 변환하여 전달하는 물리적 통로이자 가장 하드웨어에 의존적인 계층입니다.
> 2. **가치**= 유도 매체는 보안성과 고속성을, 비유도 매체는 이동성과 광역 커버리지를 제공하며, **대역폭(Bandwidth)**, **노이즈 면역력(Noise Immunity)**, **비용(Cost)**의 상관관계에 따른 최적의 매체 선택이 네트워크 성능을 좌우합니다.
> 3. **융합**= 단순한 구리선(Twin-lead)에서 시작된 유도 매체 기술은 쉴딩(Shielding)과 광학 기술로 진화하여 현대의 **고속 이더넷(Gigabit Ethernet)**과 **FTTH (Fiber To The Home)** 인프라의 기반이 되었습니다.

## Ⅰ. 개요 (Context & Background) - 물리적 연결의 근간

### 1. 개념 및 철학
**전송 매체(Transmission Medium)**란 데이터 통신 시스템에서 송신자(Transmitter)와 수신자(Receiver) 사이에 존재하는 물리적인 경로를 의미합니다. OSI 7계층 참조 모델(Open Systems Interconnection Reference Model)의 가장 하위 계층인 **물리 계층(Physical Layer)**에 속하며, 상위 계층에서 내려온 비트 열(Bit stream)을 전압의 변화, 빛의 깜빡임, 전파의 진동과 같은 **물리적 신호(Physical Signal)**로 바꾸어 전달하는 역할을 수행합니다. 즉, 소프트웨어적인 논리가 아닌 전기적, 기계적, 광학적인 특성에 직접적으로 영향을 받는 하드웨어 영역입니다.

### 2. 💡 비유: 물의 흐름과 도관
데이터를 '물'이라고 가정한다면, 전송 매체는 물이 흐르는 '파이프'나 '수로'와 같습니다. 파이프가 두꺼울수록(대역폭이 넓을수록) 한 번에 많은 물을 보낼 수 있고, 파이프 내부가 매끄러울수록(노이즈가 적을수록) 물이 막히지 않고 빠르게 흐릅니다.

### 3. 등장 배경 및 진화
① **초기 통신의 한계**: 19세기 초, 전신과 전화의 등장으로 전기 신호를 먼 거리 보낼 필요성이 대두되었습니다. 가장 간단한 구조인 두 개의 구리선을 나란히 배치한 **평행 2선식(Twin-lead)**이 사용되었으나, 거리가 멀어질수록 신호 감쇠(Attenuation)와 외부 잡음(Noise)이 심각한 문제로 대두되었습니다.
② **구조적 혁신**: 이를 극복하기 위해 신호선과 접지선을 동심원으로 배치한 **동축 케이블(Coaxial Cable)**이 개발되었고, 이후 노이즈를 상쇄하는 **꼬임 쌍선(Twisted Pair)**과 전기적 영향을 완전히 차단하는 **광섬유(Optical Fiber)**로 진화했습니다.
③ **무선의 필요성**: 이와 동시에 유선 선로를 매설하기 힘든 이동통신과 위성 통신의 요구로 인해 공기 중을 떠다니는 전파를 이용한 **비유도 매체(Unguided Media)** 기술이 발전하였습니다.

### 4. 섹션 요약 비유
📢 **섹션 요약 비유**: 전송 매체의 발전 과정은 **진흙길에서 포장도로로, 그리고 고속철도로 변하는 과정과 같습니다.** 평행 2선식은 자갈이 깔린 비포장도로처럼 충격에 취약하고 비효율적이었으나, 비유도 매체(비행기)와 유도 매체(고속철도)의 기술적 분화와 진화를 통해 현재의 초고속 인터넷 인프라가 구축되었습니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive) - 구조와 특성

전송 매체는 크게 신호가 물리적인 경계 내를 이동하는 **유도 매체(Guided Media)**와 공간으로 퍼져나가는 **비유도 매체(Unguided Media)**로 나뉩니다.

### 1. 유도 매체 (Guided Media) 상세 분석
물리적인 선로에 의해 신호의 경로가 제한되는 매체입니다. 전도체(Conductor)의 재질과 배치 방식에 따라 특성이 결정됩니다.

#### A. 구성 요소 및 특성 비교표
| 요소명 | 평행 2선식 (Twin-lead) | 꼬임 쌍선 (Twisted Pair) | 동축 케이블 (Coaxial) | 광섬유 (Optical Fiber) |
|:---|:---|:---|:---|:---|
| **구조** | 두 줄의 평행선 | 두 선을 꼬아놓음 | 중심선+차폐링+외피 | 유리/플라스틱 심자 |
| **전송 매체** | 전기(구리) | 전기(구리) | 전기(구리) | **빛(Light)** |
| **대역폭** | 극히 낮음 (Low) | 중간~높음 (UTP, STP) | 높음 (High) | **极高 (THz급)** |
| **간섭 면역** | 취약함 (Very Poor) | 보통~좋음 (차폐에 따름) | 우수함 (Shielding 효과) | 완벽함 (EMI 무관) |
| **주요 용도 | 구형 TV 수신 안테나 | **LAN (Ethernet)** | TV 유선 방송, 케이블 모뎀 | 장거리 통신, 백본망 |

#### B. 아키텍처 ASCII 다이어그램
아래는 각 매체의 단면 구조를 시각화한 것입니다.

```ascii
      [평행 2선식]               [꼬임 쌍선]               [동축 케이블]              [광섬유]
      (Open Wire)               (Twisted Pair)            (Coaxial)                 (Fiber)
      
   +---------+---------+      +------------------+     +-------------------+     +---------------+
   |  [전선1] |  [전선2] |     | (꼬인 형태)       |     |  내부 절연체        |     |   코어(Cladding) |
   |   (신호) |   (GND)  |     | +--------------+ |     | +--------------+  |     |   // 글래스 //  |
   +---------+---------+     | | (Signal)     | |     | | [Core]       |  |     |   //  (Core) //  |  <-- 빛 이동 경로
      ^      ^                | | (Ground)     | |     | | (Signal)     |  |     |   ////////////// |
      |      |                | +--------------+ |     | +--------------+  |     |   코어(Clad)    |
      |______|                |   (Pair 1)       |     |   차폐망(Shield)----|     |   (Cladding)    |
    절연체(Dielectric)        |   (Pair 2)       |     |   외부 절연체(Jacket)|     |   (Buffer)      |
                             |   ...             |     +-------------------+     |   외부 보호재    |
                             +------------------+                                +---------------+
```
*(해설: 평행 2선식은 노출되어 있어 간섭에 취약하고, 동축 케이블은 쉴드(Shield)가 중심선을 감싸 보호하며, 광섬유는 전자기적 영향을 전혀 받지 않는 코어와 클래딩 구조임)*

#### C. 심층 동작 원리 (평행 2선식)
평행 2선식은 두 개의 도선이 평행하게 나란히 뻗어 있는 구조입니다.
1.  **신호 전달**: 한쪽 선은 전압 변화(신호)를, 다른 한쪽 선은 **그라운드(Ground, 0V)** 역할을 하며 회로를 형성합니다.
2.  **전자기장 유도**: 신호가 흐르는 도선 주변에는 자기장이 형성되는데, 두 선이 나란히 멀리 떨어져 있어 이 자기장이 서로 상쇄되지 못하고 주변의 금속이나 다른 선로에 **유도 기전력(Induced EMF)**을 일으켜 **노이즈(Noise)**로 작용합니다.
3.  **캐패시턴스(Capacitance) 효과**: 두 선이 마치 평판 콘덴서처럼 작용하여 고주파 신호일수록 선 사이의 누설 전류가 발생하여 신호가 뭉개지는 현상이 발생합니다.

#### D. 핵심 수식 및 코드
신호 감쇠(Attenuation)는 거리에 따라 신호가 약해지는 정도를 나타내며, **dB (Decibel)** 단위로 측정됩니다. 평행 2선식은 거리 제곱에 비례하여 신호가 급격히 약해지는 특성을 보입니다.

```python
# Python 유사 코드: 전송 매체별 신호 감쇠율 시뮬레이션
def calculate_signal_power(initial_power, distance, attenuation_factor):
    """
    initial_power: 송신 전력 (mW)
    distance: 전송 거리 (km)
    attenuation_factor: 매체별 감쇠 계수 (dB/km)
    """
    # 감쇠분(dB) = 거리 * 계수
    loss_db = distance * attenuation_factor
    
    # 전력비 = 10^(-loss_db / 10)
    remaining_power = initial_power * (10 ** (-loss_db / 10))
    return remaining_power

# 매체별 감쇠 계수 (예시)
twin_lead_loss = 15   # 평행 2선식 (고주파에서 매우 큰 손실)
utp_loss = 10         # UTP 케이블 (Cat5 기준)
coax_loss = 5         # 동축 케이블
fiber_loss = 0.2      # 광섬유 (매우 적은 손실)

# 결과: 평행 2선식은 짧은 거리에서도 신호가 급격히 떨어짐
```

### 2. 비유도 매체 (Unguided Media) 상세 분석
**무선 통신(Wireless Communication)**에 사용되며, 신호가 **전자기파(Electromagnetic Wave)** 형태로 **전리권(Ionosphere)**이나 대기를 통해 방사됩니다.

#### A. 주파수 대역별 특성
무선 매체는 **주파수(Frequency)** 대역에 따라 특성이 완전히 달라집니다.
*   **라디오파 (Radio Wave)**: **전방향성(Omnidirectional)** 특성을 가지며, 장애물을 잘 통과하지만 **다중 경로 페이딩(Multipath Fading)** 현상이 발생합니다. (예: FM 라디오, 와이파이)
*   **마이크로파 (Microwave)**: 고주파수로 직진성이 강하여 **지구 곡률** 때문에 중계기가 필요하며, 위성 통신이나 **LTE(Long Term Evolution)** 통신에 사용됩니다.
*   **적외선 (Infrared)**: 시야 확보(Line of Sight)가 필요하여 리모컨 등 단거리 통신에 쓰입니다.

### 3. 섹션 요약 비유
📢 **섹션 요약 비유**: **유도 매체는 '수도관'이고 비유도 매체는 '물총'입니다.** 유도 매체(수도관)는 물(데이터)이 새지 않고 목욕탕(수신자)로 정확히 배달되지만, 비유도 매체(물총)는 물이 허공에 흩어져 떨어지니까 밑에 컵(안테나)을 받쳐두어야만 물을 받을 수 있습니다. 평행 2선식은 구멍이 숭숭 뚫린 낡은 낡은 관이어서 물이 샌다고 보면 됩니다.

## Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

### 1. 기술적 비교 분석표
지난 기술과 현대 기술의 정량적, 정성적 비교를 통해 기술적 트레이드오프(Trade-off)를 분석합니다.

| 비교 항목 | 평행 2선식 (Twin-lead) | 현대 UTP (Unshielded Twisted Pair) | 광섬유 (Optical Fiber) |
|:---|:---|:---|:---|
| **노이즈 억제 원리** | 없음 (노출됨) | **비틀림(Twist)으로 상쇄** | 전기적 노이즈 불가능 |
| **전송 대역폭** | < 1 MHz (음성 수준) | 100 MHz ~ 1 GHz (Cat6e 기준) | 100 THz 이상 (초고광대역) |
| **Baud Rate** | 낮음 (Kbps 급) | 높음 (1Gbps~10Gbps) |极高 (Tbps~Pbps 급) |
| **거리 제한** | 수백 미터 이내 | 100m (Ethernet) | 수십 km (중계기 없이) |
| **보안성** | 취약 (도청 용이) | 취약 (탭핑 가능) | **우수함 (도청 감지 용이)** |
| **설치 비용** | 매우 낮음 | 저렴함 | **고가임** |

### 2. 타 과목 융합 관점

#### A. 물리 계층 (Layer 1) vs 데이터 링크 계층 (Layer 2)
*   **OSI 7계층 관점**: 전송 매체는 **Layer 1 (Physical Layer)**에 속하며 단순히 비트를 전송만 합니다. 하지만 매체의 품질이 나쁘면(예: 평행 2선식의 잡음), **Layer 2 (Data Link Layer)**에서 **오류 제어(Error Control)**를 위해 `ACK` 패킷을 재전송해야 하므로, 결과적으로 **처리량(Throughput)**이 급격히 저하됩니다.
*   **비유**: 도로(전송 매체)가 울퉁불퉁하면 운전자(Layer 2 프로토콜)가 계속 브레이크를 밟고 조금만 가도 다시 출발해야 하니 전체적인 이동 속도가 느려지는 것과 같습니다.

#### B. 신호 처리 (Signal Processing
