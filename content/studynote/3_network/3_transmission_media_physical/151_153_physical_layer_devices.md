# 151-153. 물리 계층(1계층) 통신 장비

> **핵심 인사이트**: 물리 계층(OSI 1계층) 장비들은 데이터의 의미나 주소(MAC, IP)를 전혀 이해하지 못하며, 오직 전기적 신호를 증폭하고 물리적인 연결 통로를 제공하는 역할만 수행한다.

---

## Ⅰ. 리피터 (Repeater, 중계기)
네트워크 케이블의 물리적인 전송 거리 한계(예: UTP 100m)를 극복하기 위해 중간에 설치하는 장비입니다.

* **동작 원리**: 케이블을 타고 오면서 감쇠(약해진)된 전기 신호를 수신한 뒤, 그 신호를 **원래의 세기로 증폭(Regenerate)**하여 다음 구간으로 다시 쏘아 보냅니다.
* **특징**: 단순히 신호의 형태만 복원할 뿐, 데이터의 내용을 검사하거나 필터링하는 기능은 전혀 없습니다. 1포트 입력, 1포트 출력의 단순한 구조입니다.

---

## Ⅱ. 허브 (Hub)
여러 대의 컴퓨터를 하나의 네트워크(LAN)로 묶어주는 집선 장비(Concentrator)입니다. 본질적으로는 **'포트가 여러 개인 멀티포트 리피터(Multi-port Repeater)'**와 같습니다.

### 1. 더미 허브 (Dummy Hub)
* **작동 방식 (Flooding)**: 한 포트에서 데이터 신호가 들어오면, 목적지가 누구인지 상관하지 않고 나머지 **모든 포트로 신호를 복사해서 무조건 뿌립니다(Broadcast/Flooding)**.
* **문제점**:
  - **대역폭 분할**: 100Mbps 허브에 10대의 PC가 물려있다면, 실제 속도는 $100 / 10 = 10Mbps$ 로 떨어집니다. (Shared Media)
  - **충돌(Collision) 증가**: 모든 PC가 하나의 묶인 공간(하나의 Collision Domain)에 있으므로, 동시에 데이터를 보내면 충돌이 발생합니다. CSMA/CD 알고리즘에 의존해야 합니다.

### 2. 스위칭 허브 (Switching Hub = L2 스위치)
* **발전**: 더미 허브의 충돌 문제를 해결하기 위해 등장했습니다.
* **특징**: 1계층이 아닌 **2계층(데이터 링크 계층)** 장비입니다. 각 포트에 연결된 PC의 MAC 주소를 기억해 두었다가, 데이터가 들어오면 **목적지 포트로만 정확하게 전달(Switching)**합니다. (대역폭이 분할되지 않고, 충돌이 발생하지 않음)

### 3. 인텔리전트 허브 (Intelligent Hub)
* 더미 허브나 스위치에 **네트워크 관리 기능(SNMP)**을 추가한 고급 장비입니다.
* 원격에서 관리자가 허브의 상태를 모니터링하고, 특정 포트를 차단하거나 제어할 수 있는 똑똑한(Intelligent) 장비입니다.

---

## Ⅲ. 트랜시버 (Transceiver)와 MAU
두 가지 다른 형태의 물리적 매체를 연결해 주는 변환기입니다.

### 1. 트랜시버 (Transceiver)
* **Trans**mitter(송신기) + Re**ceiver**(수신기)의 합성어입니다.
* 컴퓨터의 랜카드(NIC)에서 나오는 신호를 케이블이 전송할 수 있는 신호로 변환해 줍니다. 

### 2. MAU (Medium Attachment Unit)
* 과거 굵은 동축 케이블(Thicknet, 10BASE5)을 사용하던 시절, 케이블에 구멍을 뚫고 뱀파이어 탭(Vampire Tap)이라는 장치로 물리적으로 선을 연결해주던 장치(트랜시버의 일종)를 MAU라고 불렀습니다.
* 현대에는 랜카드(NIC) 칩셋 내부에 이 트랜시버 기능이 모두 통합되어 있어, 외부에서 따로 볼 수 없습니다. (다만 광통신 모듈인 SFP 트랜시버 형태로는 흔하게 사용됩니다.)

---

## Ⅳ. 개념 맵 및 요약

```ascii
[1계층 장비의 기능과 한계]

1. 리피터 (Repeater)
   [ PC A ] --(약해진 신호)--> [ 리피터 ] --(증폭된 신호)--> [ PC B ]
   (거리 연장용)

2. 더미 허브 (Dummy Hub)
               ┌──> [ PC B ] (나 아니네, 폐기)
   [ PC A ] ───┼──> [ PC C ] (내꺼다!, 수신)
   (목적지 1명) └──> [ PC D ] (나 아니네, 폐기)
   (무식하게 모두에게 소리침 -> 충돌 도메인 1개)

3. 트랜시버 (Transceiver)
   [ PC 랜카드 ] <--(전기/빛 신호 변환)--> [ 광케이블 / 동축케이블 ]
```

📢 **섹션 요약 비유**: **리피터**는 메아리가 잘 안 들리는 산 중턱에서 앞사람의 말을 듣고 뒷사람에게 똑같이 크게 소리쳐 주는(증폭) 사람입니다. **더미 허브**는 교실 한가운데 스피커를 놓고 한 명이 말하면 반 전체가 듣게 만드는 장치(모두에게 뿌림)입니다. 반면 **스위칭 허브**는 우체부 아저씨처럼 편지 봉투의 이름(MAC 주소)을 보고 정확히 그 친구 자리에만 편지를 배달해 줍니다.
