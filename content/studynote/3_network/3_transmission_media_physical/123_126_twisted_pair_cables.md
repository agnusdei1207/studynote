# 123-126. 꼬임 쌍선 케이블(Twisted Pair)과 UTP 카테고리

### # 꼬임 쌍선(Twisted Pair)과 UTP 카테고리

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 두 구리 선을 물리적으로 꼬아 유도 결합(Inductive Coupling)을 상쇄하는 수동적인 신호 무결성 유지 장치로, 전자기적 간섭(EMI) 및 누화(Crosstalk)를 억제하는 물리 계층(Physical Layer)의 핵심 매체이다.
> 2. **가치**: 차폐(Shielding) 유무와 꼬임 피치(Twist Pitch)의 정밀도에 따라 10Mbps부터 40Gbps까지 광범위한 대역폭을 제공하며, 비용 대비 성능(Benefit-Cost Ratio)이 우수하여 LAN 구축의 표준으로 자리 잡았다.
> 3. **융합**: 이더넷(Ethernet)의 물리적 전송로로서 MAC 계층의 CSMA/CD 또는 Full-Duplex 스위칭 기술과 직결되며, 고속화될수록 PHY 칩의 DSP(Digital Signal Processing) 기술과 복합적으로 발전하고 있다.

## Ⅰ. 개요 (Context & Background)

### 1. 개념 및 정의
**꼬임 쌍선 (Twisted Pair, TP)**은 두 개의 절연된 구리 도선을 일정한 간격으로 나선형으로 꼬아서 만든 평형 신호선(Balanced Line) 케이블을 의미한다. 구리선이라는 아날로그 매체의 물리적 한계를 극복하기 위해 고안된 가장 기초적이면서도 강력한 전송 매체로, 현대의 구조화된 배선 시스템의 기반이 된다.

### 2. 💡 비유
마치 시끄러운 장터에서 두 사람이 대화할 때, 서로 팔짱을 끼고 걷는 것과 같다. 팔짱을 꼬고 걸으면(QI회로 형성) 밀치는 사람이나 소음(노이즈)이 와도 두 사람이 같은 힘을 받아 넘어지지 않듯, 꼬임 쌍선도 두 선에 동일하게 들어오는 노이즈를 상쇄하여 깨끗한 신호를 전달한다.

### 3. 등장 배경 및 역사
- **① 기존 한계 (평행선)**: 초기 전신/전화에서 사용된 평행 구리선은 외부 전자기파에 의해 유도되는 노이즈를 제거할 수 없었고, 장거리 전송 시 신호 감쇠가 심했다.
- **② 혁신적 패러다임 (차동 신호와 꼬임)**: 19세기 말 알렉산더 그레이엄 벨 등에 의해 평형 회로(Balanced Circuit) 개념이 도입되면서, 두 선을 꼬아 상호 유도 작용을 이용해 간섭을 상쇄하는 기술이 개발되었다.
- **③ 현재의 비즈니스 요구**: 1Gbps, 10Gbps, 25Gbps 이상의 초고속 이더넷 환경에서도 광케이블(Fiber)의 경제성 문제로 인해 여전히 구리선 기반의 TP 케이블이 엔트리 레벨 전송 매체로 강력히 요구되고 있다.

### 4. ASCII 다이어그램: 간섭 상쇄 메커니즘

```ascii
   [전자기파 간섭(EMI)의 영향 비교]

      [A. 평행선 (Parallel Line)]          [B. 꼬임 쌍선 (Twisted Pair)]
      
         (선 1) -------- N1 +                 /-- (선 1) ---- (+) N1 +
         (선 2) -------- N2 +               /    (선 2) ---- (+) N2 +
                                          /    \
      > 문제점:                          /      \
        N1 ≠ N2 (노이즈 크기가 다름)     /        \
        수신부: Signal + (N1 - N2)      /          \
        신호 왜곡 발생!                 /            \
                                      /              \
      > 해결책:                       /                \
        N1 ≈ N2 (노이즈가 거의 같음)   /                  \
        수신부: Signal + (N1 - N2) ≈ 0                    \
        차동 증폭기(Differential Amp)로 노이즈 제거!
```
*도해 설명: A. 평행선은 노이즈원과의 거리 차이로 인해 두 선에 걸리는 전압이 다르다. B. 꼬임 쌍선은 두 선이 거의 같은 위치를 지나므로 노이즈 크기가 거의 동일하게 유도된다. 이를 '상태 모드 노이즈(Common Mode Noise)'라 하며, 수신부에서 두 선의 전압 차를 계산하여 신호만 복원한다.*

#### 📢 섹션 요약 비유
이는 복잡한 고속도로 톨게이트에서 하이패스 차선(고속 패스)을 별도로 운영하여 병목을 해결하는 것과 같습니다. 꼬임 구조는 물리적 차선 구분을 통해 정보(차량) 흐름의 충돌(간섭)을 원천적으로 차단합니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 1. 구성 요소 상세 분석
꼬임 쌍선 케이블은 단순한 구리선이 아니라 정밀하게 설계된 복합 시스템이다. 특히 UTP (Unshielded Twisted Pair)는 4쌍(8선)의 구조를 가지며, 각 요소가 성능에 결정적인 영향을 미친다.

| 구성 요소 (Module) | 역할 및 기능 | 내부 동작 및 파라미터 | 비고 |
|:---|:---|:---|:---|
| **Conductor (도체)** | 신호 전달 경로 | 순도 99.9% 이상의 구리(Cu) 혹은 알루미늄 사용. 굵기는 AWG(American Wire Gauge) 23~26 정도. | 굵을수록 저항이 낮아 장거리 전송 유리 |
| **Insulation (절연체)** | 선 간 절연 및 보호 | PE (Polyethylene) 또는 FEP 사용. 유전율(Dielectric Constant)을 낮춰 신호 속도 향상. | 색상 코드(Blue, Orange, Green, Brown)로 Pair 식별 |
| **Twist (꼬임)** | 노이즈 제거 핵심 | 1피트당 꼬임 횟수(Turn Rate)를 조절. 인접한 Pair끼리는 꼬임 피치(Pitch)를 다르게 하여 NEXT(Near End Crosstalk) 억제. | Cat6 이상은 꼬임 풀림 방지를 위해 흔적(Scaling) 남기기도 함 |
| **Cross-Filler (개재물)** | Pair 간 간섭 차단 | Cat 6 이상에서 채택. 케이블 중앙에 십자형 플라스틱을 넣어 4개의 Pair를 물리적으로 분리. | 구조적 안정성 제공 및 FEXT 감소 |
| **Drain Wire (배선)** | 접지 경로 확보 | 차폐형 케이블(FTP/STP)에서 금속 호일과 접촉하여 접지로 노이즈를 유도. | |
| **Sheath (외피)** | 기계적/화학적 보호 | PVC(일반) 혹은 Plenum(난연) 재질. CM/CMR/CMMP 등 등급으로 화기 안전성 구분. | |

### 2. 꼬임(Twist)의 심층 원리
꼬임의 핵심은 **유도 결합(Inductive Coupling)**의 상쇄에 있다.
- **작동 원리**: 외부 전자기장(Magnetic Field)이 케이블에 근접하면, 두 도체에는 거의 동일한 위상과 크기의 노이즈 전압이 유도된다. 수신 장치의 **차동 증폭기 (Differential Amplifier)**는 두 선의 전압차($V_+ - V_-$)만을 취하므로, 공통으로 유입된 노이즈는 소거되고 송신단에서 가한 차동 신호만 남게 된다.
- **꼬임 간격(Twist Pitch)의 차이**: 케이블 내부의 4쌍이 모두 똑같이 꼬여 있다면, 서로의 전자기장 영향력이 강해져 '꼬임 누화'가 발생한다. 따라서 각 Pair(녹색-흰색/녹색, 주황-흰색/주황 등)마다 **꼬임 길이(Period)**를 다르게 설계하여 특정 지점에서의 결합을 최소화한다.

### 3. ASCII 다이어그램: UTP Cat6 내부 구조

```ascii
       [UTP Cat 6 단면 구조 및 데이터 흐름]
       
            -----------------------------------
           |           Outer Jacket          |  <-- 보호 및 내연성
            -----------------------------------
           |    /|\      /|\      /|\      |    
           |   / | \    / | \    / | \     |
           |  /  |  \  /  |  \  /  |  \    | <-- Cross-Filler (십자 개재)
           | /   |   \/   |   \/   |   \   |     : Pair 간 간섭 차단
           |/    |   /\   |   /\   |    \  |
           |     |  /  \  |  /  \  |     \ |
           |  P3 | / P4 \ | / P1 \ |  P2  \| <-- P1, P2, P3, P4 (Twisted Pairs)
           |     |/      \|/      \|      |
            -----------------------------------
           
    [Pair별 Pinout (T568B 표준)]
    P1 (White/Blue, Blue)    : Pin 4, 5      (TX/RX - 100BaseTX/1000BaseT)
    P2 (White/Orange, Orange): Pin 1, 2      (- Data+)
    P3 (White/Green, Green)  : Pin 3, 6      (+ Data-)
    P4 (White/Brown, Brown)  : Pin 7, 8      ( Spare / Bi-directional)
```
*해설: Cat6 케이블의 핵심은 중앙의 십자형 다이아프램(Diaphragm)이다. 이것은 4개의 꼬임 쌍이 서로 엉키지 않고 일정한 거리를 유지하도록 물리적 격벽 역할을 하여, 고속 주파수(250MHz)에서 발생하는 신호 혼선을 막는다.*

### 4. 차폐(Shielding) 기술: UTP vs STP vs FTP
케이블의 전자기적 성숙도(EMC)를 결정하는 또 다른 축은 차폐층의 존재다.

```ascii
   [차폐 유형별 비교 구조도]

   1. UTP (Unshielded TP)              2. FTP (Foil TP)
   [   O   O   ] <--------------->      [  ( O   O )  ] <---------------> 
   [   O   O   ]  No Shield            [  ( O   O )  ]  Aluminum Foil
   [   O   O   ]                        [  ( O   O )  ]  (Overall Shield)
   저렴, 유연함                          내부 노이즈 차단
   가정/사무실 표준                       UTP보다 비쌈
   
   3. STP (Shielded TP)                 4. S/FTP (Screened Foil TP)
   [ [ O   O ] ] <--------------->      [ {( O   O )} ] <---------------> 
   [ [ O   O ] ]  Braid/Pair Shield    [ {( O   O )} ]  Foil + Braid
   [ [ O   O ] ]                        [ {( O   O )} ]  (Max Shielding)
   강력한 차단력                          Cat7/Cat8 표준
   접지(Ground) 중요                     고가/뻣뻣함
```
*해설: UTP는 꼬임만으로 노이즈를 잡지만, STP는 금속 호일(Foil)이나 그물망(Braid)이 각 쌍이나 전체를 감싸 추가적인 방패를 제공한다. 단, 차폐 케이블은 반드시 양 끝단(Connector Jack, Patch Panel)에서 접지(Grounding)가 완벽히 되어야 하며, 그렇지 않으면 안테나 역할을 하여 오히려 노이즈를 유입할 수 있다.*

#### 📢 섹션 요약 비유
UTP는 '여름용 통기성 있는 운동복'처럼 가볍고 시원하지만(설치 용이), 찬바람(노이즈)에 취약합니다. STP/FTP는 '재킷이나 패딩 점퍼'를 입은 것과 같아서 뻣뻣하고 비싸지만, 혹한의 강풍(EMI) 속에서도 체온(신호)을 지킵니다.
