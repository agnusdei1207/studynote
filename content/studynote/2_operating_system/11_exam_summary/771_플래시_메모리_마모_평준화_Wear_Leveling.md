# 플래시 메모리 마모 평준화 (Wear Leveling)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 쓰기 사이클이 제한된 플래시 셀의 수명을 균등하게 분배
> 2. **가치**: SSD 수명 연장, 신뢰성 향상, 내구성 보장
> 3. **융합**: FTL, 가비지 컬렉션, BAD 블록 관리와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

마모 평준화(Wear Leveling)는 플래시 메모리의 각 셀이 고르게磨损되도록 쓰기를 분산시키는 기술입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    플래시 메모리 마모 평준화                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              문제: 플래시 셀 수명 제한                          │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  NAND Flash 특성:                                    │   │ │
│   │   │                                                      │   │ │
│   │   │  SLC: ~100,000 P/E 사이클                            │   │ │
│   │   │  MLC: ~3,000 - 10,000 P/E 사이클                     │   │ │
│   │   │  TLC: ~500 - 1,000 P/E 사이클                        │   │ │
│   │   │  QLC: ~100 - 1,000 P/E 사이클                        │   │ │
│   │   │                                                      │   │ │
│   │   │  → 특정 셀에 집중된 쓰기 = 조기 고장                   │   │ │
│   │   │  → 웨어 레벨링으로 고르게 분배 필요                    │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              웨어 레벨링 종류                                  │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  Dynamic Wear Leveling:                              │   │ │
│   │   │  • 활성 블록 간 쓰기 분산                             │   │ │
│   │   │  • 구현 간단                                          │   │ │
│   │   │  • 정적 데이터는 이동 안 함                           │   │ │
│   │   │                                                      │   │ │
│   │   │  Static Wear Leveling:                               │   │ │
│   │   │  • 전체 블록(정적 포함) 분산                          │   │ │
│   │   │  • 정적 데이터도 주기적 이동                          │   │ │
│   │   │  • 더 균등하지만 오버헤드 큼                          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 웨어 레벨링 없이는 핫 데이터가 특정 셀만 빨리 마모시킵니다.

**💡 비유**: 웨어 레벨링은 타이어를 주기적 교체하여 고르게 닳게 하는 것과 같습니다.

**📢 섹션 요약 비유**: 쓰기를 고르게!

---

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **FTL** | 변환 | 논리→물리 | 지도 |
| **P/E Count** | 카운트 | 사이클 수 | 주행거리 |
| **Hot/Cold** | 분류 | 빈도 | 인기도 |
| **Reserved** | 예비 | 여유 블록 | 예비 |

**알고리즘 동작**

1. 쓰기 요청 → 논리 주소
2. P/E 카운트 확인
3. 가장 적은 블록 선택
4. 매핑 테이블 갱신

**핵심 알고리즘**: min(P/E) 블록 선택 → 쓰기 → 카운트 증가

**📢 섹션 요약 비유**: FTL이 마모 카운트 관리!

---

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

**기술 비교: 레벨링 방식**

| 방식 | 범위 | 효율 | 오버헤드 |
|:---|:---|:---:|:---:|
| **Dynamic** | 활성만 | 낮음 | 낮음 |
| **Static** | 전체 | 높음 | 높음 |
| **Global** | 전체 디바이스 | 최고 | 최고 |

**📢 섹션 요약 비유**: Static이 더 균등!

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

**시나리오 1: SSD** → Dynamic + Static 혼합
**시나리오 2: eMMC** → Simple Dynamic
**시나리오 3: Enterprise SSD** → Global Wear Leveling

**도입 체크리스트**: 예비 블록 비율, GC 정책

**📢 섹션 요약 비유**: Enterprise는 Global!

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

**정량/정성 기대효과**: SSD 수명 3-10배 연장

**미래 전망**: 3D NAND, QLC 최적화

**참고 표준**: NVM Express, ONFI

**📢 섹션 요약 비유**: 웨어 레벨링은 SSD의 수명 보험!
