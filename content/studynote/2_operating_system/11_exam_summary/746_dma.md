# 510. DMA (Direct Memory Access)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU 개입 없이 장치-메모리 간 직접 전송
> 2. **가치**: CPU 부하 감소와 고속 데이터 전송
> 3. **융합**: 인터럽트, 버스, 캐시와 연관

---

## Ⅰ. 개요

### 개념 정의

DMA(Direct Memory Access)는 **CPU의 개입 없이 I/O 장치와 메모리 간에 데이터를 직접 전송하는 기술**이다.

### 💡 비유: 비서에게 위임
DMA는 **비서에게 복사 업무를 맡기는 것**과 같다. 상사(CPU)는 다른 일을 하고, 비서(DMA)가 직접 복사한다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                DMA 상세                                               │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【DMA 전송 모드】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  모드              설명                    특징                             │ │
│  │  ────              ────                    ────                             │ │
│  │  버스트 모드        버스 독점 사용          빠름, CPU 대기                  │ │
│  │  사이클 스틸링      CPU와 교대로 사용        균형, CPU 느려짐                │ │
│  │  투명 모드         CPU 사용 안 함           느림, CPU 영향 없음             │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【DMA 컨트롤러 레지스터】                                               │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  레지스터           용도                                                   │ │
│  │  ────────           ────                                                   │ │
│  │  MAR (Memory Address)  메모리 주소                                         │ │
│  │  DAR (Device Address)  장치 주소/포트                                      │ │
│  │  BCR (Byte Count)      전송할 바이트 수                                     │ │
│  │  CSR (Control/Status)  제어 및 상태                                        │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【DMA 전송 단계】                                                        │**
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. CPU가 DMA 컨트롤러 초기화                                            │ │
│  │     - 소스 주소, 목적지 주소, 전송 크기                                   │ │
│  │     - 전송 방향, 모드 설정                                                │ │
│  │                                                             │ │
│  │  2. CPU가 I/O 장치에 명령                                                │ │
│  │     - 읽기/쓰기 시작                                                     │ │
│  │                                                             │ │
│  │  3. DMA 컨트롤러가 버스 제어권 획득                                       │ │
│  │     - BR (Bus Request) → BG (Bus Grant)                                  │ │
│  │                                                             │ │
│  │  4. DMA가 직접 전송 수행                                                  │ │
│  │     - 장치 ↔ 메모리                                                      │ │
│  │                                                             │ │
│  │  5. 전송 완료 시 인터럽트                                                 │ │
│  │     - CPU가 후처리                                                       │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【DMA 채널 확인】                                                      │
│  ──────────────────                                                  │
│  // ISA DMA 채널                                                       │
│  $ cat /proc/dma                                                     │
│  4: cascade                                                          │
│                                                                     │
│  // PCIe DMA 확인                                                      │
│  $ lspci -vv | grep -i dma                                           │
│                                                                     │
│  // IOMMU 상태                                                         │
│  $ dmesg | grep -i iommu                                             │
│  $ cat /sys/kernel/debug/iommu/dmar0/translation_structs            │
│                                                                     │
│  【Linux 커널 DMA API】                                                 │
│  ──────────────────                                                  │
│  #include <linux/dma-mapping.h>                                      │
│                                                                     │
│  // 일관성 있는 DMA 매핑 (Coherent)                                     │
│  void *dma_alloc_coherent(struct device *dev, size_t size,          │
│                           dma_addr_t *dma_handle, gfp_t flag);       │
│  void dma_free_coherent(struct device *dev, size_t size,            │
│                         void *cpu_addr, dma_addr_t dma_handle);      │
│                                                                     │
│  // 스트림 DMA 매핑 (Streaming)                                         │
│  dma_addr_t dma_map_single(struct device *dev, void *ptr,           │
│                           size_t size, enum dma_data_direction dir); │
│  void dma_unmap_single(struct device *dev, dma_addr_t dma_addr,     │
│                        size_t size, enum dma_data_direction dir);    │
│                                                                     │
│  // DMA 전송 방향                                                       │
│  enum dma_data_direction {                                           │
│      DMA_BIDIRECTIONAL = 0,  // 양방향                                │
│      DMA_TO_DEVICE = 1,       // 메모리 → 장치                         │
│      DMA_FROM_DEVICE = 2,     // 장치 → 메모리                         │
│      DMA_NONE = 3,            // 방향 없음                             │
│  };                                                                  │
│                                                                     │
│  【DMA 사용 예시】                                                      │
│  ──────────────────                                                  │
│  // 버퍼 할당                                                          │
│  dma_addr_t dma_handle;                                              │
│  void *virt_addr = dma_alloc_coherent(dev, BUF_SIZE,                 │
│                                        &dma_handle, GFP_KERNEL);     │
│                                                                     │
│  // 장치에 DMA 주소 전달                                                 │
│  writel(dma_handle, dev->mmio + DMA_SRC_ADDR);                       │
│  writel(BUF_SIZE, dev->mmio + DMA_COUNT);                            │
│  writel(DMA_START, dev->mmio + DMA_CTRL);                            │
│                                                                     │
│  // 전송 완료 대기                                                      │
│  wait_for_completion(&dev->done);                                    │
│                                                                     │
│  // 버퍼 해제                                                          │
│  dma_free_coherent(dev, BUF_SIZE, virt_addr, dma_handle);            │
│                                                                     │
│  【Scatter-Gather DMA】                                                │
│  ──────────────────                                                  │
│  #include <linux scatterlist.h>                                      │
│                                                                     │
│  // Scatter-Gather 리스트                                               │
│  struct scatterlist sg[2];                                           │
│  sg_init_table(sg, 2);                                               │
│  sg_set_buf(&sg[0], buf1, size1);                                    │
│  sg_set_buf(&sg[1], buf2, size2);                                    │
│                                                                     │
│  // DMA 매핑                                                           │
│  int nents = dma_map_sg(dev, sg, 2, DMA_TO_DEVICE);                  │
│                                                                     │
│  // 장치에 SG 리스트 전달                                                │
│  for (i = 0; i < nents; i++) {                                       │
│      writel(sg_dma_address(&sg[i]), dev->mmio + ADDR);               │
│      writel(sg_dma_len(&sg[i]), dev->mmio + LEN);                    │
│  }                                                                   │
│                                                                     │
│  // 해제                                                               │
│  dma_unmap_sg(dev, sg, 2, DMA_TO_DEVICE);                            │
│                                                                     │
│  【DMA 버그 디버깅】                                                     │
│  ──────────────────                                                  │
│  // DMA 디버그 활성화                                                   │
│  $ echo 1 > /sys/kernel/debug/dma_debug/enabled                     │
│  $ cat /sys/kernel/debug/dma_debug/dump                             │
│                                                                     │
│  // 캐시 문제 확인                                                      │
│  $ dmesg | grep -i "cache\|dma"                                     │
│                                                                     │
│  // DMA 오류 확인                                                      │
│  $ dmesg | grep -i "DMA error"                                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: CPU 개입 없이 장치-메모리 직접 전송
• 장점: CPU 부하 감소, 고속 전송
• 모드: 버스트, 사이클 스틸링, 투명
• 레지스터: MAR, DAR, BCR, CSR
• 전송: 설정 → 실행 → 인터럽트
• API: dma_alloc_coherent, dma_map_single
• SG DMA: 불연속 메모리 한 번에 전송
• IOMMU: 가상 주소 DMA 지원
```
