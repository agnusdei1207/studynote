# 자원 할당 그래프 사이클

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 교착 상태 탐지를 위해 프로세스와 자원 간의 할당/요청 관계를 그래프로 표현한 분석 도구
> 2. **가치**: 사이클 존재 여부로 교착 상태 판별, 단일/다중 인스턴스 환경 구분 필요
> 3. **융합**: 교착 상태 탐지, 은행원 알고리즘, 대기 그래프와 통합된 데드락 분석

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

자원 할당 그래프(Resource-Allocation Graph)는 교착 상태 탐지를 위해 프로세스와 자원 간의 할당/요청 관계를 그래프로 표현한 분석 도구입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    자원 할당 그래프 (Resource-Allocation Graph)       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              그래프 구성 요소                                   │ │
│   │                                                              │ │
│   │   정점(Vertex):                                               │ │
│   │   ┌─────────────┐      ┌─────────────┐                      │ │
│   │   │ 프로세스(P) │      │  자원(R)    │                      │ │
│   │   │   ● 원형    │      │  □ 사각형   │                      │ │
│   │   │  P1, P2...  │      │ R1, R2...   │                      │ │
│   │   └─────────────┘      └─────────────┘                      │ │
│   │                                                              │ │
│   │   간선(Edge):                                                 │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  요청 간선: P → R (프로세스가 자원 요청)              │   │ │
│   │   │  할당 간선: R → P (자원이 프로세스에 할당됨)          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              그래프 예시                                       │ │
│   │                                                              │ │
│   │        ┌───┐                          ┌───┐                 │ │
│   │        │ R1│ ←─────┐         ┌─────→ │ R2│                 │ │
│   │        └───┘       │         │       └───┘                 │ │
│   │          │         │         │         │                    │ │
│   │          ↓         │         │         ↓                    │ │
│   │        ┌───┐     ┌───┐     ┌───┐    ┌───┐                  │ │
│   │        │ P1│ ──→ │ P2│ ──→ │ P3│ ←── │ P4│                  │ │
│   │        └───┘     └───┘     └───┘    └───┘                  │ │
│   │                              │                               │ │
│   │                              ↓                               │ │
│   │                            ┌───┐                            │ │
│   │                            │ R3│                            │ │
│   │                            └───┘                            │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 자원 할당 그래프는 교착 상태를 시각적으로 분석하는 도구입니다.

**💡 비유**: 자원 할당 그래프는 대기표와 같습니다. 누가 무엇을 기다리는지 한눈에 보여요!

**📢 섹션 요약 비유**: 그래프로 교착 상태를 한눈에 봐요!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **프로세스 정점** | 자원 요청자 | 원형 노드 | 고객 |
| **자원 정점** | 자원 제공자 | 사각형 노드 | 서비스 |
| **요청 간선** | 요청 표시 | P → R 화살표 | 대기표 |
| **할당 간선** | 보유 표시 | R → P 화살표 | 이용증 |

**사이클과 교착 상태 관계**

| 환경 | 사이클 존재 | 교착 상태 |
|:---|:---:|:---:|
| **단일 인스턴스** | ⇔ 교착 상태 | 필연적 |
| **다중 인스턴스** | → 가능성 | 필요조건 |

**핵심 알고리즘**: 그래프 생성 → 사이클 탐지 → 단일/다중 구분 → 교착 판별

**📢 섹션 요약 비유**: 사이클이 있으면 의심해요!
