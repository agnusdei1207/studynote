# 225. 원자성 (Atomicity)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 나눌 수 없는 연산 단위
> 2. **가치**: 중단 없이 완전히 실행되거나 전혀 실행되지 않음
> 3. **융합**: 트랜잭션, 원자적 연산, CAS와 연관

---

## Ⅰ. 개요

### 개념 정의

원자성(Atomicity)은 **연산이 중간에 중단되지 않고, 완전히 실행되거나 전혀 실행되지 않는 all-or-nothing 속성**이다. 그리스어 아톰(나눌 수 없는)에서 유래했다.

### 💡 비유: 전구 스위치
원자성은 **전구 스위치**와 같다. 켜거나 끄는 동작이 한 번에 완료된다. 중간 상태가 없다.

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【C11 stdatomic.h】                                                  │
│  ──────────────────                                                  │
│  #include <stdatomic.h>                                             │
│                                                                     │
│  atomic_int counter = 0;                                            │
│                                                                     │
│  void increment() {                                                 │
│      atomic_fetch_add(&counter, 1);  // 원자적 증가                  │
│  }                                                                  │
│                                                                     │
│  int get_and_reset() {                                              │
│      return atomic_exchange(&counter, 0);  // 원자적 교환            │
│  }                                                                  │
│                                                                     │
│  【C++ std::atomic】                                                  │
│  ──────────────────                                                  │
│  #include <atomic>                                                   │
│                                                                     │
│  std::atomic<int> value{0};                                         │
│                                                                     │
│  void update() {                                                    │
│      value.fetch_add(1, std::memory_order_relaxed);                 │
│  }                                                                  │
│                                                                     │
│  bool try_update(int expected, int new_val) {                       │
│      return value.compare_exchange_weak(expected, new_val);         │
│  }                                                                  │
│                                                                     │
│  【GCC Built-in】                                                     │
│  ──────────────────                                                  │
│  int counter = 0;                                                   │
│                                                                     │
│  void increment() {                                                 │
│      __sync_fetch_and_add(&counter, 1);                             │
│  }                                                                  │
│                                                                     │
│  // Compare and Swap                                                │
│  bool cas(int* ptr, int old, int new_val) {                         │
│      return __sync_bool_compare_and_swap(ptr, old, new_val);        │
│  }                                                                  │
│                                                                     │
│  【Linux 커널】                                                        │
│  ──────────────────                                                  │
│  atomic_t counter = ATOMIC_INIT(0);                                 │
│                                                                     │
│  atomic_inc(&counter);        // 원자적 증가                          │
│  atomic_dec(&counter);        // 원자적 감소                          │
│  atomic_read(&counter);       // 읽기                                │
│  atomic_xchg(&counter, 0);    // 교환                                │
│                                                                     │
│  // 64-bit                                                          │
│  atomic64_t big_counter;                                            │
│  atomic64_inc(&big_counter);                                        │
│                                                                     │
│  【메모리 순서 옵션】                                                  │
│  ──────────────────                                                  │
│  memory_order_relaxed  // 순서 보장 없음, 원자성만                    │
│  memory_order_acquire  // 읽기 후 다른 읽기/쓰기가 오도록              │
│  memory_order_release  // 쓰기 전 다른 읽기/쓰기가 완료되도록          │
│  memory_order_acq_rel  // acquire + release                         │
│  memory_order_seq_cst  // 순차적 일관성 (기본)                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
