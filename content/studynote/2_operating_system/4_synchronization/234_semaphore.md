# 231. 세마포어 (Semaphore)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 카운팅 동기화 객체, n개 자원 관리
> 2. **가치**: 자원 수 제한, 생산자-소비자 패턴
> 3. **융합**: 뮤텍스, 조건 변수, 동기화와 연관

---

## Ⅰ. 개요

### 개념 정의

세마포어(Semaphore)는 **정수 카운터와 P(wait)/V(signal) 연산을 가진 동기화 객체**다. 카운터로 n개의 자원 접근을 제어한다. Dijkstra가 1965년 발명했다.

### 💡 비유: 주차장
세마포어는 **주차장**과 같다. 주차 공간 수만큼 차가 들어갈 수 있다. 꽉 차면 입구에서 대기한다. 차가 나가면 다음 차가 들어간다.

### 세마포어 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                세마포어 구조                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【세마포어 연산】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  P(wait): 카운터 감소, 0 이하면 대기                              │ │
│  │  V(signal): 카운터 증가, 대기 중인 스레드 깨움                    │ │
│  │                                                             │ │
│  │  P 연산:                                                         │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  if (counter > 0) {                                     │ │ │
│  │  │      counter--;                                          │ │ │
│  │  │      return;  // 진입 허용                                │ │ │
│  │  │  } else {                                               │ │ │
│  │  │      wait();  // 대기                                    │ │ │
│  │  │  }                                                      │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                             │ │
│  │  V 연산:                                                         │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  counter++;                                              │ │ │
│  │  │  if (waiters > 0) {                                      │ │ │
│  │  │      wakeup_one();                                       │ │ │
│  │  │  }                                                      │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【세마포어 종류】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  종류              초기값                용도                    │ │
│  │  ────                  ────                    ────                │ │
│  │  이진 세마포어      1                  상호 배제                  │ │
│  │  카운팅 세마포어    n (자원 수)          자원 풀 관리               │ │
│  │  0 초기화 세마포어  0                  이벤트 동기화               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【주차장 비유】                                                       │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  초기값: 5 (주차 공간 5개)                                       │ │
│  │                                                             │ │
│  │  차1 진입: P() → count=4 → 진입                                │ │
│  │  차2 진입: P() → count=3 → 진입                                │ │
│  │  차3 진입: P() → count=2 → 진입                                │ │
│  │  차4 진입: P() → count=1 → 진입                                │ │
│  │  차5 진입: P() → count=0 → 진입 (만차)                          │ │
│  │  차6 진입: P() → count=0 → 대기...                             │ │
│  │  차2 나감: V() → count=1 → 차6 깨어남 → 진입                     │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                세마포어 상세                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【세마포어 vs 뮤텍스】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  구분              세마포어              뮤텍스                  │ │
│  │  ────                  ────                    ────              │ │
│  │  카운터            정수 (음수 가능)      0 또는 1                 │ │
│  │  소유권            없음                있음 (획득한 스레드)        │ │
│  │  해제              누구나 가능          소유자만                  │ │
│  │  재귀              불가능               가능 (recursive mutex)    │ │
│  │  용도              자원 수 관리          상호 배제                  │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【사용 패턴】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 상호 배제 (초기값=1)                                          │ │
│  │     P(s);   // 임계 영역 진입                                     │ │
│  │     // 임계 영역                                                  │ │
│  │     V(s);   // 임계 영역 퇴출                                     │ │
│  │                                                             │ │
│  │  2. 이벤트 동기화 (초기값=0)                                       │ │
│  │     스레드 A: V(s);   // 이벤트 발생                               │ │
│  │     스레드 B: P(s);   // 이벤트 대기                               │ │
│  │                                                             │ │
│  │  3. 자원 풀 (초기값=n)                                            │ │
│  │     P(pool);   // 자원 획득                                       │ │
│  │     // 자원 사용                                                  │ │
│  │     V(pool);   // 자원 반납                                       │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【생산자-소비자】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  semaphore empty = N;    // 빈 버퍼 수                           │ │
│  │  semaphore full = 0;     // 찬 버퍼 수                           │ │
│  │  semaphore mutex = 1;    // 버퍼 접근 보호                       │ │
│  │                                                             │ │
│  │  생산자:                    소비자:                               │ │
│  │  P(empty);                 P(full);                             │ │
│  │  P(mutex);                 P(mutex);                             │ │
│  │  버퍼에 데이터 추가;         버퍼에서 데이터 꺼냄;                 │ │
│  │  V(mutex);                 V(mutex);                             │ │
│  │  V(full);                  V(empty);                             │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【POSIX 세마포어】                                                   │
│  ──────────────────                                                  │
│  #include <semaphore.h>                                              │
│                                                                     │
│  // 이름 없는 세마포어 (프로세스 내)                                   │
│  sem_t sem;                                                         │
│  sem_init(&sem, 0, 5);  // 초기값 5                                  │
│                                                                     │
│  sem_wait(&sem);      // P 연산                                      │
│  // 자원 사용                                                        │
│  sem_post(&sem);      // V 연산                                      │
│                                                                     │
│  sem_destroy(&sem);                                                 │
│                                                                     │
│  // 이름 있는 세마포어 (프로세스 간)                                   │
│  sem_t *sem = sem_open("/mysem", O_CREAT, 0644, 5);                 │
│  sem_wait(sem);                                                     │
│  sem_post(sem);                                                     │
│  sem_close(sem);                                                    │
│  sem_unlink("/mysem");                                              │
│                                                                     │
│  【System V 세마포어】                                                │
│  ──────────────────                                                  │
│  #include <sys/sem.h>                                                │
│                                                                     │
│  // 세마포어 집합 생성                                                 │
│  int semid = semget(IPC_PRIVATE, 1, IPC_CREAT | 0600);             │
│                                                                     │
│  // 초기화                                                           │
│  semctl(semid, 0, SETVAL, 5);                                       │
│                                                                     │
│  // P 연산                                                           │
│  struct sembuf op = {0, -1, 0};                                     │
│  semop(semid, &op, 1);                                              │
│                                                                     │
│  // V 연산                                                           │
│  op.sem_op = 1;                                                     │
│  semop(semid, &op, 1);                                              │
│                                                                     │
│  semctl(semid, 0, IPC_RMID);                                        │
│                                                                     │
│  【C++ 세마포어 (C++20)】                                             │
│  ──────────────────                                                  │
│  #include <semaphore>                                                │
│  std::counting_semaphore<10> sem(5);  // 최대 10, 초기 5              │
│                                                                     │
│  sem.acquire();  // P                                               │
│  // 자원 사용                                                        │
│  sem.release();  // V                                               │
│                                                                     │
│  // 이진 세마포어                                                     │
│  std::binary_semaphore bsem(1);                                     │
│                                                                     │
│  【자원 풀 예시】                                                      │
│  ──────────────────                                                  │
│  #define POOL_SIZE 5                                                │
│  sem_t pool_sem;                                                    │
│  Resource* pool[POOL_SIZE];                                         │
│                                                                     │
│  Resource* acquire_resource() {                                     │
│      sem_wait(&pool_sem);                                           │
│      // 사용 가능한 자원 반환                                         │
│      return get_free_resource();                                    │
│  }                                                                  │
│                                                                     │
│  void release_resource(Resource* r) {                               │
│      mark_free(r);                                                  │
│      sem_post(&pool_sem);                                           │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 카운팅 동기화 객체
• 연산: P(wait), V(signal)
• 종류: 이진, 카운팅, 0 초기화
• vs 뮤텍스: 소유권 없음, 누구나 해제
• 용도: 자원 풀, 생산자-소비자, 이벤트
• POSIX: sem_init, sem_wait, sem_post
```
