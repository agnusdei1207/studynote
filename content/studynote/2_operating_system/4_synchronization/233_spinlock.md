# 230. 스핀락 (Spinlock)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 락을 획득할 때까지 루프를 돌며 대기
> 2. **가치**: 짧은 대기에 컨텍스트 스위치 없이 빠른 획득
> 3. **융합**: 원자적 연산, 커널 동기화, 멀티코어와 연관

---

## Ⅰ. 개요

### 개념 정의

스핀락(Spinlock)은 **락을 획득할 수 있을 때까지 루프를 돌며(busy-wait) 대기하는 락**이다. 컨텍스트 스위치 오버헤드가 없어 짧은 임계 영역에 적합하다.

### 💡 비유: 줄서서 기다리기
스핀락은 **줄서서 계속 기다리기**와 같다. 자리를 뜨지 않고 계속 서서 순서를 기다린다. 자리가 나면 바로 들어간다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                스핀락 상세                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【구현 최적화】                                                       │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 단순 스핀                                                     │ │
│  │     while (lock == 1);  // 캐시 경합 심함                         │ │
│  │                                                             │ │
│  │  2. Read + Pause                                                 │ │
│  │     do {                                                         │
│  │         while (lock == 1) pause();  // x86 pause 명령            │ │
│  │     } while (test_and_set(&lock));                              │ │
│  │                                                             │ │
│  │  3. Exponential Backoff                                          │ │
│  │     delay = 1;                                                   │ │
│  │     while (try_lock()) {                                         │ │
│  │         for (i = 0; i < delay; i++) pause();                    │ │
│  │         delay = min(delay * 2, MAX);                            │ │
│  │     }                                                             │ │
│  │                                                             │ │
│  │  4. 큐 기반 (MCS, CLH)                                           │ │
│  │     각 CPU가 로컬 변수에서 대기                                   │ │
│  │     캐시 경합 감소                                                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【사용 규칙】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 절대 오래 잡고 있지 않기                                        │ │
│  │  • 슬립 가능한 코드 안에서 사용 금지                               │ │
│  │  • 재귀적으로 획득 금지                                            │ │
│  │  • 다른 스핀락을 중첩해서 획득 금지                                │ │
│  │  • 인터럽트 핸들러에서 주의                                        │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【x86 Pause 명령어】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 스핀 루프 최적화                                                │ │
│  │  • 전력 소모 감소                                                  │ │
│  │  • 메모리 순서 위반 처리                                           │ │
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux 커널 스핀락】                                                 │
│  ──────────────────                                                  │
│  #include <linux/spinlock.h>                                         │
│                                                                     │
│  spinlock_t my_lock;                                                │
│  spin_lock_init(&my_lock);                                          │
│                                                                     │
│  // 일반 사용                                                        │
│  void critical_section(void) {                                      │
│      spin_lock(&my_lock);                                           │
│      // 임계 영역 (짧아야 함)                                         │
│      spin_unlock(&my_lock);                                         │
│  }                                                                  │
│                                                                     │
│  // 인터럽트 컨텍스트에서                                              │
│  void irq_handler(void) {                                           │
│      unsigned long flags;                                           │
│      spin_lock_irqsave(&my_lock, flags);                            │
│      // 임계 영역                                                    │
│      spin_unlock_irqrestore(&my_lock, flags);                       │
│  }                                                                  │
│                                                                     │
│  【C++ std::atomic_flag】                                             │
│  ──────────────────                                                  │
│  #include <atomic>                                                   │
│  class Spinlock {                                                   │
│      std::atomic_flag flag = ATOMIC_FLAG_INIT;                      │
│  public:                                                            │
│      void lock() {                                                  │
│          while (flag.test_and_set(std::memory_order_acquire)) {    │
│              // 스핀                                                 │
│          }                                                          │
│      }                                                              │
│      void unlock() {                                                │
│          flag.clear(std::memory_order_release);                     │
│      }                                                              │
│  };                                                                 │
│                                                                     │
│  【pthread_spinlock_t】                                               │
│  ──────────────────                                                  │
│  pthread_spinlock_t spinlock;                                       │
│  pthread_spin_init(&spinlock, PTHREAD_PROCESS_PRIVATE);             │
│                                                                     │
│  pthread_spin_lock(&spinlock);                                      │
│  // 임계 영역                                                        │
│  pthread_spin_unlock(&spinlock);                                    │
│                                                                     │
│  pthread_spin_destroy(&spinlock);                                   │
│                                                                     │
│  【성능 측정】                                                         │
│  ──────────────────                                                  │
│  # 스핀락 경합 분석                                                   │
│  $ perf record -e lock:spin_lock ./program                         │
│  $ perf report                                                      │
│                                                                     │
│  # ftrace로 스핀락 추적                                               │
│  $ trace-cmd record -e lock:spin_lock ./program                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
