## Ⅰ. 개요

### 개념 정의

대체 시그널 스택(Alternate Signal Stack)은 **시그널 핸들러 실행을 위한 별도의 스택**이다. 메인 스택이 오버플로우되었을 때도 핸들러를 실행할 수 있다.

### 💡 비유: 비상 탈출구
대체 시그널 스택은 **비상 탈출구**와 같다. 본 출구(메인 스택)가 막혀도 다른 길로 나갈 수 있다.

### 대체 시그널 스택 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                대체 시그널 스택 구조                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【일반 스택 vs 대체 스택】                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  일반 스택                        대체 시그널 스택              │ │
│  │  ┌─────────────────┐            ┌─────────────────┐          │ │
│  │  │                 │            │                 │          │ │
│  │  │   ←── 스택 성장  │            │   ←── 스택 성장  │          │ │
│  │  │                 │            │                 │          │ │
│  │  │   지역 변수      │            │  핸들러 지역변수 │          │ │
│  │  │                 │            │                 │          │ │
│  │  │   호출 프레임    │            │  핸들러 프레임   │          │ │
│  │  │                 │            │                 │          │ │
│  │  └─────────────────┘            └─────────────────┘          │ │
│  │        ▲                              ▲                       │ │
│  │        │                              │                       │ │
│  │    메인 실행                        시그널                     │ │
│  │                                   핸들러                       │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【필요한 상황】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 스택 오버플로우 처리 (SIGSEGV)                                 │ │
│  │  • 제한된 스택 크기 환경                                          │ │
│  │  • 깊은 재귀 중 시그널                                            │ │
│  │  • siglongjmp 사용                                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【sigaltstack 구조체】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  typedef struct {                                               │ │
│  │      void  *ss_sp;      // 스택 메모리 시작                      │ │
│  │      size_t ss_size;    // 스택 크기                            │ │
│  │      int    ss_flags;   // SS_ONSTACK, SS_DISABLE               │ │
│  │  } stack_t;                                                     │ │
│  │                                                               │ │
│  │  // 최소 크기: SIGSTKSZ (보통 8KB)                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                대체 시그널 스택 상세                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【sigaltstack API】                                                   │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  #include <signal.h>                                            │ │
│  │                                                               │ │
│  │  int sigaltstack(const stack_t *ss, stack_t *oss);              │ │
│  │                                                               │ │
│  │  ss: 새 스택 설정 (NULL이면 변경 없음)                            │ │
│  │  oss: 이전 스택 정보 반환 (NULL이면 무시)                         │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【SA_ONSTACK 플래그】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  struct sigaction sa;                                           │ │
│  │  sa.sa_flags = SA_ONSTACK;  // 대체 스택 사용                    │ │
│  │  sigaction(SIGSEGV, &sa, NULL);                                 │ │
│  │                                                               │ │
│  │  • 대체 스택이 설정되어 있어야 작동                                │ │
│  │  • 없으면 기본 스택 사용                                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【스택 오버플로우 처리】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 대체 스택 설정                                                │ │
│  │  2. SIGSEGV 핸들러 등록 (SA_ONSTACK)                             │ │
│  │  3. 핸들러에서 siglongjmp로 복구                                  │ │
│  │                                                               │ │
│  │  주의:                                                           │ │
│  │  • 무한 루프 방지 (처리 중 다시 오버플로우)                         │ │
│  │  • sigsetjmp/siglongjmp 사용                                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 오버플로우 복구           • 추가 메모리                       │ │
│  │  • 안전한 핸들러 실행         • 복잡한 설정                       │ │
│  │  • 제한 환경에서 유용          • 신중한 사용 필요                  │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 설정】                                                           │
│  ──────────────────                                                  │
│  #include <signal.h>                                                │
│  #include <stdlib.h>                                                │
│  #include <unistd.h>                                                │
│                                                                     │
│  #define STACK_SIZE (SIGSTKSZ * 2)                                  │
│                                                                     │
│  char alt_stack[STACK_SIZE];                                        │
│                                                                     │
│  void setup_alt_stack() {                                           │
│      stack_t ss;                                                    │
│      ss.ss_sp = alt_stack;                                          │
│      ss.ss_size = STACK_SIZE;                                       │
│      ss.ss_flags = 0;                                               │
│      if (sigaltstack(&ss, NULL) < 0) {                              │
│          perror("sigaltstack");                                     │
│          exit(1);                                                   │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  【스택 오버플로우 처리 예시】                                           │
│  ──────────────────                                                  │
│  #include <setjmp.h>                                                │
│                                                                     │
│  sigjmp_buf jump_buffer;                                            │
│  volatile int handling_overflow = 0;                                │
│                                                                     │
│  void segv_handler(int sig) {                                       │
│      if (handling_overflow)                                         │
│          _exit(1);  // 무한 루프 방지                                 │
│      handling_overflow = 1;                                         │
│      write(1, "Stack overflow!\n", 16);                             │
│      siglongjmp(jump_buffer, 1);                                    │
│  }                                                                  │
│                                                                     │
│  void deep_recursion(int n) {                                       │
│      char buf[4096];                                                │
│      buf[0] = n;  // 사용                                            │
│      deep_recursion(n + 1);  // 오버플로우 발생 가능                   │
│  }                                                                  │
│                                                                     │
│  int main() {                                                        │
│      // 대체 스택 설정                                                │
│      setup_alt_stack();                                             │
│                                                                     │
│      // 핸들러 등록                                                   │
│      struct sigaction sa;                                           │
│      sa.sa_handler = segv_handler;                                  │
│      sigemptyset(&sa.sa_mask);                                      │
│      sa.sa_flags = SA_ONSTACK;  // 대체 스택 사용                     │
│      sigaction(SIGSEGV, &sa, NULL);                                 │
│                                                                     │
│      if (sigsetjmp(jump_buffer, 1) == 0) {                          │
│          deep_recursion(0);                                         │
│      } else {                                                        │
│          printf("복구됨!\n");                                         │
│      }                                                              │
│                                                                     │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
