# 159. 컨테이너 (Container)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네임스페이스와 cgroup으로 격리된 실행 환경
> 2. **가치**: 경량 가상화, 이식성, 일관성
> 3. **융합**: 네임스페이스, cgroup, 이미지와 연관

---

## Ⅰ. 개요

### 개념 정의

컨테이너(Container)는 **네임스페이스와 cgroup을 활용해 격리된 환경에서 실행되는 애플리케이션 패키지**다. VM보다 가볍고 빠르다.

### 💡 비유: 이삿짐 컨테이너
컨테이너는 **이삿짐 컨테이너**와 같다. 어디서든 같은 내용물(앱)을 꺼내 쓸 수 있다.

│  │  • 이미지 형식 표준 (OCI Image Spec)                            │ │

│  └───────────────────────────────────────────────────────────────┘ │

│  │  ────                        ────                                │ │
│  │  • 빠른 시작              • 완전한 격리 아님                      │ │
│  │  • 낮은 오버헤드            • 보안 이슈 가능                     │ │
│  │  • 이식성                  • 리눅스 의존                         │ │

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Docker 기본 명령어】                                                  │
│  ──────────────────                                                  │
│  # 이미지 빌드                                                        │
│  $ docker build -t myapp:1.0 .                                        │
│                                                                     │
│  # 컨테이너 실행                                                       │
│  $ docker run -d -p 8080:80 --name web nginx                         │
│                                                                     │
│  # 자원 제한 실행                                                      │
│  $ docker run -d --cpus=1 --memory=512m myapp                        │
│                                                                     │
│  # 컨테이너 목록                                                        │
│  $ docker ps -a                                                       │
│                                                                     │
│  # 컨테이너 중지/삭제                                                   │
│  $ docker stop web                                                    │
│  $ docker rm web                                                      │
│                                                                     │
│  【Dockerfile 예시】                                                     │
│  ──────────────────                                                  │
│  FROM python:3.9-slim                                                 │
│  WORKDIR /app                                                         │
│  COPY requirements.txt .                                              │
│  RUN pip install -r requirements.txt                                  │
│  COPY . .                                                             │
│  EXPOSE 8000                                                          │
│  CMD ["python", "app.py"]                                             │
│                                                                     │
│  【docker-compose.yml】                                                  │
│  ──────────────────                                                  │
│  version: '3'                                                         │
│  services:                                                            │
│    web:                                                               │
│      image: nginx                                                     │
│      ports:                                                           │
│        - "8080:80"                                                    │
│      networks:                                                        │
│        - frontend                                                     │
│    db:                                                                │
│      image: postgres                                                  │
│      volumes:                                                         │
│        - db-data:/var/lib/postgresql/data                             │
│      networks:                                                        │
│        - backend                                                      │
│  networks:                                                            │
│    frontend:                                                          │
│    backend:                                                           │
│  volumes:                                                             │
│    db-data:                                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
