# 149. PID 파일 (PID File)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데몬의 프로세스 ID를 저장하는 파일
> 2. **가치**: 중복 실행 방지, 프로세스 관리
> 3. **융합**: 데몬, 파일 락, 프로세스 관리와 연관

---

## Ⅰ. 개요

### 개념 정의

PID 파일(PID File)은 **데몬 프로세스의 PID를 저장하는 파일**이다. 보통 /var/run/ 디렉토리에 저장되며, 중복 실행 방지와 프로세스 관리에 사용된다.

### 💡 비유: 명찰
PID 파일은 **직원의 명찰**와 같다. 누가(어떤 PID가) 일하고 있는지 식별한다.

### PID 파일 구조

```

│  ──────────────────                                                  │

│  │  표준 위치:                                                      │ │
│  │  /var/run/daemon.pid                                            │ │
│  │  /run/daemon.pid                                                │ │
│  │  /var/run/daemon/daemon.pid                                     │ │
│  │                                                               │ │
│  │  예시:                                                           │ │
│  │  /var/run/sshd.pid       # sshd 데몬                            │ │

│  │  /var/run/crond.pid      # cron 데몬                            │ │
│  │                                                               │ │

│                                                                     │
│  【PID 파일 내용】                                                      │

│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                PID 파일 상세                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【PID 파일 생성 과정】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 파일 존재 확인                                                │ │
│  │     if (access("/var/run/mydaemon.pid", F_OK) == 0) {           │ │
│  │         // 이미 실행 중일 수 있음                                  │ │
│  │     }                                                           │ │
│  │                                                               │ │
│  │  2. 파일 락 획득 (중복 방지)                                        │ │
│  │     int fd = open("/var/run/mydaemon.pid",                      │ │
│  │                    O_RDWR | O_CREAT, 0644);                     │ │
│  │     flock(fd, LOCK_EX | LOCK_NB);                               │ │
│  │                                                               │ │
│  │  3. PID 기록                                                     │ │
│  │     char buf[16];                                               │ │
│  │     sprintf(buf, "%d\n", getpid());                             │ │
│  │     write(fd, buf, strlen(buf));                                │ │
│  │                                                               │ │
│  │  4. 종료 시 파일 삭제                                              │ │
│  │     unlink("/var/run/mydaemon.pid");                            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【Stale PID 파일 문제】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 데몬이 비정상 종료 시 PID 파일이 남음                            │ │
│  │  • PID가 재사용되어 다른 프로세스를 가리킬 수 있음                   │ │
│  │  • 해결:                                                         │ │
│  │    1. PID로 프로세스 존재 확인 (kill(pid, 0))                    │ │
│  │    2. /proc/[pid]/cmdline 등으로 검증                            │ │
│  │    3. 파일 락으로 동시 접근 제어                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【파일 락과 함께 사용】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • flock() 또는 fcntl()로 락 획득                                 │ │
│  │  • 락 획득 실패 = 이미 실행 중                                     │ │
│  │  • 프로세스 종료 시 락 자동 해제                                    │ │
│  │  • 가장 안전한 중복 실행 방지                                      │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 쉬운 관리               • Stale 파일 문제                     │ │
│  │  • 표준 관행               • 파일 시스템 의존                     │ │
│  │  • 스크립트 활용            • 권한 필요                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【PID 파일 생성 함수】                                                 │
│  ──────────────────                                                  │
│  #include <stdio.h>                                                 │
│  #include <stdlib.h>                                                │
│  #include <unistd.h>                                                │
│  #include <fcntl.h>                                                 │
│  #include <sys/file.h>                                              │
│  #include <errno.h>                                                 │
│                                                                     │
│  int create_pid_file(const char *path) {                            │
│      int fd = open(path, O_RDWR | O_CREAT, 0644);                   │
│      if (fd < 0) return -1;                                         │
│                                                                     │
│      // 락 획득 (비차단)                                               │
│      if (flock(fd, LOCK_EX | LOCK_NB) < 0) {                        │
│          close(fd);                                                 │
│          return -1;  // 이미 실행 중                                  │
│      }                                                              │
│                                                                     │
│      // 파일 비우고 PID 기록                                            │
│      ftruncate(fd, 0);                                              │
│      char buf[16];                                                  │
│      int len = snprintf(buf, sizeof(buf), "%d\n", getpid());        │
│      write(fd, buf, len);                                           │
│                                                                     │
│      // fd 유지 (락 유지)                                             │
│      return fd;                                                     │
│  }                                                                  │
│                                                                     │
│  【PID 확인으로 데몬 체크】                                              │
│  ──────────────────                                                  │
│  int is_daemon_running(const char *pidfile) {                       │
│      FILE *f = fopen(pidfile, "r");                                 │
│      if (!f) return 0;  // 파일 없음                                  │
│                                                                     │
│      pid_t pid;                                                     │
│      if (fscanf(f, "%d", &pid) != 1) {                              │
│          fclose(f);                                                 │
│          return 0;                                                  │
│      }                                                              │
│      fclose(f);                                                     │
│                                                                     │
│      // 프로세스 존재 확인                                              │
│      if (kill(pid, 0) == 0) {                                       │
│          return 1;  // 실행 중                                        │ │
│      }                                                              │
│      return 0;  // 실행 안 됨                                         │
│  }                                                                  │
│                                                                     │
│  【셸 스크립트에서 사용】                                                │
│  ──────────────────                                                  │
│  PIDFILE="/var/run/mydaemon.pid"                                    │
│                                                                     │
│  start() {                                                           │
│      if [ -f "$PIDFILE" ]; then                                     │
│          PID=$(cat "$PIDFILE")                                       │
│          if kill -0 "$PID" 2>/dev/null; then                        │
│              echo "이미 실행 중: $PID"                                  │
│              return 1                                                │
│          fi                                                          │
│      fi                                                              │
│      ./mydaemon                                                      │
│  }                                                                   │
│                                                                     │
│  stop() {                                                            │
│      if [ -f "$PIDFILE" ]; then                                     │
│          PID=$(cat "$PIDFILE")                                       │
│          kill "$PID" && rm -f "$PIDFILE"                            │
│      fi                                                              │
│  }                                                                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
