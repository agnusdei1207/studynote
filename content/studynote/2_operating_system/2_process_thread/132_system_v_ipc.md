# 132. System V IPC

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: UNIX System V에서 도입된 IPC 메커니즘 집합
> 2. **가치**: 공유 메모리, 세마포어, 메시지 큐
> 3. **융합**: POSIX IPC, 키, IPC 객체와 연관

---

## Ⅰ. 개요

### 개념 정의

System V IPC는 **UNIX System V에서 도입된 세 가지 IPC 메커니즘**이다: 공유 메모리, 세마포어, 메시지 큐. 키(key)로 식별되는 IPC 객체를 사용한다.

### 💡 비유: 공용 시설
System V IPC는 **공용 시설(도서관, 체육관, 게시판)**과 같다. 누구나 이용할 수 있지만, 시설 ID(키)를 알아야 한다.

### System V IPC 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                System V IPC 구조                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【System V IPC 메커니즘】                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  공유 메모리 (Shared Memory)                                    │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  shmget(), shmat(), shmdt(), shmctl()                    │ │ │
│  │  │  가장 빠른 IPC, 직접 메모리 접근                           │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                             │ │
│  │  세마포어 (Semaphore)                                           │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  semget(), semop(), semctl()                             │ │ │
│  │  │  동기화, 자원 관리                                        │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                             │ │
│  │  메시지 큐 (Message Queue)                                      │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  msgget(), msgsnd(), msgrcv(), msgctl()                  │ │ │
│  │  │  구조화된 메시지 전달                                      │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【IPC 키와 식별자】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  키 (key_t)              식별자 (ID)                          │ │
│  │  ────                        ────                                │ │
│  │  ftok()로 생성           shmget/msgget/semget 반환              │ │
│  │  IPC_PRIVATE             시스템 전역 고유 번호                    │ │
│  │                                                             │ │
│  │  key = ftok("/tmp/file", 'A');                                 │ │
│  │  shmid = shmget(key, size, flags);                            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【IPC 객체 관리】                                                       │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  명령어                      설명                               │ │
│  │  ────                        ────                                │ │
│  │  ipcs                       IPC 객체 목록                       │ │
│  │  ipcrm                      IPC 객체 제거                       │ │
│  │  ipcs -m                    공유 메모리                         │ │
│  │  ipcs -s                    세마포어                            │ │
│  │  ipcs -q                    메시지 큐                           │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

│  │  struct ipc_perm {                                              │ │

│  │  ────                        ────                                │ │
│  │  • 널리 사용됨             • 수동 삭제                          │ │
│  │  • 강력한 기능             • 복잡한 API                         │ │
│  │  • 커널 관리               • 키 충돌 가능                       │ │
│  │                                                               │ │

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【IPC 객체 확인】                                                      │
│  ──────────────────                                                  │
│  # 모든 IPC 객체                                                      │
│  $ ipcs -a                                                           │
│                                                                     │
│  # 공유 메모리                                                        │
│  $ ipcs -m                                                           │
│  $ ipcs -m -i 32768                                                  │
│                                                                     │
│  # 세마포어                                                           │
│  $ ipcs -s                                                           │
│                                                                     │
│  # 메시지 큐                                                          │
│  $ ipcs -q                                                           │
│                                                                     │
│  【IPC 객체 제거】                                                       │
│  ──────────────────                                                  │
│  # 공유 메모리 제거                                                    │
│  $ ipcrm -m 32768                                                    │
│                                                                     │
│  # 세마포어 제거                                                       │
│  $ ipcrm -s 65536                                                    │
│                                                                     │
│  # 메시지 큐 제거                                                      │
│  $ ipcrm -q 98304                                                    │
│                                                                     │
│  # 권한 모두 제거                                                      │
│  $ ipcrm -a                                                          │
│                                                                     │
│  【키 생성】                                                             │
│  ──────────────────                                                  │
│  # ftok 대신 직접 키 사용                                              │
│  key_t key = 0x12345678;                                             │
│                                                                     │
│  # IPC_PRIVATE (부모-자식 간만)                                        │
│  int shmid = shmget(IPC_PRIVATE, size, 0666);                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
