# 152. 시그널 핸들러 (Signal Handler)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시그널 수신 시 실행되는 사용자 정의 함수
> 2. **가치**: 비동기 이벤트 처리, 사용자 정의 동작
> 3. **융합**: 시그널, sigaction, 비동기 안전성과 연관

---

## Ⅰ. 개요

### 개념 정의

시그널 핸들러(Signal Handler)는 **시그널이 수신될 때 실행되는 사용자 정의 함수**다. 기본 동작 대신 원하는 동작을 수행할 수 있다.

### 💡 비유: 비상 대피 훈련 담당
시그널 핸들러는 **비상 대피 훈련 담당자**와 같다. 비상벨(시그널)이 울리면 미리 정한 절차를 실행한다.

### 시그널 핸들러 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                시그널 핸들러 구조                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【핸들러 등록 과정】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 핸들러 함수 정의                                              │ │
│  │     void handler(int sig) { ... }                               │ │
│  │                                                             │ │
│  │  2. sigaction 구조체 설정                                        │ │
│  │     struct sigaction sa;                                        │ │
│  │     sa.sa_handler = handler;                                    │ │
│  │     sigemptyset(&sa.sa_mask);                                   │ │
│  │     sa.sa_flags = 0;                                            │ │
│  │                                                             │ │
│  │  3. 핸들러 등록                                                   │ │
│  │     sigaction(SIGINT, &sa, NULL);                               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【핸들러 종류】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  설정                      동작                                │ │
│  │  ────                        ────                                │ │
│  │  SIG_DFL                 기본 동작                              │ │
│  │  SIG_IGN                 무시                                   │ │
│  │  handler 함수            사용자 정의 동작                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【핸들러 실행 흐름】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  정상 실행                                                       │ │
│  │       │                                                          │ │
│  │       ▼                                                          │ │
│  │  시그널 도착 ────► 실행 중단                                       │ │
│  │       │                                                          │ │
│  │       ▼                                                          │ │
│  │  핸들러 실행 (다른 시그널 블록)                                     │ │
│  │       │                                                          │ │
│  │       ▼                                                          │ │
│  │  핸들러 반환 ────► 원래 위치로 복귀                                 │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                시그널 핸들러 상세                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【비동기 안전 함수 (Async-Signal-Safe)】                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  핸들러에서 안전하게 호출 가능한 함수:                              │ │
│  │  • write(), read(), _exit()                                    │ │
│  │  • sigprocmask(), sigaction()                                  │ │
│  │  • kill(), getpid(), getppid()                                 │ │
│  │  • signal-safe 목록 확인 필요                                    │ │
│  │                                                               │ │
│  │  위험한 함수 (사용 금지):                                          │ │
│  │  • printf(), malloc(), free()                                  │ │
│  │  • lock 관련 함수                                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【sigaction 플래그】                                                   │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  플래그                    효과                                │ │
│  │  ────                        ────                                │ │
│  │  SA_SIGINFO             3인자 핸들러 사용                        │ │
│  │  SA_RESTART             시스템 콜 자동 재시작                     │ │
│  │  SA_NODEFER             핸들러 실행 중 같은 시그널 허용            │ │
│  │  SA_RESETHAND           핸들러 실행 후 기본값으로 리셋             │ │
│  │  SA_NOCLDSTOP           자식 STOP 시 SIGCHLD 안 보냄             │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【3인자 핸들러】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  void handler(int sig, siginfo_t *info, void *ucontext) {       │ │
│  │      // sig: 시그널 번호                                         │ │
│  │      // info: 추가 정보                                          │ │
│  │      //   info->si_pid: 송신자 PID                               │ │
│  │      //   info->si_uid: 송신자 UID                               │ │
│  │      //   info->si_value: sigqueue로 전달된 데이터                │ │
│  │      // ucontext: 컨텍스트 정보                                  │ │
│  │  }                                                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 유연한 이벤트 처리         • 비동기 안전 제약                   │ │
│  │  • 사용자 정의 동작          • 디버깅 어려움                      │ │
│  │  • 자원 정리 가능            • 재진입 문제                       │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 핸들러 등록】                                                    │
│  ──────────────────                                                  │
│  #include <signal.h>                                                │
│  #include <unistd.h>                                                │
│                                                                     │
│  volatile sig_atomic_t stop = 0;                                    │
│                                                                     │
│  void handler(int sig) {                                            │
│      if (sig == SIGINT)                                             │
│          stop = 1;  // 비동기 안전: 단순 플래그만                       │
│  }                                                                  │
│                                                                     │
│  int main() {                                                        │
│      struct sigaction sa;                                           │
│      sa.sa_handler = handler;                                       │
│      sigemptyset(&sa.sa_mask);                                      │
│      sa.sa_flags = SA_RESTART;                                      │
│      sigaction(SIGINT, &sa, NULL);                                  │
│                                                                     │
│      while (!stop) {                                                │
│          // 메인 루프                                                │
│      }                                                              │
│      write(1, "종료\n", 4);                                          │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
│  【고급 핸들러 (siginfo_t)】                                             │
│  ──────────────────                                                  │
│  void advanced_handler(int sig, siginfo_t *info, void *ctx) {       │
│      char buf[64];                                                  │
│      int len = snprintf(buf, sizeof(buf),                           │
│                         "Signal %d from PID %d\n",                   │
│                         sig, info->si_pid);                         │
│      write(STDOUT_FILENO, buf, len);                                │
│  }                                                                  │
│                                                                     │
│  struct sigaction sa;                                               │
│  sa.sa_sigaction = advanced_handler;                                │
│  sigemptyset(&sa.sa_mask);                                          │
│  sa.sa_flags = SA_SIGINFO;                                          │
│  sigaction(SIGUSR1, &sa, NULL);                                     │
│                                                                     │
│  【자식 프로세스 정리】                                                  │
│  ──────────────────                                                  │
│  void child_handler(int sig) {                                      │
│      int status;                                                    │
│      pid_t pid;                                                     │
│      while ((pid = waitpid(-1, &status, WNOHANG)) > 0) {            │
│          // 자식 정리                                                 │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  struct sigaction sa;                                               │
│  sa.sa_handler = child_handler;                                     │
│  sigemptyset(&sa.sa_mask);                                          │
│  sa.sa_flags = SA_RESTART | SA_NOCLDSTOP;                           │
│  sigaction(SIGCHLD, &sa, NULL);                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 시그널 수신 시 실행될 사용자 함수
• 등록: sigaction 구조체
• 종류: SIG_DFL, SIG_IGN, 사용자 함수
• 제약: 비동기 안전 함수만 사용
• 플래그: SA_SIGINFO, SA_RESTART 등
• 변수: volatile sig_atomic_t 사용
```
