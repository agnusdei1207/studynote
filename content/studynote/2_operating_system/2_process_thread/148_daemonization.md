# 148. 데몬화 (Daemonization)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 일반 프로세스를 데몬으로 변환하는 과정
> 2. **가치**: 백그라운드 서비스 실행
> 3. **융합**: fork, setsid, 리다이렉션과 연관

---

## Ⅰ. 개요

### 개념 정의

데몬화(Daemonization)는 **일반 프로세스를 데몬으로 변환하는 과정**이다. 터미널에서 분리하고 백그라운드에서 실행되도록 한다.

### 💡 비유: 은둔 생활
데몬화는 **사람이 은둔 생활을 시작하는 것**과 같다. 사회(터미널)에서 떨어져 혼자 일한다.

### 데몬화 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                데몬화 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【데몬화 과정】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  시작                                                          │ │
│  │    │                                                            │ │
│  │    ▼                                                            │ │
│  │  1. fork() ─────────────────────────────────────┐              │ │
│  │    │                                             │              │ │
│  │    ├──────► 부모 exit(0) ──► 셸로 복귀            │              │ │
│  │    │                                             │              │ │
│  │    ▼                                             │              │ │
│  │  2. setsid() ──► 새 세션 생성                     │              │ │
│  │    │             (제어 터미널 없음)               │              │ │
│  │    ▼                                             │              │ │
│  │  3. (선택) fork() + 부모 종료                     │              │ │
│  │    │             (세션 리더 방지)                 │              │ │
│  │    ▼                                             │              │ │
│  │  4. chdir("/") ──► 작업 디렉토리 변경             │              │ │
│  │    │                                             │              │ │
│  │    ▼                                             │              │ │
│  │  5. umask(0) ──► 권한 마스크 초기화               │              │ │
│  │    │                                             │              │ │
│  │    ▼                                             │              │ │
│  │  6. close(0,1,2) ──► 표준 입출력 닫기             │              │ │
│  │    │             /dev/null 리다이렉트             │              │ │
│  │    │                                             │              │ │
│  │    ▼                                             │              │ │
│  │  데몬 실행                                       │              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【각 단계의 목적】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  단계                      목적                                │ │
│  │  ────                        ────                                │ │
│  │  fork()                 백그라운드 전환                         │ │
│  │  setsid()               제어 터미널 분리                        │ │
│  │  2차 fork()             터미널 재획득 방지                       │ │
│  │  chdir("/")             파일 시스템 언마운트 방지                 │ │
│  │  umask(0)               파일 생성 권한 제어                     │ │
│  │  close()                입출력 차단                             │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                데몬화 상세                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【setsid() 세부 사항】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  성공 시:                                                        │ │
│  │  • 새 세션 생성                                                  │ │
│  │  • 새 프로세스 그룹 생성                                          │ │
│  │  • 호출자가 세션 리더이자 그룹 리더                                │ │
│  │  • 제어 터미널 없음                                               │ │
│  │                                                               │ │
│  │  실패 시:                                                        │ │
│  │  • 호출자가 이미 프로세스 그룹 리더                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【표준 입출력 처리】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  방법 1: 모두 닫기                                               │ │
│  │  close(STDIN_FILENO);                                           │ │
│  │  close(STDOUT_FILENO);                                          │ │
│  │  close(STDERR_FILENO);                                          │ │
│  │                                                               │ │
│  │  방법 2: /dev/null로 리다이렉트                                   │ │
│  │  int fd = open("/dev/null", O_RDWR);                            │ │
│  │  dup2(fd, STDIN_FILENO);                                        │ │
│  │  dup2(fd, STDOUT_FILENO);                                       │ │
│  │  dup2(fd, STDERR_FILENO);                                       │ │
│  │  close(fd);                                                     │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【현대적 데몬화】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • systemd의 Type=notify                                        │ │
│  │  • sd_notify()로 준비 알림                                       │ │
│  │  • systemd가 데몬화 처리                                         │ │
│  │  • 더 간단한 코드                                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【주의사항】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • PID 파일 생성 (중복 실행 방지)                                 │ │
│  │  • 시그널 핸들러 설정                                             │ │
│  │  • 로그 파일 또는 syslog 사용                                     │ │
│  │  • 권한 설정 (root 필요 시)                                       │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【완전한 데몬화 함수】                                                  │
│  ──────────────────                                                  │
│  #include <unistd.h>                                                │
│  #include <stdlib.h>                                                │
│  #include <sys/stat.h>                                              │
│  #include <fcntl.h>                                                 │
│  #include <syslog.h>                                                │
│                                                                     │
│  int become_daemon() {                                              │
│      int fd;                                                        │
│                                                                     │
│      // 1차 fork                                                     │
│      switch (fork()) {                                              │
│          case -1: return -1;                                        │
│          case 0: break;      // 자식 계속                             │
│          default: _exit(0);  // 부모 종료                             │
│      }                                                              │
│                                                                     │
│      // 새 세션                                                      │
│      if (setsid() == -1) return -1;                                 │
│                                                                     │
│      // 2차 fork (선택)                                               │
│      switch (fork()) {                                              │
│          case -1: return -1;                                        │
│          case 0: break;                                             │
│          default: _exit(0);                                         │
│      }                                                              │
│                                                                     │
│      // 작업 디렉토리                                                  │
│      if (chdir("/") == -1) return -1;                               │
│                                                                     │
│      // umask                                                       │
│      umask(0);                                                      │
│                                                                     │
│      // 표준 입출력 닫기                                               │
│      for (fd = 0; fd < 3; fd++)                                     │
│          close(fd);                                                 │
│                                                                     │
│      // /dev/null 리다이렉트                                          │
│      fd = open("/dev/null", O_RDWR);                                │
│      if (fd != STDIN_FILENO) return -1;                             │
│      if (dup2(fd, STDOUT_FILENO) != STDOUT_FILENO) return -1;       │
│      if (dup2(fd, STDERR_FILENO) != STDERR_FILENO) return -1;       │
│                                                                     │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
│  【BSD daemon() 함수】                                                  │
│  ──────────────────                                                  │
│  // BSD 시스템에서 제공                                                │
│  int daemon(int nochdir, int noclose);                              │
│                                                                     │
│  // 사용 예                                                          │
│  if (daemon(0, 0) == -1) {                                          │
│      perror("daemon");                                              │
│      exit(1);                                                       │
│  }                                                                  │
│                                                                     │
│  【systemd 서비스에서】                                                 │
│  ──────────────────                                                  │
│  # Type=simple면 데몬화 불필요                                        │
│  [Service]                                                           │
│  Type=simple                                                         │
│  ExecStart=/usr/bin/myapp                                            │
│  # systemd가 백그라운드 실행 담당                                      │
│                                                                     │
│  # Type=notify면 sd_notify 사용                                      │
│  sd_notify(0, "READY=1");                                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
