# 140. 고루틴 (Goroutine)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: Go 언어의 경량 사용자 수준 스레드
> 2. **가치**: 낮은 생성 비용, 효율적인 동시성
> 3. **융합**: 채널, 스케줄러, M:N 스레딩과 연관

---

## Ⅰ. 개요

### 개념 정의

고루틴(Goroutine)은 **Go 언어에서 제공하는 경량 사용자 수준 스레드**다. Go 런타임이 관리하며, 적은 메모리로 수백만 개의 고루틴을 실행할 수 있다.

### 💡 비유: 작은 일꾼들
고루틴은 **작고 가벼운 일꾼들**과 같다. 수천 명을 고용해도(생성) 비용이 거의 안 든다.

### 고루틴 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                고루틴 구조                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【고루틴 vs 스레드】                                                   │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  OS 스레드                고루틴                              │ │
│  │  ┌───────────────────┐  ┌───────────────────┐               │ │
│  │  │ 스택: 1-8 MB       │  │ 스택: 2 KB (동적)  │               │ │
│  │  │ 생성: ~1 ms        │  │ 생성: ~0.5 µs     │               │ │
│  │  │ 커널 관리          │  │ Go 런타임 관리     │               │ │
│  │  │ 전환: ~1 µs        │  │ 전환: ~0.2 µs     │               │ │
│  │  └───────────────────┘  └───────────────────┘               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【Go 스케줄러 (GMP 모델)】                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  G (Goroutine)          M (Machine)         P (Processor)    │ │
│  │  ┌───┐┌───┐┌───┐       ┌─────────┐        ┌───────────┐     │ │
│  │  │ G ││ G ││ G │──────►│  M (OS  │◄──────►│ P (Context│     │ │
│  │  └───┘└───┘└───┘       │ Thread) │        │   Queue)  │     │ │
│  │                        └─────────┘        └───────────┘     │ │
│  │                                                             │ │
│  │  G: 고루틴 (실행 단위)                                         │ │
│  │  M: OS 스레드 (실행 기계)                                      │ │
│  │  P: 프로세서 (실행 컨텍스트, 로컬 큐)                            │ │
│  │                                                             │ │
│  │  기본 P 개수 = GOMAXPROCS (= CPU 코어 수)                      │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【스케줄링 과정】                                                       │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. P의 로컬 큐에서 G 가져오기                                   │ │
│  │  2. 로컬 큐 비면 글로벌 큐에서 가져오기                          │ │
│  │  3. 다른 P에서 작업 훔치기 (Work Stealing)                       │ │
│  │  4. G가 블록되면 M이 다른 G 실행                                 │ │
│  │  5. 시스템 콜 시 M이 블록되면 새 M 생성                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

```
│                                                                     │
│  【선점형 스케줄링】                                                     │
│  ──────────────────                                                  │

│  │  • Go 1.14+ 협력적 → 선점형 전환                                  │ │

│                                                                     │

│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • CSP (Communicating Sequential Processes) 기반                │ │
│  │  • 채널로 메시지 전달                                             │ │
│  │  • "Do not communicate by sharing memory;                      │ │
│  │    share memory by communicating."                             │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                           │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 매우 경량               • Go 언어 전용                        │ │
│  │  • 간단한 문법             • CGO 호출 시 스레드 차지             │ │
│  │  • 효융적 스케줄링          • 데이터 경쟁 가능                   │ │
│  │  • 수백만 개 실행 가능                                         │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【고루틴 생성】                                                        │
│  ──────────────────                                                  │
│  package main                                                        │
│                                                                     │
│  import (                                                            │
│      "fmt"                                                           │
│      "time"                                                          │
│  )                                                                   │
│                                                                     │
│  func sayHello(id int) {                                            │
│      fmt.Printf("Hello from %d\n", id)                              │
│  }                                                                  │
│                                                                     │
│  func main() {                                                       │
│      // go 키워드로 고루틴 생성                                       │
│      go sayHello(1)                                                 │
│      go func(id int) {                                              │
│          fmt.Printf("Hello from %d\n", id)                          │
│      }(2)                                                           │
│                                                                     │
│      time.Sleep(time.Second)  // 대기                                │
│  }                                                                  │
│                                                                     │
│  【채널 통신】                                                           │
│  ──────────────────                                                  │
│  func main() {                                                       │
│      ch := make(chan int)                                           │
│                                                                     │
│      // 생산자                                                        │
│      go func() {                                                    │
│          ch <- 42  // 송신                                           │
│      }()                                                             │
│                                                                     │
│      // 소비자                                                        │
│      value := <-ch  // 수신                                          │
│      fmt.Println(value)                                             │
│  }                                                                  │
│                                                                     │
│  【버퍼 채널과 Select】                                                  │
│  ──────────────────                                                  │
│  func main() {                                                       │
│      ch1 := make(chan string, 2)  // 버퍼 크기 2                     │
│      ch2 := make(chan int)                                          │
│                                                                     │
│      ch1 <- "hello"  // 블록 없음                                    │
│      ch1 <- "world"  // 블록 없음                                    │
│                                                                     │
│      select {                                                        │
│      case msg := <-ch1:                                             │
│          fmt.Println(msg)                                           │
│      case val := <-ch2:                                             │
│          fmt.Println(val)                                           │
│      default:                                                        │
│          fmt.Println("no message")                                  │
│      }                                                               │
│  }                                                                  │
│                                                                     │
│  【동시성 패턴】                                                         │
│  ──────────────────                                                  │
│  // Worker Pool                                                     │
│  func worker(id int, jobs <-chan int, results chan<- int) {         │
│      for j := range jobs {                                          │
│          results <- j * 2                                           │
│      }                                                               │
│  }                                                                  │
│                                                                     │
│  // WaitGroup                                                       │
│  var wg sync.WaitGroup                                              │
│  for i := 0; i < 5; i++ {                                           │
│      wg.Add(1)                                                      │
│      go func() {                                                    │
│          defer wg.Done()                                            │
│          // 작업                                                     │
│      }()                                                             │
│  }                                                                   │
│  wg.Wait()                                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
