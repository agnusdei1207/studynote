# 130. 시그널 (Signal)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스에 비동기 이벤트를 통지하는 소프트웨어 인터럽트
> 2. **가치**: 예외 처리, IPC, 프로세스 제어
> 3. **융합**: 인터럽트, 핸들러, kill 명령과 연관

---

## Ⅰ. 개요

### 개념 정의

시그널(Signal)은 **프로세스에게 비동기적으로 이벤트를 통지하는 소프트웨어 인터럽트**다. 커널이나 다른 프로세스가 특정 프로세스에게 이벤트를 전달한다.

### 💡 비유: 비상벨
시그널은 **비상벨**과 같다. 언제 울릴지 모르지만, 울리면 즉시 현재 일을 멈추고 대응해야 한다.

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【시그널 핸들러 설정】                                                  │
│  ──────────────────                                                  │
│  #include <signal.h>                                                │
│                                                                     │
│  void handler(int sig) {                                            │
│      printf("시그널 %d 수신\n", sig);                                  │
│  }                                                                  │
│                                                                     │
│  // signal() 사용                                                    │
│  signal(SIGINT, handler);                                           │
│                                                                     │
│  // sigaction() 사용 (권장)                                          │
│  struct sigaction sa;                                               │
│  sa.sa_handler = handler;                                           │
│  sigemptyset(&sa.sa_mask);                                          │
│  sa.sa_flags = 0;                                                   │
│  sigaction(SIGINT, &sa, NULL);                                      │
│                                                                     │
│  【시그널 전송】                                                         │
│  ──────────────────                                                  │
│  // 쉘에서                                                           │
│  $ kill -9 1234       # SIGKILL                                     │
│  $ kill -15 1234      # SIGTERM                                     │
│  $ kill -l            # 시그널 목록                                  │
│                                                                     │
│  // C에서                                                            │
│  kill(1234, SIGTERM);                                               │
│  raise(SIGINT);        // 자신에게                                   │
│                                                                     │
│  【시그널 마스크】                                                       │
│  ──────────────────                                                  │
│  sigset_t set;                                                      │
│  sigemptyset(&set);                                                 │
│  sigaddset(&set, SIGINT);                                           │
│  sigprocmask(SIG_BLOCK, &set, NULL);                                │
│  // SIGINT 블록                                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
