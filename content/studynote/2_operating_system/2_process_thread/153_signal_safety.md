# 153. 시그널 안전성 (Signal Safety)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시그널 핸들러에서 안전하게 호출 가능한 함수
> 2. **가치**: 경쟁 조건, 교착 상태 방지
> 3. **융합**: 재진입, 비동기 안전, 핸들러와 연관

---

## Ⅰ. 개요

### 개념 정의

시그널 안전성(Signal Safety)은 **시그널 핸들러에서 안전하게 호출할 수 있는 특성**이다. 비동기 안전(async-signal-safe) 함수만 핸들러에서 사용해야 한다.

### 💡 비유: 비상구 규정
시그널 안전성은 **비상구 규정**과 같다. 비상 시(시그널)에는 정해진 경로(안전 함수)만 사용할 수 있다.

### 시그널 안전성 구조

```

│                시그널 안전성 구조                                       │

│                                                                     │
│  【비동기 안전 vs 위험】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  안전 (Signal-Safe)              위험 (Non-Safe)              │ │
│  │  ─────────────────             ─────────────────              │ │
│  │                                                             │ │
│  │  • write(), read()             • printf(), scanf()           │ │
│  │  • _exit()                     • malloc(), free()            │ │
│  │  • getpid(), getppid()          • strtok(), localtime()      │ │
│  │  • kill(), signal()            • pthread_mutex_lock()        │ │
│  │  • sigaction()                 • fopen(), fclose()           │ │
│  │  • sigprocmask()               • sprintf(), snprintf()       │ │
│  │  • wait(), waitpid()            • log() 함수들                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【위험한 상황 예시】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  main()                          handler()                   │ │
│  │    │                                │                        │ │
│  │    ▼                                ▼                        │ │
│  │  malloc() 진입                   시그널 도착                   │ │
│  │    │                          (malloc 중)                     │ │
│  │    │                                │                        │ │
│  │    │                          malloc() 호출?                  │ │
│  │    │                          → 교착 상태!                    │ │
│  │    │                                                        │ │
│  │  (힙 락 보유 중)                                                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【재진입 (Reentrancy)】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  재진입 가능 함수:                                                │ │
│  │  • 정적/전역 변수 사용 안 함                                       │ │
│  │  • 락을 사용하지 않음                                              │ │
│  │  • 다른 비재진입 함수 호출 안 함                                    │ │
│  │  • 언제든 중단 후 재진입해도 안전                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【플래그 패턴】                                                         │
│  ──────────────────                                                  │
│  #include <signal.h>                                                │
│  #include <unistd.h>                                                │
│                                                                     │
│  volatile sig_atomic_t got_signal = 0;                              │
│                                                                     │
│  void handler(int sig) {                                            │
│      got_signal = 1;  // 단순 플래그만 설정                            │
│  }                                                                  │
│                                                                     │
│  int main() {                                                        │
│      signal(SIGINT, handler);                                       │
│                                                                     │
│      while (1) {                                                    │
│          if (got_signal) {                                          │
│              got_signal = 0;                                        │
│              // 여기서 안전하게 복잡한 작업                              │
│              printf("시그널 수신!\n");  // 메인에서는 안전               │
│          }                                                          │
│          pause();                                                   │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  【Self-Pipe 트릭】                                                      │
│  ──────────────────                                                  │
│  int pipe_fd[2];                                                    │
│                                                                     │
│  void handler(int sig) {                                            │
│      char c = 'x';                                                  │
│      write(pipe_fd[1], &c, 1);  // 안전한 write                      │
│  }                                                                  │
│                                                                     │
│  int main() {                                                        │
│      pipe(pipe_fd);                                                 │
│      signal(SIGINT, handler);                                       │
│                                                                     │
│      fd_set rfds;                                                   │
│      FD_ZERO(&rfds);                                                │
│      FD_SET(pipe_fd[0], &rfds);                                     │
│                                                                     │
│      while (select(pipe_fd[0] + 1, &rfds, NULL, NULL, NULL) > 0) {  │
│          char c;                                                    │
│          read(pipe_fd[0], &c, 1);                                   │
│          // 안전하게 복잡한 작업                                       │
│          printf("시그널로 깨어남\n");                                   │
│      }                                                              │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
│  【signalfd (Linux)】                                                   │
│  ──────────────────                                                  │
│  #include <sys/signalfd.h>                                          │
│                                                                     │
│  sigset_t mask;                                                     │
│  sigemptyset(&mask);                                                │
│  sigaddset(&mask, SIGINT);                                          │
│  sigprocmask(SIG_BLOCK, &mask, NULL);                               │
│                                                                     │
│  int fd = signalfd(-1, &mask, 0);                                   │
│  // fd를 select/poll/epoll로 감시                                   │
│                                                                     │
│  struct signalfd_siginfo si;                                        │
│  read(fd, &si, sizeof(si));                                         │
│  // si.ssi_signo로 시그널 번호 확인                                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 핸들러에서 안전한 함수만 사용
• 안전: write, read, _exit, getpid 등
• 위험: printf, malloc, mutex 등
• 패턴: 플래그, self-pipe, signalfd
• 재진입: 전역 변수/락 사용 안 함
• 변수: volatile sig_atomic_t
```
