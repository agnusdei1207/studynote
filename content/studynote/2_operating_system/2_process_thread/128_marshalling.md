# 128. 마샬링/언마샬링 (Marshalling/Unmarshalling)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터를 전송 가능한 형식으로 변환/복원
> 2. **가치**: 이기종 시스템 간 데이터 교환
> 3. **융합**: RPC, 직렬화, XDR과 연관

---

## Ⅰ. 개요

### 개념 정의

마샬링(Marshalling)은 **메모리 내 데이터를 네트워크 전송 가능한 바이트 스트림으로 변환**하는 과정이다. 언마샬링(Unmarshalling)은 **받은 바이트 스트림을 원래 데이터로 복원**하는 과정이다.

### 💡 비유: 짐 싸기
마샬링은 **이사 짐 싸기**와 같다. 가구(데이터)를 박스(바이트 스트림)에 넣어서(마샬링), 트럭으로 옮기고, 다시 꺼낸다(언마샬링).

### 마샬링 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                마샬링 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【마샬링 과정】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  송신측                          수신측                       │ │
│  │  ┌─────────────────┐            ┌─────────────────┐          │ │
│  │  │ struct Person { │            │ struct Person { │          │ │
│  │  │   int id;       │            │   int id;       │          │ │
│  │  │   char name[20];│            │   char name[20];│          │ │
│  │  │   float score;  │            │   float score;  │          │ │
│  │  │ } p = {1, "Kim", 95.5};      │ } p;            │          │ │
│  │  └────────┬────────┘            └────────▲────────┘          │ │
│  │           │                              │                     │ │
│  │           ▼                              │                     │ │
│  │  ┌─────────────────┐            ┌────────┴────────┐          │ │
│  │  │ 마샬링          │            │ 언마샬링        │          │ │
│  │  │                 │            │                 │          │ │
│  │  │ - 바이트 순서   │            │ - 바이트 순서   │          │ │
│  │  │   변환          │            │   복원          │          │ │
│  │  │ - 직렬화        │            │ - 역직렬화      │          │ │
│  │  │ - 패딩 추가     │            │ - 패딩 제거     │          │ │
│  │  └────────┬────────┘            └────────▲────────┘          │ │
│  │           │                              │                     │ │
│  │           ▼                              │                     │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │        바이트 스트림 (네트워크 전송)                      │ │ │
│  │  │  00 01 00 00 │ 4B 69 6D 00 ... │ 42 BF 00 00            │ │ │
│  │  │    id          name               score                  │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【마샬링이 처리하는 것】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  항목                      설명                                │ │
│  │  ────                        ────                                │ │
│  │  바이트 순서             Big/Little Endian 변환                 │ │
│  │  데이터 정렬             패딩 추가/제거                         │ │
│  │  포인터                  실제 데이터로 변환                     │ │
│  │  문자열                  길이 + 내용                            │ │
│  │  구조체                  중첩 구조 평탄화                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Protocol Buffers 마샬링】                                            │
│  ──────────────────                                                  │
│  // .proto 정의                                                       │
│  message Person {                                                    │
│      int32 id = 1;                                                   │
│      string name = 2;                                                │
│      float score = 3;                                                │
│  }                                                                   │
│                                                                     │
│  // C++ 마샬링                                                        │
│  Person p;                                                           │
│  p.set_id(1);                                                        │
│  p.set_name("Kim");                                                  │
│  p.set_score(95.5);                                                  │
│                                                                     │
│  // 직렬화 (마샬링)                                                   │
│  std::string data;                                                   │
│  p.SerializeToString(&data);                                         │
│                                                                     │
│  // 역직렬화 (언마샬링)                                                │
│  Person p2;                                                          │
│  p2.ParseFromString(data);                                           │
│                                                                     │
│  【JSON 마샬링 (Go)】                                                   │
│  ──────────────────                                                  │
│  type Person struct {                                                │
│      ID    int     `json:"id"`                                       │
│      Name  string  `json:"name"`                                     │
│      Score float64 `json:"score"`                                    │
│  }                                                                   │
│                                                                     │
│  // 마샬링                                                            │
│  data, _ := json.Marshal(p)                                          │
│                                                                     │
│  // 언마샬링                                                          │
│  var p2 Person                                                       │
│  json.Unmarshal(data, &p2)                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 마샬링: 데이터 → 바이트 스트림
• 언마샬링: 바이트 스트림 → 데이터
• 처리: 바이트 순서, 정렬, 포인터
• 형식: XDR, Protobuf, JSON
• 장점: 이기종 호환, 타입 보존
• 단점: 오버헤드, 버전 호환
```
