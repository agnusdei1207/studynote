## Ⅰ. 개요

### 개념 정의

안드로이드 바인더(Android Binder)는 **안드로이드 시스템에서 프로세스 간 통신을 위한 핵심 메커니즘**이다. 커널 모듈로 구현되며, RPC와 공유 메모리 기반 통신을 효율적으로 지원한다.

### 💡 비유: 우편 배달부
바인더는 **우편 배달부**와 같다. 앱(송신자)의 메시지를 시스템 서비스(수신자)에게 정확하게 전달한다.

### 안드로이드 바인더 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                안드로이드 바인더 구조                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【바인더 아키텍처】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  클라이언트 프로세스           커널              서버 프로세스    │ │
│  │  ┌─────────────────┐     ┌──────────┐     ┌─────────────────┐ │ │
│  │  │                 │     │          │     │                 │ │ │
│  │  │  프록시 객체     │     │          │     │  스텁 (Stub)    │ │ │
│  │  │  (Proxy)        │────►│  바인더   │────►│                 │ │ │
│  │  │                 │     │  드라이버 │     │  실제 구현      │ │ │
│  │  │  메서드 호출    │     │  (/dev/  │     │  (Service)      │ │ │
│  │  │  ↓             │     │  binder) │     │  ↓             │ │ │
│  │  │  Parcel로 직렬화│     │          │     │  Parcel 역직렬화│ │ │
│  │  │                 │◄────│  라우팅  │◄────│  결과 반환      │ │ │
│  │  └─────────────────┘     └──────────┘     └─────────────────┘ │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【바인더 통신 과정】                                                   │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. 클라이언트: 프록시로 메서드 호출                              │ │
│  │  2. 프록시: 인자를 Parcel로 직렬화                               │ │
│  │  3. ioctl(): /dev/binder에 쓰기                                 │ │
│  │  4. 바인더 드라이버: 대상 프로세스 찾기                           │ │
│  │  5. 대상 스레드: 깨어나서 처리                                   │ │
│  │  6. 스텁: Parcel 역직렬화, 메서드 실행                           │ │
│  │  7. 결과: Parcel로 직렬화, 반환                                  │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【주요 용도】                                                           │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 시스템 서비스 접근 (ActivityManager, PackageManager)          │ │
│  │  • 콘텐트 프로바이더                                            │ │
│  │  • AIDL (Android Interface Definition Language)                │ │
│  │  • 인텐트 전달                                                  │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【AIDL 인터페이스 정의】                                               │
│  ──────────────────                                                  │
│  // IRemoteService.aidl                                             │
│  package com.example;                                               │
│                                                                     │
│  interface IRemoteService {                                         │
│      int getPid();                                                  │
│      void basicTypes(int anInt, long aLong, boolean aBoolean,       │
│                      float aFloat, double aDouble,                  │
│                      String aString);                               │
│  }                                                                  │
│                                                                     │
│  【서비스 구현】                                                        │
│  ──────────────────                                                  │
│  public class RemoteService extends Service {                       │
│      private final IRemoteService.Stub binder = new                 │
│          IRemoteService.Stub() {                                    │
│          @Override                                                  │
│          public int getPid() {                                      │
│              return Process.myPid();                                │
│          }                                                          │
│      };                                                             │
│                                                                     │
│      @Override                                                      │
│      public IBinder onBind(Intent intent) {                         │
│          return binder;                                             │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  【클라이언트 사용】                                                    │
│  ──────────────────                                                  │
│  private ServiceConnection connection = new ServiceConnection() {   │
│      public void onServiceConnected(ComponentName name,             │
│                                     IBinder service) {              │
│          IRemoteService remote = IRemoteService.Stub                │
│                                   .asInterface(service);            │
│          int pid = remote.getPid();                                 │
│      }                                                              │
│  };                                                                 │
│                                                                     │
│  // 서비스 바인딩                                                     │
│  Intent intent = new Intent(this, RemoteService.class);             │
│  bindService(intent, connection, Context.BIND_AUTO_CREATE);         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
