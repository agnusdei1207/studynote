# 157. cgroup (Control Group)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스 그룹별 자원 제한/격리 메커니즘
> 2. **가치**: 자원 관리, 컨테이너 기반 기술
> 3. **융합**: 프로세스, 자원 제한, 컨테이너와 연관

---

## Ⅰ. 개요

### 개념 정의

cgroup(Control Group)은 **프로세스 그룹에 대해 자원 사용을 제한, 격리, 계측하는 리눅스 커널 기능**이다. CPU, 메모리, I/O 등을 그룹별로 제어한다.

### 💡 비유: 버스 전용 차로
cgroup은 **버스 전용 차로**와 같다. 특정 그룹(버스)에만 할당된 자원(차로)을 보장한다.

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                cgroup 상세                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │

│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  파일                      설명                                │ │
│  │  ────                        ────                                │ │
│  │  cgroup.procs           그룹에 속한 PID 목록                    │ │
│  │  cpu.shares             CPU 시간 비율 (가중치)                   │ │
│  │  cpu.cfs_quota_us       CPU 시간 제한 (마이크로초)               │ │
│  │  memory.limit_in_bytes  메모리 제한                            │ │
│  │  memory.usage_in_bytes  현재 메모리 사용량                      │ │
│  │  pids.max               최대 프로세스 개수                       │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【Docker/Kubernetes와 관계】                                           │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 컨테이너 런타임이 cgroup 사용                                  │ │
│  │  • docker run --cpus, --memory                                 │ │
│  │  • Kubernetes: resources.limits, resources.requests             │ │
│  │  • 컨테이너별 자원 격리                                           │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【cgroup 생성/관리】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  # 디렉토리 생성 = cgroup 생성                                   │ │
│  │  mkdir /sys/fs/cgroup/cpu/mygroup                               │ │
│  │                                                               │ │
│  │  # 프로세스 추가                                                 │ │
│  │  echo $$ > /sys/fs/cgroup/cpu/mygroup/cgroup.procs              │ │
│  │                                                               │ │
│  │  # 제한 설정                                                     │ │
│  │  echo 50000 > /sys/fs/cgroup/cpu/mygroup/cpu.cfs_quota_us       │ │
│  │  echo 100000 > /sys/fs/cgroup/cpu/mygroup/cpu.cfs_period_us     │ │
│  │  # → CPU 50%로 제한                                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
```

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【cgcreate/cgset 명령어 (libcgroup)】                                  │
│  ──────────────────                                                  │
│  # cgroup 생성                                                       │
│  $ cgcreate -g cpu,memory:/mygroup                                   │
│                                                                     │
│  # 제한 설정                                                          │
│  $ cgset -r cpu.shares=512 mygroup                                   │
│  $ cgset -r memory.limit_in_bytes=1G mygroup                         │
│                                                                     │
│  # 프로세스 실행                                                      │
│  $ cgexec -g cpu,memory:mygroup ./my_program                         │
│                                                                     │
│  【Docker로 자원 제한】                                                  │
│  ──────────────────                                                  │
│  # CPU 0.5개, 메모리 512MB                                           │
│  $ docker run --cpus=0.5 --memory=512m nginx                         │
│                                                                     │
│  # CPU 코어 0-1만 사용                                                │
│  $ docker run --cpuset-cpus=0,1 nginx                                │
│                                                                     │
│  【systemd로 cgroup 관리】                                              │
│  ──────────────────                                                  │
│  # /etc/systemd/system/myapp.slice                                   │
│  [Slice]                                                             │
│  CPUQuota=50%                                                        │
│  MemoryMax=1G                                                        │
│                                                                     │
│  $ systemctl start myapp.slice                                       │
│                                                                     │
│  【현재 상태 확인】                                                      │
│  ──────────────────                                                  │
│  # cgroup 목록                                                        │
│  $ cat /proc/cgroups                                                  │
│                                                                     │
│  # 프로세스의 cgroup                                                   │
│  $ cat /proc/$$/cgroup                                                │
│                                                                     │
│  # systemd-cgls로 계층 보기                                            │
│  $ systemd-cgls                                                       │
│                                                                     │
│  # 사용량 확인                                                         │
│  $ cat /sys/fs/cgroup/memory/mygroup/memory.usage_in_bytes           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
