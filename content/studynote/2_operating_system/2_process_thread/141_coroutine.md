# 141. 코루틴 (Coroutine)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 실행을 일시 중단하고 재개할 수 있는 함수
> 2. **가치**: 비동기 코드를 동기처럼 작성
> 3. **융합**: async/await, 제너레이터, 경량 스레드와 연관

---

## Ⅰ. 개요

### 개념 정의

코루틴(Coroutine)은 **실행을 일시 중단(suspend)하고 나중에 재개(resume)할 수 있는 함수**다. 협력적 멀티태스킹을 구현하며, 비동기 코드를 동기처럼 작성할 수 있다.

### 💡 비유: 책갈피
코루틴은 **책갈피를 꽂은 독서**와 같다. 읽던 페이지(실행 지점)를 표시해두고 나중에 그대로 이어서 읽는다.

### 코루틴 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                코루틴 구조                                             │
├─────────────────────────────────────────────────────────────────────┤

│  【서브루틴 vs 코루틴】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  서브루틴 (Subroutine)         코루틴 (Coroutine)              │ │
│  │  ──────────────────           ──────────────────              │ │
│  │                                                             │ │
│  │  main()                       main()                         │ │
│  │    │                            │                            │ │
│  │    ▼                            ▼                            │ │
│  │  func()                        coroutine()                   │ │
│  │    │                            │                            │ │
│  │    │ 실행 완료까지               │ yield (일시 중단)            │ │
│  │    │ 진행                        │                            │ │
│  │    │                            ▼                            │ │
│  │    │◄───────────────────────────┘                            │ │
│  │    │                            │ resume (재개)               │ │
│  │    │                            ▼                            │ │
│  │    │                            │ yield                       │ │
│  │    │                            ▼                            │ │
│  │    ▼                            ▼                            │ │
│  │  return                       return                         │ │
│  │                                                             │ │
│  │  • 하나의 진입점               • 여러 진입점                   │ │
│  │  • 호출 시 처음부터             • 이전 상태에서 재개            │ │
│  │  • 스택 기반                   • 힙/스택 저장                  │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【코루틴 상태】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  ┌──────────┐   yield    ┌──────────┐                       │ │
│  │  │  실행 중  │───────────►│ 일시 중단 │                       │ │
│  │  │(Running) │            │(Suspended)│                       │ │
│  │  └──────────┘            └─────┬────┘                       │ │
│  │       │                        │                             │ │
│  │       │                        │ resume                      │ │
│  │       ▼                        ▼                             │ │
│  │  ┌──────────┐            ┌──────────┐                       │ │
│  │  │  완료    │◄───────────│  재개    │                       │ │
│  │  │(Dead)   │   return   │(Resumed) │                       │ │
│  │  └──────────┘            └──────────┘                       │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【종류】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  종류                      설명                                │ │
│  │  ────                        ────                                │ │
│  │  Stackful                자체 스택, 중첩 호출 가능              │ │
│  │  Stackless               상태만 저장, 제너레이터                │ │
│  │  Symmetric               caller를 직접 지정                    │ │
│  │  Asymmetric              yield로 caller 복귀                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Python 제너레이터】                                                  │
│  ──────────────────                                                  │
│  def countdown(n):                                                  │
│      while n > 0:                                                   │
│          yield n          # 일시 중단하고 값 반환                     │
│          n -= 1                                                     │
│                                                                     │
│  for i in countdown(5):                                             │
│      print(i)  # 5, 4, 3, 2, 1                                      │
│                                                                     │
│  【Python async/await】                                                │
│  ──────────────────                                                  │
│  import asyncio                                                     │
│                                                                     │
│  async def fetch_data(url):                                         │
│      print(f"Fetching {url}")                                       │
│      await asyncio.sleep(1)  # 비동기 대기                           │
│      return f"data from {url}"                                      │
│                                                                     │
│  async def main():                                                  │
│      # 동시 실행                                                     │
│      results = await asyncio.gather(                                │
│          fetch_data("url1"),                                        │
│          fetch_data("url2"),                                        │
│      )                                                              │
│      print(results)                                                 │
│                                                                     │
│  asyncio.run(main())                                                │
│                                                                     │
│  【Kotlin 코루틴】                                                      │
│  ──────────────────                                                  │
│  import kotlinx.coroutines.*                                        │
│                                                                     │
│  suspend fun fetchData(url: String): String {                       │
│      delay(1000)  // 비동기 대기                                     │
│      return "data from $url"                                        │
│  }                                                                  │
│                                                                     │
│  fun main() = runBlocking {                                         │
│      val job = launch {      // 새 코루틴 시작                       │
│          val data = fetchData("url1")                               │
│          println(data)                                              │
│      }                                                              │
│      job.join()            // 완료 대기                              │
│  }                                                                  │
│                                                                     │
│  【JavaScript async/await】                                            │
│  ──────────────────                                                  │
│  async function fetchUser(id) {                                     │
│      const response = await fetch(`/api/users/${id}`);              │
│      return response.json();                                        │
│  }                                                                  │
│                                                                     │
│  async function main() {                                            │
│      const user = await fetchUser(1);                               │
│      console.log(user);                                             │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
