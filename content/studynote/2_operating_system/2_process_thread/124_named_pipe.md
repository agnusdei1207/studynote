# 124. 지명 파이프 (Named Pipe, FIFO)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일 시스템에 이름이 있는 파이프
> 2. **가치**: 부모-자식 관계 없는 프로세스 간 통신
> 3. **융합**: 익명 파이프, 파일 시스템, IPC와 연관

---

## Ⅰ. 개요

### 개념 정의

지명 파이프(Named Pipe, FIFO)는 **파일 시스템에 이름이 있는 특수 파일로, 관계없는 프로세스 간 통신을 가능하게 하는 파이프**다. 익명 파이프와 달리 파일 경로로 접근한다.

### 💡 비유: 공용 우편함
지명 파이프는 **공용 우편함**과 같다. 주소(파일 경로)를 알면 누구나 편지를 넣고 뺄 수 있다.

### 지명 파이프 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                지명 파이프 구조                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【지명 파이프 vs 익명 파이프】                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  익명 파이프                지명 파이프 (FIFO)                 │ │
│  │  ────────────               ─────────────────                 │ │
│  │  부모-자식만                관계없는 프로세스                   │ │
│  │  파일 시스템 없음           파일 시스템에 존재                   │ │
│  │  fork()로 상속              경로로 접근                        │ │
│  │  프로세스 종료 시 소멸       명시적 삭제까지 존재                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【지명 파이프 동작】                                                   │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  파일 시스템                                                  │ │
│  │  ┌───────────────────────────────────────────────────────┐   │ │
│  │  │  /tmp/myfifo (FIFO 특수 파일)                          │   │ │
│  │  │                    ▲                                    │   │ │
│  │  │                    │                                    │   │ │
│  │  ┌─────────────┐     │      ┌─────────────┐               │   │ │
│  │  │ Process A   │     │      │ Process B   │               │   │ │
│  │  │ (무관)      │     │      │ (무관)      │               │   │ │
│  │  │             │     │      │             │               │   │ │
│  │  │ fd = open() ├─────┼──────┤ open()      │               │   │ │
│  │  │ write()     │     │      │ read()      │               │   │ │
│  │  └─────────────┘     │      └─────────────┘               │   │ │
│  │                      │                                    │   │ │
│  │                      ▼                                    │   │ │
│  │  ┌───────────────────────────────────────────────────────┐   │ │
│  │  │  커널 파이프 버퍼                                       │   │ │
│  │  └───────────────────────────────────────────────────────┘   │ │
│  │                                                             │   │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【특징】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 파일 타입: 'p' (ls -l 로 확인)                               │ │
│  │  • open() 시 양쪽이 모두 열릴 때까지 블록                        │ │
│  │  • 단방향 (읽기 또는 쓰기 중 하나)                               │ │
│  │  • 양방향 통신: 두 개의 FIFO 필요                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                지명 파이프 상세                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【생성 방법】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  // 쉘 명령어                                                   │ │
│  │  $ mkfifo /tmp/myfifo                                          │ │
│  │  $ mknod /tmp/myfifo p                                         │ │
│  │                                                               │ │
│  │  // C 함수                                                      │ │
│  │  #include <sys/stat.h>                                         │ │
│  │  int mkfifo(const char *pathname, mode_t mode);                │ │
│  │  int mknod(const char *path, mode_t mode | S_IFIFO, dev_t);   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【사용 패턴】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  // 송신측                                                      │ │
│  │  int fd = open("/tmp/myfifo", O_WRONLY);                       │ │
│  │  write(fd, "Hello", 5);                                        │ │
│  │  close(fd);                                                    │ │
│  │                                                               │ │
│  │  // 수신측                                                      │ │
│  │  int fd = open("/tmp/myfifo", O_RDONLY);                       │ │
│  │  read(fd, buf, sizeof(buf));                                   │ │
│  │  close(fd);                                                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                           │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 관계없는 프로세스 통신   • 단방향                             │ │
│  │  • 파일 경로로 접근         • 파일 시스템 오버헤드               │ │
│  │  • 영구적 존재              • 수동 삭제 필요                     │ │
│  │  • 쉘에서 사용 가능          • 동시 접근 제한                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【쉘에서 FIFO 사용】                                                   │
│  ──────────────────                                                  │
│  # FIFO 생성                                                         │
│  $ mkfifo /tmp/myfifo                                               │
│                                                                     │
│  # 한 터미널에서 쓰기                                                  │
│  $ echo "Hello World" > /tmp/myfifo                                 │
│                                                                     │
│  # 다른 터미널에서 읽기                                                │
│  $ cat /tmp/myfifo                                                  │
│  Hello World                                                        │
│                                                                     │
│  # FIFO 삭제                                                         │
│  $ rm /tmp/myfifo                                                   │
│                                                                     │
│  【FIFO 확인】                                                         │
│  ──────────────────                                                  │
│  $ ls -l /tmp/myfifo                                                │
│  prw-r--r-- 1 user user 0 ... /tmp/myfifo                           │
│  #    ^ 파일 타입 'p' = FIFO                                        │
│                                                                     │
│  $ file /tmp/myfifo                                                 │
│  /tmp/myfifo: fifo (named pipe)                                     │
│                                                                     │
│  【비블로킹 열기】                                                       │
│  ──────────────────                                                  │
│  // O_NONBLOCK로 열면 상대방 없어도 즉시 반환                          │
│  int fd = open("/tmp/myfifo", O_RDONLY | O_NONBLOCK);              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
