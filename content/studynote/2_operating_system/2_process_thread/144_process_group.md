# 144. 프로세스 그룹 (Process Group)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 관련 프로세스들의 논리적 집합

> 3. **융합**: 세션, 작업 제어, 시그널과 연관

---

## Ⅰ. 개요

### 개념 정의

프로세스 그룹(Process Group)은 **관련된 프로세스들의 논리적 집합**이다. 같은 그룹의 프로세스는 동일한 프로세스 그룹 ID(PGID)를 공유하며, 시그널을 일괄 수신할 수 있다.

### 💡 비유: 팀 프로젝트
프로세스 그룹은 **팀 프로젝트 팀**과 같다. 같은 팀원(프로세스)은 같은 공지(시그널)를 받는다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                프로세스 그룹 상세                                       │

│                                                                     │
│  【관련 API】                                                           │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  함수                      설명                                │ │
│  │  ────                        ────                                │ │
│  │  getpgrp()              현재 프로세스 그룹 ID 반환               │ │
│  │  getpgid(pid)           지정 프로세스의 PGID 반환                │ │
│  │  setpgid(pid, pgid)     프로세스 그룹 설정                       │ │
│  │  setpgrp()              새 프로세스 그룹 생성                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【setpgid 규칙】                                                       │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 호출 프로세스 또는 자식만 변경 가능                            │ │
│  │  • 자식이 exec() 전에만 가능                                     │ │
│  │  • 세션 이동 불가 (같은 세션 내에서만)                            │ │
│  │  • pgid == pid면 새 그룹 생성 (리더)                             │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【작업 제어 (Job Control)】                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • Ctrl+C: SIGINT → Foreground 그룹 전체                        │ │
│  │  • Ctrl+Z: SIGTSTP → Foreground 그룹 일시 중단                   │ │
│  │  • fg/bg: 그룹을 포그라운드/백그라운드로 이동                     │ │
│  │  • Shell이 작업 관리                                            │ │
│  │                                                               │ │

│                                                                     │
│  【장단점】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 시그널 일괄 처리         • 복잡한 계층                        │ │
│  │  • 작업 제어 가능           • 디버깅 어려움                     │ │
│  │  • 파이프라인 관리          • 세션 규칙 준수                     │ │

│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【프로세스 그룹 생성과 시그널】                                         │
│  ──────────────────                                                  │
│  #include <unistd.h>                                                │
│  #include <signal.h>                                                │
│  #include <stdio.h>                                                 │
│                                                                     │
│  int main() {                                                       │
│      pid_t pid;                                                     │
│      int i;                                                         │
│                                                                     │
│      // 자식 프로세스들 생성                                          │
│      for (i = 0; i < 3; i++) {                                      │
│          pid = fork();                                              │
│          if (pid == 0) {                                            │
│              // 자식                                                │
│              pause();  // 시그널 대기                                 │
│              printf("자식 %d: 시그널 수신\n", getpid());               │
│              _exit(0);                                              │
│          }                                                          │
│      }                                                              │
│                                                                     │
│      // 부모: 자식들을 같은 그룹으로                                   │
│      setpgid(pid, pid);  // 새 그룹 생성                             │
│      for (i = 0; i < 3; i++) {                                      │
│          setpgid(-1, pid);  // 다른 자식도 같은 그룹으로               │
│      }                                                              │
│                                                                     │
│      sleep(1);                                                      │
│                                                                     │
│      // 그룹 전체에 시그널 전송                                        │
│      kill(-pid, SIGUSR1);  // 음수 PGID                              │
│                                                                     │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
│  【셸 명령어】                                                           │
│  ──────────────────                                                  │
│  # 프로세스 그룹 확인                                                  │
│  $ ps -j                        # PGID, SID 표시                    │
│  $ ps -ef --forest              # 프로세스 트리                       │
│                                                                     │
│  # 파이프라인 (같은 그룹)                                              │
│  $ cat file | grep pattern | wc -l                                   │
│  # → 세 프로세스가 같은 프로세스 그룹                                   │
│                                                                     │
│  # 그룹에 시그널 전송                                                   │
│  $ kill -SIGTERM -1234          # PGID 1234에 전송                   │
│                                                                     │
│  # 작업 제어                                                          │
│  $ sleep 100 &                     # 백그라운드                      │
│  $ jobs                            # 작업 목록                       │
│  $ fg %1                           # 포그라운드로                     │
│  $ bg %1                           # 백그라운드로                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
