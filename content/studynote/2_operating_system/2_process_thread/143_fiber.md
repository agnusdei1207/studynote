# 143. 파이버 (Fiber)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: Windows의 사용자 수준 협력적 스레드
> 2. **가치**: 세밀한 스케줄링 제어, 코루틴 구현
> 3. **융합**: 스레드, 코루틴, 그린 스레드와 연관

---

## Ⅰ. 개요

### 개념 정의

파이버(Fiber)는 **Windows에서 제공하는 사용자 수준의 협력적 멀티태스킹 단위**다. 스레드 내에서 실행되며 명시적으로 전환(yield)해야 한다.

### 💡 비유: 수동 변속기
파이버는 **수동 변속기 자동차**와 같다. 기어를 언제 바꿀지(전환할지) 운전자가 직접 결정한다.

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Windows 파이버 예시】                                               │
│  ──────────────────                                                  │
│  #include <windows.h>                                               │
│  #include <stdio.h>                                                 │
│                                                                     │
│  LPVOID fiber1, fiber2, mainFiber;                                  │
│                                                                     │
│  void WINAPI FiberFunc1(LPVOID lpParam) {                           │
│      printf("Fiber 1 시작\n");                                       │
│      SwitchToFiber(fiber2);      // 파이버 2로 전환                  │
│      printf("Fiber 1 재개\n");                                       │
│      SwitchToFiber(mainFiber);    // 메인으로 복귀                   │
│  }                                                                  │
│                                                                     │
│  void WINAPI FiberFunc2(LPVOID lpParam) {                           │
│      printf("Fiber 2 시작\n");                                       │
│      SwitchToFiber(fiber1);      // 파이버 1로 전환                  │
│      printf("Fiber 2 재개\n");                                       │
│  }                                                                  │
│                                                                     │
│  int main() {                                                       │
│      // 스레드를 파이버로 변환                                        │
│      mainFiber = ConvertThreadToFiber(NULL);                        │
│                                                                     │
│      // 파이버 생성                                                   │
│      fiber1 = CreateFiber(0, FiberFunc1, NULL);                     │
│      fiber2 = CreateFiber(0, FiberFunc2, NULL);                     │
│                                                                     │
│      // 파이버 1로 전환                                               │
│      SwitchToFiber(fiber1);                                         │
│                                                                     │
│      printf("메인으로 복귀\n");                                       │
│                                                                     │
│      // 정리                                                         │
│      DeleteFiber(fiber1);                                           │
│      DeleteFiber(fiber2);                                           │
│      ConvertFiberToThread();                                        │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
│  【유닉스 ucontext 예시】                                              │
│  ──────────────────                                                  │
│  #include <ucontext.h>                                              │
│                                                                     │
│  ucontext_t ctx1, ctx2, main_ctx;                                   │
│                                                                     │
│  void func1() {                                                     │
│      printf("Context 1\n");                                         │
│      swapcontext(&ctx1, &ctx2);  // ctx2로 전환                      │
│      printf("Back to 1\n");                                         │
│  }                                                                  │
│                                                                     │
│  int main() {                                                       │
│      char stack1[8192], stack2[8192];                               │
│                                                                     │
│      getcontext(&ctx1);                                             │
│      ctx1.uc_stack.ss_sp = stack1;                                  │
│      ctx1.uc_stack.ss_size = sizeof(stack1);                        │
│      makecontext(&ctx1, func1, 0);                                  │
│                                                                     │
│      // ... ctx2도 유사하게 설정                                     │
│                                                                     │
│      swapcontext(&main_ctx, &ctx1);  // ctx1로 전환                  │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
