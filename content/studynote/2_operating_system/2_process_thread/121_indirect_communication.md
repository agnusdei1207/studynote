# 121. 간접 통신 (Indirect Communication)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 메일박스/포트를 통해 메시지를 간접 전달
> 2. **가치**: 모듈화, 유연성, 다중 통신 지원
> 3. **융합**: 직접 통신, 메시지 큐, 메일박스와 연관

---

## Ⅰ. 개요

### 개념 정의

간접 통신(Indirect Communication)은 **메일박스나 포트를 통해 메시지를 간접적으로 전달**하는 방식이다. 송신자와 수신자가 서로를 직접 지정하지 않고 중개자를 통해 통신한다.

### 💡 비유: 사서함
간접 통신은 **사서함(PO Box)**과 같다. 편지를 특정 사람에게 직접 주지 않고, 사서함에 넣으면 상대방이 나중에 가져간다.

### 간접 통신 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                간접 통신 구조                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【간접 통신 동작】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  Process A                    Mailbox                    Process B │
│  │  ┌─────────────┐           ┌─────────────┐           ┌─────────────┐ │
│  │  │             │           │             │           │             │ │
│  │  │ send(A, msg)│ ────────► │  Message    │ ◄───────  │ receive(A,  │ │
│  │  │             │           │   Queue     │           │    msg)     │ │
│  │  │             │           │             │           │             │ │
│  │  │             │           │  ┌───────┐  │           │             │ │
│  │  │             │           │  │ Msg 1 │  │           │             │ │
│  │  │             │           │  │ Msg 2 │  │           │             │ │
│  │  │             │           │  │ Msg 3 │  │           │             │ │
│  │  │             │           │  └───────┘  │           │             │ │
│  │  └─────────────┘           └─────────────┘           └─────────────┘ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【프리미티브】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  create(mailbox)        - 메일박스 생성                        │ │
│  │  send(mailbox, msg)     - 메일박스로 메시지 전송                │ │
│  │  receive(mailbox, msg)  - 메일박스에서 메시지 수신              │ │
│  │  delete(mailbox)        - 메일박스 삭제                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【메일박스 소유권】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  유형                      설명                                │ │
│  │  ────                        ────                                │ │
│  │  프로세스 소유            생성 프로세스 종료 시 삭제             │ │
│  │  시스템 소유              독립적 존재, 명시적 삭제 필요           │ │
│  │  공유 메일박스            여러 프로세스 접근 가능                 │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【System V 메시지 큐 (간접 통신)】                                     │
│  ──────────────────                                                  │
│  // 메일박스(메시지 큐) 생성                                          │
│  key_t key = ftok("/tmp", 'A');                                     │
│  int msqid = msgget(key, IPC_CREAT | 0666);  // 메일박스 ID         │
│                                                                     │
│  // 송신: 메일박스로 전송 (수신자 모름)                                 │
│  struct msgbuf msg = {1, "Hello"};                                  │
│  msgsnd(msqid, &msg, sizeof(msg.mtext), 0);                        │
│                                                                     │
│  // 수신: 메일박스에서 수신 (송신자 모름)                               │
│  msgrcv(msqid, &msg, sizeof(msg.mtext), 1, 0);                     │
│                                                                     │
│  【POSIX 메시지 큐】                                                   │
│  ──────────────────                                                  │
│  // 메일박스 생성                                                     │
│  mqd_t mq = mq_open("/mailbox", O_CREAT | O_RDWR, 0666, NULL);     │
│                                                                     │
│  // 송신                                                            │
│  mq_send(mq, "Hello", 5, 0);                                        │
│                                                                     │
│  // 수신                                                            │
│  mq_receive(mq, buffer, 100, NULL);                                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
