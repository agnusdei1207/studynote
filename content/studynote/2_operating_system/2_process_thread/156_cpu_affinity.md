# 156. CPU 선호도 (CPU Affinity)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스가 실행될 CPU를 지정하는 메커니즘
> 2. **가치**: 캐시 지역성, 성능 최적화
> 3. **융합**: 스케줄링, 멀티코어, NUMA와 연관

---

## Ⅰ. 개요

### 개념 정의

CPU 선호도(CPU Affinity)는 **프로세스나 스레드가 특정 CPU 코어에서 실행되도록 제한하는 메커니즘**이다. 소프트 선호도(soft affinity)와 하드 선호도(hard affinity)가 있다.

### 💡 비유: 담당 구역
CPU 선호도는 **우편 배달원의 담당 구역**과 같다. 특정 배달원(프로세스)은 정해진 구역(CPU)만 담당한다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                CPU 선호도 상세                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【CPU_SET 매크로】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  #define _GNU_SOURCE                                            │ │
│  │  #include <sched.h>                                             │ │
│  │                                                               │ │
│  │  cpu_set_t mask;                                                │ │
│  │                                                               │ │
│  │  CPU_ZERO(&mask);          // 초기화                            │ │
│  │  CPU_SET(cpu, &mask);      // CPU 추가                          │ │
│  │  CPU_CLR(cpu, &mask);      // CPU 제거                          │ │
│  │  CPU_ISSET(cpu, &mask);    // CPU 포함 확인                     │ │
│  │  CPU_COUNT(&mask);         // 설정된 CPU 개수                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【sched_setaffinity API】                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  int sched_setaffinity(pid_t pid, size_t cpusetsize,            │ │
│  │                        const cpu_set_t *mask);                  │ │
│  │                                                               │ │
│  │  int sched_getaffinity(pid_t pid, size_t cpusetsize,            │ │
│  │                        cpu_set_t *mask);                        │ │
│  │                                                               │ │
│  │  pid = 0: 호출 스레드/프로세스                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【pthread 버전】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  #include <pthread.h>                                           │ │
│  │                                                               │ │
│  │  int pthread_setaffinity_np(pthread_t thread,                   │ │
│  │                            size_t cpusetsize,                   │ │
│  │                            const cpu_set_t *cpuset);            │ │
│  │                                                               │ │
│  │  int pthread_getaffinity_np(pthread_t thread,                   │ │
│  │                            size_t cpusetsize,                   │ │

│  │                                                               │ │
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【CPU 선호도 설정】                                                     │
│  ──────────────────                                                  │
│  #define _GNU_SOURCE                                                │
│  #include <sched.h>                                                 │
│  #include <unistd.h>                                                │
│  #include <stdio.h>                                                 │
│                                                                     │
│  void set_cpu_affinity(int cpu) {                                   │
│      cpu_set_t mask;                                                │
│      CPU_ZERO(&mask);                                               │
│      CPU_SET(cpu, &mask);                                           │
│                                                                     │
│      if (sched_setaffinity(0, sizeof(mask), &mask) < 0) {           │
│          perror("sched_setaffinity");                               │
│          return;                                                    │
│      }                                                              │
│      printf("CPU %d에 바인딩됨\n", cpu);                               │
│  }                                                                  │
│                                                                     │
│  void print_cpu_affinity() {                                        │
│      cpu_set_t mask;                                                │
│      CPU_ZERO(&mask);                                               │
│                                                                     │
│      if (sched_getaffinity(0, sizeof(mask), &mask) == 0) {          │
│          printf("허용된 CPU: ");                                      │
│          for (int i = 0; i < CPU_SETSIZE; i++) {                    │
│              if (CPU_ISSET(i, &mask))                               │
│                  printf("%d ", i);                                   │
│          }                                                          │
│          printf("\n");                                               │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  【taskset 명령어】                                                      │
│  ──────────────────                                                  │
│  # 특정 CPU에서 실행 (CPU 0)                                          │
│  $ taskset -c 0 ./my_program                                        │
│                                                                     │
│  # 여러 CPU에서 실행 (CPU 0, 2)                                        │
│  $ taskset -c 0,2 ./my_program                                      │
│                                                                     │
│  # 16진수 마스크로 실행                                                │
│  $ taskset 0x5 ./my_program      # 0101 = CPU 0, 2                  │
│                                                                     │
│  # 실행 중인 프로세스 선호도 확인/변경                                    │
│  $ taskset -p 1234                                                   │
│  $ taskset -cp 0,1 1234                                              │
│                                                                     │
│  【스레드 선호도 설정】                                                  │
│  ──────────────────                                                  │
│  #define _GNU_SOURCE                                                │
│  #include <pthread.h>                                               │
│                                                                     │
│  void* thread_func(void* arg) {                                     │
│      int cpu = *(int*)arg;                                          │
│      cpu_set_t mask;                                                │
│      CPU_ZERO(&mask);                                               │
│      CPU_SET(cpu, &mask);                                           │
│      pthread_setaffinity_np(pthread_self(), sizeof(mask), &mask);   │
│      // ...                                                         │
│      return NULL;                                                   │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 특정 CPU에서 실행하도록 제한
• 종류: 소프트(선호), 하드(강제)
• API: sched_setaffinity, CPU_SET
• 명령: taskset
• 장점: 캐시 지역성, 예측 가능성
• 용도: 실시간, NUMA, 성능 최적화
```
