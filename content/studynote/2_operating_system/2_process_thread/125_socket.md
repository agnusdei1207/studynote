# 125. 소켓 (Socket)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 네트워크 통신을 위한 엔드포인트 추상화
> 2. **가치**: 로컬/원격 프로세스 간 양방향 통신
> 3. **융합**: TCP/UDP, 클라이언트-서버, IPC와 연관

---

## Ⅰ. 개요

### 개념 정의

소켓(Socket)은 **네트워크 통신을 위한 엔드포인트 추상화**다. 로컬 및 원격 프로세스 간 양방향 통신을 가능하게 하며, TCP/IP 프로토콜의 기반이다.

### 💡 비유: 전화기
소켓은 **전화기**와 같다. 전화번호(IP:Port)로 상대방에게 연결하고, 양방향으로 대화한다.

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【TCP 서버】                                                           │
│  ──────────────────                                                  │
│  int server_fd = socket(AF_INET, SOCK_STREAM, 0);                  │
│                                                                     │
│  struct sockaddr_in addr;                                           │
│  addr.sin_family = AF_INET;                                         │
│  addr.sin_addr.s_addr = INADDR_ANY;                                │
│  addr.sin_port = htons(8080);                                       │
│                                                                     │
│  bind(server_fd, &addr, sizeof(addr));                              │
│  listen(server_fd, 5);                                              │
│                                                                     │
│  int client_fd = accept(server_fd, NULL, NULL);                    │
│  recv(client_fd, buffer, sizeof(buffer), 0);                        │
│  send(client_fd, "Hello", 5, 0);                                    │
│  close(client_fd);                                                  │
│  close(server_fd);                                                  │
│                                                                     │
│  【TCP 클라이언트】                                                     │
│  ──────────────────                                                  │
│  int sock = socket(AF_INET, SOCK_STREAM, 0);                       │
│                                                                     │
│  struct sockaddr_in server;                                         │
│  server.sin_family = AF_INET;                                       │
│  server.sin_port = htons(8080);                                     │
│  inet_pton(AF_INET, "127.0.0.1", &server.sin_addr);                │
│                                                                     │
│  connect(sock, &server, sizeof(server));                            │
│  send(sock, "Hello", 5, 0);                                         │
│  recv(sock, buffer, sizeof(buffer), 0);                             │
│  close(sock);                                                       │
│                                                                     │
│  【소켓 상태 확인】                                                     │
│  ──────────────────                                                  │
│  $ netstat -tulpn               # 리스닝 소켓                        │
│  $ ss -tulpn                    # 소켓 통계                          │
│  $ lsof -i :8080                # 특정 포트 소켓                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
