# 520. 이벤트 루프 (Event Loop)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 이벤트를 폴링하고 디스패치하는 무한 루프
> 2. **가치**: 단일 스레드로 높은 동시성
> 3. **융합**: I/O 멀티플렉싱, 콜백, 리액터와 연관

---

## Ⅰ. 개요

### 개념 정의

이벤트 루프(Event Loop)는 **이벤트를 대기하고, 가져와서, 처리하는 패턴을 반복하는 프로그래밍 구조**이다.

### 💡 비유: 식당 웨이터
이벤트 루프는 **식당 웨이터**와 같다. 테이블을 돌며 주문(이벤트)을 확인하고, 처리한다.

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 이벤트 루프 구현】                                               │
│  ──────────────────                                                  │
│  #include <sys/epoll.h>                                              │
│  #include <unistd.h>                                                 │
│                                                                     │
│  #define MAX_EVENTS 1024                                              │
│                                                                     │
│  void event_loop(int epfd) {                                          │
│      struct epoll_event events[MAX_EVENTS];                          │
│                                                                     │
│      while (running) {                                                │
│          // 1. 이벤트 대기                                               │
│          int n = epoll_wait(epfd, events, MAX_EVENTS, timeout);       │
│                                                                     │
│          // 2. 이벤트 처리                                               │
│          for (int i = 0; i < n; i++) {                                │
│              handler_t *handler = (handler_t *)events[i].data.ptr;    │
│              handler->callback(handler->fd, events[i].events);         │
│          }                                                             │
│                                                                     │
│          // 3. 타이머 처리 (필요시)                                      │
│          process_timers();                                             │
│      }                                                               │
│  }                                                                   │
│                                                                     │
│  【Node.js 이벤트 루프 예시】                                            │
│  ──────────────────                                                  │
│  const fs = require('fs');                                           │
│  const http = require('http');                                        │
│                                                                     │
│  // I/O 이벤트                                                         │
│  fs.readFile('file.txt', (err, data) => {                            │
│      console.log('File read complete');                              │
│  });                                                                 │
│                                                                     │
│  // 타이머 이벤트                                                        │
│  setTimeout(() => {                                                   │
│      console.log('Timer fired');                                      │
│  }, 1000);                                                           │
│                                                                     │
│  // 즉시 실행                                                           │
│  setImmediate(() => {                                                 │
│      console.log('Immediate');                                        │
│  });                                                                 │
│                                                                     │
│  // HTTP 서버                                                          │
│  const server = http.createServer((req, res) => {                    │
│      res.writeHead(200);                                              │
│      res.end('Hello World');                                          │
│  });                                                                 │
│  server.listen(8080);                                                │
│                                                                     │
│  console.log('Server started');  // 동기 실행                          │
│                                                                     │
│  【Python asyncio 이벤트 루프】                                          │
│  ──────────────────                                                  │
│  import asyncio                                                      │
│                                                                     │
│  async def handle_client(reader, writer):                            │
│      data = await reader.read(100)                                   │
│      writer.write(b'Response')                                        │
│      await writer.drain()                                             │
│      writer.close()                                                   │
│                                                                     │
│  async def main():                                                    │
│      server = await asyncio.start_server(                            │
│          handle_client, 'localhost', 8888)                            │
│      async with server:                                               │
│          await server.serve_forever()                                 │
│                                                                     │
│  asyncio.run(main())                                                  │
│                                                                     │
│  【libuv (Node.js 이벤트 루프 엔진)】                                     │
│  ──────────────────                                                  │
│  #include <uv.h>                                                      │
│                                                                     │
│  int main() {                                                        │
│      uv_loop_t *loop = uv_default_loop();                            │
│                                                                     │
│      // 타이머                                                          │
│      uv_timer_t timer;                                                │
│      uv_timer_init(loop, &timer);                                     │
│      uv_timer_start(&timer, timer_cb, 1000, 1000);                   │
│                                                                     │
│      // TCP 서버                                                        │
│      uv_tcp_t server;                                                 │
│      uv_tcp_init(loop, &server);                                      │
│                                                                     │
│      struct sockaddr_in addr;                                         │
│      uv_ip4_addr("0.0.0.0", 8080, &addr);                           │
│      uv_tcp_bind(&server, (const struct sockaddr*)&addr, 0);          │
│      uv_listen((uv_stream_t*)&server, 128, on_connection);           │
│                                                                     │
│      // 이벤트 루프 실행                                                 │
│      uv_run(loop, UV_RUN_DEFAULT);                                   │
│                                                                     │
│      return 0;                                                        │
│  }                                                                   │
│                                                                     │
│  【이벤트 루프 차단 방지】                                                │
│  ──────────────────                                                  │
│  // 나쁜 예: 동기 작업                                                   │
│  while (true) { /* CPU 연산 */ }  // 이벤트 루프 차단                   │
│                                                                     │
│  // 좋은 예: 비동기 분할                                                 │
│  async function cpuIntensive(data) {                                 │
│      for (let chunk of data) {                                        │
│          process(chunk);                                              │
│          await new Promise(r => setImmediate(r));  // 양보            │
│      }                                                               │
│  }                                                                   │
│                                                                     │
│  // 워커 스레드 사용                                                     │
│  const { Worker } = require('worker_threads');                        │
│  const worker = new Worker('./cpu_intensive.js');                    │
│  worker.on('message', result => { /* 처리 */ });                      │
│                                                                     │
│  【이벤트 루프 모니터링】                                                  │
│  ──────────────────                                                  │
│  // Node.js 이벤트 루프 지연 측정                                        │
│  const { monitorEventLoopDelay } = require('perf_hooks');            │
│  const h = monitorEventLoopDelay();                                  │
│  h.enable();                                                         │
│  // ... 코드 실행 ...                                                  │
│  console.log(h.min);  // 최소 지연                                     │
│  console.log(h.max);  // 최대 지연                                     │
│  console.log(h.mean);  // 평균 지연                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 이벤트 대기 → 가져오기 → 디스패치 반복
• 구조: 이벤트 큐 + 이벤트 루프 + 콜백
• Node.js: timers → pending → idle → poll → check → close
• 장점: 높은 동시성, 단순한 동기화
• 단점: CPU 바운드 부적합, 차단 위험
• 구현: libuv, epoll, kqueue
• 언어: Node.js, Python asyncio, Rust tokio
• 모니터링: 이벤트 루프 지연 측정
```
