# 150. 실시간 시그널 (Real-time Signal)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 큐잉되는 확장 시그널 메커니즘
> 2. **가치**: 신호 손실 방지, 데이터 전달
> 3. **융합**: 표준 시그널, 시그널 큐, sigqueue와 연관

---

## Ⅰ. 개요

### 개념 정의

실시간 시그널(Real-time Signal)은 **큐에 저장되어 손실되지 않는 시그널**이다. 표준 시그널과 달리 여러 번 전송하면 모두 큐잉된다.

### 💡 비유: 메시지 큐
실시간 시그널은 **메시지 큐**와 같다. 보낸 메시지가 모두 저장되어 하나도 잃지 않는다.

### 실시간 시그널 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                실시간 시그널 구조                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【표준 시그널 vs 실시간 시그널】                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  표준 시그널                  실시간 시그널                     │ │
│  │  ────────────               ────────────                      │ │
│  │                                                             │ │
│  │  SIGINT, SIGTERM 등          SIGRTMIN ~ SIGRTMAX              │ │
│  │  큐잉 안 됨                   큐잉 됨                           │ │
│  │  같은 시그널 중복 → 1회        모두 큐에 저장                    │ │
│  │  우선순위 정해짐              번호순 전달                        │ │
│  │  데이터 없음                  정수/포인터 데이터 전달 가능         │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【실시간 시그널 범위】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • SIGRTMIN (보통 34) ~ SIGRTMAX (보통 64)                      │ │
│  │  • POSIX는 최소 8개 보장 (RTSIG_MAX >= 8)                       │ │
│  │  • Linux: 32개 (SIGRTMIN=34, SIGRTMAX=64)                      │ │
│  │  • 애플리케이션에서 자유롭게 사용                                  │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【시그널 전달 예시】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  표준 시그널 (SIGUSR1):                                          │ │
│  │  보낸 횟수: 5회 → 수신: 1회 (나머지 무시)                          │ │
│  │                                                             │ │
│  │  실시간 시그널 (SIGRTMIN):                                        │ │
│  │  보낸 횟수: 5회 → 수신: 5회 (모두 큐잉)                            │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                실시간 시그널 상세                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【sigqueue() API】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  #include <signal.h>                                            │ │
│  │                                                               │ │
│  │  union sigval {                                                 │ │
│  │      int   sival_int;    // 정수 데이터                          │ │
│  │      void *sival_ptr;    // 포인터 데이터                        │ │
│  │  };                                                             │ │
│  │                                                               │ │
│  │  int sigqueue(pid_t pid, int sig, union sigval value);          │ │
│  │                                                               │ │
│  │  // 사용 예                                                     │ │
│  │  union sigval sv;                                               │ │
│  │  sv.sival_int = 42;                                             │ │
│  │  sigqueue(pid, SIGRTMIN, sv);                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【실시간 시그널 핸들러】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  void handler(int sig, siginfo_t *info, void *ucontext) {       │ │
│  │      // 데이터 접근                                              │ │
│  │      int value = info->si_value.sival_int;                      │ │
│  │      pid_t sender = info->si_pid;                               │ │
│  │      // ...                                                     │ │
│  │  }                                                              │ │
│  │                                                               │ │
│  │  // SA_SIGINFO 플래그 필수                                       │ │
│  │  struct sigaction sa;                                           │ │
│  │  sa.sa_sigaction = handler;                                     │ │
│  │  sa.sa_flags = SA_SIGINFO;                                      │ │
│  │  sigaction(SIGRTMIN, &sa, NULL);                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【큐 크기 제한】                                                         │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • RLIMIT_SIGPENDING: 대기 중인 시그널 최대 개수                   │ │
│  │  • /proc/sys/kernel/rtsig-max                                   │ │
│  │  • 초과 시 EAGAIN 에러                                           │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 신호 손실 없음            • 큐 크기 제한                      │ │
│  │  • 데이터 전달              • 복잡한 핸들러                       │ │
│  │  • 우선순위 보장            • 표준 시그널과 혼용 주의              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【실시간 시그널 송수신 예시】                                           │
│  ──────────────────                                                  │
│  #include <signal.h>                                                │
│  #include <stdio.h>                                                 │
│  #include <unistd.h>                                                │
│                                                                     │
│  volatile int received = 0;                                         │
│                                                                     │
│  void rt_handler(int sig, siginfo_t *info, void *ctx) {             │
│      printf("시그널 %d, 데이터: %d, 송신자: %d\n",                     │ │
│             sig, info->si_value.sival_int, info->si_pid);           │
│      received++;                                                    │
│  }                                                                  │
│                                                                     │
│  int main() {                                                        │
│      struct sigaction sa;                                           │
│      sa.sa_sigaction = rt_handler;                                  │
│      sigemptyset(&sa.sa_mask);                                      │
│      sa.sa_flags = SA_SIGINFO;                                      │
│      sigaction(SIGRTMIN, &sa, NULL);                                │
│                                                                     │
│      pid_t pid = fork();                                            │
│      if (pid == 0) {                                                │
│          // 자식: 실시간 시그널 3회 전송                                │
│          union sigval sv;                                           │
│          for (int i = 0; i < 3; i++) {                              │
│              sv.sival_int = i;                                      │
│              sigqueue(getppid(), SIGRTMIN, sv);                     │
│          }                                                          │
│          _exit(0);                                                  │
│      } else {                                                        │
│          // 부모: 대기                                                │
│          while (received < 3)                                       │
│              pause();                                               │
│          printf("총 %d회 수신\n", received);                          │
│      }                                                              │
│      return 0;                                                      │
│  }                                                                  │
│                                                                     │
│  【시그널 큐 제한 확인】                                                 │
│  ──────────────────                                                  │
│  # shell에서                                                         │
│  $ ulimit -i               # 대기 시그널 최대                          │
│  $ cat /proc/sys/kernel/rtsig-max                                    │
│  $ cat /proc/sys/kernel/rtsig-nr                                     │
│                                                                     │
│  【C에서 제한 확인/설정】                                               │
│  ──────────────────                                                  │
│  struct rlimit rl;                                                  │
│  getrlimit(RLIMIT_SIGPENDING, &rl);                                 │
│  printf("현재: %lu, 최대: %lu\n", rl.rlim_cur, rl.rlim_max);         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 큐잉되는 확장 시그널
• 범위: SIGRTMIN ~ SIGRTMAX
• 특징: 손실 없음, 데이터 전달
• API: sigqueue, siginfo_t
• 핸들러: SA_SIGINFO 필수
• 제한: RLIMIT_SIGPENDING
```
