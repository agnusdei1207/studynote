# 151. 시그널 마스크 (Signal Mask)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 차단할 시그널 집합
> 2. **가치**: 임계 영역 보호, 시그널 제어
> 3. **융합**: 시그널, 핸들러, sigprocmask와 연관

---

## Ⅰ. 개요

### 개념 정의

시그널 마스크(Signal Mask)는 **현재 차단(블록)된 시그널들의 집합**이다. 마스크된 시그널은 전달되지 않고 대기(pending) 상태가 된다.

### 💡 비유: 방해 금지 모드
시그널 마스크는 **스마트폰의 방해 금지 모드**와 같다. 특정 알림(시그널)을 차단하고, 나중에 확인한다.

### 시그널 마스크 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                시그널 마스크 구조                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【시그널 마스크 동작】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  시그널 발생                 시그널 마스크                     │ │
│  │       │                     ┌───────────┐                    │ │
│  │       │                     │ SIGINT  ○│ ← 블록됨            │ │
│  │       ▼                     │ SIGTERM ●│ ← 허용됨            │ │
│  │  ┌──────────┐               │ SIGUSR1 ○│ ← 블록됨            │ │
│  │  │ SIGINT   │──────────────►│   ...    │                     │ │
│  │  └──────────┘               └───────────┘                    │ │
│  │       │                           │                          │ │
│  │       │ 블록됨                      │                          │ │
│  │       ▼                           ▼ 허용됨                    │ │
│  │  ┌──────────┐               ┌───────────┐                    │ │
│  │  │ Pending  │               │ Handler   │                    │ │
│  │  │ 대기     │               │ 즉시 실행 │                    │ │
│  │  └──────────┘               └───────────┘                    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【sigset_t 조작 함수】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  함수                      설명                                │ │
│  │  ────                        ────                                │ │
│  │  sigemptyset(set)       집합 초기화 (비움)                      │ │
│  │  sigfillset(set)        모든 시그널 추가                        │ │
│  │  sigaddset(set, sig)    시그널 추가                            │ │
│  │  sigdelset(set, sig)    시그널 제거                            │ │
│  │  sigismember(set, sig)  멤버 확인                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【sigprocmask() 함수】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  int sigprocmask(int how, const sigset_t *set,                  │ │
│  │                  sigset_t *oldset);                             │ │
│  │                                                               │ │
│  │  how:                                                           │ │
│  │  • SIG_BLOCK: set의 시그널을 블록에 추가                         │ │
│  │  • SIG_UNBLOCK: set의 시그널을 블록에서 제거                      │ │
│  │  • SIG_SETMASK: 블록 집합을 set으로 교체                         │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                시그널 마스크 상세                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【블록 불가 시그널】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • SIGKILL (9): 강제 종료                                       │ │
│  │  • SIGSTOP (19): 일시 중단                                       │ │
│  │  • sigprocmask에 넣어도 무시됨                                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【핸들러 실행 중 자동 블록】                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • 시그널 핸들러 실행 중 해당 시그널은 자동 블록                    │ │
│  │  • sa_mask에 추가 시그널 지정 가능                               │ │
│  │  • SA_NODEFER 플래그로 비활성화 가능                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【임계 영역 보호】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  sigset_t oldmask, blockmask;                                   │ │
│  │  sigemptyset(&blockmask);                                       │ │
│  │  sigaddset(&blockmask, SIGINT);                                 │ │
│  │                                                               │ │
│  │  // 임계 영역 시작                                                │ │
│  │  sigprocmask(SIG_BLOCK, &blockmask, &oldmask);                  │ │
│  │                                                               │ │
│  │  // 임계 영역 - SIGINT 차단됨                                     │ │
│  │  ... critical code ...                                          │ │
│  │                                                               │ │
│  │  // 임계 영역 종료 - 원래 마스크 복원                               │ │
│  │  sigprocmask(SIG_SETMASK, &oldmask, NULL);                      │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【대기 중인 시그널 확인】                                               │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  int sigpending(sigset_t *set);                                 │ │
│  │  // 현재 대기 중인 시그널 집합 반환                               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  장점                      단점                                │ │
│  │  ────                        ────                                │ │
│  │  • 임계 영역 보호            • SIGKILL/SIGSTOP 불가              │ │
│  │  • 핸들러 중복 실행 방지       • 신중한 복원 필요                  │ │
│  │  • 유연한 제어                                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 사용법】                                                        │
│  ──────────────────                                                  │
│  #include <signal.h>                                                │
│  #include <stdio.h>                                                 │
│                                                                     │
│  int main() {                                                        │
│      sigset_t set, oldset;                                          │
│                                                                     │
│      // 시그널 집합 초기화                                             │
│      sigemptyset(&set);                                             │
│      sigaddset(&set, SIGINT);                                       │
│      sigaddset(&set, SIGTERM);                                      │
│                                                                     │
│      // 시그널 블록                                                   │
│      sigprocmask(SIG_BLOCK, &set, &oldset);                         │
│      printf("SIGINT, SIGTERM 블록됨\n");                              │
│                                                                     │
│      // 대기 중인 시그널 확인                                          │
│      sigset_t pending;                                              │
│      sigpending(&pending);                                          │
│      if (sigismember(&pending, SIGINT))                             │
│          printf("SIGINT 대기 중\n");                                   │
│                                                                     │
│      // 원래 마스크 복원                                               │
│      sigprocmask(SIG_SETMASK, &oldset, NULL);                       │
│                                                                     │
│      return 0;                                                       │
│  }                                                                  │
│                                                                     │
│  【임계 영역 보호 패턴】                                                 │
│  ──────────────────                                                  │
│  void critical_section() {                                          │
│      sigset_t block, old;                                           │
│      sigfillset(&block);        // 모든 시그널                       │
│      sigdelset(&block, SIGKILL); // 불가 시그널 제외                   │
│      sigdelset(&block, SIGSTOP);                                    │
│                                                                     │
│      sigprocmask(SIG_BLOCK, &block, &old);                          │
│                                                                     │
│      // 모든 시그널 차단된 임계 영역                                     │
│      ...                                                             │
│                                                                     │
│      sigprocmask(SIG_SETMASK, &old, NULL);                          │
│  }                                                                  │
│                                                                     │
│  【sigsuspend로 대기】                                                  │
│  ──────────────────                                                  │
│  // 마스크 교체 + 대기를 원자적으로                                      │
│  int sigsuspend(const sigset_t *mask);                              │
│                                                                     │
│  // 핸들러에서 플래그 설정 시 사용                                       │
│  volatile sig_atomic_t flag = 0;                                    │
│  sigset_t mask;                                                     │
│  sigfillset(&mask);                                                 │
│  sigdelset(&mask, SIGUSR1);                                         │
│  while (!flag)                                                      │
│      sigsuspend(&mask);  // SIGUSR1만 허용하고 대기                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 차단된 시그널 집합
• 조작: sigemptyset, sigaddset, sigdelset
• 설정: sigprocmask (BLOCK/UNBLOCK/SETMASK)
• 확인: sigpending
• 대기: sigsuspend
• 제외: SIGKILL, SIGSTOP 블록 불가
```
