# 134. D-Bus

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 리눅스 데스크톱 환경의 IPC 메시지 버스 시스템
> 2. **가치**: 프로세스 간 느슨한 결합, 이벤트 기반 통신
> 3. **융합**: IPC, RPC, 이벤트 시스템과 연관

---

## Ⅰ. 개요

### 개념 정의

D-Bus는 **리눅스 데스크톱 환경에서 프로세스 간 통신을 위한 메시지 버스 시스템**이다. 애플리케이션 간 느슨한 결합으로 이벤트와 메서드 호출을 지원한다.

### 💡 비유: 공지판
D-Bus는 **공지판**과 같다. 누구나 공지를 올리고(시그널), 질문하고(메서드 호출), 답을 받는다.

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【D-Bus 명령줄 도구】                                                  │
│  ──────────────────                                                  │
│  # 버스 목록 확인                                                     │
│  $ dbus-send --session --print-reply \                               │
│      --dest=org.freedesktop.DBus \                                   │
│      /org/freedesktop/DBus \                                         │
│      org.freedesktop.DBus.ListNames                                  │
│                                                                     │
│  # 시스템 버스 서비스 목록                                              │
│  $ busctl --system list                                              │
│                                                                     │
│  # 인트로스펙션                                                        │
│  $ gdbus introspect --session \                                      │
│      --dest org.freedesktop.DBus \                                   │
│      --object-path /org/freedesktop/DBus                             │
│                                                                     │
│  # 메서드 호출                                                        │
│  $ dbus-send --session --print-reply \                               │
│      --dest=org.example.Service \                                    │
│      /org/example/Object \                                           │
│      org.example.Interface.Method string:"argument"                  │
│                                                                     │
│  # 시그널 모니터링                                                     │
│  $ dbus-monitor --session                                            │
│                                                                     │
│  【Qt D-Bus 사용 예시】                                                 │
│  ──────────────────                                                  │
│  // 메서드 호출                                                       │
│  QDBusInterface iface("org.example.Service",                         │
│                       "/org/example/Object",                         │
│                       "org.example.Interface");                      │
│  QDBusReply<QString> reply = iface.call("Method", arg);              │
│                                                                     │
│  // 시그널 연결                                                        │
│  QDBusConnection::sessionBus().connect(                              │
│      "org.example.Service",                                          │
│      "/org/example/Object",                                          │
│      "org.example.Interface",                                        │
│      "Signal",                                                       │
│      this, SLOT(onSignal(QString)));                                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
