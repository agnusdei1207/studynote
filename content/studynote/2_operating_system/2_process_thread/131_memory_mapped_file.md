# 131. 메모리 맵 파일 (Memory-Mapped File)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일을 메모리처럼 직접 접근하는 메커니즘
> 2. **가치**: 빠른 I/O, 파일 공유, IPC
> 3. **융합**: mmap, 가상 메모리, 페이지 캐시와 연관

---

## Ⅰ. 개요

### 개념 정의

메모리 맵 파일(Memory-Mapped File)은 **파일을 프로세스의 가상 메모리 공간에 매핑하여 메모리처럼 직접 접근**하는 메커니즘이다. read/write 대신 포인터로 파일에 접근한다.

### 💡 비유: 큰 화이트보드
메모리 맵 파일은 **큰 화이트보드**와 같다. 화이트보드(파일)를 책상 앞에 걸어두고(매핑), 필요한 부분을 바로 보고 쓴다.

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【파일 매핑 예시】                                                      │
│  ──────────────────                                                  │
│  int fd = open("file.txt", O_RDWR);                                 │
│  struct stat st;                                                    │
│  fstat(fd, &st);                                                    │
│                                                                     │
│  // 파일을 메모리에 매핑                                               │
│  char *data = mmap(NULL, st.st_size,                               │
│                     PROT_READ | PROT_WRITE,                         │
│                     MAP_SHARED, fd, 0);                             │
│                                                                     │
│  // 포인터로 직접 접근                                                │
│  data[0] = 'H';  // 첫 바이트 수정                                   │
│  printf("%s\n", data);                                              │
│                                                                     │
│  // 변경사항 동기화                                                   │
│  msync(data, st.st_size, MS_SYNC);                                  │
│                                                                     │
│  // 매핑 해제                                                        │
│  munmap(data, st.st_size);                                          │
│  close(fd);                                                         │
│                                                                     │
│  【익명 매핑 (공유 메모리)】                                             │
│  ──────────────────                                                  │
│  void *shared = mmap(NULL, 4096,                                   │
│                       PROT_READ | PROT_WRITE,                        │
│                       MAP_SHARED | MAP_ANONYMOUS,                    │
│                       -1, 0);                                        │
│                                                                     │
│  // fork 후 자식과 공유                                               │
│  if (fork() == 0) {                                                 │
│      strcpy(shared, "Hello from child");                            │
│  } else {                                                           │
│      wait(NULL);                                                    │
│      printf("%s\n", shared);  // "Hello from child"                 │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
