# 133. POSIX IPC

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: POSIX 표준이 정의한 IPC 메커니즘
> 2. **가치**: 파일 시스템 기반, 간단한 API
> 3. **융합**: System V IPC, 메시지 큐, 세마포어와 연관

---

## Ⅰ. 개요

### 개념 정의

POSIX IPC는 **POSIX 표준이 정의한 IPC 메커니즘**이다. 메시지 큐, 세마포어, 공유 메모리를 포함하며, 파일 시스템 기반 이름을 사용한다.

### 💡 비유: 이름표가 있는 사물함
POSIX IPC는 **이름표가 있는 사물함**과 같다. 사물함 이름(/my_queue)으로 쉽게 찾고 사용한다.

│  │                                                             │ │

│  │  • mmap()으로 주소 공간에 매핑                                   │ │

│  │  ────                        ────                                │ │
│  │  • 간단한 API              • 일부 시스템 미지원                  │ │
│  │  • 자동 정리               • 성능 차이                          │ │
│  │  • 이름 기반               • 구현 의존                          │ │

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【POSIX 메시지 큐】                                                    │
│  ──────────────────                                                  │
│  #include <mqueue.h>                                                │
│                                                                     │
│  // 메시지 큐 생성                                                    │
│  mqd_t mq = mq_open("/my_queue", O_CREAT | O_RDWR, 0644, NULL);     │
│                                                                     │
│  // 메시지 송신                                                       │
│  char msg[] = "Hello";                                              │
│  mq_send(mq, msg, strlen(msg) + 1, 0);  // 우선순위 0                │
│                                                                     │
│  // 메시지 수신                                                       │
│  char buf[256];                                                     │
│  unsigned int prio;                                                 │
│  ssize_t len = mq_receive(mq, buf, 256, &prio);                     │
│                                                                     │
│  mq_close(mq);                                                      │
│  mq_unlink("/my_queue");                                            │
│                                                                     │
│  【POSIX 세마포어】                                                     │
│  ──────────────────                                                  │
│  #include <semaphore.h>                                             │
│                                                                     │
│  // 유명 세마포어                                                     │
│  sem_t *sem = sem_open("/my_sem", O_CREAT, 0644, 1);                │
│  sem_wait(sem);     // P 연산                                        │
│  // 임계 영역                                                        │
│  sem_post(sem);     // V 연산                                        │
│  sem_close(sem);                                                    │
│  sem_unlink("/my_sem");                                             │
│                                                                     │
│  // 무명 세마포어                                                     │
│  sem_t sem;                                                         │
│  sem_init(&sem, 1, 1);  // pshared=1 (프로세스 간 공유)               │
│  sem_wait(&sem);                                                    │
│  sem_post(&sem);                                                    │
│  sem_destroy(&sem);                                                 │
│                                                                     │
│  【POSIX 공유 메모리】                                                  │
│  ──────────────────                                                  │
│  #include <sys/mman.h>                                              │
│  #include <fcntl.h>                                                 │
│                                                                     │
│  // 공유 메모리 생성                                                   │
│  int fd = shm_open("/my_shm", O_CREAT | O_RDWR, 0644);              │
│  ftruncate(fd, 4096);                                               │
│                                                                     │
│  // 매핑                                                             │
│  void *ptr = mmap(NULL, 4096, PROT_READ | PROT_WRITE,               │
│                   MAP_SHARED, fd, 0);                               │
│                                                                     │
│  // 사용                                                             │
│  strcpy(ptr, "Hello");                                              │
│                                                                     │
│  // 정리                                                             │
│  munmap(ptr, 4096);                                                 │
│  close(fd);                                                         │
│  shm_unlink("/my_shm");                                             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
