# 549. HID (Human Interface Device)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 사용자 입력 장치 표준 프로토콜
> 2. **가치**: 플러그 앤 플레이, 드라이버 자동 인식
> 3. **융합**: USB, 블루투스, 입력 장치와 연관

---

## Ⅰ. 개요

### 개념 정의
**HID(Human Interface Device)**는 **키보드, 마우스 등 사용자 입력 장치를 위한 표준 프로토콜**입니다.

### 💡 비유: 만능 리모컨
HID는 **만능 리모컨**과 같습니다. 어떤 장치든 같은 방식으로 제어합니다.

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                HID 상세                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【HID 보고서 구조】                                                 │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  보고서 타입         설명                    방향                  │ │   │
│  │  ────────            ────                    ────                  │ │   │
│  │  Input Report       장치 → 호스트            입력                  │ │   │
│  │  Output Report      호스트 → 장치            출력 (LED 등)         │ │   │
│  │  Feature Report     양방향                   설정                  │ │   │
│  │                                                             │ │   │
│  │  입력 보고서 예시 (마우스):                                     │ │   │
│  │  ┌────┬────┬────┬────┬────┐                                  │ │   │
│  │  │버튼│ X  │ Y  │휠  │예약│                                   │ │   │
│  │  │1B  │1B  │1B  │1B  │1B │                                   │ │   │
│  │  └────┴────┴────┴────┴────┘                                  │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【HID Usage Page】                                                  │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  Usage Page         코드              장치 예시                  │ │   │
│  │  ──────────         ────              ────────                   │ │   │
│  │  Generic Desktop    0x01             키보드, 마우스              │ │   │
│  │  Simulation         0x02             조이스틱, 핸들              │ │   │
│  │  VR                 0x03             VR 컨트롤러                 │ │   │
│  │  Sport              0x04             스포츠 장비                 │ │   │
│  │  Game               0x05             게임패드                   │ │   │
│  │  Generic Device     0x06             일반 장치                  │ │   │
│  │  Keyboard           0x07             키보드                     │ │   │
│  │  LED                0x08             LED 표시                   │ │   │
│  │  Button             0x09             버튼                       │ │   │
│  │  Ordinal            0x0A             순서                       │ │   │
│  │  Telephony          0x0B             전화                       │ │   │
│  │  Consumer           0x0C             멀티미디어 키               │ │   │
│  │  Digitizer          0x0D             터치, 태블릿                │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【HID 설명자 (Descriptor)】                                         │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  struct hid_device {                                            │ │   │
│  │      __u16 bustype;      // BUS_USB, BUS_BLUETOOTH             │ │   │
│  │      __u16 vendor;       // 제조사 ID                          │ │   │
│  │      __u16 product;      // 제품 ID                            │ │   │
│  │      __u16 version;      // 버전                               │ │   │
│  │      char name[128];     // 장치 이름                          │ │   │
│  │      char phys[64];      // 물리적 경로                        │ │   │
│  │      char uniq[64];      // 고유 식별자                        │ │   │
│  │  };                                                             │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

│                실무 적용                                            │

│  【Linux HID 장치 확인】                                             │

│  // HID 장치 목록                                                     │
│  $ ls /dev/hidraw*                                                  │
│  /dev/hidraw0  /dev/hidraw1  /dev/hidraw2                           │
│                                                                     │
│  // 입력 장치 목록                                                    │
│  $ ls /dev/input/                                                   │
│  event0  event1  event2  mouse0  mice                               │
│                                                                     │
│  // 장치 정보                                                         │
│  $ cat /proc/bus/input/devices                                      │

│  N: Name="Logitech USB Receiver"                                    │
│  P: Phys=usb-0000:00:1a.0-1.2/input0                                │
│                                                                     │
│  【evtest (입력 이벤트 테스트)】                                       │
│  ──────────────────                                                │
│  $ sudo apt install evtest                                          │
│  $ sudo evtest /dev/input/event0                                    │
│  Input driver version is 1.0.1                                      │
│  Input device ID: bus 0x3 vendor 0x46d product 0xc52b version 0x11  │
│  Input device name: "Logitech USB Receiver"                         │
│  Supported events:                                                  │
│    Event type 0 (EV_SYN)                                            │
│    Event type 1 (EV_KEY)                                            │
│    Event type 2 (EV_REL)                                            │
│                                                                     │
│  // 이벤트 캡처                                                       │
│  Event: time 1234567890.123456, type 1 (EV_KEY), code 30 (KEY_A),   │
│         value 1                                                     │
│  Event: time 1234567890.125000, type 1 (EV_KEY), code 30 (KEY_A),   │
│         value 0                                                     │
│                                                                     │
│  【libinput】                                                        │
│  ──────────────────                                                │
│  // 장치 목록                                                         │
│  $ libinput list-devices                                            │
│  Device:           Logitech USB Receiver                            │
│  Kernel:           /dev/input/event0                                │
│  Capabilities:     keyboard                                         │
│                                                                     │
│  // debug-events                                                     │
│  $ sudo libinput debug-events                                       │
│  -event10  KEYBOARD_KEY    +1.2s    KEY_A (30) pressed              │
│  -event10  KEYBOARD_KEY    +1.3s    KEY_A (30) released             │
│                                                                     │
│  【Python HID 프로그래밍】                                            │
│  ──────────────────                                                │
│  import hid                                                         │
│                                                                     │
│  # 장치 열기                                                         │
│  device = hid.device()                                              │
│  device.open(0x046d, 0xc52b)  # vendor, product                     │
│                                                                     │
│  # 장치 정보                                                         │
│  print(f"Manufacturer: {device.get_manufacturer_string()}")         │
│  print(f"Product: {device.get_product_string()}")                   │
│                                                                     │
│  # 데이터 읽기                                                       │
│  while True:                                                        │
│      data = device.read(64, timeout_ms=1000)                        │
│      if data:                                                       │
│          print(f"Received: {data}")                                 │
│                                                                     │
│  # 장치 닫기                                                         │
│  device.close()                                                     │
│                                                                     │
│  【udev 규칙 (HID 권한)】                                             │
│  ──────────────────                                                │
│  /etc/udev/rules.d/99-hid.rules:                                   │
│  SUBSYSTEM=="hidraw", ATTRS{idVendor}=="046d", \                    │
│      ATTRS{idProduct}=="c52b", MODE="0666"                          │
│                                                                     │
│  // 규칙 적용                                                        │
│  $ sudo udevadm control --reload-rules                              │
│  $ sudo udevadm trigger                                             │
│                                                                     │
│  【입력 장치 모니터링】                                               │
│  ──────────────────                                                │
│  // input-tools                                                      │
│  $ sudo input-kbd -f /dev/input/event0                              │
│                                                                     │
│  // hid-recorder                                                     │
│  $ sudo hid-recorder /dev/hidraw0                                   │
│                                                                     │
│  // 커널 메시지                                                       │
│  $ dmesg -w | grep -i hid                                           │
│  hid-generic 0003:046D:C52B.0001: input,hidraw0: USB HID v1.11      │
│                                                                     │

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 사용자 입력 장치 표준 프로토콜
• 장치: 키보드, 마우스, 게임패드, 터치
• 보고서: Input, Output, Feature
• 전송: USB, Bluetooth, I2C, SPI
• Usage Page: 장치 분류 체계
• 지연: 1-30 ms
• Linux: /dev/hidraw*, /dev/input/
• 도구: evtest, libinput, hid-api
• 드라이버: 표준 HID 드라이버 자동 로드
• PnP: 플러그 앤 플레이 지원
```
