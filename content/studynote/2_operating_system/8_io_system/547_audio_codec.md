# 547. 오디오 코덱 (Audio Codec)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 오디오 압축 및 해제 알고리즘
> 2. **가치**: 대역폭 절약, 저장 공간 최적화
> 3. **융합**: 인코딩, 디코딩, 트랜스코딩과 연관

---

## Ⅰ. 개요

### 개념 정의
**오디오 코덱(Codec)**은 **오디오 데이터를 압축(인코딩)하고 복원(디코딩)하는 알고리즘**입니다.

### 💡 비유: 소리 압축기
오디오 코덱은 **소리 압축기**와 같습니다. 큰 소리 파일을 작게 만들고 다시 원래대로 풉니다.

### 오디오 코덱 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                오디오 코덱 구조                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【인코딩/디코딩 과정】                                               │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  [원본 오디오]                                                │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  ┌─────────────┐                                            │ │   │
│  │  │   인코더    │  → 압축 (무손실/손실)                        │ │   │
│  │  └─────────────┘                                            │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  [압축 오디오] (전송/저장)                                     │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  ┌─────────────┐                                            │ │   │
│  │  │   디코더    │  → 복원                                     │ │   │
│  │  └─────────────┘                                            │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  [재생 오디오]                                                │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【무손실 vs 손실 코덱】                                             │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  구분              무손실                  손실                   │ │   │
│  │  ────              ────                  ────                   │ │   │
│  │  압축률            30-70%                80-95%                  │ │   │
│  │  품질 보존         완벽                  일부 손실                │ │   │
│  │  예시              FLAC, ALAC            MP3, AAC                │ │   │
│  │  용도              아카이브               스트리밍                 │ │   │
│  │  파일 크기         큼                    작음                    │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【주요 코덱 비교】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  코덱             비트레이트       지연          용도               │ │   │
│  │  ────             ────────       ────          ────               │ │   │
│  │  MP3              128-320 kbps   ~100ms        음악               │ │   │
│  │  AAC              96-320 kbps    ~50ms         스트리밍            │ │   │
│  │  Opus             6-510 kbps     5-60ms        VoIP, 스트리밍      │ │   │
│  │  FLAC             800-1000 kbps  ~50ms         아카이브            │ │   │
│  │  ALAC             800-1000 kbps  ~50ms         Apple              │ │   │
│  │  Vorbis           64-500 kbps    ~100ms        게임, 웹            │ │   │
│  │  LAME (MP3)       128-320 kbps   ~100ms        고품질 MP3          │ │   │
│  │  LDAC             330-990 kbps   ~30ms         블루투스           │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                오디오 코덱 상세                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【손실 압축 기법】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  기법                 설명                                       │ │   │

│  │  지각 코딩            사람 귀에 들리지 않는 정보 제거                │ │   │
│  │  마스킹 효과          큰 소리 근처 작은 소리 제거                   │ │   │
│  │  MDCT                주파수 도메인 변환                           │ │   │
│  │  양자화              정밀도 낮춤                                   │ │   │
│  │  허프만 코딩          자주 나오는 값 짧게 인코딩                    │ │   │
│  │  비트 할당            중요 주파수에 더 많은 비트                    │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【MP3 인코딩 과정】                                                  │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  PCM 입력                                                    │ │   │
│  │     │                                                       │ │   │
│  │     ▼                                                       │ │   │
│  │  [분석 필터뱅크] → 32개 서브밴드                              │ │   │
│  │     │                                                       │ │   │
│  │     ▼                                                       │ │   │
│  │  [MDCT] → 주파수 변환                                        │ │   │
│  │     │                                                       │ │   │
│  │     ▼                                                       │ │   │
│  │  [지각 모델] → 마스킹 임계값 계산                              │ │   │
│  │     │                                                       │ │   │
│  │     ▼                                                       │ │   │
│  │  [양자화 + 스케일] → 비트 할당                                │ │   │
│  │     │                                                       │ │   │
│  │     ▼                                                       │ │   │
│  │  [허프만 코딩] → 엔트로피 코딩                                │ │   │
│  │     │                                                       │ │   │
│  │     ▼                                                       │ │   │
│  │  MP3 비트스트림 출력                                          │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【Opus 코덱 모드】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  모드               비트레이트          용도                     │ │   │
│  │  ────               ────────          ────                     │ │   │
│  │  SILK               6-40 kbps         음성 (저비트레이트)        │ │   │
│  │  Hybrid             40-60 kbps        음성 + 음악               │ │   │
│  │  CELT               60-510 kbps       음악 (고비트레이트)        │ │   │
│  │                                                             │ │   │
│  │  Opus는 자동으로 최적 모드 선택                                │ │   │
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【FFmpeg 코덱 사용】                                                │
│  ──────────────────                                                │
│  // 지원 코덱 확인                                                    │
│  $ ffmpeg -codecs | grep -E "mp3|aac|opus|flac"                     │
│  DEA.L. aac                  AAC (Advanced Audio Coding)            │
│  DEA.L. mp3                  MP3 (MPEG audio layer 3)              │
│  DEA.L. opus                 Opus (Opus Interactive Audio Codec)   │
│  DEA.L. flac                 FLAC (Free Lossless Audio Codec)      │
│                                                                     │
│  // MP3 인코딩                                                        │
│  $ ffmpeg -i input.wav -c:a libmp3lame -b:a 192k output.mp3         │
│  $ ffmpeg -i input.wav -c:a libmp3lame -q:a 2 output.mp3  // VBR    │
│                                                                     │
│  // AAC 인코딩                                                        │
│  $ ffmpeg -i input.wav -c:a aac -b:a 128k output.m4a                │
│  $ ffmpeg -i input.wav -c:a libfdk_aac -vbr 3 output.m4a            │
│                                                                     │
│  // Opus 인코딩                                                       │
│  $ ffmpeg -i input.wav -c:a libopus -b:a 128k output.opus           │
│  $ ffmpeg -i input.wav -c:a libopus -vbr on -b:a 64k output.opus    │
│                                                                     │
│  // FLAC 무손실 인코딩                                                │
│  $ ffmpeg -i input.wav -c:a flac -compression_level 8 output.flac   │
│                                                                     │
│  【GStreamer】                                                       │
│  ──────────────────                                                │
│  // MP3 재생                                                         │
│  $ gst-launch-1.0 filesrc location=audio.mp3 ! \                   │
│      decodebin ! autoaudiosink                                      │
│                                                                     │
│  // WAV → Opus 변환                                                   │
│  $ gst-launch-1.0 filesrc location=input.wav ! \                    │
│      wavparse ! opusenc ! oggmux ! filesink location=output.opus    │
│                                                                     │
│  // 블루투스 오디오 스트리밍                                          │
│  $ gst-launch-1.0 filesrc location=music.mp3 ! \                    │
│      decodebin ! audioconvert ! avenc_aac ! \                       │
│      rtpmp4gpay ! udpsink host=192.168.1.100 port=5000              │
│                                                                     │
│  【Linux 사운드 서버 코덱 설정】                                       │
│  ──────────────────                                                │
│  // PulseAudio 코덱                                                  │
│  $ pactl list cards                                                 │
│  $ pactl set-card-profile bluez_card.XX a2dp-sink                   │
│                                                                     │
│  // PipeWire 코덱                                                    │
│  $ wpctl status                                                     │
│  $ pipewire -v  // 지원 코덱 확인                                    │
│                                                                     │
│  【Python 오디오 처리】                                               │
│  ──────────────────                                                │
│  import pydub                                                       │
│  from pydub import AudioSegment                                     │
│                                                                     │
│  # 파일 로드                                                         │
│  audio = AudioSegment.from_wav("input.wav")                         │
│                                                                     │
│  # MP3로 인코딩                                                      │
│  audio.export("output.mp3", format="mp3", bitrate="192k")           │
│                                                                     │
│  # AAC로 인코딩                                                      │
│  audio.export("output.m4a", format="adts", bitrate="128k")          │
│                                                                     │
│  # Opus로 인코딩                                                     │
│  audio.export("output.opus", format="opus", bitrate="128k")         │
│                                                                     │
│  # FLAC으로 인코딩                                                   │
│  audio.export("output.flac", format="flac")                         │
│                                                                     │
│  【스트리밍 서버 코덱 설정】                                           │
│  ──────────────────                                                │
│  # Icecast 설정 (ogg/vorbis)                                        │
│  $ darkice -c /etc/darkice.cfg                                      │
│                                                                     │
│  # HLS 스트리밍                                                      │
│  $ ffmpeg -i input.wav -c:a aac -b:a 128k \                         │
│    -f hls -hls_time 10 -hls_list_size 5 playlist.m3u8              │
│                                                                     │
│  # DASH 스트리밍                                                      │
│  $ ffmpeg -i input.wav -c:a libopus -b:a 128k \                     │
│    -f dash -seg_duration 10 manifest.mpd                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 오디오 압축/해제 알고리즘
• 분류: 무손실(FLAC, ALAC), 손실(MP3, AAC, Opus)
• 압축 기법: 지각 코딩, 마스킹, MDCT, 양자화
• MP3: 128-320 kbps, 보편적
• AAC: 96-320 kbps, 고품질 스트리밍
• Opus: 6-510 kbps, 저지연, VoIP
• FLAC: 무손실, 30-70% 압축
• 지각 코딩: 사람 귀에 안 들리는 정보 제거
• 도구: FFmpeg, GStreamer, pydub
• 용도: 음악, VoIP, 스트리밍, 아카이브
```
