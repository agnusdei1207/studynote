# 560. 1-Wire

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단일 선 기반 저속 직렬 버스
> 2. **가치**: 최소 배선, 기생 전력 지원
> 3. **융합**: 온도 센서, EEPROM, iButton과 연관

---

## Ⅰ. 개요

### 개념 정의
**1-Wire**는 **단일 데이터 선으로 통신과 전력을 동시에 전달하는 버스**입니다.

### 💡 비유: 단일 전화선
1-Wire는 **단일 전화선**과 같습니다. 한 선으로 여러 사람이 통화합니다.

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                1-Wire 상세                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【신호 타이밍】                                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  Reset Pulse:                                                   │ │   │
│  │  마스터: ──┐                               ┌──────────────     │ │   │
│  │            └──────── 480us ────────────────┘                   │ │   │
│  │                                                             │ │   │
│  │  Presence Pulse (슬레이브 응답):                                │ │   │
│  │            ┌──────── 60-240us ─────────────┐                   │ │   │
│  │  ──────────┘                              └──────────────     │ │   │
│  │                                                             │ │   │
│  │  Write 0:                                                       │ │   │
│  │  ─────┐                                           ┌────────  │ │   │
│  │       └──────── 60us (Low) ────────────────────────┘           │ │   │
│  │                                                             │ │   │
│  │  Write 1:                                                       │ │   │
│  │  ─────┐   ┌───────────────────────────────────────────────    │ │   │
│  │       └───┘ <15us (Low)                                        │ │   │
│  │                                                             │ │   │
│  │  Read:                                                          │ │   │
│  │  ─────┐   ┌───────────────────────────────────────────────    │ │   │
│  │       └───┘ <15us → 샘플링 (슬레이브가 유지)                    │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【ROM 명령어】                                                       │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  명령어            코드              설명                        │ │   │
│  │  ────              ────              ────                        │ │   │
│  │  SEARCH ROM        0xF0             장치 검색                   │ │   │
│  │  READ ROM          0x33             단일 장치 주소 읽기          │ │   │
│  │  MATCH ROM         0x55             특정 장치 선택              │ │   │
│  │  SKIP ROM          0xCC             주소 없이 (단일 장치)        │ │   │
```

```
│  // 온도 읽기 (DS18B20)                                              │
│  $ cat /sys/bus/w1/devices/28-*/w1_slave                           │
│  50 05 4b 46 7f ff 0c 10 1c : crc=1c YES                           │
│  50 05 4b 46 7f ff 0c 10 1c t=85000                               │
│  // t=85000 → 85.000°C                                             │
│                                                                     │
│  // 온도 계산                                                        │
│  // 85000 = 85.000°C                                               │
│  // 23456 = 23.456°C                                               │
│  // -50000 = -50.000°C                                             │
│                                                                     │
│  【owfs (1-Wire File System)】                                       │
│  ──────────────────                                                │
│  // 설치                                                            │
│  $ sudo apt install owfs                                           │
│                                                                     │
│  // 마운트                                                          │
│  $ sudo mkdir /mnt/1wire                                           │
│  $ sudo owfs -d /dev/i2c-1 -m /mnt/1wire  // I2C 1-Wire 어댑터      │
│                                                                     │
│  // 온도 읽기                                                        │
│  $ cat /mnt/1wire/28.0000052E4B81/temperature                      │
│  23.5                                                               │
│                                                                     │
│  // 장치 목록                                                        │
│  $ ls /mnt/1wire/                                                  │
│  28.0000052E4B81  bus.0  settings  structure                       │
│                                                                     │
│  【Python 1-Wire】                                                   │
│  ──────────────────                                                │
│  import os                                                          │
│                                                                     │
│  def read_temp(device_path):                                        │
│      with open(device_path, 'r') as f:                             │
│          lines = f.readlines()                                     │
│                                                                     │
│      # CRC 확인                                                     │
│      while lines[0].strip()[-3:] != 'YES':                         │
│          time.sleep(0.2)                                           │
│          with open(device_path, 'r') as f:                         │
│              lines = f.readlines()                                 │
│                                                                     │
│      # 온도 추출                                                     │
│      temp_pos = lines[1].find('t=')                                │
│      if temp_pos != -1:                                            │

│  base_dir = '/sys/bus/w1/devices/'                                 │

│  device_file = device_folder + '/w1_slave'                         │
│                                                                     │
│  temp = read_temp(device_file)                                     │
│  print(f"Temperature: {temp}°C")                                   │
│                                                                     │
│  【C (w1-gpio)】                                                      │
│  ──────────────────                                                │
│  #include <fcntl.h>                                                 │
│  #include <unistd.h>                                                │
│  #include <dirent.h>                                                │
│  #include <string.h>                                                │
│                                                                     │
│  #define W1_PATH "/sys/bus/w1/devices/"                            │
│                                                                     │
│  int main() {                                                        │
│      DIR *dir = opendir(W1_PATH);                                  │
│      struct dirent *entry;                                          │
│                                                                     │
│      while ((entry = readdir(dir)) != NULL) {                      │
│          if (entry->d_name[0] == '2' && entry->d_name[1] == '8') { │
│              char path[256];                                        │
│              sprintf(path, "%s%s/w1_slave", W1_PATH, entry->d_name);│
│                                                                     │
│              int fd = open(path, O_RDONLY);                        │
│              char buf[256];                                        │
│              read(fd, buf, sizeof(buf));                           │
│              close(fd);                                            │
│                                                                     │
│              char *temp_str = strstr(buf, "t=") + 2;               │
│              int temp = atoi(temp_str);                            │

│      }                                                              │
│      closedir(dir);                                                 │
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 단일 선 기반 저속 직렬 버스
• 선: DQ (데이터+전력) + GND
• 속도: 16 kbps (표준)
• 전력: 기생 전력 지원
• 주소: 64비트 고유 주소
• 거리: ~100m
• 풀업: 4.7kΩ 필수
• 장치: DS18B20, iButton, EEPROM
• ROM 명령: SEARCH, READ, MATCH, SKIP
• Linux: w1-gpio, w1-therm 모듈
• 경로: /sys/bus/w1/devices/
```
