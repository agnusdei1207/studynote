# 513. 버퍼링 (Buffering)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 전송 속도 차이를 완화하는 임시 저장소
> 2. **가치**: 전송 효율 향상과 비동기 처리
> 3. **융합**: 캐싱, 스풀링, DMA와 연관

---

## Ⅰ. 개요

### 개념 정의

버퍼링(Buffering)은 **속도 차이가 있는 장치 간 데이터 전송 시 임시 저장 공간(버퍼)을 사용하여 전송 효율을 높이는 기술**이다.

### 💡 비유: 저수지
버퍼링은 **저수지**와 같다. 물이 흘러오는 속도와 나가는 속도가 달라도 중간에서 조절한다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                버퍼링 상세                                             │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【이중 버퍼링 (Double Buffering)】                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  시간 T1:                                                             │ │
│  │  ┌─────────┐     ┌─────────┐                                        │ │
│  │  │버퍼 A   │     │버퍼 B   │                                        │ │
│  │  │(생성 중)│     │(비움)   │                                        │ │
│  │  └─────────┘     └─────────┘                                        │ │
│  │       ▲                │                                            │ │
│  │       │                ▼                                            │ │
│  │    생산자           소비자                                           │ │
│  │                                                             │ │
│  │  시간 T2:                                                             │ │
│  │  ┌─────────┐     ┌─────────┐                                        │ │
│  │  │버퍼 A   │     │버퍼 B   │                                        │ │
│  │  │(비움)   │     │(생성 중)│                                        │ │
│  │  └─────────┘     └─────────┘                                        │ │
│  │       │                ▲                                            │ │
│  │       ▼                │                                            │ │
│  │    소비자           생산자                                           │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【순환 버퍼 (Circular Buffer)】                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │         ┌───────────────────────────────┐                     │ │
│  │         │     ┌───┬───┬───┬───┬───┬───┐ │                     │ │
│  │    H ───►    │ A │ B │ C │ D │   │   │ │                     │ │
│  │         │     └───┴───┴───┴───┴───┴───┘ │                     │ │
│  │         │       ▲               ▲       │                     │ │
│  │         │       │               │       │                     │ │
│  │         │     Tail            Head      │                     │ │
│  │         └───────────────────────────────┘                     │ │
│  │                                                             │ │
│  │  • Head: 데이터를 꺼내는 위치 (소비자)                                 │ │
│  │  • Tail: 데이터를 넣는 위치 (생산자)                                   │ │
│  │  • 꽉 참: (Tail + 1) % size == Head                                   │ │
│  │  • 비어있음: Head == Tail                                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【버퍼링 vs 캐싱】                                                       │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  구분        버퍼링               캐싱                                   │ │
│  │  ────        ────────             ────                                   │ │
│  │  목적        속도 차이 완화         재사용                                │ │
│  │  데이터      일시적 저장            반복 접근                             │ │
│  │  교체        FIFO                  LRU 등                               │ │
│  │  예시        I/O 버퍼, 네트워크 버퍼  CPU 캐시, 페이지 캐시             │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【버퍼 크기 확인】                                                      │
│  ──────────────────                                                  │
│  // 파일 시스템 버퍼                                                    │
│  $ free -h                                                           │
│                total        used        free      shared  buff/cache │
│  Mem:           7.7G        2.1G        3.2G        256M        2.4G│
│                                                                     │
│  // TCP 버퍼 크기                                                       │
│  $ cat /proc/sys/net/ipv4/tcp_rmem                                  │
│  4096    87380   6291456                                             │
│  $ cat /proc/sys/net/ipv4/tcp_wmem                                  │
│  4096    16384   4194304                                             │
│                                                                     │
│  // 블록 장치 버퍼                                                      │
│  $ cat /sys/block/sda/queue/nr_requests                             │
│  256                                                                 │
│                                                                     │
│  【순환 버퍼 구현】                                                      │
│  ──────────────────                                                  │
│  #include <stdlib.h>                                                 │
│  #include <string.h>                                                 │
│                                                                     │
│  typedef struct {                                                    │
│      char *buffer;                                                   │
│      size_t size;                                                    │
│      size_t head;                                                    │
│      size_t tail;                                                    │
│  } CircularBuffer;                                                   │
│                                                                     │
│  void cb_init(CircularBuffer *cb, size_t size) {                     │
│      cb->buffer = malloc(size);                                      │
│      cb->size = size;                                                │
│      cb->head = 0;                                                   │
│      cb->tail = 0;                                                   │
│  }                                                                   │
│                                                                     │
│  int cb_write(CircularBuffer *cb, const char *data, size_t len) {    │
│      for (size_t i = 0; i < len; i++) {                              │
│          size_t next = (cb->tail + 1) % cb->size;                    │
│          if (next == cb->head) return -1;  // 꽉 참                   │
│          cb->buffer[cb->tail] = data[i];                             │
│          cb->tail = next;                                            │
│      }                                                               │
│      return len;                                                     │
│  }                                                                   │
│                                                                     │
│  int cb_read(CircularBuffer *cb, char *data, size_t len) {           │
│      size_t count = 0;                                               │
│      while (count < len && cb->head != cb->tail) {                   │
│          data[count++] = cb->buffer[cb->head];                       │
│          cb->head = (cb->head + 1) % cb->size;                       │
│      }                                                               │
│      return count;                                                   │
│  }                                                                   │
│                                                                     │
│  【이중 버퍼링 (그래픽)】                                                │
│  ──────────────────                                                  │
│  // 더블 버퍼링 구조                                                     │
│  struct DoubleBuffer {                                               │
│      char *front;  // 화면에 표시 중                                   │
│      char *back;   // 그리기 중                                        │
│      size_t size;                                                    │
│  };                                                                  │
│                                                                     │
│  void swap_buffers(struct DoubleBuffer *db) {                        │
│      char *temp = db->front;                                         │
│      db->front = db->back;                                           │
│      db->back = temp;                                                │
│  }                                                                   │
│                                                                     │
│  【Linux 커널 버퍼】                                                    │
│  ──────────────────                                                  │
│  // kfifo (커널 FIFO 버퍼)                                              │
│  #include <linux/kfifo.h>                                            │
│                                                                     │
│  DECLARE_KFIFO(my_fifo, char, 1024);                                 │
│  INIT_KFIFO(my_fifo);                                                │
│                                                                     │
│  // 쓰기                                                               │
│  kfifo_in(&my_fifo, data, len);                                      │
│                                                                     │
│  // 읽기                                                               │
│  kfifo_out(&my_fifo, buf, len);                                      │
│                                                                     │
│  // 사용량 확인                                                        │
│  size = kfifo_len(&my_fifo);                                         │
│                                                                     │
│  【버퍼 튜닝】                                                           │
│  ──────────────────                                                  │
│  // TCP 버퍼 크기 조정                                                  │
│  $ sysctl -w net.ipv4.tcp_rmem="4096 131072 6291456"                │
│  $ sysctl -w net.ipv4.tcp_wmem="4096 65536 4194304"                 │
│                                                                     │
│  // 블록 장치 read-ahead                                               │
│  $ blockdev --setra 256 /dev/sda   // 256 * 512 = 128KB             │
│  $ blockdev --getra /dev/sda                                         │
│                                                                     │
│  // 드라이버 버퍼 카운                                                   │
│  $ cat /proc/meminfo | grep -i buffer                                │
│  Buffers:          123456 kB                                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 속도 차이 완화를 위한 임시 저장소
• 종류: 단일, 이중, 순환, 다중 버퍼링
• 이중 버퍼링: 교대로 채우고 비움
• 순환 버퍼: Head/Tail 포인터로 관리
• vs 캐싱: 버퍼링(속도차), 캐싱(재사용)
• 확인: free, /proc/sys/net/ipv4/tcp_*
• 튜닝: tcp_rmem, blockdev --setra
• 커널: kfifo API 사용
```
