# 553. 스캐너 (Scanner)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 물리적 문서를 디지털로 변환하는 장치
> 2. **가치**: 광학 인식, 해상도, 컬러 깊이 지원
> 3. **융합**: OCR, 이미지 처리, SANE와 연관

---

## Ⅰ. 개요

### 개념 정의
**스캐너(Scanner)**는 **물리적 문서나 이미지를 디지털로 변환하는 입력 장치**입니다.

### 💡 비유: 디지털 복사기 (입력)
스캐너는 **디지털 복사기의 입력부**와 같습니다. 종이를 디지털로 바꿉니다.

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【SANE (Scanner Access Now Easy)】                                 │
│  ──────────────────                                                │
│  // 스캐너 목록                                                       │
│  $ scanimage -L                                                     │
│  device `hpaio:/usb/HP_LaserJet_MFP?serial=XXX' is a HP ...        │
│                                                                     │
│  // 스캐너 확인                                                       │
│  $ sane-find-scanner                                                │
│  found USB scanner (vendor=0x03f0, product=0x0a2a) at libusb:001:006│
│                                                                     │
│  // 스캔 파라미터 확인                                                │
│  $ scanimage --help -d 'hpaio:/usb/HP_LaserJet_MFP'                 │
│  Options specific to device `hpaio:/usb/HP_LaserJet_MFP':           │
│    --mode Color|Gray|Lineart [Lineart]                              │
│    --resolution 75..1200dpi [75]                                    │
│                                                                     │
│  【이미지 스캔】                                                      │
│  ──────────────────                                                │
│  // 기본 스캔                                                         │
│  $ scanimage -d 'hpaio:/usb/HP_LaserJet_MFP' > scan.pnm             │
│                                                                     │
│  // 해상도 지정                                                       │
│  $ scanimage -d 'hpaio:/usb/HP_LaserJet_MFP' --resolution 300 \     │
│    --mode Color > scan.pnm                                          │
│                                                                     │
│  // 영역 지정                                                         │
│  $ scanimage -d 'hpaio:/usb/HP_LaserJet_MFP' \                      │
│    -x 100 -y 150 --resolution 300 > scan.pnm                        │
│                                                                     │
│  // JPEG으로 변환                                                    │
│  $ scanimage -d 'hpaio:/usb/HP_LaserJet_MFP' --resolution 300 | \   │
│    convert - scan.jpg                                               │
│                                                                     │
│  // PDF 생성                                                         │
│  $ scanimage -d 'hpaio:/usb/HP_LaserJet_MFP' --resolution 300 | \   │
│    convert - scan.pdf                                               │
│                                                                     │
│  【OCR (Tesseract)】                                                 │
│  ──────────────────                                                │
│  // OCR 실행                                                         │
│  $ tesseract scan.jpg output                                        │
│  $ cat output.txt                                                   │
│                                                                     │
│  // 언어 지정                                                         │
│  $ tesseract scan.jpg output -l eng+kor                             │
│                                                                     │
│  // PDF with text layer                                             │
│  $ tesseract scan.jpg output pdf                                    │
│                                                                     │
│  // 배치 처리                                                        │
│  $ for f in *.jpg; do tesseract "$f" "${f%.jpg}"; done              │
│                                                                     │
│  【scanimage 스크립트】                                              │
│  ──────────────────                                                │
│  #!/bin/bash                                                        │
│  DEVICE="hpaio:/usb/HP_LaserJet_MFP"                                │
│  OUTPUT_DIR="$HOME/scan"                                            │
│  TIMESTAMP=$(date +%Y%m%d_%H%M%S)                                   │
│                                                                     │
│  # 컬러 스캔 (300dpi)                                                │
│  scanimage -d "$DEVICE" \                                           │
│    --mode Color \                                                   │
│    --resolution 300 \                                               │
│    --format=tiff > "$OUTPUT_DIR/scan_$TIMESTAMP.tiff"               │
│                                                                     │
│  # OCR 실행                                                         │
│  tesseract "$OUTPUT_DIR/scan_$TIMESTAMP.tiff" \                     │
│    "$OUTPUT_DIR/scan_$TIMESTAMP" -l eng+kor                         │
│                                                                     │
│  echo "Scanned and OCR completed: $OUTPUT_DIR/scan_$TIMESTAMP.txt"  │
│                                                                     │
│  【Python 스캐너 제어】                                               │
│  ──────────────────                                                │
│  from PIL import Image                                              │
│  import pytesseract                                                 │
│  import subprocess                                                  │
│                                                                     │
│  # 스캔 실행                                                         │
│  result = subprocess.run([                                          │
│      'scanimage', '-d', 'hpaio:/usb/HP_LaserJet_MFP',               │
│      '--resolution', '300', '--mode', 'Color'                       │
│  ], capture_output=True)                                            │
│                                                                     │
│  # 이미지 처리                                                       │
│  from io import BytesIO                                             │
│  img = Image.open(BytesIO(result.stdout))                           │
│                                                                     │
│  # OCR 실행                                                         │
│  text = pytesseract.image_to_string(img, lang='eng+kor')            │
│  print(text)                                                        │
│                                                                     │
│  【네트워크 스캐너】                                                  │
│  ──────────────────                                                │
│  // 네트워크 스캐너 추가                                              │
│  $ sudo sane-config --add-device="net:192.168.1.100:hp"             │
│                                                                     │
│  // saned 서버 설정                                                  │
│  /etc/sane.d/saned.conf:                                           │
│  192.168.1.0/24                                                     │
│                                                                     │
│  // saned 시작                                                      │
│  $ sudo systemctl start saned                                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 물리적 문서를 디지털로 변환
• 종류: 플랫베드, 시트피드, 핸드헬드, 드럼
• 구성: 광원 → 센서(CCD/CIS) → ADC → 디지털
• 해상도: 300-6400 dpi (광학)
• 컬러 깊이: 24-48 bit
• OCR: 광학 문자 인식, Tesseract
• 포맷: TIFF, PDF, JPEG, PNG
• SANE: Linux 스캐너 API
• 명령어: scanimage, sane-find-scanner
• 속도: 10-60 ppm
```
