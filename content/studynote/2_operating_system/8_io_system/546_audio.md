# 546. 오디오 (Audio)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 소리의 디지털 변환 및 재생 시스템
> 2. **가치**: ADC/DAC, 샘플링, 믹싱 지원
> 3. **융합**: 사운드 카드, ALSA, PulseAudio와 연관

---

## Ⅰ. 개요

### 개념 정의
**오디오(Audio)**는 **아날로그 소리를 디지털로 변환하고 재생하는 I/O 시스템**입니다.

### 💡 비유: 디지털 녹음기
오디오 시스템은 **디지털 녹음기**와 같습니다. 소리를 숫자로 바꾸고 다시 소리로 만듭니다.

### 오디오 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                오디오 구조                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【오디오 파이프라인】                                               │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  [입력]                    [처리]                    [출력]  │ │   │
│  │                                                             │ │   │
│  │  마이크                    DSP                     스피커    │ │   │
│  │    │                        │                        │      │ │   │
│  │    ▼                        ▼                        ▼      │ │   │
│  │  ┌─────┐    ┌─────┐    ┌─────┐    ┌─────┐    ┌─────┐       │ │   │
│  │  │아날로그│→ │ ADC │→ │디지털│→ │ DAC │→ │아날로그│       │ │   │
│  │  │소리  │    │     │    │처리 │    │     │    │소리  │       │ │   │
│  │  └─────┘    └─────┘    └─────┘    └─────┘    └─────┘       │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【샘플링 파라미터】                                                 │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  파라미터           CD 품질         전문가         용도          │ │   │
│  │  ─────────           ──────         ──────         ────          │ │   │
│  │  샘플레이트         44.1 kHz       96/192 kHz    주파수 범위      │ │   │
│  │  비트 깊이          16-bit         24/32-bit     동적 범위       │ │   │
│  │  채널              2 (스테레오)     5.1/7.1       공간감          │ │   │
│  │  비트레이트         1411 kbps      4608+ kbps     데이터 양       │ │   │
│  │                                                             │ │   │
│  │  비트레이트 = 샘플레이트 × 비트깊이 × 채널                       │ │   │
│  │  예: 44100 × 16 × 2 = 1,411,200 bps = 1411 kbps              │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【오디오 포맷】                                                     │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  포맷             압축          특징                    용도     │ │   │
│  │  ────             ────          ────                    ────     │ │   │
│  │  WAV             무손실        PCM 기본              일반      │ │   │
│  │  FLAC            무손실        압축率高              음악 감상    │ │   │
│  │  MP3             손실          높은 압축률            스트리밍    │ │   │
│  │  AAC             손실          고품질                 스트리밍    │ │   │
│  │  OGG             손실          오픈 소스              게임       │ │   │
│  │  Opus            손실          저지연                 VoIP       │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                오디오 상세                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【ADC/DAC】                                                         │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  ADC (Analog-to-Digital Converter):                           │ │   │
│  │  • 샘플링: 연속 신호 → 이산 샘플                               │ │   │
│  │  • 양자화: 연속 값 → 이산 레벨                                 │ │   │
│  │  • 인코딩: 이진 코드 변환                                     │ │   │
│  │                                                             │ │   │
│  │  DAC (Digital-to-Analog Converter):                           │ │   │
│  │  • 디코딩: 이진 → 샘플 값                                     │ │   │
│  │  • 재구성: 샘플 → 연속 신호                                   │ │   │
│  │  • 필터링: 계단 현상 제거                                     │ │   │
│  │                                                             │ │   │
│  │  나이퀴스트 정리: fs ≥ 2 × fmax                              │ │   │
│  │  (사람 가청 범위 20Hz-20kHz → 44.1kHz 샘플레이트)              │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【오디오 믹싱】                                                     │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  기능              설명                                         │ │   │
│  │  ────              ────                                         │ │   │
│  │  볼륨 제어         PCM, Master, Capture 게인                   │ │   │
│  │  채널 믹싱         스테레오 → 모노, 업믹싱/다운믹싱              │ │   │
│  │  이퀄라이저        주파수 대역별 조절                          │ │   │
│  │  이펙트            리버브, 에코, 컴프레서                       │ │   │
│  │  라우팅            입력 → 출력 매핑                            │ │   │

│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【서라운드 사운드】                                                  │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  포맷              채널              용도                       │ │   │
│  │  ────              ────              ────                       │ │   │
│  │  2.0 (스테레오)     L, R              음악                      │ │   │

│  │  5.1               L,R,C,LS,RS,LFE    홈시어터                   │ │   │
│  │  7.1               L,R,C,LS,RS,LB,RB,LFE  고급 홈시어터          │ │   │

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 소리의 디지털 변환 및 재생
• 파이프라인: 입력 → ADC → DSP → DAC → 출력
• 샘플링: 44.1kHz(CD) ~ 192kHz(전문가)
• 비트 깊이: 16-bit ~ 32-bit
• 채널: 모노, 스테레오, 5.1, 7.1
• ADC: 아날로그 → 디지털 변환
• DAC: 디지털 → 아날로그 변환
• 나이퀴스트: fs ≥ 2 × fmax
• Linux: ALSA, PulseAudio, PipeWire
• 포맷: WAV, FLAC, MP3, AAC, Opus
```
