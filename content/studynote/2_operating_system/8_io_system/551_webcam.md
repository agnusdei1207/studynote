# 551. 웹캠 (Webcam)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 실시간 비디오 캡처 장치
> 2. **가치**: 영상 통화, 녹화, 스트리밍 지원
> 3. **융합**: V4L2, USB, 비디오 코덱과 연관

---

## Ⅰ. 개요

### 개념 정의
**웹캠(Webcam)**은 **실시간 비디오 캡처를 위한 입력 장치**입니다.

### 💡 비유: 디지털 눈
웹캠은 **디지털 눈**과 같습니다. 현실 세계를 디지털 영상으로 바꿉니다.

### 웹캠 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                웹캠 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【웹캠 구성 요소】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  [렌즈] → [이미지 센서] → [ISP] → [인코더] → [USB]             │ │   │
│  │                                                             │ │   │
│  │  • 렌즈: 빛 모으기                                              │ │   │
│  │  • 이미지 센서: CMOS/CCD (빛 → 전기 신호)                        │ │   │
│  │  • ISP (Image Signal Processor): 색상 보정, 노이즈 제거          │ │   │
│  │  • 인코더: 비디오 압축 (MJPEG, H.264, YUYV)                      │ │   │
│  │  • USB: 호스트로 데이터 전송                                    │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【해상도/프레임레이트】                                              │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  해상도            프레임레이트       대역폭           용도          │ │   │
│  │  ────              ──────────       ──────           ────          │ │   │
│  │  640x480 (VGA)     30 fps           ~18 MB/s         일반          │ │   │
│  │  1280x720 (HD)     30 fps           ~55 MB/s         화상회의       │ │   │
│  │  1920x1080 (FHD)   30 fps           ~124 MB/s        고화질        │ │   │
│  │  3840x2160 (4K)    30 fps           ~497 MB/s        전문가용       │ │   │
│  │                                                             │ │   │
│  │  압축 시 대역폭 크게 감소 (MJPEG: ~1/10, H.264: ~1/50)           │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【비디오 포맷】                                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  포맷             FourCC        압축         특징                  │ │   │
│  │  ────             ──────        ────         ────                  │ │   │
│  │  YUYV             YUY2          무압축       4:2:2 YUV            │ │   │
│  │  MJPEG            MJPG          손실         프레임별 JPEG         │ │   │
│  │  H.264            H264          손실         고효율 압축           │ │   │
│  │  NV12             NV12          무압축       4:2:0 YUV            │ │   │
│  │  UYVY             UYVY          무압축       4:2:2 YUV            │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                웹캠 상세                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【V4L2 (Video4Linux2)】                                             │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  계층              설명                                         │ │   │
│  │  ────              ────                                         │ │   │
│  │  Application      /dev/videoX 장치 파일 사용                    │ │   │
│  │  V4L2 core        ioctl 기반 장치 제어                          │ │   │
│  │  V4L2 subdev      센서, ISP 제어                                │ │   │
│  │  Driver           하드웨어별 드라이버                            │ │   │
│  │  Hardware         웹캠 장치                                     │ │   │
│  │                                                             │ │   │
│  │  주요 ioctl:                                                      │ │   │
│  │  • VIDIOC_QUERYCAP: 장치 기능 조회                              │ │   │
│  │  • VIDIOC_G_FMT/VIDIOC_S_FMT: 포맷 설정                         │ │   │
│  │  • VIDIOC_REQBUFS: 버퍼 요청                                    │ │   │
│  │  • VIDIOC_QBUF/VIDIOC_DQBUF: 버퍼 큐잉                          │ │   │
│  │  • VIDIOC_STREAMON/VIDIOC_STREAMOFF: 스트림 제어                │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【웹캠 기능】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  기능              설명                    V4L2 컨트롤            │ │   │
│  │  ────              ────                    ───────────           │ │   │
│  │  밝기              이미지 밝기             V4L2_CID_BRIGHTNESS   │ │   │
│  │  대비              명암 비                 V4L2_CID_CONTRAST     │ │   │
│  │  채도              색상 강도               V4L2_CID_SATURATION   │ │   │
│  │  노출              빛 양 조절              V4L2_CID_EXPOSURE     │ │   │
│  │  화이트밸런스       색 온도 조절            V4L2_CID_WHITE_BALANCE│ │   │
│  │  포커스            초점 조절               V4L2_CID_FOCUS        │ │   │
│  │  줌                확대/축소               V4L2_CID_ZOOM         │ │   │
│  │  팬/틸트           좌우/상하 회전           V4L2_CID_PAN/TILT     │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【USB 비디오 클래스 (UVC)】                                         │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  • USB 표준 비디오 클래스 (Class 0x0E)                          │ │   │
│  │  • 대부분의 웹캠이 UVC 호환                                      │ │   │
│  │  • 표준 드라이버로 자동 인식 (Linux: uvcvideo)                   │ │   │
│  │  • 플러그 앤 플레이 지원                                         │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux 웹캠 확인】                                                 │
│  ──────────────────                                                │
│  // 웹캠 장치 목록                                                    │
│  $ ls /dev/video*                                                   │
│  /dev/video0  /dev/video1                                           │
│                                                                     │
│  // 장치 정보                                                         │
│  $ v4l2-ctl --list-devices                                          │
│  Integrated Camera: Integrated Camera (usb-0000:00:1a.0-1.6):       │
│      /dev/video0                                                    │
│                                                                     │
│  // 지원 포맷                                                         │
│  $ v4l2-ctl --device=/dev/video0 --list-formats                     │
│  Pixel Format  Name                  Desc                           │
│  YUYV          YUYV 4:2:2           YUYV 4:2:2                     │
│  MJPG          MJPEG                 MJPEG                          │
│                                                                     │
│  // 지원 해상도                                                       │
│  $ v4l2-ctl --device=/dev/video0 --list-framesizes=YUYV             │
│  Size: Discrete 640x480                                             │
│  Size: Discrete 1280x720                                            │
│  Size: Discrete 1920x1080                                           │
│                                                                     │
│  【웹캠 컨트롤 설정】                                                 │
│  ──────────────────                                                │
│  // 컨트롤 목록                                                       │
│  $ v4l2-ctl --device=/dev/video0 --list-ctrls                       │
│  brightness (int)    : min=0 max=255 default=128 step=1             │
│  contrast (int)      : min=0 max=255 default=128 step=1             │
│  saturation (int)    : min=0 max=255 default=128 step=1             │
│                                                                     │
│  // 컨트롤 변경                                                       │
│  $ v4l2-ctl --device=/dev/video0 --set-ctrl=brightness=150          │
│  $ v4l2-ctl --device=/dev/video0 --set-ctrl=contrast=140            │
│                                                                     │
│  // 현재 값 확인                                                      │
│  $ v4l2-ctl --device=/dev/video0 --get-ctrl=brightness              │
│  brightness: 150                                                    │
│                                                                     │
│  【ffmpeg로 캡처】                                                   │
│  ──────────────────                                                │
│  // 비디오 캡처                                                       │
│  $ ffmpeg -f v4l2 -i /dev/video0 -t 10 -c:v libx264 output.mp4      │
│                                                                     │
│  // 특정 해상도/포맷                                                   │
│  $ ffmpeg -f v4l2 -video_size 1920x1080 -input_format mjpeg \       │
│    -i /dev/video0 -t 10 output.mp4                                  │
│                                                                     │
│  // 이미지 캡처                                                       │
│  $ ffmpeg -f v4l2 -i /dev/video0 -frames:v 1 capture.jpg            │
│                                                                     │
│  // 스트리밍 (RTMP)                                                   │
│  $ ffmpeg -f v4l2 -i /dev/video0 -c:v libx264 -f flv \              │
│    rtmp://live.example.com/live/stream                              │
│                                                                     │
│  【Python 웹캠 프로그래밍】                                           │
│  ──────────────────                                                │
│  import cv2                                                         │
│                                                                     │
│  # 웹캠 열기                                                         │
│  cap = cv2.VideoCapture(0)                                          │
│  cap.set(cv2.CAP_PROP_FRAME_WIDTH, 1920)                            │
│  cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 1080)                           │
│                                                                     │
│  while True:                                                        │
│      ret, frame = cap.read()                                        │
│      if not ret:                                                    │
│          break                                                      │
│      cv2.imshow('Webcam', frame)                                    │
│      if cv2.waitKey(1) & 0xFF == ord('q'):                          │
│          break                                                      │
│                                                                     │
│  cap.release()                                                      │
│  cv2.destroyAllWindows()                                            │
│                                                                     │
│  【GStreamer 파이프라인】                                            │
│  ──────────────────                                                │
│  // 웹캠 → 화면 표시                                                  │
│  $ gst-launch-1.0 v4l2src device=/dev/video0 ! \                    │
│      videoconvert ! autovideosink                                   │
│                                                                     │
│  // 웹캠 → H.264 인코딩 → 파일                                        │
│  $ gst-launch-1.0 v4l2src ! videoconvert ! x264enc ! \              │
│      mp4mux ! filesink location=output.mp4                          │
│                                                                     │
│  // 네트워크 스트리밍                                                  │
│  $ gst-launch-1.0 v4l2src ! videoconvert ! x264enc ! \              │
│      rtph264pay ! udpsink host=192.168.1.100 port=5000              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 실시간 비디오 캡처 장치
• 구성: 렌즈 → 센서(CMOS/CCD) → ISP → 인코더 → USB
• 해상도: VGA(640x480) ~ 4K(3840x2160)
• 포맷: YUYV, MJPEG, H.264, NV12
• 프레임레이트: 30/60 fps
• V4L2: Linux 비디오 API (ioctl)
• UVC: USB Video Class 표준
• 컨트롤: 밝기, 대비, 채도, 노출, 포커스
• 도구: v4l2-ctl, ffmpeg, OpenCV, GStreamer
• 장치: /dev/videoX
```
