# 556. GPIO (General Purpose Input/Output)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 범용 디지털 입출력 핀
> 2. **가치**: 하드웨어 제어, 센서 인터페이스 지원
> 3. **융합**: 임베디드, 라즈베리파이, 디바이스 드라이버와 연관

---

## Ⅰ. 개요

### 개념 정의
**GPIO(General Purpose Input/Output)**는 **범용 디지털 입출력을 위한 핀 인터페이스**입니다.

### 💡 비유: 만능 스위치
GPIO는 **만능 스위치**와 같습니다. 켜고 끄거나 상태를 읽을 수 있습니다.

### GPIO 구조
```

│                GPIO 구조                                            │
├─────────────────────────────────────────────────────────────────────┤

│  │  INPUT_PULLUP      입력 + 풀업        버튼 (기본 HIGH)           │ │   │

│  │                                                             │ │   │
│  │  특성              일반적인 값              설명                   │ │   │
│  │  ────              ────────────             ────                   │ │   │
│  │  전압 레벨          3.3V / 5V / 1.8V        로직 레벨              │ │   │
│  │  최대 전류          8-25 mA / 핀            싱크/소스              │ │   │
│  │  총 전류            50-150 mA              칩 전체                 │ │   │
│  │  입력 임피던스      1-10 MΩ               높은 임피던스            │ │   │
│  │  전환 속도          1-100 MHz             최대 토글 주파수         │ │   │
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux sysfs GPIO】                                               │
│  ──────────────────                                                │
│  // GPIO 내보내기                                                    │
│  $ echo 18 > /sys/class/gpio/export                                │
│                                                                     │
│  // 방향 설정                                                        │
│  $ echo out > /sys/class/gpio/gpio18/direction                     │
│  $ echo in > /sys/class/gpio/gpio18/direction                      │
│                                                                     │
│  // 값 설정/읽기                                                      │
│  $ echo 1 > /sys/class/gpio/gpio18/value   // HIGH                 │
│  $ echo 0 > /sys/class/gpio/gpio18/value   // LOW                  │
│  $ cat /sys/class/gpio/gpio18/value        // 읽기                  │
│                                                                     │
│  // 엣지 설정 (인터럽트)                                             │
│  $ echo rising > /sys/class/gpio/gpio18/edge                       │
│                                                                     │
│  // GPIO 해제                                                        │
│  $ echo 18 > /sys/class/gpio/unexport                              │
│                                                                     │
│  【libgpiod (새로운 API)】                                          │
│  ──────────────────                                                │
│  // gpioinfo로 정보 확인                                             │
│  $ gpioinfo gpiochip0                                              │
│  gpiochip0 - 54 lines:                                             │
│    line  0: "GPIO0" unused input active-high                       │
│    line  1: "GPIO1" used output active-high [used]                 │
│                                                                     │
│  // 값 설정                                                          │
│  $ gpioset gpiochip0 18=1                                          │
│  $ gpioset gpiochip0 18=0                                          │
│                                                                     │
│  // 값 읽기                                                          │
│  $ gpioget gpiochip0 18                                            │
│  1                                                                  │
│                                                                     │
│  // 모니터링 (인터럽트 대기)                                          │
│  $ gpiomon gpiochip0 18                                            │
│  event: RISING EDGE offset: 18 timestamp: 1234567890.123456       │
│                                                                     │
│  【Python GPIO 제어 (RPi.GPIO)】                                     │
│  ──────────────────                                                │
│  import RPi.GPIO as GPIO                                           │
│  import time                                                        │
│                                                                     │
│  # 핀 번호 체계                                                      │
│  GPIO.setmode(GPIO.BCM)    # BCM 핀 번호                           │
│  # GPIO.setmode(GPIO.BOARD)  # 물리적 핀 번호                       │
│                                                                     │
│  # 출력 설정                                                        │
│  GPIO.setup(18, GPIO.OUT)                                          │
│  GPIO.output(18, GPIO.HIGH)                                        │
│  GPIO.output(18, GPIO.LOW)                                         │
│                                                                     │
│  # 입력 설정 (풀업)                                                  │
│  GPIO.setup(24, GPIO.IN, pull_up_down=GPIO.PUD_UP)                 │
│  if GPIO.input(24) == GPIO.LOW:                                    │
│      print("Button pressed")                                       │
│                                                                     │
│  # 인터럽트                                                          │
│  def button_callback(channel):                                     │
│      print("Button was pushed!")                                   │
│                                                                     │
│  GPIO.add_event_detect(24, GPIO.FALLING, callback=button_callback) │
│                                                                     │
│  # PWM                                                              │
│  pwm = GPIO.PWM(18, 1000)   # 1kHz                                 │
│  pwm.start(50)              # 50% 듀티                              │
│  pwm.ChangeDutyCycle(75)    # 75%                                   │
│  pwm.stop()                                                         │
│                                                                     │
│  GPIO.cleanup()                                                     │
│                                                                     │
│  【C GPIO 제어 (메모리 매핑)】                                        │
│  ──────────────────                                                │
│  #include <fcntl.h>                                                 │
│  #include <sys/mman.h>                                              │
│  #include <unistd.h>                                                │
│                                                                     │
│  #define GPIO_BASE 0x3F200000  // BCM2835                           │
│  #define GPIO_SIZE 4096                                             │
│                                                                     │
│  volatile unsigned int *gpio;                                       │
│                                                                     │
│  int fd = open("/dev/mem", O_RDWR);                                │
│  gpio = mmap(NULL, GPIO_SIZE, PROT_READ|PROT_WRITE,                │
│              MAP_SHARED, fd, GPIO_BASE);                            │
│                                                                     │
│  // 핀 18을 출력으로 설정                                            │
│  *(gpio + 1) = (*(gpio + 1) & ~(7 << 24)) | (1 << 24);             │
│                                                                     │
│  // 핀 18 HIGH                                                       │
│  *(gpio + 7) = 1 << 18;                                            │
│                                                                     │
│  // 핀 18 LOW                                                        │
│  *(gpio + 10) = 1 << 18;                                           │
│                                                                     │
│  munmap((void*)gpio, GPIO_SIZE);                                   │
│  close(fd);                                                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 범용 디지털 입출력 핀
• 모드: INPUT, OUTPUT, PULLUP, PULLDOWN, ALTERNATE
• 레지스터: MODER, ODR, IDR, BSRR
• 전압: 1.8V, 3.3V, 5V 로직 레벨
• 전류: 8-25 mA/핀 (최대)
• 인터럽트: Rising, Falling, Both Edge
• 풀업/풀다운: 플로팅 방지
• Linux: sysfs, libgpiod
• Python: RPi.GPIO, gpiozero
• 용도: LED, 버튼, 센서, 릴레이 제어
```
