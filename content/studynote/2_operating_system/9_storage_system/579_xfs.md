# 579. XFS (Extents File System)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고성능 64비트 저널링 파일시스템
> 2. **가치**: 대용량 파일, 병렬 I/O, 온라인 확장
> 3. **융합**: extent, B+ 트리, 할당 그룹과 연관

---

## Ⅰ. 개요

### 개념 정의
**XFS**는 **SGI가 개발한 고성능 64비트 저널링 파일시스템**입니다.

### 💡 비유: 초대형 창고
XFS는 **초대형 창고**와 같습니다. 대량의 물건을 효율적으로 관리합니다.

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【XFS 파일시스템 생성】                                              │
│  ──────────────────                                                │
│  // 기본 생성                                                        │
│  $ sudo mkfs.xfs /dev/sdb1                                         │
│                                                                     │
│  // 옵션 지정                                                        │
│  $ sudo mkfs.xfs -L mylabel /dev/sdb1        // 라벨                  │
│  $ sudo mkfs.xfs -b size=4096 /dev/sdb1      // 블록 크기              │
│  $ sudo mkfs.xfs -d agcount=4 /dev/sdb1      // AG 수                  │
│  $ sudo mkfs.xfs -l size=256m /dev/sdb1      // 로그 크기              │
│  $ sudo mkfs.xfs -f /dev/sdb1                // 강제 덮어쓰기           │
│                                                                     │
│  // 외부 로그 사용                                                    │
│  $ sudo mkfs.xfs -l logdev=/dev/sdc1 /dev/sdb1                     │
│                                                                     │
│  【XFS 마운트】                                                       │
│  ──────────────────                                                │
│  // 기본 마운트                                                       │
│  $ sudo mount /dev/sdb1 /mnt/data                                  │
│                                                                     │
│  // 성능 옵션                                                        │
│  $ sudo mount -o noatime /dev/sdb1 /mnt/data                       │
│  $ sudo mount -o nobarrier /dev/sdb1 /mnt/data                     │
│  $ sudo mount -o allocsize=64m /dev/sdb1 /mnt/data                 │
│                                                                     │
│  // /etc/fstab                                                       │
│  /dev/sdb1  /mnt/data  xfs  defaults,noatime  0  0                 │
│  UUID=xxx   /mnt/data  xfs  defaults,noatime  0  0                 │
│                                                                     │
│  【XFS 정보 확인】                                                    │
│  ──────────────────                                                │
│  // 파일시스템 정보                                                    │
│  $ sudo xfs_info /mnt/data                                         │
│  $ sudo xfs_info /dev/sdb1                                         │
│                                                                     │
│  // 상세 정보                                                         │
│  $ sudo xfs_db -c "info" /dev/sdb1                                 │
│  $ sudo xfs_admin -l /dev/sdb1          // 라벨 확인                  │
│  $ sudo xfs_admin -u /dev/sdb1          // UUID 확인                  │
│                                                                     │
│  // 사용량                                                           │
│  $ df -h /mnt/data                                                  │
│  $ xfs_quota -x -c 'df -h' /mnt/data                               │
│                                                                     │
│  【XFS 크기 확장】                                                    │
│  ──────────────────                                                │
│  // 온라인 확장 (마운트 상태)                                          │
│  $ sudo xfs_growfs /mnt/data             // 전체 크기로                │
│  $ sudo xfs_growfs -D 1000000 /mnt/data  // 블록 수 지정              │
│                                                                     │
│  // LVM과 함께 사용                                                   │
│  $ sudo lvextend -L +10G /dev/vg0/lv0                              │
│  $ sudo xfs_growfs /mnt/data                                       │
│                                                                     │
│  ※ XFS는 축소 불가능! 확장만 가능                                     │
│                                                                     │
│  【XFS 복구】                                                         │
│  ──────────────────                                                │
│  // 파일시스템 검사                                                   │
│  $ sudo xfs_repair -n /dev/sdb1          // 읽기 전용 검사            │
│  $ sudo xfs_repair /dev/sdb1             // 복구                     │
│  $ sudo xfs_repair -L /dev/sdb1          // 로그 초기화 (위험)         │
│                                                                     │
│  // 메타데이터 덤프                                                   │
│  $ sudo xfs_metadump /dev/sdb1 dump.meta                           │
│  $ sudo xfs_mdrestore dump.meta /dev/sdb1                          │
│                                                                     │
│  【XFS 할당량 (Quota)】                                               │
│  ──────────────────                                                │
│  // 할당량 활성화                                                     │
│  $ sudo mount -o uquota /dev/sdb1 /mnt/data                        │
│  $ sudo mount -o gquota /dev/sdb1 /mnt/data                        │
│                                                                     │
│  // 할당량 설정                                                       │
│  $ sudo xfs_quota -x -c 'limit bsoft=1g bhard=2g username' /mnt    │
│  $ sudo xfs_quota -x -c 'limit isoft=1000 ihard=2000 username' /mnt│
│                                                                     │
│  // 할당량 확인                                                       │
│  $ sudo xfs_quota -x -c 'report -h' /mnt/data                      │
│  $ sudo xfs_quota -x -c 'quota -h username' /mnt/data              │
│                                                                     │
│  【XFS 조각 모음】                                                    │ |
│  ──────────────────                                                │
│  // 조각화 확인                                                       │
│  $ sudo xfs_db -c "frag -f" /dev/sdb1                              │
│                                                                     │
│  // 조각 모음                                                         │
│  $ sudo xfs_fsr /mnt/data                  // 전체                   │
│  $ sudo xfs_fsr /mnt/data/file.txt         // 단일 파일              │
│                                                                     │
│  // 파일 시스템 동결 (백업용)                                          │
│  $ sudo xfs_freeze -f /mnt/data            // 동결                   │
│  $ sudo xfs_freeze -u /mnt/data            // 해제                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: SGI 개발 고성능 64비트 파일시스템
• 최대 크기: 파일/볼륨 8 EB
• AG (Allocation Group): 병렬 공간 관리
• Extent: 시작 블록 + 길이 구조
• B+ Tree: 디렉토리, Extent 인덱싱
• 지연 할당: 실제 쓰기 시점 할당
• 확장: xfs_growfs (온라인 가능)
• 축소: 불가능
• 생성: mkfs.xfs
• 복구: xfs_repair
• 할당량: xfs_quota
• 조각모음: xfs_fsr
• RHEL/CentOS 기본
```
