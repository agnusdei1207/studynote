# 019. 예외 (Exception)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로그램 실행 중 발생하는 동기적 오류 이벤트
> 2. **가치**: 오류 감지, 보호, 디버깅 지원
> 3. **융합**: 트랩, 페이지 폴트, 시스템 콜과 연관

---

## Ⅰ. 개요

### 개념 정의

예외(Exception)는 **프로그램 실행 중 발생하는 오류나 특수 상황으로 인해 CPU가 정상 실행 흐름을 중단하고 커널로 제어를 넘기는 동기적 이벤트**다.

### 💡 비유: 경기장 보안 요원
예외는 **경기장 규칙 위반**과 같다. 선수(프로그램)가 반칙(예외)을 하면 심판(CPU)이 즉시 경기를 중단하고 보안 요원(커널)이 개입한다. 퇴장(프로세스 종료)하거나 경고(시그널)를 준다.

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 예외: 프로그램 실행 중 오류로 발생하는 동기적 이벤트
• Fault: 복구 가능, 오류 명령으로 복귀
• Trap: 의도적, 다음 명령으로 복귀
• Abort: 복구 불가, 프로세스 종료
• 주요 예외: Divide, GPF, Page Fault, Machine Check
• 시그널: SIGFPE, SIGSEGV, SIGILL, SIGTRAP
```
