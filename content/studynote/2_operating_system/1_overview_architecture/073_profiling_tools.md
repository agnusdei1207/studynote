# 073. 프로파일링 및 트레이싱 도구
#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시스템 성능 분석, 병목 식별, 문제 진단 도구
> 2. **가치**: 성능 최적화, 디버깅, 관찰 가능성
> 3. **융합**: perf, ftrace, eBPF, strace와 연관
---

## Ⅰ. 개요
### 개념 정의
프로파일링 및 트레이싱 도구는 **시스템 성능을 분석하고 병목을 식별하며 문제를 진단하는 도구들**이다. perf, ftrace, strace, DTrace 등이 포함된다.
### 💡 비유: 의료 진단 장비
프로파일링 도구는 **의료 진단 장비**와 같다. MRI, CT(perf, ftrace)로 환자(시스템) 내부를 들여다보고, 어디가 아픈지(병목) 찾는다.
### 프로파일링 도구 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                프로파일링 및 트레이싱 도구 구조                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【주요 프로파일링 도구】                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  도구              설명                    용도                │ │
│  │  ────                ────                    ────              │ │
│  │  perf              Linux 성능 분석      CPU, 메모리, 캐시     │ │
│  │  ftrace            커널 함수 추적        함수 호출, 지연       │ │
│  │  strace            시스템 콜 추적        프로세스 디버깅     │ │
│  │  ltrace            라이브러리 호출 추적      함수 호출            │ │
│  │  DTrace            동적 추적 (Solar/BSD)  시스템 전체           │ │
│  │  eBPF             eBPF 기반 추적        네트워크, 보안        │ │
│  │  bpftrace          eBPF 고수준 언어        스크립트 추적          │ │
│  │  BCC              eBPF 컴파일러 모음        추적 도구            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【perf 주요 기능】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  # CPU 사용량 분석                                                  │ │
│  │  $ perf top                                                     │ │
│  │  $ perf record -g -a -- sleep 10                               │ │
│  │  $ perf report                                                  │ │
│  │                                                               │ │
│  │  # 하드웨어 카운터                                                │ │
│  │  $ perf stat -e cycles,instructions,cache-misses ./program     │ │
│  │                                                               │ │
│  │  # 플레임 그래프                                                  │ │
│  │  $ perf record -g -a -- sleep 10                               │ │
│  │  $ perf script | stackcollapse-perf.pl | flamegraph.pl > out.svg │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【ftrace 주요 기능】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  # 함수 추적 활성화                                                │ │
│  │  $ echo function > /sys/kernel/debug/tracing/current_tracer   │ │
│  │  $ echo 1 > /sys/kernel/debug/tracing/tracing_on              │ │
│  │                                                               │ │
│  │  # 특정 함수 추적                                                  │ │
│  │  $ echo schedule > /sys/kernel/debug/tracing/set_ftrace_filter│ │
│  │                                                               │ │
│  │  # 결과 확인                                                        │ │
│  │  $ cat /sys/kernel/debug/tracing/trace                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【strace 주요 기능】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  # 시스템 콜 추적                                                  │ │
│  │  $ strace -f ./program                                          │ │
│  │  $ strace -p <pid>                                              │ │
│  │                                                               │ │
│  │  # 필터링                                                          │ │
│  │  $ strace -e trace=open,read,write ./program                   │ │
│  │  $ strace -e trace=network ./program                           │ │
│  │                                                               │ │
│  │  # 통계                                                            │ │
│  │  $ strace -c ./program                                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
