# 030. 펌웨어 (Firmware)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 하드웨어에 내장된 저수준 소프트웨어, 하드웨어 제어 담당
> 2. **가치**: 하드웨어 초기화, 부팅, 장치 제어
> 3. **융합**: BIOS, UEFI, 장치 펌웨어와 연관

---

## Ⅰ. 개요

### 개념 정의

펌웨어(Firmware)는 **하드웨어 장치에 내장된 소프트웨어로, 하드웨어의 기본적인 동작을 제어하고 초기화하는 저수준 프로그램**이다. 소프트웨어(Software)와 하드웨어(Hardware)의 합성어로, ROM, 플래시 메모리 등에 저장된다.

### 💡 비유: 기계의 두뇌
펌웨어는 **기계의 두뇌**와 같다. 로봇(하드웨어)이 움직이려면 두뇌(펌웨어)의 지시가 필요하다. 두뇌는 로봇의 각 부분에 어떻게 움직이라고 명령한다. 두뇌가 없으면 로봇은 작동하지 않는다.

## Ⅳ. 실무 적용

### 펌웨어 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                펌웨어 분석                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【시스템 펌웨어 확인】                                              │
│  ──────────────────                                                  │
│  # BIOS/UEFI 버전                                                   │
│  $ sudo dmidecode -t bios                                      │
│  $ sudo efivar -l  # UEFI 변수                                 │
│                                                                     │
│  # 펌웨어 정보                                                      │
│  $ cat /sys/class/dmi/id/bios_version                           │
│  $ cat /sys/class/dmi/id/bios_date                             │
│                                                                     │
│  【장치 펌웨어 확인】                                                │
│  ──────────────────                                                  │
│  # SSD 펌웨어                                                       │
│  $ sudo smartctl -a /dev/sda | grep Firmware                    │
│                                                                     │
│  # NIC 펌웨어                                                        │
│  $ ethtool -i eth0                                              │
│                                                                     │
│  # GPU 펌웨어                                                        │
│  $ nvidia-smi -q | grep "FB Memory"                             │
│                                                                     │
│  【CPU 마이크로코드】                                               │
│  ──────────────────                                                  │
│  # 현재 마이크로코드 확인                                           │
│  $ cat /proc/cpuinfo | grep microcode                           │
│                                                                     │
│  # 마이크로코드 업데이트 (Linux)                                     │
│  $ sudo apt install intel-microcode                              │
│  $ sudo apt install amd64-microcode                               │
│                                                                     │
│  【펌웨어 업데이트 도구】                                             │
│  ──────────────────                                                  │
│  # Linux fwupd (LVFS)                                              │
│  $ sudo fwupdmgr refresh                                        │
│  $ sudo fwupdmgr get-updates                                    │
│  $ sudo fwupdmgr update                                         │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 펌웨어: 하드웨어 내장 저수준 소프트웨어
• 저장: ROM, EPROM, Flash Memory
• 유형: BIOS/UEFI, Device FW, Microcode, EC, BMC
• 역할: 하드웨어 초기화, 부팅, 장치 제어
• 보안: Secure Boot, TPM, 루트킷 방지
• 업데이트: fwupd, OTA, 플래시 도구
• 예시: SSD FTL, NIC FW, GPU FW
```
