# 038. 셸 (Shell)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 사용자와 운영체제 커널 사이의 인터페이스
> 2. **가치**: 명령어 해석, 스크립트 실행, 시스템 제어
> 3. **융합**: bash, zsh, PowerShell, 스크립팅과 연관

---

## Ⅰ. 개요

### 개념 정의

셸(Shell)은 **사용자가 입력한 명령어를 해석하여 커널에 전달하고 결과를 사용자에게 보여주는 명령어 인터프리터**다. 사용자와 커널 사이에서 중개 역할을 하며, 대화형 명령어 실행과 스크립트 실행 기능을 제공한다.

### 💡 비유: 식당의 웨이터
셸은 **식당의 웨이터**와 같다. 손님(사용자)이 주문(명령어)을 하면 웨이터(셸)가 주방(커널)에 전달한다. 주방에서 요리(실행)가 완료되면 웨이터가 손님에게 음식(결과)을 가져다준다. 손님은 주방에 직접 들어가지 않아도 된다.

## Ⅳ. 실무 적용

### 셸 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                셸 분석                                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【현재 셸 확인】                                                    │
│  ──────────────────                                                  │
│  $ echo $SHELL            # 현재 셸 경로                            │
│  $ echo $0                # 현재 셸 이름                            │
│  $ ps -p $$               # PID로 셸 확인                           │
│  $ cat /etc/shells        # 사용 가능한 셸                          │
│                                                                     │
│  【셸 설정 파일】                                                    │
│  ──────────────────                                                  │
│  # bash                                                            │
│  ~/.bashrc                 # 대화형 비로그인                       │
│  ~/.bash_profile           # 로그인 셸                             │
│  ~/.profile               # 범용 로그인                            │
│  /etc/bash.bashrc          # 시스템 전체                           │
│                                                                     │
│  # zsh                                                             │
│  ~/.zshrc                  # 주 설정 파일                          │
│  ~/.zprofile              # 로그인                                 │
│                                                                     │
│  【유용한 셸 기능】                                                  │
│  ──────────────────                                                  │
│  # 히스토리                                                         │
│  $ history                  # 명령어 기록                           │
│  $ !!                       # 이전 명령어 반복                      │
│  $ !n                        # n번째 명령어                         │
│  $ !string                   # string으로 시작하는 명령어           │
│                                                                     │
│  # 작업 제어                                                        │
│  $ jobs                     # 백그라운드 작업                       │
│  $ fg %1                    # 포그라운드로                          │
│  $ bg %1                    # 백그라운드로                          │
│  $ Ctrl+Z                   # 일시 중지                             │
│  $ Ctrl+C                   # 종료                                 │
│                                                                     │
│  # 자동완성                                                         │
│  $ Tab                      # 파일/명령어 완성                     │
│  $ Tab Tab                  # 가능한 완성 목록                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 셸: 사용자와 커널 사이의 인터페이스
• 종류: bash, zsh, fish, PowerShell
• 기능: 명령어 실행, 파이프, 리다이렉션, 변수, 스크립트
• 특수문자: | > < & $ * ?
• 설정: .bashrc, .zshrc
• 스크립트: #!/bin/bash, 조건문, 반복문, 함수
```
