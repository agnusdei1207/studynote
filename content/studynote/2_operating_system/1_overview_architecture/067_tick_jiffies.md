# 067. 틱 (Tick) / 지피스 (Jiffies)
#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 커널 시간의 기본 단위와 부팅 후 경과 틱 수
> 2. **가치**: 시간 측정, 타임아웃, 스케줄링 기준
> 3. **융합**: HZ, timer interrupt, uptime과 연관
---

## Ⅰ. 개요
### 개념 정의
틱(Tick)은 **커널 시간의 기본 단위**로, 타이머 인터럽트가 발생할 때마다 증가한다. 지피스(Jiffies)는 **시스템 부팅 후 경과한 틱 수**를 저장하는 전역 변수다.
### 💡 비유: 심박동수
틱/지피스는 **심박동수**와 같다. 심장이 뛸 때마다(틱) 카운터(지피스)가 올라간다. 카운터로 얼마나 시간이 지났는지 알 수 있다.
### 틱/지피스 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                틱/지피스 구조                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【틱과 지피스 관계】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  jiffies_64 (64비트)                                         │ │
│  │  • 부팅 후 경과한 틱 수                                        │ │
│  │  • 타이머 인터럽트마다 증가                                      │ │
│  │                           │                                   │                  │ │
│  │                           ▼                                   ▼                  │ │
│  │  시간 계산                                                    │ │
│  │  • 시간(초) = jiffies / HZ                                    │ │
│  │  • 예: jiffies=1000, HZ=250 → 4초                             │ │
│  │                           │                                   │                  │ │
│  │                           ▼                                   ▼                  │ │
│  │  uptime 명령어                                                │ │
│  │  $ cat /proc/uptime                                           │ │
│  │  3600.45 12345.67  ← (초, idle 시간)                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【틱 관련 매크로/함수】                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  함수/매크로         설명                                        │ │
│  │  ────                ────                                        │ │
│  │  jiffies             현재 틱 수 (64비트)                          │ │

│  │  time_after(a)       a 이후 경과 확인                            │ │

│  │  time_in_range()     범위 내 시간 확인                          │ │
│  │  msecs_to_jiffies()  밀리초 → 지피스 변환                           │ │
│  │  jiffies_to_msecs()  지피스 → 밀리초 변환                           │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │

│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  설정              Hz             틱 간격                          │ │
