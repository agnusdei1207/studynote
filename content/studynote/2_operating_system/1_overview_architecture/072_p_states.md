# 072. 프로세서 성능 상태 (P-States)
#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU 동작 주파수와 전압을 조절하는 ACPI 상태
> 2. **가치**: 성능 관리, 전력 효율, 열 관리
> 3. **융합**: ACPI, C-States, DVFS, Turbo Boost와 연관
---

## Ⅰ. 개요
### 개념 정의
P-States(Performance States)은 **ACPI에서 정의하는 CPU 동작 주파수와 전압 조합으로, 성능과 전력 소모의 균형을 조절**한다.
### 💡 비유: 자동차 기어
P-States는 **자동차 기어**와 같다. 고기어(P0)는 높은 성능, 낮은 연비. 저기어(Pn)는 낮은 성능, 높은 연비. 상황에 맞게 기어를 바꾼다.
### P-States 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                프로세서 성능 상태 (P-States) 구조                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【P-States 구성】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  P-State               주파수              전압        전력      │ │
│  │  ────                    ────                ────        ────      │ │
│  │  P0 (Turbo)            최대                높음        높음      │ │
│  │  P1                   높음                높음        높음      │ │
│  │  P2                   보통                보통        보통      │ │
│  │  P3                   낮음                낮음        낮음      │ │
│  │  Pn                   최소                낮음        최저      │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【P-State 전환 요인】                                                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  요인                설명                                        │ │
│  │  ────                ────                                        │ │
│  │  CPU 부하            높음 → 높은 P-State                          │ │
│  │  온도                높음 → 낮은 P-State                          │ │
│  │  배터리              낮음 → 낮은 P-State                          │ │
│  │  전원 정책            성능 → 높은 P-State                          │ │
│  │                    절전 → 낮은 P-State                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【P-States 관리 도구】                                               │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  # 현재 P-State 확인                                                │ │
│  │  $ cat /sys/devices/system/cpu/cpu*/cpufreq/scaling_cur_freq   │ │
│  │                                                               │ │
│  │  # 사용 가능한 주파수                                                │ │
│  │  $ cat /sys/devices/system/cpu/cpu*/cpufreq/scaling_available_frequencies │ │
│  │                                                               │ │
│  │  # governor 설정                                                    │ │
│  │  $ cat /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor    │ │
│  │  $ echo performance > .../scaling_governor                      │ │
│  │  $ echo powersave > .../scaling_governor                         │ │
│  │  $ echo ondemand > .../scaling_governor                          │ │
│  │                                                               │ │
│  │  # cpupower 도구                                                    │ │
│  │  $ cpupower frequency-info                                      │ │
│  │  $ cpupower set -g performance                                  │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【Governor 종류】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  Governor            설명                                        │ │
│  │  ────                ────                                        │ │
│  │  performance         최대 성능 유지                              │ │
│  │  powersave           최소 전력 소모                              │ │
│  │  ondemand            부하에 따라 동적 조절                        │ │
│  │  conservative        점진적 조절                                  │ │
│  │  userspace           사용자 지정                                  │ │
│  │  schedutil           스케줄러 기반                                │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
