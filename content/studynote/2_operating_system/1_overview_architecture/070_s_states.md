# 070. 시스템 전원 상태 (S-States)
#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시스템 전체의 절전 수준을 정의하는 ACPI 상태
> 2. **가치**: 전력 절약, 빠른 시작, 상태 보존
> 3. **융합**: ACPI, C-States, P-States, 절전 모드와 연관
---

## Ⅰ. 개요
### 개념 정의
S-States(System Power States)은 **ACPI에서 정의하는 시스템 전체의 절전 수준**으로, S0(작동)부터 S5(종료)까지 6단계로 구성된다.
### 💡 비유: 건물 전원 상태
S-States는 **건물 전원 상태**와 같다. 모든 조명 켜짐(S0), 일부만 켜짐(S1-S3), 비상시만 켜짐(S4), 모든 전원 차단(S5)으로 단계적 절전이 가능하다.
### S-States 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                시스템 전원 상태 (S-States) 구조                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【S-States 종류】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  상태              설명                    전력 소모            │ │
│  │  ────                ────                    ────              │ │
│  │  S0 (Working)        정상 작동                최대              │ │
│  │  S1 (Power On       대기, 깨우기 빠름        중간              │ │
│  │    Suspend)                                                  │ │
│  │  S2 (Standby)       메모리 유지, CPU 정지     낮음              │ │
│  │  S3 (Sleep)        메모리 유지, 디스크 정지  매우 낮음          │ │
│  │  S4 (Hibernate)     디스크에 메모리 저장         극히 낮음          │ │
│  │  S5 (Soft Off)      완전 종료                0                 │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【상태 전환】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  전환              소요 시간                    특징                │ │
│  │  ────                ────                    ────              │ │
│  │  S0 → S1          즉각                      즉시 깨어기          │ │
│  │  S0 → S3          1-2초                      메모리 유지          │ │
│  │  S0 -> S4          수 초~수십 분             디스크 저장          │ │
│  │  S4 -> S0          수 초~수십 분             빠른 복구          │ │
│  │  S0 -> S5          즉각                      상태 손실            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【리눅스 명령어】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  # 현재 상태 확인                                                │ │
│  │  $ cat /sys/power/state                                          │ │
│  │                                                               │ │
│  │  # 절전 모드 진입                                                │ │
│  │  $ echo mem > /sys/power/state                            │ │
│  │  $ echo disk > /sys/power/state                             │ │
│  │                                                               │ │
│  │  # 절전 통계                                                        │ │
│  │  $ cat /sys/power/pm_stats                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
