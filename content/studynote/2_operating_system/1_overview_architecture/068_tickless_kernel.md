# 068. 틱리스 커널 (Tickless Kernel)
#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 불필요한 타이머 틱을 제거하여 전력 소모 감소
> 2. **가치**: 노트북 배터리 수명 증가, 데이터센터 전력 절감
> 3. **융합**: NO_HZ, dynamic tick, idle state와 연관
---

## Ⅰ. 개요
### 개념 정의
틱리스 커널(Tickless Kernel)은 **시스템이 유휴 상태일 때 불필요한 타이머 틱을 제거하여 전력 소모를 줄이는 커널 기능**이다. NO_HZ(No HZ) 설정으로 활성화한다.
### 💡 비유: 대기 모드
틱리스 커널은 **휴대폰 대기 모드**와 같다. 사용하지 않을 때는 화면을 끄고(틱 중지) 배터리를 아낀다. 깨워야 할 때만(필요할 때만) 화면이 켜진다.
### 틱리스 커널 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                틱리스 커널 구조                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Periodic vs Tickless】                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  Periodic Tick                                               │ │
│  │  ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐              │ │
│  │  │ T │ │ T │ │ T │ │ T │ │ T │ │ T │ │ T │ │ T │              │ │
│  │  └───┘ └───┘ └───┘ └───┘ └───┘ └───┘ └───┘ └───┘              │ │
│  │  • 주기적 인터럽트 계속 발생                                    │ │
│  │  • CPU 절전 모드 진입 불가                                      │ │
│  │  • 배터리 소모 증가                                            │ │
│  │                                                               │ │
│  │  Tickless (NO_HZ)                                              │ │
│  │  ┌───┐                           ┌───┐                           │ │
│  │  │ T │                           │ T │                           │ │
│  │  └───┘                           └───┘                           │ │
│  │  • 다음 이벤트까지 틱 중지                                      │ │
│  │  • CPU 깊은 절전 모드 진입 가능                                │ │
│  │  • 전력 소모 대폭 감소                                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【NO_HZ 모드】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  모드              설명                    효과                  │ │
│  │  ────                ────                    ────              │ │
│  │  NO_HZ_IDLE        유휴 CPU 틱 중지            절전              │ │
│  │  NO_HZ_FULL        CPU당 틱 중지 (RT)         지연 감소          │ │
│  │  NO_HZ            틱리스 활성화              기본 설정          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
