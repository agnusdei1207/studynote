# 042. SAN (Storage Area Network)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서버와 스토리지를 연결하는 전용 고속 네트워크
> 2. **가치**: 스토리지 집중화, 유연성, 성능, 확장성
> 3. **융합**: Fibre Channel, iSCSI, NAS와 연관

---

## Ⅰ. 개요

### 개념 정의

SAN(Storage Area Network)은 **서버와 스토리지 장치를 연결하는 전용 고속 네트워크**로, 서버에서 스토리지를 직접 연결된 것처럼 블록 단위로 접근할 수 있게 한다. Fibre Channel이나 iSCSI를 사용하여 구현된다.

### 💡 비유: 중앙 창고 시스템
SAN은 **대형 마트의 중앙 창고**와 같다. 각 매장(서버)에서 창고(SAN 스토리지)에 물건을 보관하고 꺼내 쓴다. 창고는 모든 매장이 공유하지만, 각 매장은 자기 물건만 꺼내 쓴다. 창고가 클 수록, 매장이 늘어도 물건은 충분하다.

## Ⅱ. SAN 상세

```
┌─────────────────────────────────────────────────────────────────────┐

│  │  확장성            낮음               보통               높음      │ │
│  │  공유              어려움             쉬움               가능      │ │
```

## Ⅲ. SAN × 타 과목 융합

### 과목 융합 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                SAN × 타 과목 융합                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【네트워킹】                                                        │
│  • Fibre Channel: 광섬유 채널, FCP                                  │
│  • iSCSI: IP 기반 SCSI                                              │
│  • FCoE: Fibre Channel over Ethernet                               │
│  • Multipathing: MPIO, DM-Multipath                                │
│                                                                     │
│  【파일시스템】                                                      │
│  • 클러스터 FS: GFS2, OCFS2                                        │
│  • 공유 스토리지: 동시 접근                                          │
│  • LUN 마스킹: 접근 제어                                            │
│                                                                     │
│  【가상화】                                                          │
│  • VM 스토리지: vSAN, RDM                                           │
│  • vMotion: 라이브 마이그레이션                                     │
│  • 스토리지 vMotion                                                  │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  【iSCSI 설정 예시 (Linux)】                                        │
│  ──────────────────                                                  │

│  $ apt install open-iscsi                                        │
│                                                                     │
│  # 타겟 발견                                                         │
│  $ iscsiadm -m discovery -t st -p 192.168.1.100                  │
│                                                                     │
│  # 타겟 로그인                                                       │
│  $ iscsiadm -m node -T iqn.target -p 192.168.1.100 -l            │
│                                                                     │
│  # 세션 확인                                                         │
│  $ iscsiadm -m session                                            │
│                                                                     │
│  # 디스크 확인                                                       │
│  $ fdisk -l                                                        │
```

## Ⅳ. 실무 적용

### SAN 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                SAN 분석                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【FC 정보 확인】                                                    │
│  ──────────────────                                                  │
│  # HBA 정보                                                         │
│  $ cat /sys/class/fc_host/host*/port_name                        │
│  $ systool -c fc_host -v                                          │
│                                                                     │
│  # FC 통계                                                          │
│  $ cat /sys/class/fc_host/host*/statistics/*                     │
│                                                                     │
│  【iSCSI 관리】                                                      │
│  ──────────────────                                                  │
│  # 발견된 타겟                                                       │
│  $ iscsiadm -m node                                               │
│                                                                     │
│  # 세션 상태                                                         │
│  $ iscsiadm -m session -P 3                                       │
│                                                                     │
│  # 로그아웃                                                          │
│  $ iscsiadm -m node -T iqn.target -p 192.168.1.100 -u            │
│                                                                     │
│  【Multipathing】                                                   │
│  ──────────────────                                                  │
│  # 멀티패스 상태                                                     │
│  $ multipath -ll                                                   │
│  $ multipath -v3                                                   │
│                                                                     │
│  # 장치 확인                                                        │
│  $ lsblk                                                           │
│  $ fdisk -l                                                        │
│                                                                     │
│  # I/O 통계                                                         │
│  $ iostat -x                                                       │
└─────────────────────────────────────────────────────────────────────┘
```
