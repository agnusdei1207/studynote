# 071. 프로세서 전원 상태 (C-States)
#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: CPU 코어의 절전 수준을 정의하는 ACPI 상태
> 2. **가치**: 전력 절약, 열 관리, 성능 조절
> 3. **융합**: ACPI, P-States, idle, tickless와 연관
---

## Ⅰ. 개요
### 개념 정의
C-States(CPU Power States)은 **ACPI에서 정의하는 CPU 코어의 절전 수준**으로, C0(활성)에서 Cn(심각 절전)까지 여러 단계로 구성된다.
### 💡 비유: 사람 휴식
C-States는 **사람 휴식**과 같다. 활동(C0) → 가벼운 휴식(C1) → 낮잠(C2) → 깊은 잠(C3+) 순서로 점점 쉬어간다.
