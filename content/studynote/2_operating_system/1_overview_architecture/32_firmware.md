# 569. 펌웨어 (Firmware)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 하드웨어 제어용 내장 소프트웨어
> 2. **가치**: 하드웨어 초기화, 부팅, 기능 구현
> 3. **융합**: BIOS, 드라이버, 임베디드와 연관

---

## Ⅰ. 개요

### 개념 정의
**펌웨어(Firmware)**는 **하드웨어 장치에 내장된 소프트웨어**입니다.

### 💡 비유: 기계의 두뇌
펌웨어는 **기계의 두뇌**와 같습니다. 하드웨어가 어떻게 작동할지 정합니다.

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 하드웨어 내장 소프트웨어
• 종류: BIOS/UEFI, BMC, NIC, SSD, GPU, RAID
• 저장: ROM, EPROM, EEPROM, Flash
• SSD 기능: FTL, Wear Leveling, GC, TRIM
• BMC: IPMI, 원격 전원, KVM, 센서
• 업데이트: 플래시, fwupd, USB
• Linux: /lib/firmware/, request_firmware
• 도구: fwupdmgr, ipmitool, efibootmgr
• 확인: hdparm, smartctl, nvme-cli
• 위험: 업데이트 실패 시 장치 벽돌
```
