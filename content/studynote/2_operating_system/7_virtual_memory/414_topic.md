# 414. 파일 매핑 (File Mapping)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일을 메모리 주소 공간에 매핑
> 2. **가치**: 파일 접근 간소화와 성능 향상
> 3. **융합**: 메모리 사상 파일, 페이지 캐시와 연관

---

## Ⅰ. 개요

### 개념 정의

파일 매핑(File Mapping)은 **파일의 내용을 프로세스의 가상 주소 공간에 직접 매핑하는 기법**이다. 메모리 접근으로 파일을 읽고 쓸 수 있다.

### 💡 비유: 창문
파일 매핑은 **창문**과 같다. 창 밖 풍경(파일)을 창 안(메모리)에서 본다. 창을 통해 바로 접근한다.

### 파일 매핑 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 매핑 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【전통적 vs 매핑】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  전통적 파일 접근:                                                      │ │
│  │  read() ──► 커널 버퍼 ──► 사용자 버퍼                                  │ │
│  │  (데이터 2번 복사)                                                       │ │
│  │                                                             │ │
│  │  파일 매핑:                                                              │ │
│  │  mmap() ──► 사용자 주소 공간                                            │ │
│  │  (직접 접근, 복사 없음)                                                  │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【매핑 구조】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  프로세스 주소 공간:                                                     │ │
│  │  ┌─────────────────────────────────────────────────────────────┐│ │
│  │  │ 0x1000 ───────────────────────────────────────── 0x2000   ││ │
│  │  │           ↑ mmap으로 매핑된 영역                          ││ │
│  │  │           │                                               ││ │
│  │  │           │  페이지 폴트 시                                ││ │
│  │  │           │  파일에서 로드                                  ││ │
│  │  │           │                                               ││ │
│  │  │           ▼                                               ││ │
│  │  └─────────────────────────────────────────────────────────────┘│ │
│  │                        │                                           │ │
│  │                        │                                           │ │
│  │                        ▼                                           │ │
│  │  ┌─────────────────────────────────────────────────────────────┐│ │
│  │  │                   파일 (디스크)                               ││ │
│  │  │  [Block 0] [Block 1] [Block 2] [Block 3]                    ││ │
│  │  └─────────────────────────────────────────────────────────────┘│ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【매핑 종류】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  MAP_SHARED:                                                            │ │
│  │  • 수정이 파일에 반영                                                    │ │
│  │  • 다른 프로세스와 공유                                                  │ │
│  │  • IPC에 활용                                                          │ │
│  │                                                             │ │
│  │  MAP_PRIVATE:                                                           │ │
│  │  • COW로 수정                                                           │ │
│  │  • 파일은 변경되지 않음                                                  │ │
│  │  • 프로세스 전용 복사본                                                  │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 매핑 상세                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【매핑 속성】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  PROT_READ:   읽기 권한                                                 │ │
│  │  PROT_WRITE:  쓰기 권한                                                 │ │
│  │  PROT_EXEC:   실행 권한                                                 │ │
│  │  PROT_NONE:   접근 불가                                                  │ │
│  │                                                             │ │
│  │  MAP_SHARED:   공유 매핑                                                 │ │
│  │  MAP_PRIVATE:  비공유 매핑                                               │ │
│  │  MAP_FIXED:    특정 주소에 매핑                                          │ │
│  │  MAP_ANONYMOUS: 파일 없이 매핑 (익명)                                    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【매핑 수명 주기】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  1. mmap() 호출 → VMA 생성                                              │ │
│  │  2. 접근 시 페이지 폴트 → 실제 매핑                                      │ │
│  │  3. 사용                                                                │ │
│  │  4. munmap() 호출 → 매핑 해제                                            │ │
│  │  5. 프로세스 종료 시 자동 해제                                            │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【동기화】                                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  msync(): 수정 내용을 파일에 동기화                                       │ │
│  │                                                             │ │
│  │  MS_SYNC:    동기 플러시                                                 │ │
│  │  MS_ASYNC:   비동기 플러시                                               │ │
│  │  MS_INVALIDATE: 캐시 무효화                                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【C mmap】                                                               │
│  ──────────────────                                                  │
│  #include <sys/mman.h>                                               │
│  #include <fcntl.h>                                                   │
│  #include <unistd.h>                                                 │
│                                                                     │
│  // 파일 매핑                                                            │
│  int fd = open("data.bin", O_RDWR);                                 │
│  size_t size = lseek(fd, 0, SEEK_END);                              │
│                                                                     │
│  void* map = mmap(NULL, size, PROT_READ | PROT_WRITE,               │
│                    MAP_SHARED, fd, 0);                              │
│  close(fd);  // 매핑 후 fd 닫기                                          │
│                                                                     │
│  // 사용 (마치 배열처럼)                                                  │
│  char* data = (char*)map;                                           │
│  data[0] = 'H';                                                     │
│                                                                     │
│  // 동기화                                                              │
│  msync(map, size, MS_SYNC);                                         │
│                                                                     │
│  // 해제                                                                │
│  munmap(map, size);                                                 │
│                                                                     │
│  【매핑 정보 확인】                                                        │
│  ──────────────────                                                  │
│  // 프로세스의 매핑 확인                                                   │
│  $ cat /proc/self/maps | head -5                                    │
│  555555554000-555555555000 r--p 00000000 08:01 12345 /bin/cat        │
│  555555555000-555555556000 r-xp 00001000 08:01 12345 /bin/cat        │
│  ...                                                                  │
│  // r--p: 읽기 전용, 공유                                                 │
│  // r-xp: 읽기+실행, 비공유                                               │
│  // rw-p: 읽기+쓰기, 비공유                                               │
│                                                                     │
│  【Java 파일 매핑】                                                        │
│  ──────────────────                                                  │
│  import java.nio.*;                                                   │
│  import java.nio.channels.*;                                          │
│  import java.io.*;                                                    │
│                                                                     │
│  RandomAccessFile file = new RandomAccessFile("data.bin", "rw");   │
│  FileChannel channel = file.getChannel();                           │
│                                                                     │
│  MappedByteBuffer buffer = channel.map(                             │
│      FileChannel.MapMode.READ_WRITE, 0, channel.size());           │
│                                                                     │
│  // 사용                                                                │
│  buffer.put(0, (byte)'H');                                          │
│  buffer.force();  // 동기화                                             │
│                                                                     │
│  channel.close();                                                   │
│  file.close();                                                      │
│                                                                     │
│  【매핑 크기 변경】                                                        │
│  ──────────────────                                                  │
│  // Linux: mremap()                                                     │
│  void* new_map = mremap(old_map, old_size, new_size, MREMAP_MAYMOVE);│
│                                                                     │
│  // POSIX: munmap + mmap                                                │
│  munmap(map, old_size);                                               │
│  map = mmap(NULL, new_size, ...);                                     │
│                                                                     │
│  【매핑 vs read/write 성능】                                              │
│  ──────────────────                                                  │
│  // 큰 파일 순차 읽기                                                      │
│  // read():   시스템 콜 오버헤드                                          │
│  // mmap():   페이지 폴트만 발생, 더 빠름                                  │
│                                                                     │
│  // 작은 랜덤 읽기                                                         │
│  // read():   오버헤드 큼                                                 │
│  // mmap():   오버헤드 작음                                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 파일을 메모리 주소 공간에 매핑
• API: mmap(), munmap(), msync()
• 속성: PROT_*, MAP_*
• 종류: 공유 매핑, 비공유 매핑
• 장점: 시스템 콜 감소, 성능 향상
• 활용: 대용량 파일, 공유 메모리
```
