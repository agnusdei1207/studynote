## 💡 Insight
- ZRAM은 물리 메모리의 일부를 가상 블록 장치로 사용하여, 데이터를 압축(Compression)한 상태로 RAM 내에 저장하는 기술이다.
- 저성능 저장장치(HDD/eMMC)로의 스왑(I/O)을 줄이고, CPU 연산력을 활용해 실질적인 가용 RAM 용량을 늘리는 효과를 준다.
- 모바일 기기(Android) 및 임베디드 시스템에서 시스템 체감 속도를 높이는 핵심 솔루션으로 자리 잡았다.

---

## Ⅰ. ZRAM (Compressed RAM) 개요
### 1. 정의
- RAM의 일부분을 스왑 영역(Swap Space)처럼 할당하되, 저장되는 데이터를 실시간으로 압축하여 담는 소프트웨어 기술.
### 2. 배경
- 물리적 RAM이 부족할 때 디스크로 페이지를 내보내는 전통적인 스왑은 I/O 속도가 너무 느려 시스템이 멈추는 현상이 발생함.
- CPU의 남는 성능으로 압축/해제를 수행하는 것이 디스크 I/O를 기다리는 것보다 훨씬 빠르다는 점에 착안.

📢 섹션 요약 비유: 물건을 창고(디스크)로 옮기는 대신, 집 안에 진공 압축 팩(ZRAM)에 담아 보관하여 집을 더 넓게 쓰는 것과 같습니다.

---

## Ⅱ. ZRAM 작동 매커니즘
### 1. 데이터 흐름 (I/O Path)
- 페이지 아웃(Page Out) 발생 시, 커널은 해당 데이터를 ZRAM 장치로 보냄.
- LZ4, ZSTD, LZO 등의 알고리즘을 사용해 데이터를 1/2 ~ 1/4 크기로 압축하여 RAM의 특정 영역에 저장.

### 2. ASCII 다이어그램: Memory vs ZRAM vs Disk
```text
[ Physical RAM (8GB) ]
+----------------------+----------------------+
|  Active Apps (4GB)   |   ZRAM Device (2GB)  |
|                      |  [Compressed Data]   |
+----------------------+----------------------+
           | (Fast CPU Op)         | (Slow Disk I/O)
           V                       V
    [ LZ4/ZSTD Comp ]      [ Disk Swap Area ]
```

📢 섹션 요약 비유: 쓰지 않는 이불을 그대로 두면 자리를 많이 차지하니까, 압축 주머니에 넣어 부피를 줄인 뒤 옷장(RAM) 구석에 넣어두는 것입니다.

---

## Ⅲ. ZRAM의 장점과 이점
### 1. 실질 가용 메모리 증가
- 압축률이 3:1일 경우, 1GB의 ZRAM 영역은 실제로는 3GB 분량의 메모리 페이지를 수용할 수 있음.
### 2. 지연 시간 (Latency) 감소
- 디스크 I/O는 밀리초(ms) 단위지만, RAM 내 압축/해제는 마이크로초(μs) 단위로 약 1,000배 이상 빠름.
### 3. 저장장치 수명 연장
- 쓰기 횟수가 제한된 SSD/eMMC에 스왑 데이터를 기록하지 않으므로 수명 단축을 방지함.

📢 섹션 요약 비유: 멀리 있는 유료 창고(SSD)까지 짐을 옮기는 트럭비(I/O 비용)를 아끼고, 집안 정리 정돈(CPU 연산)만으로 공간을 확보하는 경제적인 방법입니다.

---

## Ⅳ. 주요 압축 알고리즘 비교
| 알고리즘 | 특징 | 성능 지표 |
| :--- | :--- | :--- |
| **LZO** | 호환성이 좋고 균형 잡힘 | 표준 |
| **LZ4** | 압축/해제 속도가 매우 빠름 | 속도 중시 |
| **ZSTD** | 높은 압축률 제공 | 효율 중시 |

📢 섹션 요약 비유: LZ4는 "손으로 대충 구겨 넣기(빠름)"이고, ZSTD는 "진공청소기로 꽉 짜내기(효율)"입니다.

---

## Ⅴ. ZRAM vs ZSWAP vs ZCACHE
### 1. ZRAM
- 독립된 가상 블록 장치로서 동작함. (디스크 대용)
### 2. ZSWAP
- 디스크 스왑 장치의 앞단에서 압축 캐시(Cache) 역할을 하며, 압축해도 공간이 부족하면 실제 디스크로 밀어냄(Eviction).
### 3. ZCACHE
- 파일 페이지 캐시(Page Cache)를 압축하는 기술.

📢 섹션 요약 비유: ZRAM은 "전용 압축 서랍"이고, ZSWAP은 창고로 가기 전 잠시 대기하는 "압축 대기실"입니다.
