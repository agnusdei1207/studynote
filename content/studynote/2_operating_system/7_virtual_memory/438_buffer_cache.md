# 438. 메모리 압력 (Memory Pressure)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 메모리 부족 상태의 정도
> 2. **가치**: 선제적 메모리 관리
> 3. **융합**: 스와핑, OOM과 연관

---

## Ⅰ. 개요

### 개념 정의

메모리 압력(Memory Pressure)은 **시스템의 메모리 부족 정도를 나타내는 지표**다. 높을수록 메모리가 부족하다. PSI(Pressure Stall Information)로 측정한다.

### 💡 비유: 수도관 압력
메모리 압력은 **수도관 압력**과 같다. 압력이 높으면 물이 잘 안 나온다. 압력을 낮추려면 물을 아껴써야 한다.

### 메모리 압력 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                메모리 압력 구조                                         │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【압력 수준】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  낮음 (Low):                                                             │ │
│  │  • 충분한 여유 메모리                                                     │ │
│  │  • 정상 동작                                                              │ │
│  │                                                             │ │
│  │  중간 (Medium):                                                           │ │
│  │  • 메모리 회수 시작                                                       │ │
│  │  • 페이지 캐시 축소                                                       │ │
│  │                                                             │ │
│  │  높음 (High):                                                             │ │
│  │  • 심각한 메모리 부족                                                     │ │
│  │  • 스와핑 활발                                                           │ │
│  │  • OOM 위험                                                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【PSI (Pressure Stall Information)】                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  some: 일부 작업이 메모리 대기                                          │ │
│  │  full: 모든 작업이 메모리 대기                                           │ │
│  │                                                             │ │
│  │  avg10:  최근 10초 평균                                                  │ │
│  │  avg60:  최근 60초 평균                                                  │ │
│  │  avg300: 최근 300초 평균                                                 │ │
│  │  total:  총 대기 시간 (마이크로초)                                        │ │
│  │                                                             │ │
│  │  $ cat /proc/pressure/memory                                         │ │
│  │  some avg10=0.00 avg60=0.00 avg300=0.00 total=0                      │ │
│  │  full avg10=0.00 avg60=0.00 avg300=0.00 total=0                      │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【메모리 압력과 대응】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  압력 증가 시 OS 대응:                                                   │ │
│  │                                                             │ │
│  │  1. 페이지 캐시 축소                                                     │ │
│  │  2. 슬랩 캐시 회수                                                       │ │
│  │  3. 스와핑 시작                                                          │ │
│  │  4. 프로세스 종료 (OOM)                                                 │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【메모리 압력 확인】                                                    │
│  ──────────────────                                                  │
│  // PSI 확인                                                             │
│  $ cat /proc/pressure/memory                                         │
│  some avg10=2.50 avg60=1.25 avg300=0.50 total=12345678              │
│  full avg10=0.10 avg60=0.05 avg300=0.02 total=123456                │
│                                                                     │
│  // PSI 활성화 확인 (커널 설정)                                          │
│  $ cat /proc/config.gz | gunzip | grep PSI                          │
│  CONFIG_PSI=y                                                        │
│  CONFIG_PSI_DEFAULT_DISABLED is not set                              │
│                                                                     │
│  // 메모리 상태                                                          │
│  $ cat /proc/meminfo | head -10                                      │
│  MemTotal:       16383892 kB                                         │
│  MemFree:         1234567 kB                                         │
│  MemAvailable:    8192456 kB                                         │
│  // MemAvailable이 낮으면 압력 높음                                      │
│                                                                     │
│  【메모리 압력 모니터링 스크립트】                                        │
│  ──────────────────                                                  │
│  #!/bin/bash                                                          │
│  while true; do                                                       │
│      psi=$(cat /proc/pressure/memory | head -1)                      │
│      some=$(echo $psi | grep -oP 'avg10=\K[0-9.]+')                  │
│      if (( $(echo "$some > 10" | bc -l) )); then                     │
│          echo "High memory pressure: $some%"                          │
│          # 알림 또는 조치                                                 │
│      fi                                                               │
│      sleep 5                                                          │
│  done                                                                 │
│                                                                     │
│  【cgroup 메모리 제한으로 압력 관리】                                     │
│  ──────────────────                                                  │
│  // 메모리 제한 설정                                                      │
│  $ mkdir /sys/fs/cgroup/memory/myapp                                 │
│  $ echo 1G > /sys/fs/cgroup/memory/myapp/memory.limit_in_bytes       │
│                                                                     │
│  // cgroup PSI 확인                                                     │
│  $ cat /sys/fs/cgroup/memory/myapp/memory.pressure                  │
│                                                                     │
│  // 프로세스 추가                                                        │
│  $ echo <pid> > /sys/fs/cgroup/memory/myapp/tasks                    │
│                                                                     │
│  【시스템 튜닝】                                                          │
│  ──────────────────                                                  │
│  // swappiness (스왑 vs 회수 선호도)                                     │
│  $ cat /proc/sys/vm/swappiness                                      │
│  60  // 0~100, 낮을수록 회수 선호                                        │
│                                                                     │
│  // min_free_kbytes                                                    │
│  $ cat /proc/sys/vm/min_free_kbytes                                 │
│  // 최소 여유 메모리 확보                                                 │
│                                                                     │
│  // vfs_cache_pressure                                                 │
│  $ cat /proc/sys/vm/vfs_cache_pressure                              │
│  100  // 높을수록 캐시 적극 회수                                          │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 메모리 부족 상태의 정도
• 측정: PSI (Pressure Stall Information)
• 지표: some, full, avg10/60/300
• 수준: Low → Medium → High
• 대응: 캐시 축소 → 스왑 → OOM
• 모니터링: /proc/pressure/memory
• 조치: 메모리 증설, 프로세스 관리
```
