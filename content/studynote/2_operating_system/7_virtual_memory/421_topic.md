# 421. LRU 페이지 교체 (LRU Page Replacement)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 가장 오랫동안 사용하지 않은 페이지 교체
> 2. **가치**: 지역성 활용, 성능 우수
> 3. **융합**: 페이지 교체, 캐시와 연관

---

## Ⅰ. 개요

### 개념 정의

LRU(Least Recently Used) 페이지 교체는 **가장 오랫동안 사용되지 않은 페이지를 교체하는 알고리즘**이다. 시간 지역성을 활용한다.

### 💡 비유: 책상 위 책
LRU는 **책상 위 책 정리**와 같다. 오랫동안 안 읽은 책을 치운다. 최근 읽은 책은 둔다.

### LRU 교체 구조

```

│                LRU 페이지 교체 구조                                     │

│  ──────────────────                                                  │

│  │                                                             │ │
│  │  "과거를 보면 미래를 안다"                                                │ │
│  │                                                             │ │
│  │  • 최근에 사용된 페이지 → 곧 다시 사용 확률 높음                          │ │
│  │  • 오래 사용 안 한 페이지 → 계속 안 쓸 확률 높음                          │ │
│  │                                                             │ │
│  │  시간 지역성(Temporal Locality) 활용                                    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【LRU 리스트】                                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  가장 최근 사용 ←────────────────────→ 가장 오래 사용 안 함        │ │
│  │                                                             │ │
│  │  [A] ← [B] ← [C] ← [D] ← [E]                                        │ │
│  │   ↑                                       ↑                          │ │
│  │   │                                       │                          │ │
│  │  히트 시 여기로                           교체 대상                   │ │
│  │  이동                                                               │ │
│  │                                                             │ │
│  │  페이지 히트: 해당 페이지를 리스트 맨 앞으로 이동                        │ │
│  │  페이지 미스: 리스트 끝 페이지 교체, 새 페이지 맨 앞에 추가               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【예시: 참조 문자열 1,2,3,4,1,2,5,1,2,3,4,5 (3프레임)】                 │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  참조  LRU 리스트(최근→오래)      폴트?  설명                          │ │
│  │  ────  ─────────────────────  ────  ───────                       │ │
│  │   1    [1, -, -]              O    1 로드                           │ │
│  │   2    [2, 1, -]              O    2 로드                           │ │
│  │   3    [3, 2, 1]              O    3 로드                           │ │
│  │   4    [4, 3, 2]              O    1 교체 (가장 오래됨)              │ │
│  │   1    [1, 4, 3]              O    2 교체                           │ │
│  │   2    [2, 1, 4]              O    3 교체                           │ │
│  │   5    [5, 2, 1]              O    4 교체                           │ │
│  │   1    [1, 5, 2]              X    1 히트, 앞으로 이동                │ │
│  │   2    [2, 1, 5]              X    2 히트, 앞으로 이동                │ │
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【간단한 LRU 구현】                                                    │
│  ──────────────────                                                  │
│  #include <stdio.h>                                                  │
│  #include <stdlib.h>                                                 │
│                                                                     │
│  typedef struct Node {                                               │
│      int page;                                                       │
│      struct Node *prev, *next;                                       │
│  } Node;                                                             │
│                                                                     │
│  Node *head = NULL, *tail = NULL;                                    │
│  int lru_faults = 0;                                                 │
│                                                                     │
│  void move_to_front(Node *node) {                                    │
│      if (node == head) return;                                       │
│      if (node->prev) node->prev->next = node->next;                  │
│      if (node->next) node->next->prev = node->prev;                  │
│      if (node == tail) tail = node->prev;                            │
│      node->next = head;                                              │
│      node->prev = NULL;                                              │
│      if (head) head->prev = node;                                    │
│      head = node;                                                    │
│  }                                                                   │
│                                                                     │
│  Node* find_page(int page) {                                         │
│      for (Node *n = head; n; n = n->next)                           │
│          if (n->page == page) return n;                              │
│      return NULL;                                                    │
│  }                                                                   │
│                                                                     │
│  void access_page(int page) {                                        │
│      Node *node = find_page(page);                                   │
│      if (node) {                                                     │
│          move_to_front(node);  // 히트: 앞으로 이동                    │
│      } else {                                                        │
│          lru_faults++;                                               │
│          node = malloc(sizeof(Node));                                │
│          node->page = page;                                          │
│          node->prev = node->next = NULL;                             │
│          // 리스트 앞에 추가                                            │
│          node->next = head;                                          │
│          if (head) head->prev = node;                                │
│          head = node;                                                │
│          if (!tail) tail = node;                                     │
│      }                                                               │
│  }                                                                   │
│                                                                     │
│  【Linux LRU 구현】                                                    │
│  ──────────────────                                                  │
│  // Linux는 실제 LRU 리스트 사용                                        │
│  // Active List + Inactive List                                      │
│                                                                     │
│  // LRU 리스트 상태 확인                                                 │
│  $ cat /proc/zoneinfo | grep -A5 "pages"                            │
│  // Active(anon), Inactive(anon), Active(file), Inactive(file)       │
│                                                                     │
│  // 페이지 교체 통계                                                      │
│  $ cat /proc/vmstat | grep -E "pgmajfault|pgfault"                  │
│  pgfault 1234567                                                     │
│  pgmajfault 1234                                                     │
│                                                                     │
│  【LRU 근사 알고리즘】                                                   │
│  ──────────────────                                                  │
│  // Clock 알고리즘이 실제로 많이 사용됨                                    │
│  // LRU의 근사로 구현이 간단함                                            │
│                                                                     │
│  // 참조 비트(Reference Bit) 활용                                        │
│  // 하드웨어가 자동으로 설정                                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
