# 423. Clock 페이지 교체 (Clock Page Replacement)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 참조 비트로 LRU 근사
> 2. **가치**: 구현 간단, 성능 양호
> 3. **융합**: 페이지 교체, 하드웨어 지원과 연관

---

## Ⅰ. 개요

### 개념 정의

Clock 페이지 교체는 **참조 비트를 사용하여 LRU를 근사하는 알고리즘**이다. 원형 리스트와 포인터로 시계바늘처럼 동작한다.

### 💡 비유: 회전하는 정원사
Clock은 **회전하는 정원사**와 같다. 화분을 돌며 확인한다. 표시된 화분은 남기고, 안 표시된 건 교체한다.

### Clock 교체 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                Clock 페이지 교체 구조                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 구조】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │                        포인터(시계바늘)                                │ │
│  │                            ↓                                         │ │
│  │                    ┌───────────────┐                               │ │
│  │            ┌──────►│  Page 1  [1]  │──────┐                       │ │
│  │            │       └───────────────┘       │                       │ │
│  │            │                               │                       │ │
│  │   ┌────────┴───┐                 ┌────────┴───┐                   │ │
│  │   │ Page 4 [0] │◄────────────────│ Page 2 [1] │                   │ │
│  │   └────────┬───┘                 └────────┬───┘                   │ │
│  │            │                               │                       │ │
│  │            │       ┌───────────────┐       │                       │ │
│  │            └───────│  Page 3  [1]  │◄──────┘                       │ │
│  │                    └───────────────┘                               │ │
│  │                                                             │ │
│  │  [1] = 참조 비트 설정됨 (최근 사용됨)                                   │ │
│  │  [0] = 참조 비트 해제됨 (오래 사용 안 됨)                                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【동작 과정】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  페이지 폴트 발생 시:                                                   │ │
│  │                                                             │ │
│  │  1. 포인터가 가리키는 페이지 확인                                        │ │
│  │  2. 참조 비트가 0이면 → 교체                                           │ │
│  │  3. 참조 비트가 1이면 → 0으로 설정, 다음 페이지로 이동                      │ │
│  │  4. 참조 비트가 0인 페이지 찾을 때까지 반복                                │ │
│  │                                                             │ │
│  │  페이지 접근 시:                                                        │ │
│  │  • 참조 비트를 1로 설정                                                 │ │
│  │  • 포인터 이동 없음                                                     │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【예시】                                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  현재: [1][1][1][0][1]  포인터 위치: Page 1                          │ │
│  │                                                             │ │
│  │  Step 1: Page 1 [1] → 0으로 변경, 이동                               │ │
│  │  Step 2: Page 2 [1] → 0으로 변경, 이동                               │ │
│  │  Step 3: Page 3 [1] → 0으로 변경, 이동                               │ │
│  │  Step 4: Page 4 [0] → 교체! 새 페이지로 대체                           │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

│                Clock 페이지 교체 상세                                   │

│  【장점】                                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │

│  │  • 오버헤드 작음: O(1) 교체 결정                                        │ │
│  │  • 하드웨어 지원: 참조 비트는 CPU가 자동 관리                             │ │

│                                                                     │
│  【개선된 Clock】                                                          │
│  ──────────────────                                                  │

│  │                                                             │ │
│  │  (참조 비트, 수정 비트) 조합:                                            │ │
│  │  (0, 0): 교체 우선순위 1 (최적)                                        │ │
│  │  (0, 1): 교체 우선순위 2                                               │ │
│  │  (1, 0): 교체 우선순위 3                                               │ │
│  │  (1, 1): 교체 우선순위 4 (최악)                                        │ │
│  │                                                             │ │
│  │  수정된 페이지는 디스크 쓰기 필요 → 피하는 게 좋음                         │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【성능 특성】                                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • LRU보다 구현 쉬움                                                    │ │
│  │  • FIFO보다 성능 좋음                                                   │ │
│  │  • 참조 비트 없으면 소프트웨어로 시뮬레이션                               │ │

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Clock 알고리즘 구현】                                                │
│  ──────────────────                                                  │
│  #include <stdio.h>                                                  │
│  #include <stdlib.h>                                                 │
│                                                                     │
│  typedef struct {                                                    │
│      int page;                                                       │
│      int ref_bit;    // 참조 비트                                      │
│  } Frame;                                                            │
│                                                                     │
│  Frame *frames;                                                      │
│  int num_frames;                                                     │
│  int pointer = 0;    // 시계바늘                                       │
│  int clock_faults = 0;                                               │
│                                                                     │
│  void init_clock(int n) {                                            │
│      num_frames = n;                                                 │
│      frames = calloc(n, sizeof(Frame));                              │
│      for (int i = 0; i < n; i++) {                                  │
│          frames[i].page = -1;                                        │
│          frames[i].ref_bit = 0;                                      │
│      }                                                               │
│  }                                                                   │
│                                                                     │
│  int find_page(int page) {                                           │
│      for (int i = 0; i < num_frames; i++) {                         │
│          if (frames[i].page == page) return i;                       │
│      }                                                               │
│      return -1;                                                      │
│  }                                                                   │
│                                                                     │
│  void access_page(int page) {                                        │
│      int idx = find_page(page);                                      │
│      if (idx >= 0) {                                                 │
│          // 페이지 히트: 참조 비트 설정                                   │
│          frames[idx].ref_bit = 1;                                    │
│      } else {                                                        │
│          // 페이지 폴트                                                 │
│          clock_faults++;                                             │
│                                                                     │
│          // 참조 비트가 0인 프레임 찾기                                    │
│          while (frames[pointer].ref_bit == 1) {                     │
│              frames[pointer].ref_bit = 0;  // 0으로 변경               │
│              pointer = (pointer + 1) % num_frames;                  │
│          }                                                           │
│                                                                     │
│          // 교체                                                       │
│          frames[pointer].page = page;                                │
│          frames[pointer].ref_bit = 1;  // 새 페이지는 1                 │
│          pointer = (pointer + 1) % num_frames;                      │
│      }                                                               │
│  }                                                                   │
│                                                                     │
│  【Linux의 Clock 구현】                                                │
│  ──────────────────                                                  │
│  // Linux는 변형된 Clock 알고리즘 사용                                   │
│  // Active/Inactive List + 참조 비트                                   │
│                                                                     │
│  // 페이지 상태 확인                                                      │
│  $ cat /proc/vmstat | grep -E "pgscan|pgsteal"                      │
│  pgscan_kswapd_dma 0                                                 │
│  pgscan_kswapd_normal 12345                                         │
│  pgsteal_kswapd 10000                                                │
│                                                                     │
│  // Active vs Inactive 비율                                            │
│  $ cat /proc/sys/vm/page-queue-size                                 │
│                                                                     │
│  【성능 측정】                                                            │
│  ──────────────────                                                  │
│  // 페이지 폴트 통계                                                      │
│  $ perf stat -e page-faults,major-faults ./program                  │
│                                                                     │
│  // Clock vs LRU 비교                                                  │
│  $ ./bench_page_replace -a clock,lru,optimal -f trace.txt           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 참조 비트로 LRU 근사
• 구조: 원형 버퍼 + 포인터(시계바늘)
• 동작: 참조 비트 0인 페이지 교체
• 장점: 구현 간단, 오버헤드 작음
• 개선: 이중 비트로 성능 향상
• 실제: Linux 등 실제 OS에서 사용
• 성능: FIFO < Clock < LRU ≈ Optimal
```
