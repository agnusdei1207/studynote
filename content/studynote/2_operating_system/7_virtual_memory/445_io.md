# 445. 경로 (Path)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일의 위치를 나타내는 문자열
> 2. **가치**: 파일 시스템 내 파일 식별
> 3. **융합**: 디렉토리, 파일 이름과 연관

---

## Ⅰ. 개요

### 개념 정의

경로(Path)는 **파일 시스템에서 파일이나 디렉토리의 위치를 나타내는 문자열**이다. 디렉토리 구조를 따라 이동하는 방법이다.

### 💡 비유: 주소
경로는 **주소**와 같다. 집(파일)을 찾으려면 국가-도시-거리-번호로 간다. 루트부터 차례대로 간다.

### 경로 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                경로 구조                                                │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【절대 경로 vs 상대 경로】                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  절대 경로 (Absolute Path):                                              │ │
│  │  • 루트 디렉토리(/)부터 시작                                              │ │
│  │  • 전체 경로 명시                                                        │ │
│  │  • 위치에 독립적                                                          │ │
│  │  • 예: /home/user/documents/file.txt                                   │ │
│  │                                                             │ │
│  │  상대 경로 (Relative Path):                                              │ │
│  │  • 현재 디렉토리부터 시작                                                 │ │
│  │  • 현재 위치에 의존                                                       │ │
│  │  • 더 짧음                                                               │ │
│  │  • 예: ./documents/file.txt, ../parent/file.txt                        │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【경로 구성 요소】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  /home/user/documents/file.txt                                         │ │
│  │  │    │    │           │                                               │ │
│  │  │    │    │           └── 파일 이름                                    │ │
│  │  │    │    └── 디렉토리 이름                                             │ │
│  │  │    └── 디렉토리 이름                                                   │ │
│  │  └── 디렉토리 이름 (루트 아래)                                            │ │
│  │  / : 경로 구분자 (유닉스/Linux)                                          │ │
│  │  \ : 경로 구분자 (Windows)                                               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【특수 경로 표현】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  .   : 현재 디렉토리                                                     │ │
│  │  ..  : 부모 디렉토리                                                     │ │
│  │  ~   : 홈 디렉토리 (셸 확장)                                              │ │
│  │  -   : 이전 디렉토리 (cd만 해당)                                          │ │
│  │  /   : 루트 디렉토리                                                     │ │
│  │  *   : 모든 파일 (와일드카드)                                             │ │
│  │  ?   : 한 글자 (와일드카드)                                               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                경로 상세                                                │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【경로 분해】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  /home/user/docs/file.txt                                              │ │
│  │                                                             │ │
│  │  디렉토리 경로 (dirname):  /home/user/docs                              │ │
│  │  파일 이름 (basename):    file.txt                                      │ │
│  │  확장자 (extension):      .txt                                          │ │
│  │  전체 경로 (fullname):    /home/user/docs/file.txt                     │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【경로 정규화】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  입력: /home/user/../user/./docs//file.txt                             │ │
│  │                                                             │ │
│  │  정규화 과정:                                                            │ │
│  │  1. //를 /로 변환                                                       │ │
│  │  2. /./를 /로 변환 (현재 디렉토리 제거)                                   │ │
│  │  3. /dir/..를 제거 (부모로 이동)                                         │ │
│  │                                                             │ │
│  │  결과: /home/user/docs/file.txt                                         │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【최대 경로 길이】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  PATH_MAX: 4096 바이트 (POSIX 표준)                                      │ │
│  │  NAME_MAX: 255 바이트 (파일 이름)                                         │ │
│  │                                                             │ │
│  │  Windows:                                                               │ │
│  │  MAX_PATH: 260 문자 (확장 시 더 김)                                       │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【심볼릭 링크와 경로】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  논리 경로 (Logical Path):                                               │ │
│  │  • 심볼릭 링크를 따라가는 경로                                            │ │
│  │                                                             │ │
│  │  물리 경로 (Physical/Canonical Path):                                   │ │
│  │  • 심볼릭 링크를 해결한 실제 경로                                         │ │
│  │  • realpath() 함수 사용                                                 │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【경로 조작 명령】                                                      │
│  ──────────────────                                                  │
│  // 현재 작업 디렉토리                                                     │
│  $ pwd                                                               │
│  /home/user                                                          │
│                                                                     │
│  // 절대 경로로 이동                                                      │
│  $ cd /var/log                                                       │
│                                                                     │
│  // 상대 경로로 이동                                                      │
│  $ cd ./subdir                                                       │
│  $ cd ../parent                                                      │
│                                                                     │
│  // 홈 디렉토리로 이동                                                     │
│  $ cd ~                                                              │
│  $ cd                                                                │
│                                                                     │
│  // 경로의 일부 추출                                                      │
│  $ dirname /home/user/file.txt                                       │
│  /home/user                                                          │
│                                                                     │
│  $ basename /home/user/file.txt                                      │
│  file.txt                                                            │
│                                                                     │
│  $ basename /home/user/file.txt .txt                                 │
│  file                                                                │
│                                                                     │
│  【프로그래밍에서 경로 다루기】                                            │
│  ──────────────────                                                  │
│  #include <limits.h>                                                 │
│  #include <stdlib.h>                                                 │
│  #include <libgen.h>                                                 │
│  #include <unistd.h>                                                 │
│                                                                     │
│  // 현재 작업 디렉토리 얻기                                                 │
│  char cwd[PATH_MAX];                                                 │
│  getcwd(cwd, sizeof(cwd));                                           │
│  printf("Current: %s\n", cwd);                                        │
│                                                                     │
│  // 작업 디렉토리 변경                                                     │
│  chdir("/home/user");                                                │
│                                                                     │
│  // 경로 분해                                                            │
│  char path[] = "/home/user/file.txt";                                 │
│  char *dir = dirname(path);   // /home/user                            │
│  char *base = basename(path); // file.txt                              │
│                                                                     │
│  // 실제 경로 얻기                                                        │
│  char real[PATH_MAX];                                                │
│  realpath("./file.txt", real);                                        │
│  // /home/user/file.txt                                               │
│                                                                     │
│  // 경로 구성                                                            │
│  #include <stdio.h>                                                  │
│  char fullpath[PATH_MAX];                                            │
│  snprintf(fullpath, sizeof(fullpath), "%s/%s", dir, filename);        │
│                                                                     │
│  【Python에서 경로 다루기】                                              │
│  ──────────────────                                                  │
│  import os.path                                                      │
│  from pathlib import Path                                            │
│                                                                     │
│  # 경로 분해                                                            │
│  os.path.dirname('/home/user/file.txt')  # '/home/user'              │
│  os.path.basename('/home/user/file.txt') # 'file.txt'                │
│  os.path.splitext('file.txt')            # ('file', '.txt')            │
│                                                                     │
│  # 경로 구성                                                            │
│  os.path.join('/home', 'user', 'file.txt')  # '/home/user/file.txt'   │
│                                                                     │
│  # 실제 경로                                                            │
│  os.path.realpath('./file.txt')                                     │
│                                                                     │
│  # pathlib (객체 지향)                                                  │
│  p = Path('/home/user/file.txt')                                    │
│  p.parent      # Path('/home/user')                                  │
│  p.name        # 'file.txt'                                          │
│  p.suffix      # '.txt'                                              │
│  p.resolve()   # 실제 경로                                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 파일 위치를 나타내는 문자열
• 절대경로: 루트(/)부터 시작
• 상대경로: 현재 디렉토리부터 시작
• 특수문자: ., .., ~, -, /
• 구분자: 유닉스(/), Windows(\)
• 제한: PATH_MAX(4096), NAME_MAX(255)
• 분해: dirname, basename
• 정규화: 중복 제거, 실제 경로 변환
```
