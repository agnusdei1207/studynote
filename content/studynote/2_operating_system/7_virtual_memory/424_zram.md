# 424. Second Chance (2차 기회)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: FIFO에 참조 비트 추가
> 2. **가치**: FIFO 단점 보완
> 3. **융합**: Clock 알고리즘과 동일

---

## Ⅰ. 개요

### 개념 정의

Second Chance는 **FIFO에 참조 비트를 추가하여 개선한 알고리즘**이다. 교체 대상 페이지에 두 번째 기회를 준다. Clock 알고리즘과 동일하다.

### 💡 비유: 재시험 기회
Second Chance는 **재시험 기회**와 같다. 떨어질 학생도 참고서를 읽었으면(참조 비트=1) 기회를 준다.

### Second Chance 구조

│                Second Chance 구조                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │

│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  FIFO:                                                                 │ │
│  │  • 무조건 가장 오래된 페이지 교체                                        │ │
│  │  • 참조 비트 고려 안 함                                                 │ │
│  │                                                             │ │
│  │  Second Chance:                                                        │ │
│  │  • 가장 오래된 페이지 확인                                              │ │
│  │  • 참조 비트 = 1 → 0으로 변경, 큐 뒤로 이동 (2차 기회)                    │ │
│  │  • 참조 비트 = 0 → 교체                                                 │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【동작 예시】                                                              │

│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  큐 상태: [A,1] → [B,0] → [C,1] → [D,0]  (페이지, 참조비트)           │ │
│  │           ↑                                                           │ │
│  │        교체 대상                                                       │
│  │                                                             │ │
│  │  Step 1: A 확인, ref=1 → 0으로 변경, 큐 뒤로 이동                     │ │
│  │          [B,0] → [C,1] → [D,0] → [A,0]                               │ │
│  │                                                             │ │
│  │  Step 2: B 확인, ref=0 → 교체!                                        │ │
│  │          [새페이지,1] → [C,1] → [D,0] → [A,0]                         │ │
│  │                                                             │ │
│  │  새 페이지는 참조 비트 = 1로 시작                                        │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【Clock과의 동일성】                                                      │

│  │                                                             │ │
│  │  Second Chance = Clock Algorithm                                      │ │
│  │                                                             │ │
│  │  • 큐를 원형으로 연결 = 원형 버퍼                                        │ │
│  │  • 큐 앞에서 뒤로 이동 = 포인터 회전                                     │ │
│  │  • 동일한 알고리즘의 다른 표현                                           │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Second Chance 구현】                                                │
│  ──────────────────                                                  │
│  #include <stdio.h>                                                  │
│  #include <stdlib.h>                                                 │
│                                                                     │
│  typedef struct {                                                    │
│      int page;                                                       │
│      int ref_bit;                                                    │
│  } Frame;                                                            │
│                                                                     │
│  Frame *queue;                                                       │
│  int front = 0, rear = 0, count = 0;                                 │
│  int capacity;                                                       │
│  int sc_faults = 0;                                                  │
│                                                                     │
│  void enqueue(int page, int ref) {                                   │
│      queue[rear].page = page;                                        │
│      queue[rear].ref_bit = ref;                                      │
│      rear = (rear + 1) % capacity;                                   │
│      count++;                                                        │
│  }                                                                   │
│                                                                     │
│  int dequeue() {                                                     │
│      int page = queue[front].page;                                   │
│      front = (front + 1) % capacity;                                 │
│      count--;                                                        │
│      return page;                                                    │
│  }                                                                   │
│                                                                     │
│  int find_page(int page) {                                           │
│      for (int i = 0, idx = front; i < count; i++) {                 │
│          if (queue[idx].page == page) {                              │
│              queue[idx].ref_bit = 1;  // 히트 시 참조 비트 설정        │
│              return 1;                                               │
│          }                                                           │
│          idx = (idx + 1) % capacity;                                 │
│      }                                                               │
│      return 0;                                                       │
│  }                                                                   │
│                                                                     │
│  void replace_page(int new_page) {                                   │
│      sc_faults++;                                                    │
│      if (count < capacity) {                                         │
│          enqueue(new_page, 1);                                       │
│          return;                                                     │
│      }                                                               │
│                                                                     │
│      // Second Chance 로직                                            │
│      while (1) {                                                     │
│          if (queue[front].ref_bit == 0) {                            │
│              dequeue();  // 교체                                      │
│              enqueue(new_page, 1);                                   │
│              return;                                                 │
│          } else {                                                    │
│              // 2차 기회: 뒤로 이동                                     │
│              queue[front].ref_bit = 0;                               │
│              Frame temp = queue[front];                              │
│              dequeue();                                              │
│              enqueue(temp.page, temp.ref_bit);                       │
│          }                                                           │
│      }                                                               │
│  }                                                                   │
│                                                                     │
│  【실제 시스템에서의 활용】                                              │
│  ──────────────────                                                  │
│  // 대부분의 OS가 Second Chance/Clock 변형 사용                         │
│                                                                     │
│  // BSD Unix: Clock 알고리즘                                           │
│  // Linux: Two-List Strategy (Active + Inactive)                     │
│  // Windows: Modified Clock                                           │
│                                                                     │
│  // Linux 페이지 교체 동작 확인                                          │
│  $ cat /proc/vmstat | grep pgrefill                                 │
│  $ cat /proc/vmstat | grep pgsteal                                  │
│                                                                     │
│  【성능 비교】                                                            │
│  ──────────────────                                                  │
│  // 같은 참조 문자열로 비교                                               │
│  FIFO:           10 폴트                                              │
│  Second Chance:   8 폴트                                              │
│  LRU:             7 폴트                                              │
│  Optimal:         6 폴트                                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: FIFO + 참조 비트
• 동작: 참조 비트=1이면 2차 기회
• 동일: Clock 알고리즘과 같음
• 장점: FIFO보다 성능 좋음
• 구현: 큐 + 참조 비트
• 실제: 많은 OS에서 사용
• 위치: FIFO < Second Chance < LRU
```
