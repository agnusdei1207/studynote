# 604. HDD (Hard Disk Drive)
#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 자기장 회전하는 디스크
> 2. **가치**: 대용량, 저가격, 높은 신뢰성
> 3. **융합**: SSD, RAID, 파일 시스템과 연관

---

## Ⅰ. 개요

### 개념 정의
**HDD(Hard Disk Drive)**는 **회전하는 자기장과 플래터(Platter)로 구성된 디스크 장치**입니다.
### 💡 비유: 거대한 도서관
HDD는 **거대한 도서관**과 같습니다. 많은 책장이 한꺍에 직접 접근할 수 있습니다.
### HDD 구조

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                HDD 상세 분석                                          │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【HDD 스마트 특성】                                             │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  • CMR (Cache Management): 디스크 캐시 자동 관리                      │ │   │
│  │  • S.M.A.R.T. (Self-Monitoring): 디스크 상태 모니터링              │ │   │
│  │  • NCQ (Native Command Queuing): 다중 I/O 최적화                    │ │   │
│  │  • Advanced Format (4KB 섹터): 물리 섹터 4KB 지원                   │ │   │
│  │  • ECC (Error Correction): 오류 정정 코드                          │ │   │
│  │  • 터보 기술: 진동 방지, 소음 감소                         │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【HDD 인터페이스 진화】                                            │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  인터페이스          특징                                          │ │   │
│  │  ────              ────                                          │ │   │
│  │  IDE/PATA          병렬, 40/80핀, 저속, 높은 호환성               │ │   │
│  │  SATA             직렬, 7핀, 1포트당, 6Gb/s, 750MB/s            │ │   │
│  │  SAS (SAS)        병렬, 50핀, 높은 신뢰성, 서버급               │ │   │
│  │  NVMe             직렬, PCIe, 4레인, 32Gb/s, 6400MB/s          │ │   │
│  │  SCSI             병렬, 50핀, 320MB/s, 640MB/s            │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【HDD 정보 확인】                                                 │
│  ──────────────────                                                │
│  // lsblk                                                            │
│  $ lsblk                                                            │
│  NAME   MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT           │
│  sda      8:0    465.8G    disk                           │
│  ├─0    8:0      465.8G    disk                        │
│  sr0     179:0  179G    disk  part                         │
│  sdb      8:16   931.5G   disk                        │
│  sr0     8:16   931.5G   disk                        │
│                                                                     │
│  // fdisk                                                            │
│  $ sudo fdisk -l /dev/sda                                           │
│                                                                     │
│  // hdparm (ATA)                                                  │
│  $ sudo hdparm -I /dev/sda               // 상세 정보                  │
│  $ sudo hdparm -t /dev/sda               // 읽기 성능                  │
│  $ sudo hdparm -Tt /dev/sda             // 캐시 성능                  │
│                                                                     │
│  // smartctl (SATA/AHCI)                                             │
│  $ sudo smartctl -a /dev/sda                                    │
│  $ sudo smartctl -H /dev/sda                                     │
│  $ sudo smartctl -t /dev/sda 1                // 1초 값스 5회 테스트             │
│  $ sudo smartctl -c /dev/sda -q                           │
│  $ sudo smartctl -x /dev/sda              // 상세 정보                  │
│                                                                     │
│  // HDD 건강 상태                                                  │
│  $ sudo smartctl -H /dev/sda | grep -i "SMART"                 │
│                                                                     │
│  // 배드 섹터 재매핑                                                  │
│  $ sudo smartctl -l /dev/sda                                     │
│                                                                     │
│  // 온도 확인                                                          │
│  $ sudo smartctl -a /dev/sda | grep -i "Temperature"              │
│  Current Drive Temperature:  35 (정상)                      │
│  Temperature:  36 Celsius                                      │
│                                                                     │
│  // SMRLL/SCT/LLC/RC 상태                                                 │
│  $ sudo smartctl -a /dev/sda | grep -i "SMART"                │
│  Available Spare: 100%                                │
│  Reallocated Sectors: 0                               │
│  Power on Hours: 0                                             │
│                                                                     │
│  // 에러 로그 확인                                                    │
│  $ sudo smartctl -l error /dev/sda                               │
│                                                                     │
│  // 식별 정보 확인                                                      │
│  $ sudo smartctl -i /dev/sda                                          │
│  Model Number: ST1000DM001                          │
│  Serial Number:    S1WMJXG2B0F5D               │
│  Firmware:    1.01                                        │
│                                                                     │
│  // 쓰기 테스트                                                          │
│  $ sudo dd if=/dev/zero of=/dev/null bs=1M count=1                │
│  $ sudo dd if=/dev/zero of=/tmp/testfile bs=1M count=1             │
│                                                                     │
│  // 읽기 테스트                                                          │
│  $ sudo hdparm -t /dev/sda                                           │
│  $ sudo hdparm -tT /dev/sda                                          │
│                                                                     │
│  // 캐시 드롭 확인                                                      │
│  $ sudo hdparm -I /dev/sda | grep -i "drive"                │
│  *   drive buffer size:  16384 KBytes                       │
│  Max drive buffer size:  1                            │
│                                                                     │
│  // APM 설정 (hdparm)                                              │
│  $ sudo hdparm -B /dev/sda                     // Advanced Power Management         │
│  $ sudo hdparm -y /dev/sda                     // Display power mode                 │
│                                                                     │
│  // APM 레벨 확인 (hdparm)                                              │
│  $ sudo hdparm -I /dev/sda | grep -i "Advanced"              │
│  *   Advanced: yes                                              │
│  *   Lookahead: yes (read look-ahead)                       │
│  *   Write cache: yes (write-back)                        │
│  *   Hybrid: enhanced (host-aware)                     │
│  *   Power management: yes                                │
│  *   APM Level: 254 (maximum)                      │
│                                                                     │
│  // APM 설정 (sdparm)                                               │
│  $ sudo sdparm --set-apm 254 /dev/sda                             │
│                                                                     │
│  // APM 상태 모니터링                                                │
│  $ watch -n 1 "sudo hdparm -B /dev/sda | grep -i "APM"     │
│                                                                     │
│  【HDD 성능 최적화】                                             │
│  ──────────────────                                                │
│  // I/O 스케줄러 변경                                                  │
│  $ cat /sys/block/sda/queue/scheduler                              │
│  [cfq] deadline none                                         │
│  $ echo cfq | sudo tee /sys/block/sda/queue/scheduler      │
│                                                                     │
│  // read-ahead 조정                                                  │
│  $ sudo blockdev --setra 256 /dev/sda    // 128KB read-ahead      │
│  $ sudo blockdev --getra /dev/sda                               │
│                                                                     │
│  // NCQ 설정 (NVMe SSD)                                              │
│  $ sudo nvme set-feature -f 0x /dev/nvme0n1       │
│  $ sudo nvme get-feature -f 0 /dev/nvme0n1                         │
│                                                                     │
│  // fstrim (SSD 최적화)                                               │
│  $ sudo fstrim -v /dev/sda                                           │
│  $ sudo fstrim / /dev/sda                                          │
│                                                                     │
│  // 파일시스템 마운트 옵션                                                │
│  $ sudo mount -o noatime /dev/sda1 /mnt                           │
│  $ sudo mount -o discard /dev/sda1 /mnt                            │
│                                                                     │
│  // badblocks 복구                                                  │
│  $ sudo badblocks -v /dev/sda                                         │
│                                                                     │
│  // 디스크 동기화                                                    │
│  $ sudo hdparm -f /dev/sda                                           │
│  $ sudo hdparm -F /dev/sda                                           │
│                                                                     │
│  【HDD 진단】                                                       │
│  ──────────────────                                                │
│  // smartctl (S.M.A.R.T.)                                               │
│  $ sudo smartctl -t short /dev/sda                                 │
│  $ sudo smartctl -a /dev/sda                                        │
│  $ sudo smartctl -H /dev/sda | grep -i "SMART"                │
│  SMART support: Available - device lacks SMART capability.           │
│                                                                     │
│  // 디스크 오류 확인                                                    │
│  $ sudo smartctl -l error /dev/sda                                 │
│  $ sudo smartctl -a /dev/sda | grep -i "Error"                  │
│  No Errors Logged                                                 │
│                                                                     │
│  // 불량 섹터 확인                                                    │
│  $ sudo smartctl -l /dev/sda                                        │
│                                                                     │
│  // 디스크 테스트                                                        │
│  $ sudo badblocks -v /dev/sda                                       │
│  $ sudo badblocks -n /dev/sda                                       │
│                                                                     │
│  // 완전 삭제 (보안)                                                     │
│  $ sudo hdparm --security-erase /dev/sda                            │
│  $ sudo shred -vfz -n 3 /dev/sda                                   │
│                                                                     │
│  // 수명 확인                                                          │
│  $ sudo smartctl -a /dev/sda | grep -i "Power_On"               │
│  PowerOnHours: 35151                                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 회전하는 자기장과 플래터로 구성된 디스크
• 구조: 플래터, 스핀들, 액추, 커넥터, PCB
• 인터페이스: PATA, SATA, SAS, NVMe
• 속도: 80-250 MB/s (PATA) ~ 750 MB/s (SATA 3.2)
• 장점: 대용량, 저가격, 호환성
• 단점: 기계식, 느린 속도, 소음/발열
• 최적화: APM, NCQ, TRIM
• 확인: lsblk, fdisk, hdparm, smartctl
• 진단: smartctl -a, badblocks
• 성능: hdparm -t, dd
• 수명: 3-5년 (HDD), 5-10년 (SSD)
• 서버: SATA/SAS (호환성), NVMe (성능)
```
