# 469. 자동 마운트 (Auto Mount)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 장치 연결 시 자동으로 파일 시스템 마운트
> 2. **가치**: 사용자 편의성 향상
> 3. **융합**: 마운트, fstab, udev와 연관

---

## Ⅰ. 개요

### 개념 정의

자동 마운트(Auto Mount)는 **저장 장치 연결 감지 시 자동으로 파일 시스템을 마운트하는 기능**이다.

### 💡 비유: 자동 문
자동 마운트는 **자동 문**과 같다. 사람(장치)이 다가오면 자동으로 열린다(마운트).

### 자동 마운트 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                자동 마운트                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【자동 마운트 방식】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. /etc/fstab 자동 마운트:                                           │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  부팅 시 fstab에 등록된 파일시스템 자동 마운트                │    │ │
│  │  │  # <device>  <mount>  <type>  <options>  <dump>  <pass> │    │ │
│  │  │  /dev/sda1   /        ext4   defaults    0       1       │    │ │
│  │  │  /dev/sdb1   /data    ext4   defaults    0       2       │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  2. udev + udisks (데스크탑):                                         │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  USB 연결 → udev 감지 → udisks2 마운트 → 파일 매니저 표시  │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  3. autofs (주문형):                                                 │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  디렉토리 접근 → 마운트 → 일정 시간 후 자동 언마운트          │    │ │
│  │  │  /net/server1  접근 시 → NFS 마운트                       │    │ │
│  │  │  /mnt/cdrom   접근 시 → CD-ROM 마운트                     │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  4. systemd-automount:                                               │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │  systemd 마운트 유닛 + automount 유닛                      │    │ │
│  │  │  접근 시 마운트, 유휴 시 언마운트                              │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【자동 마운트 흐름 (udev)】                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  USB 장치 연결                                                      │
│  │       │                                                             │
│  │       ▼                                                             │
│  │  ┌─────────┐                                                      │ │
│  │  │  커널   │ → /sys/block/sdb 생성                                 │ │
│  │  └─────────┘                                                      │ │
│  │       │                                                             │
│  │       ▼                                                             │
│  │  ┌─────────┐                                                      │ │
│  │  │  udev   │ → 규칙 매칭, 이벤트 생성                               │ │
│  │  └─────────┘                                                      │ │
│  │       │                                                             │
│  │       ▼                                                             │
│  │  ┌───────────────┐                                                │ │
│  │  │  udisks2      │ → /media/user/UUID에 마운트                     │ │
│  │  │  (D-Bus 서비스)│                                                │ │
│  │  └───────────────┘                                                │ │
│  │       │                                                             │
│  │       ▼                                                             │
│  │  ┌───────────────┐                                                │ │
│  │  │  파일 매니저  │ → 아이콘 표시                                    │ │
│  │  └───────────────┘                                                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                자동 마운트 상세                                        │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【/etc/fstab 옵션】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  auto: 부팅 시 자동 마운트 (기본)                                     │ │
│  │  noauto: 자동 마운트 안 함                                            │ │
│  │  user: 일반 사용자 마운트 가능                                        │ │
│  │  users: 모든 사용자 마운트/언마운트 가능                                │ │
│  │  x-systemd.automount: 접근 시 마운트                                  │ │
│  │  x-systemd.idle-timeout=10s: 10초 유휴 시 언마운트                     │ │
│  │                                                             │ │
│  │  예시:                                                               │
│  │  /dev/sdb1  /mnt/usb  vfat  noauto,users,x-systemd.automount 0 0   │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【autofs 설정】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  /etc/auto.master:                                                  │ │
│  │  /mnt    /etc/auto.misc --timeout=60                               │ │
│  │  /net    -hosts                                                     │ │
│  │                                                             │ │
│  │  /etc/auto.misc:                                                     │
│  │  cdrom    -fstype=iso9660,ro    :/dev/cdrom                        │ │
│  │  usb      -fstype=vfat         :/dev/sdb1                          │
│  │  server   -fstype=nfs,soft     server1:/export/data               │ │
│  │                                                             │ │
│  │  /mnt/cdrom 접근 → 60초간 사용 없으면 자동 언마운트                      │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  장점:                                                               │
│  │  • 사용자 편의성 향상                                                  │ │
│  │  • 수동 작업 감소                                                      │ │
│  │  • 리소스 효율성 (필요할 때만 마운트)                                    │ │
│  │                                                             │ │
│  │  단점:                                                               │
│  │  • 보안 위험 (악성 장치 자동 실행)                                      │ │
│  │  • 예기치 않은 동작                                                    │ │
│  │  • 설정 오류 시 부팅 문제                                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【fstab 자동 마운트 설정】                                          │
│  ──────────────────                                                  │
│  // UUID로 영구 마운트 설정                                            │
│  $ sudo blkid /dev/sdb1                                             │
│  /dev/sdb1: UUID="1234-ABCD" TYPE="vfat"                            │
│                                                                     │
│  // /etc/fstab 추가                                                   │
│  UUID=1234-ABCD  /mnt/usb  vfat  defaults,users  0  0               │
│                                                                     │
│  // systemd automount 사용                                            │
│  UUID=1234-ABCD  /mnt/usb  vfat  noauto,x-systemd.automount 0 0     │
│                                                                     │
│  【autofs 설정】                                                      │
│  ──────────────────                                                  │
│  // 설치                                                              │
│  $ sudo apt install autofs                                          │
│                                                                     │
│  // /etc/auto.master 설정                                             │
│  /mnt    /etc/auto.misc --timeout=300                               │
│                                                                     │
│  // /etc/auto.misc 설정                                               │
│  data    -fstype=ext4    :/dev/sdb1                                 │
│  nfs     -fstype=nfs     server:/export                             │
│                                                                     │
│  // 서비스 재시작                                                      │
│  $ sudo systemctl restart autofs                                    │
│                                                                     │
│  // 접근 시 자동 마운트                                                │
│  $ ls /mnt/data  // 자동으로 /dev/sdb1 마운트                         │
│                                                                     │
│  【systemd 마운트 유닛】                                              │
│  ──────────────────                                                  │
│  // /etc/systemd/system/mnt-data.mount                              │
│  [Unit]                                                              │
│  Description=Data Partition                                         │
│                                                                     │
│  [Mount]                                                             │
│  What=/dev/sdb1                                                     │
│  Where=/mnt/data                                                    │
│  Type=ext4                                                          │
│  Options=defaults                                                   │
│                                                                     │
│  [Install]                                                           │
│  WantedBy=multi-user.target                                         │
│                                                                     │
│  // /etc/systemd/system/mnt-data.automount                          │
│  [Unit]                                                              │
│  Description=Automount Data Partition                               │
│                                                                     │
│  [Automount]                                                         │
│  Where=/mnt/data                                                    │
│  TimeoutIdleSec=300                                                 │
│                                                                     │
│  [Install]                                                           │
│  WantedBy=multi-user.target                                         │
│                                                                     │
│  // 활성화                                                            │
│  $ sudo systemctl enable mnt-data.automount                         │
│  $ sudo systemctl start mnt-data.automount                          │
│                                                                     │
│  【자동 마운트 확인】                                                  │
│  ──────────────────                                                  │
│  // 마운트 목록                                                        │
│  $ findmnt -A  // automount 목록                                     │
│                                                                     │
│  // systemd 마운트 상태                                                │
│  $ systemctl list-units --type=mount                                │
│  $ systemctl status mnt-data.automount                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 장치 연결 시 자동 마운트
• 방식: fstab, udev+udisks, autofs, systemd
• fstab: defaults, auto, noauto, users
• autofs: 접근 시 마운트, 유휴 시 언마운트
• systemd: .mount + .automount 유닛
• 장점: 편의성, 리소스 효율
• 단점: 보안 위험, 예기치 않은 동작
• 확인: findmnt -A, systemctl list-units
```
