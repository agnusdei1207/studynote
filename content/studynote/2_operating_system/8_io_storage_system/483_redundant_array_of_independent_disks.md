# 584. RAID (Redundant Array of Independent Disks)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 여러 디스크를 논리적으로 결합
> 2. **가치**: 성능 향상, 데이터 보호, 가용성
> 3. **융합**: 스트라이핑, 미러링, 패리티와 연관

---

## Ⅰ. 개요

### 개념 정의
**RAID**는 **여러 개의 물리적 디스크를 논리적으로 결합하여 성능과 신뢰성을 향상시키는 기술**입니다.

### 💡 비유: 팀워크
RAID는 **팀워크**와 같습니다. 여러 사람이 협력해서 더 빠르고 안전하게 일합니다.

### RAID 구조
```

├─────────────────────────────────────────────────────────────────────┤
│                                                                     │

│  │                                                             │ │   │

│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  데이터: [A1][A2][A3][A4]                                       │ │   │
│  │                                                             │ │   │
│  │  ┌─────────────┐    ┌─────────────┐                          │ │   │

│  │  └─────────────┘    └─────────────┘                          │ │   │
│  │                                                             │ │   │

│  │                                                             │ │   │
│  │  ┌─────────────────────┐    ┌─────────────────────┐        │ │   │
│  │  │     Mirror Set 1    │    │     Mirror Set 2    │        │ │   │
│  │  │  ┌─────┐  ┌─────┐   │    │  ┌─────┐  ┌─────┐   │        │ │   │
│  │  │  │ D1  │═│ D2  │   │════│  │ D3  │═│ D4  │   │        │ │   │
│  │  │  │[A1] │ │[A1] │   │    │  │[A2] │ │[A2] │   │        │ │   │

│  │  │  └─────┘  └─────┘   │    │  └─────┘  └─────┘   │        │ │   │
│  │  └─────────────────────┘    └─────────────────────┘        │ │   │
│  │           │                          │                       │ │   │

│  │                                                             │ │   │
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【mdadm RAID 생성】                                                 │
│  ──────────────────                                                │
│  // RAID 0 (스트라이핑)                                               │
│  $ sudo mdadm --create /dev/md0 --level=0 --raid-devices=2 \       │
│    /dev/sdb1 /dev/sdc1                                             │
│                                                                     │
│  // RAID 1 (미러링)                                                  │
│  $ sudo mdadm --create /dev/md0 --level=1 --raid-devices=2 \       │
│    /dev/sdb1 /dev/sdc1                                             │
│                                                                     │
│  // RAID 5 (분산 패리티)                                              │
│  $ sudo mdadm --create /dev/md0 --level=5 --raid-devices=4 \       │
│    /dev/sd{b,c,d,e}1                                               │
│                                                                     │
│  // RAID 10                                                         │
│  $ sudo mdadm --create /dev/md0 --level=10 --raid-devices=4 \      │
│    /dev/sd{b,c,d,e}1                                               │
│                                                                     │
│  // Hot Spare 추가                                                   │
│  $ sudo mdadm --create /dev/md0 --level=5 --raid-devices=3 \       │
│    /dev/sd{b,c,d}1 --spare-devices=1 /dev/sde1                     │
│                                                                     │
│  【RAID 관리】                                                       │
│  ──────────────────                                                │
│  // RAID 상태 확인                                                    │
│  $ cat /proc/mdstat                                                 │
│  $ sudo mdadm --detail /dev/md0                                    │
│  $ sudo mdadm -D /dev/md0                                          │
│                                                                     │
│  // RAID 배열 스캔                                                    │
│  $ sudo mdadm --detail --scan                                      │
│  $ sudo mdadm -Es                                                  │
│                                                                     │
│  // 설정 저장                                                        │
│  $ sudo mdadm --detail --scan >> /etc/mdadm.conf                   │
│  $ sudo mdadm -Es >> /etc/mdadm.conf                               │
│                                                                     │
│  // 디스크 추가                                                       │
│  $ sudo mdadm --add /dev/md0 /dev/sdf1                             │
│  $ sudo mdadm --grow /dev/md0 --raid-devices=5                     │
│                                                                     │
│  // 디스크 제거                                                       │
│  $ sudo mdadm --fail /dev/md0 /dev/sdb1                            │
│  $ sudo mdadm --remove /dev/md0 /dev/sdb1                          │
│                                                                     │
│  // Hot Spare 수동 리빌드                                             │
│  $ sudo mdadm --rebuild /dev/md0 /dev/sdb1                         │
│                                                                     │
│  【RAID 모니터링】                                                    │ |
│  ──────────────────                                                │
│  // 실시간 모니터링                                                    │
│  $ watch cat /proc/mdstat                                          │
│                                                                     │
│  // 이메일 알림 설정                                                   │
│  $ sudo mdadm --monitor --daemonise --mail=admin@example.com       │
│                                                                     │
│  // /etc/mdadm/mdadm.conf                                           │
│  MAILADDR admin@example.com                                        │
│                                                                     │
│  【RAID 삭제】                                                       │
│  ──────────────────                                                │
│  // RAID 중지                                                        │
│  $ sudo mdadm --stop /dev/md0                                      │
│                                                                     │
│  // 슈퍼블록 삭제                                                      │
│  $ sudo mdadm --zero-superblock /dev/sdb1                          │
│  $ sudo mdadm --zero-superblock /dev/sdc1                          │
│                                                                     │
│  【ZFS RAID-Z】                                                      │
│  ──────────────────                                                │
│  // RAID-Z1 (RAID 5 equivalent)                                     │
│  $ sudo zpool create mypool raidz /dev/sdb /dev/sdc /dev/sdd       │
│                                                                     │
│  // RAID-Z2 (RAID 6 equivalent)                                     │
│  $ sudo zpool create mypool raidz2 /dev/sd{b,c,d,e}                │
│                                                                     │
│  // RAID-Z3 (트리플 패리티)                                           │
│  $ sudo zpool create mypool raidz3 /dev/sd{b,c,d,e,f}              │
│                                                                     │
│  // 미러 (RAID 1)                                                    │
│  $ sudo zpool create mypool mirror /dev/sdb /dev/sdc               │
│                                                                     │
│  // 상태 확인                                                        │
│  $ sudo zpool status                                                │
│  $ sudo zpool list                                                  │
│                                                                     │
│  【LVM + RAID】                                                      │
│  ──────────────────                                                │
│  // LVM RAID 생성                                                    │
│  $ sudo lvcreate --type raid1 -L 100G -n mylv vg0                  │
│  $ sudo lvcreate --type raid5 -L 100G -n mylv vg0                  │
│                                                                     │
│  // LVM RAID 상태                                                    │
│  $ sudo lvs -a -o +devices                                         │
│  $ sudo lvdisplay /dev/vg0/mylv                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 여러 디스크를 논리적 결합
• RAID 0: 스트라이핑, 최고 성능, 무보호
• RAID 1: 미러링, 50% 용량, 높은 보호
• RAID 5: 분산 패리티, 1디스크 보호
• RAID 6: 이중 패리티, 2디스크 보호
• RAID 10: 미러+스트라이프, DB 권장
• 소프트웨어 RAID: mdadm
• ZFS: RAID-Z, RAID-Z2, RAID-Z3
• Hot Spare: 예비 디스크 자동 투입
• 생성: mdadm --create
• 확인: /proc/mdstat, mdadm --detail
• 복구: mdadm --add, --grow
• 모니터링: mdadm --monitor
```
