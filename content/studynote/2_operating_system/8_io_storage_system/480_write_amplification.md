# 480. 디스크 쿼터 (Disk Quota)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 사용자별 디스크 사용량 제한
> 2. **가치**: 공정한 자원 분배와 시스템 보호
> 3. **융합**: 파일 시스템, 사용자 관리와 연관

---

## Ⅰ. 개요

### 개념 정의

디스크 쿼터(Disk Quota)는 **사용자나 그룹별로 디스크 사용량(블록, 아이노드)을 제한하는 시스템**이다.

### 💡 비유: 식당 주문 제한
디스크 쿼터는 **식당 주문 제한**과 같다. 각 손님(사용자)이 주문할 수 있는 음식(디스크) 양에 한도가 있다.

### 디스크 쿼터 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                디스크 쿼터                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【쿼터 종류】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 블록 쿼터 (Block Quota):                                          │ │
│  │     • 디스크 공간 사용량 제한                                           │ │
│  │     • KB, MB, GB 단위                                                 │ │
│  │     • "용량 얼마까지 사용 가능"                                          │ │
│  │                                                             │ │
│  │  2. 아이노드 쿼터 (Inode Quota):                                      │ │
│  │     • 파일/디렉토리 개수 제한                                           │ │
│  │     • "몇 개의 파일까지 생성 가능"                                       │ │
│  │     • 많은 작은 파일 방지                                               │ │
│  │                                                             │ │
│  │  3. 사용자 쿼터 (User Quota):                                         │ │
│  │     • 개별 사용자별 제한                                                │ │
│  │     • UID 기반                                                        │ │
│  │                                                             │ │
│  │  4. 그룹 쿼터 (Group Quota):                                          │ │
│  │     • 그룹 전체 합계 제한                                               │ │
│  │     • GID 기반                                                        │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【쿼터 한도 구조】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │                    사용량                                 │    │ │
│  │  │    0        soft       hard                               │    │ │
│  │  │    ├─────────┼─────────┼─────────┤                        │    │ │
│  │  │    │  정상   │  경고   │  차단   │                        │    │ │
│  │  │    │         │(유예기간)│         │                        │    │ │
│  │  │    │         │         │         │                        │    │ │
│  │  │  grace period: soft 초과 시 허용되는 유예 기간               │    │ │
│  │  │  (기본 7일, 변경 가능)                                       │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  soft limit: 소프트 한도 (경고, 유예기간 있음)                        │ │
│  │  hard limit: 하드 한도 (절대 한도, 초과 불가)                         │ │
│  │  grace period: 유예 기간 (soft 초과 시)                               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                디스크 쿼터 상세                                        │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【쿼터 설정 파일】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  /etc/fstab에 쿼터 옵션 추가:                                          │ │
│  │  /dev/sda1  /home  ext4  defaults,usrquota,grpquota  0  2        │ │
│  │                                                             │ │
│  │  쿼터 파일:                                                             │ │
│  │  • aquota.user: 사용자 쿼터                                            │ │
│  │  • aquota.group: 그룹 쿼터                                            │ │
│  │  • 파일 시스템 루트에 위치                                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【쿼터 관리 명령어】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  quotaon/quotaoff: 쿼터 활성화/비활성화                                │ │
│  │  quotacheck: 쿼터 파일 생성/갱신                                       │ │
│  │  edquota: 쿼터 편집                                                    │ │
│  │  repquota: 쿼터 보고서                                                  │ │
│  │  setquota: 명령줄로 쿼터 설정                                           │ │
│  │  warnquota: 쿼터 초과 사용자에게 경고                                   │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【쿼터 장점】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  • 공정한 자원 분배                                                    │ │
│  │  • 디스크 가득 참 방지                                                  │ │
│  │  • 악의적 사용 방지                                                    │ │
│  │  • 사용량 추적                                                         │ │
│  │  • 시스템 안정성 확보                                                   │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【쿼터 설정 과정】                                                    │
│  ──────────────────                                                  │
│  // 1. 패키지 설치                                                    │
│  $ sudo apt install quota                                           │
│                                                                     │
│  // 2. /etc/fstab 수정                                                │
│  /dev/sda1  /home  ext4  defaults,usrquota,grpquota  0  2          │
│                                                                     │
│  // 3. 재마운트                                                       │
│  $ sudo mount -o remount /home                                      │
│                                                                     │
│  // 4. 쿼터 파일 생성                                                  │
│  $ sudo quotacheck -cugm /home                                      │
│  // aquota.user, aquota.group 생성                                   │
│                                                                     │
│  // 5. 쿼터 활성화                                                     │
│  $ sudo quotaon -vug /home                                          │
│                                                                     │
│  【사용자 쿼터 설정】                                                  │
│  ──────────────────                                                  │
│  // 대화형 편집                                                        │
│  $ sudo edquota -u username                                         │
│  // 에디터에서 설정                                                    │
│  // Filesystem  blocks  soft  hard  inodes  soft  hard             │
│  // /dev/sda1       100  5000  6000      20    100   120           │
│  // (블록: KB, 5GB soft, 6GB hard)                                    │
│                                                                     │
│  // 명령줄로 설정                                                       │
│  $ sudo setquota -u username 5G 6G 100 120 /home                   │
│  // setquota -u user 블록_soft 블록_hard inode_soft inode_hard FS    │
│                                                                     │
│  // 유예 기간 설정                                                      │
│  $ sudo edquota -t                                                   │
│  // block grace: 7days                                               │
│  // inode grace: 7days                                               │
│                                                                     │
│  【그룹 쿼터 설정】                                                      │
│  ──────────────────                                                  │
│  $ sudo edquota -g groupname                                        │
│  $ sudo setquota -g groupname 10G 12G 200 250 /home                │
│                                                                     │
│  【쿼터 확인】                                                          │
│  ──────────────────                                                  │
│  // 전체 보고서                                                        │
│  $ sudo repquota -aug                                               │
│  *** Report for user quotas on device /dev/sda1                    │
│  Block grace time: 7days                                           │
│  Block limits                File limits                          │
│  User  used   soft   hard  grace  used  soft  hard  grace          │
│  ────  ────   ────   ────  ─────  ────  ────  ────  ─────          │
│  user1  1000   5000   6000        50    100   120                  │
│  user2  2000   5000   6000        80    100   120                  │
│                                                                     │
│  // 개인 사용자 확인                                                   │
│  $ quota -s                                                          │
│  Filesystem  blocks  quota  limit  grace  files  quota  limit      │
│  /dev/sda1    1M     5G    6G           50     100   120           │
│                                                                     │
│  // 쿼터 상태 확인                                                      │
│  $ sudo quotaon -p /home                                             │
│  group quota on /home (/dev/sda1) is on                             │
│  user quota on /home (/dev/sda1) is on                              │
│                                                                     │
│  【쿼터 복사】                                                          │
│  ──────────────────                                                  │
│  // 다른 사용자에게 동일한 쿼터 복사                                     │
│  $ sudo edquota -p reference_user target_user                       │
│                                                                     │
│  // 여러 사용자에게 복사                                                │
│  $ for user in user1 user2 user3; do                                │
│        sudo edquota -p prototype $user                              │
│    done                                                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 사용자/그룹별 디스크 사용량 제한
• 종류: 블록 쿼터, 아이노드 쿼터
• 대상: 사용자 쿼터, 그룹 쿼터
• 한도: soft limit (경고), hard limit (절대)
• 유예: grace period (기본 7일)
• 설정: /etc/fstab에 usrquota/grpquota
• 명령: quotacheck, edquota, repquota
• 파일: aquota.user, aquota.group
```
