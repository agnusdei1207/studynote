# 497. 클라우드 스토리지 (Cloud Storage)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 인터넷을 통한 원격 데이터 저장
> 2. **가치**: 확장성, 가용성, 접근성
> 3. **융합**: 분산 시스템, 가상화, 네트워크와 연관

---

## Ⅰ. 개요

### 개념 정의

클라우드 스토리지(Cloud Storage)는 **인터넷을 통해 원격 서버에 데이터를 저장하고 언제 어디서나 접근할 수 있는 서비스**이다.

### 💡 비유: 은행 금고
클라우드 스토리지는 **은행 금고**와 같다. 내 물건을 안전한 곳에 맡기고, 필요할 때 어디서든 찾아간다.

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【AWS S3 사용】                                                        │
│  ──────────────────                                                  │
│  // 버킷 생성                                                          │
│  $ aws s3 mb s3://my-bucket                                         │
│                                                                     │
│  // 파일 업로드                                                        │
│  $ aws s3 cp file.txt s3://my-bucket/                               │
│  $ aws s3 sync ./local s3://my-bucket/                              │
│                                                                     │
│  // 파일 다운로드                                                      │
│  $ aws s3 cp s3://my-bucket/file.txt ./                             │
│  $ aws s3 sync s3://my-bucket/ ./local                              │
│                                                                     │
│  // 버킷 목록                                                          │
│  $ aws s3 ls                                                        │
│  $ aws s3 ls s3://my-bucket/                                        │
│                                                                     │
│  // 스토리지 클래스 변경                                                │
│  $ aws s3 cp file.txt s3://my-bucket/ --storage-class GLACIER      │
│                                                                     │
│  【Azure Blob Storage 사용】                                           │
│  ──────────────────                                                  │
│  // 컨테이너 생성                                                      │
│  $ az storage container create --name mycontainer                   │
│                                                                     │
│  // 파일 업로드                                                        │
│  $ az storage blob upload --container-name mycontainer \            │
│        --name blobname --file localfile.txt                         │
│                                                                     │
│  // 파일 다운로드                                                      │
│  $ az storage blob download --container-name mycontainer \          │
│        --name blobname --file localfile.txt                         │
│                                                                     │
│  // 목록 확인                                                          │
│  $ az storage blob list --container-name mycontainer                │
│                                                                     │
│  【GCP Cloud Storage 사용】                                            │
│  ──────────────────                                                  │
│  // 버킷 생성                                                          │
│  $ gsutil mb gs://my-bucket                                         │
│                                                                     │
│  // 파일 업로드                                                        │
│  $ gsutil cp file.txt gs://my-bucket/                               │
│  $ gsutil -m rsync -r ./local gs://my-bucket/                       │
│                                                                     │
│  // 파일 다운로드                                                      │
│  $ gsutil cp gs://my-bucket/file.txt ./                             │
│                                                                     │
│  // 스토리지 클래스 설정                                                │
│  $ gsutil defstorageclass set nearline gs://my-bucket              │
│                                                                     │
│  【MinIO (온프레미스 S3 호환)】                                        │
│  ──────────────────                                                  │
│  // 서버 시작                                                          │
│  $ minio server /data                                               │
│                                                                     │
│  // mc 클라이언트 설정                                                  │
│  $ mc alias set myminio http://localhost:9000 minioadmin minioadmin│
│                                                                     │
│  // 버킷 생성                                                          │
│  $ mc mb myminio/mybucket                                           │
│                                                                     │
│  // 파일 복사                                                          │
│  $ mc cp file.txt myminio/mybucket/                                 │
│                                                                     │
│  // 파일 동기화                                                        │
│  $ mc mirror ./local myminio/mybucket                               │
│                                                                     │
│  【rclone (멀티 클라우드 동기화)】                                      │
│  ──────────────────                                                  │
│  // 설정                                                              │
│  $ rclone config                                                    │
│                                                                     │
│  // S3 동기화                                                          │
│  $ rclone sync ./local s3:my-bucket                                 │
│                                                                     │
│  // 클라우드 간 복사                                                    │
│  $ rclone copy s3:bucket1 gcs:bucket2                               │
│                                                                     │
│  // 암호화 동기화                                                       │
│  $ rclone crypt ./local remote:encrypted                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 핵심 요약

