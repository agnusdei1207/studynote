# 493. 파일 시스템 모니터링 (File System Monitoring)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일 시스템 상태 실시간 관찰
> 2. **가치**: 장애 예방과 성능 최적화
> 3. **융합**: 로깅, 알림, 용량 관리와 연관

---

## Ⅰ. 개요

### 개념 정의

파일 시스템 모니터링(File System Monitoring)은 **파일 시스템의 상태, 성능, 용량을 실시간으로 관찰하고 분석하는 활동**이다.

### 💡 비유: 자동차 계기판
파일 시스템 모니터링은 **계기판**과 같다. 속도, 연료, 엔진 상태를 실시간으로 확인한다.

### 파일 시스템 모니터링 구조


## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 시스템 모니터링 상세                                │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【주요 명령어】                                                        │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 용량 모니터링:                                                      │ │
│  │     df -h          // 파일시스템별 사용량                                │ │
│  │     df -i          // 아이노드 사용량                                    │ │
│  │     du -sh *       // 디렉토리별 크기                                    │ │
│  │     ncdu           // 대화형 디스크 사용량                               │ │
│  │                                                             │ │
│  │  2. I/O 모니터링:                                                       │ │
│  │     iostat -x 1    // 확장 I/O 통계                                     │ │
│  │     iostat -d -x   // 장치별 상세                                       │ │
│  │     iotop          // 프로세스별 I/O                                    │ │
│  │     pidstat -d     // 프로세스 디스크 활동                               │ │
│  │                                                             │ │
│  │  3. 파일 시스템 이벤트:                                                  │ │
│  │     inotifywait    // 파일 변경 감지                                    │ │
│  │     lsof           // 열린 파일 목록                                    │ │
│  │     fuser          // 파일 사용 프로세스                                 │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【임계치 설정 예시】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  지표                경고(Warning)   위험(Critical)                       │ │
│  │  ────                ─────────────   ──────────────                       │ │
│  │  디스크 사용량        80%             95%                                   │ │
│  │  아이노드 사용량      80%             95%                                   │ │
│  │  I/O 대기 시간        20ms            50ms                                  │ │
│  │  읽기 에러            > 0             > 10                                  │ │
│  │  쓰기 에러            > 0             > 10                                  │ │
│  │  %util               80%             95%                                   │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【알림 통합】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  • Prometheus + Grafana: 시계열 데이터 시각화                           │ │
│  │  • Nagios/Zabbix: 임계치 기반 알림                                      │ │
│  │  • ELK Stack: 로그 분석                                                 │ │
│  │  • Slack/Email: 알림 채널                                               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【용량 모니터링】                                                      │
│  ──────────────────                                                  │
│  // 파일시스템 사용량                                                   │
│  $ df -h                                                             │
│  Filesystem      Size  Used Avail Use% Mounted on                    │
│  /dev/sda1       100G   75G   25G  75% /                             │
│                                                                     │
│  // 아이노드 확인                                                       │
│  $ df -i                                                             │
│  Filesystem     Inodes  IUsed  IFree IUse% Mounted on                │
│  /dev/sda1      6553600 123456 6430144    2% /                       │
│                                                                     │
│  // 디렉토리별 크기                                                      │
│  $ du -sh /* | sort -rh | head -10                                  │
│                                                                     │
│  // ncdu (대화형)                                                      │
│  $ ncdu /                                                            │
│                                                                     │
│  【I/O 모니터링】                                                        │
│  ──────────────────                                                  │
│  // I/O 통계                                                           │
│  $ iostat -x 1                                                       │
│  Device  r/s   w/s   rkB/s  wkB/s  rrqm/s  wrqm/s  %util  await     │
│  sda     100   50    400    200    10      5       85     5.0       │
│                                                                     │
│  // 프로세스별 I/O                                                       │
│  $ iotop -o                                                          │
│  PID   PRIO  USER     DISK READ  DISK WRITE  COMMAND                 │
│  1234  be/4  mysql    50.0 M/s   10.0 M/s    mysqld                  │
│                                                                     │
│  // 특정 프로세스 I/O                                                    │
│  $ pidstat -d -p 1234 1                                              │
│                                                                     │
│  【파일 접근 모니터링】                                                  │
│  ──────────────────                                                  │
│  // 열린 파일 목록                                                       │
│  $ lsof /mnt/data                                                    │
│  $ lsof -u username                                                  │
│                                                                     │
│  // 파일 사용 프로세스                                                    │
│  $ fuser -v /mnt/data                                                │
│                                                                     │
│  // 실시간 파일 변경 감지                                                │
│  $ inotifywait -m -r /var/log                                        │
│  /var/log/ MODIFY syslog                                             │
│                                                                     │
│  【디스크 건강 모니터링】                                                │
│  ──────────────────                                                  │
│  // SMART 상태 확인                                                     │
│  $ sudo smartctl -a /dev/sda                                         │
│  $ sudo smartctl -H /dev/sda                                         │
│                                                                     │
│  // dmesg로 에러 확인                                                   │
│  $ dmesg | grep -i error                                            │
│  $ dmesg | grep -i "I/O error"                                      │
│                                                                     │
│  【자동화 스크립트】                                                      │
│  ──────────────────                                                  │
│  // 용량 체크 스크립트                                                    │
│  #!/bin/bash                                                         │
│  THRESHOLD=90                                                        │
│  df -H | grep -vE '^Filesystem|tmpfs|cdrom' | awk '{               │
│    use=$5; sub(/%/,"",use)                                          │
│    if (use > '$THRESHOLD') print $0                                 │
│  }' | while read line; do                                           │
│    echo "WARNING: $line" | mail -s "Disk Alert" admin@example.com   │
│  done                                                                │
│                                                                     │
│  // 크론에 등록                                                          │
│  */5 * * * * /usr/local/bin/disk_monitor.sh                        │
│                                                                     │
│  【ZFS 모니터링】                                                        │
│  ──────────────────                                                  │
│  $ zpool status                                                      │
│  $ zpool list                                                        │
│  $ zfs list                                                          │
│  $ zpool iostat 1                                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
