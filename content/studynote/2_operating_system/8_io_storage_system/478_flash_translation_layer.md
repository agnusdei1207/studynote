# 478. CIFS/SMB (Common Internet File System / Server Message Block)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 윈도우 중심 파일 공유 프로토콜
> 2. **가치**: 윈도우-유닉스 간 파일 공유
> 3. **융합**: NFS, 네트워킹과 연관

---

## Ⅰ. 개요

### 개념 정의

SMB(Server Message Block)/CIFS(Common Internet File System)는 **윈도우 환경에서 파일, 프린터, 시리얼 포트를 공유하기 위한 프로토콜**이다.

### 💡 비유: 윈도우 네트워크 공유
CIFS/SMB는 **윈도우 네트워크 공유**와 같다. "네트워크 드라이브"로 다른 컴퓨터의 폴더를 내 컴퓨터처럼 쓴다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                CIFS/SMB 상세                                          │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【SMB 버전 특징】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  SMB 1.0:                                                             │ │
│  │  • 레거시, 보안 문제로 사용 비권장                                      │ │
│  │  • WannaCry 등 악용                                                   │ │
│  │                                                             │ │
│  │  SMB 2.0+:                                                             │ │
│  │  • 성능 개선 (파이프라이닝)                                            │ │
│  │  • 더 적은 명령어                                                      │ │
│  │  • 영속 핸들 (Persistent Handles)                                     │ │
│  │                                                             │ │
│  │  SMB 3.0+:                                                             │ │
│  │  • SMB Direct (RDMA)                                                  │ │
│  │  • 암호화 (SMB Encryption)                                            │ │
│  │  • 다중 채널 (Multichannel)                                           │ │
│  │  • SMB Multichannel, Transparent Failover                            │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【Samba (리눅스 SMB 구현)】                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  주요 구성 요소:                                                       │ │
│  │  • smbd: 파일/프린터 공유 서비스                                       │ │
│  │  • nmbd: NetBIOS 이름 서비스                                          │ │
│  │  • winbindd: 윈도우 도메인 통합                                        │ │
│  │                                                             │ │
│  │  기능:                                                               │
│  │  • 윈도우 파일 공유 서버                                               │ │
│  │  • 윈도우 도메인 컨트롤러                                              │ │
│  │  • Active Directory 통합                                              │ │
│  │  • 윈도우 클라이언트로 리눅스 공유                                      │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【보안 모델】                                                          │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  security=user: 로컬 사용자 인증                                       │ │
│  │  security=domain: 도메인 컨트롤러 인증                                 │ │
│  │  security=ads: Active Directory 인증                                  │ │
│  │  security=share: 공유 레벨 인증 (레거시)                                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Samba 서버 설정】                                                  │
│  ──────────────────                                                  │
│  // 설치                                                              │
│  $ sudo apt install samba                                           │
│                                                                     │
│  // /etc/samba/smb.conf 설정                                          │
│  [global]                                                             │
│     workgroup = WORKGROUP                                            │
│     server string = Samba Server                                     │
│     security = user                                                  │
│     passdb backend = tdbsam                                          │
│                                                                     │
│  [share]                                                              │
│     path = /srv/samba/share                                          │
│     browseable = yes                                                  │
│     writable = yes                                                    │
│     guest ok = no                                                     │
│     valid users = @smbgroup                                           │
│                                                                     │
│  // 사용자 추가                                                        │
│  $ sudo smbpasswd -a username                                        │
│                                                                     │
│  // 서비스 시작                                                        │
│  $ sudo systemctl restart smbd nmbd                                  │
│                                                                     │
│  【리눅스에서 SMB 마운트】                                            │
│  ──────────────────                                                  │
│  // cifs-utils 설치                                                   │
│  $ sudo apt install cifs-utils                                       │
│                                                                     │
│  // 마운트                                                            │
│  $ sudo mount -t cifs //server/share /mnt/smb \                      │
│        -o username=user,password=pass                               │
│                                                                     │
│  // credentials 파일 사용                                              │
│  // /etc/smb.credentials                                              │
│  username=user                                                        │
│  password=pass                                                        │
│  domain=WORKGROUP                                                     │
│                                                                     │
│  // fstab 설정                                                        │
│  //server/share  /mnt/smb  cifs  credentials=/etc/smb.credentials 0 0│
│                                                                     │
│  【윈도우에서 접근】                                                    │
│  ──────────────────                                                  │
│  // 탐색기에서                                                         │
│  \\server\share                                                       │
│                                                                     │
│  // 네트워크 드라이브 연결                                              │
│  net use Z: \\server\share /user:username password                  │
│                                                                     │
│  【Samba 상태 확인】                                                  │
│  ──────────────────                                                  │
│  // 설정 테스트                                                        │
│  $ testparm                                                           │
│                                                                     │
│  // 연결된 클라이언트                                                  │
│  $ smbstatus                                                          │
│                                                                     │
│  // 공유 목록 확인                                                     │
│  $ smbclient -L localhost -U username                               │
│                                                                     │
│  // SMB 버전 확인                                                      │
│  $ smbclient -L server -m SMB3                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 윈도우 파일 공유 프로토콜
• 버전: SMB 1.0, 2.x, 3.x
• 포트: 445 (Direct), 139 (NetBIOS)
• 리눅스: Samba로 구현
• 구성: smbd, nmbd, winbindd
• 인증: user, domain, ads
• 마운트: mount -t cifs
• 접근: \\server\share
```
