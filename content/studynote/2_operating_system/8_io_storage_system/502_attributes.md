# 444. 파일 속성 (File Attributes)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일의 메타데이터
> 2. **가치**: 파일 관리와 접근 제어
> 3. **융합**: 아이노드, 권한과 연관

---

## Ⅰ. 개요

### 개념 정의

파일 속성(File Attributes)은 **파일에 대한 메타데이터로, 파일의 특성과 관리 정보**다. 파일 이름 외에 저장되는 모든 정보다.

### 💡 비유: 신분증
파일 속성은 **신분증**과 같다. 이름뿐 아니라 생년월일, 주소, 사진 등 여러 정보가 있다.

### 파일 속성 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 속성 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【주요 파일 속성】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 식별 정보:                                                            │ │
│  │     • 파일 이름 (File Name)                                              │ │
│  │     • 식별자 (Identifier, inode 번호)                                     │ │
│  │     • 소유자 (Owner, UID)                                                │ │
│  │     • 그룹 (Group, GID)                                                  │ │
│  │                                                             │ │
│  │  2. 타입 정보:                                                            │ │
│  │     • 파일 타입 (Type: 일반, 디렉토리, 링크, 장치...)                      │ │
│  │     • 파일 형식 (Format, 매직 넘버)                                        │ │
│  │                                                             │ │
│  │  3. 접근 제어:                                                            │ │
│  │     • 접근 권한 (Permissions: rwx)                                        │ │
│  │     • ACL (Access Control List)                                          │ │
│  │     • 특수 권한 (SUID, SGID, Sticky)                                     │ │
│  │                                                             │ │
│  │  4. 크기 정보:                                                            │ │
│  │     • 현재 크기 (Size)                                                    │ │
│  │     • 최대 크기 (Maximum size)                                           │ │
│  │     • 할당된 블록 수                                                     │ │
│  │                                                             │ │
│  │  5. 시간 정보:                                                            │ │
│  │     • 생성 시간 (Creation/Birth time)                                     │ │
│  │     • 수정 시간 (Modification time, mtime)                               │ │
│  │     • 접근 시간 (Access time, atime)                                      │ │
│  │     • 변경 시간 (Change time, ctime)                                      │ │
│  │                                                             │ │
│  │  6. 위치 정보:                                                            │ │
│  │     • 파일 위치 (Location: 블록 번호들)                                    │ │
│  │     • 장치 번호 (Device number)                                          │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【stat 구조체 (유닉스)】                                               │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  struct stat {                                                          │ │
│  │      dev_t     st_dev;     // 장치 번호                                   │ │
│  │      ino_t     st_ino;     // 아이노드 번호                                │ │
│  │      mode_t    st_mode;    // 타입 + 권한                                  │ │
│  │      nlink_t   st_nlink;   // 하드링크 수                                  │ │
│  │      uid_t     st_uid;     // 소유자 UID                                   │ │
│  │      gid_t     st_gid;     // 그룹 GID                                     │ │
│  │      dev_t     st_rdev;    // 장치 파일의 장치 번호                         │ │
│  │      off_t     st_size;    // 크기 (바이트)                                 │ │
│  │      blksize_t st_blksize; // 블록 크기                                    │ │
│  │      blkcnt_t  st_blocks;  // 할당된 블록 수                                │ │
│  │      time_t    st_atime;   // 마지막 접근 시간                              │ │
│  │      time_t    st_mtime;   // 마지막 수정 시간                              │ │
│  │      time_t    st_ctime;   // 상태 변경 시간                                │ │
│  │  };                                                                      │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 속성 상세                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【파일 타입 (st_mode 상위 비트)】                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  S_IFREG (0100000): 일반 파일 (Regular file)                            │ │
│  │  S_IFDIR (0040000): 디렉토리 (Directory)                                 │ │
│  │  S_IFLNK (0120000): 심볼릭 링크 (Symbolic link)                          │ │
│  │  S_IFBLK (0060000): 블록 장치 (Block device)                             │ │
│  │  S_IFCHR (0020000): 문자 장치 (Character device)                         │ │
│  │  S_IFIFO (0010000): FIFO/파이프 (Named pipe)                             │ │
│  │  S_IFSOCK (0140000): 소켓 (Socket)                                       │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【세 가지 시간의 차이】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  atime (Access Time):                                                   │ │
│  │  • 파일 내용을 읽은 시간                                                 │ │
│  │  • cat, less 등                                                         │ │
│  │                                                             │ │
│  │  mtime (Modification Time):                                             │ │
│  │  • 파일 내용을 수정한 시간                                                │ │
│  │  • write, echo > 등                                                     │ │
│  │                                                             │ │
│  │  ctime (Change Time):                                                   │ │
│  │  • 메타데이터를 변경한 시간                                               │ │
│  │  • chmod, chown 등                                                      │ │
│  │  • mtime 변경 시 ctime도 변경                                            │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【Windows 파일 속성】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  추가 속성:                                                               │ │
│  │  • 읽기 전용 (Read-only)                                                  │ │
│  │  • 숨김 (Hidden)                                                          │ │
│  │  • 시스템 (System)                                                        │ │
│  │  • 보관 (Archive)                                                        │ │
│  │  • 압축 (Compressed)                                                     │ │
│  │  • 암호화 (Encrypted)                                                     │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【파일 속성 확인】                                                      │
│  ──────────────────                                                  │
│  // ls로 속성 확인                                                        │
│  $ ls -la file.txt                                                    │
│  -rw-r--r-- 1 user group 1234 Mar 8 10:00 file.txt                   │
│  // [타입권한] [링크수] [소유자] [그룹] [크기] [수정시간] [이름]            │
│                                                                     │
│  // stat으로 상세 확인                                                     │
│  $ stat file.txt                                                      │
│    File: file.txt                                                      │
│    Size: 1234          Blocks: 8          IO Block: 4096              │
│  Device: 801h/2049d    Inode: 1234567     Links: 1                     │
│  Access: (0644/-rw-r--r--)  Uid: (1000/user)  Gid: (1000/group)       │
│  Access: 2024-03-08 10:00:00.000000000 +0900                           │
│  Modify: 2024-03-08 10:00:00.000000000 +0900                           │
│  Change: 2024-03-08 10:00:00.000000000 +0900                           │
│  Birth: 2024-03-08 09:55:00.000000000 +0900                            │
│                                                                     │
│  // 파일 타입만 확인                                                        │
│  $ file file.txt                                                      │
│  file.txt: ASCII text                                                 │
│                                                                     │
│  【파일 속성 변경】                                                      │
│  ──────────────────                                                  │
│  // 권한 변경                                                            │
│  $ chmod 755 file.txt                                                 │
│  $ chmod +x script.sh                                                 │
│                                                                     │
│  // 소유자 변경                                                          │
│  $ chown user file.txt                                                │
│  $ chown user:group file.txt                                          │
│  $ chgrp group file.txt                                               │
│                                                                     │
│  // 시간 변경                                                            │
│  $ touch file.txt            // 현재 시간으로                            │
│  $ touch -t 202403081000 file.txt  // 지정 시간으로                       │
│  $ touch -d "2024-03-08 10:00" file.txt                                │
│                                                                     │
│  【프로그래밍에서 속성 다루기】                                            │
│  ──────────────────                                                  │
│  #include <sys/stat.h>                                               │
│  #include <sys/types.h>                                              │
│  #include <stdio.h>                                                  │
│                                                                     │
│  int main() {                                                         │
│      struct stat st;                                                  │
│                                                                     │
│      if (stat("file.txt", &st) == 0) {                                │
│          printf("Size: %ld bytes\n", st.st_size);                     │
│          printf("Inode: %ld\n", st.st_ino);                           │
│          printf("Links: %ld\n", st.st_nlink);                         │
│          printf("Mode: %o\n", st.st_mode & 0777);                     │
│                                                                     │
│          // 파일 타입 확인                                               │
│          if (S_ISREG(st.st_mode))                                     │
│              printf("Type: Regular file\n");                           │
│          else if (S_ISDIR(st.st_mode))                                │
│              printf("Type: Directory\n");                              │
│          else if (S_ISLNK(st.st_mode))                                │
│              printf("Type: Symbolic link\n");                          │
│      }                                                                │
│      return 0;                                                        │
│  }                                                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

