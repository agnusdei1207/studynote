# 571. 파일 (File)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터를 저장하는 논리적 단위
> 2. **가치**: 영속성, 구조화, 접근 제어
> 3. **융합**: 파일 시스템, inode, 메타데이터와 연관

---

## Ⅰ. 개요

### 개념 정의
**파일(File)**은 **보조기억장치에 저장된 관련 데이터의 논리적 집합**입니다.

### 💡 비유: 서류철의 문서
파일은 **서류철의 문서**와 같습니다. 이름이 있고 내용이 담겨 있습니다.

### 파일 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【파일 구성 요소】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  구성 요소          설명                                        │ │   │
│  │  ────────            ────                                        │ │   │
│  │  파일 이름           사용자 식별용 (경로 포함)                     │ │   │
│  │  파일 내용           실제 데이터                                 │ │   │
│  │  메타데이터           속성 정보 (크기, 시간, 권한)                  │ │   │
│  │  파일 포인터         현재 읽기/쓰기 위치                          │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【파일 속성 (메타데이터)】                                           │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  속성               설명                    예시                   │ │   │
│  │  ────               ────                    ────                   │ │   │
│  │  이름               파일 식별자              document.txt          │ │   │
│  │  확장자             파일 타입               .txt, .pdf            │ │   │
│  │  크기               바이트 단위              1024 bytes           │ │   │
│  │  유형               일반/디렉토리/장치        regular file         │ │   │
│  │  권한               읽기/쓰기/실행           rw-r--r--            │ │   │
│  │  소유자             UID/GID                 1000:1000            │ │   │
│  │  타임스탬프         생성/수정/접근           atime, mtime, ctime  │ │   │
│  │  링크 수            하드 링크 수             1                    │ │   │
│  │  블록 수            할당된 블록              8 blocks             │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【파일 타입】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  타입               기호              설명                       │ │   │
│  │  ────               ────              ────                       │ │   │
│  │  일반 파일          -                 텍스트, 바이너리             │ │   │
│  │  디렉토리           d                 폴더                        │ │   │
│  │  심볼릭 링크        l                 소프트 링크                  │ │   │
│  │  블록 장치          b                 HDD, SSD                    │ │   │
│  │  문자 장치          c                 터미널, 시리얼               │ │   │
│  │  파이프             p                 Named pipe                  │ │   │
│  │  소켓               s                 Unix socket                 │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 상세                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【파일 접근 방식】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  방식              설명                    특징                   │ │   │
│  │  ────              ────                    ────                   │ │   │
│  │  순차 접근          처음부터 순서대로         간단, 테이프 유사        │ │   │
│  │  직접 접근          임의 위치 바로 이동        빠름, 레코드 지향        │ │   │
│  │  인덱스 접근        인덱스 통해 위치 찾기      데이터베이스            │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【파일 연산】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  연산              시스템 콜              설명                     │ │   │
│  │  ────              ─────────              ────                     │ │   │
│  │  생성              creat()               새 파일 생성               │ │   │
│  │  열기              open()                파일 열기                  │ │   │
│  │  읽기              read()                데이터 읽기                │ │   │
│  │  쓰기              write()               데이터 쓰기                │ │   │
│  │  닫기              close()               파일 닫기                  │ │   │
│  │  삭제              unlink()              파일 삭제                  │ │   │
│  │  이동              lseek()               파일 포인터 이동            │ │   │
│  │  속성              stat()                메타데이터 조회             │ │   │
│  │  권한              chmod()               권한 변경                  │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【파일 권한 (Unix)】                                                 │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  rwx    rwx    rwx                                           │ │   │
│  │  소유자  그룹   기타                                           │ │   │
│  │  421    421    421                                           │ │   │
│  │                                                             │ │   │
│  │  예: 755 = rwxr-xr-x                                          │ │   │
│  │      644 = rw-r--r--                                          │ │   │
│  │      700 = rwx------                                          │ │   │
│  │                                                             │ │   │
│  │  특수 권한:                                                     │ │   │
│  │  SUID (4000): 실행 시 소유자 권한                               │ │   │
│  │  SGID (2000): 실행 시 그룹 권한                                 │ │   │
│  │  Sticky (1000): 디렉토리 내 파일 삭제 제한                        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【파일 정보 확인】                                                   │
│  ──────────────────                                                │
│  // 파일 상세 정보                                                    │
│  $ ls -la document.txt                                              │
│  -rw-r--r-- 1 user group 1024 Jan 15 12:00 document.txt            │
│                                                                     │
│  // 파일 타입 확인                                                    │
│  $ file document.txt                                                │
│  document.txt: ASCII text                                          │
│                                                                     │
│  // stat 명령어                                                      │
│  $ stat document.txt                                                │
│  File: document.txt                                                 │
│  Size: 1024          Blocks: 8          IO Block: 4096   regular file│
│  Inode: 1234567       Links: 1                                      │
│  Access: (0644/-rw-r--r--)  Uid: (1000/user)  Gid: (1000/group)   │
│  Access: 2024-01-15 12:00:00.000000000 +0900                       │
│  Modify: 2024-01-15 12:00:00.000000000 +0900                       │
│  Change: 2024-01-15 12:00:00.000000000 +0900                       │
│                                                                     │
│  【파일 권한 관리】                                                   │
│  ──────────────────                                                │
│  // 권한 변경                                                        │
│  $ chmod 755 script.sh                                              │
│  $ chmod u+x script.sh          // 소유자 실행 권한 추가              │
│  $ chmod go-w file.txt          // 그룹/기타 쓰기 권한 제거           │
│                                                                     │
│  // 소유권 변경                                                      │
│  $ chown user:group file.txt                                        │
│  $ chown user file.txt          // 소유자만 변경                      │
│  $ chgrp group file.txt         // 그룹만 변경                        │
│                                                                     │
│  // 특수 권한                                                        │
│  $ chmod 4755 suid_program     // SUID                              │
│  $ chmod 2755 sgid_dir         // SGID                              │
│  $ chmod 1755 sticky_dir       // Sticky                            │
│                                                                     │
│  【파일 연산】                                                        │
│  ──────────────────                                                │
│  // 파일 생성/삭제                                                   │
│  $ touch newfile.txt            // 빈 파일 생성                       │
│  $ rm file.txt                  // 파일 삭제                         │
│  $ rm -r directory/             // 디렉토리 삭제                      │
│                                                                     │
│  // 파일 복사/이동                                                    │
│  $ cp source.txt dest.txt       // 복사                              │
│  $ mv old.txt new.txt           // 이동/이름 변경                     │
│                                                                     │
│  // 링크 생성                                                        │
│  $ ln target hardlink           // 하드 링크                          │
│  $ ln -s target symlink         // 심볼릭 링크                        │
│                                                                     │
│  【파일 검색】                                                        │
│  ──────────────────                                                │
│  // 파일 찾기                                                        │
│  $ find /path -name "*.txt"                                         │
│  $ find /path -size +1M        // 1MB 이상                          │
│  $ find /path -mtime -7        // 7일 이내 수정                       │
│                                                                     │
│  // 내용 검색                                                        │
│  $ grep "pattern" file.txt                                          │
│  $ grep -r "pattern" directory/                                     │
│                                                                     │
│  【Python 파일 처리】                                                 │
│  ──────────────────                                                │
│  # 파일 읽기/쓰기                                                     │
│  with open('file.txt', 'r') as f:                                   │
│      content = f.read()                                             │
│                                                                     │
│  with open('file.txt', 'w') as f:                                   │
│      f.write('Hello')                                               │
│                                                                     │
│  # 파일 메타데이터                                                    │
│  import os                                                           │
│  stat = os.stat('file.txt')                                         │
│  print(f"Size: {stat.st_size}")                                     │
│  print(f"Mode: {oct(stat.st_mode)}")                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 보조기억장치에 저장된 데이터 집합
• 구성: 이름, 내용, 메타데이터, 포인터
• 속성: 크기, 권한, 소유자, 타임스탬프
• 타입: 일반, 디렉토리, 링크, 장치, 파이프, 소켓
• 접근: 순차, 직접, 인덱스
• 연산: open, read, write, close, seek
• 권한: rwx (777, 755, 644)
• 특수 권한: SUID, SGID, Sticky
• 명령어: ls, stat, chmod, chown, file
• 시스템 콜: open(), read(), write(), close()
```
