# 468. 언마운트 (Unmount)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일 시스템과 디렉토리 트리 분리
> 2. **가치**: 안전한 장치 제거와 데이터 보호
> 3. **융합**: 마운트, 동기화와 연관

---

## Ⅰ. 개요

### 개념 정의

언마운트(Unmount)는 **마운트된 파일 시스템을 디렉토리 트리에서 분리하는 작업**이다.

### 💡 비유: 선반 제거
언마운트는 **선반 제거**와 같다. 선반(파일 시스템)을 벽(디렉토리 트리)에서 떼어낸다.

### 언마운트 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                언마운트                                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【언마운트 과정】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 언마운트 요청:                                                    │ │
│  │     $ umount /mnt/usb                                              │ │
│  │                                                             │ │
│  │  2. 사용 중인지 확인:                                                 │ │
│  │     • 열린 파일 확인                                                   │ │
│  │     • 현재 디렉토리 확인                                               │ │
│  │     • 사용 중인 프로세스 확인                                          │ │
│  │                                                             │ │
│  │  3. 데이터 동기화:                                                    │ │
│  │     • 버퍼 캐시 flush                                                 │ │
│  │     • 메타데이터 동기화                                                │ │
│  │     • 저널 커밋                                                       │ │
│  │                                                             │ │
│  │  4. 파일 시스템 분리:                                                 │ │
│  │     • 마운트 테이블에서 제거                                            │ │
│  │     • 디렉토리 엔트리 복원                                             │ │
│  │                                                             │ │
│  │  5. 장치 해제:                                                       │
│  │     • 장치 사용 중단                                                  │ │
│  │     • 안전한 제거 가능                                                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【언마운트 전/후】                                                   │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  언마운트 전:                                                          │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │   /                                                      │    │ │
│  │  │  ├── bin                                                 │    │ │
│  │  │  ├── mnt                                                 │    │ │
│  │  │  │   └── usb  ←────── /dev/sdb1 (마운트됨)                 │    │ │
│  │  │  │       ├── file1.txt                                   │    │ │
│  │  │  │       └── file2.txt                                   │    │ │
│  │  │  └── home                                                │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  언마운트 후: umount /mnt/usb                                        │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │   /                                                      │    │ │
│  │  │  ├── bin                                                 │    │ │
│  │  │  ├── mnt                                                 │    │ │
│  │  │  │   └── usb  ←────── 빈 디렉토리 (원래 내용 복원)            │    │ │
│  │  │  └── home                                                │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  /dev/sdb1은 이제 분리됨 → 안전하게 제거 가능                        │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                언마운트 상세                                           │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【언마운트 실패 원인】                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 장치 사용 중 (device busy):                                       │ │
│  │     • 파일이 열려 있음                                                 │ │
│  │     • 프로세스의 현재 디렉토리                                          │ │
│  │     • 다른 마운트가 하위에 있음                                         │ │
│  │                                                             │ │
│  │  2. 오류 메시지:                                                      │ │
│  │     umount: /mnt/usb: target is busy                                 │
│  │                                                             │ │
│  │  3. 원인 파악:                                                        │ │
│  │     $ lsof /mnt/usb                                                 │
│  │     $ fuser -v /mnt/usb                                             │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【강제 언마운트】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 지연 언마운트 (lazy unmount):                                      │ │
│  │     $ umount -l /mnt/usb                                            │
│  │     • 즉시 분리, 사용 중인 프로세스가 끝나면 완전 해제                     │ │
│  │     • 데이터 손실 가능성 있음                                          │ │
│  │                                                             │ │
│  │  2. 강제 언마운트:                                                      │ │
│  │     $ umount -f /mnt/usb                                            │
│  │     • 강제로 분리 (NFS용)                                             │ │
│  │     • 로컬 파일시스템에는 효과 없을 수 있음                               │ │
│  │                                                             │ │
│  │  주의: 강제 언마운트는 데이터 손실 위험!                                  │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【안전한 언마운트 절차】                                              │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 사용 중인 프로세스 확인:                                           │
│  │     $ lsof /mnt/usb                                                 │
│  │     $ fuser -v /mnt/usb                                             │
│  │                                                             │ │
│  │  2. 프로세스 종료:                                                     │
│  │     $ kill -9 <PID>                                                 │
│  │     $ fuser -k /mnt/usb  // 사용 중인 프로세스 종료                     │ │
│  │                                                             │ │
│  │  3. 데이터 동기화:                                                     │
│  │     $ sync                                                           │ │
│  │                                                             │ │
│  │  4. 언마운트:                                                         │
│  │     $ umount /mnt/usb                                               │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 언마운트】                                                      │
│  ──────────────────                                                  │
│  // 마운트 포인트로 언마운트                                            │
│  $ sudo umount /mnt/usb                                             │
│                                                                     │
│  // 장치 파일로 언마운트                                                │
│  $ sudo umount /dev/sdb1                                            │
│                                                                     │
│  // 모든 마운트 정보 표시 후 언마운트                                    │
│  $ sudo umount -v /mnt/usb                                          │
│                                                                     │
│  【사용 중인 장치 확인】                                              │
│  ──────────────────                                                  │
│  // lsof로 열린 파일 확인                                              │
│  $ sudo lsof /mnt/usb                                               │
│  COMMAND   PID USER   FD   TYPE DEVICE SIZE/OFF NODE NAME           │
│  bash     1234 user  cwd    DIR   8,17     4096    ...              │
│  vim      5678 user   4r   REG   8,17    10240    ...               │
│                                                                     │
│  // fuser로 사용 중인 프로세스 확인                                     │
│  $ sudo fuser -v /mnt/usb                                           │
│                       USER        PID ACCESS COMMAND                │
│  /mnt/usb:           user       1234 ..c.. bash                     │
│                      user       5678 ...e. vim                      │
│                                                                     │
│  // 사용 중인 프로세스 종료                                              │
│  $ sudo fuser -k /mnt/usb                                           │
│                                                                     │
│  【강제 언마운트】                                                      │
│  ──────────────────                                                  │
│  // 지연 언마운트 (lazy)                                               │
│  $ sudo umount -l /mnt/usb                                          │
│                                                                     │
│  // 강제 언마운트 (NFS용)                                               │
│  $ sudo umount -f /mnt/nfs                                          │
│                                                                     │
│  // 모든 마운트 해제 (단일 사용자 모드)                                  │
│  $ sudo umount -a                                                   │
│                                                                     │
│  【프로그래밍에서 언마운트】                                            │
│  ──────────────────                                                  │
│  #include <sys/mount.h>                                             │
│                                                                     │
│  // 일반 언마운트                                                       │
│  if (umount("/mnt/usb") == -1) {                                    │
│      perror("umount failed");                                        │
│      // 실패 원인: EBUSY (사용 중)                                      │ │
│  }                                                                   │
│                                                                     │
│  // 지연 언마운트                                                       │
│  if (umount2("/mnt/usb", MNT_DETACH) == -1) {                       │
│      perror("umount2 failed");                                       │
│  }                                                                   │
│                                                                     │
│  // 강제 언마운트 (root만)                                              │
│  umount2("/mnt/usb", MNT_FORCE);                                    │
│                                                                     │
│  【동기화 후 언마운트】                                                │
│  ──────────────────                                                  │
│  // 안전한 언마운트 스크립트                                            │
│  #!/bin/bash                                                         │
│  MOUNT_POINT="/mnt/usb"                                              │
│                                                                     │
│  # 1. 사용 중인지 확인                                                 │
│  if lsof $MOUNT_POINT > /dev/null 2>&1; then                        │
│      echo "Device is busy. Killing processes..."                    │
│      fuser -k $MOUNT_POINT                                           │
│      sleep 2                                                         │
│  fi                                                                  │
│                                                                     │
│  # 2. 데이터 동기화                                                    │
│  sync                                                                │
│                                                                     │
│  # 3. 언마운트                                                        │
│  umount $MOUNT_POINT                                                 │
│  echo "Unmounted successfully"                                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 파일 시스템을 디렉토리 트리에서 분리
• 과정: 사용 확인 → 동기화 → 분리
• 실패 원인: 장치 사용 중 (파일 열림, 프로세스)
• 확인: lsof, fuser -v
• 강제: -l (지연), -f (강제)
• 주의: 강제 언마운트는 데이터 손실 위험
• 안전: sync 후 umount
• 시스템콜: umount(), umount2()
```
