# 580. 마운트 (Mount)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일시스템을 디렉토리 트리에 연결
> 2. **가치**: 통합 접근, 저장소 확장, 논리적 구조
> 3. **융합**: 파일시스템, fstab, 마운트 포인트와 연관

---

## Ⅰ. 개요

### 개념 정의
**마운트(Mount)**는 **파일시스템을 디렉토리 계층 구조의 특정 위치에 연결하는 작업**입니다.

### 💡 비유: 책장에 책 꽂기
마운트는 **책장에 책을 꽂는 것**과 같습니다. 새 책을 특정 위치에 배치합니다.

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                마운트 상세                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【마운트 옵션】                                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  옵션              설명                                          │ │   │
│  │  ────              ────                                          │ │   │
│  │  ro               읽기 전용                                       │ │   │
│  │  rw               읽기/쓰기 (기본)                                 │ │   │
│  │  noatime          접근 시간 갱신 안 함                             │ │   │
│  │  nodiratime       디렉토리 접근 시간 갱신 안 함                      │ │   │
│  │  sync             동기 쓰기                                       │ │   │
│  │  async            비동기 쓰기 (기본)                                │ │   │
│  │  noexec           실행 파일 실행 금지                              │ │   │
│  │  nosuid           SUID/SGID 무시                                  │ │   │
│  │  nodev            장치 파일 무시                                   │ │   │
│  │  user             일반 사용자 마운트 허용                           │ │   │
│  │  users            모든 사용자 마운트/언마운트 허용                    │ │   │
│  │  auto             부팅 시 자동 마운트                              │ │   │
│  │  noauto           자동 마운트 안 함                                │ │   │
│  │  defaults         rw,suid,dev,exec,auto,nouser,async              │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【/etc/fstab 구조】                                                  │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  [장치] [마운트포인트] [파일시스템] [옵션] [dump] [fsck]           │ │   │
│  │                                                             │ │   │
│  │  예시:                                                         │ │   │
│  │  /dev/sda1  /        ext4   defaults        0  1             │ │   │
│  │  /dev/sda2  /home    ext4   defaults        0  2             │ │   │
│  │  /dev/sdb1  /data    xfs    defaults,noatime 0  0            │ │   │
│  │  /dev/sr0   /mnt/cd  iso9660 noauto,ro       0  0            │ │   │
│  │  UUID=xxx   /mnt/usb ext4   defaults,noatime 0  0            │ │   │
│  │  tmpfs      /tmp     tmpfs  defaults        0  0             │ │   │
│  │  proc       /proc    proc   defaults        0  0             │ │   │
│  │                                                             │ │   │
│  │  dump: 0 = dump 불필요, 1 = dump 필요                            │ │   │
│  │  fsck: 0 = fsck 불필요, 1 = 루트, 2 = 기타                        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【특수 마운트】                                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  바인드 마운트:                                                  │ │   │
│  │  $ mount --bind /old /new                                        │ │   │
│  │  $ mount -o bind /old /new                                       │ │   │
│  │                                                             │ │   │
│  │  오버레이 마운트 (Union):                                         │ │   │
│  │  $ mount -t overlay overlay -o lowerdir=/lower,upperdir=/upper,  │ │   │
│  │    workdir=/work /merged                                         │ │   │
│  │                                                             │ │   │
│  │  네트워크 마운트:                                                 │ │   │
│  │  $ mount -t nfs server:/share /mnt/nfs                           │ │   │
│  │  $ mount -t cifs //server/share /mnt/smb -o user=username        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【마운트 명령어】                                                    │
│  ──────────────────                                                │
│  // 기본 마운트                                                       │
│  $ sudo mount /dev/sdb1 /mnt/usb                                   │
│  $ sudo mount -t ext4 /dev/sdb1 /mnt/usb                           │
│  $ sudo mount -o ro /dev/sdb1 /mnt/usb       // 읽기 전용            │
│  $ sudo mount -o noatime /dev/sdb1 /mnt/usb                        │
│                                                                     │
│  // UUID로 마운트                                                     │
│  $ sudo mount UUID=xxx-xxx /mnt/usb                                │
│  $ sudo mount -U xxx-xxx /mnt/usb                                  │
│                                                                     │
│  // 라벨로 마운트                                                      │
│  $ sudo mount LABEL=mydata /mnt/usb                                │
│  $ sudo mount -L mydata /mnt/usb                                   │
│                                                                     │
│  【언마운트】                                                         │
│  ──────────────────                                                │
│  // 언마운트                                                          │
│  $ sudo umount /mnt/usb                                            │
│  $ sudo umount /dev/sdb1                                           │
│                                                                     │
│  // 강제 언마운트                                                      │
│  $ sudo umount -f /mnt/usb          // 강제                         │
│  $ sudo umount -l /mnt/usb          // lazy (사용 중단 후 해제)       │
│                                                                     │
│  // 사용 중인 프로세스 확인                                            │
│  $ lsof /mnt/usb                                                    │
│  $ fuser -v /mnt/usb                                               │
│  $ fuser -km /mnt/usb              // 프로세스 종료 후 언마운트        │
│                                                                     │
│  【마운트 정보 확인】                                                  │
│  ──────────────────                                                │
│  // 마운트된 파일시스템                                                │
│  $ mount                                                            │
│  $ mount | grep ext4                                               │
│  $ cat /proc/mounts                                                │
│  $ cat /etc/mtab                                                   │
│                                                                     │
│  // df 명령어                                                        │
│  $ df -h                           // 인간 읽기 쉬운 형태              │
│  $ df -T                           // 파일시스템 타입                 │
│  $ df -h /mnt/usb                  // 특정 마운트만                   │
│                                                                     │
│  // findmnt                                                          │
│  $ findmnt                          // 트리 형태                      │
│  $ findmnt /mnt/usb                                                │
│  $ findmnt -t ext4                                                 │
│                                                                     │
│  // lsblk                                                            │
│  $ lsblk -f                         // 파일시스템 정보                │
│  $ lsblk -o NAME,FSTYPE,MOUNTPOINT                                 │
│                                                                     │
│  【fstab 관리】                                                       │
│  ──────────────────                                                │
│  // fstab 편집                                                       │
│  $ sudo nano /etc/fstab                                            │
│                                                                     │
│  // fstab 마운트                                                     │
│  $ sudo mount -a                    // 전체 마운트                    │
│  $ sudo mount /mnt/usb              // fstab 항목 마운트              │
│                                                                     │
│  // fstab 언마운트                                                    │
│  $ sudo umount -a                   // 전체 언마운트 (루트 제외)       │
│                                                                     │
│  // fstab 검증                                                       │
│  $ sudo findmnt --verify                                           │
│  $ sudo mount -a -v                // 상세 출력                      │
│                                                                     │
│  【UUID/라벨 관리】                                                   │
│  ──────────────────                                                │
│  // UUID 확인                                                        │
│  $ sudo blkid /dev/sdb1                                            │
│  $ ls -la /dev/disk/by-uuid/                                       │
│  $ sudo tune2fs -l /dev/sdb1 | grep UUID                           │
│                                                                     │
│  // 라벨 변경                                                        │
│  $ sudo e2label /dev/sdb1 mylabel    // ext4                        │
│  $ sudo xfs_admin -L mylabel /dev/sdb1  // XFS                     │
│                                                                     │
│  // UUID 변경                                                        │
│  $ sudo tune2fs -U random /dev/sdb1                                │
│                                                                     │
│  【자동 마운트 (autofs)】                                             │ |
│  ──────────────────                                                │
│  // 설치                                                            │
│  $ sudo apt install autofs                                         │
│                                                                     │
│  // /etc/auto.master                                                │
│  /mnt    /etc/auto.misc                                            │
│                                                                     │
│  // /etc/auto.misc                                                  │
│  usb     -fstype=auto      :/dev/sdb1                              │
│  nfs     -fstype=nfs       server:/share                           │
│                                                                     │
│  // 접근 시 자동 마운트                                                │
│  $ ls /mnt/usb                                                     │
│                                                                     │
│  【Windows 마운트】                                                   │
│  ──────────────────                                                │
│  // diskpart                                                        │
│  > list volume                                                      │
│  > assign letter=Z                                                 │
│  > remove letter=Z                                                 │
│                                                                     │
│  // subst                                                           │
│  > subst Z: C:\Users\username\Documents                            │
│  > subst Z: /d                                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 파일시스템을 디렉토리 트리에 연결
• 마운트 포인트: 연결될 디렉토리 위치
• 명령어: mount, umount
• 영구 마운트: /etc/fstab
• 옵션: ro, rw, noatime, noexec, nosuid
• 바인드 마운트: mount --bind
• 네트워크 마운트: NFS, CIFS
• 확인: mount, df, findmnt, lsblk
• 언마운트: umount, umount -l (lazy)
• 자동 마운트: autofs
• UUID: blkid, /dev/disk/by-uuid/
```
