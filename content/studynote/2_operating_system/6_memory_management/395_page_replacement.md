# 392. 페이지 교체 (Page Replacement)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 메모리 부족 시 페이지를 교체하는 알고리즘
> 2. **가치**: 한정된 메모리의 효율적 활용
> 3. **융합**: 페이지 폴트, 스래싱과 연관

---

## Ⅰ. 개요

### 개념 정의

페이지 교체(Page Replacement)는 **물리 메모리가 부족할 때 기존 페이지를 디스크로 내보내고 새 페이지를 가져오는 작업**이다. 어떤 페이지를 교체할지 결정하는 알고리즘이 핵심이다.

### 💡 비유: 책장 바꾸기
페이지 교체는 **책장 바꾸기**와 같다. 책장이 꽉 차면 안 읽는 책장을 내리고 새 책장을 올린다. 어떤 책장을 내릴지가 핵심이다.

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux 페이지 교체】                                                    │
│  ──────────────────                                                  │
│  // 현재 교체 정책 확인                                                   │
│  $ cat /proc/sys/vm/swappiness                                         │
│  60                                                                  │
│                                                                     │
│  // 페이지 교체 통계                                                       │
│  $ cat /proc/vmstat | grep -E "pgscan|pgsteal"                           │
│  pgscan_kswapd 12345                                                 │
│  pgsteal_kswapd 5678                                                 │
│                                                                     │
│  【LRU 근사 구현】                                                        │
│  ──────────────────                                                  │
│  class LRUPageReplacer {                                              │
│      LinkedHashMap<Page, Long> accessTime = new LinkedHashMap<>();     │
│                                                                     │
│      Page selectVictim() {                                           │
│          // 가장 오래된 접근 시간 가진 페이지 선택                           │ │
│          long oldest = Long.MAX_VALUE;                                │
│          Page victim = null;                                          │
│          for (Map.Entry<Page, Long> entry : accessTime.entrySet()) {  │
│              if (entry.getValue() < oldest) {                           │
│                  oldest = entry.getValue();                            │
│                  victim = entry.getKey();                              │
│              }                                                          │
│          }                                                              │
│          return victim;                                                 │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  【Clock 알고리즘 구현】                                                   │
│  ──────────────────                                                  │
│  class ClockReplacer {                                               │
│      List<Page> circularList;                                        │
│      int pointer = 0;                                                │
│                                                                     │
│      Page selectVictim() {                                           │
│          while (true) {                                               │
│              Page page = circularList.get(pointer);                   │
│              if (!page.referenced) {                                  │
│                  pointer = (pointer + 1) % circularList.size();       │
│                  return page;                                         │
│              }                                                          │
│              page.referenced = false;  // 2차 기회                    │ │
│              pointer = (pointer + 1) % circularList.size();           │
│          }                                                              │
│      }                                                              │
│  }                                                                  │
│                                                                     │
│  【스래싱 방지】                                                           │
│  ──────────────────                                                  │
│  // 작업 집합 모니터링                                                  │
│  class WorkingSetMonitor {                                           │
│      int windowSize = 1000;  // 참조 윈도우                             │
│      Map<Process, Set<Page>> workingSets = new HashMap<>();          │
│                                                                     │
│      boolean isThrashing(Process p) {                                │
│          Set<Page> workingSet = workingSets.get(p);                   │
│          int memoryFrames = getFreeFrames();                          │
│          return workingSet.size() > memoryFrames * 0.5;               │
│      }                                                              │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
