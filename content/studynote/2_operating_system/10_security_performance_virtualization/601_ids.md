# 601. SCSI (Small Computer System Interface)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고성능 저장 장치 인터페이스 표준
> 2. **가치**: 높은 신뢰성, 다중 장치 지원, 서버급
> 3. **융합**: SATA, NVMe, iSCSI와 연관

---

## Ⅰ. 개요

### 개념 정의
**SCSI(Small Computer System Interface)**는 **컴퓨터와 주변 장치 간 데이터 전송을 위한 고성능 인터페이스 표준**입니다.

### 💡 비유: 고속도로
SCSI는 **고속도로**와 같습니다. 많은 차선으로 빠르고 안정적으로 이동합니다.

### SCSI 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                SCSI 구조                                              │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【SCSI 표준 진화】                                                    │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  표준              속도           대역폭        연결         │ │   │
│  │  ────              ────           ────         ────         │ │   │
│  │  SCSI-1           5 MB/s         8-bit        Narrow        │ │   │
│  │  SCSI-2 (Fast)    10 MB/s        8-bit        Narrow        │ │   │
│  │  Fast-Wide        20 MB/s        16-bit       Wide          │ │   │
│  │  Ultra SCSI       20 MB/s        8-bit        Narrow        │ │   │
│  │  Ultra-Wide       40 MB/s        16-bit       Wide          │ │   │
│  │  Ultra2 SCSI      40 MB/s        8-bit        Narrow        │ │   │
│  │  Ultra2 Wide      80 MB/s        16-bit       Wide          │ │   │
│  │  Ultra3 (Ultra160) 160 MB/s     16-bit       Wide          │ │   │
│  │  Ultra320         320 MB/s       16-bit       Wide          │ │   │
│  │  Ultra640         640 MB/s       16-bit       Wide          │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【SCSI 장치 ID】                                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  SCSI ID: 0-7 (Narrow) 또는 0-15 (Wide)                         │ │   │
│  │  • ID 7: 보통 호스트 어댑터 (HBA)                                 │ │   │
│  │  • ID 0-6: 주변 장치 (디스크, 테이프 등)                            │ │   │
│  │                                                             │ │   │
│  │  LUN (Logical Unit Number):                                    │ │   │
│  │  • 장치 내 논리 단위 식별                                         │ │   │
│  │  • RAID 컨트롤러에서 각 논리 드라이브                                │ │   │
│  │                                                             │ │   │
│  │  예: /dev/sda = SCSI ID 0, LUN 0                               │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【SCSI 명령 구조】                                                    │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  SCSI 명령어 (CDB - Command Descriptor Block):                  │ │   │
│  │  • READ(6/10/16): 데이터 읽기                                   │ │   │
│  │  • WRITE(6/10/16): 데이터 쓰기                                  │ │   │
│  │  • INQUIRY: 장치 정보 조회                                       │ │   │
│  │  • TEST UNIT READY: 장치 준비 상태 확인                          │ │   │
│  │  • START STOP UNIT: 장치 시작/정지                               │ │   │
│  │  • MODE SENSE/SELECT: 장치 모드 설정                             │ │   │
│  │  • FORMAT UNIT: 장치 포맷                                       │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                SCSI 상세                                              │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【SCSI 버스 구조】                                                    │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  ┌─────────┐     ┌─────────┐     ┌─────────┐               │ │   │
│  │  │  HBA    │────│ Terminator │────│  Disk   │               │ │   │
│  │  │(Host)   │     │           │     │  ID 0   │               │ │   │
│  │  └────┬────┘     └─────────┘     └─────────┘               │ │   │
│  │       │ SCSI Bus (50/68/80핀 케이블)                         │ │   │
│  │       │                                                      │ │   │
│  │  ┌────┴────┐     ┌─────────┐     ┌─────────┐               │ │   │
│  │  │  Tape   │────│  Disk   │────│  CD-ROM │               │ │   │
│  │  │  ID 5   │     │  ID 1   │     │  ID 3   │               │ │   │
│  │  └─────────┘     └─────────┘     └─────────┘               │ │   │
│  │                                                             │ │   │
│  │  • 최대 7/15개 장치 연결 (Narrow/Wide)                           │ │   │
│  │  • 종단자(Terminator) 양 끝에 필수                                │ │   │
│  │  • 병렬 전송 (8/16비트)                                          │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【SCSI vs SATA】                                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  항목            SCSI              SATA                      │ │   │
│  │  ────            ────              ────                      │ │   │
│  │  용도            서버/워크스테이션    데스크탑/소비자            │ │   │
│  │  가격            높음               낮음                       │ │   │
│  │  장치 수          7~15              1 (포트당)                 │ │   │
│  │  케이블 길이       12m (최대)         1m                        │ │   │
│  │  신뢰성           높음               보통                       │ │   │
│  │  명령 큐          Tagged Command     NCQ                       │ │   │
│  │  핫 스왑          지원               일부 지원                   │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【Serial Attached SCSI (SAS)】                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  SAS: SCSI의 직렬 버전                                          │ │   │
│  │  • 포인트 투 포인트 연결                                         │ │   │
│  │  • 3Gb/s, 6Gb/s, 12Gb/s, 24Gb/s                               │ │   │
│  │  • SATA 디스크와 호환 (SAS 컨트롤러에서 SATA 인식)                │ │   │
│  │  • 엔터프라이즈급 신뢰성                                          │ │   │
│  │  • 듀얼 포트 (이중화)                                            │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【SCSI 장치 확인】                                                    │
│  ──────────────────                                                │
│  // lsscsi                                                           │
│  $ lsscsi                                                            │
│  [0:0:0:0]    disk    ATA      Samsung SSD   1.0  /dev/sda         │
│  [0:0:1:0]    disk    ATA      WDC WD1000    2.0  /dev/sdb         │
│  //  ↑        ↑                                         ↑           │
│  // HBA:Bus:Target:LUN  타입                          장치 파일       │
│                                                                     │
│  // sg_map (SCSI Generic)                                            │
│  $ sg_map                                                            │
│  /dev/sg0   /dev/sda                                                 │
│  /dev/sg1   /dev/sdb                                                 │
│                                                                     │
│  // SCSI 장치 정보                                                    │
│  $ sg_inq /dev/sg0                                                   │
│  $ scsiinfo /dev/sda                                                 │
│                                                                     │
│  【/proc/scsi 확인】                                                  │
│  ──────────────────                                                │
│  // SCSI 장치 목록                                                     │
│  $ cat /proc/scsi/scsi                                               │
│  Attached devices:                                                   │
│  Host: scsi0 Channel: 00 Id: 00 Lun: 00                             │
│    Vendor: ATA      Model: Samsung SSD   Rev: 1.0                   │
│                                                                     │
│  // SCSI 호스트 어댑터                                                 │
│  $ ls /proc/scsi/                                                    │
│  scsi  sg  device_info                                               │
│                                                                     │
│  // dmesg로 SCSI 이벤트                                               │
│  $ dmesg | grep -i scsi                                              │
│                                                                     │
│  【SCSI 장치 관리】                                                    │
│  ──────────────────                                                │
│  // 장치 스캔 (새 장치 인식)                                            │
│  $ echo "- - -" | sudo tee /sys/class/scsi_host/host0/scan         │
│                                                                     │
│  // 장치 삭제 (안전 제거)                                               │
│  $ echo 1 | sudo tee /sys/block/sdb/device/delete                  │
│                                                                     │
│  // 장치 리셋                                                          │
│  $ sg_reset /dev/sg0                                                 │
│                                                                     │
│  // SCSI 에러 로그                                                    │
│  $ dmesg | grep -i "scsi error"                                     │
│                                                                     │
│  【sg3_utils 활용】                                                    │
│  ──────────────────                                                │
│  // 설치                                                              │
│  $ sudo apt install sg3-utils                                       │
│                                                                     │
│  // INQUIRY                                                           │
│  $ sg_inq /dev/sg0                                                   │
│                                                                     │
│  // 장치 정보                                                          │
│  $ sginfo /dev/sg0                                                   │
│                                                                     │
│  // READ CAPACITY                                                     │
│  $ sg_readcap /dev/sg0                                               │
│                                                                     │
│  // MODE 페이지                                                       │
│  $ sg_modes /dev/sg0                                                 │
│                                                                     │
│  // VPD (Vital Product Data)                                         │
│  $ sg_vpd /dev/sg0                                                   │
│                                                                     │
│  【iSCSI 연결】                                                        │
│  ──────────────────                                                │
│  // iSCSI 타겟 검색                                                    │
│  $ sudo iscsiadm -m discovery -t st -p 192.168.1.100               │
│                                                                     │
│  // 타겟 로그인                                                         │
│  $ sudo iscsiadm -m node -T iqn.2024.example:storage --login       │
│                                                                     │
│  // 세션 확인                                                          │
│  $ sudo iscsiadm -m session                                         │
│                                                                     │
│  // 로그아웃                                                           │
│  $ sudo iscsiadm -m node -T iqn.2024.example:storage --logout      │
│                                                                     │
│  【SAS 컨트롤러 관리】                                                  │
│  ──────────────────                                                │
│  // SAS 컨트롤러 정보                                                   │
│  $ sudo sas2ircu 0 DISPLAY                                          │
│  $ sudo sas3ircu 0 DISPLAY                                          │
│                                                                     │
│  // MegaCLI (LSI/Avago/Broadcom)                                     │
│  $ sudo megacli -AdpAllInfo -aAll                                   │
│  $ sudo megacli -PDList -aAll                                       │
│  $ sudo megacli -LDInfo -Lall -aAll                                 │
│                                                                     │
│  【SMART 상태 확인】                                                   │
│  ──────────────────                                                │
│  // smartctl로 SCSI 디스크                                            │
│  $ sudo smartctl -a /dev/sda                                        │
│  $ sudo smartctl -H /dev/sda                                        │
│  $ sudo smartctl -l error /dev/sda                                  │
│  $ sudo smartctl -t long /dev/sda                                   │
│                                                                     │
│  【성능 테스트】                                                        │
│  ──────────────────                                                │
│  // hdparm                                                            │
│  $ sudo hdparm -tT /dev/sda                                         │
│                                                                     │
│  // fio (유연한 I/O 테스터)                                             │
│  $ sudo fio --name=randread --ioengine=libaio \                     │
│    --iodepth=16 --rw=randread --bs=4k \                            │
│    --direct=1 --size=1G --numjobs=4 /dev/sda                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 고성능 저장 장치 인터페이스 표준
• 속도: 5 MB/s (SCSI-1) ~ 640 MB/s (Ultra640)
• 장치 ID: 0-7 (Narrow), 0-15 (Wide)
• LUN: 논리 단위 번호
• 명령어: READ, WRITE, INQUIRY, TEST UNIT READY
• 버스: 병렬 전송, 종단자 필수
• 장점: 높은 신뢰성, 다중 장치, 명령 큐잉
• SAS: 직렬 SCSI, SATA 호환
• 확인: lsscsi, /proc/scsi
• 관리: sg3_utils, iscsiadm
• 스캔: /sys/class/scsi_host/host*/scan
• 삭제: /sys/block/*/device/delete
• 성능: hdparm, fio
• 서버급: 엔터프라이즈 환경
```
