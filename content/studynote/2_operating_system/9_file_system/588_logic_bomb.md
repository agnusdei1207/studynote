# 588. 스왑 (Swap)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 메모리 부족 시 디스크를 메모리처럼 사용
> 2. **가치**: 메모리 확장, OOM 방지
> 3. **융합**: 가상 메모리, 페이지 교체, thrashing과 연관

---

## Ⅰ. 개요

### 개념 정의
**스왑(Swap)**은 **물리적 메모리가 부족할 때 디스크 공간을 메모리처럼 사용하는 기술**입니다.

### 💡 비유: 임시 창고
스왑은 **임시 창고**와 같습니다. 본 창고가 꽉 차면 임시로 다른 곳을 씁니다.

### 스왑 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                스왑 구조                                              │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【스왑 동작 원리】                                                    │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  RAM (물리 메모리)          스왑 영역 (디스크)                   │ │   │
│  │  ┌─────────────────┐       ┌─────────────────┐               │ │   │
│  │  │  [페이지 A]      │       │  [페이지 X]      │               │ │   │
│  │  │  [페이지 B]      │  ←──→  │  [페이지 Y]      │               │ │   │
│  │  │  [페이지 C]      │       │  [페이지 Z]      │               │ │   │
│  │  │  ...             │       │                 │               │ │   │
│  │  └─────────────────┘       └─────────────────┘               │ │   │
│  │                                                             │ │   │
│  │  Swap Out: RAM → 디스크 (페이지 아웃)                            │ │   │
│  │  Swap In:  디스크 → RAM (페이지 인)                              │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【스왑 종류】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  종류              설명                    장점          단점          │ │   │
│  │  ────              ────                    ────          ────          │ │   │
│  │  스왑 파티션        전용 파티션              성능          유연성 낮음     │ │   │
│  │  스왑 파일          파일로 생성              유연함        약간 느림      │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【스왑 권장 크기】                                                    │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  RAM 크기         스왑 권장 크기          용도                   │ │   │
│  │  ────────         ─────────────          ────                   │ │   │
│  │  ≤ 2 GB           RAM × 2                                     │ │   │
│  │  2 GB ~ 8 GB      RAM 크기 또는 그 이상   일반                   │ │   │
│  │  8 GB ~ 64 GB     4 GB ~ 8 GB                                    │ │   │
│  │  > 64 GB          8 GB 최소             최소                   │ │   │
│  │                                                             │ │   │
│  │  Hibernate 사용 시: RAM 크기 이상 권장                            │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                스왑 상세                                              │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【swappiness 설정】                                                  │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  /proc/sys/vm/swappiness (0~100)                                │ │   │
│  │  • 0: 스왑 최소화, 물리 메모리 거의 다 쓸 때까지                    │ │   │
│  │  • 10~20: 낮은 스왑 사용 (서버 권장)                               │ │   │
│  │  • 60: 기본값                                                     │ │   │
│  │  • 100: 적극적 스왑 사용                                           │ │   │
│  │                                                             │ │   │
│  │  // 현재 값 확인                                                   │ │   │
│  │  $ cat /proc/sys/vm/swappiness                                  │ │   │
│  │                                                             │ │   │
│  │  // 일시적 변경                                                    │ │   │
│  │  $ sudo sysctl vm.swappiness=10                                 │ │   │
│  │                                                             │ │   │
│  │  // 영구 변경 (/etc/sysctl.conf)                                   │ │   │
│  │  vm.swappiness=10                                                │ │   │
│  │  $ sudo sysctl -p                                                │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【Thrashing】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  정의: 과도한 스왑 인/아웃으로 시스템이 느려지는 현상                  │ │   │
│  │                                                             │ │   │
│  │  원인:                                                           │ │   │
│  │  • 물리 메모리 부족                                                │ │   │
│  │  • 너무 많은 프로세스 실행                                          │ │   │
│  │  • 메모리 누수                                                     │ │   │
│  │                                                             │ │   │
│  │  증상:                                                           │ │   │
│  │  • 디스크 I/O 급증                                                 │ │   │
│  │  • CPU 사용률 저하                                                 │ │   │
│  │  • 시스템 응답 없음                                                 │ │   │
│  │                                                             │ │   │
│  │  해결:                                                           │ │   │
│  │  • 메모리 증설                                                     │ │   │
│  │  • 불필요한 프로세스 종료                                           │ │   │
│  │  • 스왑 크기 증설                                                   │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【스왑 우선순위】                                                     │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  pri 옵션으로 우선순위 설정 (-1 ~ 32767)                           │ │   │
│  │  • 높은 값: 먼저 사용                                              │ │   │
│  │  • 같은 값: 라운드로빈                                              │ │   │
│  │                                                             │ │   │
│  │  // 빠른 SSD 스왑 먼저 사용                                         │ │   │
│  │  /dev/sdb1  swap  swap  pri=100  0  0                            │ │   │
│  │  /dev/sdc1  swap  swap  pri=50   0  0                            │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【스왑 파티션 생성】                                                  │
│  ──────────────────                                                │
│  // 파티션 생성                                                       │
│  $ sudo fdisk /dev/sdb                                             │
│  // n (새 파티션) → t (타입) → 82 (Linux swap)                        │
│                                                                     │
│  // 스왑 영역 설정                                                     │
│  $ sudo mkswap /dev/sdb1                                           │
│  $ sudo mkswap -L myswap /dev/sdb1      // 라벨 지정                  │
│                                                                     │
│  // 활성화                                                           │
│  $ sudo swapon /dev/sdb1                                           │
│  $ sudo swapon -L myswap                                           │
│                                                                     │
│  // /etc/fstab 등록                                                   │
│  /dev/sdb1  swap  swap  defaults  0  0                             │
│  UUID=xxx   swap  swap  defaults  0  0                             │
│                                                                     │
│  【스왑 파일 생성】                                                    │
│  ──────────────────                                                │
│  // 방법 1: dd                                                        │
│  $ sudo dd if=/dev/zero of=/swapfile bs=1M count=4096              │
│  $ sudo chmod 600 /swapfile                                         │
│  $ sudo mkswap /swapfile                                            │
│  $ sudo swapon /swapfile                                            │
│                                                                     │
│  // 방법 2: fallocate                                                 │
│  $ sudo fallocate -l 4G /swapfile                                   │
│  $ sudo chmod 600 /swapfile                                         │
│  $ sudo mkswap /swapfile                                            │
│  $ sudo swapon /swapfile                                            │
│                                                                     │
│  // /etc/fstab 등록                                                   │
│  /swapfile  swap  swap  defaults  0  0                             │
│                                                                     │
│  【스왑 관리】                                                         │
│  ──────────────────                                                │
│  // 스왑 상태 확인                                                     │
│  $ swapon --show                                                    │
│  $ swapon -s                                                        │
│  $ cat /proc/swaps                                                  │
│  $ free -h                                                          │
│                                                                     │
│  // 스왑 사용량 확인                                                   │
│  $ vmstat 1                                                         │
│  $ top                                                              │
│  $ htop                                                             │
│                                                                     │
│  // 스왑 비활성화                                                      │
│  $ sudo swapoff /dev/sdb1                                           │
│  $ sudo swapoff /swapfile                                           │
│  $ sudo swapoff -a                  // 전체                          │
│                                                                     │
│  // 스왑 활성화                                                        │
│  $ sudo swapon -a                   // 전체 (fstab 기준)              │
│                                                                     │
│  // 스왑 파일 삭제                                                     │
│  $ sudo swapoff /swapfile                                           │
│  $ sudo rm /swapfile                                                │
│                                                                     │
│  【스왑 크기 변경】                                                    │ |
│  ──────────────────                                                │
│  // 스왑 파일 확장                                                     │
│  $ sudo swapoff /swapfile                                           │
│  $ sudo dd if=/dev/zero of=/swapfile bs=1M count=8192              │
│  $ sudo chmod 600 /swapfile                                         │
│  $ sudo mkswap /swapfile                                            │
│  $ sudo swapon /swapfile                                            │
│                                                                     │
│  // 또는 새 스왑 파일 추가                                              │
│  $ sudo fallocate -l 4G /swapfile2                                 │
│  $ sudo chmod 600 /swapfile2                                        │
│  $ sudo mkswap /swapfile2                                           │
│  $ sudo swapon /swapfile2                                           │
│                                                                     │
│  【스왑 성능 최적화】                                                  │ |
│  ──────────────────                                                │
│  // swappiness 조정 (서버용)                                          │
│  $ sudo sysctl vm.swappiness=10                                     │
│                                                                     │
│  // vfs_cache_pressure 조정                                          │
│  $ sudo sysctl vm.vfs_cache_pressure=50                            │
│  // 낮은 값: inode/dentry 캐시 유지                                    │
│                                                                     │
│  // /etc/sysctl.conf                                                 │
│  vm.swappiness=10                                                    │
│  vm.vfs_cache_pressure=50                                            │
│                                                                     │
│  // SSD에 스왑 배치                                                   │
│  // - 빠른 I/O로 스왑 성능 향상                                         │
│  // - discard 옵션으로 TRIM                                           │
│  /dev/sdb1  swap  swap  defaults,discard  0  0                      │
│                                                                     │
│  【ZRAM (압축 스왑)】                                                 │ |
│  ──────────────────                                                │
│  // zram 모듈 로드                                                    │
│  $ sudo modprobe zram                                               │
│                                                                     │
│  // zram 장치 생성                                                    │
│  $ sudo zramctl --find --size 2G                                    │
│  $ sudo mkswap /dev/zram0                                           │
│  $ sudo swapon -p 100 /dev/zram0                                   │
│                                                                     │
│  // systemd-zram-generator (Fedora)                                 │
│  $ sudo dnf install zram-generator                                  │
│  // /etc/systemd/zram-generator.conf 설정                            │
│                                                                     │
│  // Ubuntu                                                          │
│  $ sudo apt install zram-config                                    │
│                                                                     │
│  【OOM (Out of Memory) 방지】                                        │ |
│  ──────────────────                                                │
│  // OOM killer 튜닝                                                   │
│  $ cat /proc/sys/vm/overcommit_memory                               │
│  // 0: 휴리스틱, 1: 항상 허용, 2: 엄격                                  │
│                                                                     │
│  // 프로세스 OOM 점수                                                  │
│  $ cat /proc/<pid>/oom_score                                        │
│  $ echo -1000 > /proc/<pid>/oom_score_adj    // 보호                  │
│                                                                     │
│  // 스왑 없이 OOM 방지                                                 │
│  $ sudo sysctl vm.overcommit_memory=2                               │
│  $ sudo sysctl vm.overcommit_ratio=50                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 메모리 부족 시 디스크를 메모리처럼 사용
• 동작: Swap Out (RAM→디스크), Swap In (디스크→RAM)
• 종류: 스왑 파티션, 스왑 파일
• 권장 크기: RAM ≤ 2GB: 2×, RAM > 8GB: 4~8GB
• swappiness: 0~100 (기본 60, 서버 10~20)
• Thrashing: 과도한 스왑으로 느려짐
• 생성: mkswap, swapon
• 확인: swapon --show, free -h, /proc/swaps
• 비활성화: swapoff
• fstab: /dev/sdb1 swap swap defaults 0 0
• 우선순위: pri= 옵션
• SSD 스왑: 빠른 I/O, discard 옵션
• ZRAM: 압축 스왑, 메모리 내
```
