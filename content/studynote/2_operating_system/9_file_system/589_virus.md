# 589. 디스크 파티션 (Disk Partition)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 물리적 디스크를 논리적 영역으로 분할
> 2. **가치**: 데이터 격리, 멀티부팅, 관리 편의성
> 3. **융합**: MBR, GPT, 파일시스템과 연관

---

## Ⅰ. 개요

### 개념 정의
**디스크 파티션(Disk Partition)**은 **물리적 디스크를 논리적인 영역으로 나누는 것**입니다.

### 💡 비유: 방 나누기
디스크 파티션은 **방 나누기**와 같습니다. 큰 공간을 용도별로 구분합니다.

### 파티션 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                파티션 방식                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【MBR (Master Boot Record)】                                         │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  ┌───────────────────────────────────────────────────────┐  │ │   │
│  │  │                    MBR 디스크                           │  │ │   │
│  │  ├───────────────────────────────────────────────────────┤  │ │   │
│  │  │ [MBR][1차 파티션1][1차 파티션2][1차 파티션3][확장][논리...] │  │ │   │
│  │  │ 512B                                                 │  │ │   │
│  │  └───────────────────────────────────────────────────────┘  │ │   │
│  │                                                             │ │   │
│  │  특징:                                                           │ │   │
│  │  • 최대 4개 1차 파티션 (또는 3 + 확장)                             │ │   │
│  │  • 확장 파티션 내 논리 파티션 (최대 15개)                           │ │   │
│  │  • 최대 디스크 크기: 2TB                                          │ │   │
│  │  • 파티션 시작 섹터: 63번째 (CHS 주소)                             │ │   │
│  │  • BIOS 부팅                                                     │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【GPT (GUID Partition Table)】                                       │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  ┌───────────────────────────────────────────────────────┐  │ │   │
│  │  │                    GPT 디스크                           │  │ │   │
│  │  ├───────────────────────────────────────────────────────┤  │ │   │
│  │  │ [보호MBR][GPT헤더][파티션1][파티션2]...[파티션128][백업GPT] │  │ │   │
│  │  └───────────────────────────────────────────────────────┘  │ │   │
│  │                                                             │ │   │
│  │  특징:                                                           │ │   │
│  │  • 최대 128개 파티션                                              │ │   │
│  │  • 최대 디스크 크기: 18 EB                                        │ │   │
│  │  • 파티션 시작 섹터: 2048번째 (4KB 정렬)                           │ │   │
│  │  • UEFI 부팅                                                     │ │   │
│  │  • CRC32 체크섬으로 무결성 보장                                    │ │   │
│  │  • 백업 GPT 헤더/테이블                                           │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【MBR vs GPT 비교】                                                  │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  항목              MBR              GPT                        │ │   │
│  │  ────              ────              ────                       │ │   │
│  │  최대 디스크        2 TB              18 EB                      │ │   │
│  │  최대 파티션        4 (또는 15)        128                        │ │   │
│  │  부팅 방식          BIOS              UEFI                       │ │   │
│  │  백업              없음              있음 (끝에)                   │ │   │
│  │  보안              없음              Secure Boot                 │ │   │
│  │  호환성            높음              Windows 8+                  │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【fdisk - MBR 파티션】                                               │ |
│  ──────────────────                                                │
│  // fdisk 실행                                                       │
│  $ sudo fdisk /dev/sda                                              │
│                                                                     │
│  // 주요 명령어                                                        │
│  m - 도움말                                                          │
│  p - 파티션 목록                                                       │
│  n - 새 파티션                                                        │
│  d - 파티션 삭제                                                       │
│  t - 파티션 타입 변경                                                   │
│  l - 파티션 타입 목록                                                   │
│  w - 저장 후 종료                                                      │
│  q - 저장 없이 종료                                                     │
│                                                                     │
│  // 예시: 새 파티션 생성                                                │
│  n → p (1차) → 1 (번호) → 엔터 (시작) → +50G (크기) → w               │
│                                                                     │
│  // 파티션 타입 변경                                                    │
│  t → 82 (Linux swap) → w                                           │
│                                                                     │
│  【gdisk - GPT 파티션】                                               │ |
│  ──────────────────                                                │
│  // gdisk 실행                                                        │
│  $ sudo gdisk /dev/sda                                              │
│                                                                     │
│  // 주요 명령어                                                        │
│  ? - 도움말                                                          │
│  p - 파티션 목록                                                       │
│  n - 새 파티션                                                        │
│  d - 파티션 삭제                                                       │
│  t - 파티션 타입 변경                                                   │
│  L - 파티션 타입 목록                                                   │
│  w - 저장 후 종료                                                      │
│  q - 저장 없이 종료                                                     │
│                                                                     │
│  // GPT 타입 코드                                                      │
│  8300 - Linux filesystem                                            │
│  8200 - Linux swap                                                  │
│  8E00 - Linux LVM                                                   │
│  EF00 - EFI System                                                  │
│  EF02 - BIOS boot partition                                         │
│                                                                     │
│  【parted - 통합 파티션 도구】                                          │ |
│  ──────────────────                                                │
│  // 대화형 모드                                                        │
│  $ sudo parted /dev/sda                                             │
│                                                                     │
│  // GPT 레이블 생성                                                    │
│  (parted) mklabel gpt                                               │
│                                                                     │
│  // 파티션 생성                                                        │
│  (parted) mkpart primary ext4 1MiB 50GiB                           │
│  (parted) mkpart primary linux-swap 50GiB 58GiB                    │
│                                                                     │
│  // 파티션 확인                                                        │
│  (parted) print                                                     │
│                                                                     │
│  // 비대화형 모드                                                       │
│  $ sudo parted -s /dev/sda mklabel gpt                              │
│  $ sudo parted -s /dev/sda mkpart primary ext4 1MiB 100%            │
│                                                                     │
│  // 정렬 확인                                                          │
│  $ sudo parted /dev/sda align-check optimal 1                       │
│                                                                     │
│  【파티션 확인】                                                       │ |
│  ──────────────────                                                │
│  // fdisk                                                            │
│  $ sudo fdisk -l /dev/sda                                           │
│  $ sudo fdisk -l                                                    │
│                                                                     │
│  // parted                                                            │
│  $ sudo parted -l                                                   │
│  $ sudo parted /dev/sda print                                       │
│                                                                     │
│  // lsblk                                                            │
│  $ lsblk /dev/sda                                                   │
│  $ lsblk -f /dev/sda                  // 파일시스템 포함               │
│  $ lsblk -o NAME,SIZE,TYPE,FSTYPE,MOUNTPOINT                        │
│                                                                     │
│  // blkid                                                            │
│  $ sudo blkid                                                       │
│  $ sudo blkid /dev/sda1                                             │
│                                                                     │
│  【파티션 변경사항 적용】                                               │ |
│  ──────────────────                                                │
│  // 커널에 파티션 테이블 다시 읽기                                        │
│  $ sudo partprobe /dev/sda                                          │
│  $ sudo partx -u /dev/sda                                           │
│                                                                     │
│  // 또는 재부팅                                                        │
│  $ sudo reboot                                                      │
│                                                                     │
│  【MBR ↔ GPT 변환】                                                   │ |
│  ──────────────────                                                │
│  // gdisk으로 변환 (데이터 보존)                                        │
│  $ sudo gdisk /dev/sda                                              │
│  // r (복구) → f (MBR→GPT 변환) → w                                   │
│                                                                     │
│  // sgdisk으로 변환                                                    │
│  $ sudo sgdisk -g /dev/sda          // MBR → GPT                     │
│  $ sudo sgdisk -m /dev/sda          // GPT → MBR                     │
│                                                                     │
│  // 주의: 백업 권장                                                     │
│                                                                     │
│  【파티션 크기 조절】                                                   │ |
│  ──────────────────                                                │
│  // growpart (cloud-init)                                            │
│  $ sudo growpart /dev/sda 1                                         │
│                                                                     │
│  // parted                                                            │
│  $ sudo parted /dev/sda resizepart 1 100%                           │
│                                                                     │
│  // fdisk (삭제 후 재생성)                                             │
│  $ sudo fdisk /dev/sda                                              │
│  // d (삭제) → n (재생성, 같은 시작 섹터) → w                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 물리적 디스크를 논리적 영역으로 분할
• MBR: 최대 2TB, 4개 1차 파티션, BIOS
• GPT: 최대 18EB, 128개 파티션, UEFI
• MBR 타입: 83(Linux), 82(swap), 07(NTFS)
• GPT GUID: EFI, Linux, swap, LVM
• 도구: fdisk(MBR), gdisk(GPT), parted
• 확인: fdisk -l, lsblk -f, blkid
• 적용: partprobe, partx -u
• 변환: gdisk r→f, sgdisk -g/-m
• 4KB 정렬: GPT 기본 (섹터 2048)
• UEFI: EFI 파티션 필수 (vfat)
• BIOS boot: EF02 타입 (GRUB)
```
