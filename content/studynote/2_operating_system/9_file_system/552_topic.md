# 552. 프린터 (Printer)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디지털 문서를 종이로 출력하는 장치
> 2. **가치**: 인쇄 품질, 속도, 용지 규격 지원
> 3. **융합**: 스풀러, 드라이버, 포맷과 연관

---

## Ⅰ. 개요

### 개념 정의
**프린터(Printer)**는 **디지털 문서를 물리적 매체(종이)로 출력하는 I/O 장치**입니다.

### 💡 비유: 디지털 복사기
프린터는 **디지털 복사기**와 같습니다. 화면의 내용을 종이로 옮깁니다.

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【CUPS (Common Unix Printing System)】                             │
│  ──────────────────                                                │
│  // 프린터 목록                                                       │
│  $ lpstat -p                                                        │
│  printer HP_LaserJet is idle.  enabled since ...                   │
│                                                                     │
│  // 기본 프린터 설정                                                  │
│  $ lpoptions -d HP_LaserJet                                         │
│                                                                     │
│  // 프린터 추가                                                       │
│  $ lpadmin -p MyPrinter -E -v socket://192.168.1.100 \              │
│    -m drv:///hp/hp-laserjet.ppd                                     │
│                                                                     │
│  // 웹 인터페이스                                                     │
│  $ firefox http://localhost:631                                     │
│                                                                     │
│  【인쇄 명령어】                                                      │
│  ──────────────────                                                │
│  // 문서 인쇄                                                         │
│  $ lpr document.pdf                                                 │
│  $ lpr -P HP_LaserJet document.pdf                                  │
│                                                                     │
│  // 인쇄 옵션                                                         │
│  $ lpr -o sides=two-sided-long-edge document.pdf  // 양면           │
│  $ lpr -o number-up=2 document.pdf                // 2페이지/시트    │
│  $ lpr -o media=A4 document.pdf                   // 용지 크기       │
│  $ lpr -o quality=high document.pdf               // 고품질          │
│  $ lpr -# 3 document.pdf                          // 3부             │
│                                                                     │
│  // 인쇄 큐 확인                                                      │
│  $ lpq                                                              │
│  Rank    Owner   Job     File(s)          Total size                │
│  active  pf      123     document.pdf     1024 bytes                │
│                                                                     │
│  // 인쇄 취소                                                         │
│  $ lprm 123                                                         │
│  $ cancel -a                         // 모든 작업 취소               │
│                                                                     │
│  【IPP (Internet Printing Protocol)】                               │
│  ──────────────────                                                │
│  // IPP 프린터 URI                                                   │
│  ipp://printer.example.com/ipp/print                                │
│  ipps://printer.example.com/ipp/print  // TLS                       │
│                                                                     │
│  // IPP 프린터 추가                                                   │
│  $ lpadmin -p IPP_Printer -E -v ipp://192.168.1.100/ipp/print       │
│                                                                     │
│  // ippfind로 검색                                                   │
│  $ ippfind                                                          │
│  ipp://HP LaserJet._ipp._tcp.local/ipp/print                        │
│                                                                     │
│  【프린터 드라이버 없는 인쇄】                                         │
│  ──────────────────                                                │
│  // AirPrint / IPP Everywhere                                       │
│  $ lpadmin -p AirPrint_Printer -E -v ipp://192.168.1.100/ipp/print \│
│    -m everywhere                                                    │
│                                                                     │
│  // 드라이버 없이 PDF 직접 전송                                        │
│  $ lp -d printer_name document.pdf                                  │
│                                                                     │
│  【Python 인쇄】                                                      │
│  ──────────────────                                                │
│  import subprocess                                                  │
│                                                                     │
│  # lpr로 인쇄                                                        │
│  subprocess.run(['lpr', 'document.pdf'])                            │
│                                                                     │
│  # 옵션과 함께                                                        │
│  subprocess.run([                                                   │
│      'lpr', '-P', 'HP_LaserJet',                                    │
│      '-o', 'sides=two-sided-long-edge',                             │
│      'document.pdf'                                                 │
│  ])                                                                 │
│                                                                     │
│  【인쇄 문제 해결】                                                   │
│  ──────────────────                                                │
│  // CUPS 로그 확인                                                   │
│  $ tail -f /var/log/cups/error_log                                  │
│                                                                     │
│  // 프린터 상태                                                       │
│  $ lpstat -t                                                        │
│                                                                     │
│  // CUPS 재시작                                                      │
│  $ sudo systemctl restart cups                                      │
│                                                                     │
│  // 프린터 삭제 후 재추가                                             │
│  $ lpadmin -x HP_LaserJet                                           │
│  $ lpadmin -p HP_LaserJet -E -v socket://192.168.1.100              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 디지털 문서를 종이로 출력
• 종류: 잉크젯, 레이저, 열전사, 도트 매트릭스
• 인터페이스: USB, 이더넷, WiFi, IPP
• PDL: PCL, PostScript, PDF
• 스풀러: 인쇄 큐잉, 렌더링
• 사양: dpi(해상도), ppm(속도)
• CUPS: Linux 프린팅 시스템
• IPP: 인터넷 프린팅 프로토콜
• 용지: A4, Letter, Legal, A3
• 명령어: lpr, lpstat, lpadmin, cancel
```
