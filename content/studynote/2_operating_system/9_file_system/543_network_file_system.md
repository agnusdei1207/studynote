# 543. 썬더볼트 (Thunderbolt)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: PCIe와 DisplayPort 결합 고속 인터페이스
> 2. **가치**: 40Gbps+ 대역폭, 데이지 체인 지원
> 3. **융합**: USB4, PCIe, 디스플레이와 연관

---

## Ⅰ. 개요

### 개념 정의
**썬더볼트(Thunderbolt)**는 **Intel이 개발한 PCIe와 DisplayPort를 결합한 고속 데이터 및 비디오 인터페이스**입니다.

### 💡 비유: 초고속 멀티레인 고속도로
썬더볼트는 **데이터와 영상을 동시에 전송하는 초고속 멀티레인 고속도로**와 같습니다.

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: PCIe + DisplayPort 결합 고속 인터페이스
• 버전: 1(10G) → 5(80G/120G Boost)
• 커넥터: Mini DisplayPort (1-2), USB-C (3-5)
• 프로토콜: PCIe, DP, USB 터널링
• 데이지 체인: 최대 6장치
• USB4: Thunderbolt 3+ 호환
• 전력: 최대 100W PD
• 보안: user, secure 수준
• Linux: thunderbolt 모듈, bolt
• 용도: eGPU, 외장 저장, 4K/8K 디스플레이
```
