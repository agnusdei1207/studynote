# 559. UART (Universal Asynchronous Receiver-Transmitter)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 비동기 직렬 통신 컨트롤러
> 2. **가치**: 간단한 2선 통신, 널리 사용
> 3. **융합**: 시리얼 콘솔, 모뎀, 센서와 연관

---

## Ⅰ. 개요

### 개념 정의
**UART(Universal Asynchronous Receiver-Transmitter)**는 **비동기 직렬 통신을 위한 하드웨어 컨트롤러**입니다.

### 💡 비유: 편지 교환
UART는 **편지 교환**과 같습니다. 주고받는 타이밍이 맞아야 합니다.

### UART 구조

│  │  └─────────┘                      └─────────┘               │ │   │

│  │  TX: Transmit (전송)        RX: Receive (수신)               │ │   │

│  │                                                             │ │   │
│  │  ┌────┬──┬──┬──┬──┬──┬──┬──┬──┬──┬──┬─────┐                   │ │   │

│  │  Idle: 1 (High)                                               │ │   │
│  │  Start: 0 (Low) - 전송 시작                                    │ │   │

│  │  Parity: 짝수/홀수/없음                                        │ │   │
│  │  Stop: 1-2 비트 (High)                                        │ │   │
│  │                                                             │ │   │

│  【주요 파라미터】                                                   │

│  │                                                             │ │   │

│  │  ─────────           ──────              ────                    │ │   │
│  │  Baud Rate          9600, 115200        비트 전송 속도           │ │   │
│  │  Data Bits          7, 8                데이터 비트              │ │   │

│  │  Flow Control       None, RTS/CTS, XON/XOFF  흐름 제어          │ │   │

│  │  표준 설정: 8N1 (8 data, No parity, 1 stop)                    │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux UART 장치】                                                 │
│  ──────────────────                                                │
│  // 시리얼 장치 목록                                                   │
│  $ ls /dev/tty{S,USB,ACM}*                                         │
│  /dev/ttyS0   /dev/ttyS1    // 내장 UART                            │
│  /dev/ttyUSB0  /dev/ttyUSB1 // USB-시리얼 변환기                    │
│  /dev/ttyACM0             // 아두이노 (USB CDC ACM)                  │
│                                                                     │
│  // 장치 정보                                                         │
│  $ stty -F /dev/ttyS0 -a                                           │
│  speed 115200 baud; line = 0;                                     │
│  intr = ^C; quit = ^\; ...                                        │
│                                                                     │
│  【stty 설정】                                                        │
│  ──────────────────                                                │
│  // Baud rate 설정                                                   │
│  $ stty -F /dev/ttyS0 115200                                       │
│                                                                     │
│  // 설정 확인                                                         │
│  $ stty -F /dev/ttyS0                                              │
│  speed 115200 baud; line = 0;                                     │
│  -brkint -imaxbel                                                  │
│                                                                     │
│  // Raw 모드                                                         │
│  $ stty -F /dev/ttyS0 raw -echo                                   │
│                                                                     │
│  // 캐노니컬 모드                                                     │
│  $ stty -F /dev/ttyS0 cooked echo                                  │
│                                                                     │
│  【screen/minicom】                                                 │
│  ──────────────────                                                │
│  // screen으로 연결                                                   │
│  $ screen /dev/ttyUSB0 115200                                      │
│  $ screen /dev/ttyACM0 115200,cs8                                  │
│                                                                     │
│  // minicom                                                         │
│  $ minicom -s                        // 설정                        │
│  $ minicom -D /dev/ttyUSB0           // 연결                        │
│                                                                     │
│  // cu                                                              │
│  $ cu -l /dev/ttyUSB0 -s 115200                                    │
│                                                                     │
│  【Python (pyserial)】                                               │
│  ──────────────────                                                │
│  import serial                                                      │
│  import serial.tools.list_ports                                    │
│                                                                     │
│  # 사용 가능한 포트 목록                                              │
│  ports = serial.tools.list_ports.comports()                        │
│  for p in ports:                                                    │
│      print(f"{p.device}: {p.description}")                         │
│                                                                     │
│  # 시리얼 포트 열기                                                   │
│  ser = serial.Serial(                                              │
│      port='/dev/ttyUSB0',                                          │
│      baudrate=115200,                                              │
│      bytesize=serial.EIGHTBITS,                                    │
│      parity=serial.PARITY_NONE,                                    │
│      stopbits=serial.STOPBITS_ONE,                                 │
│      timeout=1                                                      │
│  )                                                                  │
│                                                                     │
│  # 데이터 쓰기                                                        │
│  ser.write(b'Hello\n')                                             │
│  ser.write(bytes([0x01, 0x02, 0x03]))                              │
│                                                                     │
│  # 데이터 읽기                                                        │
│  data = ser.read(10)          # 10바이트 읽기                        │
│  line = ser.readline()        # 라인 읽기                           │
│  all_data = ser.read_all()    # 모두 읽기                           │
│                                                                     │
│  # 닫기                                                             │
│  ser.close()                                                        │
│                                                                     │
│  【C (termios)】                                                      │
│  ──────────────────                                                │
│  #include <fcntl.h>                                                 │
│  #include <termios.h>                                               │
│  #include <unistd.h>                                                │
│                                                                     │
│  int fd = open("/dev/ttyUSB0", O_RDWR | O_NOCTTY);                 │
│                                                                     │
│  struct termios tty;                                                │
│  tcgetattr(fd, &tty);                                              │
│                                                                     │
│  cfsetispeed(&tty, B115200);                                       │
│  cfsetospeed(&tty, B115200);                                       │
│                                                                     │
│  tty.c_cflag &= ~PARENB;      // No parity                         │
│  tty.c_cflag &= ~CSTOPB;      // 1 stop bit                        │
│  tty.c_cflag &= ~CSIZE;                                             │
│  tty.c_cflag |= CS8;          // 8 data bits                       │
│  tty.c_cflag &= ~CRTSCTS;     // No flow control                   │
│  tty.c_cflag |= CREAD | CLOCAL;                                     │
│                                                                     │
│  tcsetattr(fd, TCSANOW, &tty);                                      │
│                                                                     │
│  // 쓰기                                                            │
│  write(fd, "Hello\n", 6);                                          │
│                                                                     │
│  // 읽기                                                            │
│  char buf[100];                                                    │
│  int n = read(fd, buf, sizeof(buf));                               │
│                                                                     │
│  close(fd);                                                         │
│                                                                     │
│  【UART 문제 해결】                                                   │
│  ──────────────────                                                │
│  // 권한                                                            │
│  $ sudo usermod -aG dialout $USER                                  │
│                                                                     │
│  // 장치 인식 확인                                                    │
│  $ dmesg | grep tty                                                │
│  usb 1-1: cp210x converter now attached to ttyUSB0                 │
│                                                                     │
│  // 파일럿 오류 확인                                                  │
│  $ cat /proc/tty/driver/serial                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 비동기 직렬 통신 컨트롤러
• 신호: TX, RX, GND (+ RTS/CTS 선택)
• 프레임: Start + Data + Parity + Stop
• Baud Rate: 9600 - 921600+ bps
• 설정: 8N1 (8bit, No parity, 1 stop)
• 흐름 제어: 없음, RTS/CTS, XON/XOFF
• 장치: /dev/ttyS*, /dev/ttyUSB*, /dev/ttyACM*
• 도구: screen, minicom, stty
• Python: pyserial (serial.Serial)
• 용도: 콘솔, 모뎀, 센서, 아두이노
```
