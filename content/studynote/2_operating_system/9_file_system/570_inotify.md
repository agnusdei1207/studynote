# 570. BIOS/UEFI

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시스템 부팅 및 하드웨어 초기화 펌웨어
> 2. **가치**: POST, 부트 로더, 하드웨어 구성
> 3. **융합**: 부팅, ACPI, 드라이버와 연관

---

## Ⅰ. 개요

### 개념 정의
**BIOS/UEFI**는 **시스템 시작 시 하드웨어 초기화 및 부팅을 담당하는 펌웨어**입니다.

### 💡 비유: 오케스트라 지휘자
BIOS/UEFI는 **오케스트라 지휘자**와 같습니다. 모든 하드웨어를 준비시키고 시작합니다.

### BIOS/UEFI 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                BIOS/UEFI 구조                                       │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【BIOS vs UEFI】                                                    │ |
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  특성              BIOS               UEFI                    │ │   │
│  │  ────              ────               ────                    │ │   │
│  │  아키텍처          16비트             32/64비트                │ │   │
│  │  주소 공간          1 MB              4 GB+                   │ │   │
│  │  파티션            MBR (2TB 제한)     GPT (무제한)             │ │   │
│  │  부팅 방식          MBR 부트 코드       EFI 실행 파일           │ │   │
│  │  드라이브 지원       최대 4개 파티션     무제한                 │ │   │
│  │  보안              없음               Secure Boot             │ │   │
│  │  네트워크 부팅       PXE               PXE/HTTP                │ │   │
│  │  UI                텍스트             그래픽                   │ │   │
│  │  확장              제한적             드라이버, 앱              │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【부팅 과정】                                                        │ |
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  [전원 ON]                                                      │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  ┌─────────┐                                                │ │   │
│  │  │  POST   │  → 전원 자가 진단 (Power-On Self-Test)           │ │   │
│  │  └─────────┘                                                │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  ┌─────────┐                                                │ │   │
│  │  │하드웨어 초기화│ → CPU, 메모리, 장치 검색/구성                │ │   │
│  │  └─────────┘                                                │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  ┌─────────┐                                                │ │   │
│  │  │부트 장치 선택│ → BIOS: MBR / UEFI: EFI 파티션               │ │   │
│  │  └─────────┘                                                │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  ┌─────────┐                                                │ │   │
│  │  │부트 로더 로드│ → GRUB, Windows Boot Manager               │ │   │
│  │  └─────────┘                                                │ │   │
│  │       │                                                     │ │   │
│  │       ▼                                                     │ │   │
│  │  [OS 부팅]                                                      │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【UEFI 파티션 (ESP)】                                               │ |
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  파티션 타입        GUID                   크기                 │ │   │
│  │  ──────────        ────                   ────                 │ │   │
│  │  EFI System        C12A7328-F81F-11D2-BA4B-00A0C93EC93B     100-500MB│
│  │  Microsoft Basic   EBD0A0A2-B9E5-4433-87C0-68B6B72699C7     -       │
│  │  Linux Filesystem  0FC63DAF-8483-4772-8E79-3D69D8477DE4     -       │
│  │  Linux Swap        0657FD6D-A4AB-43C4-84E5-0933C84B4F4F     -       │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                BIOS/UEFI 상세                                       │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【POST 과정】                                                        │ |
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  단계              검사 항목              오류 코드              │ │   │
│  │  ────              ────────              ────────               │ │   │
│  │  1                 CPU                   비프음 1번               │ │   │
│  │  2                 메모리                 비프음 3번              │ │   │
│  │  3                 VGA                   비프음 1긴 2짧           │ │   │
│  │  4                 키보드                 비프음 6번              │ │   │
│  │  5                 디스크                 화면 메시지             │ │   │
│  │  6                 기타 장치               화면 메시지            │ │   │
│  │                                                             │ │   │
│  │  비프음 코드는 제조사마다 다름                                   │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【Secure Boot】                                                      │ |
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  목적: 악성 코드/변조된 OS 부팅 방지                             │ │   │
│  │                                                             │ │   │
│  │  작동 방식:                                                      │ │   │
│  │  1. UEFI 펌웨어에 저장된 키로 EFI 실행 파일 서명 확인             │ │   │
│  │  2. 신뢰할 수 있는 서명이면 부팅 허용                             │ │   │
│  │  3. 아니면 부팅 차단                                             │ │   │
│  │                                                             │ │   │
│  │  키 종류:                                                        │ │   │
│  │  • KEK (Key Exchange Key): 키 업데이트용                        │ │   │
│  │  • db (Signature Database): 허용된 서명                          │ │   │
│  │  • dbx (Forbidden Signatures): 차단된 서명                       │ │   │
│  │  • PK (Platform Key): 최상위 키                                 │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【MBR vs GPT】                                                       │ |
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  특성              MBR                 GPT                     │ │   │
│  │  ────              ───                 ────                     │ │   │
│  │  최대 디스크        2 TB               9.4 ZB                  │ │   │
│  │  최대 파티션        4개 (기본)          128개 (Windows)          │ │   │
│  │  예약 영역          없음               MBR 보호 파티션           │ │   │
│  │  백업              없음               끝에 백업                 │ │   │
│  │  CRC              없음               있음                      │ │   │
│  │  호환성            BIOS               UEFI (CSM으로 BIOS도)     │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【UEFI 부팅 항목 관리】                                              │ |
│  ──────────────────                                                │
│  // efibootmgr 설치                                                   │
│  $ sudo apt install efibootmgr                                     │
│                                                                     │
│  // 부팅 항목 목록                                                     │
│  $ efibootmgr                                                       │
│  BootCurrent: 0001                                                  │
│  BootOrder: 0001,0000,0002                                         │
│  Boot0000* Windows Boot Manager                                    │
│  Boot0001* ubuntu                                                  │
│  Boot0002* UEFI: Built-in EFI Shell                                │
│                                                                     │
│  // 부팅 순서 변경                                                    │
│  $ sudo efibootmgr -o 0000,0001                                    │
│                                                                     │
│  // 부팅 항목 추가                                                    │
│  $ sudo efibootmgr -c -d /dev/sda -p 1 -L "MyBoot" -l '\EFI\myboot\bootx64.efi'│
│                                                                     │
│  // 부팅 항목 삭제                                                    │
│  $ sudo efibootmgr -b 0002 -B                                      │
│                                                                     │
│  【ESP (EFI System Partition)】                                       │ |
│  ──────────────────                                                │
│  // ESP 마운트                                                        │
│  $ sudo mount /dev/sda1 /mnt/efi                                   │
│                                                                     │
│  // ESP 내용                                                          │
│  $ ls /mnt/efi/EFI/                                                 │
│  BOOT/  ubuntu/  Microsoft/                                        │
│                                                                     │
│  // EFI 실행 파일                                                    │
│  $ ls /mnt/efi/EFI/ubuntu/                                         │
│  grubx64.efi  shimx64.efi  mmx64.efi                               │
│                                                                     │
│  【Secure Boot 관리】                                                 │ |
│  ──────────────────                                                │
│  // Secure Boot 상태 확인                                             │
│  $ mokutil --sb-state                                               │
│  SecureBoot enabled                                                 │
│                                                                     │
│  // 키 상태                                                           │
│  $ mokutil --list-enrolled                                          │
│                                                                     │
│  // MOK (Machine Owner Key) 등록                                     │
│  $ sudo mokutil --import /path/to/key.der                          │
│  // 재부팅 후 비밀번호 입력                                           │
│                                                                     │
│  // Secure Boot 비활성화 (BIOS에서)                                   │
│  // 또는                                                             │
│  $ sudo mokutil --disable-validation                                │
│                                                                     │
│  【BIOS/UEFI 설정 덤프】                                              │ |
│  ──────────────────                                                │
│  // dmidecode                                                        │
│  $ sudo dmidecode -t bios                                           │
│  BIOS Information                                                   │
│    Vendor: American Megatrends Inc.                                 │
│    Version: 2.10                                                    │
│    Release Date: 01/15/2024                                        │
│    ROM Size: 16 MB                                                 │
│    Characteristics:                                                 │
│      PCI is supported                                               │
│      Boot from CD is supported                                      │
│      ACPI is supported                                              │
│      UEFI is supported                                              │
│                                                                     │
│  // 부팅 모드 확인                                                    │
│  $ [ -d /sys/firmware/efi ] && echo "UEFI" || echo "BIOS"          │
│  UEFI                                                               │
│                                                                     │
│  【GRUB 재설치】                                                       │ |
│  ──────────────────                                                │
│  // BIOS 모드                                                         │
│  $ sudo grub-install /dev/sda                                      │
│                                                                     │
│  // UEFI 모드                                                         │
│  $ sudo grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=ubuntu│
│                                                                     │
│  // GRUB 업데이트                                                     │
│  $ sudo update-grub                                                 │
│                                                                     │
│  【UEFI 변수 백업/복원】                                              │ |
│  ──────────────────                                                │
│  // 변수 백업                                                         │
│  $ sudo efivar -L > efivar-list.txt                                │
│                                                                     │
│  // 특정 변수 읽기                                                    │
│  $ cat /sys/firmware/efi/efivars/BootCurrent-8be4df61-93ca-11d2-aa0d-00e098032b8c│
│                                                                     │
│  // 변수 쓰기 (주의!)                                                  │
│  # printf '\x07\x00\x00\x00\x01\x00\x00' > /sys/firmware/efi/efivars/...│
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 시스템 부팅 및 하드웨어 초기화 펌웨어
• BIOS: 16비트, MBR, 2TB 제한, 텍스트 UI
• UEFI: 32/64비트, GPT, Secure Boot, 그래픽 UI
• POST: 전원 자가 진단, 비프음 코드
• 부팅 과정: POST → 하드웨어 초기화 → 부트 로더 → OS
• Secure Boot: 서명된 OS만 부팅 허용
• ESP: EFI System Partition (FAT32)
• 도구: efibootmgr, mokutil, dmidecode
• 파티션: MBR vs GPT
• 경로: /sys/firmware/efi/efivars/
```
