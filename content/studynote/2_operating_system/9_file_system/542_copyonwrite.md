# 542. 블루투스 (Bluetooth)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단거리 무선 통신 기술
> 2. **가치**: 저전력, 저비용, 무선 연결
> 3. **융합**: PAN, 무선 오디오, IoT와 연관

---

## Ⅰ. 개요

### 개념 정의
**블루투스(Bluetooth)**는 **2.4GHz 대역을 사용하는 단거리 무선 통신 기술**입니다.

### 💡 비유: 보이지 않는 케이블
블루투스는 **보이지 않는 케이블**과 같습니다. 선 없이 장치를 연결합니다.

```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                블루투스 상세                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【주파수 대역】                                                      │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  ISM 대역: 2.4 GHz (2.4000 - 2.4835 GHz)                      │ │   │
│  │  채널: 79개 (1 MHz 간격, 채널 0-78)                             │ │   │
│  │  호핑: 초당 1600회 주파수 변경 (FHSS)                           │ │   │
│  │                                                             │ │   │
│  │  BLE: 40개 채널 (2 MHz 간격, 채널 37-39: 광고용)               │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【블루투스 Low Energy (BLE)】                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  특징:                                                       │ │   │
│  │  • 매우 낮은 전력 소비 (코인 건전지로 수년 사용)                 │ │   │
│  │  • 빠른 연결/해제                                              │ │   │
│  │  • GATT 기반 데이터 교환                                        │ │   │
│  │  • 최대 50m 범위                                               │ │   │
│  │                                                             │ │   │
│  │  GATT 구조:                                                   │ │   │
│  │  ┌─────────────────────────────────────────────────────┐       │ │   │
│  │  │  Service (서비스)                                    │       │ │   │
│  │  │    └── Characteristic (특성)                         │       │ │   │
│  │  │          ├── Value (값)                              │       │ │   │
│  │  │          ├── Properties (속성: 읽기/쓰기/알림)        │       │ │   │
│  │  │          └── Descriptor (설명자)                     │       │ │   │
│  │  └─────────────────────────────────────────────────────┘       │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【클래식 vs BLE 비교】                                               │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  특징              클래식              BLE                        │ │   │
│  │  ────              ──────              ────                        │ │   │
│  │  전력              높음 (1W)           낮음 (0.01-0.5W)              │ │   │
│  │  연결 시간          수 초               수 밀리초                    │ │   │
│  │  데이터 전송        2-3 Mbps           1 Mbps                      │ │   │
│  │  범위              ~100m              ~50m                        │ │   │
│  │  용도              오디오, 데이터      센서, 비콘                    │ │   │
│  │  프로파일          다양함              GATT 기반                     │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 2.4GHz 단거리 무선 통신
• 버전: 1.0 → 5.3 (속도 및 기능 향상)
• 구조: 피코넷 (마스터 + 최대 7 슬레이브)
• 프로파일: A2DP, HFP, HID, PAN, GATT
• BLE: 저전력, 빠른 연결, 센서/IoT
• 주파수: 2.4GHz ISM 대역, 79채널 (BLE: 40채널)
• 도구: hcitool, bluetoothctl, bluepy
• 용도: 오디오, 입력장치, 센서, 테더링
```
