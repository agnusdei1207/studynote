# 534. Scatter/Gather I/O (벡터화 I/O)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 단일 요청으로 여러 I/O 작업 수행
> 2. **가치**: I/O 성능 최적화 및 오버헤드 감소
> 3. **융합**: 비동기 I/O, DMA, 벡터화와 연관

---

## Ⅰ. 개요

### 개념 정의
**Scatter/Gather I/O**는 **단일 요청으로 여러 비동기 I/O 작업을 효율적으로 수행하는 기법**입니다.

### 💡 비유: 쇼핑 배달
Scatter/Gather I/O는 **한 번의 주문으로 여러 매장에 배달하는 것**과 같습니다. 여러 배달원이 각자 매장으로 흩어지고, 완료 후 다시 모입니다.

### Scatter/Gather I/O 구조

│  │                                                             │ │   │

│  │     • 결과를 단일 응답으로 통합               │ │   │

│  │       │                                                         │ │   │

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                Scatter/Gather I/O 상세                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【readv (벡터화 읽기)】                                              │
│  ──────────────────                                                │
│  #include <sys/uio.h>                                              │
│                                                                     │
│  ssize_t readv(int fd, const struct iovec *iov, int iovcnt);        │
│                                                                     │
│  struct iovec {                                                      │
│      void  *iov_base;    // 버퍼 시작 주소                         │
│      size_t iov_len;    // 버퍼 길이                              │
│  };                                                                │
│                                                                     │
│  【writev (벡터화 쓰기)】                                             │
│  ──────────────────                                                │
│  ssize_t writev(int fd, const struct iovec *iov, int iovcnt);       │
│                                                                     │
│  【장단점】                                                             │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  장점:                                                       │ │   │
│  │  • 단일 시스템 콜로 여러 버퍼 처리                          │ │   │
│  │  • 시스템 콜 오버헤드 감소                          │ │   │
│  │  • 연속된 파일 영역 효율적 처리                        │ │   │
│  │  • 네트워크 프로토콜 구현에 적합                        │ │   │
│  │                                                             │ │   │
│  │  단점:                                                       │ │   │
│  │  • 복잡한 버퍼 관리                                        │ │   │
│  │  • iovcnt 제한 (UIO_MAXIOV)                                │ │   │
│  │  • 부분적 읽기/쓰기 처리 필요                            │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【Scatter/Gather DMA】                                              │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  일반 DMA: 연속된 메모리 영역 전송                              │ │   │
│  │  Scatter-Gather DMA: 불연속 메모리 영역을 단일 전송으로 처리     │ │   │
│  │                                                             │ │   │
│  │  장점:                                                       │ │   │
│  │  • 불연속 메모리를 단일 전송으로 처리                        │ │   │
│  │  • DMA 설정 횟수 감소                                    │ │   │
│  │  • 프로토콜 헤더/페이로드 분리 용이                        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【writev 예시】                                                     │
│  ──────────────────                                                │
│  #include <sys/uio.h>                                              │
│  #include <fcntl.h>                                                │
│  #include <unistd.h>                                               │
│                                                                     │
│  int main() {                                                        │
│      int fd = open("output.txt", O_WRONLY | O_CREAT, 0644);          │
│                                                                     │
│      // 여러 버퍼 정의                                          │
│      struct iovec iov[3];                                          │
│      char header[] = "HEADER\n";                                   │
│      char body[] = "This is the body content.\n";                  │
│      char footer[] = "FOOTER\n";                                   │
│                                                                     │
│      iov[0].iov_base = header;                                    │
│      iov[0].iov_len = strlen(header);                             │
│      iov[1].iov_base = body;                                      │
│      iov[1].iov_len = strlen(body);                               │
│      iov[2].iov_base = footer;                                    │
│      iov[2].iov_len = strlen(footer);                             │
│                                                                     │
│      // 단일 호출로 모든 버퍼 쓰기                              │
│      ssize_t written = writev(fd, iov, 3);                           │
│      printf("Total written: %zd bytes\n", written);                 │
│                                                                     │
│      close(fd);                                                     │
│      return 0;                                                        │
│  }                                                                   │
│                                                                     │
│  【readv 예시】                                                     │
│  ──────────────────                                                │
│  #include <sys/uio.h>                                              │
│                                                                     │
│  int main() {                                                        │
│      int fd = open("input.txt", O_RDONLY);                         │
│                                                                     │
│      // 여러 버퍼 준비                                          │
│      struct iovec iov[2];                                          │
│      char header[10];                                              │
│      char data[100];                                               │
│                                                                     │
│      iov[0].iov_base = header;                                    │
│      iov[0].iov_len = sizeof(header);                              │
│      iov[1].iov_base = data;                                      │
│      iov[1].iov_len = sizeof(data);                                │
│                                                                     │
│      // 단일 호출로 여러 버퍼에 읽기              │
│      ssize_t nread = readv(fd, iov, 2);                                │
│      if (nread > 0) {                                              │
│          printf("Header: %.10s\n", header);                       │
│          printf("Data: %.100s\n", data);                         │
│      }                                                             │
│      close(fd);                                                     │
│      return 0;                                                │
│  }                                                                   │
│                                                                     │
│  【네트워크 송신 예시 (TCP)】                                       │
│  ──────────────────                                                │
│  #include <sys/socket.h>                                             │
│  #include <sys/uio.h>                                              │
│                                                                     │
│  void send_message(int sock, const char *header, int hlen,                   │
│                     const char *body, int blen) {                │
│      struct iovec iov[2];                                            │
│      iov[0].iov_base = (void*)header;                                 │
│      iov[0].iov_len = hlen;                                       │
│      iov[1].iov_base = (void*)body;                                   │
│      iov[1].iov_len = blen;                                       │
│                                                             │
│      writev(sock, iov, 2);  // 헤더 + 본문 한 번에 전송 │
│  }                                                             │
│                                                                     │
│  【preadv/pwritev (분산 읽기/쓰기)】                                        │
│  ──────────────────                                                │
│  //preadv: 파일의 여러 위치에서 읽기                      │
│  ssize_t preadv(int fd, const struct iovec *iov, int iovcnt,       │
│               off_t offset);                                              │
│                                                                     │
│  //pwritev: 파일의 여러 위치에 쓰기                     │
│  ssize_t pwritev(int fd, const struct iovec *iov, int iovcnt,      │
│                off_t offset);                                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

 DMA: Scatter-Gather DMA로 불연속 전송
```
