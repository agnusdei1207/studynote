# 548. 사운드 카드 (Sound Card)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 오디오 입출력 전용 하드웨어
> 2. **가치**: ADC/DAC, DSP, 믹싱 기능 제공
> 3. **융합**: 오디오, 코덱, ALSA와 연관

---

## Ⅰ. 개요

### 개념 정의
**사운드 카드(Sound Card)**는 **오디오 입출력을 전담하는 확장 카드 하드웨어**입니다.

### 💡 비유: 소리 공장
사운드 카드는 **소리 공장**과 같습니다. 디지털 신호를 소리로, 소리를 디지털로 바꿉니다.

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 오디오 입출력 전용 하드웨어
• 구성: DSP, ADC, DAC, 믹서, 증폭기
• 인터페이스: Line In/Out, Mic, SPDIF, MIDI
• 종류: 온보드, PCIe, USB 외장, 전문가용
• DAC: 16-32 bit, 48-384 kHz
• SNR: 90-120 dB
• 칩셋: Realtek, Creative, Focusrite
• Linux: ALSA, /proc/asound/
• DSP: EQ, 리버브, 컴프레서, 3D 오디오
• 지연: 1-20 ms
```
