# 550. 터치스크린 (Touchscreen)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 터치 입력을 감지하는 디스플레이
> 2. **가치**: 직관적 UI, 멀티터치 지원
> 3. **융합**: HID, 디스플레이, 제스처와 연관

---

## Ⅰ. 개요

### 개념 정의
**터치스크린(Touchscreen)**은 **터치 입력을 감지하고 시각적 출력을 동시에 제공하는 장치**입니다.

### 💡 비유: 마법 스크린
터치스크린은 **마법 스크린**과 같습니다. 손가락으로 그리면 반응합니다.

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux 터치스크린 확인】                                           │
│  ──────────────────                                                │
│  // 터치 장치 찾기                                                    │
│  $ cat /proc/bus/input/devices | grep -A 10 -i touch                │
│  N: Name="ELAN Touchscreen"                                         │
│  B: PROP=2                                                          │
│  A: ABS=661800011000003                                             │
│                                                                     │
│  // 입력 장치 목록                                                    │
│  $ xinput list                                                      │
│  ⎡ Virtual core pointer                    	id=2	[master pointer]│
│  ⎜   ↳ ELAN Touchscreen                   	id=9	[slave  pointer]│
│                                                                     │
│  // libinput 장치                                                    │
│  $ libinput list-devices | grep -A 20 Touch                         │
│  Device:           ELAN Touchscreen                                 │
│  Capabilities:     touch                                            │
│                                                                     │
│  【터치스크린 캘리브레이션】                                           │
│  ──────────────────                                                │
│  // xinput 캘리브레이션                                               │
│  $ xinput --list-props "ELAN Touchscreen"                           │
│  $ xinput --set-prop "ELAN Touchscreen" "Coordinate Transformation" │
│                                                                     │
│  // xorg.conf 설정                                                   │
│  Section "InputClass"                                               │
│      Identifier "touchscreen"                                       │
│      MatchIsTouchscreen "on"                                        │
│      Option "TransformationMatrix" "1 0 0 0 1 0 0 0 1"              │
│  EndSection                                                         │
│                                                                     │
│  // tslib (터치스크린 라이브러리)                                      │
│  $ ts_calibrate                                                     │
│  $ ts_test                                                          │
│                                                                     │
│  【evdev로 터치 이벤트 캡처】                                          │
│  ──────────────────                                                │
│  $ sudo evtest /dev/input/event5                                    │
│  Event: time 1234567890.123456, type 3 (EV_ABS), code 53 (ABS_MT_), │
│         value 500                                                   │
│  Event: time 1234567890.123456, type 3 (EV_ABS), code 54 (ABS_MT_), │
│         value 300                                                   │
│  Event: time 1234567890.123456, type 1 (EV_KEY), code 330 (BTN_T),  │
│         value 1                                                     │
│                                                                     │
│  【Python 터치 이벤트 처리】                                          │
│  ──────────────────                                                │
│  import evdev                                                       │
│                                                                     │
│  # 터치 장치 찾기                                                     │
│  devices = [evdev.InputDevice(path) for path in evdev.list_devices()]│
│  touch_dev = None                                                   │
│  for dev in devices:                                                │
│      if 'touch' in dev.name.lower():                                │
│          touch_dev = dev                                            │
│          break                                                      │
│                                                                     │
│  # 이벤트 수신                                                        │
│  for event in touch_dev.read_loop():                                │
│      if event.type == evdev.ecodes.EV_ABS:                          │
│          if event.code == evdev.ecodes.ABS_MT_POSITION_X:           │
│              print(f"Touch X: {event.value}")                       │
│          elif event.code == evdev.ecodes.ABS_MT_POSITION_Y:         │
│              print(f"Touch Y: {event.value}")                       │
│                                                                     │
│  【터치스크린 회전】                                                  │
│  ──────────────────                                                │
│  // xrandr로 화면 회전                                                │
│  $ xrandr --output eDP-1 --rotate right                             │
│                                                                     │
│  // 터치 입력도 함께 회전                                              │
│  $ xinput --set-prop "ELAN Touchscreen" \                           │
│      "Coordinate Transformation Matrix" 0 1 0 -1 0 1 0 0 1          │
│                                                                     │
│  // 90도 회전 행렬                                                   │
│  # 0 1 0                                                            │
│  # -1 0 1                                                           │
│  # 0 0 1                                                            │
│                                                                     │
│  【멀티터치 테스트】                                                  │
│  ──────────────────                                                │
│  $ sudo apt install mtview                                          │
│  $ sudo mtview /dev/input/event5                                    │
│                                                                     │
│  // 또는                                                            │
│  $ sudo apt install evtest                                          │
│  $ sudo evtest /dev/input/event5                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 터치 입력 감지 디스플레이
• 기술: 정전식, 저항식, 적외선, 광학식
• 정전식: 자기정전식, 상호정전식
• 멀티터치: 프로토콜 B (ABS_MT_*)
• 제스처: 탭, 스와이프, 핀치, 회전
• 이벤트: BTN_TOUCH, ABS_X/Y, ABS_MT_*
• Linux: evdev, libinput, xinput
• 캘리브레이션: tslib, 변환 행렬
• 회전: Coordinate Transformation Matrix
• HID: Digitizer Usage Page
```
