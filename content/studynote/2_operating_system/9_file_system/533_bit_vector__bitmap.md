# 533. 메모리 맵핑 파일 (Memory-Mapped File)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일을 메모리 주소 공간에 매핑
> 2. **가치**: 간단한 파일 액세 인터페이스 제공
> 3. **융합**: 메모리 관리, mmap, 파일 시스템과 연관

---

## Ⅰ. 개요

### 개념 정의
**메모리 맵핑 파일(Memory-Mapped File)**은 **파일의 내용을 프로세스의 가상 주소 공간에 직접 매핑하여, 일반 메모리 접근 명령으로 파일에 접근하는 기술**입니다.

### 💡 비유: 책을 책상에 올려놓기
메모리 맵핑 파일은 **책을 책장에서 책상으로 옮겨놓은 것**과 같습니다. 책상에서 바로 읽을 수 있습니다.

### 메모리 맵핑 파일 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                메모리 맵핑 파일 구조                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 개념】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │       파일 시스템                프로세스 주소 공간                │ │   │
│  │  ┌─────────────────────┐     ┌─────────────────────────────┐     │ │   │
│  │  │                     │     │                             │     │ │   │
│  │  │   파일 데이터        │────│▶  가상 메모리 영역             │     │ │   │
│  │  │   (디스크)           │     │   (직접 매핑)               │     │ │   │
│  │  │                     │     │                             │     │ │   │
│  │  └─────────────────────┘     └─────────────────────────────┘     │ │   │
│  │                                                             │ │   │
│  │  장점:                                                       │ │   │
│  │  • read()/write() 시스템 콜 없이 메모리처럼 접근               │ │   │
│  │  • 복사 오버헤드 없음                                        │ │   │
│  │  • 여러 프로세스 간 메모리 공유 가능                           │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【mmap() 시스템 콜】                                                │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  #include <sys/mman.h>                                        │ │   │
│  │                                                             │ │   │
│  │  void *mmap(                                                │ │   │
│  │      void *addr,      // 매핑 시작 주소 (NULL=자동)               │ │   │
│  │      size_t length,   // 매핑 길이                               │ │   │
│  │      int prot,        // 보호 플래그 (PROT_READ/WRITE/EXEC)          │ │   │
│  │      int flags,       // 매핑 플래그 (MAP_SHARED/PRIVATE/ANON)         │ │   │
│  │      int fd,          // 파일 디스크립터                         │ │   │
│  │      off_t offset     // 파일 오프셋                              │ │   │
│  │  );                                                          │ │   │
│  │                                                             │ │   │
│  │  int munmap(void *addr, size_t length);  // 매핑 해제              │ │   │
│  │  int msync(void *addr, size_t length, int flags);  // 동기화        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【매핑 플래그】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  플래그              설명                                        │ │   │
│  │  ────              ────                                        │ │   │
│  │  MAP_SHARED        변경사항이 파일에 반영                          │ │   │
│  │  MAP_PRIVATE       변경사항이 파일에 반영 안 됨 (COW)               │ │   │
│  │  MAP_ANONYMOUS     파일 없이 메모리만 매핑                           │ │   │
│  │  MAP_FIXED         지정된 주소에 정확히 매핑                           │ │   │
│  │  MAP_POPULATE      페이지를 미리 로드                               │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                메모리 맵핑 파일 상세                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【페이지 폴트 처리】                                                │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  1. 파일에서 읽기:                                             │ │   │
│  │     • 페이지 폴트 발생 시 파일에서 데이터 로드                        │ │   │
│  │     • 페이지 테이블 업데이트                                  │ │   │
│  │     • 프로세스 재개                                            │ │   │
│  │                                                             │ │   │
│  │  2. 쓰기 시:                                                   │ │   │
│  │     • MAP_PRIVATE: COW (Copy-on-Write) 발생                       │ │   │
│  │     • MAP_SHARED: 더티 페이지로 표시                              │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【동기화 (msync)】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  // 메모리 변경 사항을 파일에 동기화                             │ │   │
│  │  msync(addr, length, MS_SYNC);      // 동기 쓰기                    │ │   │
│  │  msync(addr, length, MS_ASYNC);    // 비동기 쓰기                   │ │   │
│  │  msync(addr, length, MS_INVALIDATE); // 캐시 무효화                   │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【mmap vs read/write】                                               │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  특징                read/write               mmap                          │ │   │
│  │  ────                ──────────                ────                          │ │   │
│  │  시스템 콜              매번 호출                   초기 설정 후                │ │   │
│  │  데이터 복사           커널 버퍼 ↔ 사용자 공간            메모리 매핑               │ │   │
│  │  성능                작은 파일에 유리            큰 파일에 유리              │ │   │
│  │  메모리 사용            커널 버퍼              가상 주소 공간               │ │   │
│  │  동시성               제한적                       페이지 단위 잠금               │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기본 mmap 예시】                                                │
│  ──────────────────                                                │
│  #include <sys/mman.h>                                              │
│  #include <sys/stat.h>                                               │
│  #include <fcntl.h>                                               │
│  #include <unistd.h>                                               │
│                                                                     │
│  int main() {                                                        │
│      int fd = open("test.txt", O_RDWR | O_CREAT, 0644);              │
│      if (fd < 0) {                                                    │
│          perror("open");                                         │
│          return 1;                                                │
│      }                                                             │
│                                                                     │
│      // 파일 크기 설정                                               │
│      ftruncate(fd, 4096);                                            │
│                                                                     │
│      // 메모리 매핑                                               │
│      char *map = mmap(NULL, 4096, PROT_READ | PROT_WRITE,       │
│                         MAP_SHARED, fd, 0);                                 │
│      if (map == MAP_FAILED) {                                         │
│          perror("mmap");                                         │
│          return 1;                                                │
│      }                                                             │
│                                                                     │
│      // 파일에 데이터 쓰기                                               │
│      const char *data = "Hello, mmap!";                              │
│      memcpy(map, data, strlen(data));                             │
│                                                                     │
│      // 동기화                                                  │
│      msync(map, 4096, MS_SYNC);                                   │
│                                                                     │
│      // 메모리 읽기                                               │
│      printf("Mapped data: %s\n", map);                            │
│                                                                     │
│      // 매핑 해제                                               │
│      munmap(map, 4096);                                              │
│      close(fd);                                                     │
│      return 0;                                                        │
│  }                                                                   │
│                                                                     │
│  【익명 매핑 (malloc 대체)】                                           │
│  ──────────────────                                                │
│  // MAP_ANONYMOUS 사용 - 스왑/힙 영역                               │
│  void *mem = mmap(NULL, size, PROT_READ | PROT_WRITE,                 │
│                        MAP_PRIVATE | MAP_ANONYMOUS, -1, 0);     │
│  // ... 사용 ...                                                  │
│  munmap(mem, size);                                                │
│                                                                     │
│  【IPC 공유 메모리】                                                │
│  ──────────────────                                                │
│  // 부모 프로세스                                                  │
│  int fd = open("/dev/shm/myshm", O_RDWR | O_CREAT, 0644);               │
│  ftruncate(fd, size);                                              │
│  char *shared = mmap(NULL, size, PROT_READ | PROT_WRITE,                 │
│                        MAP_SHARED, fd, 0);                           │
│                                                                     │
│  // 자식 프로세스                                                  │
│  int fd = open("/dev/shm/myshm", O_RDWR);                          │
│  char *shared = mmap(NULL, size, PROT_READ | PROT_WRITE,                 │
│                        MAP_SHARED, fd, 0);                           │
│                                                                     │
│  【파일 크기 변경】                                                │
│  ──────────────────                                                │
│  #include <unistd.h>                                               │
│                                                                     │
│  void set_file_size(int fd, off_t new_size) {                          │
│      // 파일 크기 변경                                                  │
│      if (ftruncate(fd, new_size) < 0) {                              │
│          // 실패 시 munmap 후 다시 mmap                           │
│      }                                                             │
│      // 또는 posix_fallocate 사용                               │
│  }                                                             │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 파일을 메모리 주소 공간에 직접 매핑
• 시스템 콜: mmap, munmap, msync
• 플래그: MAP_SHARED, MAP_PRIVATE, MAP_ANONYMOUS
• 장점: 시스템 콜 없음, 복사 없음, 공유 메모리
• 페이지 폴트: 온디맨드 로드
• COW: MAP_PRIVATE의 Copy-on-Write
• 용도: 대용량 파일 처리, IPC, 메모리 할당
```
