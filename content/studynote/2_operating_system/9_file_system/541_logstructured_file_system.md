# 541. USB (Universal Serial Bus)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 범용 직렬 버스 인터페이스
> 2. **가치**: 플러그 앤 플레이, 핫 플러그 지원
> 3. **융합**: 버스, 프로토콜, 장치 드라이버와 연관

---

## Ⅰ. 개요

### 개념 정의
**USB(Universal Serial Bus)**는 **컴퓨터와 주변 장치 간 연결을 위한 표준 직렬 버스 인터페이스**입니다.

### 💡 비유: 만능 콘센트
USB는 **만능 콘센트**와 같습니다. 어떤 전자 기기든 같은 콘센트에 연결할 수 있습니다.

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                USB 상세                                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【USB 전송 타입】                                                    │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  타입              특징                용도                       │ │   │
│  │  ────              ────                ────                       │ │   │
│  │  Control          설정/상태 전용         장치 초기화, 설정          │ │   │
│  │  Interrupt        낮은 지연 보장         키보드, 마우스              │ │   │
│  │  Bulk             최선형 전달            대용량 데이터             │ │   │
│  │  Isochronous      대역폭 보장           오디오, 비디오             │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【USB 디바이스 클래스】                                              │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  클래스          코드       설명                               │ │   │
│  │  ──────          ────       ────                                │ │   │
│  │  HID             0x03       휴먼 인터페이스 장치 (키보드, 마우스)    │ │   │
│  │  Mass Storage    0x08       대용량 저장 장치 (USB 드라이브)        │ │   │
│  │  CDC             0x02       통신 장치 (모뎀, 이더넷)                │ │   │
│  │  Audio           0x01       오디오 장치                          │ │   │
│  │  Video           0x0E       비디오 장치 (웹캠)                      │ │   │
│  │  Printer         0x07       프린터                                │ │   │
│  │  Hub             0x09       USB 허브                              │ │   │
│  │  Bluetooth       0xE0       블루투스 어댑터                        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【USB 전력 공급 (USB PD)】                                           │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  프로파일       전압           전류         전력                │ │   │
│  │  ──────        ────           ────        ────                 │ │   │
│  │  Standard      5V            3A          15W                   │ │   │
│  │  PD 2.0        5-20V         5A          100W                  │ │   │
│  │  PD 3.0        5-20V         5A          100W                  │ │   │
│  │  PD 3.1        5-48V         5A          240W                  │ │   │
│  │                                                             │ │   │
│  │  PPS (Programmable Power Supply): 20mV 단계로 세밀 조정          │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux USB 장치 확인】                                             │
│  ──────────────────                                                │
│  // USB 장치 목록                                                    │
│  $ lsusb                                                            │
│  Bus 002 Device 003: ID 8087:0024 Intel Corp.                       │
│  Bus 001 Device 004: ID 046d:c52b Logitech, Inc.                    │
│  Bus 001 Device 006: ID 1058:25a2 Western Digital                   │
│                                                                     │
│  // 상세 정보                                                        │
│  $ lsusb -v -d 046d:c52b                                            │
│                                                                     │
│  // 커널 메시지 모니터링                                              │
│  $ dmesg -w | grep usb                                              │
│  usb 1-1: new high-speed USB device number 3 using xhci_hcd         │
│  usb 1-1: New USB device found, idVendor=046d, idProduct=c52b       │
│                                                                     │
│  【USB 장치 정보】                                                   │
│  ──────────────────                                                │
│  // sysfs에서 장치 정보                                                │
│  $ ls /sys/bus/usb/devices/                                        │
│  1-1  1-1:1.0  usb1  usb2                                          │
│                                                                     │
│  $ cat /sys/bus/usb/devices/1-1/idVendor                           │
│  046d                                                               │
│  $ cat /sys/bus/usb/devices/1-1/idProduct                          │
│  c52b                                                               │
│  $ cat /sys/bus/usb/devices/1-1/speed                              │
│ 480  // USB 2.0 High Speed                                          │
│                                                                     │
│  【USB 장치 마운트】                                                 │
│  ──────────────────                                                │
│  // 장치 확인                                                        │
│  $ lsblk                                                            │
│  sdb      8:16   0   14.9G  0                                       │
│  └─sdb1   8:17   0   14.9G  0                                       │
│                                                                     │
│  // 마운트                                                           │
│  $ sudo mount /dev/sdb1 /mnt/usb                                   │
│                                                                     │
│  // 자동 마운트 (udev 규칙)                                          │
│  /etc/udev/rules.d/99-usb.rules:                                   │
│  ACTION=="add", KERNEL=="sd[a-z][0-9]", SUBSYSTEMS=="usb", \        │
│      RUN+="/usr/local/bin/usb-mount.sh"                             │
│                                                                     │
│  【USB 권한 설정 (udev)】                                             │
│  ──────────────────                                                │
│  // 특정 장치 권한                                                    │
│  /etc/udev/rules.d/99-mydevice.rules:                              │
│  SUBSYSTEM=="usb", ATTR{idVendor}=="046d", \                        │
│      ATTR{idProduct}=="c52b", MODE="0666"                           │
│                                                                     │
│  // 규칙 적용                                                        │
│  $ sudo udevadm control --reload-rules                             │
│  $ sudo udevadm trigger                                            │
│                                                                     │
│  【USB CDC ACM (시리얼)】                                             │
│  ──────────────────                                                │
│  // 아두이노/시리얼 변환기                                            │
│  $ ls /dev/ttyACM*                                                  │
│  /dev/ttyACM0                                                       │
│                                                                     │
│  // screen으로 연결                                                  │
│  $ screen /dev/ttyACM0 115200                                      │
│                                                                     │
│  // cu 명령어                                                        │
│  $ cu -l /dev/ttyACM0 -s 115200                                    │
│                                                                     │
│  【USB 네트워킹】                                                     │
│  ──────────────────                                                │
│  // USB 이더넷 (cdc_ether)                                          │
│  $ ip link show usb0                                                │
│  usb0: <BROADCAST,MULTICAST> mtu 1500                              │
│                                                                     │
│  // 안드로이드 USB 테더링                                             │
│  $ ip addr show usb0                                                │
│  inet 192.168.42.2/24 scope global usb0                            │
│                                                                     │
│  【USB 장치 디버깅】                                                  │
│  ──────────────────                                                │
│  // usbmon (USB 모니터링)                                            │
│  $ sudo modprobe usbmon                                            │
│  $ sudo cat /sys/kernel/debug/usb/usbmon/1u                        │
│                                                                     │
│  // wireshark로 USB 패킷 캡처                                        │
│  $ sudo wireshark -k -i usbmon1                                    │
│                                                                     │
│  【USB 문제 해결】                                                    │
│  ──────────────────                                                │
│  // 장치 리셋                                                        │
│  $ usbreset "Logitech USB Receiver"                                │
│  $ usbreset /dev/bus/usb/001/004                                   │
│                                                                     │
│  // 자동 서스펜드 비활성화                                            │
│  $ echo -1 > /sys/module/usbcore/parameters/autosuspend            │
│                                                                     │
│  // 특정 장치 서스펜드 비활성화                                        │
│  $ echo on > /sys/bus/usb/devices/1-1/power/control               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 범용 직렬 버스 인터페이스
• 버전: 1.0(1.5Mbps) → 4.0(40Gbps)
• 토폴로지: 호스트-허브-장치, 최대 127개 장치
• 전송 타입: Control, Interrupt, Bulk, Isochronous
• 커넥터: Type-A, Type-B, Micro-B, Type-C
• 클래스: HID, Mass Storage, CDC, Audio, Video
• 전력: USB PD (최대 240W)
• Type-C: 양방향, 전력 공급, 대체 모드
• Linux: lsusb, usbmon, udev
```
