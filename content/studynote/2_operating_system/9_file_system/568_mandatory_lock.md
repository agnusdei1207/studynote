# 568. 핫 플러그 (Hot Plug)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 시스템 실행 중 장치 추가/제거
> 2. **가치**: 서비스 중단 없이 장치 교체
> 3. **융합**: udev, ACPI, PCIe와 연관

---

## Ⅰ. 개요

### 개념 정의
**핫 플러그(Hot Plug)**는 **시스템 전원이 켜진 상태에서 장치를 추가/제거하는 기술**입니다.

### 💡 비유: 주행 중 타이어 교체
핫 플러그는 **주행 중 타이어 교체**와 같습니다. 멈추지 않고 교체합니다.

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【USB 핫 플러그】                                                    │
│  ──────────────────                                                │
│  // 이벤트 모니터링                                                    │
│  $ sudo udevadm monitor --property                                 │
│  KERNEL[1234567.890] add /devices/pci0000:00/.../usb1/1-1         │
│    DEVTYPE=usb_device                                              │
│    ID_VENDOR=Logitech                                              │
│    ID_MODEL=USB_Receiver                                           │
│                                                                     │
│  // 장치 노드 생성 확인                                               │
│  $ ls -la /dev/bus/usb/001/                                        │
│  crw-rw-r-- 1 root root 189, 5 Jan 1 12:00 006                     │
│                                                                     │
│  【PCIe 핫 플러그】                                                   │
│  ──────────────────                                                │
│  // PCIe 핫 플러그 슬롯 확인                                          │
│  $ ls /sys/bus/pci/slots/                                          │
│  1  2  3  4                                                         │
│                                                                     │
│  // 슬롯 상태                                                         │
│  $ cat /sys/bus/pci/slots/1/attention                              │
│ 0                                                                   │
│                                                                     │
│  // 카드 제거 (소프트웨어)                                            │
│  $ echo 0 > /sys/bus/pci/slots/1/power                             │
│                                                                     │
│  // 카드 장착 (소프트웨어)                                            │
│  $ echo 1 > /sys/bus/pci/slots/1/power                             │
│                                                                     │
│  // 버스 재스캔                                                       │
│  $ echo 1 > /sys/bus/pci/rescan                                    │
│                                                                     │
│  【SATA 핫 스왑】                                                      │
│  ──────────────────                                                │
│  // 핫 스왑 지원 확인                                                  │
│  $ cat /sys/class/scsi_host/host0/ahci_hotplug                    │
│  1                                                                   │
│                                                                     │
│  // 안전 제거                                                         │
│  $ echo 1 > /sys/block/sdb/device/delete                          │
│                                                                     │
│  // 새 장치 스캔                                                       │
│  $ echo "- - -" > /sys/class/scsi_host/host0/scan                 │
│                                                                     │
│  // udisksctl                                                        │
│  $ udisksctl power-off -b /dev/sdb                                 │
│                                                                     │
│  【CPU 핫 플러그】                                                      │ |
│  ──────────────────                                                │
│  // CPU 온라인 상태                                                   │
│  $ cat /sys/devices/system/cpu/cpu*/online                         │
│  1                                                                   │
│  1                                                                   │
│                                                                     │
│  // CPU 오프라인                                                      │
│  $ echo 0 > /sys/devices/system/cpu/cpu3/online                    │
│                                                                     │
│  // CPU 온라인                                                        │
│  $ echo 1 > /sys/devices/system/cpu/cpu3/online                    │
│                                                                     │
│  // lscpu로 확인                                                      │
│  $ lscpu | grep "CPU(s)"                                            │
│  CPU(s):                4                                           │
│  On-line CPU(s) list:   0-2                                        │
│  Off-line CPU(s) list:  3                                          │
│                                                                     │
│  【컨테이너/가상화 핫 플러그】                                          │ |
│  ──────────────────                                                │
│  // virtio 장치 핫 플러그                                             │
│  $ echo 1 > /sys/bus/virtio/drivers/virtio_balloon/bind           │
│                                                                     │
│  // USB 패스스루 (VM)                                                │
│  $ virsh attach-device vm1 usb.xml                                 │
│  $ virsh detach-device vm1 usb.xml                                 │
│                                                                     │
│  // PCI 패스스루                                                      │
│  $ virsh nodedev-detach pci_0000_01_00_0                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 시스템 실행 중 장치 추가/제거
• 지원: USB, SATA, PCIe, NVMe, CPU, Memory
• 과정: 감지 → 스캔 → 할당 → 바인딩 → 초기화
• Linux: udev, uevent, sysfs
• PCIe: Presence Detect, Power Control
• CPU: /sys/devices/system/cpu/cpu*/online
• Memory: /sys/devices/system/memory/
• SATA: 안전 제거 후 스캔
• 장점: 서비스 중단 없이 유지보수
• 용도: 서버, 가상화, 모바일
```
