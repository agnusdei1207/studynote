# 380. 데이터 이관(Migration) 및 검증 - 끊김 없는 데이터 이동

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 이관은 서비스 운영 중 또는 시스템 교체 시, **소스 DB의 데이터를 타겟 DB로 정해진 절차에 따라 정확하고 안전하게 옮기는 일련의 과정**이다.
> 2. **가치**: 데이터 유실이나 변조를 원천 차단하는 **정밀한 검증(Verification)** 절차를 통해 비즈니스 연속성을 보장하며, 마이그레이션 중 발생하는 서비스 중단 시간(Downtime)을 최소화한다.
> 3. **융합**: ETL/ELT 기술, CDC(실시간 복제) 기술, 그리고 데이터 품질 관리 기법이 융합되어 대규모 엔터프라이즈 시스템의 현대화를 지원하는 핵심 공정이다.

+++

### Ⅰ. 데이터 이관의 4대 핵심 단계

1. **분석 및 설계**: 소스-타겟 간 스키마 매핑, 이관 대상 및 범위 확정, 이관 전략(Big-bang vs Phased) 수립.
2. **추출 및 변환 (Extraction & Transformation)**: 소스에서 데이터를 뽑아내어 타겟 형식에 맞게 클렌징하고 변환.
3. **적재 (Loading)**: 타겟 DB에 데이터 주입. 성능을 위해 인덱스와 제약 조건을 일시 해제 후 재구축.
4. **검증 및 안정화**: 이관된 데이터의 정합성을 100% 확인하고 업무 영향도를 최종 체크.

+++

### Ⅱ. 이관 및 검증 프로세스 시각화 (ASCII Flow)

```text
[ Data Migration Pipeline ]

  (Source DB) ──▶ [ Extraction ] ──▶ [ Staging / Transform ]
                         │                   │
                         ▼                   ▼
                  [ Target DB ] ◀──── [ Loading ]
                         │
          ┌──────────────┴──────────────┐
          ▼                             ▼
  [ 1. Count Check ]            [ 2. Hash Check ] ✅
  (건수 일치 확인)               (데이터 내용 일치 확인)
          │                             │
          └──────────────┬──────────────┘
                         ▼
                [ Final Switch-over ] (서비스 전환)
```

+++

### Ⅲ. 치명적인 이관 검증 기법

- **건수 검증**: 소스와 타겟의 테이블별 레코드 수가 정확히 일치하는지 확인.
- **합계 검증**: 금액, 수량 등 주요 수치형 컬럼의 총합(Sum) 비교.
- **데이터 샘플링 검증**: 무작위로 추출한 특정 레코드들의 모든 컬럼 값을 1:1로 정밀 대조.
- **양방향 비교**: 이관 완료 후 일정 기간 소스와 타겟에 동시에 데이터를 쓰고 결과를 비교하는 '병행 운용'.

- **📢 섹션 요약 비유**: 데이터 이관은 **'심장 이식 수술'**과 같습니다. 환자(서비스)가 죽지 않도록 인공심폐기(CDC)를 달아 피(데이터)를 돌리면서, 새로운 심장(타겟 DB)으로 교체해야 합니다. 수술 후에는 모든 혈관이 잘 연결되었는지(정합성 검증) 확인하고, 환자가 스스로 숨을 쉴 수 있을 때 비로소 수술을 마칩니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[CDC]**: 실시간 이관을 가능케 하는 기술 (Change Data Capture).
- **[Fallback]**: 이관 실패 시 원래대로 되돌리는 비상 계획.
- **[Schema Mapping]**: 소스-타겟 간의 데이터 지도.

📢 **마무리 요약**: **Data Migration**은 기술력과 치밀함의 집약체입니다. 완벽한 **검증 절차**만이 데이터 이동의 리스크를 제거하고 시스템의 새로운 출발을 축복할 수 있습니다.
