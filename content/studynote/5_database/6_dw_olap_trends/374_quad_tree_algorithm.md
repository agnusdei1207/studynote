# 374. Quad-tree 알고리즘 - 공간을 4등분하는 지혜

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 쿼드트리(Quad-tree)는 2차원 평면을 **4개의 구역(Quadrant)으로 재귀적으로 분할하여 데이터를 조직화**하는 트리 기반의 공간 인덱스 자료구조다.
> 2. **가치**: 특정 구역에 데이터가 집중되면 해당 구역만 다시 4등분하여 세분화함으로써, **불균형한 공간 데이터 분포 속에서도 효율적인 탐색 성능**을 유지한다.
> 3. **융합**: 이미지 압축, 지리정보시스템(GIS)의 충돌 감지, 게임 엔진의 가시성 판단 기술과 융합되어 공간 데이터 처리의 핵심 알고리즘으로 사용된다.

+++

### Ⅰ. 쿼드트리의 분할 원리

- **재귀적 분할**: 하나의 노드는 최대 4개의 자식 노드(NW, NE, SW, SE)를 가집니다.
- **분할 기준**: 특정 구역 내의 객체 수가 임계치(Threshold)를 넘으면 해당 사각형 공간을 십자(+) 모양으로 4등분합니다.
- **정지 조건**: 구역 내 객체가 적거나, 최소 분할 크기(Depth)에 도달하면 분할을 멈춥니다.

+++

### Ⅱ. 쿼드트리 공간 분할 시각화 (ASCII Model)

```text
[ Quad-tree Spatial Decomposition ]

  (Level 0: Root)        (Level 1: Split)       (Level 2: Detail)
  ┌───────┬───────┐      ┌───┬───┬───────┐      ┌───┬───┬───┬───┐
  │       │       │      │NW │NE │       │      │   │   │   │   │
  │       │       │      ├───┼───┤       │      ├───┼───┼───┼───┐
  │       │       │      │SW │SE │       │      │   │   │   │   │
  ├───────┼───────┤      ├───┴───┼───────┤      ├───┼───┼───┼───┐
  │       │       │      │       │       │      │   │   │   │   │
  │       │       │      │       │       │      ├───┼───┼───┼───┐
  │       │       │      │       │       │      │   │   │   │   │
  └───────┴───────┘      └───────┴───────┘      └───┴───┴───┴───┘
```

### Ⅲ. 쿼드트리 vs R-Tree

| 비교 항목 | Quad-tree (쿼드트리) | R-Tree |
|:---|:---|:---|
| **분할 방식** | **정적인 공간 분할** (항상 4등분) | 데이터 분포에 따른 가변 분할 |
| **객체 겹침** | MBR 개념 없음 (공간 중심) | **MBR 사용** (객체 중심) |
| **차원 확장** | 3차원은 Octree로 확장 | 다차원 확장 용이 |
| **주요 강점** | 구현이 단순하고 속도가 빠름 | 검색 효율과 정확도가 높음 |

- **📢 섹션 요약 비유**: 쿼드트리는 **'지도의 줌인(Zoom-in) 기능'**과 같습니다. 멀리서 볼 때는 큰 사각형 하나로 보이지만, 사람들이 많이 모여있는 도심지(데이터 집중 구역)를 자세히 보려고 하면 지도가 자동으로 4개의 작은 지도로 쪼개지며 아주 상세한 골목길까지 보여주는 방식입니다. 한산한 시골은 굳이 쪼개지 않고 큰 지도로 놔두는 영리한 지도 제작법입니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Octree]**: 쿼드트리를 3차원 공간(8등분)으로 확장한 자료구조.
- **[Collision Detection]**: 게임 등에서 물체끼리 부딪혔는지 확인할 때 탐색 범위를 줄이는 용도.
- **[Z-Order Curve]**: 2차원 공간을 1차원 선으로 변환하여 쿼드트리 효율을 높이는 기술.

📢 **마무리 요약**: **Quad-tree**는 평면을 지배하는 논리입니다. 단순한 4분할의 반복을 통해 복잡한 세상을 체계적으로 담아내는 공간 컴퓨팅의 고전이자 필독 알고리즘입니다.
