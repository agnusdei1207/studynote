# 354. 데이터 가상화 (Data Virtualization) - 물리적 경계를 허물다

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 가상화는 서로 다른 위치와 형식을 가진 데이터 원천들을 물리적으로 복제하거나 이동하지 않고, **논리적인 가상 계층을 통해 마치 하나의 데이터베이스인 것처럼 통합 조회할 수 있게 하는 기술**이다.
> 2. **가치**: 데이터 웨어하우스(DW) 구축에 필요한 고비용의 **ETL 과정을 획기적으로 줄이며**, 소스 시스템의 최신 데이터를 실시간으로 즉시 조회할 수 있는 '제로 지연(Zero Latency)' 통합을 실현한다.
> 3. **융합**: 클라우드, 온프레미스, SaaS 등 하이브리드 인프라 환경에서 흩어진 데이터를 연결하는 '논리적 데이터 웨어하우스' 아키텍처의 핵심 엔진으로 융합된다.

+++

### Ⅰ. 데이터 가상화의 작동 원리

1. **연결 (Connect)**: 다양한 데이터 소스(SQL, NoSQL, API, File)에 어댑터를 통해 연결합니다.
2. **추상화 (Abstract)**: 소스별 고유 문법을 무시하고 공통된 메타데이터 모델로 변환합니다.
3. **통합 (Combine)**: 여러 소스의 데이터를 논리적으로 조인(Join)하여 가상 뷰(Virtual View)를 만듭니다.
4. **전달 (Deliver)**: 사용자가 SQL이나 API로 가상 뷰를 조회하면, 엔진이 실시간으로 소스 DB에 질의하여 결과를 취합해 반환합니다.

+++

### Ⅱ. 가상화 아키텍처 시각화 (ASCII Flow)

```text
[ Data Virtualization Architecture ]

      (User / BI Tool) : "SELECT * FROM Integrated_Sales"
                │
      ┌─────────▼─────────┐
      │ Virtualization    │ (Query Optimizer & Executor)
      │ Layer (Logical)   │ ──▶ [ Global Metadata ]
      └─────────┬─────────┘
                │ (Push-down Queries)
      ┌─────────┴─────────┬──────────────┐
      ▼                   ▼              ▼
  [ Oracle ]          [ Hadoop ]     [ Salesforce ]
 (Sales Table)       (Log Data)     (Customer API)
```

### Ⅲ. 가상화의 장점과 한계

- **장점**: 
    - **속도**: 인프라 구축 없이 며칠 내에 데이터 통합 완료 가능.
    - **비용**: 복제본을 위한 추가 저장 공간이나 관리 비용 절감.
    - **민첩성**: 원천 시스템의 변화에 즉각 대응 가능.
- **한계**: 
    - **성능 부하**: 실시간으로 원천 DB를 직접 찌르기 때문에 소스 시스템 성능에 영향을 줄 수 있음.
    - **네트워크 의존성**: 대량 데이터 조인 시 네트워크 지연이 병목이 됨.

- **📢 섹션 요약 비유**: 데이터 가상화는 **'전 세계 도서관을 연결한 통합 온라인 검색대'**와 같습니다. 책을 우리 집으로 다 사 올(ETL) 필요 없이, 검색창에 입력만 하면 각 나라 도서관에 있는 책 내용을 실시간으로 화면에 띄워주는 것과 같습니다. 책은 제자리에 있지만, 나는 내 방에서 전 세계의 지식을 한꺼번에 읽는 셈입니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Federated Query]**: 가상화의 기술적 기초가 되는 분산 질의.
- **[Zero Copy]**: 데이터 복제 없이 통합을 달성하는 데이터 관리 철학.
- **[Denodo / Presto]**: 데이터 가상화를 구현하는 대표적인 솔루션 및 엔진.

📢 **마무리 요약**: **Data Virtualization**은 데이터 통합의 지름길입니다. 물리적 통합의 무거움을 벗어던지고, 논리적 연결을 통해 데이터 활용의 속도를 극대화하는 영리한 아키텍처입니다.
