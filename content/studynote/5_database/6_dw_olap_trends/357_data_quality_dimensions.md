# 357. 데이터 품질(Data Quality) 6대 차원 - 정량적 평가의 기준

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 품질 6대 차원은 데이터가 비즈니스 목적에 얼마나 부합하는지를 객관적으로 측정하기 위해 정의된 **정확성, 완전성, 일관성, 최신성, 유효성, 무결성**의 평가 기준이다.
> 2. **가치**: "우리 데이터는 믿을만한가?"에 대한 모호한 판단을 정량적 수치(지표)로 변환하여, 지속적인 데이터 정화(Cleansing) 및 관리 활동의 근거를 제공한다.
> 3. **융합**: 데이터 거버넌스 체계와 결합되어 기업의 전사적 데이터 자산 가치를 유지하며, 고도화된 AI 모델 학습 데이터의 신뢰성을 담보하는 핵심 잣대가 된다.

+++

### Ⅰ. 데이터 품질의 6대 핵심 차원 (DAMA 기준 등)

| 차원 (Dimension) | 정의 | 측정 예시 |
|:---|:---|:---|
| **1. 정확성 (Accuracy)** | 실제 세계의 값과 일치하는 정도 | 고객 주소가 실제 배송지와 일치하는가? |
| **2. 완전성 (Completeness)** | 필수 항목에 누락이 없는 상태 | 고객 정보 중 전화번호가 비어있지 않은가? |
| **3. 일관성 (Consistency)** | 서로 다른 시스템 간 데이터 불일치 부재 | ERP와 CRM의 고객 등급이 동일한가? |
| **4. 최신성 (Timeliness)** | 필요한 시점에 데이터가 공급되는 정도 | 어제 매출이 오늘 아침 보고서에 반영됐는가? |
| **5. 유효성 (Validity)** | 정해진 형식이나 범위 준수 여부 | 날짜 필드에 '9999-99-99' 같은 값은 없는가? |
| **6. 무결성 (Integrity)** | 데이터 간의 관계 및 제약 조건 준수 | 주문 내역이 있는데 주문 고객 정보가 없는가? |

+++

### Ⅱ. 품질 측정 및 관리 구조 (ASCII Model)

```text
[ Data Quality Management Cycle ]

  ┌─────────────────────────────────────────┐
  │  1. 품질 진단 (Profiling & Assessment)   │ ──▶ (얼마나 나쁜가?)
  ├─────────────────────────────────────────┤
  │  2. 규칙 수립 (Quality Rule Design)      │ ──▶ (어떻게 측정할까?)
  ├─────────────────────────────────────────┤
  │  3. 데이터 정제 (Data Cleansing)         │ ──▶ (틀린 걸 고치자!)
  ├─────────────────────────────────────────┤
  │  4. 상시 모니터링 (Continuous Monitor)   │ ──▶ (계속 깨끗한가?)
  └─────────────────────────────────────────┘
```

+++

### Ⅲ. 데이터 품질이 비즈니스에 미치는 영향

- **의사결정 오류**: 정확성이나 일관성이 낮으면 잘못된 데이터로 인해 경영진이 엉뚱한 전략을 수립하게 됩니다.
- **운영 효율 저하**: 완전성이 떨어지면 고객에게 전화를 걸거나 메일을 보낼 때 주소지를 다시 확인해야 하는 수고가 발생합니다.
- **AI 성능 저하**: 인공지능 모델은 학습 데이터의 품질(특히 정확성과 유효성)에 따라 그 성능이 결정됩니다 (GIGO).

- **📢 섹션 요약 비유**: 데이터 품질 6대 차원은 **'건강 검진 지표'**와 같습니다. 혈압(정확성), 맥박(최신성), 비만도(유효성) 등 여러 수치를 종합적으로 검사해야 몸 전체의 건강(데이터 신뢰도)을 알 수 있는 것과 같습니다. 어느 한 수치만 좋다고 해서 건강한 몸이라 할 수 없듯, 6대 차원이 골고루 갖춰져야 진짜 쓸모 있는 데이터가 됩니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[GIGO (Garbage In, Garbage Out)]**: 품질 관리의 중요성을 나타내는 대명사.
- **[Data Profiling]**: 데이터의 실제 값들을 통계적으로 분석하여 품질을 진단하는 기술.
- **[Golden Record]**: 중복과 오류를 제거하여 품질이 완벽해진 마스터 데이터.

📢 **마무리 요약**: **Data Quality Dimensions**는 데이터의 등급을 매기는 기준입니다. 양적인 팽창보다 질적인 정교함을 추구할 때, 데이터는 비로소 기업의 강력한 무기가 됩니다.
