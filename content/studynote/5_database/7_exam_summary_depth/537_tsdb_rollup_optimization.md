# 537. 시계열 DB의 다운샘플링과 롤업 - 분석 속도의 가속기

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 다운샘플링(Downsampling)과 롤업(Rollup)은 방대한 시계열 데이터를 시간 단위(분, 시, 일)로 묶어 **대표값(평균, 최대 등)으로 요약하여 저장함으로써 조회 범위를 축소하는 데이터 최적화 기술**이다.
> 2. **가치**: 1년치 초 단위 데이터를 읽는 대신 미리 계산된 '일 단위 롤업' 데이터를 읽음으로써 **쿼리 응답 속도를 수백 배 향상**시키고, 오래된 데이터의 정밀도를 낮춰 스토리지 비용을 절감한다.
> 3. **융합**: 연속 쿼리(Continuous Query) 기술과 융합되어, 데이터가 적재되는 즉시 백그라운드에서 자동으로 요약본을 생성하는 지능형 시계열 파이프라인을 형성한다.

+++

### Ⅰ. 다운샘플링(Downsampling)의 작동 원리

- **정의**: 고해상도(High Resolution) 데이터를 저해상도(Low Resolution) 데이터로 변환하는 과정.
- **예시**:
    - **Raw Data**: 1초마다 수집되는 온도 센서 값 (1일 86,400개).
    - **Downsampled**: 1시간당 평균/최대/최소 온도 (1일 24개). ✅ "데이터량 3,600배 압축"
- **연산 종류**: `AVG`, `SUM`, `MIN`, `MAX`, `FIRST`, `LAST`.

+++

### Ⅱ. 롤업 및 쿼리 효율화 시각화 (ASCII Model)

```text
[ Hierarchical Time Series Aggregation ]

  (Level 1: Raw)   [1s][1s][1s]...[1s]  ──▶ 💾 Massive Disk I/O
                          │
          ┌───────────────┴───────────────┐
          ▼ (Background Rollup Process)   ▼
  (Level 2: 1 Min) [ SUM / AVG / MAX ]  ──▶ 💾 Reduced I/O
                          │
          ┌───────────────┴───────────────┐
          ▼ (Daily Rollup)                ▼
  (Level 3: 1 Day) [ Final Summary ]    ──▶ ⚡️ Instant Query ✅

  * Query "Show me last year trend" ──▶ Level 3 읽기 (광속 응답)
```

+++

### Ⅲ. 실무적 활용: 대시보드 최적화

- **자동 쿼리 전환**: 사용자가 차트에서 '최근 1시간'을 보면 원본 데이터를, '최근 1년'을 보면 롤업 데이터를 자동으로 조회하도록 쿼리 엔진을 설계합니다.
- **보존 정책 연계**: 원본 데이터는 7일만 보관하고 삭제(Drop)하되, 롤업된 통계 데이터는 수년간 보존하여 장기 추세 분석을 지원합니다.
- **미리 계산된 뷰 (CQ)**: InfluxDB의 Continuous Query나 Prometheus의 Recording Rules 등을 활용하여 런타임 집계 부하를 0으로 만듭니다.

- **📢 섹션 요약 비유**: 시계열 롤업은 **'가계부 요약'**과 같습니다. 매일 쓴 커피값, 밥값 내역(Raw Data)을 모두 들여다보는 것은 최근 며칠간은 유용하지만, 10년 전의 소비 패턴을 분석할 때는 매일의 영수증을 다 뒤질 필요가 없습니다. 대신 '월별 총액'이나 '연도별 합계'로 미리 요약해 둔 장부(Rollup)만 보면, 10년치 경제 흐름을 단 1초 만에 파악할 수 있는 것과 같습니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Granularity]**: 데이터가 얼마나 촘촘한지를 나타내는 입도.
- **[Continuous Query]**: 실시간으로 롤업을 수행하는 자동화된 질의.
- **[Retention Policy]**: 롤업 단계에 따라 다르게 적용하는 데이터 보존 기간.

📢 **마무리 요약**: **Downsampling & Rollup**은 시계열 데이터의 해상도를 조절하는 렌즈입니다. 필요한 만큼만 보여주고 나머지는 과감히 요약할 때, 거대한 시간의 흐름을 가장 빠르게 통찰할 수 있습니다.
