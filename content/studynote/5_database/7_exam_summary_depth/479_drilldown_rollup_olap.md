# 479. 드릴 다운(Drill-down)과 롤업(Roll-up) - 분석의 깊이와 넓이

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 드릴 다운과 롤업은 다차원 데이터 분석(OLAP)에서 데이터의 **상세도(Granularity)를 조절하며 정보를 탐색하는 상호 보완적인 집계 연산**이다.
> 2. **가치**: 전체적인 트렌드(롤업)를 파악한 뒤 문제의 원인을 심층 분석(드릴 다운)할 수 있게 함으로써, 데이터 속에 숨겨진 상관관계와 이상 징후를 입체적으로 발견하게 돕는다.
> 3. **융합**: 계층 구조(Hierarchy)를 가진 차원 데이터와 결합되어, 사용자가 클릭 한 번으로 요약 보고서와 상세 내역을 오가는 대화형 비즈니스 인텔리전스(BI)를 구현한다.

+++

### Ⅰ. 드릴 다운 (Drill-down)

- **정의**: 요약된 데이터에서 하위의 구체적인 상세 데이터로 내려가는 과정입니다.
- **방향**: 상위 계층 ──▶ 하위 계층 (예: 연도별 매출 ──▶ 월별 매출 ──▶ 일별 매출).
- **효과**: "매출이 왜 줄었지?"라는 질문에 대해 특정 지역이나 특정 상품의 부진을 찾아내는 '원인 분석'에 탁월합니다.

+++

### Ⅱ. 롤업 (Roll-up / Drill-up)

- **정의**: 상세 데이터들을 결합하여 상위 레벨의 요약된 정보를 만드는 과정입니다.
- **방향**: 하위 계층 ──▶ 상위 계층 (예: 시 단위 매출 ──▶ 도 단위 매출 ──▶ 전국 매출).
- **효과**: 사소한 데이터의 변화(노이즈)를 걷어내고 비즈니스의 전반적인 '큰 흐름'을 파악하는 데 유리합니다.

+++

### Ⅲ. 계층적 분석 시각화 (ASCII Model)

```text
[ Data Exploration Hierarchy ]

      (Roll-up: Summarize) ▲
                           │
  ┌────────────────────────┴────────────────────────┐
  │  Level 1: Year (2026)                           │
  ├────────────────────────┬────────────────────────┤
  │  Level 2: Quarter (Q1) │  Level 2: Quarter (Q2) │
  ├───────────┬────────────┼───────────┬────────────┤
  │ Month (1) │ Month (2)  │ Month (3) │ ...        │
  └───────────┴────────────┴───────────┴────────────┘
                           │
      (Drill-down: Detailed) ▼
```

+++

### Ⅳ. 실무적 의의: BI 도구에서의 활용

- **대화식 분석**: 사용자가 차트의 특정 막대를 클릭하면 자동으로 드릴 다운된 결과가 나타나는 동적 UI의 핵심 로직입니다.
- **성능 고려**: 드릴 다운 시 실시간 집계 연산 부하가 커질 수 있으므로, 미리 요약 테이블(Summary Table)을 만들어 두는 최적화가 병행됩니다.

- **📢 섹션 요약 비유**: 드릴 다운과 롤업은 **'지도의 확대와 축소'**와 같습니다. 롤업은 지도를 축소하여 나라 전체의 지형(트렌드)을 보는 것이고, 드릴 다운은 돋보기를 들이대어 특정 동네의 골목길(상세 데이터)까지 줌인하여 들여다보는 것과 같습니다. 멀리서 숲을 보고, 가까이서 나무를 보는 분석의 기본기입니다.

+++

### Ⅴ. 개념 맵 및 요약

- **[Hierarchy]**: 드릴 다운/롤업이 가능하기 위한 차원의 계층 구조.
- **[Aggregation]**: 롤업 시 수행되는 합계, 평균 등의 연산.
- **[Granularity]**: 데이터가 얼마나 잘게 쪼개져 있는지를 나타내는 정도.

📢 **마무리 요약**: **Drill-down & Roll-up**은 데이터 분석가의 시력을 교정해 줍니다. 거시적 관점과 미시적 관점을 자유롭게 오갈 때, 데이터는 비로소 의미 있는 전략으로 바뀝니다.
