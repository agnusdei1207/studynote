# 524. EER 모델(Enhanced ER) - 계층과 상속의 데이터 모델링

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: EER 모델은 기존 ER 모델에 객체 지향 개념인 **일반화(Generalization), 특수화(Specialization), 상속(Inheritance)**을 추가하여 더욱 복잡하고 정교한 데이터 요구사항을 표현할 수 있게 확장한 모델이다.
> 2. **가치**: 공통 속성을 가진 슈퍼타입(Supertype)과 개별 특징을 가진 서브타입(Subtype)을 구분함으로써, 데이터의 중복을 줄이고 논리적 계층 구조를 명확히 시각화한다.
> 3. **융합**: 서브타입 간의 중복 허용 여부(Disjoint/Overlap)와 참여 제약 조건(Total/Partial) 기술이 융합되어, 현실 세계의 복잡한 개체 분류 체계를 데이터베이스 설계에 완벽히 반영한다.

+++

### Ⅰ. EER 모델의 핵심 개념

1. **특수화 (Specialization)**: 상위 개체(슈퍼타입)를 특정 기준에 따라 하위 개체(서브타입)들로 나누는 과정. (Top-down)
2. **일반화 (Generalization)**: 여러 하위 개체들의 공통점을 찾아 하나의 상위 개체로 묶는 과정. (Bottom-up)
3. **속성 상속 (Inheritance)**: 서브타입은 슈퍼타입의 모든 속성과 관계를 자동으로 물려받으며, 자신만의 고유 속성만 추가로 정의합니다.

+++

### Ⅱ. EER 계층 구조 시각화 (ASCII Model)

```text
[ EER Diagram: Employee hierarchy ]

          ┌──────────────────────────┐
          │    (Supertype) Person    │
          │  - ID, Name, BirthDate   │
          └─────────────┬────────────┘
                        │
          ┌─────────────┴─────────────┐
          ▼ (Specialization)          ▼
  ┌───────────────┐           ┌───────────────┐
  │ (Subtype) Std │           │ (Subtype) Emp │ ✅ "Inherit ID, Name"
  │ - Major, GPA  │           │ - Salary, Hire│ ✅ "Has unique data"
  └───────────────┘           └───────────────┘
```

+++

### Ⅲ. 서브타입 제약 조건 (Constraints)

- **상호 배제 제약 (Disjointness)**:
    - **Disjoint (d)**: 한 개체는 단 하나의 서브타입에만 속할 수 있음. (예: 남성 또는 여성)
    - **Overlap (o)**: 한 개체가 여러 서브타입에 동시에 속할 수 있음. (예: 학생이면서 동시에 직원)
- **참여 제약 (Completeness)**:
    - **Total (전체 참여)**: 슈퍼타입의 모든 개체는 반드시 하나 이상의 서브타입에 속해야 함.
    - **Partial (부분 참여)**: 어느 서브타입에도 속하지 않는 슈퍼타입 개체가 존재할 수 있음.

- **📢 섹션 요약 비유**: EER 모델은 **'생물 분류 도감'**과 같습니다. '동물(슈퍼타입)'이라는 공통의 특징(숨을 쉰다, 먹는다)을 가진 큰 틀 아래, '포유류', '조류', '파충류'라는 고유한 특징(서브타입)을 가진 그룹으로 나누어 정리하는 것입니다. 이렇게 정리하면 공통점은 한 번만 설명(상속)하면 되고, 차이점만 따로 기록하면 되니 도감(데이터 모델)이 아주 효율적이고 체계적이 됩니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[ISA Relationship]**: 서브타입과 슈퍼타입 간의 "is-a" 관계.
- **[Subtype Discriminator]**: 서브타입을 구분 짓는 기준이 되는 컬럼.
- **[Attribute Inheritance]**: 하위 개체가 상위의 유산을 물려받는 물리적 원리.

📢 **마무리 요약**: **Enhanced ER Model**은 복잡한 데이터 세상의 질서입니다. 계층과 상속의 논리를 통해 데이터의 의미를 깊고 넓게 확장하여 완벽한 설계의 기초를 닦습니다.
