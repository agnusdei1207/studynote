# 516. GNN 기반 추천 시스템 - 관계를 학습하는 인공지능

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: GNN(Graph Neural Network) 기반 추천 시스템은 사용자(User)와 상품(Item)을 노드로, 구매나 클릭 등의 활동을 엣지(Edge)로 모델링하여 **데이터 간의 복잡한 연결 맥락을 직접 학습하는 지능형 추천 기술**이다.
> 2. **가치**: 단순히 비슷한 사람을 찾는 고전적 방식(CF)을 넘어, "내 친구의 친구가 좋아하는 브랜드" 같은 **고차원적인 인접 관계의 특징을 벡터로 추출**하여 예측 정확도를 획기적으로 높인다.
> 3. **융합**: 그래프 데이터베이스(Neo4j 등)와 딥러닝 프레임워크(PyTorch Geometric 등)가 융합되어, 데이터의 기하학적 구조를 파악하고 실시간 개인화 추천을 수행한다.

+++

### Ⅰ. GNN 추천의 핵심: 메시지 패싱 (Message Passing)

- **작동 원리**: 각 노드(사용자)는 연결된 이웃 노드(상품, 친구)들로부터 특징 정보를 전달받아 자신의 임베딩 벡터를 업데이트합니다.
- **특징 추출**: 층(Layer)이 깊어질수록 더 멀리 떨어진 노드와의 관계 정보까지 나의 벡터에 담기게 됩니다 (Multi-hop interaction).

+++

### Ⅱ. GNN 추천 시스템 아키텍처 시각화 (ASCII Model)

```text
[ Graph Interaction & Recommendation ]

  (User 1) ──[:Bought]──▶ (Item A) ◀──[:Likes]── (User 2)
     │                                             │
  [:Friend]                                     [:Follows]
     │                                             │
     ▼                                             ▼
  (User 3) ──[:Views]──▶ (Item B) ──[:Category]──▶ (Node: Fashion)

  [ GNN Learning ]
  1. Aggregate neighbors' info ──▶ 2. Update Node Vector ──▶ 3. Predict Edge ✅
  "User 1과 User 2는 구조적으로 닮았으니 Item B를 추천하라!" ✅
```

+++

### Ⅲ. 기존 추천 방식(CF)과의 차이점

| 비교 항목 | 협업 필터링 (Collaborative Filtering) | GNN 기반 추천 시스템 |
|:---|:---|:---|
| **데이터 표현** | 사용자-아이템 행렬 (Matrix) | **사용자-아이템 그래프 (Graph)** |
| **관계 활용** | 직접적인 1단계 관계 중심 | **다단계(Multi-hop) 관계 맥락 학습** |
| **콜드 스타트** | 신규 데이터에 취약함 | **그래프 구조를 통해 어느 정도 완화** |
| **복잡도** | 상대적으로 낮음 | 매우 높음 (그래프 연산 부하) |

- **📢 섹션 요약 비유**: GNN 기반 추천은 **'발 넓은 마당발 친구의 조언'**과 같습니다. 일반 추천이 "너 어제 피자 먹었으니 오늘도 이 피자 먹어봐"라고 단순하게 말한다면, GNN은 "네 친구 철수가 자주 가는 피자집 사장님이 사실은 네가 좋아하는 패션 브랜드를 운영하신대. 너 그 브랜드 좋아할 것 같아"라고 복잡하게 얽힌 인맥과 관심사를 꿰뚫어 보고 추천해 주는 것과 같습니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Embedding]**: 그래프 구조를 수치로 표현한 결과물.
- **[Link Prediction]**: 사용자-아이템 사이에 존재하지 않는 연결(추천)을 예측하는 것.
- **[GraphSAGE / PinSage]**: 대규모 추천 시스템에 적용된 대표적 GNN 모델.

📢 **마무리 요약**: **GNN Recommendation**은 데이터의 '연결' 속에 숨겨진 의도를 읽어냅니다. 초연결 시대의 파편화된 데이터들을 하나의 지능형 망으로 엮어, 가장 인간다운 추천을 실현합니다.
