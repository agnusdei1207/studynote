# 436. 윈도우 함수(Window Function) - 행을 유지하는 집계의 마법

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 윈도우 함수는 행과 행 사이의 관계를 정의하여 **개별 행(Row)의 세부 정보를 유지하면서도, 특정 범위(Window) 내의 집계나 순위 연산을 수행**하는 고급 SQL 기능이다.
> 2. **가치**: 기존 `GROUP BY`가 여러 행을 하나로 합쳐버리는(Collapse) 것과 달리, 전체 행 수를 유지하며 현재 행의 값과 그룹의 통계값(합계, 순위 등)을 1:1로 비교 분석할 수 있게 한다.
> 3. **융합**: `OVER`, `PARTITION BY`, `ORDER BY` 구문이 융합되어 데이터를 논리적으로 그룹화하고 정렬함으로써, 이동 평균이나 누적 합계 등 복잡한 분석 쿼리를 단순화한다.

+++

### Ⅰ. 윈도우 함수의 기본 문법 구조

- **PARTITION BY**: 전체 결과 집합을 소그룹으로 나눕니다. (`GROUP BY`와 유사한 역할)
- **ORDER BY**: 윈도우 안에서 행의 순서를 정합니다. (순위 결정 시 필수)
- **WINDOWing Clause**: 현재 행을 기준으로 계산에 포함할 행의 범위를 더 세밀하게 지정합니다. (예: `ROWS BETWEEN 2 PRECEDING AND CURRENT ROW`)

+++

### Ⅱ. 윈도우 함수 작동 시각화 (ASCII Model)

```text
[ Window Function Operation ]

  (Data Set)
  ┌────┬────────┬────────┬─────────────────────────────┐
  │ ID │ Dept   │ Salary │ AVG(Sal) OVER(PARTITION BY Dept) │
  ├────┼────────┼────────┼─────────────────────────────┤
  │ 1  │ Sales  │ 500    │ 450 (Sales 부서 평균)        │ ✅ Individual Rows
  │ 2  │ Sales  │ 400    │ 450                         │ ✅ ARE MAINTAINED!
  ├────┼────────┼────────┼─────────────────────────────┤
  │ 3  │ IT     │ 800    │ 850 (IT 부서 평균)           │
  │ 4  │ IT     │ 900    │ 850                         │
  └────┴────────┴────────┴─────────────────────────────┘
```

+++

### Ⅲ. 주요 윈도우 함수 종류

1. **순위 함수**: `RANK()`, `DENSE_RANK()`, `ROW_NUMBER()`.
2. **집계 함수**: `SUM()`, `AVG()`, `MAX()`, `MIN()`, `COUNT()`.
3. **행 간격 함수**: 
    - `LAG()`: 현재 행보다 앞선 행의 데이터를 가져옴.
    - `LEAD()`: 현재 행보다 뒤에 있는 행의 데이터를 가져옴.

- **📢 섹션 요약 비유**: 윈도우 함수는 **'투명한 이동 돋보기'**와 같습니다. 데이터를 한 줄 한 줄 읽어나가면서(개별 행 유지), 돋보기를 통해 주변 행들의 상태(윈도우 범위)를 슥 훑어서 계산 결과를 현재 내 위치 옆에 메모해 두는 것과 같습니다. 돋보기 덕분에 내 자리를 떠나지 않고도 주변 상황과의 차이를 명확히 알 수 있습니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Analytical Function]**: 오라클 등에서 윈도우 함수를 부르는 용어.
- **[Partitioning]**: 계산의 단위를 쪼개는 행위.
- **[Grouping vs Windowing]**: 행이 합쳐지느냐(Grouping) 유지되느냐(Windowing)의 결정적 차이.

📢 **마무리 요약**: **Window Function**은 SQL의 한계를 넘어선 분석의 정수입니다. 데이터의 상세함과 통계적 통찰을 한 화면에 담아낼 수 있는 가장 현대적이고 강력한 도구입니다.
