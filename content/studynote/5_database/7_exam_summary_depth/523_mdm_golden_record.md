# 523. 마스터 데이터 관리(MDM) - 단일 진실의 황금 레코드

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: MDM(Master Data Management)은 기업 내 여러 시스템에 파편화되어 흩어져 있는 **핵심 비즈니스 개체(고객, 상품, 자산 등) 정보를 통합하여 단일한 표준 데이터(Golden Record)로 관리하는 체계**다.
> 2. **가치**: 시스템마다 다른 고객 정보나 상품 코드로 인한 혼란을 제거하여 **전사적인 데이터 일관성**을 확보하고, 비즈니스 프로세스의 효율성과 분석 결과의 신뢰도를 극대화한다.
> 3. **융합**: 데이터 거버넌스 정책과 실시간 동기화 기술(CDC, API)이 융합되어, 데이터의 탄생부터 변경까지 전 생애주기에 걸쳐 최신성을 유지하는 기업의 데이터 허브 역할을 수행한다.

+++

### Ⅰ. 마스터 데이터의 정의

- **정의**: 비즈니스를 수행하는 데 반드시 필요한 **기준이 되는 데이터**입니다. (자주 변하지 않는 속성 중심)
- **종류**:
    - **Party**: 고객, 협력사, 직원.
    - **Thing**: 상품, 부품, 자산.
    - **Place**: 매장, 지점, 창고.
- **특징**: 트랜잭션 데이터(주문, 결제)의 근간이 되며, 기업 전체에서 공유됩니다.

+++

### Ⅱ. 골든 레코드(Golden Record) 생성 과정 (ASCII Model)

여러 소스에서 온 정보를 비교, 정제, 통합하여 '가장 믿을만한 하나의 데이터'를 만듭니다.

```text
[ MDM Consolidation Workflow ]

  (Source: CRM) [ 홍길동 | 010-111 | 서울 ] ──┐
                                            │
  (Source: ERP) [ 홍길동 | 010-222 | 경기 ] ──┼─▶ [ MDM Hub ]
                                            │   (Match & Merge)
  (Source: Web) [ 길동홍 | 010-111 | 서울 ] ──┘          │
                                                       ▼
                                            ┌──────────────────────┐
                                            │  GOLDEN RECORD (SSOT)│ ✅
                                            │ [ 홍길동 | 010-111 ] │
                                            └──────────┬───────────┘
                                                       │ (Distribute)
                                            ◀──────────┴──────────▶
                                            [ 다시 각 시스템으로 배포 ]
```

+++

### Ⅲ. MDM의 핵심 기술 요소

1. **식별 및 매칭 (Matching)**: 이름은 같지만 전화번호가 다른 경우, 혹은 오타가 있는 경우 동일인인지 판별하는 확률적 알고리즘.
2. **생존 규칙 (Survivorship)**: 여러 시스템 중 어떤 데이터가 가장 최신이고 신뢰할 수 있는지 정하는 규칙.
3. **데이터 동기화**: MDM에서 완성된 골든 레코드를 다시 운영 시스템들에 실시간으로 뿌려주는 배포 기술.

- **📢 섹션 요약 비유**: MDM은 **'기업의 주민등록 시스템'**과 같습니다. 동네마다(각 부서 DB) 부르는 이름이나 별명이 달라도, 국가(MDM)에서 정한 단 하나의 표준 주민등록 정보(골든 레코드)를 기준으로 모든 행정 서비스(비즈니스)를 처리하는 것과 같습니다. 이 기준이 명확해야 세금(정산)도 정확히 매기고 복지(마케팅)도 효율적으로 제공할 수 있는 것과 같습니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Golden Record]**: MDM이 도달하고자 하는 완벽한 데이터 한 행.
- **[Single Source of Truth (SSOT)]**: 전사에서 오직 하나만 존재하는 진실된 데이터 원천.
- **[Data Cleansing]**: 골든 레코드를 만들기 위해 선행되는 데이터 세척 작업.

📢 **마무리 요약**: **MDM**은 데이터 경영의 '기준점'입니다. 파편화된 정보를 정제된 지식으로 바꾸는 이 과정을 통해, 기업은 비로소 하나의 유기체처럼 일관되게 움직일 수 있습니다.
