# 596. 데이터 디스커버리와 카탈로그 - 정보의 마켓플레이스

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 디스커버리(Discovery)는 흩어진 데이터 자산을 자동으로 찾아내고 분석하는 과정이며, **데이터 카탈로그(Catalog)는 이를 체계적으로 분류하고 검색 가능하게 만든 통합 저장소**다.
> 2. **가치**: 사용자가 데이터의 위치와 의미를 스스로 파악할 수 있는 **셀프 서비스(Self-serve)** 환경을 제공하며, 전문가의 도움 없이도 분석에 필요한 최적의 데이터를 1초 내에 검색하게 한다.
> 3. **융합**: AI 기반의 자동 큐레이션, 데이터 리니지 시각화, 그리고 거버넌스 워크플로우가 융합되어 데이터 활용의 편의성과 보안성을 동시에 확보한다.

+++

### Ⅰ. 데이터 카탈로그의 주요 기능

- **데이터 검색 (Search)**: 구글처럼 키워드 검색을 통해 테이블, 컬럼, 보고서를 찾아냅니다.
- **데이터 큐레이션**: 데이터에 대한 설명, 비즈니스 용어, 샘플 데이터를 제공하여 이해를 돕습니다.
- **신뢰 지표 제공**: 데이터의 품질 점수, 인기척(조회수), 리뷰 등을 통해 믿을만한 데이터인지 보여줍니다. ✅
- **접근 권한 워크플로우**: 카탈로그에서 바로 데이터 사용 권한을 요청하고 소유자(Owner)의 승인을 받는 절차를 자동화합니다.

### Ⅲ. 실무적 의의: 데이터 민주화 (Democratization)

- **검색 비용 절감**: 분석가가 데이터를 찾는 데 소요되는 시간(데이터 워크로드의 약 80%)을 획기적으로 줄여줍니다.
- **지식 자산화**: 퇴사한 담당자의 머릿속에 있던 데이터 지식을 카탈로그에 명문화하여 영구적인 기업 자산으로 만듭니다.
- **거버넌스 연계**: 개인정보나 기밀 데이터가 포함된 자산을 자동으로 식별(Classification)하고 검색 결과에서 필터링하거나 보안 경고를 띄웁니다.

- **📢 섹션 요약 비유**: 데이터 카탈로그는 **'온라인 쇼핑몰의 카테고리와 리뷰'**와 같습니다. 거대한 물류 창고(데이터 인프라)에 어떤 물건이 있는지 일일이 돌아다닐 필요 없이, 쇼핑몰 앱(카탈로그)에서 검색하고 상품 상세 설명과 다른 사람들의 구매 후기(리뷰/평점)를 보고 장바구니에 담는(분석 활용) 것과 같습니다. 쇼핑이 쉬워지듯 데이터 활용도 즐거운 경험이 됩니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Active Metadata]**: 사용자의 활용 패턴을 분석하여 카탈로그를 스스로 업데이트하는 기술.
- **[Data Lineage]**: 카탈로그 내에서 데이터의 출처를 시각화해 보여주는 기능.
- **[Self-serve Data]**: 전문가 없이 누구나 데이터를 발견하고 사용할 수 있는 상태.

📢 **마무리 요약**: **Data Discovery & Catalog**는 데이터 지능형 기업의 입구입니다. 방대한 데이터의 홍수 속에서 길을 잃지 않고, 정보의 가치를 즉각적으로 실현하게 돕는 디지털 이정표입니다.
