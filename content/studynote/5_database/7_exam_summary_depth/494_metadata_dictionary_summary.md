# 494. 메타데이터 사전 통합 관리 - 데이터의 신분증

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 메타데이터 사전 통합 관리는 기업 내 흩어진 데이터의 **기술적 속성(타입, 길이)과 비즈니스 속성(정의, 담당자) 정보를 하나의 중앙 저장소에서 표준화하여 관리하는 체계**다.
> 2. **가치**: "데이터가 어디에 있으며 무엇을 의미하는가?"에 대한 즉각적인 답을 제공하여 **데이터 탐색(Discovery)** 시간을 줄이고, 부서 간 용어 불일치로 인한 소통 오류를 해결한다.
> 3. **융합**: 데이터 카탈로그(Data Catalog) 기술과 융합되어 데이터 리니지 추적과 품질 관리의 핵심 지식 베이스(Knowledge Base) 역할을 수행한다.

+++

### Ⅰ. 메타데이터 사전의 관리 항목

1. **기술적 메타데이터 (Technical)**:
    - DB명, 테이블명, 컬럼명, 데이터 타입, 인덱스 정보, 생성일 등.
2. **비즈니스 메타데이터 (Business)**:
    - 데이터 정의, 업무 설명, 데이터 소유자(Owner), 보안 등급, 비즈니스 용어집(Glossary).
3. **운영 메타데이터 (Operational)**:
    - ETL 실행 이력, 데이터 크기 변화 추이, 접근 로그, 백업 주기.

+++

### Ⅱ. 메타데이터 통합 관리 아키텍처 (ASCII Model)

파편화된 원천 정보를 하나로 수집하여 서비스합니다.

```text
[ Centralized Metadata Repository ]

  (Source A) ──▶ [ Harvesting ] ──┐
  (Source B) ──▶ [ Extraction ] ──┼─▶ [ Metadata Registry ]
  (Source C) ──▶ [ API Integration ] ┘   (Standardized Store) ✅
                                               │
          ┌────────────────────────────────────┼────────────────────────────────┐
          ▼                                    ▼                                ▼
  [ Data Catalog ]                     [ Data Lineage ]                 [ Quality Tool ]
  (Search & Found)                     (Where it came from)             (Is it correct?)
```

+++

### Ⅲ. 통합 관리의 이점

- **데이터 가시성**: 수천 개의 테이블 속에서 내가 필요한 데이터를 1초 만에 검색하여 찾아낼 수 있습니다.
- **영향도 분석**: 특정 컬럼의 형식을 바꿀 때, 이와 연결된 하위 리포트나 API가 무엇인지 즉시 파악하여 장애를 예방합니다.
- **용어 표준화**: '고객번호', 'CUST_ID', '회원코드' 등으로 제각각 불리던 명칭을 전사 표준으로 통일합니다.

- **📢 섹션 요약 비유**: 메타데이터 사전은 **'도서관의 통합 검색 시스템'**과 같습니다. 수만 권의 책(데이터)이 꽂혀 있는 거대한 도서관에서 책의 제목, 저자, 장르, 위치(메타데이터)가 적힌 컴퓨터 시스템이 없다면 책 한 권을 찾는 데 하루가 꼬박 걸릴 것입니다. 메타데이터 사전은 데이터라는 미로 속에서 원하는 보물을 찾아주는 친절한 가이드북입니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Information Schema]**: DB 엔진이 제공하는 기본 메타데이터 테이블.
- **[Business Glossary]**: 현업 사용자를 위한 데이터 용어 해설집.
- **[Data Discovery]**: 메타데이터 관리를 통해 얻게 되는 데이터 탐색 능력.

📢 **마무리 요약**: **Metadata Management**는 데이터 거버넌스의 출발점입니다. 데이터의 신분증을 확실히 관리할 때, 비로소 데이터는 누구나 안심하고 쓸 수 있는 기업의 공용 자산이 됩니다.
