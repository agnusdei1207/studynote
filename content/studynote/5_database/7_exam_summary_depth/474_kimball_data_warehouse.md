# 474. 킴벌(Kimball)식 데이터 마트 - 상향식 분석의 실천

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 랄프 킴벌(Ralph Kimball)의 데이터 웨어하우스 아키텍처는 부서별 또는 프로세스별 **데이터 마트(Data Mart)를 먼저 구축한 뒤, 이를 결합하여 전사적 DW를 완성하는 'Bottom-up(상향식)' 접근 방식**이다.
> 2. **가치**: **다차원 모델링(스타 스키마)**을 적극 활용하여 비즈니스 사용자의 분석 편의성을 극대화하며, 초기 구축 비용이 적고 결과물을 빠르게 확인할 수 있는 높은 민첩성을 제공한다.
> 3. **융합**: '데이터 버스(Data Bus)' 아키텍처와 **공통 차원(Conformed Dimension)** 기술이 융합되어, 분산 구축된 마트들 사이의 데이터 일관성을 논리적으로 연결한다.

+++

### Ⅰ. 킴벌(Kimball) 아키텍처의 핵심 특징

- **사용자 중심**: 현업 사용자가 이해하기 쉬운 비즈니스 관점의 모델링을 지향합니다.
- **스타 스키마 (Star Schema)**: 조인을 최소화하고 성능을 높이기 위해 비정규화된 다차원 모델을 사용합니다.
- **상향식 통합**: 전사 통합보다는 당장 필요한 부서의 분석 니즈 해결에 집중하여 프로젝트를 시작합니다.

+++

### Ⅱ. 킴벌 아키텍처 시각화 (ASCII Flow)

```text
[ Kimball Bottom-up Architecture ]

  (Source Systems)       (Data Bus - Conformed Dimensions)
  ┌────────────┐       ┌─────────────────────────────────┐
  │  ERP / CRM │ ──▶   │  [ Time ]  [ Customer ]  [ Product ] │ ✅ (Shared)
  ├────────────┤       └────────────────┬────────────────┘
  │  Web Logs  │ ──▶                    │ (Integration)
  └────────────┘       ┌────────────────┴────────────────┐
                       ▼                                 ▼
              [ Marketing Mart ]                 [ Finance Mart ]
              (Star Schema)                      (Star Schema)
              - Sales Fact                       - Budget Fact
```

+++

### Ⅲ. 장단점 및 실무 적용

- **장점**: 
    - 투자 대비 효과(ROI)가 매우 빠릅니다 (신속한 가치 증명).
    - 쿼리 성능이 우수하고 분석 도구(BI)와의 호환성이 뛰어납니다.
- **단점**: 
    - 공통 차원 관리가 실패할 경우, 부서마다 데이터 수치가 달라지는 불일치 위험이 큽니다.
    - 데이터 중복 저장으로 인한 스토리지 비용이 상대적으로 높습니다.

- **📢 섹션 요약 비유**: 킴벌 방식은 **'필요한 곳마다 우물을 파는 것'**과 같습니다. 마을 전체 상수도를 짓느라(Inmon) 기다리는 대신, 목마른 집(부서) 옆에 우물(데이터 마트)을 먼저 파서 바로 물을 마시는 실용적인 방식입니다. 나중에 이 우물들을 수로(Data Bus)로 잘 연결하기만 하면 훌륭한 급수 시스템(전사 DW)이 완성되는 원리입니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Conformed Dimension]**: 서로 다른 마트 간에 공유되는 '기준 정보' (킴벌 방식의 성패를 가름).
- **[Star Schema]**: 킴벌이 사랑한 최고의 분석용 데이터 모델.
- **[Bus Architecture]**: 분산된 마트들을 연결하는 논리적 공통 분모.

📢 **마무리 요약**: **Kimball's Approach**는 실전 분석의 바이블입니다. 비즈니스의 속도에 맞춰 유연하게 성장하며, 사용자에게 가장 직관적인 데이터 통찰을 제공하는 현대 분석 환경의 대세입니다.
