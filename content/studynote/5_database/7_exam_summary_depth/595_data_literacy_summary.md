# 595. 데이터 리터러시(Data Literacy) - 데이터와 소통하는 능력

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 리터러시는 목적에 맞는 데이터를 찾고, 분석하여 비판적으로 해석하며, 그 결과를 바탕으로 **비즈니스 의사결정에 활용하고 소통할 수 있는 전사적 역량**을 의미한다.
> 2. **가치**: 특정 전문가(데이터 사이언티스트)의 영역을 넘어 현업 실무자 모두가 데이터를 다룰 수 있게 함으로써 **데이터 기반 의사결정 문화(Data-driven Culture)**를 정착시킨다.
> 3. **융합**: 데이터 카탈로그, 시각화 도구(BI), 그리고 조직 내 교육 프로세스가 융합되어, 데이터가 단순한 숫자가 아닌 공용어로 기능하게 한다.

+++

### Ⅰ. 데이터 리터러시의 4대 핵심 구성 요소

1. **데이터 탐색 (Read)**: 어떤 데이터가 어디에 있는지 알고, 데이터의 구조와 의미를 이해하는 능력.
2. **데이터 작업 (Work)**: 분석 도구(SQL, Excel, Python 등)를 활용해 데이터를 추출하고 가공하는 능력.
3. **데이터 분석 (Analyze)**: 통계적 기법이나 차트를 통해 데이터 속의 패턴과 추세를 발견하는 능력.
4. **데이터 소통 (Argue)**: 분석된 결과를 시각화하여 타인에게 설득력 있게 전달하고 토론하는 능력. ✅

+++

### Ⅱ. 데이터 리터러시 확산 아키텍처 (ASCII Model)

단순 도구 보급을 넘어 문화적 확산이 필요합니다.

```text
[ Data-Driven Culture Framework ]

  (Strategy) ──▶ [ Executive Support ] (데이터 경영 선포)
                        │
          ┌─────────────┴─────────────┐
          ▼ (Capability Build)        ▼ (Infrastructure)
  ┌──────────────────┐        ┌──────────────────┐
  │ Data Education   │        │ Self-Serve BI    │ ✅
  │ (리터러시 교육)   │        │ (쉬운 분석 도구)  │
  └────────┬─────────┘        └────────┬─────────┘
           │                           │
           └────────────┬──────────────┘
                        ▼
            [ Data Democratization ] 🔓 ✅
            "누구나 데이터를 보고 말한다"
```

+++

### Ⅲ. 기업 내 확산 전략 및 고려사항

- **데이터 카탈로그 구축**: 어려운 DB 용어 대신 비즈니스 용어로 데이터를 검색할 수 있는 환경을 제공합니다.
- **분석의 민주화**: 코딩을 몰라도 드래그 앤 드롭으로 차트를 만들 수 있는 No-code 분석 툴을 보급합니다.
- **데이터 품질 신뢰**: 리터러시가 높아져도 데이터가 틀리면 의미가 없으므로, 강력한 데이터 거버넌스가 뒷받침되어야 합니다.

- **📢 섹션 요약 비유**: 데이터 리터러시는 **'외국어 구사 능력'**과 같습니다. 예전에는 소수의 통역사(데이터 전문가)만이 외국인(데이터)과 대화할 수 있었다면, 이제는 모든 직원이 영어를 배워 직접 소통하는 시대입니다. 영어를 잘할수록 더 넓은 세상(시장 기회)을 보고 더 정확한 정보(비즈니스 통찰)를 얻을 수 있는 것과 같습니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Self-serve BI]**: 리터러시를 실천하기 위한 기술적 도구 (Tableau, PowerBI 등).
- **[Critical Thinking]**: 데이터를 맹신하지 않고 오류 가능성을 의심하는 비판적 사고.
- **[Data Storytelling]**: 수치에 서사를 입혀 의미를 전달하는 소통 기술.

📢 **마무리 요약**: **Data Literacy**는 4차 산업혁명 시대의 새로운 문해력입니다. 기술보다 중요한 것은 '데이터로 대화하려는 의지'이며, 이 역량이 기업의 미래 경쟁력을 결정합니다.
