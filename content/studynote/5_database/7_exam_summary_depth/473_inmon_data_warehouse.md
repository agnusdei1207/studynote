# 473. 인먼(Inmon)식 데이터 웨어하우스 - 전사적 통합의 정석

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 빌 인먼(Bill Inmon)의 데이터 웨어하우스 아키텍처는 기업 전체의 데이터를 **제3정규형(3NF) 수준으로 중앙 집중화하여 통합 저장하는 'Top-down(하향식)' 접근 방식**이다.
> 2. **가치**: 데이터 중복을 최소화하고 전사적인 정합성을 보장하여 '단일 진실 공급원(Single Source of Truth)'을 구축하며, 장기적인 데이터 거버넌스와 유지보수성에 최적화되어 있다.
> 3. **융합**: 기업 정보 공장(CIF, Corporate Information Factory) 사상을 바탕으로, 원천 데이터 수집 후 전사 DW(EDW)를 먼저 짓고 필요한 부서별 데이터 마트(DM)를 파생시키는 구조로 융합된다.

+++

### Ⅰ. 인먼(Inmon) 아키텍처의 핵심 특징

- **중앙 집중형 (EDW)**: 모든 부서의 데이터를 한곳에 모으는 것을 최우선으로 합니다.
- **정규화된 모델 (3NF)**: 중복을 없애기 위해 관계형 모델링 원칙을 철저히 따릅니다.
- **데이터 마트의 종속성**: 데이터 마트는 반드시 중앙 EDW로부터 추출되어 생성되어야 합니다. (Data integrity 유지)

+++

### Ⅱ. 인먼 아키텍처 시각화 (ASCII Flow)

```text
[ Inmon Top-down Architecture ]

  (Source Systems)       (Integration)       (Unified Repository)      (Analysis)
  ┌────────────┐       ┌───────────┐       ┌─────────────────┐      ┌────────────┐
  │  ERP / CRM │ ──▶   │    ETL    │ ──▶   │  Corporate DW   │ ──▶  │ Data Marts │
  ├────────────┤       │ (Cleaning)│       │ (3NF Integrated)│      ├────────────┤
  │  Legacy DB │ ──▶   │ (Mapping) │       └────────┬────────┘      │ BI / Query │
  └────────────┘       └───────────┘                │               └────────────┘
                                            (Single Source of Truth) ✅
```

+++

### Ⅲ. 장단점 및 실무 적용

- **장점**: 
    - 전사 데이터의 일관성이 매우 높습니다 (숫자가 꼬이지 않음).
    - 중앙 제어가 용이하여 데이터 보안 및 거버넌스 수립에 유리합니다.
- **단점**: 
    - 초기 구축 비용이 비싸고 시간이 오래 걸립니다 (High Initial ROI Low).
    - 변화하는 비즈니스 요구사항에 유연하게 대처하기 어렵습니다.

- **📢 섹션 요약 비유**: 인먼 방식은 **'거대한 정부 중앙 도서관'**을 짓는 것과 같습니다. 나라 안의 모든 책(데이터)을 한곳에 모아 엄격한 분류 체계(정규화)에 따라 정리해 둡니다. 누군가 특정 분야의 책이 필요하면(데이터 마트), 중앙 도서관에서 복사본을 만들어 나누어 주는 방식입니다. 짓기는 힘들어도 한 번 지어놓으면 온 나라의 지식이 질서 정연하게 관리되는 장점이 있습니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[CIF (Corporate Information Factory)]**: 인먼이 제안한 데이터 처리 인프라 체계.
- **[3NF (3rd Normal Form)]**: 인먼식 DW의 표준 데이터 모델.
- **[Enterprise Data Warehouse (EDW)]**: 전사적 통합 저장소를 일컫는 말.

📢 **마무리 요약**: **Inmon's Approach**는 데이터 무결성의 끝판왕입니다. 기업의 근간이 되는 '기준 데이터'를 확립하여, 흔들리지 않는 데이터 경영의 토대를 만드는 가장 정석적인 방법입니다.
