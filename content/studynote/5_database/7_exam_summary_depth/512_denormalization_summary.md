# 512. 반정규화(Denormalization) - 성능과 무결성의 정교한 타협

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 반정규화는 정규화된 데이터 모델에서 **시스템의 성능 향상, 개발 및 운영의 단순화를 위해 의도적으로 중복, 통합, 분리를 수행하는 데이터 모델링 기법**이다.
> 2. **가치**: 고비용의 조인(Join) 연산을 제거하고 디스크 I/O를 줄여 조회 성능을 극대화하며, 읽기 비중이 압도적인 대규모 서비스 환경에서 필수적인 최적화 수단이 된다.
> 3. **융합**: 데이터 중복으로 인한 **정합성 유실 위험**이라는 역설을 안고 있으므로, 정규화가 완료된 모델을 기반으로 성능 측정 후 꼭 필요한 곳에만 전략적으로 적용해야 한다.

+++

### Ⅰ. 반정규화의 주요 기법 분류

1. **테이블 통합/분할**:
    - **통합**: 두 테이블을 하나로 합쳐 조인 제거 (1:1, 1:N 관계).
    - **분할**: 수평 분할(범위별), 수직 분할(컬럼별)을 통해 I/O 부하 분산.
2. **중복 컬럼 추가**:
    - 자주 조인되는 속성을 참조 테이블에 미리 복사해 둠.
3. **파생 컬럼(Derived Column) 생성**:
    - 총합, 평균 등 연산 결과를 컬럼에 미리 저장하여 런타임 계산 부하 제거.

+++

### Ⅱ. 반정규화(파생 컬럼) 시각화 (ASCII Model)

```text
[ 정규화 모델 (Normalized) ]           [ 반정규화 모델 (Denormalized) ]

  Table: [ Orders ]                     Table: [ Orders_Optimized ]
  - OrderID                             - OrderID
  - ProductID                           - ProductID
  - Qty                                 - Qty
  - Price                               - Price
                                        - Total_Amount ✅ (파생 컬럼 추가)
                                        
  * Query: SELECT Qty * Price...        * Query: SELECT Total_Amount...
  (런타임 계산 필요 -> 부하)               (이미 계산됨 -> 즉시 응답) ✅
```

+++

### Ⅲ. 반정규화 수행 시 고려사항 (성능 트레이드오프)

- **데이터 정합성 리스크**: 원본 값이 바뀔 때 중복된 모든 곳을 업데이트해야 하므로 '갱신 이상' 위험이 큽니다.
- **성능 반비례**: 조회(Read) 성능은 좋아지지만, 중복 데이터 관리로 인해 입력/수정/삭제(Write) 성능은 떨어질 수 있습니다.
- **결정 기준**: 조인 빈도가 너무 높아 응답 시간이 비즈니스 요건을 초과할 때, 혹은 쿼리가 너무 복잡하여 유지보수가 불가능할 때만 최후의 수단으로 선택합니다.

- **📢 섹션 요약 비유**: 반정규화는 **'요리 재료를 미리 손질해 두는 것'**과 같습니다. 손님이 올 때마다 재료를 씻고 다듬는 것(정규화)이 가장 신선하고 정확하지만, 손님이 수만 명이라면 미리 다 썰어서 통에 담아두는(반정규화) 것이 훨씬 빠릅니다. 다만, 재료가 상했을 때(데이터 변경) 모든 통을 다 확인해서 바꿔야 하는 수고를 감수해야 하는 것과 같습니다.

+++

### Ⅳ. 개념 맵 및 요약

- **[Trade-off]**: 성능 향상과 무결성 유지 사이의 줄다리기.
- **[Redundancy]**: 반정규화가 의도적으로 허용하는 데이터 중복.
- **[Anomalies]**: 반정규화 시 발생할 수 있는 삽입/삭제/갱신 이상 현상.

📢 **마무리 요약**: **Denormalization**은 모델링의 실패가 아닌, 현실을 직시한 공학적 결단입니다. 완벽한 정규화라는 이상을 바탕으로, 비즈니스 성과라는 실리를 챙기는 정교한 균형 감각이 필요합니다.
