## 핵심 인사이트 (3줄 요약)

### Ⅰ. 개요

#### 1. 개념
하이브리드 컴퓨터(Hybrid Computer)는 **아날로그 컴퓨터와 디지털 컴퓨터의 기능을 결합**한 시스템이다.

> 비유: "디지털 카메라 + 필름 카메라" - 두 방식의 장점 활용

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 3. 구조
```
┌─────────────────────────────────────┐
│           하이브리드 컴퓨터          │
│  ┌─────────┐     ┌─────────┐       │
│  │ 아날로그 │ ←→  │  디지털  │       │
│  │  프로세서│     │  프로세서│       │
│  └─────────┘     └─────────┘       │
│       ↑               │            │
│    센서              ↓             │
│  (온도,압력)      [ADC]            │
│       │              ↓             │
│       └────────── [DAC] ──→ 제어기  │
└─────────────────────────────────────┘

ADC: Analog-to-Digital Converter
DAC: Digital-to-Analog Converter
```

#### 4. 동작 과정
```
1. 아날로그 입력 (온도, 압력, 속도)
2. ADC로 디지털 변환
3. 디지털 연산 (정확한 계산)
4. DAC로 아날로그 변환
5. 아날로그 출력 (모터 제어 등)
```

#### 5. 응용 분야
| 분야 | 용도 |
|------|------|
| 의료 | ECG, MRI 신호 처리 |
| 군사 | 레이더, 미사일 유도 |
| 공장 | PLC, 공정 제어 |
| 항공 | 비행 시뮬레이터 |
| 통신 | 신호 변환 |

#### 7. 예시: 의료 장비
```
ECG (심전도) 시스템:

심장 신호 (아날로그)
    ↓
증폭기
    ↓
ADC (디지털 변환)
    ↓
디지털 신호 처리
    ↓
화면 표시 + 알람

→ 실시간 모니터링 + 정확한 진단
```

#### 8. PLC (Programmable Logic Controller)
```
공장 자동화의 하이브리드:

┌─────────┐
│   PLC   │
├─────────┤
│ 센서 입력│ → 디지털 처리 → 모터 제어
└─────────┘

온도, 압력 등 아날로그 입력
→ 디지털 연산
→ 아날로그 제어 출력
```

### Ⅲ. 기술 비교 분석

#### 2. 아날로그 vs 디지털 vs 하이브리드
```
아날로그:
- 연속적 신호
- 빠른 실시간 처리
- 정확도 낮음

디지털:
- 이산적 신호 (0, 1)
- 정확한 계산
- 처리 속도 느림

하이브리드:
- 아날로그 입력 → 디지털 처리 → 아날로그 출력
- 실시간 + 정확성
```

#### 6. 장단점
| 장점 | 단점 |
|-----|------|
| 실시간 처리 | 복잡한 구조 |
| 높은 정확도 | 비용 높음 |
| 유연성 | 유지보수 어려움 |
