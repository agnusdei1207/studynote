# T 플립플롭 (T Flip-Flop)

## 핵심 인사이트 (3줄 요약)
> **토글(Toggle) 동작만 수행**하는 플립플롭. T=1일 때마다 출력이 반전되어 주파수 분주기, 카운터에 최적화되어 있다. JK 플립플롭에서 J=K로 연결하여 쉽게 구현 가능하다.

### Ⅰ. 개요

#### 1. 개념
T 플립플롭(Toggle Flip-Flop)은 **T 입력이 1일 때마다 출력을 반전**시킨다. T=0이면 상태 유지, T=1이면 토글 동작을 수행한다.

> 비유: "매번 누르면 반대로 바뀌는 스위치" - 한 번 누르면 켜지고, 또 누르면 꺼지고

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 진리표
| T | Q(n) | Q(n+1) | 동작 |
|---|------|--------|------|
| 0 | 0 | 0 | 유지 |
| 0 | 1 | 1 | 유지 |
| 1 | 0 | 1 | 토글 |
| 1 | 1 | 0 | 토글 |

#### 3. 논리 기호
```
         ┌───┐
    T ───┤   │
         │ T ├─── Q
  CLK ───┤>  │
         └───┘
```

#### 4. 회로 구현
```
T 플립플롭 = JK 플립플롭에서 J = K = T

또는 D 플립플롭에서:
D = T XOR Q = T·Q̄ + T̄·Q
```

#### 5. 주파수 분주 원리
```
입력 클럭:  │┃│┃│┃│┃│┃│┃│┃│┃
T=1 출력:  │██│  │██│  │██│
           └─┘ 2배 느림

입력 주파수 f → 출력 주파수 f/2
```

#### 9. 리플 카운터 구조
```
        ┌───┐     ┌───┐     ┌───┐
CLK ────┤ T ├──Q0─┤ T ├──Q1─┤ T ├──Q2
        └───┘     └───┘     └───┘
          T=1       T=1       T=1

Q0: 2분주
Q1: 4분주
Q2: 8분주
```

#### 6. 장단점

|-----|------|
| 카운터 설계가 간단함 | 단독 IC로 드묾 |
| 주파수 분주에 최적 | D FF로 대체 가능 |
| 회로가 간단함 | 기능이 제한적 |

---

### Ⅳ. 실무 적용 방안

#### 7. 활용 사례
1. **주파수 분주기**: 2^n 분주 회로
2. **이진 카운터**: 리플 카운터의 기본
3. **위상 검출**: 위상 비교 회로
4. **클럭 생성**: 저주파 클럭 생성

#### 10. 실무에선? (기술사적 판단)
- **FPGA**: T FF는 직접 제공하지 않고 D FF로 구현
- **ASIC**: 표준 셀 라이브러리에 T FF 포함될 수 있음
- **동기 카운터**: 리플 카운터보다 동기식 선호
