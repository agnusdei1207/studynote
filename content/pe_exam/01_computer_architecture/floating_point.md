### Ⅰ. 개요

#### 1. 개념
부동 소수점 수(Floating Point Number)는 **소수점이 움직일 수 있는(浮動)** 형태로 실수를 표현한다. 과학적 표기법을 2진수로 구현한 것이다.

> 비유: "과학적 표기법" - 1230000 = 1.23 × 10^6

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 표현 방식
```
부동 소수점 표현:

값 = (-1)^부호 × 가수 × 밑수^지수

IEEE 754 단정도 (32-bit):
┌────┬─────────┬───────────────────┐
│부호│  지수   │       가수        │
│1bit│ 8 bits  │     23 bits       │
└────┴─────────┴───────────────────┘

IEEE 754 배정도 (64-bit):
┌────┬──────────┬───────────────────────┐
│부호│   지수   │        가수           │
│1bit│ 11 bits  │      52 bits          │
└────┴──────────┴───────────────────────┘
```

#### 3. IEEE 754 형식
```
단정도 (32-bit):
- 부호: 1 bit (0=양수, 1=음수)
- 지수: 8 bits (bias = 127)
- 가수: 23 bits (1.xxxxx의 xxxxx)

예: -6.625 표현
1. 2진수 변환: -110.101
2. 정규화: -1.10101 × 2^2
3. 부호: 1
4. 지수: 2 + 127 = 129 = 10000001
5. 가수: 10101000...

결과: 1 10000001 10101000000000000000000
```

#### 4. 특수 값
| 값 | 지수 | 가수 | 의미 |
|----|------|------|------|
| 0 | 0 | 0 | 0 |
| 비정규수 | 0 | ≠0 | 아주 작은 수 |
| 정규수 | 1~254 | 임의 | 일반 수 |
| 무한대 | 255 | 0 | ±∞ |
| NaN | 255 | ≠0 | Not a Number |

#### 5. 정밀도
```
단정도 (float):
- 유효 자릿수: 약 7자리
- 범위: ±3.4 × 10^38

배정도 (double):
- 유효 자릿수: 약 15자리
- 범위: ±1.8 × 10^308
```

#### 6. 정밀도 문제
```python
>>> 0.1 + 0.2
0.30000000000000004  # 정확히 0.3이 아님!

>>> 0.1 + 0.2 == 0.3
False

이유: 0.1, 0.2를 2진수로 정확히 표현 불가
0.1(10진) = 0.0001100110011...(2진, 무한소수)
```

#### 7. 반올림 모드
| 모드 | 설명 |
|------|------|
| Round to Nearest | 가장 가까운 값 (기본) |
| Round toward 0 | 0 쪽으로 |
| Round toward +∞ | 양의 무한대 쪽으로 |
| Round toward -∞ | 음의 무한대 쪽으로 |
