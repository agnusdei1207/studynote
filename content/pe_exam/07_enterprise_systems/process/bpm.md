# BPM (Business Process Management)

## 핵심 인사이트 (3줄 요약)
> **비즈니스 프로세스의 지속적 최적화**. 모델링, 실행, 모니터링, 개선 순환. 프로세스 자동화와 효율성 향상.

### Ⅰ. 개요

#### 1. 개념
BPM(Business Process Management)은 **기업의 비즈니스 프로세스를 체계적으로 정의, 실행, 모니터링, 분석, 최적화하는 경영 방법론과 시스템**이다. 프로세스의 지속적인 개선을 통해 운영 효율성을 높인다.

> 비유: "운동 코치" - 기업의 업무 방식을 계속 개선해요

---

│    │(Analyze)│  │  (Execute)  │  │ (Model) │          │

│         │              ▲                               │
│         │              │                               │

#### 3. BPMN (프로세스 모델링 표기법)

┌────────────────────────────────────────────────────────┐

│                                                        │

│  │  ◉ 종료 이벤트                                  │   │
│  │  ◎ 중간 이벤트                                  │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🟦 액티비티 (Activity):                              │
│  ┌────────────────────────────────────────────────┐   │
│  │  ▭ 작업 (Task)                                  │   │
│  │  ▭▧ 서브프로세스                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  ◇ 게이트웨이 (Gateway):                              │
│  ┌────────────────────────────────────────────────┐   │
│  │  ◇ 배타적 (XOR) - 하나만 선택                   │   │
│  │  ◇ 병렬 (AND) - 모두 실행                       │   │
│  │  ◇ 포함적 (OR) - 조건에 따라 여러 개            │   │

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **경영 효율** | 프로세스 자동화·통합으로 업무 생산성 향상 | 행정 업무 처리 시간 40% 단축 |
| **의사결정** | 실시간 BI·분석으로 데이터 기반 의사결정 지원 | 의사결정 속도 50% 향상 |
| **IT 거버넌스** | 표준화된 거버넌스 체계로 IT 리스크 관리 강화 | IT 감사 지적 사항 60% 감소 |

#### 결론
> **BPM (Business Process Management)**은(는) 기업 정보 시스템은 ERP·CRM에서 시작하여 DX(디지털 전환)·초자동화(Hyper-automation)·AI 통합으로 진화하며, 기업의 모든 운영 영역을 데이터로 연결하는 디지털 기업(Digital Enterprise)의 근간이 될 것이다.

> **※ 참고 표준**: ITIL v4(AXELOS), COBIT 2019(ISACA), ISO/IEC 20000-1:2018, ISO 9001
