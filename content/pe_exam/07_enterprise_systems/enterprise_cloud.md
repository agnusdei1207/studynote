### Ⅰ. 개요

#### 1. 개념
기업용 클라우드(Enterprise Cloud)는 **인터넷을 통해 서버, 스토리지, 데이터베이스, 애플리케이션 등 IT 리소스를 필요할 때마다 주문형으로 제공받는 컴퓨팅 서비스 모델**이다. 초기 투자 없이 사용량 기반으로 비용을 지불한다.

> 비유: "IT 공공요금" - 전기처럼 쓴 만큼 내요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 클라우드 서비스 모델
```
┌────────────────────────────────────────────────────────┐
│           클라우드 서비스 모델                        │
├────────────────────────────────────────────────────────┤
│                                                        │
│  ☁️ SaaS (Software as a Service):                     │
│  ┌────────────────────────────────────────────────┐   │
│  │  완제품 소프트웨어                              │   │
│  │  • 이메일, 협업툴, CRM, ERP                    │   │
│  │  • 예: Gmail, Salesforce, Microsoft 365       │   │
│  │  • 사용자: 비즈니스 사용자                      │   │
│  └────────────────────────────────────────────────┘   │
│                         │                             │
│                         ▼                             │
│  🛠️ PaaS (Platform as a Service):                    │
│  ┌────────────────────────────────────────────────┐   │
│  │  앱 개발/실행 플랫폼                            │   │
│  │  • 런타임, 미들웨어, 개발도구                   │   │
│  │  • 예: AWS Elastic Beanstalk, Heroku           │   │
│  │  • 사용자: 개발자                               │   │
│  └────────────────────────────────────────────────┘   │
│                         │                             │
│                         ▼                             │
│  🖥️ IaaS (Infrastructure as a Service):               │
│  ┌────────────────────────────────────────────────┐   │
│  │  가상 인프라                                    │   │
│  │  • VM, 스토리지, 네트워크                       │   │
│  │  • 예: AWS EC2, Azure VM, Google Compute       │   │
│  │  • 사용자: IT 관리자                            │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 클라우드 배포 모델
| 모델 | 설명 | 특징 |
|-----|------|------|
| 퍼블릭 | 공용 클라우드 | 낮은 비용, 높은 확장성 |
| 프라이빗 | 전용 클라우드 | 높은 보안, 높은 비용 |
| 하이브리드 | 혼합 클라우드 | 유연성, 복잡성 |
| 멀티 | 다중 클라우드 | 벤더 종속 방지 |

#### 4. 주요 클라우드 공급자
| 공급자 | 특징 | 강점 |
|-------|------|------|
| AWS | 시장 선두 | 서비스 다양성 |
| Azure | MS 생태계 | 엔터프라이즈 |
| GCP | 데이터/AI | 빅데이터 |

#### 결론
> **기업용 클라우드 (Enterprise Cloud)**은(는) 기업 정보 시스템은 ERP·CRM에서 시작하여 DX(디지털 전환)·초자동화(Hyper-automation)·AI 통합으로 진화하며, 기업의 모든 운영 영역을 데이터로 연결하는 디지털 기업(Digital Enterprise)의 근간이 될 것이다.

> **※ 참고 표준**: ITIL v4(AXELOS), COBIT 2019(ISACA), ISO/IEC 20000-1:2018, ISO 9001
