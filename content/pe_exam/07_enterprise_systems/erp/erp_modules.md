### Ⅰ. 개요

#### 1. 개념
ERP 모듈은 **전사적 자원 관리(ERP) 시스템에서 특정 업무 영역을 담당하는 독립적이면서도 통합된 기능 단위**이다. 주요 모듈로는 재무회계(FI), 관리회계(CO), 영업판매(SD), 자재관리(MM), 생산관리(PP) 등이 있다.

> 비유: "ERP = 앱스토어, 모듈 = 개별 앱" - 필요한 기능을 골라 써요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. ERP 모듈 구성
```
┌────────────────────────────────────────────────────────┐
│           ERP 핵심 모듈                               │
├────────────────────────────────────────────────────────┤
│                                                        │
│  💰 FI (Financial Accounting) - 재무회계:              │
│  ┌────────────────────────────────────────────────┐   │

│  │  • 채권/채무 관리 (AR/AP)                       │   │
│  │  • 자산 관리 (AM)                               │   │
│  │  • 재무제표 작성                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📊 CO (Controlling) - 관리회계:                       │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 원가 계산                                    │   │
│  │  • 수익성 분석                                  │   │
│  │  • 예산 관리                                    │   │
│  │  • 이익 센터 회계                               │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🛒 SD (Sales & Distribution) - 영업/판매:             │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 견적/주문 관리                               │   │
│  │  • 출하/배송                                   │   │
│  │  • 청구/수금                                   │   │
│  │  • 판매 분석                                   │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📦 MM (Material Management) - 자재관리:               │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 구매 관리                                    │   │
│  │  • 재고 관리                                    │   │
│  │  • 자재 소요량 계획                             │   │
│  │  • 입고 검사                                    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🏭 PP (Production Planning) - 생산관리:               │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 생산 계획 (MPS)                              │   │
│  │  • 소요량 계획 (MRP)                            │   │
│  │  • 작업 지시                                    │   │
│  │  • 품질 관리                                    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 모듈 간 통합 흐름
```
┌────────────────────────────────────────────────────────┐
│           모듈 간 데이터 흐름                         │
├────────────────────────────────────────────────────────┤
│                                                        │
│  🛒 주문 (SD) ──────────────────────┐                 │
│      │                              │                 │
│      ▼                              ▼                 │
│  📦 자재확보 (MM) ◄─────── 🏭 생산계획 (PP)           │
│      │                         │                      │
│      ▼                         ▼                      │
│  💰 매출 (FI) ◄─────────── 📊 원가 (CO)               │
│                                                        │
│  • SD 주문 → MM 출고 → FI 매출 인식                    │
│  • PP 생산 → MM 자재 소모 → CO 원가 계산               │
│  • MM 구매 → FI 채무 발생                              │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 4. 모듈별 주요 기능
| 모듈 | 핵심 기능 | 주요 트랜잭션 |
|-----|---------|-------------|
| FI | 재무 기장, 결산 | 전표입력, B/S, P/L |
| CO | 원가관리, 수익성 | 비용배분, 손익분석 |
| SD | 주문~수금 | 견적, 주문, 출하, 청구 |
| MM | 구매~지급 | 구매요청, 발주, 입고 |
| PP | 계획~생산 | MPS, MRP, 작업지시 |
