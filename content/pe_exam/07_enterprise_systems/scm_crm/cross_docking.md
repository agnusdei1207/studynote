# 크로스도킹 (Cross-Docking)

## 핵심 인사이트 (3줄 요약)
> **입고된 화물을 보관 없이 즉시 출고**하는 물류 전략. 창고 저장 단계 생략, 리드타임 단축, 재고 비용 절감. JIT(적시생산)와 결합하여 효과 극대화.

### Ⅰ. 개요

#### 1. 개념
크로스도킹(Cross-Docking)은 **공급업체에서 들어온 화물을 창고에 보관하지 않고 즉시 분류하여 최종 목적지로 출고**하는 물류 전략이다.

> 비유: "택시 환승" - 내려서 바로 다른 택시로 갈아타는 것 (대기 없이)

### Ⅲ. 기술 비교 분석

#### 2. 전통적 물류 vs 크로스도킹
```
┌────────────────────────────────────────────────────────┐
│         전통적 물류 vs 크로스도킹                       │
├────────────────────────────────────────────────────────┤
│                                                        │
│  전통적 물류 (Traditional Warehousing):                │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  공급업체 → [입고] → [보관] → [피킹] → [출고] → 고객│   │
│  │                         ↑                      │   │
│  │                    재고 보관                    │   │
│  │                    (수일~수주)                  │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  크로스도킹 (Cross-Docking):                          │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  공급업체 → [입고] → [분류] → [출고] → 고객    │   │
│  │                ↓        ↑                      │   │
│  │            ┌────────────┐                      │   │
│  │            │ 크로스도킹 │                      │   │
│  │            │    구역   │                      │   │
│  │            └────────────┘                      │   │
│  │             (24시간 이내)                       │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  비교표:                                               │
│  ┌─────────────┬─────────────┬─────────────┐         │
│  │    구분      │   전통적    │ 크로스도킹  │         │
│  ├─────────────┼─────────────┼─────────────┤         │
│  │ 처리 시간   │ 수일~수주   │ 24시간 이내 │         │
│  │ 재고 보관   │ 필요        │ 최소화      │         │
│  │ 창고 면적   │ 대형 필요   │ 중간        │         │
│  │ 총비용      │ 높음        │ 낮음        │         │
│  │ 유연성      │ 높음        │ 낮음        │         │
│  │ 예측 필요   │ 중간        │ 높음        │         │
│  └─────────────┴─────────────┴─────────────┘         │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 결론
> **크로스도킹 (Cross-Docking)**은(는) 기업 정보 시스템은 ERP·CRM에서 시작하여 DX(디지털 전환)·초자동화(Hyper-automation)·AI 통합으로 진화하며, 기업의 모든 운영 영역을 데이터로 연결하는 디지털 기업(Digital Enterprise)의 근간이 될 것이다.

> **※ 참고 표준**: ITIL v4(AXELOS), COBIT 2019(ISACA), ISO/IEC 20000-1:2018, ISO 9001
