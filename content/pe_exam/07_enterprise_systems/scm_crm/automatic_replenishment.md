# 자동발주시스템 (Automatic Replenishment)

## 핵심 인사이트 (3줄 요약)
> **재고 수준에 따라 자동으로 발주**하는 시스템. 안전재고, 재주문점, EOQ 기반 자동화. 재고 부족 방지와 발주 효율화.

### Ⅰ. 개요

#### 1. 개념
자동발주시스템(Automatic Replenishment System)은 **재고가 일정 수준 이하로 떨어지면 미리 설정된 규칙에 따라 자동으로 발주**를 생성하는 시스템이다.

> 비유: "냉장고 센서" - 우유가 떨어지면 자동으로 주문

---

```

│                                                        │

│  │                                                │   │

│  │    └───────────────────────────→ 시간          │   │
│  │         │← LT →│                              │   │
│  │         발주   입고                            │   │
│  │                                                │   │
│  │  ROP = 일일수요 × 리드타임 + 안전재고         │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 정기발주 방식 (Periodic Review):                  │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  재고량                                        │   │
│  │    ↑                                           │   │
│  │    │     ●━━━━━━━━━━                          │   │
│  │    │    ╱                                      │   │
│  │    │   ╱                                       │   │

│  │    │  ╲     발주량 = 목표재고 - 현재재고      │   │

│  │    │     ●                                     │   │
│  │    └───────────────────────────→ 시간          │   │
│  │         검토   검토   검토                     │   │
│  │         (월)  (화)  (수)                       │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │

│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │

│  │                                                │   │
│  │  [공급업체] ←─재고정보─→ [고객사]            │   │
│  │       │                        │               │   │
│  │       └──── 자동 납품 ────────┘               │   │

│  └────────────────────────────────────────────────┘   │
│                                                        │
```

#### 3. EOQ (경제적 주문량)
```
┌────────────────────────────────────────────────────────┐
│           EOQ (Economic Order Quantity)                │
├────────────────────────────────────────────────────────┤
│                                                        │
│  목표: 총비용을 최소화하는 최적 주문량 찾기           │
│                                                        │
│  비용 요소:                                            │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  1. 발주비용 (Ordering Cost)                  │   │
│  │     • 주문 처리 비용                          │   │
│  │     • 운송비                                  │   │
│  │     • 검수 비용                               │   │
│  │                                                │   │
│  │  2. 재고유지비용 (Holding Cost)               │   │
│  │     • 창고 비용                               │   │
│  │     • 자본 비용 (기회비용)                    │   │
│  │     • 손상/진부화 비용                        │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  비용 그래프:                                          │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  비용                                          │   │
│  │    ↑                                           │   │
│  │    │    총비용                                │   │
│  │    │      ╲                                    │   │
│  │    │       ╲  ● ← 최저점 = EOQ               │   │
│  │    │      ╱  ╲                                 │   │
│  │    │     ╱    ╲                               │   │
│  │    │    ╱      ╲ 재고유지비용                │   │
│  │    │   ╱        ╲ (증가)                      │   │
│  │    │  ╱─────────╲                             │   │
│  │    │ ╱           ╲                            │   │
│  │    │╱ 발주비용    ╲                           │   │
│  │    │  (감소)                                  │   │
│  │    └─────────────────────→ 주문량             │   │
│  │             EOQ                               │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  EOQ 공식:                                             │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │         √(2 × D × S)                          │   │
│  │  EOQ = ───────────                            │   │
│  │              H                                 │   │
│  │                                                │   │
│  │  D: 연간 수요량                                │   │
│  │  S: 1회 발주비용                               │   │
│  │  H: 단위당 연간 재고유지비용                  │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```
