### Ⅰ. 개요

#### 1. 개념
MES(Manufacturing Execution System, 제조 실행 시스템)는 **공장의 생산 계획부터 완제품 출하까지 전체 생산 과정을 실시간으로 관리, 추적, 통제**하는 시스템이다.

> 비유: "공장의 교통관제센터" - 모든 생산 활동을 실시간으로 모니터링하고 제어

---

#### 4. 주요 성과 지표
```
┌────────────────────────────────────────────────────────┐
│                   OEE (전체설비효율)                    │
├────────────────────────────────────────────────────────┤
│                                                        │
│  OEE = 가동률 × 성능율 × 양품률                        │
│                                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  가동률 (Availability):                        │   │
│  │  = 가동시간 / 계획가동시간 × 100              │   │
│  │  • 고장, 셋업, 대기 시간 고려                 │   │
│  │                                                │   │
│  │  성능율 (Performance):                         │   │
│  │  = 생산량 / (가동시간 × 표준사이클) × 100     │   │
│  │  • 속도 손실, 미세정지 고려                    │   │
│  │                                                │   │
│  │  양품률 (Quality):                             │   │
│  │  = 양품수량 / 총생산량 × 100                   │   │
│  │  • 불량, 재작업 고려                           │   │
│  │                                                │   │
│  │  세계급 OEE: 85% 이상                          │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  손실 분석 (6대 손실):                                │
│  ┌────────────────────────────────────────────────┐   │
│  │ 설비 손실: 고장, 셋업/조정, 단축/공회전       │   │
│  │ 성능 손실: 관리손실, 속도저하                  │   │
│  │ 품질 손실: 불량/재작업                         │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```
