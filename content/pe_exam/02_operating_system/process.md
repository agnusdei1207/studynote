### Ⅰ. 개요

#### 1. 개념
프로세스는 **실행 중인 프로그램으로 독립된 메모리 공간을 가지는 작업 단위**이고, 스레드는 **프로세스 내에서 실행되는 실행 흐름의 최소 단위**다. 프로세스는 최소 하나의 스레드를 가진다.

> 비유: "프로세스=공장, 스레드=작업자" - 공장 안에서 여러 작업자가 일해요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 프로세스 vs 스레드
```
┌────────────────────────────────────────────────────────┐
│           프로세스 vs 스레드                           │
├────────────────────────────────────────────────────────┤
│                                                        │
│  📦 프로세스 (Process):                                │
│  ┌────────────────────────────────────────────────┐   │
│  │  ┌──────────────────────────────────────────┐  │   │
│  │  │ Code Segment (코드)                      │  │   │
│  │  ├──────────────────────────────────────────┤  │   │
│  │  │ Data Segment (데이터)                    │  │   │
│  │  ├──────────────────────────────────────────┤  │   │
│  │  │ Heap (힙)                                │  │   │
│  │  ├──────────────────────────────────────────┤  │   │
│  │  │ Stack (스택)                             │  │   │
│  │  └──────────────────────────────────────────┘  │   │
│  │  → 독립된 메모리 공간                           │   │
│  │  → 생성 비용 높음                               │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🧵 멀티스레드 프로세스:                               │
│  ┌────────────────────────────────────────────────┐   │
│  │  ┌──────────────────────────────────────────┐  │   │
│  │  │ 공유: Code, Data, Heap                   │  │   │
│  │  ├──────────────────────────────────────────┤  │   │
│  │  │ Thread1 │ Thread2 │ Thread3              │  │   │
│  │  │ Stack   │ Stack   │ Stack                │  │   │
│  │  │ Registers│Registers│Registers            │  │   │
│  │  └──────────────────────────────────────────┘  │   │
│  │  → 메모리 공유                                   │   │
│  │  → 생성 비용 낮음                               │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 프로세스 상태
| 상태 | 설명 |
|-----|------|
| New | 생성 중 |
| Ready | 실행 대기 |
| Running | CPU 사용 중 |
| Waiting | 이벤트 대기 |
| Terminated | 종료 |

#### 5. 스케줄링 알고리즘
| 알고리즘 | 설명 | 특징 |
|---------|------|------|
| FCFS | 선입선출 | 단순, Convoy |
| SJF | 최단 작업 우선 | 평균 대기 최소 |
| RR | 라운드 로빈 | 공정, 시분할 |
| Priority | 우선순위 | 기아 현상 |
| MLQ | 다단계 큐 | 우선순위 그룹 |
