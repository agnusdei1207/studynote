# HUD (헤드업 디스플레이)

## 핵심 인사이트 (3줄 요약)

### Ⅰ. 개요

#### 1. 개념
HUD(Head-Up Display)는 **투명한 화면이나 유리에 정보를 투영하여, 사용자가 시선을 돌리지 않고도 중요한 정보를 볼 수 있게 하는 디스플레이 기술**이다.

> 비유: "눈앞에 뜨는 정보 창" - 게임 UI처럼 현실에 정보가 떠요

---

#### 3. HUD 유형
```
┌────────────────────────────────────────────────────────┐
│           HUD 유형                                    │
├────────────────────────────────────────────────────────┤
│                                                        │

│  │  전면 유리에 직접 투영                         │   │

│  │  시야각: 넓음                                 │   │
│  │                                                │   │
│  │  장점: 넓은 화면, 자연스러운 시각             │   │
│  │  단점: 비용 높음, 차량 설계 변경 필요         │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 콤바이너 HUD (Combiner HUD):                      │

│  │                                                │   │
│  │  장점: 설치 쉬움, 저렴                        │   │
│  │  단점: 화면 작음, 시야 제한                    │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3. AR HUD (증강현실 HUD):                            │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  실제 도로 위에 정보 겹쳐서 표시               │   │
│  │                                                │   │
│  │  예:                                           │   │
│  │  • 내비게이션 화살표가 도로에 표시             │   │
│  │  • 앞차와 거리 표시                            │   │
│  │  • 보행자 하이라이트                           │   │
│  │                                                │   │
│  │  장점: 직관적, 안전                           │   │
│  │  단점: 기술 복잡, 비용 높음                    │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```
