# AR/VR/MR (확장현실)

## 핵심 인사이트 (3줄 요약)
> **현실과 가상을 결합하는 몰입형 기술**. AR은 현실 위에 가상 정보, VR은 완전한 가상 세계, MR은 현실과 가상의 상호작용. 메타버스의 핵심 기반.

### Ⅰ. 개요

#### 1. 개념
확장현실(XR, Extended Reality)은 **현실 세계와 가상 세계를 결합하거나 완전히 대체하여 몰입형 경험을 제공**하는 기술의 총칭이다.

> 비유: "디지털 안경" - 보는 세상이 달라진다

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 3. 핵심 기술
```
┌────────────────────────────────────────────────────────┐
│                  XR 핵심 기술                           │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 디스플레이 기술                                    │
│  ┌────────────────────────────────────────────────┐   │
│  │ HMD (Head-Mounted Display):                   │   │
│  │ • VR: 오큘러스, 바이브                        │   │
│  │ • MR: 홀로렌즈, 비전 프로                     │   │
│  │                                                │   │
│  │ 광학 방식:                                     │   │
│  │ • 비디오 패스스루 (Video Passthrough)         │   │
│  │ • 광학 패스스루 (Optical Passthrough)         │   │
│  │ • 프로젝션                                     │   │
│  │                                                │   │
│  │ 해상도/주사율:                                 │   │
│  │ • 4K 이상, 90Hz+ 권장                         │   │
│  │ • 시야각(FOV): 100도 이상                     │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 트래킹 기술                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │ 위치 추적:                                     │   │
│  │ • 6DoF (6 Degrees of Freedom)                 │   │
│  │   - 위치: X, Y, Z                             │   │
│  │   - 회전: Pitch, Yaw, Roll                    │   │
│  │                                                │   │
│  │ 센서:                                          │   │
│  │ • IMU (관성 측정 장치)                        │   │
│  │ • 카메라 (Inside-Out / Outside-In)            │   │
│  │ • Lighthouse (베이스 스테이션)                │   │
│  │ • 라이다 (LiDAR)                              │   │
│  │                                                │   │
│  │ 시선 추적 (Eye Tracking):                     │   │
│  │ • 포베이티드 렌더링                           │   │
│  │ • 인터랙션                                     │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3. 렌더링 기술                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 실시간 3D 렌더링 (60-120 FPS)               │   │
│  │ • SLAM (Simultaneous Localization & Mapping)  │   │
│  │ • 포베이티드 렌더링 (시선 중심)               │   │
│  │ • 지연 시간 < 20ms (멀미 방지)                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  4. 인터랙션                                           │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 컨트롤러 (모션 컨트롤러)                    │   │
│  │ • 핸드 트래킹 (손 추적)                        │   │
│  │ • 제스처 인식                                  │   │
│  │ • 음성 인식                                    │   │
│  │ • 햅틱 피드백                                  │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### Ⅲ. 기술 비교 분석

#### 2. AR vs VR vs MR
```
┌────────────────────────────────────────────────────────┐
│              AR / VR / MR 비교                         │
├────────────────────────────────────────────────────────┤
│                                                        │
│  현실감 ←──────────────────────────────────→ 가상감   │
│                                                        │
│  AR (Augmented Reality) 증강현실:                     │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │   현실 세계 + 디지털 정보 오버레이            │   │
│  │                                                │   │
│  │   [실제 풍경]                                  │   │
│  │      +                                         │   │
│  │   [네비게이션 화살표]                          │   │
│  │      +                                         │   │
│  │   [상점 정보 팝업]                             │   │
│  │                                                │   │
│  │   예: 포켓몬고, IKEA Place, 네이버 지도 AR    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  VR (Virtual Reality) 가상현실:                       │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │   완전히 가상의 3D 환경                        │   │
│  │   현실 세계 차단                              │   │
│  │                                                │   │
│  │   ┌─────────────────────┐                     │   │
│  │   │  가상 공간          │                     │   │
│  │   │  (게임, 교육, 등)   │                     │   │
│  │   └─────────────────────┘                     │   │
│  │                                                │   │
│  │   예: Meta Quest, HTC Vive, 게임              │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  MR (Mixed Reality) 혼합현실:                         │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │   현실 + 가상의 양방향 상호작용                │   │
│  │                                                │   │
│  │   [실제 테이블 위에]                           │   │
│  │      +                                         │   │
│  │   [가상 물체 (잡을 수 있음)]                   │   │
│  │      +                                         │   │
│  │   [물리적 충돌 감지]                           │   │
│  │                                                │   │
│  │   예: Microsoft HoloLens, Apple Vision Pro    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  XR (Extended Reality) 확장현실:                      │
│  ┌────────────────────────────────────────────────┐   │
│  │   AR + VR + MR + 기타 몰입형 기술 통칭         │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **비즈니스 혁신** | 디지털 전환 가속화 및 신규 비즈니스 모델 창출 | 시장 출시 시간(TTM) 50% 단축 |

| **경쟁력 강화** | 최신 기술 도입으로 시장 경쟁 우위 확보 | 고객 만족도(CSAT) 20점 향상 |
