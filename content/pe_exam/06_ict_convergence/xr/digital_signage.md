# 디지털 사이니지 (Digital Signage)

## 핵심 인사이트 (3줄 요약)
> **디지털 디스플레이로 정보 전달 및 광고**를 하는 시스템. 중앙 집중 관리, 실시간 콘텐츠 변경, 타겟팅 광고 가능. IoT, AI와 결합한 스마트 사이니지로 진화.

### Ⅰ. 개요

#### 1. 개념
디지털 사이니지는 **디지털 디스플레이 장치를 통해 정보, 광고, 엔터테인먼트 콘텐츠를 전달**하는 시스템이다.

> 비유: "똑똑한 전광판" - 상황에 맞춰 스스로 바뀌는 광고판

---

│  │ • 온도/습도: 날씨 연동 콘텐츠                  │   │

│  │ • 효과 분석                                    │   │

│                                                        │
│  4. 연결성                                             │
│  ┌────────────────────────────────────────────────┐   │

│                                                        │
└────────────────────────────────────────────────────────┘
