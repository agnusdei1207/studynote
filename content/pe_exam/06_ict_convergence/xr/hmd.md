# HMD (헤드 마운트 디스플레이)

## 핵심 인사이트 (3줄 요약)

### Ⅰ. 개요

#### 1. 개념
HMD(Head-Mounted Display)는 **머리에 착용하여 눈 앞에 디스플레이를 위치시켜 가상 현실이나 증강 현실을 체험할 수 있게 하는 웨어러블 디스플레이 장치**이다.

> 비유: "눈에 쓰는 영화관" - 어디를 봐도 화면이 보여요

---

#### 3. HMD 핵심 사양
| 사양 | 설명 | 중요도 |
|-----|------|-------|
| 해상도 | 픽셀 수 (4K, 8K) | 높음 |
| 시야각 (FOV) | 화면이 보이는 범위 | 높음 |
| 주사율 | 초당 화면 갱신 (90Hz, 120Hz) | 높음 |
| 지연시간 | 입력~화면 반응 시간 | 매우 높음 |
| 6DoF | 6자유도 추적 | 높음 |
| 패널 타입 | OLED, LCD, MicroLED | 보통 |
| IPD 조절 | 눈 간격 조절 | 보통 |

#### 4. 트래킹 기술
```
┌────────────────────────────────────────────────────────┐
│           HMD 트래킹 기술                             │
├────────────────────────────────────────────────────────┤
│                                                        │
│  3DoF vs 6DoF:                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  3DoF (3 Degrees of Freedom):                 │   │
│  │  • 고개 회전만 추적                            │   │
│  │  • Pitch (위아래)                              │   │
│  │  • Yaw (좌우)                                  │   │
│  │  • Roll (기울기)                               │   │
│  │                                                │   │
│  │  → 제자리에서만 경험 가능                      │   │
│  │                                                │   │
│  │  6DoF (6 Degrees of Freedom):                 │   │
│  │  • 회전 (3DoF) + 이동 (3DoF)                  │   │
│  │  • X, Y, Z 축 이동                             │   │
│  │                                                │   │
│  │  → 공간을 걸어다닐 수 있음                     │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  트래킹 방식:                                         │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  내부 아웃 (Inside-Out):                       │   │
│  │  HMD 카메라가 환경을 인식                      │   │
│  │  예: Meta Quest, Apple Vision Pro             │   │
│  │                                                │   │
│  │  외부 인 (Outside-In):                         │   │
│  │  외부 센서가 HMD를 추적                        │   │
│  │  예: HTC Vive (베이스 스테이션)                │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```
