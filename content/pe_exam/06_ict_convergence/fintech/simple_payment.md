# 간편결제 (Simple Payment)

## 핵심 인사이트 (3줄 요약)
> **카드/계좌 정보를 등록하고 간편하게 결제**하는 서비스. 토큰화로 보안 강화, 생체인증으로 편의성 확보. 카카오페이, 네이버페이, 애플페이 등이 대표적.

### Ⅰ. 개요

#### 1. 개념
간편결제는 **카드 정보나 계좌 번호를 사전에 등록해두고, 비밀번호나 생체인증만으로 결제를 완료**할 수 있는 전자결제 서비스다.

> 비유: "디지털 지갑" - 카드를 일일이 꺼내지 않고도 결제

---

#### 3. 간편결제 프로세스
```
┌────────────────────────────────────────────────────────┐
│                결제 프로세스                            │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 최초 등록 (카드 등록)                              │
│  ┌────────────────────────────────────────────────┐   │
│  │ 사용자 → [카드정보입력] → [PG사검증]           │   │
│  │      → [토큰생성] → [토큰저장] → 완료         │   │
│  │                                                │   │
│  │  실제 카드정보는 토큰으로 대체하여 저장        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 결제 요청                                          │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  사용자           간편결제           PG사      │   │
│  │    │                │                 │        │   │
│  │    │──결제요청────→│                 │        │   │
│  │    │  (상품,금액)   │                 │        │   │
│  │    │                │──인증요청──→   │        │   │
│  │    │←─생체/비번───│                 │        │   │
│  │    │                │──토큰전송──→   │        │   │
│  │    │                │                 │──승인──→│
│  │    │←───결제완료───│←──승인결과──   │        │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3. 정산                                                │
│  PG사 → 가맹점 대금 정산 (T+1 ~ T+2일)               │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 4. 보안 기술
```
┌────────────────────────────────────────────────────────┐
│                 보안 기술                               │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 토큰화 (Tokenization)                              │
│     ┌────────────────────────────────────────────┐    │
│     │ 실제 카드번호: 1234-5678-9012-3456         │    │
│     │       ↓ 암호화                              │    │
│     │ 토큰: abc123xyz789...                      │    │
│     │                                            │    │
│     │ • 원본 카드정보는 토큰 서버에만 저장       │    │
│     │ • 서비스는 토큰만 보유                     │    │
│     │ • 유출되어도 원본 복원 불가                │    │
│     └────────────────────────────────────────────┘    │
│                                                        │
│  2. 생체인증                                           │
│     • 지문 인식                                       │
│     • Face ID / 얼굴 인식                             │
│     • 홍채 인식                                       │
│     • 정맥 인식                                       │
│                                                        │
│  3. 추가 인증 (2FA)                                    │
│     • SMS 인증번호                                    │
│     • ARS 인증                                        │
│     • 앱 푸시 인증                                    │
│                                                        │
│  4. NFC / MST                                         │
│     ┌────────────────────────────────────────────┐    │
│     │ NFC (Near Field Communication):            │    │
│     │ - 비접촉 통신 (10cm 이내)                  │    │
│     │ - 암호화된 결제 정보 전송                  │    │
│     │                                            │    │
│     │ MST (Magnetic Secure Transmission):        │    │
│     │ - 자기장으로 카드 단말기 에뮬레이션        │    │
│     │ - 기존 카드 단말기 호환 (삼성페이)         │    │
│     └────────────────────────────────────────────┘    │
│                                                        │
└────────────────────────────────────────────────────────┘
```
