# 옴니채널 (Omni-Channel)

## 핵심 인사이트 (3줄 요약)
> **모든 채널을 통합하여 일관된 고객 경험** 제공. 온라인-오프라인 경계 없는 서비스. O2O, 쇼루밍의 진화된 형태.

### Ⅰ. 개요

#### 1. 개념
옴니채널(Omni-Channel)은 **온라인, 오프라인, 모바일 등 모든 판매 채널을 통합하여 고객에게 끊김 없는(Seamless) 경험**을 제공하는 전략이다.

> 비유: "물처럼 자연스러운 쇼핑" - 어디서 시작하든 어디서든 이어짐

### Ⅱ. 구성 요소 및 핵심 원리

#### 3. 옴니채널 고객 여정
```
┌────────────────────────────────────────────────────────┐
│           옴니채널 고객 여정 예시                       │
├────────────────────────────────────────────────────────┤
│                                                        │
│  시나리오: 신발 쇼핑                                   │
│                                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  1단계: 발견 (Discovery)                       │   │
│  │  ┌──────────────────────────────────────────┐ │   │
│  │  │ 📱 인스타그램에서 광고 시청               │ │   │
│  │  │    → 앱으로 이동                         │ │   │
│  │  └──────────────────────────────────────────┘ │   │
│  │                      ↓                         │   │
│  │  2단계: 탐색 (Exploration)                     │   │
│  │  ┌──────────────────────────────────────────┐ │   │
│  │  │ 💻 집에서 PC로 상세 정보 확인            │ │   │
│  │  │    → 장바구니에 추가                     │ │   │
│  │  └──────────────────────────────────────────┘ │   │
│  │                      ↓                         │   │
│  │  3단계: 체험 (Experience) - 쇼루밍            │   │
│  │  ┌──────────────────────────────────────────┐ │   │
│  │  │ 💒 매장 방문, 신발 직접 신어보기         │ │   │
│  │  │    → QR코드로 앱에서 정보 확인          │ │   │
│  │  └──────────────────────────────────────────┘ │   │
│  │                      ↓                         │   │
│  │  4단계: 구매 (Purchase)                        │   │
│  │  ┌──────────────────────────────────────────┐ │   │
│  │  │ 📱 앱으로 결제 (매장 픽업 선택)          │ │   │
│  │  │    → 포인트 적립                         │ │   │
│  │  └──────────────────────────────────────────┘ │   │
│  │                      ↓                         │   │
│  │  5단계: 수령 (Fulfillment)                     │   │
│  │  ┌──────────────────────────────────────────┐ │   │
│  │  │ 💒 매장 픽업 또는 집 배송 선택           │ │   │
│  │  │    → 클릭앤컬렉트                       │ │   │
│  │  └──────────────────────────────────────────┘ │   │
│  │                      ↓                         │   │
│  │  6단계: 서비스 (Service)                       │   │
│  │  ┌──────────────────────────────────────────┐ │   │
│  │  │ 💬 챗봇으로 교환/반품 문의               │ │   │
│  │  │    → 매장에서 바로 교환                 │ │   │
│  │  └──────────────────────────────────────────┘ │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  핵심: 모든 단계가 연결되고 데이터가 공유됨           │
│                                                        │
└────────────────────────────────────────────────────────┘
```
