# 바이오메트릭스 (Biometrics)

## 핵심 인사이트 (3줄 요약)
> **신체적, 행동적 특성으로 개인 식별**하는 기술. 지문, 얼굴, 홍채, 음성 등 활용. 보안과 편의성을 동시에 확보하는 차세대 인증 방식.

### Ⅰ. 개요

#### 1. 개념
바이오메트릭스는 **사람의 고유한 생체적, 행동적 특징을 이용하여 신원을 확인**하는 기술이다.

> 비유: "몸이 곧 열쇠" - 잃어버릴 수 없는 고유한 신분증

---

#### 2. 생체 인식 유형

│                 생체 인식 유형                          │
├────────────────────────────────────────────────────────┤
│                                                        │

│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  지문 (Fingerprint)                           │   │
│  │  ┌────────────────────────────────────────┐   │   │
│  │  │ • 가장 널리 사용                        │   │   │

│  │  │ • 특징점: 분기점, 끝점                  │   │   │
│  │  │ • 정확도: 매우 높음                     │   │   │
│  │  └────────────────────────────────────────┘   │   │
│  │                                                │   │
│  │  얼굴 (Face Recognition)                       │   │   │

│  │  │ • 2D/3D 얼굴 인식                       │   │   │

│  │  │ • 딥러닝 기반 정확도 향상               │   │   │
│  │  │ • Apple Face ID: 30,000개 점 매핑       │   │   │
│  │  └────────────────────────────────────────┘   │   │
│  │                                                │   │
│  │  홍채 (Iris) / 망막 (Retina)                  │   │   │
│  │  ┌────────────────────────────────────────┐   │   │
│  │  │ • 홍채: 눈동자의 색깔 부분 패턴          │   │   │
│  │  │ • 망막: 안저 혈관 패턴                  │   │   │
│  │  │ • 정확도: 최고 수준                     │   │   │
│  │  │ • 비용: 높음                            │   │   │
│  │  └────────────────────────────────────────┘   │   │
│  │                                                │   │
│  │  정맥 (Vein)                                  │   │   │
│  │  • 손등/손가락 혈관 패턴                      │   │   │
│  │  • 적외선 카메라로 촬영                       │   │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 행동적 (Behavioral) 특징                           │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  음성 (Voice)                                 │   │   │

│  │                                                │   │
│  │  서명 (Signature)                             │   │   │

│  │                                                │   │
│  │  키 입력 패턴 (Keystroke)                     │   │   │

#### 4. 성능 지표
```
┌────────────────────────────────────────────────────────┐
│                   성능 지표                             │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 오류 유형                                          │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  FAR (False Acceptance Rate)                  │   │
│  │  • 거짓 수락률: 타인을 본인으로 인식          │   │
│  │  • 보안 위험                                   │   │
│  │                                                │   │
│  │  FRR (False Rejection Rate)                   │   │
│  │  • 거짓 거부률: 본인을 타인으로 인식          │   │
│  │  • 사용자 불편                                 │   │
│  │                                                │   │
│  │  EER (Equal Error Rate)                       │   │
│  │  • FAR = FRR 되는 지점                        │   │
│  │  • 시스템 성능 지표                            │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 정확도 비교 (일반적 수치)                          │
│  ┌────────────────────────────────────────────────┐   │
│  │  방식        │  EER (%)   │  특징             │   │
│  │  ────────────┼────────────┼────────────────── │   │
│  │  지문        │   0.1~2    │  가장 널리 사용   │   │
│  │  홍채        │   0.1~0.5  │  최고 정확도      │   │
│  │  얼굴 (3D)   │   0.1~1    │  비접촉          │   │
│  │  음성        │   2~5      │  원격 인증        │   │
│  │  서명       │   2~5      │  행동 기반        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3. ROC 곡선 (Receiver Operating Characteristic)      │
│     • FAR vs 1-FRR 그래프                            │
│     • 곡선 아래 면적(AUC) 클수록 좋음                 │
│                                                        │
└────────────────────────────────────────────────────────┘
```
