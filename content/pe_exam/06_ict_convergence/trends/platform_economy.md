# 플랫폼 (Platform)

## 핵심 인사이트 (3줄 요약)
> **생산자와 소비자를 연결하여 가치를 창출하는 디지털 공간**. 네트워크 효과로 성장. 현대 비즈니스의 핵심 모델.

---

## 📝 기술사 모의답안 (2.5페이지 분량)

### 📌 예상 문제
> **"플랫폼 (Platform)의 개념과 핵심 원리를 설명하고, 비교 분석 및 실무 적용 방안을 기술하시오."**

---

### Ⅰ. 개요

#### 1. 개념
플랫폼(Platform)은 **서로 다른 그룹(주로 생산자와 소비자)을 연결하여 상호작용하고 가치를 창출할 수 있게 하는 디지털 기반 공간이나 서비스**이다.

> 비유: "디지털 장터" - 파는 사람과 사는 사람을 만나게 해주는 곳

---

### Ⅴ. 기대 효과 및 결론

#### 3. 네트워크 효과
```
┌────────────────────────────────────────────────────────┐
│           네트워크 효과 (Network Effect)              │
├────────────────────────────────────────────────────────┤
│                                                        │
│  정의: 사용자가 늘수록 플랫폼의 가치가 증가하는 현상 │
│                                                        │
│  직접 네트워크 효과:                                  │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  사용자 ↑ → 다른 사용자에게 가치 ↑            │   │
│  │                                                │   │
│  │  예: 전화, SNS, 메신저                        │   │
│  │      친구가 많을수록 재미있음                 │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  간접 네트워크 효과 (양면 시장):                      │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  구매자 ↑ → 판매자 ↑ → 구매자 ↑              │   │
│  │                                                │   │
│  │  예: 앱스토어, 쇼핑몰                         │   │
│  │      사용자 많으면 앱 개발자 많이 옴          │   │
│  │      앱 많으면 사용자 많이 옴                 │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  네트워크 효과 그래프:                                │
│  ┌────────────────────────────────────────────────┐   │
│  │  가치                                         │   │
│  │    ↑                                          │   │
│  │    │              ╱                           │   │
│  │    │            ╱                             │   │
│  │    │          ╱                               │   │
│  │    │        ╱                                 │   │
│  │    │      ╱                                   │   │
│  │    │    ╱                                     │   │
│  │    │  ╱                                       │   │
│  │    └───────────────────────→ 사용자 수       │   │
│  │                                              │   │
│  │  메트칼프의 법칙: 가치 ∝ 사용자²             │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

> **※ 참고 표준**: 해당 기술 관련 NIST / ISO / IEEE / 과기정통부 가이드라인
