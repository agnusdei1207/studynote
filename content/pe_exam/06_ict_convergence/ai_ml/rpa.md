# RPA (Robotic Process Automation)

## 핵심 인사이트 (3줄 요약)
> **소프트웨어 로봇으로 반복 업무 자동화**. 사람처럼 컴퓨터 조작. 업무 효율 극대화.

---

## 📝 기술사 모의답안 (2.5페이지 분량)

### 📌 예상 문제
> **"RPA (Robotic Process Automation)의 개념과 핵심 원리를 설명하고, 비교 분석 및 실무 적용 방안을 기술하시오."**

---

### Ⅰ. 개요

#### 1. 개념
RPA(Robotic Process Automation)는 **사람이 하던 규칙적이고 반복적인 컴퓨터 작업을 소프트웨어 로봇이 대신 수행하도록 하여 업무 효율을 높이는 기술**이다.

> 비유: "디지털 비서" - 지루한 일을 대신 해줘요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. RPA 동작 원리
```
┌────────────────────────────────────────────────────────┐
│           RPA 동작 과정                              │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 기록 (Recording):                                 │
│  ┌────────────────────────────────────────────────┐   │
│  │  👤 사용자 작업을 녹화                          │   │
│  │  • 클릭, 타이핑, 복사/붙여넣기                  │   │
│  │  • 앱 전환, 데이터 입력                        │   │
│  └────────────────────────────────────────────────┘   │
│                        ↓                              │
│  2. 정의 (Definition):                                │
│  ┌────────────────────────────────────────────────┐   │
│  │  📝 워크플로우 정의                             │   │
│  │  • 단계별 프로세스 구성                         │   │
│  │  • 조건, 반복, 예외 처리                       │   │
│  └────────────────────────────────────────────────┘   │
│                        ↓                              │
│  3. 실행 (Execution):                                 │
│  ┌────────────────────────────────────────────────┐   │
│  │  🤖 로봇이 자동 실행                            │   │
│  │  • 24/7 작업 가능                               │   │
│  │  • 오류 없이 일관되게                           │   │
│  └────────────────────────────────────────────────┘   │
│                        ↓                              │
│  4. 모니터링 (Monitoring):                            │
│  ┌────────────────────────────────────────────────┐   │
│  │  📊 실행 결과 모니터링                          │   │
│  │  • 성공/실패 추적                               │   │
│  │  • 성능 측정                                    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### Ⅲ. 기술 비교 분석

#### 5. 장단점
| 장점 | 단점 |
|-----|------|
| 비용 절감 | 초기 구현 비용 |
| 정확성 향상 | UI 변경 취약 |
| 24/7 작업 | 복잡한 판단 어려움 |
| 빠른 구현 | 유지보수 |

---

### Ⅳ. 실무 적용 방안

#### 3. RPA 적용 분야
| 분야 | 업무 | 효과 |
|-----|------|------|
| 재무 | 송장 처리, 결제 | 90% 시간 단축 |
| HR | 채용, 급여 | 오류 감소 |
| 고객센터 | 문의 응답 | 응답 속도 향상 |
| 물류 | 주문 처리 | 처리량 증가 |
| IT | 데이터 마이그레이션 | 자동화 |

---

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **비즈니스 혁신** | 디지털 전환 가속화 및 신규 비즈니스 모델 창출 | 시장 출시 시간(TTM) 50% 단축 |
| **운영 효율** | AI·자동화로 수작업 제거 및 의사결정 지원 강화 | 운영 비용 30~40% 절감 |
| **경쟁력 강화** | 최신 기술 도입으로 시장 경쟁 우위 확보 | 고객 만족도(CSAT) 20점 향상 |

#### 결론
> **RPA (Robotic Process Automation)**은(는) ICT 융합 기술은 AI-First 전략, 탄소 중립(Net Zero) 목표, EU AI Act 등 글로벌 규제 환경에 대응하면서 기술적 혁신과 사회적 책임을 동시에 실현하는 방향으로 발전하고 있다.

> **※ 참고 표준**: NIST AI RMF 1.0, EU AI Act(2024), ISO/IEC 42001(AI 관리 시스템), 과기정통부 AI 기본법
