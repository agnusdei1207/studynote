### Ⅰ. 개요

#### 1. 개념
이더리움(Ethereum)은 **스마트 계약(Smart Contract) 기능을 갖춘 탈중앙화 플랫폼**으로, 개발자가 블록체인 위에서 분산 애플리케이션(DApp)을 구축할 수 있게 한다.

> 비유: "세계 컴퓨터" - 누구나 프로그램을 실행할 수 있는 분산 컴퓨팅 플랫폼

#### 3. 이더리움 핵심 개념
```
┌────────────────────────────────────────────────────────┐
│               이더리움 핵심 개념                         │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 이더 (Ether, ETH)                                  │
│     - 이더리움의 기본 통화                              │
│     - 가스(Gas) 지불에 사용                            │
│     - 1 ETH = 10¹⁸ Wei                                │
│                                                        │
│  2. 계정 (Account)                                     │
│     ┌────────────────────────────────────────────┐    │
│     │ EOA (Externally Owned Account)             │    │
│     │ - 개인 키로 제어                            │    │
│     │ - 트랜잭션 시작 가능                        │    │
│     │                                            │    │
│     │ CA (Contract Account)                      │    │
│     │ - 스마트 계약 코드                          │    │
│     │ - EOA의 의해 트리거                        │    │
│     └────────────────────────────────────────────┘    │
│                                                        │
│  3. 가스 (Gas)                                         │
│     ┌────────────────────────────────────────────┐    │
│     │ 연산 비용을 나타내는 단위                    │    │
│     │                                            │    │
│     │ Gas Limit: 최대 가스 사용량                │    │
│     │ Gas Price: 가스당 가격 (Gwei)              │    │
│     │ Gas Used: 실제 사용한 가스                  │    │
│     │                                            │    │
│     │ 수수료 = Gas Used × Gas Price              │    │
│     └────────────────────────────────────────────┘    │
│                                                        │
│  4. EVM (Ethereum Virtual Machine)                    │
│     - 스마트 계약 실행 환경                            │
│     - 샌드박스 격리                                    │
│     - 모든 노드에서 동일 실행                          │
│                                                        │
│  5. 스마트 계약 (Smart Contract)                       │
│     - EVM에서 실행되는 프로그램                        │
│     - 솔리디티(Solidity)로 작성                       │
│     - 불변성 (한 번 배포하면 수정 불가)               │
│                                                        │
│  6. DApp (Decentralized App)                          │
│     - 탈중앙화 애플리케이션                            │
│     - 프론트엔드 + 스마트 계약                        │
│     - DeFi, NFT, DAO 등                              │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### Ⅲ. 기술 비교 분석

#### 2. 비트코인 vs 이더리움
```
┌────────────────────────────────────────────────────────┐
│              비트코인 vs 이더리움                        │
├────────────────────────────────────────────────────────┤
│                                                        │
│  비트코인 (Bitcoin 1.0):                               │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 디지털 화폐 (P2P 전자 화폐)                  │   │
│  │ • 가치 저장, 결제                              │   │
│  │ • 스크립트 언어 (제한적)                       │   │
│  │ • UTXO 모델                                    │   │
│  │ • 채굴: PoW (작업 증명)                        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  이더리움 (Ethereum 2.0):                              │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 프로그래밍 가능한 블록체인                    │   │
│  │ • 스마트 계약, DApp                            │   │
│  │ • 솔리디티 (Turing-complete)                   │   │
│  │ • Account 모델                                 │   │
│  │ • 채굴: PoW → PoS (지분 증명)                  │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  비교표:                                               │
│  ┌─────────────┬─────────────┬─────────────┐         │
│  │    구분      │  비트코인   │  이더리움   │         │
│  ├─────────────┼─────────────┼─────────────┤         │
│  │ 목적        │ 디지털 화폐 │ 플랫폼      │         │
│  │ 화폐        │ BTC        │ ETH        │         │
│  │ 프로그래밍   │ 제한적      │ 완전       │         │
│  │ 블록 시간   │ ~10분      │ ~15초      │         │
│  │ 합의        │ PoW        │ PoS        │         │
│  └─────────────┴─────────────┴─────────────┘         │
│                                                        │
└────────────────────────────────────────────────────────┘
```
