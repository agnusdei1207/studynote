### Ⅰ. 개요

#### 1. 개념
NFT(Non-Fungible Token)는 **블록체인상에서 고유성을 가지며 대체가 불가능한 디지털 자산을 나타내는 토큰**이다.

> 비유: "디지털 세상의 등기부등본" - 원본임을 증명해요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 3. NFT 표준
| 표준 | 블록체인 | 설명 |
|-----|---------|------|
| ERC-721 | 이더리움 | 기본 NFT 표준 |
| ERC-1155 | 이더리움 | 멀티 토큰 (FT+NFT) |
| BEP-721 | BSC | 바이낸스 NFT |
| SPL | 솔라나 | 솔라나 NFT |
| Flow | 플로우 | NBA Top Shot |

### Ⅲ. 기술 비교 분석

#### 2. 대체 가능 vs 불가능
```
┌────────────────────────────────────────────────────────┐
│           대체 가능 vs 대체 불가능                    │
├────────────────────────────────────────────────────────┤
│                                                        │
│  대체 가능 (Fungible):                                │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  비트코인, 이더리움, 달러                     │   │
│  │                                                │   │
│  │  💰 1 BTC = 💰 1 BTC                          │   │
│  │                                                │   │
│  │  • 동일한 가치                                 │   │
│  │  • 교환 가능                                   │   │
│  │  • 분할 가능                                   │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  대체 불가능 (Non-Fungible):                          │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  디지털 아트, 게임 아이템, 티켓               │   │
│  │                                                │   │
│  │  🎨 NFT A ≠ 🎨 NFT B                          │   │
│  │                                                │   │
│  │  • 고유한 가치                                 │   │
│  │  • 교환 불가능                                 │   │
│  │  • 분할 불가능                                 │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  NFT 특징:                                            │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  • 고유성: 각 NFT는 고유 ID 보유              │   │
│  │  • 희소성: 발행 수량 제한 가능                │   │
│  │  • 소유권: 블록체인으로 증명                  │   │
│  │  • 이력: 거래 내역 추적 가능                  │   │
│  │  • 프로그래밍: 로얄티, 조건 설정 가능         │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```
