# Web3

## 핵심 인사이트 (3줄 요약)
> **탈중앙화된 차세대 인터넷**. 블록체인 기반. 사용자가 데이터와 자산 소유.

---

## 📝 기술사 모의답안 (2.5페이지 분량)

### 📌 예상 문제
> **"Web3의 개념과 핵심 원리를 설명하고, 비교 분석 및 실무 적용 방안을 기술하시오."**

---

### Ⅰ. 개요

#### 1. 개념
Web3는 **블록체인과 탈중앙화 기술을 기반으로 사용자가 자신의 데이터와 디지털 자산을 직접 소유하고 통제하는 차세대 분산형 인터넷 아키텍처**이다.

> 비유: "내 집 마련" - 빌려 쓰던 내 방을 이제 내가 가져요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. Web의 진화
```
┌────────────────────────────────────────────────────────┐
│           Web의 진화                                  │
├────────────────────────────────────────────────────────┤
│                                                        │
│  Web1 (1990-2005): 읽기 전용                           │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  📖 정적 웹사이트                               │   │
│  │  • 사용자: 정보 소비만                          │   │
│  │  • 콘텐츠: 기관이 생성                          │   │
│  │  • 예: 뉴스 사이트, 포털                        │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  Web2 (2005-현재): 읽기/쓰기                           │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  ✏️ 소셜 웹                                     │   │
│  │  • 사용자: 콘텐츠 생성/공유                     │   │
│  │  • 플랫폼: 중앙 집중 (Facebook, Google)         │   │
│  │  • 문제: 데이터는 플랫폼 소유                   │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  Web3 (미래): 읽기/쓰기/소유                           │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  🔐 탈중앙화 웹                                 │   │
│  │  • 사용자: 데이터와 자산 소유                   │   │
│  │  • 플랫폼: 분산형 (블록체인)                    │   │
│  │  • 특징: 토큰 경제, DAO, DeFi                  │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. Web3 핵심 기술
| 기술 | 설명 |
|-----|------|
| 블록체인 | 분산 원장 |
| 스마트 컨트랙트 | 자동 실행 계약 |
| 암호화폐 | 디지털 화폐 |
| NFT | 대체 불가능 토큰 |
| DAO | 탈중앙화 자율 조직 |
| DApp | 탈중앙화 앱 |
| 지갑 | 디지털 자산 관리 |

### Ⅲ. 기술 비교 분석

#### 4. Web2 vs Web3
| 특징 | Web2 | Web3 |
|-----|------|------|
| 데이터 소유 | 플랫폼 | 사용자 |
| 신원 | 이메일/비밀번호 | 지갑 주소 |
| 결제 | 신용카드 | 암호화폐 |
| 백엔드 | 중앙 서버 | 스마트 컨트랙트 |
| 거버넌스 | 회사 | DAO |

#### 5. 장단점
| 장점 | 단점 |
|-----|------|
| 사용자 주권 | 복잡성 |
| 검열 저항성 | 확장성 |
| 투명성 | 사용자 경험 |
| 인센티브 | 보안 책임 |

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **비즈니스 혁신** | 디지털 전환 가속화 및 신규 비즈니스 모델 창출 | 시장 출시 시간(TTM) 50% 단축 |
