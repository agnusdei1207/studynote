# IDS/IPS (침입 탐지/방지 시스템)

## 핵심 인사이트 (3줄 요약)
> **IDS는 침입을 탐지하고 IPS는 탐지 후 차단까지 수행**. 네트워크 보안의 핵심 장비. 탐지 방식은 시그니처 기반과 이상 행위 기반.

---

### 📌 예상 문제

---

### Ⅰ. 개요

#### 1. 개념
- **IDS (Intrusion Detection System)**: 네트워크나 시스템의 침입 시도를 탐지하고 알리는 시스템
- **IPS (Intrusion Prevention System)**: 침입을 탐지하고 자동으로 차단까지 수행하는 시스템

> 비유: IDS = "CCTV (감시만)", IPS = "경비원 (감시 + 제지)"

#### 4. 탐지 방식
| 방식 | 원리 | 장점 | 단점 |
|-----|------|------|------|
| 시그니처 기반 | 알려진 공격 패턴 매칭 | 정확도 높음 | 미지 공격 탐지 불가 |
| 이상 행위 기반 | 정상 패턴 학습 후 이상 탐지 | 미지 공격 탐지 | 오탐 높음 |
| 하이브리드 | 두 방식 결합 | 장점 결합 | 복잡度高 |

#### 6. 주요 공격 탐지
| 공격 유형 | 탐지 방법 |
|----------|----------|
| DDoS | 트래픽 급증, 출발지 분산 |
| 포트 스캔 | 다수 포트 접속 시도 |
| SQL 인젝션 | 특수 문자 패턴 |
| 악성코드 | 시그니처, 행위 분석 |
| 무차별 대입 | 로그인 실패 반복 |
| C&C 통신 | 알려진 악성 IP/도메인 |
