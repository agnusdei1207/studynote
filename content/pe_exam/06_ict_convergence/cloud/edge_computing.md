# 엣지 컴퓨팅 (Edge Computing)

### Ⅰ. 개요

#### 1. 개념
엣지 컴퓨팅(Edge Computing)은 **데이터를 중앙 클라우드가 아닌 데이터 발생 지점(엣지) 근처에서 처리하는 분산 컴퓨팅 패러다임으로, 지연 시간을 줄이고 대역폭을 절약하며 실시간 처리를 가능하게 한다**.

> 비유: "지점 은행" - 모든 업무를 본점까지 가서 하지 않아요

│  └────────────────────────────────────────────────┘   │

│  🌫️ 포그 (Fog)                                       │

│  └────────────────────────────────────────────────┘   │
│                        ↑                               │

│  │  • 엔드포인트                                    │   │
│  │  • 매우 제한된 리소스                          │   │
│  │  • 지연: <1ms                                   │   │
