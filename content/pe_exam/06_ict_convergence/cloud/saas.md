# SaaS (Software as a Service)

## 핵심 인사이트 (3줄 요약)

### Ⅰ. 개요

#### 1. 개념
SaaS(Software as a Service)는 **소프트웨어를 사용자의 컴퓨터에 설치하지 않고, 클라우드 서버에서 실행하여 인터넷을 통해 서비스 형태로 제공**하는 방식이다.

> 비유: "소프트웨어를 사는 게 아니라 빌려 쓰는 것" - 넷플릭스처럼 이용료 내고 사용

---

#### 3. SaaS 특징
| 특징 | 설명 |
|-----|------|
| 접근성 | 인터넷만 있으면 어디서든 사용 |
| 설치 불필요 | 브라우저로 바로 사용 |
| 자동 업데이트 | 항상 최신 버전 |
| 구독 과금 | 사용한 만큼 비용 지불 |
| 확장성 | 필요에 따라 사용자 추가 |
| 유지보수 | 공급업체가 담당 |

### Ⅲ. 기술 비교 분석

#### 5. SaaS 장단점
| 장점 | 설명 |
|-----|------|
| 초기 비용 낮음 | 구독료만 지불 |
| 빠른 도입 | 즉시 사용 가능 |
| 접근성 | 언제 어디서나 |
| 자동 업데이트 | 항상 최신 버전 |
| 유연성 | 사용자 수 조절 |

| 단점 | 설명 |
|-----|------|
| 인터넷 의존 | 연결 필수 |
| 데이터 보안 | 외부 서버 저장 |
| 커스터마이징 제한 | 표준 기능만 |
| 장기 비용 | 구독 비용 지속 |
| 벤더 락인 | 이관 어려움 |

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **비즈니스 혁신** | 디지털 전환 가속화 및 신규 비즈니스 모델 창출 | 시장 출시 시간(TTM) 50% 단축 |
| **운영 효율** | AI·자동화로 수작업 제거 및 의사결정 지원 강화 | 운영 비용 30~40% 절감 |
| **경쟁력 강화** | 최신 기술 도입으로 시장 경쟁 우위 확보 | 고객 만족도(CSAT) 20점 향상 |

#### 결론
> **SaaS (Software as a Service)**은(는) ICT 융합 기술은 AI-First 전략, 탄소 중립(Net Zero) 목표, EU AI Act 등 글로벌 규제 환경에 대응하면서 기술적 혁신과 사회적 책임을 동시에 실현하는 방향으로 발전하고 있다.

> **※ 참고 표준**: NIST AI RMF 1.0, EU AI Act(2024), ISO/IEC 42001(AI 관리 시스템), 과기정통부 AI 기본법
