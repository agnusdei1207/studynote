### Ⅰ. 개요

#### 1. 개념
사물인터넷(Internet of Things, IoT)은 **각종 사물에 센서와 통신 기능을 탑재하여 인터넷으로 연결하고, 수집된 데이터를 분석해 지능형 서비스를 제공하는 기술**이다.

> 비유: "사물들이 대화해요" - 냉장고가 우유 없다고 알려줘요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. IoT 아키텍처
```
┌────────────────────────────────────────────────────────┐
│           IoT 4단계 아키텍처                           │
├────────────────────────────────────────────────────────┤
│                                                        │
│  4단계: 응용 계층 (Application Layer)                 │
│  ┌────────────────────────────────────────────────┐   │
│  │  스마트홈 │ 스마트시티 │ 스마트팩토리 │ 헬스케어│   │
│  └────────────────────────────────────────────────┘   │
│                        ↑                              │
│  3단계: 처리 계층 (Processing Layer)                  │
│  ┌────────────────────────────────────────────────┐   │
│  │  클라우드 │ 빅데이터 │ AI/ML │ API            │   │
│  └────────────────────────────────────────────────┘   │
│                        ↑                              │
│  2단계: 네트워크 계층 (Network Layer)                 │
│  ┌────────────────────────────────────────────────┐   │
│  │  WiFi │ BLE │ Zigbee │ LoRa │ 5G │ MQTT       │   │
│  └────────────────────────────────────────────────┘   │
│                        ↑                              │
│  1단계: 디바이스 계층 (Device Layer)                  │
│  ┌────────────────────────────────────────────────┐   │
│  │  센서 │ 액추에이터 │ 게이트웨이 │ 임베디드    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. IoT 구성 요소
| 구성 요소 | 설명 | 예시 |
|----------|------|------|
| 센서 | 데이터 수집 | 온도, 습도, 가속도 |
| 액추에이터 | 물리적 동작 | 모터, 밸브 |
| 게이트웨이 | 데이터 중계 | 라우터, 허브 |
| 통신망 | 데이터 전송 | WiFi, BLE, 5G |
| 클라우드 | 데이터 처리 | AWS IoT, Azure IoT |

#### 4. IoT 통신 프로토콜
| 프로토콜 | 특징 | 용도 |
|---------|------|------|
| MQTT | 경량, 발행/구독 | 센서 데이터 |
| CoAP | UDP 기반, REST | 제약 장치 |
| HTTP/REST | 범용, 무겁음 | 웹 연동 |
| AMQP | 신뢰성, 메시지 큐 | 엔터프라이즈 |
| WebSocket | 양방향, 실시간 | 스트리밍 |

#### 5. IoT 통신 기술
| 기술 | 범위 | 전송속도 | 용도 |
|-----|------|---------|------|
| WiFi | 100m | 1Gbps | 홈/사무실 |
| BLE | 100m | 2Mbps | 웨어러블 |
| Zigbee | 100m | 250Kbps | 스마트홈 |
| LoRa | 10km | 50Kbps | 스마트시티 |
| 5G | 1km | 10Gbps | 자율주행 |

#### 6. IoT 보안 이슈
| 위협 | 설명 | 대책 |
|-----|------|------|
| 기기 탈취 | 물리적 공격 | TPM, 보안 부팅 |
| 통신 도청 | 데이터 가로채기 | 암호화 (TLS) |
| DDoS | 봇넷 공격 | 펌웨어 업데이트 |
| 인증 우회 | 기본 비밀번호 | 강력한 인증 |

### Ⅳ. 실무 적용 방안

#### 7. 활용 분야
| 분야 | 사례 |
|-----|------|
| 스마트홈 | 조명, 난방, 가전 제어 |
| 스마트시티 | 교통, 환경, 에너지 관리 |
| 스마트팩토리 | 설비 모니터링, 예지보전 |
| 스마트헬스 | 원격 진료, 웨어러블 |
| 스마트농업 | 온실 제어, 가축 관리 |

#### 10. 실무에선? (기술사적 판단)
**구축 시 고려사항:**
- 통신 프로토콜 선정 (전력/거리/속도)
- 보안 설계 (기기 인증, 암호화)
- 확장성 (수만~수백만 디바이스)

**표준화 동향:**
- oneM2M: 글로벌 IoT 표준
- OCF: 오픈 커넥티비티 재단
- Thread: 스마트홈 네트워크

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **비즈니스 혁신** | 디지털 전환 가속화 및 신규 비즈니스 모델 창출 | 시장 출시 시간(TTM) 50% 단축 |

| **경쟁력 강화** | 최신 기술 도입으로 시장 경쟁 우위 확보 | 고객 만족도(CSAT) 20점 향상 |
