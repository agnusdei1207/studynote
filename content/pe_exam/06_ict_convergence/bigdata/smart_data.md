# 스마트데이터 (Smart Data)

## 핵심 인사이트 (3줄 요약)
> **분석과 활용에 최적화된 고품질 데이터**. 빅데이터에서 가치 있는 정보만 추출. 의사결정과 AI 학습의 핵심 자원.

### Ⅰ. 개요

#### 1. 개념
스마트데이터(Smart Data)는 **빅데이터에서 의미 있는 패턴과 인사이트를 추출할 수 있도록 정제, 구조화, 강화된 고품질 데이터**를 말한다.

> 비유: "원석에서 다이아몬드만 골라내기" - 쓸모 있는 데이터만 남겨요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 3. 스마트데이터 특성
| 특성 | 설명 |
|-----|------|
| 맥락성 | 데이터의 출처, 시간, 의미 포함 |
| 구조화 | 분석 가능한 형태로 조직화 |
| 연관성 | 다른 데이터와의 관계 정의 |
| 품질 | 정확성, 완전성, 일관성 확보 |
| 실시간 | 즉시 활용 가능한 상태 |
| 접근성 | 쉽게 검색하고 사용 가능 |

│  │                     ↓                         │   │
│  │  6. 서비스 (Serving)                           │   │
