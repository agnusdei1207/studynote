### Ⅰ. 개요

#### 1. 개념
지능형 전력망(Smart Grid)은 **전력망에 ICT 기술을 접목하여 전력 생산, 송배전, 소비를 지능적으로 관리**하는 차세대 전력망이다.

> 비유: "전력의 인터넷" - 전력과 정보가 양방향으로 흐르는 시스템

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 3. 스마트 그리드 구성요소
```

│               스마트 그리드 구성요소                     │

│     │ • PLC (전력선통신)                         │    │
│     │ • Zigbee, WiFi, LTE, 5G                   │    │
```

### Ⅲ. 기술 비교 분석

#### 2. 기존 전력망 vs 스마트 그리드
```
┌────────────────────────────────────────────────────────┐
│           기존 전력망 vs 스마트 그리드                   │
├────────────────────────────────────────────────────────┤
│                                                        │
│  기존 전력망 (Traditional Grid):                       │
│  ┌────────────────────────────────────────────┐       │
│  │                                            │       │
│  │  발전소 → 송전 → 배전 → 소비자             │       │
│  │   ↑                              │         │       │
│  │   └──────────────────────────────┘         │       │
│  │         (단방향 전력 흐름)                  │       │
│  │                                            │       │
│  │  • 수동 모니터링                            │       │
│  │  • 일방향 통신                              │       │
│  │  • 제한적인 제어                            │       │
│  │  • 신재생에너지 통합 어려움                 │       │
│  └────────────────────────────────────────────┘       │
│                                                        │
│  스마트 그리드 (Smart Grid):                            │
│  ┌────────────────────────────────────────────┐       │
│  │                                            │       │
│  │  발전소 ←→ 송전 ←→ 배전 ←→ 소비자         │       │
│  │     ↑        ↑        ↑        ↑          │       │
│  │     └────────┴────────┴────────┘          │       │
│  │         (양방향 전력 + 정보 흐름)          │       │
│  │                                            │       │
│  │  • 실시간 모니터링                          │       │
│  │  • 양방향 통신                              │       │
│  │  • 자동화된 제어                            │       │
│  │  • 분산 전원 통합                           │       │
│  └────────────────────────────────────────────┘       │
│                                                        │
└────────────────────────────────────────────────────────┘
```
