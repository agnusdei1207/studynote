# 라이다 (LiDAR)

### Ⅰ. 개요

#### 1. 개념
라이다(LiDAR, Light Detection and Ranging)는 **레이저 광선을 송출하고 반사되어 돌아오는 시간을 측정하여 거리와 형상을 파악**하는 기술이다.

> 비유: "레이저 눈" - 어둠 속에서도 3D로 세상을 본다

#### 3. 라이다 유형
```
┌────────────────────────────────────────────────────────┐
│                  라이다 유형                            │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 기계식 (Mechanical)                                │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 회전하는 센서/미러                           │   │
│  │ • 360° 수평 스캔                              │   │
│  │ • 다중 채널 (16, 32, 64, 128채널)             │   │
│  │                                                │   │
│  │ 예: Velodyne, Ouster, Hesai                   │   │
│  │ 장점: 넓은 범위, 높은 정확도                   │   │
│  │ 단점: 크기, 가격, 기계적 마모                  │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. MEMS (Micro-Electro-Mechanical System)            │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 초소형 거울로 빔 조향                        │   │
│  │ • 빠른 스캔, 저비용 잠재                      │   │
│  │                                                │   │
│  │ 예: Innoviz, Ibeo                             │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3. OPA (Optical Phased Array)                        │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 위상 배열로 빔 조향 (움직이는 부품 없음)    │   │
│  │ • 초고속 스캔                                 │   │
│  │                                                │   │
│  │ 예: Quanergy, Aurora                          │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  4. Flash LiDAR                                       │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 넓은 영역 한 번에 조사                       │   │
│  │ • 2D 배열 센서                                │   │
│  │                                                │   │
│  │ 예: LeddarTech, Continental                   │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  5. FMCW (Frequency Modulated Continuous Wave)        │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 연속파 변조 방식                             │   │
│  │ • 거리 + 속도 측정 (도플러 효과)              │   │
│  │ • 간섭 저항                                   │   │
│  │                                                │   │
│  │ 예: Aeva, Aurora                              │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 4. 주요 사양
```
┌────────────────────────────────────────────────────────┐
│                   주요 사양                             │
├────────────────────────────────────────────────────────┤
│                                                        │
│  ┌─────────────────────────────────────────────────┐  │

│  ├─────────────────┼────────────────────┼──────────┤  │
│  │ 측정 거리       │ 0.1m ~ 300m        │ 물체 감지│  │

│  │ 시야각(수평)    │ 120° ~ 360°        │ 커버리지 │  │
│  │ 시야각(수직)    │ 20° ~ 40°          │ 높이 범위│  │
│  │ 해상도          │ 0.1° ~ 0.5°        │ 각도 정밀│  │
│  │ 스캔 속도       │ 10~30 Hz           │ 초당 프레임│  │
│  │ 파장            │ 905nm / 1550nm     │ 안전/거리│  │
│  │ 포인트 수       │ 10만~200만/초      │ 데이터량 │  │

│                                                        │
│  파장 비교:                                            │
│  • 905nm: 저비용, 눈 안전 제한, 단거리                │
│  • 1550nm: 고비용, 더 안전, 장거리 가능               │
│                                                        │
└────────────────────────────────────────────────────────┘
```
