### Ⅰ. 개요

#### 1. 개념
Demand Response(수요 응답)는 **전력 공급이 부족하거나 가격이 높을 때, 소비자가 전력 사용을 줄이거나 시점을 변경하여 전력망 안정화에 참여하고 보상을 받는 시스템**이다.

> 비유: "붐비는 시간엔 식당 피하기" - 비쌀 때는 전기를 덜 써요

### Ⅱ. 구성 요소 및 핵심 원리

#### 3. 수요 응답 유형
```
┌────────────────────────────────────────────────────────┐
│           수요 응답 유형                              │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 가격 기반 (Price-Based):                         │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  전기 요금 변동에 따른 자발적 참여            │   │
│  │                                                │   │
│  │  요금                                          │   │
│  │   ↑     ╱╲                                     │   │
│  │   │    ╱  ╲   피크 요금                       │   │
│  │   │   ╱    ╲╱╲                                │   │
│  │   │  ╱        ╲                               │   │
│  │   └──────────────→ 시간                       │   │
│  │                                                │   │
│  │  • TOU (Time-of-Use): 시간대별 요금          │   │
│  │  • CPP (Critical Peak Pricing): 피크 할증    │   │
│  │  • RTP (Real-Time Pricing): 실시간 요금      │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 인센티브 기반 (Incentive-Based):                  │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  참여에 대한 보상 지급                        │   │
│  │                                                │   │
│  │  • DLC (Direct Load Control): 원격 제어       │   │
│  │  • Interruptible: 중단 가능 요금 할인        │   │
│  │  • Capacity Bidding: 용량 입찰               │   │
│  │                                                │   │
│  │  예: 에어컨 원격 끄기 → 할인 받음            │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```
