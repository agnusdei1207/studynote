### Ⅰ. 개요

#### 1. 개념
ESS(Energy Storage System)는 **전기 에너지를 배터리 등에 저장했다가, 필요한 시점에 방전하여 전력을 공급하는 시스템**이다.

> 비유: "거대한 충전 보조배터리" - 싸게 사서 비쌀 때 써요

---

#### 3. ESS 활용 방식
| 방식 | 설명 | 효과 |
|-----|------|------|
| 피크 쉐이빙 | 전력 소비 많을 때 방전 | 전기요금 절감 |
| 부하 이동 | 싼 시간에 충전, 비쌀 때 방전 | 비용 절감 |
| 정전 대비 | 무정전 전원 공급 | 안정성 확보 |
| 신재생 연계 | 태양광/풍력 저장 | 간헐성 해결 |
| 주파수 조정 | 전력망 주파수 안정화 | 계통 안정 |

│  │  └─────────────────┘                          │   │
│  │  예: 테슬라 Powerwall, LG ESS                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │  상업용 (100 kWh ~ 1 MWh):                    │   │
│  │  ┌─────────────────┐                          │   │
│  │  │ 🏢 빌딩/공장    │                          │   │

│  │  예: 빌딩, 쇼핑몰, 데이터센터                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │  산업용/계통용 (1 MWh ~ 100+ MWh):            │   │
│  │  ┌─────────────────┐                          │   │
│  │  │ ⚡ 발전소급     │                          │   │
│  │  │ 🔋 대용량 ESS   │ → 전력망 안정화         │   │
│  │  └─────────────────┘                          │   │
│  │  예: 한국전력 ESS, 테슬라 호ornsdale         │   │

└────────────────────────────────────────────────────────┘
