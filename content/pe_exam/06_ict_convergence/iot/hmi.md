### Ⅰ. 개요

#### 1. 개념
HMI(Human-Machine Interface)는 **사람과 기계/시스템 사이에서 정보를 교환하고 제어할 수 있게 하는 하드웨어와 소프트웨어의 결합체**이다.

> 비유: "사람과 기계의 통역사" - 내 말을 기계가 알아듣게, 기계 말을 내가 알아듣게

---

#### 3. HMI 유형
| 유형 | 입력 방식 | 예시 |
|-----|----------|------|
| 물리적 | 버튼, 스위치, 다이얼 | 공장 제어반 |
| 터치 | 터치스크린 | 스마트폰, 키오스크 |
| 음성 | 음성 인식 | 스마트 스피커, 내비게이션 |
| 제스처 | 손 동작 인식 | 스마트 TV, 게임기 |
| 시선 | 안구 추적 | 스티브 아이즈 |
| BCI | 뇌파 | 의료 재활 |

#### 5. HMI 설계 원칙
| 원칙 | 설명 |
|-----|------|
| 단순성 | 복잡함 최소화 |
| 일관성 | 동일한 동작은 동일한 결과 |
| 피드백 | 즉각적인 반응 제공 |
| 오류 방지 | 실수하기 어렵게 설계 |
| 접근성 | 모든 사용자 고려 |
| 가독성 | 명확한 정보 표시 |
