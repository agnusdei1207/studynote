# 오감센서 (Multi-Sensor)

## 핵심 인사이트 (3줄 요약)
> **인간의 오감을 모방한 다중 센서 시스템**. 시각, 청각, 후각, 미각, 촉각 데이터 수집. IoT, 로봇, 스마트 홈의 핵심 입력 장치.

### 📌 예상 문제

### Ⅰ. 개요

#### 1. 개념
오감센서는 **인간의 시각, 청각, 후각, 미각, 촉각에 대응하는 다양한 센서를 통합**하여 환경을 종합적으로 인식하는 시스템이다.

> 비유: "로봇의 감각 기관" - 기계가 세상을 느끼는 방식

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 오감별 센서
```
┌────────────────────────────────────────────────────────┐
│                오감별 센서 기술                         │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1. 시각 (Vision)                                      │
│  ┌────────────────────────────────────────────────┐   │
│  │ • CCD/CMOS 카메라: 이미지 캡처                │   │
│  │ • 적외선 카메라: 열화상, 야간                  │   │
│  │ • 라이다(LiDAR): 3D 거리 측정                  │   │
│  │ • ToF(Time of Flight): 깊이 감지               │   │
│  │ • 스테레오 비전: 거리 인식                      │   │
│  │                                                │   │
│  │ 응용: 얼굴 인식, 물체 감지, OCR               │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2. 청각 (Auditory)                                    │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 마이크: 소리 수집                            │   │
│  │ • 초음파 센서: 거리 측정 (20kHz 이상)          │   │
│  │ • 가청 센서: 음성 인식, 소음 감지              │   │
│  │                                                │   │
│  │ 응용: 음성 명령, 화자 인식, 이상 소리 감지    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3. 후각 (Olfactory)                                   │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 전자코 (E-nose): 냄새 감지                   │   │
│  │ • 가스 센서: CO, CO2, VOC, 메탄                │   │
│  │ • 공기질 센서: 미세먼지, 습도                  │   │
│  │                                                │   │
│  │ 응용: 가스 누출, 화재 감지, 식품 신선도       │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  4. 미각 (Gustatory)                                   │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 전자혀 (E-tongue): 맛 감지                   │   │
│  │ • pH 센서: 산도 측정                           │   │
│  │ • 전도도 센서: 염분, 당도                      │   │
│  │                                                │   │
│  │ 응용: 식품 품질, 수질 검사, 의약품            │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  5. 촉각 (Tactile)                                     │
│  ┌────────────────────────────────────────────────┐   │
│  │ • 압력 센서: 터치, 무게                        │   │
│  │ • 온도 센서: 열감지                            │   │
│  │ • 진동 센서: 움직임                            │   │
│  │ • 변형 센서: 굽힘, 스트레인                    │   │
│  │ • 슬립 센서: 미끄러짐                          │   │
│  │                                                │   │
│  │ 응용: 로봇 손, 스마트 스킨, 웨어러블          │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 센서 퓨전
```
┌────────────────────────────────────────────────────────┐
│                  센서 퓨전 (Sensor Fusion)              │
├────────────────────────────────────────────────────────┤
│                                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │   시각      청각      후각      촉각          │   │
│  │    ↓         ↓         ↓         ↓            │   │
│  │  ┌────────────────────────────────────────┐   │   │
│  │  │           센서 퓨전 엔진               │   │   │
│  │  │  • 데이터 정규화                        │   │   │
│  │  │  • 시간 동기화                          │   │   │
│  │  │  • 좌표 변환                            │   │   │
│  │  │  • 확률적 융합 (칼만 필터 등)          │   │   │
│  │  └────────────────────────────────────────┘   │   │
│  │                    ↓                          │   │
│  │  ┌────────────────────────────────────────┐   │   │
│  │  │         통합 환경 인식 결과            │   │   │
│  │  │  • 객체 식별 및 위치                    │   │   │
│  │  │  • 상황 판단                            │   │   │
│  │  │  • 이상 감지                            │   │   │
│  │  └────────────────────────────────────────┘   │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  퓨전 레벨:                                            │
│  ┌────────────────────────────────────────────────┐   │
│  │ • Low-level: 원시 데이터 결합                  │   │
│  │ • Feature-level: 특징 추출 후 결합             │   │
│  │ • Decision-level: 개별 판단 후 결합            │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```
