### Ⅰ. 개요

#### 1. 개념
측위 기술(Positioning Technology)은 **위성, 무선 신호, 센서 등을 활용하여 대상의 정확한 위치(좌표)를 파악하는 기술**이다.

> 비유: "전자 지도에서 내 위치 찾기" - 여러 신호로 "여기가 어디인지" 알아요

#### 3. 측위 방식
| 방식 | 원리 | 정확도 | 용도 |
|-----|------|-------|------|
| GPS | 위성 신호 시간차 | 3~10m | 야외 내비 |
| Wi-Fi | RSSI 핑거프린팅 | 3~5m | 실내 |
| BLE | 비콘 근접 감지 | 1~3m | 상점, 박물관 |
| UWB | ToF/삼변측량 | 10~30cm | 자산 추적 |
| Cell ID | 기지국 ID | 100m~수km | 대략적 위치 |
| Dead Reckoning | 관성 센서 | 누적 오차 | 보조용 |

#### 4. 삼변측량 원리
```
┌────────────────────────────────────────────────────────┐
│           삼변측량 (Trilateration)                    │
├────────────────────────────────────────────────────────┤
│                                                        │
│  원리: 3개 이상의 기지국과의 거리로 위치 계산        │
│                                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │         📡 A (AP1)                             │   │
│  │           ╲                                     │   │
│  │      거리a ╲                                    │   │
│  │              ╲                                  │   │
│  │                📱                               │   │
│  │              ╱   ╲                              │   │
│  │    거리b   ╱       ╲  거리c                    │   │
│  │          ╱           ╲                          │   │
│  │   📡 B (AP2) ───────── 📡 C (AP3)              │   │
│  │                                                │   │
│  │  세 원의 교점 = 내 위치                        │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  RSSI (수신 신호 강도) 방식:                          │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  신호 강도 = K / 거리^n                        │   │
│  │  거리 = (K / RSSI)^(1/n)                       │   │
│  │                                                │   │
│  │  RSSI가 강할수록 → 가까움                     │   │
│  │  RSSI가 약할수록 → 멂                         │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  TDOA (도달 시간차) 방식:                             │
│  ┌────────────────────────────────────────────────┐   │
│  │                                                │   │
│  │  신호가 각 AP에 도달하는 시간 차이 측정       │   │
│  │                                                │   │
│  │  t1 - t2 = (거리1 - 거리2) / 빛의속도        │   │
│  │                                                │   │
│  │  더 정밀하지만 시간 동기화 필요               │   │
│  │                                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 7. 장단점
| 장점 | 단점 |
|-----|------|
| 위치 기반 서비스 | 실내 정확도 낮음 |
| 자산 관리 | 프라이버시 이슈 |
| 안전/보안 | 배터리 소모 |
| 편의성 | 인프라 비용 |
