# 분산 분석 (ANOVA) - 여러 집단 간 평균 차이 검정

## 핵심 인사이트 (3줄 요약)
> **3개 이상의 집단 간 평균 차이를 한 번에 검정**하는 통계 기법. 집단 간 분산과 집단 내 분산의 비율(F-값)을 이용한다. t-검정을 여러 번 할 때 발생하는 제1종 오류의 누적(Inflation) 문제를 해결한다.

---

## 📝 기술사 모의답안 (2.5페이지 분량)

### 📌 예상 문제
> **"분산 분석 (ANOVA) - 여러 집단 간 평균 차이 검정의 원리와 동작 과정을 설명하고, 유사 알고리즘·기법과 비교하여 적합한 활용 시나리오를 기술하시오."**

---

### Ⅰ. 개요

#### 1. 개념 및 필요성
```
왜 t-검정을 3번 안 하고 ANOVA를 하나요?
- 3개 집단(A, B, C) 비교 시: (A-B), (B-C), (A-C) 총 3번의 t-검정 필요
- 각 검정의 유의수준이 α=0.05라면:
  전체 실험에서 오류가 없을 확률 = (1-0.05)^3 ≈ 0.857
  전체 제1종 오류 발생 확률 = 1 - 0.857 = 0.143 (약 14%)
  → 5%보다 훨씬 커지는 '오류의 인플레이션' 발생!

해결: ANOVA (Analysis of Variance)
- "전체적으로 차이가 있는가?"를 한 번의 검정으로 판단
```

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. ANOVA의 기본 원리 (F-검정) ★ (중요)
```
F-통계량 = (집단 간 분산, SSB) / (집단 내 분산, SSW)

1. 집단 간 분산 (Between-group Variance):
   - 각 집단의 평균들이 전체 평균으로부터 얼마나 떨어져 있는가?
   - "우리가 찾고 싶은 진짜 차이"

2. 집단 내 분산 (Within-group Variance):
   - 각 집단 내부에서 개별 데이터들이 자기 집단 평균으로부터 얼마나 떨어져 있는가?
   - "측정 오차나 개별 특성에 따른 우연한 차이"

결론:
- F-값이 크다 = (집단 간 차이) >> (집단 내 오차)
- F-값이 크면 p-value가 낮아지고, "적어도 한 집단은 평균이 다르다"고 결론 내림
```

#### 3. ANOVA의 3가지 가정 (수행 조건)
1. **정규성 (Normality)**: 각 집단의 데이터는 정규분포를 따라야 함
2. **등분산성 (Homogeneity of Variance)**: 각 집단의 분산은 서로 같아야 함 (Levene 검정 등으로 확인)
3. **독립성 (Independence)**: 각 표본은 서로 독립적으로 추출되어야 함

#### 5. 사후 검정 (Post-hoc Test) ★ (필수)
ANOVA 결과가 유의미(p < 0.05)하다면, "어떤 집단끼리 차이가 나는지" 구체적으로 알아내야 함.

1. **Tukey HSD**: 모든 집단 쌍을 일대일 비교 (가장 널리 쓰임)
2. **Scheffé**: 가장 보수적이고 엄격한 방법 (표본 크기 달라도 가능)
3. **Duncan**: 집단 간 차이를 발견하기 가장 쉬운 (너그러운) 방법
4. **Bonferroni**: t-검정의 유의수준을 수동으로 보정 (α/검정 횟수)

#### 6. ANOVA 표 (ANOVA Table) 구조
| 요인 (Source) | 자유도 (df) | 제곱합 (SS) | 평균제곱 (MS) | F-값 |
|--------------|-----------|------------|--------------|------|
| 집단 간 (Between) | k - 1 | SSB | MSB = SSB / dfB | **MSB / MSW** |
| 집단 내 (Within) | N - k | SSW | MSW = SSW / dfW | |
| 전체 (Total) | N - 1 | SST | | |

*k: 집단 수, N: 전체 데이터 수*

---

### Ⅲ. 기술 비교 분석

#### 4. 일원 분산 분석 (One-way ANOVA) vs 이원 분산 분석 (Two-way ANOVA)
| 구분 | 일원 분산 분석 (One-way) | 이원 분산 분석 (Two-way) |
|------|-----------------------|-----------------------|
| 독립변수 개수 | 1개 (예: 교육 방법 A, B, C) | 2개 (예: 교육 방법 + 성별) |
| 분석 내용 | 교육 방법에 따른 성적 차이 | 교육 방법, 성별, 그리고 **상호작용** 효과 |
| 예시 | 비타민 종류에 따른 피로도 차이 | 비타민 종류와 복용 시간대에 따른 차이 |

---

### Ⅳ. 실무 적용 방안

#### 7. 실무 활용 사례
- **마케팅**: 광고 시안 A, B, C에 따른 클릭률 차이 분석
- **제조**: 기계 설정값 1, 2, 3에 따른 생산 제품 수율 차이
- **SW공학**: 개발 방법론(Agile, Waterfall, DevSecOps)에 따른 결함 밀도 차이
- **기술사 포인트**: t-검정과의 차이(오류 인플레이션), F-값의 의미(간/내 비율), 사후 검정 종류
