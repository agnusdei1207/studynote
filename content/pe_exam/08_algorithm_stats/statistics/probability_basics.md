### Ⅰ. 개요

#### 1. 개념
확률(Probability)은 **어떤 사건이 발생할 가능성을 0과 1 사이의 수치로 나타내는 것**이다. 0은 불가능, 1은 필수 발생을 의미하며, 불확실한 상황에서 합리적인 의사결정을 돕는 수학적 도구다.

> 비유: "주사위 던지기" - 각 면이 나올 확률은 1/6이에요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 확률 기본 용어
```
┌────────────────────────────────────────────────────────┐
│           확률 기본 용어                               │
├────────────────────────────────────────────────────────┤
│                                                        │
│  📊 표본 공간 (Sample Space, S):                       │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 모든 가능한 결과의 집합                     │   │
│  │  • 예: 주사위 S = {1, 2, 3, 4, 5, 6}          │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🎯 사건 (Event, E):                                   │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 표본 공간의 부분집합                        │   │
│  │  • 예: 짝수 E = {2, 4, 6}                      │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📈 확률 (Probability, P):                             │
│  ┌────────────────────────────────────────────────┐   │
│  │  • P(E) = |E| / |S|                           │   │
│  │  • 0 ≤ P(E) ≤ 1                               │   │
│  │  • P(S) = 1                                    │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 확률 공리 (Kolmogorov)
| 공리 | 설명 | 수식 |
|-----|------|------|
| 비음수성 | 확률은 0 이상 | P(E) ≥ 0 |
| 정규화 | 전체 확률은 1 | P(S) = 1 |
| 가산성 | 서로소 사건 | P(A∪B) = P(A) + P(B) |

#### 4. 확률 종류
| 종류 | 설명 | 예시 |
|-----|------|------|
| 고전적 확률 | 동등한 가능성 | P(주사위 1) = 1/6 |
| 경험적 확률 | 관측 빈도 | 1000번 중 300번 = 0.3 |
| 주관적 확률 | 개인적 믿음 | "비 올 확률 70%" |

#### 5. 주요 확률 공식
| 공식 | 설명 |
|-----|------|
| P(A∪B) = P(A) + P(B) - P(A∩B) | 합집합 |
| P(A') = 1 - P(A) | 여사건 |
| P(A\|B) = P(A∩B) / P(B) | 조건부 확률 |
| P(A∩B) = P(A) × P(B) | 독립 사건 |
