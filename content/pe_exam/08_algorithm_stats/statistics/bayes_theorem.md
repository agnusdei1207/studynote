# 베이즈 정리 (Bayes' Theorem)

## 핵심 인사이트 (3줄 요약)
> **새로운 증거로 확률 갱신**. 사후확률 = 사전확률 × 가능도. 조건부 확률의 역 계산.

### Ⅰ. 개요

#### 1. 개념
베이즈 정리(Bayes' Theorem)는 **새로운 증거(데이터)를 바탕으로 가설의 확률을 갱신하는 방법**이다. 조건부 확률 간의 관계를 수식화한 것으로, 확률론적 추론의 기초다.

> 비유: "의사의 진단" - 증상 보고 병 확률 계산해요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 베이즈 정리 공식
```
┌────────────────────────────────────────────────────────┐
│           베이즈 정리                                 │
├────────────────────────────────────────────────────────┤
│                                                        │
│                    P(B|A) × P(A)                       │
│         P(A|B) = ─────────────────                    │
│                        P(B)                            │
│                                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │  P(A|B): 사후확률 (Posterior)                   │   │
│  │           B가 주어졌을 때 A의 확률              │   │
│  │                                                │   │
│  │  P(A): 사전확률 (Prior)                         │   │
│  │        A의 초기 확률                           │   │
│  │                                                │   │
│  │  P(B|A): 가능도 (Likelihood)                   │   │
│  │          A가 참일 때 B의 확률                  │   │
│  │                                                │   │
│  │  P(B): 증거 (Evidence)                         │   │
│  │       B의 총 확률                              │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  확장 형태:                                           │
│                    P(B|A) × P(A)                       │
│         P(A|B) = ─────────────────────────            │
│                  P(B|A)P(A) + P(B|¬A)P(¬A)            │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 확률 용어
| 용어 | 영어 | 설명 |
|-----|------|------|
| 사전확률 | Prior | 증거 전 확률 |
| 사후확률 | Posterior | 증거 후 확률 |
| 가능도 | Likelihood | 가설이 참일 때 증거 확률 |
| 증거 | Evidence | 관측된 데이터 확률 |

#### 4. 대표적 예제: 질병 진단
```
문제:
- 질병 유병률: 1% (P(질병) = 0.01)
- 검사 양성률 (질병 있을 때): 99% (P(양성|질병) = 0.99)
- 검사 위양성률 (질병 없을 때): 5% (P(양성|건강) = 0.05)

질문: 양성일 때 실제 질병일 확률?

해답:
P(질병|양성) = P(양성|질병) × P(질병) / P(양성)

P(양성) = P(양성|질병) × P(질병) + P(양성|건강) × P(건강)
        = 0.99 × 0.01 + 0.05 × 0.99
        = 0.0099 + 0.0495 = 0.0594

P(질병|양성) = 0.99 × 0.01 / 0.0594 ≈ 0.167 (16.7%)

결과: 양성이어도 실제 질병일 확률은 약 17%!
```

#### 5. 베이지안 추론 과정
```
1. 사전확률 설정
   ┌────────────────────────────────────────────────┐
   │  초기 믿음: P(가설)                            │
   └────────────────────────────────────────────────┘

2. 증거 수집
   ┌────────────────────────────────────────────────┐
   │  새로운 데이터: P(증거|가설)                   │
   └────────────────────────────────────────────────┘

3. 사후확률 계산
   ┌────────────────────────────────────────────────┐
   │  갱신된 믿음: P(가설|증거)                     │
   └────────────────────────────────────────────────┘

4. 반복 (새로운 증거로 계속 갱신)
```

#### 7. 나이브 베이즈 분류기
```
P(클래스|특징들) ∝ P(특징들|클래스) × P(클래스)

가정: 특징들이 서로 독립

예시 (스팸 분류):
P(스팸|단어들) ∝ P(단어1|스팸) × P(단어2|스팠) × ... × P(스팸)
```
