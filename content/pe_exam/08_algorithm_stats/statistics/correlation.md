### Ⅰ. 개요

#### 1. 개념
상관분석(Correlation Analysis)은 **두 변수 간의 선형적 관계의 강도와 방향을 측정하는 통계 기법**이다. 상관계수는 -1에서 +1 사이의 값을 가지며, 변수 간 인과관계가 아닌 단순한 연관성을 나타낸다.

> 비유: "키와 몸무게 관계" - 키가 크면 몸무게도 많이 나가는 경향

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 상관계수 종류
```
┌────────────────────────────────────────────────────────┐
│           상관계수 종류                                │
├────────────────────────────────────────────────────────┤
│                                                        │
│  📊 피어슨 상관계수 (Pearson r):                       │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 연속형 변수, 선형 관계                       │   │
│  │  • r = Σ(x-x̄)(y-ȳ) / √[Σ(x-x̄)²Σ(y-ȳ)²]      │   │
│  │  • 정규분포 가정                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📈 스피어만 순위상관 (Spearman ρ):                    │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 순위형 데이터                                │   │
│  │  • 비선형 단조 관계                             │   │
│  │  • 이상값에 강건                                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📉 켄달의 타우 (Kendall's τ):                         │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 순위 일치/불일치 기반                        │   │
│  │  • 소표본에 적합                                │   │
│  │  • 동순위 처리 우수                             │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🎯 상관계수 해석:                                     │
│  ┌────────────────────────────────────────────────┐   │
│  │  |r| ≥ 0.7: 강한 상관                          │   │
│  │  0.3 ≤ |r| < 0.7: 중간 상관                    │   │
│  │  |r| < 0.3: 약한 상관                          │   │
│  │                                                 │   │
│  │  r > 0: 양의 상관 (같이 증가)                  │   │
│  │  r < 0: 음의 상관 (반대)                       │   │
│  │  r = 0: 무상관                                 │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 4. 결정계수 (R²)
| R² 값 | 설명 |
|-------|------|
| 0.9+ | 매우 강한 설명력 |
| 0.7-0.9 | 강한 설명력 |
| 0.5-0.7 | 중간 설명력 |
| 0.3-0.5 | 약한 설명력 |
| < 0.3 | 설명력 낮음 |

#### 5. 주의사항
| 오해 | 진실 |
|-----|------|
| 상관=인과 | 인과 아님 |
| 높은 r=중요 | 실제 의미 확인 |
| 선형만 측정 | 비선형 관계 놓침 |
| 이상값 무시 | 결과 왜곡 가능 |

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **알고리즘 효율** | 최적 알고리즘 적용으로 복잡도 대폭 감소 | O(n²) → O(n log n) 수준 개선 |
