### Ⅰ. 개요

#### 1. 개념
회귀 분석(Regression Analysis)은 **하나 이상의 독립변수(X)와 종속변수(Y) 사이의 관계를 수학적 모델로 표현하고, 이를 통해 Y값을 예측하는 통계 기법**이다.

> 비유: "키로 몸무게 추측하기" - 키가 크면 몸무게도 많이 나가요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 회귀 분석 분류
```
┌────────────────────────────────────────────────────────┐
│           회귀 분석 분류                               │
├────────────────────────────────────────────────────────┤
│                                                        │
│  📊 단순 회귀 (Simple Regression):                     │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 독립변수 1개                                 │   │
│  │  • Y = β₀ + β₁X + ε                            │   │
│  │  • 직선 관계 모델링                             │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📈 다중 회귀 (Multiple Regression):                   │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 독립변수 여러 개                             │   │
│  │  • Y = β₀ + β₁X₁ + β₂X₂ + ... + ε             │   │
│  │  • 여러 요인의 영향 분석                        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🔀 비선형 회귀 (Non-linear Regression):               │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 곡선 관계                                    │   │
│  │  • 다항식, 지수함수 등                          │   │
│  │  • Y = β₀ + β₁X + β₂X² + ...                   │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📋 로지스틱 회귀 (Logistic Regression):               │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 종속변수가 범주형 (0/1)                      │   │
│  │  • P(Y=1) = 1 / (1 + e^(-z))                   │   │
│  │  • 분류 문제에 활용                             │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 최소제곱법 (OLS)
| 공식 | 설명 |
|-----|------|
| β₁ = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)² | 기울기 |
| β₀ = ȳ - β₁x̄ | 절편 |
| R² = 1 - SS_res / SS_tot | 결정계수 |

#### 4. 회귀 진단
| 지표 | 의미 | 좋은 값 |
|-----|------|--------|
| R² | 설명력 | 1에 가까울수록 |
| Adj R² | 조정 설명력 | 높을수록 |
| F-통계량 | 모형 유의성 | p < 0.05 |
| t-통계량 | 계수 유의성 | p < 0.05 |
| RMSE | 예측 오차 | 낮을수록 |

#### 5. 가정 검토
| 가정 | 설명 |
|-----|------|
| 선형성 | X와 Y의 선형 관계 |
| 독립성 | 오차의 독립성 |
| 정규성 | 오차의 정규분포 |
| 등분산성 | 오차 분산 일정 |
