### Ⅰ. 개요

#### 1. 개념
가설 검정(Hypothesis Testing)은 **표본 데이터를 바탕으로 모집단에 대한 주장(가설)이 참인지 거짓인지를 통계적으로 판단하는 방법**이다. 확률적 오류를 인정하면서 과학적 결론을 도출한다.

> 비유: "법정 판결" - 무죄 추정, 증거로 유죄 입증

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 가설 검정 절차
```
┌────────────────────────────────────────────────────────┐
│           가설 검정 5단계                              │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1️⃣ 가설 설정:                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 귀무가설 H₀: 차이 없음, 효과 없음            │   │
│  │  • 대립가설 H₁: 차이 있음, 효과 있음            │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2️⃣ 유의수준 설정:                                    │
│  ┌────────────────────────────────────────────────┐   │
│  │  • α = 0.05 (5%) 또는 0.01 (1%)                │   │
│  │  • 제1종 오류 허용 확률                         │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3️⃣ 검정통계량 계산:                                  │
│  ┌────────────────────────────────────────────────┐   │
│  │  • z-검정, t-검정, χ²-검정 등                  │   │
│  │  • 표본 데이터로 계산                           │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  4️⃣ p-value 계산:                                     │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 귀무가설 하에서 관측값 이상 극단적일 확률    │   │
│  │  • p < α → H₀ 기각                             │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  5️⃣ 결론 도출:                                        │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 기각: 대립가설 채택                          │   │
│  │  • 채택: 귀무가설 기각 불가                     │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 오류 유형
| 구분 | H₀ 참 | H₀ 거짓 |
|-----|-------|--------|
| H₀ 기각 | **제1종 오류 (α)** | 정상 |
| H₀ 채택 | 정상 | **제2종 오류 (β)** |

- **제1종 오류**: 참인 귀무가설을 기각 (거짓 양성)
- **제2종 오류**: 거짓인 귀무가설을 채택 (거짓 음성)
- **검정력 (Power)**: 1 - β, 참인 대립가설 채택 확률

---

### Ⅲ. 기술 비교 분석

#### 4. 검정 방법 선택
| 조건 | 모분산 | 검정법 |
|-----|--------|--------|
| n ≥ 30 | 알면/모름 | z-검정 |
| n < 30 | 앎 | z-검정 |
| n < 30 | 모름 | t-검정 |
| 비율 | - | z-검정 |
| 분산 | - | χ²-검정 |

#### 5. 단측 vs 양측 검정
| 유형 | 대립가설 | 용도 |
|-----|----------|------|
| 양측 | H₁: μ ≠ μ₀ | 차이 존재 |
| 좌측 | H₁: μ < μ₀ | 감소 |
| 우측 | H₁: μ > μ₀ | 증가 |

#### 6. 장단점
| 장점 | 단점 |
|-----|------|
| 객관적 의사결정 | p-value 오해 가능 |
| 과학적 근거 | 표본 크기 민감 |
| 재현 가능 | 유의수준 임의성 |

### Ⅴ. 기대 효과 및 결론


| 효과 영역 | 내용 | 정량적 목표 |
|---------|-----|-----------|
| **알고리즘 효율** | 최적 알고리즘 적용으로 복잡도 대폭 감소 | O(n²) → O(n log n) 수준 개선 |
| **시스템 성능** | 빠른 자료구조·탐색 알고리즘으로 응답 시간 단축 | 대용량 데이터 처리 10배 향상 |
| **의사결정 품질** | 통계적 검증으로 신뢰 있는 데이터 기반 판단 제공 | 의사결정 오류율 50% 감소 |
