### Ⅰ. 개요

#### 1. 개념
확률 분포(Probability Distribution)는 **확률변수가 취할 수 있는 모든 값과 그에 대응하는 확률 간의 관계를 나타내는 함수**이다. 데이터의 패턴을 이해하고 미래를 예측하는 기초가 된다.

> 비유: "키 분포" - 사람들 키가 어떻게 퍼져 있는지 그래프로 보여요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 확률 분포 분류
```
┌────────────────────────────────────────────────────────┐
│           확률 분포 분류                               │
├────────────────────────────────────────────────────────┤
│                                                        │
│  📊 이산형 분포 (Discrete):                            │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 확률변수가 셀 수 있는 값                     │   │
│  │  • 확률질량함수 (PMF)                           │   │
│  │  • 베르누이, 이항, 포아송, 기하                 │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📈 연속형 분포 (Continuous):                          │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 확률변수가 연속적인 값                       │   │
│  │  • 확률밀도함수 (PDF)                           │   │
│  │  • 정규, 지수, 균등, t, 카이제곱                │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🔑 핵심 특성값:                                       │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 기댓값 (Expectation): E[X] = Σx·P(x)       │   │
│  │  • 분산 (Variance): Var(X) = E[(X-μ)²]        │   │
│  │  • 표준편차 (Std Dev): σ = √Var(X)            │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 주요 이산 분포
| 분포 | 조건 | 확률질량함수 | 기댓값 | 분산 |
|-----|------|-------------|--------|------|
| 베르누이 | 성공/실패 | P(X=k) = p^k(1-p)^(1-k) | p | p(1-p) |
| 이항 | n번 시도 | P(X=k) = C(n,k)p^k(1-p)^(n-k) | np | np(1-p) |
| 포아송 | 단위시간 사건 | P(X=k) = λ^k·e^(-λ)/k! | λ | λ |
| 기하 | 첫 성공 | P(X=k) = (1-p)^(k-1)·p | 1/p | (1-p)/p² |

#### 4. 주요 연속 분포
| 분포 | 용도 | PDF | 기댓값 | 분산 |
|-----|------|-----|--------|------|
| 정규 | 자연현상 | (1/σ√2π)e^(-(x-μ)²/2σ²) | μ | σ² |
| 지수 | 대기시간 | λe^(-λx) | 1/λ | 1/λ² |
| 균등 | 범위 내 동등 | 1/(b-a) | (a+b)/2 | (b-a)²/12 |
| t | 소표본 | 복잡함 | 0 (ν>1) | ν/(ν-2) |

#### 5. 정규 분포 특징
| 특징 | 설명 |
|-----|------|
| 종 모양 | 좌우 대칭 |
| 68-95-99.7 규칙 | μ±σ, μ±2σ, μ±3σ |
| 중심극한정리 | 표본평균 → 정규분포 |
| 표준화 | Z = (X-μ)/σ |
