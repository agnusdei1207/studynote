### Ⅰ. 개요

#### 1. 개념
트리(Tree)는 **노드(Node)와 간선(Edge)으로 구성된 계층적 비선형 자료구조**로, 하나의 루트 노드에서 시작하여 자식 노드로 확장되는 구조를 가진다. 사이클이 없는 연결 그래프의 일종이다.

> 비유: "가계도" - 조상에서 자손으로 뻗어나가요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 트리 용어
```
┌────────────────────────────────────────────────────────┐
│           트리 구조와 용어                             │
├────────────────────────────────────────────────────────┤
│                                                        │
│                    ┌───┐                               │
│                    │ A │  ← 루트 (Root)               │
│                    └─┬─┘                               │
│               ┌──────┼──────┐                         │
│             ┌─┴─┐  ┌─┴─┐  ┌─┴─┐                       │
│             │ B │  │ C │  │ D │  ← 내부 노드          │
│             └─┬─┘  └─┬─┘  └─┬─┘    (Internal Node)   │
│           ┌───┴───┐   │  ┌──┴──┐                     │
│         ┌─┴─┐   ┌─┴─┐ │ ┌┴─┐ ┌─┴─┐                   │
│         │ E │   │ F │ │ │G │ │ H │  ← 리프 (Leaf)    │
│         └───┘   └───┘ │ └──┘ └───┘                   │
│                         │                              │
│                       ┌─┴─┐                           │
│                       │ I │                           │
│                       └───┘                           │
│                                                        │
│  📊 용어 정리:                                         │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 깊이 (Depth): 루트에서 해당 노드까지 거리   │   │
│  │  • 높이 (Height): 노드에서 리프까지 최장 거리  │   │
│  │  • 차수 (Degree): 자식 노드의 개수             │   │
│  │  • 레벨 (Level): 루트가 레벨 1                 │   │
│  │  • 형제 (Sibling): 같은 부모를 가진 노드       │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 트리 종류
| 종류 | 특징 | 용도 |
|-----|------|------|
| 이진 트리 | 자식 최대 2개 | 기본 구조 |
| 포화 이진 트리 | 모든 리프가 같은 레벨 | 힙 |
| 완전 이진 트리 | 왼쪽부터 채움 | 힙, 우선순위큐 |
| BST | 좌 < 루트 < 우 | 검색 |
| AVL | 균형 BST | 검색 최적화 |
| 레드블랙 | 색상 기반 균형 | 연관배열 |
| B-트리 | 다진 균형 트리 | DB 인덱스 |

#### 4. 이진 트리 순회
| 순회 | 순서 | 용도 |
|-----|------|------|
| 전위 (Preorder) | 루트→좌→우 | 복사, 출력 |
| 중위 (Inorder) | 좌→루트→우 | 정렬 (BST) |
| 후위 (Postorder) | 좌→우→루트 | 삭제, 계산 |
| 레벨 (Level) | 레벨 순서 | BFS |

#### 5. 트리 연산 복잡도
| 연산 | 일반 | BST (평균) | BST (최악) | AVL |
|-----|------|-----------|-----------|-----|
| 검색 | O(n) | O(log n) | O(n) | O(log n) |
| 삽입 | O(n) | O(log n) | O(n) | O(log n) |
| 삭제 | O(n) | O(log n) | O(n) | O(log n) |
