### Ⅰ. 개요

#### 1. 개념
탐색 알고리즘(Searching Algorithm)은 **데이터 집합에서 특정 값이나 조건을 만족하는 요소를 찾는 알고리즘**이다. 데이터 구조와 정렬 여부에 따라 적절한 알고리즘을 선택한다.

> 비유: "도서관에서 책 찾기" - 어디 있는지 찾아요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 탐색 알고리즘 분류
```
┌────────────────────────────────────────────────────────┐
│           탐색 알고리즘 분류                          │
├────────────────────────────────────────────────────────┤
│                                                        │
│  📊 배열 기반 탐색:                                    │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 선형 탐색 (Linear Search) - O(n)            │   │
│  │  • 이진 탐색 (Binary Search) - O(log n)        │   │
│  │  • 보간 탐색 (Interpolation Search)            │   │
│  │  • 지수 탐색 (Exponential Search)              │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🌲 트리 기반 탐색:                                    │
│  ┌────────────────────────────────────────────────┐   │
│  │  • BST 탐색 - O(log n) 평균                    │   │
│  │  • AVL/레드블랙 트리 - O(log n) 보장           │   │
│  │  • B-트리 탐색 - DB 인덱스                     │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🔑 해시 기반 탐색:                                    │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 해시 테이블 - O(1) 평균                      │   │
│  │  • 체이닝 / 오픈 어드레싱                       │   │
│  │  • 완전 해시 / 쿠쿠 해시                        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🔍 문자열 탐색:                                       │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 브루트 포스 - O(nm)                         │   │
│  │  • KMP 알고리즘 - O(n+m)                        │   │
│  │  • 보이어-무어 - 평균 O(n/m)                    │   │
│  │  • 라빈-카프 - O(n+m) 평균                      │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 탐색 알고리즘 비교
| 알고리즘 | 시간복잡도 | 공간복잡도 | 조건 |
|---------|-----------|-----------|------|
| 선형 탐색 | O(n) | O(1) | 없음 |
| 이진 탐색 | O(log n) | O(1) | 정렬됨 |
| 해시 탐색 | O(1) 평균 | O(n) | 해시함수 |
| BST | O(log n) 평균 | O(n) | 트리 |
| B-트리 | O(log n) | O(n) | 디스크 |

#### 4. 이진 탐색 변형
| 변형 | 용도 |
|-----|------|
| lower_bound | 이상인 첫 위치 |
| upper_bound | 초과인 첫 위치 |
| 회전 배열 탐색 | 정렬 후 회전된 배열 |
| 2D 행렬 탐색 | 행렬에서 값 찾기 |

### Ⅲ. 기술 비교 분석

#### 5. 장단점
| 알고리즘 | 장점 | 단점 |
|---------|------|------|
| 선형 | 간단, 조건 없음 | 느림 |
| 이진 | 빠름 | 정렬 필요 |
| 해시 | 가장 빠름 | 충돌, 공간 |
| BST | 순회 가능 | 균형 필요 |
