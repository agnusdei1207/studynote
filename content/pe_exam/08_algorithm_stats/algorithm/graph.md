### Ⅰ. 개요

#### 1. 개념
그래프 알고리즘(Graph Algorithm)은 **정점(Vertex)과 간선(Edge)으로 구성된 그래프 자료구조에서 탐색, 경로 찾기, 연결성 분석 등을 수행하는 알고리즘**이다. 소셜 네트워크, 지도, 웹 등에 활용된다.

> 비유: "지도와 도로" - 장소와 길로 연결돼요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 그래프 알고리즘 분류
```
┌────────────────────────────────────────────────────────┐
│           그래프 알고리즘 분류                        │
├────────────────────────────────────────────────────────┤
│                                                        │
│  🔍 그래프 탐색:                                       │
│  ┌────────────────────────────────────────────────┐   │
│  │  • DFS (깊이 우선 탐색)                         │   │
│  │  • BFS (너비 우선 탐색)                         │   │
│  │  • 시간복잡도: O(V+E)                           │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🛤️ 최단 경로:                                         │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 다익스트라 (Dijkstra) - O(E log V)           │   │
│  │  • 벨만-포드 (Bellman-Ford) - O(VE)            │   │
│  │  • 플로이드-워셜 (Floyd-Warshall) - O(V³)       │   │
│  │  • A* 알고리즘                                  │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🌲 최소 신장 트리 (MST):                              │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 크루스칼 (Kruskal) - O(E log E)              │   │
│  │  • 프림 (Prim) - O(E log V)                     │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🔗 연결성:                                            │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 위상 정렬 (Topological Sort)                 │   │
│  │  • 강연결요소 (SCC)                             │   │
│  │  • 이중 연결 요소                               │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🔄 네트워크 플로우:                                   │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 포드-풀커슨 (Ford-Fulkerson)                │   │
│  │  • 에드몬드-카프 (Edmonds-Karp)                │   │
│  │  • 이분 매칭                                   │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 최단 경로 알고리즘 비교
| 알고리즘 | 가중치 | 음수 허용 | 시간복잡도 |
|---------|--------|----------|-----------|
| BFS | 균등 | X | O(V+E) |
| 다익스트라 | 양수 | X | O(E log V) |
| 벨만-포드 | 모두 | O | O(VE) |
| 플로이드-워셜 | 모두 | O | O(V³) |

#### 4. MST 알고리즘
| 구분 | 크루스칼 | 프림 |
|-----|---------|------|
| 방식 | 간선 선택 | 정점 확장 |
| 자료구조 | Union-Find | 우선순위큐 |
| 적합 | 희소 그래프 | 밀집 그래프 |
