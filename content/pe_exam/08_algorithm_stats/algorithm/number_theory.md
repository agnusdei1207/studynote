### Ⅰ. 개요

#### 1. 개념
수리 알고리즘(Number Theory Algorithms)은 **정수의 성질을 이용한 수학적 계산 알고리즘**으로, 암호학, 해시 함수, 난수 생성 등 다양한 분야에서 활용된다.

> 비유: "수학 공식" - 숫자의 비밀을 풀어요

#### 3. 알고리즘 복잡도
| 알고리즘 | 시간복잡도 | 공간복잡도 |
|---------|-----------|-----------|
| 유클리드 호제법 | O(log min(a,b)) | O(1) |
| 소수 판별 | O(√n) | O(1) |
| 에라토스테네스 체 | O(n log log n) | O(n) |
| 모듈러 거듭제곱 | O(log n) | O(1) |
| 확장 유클리드 | O(log min(a,b)) | O(1) |

#### 4. 수학적 성질
| 성질 | 설명 |
|-----|------|
| gcd(a,b)×lcm(a,b) = a×b | GCD와 LCM 관계 |
| a^φ(n) ≡ 1 (mod n) | 오일러 정리 |
| a^(p-1) ≡ 1 (mod p) | 페르마 소정리 |
| (a×b) mod n = ((a mod n)×(b mod n)) mod n | 모듈러 곱셈 |

#### 5. 응용 분야
| 분야 | 활용 |
|-----|------|
| 암호학 | RSA, ECC |
| 해시 | 체크섬 |
| 난수 | 의사난수 생성 |
| 코딩테스트 | 수학 문제 |
