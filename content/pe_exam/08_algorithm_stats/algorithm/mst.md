# 최소 신장 트리 (MST) - Prim vs Kruskal 알고리즘

## 핵심 인사이트 (3줄 요약)
> **모든 정점을 연결하되 합계 가중치가 최소가 되는 트리**. Kruskal은 간선을 가중치 순으로 정렬(Greedy), Prim은 정점 중심으로 확장(Dijkstra와 유사). 간선이 많으면 Prim, 간선이 적으면 Kruskal이 유리하다.

### Ⅰ. 개요

#### 1. 개념 및 정의
```
신장 트리 (Spanning Tree):
- 그래프 내 모든 정점을 포함하는 트리
- 정점 수 V, 간선 수 V-1 (Cycle 없음)
- 하나의 그래프에 여러 개의 신장 트리 존재 가능

최소 신장 트리 (MST, Minimum Spanning Tree):
- 신장 트리 중 간선 가중치 합이 최소인 트리
- 네트워크 통신망 구축, 전력망 연결 등 최적화 비용 절감
```

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. Prim 알고리즘 (정점 중심) ★ (중요)
```
원리: "가까운 정점을 하나씩 더한다"
1. 임의의 정점을 선택해 집합 S에 추가
2. 집합 S와 연결된 간선 중 가중치가 가장 작은 {S 내부, S 외부} 간선 선택
3. 선택된 간선에 연결된 정점을 S에 추가 (탐욕적 선택)
4. 모든 정점이 포함될 때까지(V-1개 간선) 반복

자료구조: Priority Queue (우선순위 큐)
시간복잡도: O(E log V)
특징: 시작 정점이 필요함, 데이터가 밀집된 그래프(Dense Graph)에 유리
```

#### 3. Kruskal 알고리즘 (간선 중심) ★ (중요)
```
원리: "가장 저렴한 간선부터 일단 고른다 (단, 사이클 생기면 버림)"
1. 그래프의 모든 간선을 가중치 오름차순으로 정렬
2. 가중치가 가장 작은 간선부터 순차적으로 그래프에 추가 시도
3. (핵심) 간선을 추가했을 때 사이클(Cycle)이 발생하면 추가하지 않음
4. V-1개의 간선이 선택될 때까지 반복

자료구조: Union-Find (Disjoint Set) - 사이클 판별용
시간복잡도: O(E log E) 또는 O(E log V)
특징: 간선 정렬이 핵심, 데이터가 희소한 그래프(Sparse Graph)에 유리
```

#### 4. Prim vs Kruskal 비교 ★ (기술사 핵심)
| 구분 | Prim 알고리즘 | Kruskal 알고리즘 |
|------|--------------|-----------------|
| 방식 | 정점 중심 (Vertex-based) | 간선 중심 (Edge-based) |
| 선택 기준 | 현재 트리에서 가장 가까운 정점 | 가중치가 가장 작은 간선 |
| 핵심 자료구조 | Priority Queue (Min-Heap) | Union-Find (Disjoint Set) |
| 정렬 필요 여부 | 불필요 | **간선 정렬 필요** (O(E log E)) |
| 사이클 판별 | 트리 구성 방식상 자동 방지 | Union-Find로 명시적 판별 |
| 유리한 그래프 | **밀집 그래프 (Dense)** | **희소 그래프 (Sparse)** |
| 복잡도 | O(E log V) | O(E log V) *E가 V^2이면 유사* |

#### 5. 알고리즘 시뮬레이션
```
그래프 G:
A-B(1), A-C(3), B-C(2), B-D(4), C-D(5)

Kruskal 시뮬레이션:
1. 정렬: A-B(1), B-C(2), A-C(3), B-D(4), C-D(5)
2. A-B(1) 선택 (Union(A,B))
3. B-C(2) 선택 (Union(B,C))
4. A-C(3) 선택 시도 → A-B-C-A 사이클 발생! (버림)
5. B-D(4) 선택 (Union(B,D))
6. V-1(3)개 간선 완성!
결과: A-B-C-D 연결 (합계 7)
```

#### 6. 최소 신장 트리의 속성 (성질)
- **Cut Property**: 어떤 컷에 대해 가중치가 가장 작은 간선은 무조건 MST에 포함됨
- **Unique MST**: 모든 간선 가중치가 다르면 MST는 유일하게 하나 존재
- **Cycle**: MST 내의 임의의 두 정점 간 경로에 새로운 간선을 추가하면 사이클이 형성됨
