### Ⅰ. 개요

#### 1. 개념
문자열 알고리즘(String Algorithms)은 **문자열의 검색, 매칭, 변환, 분석을 위한 알고리즘**으로, 텍스트 처리, 검색 엔진, 생물정보학 등 다양한 분야에서 활용된다.

> 비유: "텍스트 찾기" - Ctrl+F 기능의 원리예요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 패턴 매칭 알고리즘
```
┌────────────────────────────────────────────────────────┐
│           패턴 매칭 알고리즘                           │
├────────────────────────────────────────────────────────┤
│                                                        │
│  🔍 브루트 포스 (Brute Force):                         │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 모든 위치에서 패턴 비교                     │   │
│  │  • 시간복잡도: O(n×m)                          │   │
│  │  • 구현 간단, 비효율                           │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  ⚡ KMP (Knuth-Morris-Pratt):                          │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 실패 함수(LPS) 활용                         │   │
│  │  • 시간복잡도: O(n+m)                          │   │
│  │  • 불필요한 비교 건너뜀                        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  🎲 라빈-카프 (Rabin-Karp):                            │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 해시 기반 매칭                               │   │
│  │  • 평균 O(n+m), 최악 O(n×m)                    │   │
│  │  • 다중 패턴 검색에 유리                        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📊 보이어-무어 (Boyer-Moore):                         │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 오른쪽→왼쪽 비교                             │   │
│  │  • 평균 O(n/m), 최악 O(n×m)                    │   │
│  │  • 실제로 가장 빠름                             │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 알고리즘 비교
| 알고리즘 | 최선 | 평균 | 최악 | 공간 |
|---------|------|------|------|------|
| 브루트포스 | O(n×m) | O(n×m) | O(n×m) | O(1) |
| KMP | O(n+m) | O(n+m) | O(n+m) | O(m) |
| 라빈-카프 | O(n+m) | O(n+m) | O(n×m) | O(1) |
| 보이어-무어 | O(n/m) | O(n+m) | O(n×m) | O(k) |

#### 4. 응용 분야
| 분야 | 활용 |
|-----|------|
| 검색 엔진 | 키워드 검색 |
| 텍스트 에디터 | 찾기/바꾸기 |
| 생물정보학 | DNA 서열 분석 |
| 컴파일러 | 어휘 분석 |
| 보안 | 침입 탐지 |

#### 5. 접미사 구조
| 구조 | 구축 | 검색 | 용도 |
|-----|------|------|------|
| 접미사 트리 | O(n) | O(m) | 반복 패턴 |
| 접미사 배열 | O(n log n) | O(m log n) | 정렬된 접미사 |
| LCP 배열 | O(n) | - | 공통 접두사 |
