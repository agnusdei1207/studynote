# 분할 정복 (Divide and Conquer)

### Ⅰ. 개요

#### 1. 개념
분할 정복(Divide and Conquer)은 **큰 문제를 여러 개의 작은 문제로 나누어 해결한 후, 그 결과를 결합하여 전체 문제를 해결하는 알고리즘 설계 기법**이다.

> 비유: "큰 퍼즐을 작은 조각으로 나누어 맞추기" - 나누고 합쳐요

---

### Ⅱ. 구성 요소 및 핵심 원리

#### 2. 분할 정복 단계
```
┌────────────────────────────────────────────────────────┐
│           분할 정복 3단계                              │
├────────────────────────────────────────────────────────┤
│                                                        │
│  1️⃣ 분할 (Divide):                                    │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 문제를 작은 하위 문제로 분할                 │   │
│  │  • 하위 문제는 원래 문제와 동일한 구조          │   │
│  │  • 보통 크기를 반으로 나눔                     │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  2️⃣ 정복 (Conquer):                                   │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 하위 문제를 재귀적으로 해결                  │   │
│  │  • 충분히 작으면 직접 해결                     │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  3️⃣ 결합 (Combine):                                   │
│  ┌────────────────────────────────────────────────┐   │
│  │  • 하위 문제의 해를 결합                        │   │
│  │  • 전체 문제의 해를 구성                        │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
│  📊 시간 복잡도 분석:                                  │
│  ┌────────────────────────────────────────────────┐   │
│  │  T(n) = aT(n/b) + f(n)                         │   │
│  │  • a: 하위 문제 개수                            │   │
│  │  • b: 하위 문제 크기 비율                       │   │
│  │  • f(n): 분할/결합 비용                         │   │
│  └────────────────────────────────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

#### 3. 대표 알고리즘
| 알고리즘 | 분할 방식 | 시간복잡도 | 결합 비용 |
|---------|----------|-----------|----------|
| 병합 정렬 | 반반 분할 | O(n log n) | O(n) |
| 퀵 정렬 | 피벗 기준 | O(n log n) 평균 | O(n) |
| 이진 탐색 | 반반 분할 | O(log n) | O(1) |
| 스트라센 | 2×2 → 7개 | O(n^2.81) | O(n²) |
| FFT | 짝수/홀수 | O(n log n) | O(n) |
| 카라츠바 | 3개 부분 | O(n^1.58) | O(n) |

#### 4. Master Theorem
| 경우 | 조건 | 시간복잡도 |
|-----|------|-----------|
| 1 | f(n) < n^(log_b a) | O(n^(log_b a)) |
| 2 | f(n) = n^(log_b a) | O(n^(log_b a) * log n) |
| 3 | f(n) > n^(log_b a) | O(f(n)) |
